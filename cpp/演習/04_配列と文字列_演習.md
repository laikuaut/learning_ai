## 問題1：配列の最大値・最小値を求めよう（基本）

整数の配列（要素数10）が与えられたとき、最大値・最小値・平均値を求めて表示するプログラムを作成してください。

- 配列の値はプログラム内で初期化してかまいません
- 平均値は小数点以下1桁で表示してください

### 期待される出力例

```
配列: 34 78 12 95 23 67 45 89 56 41
要素数: 10
最大値: 95
最小値: 12
合計: 540
平均値: 54.0
```

<details>
<summary>ヒント</summary>

- 最大値用・最小値用の変数を配列の最初の要素で初期化します
- ループで各要素と比較し、必要に応じて更新します
- `std::array<int, 10>` もしくは C スタイルの `int arr[10]` どちらでもOKです
- 範囲 for 文（`for (int x : arr)`）を使うとシンプルに書けます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>
#include <array>

int main() {
    // std::array で配列を定義（C++11以降）
    std::array<int, 10> arr = {34, 78, 12, 95, 23, 67, 45, 89, 56, 41};

    // 配列の内容を表示
    std::cout << "配列: ";
    for (int x : arr) {
        std::cout << x << " ";
    }
    std::cout << std::endl;

    // 最大値・最小値・合計の計算
    int max_val = arr[0]; // 最初の要素で初期化
    int min_val = arr[0];
    int sum = 0;

    for (int x : arr) {
        if (x > max_val) max_val = x; // より大きい値があれば更新
        if (x < min_val) min_val = x; // より小さい値があれば更新
        sum += x;
    }

    // 平均値の計算（doubleにキャストして小数点以下も計算）
    double average = static_cast<double>(sum) / arr.size();

    // 結果表示
    std::cout << "要素数: " << arr.size() << std::endl;
    std::cout << "最大値: " << max_val << std::endl;
    std::cout << "最小値: " << min_val << std::endl;
    std::cout << "合計: " << sum << std::endl;
    std::cout << std::fixed << std::setprecision(1);
    std::cout << "平均値: " << average << std::endl;

    return 0;
}
```

</details>

## 問題2：配列を逆順に並べ替えよう（基本）

`std::vector<int>` に格納された整数を逆順に並べ替えて表示するプログラムを作成してください。

- **標準ライブラリの `reverse` 関数は使わず**、自力でスワップ処理を実装してください
- 元の配列と逆順後の配列の両方を表示してください

### 期待される出力例

```
元の配列: 1 2 3 4 5 6 7 8
逆順後:   8 7 6 5 4 3 2 1
```

<details>
<summary>ヒント</summary>

- 先頭と末尾から同時にインデックスを動かし、要素を交換（スワップ）していきます
- `left` を 0 から増やし、`right` を `size - 1` から減らします
- `left < right` の間だけ交換を続けます
- 一時変数 `temp` を使って2つの値を入れ替えます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <vector>

int main() {
    std::vector<int> vec = {1, 2, 3, 4, 5, 6, 7, 8};

    // 元の配列を表示
    std::cout << "元の配列: ";
    for (int x : vec) {
        std::cout << x << " ";
    }
    std::cout << std::endl;

    // 逆順に並べ替え（先頭と末尾から交換）
    int left = 0;
    int right = static_cast<int>(vec.size()) - 1;

    while (left < right) {
        // 一時変数を使ってスワップ
        int temp = vec[left];
        vec[left] = vec[right];
        vec[right] = temp;

        left++;
        right--;
    }

    // 逆順後の配列を表示
    std::cout << "逆順後:   ";
    for (int x : vec) {
        std::cout << x << " ";
    }
    std::cout << std::endl;

    return 0;
}
```

</details>

## 問題3：回文判定プログラム（応用）

文字列を入力として受け取り、その文字列が回文（前から読んでも後ろから読んでも同じ）かどうかを判定するプログラムを作成してください。

- 大文字・小文字は区別しない（`"Aba"` も回文とする）
- スペースは無視する（`"a man a plan a canal panama"` は回文とみなす場合はチャレンジ問題として取り組んでみましょう）

### 期待される出力例

```
文字列を入力してください: level
"level" は回文です
```

```
文字列を入力してください: hello
"hello" は回文ではありません
```

```
文字列を入力してください: Madam
"Madam" は回文です
```

<details>
<summary>ヒント</summary>

- `std::string` の `size()` メソッドで文字列の長さを取得できます
- `tolower(c)` で文字を小文字に変換できます（`#include <cctype>`）
- 先頭と末尾から比較していく方法が効率的です
- `std::getline(std::cin, str)` を使うとスペースを含む文字列も入力できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
#include <cctype>

int main() {
    std::string input;
    std::cout << "文字列を入力してください: ";
    std::getline(std::cin, input);

    // 小文字に変換したコピーを作る
    std::string lower_str = input;
    for (char& c : lower_str) {
        c = static_cast<char>(std::tolower(c));
    }

    // 回文判定（先頭と末尾から比較）
    bool is_palindrome = true;
    int left = 0;
    int right = static_cast<int>(lower_str.size()) - 1;

    while (left < right) {
        if (lower_str[left] != lower_str[right]) {
            is_palindrome = false;
            break; // 不一致が見つかった時点で判定終了
        }
        left++;
        right--;
    }

    // 結果表示
    if (is_palindrome) {
        std::cout << "\"" << input << "\" は回文です" << std::endl;
    } else {
        std::cout << "\"" << input << "\" は回文ではありません" << std::endl;
    }

    return 0;
}
```

</details>

## 問題4：単語カウントプログラム（応用）

文章（1行の文字列）を入力として受け取り、以下の情報を表示するプログラムを作成してください。

- 文字数（スペースを含む）
- 文字数（スペースを含まない）
- 単語数（スペースで区切られた単語の数）

### 期待される出力例

```
文章を入力してください: Hello World C++ Programming
--- 分析結果 ---
文字数（スペース込み）: 27
文字数（スペース除く）: 24
単語数: 4
```

```
文章を入力してください: I love C++
--- 分析結果 ---
文字数（スペース込み）: 10
文字数（スペース除く）: 8
単語数: 3
```

<details>
<summary>ヒント</summary>

- `std::getline(std::cin, str)` でスペースを含む1行全体を入力できます
- `str.size()` で全体の文字数がわかります
- ループで各文字をチェックし、スペース（`' '`）の個数を数えると、スペースなしの文字数が求められます
- 単語数はスペースの個数 + 1 で概算できますが、連続するスペースがある場合は工夫が必要です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>

int main() {
    std::string text;
    std::cout << "文章を入力してください: ";
    std::getline(std::cin, text);

    int total_chars = static_cast<int>(text.size()); // スペース込みの文字数
    int space_count = 0;   // スペースの数
    int word_count = 0;    // 単語数
    bool in_word = false;  // 現在単語の中にいるかどうか

    for (char c : text) {
        if (c == ' ') {
            space_count++;
            in_word = false;
        } else {
            // スペース以外の文字に出会い、かつ直前が単語の外なら新しい単語
            if (!in_word) {
                word_count++;
                in_word = true;
            }
        }
    }

    int non_space_chars = total_chars - space_count;

    // 結果表示
    std::cout << "--- 分析結果 ---" << std::endl;
    std::cout << "文字数（スペース込み）: " << total_chars << std::endl;
    std::cout << "文字数（スペース除く）: " << non_space_chars << std::endl;
    std::cout << "単語数: " << word_count << std::endl;

    return 0;
}
```

</details>

## 問題5：成績管理プログラム（チャレンジ）

`std::vector` を使って、複数の生徒のテスト成績を管理するプログラムを作成してください。

**機能：**
1. 生徒の名前と点数を複数入力（「end」と入力したら入力終了）
2. 全生徒の情報を一覧表示
3. 平均点、最高点の生徒、最低点の生徒を表示

### 期待される出力例

```
=== 成績管理システム ===
生徒の名前と点数を入力してください（終了するには「end」と入力）

名前: 田中
点数: 85

名前: 佐藤
点数: 92

名前: 鈴木
点数: 78

名前: 高橋
点数: 95

名前: end

--- 成績一覧 ---
1. 田中: 85点
2. 佐藤: 92点
3. 鈴木: 78点
4. 高橋: 95点

--- 統計情報 ---
生徒数: 4人
平均点: 87.5点
最高点: 高橋（95点）
最低点: 鈴木（78点）
```

<details>
<summary>ヒント</summary>

- 名前用の `std::vector<std::string>` と点数用の `std::vector<int>` を並行して使います
- `while` ループで「end」が入力されるまで繰り返します
- 最高点・最低点はインデックスを記録しておくと、対応する名前を取得できます
- `push_back()` でベクターに要素を追加できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
#include <vector>
#include <iomanip>

int main() {
    // 名前と点数を格納するベクター
    std::vector<std::string> names;
    std::vector<int> scores;

    std::cout << "=== 成績管理システム ===" << std::endl;
    std::cout << "生徒の名前と点数を入力してください（終了するには「end」と入力）"
              << std::endl;
    std::cout << std::endl;

    // 入力ループ
    while (true) {
        std::string name;
        std::cout << "名前: ";
        std::cin >> name;

        // 終了条件
        if (name == "end") {
            break;
        }

        int score;
        std::cout << "点数: ";
        std::cin >> score;
        std::cout << std::endl;

        // ベクターに追加
        names.push_back(name);
        scores.push_back(score);
    }

    // データがない場合
    if (names.empty()) {
        std::cout << "データが入力されていません。" << std::endl;
        return 0;
    }

    // --- 一覧表示 ---
    std::cout << std::endl;
    std::cout << "--- 成績一覧 ---" << std::endl;
    for (size_t i = 0; i < names.size(); i++) {
        std::cout << i + 1 << ". " << names[i] << ": "
                  << scores[i] << "点" << std::endl;
    }

    // --- 統計計算 ---
    int sum = 0;
    int max_idx = 0; // 最高点のインデックス
    int min_idx = 0; // 最低点のインデックス

    for (size_t i = 0; i < scores.size(); i++) {
        sum += scores[i];

        if (scores[i] > scores[max_idx]) {
            max_idx = static_cast<int>(i);
        }
        if (scores[i] < scores[min_idx]) {
            min_idx = static_cast<int>(i);
        }
    }

    double average = static_cast<double>(sum) / scores.size();

    // --- 統計表示 ---
    std::cout << std::endl;
    std::cout << "--- 統計情報 ---" << std::endl;
    std::cout << "生徒数: " << names.size() << "人" << std::endl;
    std::cout << std::fixed << std::setprecision(1);
    std::cout << "平均点: " << average << "点" << std::endl;
    std::cout << "最高点: " << names[max_idx] << "（"
              << scores[max_idx] << "点）" << std::endl;
    std::cout << "最低点: " << names[min_idx] << "（"
              << scores[min_idx] << "点）" << std::endl;

    return 0;
}
```

</details>

## 問題6：シーザー暗号（チャレンジ）

文字列とシフト数を入力として受け取り、シーザー暗号で暗号化・復号化するプログラムを作成してください。

**シーザー暗号のルール：**
- 各アルファベットを指定した数だけ後ろにずらす（暗号化）
- `z` の次は `a` に戻る（ラップアラウンド）
- アルファベット以外の文字（スペース、数字、記号）はそのまま残す
- 大文字・小文字はそれぞれの範囲内でシフトする

### 期待される出力例

```
文字列を入力してください: Hello World!
シフト数を入力してください: 3
--- 結果 ---
元の文字列: Hello World!
暗号化:     Khoor Zruog!
復号化:     Hello World!
```

```
文字列を入力してください: ABC xyz
シフト数を入力してください: 5
--- 結果 ---
元の文字列: ABC xyz
暗号化:     FGH cde
復号化:     ABC xyz
```

<details>
<summary>ヒント</summary>

- アルファベットかどうかの判定には `std::isalpha(c)` が使えます
- 大文字・小文字の判定には `std::isupper(c)` / `std::islower(c)` が使えます
- シフトの計算：`'a' + (c - 'a' + shift) % 26` のように、基準文字からのオフセットで考えます
- 復号化は暗号化と逆方向（`26 - shift`）にずらすだけです
- 負のシフトを避けるため `(c - 'a' + shift % 26 + 26) % 26` のようにするのが安全です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
#include <cctype>

// シーザー暗号の暗号化関数
std::string caesar_encrypt(const std::string& text, int shift) {
    std::string result = "";

    // シフト値を正の値に正規化（負の値にも対応）
    shift = ((shift % 26) + 26) % 26;

    for (char c : text) {
        if (std::isupper(c)) {
            // 大文字の場合: 'A'を基準にシフト
            result += static_cast<char>('A' + (c - 'A' + shift) % 26);
        }
        else if (std::islower(c)) {
            // 小文字の場合: 'a'を基準にシフト
            result += static_cast<char>('a' + (c - 'a' + shift) % 26);
        }
        else {
            // アルファベット以外はそのまま
            result += c;
        }
    }

    return result;
}

int main() {
    std::string text;
    int shift;

    std::cout << "文字列を入力してください: ";
    std::getline(std::cin, text);

    std::cout << "シフト数を入力してください: ";
    std::cin >> shift;

    // 暗号化
    std::string encrypted = caesar_encrypt(text, shift);

    // 復号化（暗号化の逆方向にシフト）
    std::string decrypted = caesar_encrypt(encrypted, -shift);

    // 結果表示
    std::cout << "--- 結果 ---" << std::endl;
    std::cout << "元の文字列: " << text << std::endl;
    std::cout << "暗号化:     " << encrypted << std::endl;
    std::cout << "復号化:     " << decrypted << std::endl;

    return 0;
}
```

</details>
