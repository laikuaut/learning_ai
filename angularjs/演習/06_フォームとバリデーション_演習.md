## 演習6-1：必須バリデーション付きログインフォーム（基本）

### 問題

以下の要件を満たすログインフォームを作成してください。

1. ユーザー名（必須、3文字以上）とパスワード（必須、8文字以上）の入力欄
2. 各フィールドにバリデーションエラーメッセージを表示（フィールドに触れた後のみ）
3. フォームが無効な場合はログインボタンを無効化
4. 有効な場合にログインボタンをクリックするとアラートで内容を表示

### 期待される動作

- 未入力のまま入力欄にフォーカスして外すとエラーが表示される
- 条件を満たすとエラーが消え、ボタンが有効になる

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 400px; margin: 40px auto; }
    .form-group { margin: 16px 0; }
    label { display: block; font-weight: bold; margin-bottom: 4px; }
    input { width: 100%; padding: 10px; border: 2px solid #ddd; border-radius: 4px; box-sizing: border-box; font-size: 14px; }
    input.ng-invalid.ng-touched { border-color: #e74c3c; }
    input.ng-valid.ng-touched { border-color: #42b883; }
    .error { color: #e74c3c; font-size: 13px; margin-top: 4px; }
    .btn { width: 100%; padding: 12px; background: #42b883; color: white; border: none; border-radius: 4px; font-size: 16px; cursor: pointer; }
    .btn:disabled { background: #ccc; cursor: not-allowed; }
  </style>
</head>
<body ng-controller="LoginCtrl">
  <h2>ログイン</h2>
  
  <form name="loginForm" ng-submit="login()" novalidate>
    <div class="form-group">
      <label>ユーザー名</label>
      <input type="text" name="username" ng-model="user.name"
             required ng-minlength="3">
      <!-- $touched でフォーカスが外れた後のみ表示 -->
      <div class="error" ng-show="loginForm.username.$touched && loginForm.username.$invalid">
        <p ng-show="loginForm.username.$error.required">ユーザー名は必須です</p>
        <p ng-show="loginForm.username.$error.minlength">3文字以上で入力してください</p>
      </div>
    </div>
    
    <div class="form-group">
      <label>パスワード</label>
      <input type="password" name="password" ng-model="user.password"
             required ng-minlength="8">
      <div class="error" ng-show="loginForm.password.$touched && loginForm.password.$invalid">
        <p ng-show="loginForm.password.$error.required">パスワードは必須です</p>
        <p ng-show="loginForm.password.$error.minlength">8文字以上で入力してください</p>
      </div>
    </div>
    
    <!-- $invalid の場合はボタンを無効化 -->
    <button class="btn" type="submit" ng-disabled="loginForm.$invalid">ログイン</button>
  </form>

  <script>
    angular.module('myApp', [])
      .controller('LoginCtrl', function($scope) {
        $scope.user = {};
        $scope.login = function() {
          if ($scope.loginForm.$valid) {
            alert('ログイン: ' + $scope.user.name);
          }
        };
      });
  </script>
</body>
</html>
```

</details>

## 演習6-2：フォーム状態のCSSスタイリング（基本）

### 問題

入力フィールドの状態に応じてスタイルを変更してください。

1. 未入力（pristine）: グレーのボーダー
2. 入力中で有効（dirty + valid）: 緑のボーダー + 緑の背景
3. 入力中で無効（dirty + invalid）: 赤のボーダー + 薄い赤の背景
4. フォームの状態（$pristine, $dirty, $valid, $invalid）をリアルタイムに表示するデバッグパネルを追加

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 500px; margin: 40px auto; }
    .form-group { margin: 16px 0; }
    input { width: 100%; padding: 10px; border: 2px solid #ddd; border-radius: 4px; box-sizing: border-box; font-size: 14px; transition: all 0.2s; }
    /* CSSクラスによるスタイリング */
    input.ng-pristine { border-color: #ccc; background: white; }
    input.ng-dirty.ng-valid { border-color: #42b883; background: #e8f5e9; }
    input.ng-dirty.ng-invalid { border-color: #e74c3c; background: #ffebee; }
    .debug { background: #f5f5f5; padding: 16px; border-radius: 8px; margin-top: 20px; font-size: 13px; font-family: monospace; }
    .true { color: #42b883; font-weight: bold; }
    .false { color: #e74c3c; }
  </style>
</head>
<body ng-controller="StyleCtrl">
  <h2>フォーム状態のスタイリング</h2>
  
  <form name="demoForm" novalidate>
    <div class="form-group">
      <label>名前（必須、2文字以上）</label>
      <input type="text" name="userName" ng-model="name" required ng-minlength="2">
    </div>
    <div class="form-group">
      <label>メール（必須、メール形式）</label>
      <input type="email" name="userEmail" ng-model="email" required>
    </div>
  </form>
  
  <!-- デバッグパネル -->
  <div class="debug">
    <h4>フォーム全体</h4>
    <p>$pristine: <span ng-class="{ true: demoForm.$pristine, false: !demoForm.$pristine }">{{ demoForm.$pristine }}</span></p>
    <p>$dirty: <span ng-class="{ true: demoForm.$dirty, false: !demoForm.$dirty }">{{ demoForm.$dirty }}</span></p>
    <p>$valid: <span ng-class="{ true: demoForm.$valid, false: !demoForm.$valid }">{{ demoForm.$valid }}</span></p>
    <p>$invalid: <span ng-class="{ true: !demoForm.$invalid, false: demoForm.$invalid }">{{ demoForm.$invalid }}</span></p>
    
    <h4>名前フィールド</h4>
    <p>$error: {{ demoForm.userName.$error | json }}</p>
    
    <h4>メールフィールド</h4>
    <p>$error: {{ demoForm.userEmail.$error | json }}</p>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('StyleCtrl', function($scope) {
        // 初期値なし（pristine 状態を見せるため）
      });
  </script>
</body>
</html>
```

## 演習6-3：ユーザー登録フォーム（応用）

### 問題

以下のフィールドを持つユーザー登録フォームを作成してください。

1. 名前（必須、2〜20文字）
2. メール（必須、メール形式）
3. 電話番号（任意、`/^0\d{1,4}-\d{1,4}-\d{4}$/` パターン）
4. 年齢（必須、数値、18〜100）
5. 性別（ラジオボタン: 男性/女性/その他）
6. 利用規約への同意（チェックボックス、必須）
7. すべて有効な場合のみ送信ボタンが有効になる
8. 送信成功時に「登録完了」メッセージを表示

## 演習6-4：カスタムバリデータ — パスワード強度チェック（応用）

### 問題

パスワードの強度をチェックするカスタムバリデータディレクティブ `password-strength` を作成してください。

**要件：**
- 8文字以上
- 大文字を1つ以上含む
- 数字を1つ以上含む
- 上記すべてを満たさないと `$error.passwordStrength` が true になる
- 強度レベル（弱/中/強）を視覚的に表示する

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 400px; margin: 40px auto; }
    input { width: 100%; padding: 10px; border: 2px solid #ddd; border-radius: 4px; box-sizing: border-box; margin: 8px 0; }
    input.ng-dirty.ng-valid { border-color: #42b883; }
    input.ng-dirty.ng-invalid { border-color: #e74c3c; }
    .strength-bar { height: 6px; border-radius: 3px; margin-top: 4px; transition: all 0.3s; }
    .strength-weak { width: 33%; background: #e74c3c; }
    .strength-medium { width: 66%; background: #f39c12; }
    .strength-strong { width: 100%; background: #42b883; }
    .error { color: #e74c3c; font-size: 13px; }
    .strength-label { font-size: 13px; margin-top: 4px; }
  </style>
</head>
<body ng-controller="PwCtrl">
  <h2>パスワード強度チェック</h2>
  
  <form name="pwForm" novalidate>
    <label>パスワード</label>
    <input type="password" name="pw" ng-model="password" required password-strength>
    
    <!-- 強度バー -->
    <div class="strength-bar" ng-show="password"
         ng-class="{
           'strength-weak': strength === 'weak',
           'strength-medium': strength === 'medium',
           'strength-strong': strength === 'strong'
         }"></div>
    <p class="strength-label" ng-show="password">
      強度: {{ strengthLabel }}
    </p>
    
    <div class="error" ng-show="pwForm.pw.$dirty && pwForm.pw.$invalid">
      <p ng-show="pwForm.pw.$error.required">パスワードは必須です</p>
      <p ng-show="pwForm.pw.$error.passwordStrength">8文字以上、大文字1つ以上、数字1つ以上が必要です</p>
    </div>
  </form>

  <script>
    angular.module('myApp', [])
      .controller('PwCtrl', function($scope) {
        $scope.password = '';
        $scope.strength = '';
        $scope.strengthLabel = '';
      })
      .directive('passwordStrength', function() {
        return {
          require: 'ngModel',
          link: function(scope, elem, attrs, ngModel) {
            ngModel.$validators.passwordStrength = function(value) {
              if (!value) return false;
              var hasLength = value.length >= 8;
              var hasUpper = /[A-Z]/.test(value);
              var hasNumber = /\d/.test(value);
              
              // 強度計算
              var score = (hasLength ? 1 : 0) + (hasUpper ? 1 : 0) + (hasNumber ? 1 : 0);
              if (score <= 1) {
                scope.strength = 'weak';
                scope.strengthLabel = '弱い';
              } else if (score === 2) {
                scope.strength = 'medium';
                scope.strengthLabel = '普通';
              } else {
                scope.strength = 'strong';
                scope.strengthLabel = '強い';
              }
              
              return hasLength && hasUpper && hasNumber;
            };
          }
        };
      });
  </script>
</body>
</html>
```

## 演習6-5：多機能お問い合わせフォーム（チャレンジ）

### 問題

以下の機能を持つ完全なお問い合わせフォームを作成してください。

1. 名前（必須）、メール（必須、メール形式）、件名（必須）、カテゴリ（select、必須）、メッセージ（必須、10文字以上）
2. すべてのフィールドにリアルタイムバリデーション
3. 送信ボタンクリック時に未入力フィールドのエラーもすべて表示
4. 送信成功時にフォームをリセットし、成功メッセージを表示
5. 文字数カウンターをメッセージ欄に表示

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 500px; margin: 40px auto; padding: 0 20px; }
    .form-group { margin: 14px 0; }
    label { display: block; font-weight: bold; margin-bottom: 4px; font-size: 14px; }
    input, select, textarea { width: 100%; padding: 10px; border: 2px solid #ddd; border-radius: 4px; box-sizing: border-box; font-size: 14px; font-family: inherit; }
    textarea { height: 120px; resize: vertical; }
    .ng-invalid.ng-touched, .ng-invalid.submitted { border-color: #e74c3c !important; }
    .ng-valid.ng-touched { border-color: #42b883; }
    .error { color: #e74c3c; font-size: 12px; margin-top: 2px; }
    .btn { padding: 12px 28px; background: #42b883; color: white; border: none; border-radius: 4px; font-size: 16px; cursor: pointer; }
    .btn:disabled { background: #ccc; }
    .success { background: #e8f5e9; padding: 20px; border-radius: 8px; text-align: center; border: 1px solid #42b883; }
    .char-count { text-align: right; font-size: 12px; color: #999; }
  </style>
</head>
<body ng-controller="ContactCtrl">
  <h2>お問い合わせ</h2>
  
  <div class="success" ng-show="sent">
    <h3>送信完了しました！</h3>
    <p>{{ sentName }}様、お問い合わせありがとうございます。</p>
    <button class="btn" ng-click="sent = false">新しいお問い合わせ</button>
  </div>
  
  <form name="contactForm" ng-submit="submit()" ng-hide="sent" novalidate>
    <div class="form-group">
      <label>お名前 *</label>
      <input type="text" name="name" ng-model="form.name" required
             ng-class="{ submitted: contactForm.$submitted }">
      <div class="error" ng-show="(contactForm.name.$touched || contactForm.$submitted) && contactForm.name.$error.required">
        お名前は必須です
      </div>
    </div>
    
    <div class="form-group">
      <label>メールアドレス *</label>
      <input type="email" name="email" ng-model="form.email" required
             ng-class="{ submitted: contactForm.$submitted }">
      <div class="error" ng-show="(contactForm.email.$touched || contactForm.$submitted) && contactForm.email.$invalid">
        <span ng-show="contactForm.email.$error.required">メールは必須です</span>
        <span ng-show="contactForm.email.$error.email">正しい形式で入力してください</span>
      </div>
    </div>
    
    <div class="form-group">
      <label>カテゴリ *</label>
      <select name="category" ng-model="form.category" required
              ng-class="{ submitted: contactForm.$submitted }">
        <option value="">選択してください</option>
        <option value="general">一般的なお問い合わせ</option>
        <option value="support">技術サポート</option>
        <option value="billing">お支払いについて</option>
        <option value="other">その他</option>
      </select>
      <div class="error" ng-show="(contactForm.category.$touched || contactForm.$submitted) && contactForm.category.$error.required">
        カテゴリを選択してください
      </div>
    </div>
    
    <div class="form-group">
      <label>件名 *</label>
      <input type="text" name="subject" ng-model="form.subject" required
             ng-class="{ submitted: contactForm.$submitted }">
      <div class="error" ng-show="(contactForm.subject.$touched || contactForm.$submitted) && contactForm.subject.$error.required">
        件名は必須です
      </div>
    </div>
    
    <div class="form-group">
      <label>メッセージ *</label>
      <textarea name="message" ng-model="form.message" required ng-minlength="10"
                ng-class="{ submitted: contactForm.$submitted }"></textarea>
      <div class="char-count">{{ (form.message || '').length }}文字（10文字以上）</div>
      <div class="error" ng-show="(contactForm.message.$touched || contactForm.$submitted) && contactForm.message.$invalid">
        <span ng-show="contactForm.message.$error.required">メッセージは必須です</span>
        <span ng-show="contactForm.message.$error.minlength">10文字以上で入力してください</span>
      </div>
    </div>
    
    <button class="btn" type="submit">送信する</button>
  </form>

  <script>
    angular.module('myApp', [])
      .controller('ContactCtrl', function($scope) {
        $scope.form = {};
        $scope.sent = false;
        
        $scope.submit = function() {
          // $submitted をセットしてすべてのエラーを表示
          if ($scope.contactForm.$invalid) {
            return;
          }
          $scope.sentName = $scope.form.name;
          $scope.sent = true;
          // フォームリセット
          $scope.form = {};
          $scope.contactForm.$setPristine();
          $scope.contactForm.$setUntouched();
        };
      });
  </script>
</body>
</html>
```
