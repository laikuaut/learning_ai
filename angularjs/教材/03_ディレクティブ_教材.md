# 第3章：ディレクティブ

## この章のゴール

- ディレクティブ（Directive）の概念を理解する
- `ng-repeat` でリストを動的に表示できるようになる
- `ng-if` と `ng-show/ng-hide` の違いを把握する
- `ng-class`、`ng-style` で動的にスタイルを適用する
- イベント系ディレクティブ（`ng-click` など）を使いこなす
- `ng-cloak` でちらつきを防止する

---

## 3.1 ディレクティブとは

ディレクティブ（Directive）とは、**HTMLに特別な振る舞いを追加する仕組み**です。AngularJS の最も強力な機能の一つで、HTML を拡張して独自のタグや属性を作れます。

### ディレクティブの種類

| 種類 | 例 | 説明 |
|------|-----|------|
| 組み込みディレクティブ | `ng-model`, `ng-repeat` | AngularJS が標準で提供 |
| カスタムディレクティブ | `<my-widget>` | 自分で定義（第5章で学習） |

### よく使う組み込みディレクティブ一覧

| ディレクティブ | 用途 |
|----------------|------|
| `ng-app` | AngularJS アプリのルートを指定 |
| `ng-controller` | コントローラを割り当て |
| `ng-model` | 双方向バインディング |
| `ng-repeat` | 配列のループ表示 |
| `ng-if` | 条件による要素の追加・削除 |
| `ng-show` / `ng-hide` | 条件による表示・非表示 |
| `ng-class` | 動的なCSS クラス付与 |
| `ng-style` | 動的なインラインスタイル |
| `ng-click` | クリックイベント |
| `ng-submit` | フォーム送信イベント |
| `ng-cloak` | 初期表示のちらつき防止 |

---

## 3.2 ng-repeat — リストの表示

`ng-repeat` は配列やオブジェクトを**ループして繰り返し表示**するディレクティブです。

### 基本の使い方

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="ListCtrl">
  <h2>フルーツリスト</h2>
  <ul>
    <li ng-repeat="fruit in fruits">
      {{ $index + 1 }}. {{ fruit }}
    </li>
  </ul>

  <script>
    angular.module('myApp', [])
      .controller('ListCtrl', function($scope) {
        $scope.fruits = ['りんご', 'バナナ', 'みかん', 'ぶどう', 'もも'];
      });
  </script>
</body>
</html>
```

### ng-repeat の特殊変数

| 変数 | 型 | 説明 |
|------|-----|------|
| `$index` | 数値 | 現在のインデックス（0始まり） |
| `$first` | 真偽値 | 最初の要素なら `true` |
| `$last` | 真偽値 | 最後の要素なら `true` |
| `$middle` | 真偽値 | 最初でも最後でもなければ `true` |
| `$even` | 真偽値 | 偶数番目（0, 2, 4...）なら `true` |
| `$odd` | 真偽値 | 奇数番目（1, 3, 5...）なら `true` |

### オブジェクト配列のループ

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    table { border-collapse: collapse; width: 100%; }
    th, td { border: 1px solid #ddd; padding: 8px; text-align: left; }
    th { background: #42b883; color: white; }
    .even { background: #f9f9f9; }
  </style>
</head>
<body ng-controller="StudentCtrl">
  <h2>生徒一覧</h2>
  <table>
    <tr>
      <th>No.</th><th>名前</th><th>学年</th><th>点数</th>
    </tr>
    <tr ng-repeat="student in students" ng-class="{ even: $even }">
      <td>{{ $index + 1 }}</td>
      <td>{{ student.name }}</td>
      <td>{{ student.grade }}年</td>
      <td>{{ student.score }}点</td>
    </tr>
  </table>

  <script>
    angular.module('myApp', [])
      .controller('StudentCtrl', function($scope) {
        $scope.students = [
          { name: '田中太郎', grade: 1, score: 85 },
          { name: '鈴木花子', grade: 2, score: 92 },
          { name: '佐藤一郎', grade: 1, score: 78 },
          { name: '高橋美咲', grade: 3, score: 95 },
          { name: '渡辺健一', grade: 2, score: 88 }
        ];
      });
  </script>
</body>
</html>
```

### track by — パフォーマンスの最適化

`ng-repeat` はデフォルトでオブジェクトの参照でアイテムを追跡します。`track by` を指定すると、一意のキーで追跡でき、パフォーマンスが向上します。

```html
<!-- 基本形（デフォルト） -->
<li ng-repeat="item in items">{{ item.name }}</li>

<!-- track by で ID を指定（推奨） -->
<li ng-repeat="item in items track by item.id">{{ item.name }}</li>

<!-- プリミティブ配列の場合は $index を使う -->
<li ng-repeat="tag in tags track by $index">{{ tag }}</li>
```

> **よくある間違い：** 配列に重複する値がある場合、`track by` を付けないとエラーになります。`track by $index` で解決できます。

### ネストした ng-repeat

```html
<div ng-repeat="category in categories">
  <h3>{{ category.name }}</h3>
  <ul>
    <li ng-repeat="item in category.items">{{ item }}</li>
  </ul>
</div>
```

### ポイントまとめ

- `ng-repeat` は配列やオブジェクトをループ表示する
- `$index`, `$first`, `$last`, `$even`, `$odd` の特殊変数が使える
- `track by` でパフォーマンスを最適化し、重複エラーを防げる

## 3.3 ng-if vs ng-show / ng-hide

条件による表示制御には2つのアプローチがあります。

### 比較表

| 特性 | ng-if | ng-show / ng-hide |
|------|-------|-------------------|
| DOM操作 | 要素を追加/削除する | `display: none` で隠すだけ |
| スコープ | 新しい子スコープを作る | スコープを作らない |
| 初期コスト | 高い（DOM操作） | 低い（CSS切替のみ） |
| 切替コスト | 高い | 低い |
| 適する場面 | めったに切り替えない場合 | 頻繁に切り替える場合 |

```
【ng-if の動作】
条件 true  → <div>内容</div>     ← DOM に存在
条件 false → <!-- ng-if: -->     ← DOM から完全に削除

【ng-show の動作】
条件 true  → <div>内容</div>                    ← 表示
条件 false → <div style="display:none">内容</div> ← 非表示（DOMには残る）
```

### 実例

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .box { padding: 15px; margin: 10px 0; border-radius: 8px; }
    .info { background: #e3f2fd; border: 1px solid #2196f3; }
    .warning { background: #fff3e0; border: 1px solid #ff9800; }
  </style>
</head>
<body ng-controller="ToggleCtrl">
  <h2>ng-if vs ng-show の比較</h2>
  
  <label>
    <input type="checkbox" ng-model="isVisible"> 表示する
  </label>

  <div class="box info" ng-if="isVisible">
    <strong>ng-if:</strong> この要素はDOMから完全に追加/削除されます
  </div>
  
  <div class="box warning" ng-show="isVisible">
    <strong>ng-show:</strong> この要素はCSSで表示/非表示が切り替わります
  </div>
  
  <div class="box warning" ng-hide="!isVisible">
    <strong>ng-hide:</strong> ng-show の逆（条件が true で非表示）
  </div>

  <script>
    angular.module('myApp', [])
      .controller('ToggleCtrl', function($scope) {
        $scope.isVisible = true;
      });
  </script>
</body>
</html>
```

### ポイントまとめ

- **頻繁に切り替える場合** → `ng-show` / `ng-hide`（軽量）
- **めったに表示しない場合** → `ng-if`（不要時にDOMを削減）
- `ng-if` は新しいスコープを作るため、ドット記法の注意が必要

## 3.4 ng-class — 動的なCSSクラス

`ng-class` は条件に応じて**CSSクラスを動的に付与**するディレクティブです。

### 3つの書き方

```html
<!-- 1. オブジェクト構文（最もよく使う） -->
<p ng-class="{ 'active': isActive, 'highlight': isHighlighted }">
  条件が true のクラスが付与される
</p>

<!-- 2. 配列構文 -->
<p ng-class="[style1, style2]">
  変数に格納されたクラス名が付与される
</p>

<!-- 3. 三項演算子 -->
<p ng-class="isError ? 'text-danger' : 'text-success'">
  条件で切り替え
</p>
```

### 実用例：スコアに応じた色分け

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .score { padding: 4px 12px; border-radius: 4px; font-weight: bold; display: inline-block; }
    .score-high { background: #c8e6c9; color: #2e7d32; }
    .score-mid { background: #fff9c4; color: #f57f17; }
    .score-low { background: #ffcdd2; color: #c62828; }
    table { border-collapse: collapse; margin-top: 10px; }
    td, th { padding: 8px 16px; border: 1px solid #ddd; }
    th { background: #35495e; color: white; }
  </style>
</head>
<body ng-controller="ScoreCtrl">
  <h2>成績表</h2>
  <table>
    <tr><th>名前</th><th>点数</th></tr>
    <tr ng-repeat="s in students">
      <td>{{ s.name }}</td>
      <td>
        <span class="score" ng-class="{
          'score-high': s.score >= 80,
          'score-mid': s.score >= 60 && s.score < 80,
          'score-low': s.score < 60
        }">{{ s.score }}点</span>
      </td>
    </tr>
  </table>

  <script>
    angular.module('myApp', [])
      .controller('ScoreCtrl', function($scope) {
        $scope.students = [
          { name: '田中', score: 95 },
          { name: '鈴木', score: 72 },
          { name: '佐藤', score: 48 },
          { name: '高橋', score: 83 },
          { name: '渡辺', score: 61 }
        ];
      });
  </script>
</body>
</html>
```

## 3.5 ng-style — 動的なインラインスタイル

```html
<div ng-style="{ 'background-color': bgColor, 'font-size': fontSize + 'px' }">
  動的スタイル
</div>
```

### 実例：スライダーでスタイルを変更

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="StyleCtrl">
  <h2>ng-style デモ</h2>
  
  <label>フォントサイズ: {{ fontSize }}px</label><br>
  <input type="range" ng-model="fontSize" min="12" max="48"><br><br>
  
  <label>色: </label>
  <select ng-model="textColor">
    <option value="#333">黒</option>
    <option value="#e74c3c">赤</option>
    <option value="#2196f3">青</option>
    <option value="#42b883">緑</option>
  </select>
  
  <p ng-style="{ 'font-size': fontSize + 'px', 'color': textColor }">
    このテキストのスタイルが動的に変わります！
  </p>

  <script>
    angular.module('myApp', [])
      .controller('StyleCtrl', function($scope) {
        $scope.fontSize = 16;
        $scope.textColor = '#333';
      });
  </script>
</body>
</html>
```

## 3.6 イベント系ディレクティブ

### 主なイベントディレクティブ

| ディレクティブ | 対応するDOMイベント |
|----------------|---------------------|
| `ng-click` | click |
| `ng-dblclick` | dblclick |
| `ng-change` | change（`ng-model` と併用） |
| `ng-submit` | submit（`form` 要素に使用） |
| `ng-keydown` | keydown |
| `ng-keyup` | keyup |
| `ng-keypress` | keypress |
| `ng-mouseenter` | mouseenter |
| `ng-mouseleave` | mouseleave |
| `ng-focus` | focus |
| `ng-blur` | blur |

### `ng-click` と `$event`

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="EventCtrl">
  <h2>イベントデモ</h2>
  
  <!-- ng-click -->
  <button ng-click="count = count + 1">クリック回数: {{ count }}</button>
  
  <!-- ng-submit（フォーム送信） -->
  <form ng-submit="addItem()">
    <input ng-model="newItem" placeholder="アイテムを追加">
    <button type="submit">追加</button>
  </form>
  <ul>
    <li ng-repeat="item in items">{{ item }}</li>
  </ul>
  
  <!-- ng-keyup でリアルタイム検索 -->
  <input ng-model="query" ng-keyup="search()" placeholder="検索">
  <p>検索中: {{ query }}</p>

  <script>
    angular.module('myApp', [])
      .controller('EventCtrl', function($scope) {
        $scope.count = 0;
        $scope.items = [];
        $scope.newItem = '';
        
        $scope.addItem = function() {
          if ($scope.newItem.trim()) {
            $scope.items.push($scope.newItem);
            $scope.newItem = '';
          }
        };
        
        $scope.search = function() {
          console.log('検索:', $scope.query);
        };
      });
  </script>
</body>
</html>
```

## 3.7 ng-switch — 条件分岐の表示

`ng-switch` は値に応じて**表示する要素を切り替える**ディレクティブです。JavaScript の `switch` 文に相当します。

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="SwitchCtrl">
  <h2>ステータス表示</h2>
  <select ng-model="status">
    <option value="active">アクティブ</option>
    <option value="pending">保留中</option>
    <option value="inactive">無効</option>
  </select>

  <div ng-switch="status">
    <p ng-switch-when="active" style="color: green;">✓ アクティブ — 正常に動作中です</p>
    <p ng-switch-when="pending" style="color: orange;">⏳ 保留中 — 承認を待っています</p>
    <p ng-switch-when="inactive" style="color: red;">✗ 無効 — サービスは停止中です</p>
    <p ng-switch-default>状態を選択してください</p>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('SwitchCtrl', function($scope) {
        $scope.status = 'active';
      });
  </script>
</body>
</html>
```

## 3.8 ng-include — テンプレートの読み込み

`ng-include` は外部HTMLテンプレートやインラインテンプレートを読み込んで表示します。

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="IncludeCtrl">
  <!-- script タグでインラインテンプレートを定義 -->
  <script type="text/ng-template" id="header.html">
    <div style="background: #35495e; color: white; padding: 15px;">
      <h1>{{ title }}</h1>
    </div>
  </script>

  <script type="text/ng-template" id="footer.html">
    <div style="background: #eee; padding: 10px; text-align: center; margin-top: 20px;">
      <p>&copy; 2024 {{ title }}</p>
    </div>
  </script>

  <ng-include src="'header.html'"></ng-include>
  <div style="padding: 20px;">
    <p>メインコンテンツ</p>
  </div>
  <ng-include src="'footer.html'"></ng-include>

  <script>
    angular.module('myApp', [])
      .controller('IncludeCtrl', function($scope) {
        $scope.title = 'マイアプリ';
      });
  </script>
</body>
</html>
```

> **注意：** `ng-include` の `src` には**文字列リテラル**を渡すため、シングルクォートで囲む必要があります（`src="'template.html'"`）。

## 3.9 ng-cloak — ちらつき防止

AngularJS がまだ読み込まれていない瞬間、`{{ expression }}` がそのまま画面に一瞬表示される（ちらつき/FOUC）ことがあります。`ng-cloak` でこれを防ぎます。

```html
<style>
  [ng-cloak] { display: none !important; }
</style>

<div ng-cloak>
  {{ message }}
</div>
```

### ポイントまとめ

- `ng-cloak` は CSS で `display: none` にし、AngularJS の処理完了後に自動的に表示する
- `<style>` で `[ng-cloak]` のルールを書くとさらに確実

## 3.10 まとめ

| ディレクティブ | 用途 | 使い分けのポイント |
|----------------|------|---------------------|
| `ng-repeat` | リスト表示 | `track by` でパフォーマンス向上 |
| `ng-if` | 条件によるDOM追加/削除 | めったに切り替えない場合 |
| `ng-show/hide` | 条件による表示/非表示 | 頻繁に切り替える場合 |
| `ng-class` | 動的CSSクラス | オブジェクト構文が最も柔軟 |
| `ng-style` | 動的インラインスタイル | 値が動的に変わる場合 |
| `ng-click` 他 | イベント処理 | `$event` でイベントオブジェクト取得可能 |
| `ng-switch` | 値による分岐表示 | 3つ以上の条件分岐に適する |
| `ng-include` | テンプレート読み込み | `src` は文字列リテラルで渡す |
| `ng-cloak` | ちらつき防止 | 必ずCSSと併用する |
