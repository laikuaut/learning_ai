## 3.1 アンカー（Anchor）

### アンカーとは

アンカーとは、**文字そのものではなく「位置」にマッチ**するメタ文字です。文字を消費しない（幅がゼロ）ため、「ゼロ幅アサーション（Zero-width Assertion）」とも呼ばれます。

```
【アンカーのイメージ】

通常のパターン:   \d+      → 文字列中のどこにあっても数字にマッチ
アンカー付き:     ^\d+     → 行の先頭にある数字にだけマッチ
                  \d+$     → 行の末尾にある数字にだけマッチ
```

### `^`：行頭（文字列の先頭）

`^` は、**文字列の先頭**にマッチします。`re.MULTILINE` フラグを使うと、各行の先頭にマッチします。

```python
import re

text = "Python is great"

# 先頭が "Python" で始まるかチェック
result = re.search(r"^Python", text)
print(result.group() if result else "マッチなし")  # Python

# 先頭が "is" で始まるかチェック
result = re.search(r"^is", text)
print(result.group() if result else "マッチなし")  # マッチなし
```

```python
import re

text = """1行目: Hello
2行目: World
3行目: Python"""

# re.MULTILINE なし → 文字列全体の先頭のみ
results = re.findall(r"^\d", text)
print(results)  # ['1']

# re.MULTILINE あり → 各行の先頭
results = re.findall(r"^\d", text, re.MULTILINE)
print(results)  # ['1', '2', '3']
```

### `$`：行末（文字列の末尾）

`$` は、**文字列の末尾**にマッチします。`re.MULTILINE` フラグを使うと、各行の末尾にマッチします。

```python
import re

text = "Hello World"

# 末尾が "World" で終わるかチェック
result = re.search(r"World$", text)
print(result.group() if result else "マッチなし")  # World

# 末尾が "Hello" で終わるかチェック
result = re.search(r"Hello$", text)
print(result.group() if result else "マッチなし")  # マッチなし
```

```python
import re

text = """apple 100
banana 200
cherry 300"""

# 各行末の数字を取得（re.MULTILINE使用）
results = re.findall(r"\d+$", text, re.MULTILINE)
print(results)  # ['100', '200', '300']
```

### `\b`：単語境界（Word Boundary）

`\b` は、**単語文字（`\w`）と非単語文字（`\W`）の境界**にマッチします。単語の「端」を指定できます。

```
【単語境界のイメージ】

テキスト: "cat concatenate category"
          ↑   ↑                      → 各単語の先頭と末尾に境界がある

\bcat\b   → "cat" のみマッチ（独立した単語）
\bcat     → "cat", "concatenate", "category" の先頭にマッチ
cat\b     → "cat" のみマッチ（末尾で単語境界）
```

```python
import re

text = "cat concatenate category scat"

# \bcat\b → 独立した単語 "cat" のみ
results = re.findall(r"\bcat\b", text)
print(results)  # ['cat']

# \bcat → "cat" で始まる単語の先頭部分
results = re.findall(r"\bcat", text)
print(results)  # ['cat', 'cat', 'cat']
# ← cat, concatenate, category の先頭の cat

# cat\b → "cat" で終わる単語の末尾部分
results = re.findall(r"cat\b", text)
print(results)  # ['cat', 'cat']
# ← cat, scat の末尾の cat
```

```python
import re

# 実用例：特定の単語を丸ごと置換
text = "I like Java and JavaScript"

# Java だけを Python に置換（JavaScript は変えない）
result = re.sub(r"\bJava\b", "Python", text)
print(result)  # I like Python and JavaScript
```

> **よくある間違い**: `\b` は「スペース」ではありません。単語文字（英数字・アンダースコア）と非単語文字（それ以外）の**境界**にマッチします。文字列の先頭や末尾も単語境界になります。

### アンカーのまとめ

```
┌────────┬──────────────────────────────────────────────┐
│ 記号    │ 意味                                         │
├────────┼──────────────────────────────────────────────┤
│   ^    │ 行頭（文字列の先頭、MULTILINE時は各行の先頭） │
│   $    │ 行末（文字列の末尾、MULTILINE時は各行の末尾） │
│  \b    │ 単語境界（単語文字と非単語文字の間）          │
│  \B    │ 単語境界以外（\b の否定）                    │
└────────┴──────────────────────────────────────────────┘
```

## 3.2 グループ化（Grouping）と選択（Alternation）

### グループ化 `()` とは

丸括弧 `()` を使うと、パターンの一部を**グループ化**できます。グループ化には2つの役割があります。

1. **量指定子をグループに適用する**（パターンのまとまりに対して繰り返しを指定）
2. **マッチした部分を後から取り出す**（キャプチャ）

```python
import re

text = "abcabcabc"

# (abc)+ → "abc" というかたまりが1回以上繰り返し
result = re.search(r"(abc)+", text)
print(result.group())   # abcabcabc
```

```python
import re

# グループなし：? は直前の1文字(c)にだけ適用
text = "ab abc"
results = re.findall(r"abc?", text)
print(results)  # ['ab', 'abc']

# グループあり：? はグループ全体(bc)に適用
results = re.findall(r"a(bc)?", text)
print(results)  # ['', 'bc']  ← findall はグループ部分を返す
```

### 選択 `|`（OR条件）

パイプ `|` を使うと、**左右のパターンのどちらかにマッチ**するOR条件を書けます。

```python
import re

text = "I have a cat and a dog and a bird"

# cat|dog → "cat" または "dog"
results = re.findall(r"cat|dog", text)
print(results)  # ['cat', 'dog']
```

```python
import re

text = "Monday Tuesday Wednesday Thursday Friday"

# 月|火|水|木|金 に相当する英語の曜日
results = re.findall(r"Monday|Tuesday|Wednesday|Thursday|Friday", text)
print(results)  # ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday']
```

### グループ化と選択の組み合わせ

`|` は**パターン全体**に適用されます。一部分だけにOR条件を適用したい場合は、`()` でグループ化します。

```python
import re

text = "grayscale greyscale grayish greyish"

# gray|grey → "gray" または "grey"
results = re.findall(r"gr(a|e)y", text)
print(results)  # ['a', 'e', 'a', 'e']
# ← findall はグループ内の部分を返す

# グループ全体のマッチを確認したい場合は finditer を使う
for m in re.finditer(r"gr(a|e)y", text):
    print(m.group())
# gray
# grey
# gray
# grey
```

```
【 | の適用範囲に注意】

abc|def       → "abc" または "def"（パターン全体がOR）
ab(c|d)ef     → "abcef" または "abdef"（c と d だけがOR）
```

> **よくある間違い**: `cat|dog food` は「cat」または「dog food」にマッチします（`|` はパターン全体を区切る）。「cat food」または「dog food」にマッチさせたい場合は `(cat|dog) food` とグループ化が必要です。

## 3.3 キャプチャグループと非キャプチャグループ

### キャプチャグループ

`()` で囲んだ部分はキャプチャグループ（Capture Group）となり、マッチした部分を**後から取り出す**ことができます。

```python
import re

text = "2026-04-05"

# () でグループ化して、年・月・日を個別に取得
result = re.search(r"(\d{4})-(\d{2})-(\d{2})", text)

if result:
    print(f"全体: {result.group()}")     # 2026-04-05
    print(f"年: {result.group(1)}")      # 2026
    print(f"月: {result.group(2)}")      # 04
    print(f"日: {result.group(3)}")      # 05
```

```
【キャプチャグループの番号】

パターン: (\d{4})-(\d{2})-(\d{2})
           ──┬──    ──┬──    ──┬──
          グループ1 グループ2 グループ3

group()  → 全体のマッチ "2026-04-05"
group(1) → 1番目のグループ "2026"
group(2) → 2番目のグループ "04"
group(3) → 3番目のグループ "05"
```

### re.findall() とグループの関係

`re.findall()` は、パターンにグループが含まれる場合、**グループ内の文字列だけ**を返します。グループが複数ある場合はタプルのリストになります。

```python
import re

text = "2026-04-05 と 2025-12-31"

# グループなし → マッチ全体を返す
results = re.findall(r"\d{4}-\d{2}-\d{2}", text)
print(results)  # ['2026-04-05', '2025-12-31']

# グループ1つ → グループ内の文字列を返す
results = re.findall(r"(\d{4})-\d{2}-\d{2}", text)
print(results)  # ['2026', '2025']  ← 年だけ

# グループ複数 → タプルのリストを返す
results = re.findall(r"(\d{4})-(\d{2})-(\d{2})", text)
print(results)  # [('2026', '04', '05'), ('2025', '12', '31')]
```

### 非キャプチャグループ `(?:...)`

グループ化はしたいが、キャプチャ（取得）は不要という場合は、**非キャプチャグループ（Non-capturing Group）** `(?:...)` を使います。

```python
import re

text = "gray grey grayish greyish"

# キャプチャグループ → findall がグループ内を返してしまう
results = re.findall(r"gr(a|e)y", text)
print(results)  # ['a', 'e', 'a', 'e']  ← a か e だけ返る

# 非キャプチャグループ → findall が全体のマッチを返す
results = re.findall(r"gr(?:a|e)y", text)
print(results)  # ['gray', 'grey', 'gray', 'grey']  ← 全体が返る
```

```
【キャプチャ vs 非キャプチャ】

(abc)    → グループ化 ＋ キャプチャ（後から取り出せる）
(?:abc)  → グループ化のみ（キャプチャしない）

使い分け：
  - 後から取り出したい → ()
  - 量指定子やORの適用範囲だけ指定したい → (?:...)
```

> **実務でのポイント**: `re.findall()` で全体のマッチを取得したいのに、グループ部分だけが返ってきて困ることがよくあります。キャプチャが不要なグループには `(?:...)` を使う癖をつけましょう。

## 3.4 re.findall() でグループを使った抽出テクニック

### テクニック1：必要な部分だけを抽出

```python
import re

text = "価格: 1,500円、送料: 500円、合計: 2,000円"

# 「○○円」の数字部分だけを取得
results = re.findall(r"(\d[\d,]+)円", text)
print(results)  # ['1,500', '500', '2,000']
```

### テクニック2：key=value 形式からの抽出

```python
import re

config = """
host=localhost
port=8080
debug=true
name=my_app
"""

# key=value のペアをタプルで取得
results = re.findall(r"(\w+)=(\w+)", config)
print(results)
# [('host', 'localhost'), ('port', '8080'), ('debug', 'true'), ('name', 'my_app')]

# 辞書に変換
config_dict = dict(results)
print(config_dict)
# {'host': 'localhost', 'port': '8080', 'debug': 'true', 'name': 'my_app'}
```

### テクニック3：特定の前後文脈を含む抽出

```python
import re

text = "バージョン: v1.2.3, 旧バージョン: v0.9.1"

# "v" の後にバージョン番号がある部分を取得（v は含めない）
results = re.findall(r"v(\d+\.\d+\.\d+)", text)
print(results)  # ['1.2.3', '0.9.1']
```

## 3.5 実用例

### HTMLタグと中身の抽出

```python
import re

html = """
<h1>タイトル</h1>
<p>本文テキストです。</p>
<p>2番目の段落です。</p>
"""

# タグ名と中身を同時に取得
results = re.findall(r"<(\w+)>(.+?)</\1>", html)
print(results)
# [('h1', 'タイトル'), ('p', '本文テキストです。'), ('p', '2番目の段落です。')]

# <p>タグの中身だけ取得
results = re.findall(r"<p>(.+?)</p>", html)
print(results)
# ['本文テキストです。', '2番目の段落です。']
```

### 日付パターンの抽出

```python
import re

text = """
会議日程:
  2026/04/05（月）10:00〜
  2026/04/12（月）14:00〜
  2026-05-01 は祝日
"""

# YYYY/MM/DD または YYYY-MM-DD 形式の日付を抽出
results = re.findall(r"(\d{4})[/-](\d{2})[/-](\d{2})", text)
print(results)
# [('2026', '04', '05'), ('2026', '04', '12'), ('2026', '05', '01')]

# 全体のマッチを取得（非キャプチャグループを使用）
results = re.findall(r"\d{4}[/-]\d{2}[/-]\d{2}", text)
print(results)
# ['2026/04/05', '2026/04/12', '2026-05-01']
```

### URL からドメイン名を抽出

```python
import re

text = """
参考サイト:
https://www.example.com/page1
http://docs.python.org/3/library/re.html
https://github.com/user/repo
"""

# URLからドメイン名を抽出
results = re.findall(r"https?://([^/\s]+)", text)
print(results)
# ['www.example.com', 'docs.python.org', 'github.com']
```

```
【URLパターンの分解】

https?://([^/\s]+)

https?   → "http" + "s" が0回か1回（http または https）
://      → リテラルの "://"
([^/\s]+) → / と空白以外の1文字以上（= ドメイン名）をキャプチャ
```
