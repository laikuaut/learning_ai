## この章の演習で身につくこと

- 論理演算子（`&&`, `||`, `!`）を正しく使える

## 問題1：偶数・奇数判定（基本）

整数を入力として受け取り、その数が偶数か奇数かを判定して表示するプログラムを作成してください。

さらに、0 の場合は「0は偶数です」と表示してください。

### 期待される出力例

```
整数を入力してください: 7
7は奇数です
```

```
整数を入力してください: 12
12は偶数です
```

```
整数を入力してください: 0
0は偶数です
```

<details>
<summary>ヒント</summary>

- 偶数かどうかは `n % 2 == 0` で判定できます
- `if` と `else` の2分岐で十分です
- 0 を 2 で割った余りは 0 なので、特別な処理は不要です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int n;
    std::cout << "整数を入力してください: ";
    std::cin >> n;

    // 偶数・奇数の判定
    // 2で割った余りが0なら偶数、それ以外なら奇数
    if (n % 2 == 0) {
        std::cout << n << "は偶数です" << std::endl;
    } else {
        std::cout << n << "は奇数です" << std::endl;
    }

    return 0;
}
```

</details>

## 問題2：成績判定プログラム（基本）

テストの点数（0〜100）を入力として受け取り、以下の基準で成績を判定して表示するプログラムを作成してください。

| 点数 | 成績 |
|------|------|
| 90〜100 | A（優秀） |
| 80〜89 | B（良好） |
| 70〜79 | C（普通） |
| 60〜69 | D（要努力） |
| 0〜59 | F（不合格） |

入力が 0〜100 の範囲外の場合は「不正な入力です」と表示してください。

### 期待される出力例

```
テストの点数を入力してください: 85
成績: B（良好）
```

```
テストの点数を入力してください: 42
成績: F（不合格）
```

```
テストの点数を入力してください: 105
不正な入力です
```

<details>
<summary>ヒント</summary>

- 範囲外チェックを最初に行いましょう（`score < 0 || score > 100`）
- `if` → `else if` → `else` の連鎖で上から順に判定します
- 大きい値から順に判定すると条件式がシンプルになります

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int score;
    std::cout << "テストの点数を入力してください: ";
    std::cin >> score;

    // まず範囲外チェック
    if (score < 0 || score > 100) {
        std::cout << "不正な入力です" << std::endl;
    }
    // 上の点数帯から順に判定
    else if (score >= 90) {
        std::cout << "成績: A（優秀）" << std::endl;
    }
    else if (score >= 80) {
        std::cout << "成績: B（良好）" << std::endl;
    }
    else if (score >= 70) {
        std::cout << "成績: C（普通）" << std::endl;
    }
    else if (score >= 60) {
        std::cout << "成績: D（要努力）" << std::endl;
    }
    else {
        std::cout << "成績: F（不合格）" << std::endl;
    }

    return 0;
}
```

</details>

## 問題3：うるう年判定（応用）

西暦（年）を入力として受け取り、その年がうるう年かどうかを判定するプログラムを作成してください。

**うるう年の条件：**
1. 4で割り切れる年はうるう年
2. ただし、100で割り切れる年はうるう年ではない
3. ただし、400で割り切れる年はうるう年

### 期待される出力例

```
西暦を入力してください: 2024
2024年はうるう年です
```

```
西暦を入力してください: 1900
1900年はうるう年ではありません
```

```
西暦を入力してください: 2000
2000年はうるう年です
```

```
西暦を入力してください: 2023
2023年はうるう年ではありません
```

<details>
<summary>ヒント</summary>

- 条件を整理すると：`(4で割り切れる && 100で割り切れない) || 400で割り切れる`
- `&&`（AND）と `||`（OR）を組み合わせて1つの条件式にまとめることもできます
- `if-else if-else` で段階的に判定する方法もあります

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int year;
    std::cout << "西暦を入力してください: ";
    std::cin >> year;

    // うるう年の判定
    // 条件: (4で割り切れる AND 100で割り切れない) OR 400で割り切れる
    bool is_leap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);

    if (is_leap) {
        std::cout << year << "年はうるう年です" << std::endl;
    } else {
        std::cout << year << "年はうるう年ではありません" << std::endl;
    }

    return 0;
}
```

</details>

## 問題4：じゃんけんゲーム（応用）

ユーザーがグー（0）、チョキ（1）、パー（2）のいずれかを入力し、コンピューターの手（固定値でOK）と勝敗を判定するプログラムを作成してください。

- 不正な入力（0〜2以外）にはエラーメッセージを表示してください
- コンピューターの手は `1`（チョキ）に固定してかまいません（乱数は次章以降で学びます）

### 期待される出力例

```
じゃんけん！（0:グー, 1:チョキ, 2:パー）: 0
あなた: グー
コンピューター: チョキ
結果: あなたの勝ち！
```

```
じゃんけん！（0:グー, 1:チョキ, 2:パー）: 2
あなた: パー
コンピューター: チョキ
結果: あなたの負け...
```

```
じゃんけん！（0:グー, 1:チョキ, 2:パー）: 1
あなた: チョキ
コンピューター: チョキ
結果: あいこ！
```

<details>
<summary>ヒント</summary>

- `switch` 文を使うと、数値に対応する手の名前を取得しやすくなります
- 勝敗判定のロジック：`(自分の手 - 相手の手 + 3) % 3` の結果が 0 ならあいこ、1 なら負け、2 なら勝ちです
- もしくは `if-else` で全パターンを書く方法でもOKです

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>

int main() {
    int player;
    int computer = 1; // コンピューターはチョキ固定

    std::cout << "じゃんけん！（0:グー, 1:チョキ, 2:パー）: ";
    std::cin >> player;

    // 入力チェック
    if (player < 0 || player > 2) {
        std::cout << "不正な入力です。0〜2を入力してください。" << std::endl;
        return 1;
    }

    // 手の名前を決定（switch文の練習）
    std::string player_hand, computer_hand;

    switch (player) {
        case 0: player_hand = "グー"; break;
        case 1: player_hand = "チョキ"; break;
        case 2: player_hand = "パー"; break;
    }

    switch (computer) {
        case 0: computer_hand = "グー"; break;
        case 1: computer_hand = "チョキ"; break;
        case 2: computer_hand = "パー"; break;
    }

    // 手の表示
    std::cout << "あなた: " << player_hand << std::endl;
    std::cout << "コンピューター: " << computer_hand << std::endl;

    // 勝敗判定
    // (player - computer + 3) % 3 → 0:あいこ, 1:負け, 2:勝ち
    int result = (player - computer + 3) % 3;

    if (result == 0) {
        std::cout << "結果: あいこ！" << std::endl;
    } else if (result == 2) {
        std::cout << "結果: あなたの勝ち！" << std::endl;
    } else {
        std::cout << "結果: あなたの負け..." << std::endl;
    }

    return 0;
}
```

</details>

## 問題5：簡易電卓プログラム（応用）

2つの数値と演算子（`+`, `-`, `*`, `/`）を入力として受け取り、計算結果を表示するプログラムを作成してください。

- 演算子は `char` 型で受け取ります
- `switch` 文を使って演算子ごとに処理を分けてください
- 0での除算にはエラーメッセージを表示してください
- 未対応の演算子にもエラーメッセージを表示してください

### 期待される出力例

```
1つ目の数値: 10
演算子（+, -, *, /）: *
2つ目の数値: 3
10 * 3 = 30
```

```
1つ目の数値: 10
演算子（+, -, *, /）: /
2つ目の数値: 0
エラー: 0で割ることはできません
```

```
1つ目の数値: 7
演算子（+, -, *, /）: %
2つ目の数値: 2
エラー: 未対応の演算子です
```

<details>
<summary>ヒント</summary>

- `char op;` で演算子を受け取り、`switch (op)` で分岐します
- `case '+':`のように文字リテラルを使います
- 除算の前に `b == 0` のチェックを入れましょう
- `default:` で未対応の演算子を処理します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    double a, b;
    char op;

    // 入力
    std::cout << "1つ目の数値: ";
    std::cin >> a;

    std::cout << "演算子（+, -, *, /）: ";
    std::cin >> op;

    std::cout << "2つ目の数値: ";
    std::cin >> b;

    // switch文で演算子ごとに処理を分岐
    switch (op) {
        case '+':
            std::cout << a << " + " << b << " = " << a + b << std::endl;
            break;

        case '-':
            std::cout << a << " - " << b << " = " << a - b << std::endl;
            break;

        case '*':
            std::cout << a << " * " << b << " = " << a * b << std::endl;
            break;

        case '/':
            // 0除算チェック
            if (b == 0) {
                std::cout << "エラー: 0で割ることはできません" << std::endl;
            } else {
                std::cout << a << " / " << b << " = " << a / b << std::endl;
            }
            break;

        default:
            // 未対応の演算子
            std::cout << "エラー: 未対応の演算子です" << std::endl;
            break;
    }

    return 0;
}
```

</details>

## 問題6：料金計算プログラム（チャレンジ）

映画館の入場料金を計算するプログラムを作成してください。

**料金表：**

| 区分 | 通常料金 |
|------|---------|
| 子供（0〜11歳） | 800円 |
| 中高生（12〜17歳） | 1200円 |
| 大人（18〜64歳） | 1800円 |
| シニア（65歳以上） | 1200円 |

**割引条件：**
- 毎月1日（会員デー）は全員 200円引き
- レイトショー（20時以降の上映）は大人のみ 400円引き
- 割引は重複適用可能

年齢、日付（日のみ）、上映開始時間（時のみ）を入力として受け取ってください。

### 期待される出力例

```
年齢を入力してください: 30
日付（日のみ）を入力してください: 15
上映開始時間（時のみ）を入力してください: 21
--- 料金計算 ---
区分: 大人
基本料金: 1800円
レイトショー割引: -400円
お支払い金額: 1400円
```

```
年齢を入力してください: 30
日付（日のみ）を入力してください: 1
上映開始時間（時のみ）を入力してください: 21
--- 料金計算 ---
区分: 大人
基本料金: 1800円
会員デー割引: -200円
レイトショー割引: -400円
お支払い金額: 1200円
```

```
年齢を入力してください: 10
日付（日のみ）を入力してください: 1
上映開始時間（時のみ）を入力してください: 14
--- 料金計算 ---
区分: 子供
基本料金: 800円
会員デー割引: -200円
お支払い金額: 600円
```

<details>
<summary>ヒント</summary>

- まず年齢で区分と基本料金を決定します
- 割引額を変数で管理し、条件に合うたびに加算していきます
- 三項演算子を使うと区分名の決定を簡潔に書けます
  - 例：`std::string category = (age <= 11) ? "子供" : ...;`
- 最終金額 = 基本料金 - 割引合計

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>

int main() {
    int age, day, hour;

    // 入力
    std::cout << "年齢を入力してください: ";
    std::cin >> age;

    std::cout << "日付（日のみ）を入力してください: ";
    std::cin >> day;

    std::cout << "上映開始時間（時のみ）を入力してください: ";
    std::cin >> hour;

    // 区分と基本料金の決定
    std::string category;
    int base_price;

    if (age <= 11) {
        category = "子供";
        base_price = 800;
    } else if (age <= 17) {
        category = "中高生";
        base_price = 1200;
    } else if (age <= 64) {
        category = "大人";
        base_price = 1800;
    } else {
        category = "シニア";
        base_price = 1200;
    }

    // 割引計算
    int member_discount = 0;   // 会員デー割引
    int late_discount = 0;     // レイトショー割引

    // 毎月1日は全員200円引き
    if (day == 1) {
        member_discount = 200;
    }

    // レイトショー（20時以降）は大人のみ400円引き
    if (hour >= 20 && category == "大人") {
        late_discount = 400;
    }

    // 最終金額の計算
    int total = base_price - member_discount - late_discount;

    // 結果表示
    std::cout << "--- 料金計算 ---" << std::endl;
    std::cout << "区分: " << category << std::endl;
    std::cout << "基本料金: " << base_price << "円" << std::endl;

    if (member_discount > 0) {
        std::cout << "会員デー割引: -" << member_discount << "円" << std::endl;
    }
    if (late_discount > 0) {
        std::cout << "レイトショー割引: -" << late_discount << "円" << std::endl;
    }

    std::cout << "お支払い金額: " << total << "円" << std::endl;

    return 0;
}
```

</details>
