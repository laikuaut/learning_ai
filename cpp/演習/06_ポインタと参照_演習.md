## 問題1：ポインタを使ったswap関数（基本）

2つの整数のポインタを受け取り、値を入れ替える関数 `swapPtr` を作成してください。

**期待される出力例：**
```
交換前: a=10, b=20
交換後: a=20, b=10
```

<details>
<summary>ヒント</summary>

- 関数のシグネチャは `void swapPtr(int* a, int* b)` です
- ポインタの指す値にアクセスするには `*a` のようにデリファレンス（間接参照）します
- 一時変数 `temp` を使って値を入れ替えます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// ポインタを使って2つの値を交換する関数
void swapPtr(int* a, int* b) {
    int temp = *a;  // aが指す値を一時保存
    *a = *b;        // aが指す場所にbが指す値を代入
    *b = temp;      // bが指す場所に一時保存した値を代入
}

int main() {
    int a = 10, b = 20;
    cout << "交換前: a=" << a << ", b=" << b << endl;

    swapPtr(&a, &b);  // アドレスを渡す

    cout << "交換後: a=" << a << ", b=" << b << endl;
    return 0;
}
```

</details>

## 問題2：参照を使った値の交換（基本）

問題1と同じ処理を、今度は**参照**を使って実装してください。
関数名は `swapRef` とします。

**期待される出力例：**
```
交換前: x=100, y=200
交換後: x=200, y=100
```

<details>
<summary>ヒント</summary>

- 関数のシグネチャは `void swapRef(int& a, int& b)` です
- 参照の場合、`*` や `&` を使わずにそのまま変数名でアクセスできます
- 呼び出し側も `swapRef(x, y)` のように通常の関数呼び出しと同じ書き方です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// 参照を使って2つの値を交換する関数
void swapRef(int& a, int& b) {
    int temp = a;
    a = b;
    b = temp;
}

int main() {
    int x = 100, y = 200;
    cout << "交換前: x=" << x << ", y=" << y << endl;

    swapRef(x, y);  // 通常の関数呼び出しと同じ

    cout << "交換後: x=" << x << ", y=" << y << endl;
    return 0;
}
```

</details>

## 問題3：動的配列の確保と操作（基本）

ユーザーから要素数 n を受け取り、`new` で整数配列を動的に確保してください。
1 から n までの値を格納し、合計を計算して表示した後、`delete[]` で解放してください。

**期待される出力例：**
```
要素数を入力: 5
配列: 1 2 3 4 5
合計: 15
メモリを解放しました
```

<details>
<summary>ヒント</summary>

- `int* arr = new int[n];` で動的に配列を確保します
- 使い終わったら `delete[] arr;` で必ず解放します
- `delete[]` を忘れるとメモリリーク（memory leak）が発生します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "要素数を入力: ";
    cin >> n;

    // 動的にメモリを確保
    int* arr = new int[n];

    // 1からnまでの値を格納
    for (int i = 0; i < n; i++) {
        arr[i] = i + 1;
    }

    // 配列の中身を表示
    cout << "配列:";
    for (int i = 0; i < n; i++) {
        cout << " " << arr[i];
    }
    cout << endl;

    // 合計を計算
    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += arr[i];
    }
    cout << "合計: " << sum << endl;

    // メモリを解放
    delete[] arr;
    cout << "メモリを解放しました" << endl;

    return 0;
}
```

</details>

## 問題4：unique_ptr の基本的な使い方（応用）

問題3と同じ処理を `std::unique_ptr` を使って書き直してください。
`delete` を手動で呼ぶ必要がなくなることを確認してください。

**期待される出力例：**
```
要素数を入力: 5
配列: 1 2 3 4 5
合計: 15
スコープ終了時に自動的にメモリが解放されます
```

<details>
<summary>ヒント</summary>

- `<memory>` をインクルードします
- `unique_ptr<int[]> arr(new int[n]);` または `make_unique<int[]>(n)` で作成します
- 配列要素には `arr[i]` でアクセスできます（通常の配列と同じ）
- `unique_ptr` はスコープを抜けると自動的に `delete` されます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <memory>
using namespace std;

int main() {
    int n;
    cout << "要素数を入力: ";
    cin >> n;

    // unique_ptrで動的配列を確保（C++14以降）
    auto arr = make_unique<int[]>(n);

    // 1からnまでの値を格納
    for (int i = 0; i < n; i++) {
        arr[i] = i + 1;
    }

    // 配列の中身を表示
    cout << "配列:";
    for (int i = 0; i < n; i++) {
        cout << " " << arr[i];
    }
    cout << endl;

    // 合計を計算
    int sum = 0;
    for (int i = 0; i < n; i++) {
        sum += arr[i];
    }
    cout << "合計: " << sum << endl;

    cout << "スコープ終了時に自動的にメモリが解放されます" << endl;
    // delete不要！unique_ptrが自動で解放する
    return 0;
}
```

</details>

## 問題5：ポインタで配列を逆順にする関数（応用）

配列の先頭ポインタと末尾ポインタを受け取り、配列を逆順にする関数 `reverseArray` を作成してください。
ポインタ演算を使って実装してください。

**期待される出力例：**
```
元の配列: 1 2 3 4 5
逆順の配列: 5 4 3 2 1
```

<details>
<summary>ヒント</summary>

- 関数のシグネチャは `void reverseArray(int* begin, int* end)` です
- `end` は最後の要素の「次」を指すポインタ（STLの慣例に合わせる）とします
- 先頭と末尾のポインタを近づけながら値を交換していきます
- ポインタの比較（`begin < end`）やインクリメント（`begin++`）が使えます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// ポインタを使って配列を逆順にする
void reverseArray(int* begin, int* end) {
    // endは最後の要素の「次」を指すので、1つ戻す
    end--;
    while (begin < end) {
        // 値を交換
        int temp = *begin;
        *begin = *end;
        *end = temp;
        begin++;  // 先頭側を進める
        end--;    // 末尾側を戻す
    }
}

// 配列を表示するヘルパー関数
void printArray(int* arr, int size) {
    for (int i = 0; i < size; i++) {
        if (i > 0) cout << " ";
        cout << arr[i];
    }
    cout << endl;
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int size = 5;

    cout << "元の配列: ";
    printArray(arr, size);

    reverseArray(arr, arr + size);

    cout << "逆順の配列: ";
    printArray(arr, size);

    return 0;
}
```

</details>

## 問題6：簡易リンクリスト（チャレンジ）

ポインタを使って簡易的な片方向リンクリスト（singly linked list）を実装してください。
以下の操作を実装します。

1. `pushFront` — リストの先頭にノードを追加
2. `printList` — リストの全要素を表示
3. `deleteList` — リスト全体を解放

**期待される出力例：**
```
リストに追加: 10, 20, 30
リストの内容: 30 -> 20 -> 10 -> null
メモリを解放しました
```

<details>
<summary>ヒント</summary>

- ノードは構造体で定義します: `struct Node { int data; Node* next; };`
- `pushFront` は新しいノードを作り、`next` を現在の先頭に設定します
- `deleteList` は先頭から順に `delete` しながら次のノードに進みます
- リストの先頭は `Node*` 型のポインタで管理します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// ノード構造体
struct Node {
    int data;     // データ
    Node* next;   // 次のノードへのポインタ
};

// リストの先頭にノードを追加
void pushFront(Node*& head, int value) {
    Node* newNode = new Node;   // 新しいノードを作成
    newNode->data = value;      // データを設定
    newNode->next = head;       // 次のノードを現在の先頭に
    head = newNode;             // 先頭を新しいノードに更新
}

// リストの全要素を表示
void printList(Node* head) {
    Node* current = head;
    while (current != nullptr) {
        cout << current->data << " -> ";
        current = current->next;
    }
    cout << "null" << endl;
}

// リスト全体を解放
void deleteList(Node*& head) {
    while (head != nullptr) {
        Node* temp = head;      // 現在のノードを一時保存
        head = head->next;      // 先頭を次のノードに進める
        delete temp;            // 一時保存したノードを解放
    }
}

int main() {
    Node* head = nullptr;  // 空のリスト

    // リストに追加
    cout << "リストに追加: 10, 20, 30" << endl;
    pushFront(head, 10);
    pushFront(head, 20);
    pushFront(head, 30);

    // リストの内容を表示
    cout << "リストの内容: ";
    printList(head);

    // メモリを解放
    deleteList(head);
    cout << "メモリを解放しました" << endl;

    return 0;
}
```

</details>

## 問題7：shared_ptr でリソース共有（チャレンジ）

`std::shared_ptr` を使って、複数の変数から同じオブジェクトを共有する例を作成してください。
`use_count()` で参照カウントがどのように変化するかを確認してください。

**期待される出力例：**
```
=== shared_ptr の参照カウント ===
ptr1作成後: use_count = 1
ptr2にコピー後: use_count = 2
ptr3にコピー後: use_count = 3
ptr2をリセット後: use_count = 2
ブロック内でptr4作成: use_count = 3
ブロック終了後: use_count = 2
値: 42
```

<details>
<summary>ヒント</summary>

- `<memory>` をインクルードします
- `make_shared<int>(42)` で `shared_ptr` を作成します
- `use_count()` で現在の参照カウントを取得できます
- `reset()` で所有権を放棄します
- ブロック `{}` を使うとスコープを限定できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <memory>
using namespace std;

int main() {
    cout << "=== shared_ptr の参照カウント ===" << endl;

    // shared_ptrを作成
    shared_ptr<int> ptr1 = make_shared<int>(42);
    cout << "ptr1作成後: use_count = " << ptr1.use_count() << endl;

    // コピーで共有
    shared_ptr<int> ptr2 = ptr1;
    cout << "ptr2にコピー後: use_count = " << ptr1.use_count() << endl;

    shared_ptr<int> ptr3 = ptr1;
    cout << "ptr3にコピー後: use_count = " << ptr1.use_count() << endl;

    // ptr2の所有権を放棄
    ptr2.reset();
    cout << "ptr2をリセット後: use_count = " << ptr1.use_count() << endl;

    // スコープ限定のコピー
    {
        shared_ptr<int> ptr4 = ptr1;
        cout << "ブロック内でptr4作成: use_count = " << ptr1.use_count() << endl;
    }
    // ptr4はスコープを抜けて破棄される
    cout << "ブロック終了後: use_count = " << ptr1.use_count() << endl;

    cout << "値: " << *ptr1 << endl;

    return 0;
}
```

</details>

## 問題8：ポインタの落とし穴を理解する（チャレンジ）

以下のコードにはメモリに関するバグが含まれています。
それぞれのバグを特定し、修正したコードを書いてください。

```cpp
// バグ1: ダングリングポインタ
int* createValue() {
    int x = 42;
    return &x;
}

// バグ2: メモリリーク
void processData() {
    int* data = new int[100];
    data[0] = 1;
    // ... 処理 ...
    // return してしまい delete[] されない
}

// バグ3: 二重解放
void doubleFree() {
    int* p = new int(10);
    int* q = p;
    delete p;
    delete q;
}
```

**期待される出力例（修正版）：**
```
=== バグ1修正: 動的確保を使う ===
値: 42
=== バグ2修正: unique_ptrを使う ===
data[0] = 1
=== バグ3修正: shared_ptrを使う ===
値: 10
```

<details>
<summary>ヒント</summary>

- バグ1: ローカル変数のアドレスを返してはいけません（ダングリングポインタ）。`new` で確保するか、値を返しましょう
- バグ2: `new` したら必ず `delete` が必要です。スマートポインタを使えば自動管理できます
- バグ3: 同じメモリを2回 `delete` すると未定義動作になります。`shared_ptr` なら参照カウントで安全に管理できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <memory>
using namespace std;

// バグ1修正: 動的確保したポインタを返す（またはスマートポインタを使う）
unique_ptr<int> createValue() {
    auto p = make_unique<int>(42);
    return p;  // ムーブされて呼び出し元に渡る
}

// バグ2修正: unique_ptrを使えばdelete忘れがない
void processData() {
    auto data = make_unique<int[]>(100);
    data[0] = 1;
    cout << "data[0] = " << data[0] << endl;
    // unique_ptrが自動的にメモリを解放する
}

// バグ3修正: shared_ptrで共有管理する
void fixedDoubleFree() {
    shared_ptr<int> p = make_shared<int>(10);
    shared_ptr<int> q = p;  // 参照カウントが増える
    cout << "値: " << *p << endl;
    // shared_ptrが参照カウントを管理し、最後の1つが破棄されたときに解放する
}

int main() {
    cout << "=== バグ1修正: 動的確保を使う ===" << endl;
    auto val = createValue();
    cout << "値: " << *val << endl;

    cout << "=== バグ2修正: unique_ptrを使う ===" << endl;
    processData();

    cout << "=== バグ3修正: shared_ptrを使う ===" << endl;
    fixedDoubleFree();

    return 0;
}
```

</details>
