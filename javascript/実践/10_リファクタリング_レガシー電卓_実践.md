# 実践課題10：リファクタリング ─ レガシー電卓アプリ ★4

> **難易度**: ★★★★☆（上級）
> **前提知識**: 第1章〜第13章（全章の知識を活用）
> **課題の種類**: リファクタリング
> **学習目標**: 動作するが品質の低いコードを読み解き、ES6+の機能・モジュール設計パターン・DRY原則を活用して段階的に改善する

## 課題の説明

以下の「電卓アプリ」は正しく動作しますが、多くの問題を抱えています。
コードを段階的にリファクタリング（動作を変えずにコードの品質を改善）してください。

### 問題点の概要

1. `var` が使われている
2. 関数が巨大すぎる（1つの関数に多くの責務）
3. グローバル変数が散らばっている
4. マジックナンバー（意味不明な数値）がある
5. 重複コードがある
6. `==` が使われている
7. 文字列結合に `+` を使っている
8. エラー処理がない
9. `innerHTML` に直接文字列を入れている（XSSリスク）
10. イベントハンドラが `onclick` 属性で書かれている

      previousInput = "";
      operator = "";

    }

## リファクタリング課題

以下の10個の改善を段階的に行ってください。1つずつ改善し、そのたびに電卓が正しく動作することを確認しましょう。

<details>
<summary>課題1：var を let / const に置き換える</summary>

**問題:** `var` はスコープが関数レベルで、巻き上げ（hoisting）もあるため、意図しないバグの原因になります。

**やること:**
- 再代入する変数は `let`
- 再代入しない変数は `const`

```javascript
// 修正前
var currentInput = "";
var historyList = [];

// 修正後
let currentInput = "";
const historyList = [];  // 配列自体は再代入しないがpushはOK
```

</details>

<details>
<summary>課題2：== を === に置き換える</summary>

**問題:** `==` は型変換を行うため、予期しない一致が起きることがあります。

**やること:**
- 全ての `==` を `===` に変更
- 全ての `!=` を `!==` に変更

```javascript
// 修正前
if (num == "." && currentInput.indexOf(".") != -1) return;
// 修正後
if (num === "." && currentInput.indexOf(".") !== -1) return;
```

</details>

<details>
<summary>課題3：テンプレートリテラルを使う</summary>

**問題:** 文字列結合に `+` を使うとコードが読みにくくなります。

**やること:**
- 文字列結合をテンプレートリテラルに置き換える

```javascript
// 修正前
var entry = previousInput + " " + opSymbol + " " + currentInput + " = " + result;
// 修正後
const entry = `${previousInput} ${opSymbol} ${currentInput} = ${result}`;
```

</details>

<details>
<summary>課題4：演算子のマッピングをオブジェクトで管理する</summary>

**問題:** `if-else` の連鎖で演算子を判定しており、新しい演算子の追加が面倒です。

**やること:**
- 演算子と関数のマッピングをオブジェクトで定義する

```javascript
// 修正後
const operations = {
  "+": (a, b) => a + b,
  "-": (a, b) => a - b,
  "*": (a, b) => a * b,
  "/": (a, b) => a / b,
};

const opSymbols = {
  "+": "+", "-": "-", "*": "×", "/": "÷",
};

// calculate内
const operation = operations[operator];
if (!operation) return;
const result = operation(num1, num2);
```

</details>

<details>
<summary>課題5：マジックナンバーを定数に置き換える</summary>

**問題:** `12`, `100000000`, `5` などの数値の意味が不明です。

```javascript
// 修正後
const MAX_DISPLAY_LENGTH = 12;
const DECIMAL_PRECISION = 8;
const HISTORY_DISPLAY_COUNT = 5;

// 使用箇所
if (currentInput.length > MAX_DISPLAY_LENGTH) return;
result = Math.round(result * 10 ** DECIMAL_PRECISION) / 10 ** DECIMAL_PRECISION;
const start = Math.max(0, historyList.length - HISTORY_DISPLAY_COUNT);
```

</details>

<details>
<summary>課題6：innerHTML を安全なDOM操作に置き換える</summary>

**問題:** `innerHTML` に文字列を直接入れるとXSSのリスクがあります。

```javascript
// 修正後
function renderHistory() {
  const historyEl = document.getElementById("history");
  historyEl.innerHTML = "";  // クリア

  const title = document.createElement("b");
  title.textContent = "計算履歴:";
  historyEl.appendChild(title);

  const start = Math.max(0, historyList.length - HISTORY_DISPLAY_COUNT);
  for (let i = start; i < historyList.length; i++) {
    historyEl.appendChild(document.createElement("br"));
    const text = document.createTextNode(historyList[i]);
    historyEl.appendChild(text);
  }
}
```

</details>

<details>
<summary>課題7：onclick属性を addEventListener に置き換える</summary>

**問題:** HTML属性でのイベントハンドラはグローバルスコープに依存し、保守性が悪いです。

```html
<!-- 修正前 -->
<button class="btn" onclick="press('7')">7</button>

<!-- 修正後: data属性で値を保持 -->
<button class="btn" data-value="7">7</button>
```

```javascript
// JavaScript側でイベントを登録
document.querySelectorAll(".btn").forEach(btn => {
  const value = btn.dataset.value;
  const op = btn.dataset.op;
  const action = btn.dataset.action;

  btn.addEventListener("click", () => {
    if (value !== undefined) press(value);
    else if (op !== undefined) pressOp(op);
    else if (action === "calc") calculate();
    else if (action === "clear") clearAll();
    else if (action === "back") backspace();
  });
});
```

</details>

<details>
<summary>課題8：状態をオブジェクトにまとめる</summary>

**問題:** グローバル変数が散らばっていると、状態の管理が困難です。

```javascript
// 修正後: 状態を1つのオブジェクトに集約
const state = {
  currentInput: "",
  previousInput: "",
  operator: "",
  history: [],
};

// 各関数内で state.currentInput のようにアクセス
function clearAll() {
  state.currentInput = "";
  state.previousInput = "";
  state.operator = "";
  updateDisplay();
}
```

</details>

<details>
<summary>課題9：表示更新関数を分離する</summary>

**問題:** 各関数内で `document.getElementById("display").value = ...` を繰り返しています。

```javascript
// 修正後: 表示更新を1つの関数にまとめる
function updateDisplay(value) {
  document.getElementById("display").value = value !== undefined ? value : state.currentInput;
}

// 各関数では updateDisplay() を呼ぶだけ
function press(num) {
  // ... ロジック ...
  updateDisplay();
}
```

</details>

<details>

**問題:** マウスでしか操作できません。

<summary>リファクタリング済みの完全なコード</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>電卓（リファクタリング済み）</title>
  <style>
    body { font-family: monospace; text-align: center; margin-top: 40px; }

    .btn:hover { background: #e0e0e0; }
    .btn.wide { width: 134px; }
    #history { width: 280px; margin: 20px auto; text-align: left; font-size: 14px; }
  </style>
</head>
<body>
  <h2>電卓</h2>
  <input type="text" id="display" readonly>
  <div class="btn-row">
    <button class="btn" data-value="7">7</button>
    <button class="btn" data-value="8">8</button>
    <button class="btn" data-value="9">9</button>
    <button class="btn" data-op="/">÷</button>
  </div>
  <div class="btn-row">
    <button class="btn" data-value="4">4</button>
    <button class="btn" data-value="5">5</button>
    <button class="btn" data-value="6">6</button>
    <button class="btn" data-op="*">×</button>
  </div>
  <div class="btn-row">
    <button class="btn" data-value="1">1</button>
    <button class="btn" data-value="2">2</button>
    <button class="btn" data-value="3">3</button>
    <button class="btn" data-op="-">-</button>
  </div>
  <div class="btn-row">
    <button class="btn" data-value="0">0</button>
    <button class="btn" data-value=".">.</button>
    <button class="btn" data-action="calc">=</button>
    <button class="btn" data-op="+">+</button>
  </div>
  <div class="btn-row">
    <button class="btn wide" data-action="clear">C</button>
    <button class="btn wide" data-action="back">←</button>
  </div>
  <div id="history"></div>

  <script>
    // ===== 定数 =====
    const MAX_DISPLAY_LENGTH = 12;
    const DECIMAL_PRECISION = 8;
    const HISTORY_DISPLAY_COUNT = 5;

    // 演算子のマッピング
    const operations = {
      "+": (a, b) => a + b,
      "-": (a, b) => a - b,
      "*": (a, b) => a * b,
      "/": (a, b) => a / b,
    };

    const opSymbols = { "+": "+", "-": "-", "*": "×", "/": "÷" };

    // ===== 状態 =====
    const state = {
      currentInput: "",
      previousInput: "",
      operator: "",
      history: [],
    };

    // ===== 表示更新 =====
    function updateDisplay(value) {
      const displayValue = value !== undefined ? String(value) : state.currentInput;
      document.getElementById("display").value = displayValue;
    }

    function renderHistory() {
      const historyEl = document.getElementById("history");
      historyEl.innerHTML = "";

      if (state.history.length === 0) return;

      const title = document.createElement("b");
      title.textContent = "計算履歴:";
      historyEl.appendChild(title);

      const start = Math.max(0, state.history.length - HISTORY_DISPLAY_COUNT);
      for (let i = start; i < state.history.length; i++) {
        historyEl.appendChild(document.createElement("br"));
        historyEl.appendChild(document.createTextNode(state.history[i]));
      }
    }

    // ===== 操作 =====
    function press(num) {
      if (state.currentInput.length > MAX_DISPLAY_LENGTH) return;
      if (num === "." && state.currentInput.includes(".")) return;
      if (num === "." && state.currentInput === "") {
        state.currentInput = "0";
      }
      state.currentInput += num;
      updateDisplay();
    }

    function pressOp(op) {
      if (state.currentInput === "") return;
      if (state.previousInput !== "") {
        calculate();
      }
      state.operator = op;
      state.previousInput = state.currentInput;
      state.currentInput = "";
    }

    function calculate() {
      if (state.previousInput === "" || state.currentInput === "") return;

      const num1 = parseFloat(state.previousInput);
      const num2 = parseFloat(state.currentInput);

      // ゼロ除算チェック
      if (state.operator === "/" && num2 === 0) {
        updateDisplay("Error");
        state.currentInput = "";
        state.previousInput = "";
        state.operator = "";
        return;
      }

      const operation = operations[state.operator];
      if (!operation) return;

      // 計算と丸め
      const precision = 10 ** DECIMAL_PRECISION;
      let result = Math.round(operation(num1, num2) * precision) / precision;

      // 履歴に追加
      const symbol = opSymbols[state.operator] || state.operator;
      state.history.push(`${state.previousInput} ${symbol} ${state.currentInput} = ${result}`);

      // 状態を更新
      updateDisplay(result);
      state.currentInput = String(result);
      state.previousInput = "";
      state.operator = "";

      renderHistory();
    }

    function clearAll() {
      state.currentInput = "";
      state.previousInput = "";
      state.operator = "";
      updateDisplay("");
    }

    function backspace() {
      state.currentInput = state.currentInput.slice(0, -1);
      updateDisplay();
    }

    // ===== イベント登録 =====
    document.querySelectorAll(".btn").forEach(btn => {
      btn.addEventListener("click", () => {
        const { value, op, action } = btn.dataset;
        if (value !== undefined) press(value);
        else if (op !== undefined) pressOp(op);
        else if (action === "calc") calculate();
        else if (action === "clear") clearAll();
        else if (action === "back") backspace();
      });
    });

    // キーボード入力
    document.addEventListener("keydown", (e) => {
      if ((e.key >= "0" && e.key <= "9") || e.key === ".") {
        press(e.key);
      } else if (["+", "-", "*", "/"].includes(e.key)) {
        pressOp(e.key);
      } else if (e.key === "Enter" || e.key === "=") {
        e.preventDefault();
        calculate();
      } else if (e.key === "Escape") {
        clearAll();
      } else if (e.key === "Backspace") {
        backspace();
      }
    });
  </script>
</body>
</html>
```

**リファクタリングのポイントまとめ:**

| 改善 | Before | After | メリット |
|---|---|---|---|
| 変数宣言 | `var` | `let` / `const` | スコープの明確化 |
| 等価比較 | `==` | `===` | 型安全な比較 |
| 文字列結合 | `+` 演算子 | テンプレートリテラル | 可読性の向上 |
| 条件分岐 | `if-else` の連鎖 | オブジェクトマッピング | 拡張性の向上 |
| 定数 | マジックナンバー | 名前付き定数 | 意図の明確化 |
| DOM操作 | `innerHTML` | `createElement` | XSS対策 |
| イベント | `onclick` 属性 | `addEventListener` | 関心の分離 |
| 状態管理 | グローバル変数 | `state` オブジェクト | 見通しの改善 |
| 表示更新 | コード重複 | `updateDisplay()` | DRY原則 |
| 入力手段 | マウスのみ | キーボード対応 | アクセシビリティ |

</details>
