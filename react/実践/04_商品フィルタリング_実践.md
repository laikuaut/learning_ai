# 実践課題04：商品フィルタリング ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第2章（コンポーネントとprops）、第3章（stateとイベント）、第4章（条件分岐とリスト）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 複数の検索条件（テキスト・カテゴリ・価格帯）を組み合わせたフィルタリングを実装する

---

---

## 課題の要件

1. 商品データ（名前・カテゴリ・価格・評価）を配列で管理する
2. テキスト入力でリアルタイム検索（商品名の部分一致）
3. カテゴリボタンでフィルタリング
4. 価格帯ボタンでフィルタリング
5. 複数条件を同時に適用する（AND検索）
6. 該当件数を表示する
7. `ProductCard` コンポーネントを分離する

<summary>解答例（改良版 ─ ソート機能追加＋useMemoで最適化）</summary>

初心者向けに以下を追加した改良版の差分です。

```jsx
// ソートstateを追加
const [sortBy, setSortBy] = React.useState("none");

// useMemoでフィルタ＋ソート結果をキャッシュ
const filtered = React.useMemo(() => {
  let result = PRODUCTS.filter(product => {
    if (search && !product.name.includes(search)) return false;
    if (category !== "すべて" && product.category !== category) return false;
    if (priceRange === "low" && product.price > 500) return false;
    if (priceRange === "mid" && (product.price < 500 || product.price > 1000)) return false;
    if (priceRange === "high" && product.price < 1000) return false;
    return true;
  });

  // ソート
  if (sortBy === "price_asc") {
    result = [...result].sort((a, b) => a.price - b.price);
  } else if (sortBy === "price_desc") {
    result = [...result].sort((a, b) => b.price - a.price);
  } else if (sortBy === "rating") {
    result = [...result].sort((a, b) => b.rating - a.rating);
  }

  return result;
}, [search, category, priceRange, sortBy]);
```

**初心者向けとの違い:**
- `React.useMemo` でフィルタ計算をメモ化（第10章の先取り）
- ソート機能（価格昇順/降順、評価順）を追加
- フィルタリセットボタンでワンクリック初期化
