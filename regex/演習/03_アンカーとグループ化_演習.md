## 問題1：行頭アンカー ^（基本）

以下の複数行テキストから、行頭が `#` で始まる行をすべて抽出してください。

```python
text = """# 見出し1
本文テキストです。
# 見出し2
これも本文です。
## 小見出し"""
```

**期待される出力：**
```
['# 見出し1', '# 見出し2', '## 小見出し']
```

<details>
<summary>ヒント</summary>

`^#.+` で「行頭の `#` + 何らかの文字」にマッチします。複数行に対応するには `re.MULTILINE` フラグが必要です。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = """# 見出し1
本文テキストです。
# 見出し2
これも本文です。
## 小見出し"""

# ^ で行頭、#.+ で # に続く文字列にマッチ
# re.MULTILINE で各行の先頭を ^ で認識
results = re.findall(r"^#.+", text, re.MULTILINE)
print(results)  # ['# 見出し1', '# 見出し2', '## 小見出し']
```

</details>

## 問題2：単語境界 \b（基本）

以下のテキストから、独立した単語 `is` だけを検索してください。`this` や `island` の中に含まれる `is` にはマッチさせないでください。

```python
text = "this is an island where this is normal"
```

**期待される出力：**
```
['is', 'is']
```

<details>
<summary>ヒント</summary>

`\b` を使って単語境界を指定すると、独立した単語にだけマッチさせることができます。`\bis\b` のように、前後を `\b` で囲みます。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "this is an island where this is normal"

# \bis\b → 独立した単語 "is" にだけマッチ
results = re.findall(r"\bis\b", text)
print(results)  # ['is', 'is']

# 比較：\b なしだと this や island の中の is にもマッチ
results_no_boundary = re.findall(r"is", text)
print(results_no_boundary)  # ['is', 'is', 'is', 'is', 'is']
```

</details>

## 問題3：選択（OR条件）（基本）

以下のテキストから、果物の名前（`りんご`、`みかん`、`バナナ`）をすべて抽出してください。

```python
text = "今日はりんごとバナナを買いました。みかんは売り切れでした。りんごは2個です。"
```

**期待される出力：**
```
['りんご', 'バナナ', 'みかん', 'りんご']
```

<details>
<summary>ヒント</summary>

`|` を使って「りんご|みかん|バナナ」のようにOR条件を書きます。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "今日はりんごとバナナを買いました。みかんは売り切れでした。りんごは2個です。"

# | でOR条件
results = re.findall(r"りんご|みかん|バナナ", text)
print(results)  # ['りんご', 'バナナ', 'みかん', 'りんご']
```

</details>

## 問題4：グループ化とキャプチャ（基本）

以下のテキストから日付を抽出し、年・月・日をそれぞれ個別に取得してください。

```python
text = "開始日: 2026-04-01, 終了日: 2026-04-30"
```

**期待される出力：**
```
[('2026', '04', '01'), ('2026', '04', '30')]
```

<details>
<summary>ヒント</summary>

`(\d{4})-(\d{2})-(\d{2})` のように、年・月・日をそれぞれ `()` でグループ化すると、`re.findall()` がタプルのリストを返します。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "開始日: 2026-04-01, 終了日: 2026-04-30"

# 年・月・日をそれぞれキャプチャグループで囲む
results = re.findall(r"(\d{4})-(\d{2})-(\d{2})", text)
print(results)  # [('2026', '04', '01'), ('2026', '04', '30')]

# 個別に取り出す例
for year, month, day in results:
    print(f"{year}年{month}月{day}日")
# 2026年04月01日
# 2026年04月30日
```

</details>

## 問題5：非キャプチャグループ (?:...)（応用）

以下のテキストから、`http` または `https` で始まるURLをすべて抽出してください。`re.findall()` で URL全体を取得してください（プロトコル部分だけではなく）。

```python
text = """
公式サイト: https://www.example.com
ドキュメント: http://docs.example.org/guide
API: https://api.example.com/v1
"""
```

**期待される出力：**
```
['https://www.example.com', 'http://docs.example.org/guide', 'https://api.example.com/v1']
```

<details>
<summary>ヒント</summary>

`(http|https)` だとキャプチャグループになり、`re.findall()` が `http` や `https` だけを返してしまいます。`(?:http|https)` で非キャプチャグループにするか、`https?` と書けば解決します。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = """
公式サイト: https://www.example.com
ドキュメント: http://docs.example.org/guide
API: https://api.example.com/v1
"""

# 方法1: 非キャプチャグループを使う
results = re.findall(r"(?:https|http)://\S+", text)
print(results)
# ['https://www.example.com', 'http://docs.example.org/guide', 'https://api.example.com/v1']

# 方法2: https? を使う（こちらの方がシンプル）
results = re.findall(r"https?://\S+", text)
print(results)
# ['https://www.example.com', 'http://docs.example.org/guide', 'https://api.example.com/v1']
```

</details>

## 問題6：key=value の抽出（応用）

以下の設定テキストから、`key=value` 形式のペアをタプルのリストとして抽出し、辞書に変換してください。

```python
config_text = """
# データベース設定
db_host=localhost
db_port=5432
db_name=myapp
# アプリ設定
debug=true
log_level=info
"""
```

**期待される出力：**
```
[('db_host', 'localhost'), ('db_port', '5432'), ('db_name', 'myapp'), ('debug', 'true'), ('log_level', 'info')]
{'db_host': 'localhost', 'db_port': '5432', 'db_name': 'myapp', 'debug': 'true', 'log_level': 'info'}
```

<details>
<summary>ヒント</summary>

`(\w+)=(\w+)` で key と value をそれぞれキャプチャグループで囲みます。`# コメント行` には `=` がないのでマッチしません。

</details>

<details>
<summary>解答例</summary>

```python
import re

config_text = """
# データベース設定
db_host=localhost
db_port=5432
db_name=myapp
# アプリ設定
debug=true
log_level=info
"""

# (\w+)=(\w+) で key と value をキャプチャ
pairs = re.findall(r"(\w+)=(\w+)", config_text)
print(pairs)
# [('db_host', 'localhost'), ('db_port', '5432'), ('db_name', 'myapp'), ('debug', 'true'), ('log_level', 'info')]

# 辞書に変換
config_dict = dict(pairs)
print(config_dict)
# {'db_host': 'localhost', 'db_port': '5432', 'db_name': 'myapp', 'debug': 'true', 'log_level': 'info'}
```

</details>

## 問題7：HTMLタグの中身を抽出（応用）

以下のHTMLテキストから、`<li>` タグの中身をすべて抽出してください。

```python
html = """
<ul>
  <li>Python</li>
  <li>JavaScript</li>
  <li>TypeScript</li>
</ul>
"""
```

**期待される出力：**
```
['Python', 'JavaScript', 'TypeScript']
```

<details>
<summary>ヒント</summary>

`<li>(.+?)</li>` で `<li>` と `</li>` の間の文字をキャプチャグループで取得します。非貪欲 `+?` を使って最短マッチにしましょう。

</details>

<details>
<summary>解答例</summary>

```python
import re

html = """
<ul>
  <li>Python</li>
  <li>JavaScript</li>
  <li>TypeScript</li>
</ul>
"""

# <li> と </li> の間をキャプチャ
results = re.findall(r"<li>(.+?)</li>", html)
print(results)  # ['Python', 'JavaScript', 'TypeScript']
```

</details>

## 問題8：ファイル拡張子の分類（チャレンジ）

以下のファイル名リストから、画像ファイル（拡張子が `.jpg`, `.png`, `.gif` のいずれか）だけを抽出してください。大文字・小文字は区別しないものとします。

```python
files = [
    "photo.jpg",
    "document.pdf",
    "logo.PNG",
    "script.py",
    "banner.gif",
    "data.csv",
    "icon.Jpg",
    "animation.GIF",
]
```

**期待される出力：**
```
画像ファイル: ['photo.jpg', 'logo.PNG', 'banner.gif', 'icon.Jpg', 'animation.GIF']
```

<details>
<summary>ヒント</summary>

各ファイル名に対して `re.search()` で拡張子パターンをチェックします。`\.(?:jpg|png|gif)$` で末尾の拡張子にマッチさせ、`re.IGNORECASE` で大文字小文字を無視します。

</details>

<details>
<summary>解答例</summary>

```python
import re

files = [
    "photo.jpg",
    "document.pdf",
    "logo.PNG",
    "script.py",
    "banner.gif",
    "data.csv",
    "icon.Jpg",
    "animation.GIF",
]

# \.(?:jpg|png|gif)$ → 末尾が .jpg, .png, .gif のいずれか
# re.IGNORECASE で大文字小文字を区別しない
image_files = [f for f in files if re.search(r"\.(?:jpg|png|gif)$", f, re.IGNORECASE)]
print(f"画像ファイル: {image_files}")
# ['photo.jpg', 'logo.PNG', 'banner.gif', 'icon.Jpg', 'animation.GIF']
```

</details>

## 問題9：CSV行の解析（チャレンジ）

以下のCSVデータの各行から、名前・年齢・都市を抽出し、整形して表示してください。

```python
csv_data = """名前,年齢,都市
田中太郎,30,東京
佐藤花子,25,大阪
鈴木一郎,35,名古屋"""
```

**期待される出力：**
```
田中太郎さん（30歳）- 東京在住
佐藤花子さん（25歳）- 大阪在住
鈴木一郎さん（35歳）- 名古屋在住
```

<details>
<summary>ヒント</summary>

ヘッダー行を除外するために、名前が漢字であることを利用するか、`re.MULTILINE` と `^` を組み合わせて各行を処理します。`(.+),(\d+),(.+)` で3つのフィールドをキャプチャできます。

</details>

<details>
<summary>解答例</summary>

```python
import re

csv_data = """名前,年齢,都市
田中太郎,30,東京
佐藤花子,25,大阪
鈴木一郎,35,名古屋"""

# (.+),(\d+),(.+) で名前・年齢・都市をキャプチャ
# \d+ を含めることでヘッダー行は自然に除外される
results = re.findall(r"(.+),(\d+),(.+)", csv_data)

for name, age, city in results:
    print(f"{name}さん（{age}歳）- {city}在住")
# 田中太郎さん（30歳）- 東京在住
# 佐藤花子さん（25歳）- 大阪在住
# 鈴木一郎さん（35歳）- 名古屋在住
```

</details>
