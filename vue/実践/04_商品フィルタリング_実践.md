# 実践課題04：商品フィルタリング ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第2章（リアクティビティ）、第3章（テンプレート構文）、第4章（コンポーネント基礎）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `props` によるデータの受け渡し、`$emit` によるイベント通知、複数の `computed` を組み合わせたフィルタリングを実践する

---

## 完成イメージ

カテゴリ・価格帯・キーワードで商品を絞り込むアプリです。

```
┌──────────────────────────────────────────┐
│  🛍️ 商品フィルタリング                    │
├──────────────────────────────────────────┤
│  検索: [キーワードを入力...        ]      │
│                                          │
│  カテゴリ: [全て] [家電] [書籍] [食品]    │
│  価格帯:   [全て] [~1000] [~5000] [5000~]│
│  並び替え: [名前順] [価格が安い順] [高い順]│
│                                          │
│  該当: 5件 / 全10件                       │
│                                          │
│  ┌────────┐ ┌────────┐ ┌────────┐       │
│  │ 📱     │ │ 📖     │ │ 🍎     │       │
│  │ スマホ  │ │ Vue本  │ │ りんご  │       │
│  │ 家電   │ │ 書籍   │ │ 食品   │       │
│  │ ¥80,000│ │ ¥3,200 │ │ ¥500   │       │
│  └────────┘ └────────┘ └────────┘       │
└──────────────────────────────────────────┘
```

---

## 課題の要件

1. 10件以上の商品データ（名前・カテゴリ・価格・アイコン）を用意する
2. キーワード検索（商品名の部分一致）ができる
3. カテゴリボタンでフィルタリングできる
4. 価格帯ボタンでフィルタリングできる
5. 並び替え（名前順・価格昇順・価格降順）ができる
6. 複数のフィルタを同時に適用できる（AND条件）
7. 商品カードと検索バーをそれぞれコンポーネントに分離する

**初心者向けとの違い:**
- `search-bar`、`filter-row`、`product-card` の3つのコンポーネントに分離
- `$emit` で子コンポーネントから親にイベントを通知するパターン
- `filter-row` コンポーネントを再利用してフィルタ行を汎用化
- 「フィルタをリセット」ボタンを `computed`（`hasActiveFilter`）で条件表示
- 大文字小文字を無視した検索（`toLowerCase()`）
- ソート関数をオブジェクトにまとめて簡潔に記述
