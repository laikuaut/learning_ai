## 課題の要件

1. Node.js v18以上がインストールされていることを確認する
2. Claude Codeをnpmでグローバルインストールする
3. 認証（APIキーまたはOAuth）を設定する
4. テスト用のプロジェクトディレクトリを作成する
5. Claude Codeを起動し、簡単な質問を投げて応答を確認する
6. `/help`、`/cost`、`/status`コマンドを実行して結果を確認する
7. ワンショットモード（`-p`フラグ）で1回だけ指示を実行する

---

## ステップガイド

<details>
<summary>ステップ1：前提条件の確認</summary>

Node.jsとnpmのバージョンを確認します。

```bash
# Node.jsのバージョン確認（v18以上が必要）
node --version

# npmのバージョン確認
npm --version
```

もしNode.jsがインストールされていない、またはv18未満の場合は、公式サイト（https://nodejs.org/）からインストールしてください。

</details>

<details>
<summary>ステップ2：Claude Codeのインストール</summary>

npmでグローバルインストールします。

```bash
# Claude Codeをインストール
npm install -g @anthropic-ai/claude-code

# インストール確認
claude --version
```

権限エラーが出る場合は `sudo` をつけて実行してください（macOS/Linux）。

</details>

<details>
<summary>ステップ3：認証の設定</summary>

2つの方法があります。いずれか1つを実施してください。

**方法A：APIキーを使用**

```bash
# 環境変数に設定
export ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxx"

# 永続化（bashの場合）
echo 'export ANTHROPIC_API_KEY="sk-ant-xxxxx"' >> ~/.bashrc
source ~/.bashrc
```

**方法B：Claude Pro/Teamプランを使用**

```bash
# 初回起動時にブラウザ認証が開始される
claude
# → ブラウザでCloudeアカウントにログイン
```

</details>

<details>
<summary>解答例（改良版）── 確認スクリプト</summary>

以下のシェルスクリプトで環境を一括チェックできます。

```bash
#!/bin/bash
# Claude Code 環境チェックスクリプト

echo "===== Claude Code 環境チェック ====="
echo ""

# Node.jsチェック
echo -n "Node.js: "
if command -v node &> /dev/null; then
    NODE_VER=$(node --version)
    MAJOR=$(echo "$NODE_VER" | sed 's/v//' | cut -d. -f1)
    if [ "$MAJOR" -ge 18 ]; then
        echo "OK ($NODE_VER)"
    else
        echo "NG ($NODE_VER) - v18以上が必要です"
    fi
else
    echo "NG - Node.jsがインストールされていません"
fi

# npmチェック
echo -n "npm: "
if command -v npm &> /dev/null; then
    echo "OK ($(npm --version))"
else
    echo "NG - npmがインストールされていません"
fi

# Claude Codeチェック
echo -n "Claude Code: "
if command -v claude &> /dev/null; then
    echo "OK ($(claude --version 2>/dev/null))"
else
    echo "NG - Claude Codeがインストールされていません"
fi

# APIキーチェック
echo -n "APIキー: "
if [ -n "$ANTHROPIC_API_KEY" ]; then
    echo "OK (設定済み)"
else
    echo "未設定 (OAuth認証を使用するか、ANTHROPIC_API_KEYを設定してください)"
fi

echo ""
echo "===== チェック完了 ====="
```

**初心者向けとの違い:**
- 手動で1つずつ確認する代わりに、スクリプトで一括チェックできます
- 各要件の合否が一目でわかるので、トラブルシューティングが容易です
- チーム内で共有すれば、全員の環境構築を効率化できます

</details>
