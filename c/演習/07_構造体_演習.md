# 第7章：構造体 - 演習問題

この演習では、C言語の構造体（`struct`）、列挙型（`enum`）、構造体とポインタの組み合わせについて練習します。

---

## 基本問題

### 問題1：学生構造体を定義して表示

学生の情報（名前、年齢、成績）を管理する構造体 `Student` を定義し、データを設定して表示するプログラムを作成してください。

- 名前は `char` 配列（最大50文字）
- 年齢は `int` 型
- 成績は `double` 型
- `typedef` を使って型名を簡潔にする

**期待される出力例：**
```
===== 学生情報 =====
名前: 田中太郎
年齢: 20歳
成績: 85.50点
```

<details>
<summary>ヒント</summary>

- `typedef struct { ... } Student;` の形式で定義すると、使うときに `struct` を省略できます
- 文字列のコピーには `strcpy` を使います（`<string.h>` が必要）
- 成績は `%.2f` で小数点以下2桁まで表示しましょう

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>  /* strcpy に必要 */

/* 学生構造体の定義 */
typedef struct {
    char name[50];   /* 名前 */
    int age;         /* 年齢 */
    double score;    /* 成績 */
} Student;

int main(void) {
    Student s;

    /* データを設定 */
    strcpy(s.name, "田中太郎");
    s.age = 20;
    s.score = 85.5;

    /* 表示 */
    printf("===== 学生情報 =====\n");
    printf("名前: %s\n", s.name);
    printf("年齢: %d歳\n", s.age);
    printf("成績: %.2f点\n", s.score);

    return 0;
}
```

</details>

### 問題2：構造体配列で複数の学生データを管理

問題1の `Student` 構造体を使い、3人分の学生データを構造体配列で管理するプログラムを作成してください。

- 初期化子を使って3人分のデータを設定する
- 全員の情報と、成績の平均点を表示する

**期待される出力例：**
```
===== 学生一覧 =====
No.1 田中太郎   (20歳) 成績: 85.50点
No.2 鈴木花子   (19歳) 成績: 92.00点
No.3 佐藤一郎   (21歳) 成績: 78.30点

平均点: 85.27点
```

<details>
<summary>ヒント</summary>

- 構造体配列は `Student students[3] = { {...}, {...}, {...} };` のように初期化できます
- ループで全員の情報を表示し、同時に成績を合計して平均を求めます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

/* 学生構造体の定義 */
typedef struct {
    char name[50];
    int age;
    double score;
} Student;

int main(void) {
    /* 構造体配列を初期化子で設定 */
    Student students[3] = {
        {"田中太郎", 20, 85.5},
        {"鈴木花子", 19, 92.0},
        {"佐藤一郎", 21, 78.3}
    };
    int count = 3;
    double total = 0.0;
    int i;

    printf("===== 学生一覧 =====\n");

    /* 全員の情報を表示しながら合計点を計算 */
    for (i = 0; i < count; i++) {
        printf("No.%d %-10s (%d歳) 成績: %.2f点\n",
               i + 1, students[i].name, students[i].age, students[i].score);
        total += students[i].score;
    }

    printf("\n平均点: %.2f点\n", total / count);

    return 0;
}
```

</details>

### 問題3：enumで曜日を定義して表示

列挙型（`enum`）で曜日を定義し、入力された番号（0〜6）に対応する曜日を日本語で表示するプログラムを作成してください。

- `enum` で `SUNDAY = 0` から `SATURDAY = 6` まで定義する
- 曜日名の文字列配列を用意して表示に使う

**期待される出力例：**
```
曜日番号を入力してください (0:日〜6:土): 3
3 → 水曜日
```

**範囲外の場合：**
```
曜日番号を入力してください (0:日〜6:土): 8
エラー: 0〜6の範囲で入力してください。
```

<details>
<summary>ヒント</summary>

- `enum Weekday { SUNDAY, MONDAY, ... };` のように定義すると、`SUNDAY` は `0` になります
- 曜日名は `const char *day_names[] = {"日曜日", "月曜日", ...};` のように文字列配列で管理すると便利です

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

/* 列挙型で曜日を定義 */
enum Weekday {
    SUNDAY,     /* 0 */
    MONDAY,     /* 1 */
    TUESDAY,    /* 2 */
    WEDNESDAY,  /* 3 */
    THURSDAY,   /* 4 */
    FRIDAY,     /* 5 */
    SATURDAY    /* 6 */
};

int main(void) {
    int day;

    /* 曜日名の文字列配列 */
    const char *day_names[] = {
        "日曜日", "月曜日", "火曜日", "水曜日",
        "木曜日", "金曜日", "土曜日"
    };

    printf("曜日番号を入力してください (0:日〜6:土): ");
    scanf("%d", &day);

    /* 範囲チェック */
    if (day < SUNDAY || day > SATURDAY) {
        printf("エラー: 0〜6の範囲で入力してください。\n");
    } else {
        printf("%d → %s\n", day, day_names[day]);
    }

    return 0;
}
```

</details>

## 応用問題

### 問題4：構造体をポインタで関数に渡す

`Student` 構造体へのポインタを受け取り、情報を表示する関数 `print_student` と、成績を更新する関数 `update_score` を作成してください。

- `print_student` は `const Student *` を引数にとる（読み取り専用）
- `update_score` は `Student *` を引数にとり、新しい成績をセットする
- アロー演算子（`->`）を使ってメンバにアクセスする

**期待される出力例：**
```
===== 更新前 =====
名前: 田中太郎
年齢: 20歳
成績: 75.00点

成績を更新します...

===== 更新後 =====
名前: 田中太郎
年齢: 20歳
成績: 88.50点
```

<details>
<summary>ヒント</summary>

- 構造体のポインタからメンバにアクセスするには `p->name` のようにアロー演算子を使います
- `const` をつけると、関数内でメンバを変更できなくなります（安全性が向上）
- 関数にはアドレス演算子 `&` で構造体のアドレスを渡します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>

/* 学生構造体の定義 */
typedef struct {
    char name[50];
    int age;
    double score;
} Student;

/* プロトタイプ宣言 */
void print_student(const Student *s);
void update_score(Student *s, double new_score);

int main(void) {
    Student s;
    strcpy(s.name, "田中太郎");
    s.age = 20;
    s.score = 75.0;

    printf("===== 更新前 =====\n");
    print_student(&s);  /* アドレスを渡す */

    printf("\n成績を更新します...\n\n");
    update_score(&s, 88.5);  /* ポインタ経由で値を更新 */

    printf("===== 更新後 =====\n");
    print_student(&s);

    return 0;
}

/* 構造体の情報を表示する関数（読み取り専用） */
void print_student(const Student *s) {
    /* アロー演算子でメンバにアクセス */
    printf("名前: %s\n", s->name);
    printf("年齢: %d歳\n", s->age);
    printf("成績: %.2f点\n", s->score);
}

/* 成績を更新する関数 */
void update_score(Student *s, double new_score) {
    s->score = new_score;
}
```

</details>

### 問題5：ネストした構造体（住所を含む社員情報）

住所を表す構造体 `Address` と、社員を表す構造体 `Employee` を定義してください。`Employee` は `Address` をメンバとして持ちます。

- `Address`: 都道府県（`prefecture`）、市区町村（`city`）、番地（`street`）
- `Employee`: 社員ID、名前、部署、住所（`Address` 型）
- 2人分の社員データを設定して表示する

**期待される出力例：**
```
===== 社員情報 =====
[社員ID: 1001]
名前: 山田太郎
部署: 開発部
住所: 東京都 新宿区 西新宿2-8-1

[社員ID: 1002]
名前: 佐藤花子
部署: 営業部
住所: 大阪府 中央区 本町1-3-5
```

<details>
<summary>ヒント</summary>

- ネストした構造体のメンバには `emp.address.prefecture` のようにドット演算子を連続で使います
- 初期化子でも `{"東京都", "新宿区", "西新宿2-8-1"}` のようにネストできます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

/* 住所構造体 */
typedef struct {
    char prefecture[20];  /* 都道府県 */
    char city[30];        /* 市区町村 */
    char street[50];      /* 番地 */
} Address;

/* 社員構造体（Addressをメンバとして持つ） */
typedef struct {
    int id;               /* 社員ID */
    char name[50];        /* 名前 */
    char department[30];  /* 部署 */
    Address address;      /* 住所（ネストした構造体） */
} Employee;

/* プロトタイプ宣言 */
void print_employee(const Employee *e);

int main(void) {
    /* 初期化子でデータを設定 */
    Employee employees[2] = {
        {1001, "山田太郎", "開発部", {"東京都", "新宿区", "西新宿2-8-1"}},
        {1002, "佐藤花子", "営業部", {"大阪府", "中央区", "本町1-3-5"}}
    };
    int i;

    printf("===== 社員情報 =====\n");
    for (i = 0; i < 2; i++) {
        print_employee(&employees[i]);
    }

    return 0;
}

/* 社員情報を表示する関数 */
void print_employee(const Employee *e) {
    printf("[社員ID: %d]\n", e->id);
    printf("名前: %s\n", e->name);
    printf("部署: %s\n", e->department);
    /* ネストした構造体のメンバにアロー演算子+ドット演算子でアクセス */
    printf("住所: %s %s %s\n",
           e->address.prefecture, e->address.city, e->address.street);
    printf("\n");
}
```

</details>

## チャレンジ問題

### 問題6：構造体配列のソート（成績順）

5人分の `Student` データを構造体配列に格納し、成績の降順（高い順）にソートして表示するプログラムを作成してください。

- ソートにはバブルソートまたは選択ソートを使う（`qsort` は使わない）
- ソート前・ソート後の両方を表示する
- 構造体ごと交換する

**期待される出力例：**
```
===== ソート前 =====
No.1 田中太郎   成績: 72.50点
No.2 鈴木花子   成績: 95.00点
No.3 佐藤一郎   成績: 68.30点
No.4 高橋美咲   成績: 88.00点
No.5 伊藤健太   成績: 81.50点

===== 成績順（降順）にソート後 =====
No.1 鈴木花子   成績: 95.00点
No.2 高橋美咲   成績: 88.00点
No.3 伊藤健太   成績: 81.50点
No.4 田中太郎   成績: 72.50点
No.5 佐藤一郎   成績: 68.30点
```

<details>
<summary>ヒント</summary>

- バブルソートは隣り合う要素を比較・交換するアルゴリズムです
- 構造体の交換は一時変数 `Student temp` を使って丸ごとコピーできます
- 降順にするには `students[j].score < students[j+1].score` のとき交換します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

/* 学生構造体の定義 */
typedef struct {
    char name[50];
    int age;
    double score;
} Student;

/* プロトタイプ宣言 */
void print_students(const Student students[], int count);
void sort_by_score_desc(Student students[], int count);

int main(void) {
    Student students[] = {
        {"田中太郎", 20, 72.5},
        {"鈴木花子", 19, 95.0},
        {"佐藤一郎", 21, 68.3},
        {"高橋美咲", 20, 88.0},
        {"伊藤健太", 22, 81.5}
    };
    int count = sizeof(students) / sizeof(students[0]);

    printf("===== ソート前 =====\n");
    print_students(students, count);

    /* 成績降順にソート */
    sort_by_score_desc(students, count);

    printf("===== 成績順（降順）にソート後 =====\n");
    print_students(students, count);

    return 0;
}

/* 学生一覧を表示する関数 */
void print_students(const Student students[], int count) {
    int i;
    for (i = 0; i < count; i++) {
        printf("No.%d %-10s 成績: %.2f点\n",
               i + 1, students[i].name, students[i].score);
    }
    printf("\n");
}

/* バブルソートで成績の降順に並べ替える関数 */
void sort_by_score_desc(Student students[], int count) {
    int i, j;
    Student temp;

    for (i = 0; i < count - 1; i++) {
        for (j = 0; j < count - 1 - i; j++) {
            /* 成績が小さい方を後ろに移動（降順） */
            if (students[j].score < students[j + 1].score) {
                /* 構造体を丸ごと交換 */
                temp = students[j];
                students[j] = students[j + 1];
                students[j + 1] = temp;
            }
        }
    }
}
```

</details>
