## 問題1：自己紹介プログラム（基本）

変数を使って、自分の名前・年齢・趣味を表示するプログラムを作成してください。

**期待される出力例：**
```
名前: 太郎
年齢: 20
趣味: プログラミング
```

<details>
<summary>ヒント</summary>

変数に値を代入してから `print()` で出力しましょう。f文字列を使うと便利です。

</details>

<details>
<summary>解答例</summary>

```python
name = "太郎"
age = 20
hobby = "プログラミング"

print(f"名前: {name}")
print(f"年齢: {age}")
print(f"趣味: {hobby}")
```

</details>

## 問題2：計算プログラム（基本）

2つの変数 `a = 15` と `b = 4` を使って、以下の計算結果をすべて表示してください。

- 足し算
- 引き算
- 掛け算
- 割り算（小数）
- 割り算（切り捨て）
- 余り
- べき乗

**期待される出力：**
```
足し算: 19
引き算: 11
掛け算: 60
割り算: 3.75
切り捨て除算: 3
余り: 3
べき乗: 50625
```

<details>
<summary>解答例</summary>

```python
a = 15
b = 4

print(f"足し算: {a + b}")
print(f"引き算: {a - b}")
print(f"掛け算: {a * b}")
print(f"割り算: {a / b}")
print(f"切り捨て除算: {a // b}")
print(f"余り: {a % b}")
print(f"べき乗: {a ** b}")
```

</details>

## 問題3：型の確認（基本）

以下の変数それぞれの型を `type()` で確認し、表示してください。

## 問題5：BMI計算（応用）

身長（cm）と体重（kg）を変数に設定して、BMIを計算・表示するプログラムを作成してください。

**BMIの計算式：** `BMI = 体重(kg) / (身長(m) ** 2)`

※ 身長はcmで入力するので、mに変換する必要があります。

**期待される出力例：**
```
身長: 170cm
体重: 65kg
BMI: 22.49
```

<details>
<summary>ヒント</summary>

- cm → m の変換: `height_m = height_cm / 100`
- 小数点以下2桁に丸める: `round(値, 2)` または f文字列で `f"{bmi:.2f}"`

</details>

<details>
<summary>解答例</summary>

```python
height_cm = 170
weight_kg = 65

height_m = height_cm / 100
bmi = weight_kg / (height_m ** 2)

print(f"身長: {height_cm}cm")
print(f"体重: {weight_kg}kg")
print(f"BMI: {bmi:.2f}")
```

</details>

## 問題6：文字列操作（応用）

以下の変数を使って、指示通りの出力を作成してください。

```python
first_name = "Python"
last_name = "太郎"
birth_year = 2005
current_year = 2026
```

**期待される出力：**
```
名前: Python太郎
年齢: 21歳
メッセージ: Python太郎さん（21歳）、ようこそ！

<summary>解答例</summary>

```

</details>

## 問題7：温度変換（応用）

摂氏温度を華氏温度に変換するプログラムを作成してください。

**変換式：** `華氏 = 摂氏 × 9 / 5 + 32`

摂氏 `25` 度を変換し、以下のように表示してください。

**期待される出力：**
```
摂氏 25.0°C = 華氏 77.0°F
```

<details>
<summary>解答例</summary>

```python
celsius = 25.0
fahrenheit = celsius * 9 / 5 + 32

print(f"摂氏 {celsius}°C = 華氏 {fahrenheit}°F")
```

</details>

## 問題8：入力プログラム（チャレンジ）

`input()` を使って、ユーザーから名前と生まれた年を入力してもらい、現在の年齢を計算して表示するプログラムを作成してください。

**実行例：**
```
名前を入力してください: 太郎
生まれた年を入力してください: 2005
太郎さんは2026年現在、21歳です。
```

<details>
<summary>ヒント</summary>

- `input()` の返り値は**文字列**です。年を計算に使うには `int()` で変換が必要です。
- 現在の年は `2026` としてOKです。

</details>

<details>
<summary>解答例</summary>

```python
name = input("名前を入力してください: ")
birth_year = int(input("生まれた年を入力してください: "))

current_year = 2026
age = current_year - birth_year

print(f"{name}さんは{current_year}年現在、{age}歳です。")
```

</details>

## 問題9：買い物計算（チャレンジ）

りんご1個150円、みかん1個80円です。
りんごを3個、みかんを5個買ったときの合計金額と、消費税10%を加えた税込金額を計算してください。

**期待される出力：**
```
りんご 3個: 450円
みかん 5個: 400円
小計: 850円
税込合計: 935円
```

<details>
<summary>ヒント</summary>

- 税込金額: `int(小計 * 1.1)` で整数に変換できます。

</details>

<details>
<summary>解答例</summary>

```python
apple_price = 150
orange_price = 80
apple_count = 3
orange_count = 5

apple_total = apple_price * apple_count
orange_total = orange_price * orange_count
subtotal = apple_total + orange_total
tax_total = int(subtotal * 1.1)

print(f"りんご {apple_count}個: {apple_total}円")
print(f"みかん {orange_count}個: {orange_total}円")
print(f"小計: {subtotal}円")
print(f"税込合計: {tax_total}円")
```

</details>

## 問題10：変数の入れ替え（チャレンジ）

2つの変数 `a = 10` と `b = 20` の値を入れ替えて表示してください。
**3つ目の変数を使う方法**と**使わない方法**の2通りで書いてみましょう。

**期待される出力：**
```
入れ替え前: a=10, b=20
入れ替え後: a=20, b=10
```

<details>
<summary>ヒント</summary>

- 方法1：一時変数 `temp` を使う
- 方法2：Pythonの多重代入 `a, b = b, a`

</details>

<details>
<summary>解答例</summary>

```python
# 方法1：一時変数を使う
a = 10
b = 20
print(f"入れ替え前: a={a}, b={b}")

temp = a
a = b
b = temp
print(f"入れ替え後: a={a}, b={b}")

print("---")

# 方法2：多重代入を使う（Python特有の方法）
a = 10
b = 20
print(f"入れ替え前: a={a}, b={b}")

a, b = b, a
print(f"入れ替え後: a={a}, b={b}")
```

</details>
