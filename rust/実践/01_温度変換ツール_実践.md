---

## 課題の要件

1. 摂氏（Celsius）から華氏（Fahrenheit）への変換ができる
2. 華氏から摂氏への変換ができる
3. ユーザーから「変換モード」と「温度」を入力で受け取る
4. 無効なモードが入力された場合、エラーメッセージを表示する
5. 変換公式：`F = C * 9.0 / 5.0 + 32.0`、`C = (F - 32.0) * 5.0 / 9.0`

## ステップガイド

<details>
<summary>ステップ1：標準入力を受け取る方法を確認する</summary>

Rustで標準入力を受け取るには `std::io::stdin()` を使います。

```rust
use std::io;

fn main() {
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    let input = input.trim(); // 末尾の改行を除去
    println!("入力値: {}", input);
}
```

文字列を数値に変換するには `.parse()` を使います。

```rust
let number: f64 = input.trim().parse().expect("数値を入力してください");
```

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```rust
// 温度変換ツール
// 学べる内容：変数、型変換、条件分岐、標準入出力
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

fn main() {
    // --- メニュー表示 ---
    println!("===== 温度変換ツール =====");
    println!("1. 摂氏 → 華氏");
    println!("2. 華氏 → 摂氏");
    println!();

    // --- 変換モード入力 ---
    print!("変換モードを選んでください (1 or 2): ");
    io::stdout().flush().unwrap();

    let mut choice = String::new();
    io::stdin().read_line(&mut choice).expect("入力エラー");
    let choice = choice.trim();

    if choice == "1" {
        // --- 摂氏 → 華氏 ---
        print!("温度を入力してください: ");
        io::stdout().flush().unwrap();

        let mut temp_input = String::new();
        io::stdin().read_line(&mut temp_input).expect("入力エラー");
        let celsius: f64 = temp_input.trim().parse().expect("数値を入力してください");

        let fahrenheit = celsius * 9.0 / 5.0 + 32.0;
        println!();
        println!("変換結果: {:.1}°C → {:.1}°F", celsius, fahrenheit);
    } else if choice == "2" {
        // --- 華氏 → 摂氏 ---
        print!("温度を入力してください: ");
        io::stdout().flush().unwrap();

        let mut temp_input = String::new();
        io::stdin().read_line(&mut temp_input).expect("入力エラー");
        let fahrenheit: f64 = temp_input.trim().parse().expect("数値を入力してください");

        let celsius = (fahrenheit - 32.0) * 5.0 / 9.0;
        println!();
        println!("変換結果: {:.1}°F → {:.1}°C", fahrenheit, celsius);
    } else {
        println!();
        println!("無効な選択です。1 または 2 を入力してください。");
    }
}
```

<details>
<summary>解答例（改良版 ─ 関数を使った整理）</summary>

第5章（関数とクロージャ）の知識を先取りして、変換処理を関数に切り出したバージョンです。

```rust
// 温度変換ツール（改良版）
// 関数で変換処理を分離したバージョン
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

/// 摂氏から華氏に変換します
fn celsius_to_fahrenheit(c: f64) -> f64 {
    c * 9.0 / 5.0 + 32.0
}

/// 華氏から摂氏に変換します
fn fahrenheit_to_celsius(f: f64) -> f64 {
    (f - 32.0) * 5.0 / 9.0
}

/// 標準入力から1行読み取り、trimした文字列を返します
fn read_line() -> String {
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

/// プロンプトを表示してから入力を読み取ります
fn prompt(message: &str) -> String {
    print!("{}", message);
    io::stdout().flush().unwrap();
    read_line()
}

fn main() {
    println!("===== 温度変換ツール =====");
    println!("1. 摂氏 → 華氏");
    println!("2. 華氏 → 摂氏");
    println!();

    let choice = prompt("変換モードを選んでください (1 or 2): ");

    match choice.as_str() {
        "1" => {
            let input = prompt("温度を入力してください: ");
            let celsius: f64 = input.parse().expect("数値を入力してください");
            let fahrenheit = celsius_to_fahrenheit(celsius);
            println!("\n変換結果: {:.1}°C → {:.1}°F", celsius, fahrenheit);
        }
        "2" => {
            let input = prompt("温度を入力してください: ");
            let fahrenheit: f64 = input.parse().expect("数値を入力してください");
            let celsius = fahrenheit_to_celsius(fahrenheit);
            println!("\n変換結果: {:.1}°F → {:.1}°C", fahrenheit, celsius);
        }
        _ => {
            println!("\n無効な選択です。1 または 2 を入力してください。");
        }
    }
}
```

**初心者向けとの違い:**
- 変換処理を `celsius_to_fahrenheit()` / `fahrenheit_to_celsius()` 関数に分離 → テストしやすく再利用可能
- 入力処理を `read_line()` / `prompt()` に共通化 → コードの重複を排除
- `if-else` を `match` 式に変更 → Rustらしいパターンマッチング
- ドキュメンテーションコメント（`///`）を追加

</details>
