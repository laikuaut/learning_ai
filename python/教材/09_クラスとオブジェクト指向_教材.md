## 9.1 クラスとは？

**クラス**は「設計図」、**オブジェクト（インスタンス）**は「設計図から作った実体」です。

```python
# クラスの定義
class Dog:
    def __init__(self, name, breed):
        self.name = name      # インスタンス変数
        self.breed = breed

    def bark(self):
        print(f"{self.name}:ワン！")

    def introduce(self):
        print(f"名前: {self.name}, 犬種: {self.breed}")

# インスタンスの作成
dog1 = Dog("ポチ", "柴犬")
dog2 = Dog("ラッキー", "ゴールデンレトリバー")

dog1.bark()         # ポチ:ワン！
dog2.introduce()    # 名前: ラッキー, 犬種: ゴールデンレトリバー
```

### self とは？

`self` はインスタンス自身を指します。メソッドの第1引数に必ず書きます。

```python
class Counter:
    def __init__(self):
        self.count = 0    # このインスタンスのcount

    def increment(self):
        self.count += 1   # このインスタンスのcountを+1

    def get_count(self):
        return self.count

c1 = Counter()
c2 = Counter()
c1.increment()
c1.increment()
c2.increment()
print(c1.get_count())  # 2（c1のcount）
print(c2.get_count())  # 1（c2のcount）
```

## 9.2 `__init__` メソッド

`__init__` はインスタンス作成時に自動で呼ばれるメソッドです。

```python
class Student:
    def __init__(self, name, grade=1):
        self.name = name
        self.grade = grade
        self.scores = []       # 空のリストで初期化

    def add_score(self, score):
        self.scores.append(score)

    def average(self):
        if not self.scores:
            return 0
        return sum(self.scores) / len(self.scores)

student = Student("太郎", grade=2)
student.add_score(85)
student.add_score(90)
student.add_score(78)
print(f"{student.name}の平均点: {student.average():.1f}")  # 84.3
```

## 9.3 クラス変数とインスタンス変数

```python
class Student:
    # クラス変数（全インスタンスで共有）
    school_name = "Python学園"
    student_count = 0

    def __init__(self, name):
        # インスタンス変数（各インスタンス固有）
        self.name = name
        Student.student_count += 1  # クラス変数を更新

    def introduce(self):
        print(f"{Student.school_name}の{self.name}です")

s1 = Student("太郎")
s2 = Student("花子")
s1.introduce()  # Python学園の太郎です
print(f"生徒数: {Student.student_count}")  # 2
```

## 9.4 継承（inheritance）

既存のクラスを元に、新しいクラスを作ることができます。

```python
# 親クラス（基底クラス）
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name}が鳴きました")

    def introduce(self):
        print(f"動物の名前: {self.name}")


# 子クラス（派生クラス）
class Cat(Animal):
    def __init__(self, name, color):
        super().__init__(name)  # 親クラスの__init__を呼ぶ
        self.color = color

    def speak(self):  # メソッドのオーバーライド（上書き）
        print(f"{self.name}:ニャー！")

    def show_color(self):
        print(f"{self.name}の色: {self.color}")


class Dog(Animal):
    def speak(self):
        print(f"{self.name}:ワン！")


cat = Cat("タマ", "白")
dog = Dog("ポチ")

cat.speak()       # タマ:ニャー！
dog.speak()       # ポチ:ワン！
cat.introduce()   # 動物の名前: タマ（親クラスのメソッド）
cat.show_color()  # タマの色: 白
```

### super() の活用

```python
class Employee:
    def __init__(self, name, salary):
        self.name = name
        self.salary = salary

    def info(self):
        return f"{self.name} (給与: {self.salary:,}円)"


class Manager(Employee):
    def __init__(self, name, salary, department):
        super().__init__(name, salary)  # 親の初期化を呼ぶ
        self.department = department

    def info(self):
        base_info = super().info()  # 親のメソッドを呼ぶ
        return f"{base_info} - {self.department}部長"

m = Manager("田中", 500000, "開発")
print(m.info())  # 田中 (給与: 500,000円) - 開発部長
```

### isinstance() で型チェック

```python
cat = Cat("タマ", "白")
print(isinstance(cat, Cat))     # True
print(isinstance(cat, Animal))  # True（親クラスもTrue）
print(isinstance(cat, Dog))     # False
```

## 9.5 カプセル化と @property

### アンダースコアの慣例

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner          # 公開
        self._balance = balance     # 慣例的にプライベート（アクセスは可能）
        self.__secret = "xxx"       # 名前修飾される（強いプライベート）

account = BankAccount("太郎", 1000)
print(account.owner)       # 太郎
print(account._balance)    # 1000（アクセスは可能だが非推奨）
# print(account.__secret)  # AttributeError!
```

### @property デコレータ

属性のように見えるが、実はメソッドを呼んでいるという仕組みです。

```python
class Temperature:
    def __init__(self, celsius):
        self._celsius = celsius

    @property
    def celsius(self):
        """摂氏温度の取得"""
        return self._celsius

    @celsius.setter
    def celsius(self, value):
        """摂氏温度の設定（バリデーション付き）"""
        if value < -273.15:
            raise ValueError("絶対零度以下は設定できません")
        self._celsius = value

    @property
    def fahrenheit(self):
        """華氏温度（読み取り専用）"""
        return self._celsius * 9 / 5 + 32

temp = Temperature(25)
print(temp.celsius)      # 25（プロパティとしてアクセス）
print(temp.fahrenheit)   # 77.0

temp.celsius = 30        # setterが呼ばれる
print(temp.fahrenheit)   # 86.0

# temp.celsius = -300    # ValueError!
# temp.fahrenheit = 100  # AttributeError!（setterがないため）
```

## 9.6 ポリモーフィズム（多態性）

異なるクラスのオブジェクトを同じインターフェースで扱えることです。

```python
class Circle:
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14159 * self.radius ** 2

    def describe(self):
        return f"円（半径{self.radius}）"


class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

    def describe(self):
        return f"長方形（{self.width}x{self.height}）"


class Triangle:
    def __init__(self, base, height):
        self.base = base
        self.height = height

    def area(self):
        return self.base * self.height / 2

    def describe(self):
        return f"三角形（底辺{self.base}, 高さ{self.height}）"


# 同じインターフェースで操作できる！
shapes = [Circle(5), Rectangle(3, 4), Triangle(6, 8)]

for shape in shapes:
    print(f"{shape.describe()}: 面積 = {shape.area():.2f}")
# 円（半径5）: 面積 = 78.54
# 長方形（3x4）: 面積 = 12.00
# 三角形（底辺6, 高さ8）: 面積 = 24.00
```

## 9.7 特殊メソッド（マジックメソッド）

`__xxx__` の形式のメソッドは、特定の演算子や組み込み関数に対応します。

```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __repr__(self):
        """repr()やインタプリタでの表示"""
        return f"Vector({self.x}, {self.y})"

    def __str__(self):
        """print()での表示"""
        return f"({self.x}, {self.y})"

    def __add__(self, other):
        """+ 演算子"""
        return Vector(self.x + other.x, self.y + other.y)

    def __sub__(self, other):
        """- 演算子"""
        return Vector(self.x - other.x, self.y - other.y)

    def __mul__(self, scalar):
        """* 演算子（スカラー倍）"""
        return Vector(self.x * scalar, self.y * scalar)

    def __eq__(self, other):
        """== 演算子"""
        return self.x == other.x and self.y == other.y

    def __len__(self):
        """len()"""
        return int((self.x ** 2 + self.y ** 2) ** 0.5)

    def __bool__(self):
        """bool()（if文での判定）"""
        return self.x != 0 or self.y != 0


v1 = Vector(3, 4)
v2 = Vector(1, 2)

print(v1)           # (3, 4)
print(repr(v1))     # Vector(3, 4)
print(v1 + v2)      # (4, 6)
print(v1 - v2)      # (2, 2)
print(v1 * 3)       # (9, 12)
print(v1 == v2)     # False
print(len(v1))      # 5
```

### よく使う特殊メソッド一覧

| メソッド | 対応する操作 |
|---------|------------|
| `__init__` | コンストラクタ |
| `__str__` | `str()`, `print()` |
| `__repr__` | `repr()`, デバッグ表示 |
| `__len__` | `len()` |
| `__add__` | `+` |
| `__sub__` | `-` |
| `__mul__` | `*` |
| `__eq__` | `==` |
| `__lt__` | `<` |
| `__getitem__` | `obj[key]` |
| `__setitem__` | `obj[key] = value` |
| `__contains__` | `in` |
| `__iter__` | `for ... in` |
| `__call__` | `obj()` |

## 9.8 クラスメソッドとスタティックメソッド

```python
class Date:
    def __init__(self, year, month, day):
        self.year = year
        self.month = month
        self.day = day

    def __str__(self):
        return f"{self.year}/{self.month:02d}/{self.day:02d}"

    @classmethod
    def from_string(cls, date_string):
        """文字列からDateを作成する（代替コンストラクタ）"""
        year, month, day = map(int, date_string.split("-"))
        return cls(year, month, day)

    @staticmethod
    def is_leap_year(year):
        """うるう年判定（インスタンス不要）"""
        return year % 4 == 0 and (year % 100 != 0 or year % 400 == 0)


d = Date.from_string("2024-03-15")
print(d)  # 2024/03/15
print(Date.is_leap_year(2024))  # True
```

## まとめ

| 概念 | 説明 |
|------|------|
| クラス | オブジェクトの設計図 |
| `__init__` | インスタンス作成時に呼ばれる |
| `self` | インスタンス自身を指す |
| 継承 | 既存クラスを拡張する |
| `super()` | 親クラスのメソッドを呼ぶ |
| `@property` | 属性風のアクセスでメソッドを呼ぶ |
| ポリモーフィズム | 同じ操作を異なる型で使える |
| 特殊メソッド | 演算子や組み込み関数をカスタマイズ |
