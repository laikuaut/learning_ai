### 問題1：テキストファイルの書き込み

`BufferedWriter` を使って、自分の自己紹介を3行以上テキストファイルに書き込むプログラムを作成してください。

**期待される出力例：**
```
ファイル "profile.txt" に書き込みました。

--- ファイル内容 ---
名前: 田中太郎
趣味: プログラミング
目標: Javaマスターになること
```

<details>
<summary>ヒント</summary>

- `BufferedWriter` を `try-with-resources` で使います
- `writer.write()` で文字列を書き込み、`writer.newLine()` で改行します
- 書き込み後に読み込んで内容を確認します

</details>

<details>
<summary>解答例</summary>

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;

public class WriteProfile {
    public static void main(String[] args) {
        String filename = "profile.txt";

        /* ファイルに書き込み */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filename))) {
            writer.write("名前: 田中太郎");
            writer.newLine();
            writer.write("趣味: プログラミング");
            writer.newLine();
            writer.write("目標: Javaマスターになること");
            writer.newLine();
            System.out.println("ファイル \"" + filename + "\" に書き込みました。");
        } catch (IOException e) {
            System.err.println("書き込みエラー: " + e.getMessage());
            return;
        }

        /* 内容を確認 */
        System.out.println("\n--- ファイル内容 ---");
        try (BufferedReader reader = new BufferedReader(new FileReader(filename))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            System.err.println("読み込みエラー: " + e.getMessage());
        }

        /* クリーンアップ */
        try { Files.deleteIfExists(Path.of(filename)); } catch (IOException e) {}
    }
}
```

</details>

### 問題2：テキストファイルの行番号付き表示

テキストファイルを読み込み、各行に行番号をつけて表示するプログラムを作成してください。

**期待される出力例：**
```
   1: public class Hello {
   2:     public static void main(String[] args) {
   3:         System.out.println("Hello!");
   4:     }
   5: }
合計: 5行
```

<details>
<summary>ヒント</summary>

- まずサンプルファイルを書き込みで作成してからそれを読み込みます
- `BufferedReader` の `readLine()` で1行ずつ読み込みます
- `System.out.printf("%4d: %s%n", lineNum, line)` で整形表示します

</details>

<details>
<summary>解答例</summary>

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;

public class NumberedReader {
    public static void main(String[] args) {
        String filename = "sample_code.txt";

        /* サンプルファイルを作成 */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filename))) {
            writer.write("public class Hello {\n");
            writer.write("    public static void main(String[] args) {\n");
            writer.write("        System.out.println(\"Hello!\");\n");
            writer.write("    }\n");
            writer.write("}\n");
        } catch (IOException e) {
            System.err.println("作成エラー: " + e.getMessage());
            return;
        }

        /* 行番号付きで表示 */
        try (BufferedReader reader = new BufferedReader(new FileReader(filename))) {
            String line;
            int lineNum = 1;
            while ((line = reader.readLine()) != null) {
                System.out.printf("%4d: %s%n", lineNum, line);
                lineNum++;
            }
            System.out.println("合計: " + (lineNum - 1) + "行");
        } catch (IOException e) {
            System.err.println("読み込みエラー: " + e.getMessage());
        }

        /* クリーンアップ */
        try { Files.deleteIfExists(Path.of(filename)); } catch (IOException e) {}
    }
}
```

</details>

### 問題3：Files クラスを使った簡易ファイル操作

`java.nio.file.Files` クラスを使って、ファイルの作成・内容書き込み・読み込み・サイズ確認・削除を行ってください。

**期待される出力例：**
```
ファイル作成: test_nio.txt
書き込み完了
内容:
  Hello, Java NIO!
  Files APIは便利です。
ファイルサイズ: XX バイト
ファイル削除完了
```

<details>
<summary>ヒント</summary>

- `Files.writeString()` で書き込み（Java 11+）
- `Files.readAllLines()` で全行読み込み
- `Files.size()` でサイズ取得
- `Files.delete()` で削除

</details>

<details>
<summary>解答例</summary>

```java
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.List;

public class NioFileOps {
    public static void main(String[] args) {
        Path path = Path.of("test_nio.txt");

        try {
            /* 書き込み */
            String content = "Hello, Java NIO!\nFiles APIは便利です。\n";
            Files.writeString(path, content);
            System.out.println("ファイル作成: " + path);
            System.out.println("書き込み完了");

            /* 読み込み */
            System.out.println("内容:");
            List<String> lines = Files.readAllLines(path);
            for (String line : lines) {
                System.out.println("  " + line);
            }

            /* サイズ確認 */
            System.out.println("ファイルサイズ: " + Files.size(path) + " バイト");

            /* 削除 */
            Files.delete(path);
            System.out.println("ファイル削除完了");

        } catch (IOException e) {
            System.err.println("エラー: " + e.getMessage());
        }
    }
}
```

</details>

### 問題4：単語出現回数カウンター

テキストファイルから全単語を読み込み、各単語の出現回数を集計して表示するプログラムを作成してください。

**期待される出力例：**
```
=== 単語出現回数 ===
java     : 3回
is       : 2回
a        : 2回
great    : 1回
programming: 1回
language : 1回
fun      : 1回
and      : 1回
powerful : 1回
```

<details>
<summary>ヒント</summary>

- `HashMap<String, Integer>` で単語と回数を管理します
- `line.toLowerCase().split("\\s+")` で行を単語に分割します
- `map.merge(word, 1, Integer::sum)` でカウントアップできます

</details>

<details>
<summary>解答例</summary>

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.HashMap;
import java.util.Map;

public class WordCounter {
    public static void main(String[] args) {
        String filename = "words.txt";

        /* サンプルファイル作成 */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filename))) {
            writer.write("Java is a great programming language\n");
            writer.write("Java is fun and powerful\n");
            writer.write("Java is great\n");
        } catch (IOException e) {
            System.err.println("作成エラー: " + e.getMessage());
            return;
        }

        /* 単語をカウント */
        Map<String, Integer> wordCount = new HashMap<>();
        try (BufferedReader reader = new BufferedReader(new FileReader(filename))) {
            String line;
            while ((line = reader.readLine()) != null) {
                String[] words = line.toLowerCase().split("\\s+");
                for (String word : words) {
                    if (!word.isEmpty()) {
                        wordCount.merge(word, 1, Integer::sum);
                    }
                }
            }
        } catch (IOException e) {
            System.err.println("読み込みエラー: " + e.getMessage());
            return;
        }

        /* 結果表示 */
        System.out.println("=== 単語出現回数 ===");
        wordCount.entrySet().stream()
            .sorted((a, b) -> b.getValue() - a.getValue())
            .forEach(entry ->
                System.out.printf("%-12s: %d回%n", entry.getKey(), entry.getValue()));

        /* クリーンアップ */
        try { Files.deleteIfExists(Path.of(filename)); } catch (IOException e) {}
    }
}
```

</details>

### 問題5：CSVファイルの読み書き

商品データのCSVファイルを作成し、読み込んで合計金額を計算するプログラムを作成してください。

**期待される出力例：**
```
=== 商品一覧 ===
商品名          単価    個数    小計
--------------------------------------
りんご          150       3     450
バナナ          100       5     500
みかん          200       2     400
--------------------------------------
合計金額: 1,350円
```

<details>
<summary>ヒント</summary>

- CSVはカンマ区切り：`商品名,単価,個数`
- `split(",")` でフィールドを分割します
- `Integer.parseInt()` で数値に変換します
- `String.format("%,d", total)` で3桁カンマ区切り表示

</details>

<details>
<summary>解答例</summary>

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;

public class CsvShop {
    public static void main(String[] args) {
        String filename = "products.csv";

        /* CSVファイルを作成 */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filename))) {
            writer.write("商品名,単価,個数\n");
            writer.write("りんご,150,3\n");
            writer.write("バナナ,100,5\n");
            writer.write("みかん,200,2\n");
        } catch (IOException e) {
            System.err.println("CSV作成エラー: " + e.getMessage());
            return;
        }

        /* CSVを読み込んで集計 */
        System.out.println("=== 商品一覧 ===");
        System.out.printf("%-12s %6s %6s %8s%n", "商品名", "単価", "個数", "小計");
        System.out.println("-".repeat(38));

        int grandTotal = 0;
        try (BufferedReader reader = new BufferedReader(new FileReader(filename))) {
            String line;
            boolean isHeader = true;
            while ((line = reader.readLine()) != null) {
                if (isHeader) {
                    isHeader = false;
                    continue;  /* ヘッダー行をスキップ */
                }
                String[] fields = line.split(",");
                String name = fields[0];
                int price = Integer.parseInt(fields[1]);
                int quantity = Integer.parseInt(fields[2]);
                int subtotal = price * quantity;
                grandTotal += subtotal;

                System.out.printf("%-12s %,6d %6d %,8d%n", name, price, quantity, subtotal);
            }
        } catch (IOException e) {
            System.err.println("読み込みエラー: " + e.getMessage());
            return;
        }

        System.out.println("-".repeat(38));
        System.out.printf("合計金額: %,d円%n", grandTotal);

        /* クリーンアップ */
        try { Files.deleteIfExists(Path.of(filename)); } catch (IOException e) {}
    }
}
```

### 問題6：成績ファイル処理システム

以下の処理を行う成績管理プログラムを作成してください。

1. 生徒の成績データをCSVファイルに書き込む
2. CSVファイルを読み込んで各生徒の平均点を計算する
3. 結果をレポートファイル（テキスト）に書き出す
4. レポートの内容をコンソールに表示する

**期待される出力例：**
```
=== 成績レポート ===

名前       国語  数学  英語  平均
----------------------------------
田中太郎    78    85    92   85.0
鈴木花子    92    76    88   85.3
佐藤一郎    65    90    78   77.7
山田美咲    88    82    95   88.3
----------------------------------
クラス平均:                  84.1

最高平均点: 山田美咲（88.3）
最低平均点: 佐藤一郎（77.7）

レポートを "grade_report.txt" に保存しました。
```

<details>
<summary>ヒント</summary>

- CSVファイルの形式：`名前,国語,数学,英語`
- データを読み込みながらリストや配列に格納し、集計処理を行います
- `StringBuilder` を使ってレポート文字列を組み立て、コンソール表示とファイル書き込みの両方に使います
- 最高点・最低点の追跡には変数を使います

</details>

<details>
<summary>解答例</summary>

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.ArrayList;
import java.util.List;

public class GradeReport {
    public static void main(String[] args) {
        String csvFile = "grades.csv";
        String reportFile = "grade_report.txt";

        /* 1. CSVファイルに成績データを書き込み */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(csvFile))) {
            writer.write("名前,国語,数学,英語\n");
            writer.write("田中太郎,78,85,92\n");
            writer.write("鈴木花子,92,76,88\n");
            writer.write("佐藤一郎,65,90,78\n");
            writer.write("山田美咲,88,82,95\n");
        } catch (IOException e) {
            System.err.println("CSV書き込みエラー: " + e.getMessage());
            return;
        }

        /* 2. CSVを読み込んで処理 */
        List<String> names = new ArrayList<>();
        List<double[]> scores = new ArrayList<>();
        List<Double> averages = new ArrayList<>();

        try (BufferedReader reader = new BufferedReader(new FileReader(csvFile))) {
            String line;
            boolean isHeader = true;
            while ((line = reader.readLine()) != null) {
                if (isHeader) { isHeader = false; continue; }
                String[] fields = line.split(",");
                names.add(fields[0]);
                double[] s = {
                    Double.parseDouble(fields[1]),
                    Double.parseDouble(fields[2]),
                    Double.parseDouble(fields[3])
                };
                scores.add(s);
                averages.add((s[0] + s[1] + s[2]) / 3.0);
            }
        } catch (IOException e) {
            System.err.println("CSV読み込みエラー: " + e.getMessage());
            return;
        }

        /* 3. レポート生成 */
        StringBuilder report = new StringBuilder();
        report.append("=== 成績レポート ===\n\n");
        report.append(String.format("%-10s %5s %5s %5s %6s%n", "名前", "国語", "数学", "英語", "平均"));
        report.append("-".repeat(34)).append("\n");

        double totalAvg = 0;
        String bestName = "";
        String worstName = "";
        double bestAvg = -1;
        double worstAvg = 101;

        for (int i = 0; i < names.size(); i++) {
            double[] s = scores.get(i);
            double avg = averages.get(i);
            totalAvg += avg;
            report.append(String.format("%-10s %5.0f %5.0f %5.0f %6.1f%n",
                names.get(i), s[0], s[1], s[2], avg));

            if (avg > bestAvg) { bestAvg = avg; bestName = names.get(i); }
            if (avg < worstAvg) { worstAvg = avg; worstName = names.get(i); }
        }

        double classAvg = totalAvg / names.size();
        report.append("-".repeat(34)).append("\n");
        report.append(String.format("クラス平均: %23.1f%n", classAvg));
        report.append(String.format("%n最高平均点: %s（%.1f）%n", bestName, bestAvg));
        report.append(String.format("最低平均点: %s（%.1f）%n", worstName, worstAvg));

        /* コンソールに表示 */
        System.out.println(report);

        /* 4. レポートファイルに保存 */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(reportFile))) {
            writer.write(report.toString());
            System.out.println("レポートを \"" + reportFile + "\" に保存しました。");
        } catch (IOException e) {
            System.err.println("レポート書き込みエラー: " + e.getMessage());
        }

        /* クリーンアップ */
        try {
            Files.deleteIfExists(Path.of(csvFile));
            Files.deleteIfExists(Path.of(reportFile));
        } catch (IOException e) {}
    }
}
```

### 問題7：設定ファイルリーダー

`key=value` 形式の設定ファイルを読み込み、`HashMap` に格納して利用するプログラムを作成してください。コメント行（`#` で始まる行）と空行はスキップしてください。

**期待される出力例：**
```
=== 設定ファイルの内容 ===
app.name = MyJavaApp
app.version = 1.0.0
db.host = localhost
db.port = 5432
db.name = mydb

設定値の取得:
  アプリ名: MyJavaApp
  バージョン: 1.0.0
  DBホスト: localhost
  DBポート: 5432
```

<details>
<summary>ヒント</summary>

- `#` で始まる行と空行（`trim().isEmpty()`）をスキップします
- `split("=", 2)` で key と value に分割します（value に `=` が含まれる場合に対応）
- `trim()` で前後の空白を除去します

</details>

<details>
<summary>解答例</summary>

```java
import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.FileReader;
import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Path;
import java.util.HashMap;
import java.util.Map;

public class ConfigReader {
    /* 設定ファイルを読み込んで Map に格納 */
    static Map<String, String> loadConfig(String filename) throws IOException {
        Map<String, String> config = new HashMap<>();
        try (BufferedReader reader = new BufferedReader(new FileReader(filename))) {
            String line;
            while ((line = reader.readLine()) != null) {
                line = line.trim();
                /* コメント行と空行をスキップ */
                if (line.isEmpty() || line.startsWith("#")) {
                    continue;
                }
                /* key=value 形式をパース */
                String[] parts = line.split("=", 2);
                if (parts.length == 2) {
                    config.put(parts[0].trim(), parts[1].trim());
                }
            }
        }
        return config;
    }

    public static void main(String[] args) {
        String filename = "app.conf";

        /* 設定ファイルを作成 */
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(filename))) {
            writer.write("# アプリケーション設定\n");
            writer.write("app.name = MyJavaApp\n");
            writer.write("app.version = 1.0.0\n");
            writer.write("\n");
            writer.write("# データベース設定\n");
            writer.write("db.host = localhost\n");
            writer.write("db.port = 5432\n");
            writer.write("db.name = mydb\n");
        } catch (IOException e) {
            System.err.println("作成エラー: " + e.getMessage());
            return;
        }

        /* 読み込みと表示 */
        try {
            Map<String, String> config = loadConfig(filename);

            System.out.println("=== 設定ファイルの内容 ===");
            for (Map.Entry<String, String> entry : config.entrySet()) {
                System.out.println(entry.getKey() + " = " + entry.getValue());
            }

            System.out.println("\n設定値の取得:");
            System.out.println("  アプリ名: " + config.getOrDefault("app.name", "未設定"));
            System.out.println("  バージョン: " + config.getOrDefault("app.version", "未設定"));
            System.out.println("  DBホスト: " + config.getOrDefault("db.host", "未設定"));
            System.out.println("  DBポート: " + config.getOrDefault("db.port", "未設定"));

        } catch (IOException e) {
            System.err.println("読み込みエラー: " + e.getMessage());
        }

        /* クリーンアップ */
        try { Files.deleteIfExists(Path.of(filename)); } catch (IOException e) {}
    }
}
```
