## 課題の要件

### 対応するコマンド
| コマンド | 構文 | 説明 |
|---------|------|------|
| CREATE TABLE | `CREATE TABLE テーブル名 (カラム1, カラム2, ...)` | テーブルを作成 |
| INSERT | `INSERT テーブル名 値1 値2 ...` | 行を挿入 |
| SELECT | `SELECT テーブル名 [WHERE カラム = 値]` | 行を検索 |
| UPDATE | `UPDATE テーブル名 SET カラム = 値 WHERE カラム = 値` | 行を更新 |
| DELETE | `DELETE テーブル名 WHERE カラム = 値` | 行を削除 |
| COUNT | `COUNT テーブル名` | 行数を表示 |
| TABLES | `TABLES` | テーブル一覧を表示 |
| DROP TABLE | `DROP TABLE テーブル名` | テーブルを削除 |
| EXIT | `EXIT` | 終了 |

### 技術要件
- すべての値は文字列として格納する（型の区別なし）
- 各行にはauto incrementのIDを付与する
- `WHERE` 句は `カラム = 値` の等価比較のみ対応
- カスタムエラー型でエラーを管理する
- 存在しないテーブル・カラムへのアクセスにはエラーを返す

// ミニデータベースエンジン
// 学べる内容：構造体設計、列挙型、パーサー、Result、HashMap、所有権
// 実行方法：rustc minidb.rs && ./minidb

use std::fmt;
use std::io;
use std::io::Write;

// ==================== エラー型 ====================

enum DbError {
    EmptyCommand,
    UnknownCommand(String),
    SyntaxError(String),
    TableNotFound(String),
    TableAlreadyExists(String),
    ColumnNotFound(String),
    ColumnMismatch { expected: usize, got: usize },
}

impl fmt::Display for DbError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        match self {
            DbError::EmptyCommand => write!(f, "コマンドを入力してください。"),
            DbError::UnknownCommand(cmd) => write!(f, "不明なコマンド: {}", cmd),
            DbError::SyntaxError(msg) => write!(f, "構文エラー: {}", msg),
            DbError::TableNotFound(name) => write!(f, "テーブル '{}' が見つかりません。", name),
            DbError::TableAlreadyExists(name) => {
                write!(f, "テーブル '{}' は既に存在します。", name)
            }
            DbError::ColumnNotFound(name) => write!(f, "カラム '{}' が見つかりません。", name),
            DbError::ColumnMismatch { expected, got } => {
                write!(f, "カラム数の不一致: {}個必要ですが{}個です。", expected, got)
            }
        }
    }
}

// ==================== データモデル ====================

struct Row {
    id: u32,
    values: Vec<String>,
}

struct Table {
    name: String,
    columns: Vec<String>,
    rows: Vec<Row>,
    next_id: u32,
}

impl Table {
    fn new(name: String, columns: Vec<String>) -> Table {
        Table {
            name,
            columns,
            rows: Vec::new(),
            next_id: 1,
        }
    }

    fn column_index(&self, col_name: &str) -> Result<usize, DbError> {
        self.columns
            .iter()
            .position(|c| c == col_name)
            .ok_or_else(|| DbError::ColumnNotFound(col_name.to_string()))
    }

    fn insert(&mut self, values: Vec<String>) -> Result<u32, DbError> {
        if values.len() != self.columns.len() {
            return Err(DbError::ColumnMismatch {
                expected: self.columns.len(),
                got: values.len(),
            });
        }
        let id = self.next_id;
        self.next_id += 1;
        self.rows.push(Row { id, values });
        Ok(id)
    }

    fn select(&self, condition: &Option<(String, String)>) -> Result<Vec<&Row>, DbError> {
        match condition {
            None => Ok(self.rows.iter().collect()),
            Some((col, val)) => {
                let col_idx = self.column_index(col)?;
                Ok(self
                    .rows
                    .iter()
                    .filter(|row| row.values[col_idx] == *val)
                    .collect())
            }
        }
    }

    fn update(
        &mut self,
        set_col: &str,
        set_val: &str,
        where_col: &str,
        where_val: &str,
    ) -> Result<usize, DbError> {
        let set_idx = self.column_index(set_col)?;
        let where_idx = self.column_index(where_col)?;

        let mut count = 0;
        for row in &mut self.rows {
            if row.values[where_idx] == where_val {
                row.values[set_idx] = set_val.to_string();
                count += 1;
            }
        }
        Ok(count)
    }

    fn delete(&mut self, where_col: &str, where_val: &str) -> Result<usize, DbError> {
        let where_idx = self.column_index(where_col)?;
        let before = self.rows.len();
        self.rows.retain(|row| row.values[where_idx] != where_val);
        Ok(before - self.rows.len())
    }

    fn display_rows(&self, rows: &[&Row], filter_label: &str) {
        // ヘッダ表示
        if filter_label.is_empty() {
            println!("--- {} ({}行) ---", self.name, rows.len());
        } else {
            println!(
                "--- {} ({}行, フィルタ: {}) ---",
                self.name,
                rows.len(),
                filter_label
            );
        }

        // カラム幅の計算
        let mut widths: Vec<usize> = Vec::new();
        widths.push(2); // "id" の幅
        for col in &self.columns {
            widths.push(col.len());
        }

        // データの最大幅を反映
        for row in rows {
            let id_str = row.id.to_string();
            if id_str.len() > widths[0] {
                widths[0] = id_str.len();
            }
            for (j, val) in row.values.iter().enumerate() {
                if val.len() > widths[j + 1] {
                    widths[j + 1] = val.len();
                }
            }
        }

        // 最小幅を保証
        for w in &mut widths {
            if *w < 3 {
                *w = 3;
            }
        }

        // ヘッダ行
        let mut header = String::from("| ");
        header.push_str(&format!("{:<width$}", "id", width = widths[0]));
        for (i, col) in self.columns.iter().enumerate() {
            header.push_str(&format!(" | {:<width$}", col, width = widths[i + 1]));
        }
        header.push_str(" |");
        println!("{}", header);

        // 区切り線
        let mut separator = String::from("|");
        for w in &widths {
            separator.push_str(&"-".repeat(w + 2));
            separator.push('|');
        }
        println!("{}", separator);

        // データ行
        for row in rows {
            let mut line = String::from("| ");
            line.push_str(&format!("{:>width$}", row.id, width = widths[0]));
            for (j, val) in row.values.iter().enumerate() {
                line.push_str(&format!(" | {:<width$}", val, width = widths[j + 1]));
            }
            line.push_str(" |");
            println!("{}", line);
        }
    }
}

// ==================== データベース ====================

struct Database {
    tables: Vec<Table>,
}

impl Database {
    fn new() -> Database {
        Database { tables: Vec::new() }
    }

    fn find_table(&self, name: &str) -> Result<usize, DbError> {
        self.tables
            .iter()
            .position(|t| t.name == name)
            .ok_or_else(|| DbError::TableNotFound(name.to_string()))
    }

    fn create_table(&mut self, name: String, columns: Vec<String>) -> Result<(), DbError> {
        // 同名テーブルの存在チェック
        if self.tables.iter().any(|t| t.name == name) {
            return Err(DbError::TableAlreadyExists(name));
        }
        println!(
            "テーブル '{}' を作成しました。(カラム: {})",
            name,
            columns.join(", ")
        );
        self.tables.push(Table::new(name, columns));
        Ok(())
    }

    fn drop_table(&mut self, name: &str) -> Result<(), DbError> {
        let index = self.find_table(name)?;
        self.tables.remove(index);
        println!("テーブル '{}' を削除しました。", name);
        Ok(())
    }

    fn show_tables(&self) {
        if self.tables.is_empty() {
            println!("テーブルがありません。");
            return;
        }
        println!("--- テーブル一覧 ---");
        for table in &self.tables {
            println!(
                "  {} (カラム: {}) - {}行",
                table.name,
                table.columns.join(", "),
                table.rows.len()
            );
        }
    }
}

// ==================== コマンド解析 ====================

enum Command {
    CreateTable {
        name: String,
        columns: Vec<String>,
    },
    Insert {
        table: String,
        values: Vec<String>,
    },
    Select {
        table: String,
        condition: Option<(String, String)>,
    },
    Update {
        table: String,
        set_col: String,
        set_val: String,
        where_col: String,
        where_val: String,
    },
    Delete {
        table: String,
        where_col: String,
        where_val: String,
    },
    Count {
        table: String,
    },
    Tables,
    DropTable {
        name: String,
    },
    Exit,
}

fn parse_command(input: &str) -> Result<Command, DbError> {
    let tokens: Vec<&str> = input.split_whitespace().collect();
    if tokens.is_empty() {
        return Err(DbError::EmptyCommand);
    }

    let cmd = tokens[0].to_uppercase();
    match cmd.as_str() {
        "CREATE" => {
            // CREATE TABLE name (col1, col2, ...)
            if tokens.len() < 4 || tokens[1].to_uppercase() != "TABLE" {
                return Err(DbError::SyntaxError(
                    "構文: CREATE TABLE テーブル名 (カラム1, カラム2, ...)".to_string(),
                ));
            }
            let name = tokens[2].to_string();

            // 残りの部分からカラム名を抽出
            let rest: String = tokens[3..].join(" ");
            let rest = rest.trim_start_matches('(').trim_end_matches(')');
            let columns: Vec<String> = rest
                .split(',')
                .map(|s| s.trim().to_string())
                .filter(|s| !s.is_empty())
                .collect();

            if columns.is_empty() {
                return Err(DbError::SyntaxError("カラムを1つ以上指定してください。".to_string()));
            }

            Ok(Command::CreateTable { name, columns })
        }
        "INSERT" => {
            // INSERT table val1 val2 ...
            if tokens.len() < 3 {
                return Err(DbError::SyntaxError(
                    "構文: INSERT テーブル名 値1 値2 ...".to_string(),
                ));
            }
            let table = tokens[1].to_string();
            let values: Vec<String> = tokens[2..].iter().map(|s| s.to_string()).collect();
            Ok(Command::Insert { table, values })
        }
        "SELECT" => {
            // SELECT table [WHERE col = val]
            if tokens.len() < 2 {
                return Err(DbError::SyntaxError(
                    "構文: SELECT テーブル名 [WHERE カラム = 値]".to_string(),
                ));
            }
            let table = tokens[1].to_string();
            let condition = if tokens.len() >= 6
                && tokens[2].to_uppercase() == "WHERE"
                && tokens[4] == "="
            {
                Some((tokens[3].to_string(), tokens[5].to_string()))
            } else if tokens.len() == 2 {
                None
            } else if tokens.len() > 2 {
                return Err(DbError::SyntaxError(
                    "WHERE句の構文: WHERE カラム = 値".to_string(),
                ));
            } else {
                None
            };
            Ok(Command::Select { table, condition })
        }
        "UPDATE" => {
            // UPDATE table SET col = val WHERE col = val
            if tokens.len() != 9
                || tokens[2].to_uppercase() != "SET"
                || tokens[4] != "="
                || tokens[6].to_uppercase() != "WHERE"
                || tokens[8] != "="
            {
                // tokens[8] が "=" ではなく、実際のトークンの並びを確認
                // UPDATE table SET col = val WHERE col = val
                // [0]    [1]   [2] [3] [4] [5] [6]   [7]  [8]
                // 修正: tokens[8] は値なので再チェック
            }
            if tokens.len() < 9 {
                return Err(DbError::SyntaxError(
                    "構文: UPDATE テーブル名 SET カラム = 値 WHERE カラム = 値".to_string(),
                ));
            }
            if tokens[2].to_uppercase() != "SET"
                || tokens[4] != "="
                || tokens[6].to_uppercase() != "WHERE"
            {
                return Err(DbError::SyntaxError(
                    "構文: UPDATE テーブル名 SET カラム = 値 WHERE カラム = 値".to_string(),
                ));
            }
            // UPDATE[0] table[1] SET[2] col[3] =[4] val[5] WHERE[6] col[7] =[8] val[9]
            if tokens.len() != 10 || tokens[8] != "=" {
                return Err(DbError::SyntaxError(
                    "構文: UPDATE テーブル名 SET カラム = 値 WHERE カラム = 値".to_string(),
                ));
            }
            Ok(Command::Update {
                table: tokens[1].to_string(),
                set_col: tokens[3].to_string(),
                set_val: tokens[5].to_string(),
                where_col: tokens[7].to_string(),
                where_val: tokens[9].to_string(),
            })
        }
        "DELETE" => {
            // DELETE table WHERE col = val
            if tokens.len() != 6
                || tokens[2].to_uppercase() != "WHERE"
                || tokens[4] != "="
            {
                return Err(DbError::SyntaxError(
                    "構文: DELETE テーブル名 WHERE カラム = 値".to_string(),
                ));
            }
            Ok(Command::Delete {
                table: tokens[1].to_string(),
                where_col: tokens[3].to_string(),
                where_val: tokens[5].to_string(),
            })
        }
        "COUNT" => {
            if tokens.len() != 2 {
                return Err(DbError::SyntaxError(
                    "構文: COUNT テーブル名".to_string(),
                ));
            }
            Ok(Command::Count {
                table: tokens[1].to_string(),
            })
        }
        "TABLES" => Ok(Command::Tables),
        "DROP" => {
            if tokens.len() != 3 || tokens[1].to_uppercase() != "TABLE" {
                return Err(DbError::SyntaxError(
                    "構文: DROP TABLE テーブル名".to_string(),
                ));
            }
            Ok(Command::DropTable {
                name: tokens[2].to_string(),
            })
        }
        "EXIT" | "QUIT" => Ok(Command::Exit),
        _ => Err(DbError::UnknownCommand(tokens[0].to_string())),
    }
}

// ==================== コマンド実行 ====================

fn execute(db: &mut Database, cmd: Command) -> Result<bool, DbError> {
    match cmd {
        Command::CreateTable { name, columns } => {
            db.create_table(name, columns)?;
        }
        Command::Insert { table, values } => {
            let idx = db.find_table(&table)?;
            db.tables[idx].insert(values)?;
            println!("1行を挿入しました。");
        }
        Command::Select { table, condition } => {
            let idx = db.find_table(&table)?;
            let rows = db.tables[idx].select(&condition)?;
            let filter_label = match &condition {
                Some((col, val)) => format!("{} = {}", col, val),
                None => String::new(),
            };
            db.tables[idx].display_rows(&rows, &filter_label);
        }
        Command::Update {
            table,
            set_col,
            set_val,
            where_col,
            where_val,
        } => {
            let idx = db.find_table(&table)?;
            let count = db.tables[idx].update(&set_col, &set_val, &where_col, &where_val)?;
            println!("{}行を更新しました。", count);
        }
        Command::Delete {
            table,
            where_col,
            where_val,
        } => {
            let idx = db.find_table(&table)?;
            let count = db.tables[idx].delete(&where_col, &where_val)?;
            println!("{}行を削除しました。", count);
        }
        Command::Count { table } => {
            let idx = db.find_table(&table)?;
            println!("{}: {}行", db.tables[idx].name, db.tables[idx].rows.len());
        }
        Command::Tables => {
            db.show_tables();
        }
        Command::DropTable { name } => {
            db.drop_table(&name)?;
        }
        Command::Exit => {
            println!("お疲れさまでした。");
            return Ok(true);
        }
    }
    Ok(false)
}

// ==================== メイン ====================

fn main() {
    println!("===== MiniDB - ミニデータベースエンジン =====");
    println!("コマンド一覧: CREATE TABLE, INSERT, SELECT, UPDATE, DELETE, COUNT, TABLES, DROP TABLE, EXIT");
    println!();

    let mut db = Database::new();

    loop {
        print!("minidb> ");
        io::stdout().flush().unwrap();

        let mut input = String::new();
        io::stdin().read_line(&mut input).expect("入力エラー");
        let input = input.trim();

        if input.is_empty() {
            continue;
        }

        match parse_command(input) {
            Ok(cmd) => match execute(&mut db, cmd) {
                Ok(true) => break,  // EXIT
                Ok(false) => {}      // 続行
                Err(e) => println!("エラー: {}", e),
            },
            Err(e) => println!("エラー: {}", e),
        }
    }
}

<details>
<summary>設計のポイント解説</summary>

### 1. パーサーと実行エンジンの分離

`parse_command()` と `execute()` を分離することで、コマンド解析とデータ操作の責務が明確に分かれています。これは実際のデータベースの設計パターン（パーサー → プランナー → エグゼキューター）を簡略化したものです。

### 2. `Command` 列挙型によるコマンドの型安全な表現

各コマンドが必要とするパラメータを列挙型のバリアントに含めることで、実行時に必ず必要なデータが揃っていることをコンパイラが保証します。

### 3. テーブル内のカラムインデックス解決

`column_index()` メソッドが `Result<usize, DbError>` を返すことで、存在しないカラムへのアクセスを安全にハンドリングしています。

### 4. 表示幅の動的計算

`display_rows()` ではカラム名とデータの最大幅を計算し、テーブルを整形表示しています。これにより、どのようなデータが入っても見やすいテーブル表示が可能です。

### 5. 所有権の設計判断

- `select()` は `Vec<&Row>` を返す：データのコピーを避け、参照だけを返す
- `insert()` は `Vec<String>` を受け取る：新しいデータの所有権をテーブルに移す
- `delete()` は `retain()` を使用：条件に合わない行だけを残すイディオム

### 6. エラー伝播のチェーン

```rust
match parse_command(input) {
    Ok(cmd) => match execute(&mut db, cmd) {
        Ok(true) => break,
        Ok(false) => {},
        Err(e) => println!("エラー: {}", e),
    },
    Err(e) => println!("エラー: {}", e),
}
```

パースエラーと実行エラーの両方を統一的な `DbError` 型で扱い、ユーザーに分かりやすいメッセージを表示しています。
