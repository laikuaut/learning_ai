## 課題の要件

1. トップページのナビゲーションリンクが3つ（「トップ」「サイトについて」「お問い合わせ」）表示されていることを検証する
2. 「サイトについて」リンクをクリックして、About ページに遷移することを検証する（URL とタイトルの確認）
3. About ページから `page.goBack()` でトップページに戻れることを検証する
4. 検索ボックスにテキストを入力し、検索ボタンをクリックした結果を検証する
5. トップページのスクリーンショットを撮影して保存する

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```typescript
// tests/navigation.spec.ts
// 学べる内容：ページ遷移、クリック、入力、スクリーンショット
// 実行方法：npx playwright test tests/navigation.spec.ts --project=chromium

import { test, expect } from '@playwright/test';
import path from 'path';

test.describe('ナビゲーションテスト', () => {

  test.beforeEach(async ({ page }) => {
    const filePath = path.resolve(__dirname, '../index.html');
    await page.goto(`file://${filePath}`);
  });

  test('ナビゲーションリンクがすべて表示されている', async ({ page }) => {
    const nav = page.getByRole('navigation', { name: 'メインナビゲーション' });

    // 3つのリンクが存在することを検証
    const topLink = nav.getByRole('link', { name: 'トップ' });
    const aboutLink = nav.getByRole('link', { name: 'サイトについて' });
    const contactLink = nav.getByRole('link', { name: 'お問い合わせ' });

    await expect(topLink).toBeVisible();
    await expect(aboutLink).toBeVisible();
    await expect(contactLink).toBeVisible();
  });

  test('トップページからAboutページに遷移できる', async ({ page }) => {
    // 「サイトについて」リンクをクリック
    await page.getByRole('link', { name: 'サイトについて' }).click();

    // URLの検証
    await expect(page).toHaveURL(/about\.html/);

    // タイトルの検証
    await expect(page).toHaveTitle(/サイトについて/);

    // 見出しの検証
    await expect(page.getByRole('heading', { level: 1 })).toHaveText('サイトについて');
  });

  test('ブラウザの戻るボタンで前のページに戻れる', async ({ page }) => {
    // Aboutページに遷移
    await page.getByRole('link', { name: 'サイトについて' }).click();
    await expect(page).toHaveURL(/about\.html/);

    // 戻る
    await page.goBack();

    // トップページに戻ったことを確認
    await expect(page).toHaveURL(/index\.html/);
    await expect(page).toHaveTitle(/トップページ/);
  });
});

test.describe('ページ操作テスト', () => {

  test.beforeEach(async ({ page }) => {
    const filePath = path.resolve(__dirname, '../index.html');
    await page.goto(`file://${filePath}`);
  });

  test('テキスト入力とクリックが正しく動作する', async ({ page }) => {
    // 検索ボックスにテキストを入力
    const searchInput = page.getByLabel('サイト内検索：');
    await searchInput.fill('Playwright');

    // 入力値の検証
    await expect(searchInput).toHaveValue('Playwright');

    // 検索ボタンをクリック
    await page.getByRole('button', { name: '検索' }).click();

    // 検索結果の検証
    const result = page.locator('#search-result');
    await expect(result).toHaveText('「Playwright」で検索しました');
  });

  test('スクリーンショットが保存される', async ({ page }) => {
    // ページ全体のスクリーンショットを撮影
    await page.screenshot({ path: 'screenshots/top-page.png' });

    // 特定要素のスクリーンショット
    const nav = page.getByRole('navigation');
    await nav.screenshot({ path: 'screenshots/navigation.png' });
  });
});
```

<summary>解答例（改良版 ─ より実務的なテスト）</summary>

```typescript
// tests/navigation-advanced.spec.ts
// 改良版：ページ遷移パターンを網羅的にテスト
// 実行方法：npx playwright test tests/navigation-advanced.spec.ts --project=chromium

import { test, expect } from '@playwright/test';
import path from 'path';

const BASE_PATH = path.resolve(__dirname, '..');

test.describe('ナビゲーション総合テスト', () => {

  test.beforeEach(async ({ page }) => {
    await page.goto(`file://${BASE_PATH}/index.html`);
  });

  test('全ナビゲーションリンクのhref属性が正しい', async ({ page }) => {
    // ナビゲーション内のリンクを一括検証
    const nav = page.getByRole('navigation', { name: 'メインナビゲーション' });
    const links = nav.getByRole('link');

    // リンクの数を検証
    await expect(links).toHaveCount(3);

    // 各リンクのhref属性を検証
    await expect(links.nth(0)).toHaveAttribute('href', 'index.html');
    await expect(links.nth(1)).toHaveAttribute('href', 'about.html');
    await expect(links.nth(2)).toHaveAttribute('href', 'contact.html');
  });

  test('ページ遷移の往復が正しく動作する', async ({ page }) => {
    // トップ → About
    await page.getByRole('link', { name: 'サイトについて' }).click();
    await expect(page).toHaveURL(/about\.html/);

    // About → トップ（ページ内リンク）
    await page.getByRole('link', { name: 'トップページに戻る' }).click();
    await expect(page).toHaveURL(/index\.html/);
  });

  test('入力のクリアと再入力が正しく動作する', async ({ page }) => {
    const searchInput = page.getByLabel('サイト内検索：');

    // 入力 → クリア → 再入力
    await searchInput.fill('最初の検索');
    await expect(searchInput).toHaveValue('最初の検索');

    await searchInput.clear();
    await expect(searchInput).toHaveValue('');

    await searchInput.fill('2回目の検索');
    await page.getByRole('button', { name: '検索' }).click();

    await expect(page.locator('#search-result')).toHaveText(/2回目の検索/);
  });

  test('ページの表示内容が正しい', async ({ page }) => {
    // 見出しの検証
    await expect(page.getByRole('heading', { level: 1 })).toHaveText('ようこそ！ナビ練習サイトへ');

    // 新着情報のリストアイテムが2つあることを検証
    const listItems = page.getByRole('listitem');
    await expect(listItems).toHaveCount(2);

    // 最初のリストアイテムの内容を検証
    await expect(listItems.first()).toContainText('リニューアル');
  });
});
```

**初心者向けとの違い:**
- `toHaveCount()` でリンクやリストアイテムの数を検証し、要素の欠落を検出できる
- `nth()` でインデックス指定して個別のリンクを検証
- `clear()` と `fill()` を組み合わせた入力操作パターンを使用
- テスト対象のパスを定数に切り出して管理しやすくしている
