## 課題の要件

### パート1：ケーススタディ

以下のシナリオを読み、経営判断を行ってください。

**シナリオ:**
C社（年商50億円、従業員800名の製造業）の情報システム部長は、以下の3つのIT投資案件から優先順位をつけて経営会議に提案する必要があります。IT投資予算は年間8,000万円です。

**案件A: 基幹系システム（ERP）の刷新**
- 初期投資: 6,000万円
- 年間運用コスト: 1,000万円（現行は1,500万円）
- 期待効果: 業務効率化による人件費削減 年1,500万円、在庫最適化 年500万円
- リスク: 移行失敗による業務停止リスク（影響大）
- 投資期間: 5年

**案件B: AIによる需要予測システム導入**
- 初期投資: 3,000万円
- 年間運用コスト: 600万円
- 期待効果: 需要予測精度向上による在庫削減 年800万円、廃棄ロス削減 年400万円
- リスク: AI精度が期待に届かないリスク（影響中）
- 投資期間: 5年

**案件C: セキュリティ対策強化（EDR/SIEM導入）**
- 初期投資: 2,000万円
- 年間運用コスト: 500万円
- 期待効果: インシデント対応コスト削減 年300万円、信頼性向上による受注増 年500万円
- リスク: 効果が定量化しにくい（影響小〜中）
- 投資期間: 5年

**課題1:** 各案件のROI、単純回収期間を計算してください。
**課題2:** NPV（割引率5%）を計算し、投資判断を行ってください。
**課題3:** 予算制約（8,000万円）のもとで、どの案件を優先すべきか根拠とともに提案してください。

### パート2：損益分岐点分析

C社の現在の経営指標は以下のとおりです。

- 売上高: 50億円
- 変動費: 30億円
- 固定費: 15億円

**課題4:** 損益分岐点売上高を計算してください。
**課題5:** ERP導入により固定費が年間500万円増加するが、変動費率が2%改善される場合、新しい損益分岐点と利益の変化を計算してください。

## ステップガイド

<details>
<summary>パート1のステップ：投資分析の公式</summary>

```
年間純効果 = 年間期待効果 - 年間運用コスト + 現行運用コスト削減分

単純回収期間 = 初期投資額 / 年間純効果

ROI = (投資期間中の総純効果 - 初期投資額) / 初期投資額 × 100%

NPV = Σ(年間CF / (1+r)^t) - 初期投資
  ※ r: 割引率、t: 年数、CF: キャッシュフロー
```

</details>

<details>
<summary>パート2のステップ：損益分岐点の公式</summary>

```
変動費率 = 変動費 / 売上高
限界利益率 = 1 - 変動費率
損益分岐点売上高 = 固定費 / 限界利益率
利益 = 売上高 × 限界利益率 - 固定費
```

</details>

**課題1：ROIと回収期間**

| 指標 | 案件A(ERP) | 案件B(AI) | 案件C(セキュリティ) |
|------|-----------|----------|------------------|
| 年間純効果 | 1,500+500+500-1,000=1,500万 | 800+400-600=600万 | 300+500-500=300万 |
| 単純回収期間 | 6,000/1,500=4.0年 | 3,000/600=5.0年 | 2,000/300=6.7年 |
| 5年間総純効果 | 1,500×5=7,500万 | 600×5=3,000万 | 300×5=1,500万 |
| ROI | (7,500-6,000)/6,000=25.0% | (3,000-3,000)/3,000=0.0% | (1,500-2,000)/2,000=-25.0% |

※案件Aは現行運用コスト1,500→1,000への削減分500万を含む

**課題2：NPV（割引率5%）**

年金現価係数（5年、5%）= (1-(1.05)^-5)/0.05 ≈ 4.329

| 案件 | NPV計算 | NPV |
|------|---------|-----|
| A | 1,500×4.329-6,000 | 494万円 |
| B | 600×4.329-3,000 | -403万円 |
| C | 300×4.329-2,000 | -701万円 |

案件AのみNPVが正。

**課題3：優先順位の提案**

**推奨: 案件A（ERP刷新）を最優先**

根拠:
1. NPVが唯一正の値（494万円） → 投資価値がある
2. ROIが25%で最高 → 投資効率が最も良い
3. 回収期間4.0年で投資期間5年以内 → 期間内に回収可能
4. 基幹システムの安定稼働は事業継続の基盤

ただし、案件Cは定量効果だけでは測れないリスク低減効果があるため、次年度に実施を検討すべきです。案件Bは期待効果の精度を高めたうえで再評価が必要です。

**課題4：損益分岐点**

```
変動費率 = 30億 / 50億 = 0.6（60%）
限界利益率 = 1 - 0.6 = 0.4（40%）
損益分岐点売上高 = 15億 / 0.4 = 37.5億円
現在の利益 = 50億 × 0.4 - 15億 = 5億円
```

**課題5：ERP導入後**

```
新固定費 = 15億 + 0.05億 = 15.05億円
新変動費率 = 60% - 2% = 58%
新限界利益率 = 42%
新損益分岐点 = 15.05億 / 0.42 = 35.83億円
新利益 = 50億 × 0.42 - 15.05億 = 5.95億円

利益改善 = 5.95億 - 5.00億 = 0.95億円（9,500万円/年）
損益分岐点改善 = 37.5億 - 35.83億 = 1.67億円 低下
```

ERP導入により損益分岐点が約1.67億円低下し、利益が年間約9,500万円増加します。

```python
# IT投資分析ツール
# 学べる内容：ROI、NPV、損益分岐点、投資判断
# 実行方法：python investment_analysis.py

print("===== IT投資分析ツール =====\n")

# --- 案件データ ---
projects = [
    {
        "name": "案件A: ERP刷新",
        "initial_cost": 6000,
        "annual_cost": 1000,
        "annual_benefit": 2500,  # 1500+500+500(現行運用コスト削減)
        "years": 5,
    },
    {
        "name": "案件B: AI需要予測",
        "initial_cost": 3000,
        "annual_cost": 600,
        "annual_benefit": 1200,
        "years": 5,
    },
    {
        "name": "案件C: セキュリティ強化",
        "initial_cost": 2000,
        "annual_cost": 500,
        "annual_benefit": 800,
        "years": 5,
    },
]

discount_rate = 0.05

print("========== 投資比較分析 ==========\n")

for proj in projects:
    name = proj["name"]
    init = proj["initial_cost"]
    annual_net = proj["annual_benefit"] - proj["annual_cost"]
    years = proj["years"]

    print(f"--- {name} ---")
    print(f"初期投資: {init:,}万円")
    print(f"年間純効果: {annual_net:,}万円/年")

    # 単純回収期間
    if annual_net > 0:
        payback = init / annual_net
        print(f"単純回収期間: {payback:.2f}年")
    else:
        print("単純回収期間: 回収不可")

    # ROI
    total_net = annual_net * years
    roi = (total_net - init) / init * 100
    print(f"ROI: {roi:.1f}%")

    # NPV
    npv = -init
    print(f"\nNPV分析（割引率{discount_rate*100:.0f}%）:")
    print(f"{'年':>3} | {'CF':>10} | {'現在価値':>10} | {'累計NPV':>10}")
    print("-" * 45)
    print(f"{'0':>3} | {-init:>10,} | {-init:>10,} | {npv:>10,}")

    for t in range(1, years + 1):
        pv = annual_net / (1 + discount_rate) ** t
        npv += pv
        print(f"{t:>3} | {annual_net:>10,} | {pv:>10,.0f} | {npv:>10,.0f}")

    npv_judge = "投資価値あり" if npv > 0 else "投資価値なし"
    print(f"\nNPV = {npv:,.0f}万円 → {npv_judge}")
    print()

# --- 損益分岐点分析 ---
print("\n========== 損益分岐点分析 ==========\n")

revenue = 5000  # 百万円（50億円）
variable_cost = 3000  # 百万円
fixed_cost = 1500  # 百万円

var_ratio = variable_cost / revenue
margin_ratio = 1 - var_ratio
bep = fixed_cost / margin_ratio
profit = revenue * margin_ratio - fixed_cost

print(f"--- 現状 ---")
print(f"売上高: {revenue/100:.0f}億円")
print(f"変動費率: {var_ratio*100:.0f}%")
print(f"限界利益率: {margin_ratio*100:.0f}%")
print(f"損益分岐点: {bep/100:.2f}億円")
print(f"利益: {profit/100:.2f}億円")

# ERP導入後
new_fixed = fixed_cost + 5  # 500万円増
new_var_ratio = var_ratio - 0.02  # 2%改善
new_margin = 1 - new_var_ratio
new_bep = new_fixed / new_margin
new_profit = revenue * new_margin - new_fixed

print(f"\n--- ERP導入後 ---")
print(f"固定費: +500万円 → {new_fixed/100:.2f}億円")
print(f"変動費率: {new_var_ratio*100:.0f}%（2%改善）")
print(f"限界利益率: {new_margin*100:.0f}%")
print(f"損益分岐点: {new_bep/100:.2f}億円（{(bep-new_bep)/100:.2f}億円低下）")
print(f"利益: {new_profit/100:.2f}億円（{(new_profit-profit)/100:.2f}億円増加）")
```

```python
# IT投資分析ツール（改良版）
# 学べる内容：NPV、IRR、感度分析、投資判断の定量的根拠
# 実行方法：python investment_advanced.py

def calc_npv(initial_cost, annual_cf, years, rate):
    """NPVを計算"""
    npv = -initial_cost
    for t in range(1, years + 1):
        npv += annual_cf / (1 + rate) ** t
    return npv

def calc_irr(initial_cost, annual_cf, years, precision=0.0001):
    """IRRを二分探索で計算"""
    low, high = -0.5, 5.0
    for _ in range(1000):
        mid = (low + high) / 2
        npv = calc_npv(initial_cost, annual_cf, years, mid)
        if abs(npv) < precision:
            return mid
        if npv > 0:
            low = mid
        else:
            high = mid
    return mid

def sensitivity_analysis(initial_cost, base_annual_cf, years, rate):
    """感度分析: 年間効果が±30%変動した場合のNPVを計算"""
    results = []
    for change in [-30, -20, -10, 0, 10, 20, 30]:
        adjusted_cf = base_annual_cf * (1 + change / 100)
        npv = calc_npv(initial_cost, adjusted_cf, years, rate)
        results.append((change, adjusted_cf, npv))
    return results

# --- 案件データ ---
projects = [
    {"name": "案件A: ERP刷新", "init": 6000, "cost": 1000, "benefit": 2500, "years": 5},
    {"name": "案件B: AI需要予測", "init": 3000, "cost": 600, "benefit": 1200, "years": 5},
    {"name": "案件C: セキュリティ", "init": 2000, "cost": 500, "benefit": 800, "years": 5},
]

rate = 0.05
budget = 8000

print("===== IT投資分析ツール（改良版） =====\n")

# --- 比較表 ---
print(f"{'指標':<16}", end="")
for p in projects:
    print(f" | {p['name']:<18}", end="")
print("\n" + "-" * 75)

# 各指標を計算
for p in projects:
    p["annual_net"] = p["benefit"] - p["cost"]
    p["payback"] = p["init"] / p["annual_net"] if p["annual_net"] > 0 else float("inf")
    p["roi"] = (p["annual_net"] * p["years"] - p["init"]) / p["init"] * 100
    p["npv"] = calc_npv(p["init"], p["annual_net"], p["years"], rate)
    p["irr"] = calc_irr(p["init"], p["annual_net"], p["years"])

metrics = [
    ("初期投資(万円)", lambda p: f"{p['init']:>12,}"),
    ("年間純効果(万円)", lambda p: f"{p['annual_net']:>12,}"),
    ("回収期間(年)", lambda p: f"{p['payback']:>12.2f}"),
    ("ROI(%)", lambda p: f"{p['roi']:>12.1f}"),
    ("NPV(万円)", lambda p: f"{p['npv']:>12,.0f}"),
    ("IRR(%)", lambda p: f"{p['irr']*100:>12.1f}"),
]

for label, func in metrics:
    print(f"{label:<16}", end="")
    for p in projects:
        print(f" | {func(p):<18}", end="")
    print()

# --- 感度分析（案件A） ---
print(f"\n\n--- 感度分析（案件A: 年間効果の変動に対するNPV） ---")
sens = sensitivity_analysis(projects[0]["init"], projects[0]["annual_net"],
                            projects[0]["years"], rate)
print(f"{'効果変動':>8} | {'年間純効果':>10} | {'NPV':>10} | グラフ")
print("-" * 55)
for change, cf, npv in sens:
    bar_len = max(0, int(npv / 100))
    bar = "█" * bar_len if npv > 0 else ""
    neg_bar = "▓" * abs(int(npv / 100)) if npv < 0 else ""
    mark = " ← 基準" if change == 0 else ""
    print(f"{change:>+7}% | {cf:>10,.0f} | {npv:>+10,.0f} | {neg_bar}{bar}{mark}")

# --- 投資判断 ---
print(f"\n\n{'='*60}")
print("  投資判断レポート")
print(f"{'='*60}")

# NPV正の案件をフィルタ
viable = [p for p in projects if p["npv"] > 0]
print(f"\nNPVが正の案件: {len(viable)}件")

# 予算内で最大NPVとなる組み合わせを探索（簡易ナップサック）
best_combo = None
best_npv = 0
for i in range(1 << len(projects)):
    combo = [projects[j] for j in range(len(projects)) if i & (1 << j)]
    total_cost = sum(p["init"] for p in combo)
    total_npv = sum(p["npv"] for p in combo)
    if total_cost <= budget and total_npv > best_npv:
        best_npv = total_npv
        best_combo = combo

if best_combo:
    print(f"\n予算{budget:,}万円での最適組み合わせ:")
    for p in best_combo:
        print(f"  ・{p['name']}（投資{p['init']:,}万円、NPV{p['npv']:+,.0f}万円）")
    total_init = sum(p["init"] for p in best_combo)
    print(f"\n  合計投資: {total_init:,}万円（予算残: {budget-total_init:,}万円）")
    print(f"  合計NPV:  {best_npv:+,.0f}万円")
```

**初心者向けとの違い:**
- IRR（内部収益率）を二分探索で計算
- 感度分析により、効果の不確実性に対する投資判断のロバスト性を評価
- 予算制約下での最適な投資組み合わせを探索（簡易ナップサック問題）
- 比較表形式で複数案件を一覧比較
