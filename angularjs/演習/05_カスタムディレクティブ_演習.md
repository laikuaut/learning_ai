## 演習5-1（基本）：シンプルなカスタムディレクティブを作ろう

`<my-greeting>` というカスタムディレクティブを作成してください。テンプレートを使って挨拶メッセージを表示します。

**要件：**
- ディレクティブ名は `myGreeting`（HTMLでは `<my-greeting>`）
- `restrict: 'E'`（要素として使用）
- テンプレートに `<div>` で囲んだ「こんにちは！AngularJSの世界へようこそ！」を表示
- ディレクティブを3つ並べて表示すること

**期待される表示：**
```
こんにちは！AngularJSの世界へようこそ！
こんにちは！AngularJSの世界へようこそ！
こんにちは！AngularJSの世界へようこそ！
```

<details>
<summary>ヒント</summary>

ディレクティブは `app.directive('myGreeting', function() { ... })` で定義します。`template` プロパティにHTMLテンプレートを指定します。JavaScriptではキャメルケース（`myGreeting`）で定義し、HTMLではケバブケース（`my-greeting`）で使用します。

```javascript
app.directive('myGreeting', function() {
  return {
    restrict: 'E',
    template: '<div>...</div>'
  };
});
```

</details>

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <title>演習5-1：シンプルなカスタムディレクティブ</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .greeting {
      padding: 10px;
      margin: 5px 0;
      background-color: #e8f5e9;
      border-left: 4px solid #4caf50;
      font-size: 16px;
    }
  </style>
</head>
<body ng-controller="MainCtrl">
  <h2>カスタムディレクティブの基本</h2>

  <!-- ========================================
    ディレクティブをHTML要素として使用する
    JavaScriptのキャメルケース myGreeting は
    HTMLではケバブケース my-greeting になる
  ======================================== -->
  <my-greeting></my-greeting>
  <my-greeting></my-greeting>
  <my-greeting></my-greeting>

  <script>
    // モジュール定義
    var app = angular.module('myApp', []);

    // コントローラー（今回は空でOK）
    app.controller('MainCtrl', function() {});

    // ========================================
    // カスタムディレクティブの定義
    // restrict: 'E' → 要素（Element）としてのみ使用可能
    // template: ディレクティブが展開されるHTMLテンプレート
    // ========================================
    app.directive('myGreeting', function() {
      return {
        restrict: 'E',
        template: '<div class="greeting">こんにちは！AngularJSの世界へようこそ！</div>'
      };
    });
  </script>
</body>
</html>
```

**ポイント：**
- `restrict` には `'E'`（要素）、`'A'`（属性）、`'C'`（クラス）、`'M'`（コメント）を指定できます
- 一般的には `'E'`（要素）と `'A'`（属性）がよく使われます
- ディレクティブ名の命名規則（キャメルケース↔ケバブケース）は非常に重要です

</details>

## 演習5-2（基本）：分離スコープと `@` バインディングを使おう

`<user-card>` ディレクティブを作成し、属性から文字列を受け取って表示してください。

**要件：**
- ディレクティブ名は `userCard`
- 分離スコープ（isolated scope）を使用
- `name` 属性と `role` 属性を `@` バインディングで受け取る
- テンプレートで名前と役割を表示する
- 異なるデータで3枚のカードを表示する

**期待される表示：**
```
名前：田中太郎
役割：エンジニア

名前：鈴木花子
役割：デザイナー

名前：佐藤次郎
役割：マネージャー
```

<details>
<summary>ヒント</summary>

`@` バインディングは属性の値を文字列としてスコープに渡します。`scope` オブジェクトで定義し、HTML属性名と対応させます。

```javascript
scope: {
  name: '@',
  role: '@'
}
```

HTML側では通常の属性として値を渡します：
```html
<user-card name="田中太郎" role="エンジニア"></user-card>
```

</details>

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <title>演習5-2：分離スコープと@バインディング</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .user-card {
      border: 1px solid #ccc;
      border-radius: 8px;
      padding: 15px;
      margin: 10px 0;
      max-width: 300px;
      background-color: #fafafa;
    }
    .user-card h3 { margin: 0 0 8px 0; color: #1976d2; }
    .user-card p { margin: 4px 0; }
  </style>
</head>
<body ng-controller="MainCtrl">
  <h2>ユーザーカード一覧</h2>

  <!-- ========================================
    @バインディングは属性の文字列をそのまま渡す
    コントローラーのスコープ変数を渡す場合は
    {{}} で補間する必要がある
  ======================================== -->
  <user-card name="田中太郎" role="エンジニア"></user-card>
  <user-card name="鈴木花子" role="デザイナー"></user-card>
  <user-card name="佐藤次郎" role="マネージャー"></user-card>

  <script>
    var app = angular.module('myApp', []);

    app.controller('MainCtrl', function() {});

    // ========================================
    // カスタムディレクティブ：userCard
    // scope で分離スコープを定義する
    // '@' バインディング：HTML属性の文字列をそのままスコープに渡す
    // 親スコープとは完全に分離されるため、再利用性が高い
    // ========================================
    app.directive('userCard', function() {
      return {
        restrict: 'E',
        scope: {
          name: '@',   // HTML属性 name の文字列をバインド
          role: '@'    // HTML属性 role の文字列をバインド
        },
        template:
          '<div class="user-card">' +
            '<h3>{{name}}</h3>' +
            '<p>名前：{{name}}</p>' +
            '<p>役割：{{role}}</p>' +
          '</div>'
      };
    });
  </script>
</body>
</html>
```

**ポイント：**
- `@` バインディングは文字列の一方向バインディング（親→子）です
- 分離スコープにすることで、ディレクティブ内部と外部のスコープが完全に分離されます
- これにより、同じディレクティブを異なるデータで何度でも再利用できます
- `@` で渡す値にコントローラーの変数を使う場合は `name="{{user.name}}"` のように `{{}}` が必要です

</details>

## 演習5-3（応用）：トランスクルード付きアラートボックスを作ろう

`<alert-box>` ディレクティブを作成し、中のコンテンツをトランスクルード（transclude）で差し込めるようにしてください。

**要件：**
- ディレクティブ名は `alertBox`
- `type` 属性を `@` バインディングで受け取る（`"success"`, `"warning"`, `"danger"`）
- `transclude: true` を使い、任意のコンテンツを差し込めるようにする
- `type` に応じて背景色を変える
- 閉じるボタン（×）を付けて、クリックで非表示にする

**期待される表示：**
```
[×] 操作が正常に完了しました。（緑の背景）
[×] この操作は取り消せません。ご注意ください。（黄色の背景）
[×] エラーが発生しました。再度お試しください。（赤い背景）
```

<details>
<summary>ヒント</summary>

トランスクルードを使うと、ディレクティブのタグで囲んだ内容をテンプレート内に埋め込めます。テンプレート側では `ng-transclude` ディレクティブを使います。

```javascript
return {
  restrict: 'E',
  transclude: true,
  scope: { type: '@' },
  template: '<div><span ng-transclude></span></div>'
};
```

閉じるボタンには `link` 関数で `element.remove()` を使うか、`ng-if` と内部スコープ変数を使います。

</details>

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <title>演習5-3：トランスクルード付きアラートボックス</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .alert-box {
      padding: 12px 35px 12px 15px;
      margin: 10px 0;
      border-radius: 4px;
      position: relative;
      font-size: 14px;
    }
    /* typeに応じた背景色 */
    .alert-success { background-color: #d4edda; border: 1px solid #c3e6cb; color: #155724; }
    .alert-warning { background-color: #fff3cd; border: 1px solid #ffeeba; color: #856404; }
    .alert-danger  { background-color: #f8d7da; border: 1px solid #f5c6cb; color: #721c24; }
    .alert-close {
      position: absolute;
      top: 8px;
      right: 10px;
      background: none;
      border: none;
      font-size: 18px;
      cursor: pointer;
      opacity: 0.7;
    }
    .alert-close:hover { opacity: 1; }
  </style>
</head>
<body ng-controller="MainCtrl">
  <h2>アラートボックス</h2>

  <!-- ========================================
    transclude を使うことで、タグの中身が
    テンプレートの ng-transclude 部分に挿入される
  ======================================== -->
  <alert-box type="success">
    操作が正常に完了しました。
  </alert-box>

  <alert-box type="warning">
    この操作は取り消せません。<strong>ご注意ください。</strong>
  </alert-box>

  <alert-box type="danger">
    エラーが発生しました。再度お試しください。
  </alert-box>

  <script>
    var app = angular.module('myApp', []);

    app.controller('MainCtrl', function() {});

    // ========================================
    // alertBox ディレクティブ
    // transclude: true → タグの中のコンテンツをテンプレート内に挿入
    // ng-transclude → トランスクルードされた内容が表示される場所
    // link関数 → DOM操作やイベント処理を記述する場所
    // ========================================
    app.directive('alertBox', function() {
      return {
        restrict: 'E',
        transclude: true,   // トランスクルードを有効にする
        scope: {
          type: '@'          // "success", "warning", "danger"
        },
        template:
          '<div class="alert-box alert-{{type}}" ng-if="!closed">' +
            '<button class="alert-close" ng-click="close()">&times;</button>' +
            '<span ng-transclude></span>' +
          '</div>',
        link: function(scope) {
          // ========================================
          // closed フラグで表示/非表示を制御
          // ng-if="!closed" によりDOMから完全に削除される
          // ========================================
          scope.closed = false;
          scope.close = function() {
            scope.closed = true;
          };
        }
      };
    });
  </script>
</body>
</html>
```

**ポイント：**
- `transclude: true` を指定すると、ディレクティブ要素の中身がテンプレートの `ng-transclude` の位置に挿入されます
- トランスクルードされた内容はHTMLタグもそのまま保持されます（`<strong>` など）
- `link` 関数はディレクティブのDOM操作やイベントハンドリングに使用します
- `ng-if` は要素をDOMから完全に削除し、`ng-show` は `display: none` で非表示にするだけです

</details>

## 演習5-4（応用）：折りたたみパネルディレクティブを作ろう

`<collapsible-panel>` ディレクティブを作成し、外部のコントローラーから開閉状態を制御できるようにしてください。

**要件：**
- ディレクティブ名は `collapsiblePanel`
- `title` 属性を `@` バインディングで受け取る
- `is-open` 属性を `=` バインディングで外部スコープと双方向バインドする
- ヘッダーをクリックすると開閉をトグルする
- コントローラー側に「すべて開く」「すべて閉じる」ボタンを追加
- トランスクルードでパネルの中身を自由に設定できるようにする

**期待される動作：**
- パネルのヘッダーをクリックすると中身が開閉する
- 「すべて開く」ボタンですべてのパネルが開く
- 「すべて閉じる」ボタンですべてのパネルが閉じる

<details>
<summary>ヒント</summary>

`=` バインディングは親スコープの変数と双方向バインディングを行います。ディレクティブ内で値を変更すると、親スコープにも反映されます。

```javascript
scope: {
  title: '@',
  isOpen: '='  // HTMLでは is-open
}
```

コントローラー側では各パネルの開閉状態を管理する変数を用意します：
```javascript
$scope.panel1Open = true;
$scope.panel2Open = false;
```

</details>

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <title>演習5-4：折りたたみパネルディレクティブ</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .panel { border: 1px solid #ddd; border-radius: 4px; margin: 10px 0; }
    .panel-header {
      padding: 12px 15px;
      background-color: #f5f5f5;
      cursor: pointer;
      display: flex;
      justify-content: space-between;
      align-items: center;
      font-weight: bold;
    }
    .panel-header:hover { background-color: #e8e8e8; }
    .panel-body { padding: 15px; border-top: 1px solid #ddd; }
    .toggle-icon { transition: transform 0.2s; }
    .toggle-icon.open { transform: rotate(90deg); }
    .controls { margin: 15px 0; }
    .controls button {
      padding: 8px 16px;
      margin-right: 8px;
      cursor: pointer;
      border: 1px solid #ccc;
      border-radius: 4px;
      background: #fff;
    }
    .controls button:hover { background: #f0f0f0; }
  </style>
</head>
<body ng-controller="MainCtrl">
  <h2>折りたたみパネル</h2>

  <!-- コントローラー側の一括操作ボタン -->
  <div class="controls">
    <button ng-click="openAll()">すべて開く</button>
    <button ng-click="closeAll()">すべて閉じる</button>
  </div>

  <!-- ========================================
    is-open 属性で = バインディングを使用
    コントローラーの変数と双方向に連動する
    ディレクティブ内でトグルすると親にも反映される
  ======================================== -->
  <collapsible-panel title="セクション1：はじめに" is-open="panel1Open">
    <p>AngularJSは、Googleが開発したJavaScriptフレームワークです。</p>
  </collapsible-panel>

  <collapsible-panel title="セクション2：ディレクティブ" is-open="panel2Open">
    <p>ディレクティブはAngularJSの最も強力な機能の一つです。</p>
    <p>HTMLを拡張して独自の要素や属性を作ることができます。</p>
  </collapsible-panel>

  <collapsible-panel title="セクション3：サービス" is-open="panel3Open">
    <p>サービスはアプリケーション全体で共有するロジックやデータを管理します。</p>
  </collapsible-panel>

  <p>パネル状態：1={{panel1Open}}, 2={{panel2Open}}, 3={{panel3Open}}</p>

  <script>
    var app = angular.module('myApp', []);

    app.controller('MainCtrl', ['$scope', function($scope) {
      // 各パネルの開閉状態をコントローラーで管理
      $scope.panel1Open = true;   // 初期状態：開
      $scope.panel2Open = false;  // 初期状態：閉
      $scope.panel3Open = false;  // 初期状態：閉

      // すべて開く
      $scope.openAll = function() {
        $scope.panel1Open = true;
        $scope.panel2Open = true;
        $scope.panel3Open = true;
      };

      // すべて閉じる
      $scope.closeAll = function() {
        $scope.panel1Open = false;
        $scope.panel2Open = false;
        $scope.panel3Open = false;
      };
    }]);

    // ========================================
    // collapsiblePanel ディレクティブ
    // scope.isOpen は '=' バインディングにより
    // 親スコープの変数と双方向に連動する
    // ディレクティブ内で toggle() すると親にも反映される
    // ========================================
    app.directive('collapsiblePanel', function() {
      return {
        restrict: 'E',
        transclude: true,
        scope: {
          title: '@',     // 文字列として受け取る（一方向）
          isOpen: '='     // 双方向バインディング（HTMLでは is-open）
        },
        template:
          '<div class="panel">' +
            '<div class="panel-header" ng-click="toggle()">' +
              '<span>{{title}}</span>' +
              '<span class="toggle-icon" ng-class="{open: isOpen}">▶</span>' +
            '</div>' +
            '<div class="panel-body" ng-show="isOpen">' +
              '<div ng-transclude></div>' +
            '</div>' +
          '</div>',
        link: function(scope) {
          // ========================================
          // toggle関数：isOpenを反転する
          // '=' バインディングなので、この変更は
          // 親コントローラーの変数にも自動で反映される
          // ========================================
          scope.toggle = function() {
            scope.isOpen = !scope.isOpen;
          };
        }
      };
    });
  </script>
</body>
</html>
```

**ポイント：**
- `=` バインディングは双方向バインディングです。ディレクティブ内の変更が親スコープに、親スコープの変更がディレクティブに反映されます
- `@` は文字列の一方向、`=` はオブジェクト・値の双方向という違いを理解しましょう
- HTML属性名のケバブケース `is-open` がJavaScript側のキャメルケース `isOpen` に対応します
- パネル状態の表示で、双方向バインディングが正しく動作していることを確認できます

</details>

## 演習5-5（チャレンジ）：スター評価ディレクティブを作ろう

`<star-rating>` ディレクティブを作成し、星をクリックして評価値を設定できるようにしてください。評価値が変わったときに親コントローラーにコールバックで通知します。

**要件：**
- ディレクティブ名は `starRating`
- `rating` 属性を `=` バインディングで双方向バインド（現在の評価値）
- `max` 属性を `@` バインディングで最大星数を設定（デフォルト5）
- `on-rate` 属性を `&` バインディングでコールバック関数を受け取る
- 星をクリックするとその数が評価値になり、コールバックが呼ばれる
- 星にホバーすると仮のハイライトが表示される
- 商品リスト3件でスター評価を使用し、評価変更時にログを表示

**期待される動作：**
- 各商品の横に星が5つ表示される
- 星をクリックすると評価値が変わる
- 評価変更時に「○○ の評価が ☆N に変更されました」とログ表示される
- ホバー時に星がプレビュー表示される

<details>
<summary>ヒント</summary>

`&` バインディングは親スコープの式（関数呼び出し）をディレクティブから実行するために使います。

```javascript
scope: {
  rating: '=',
  max: '@',
  onRate: '&'    // HTMLでは on-rate
}
```

HTML側では：
```html
<star-rating rating="item.rating" max="5"
  on-rate="onRatingChange({item: item, value: value})">
</star-rating>
```

ディレクティブ内からコールバックを呼ぶ：
```javascript
scope.onRate({ value: newRating });
```

`&` バインディングではオブジェクト形式で引数を渡します。

</details>

```html
<body ng-controller="MainCtrl">
  <h2>商品評価</h2>

  <!-- ========================================
    star-rating ディレクティブを各商品に適用
    on-rate に & バインディングで評価変更時のコールバックを渡す
    {item: product, value: value} でディレクティブに渡す引数名を指定
  ======================================== -->
  <div class="product-item" ng-repeat="product in products">
    <span class="product-name">{{product.name}}</span>
    <star-rating
      rating="product.rating"
      max="5"
      on-rate="onRatingChange({item: product, value: value})">
    </star-rating>
    <span class="product-rating-value">{{product.rating}} / 5</span>
  </div>

  <h3>変更ログ</h3>
  <div class="log-area">
    <div class="log-entry" ng-repeat="log in logs track by $index">
      {{log}}
    </div>
    <div ng-if="logs.length === 0" style="color: #999;">
      星をクリックすると変更ログが表示されます
    </div>
  </div>

  <script>
    var app = angular.module('myApp', []);

    app.controller('MainCtrl', ['$scope', function($scope) {
      // 商品データ
      $scope.products = [
        { name: 'ノートPC', rating: 4 },
        { name: 'マウス', rating: 2 },
        { name: 'キーボード', rating: 0 }
      ];

      // 変更ログ
      $scope.logs = [];

      // ========================================
      // 評価変更時のコールバック
      // & バインディング経由でディレクティブから呼ばれる
      // item と value はディレクティブ側から渡される引数
      // ========================================
      $scope.onRatingChange = function(params) {
        var message = params.item.name + ' の評価が ☆' + params.value + ' に変更されました';
        $scope.logs.unshift(message);
      };
    }]);

    // ========================================
    // starRating ディレクティブ
    // rating: '='  → 親と双方向バインディング
    // max: '@'     → 最大星数（文字列→数値に変換して使用）
    // onRate: '&'  → 評価変更時に親のコールバックを実行
    // ========================================
    app.directive('starRating', function() {
      return {
        restrict: 'E',
        scope: {
          rating: '=',   // 双方向バインディング（評価値）
          max: '@',       // 最大星数（文字列）
          onRate: '&'     // コールバック関数（& バインディング）
        },
        template:
          '<span class="stars">' +
            '<span class="star" ng-repeat="star in stars track by $index" ' +
              'ng-class="{filled: $index < rating, hovered: $index < hoverRating}" ' +
              'ng-click="rate($index + 1)" ' +
              'ng-mouseenter="hover($index + 1)" ' +
              'ng-mouseleave="hover(0)">&#9733;</span>' +
          '</span>',
        link: function(scope) {
          // maxを数値に変換（デフォルト5）
          var maxStars = parseInt(scope.max, 10) || 5;

          // ========================================
          // 星の配列を生成
          // ng-repeat で繰り返し表示するために使用
          // ========================================
          scope.stars = new Array(maxStars);

          // ホバー中の仮評価値（0ならホバーなし）
          scope.hoverRating = 0;

          // ========================================
          // 星をクリックしたときの処理
          // 1. rating を更新（= バインディングで親にも反映）
          // 2. onRate コールバックを呼び出す
          //    引数はオブジェクト形式で渡す（& バインディングの仕様）
          // ========================================
          scope.rate = function(value) {
            scope.rating = value;
            scope.onRate({ value: value });
          };

          // ホバー時のプレビュー
          scope.hover = function(value) {
            scope.hoverRating = value;
          };
        }
      };
    });
  </script>
</body>
</html>
```

**ポイント：**
- `&` バインディングは親スコープの式をディレクティブから実行するためのものです
- コールバック呼び出し時は `scope.onRate({ value: value })` のようにオブジェクト形式で引数を渡します
- HTML側の `on-rate="onRatingChange({item: product, value: value})"` の `value` がディレクティブから渡される値に対応します
- `@`（文字列一方向）、`=`（双方向）、`&`（コールバック）の3つのバインディングの使い分けがディレクティブ設計の要です
- ホバー表示と確定表示を分けることで、ユーザー体験を向上させています
