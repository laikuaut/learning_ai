# 実践課題06：テーマ切替アプリ ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第6章（カスタムフック）、第7章（useContextとグローバル状態）
> **課題の種類**: ミニプロジェクト
> **学習目標**: Context APIでグローバル状態を管理し、カスタムフックで共通ロジックを抽出する

---

## 完成イメージ

ダーク/ライトモードの切り替えが、ヘッダー・カード・フッターなど全コンポーネントに即座に反映されます。

```
┌───────────────────────────────────────┐
│ ☀ MyApp                [🌙 ダーク]    │  ← ヘッダー（トグルボタン）
├───────────────────────────────────────┤
│                                       │
│  ┌─────────┐  ┌─────────┐            │
│  │ カード1  │  │ カード2  │            │  ← コンテンツ（テーマ反映）
│  │ ...      │  │ ...      │            │
│  └─────────┘  └─────────┘            │
│                                       │
│  フォントサイズ: [小] [中] [大]        │  ← 設定パネル
│  カラー: [青] [緑] [紫]               │
│                                       │
├───────────────────────────────────────┤
│ © 2024 MyApp                         │  ← フッター（テーマ反映）
└───────────────────────────────────────┘
```

---

## 課題の要件

1. `ThemeContext` を作成し、ダーク/ライトモードをグローバルに管理する
2. ヘッダーのトグルボタンでテーマを切り替えられる
3. 全コンポーネント（ヘッダー、カード、フッター）にテーマが反映される
4. `useTheme` カスタムフックでContextの利用を簡潔にする
5. **追加設定**: フォントサイズ（小/中/大）とアクセントカラーもContextで管理する
6. 設定変更が即座に画面全体に反映される

---

## ステップガイド

<details>
<summary>ステップ1：ThemeContextを作成する</summary>

```jsx
const ThemeContext = React.createContext();

function ThemeProvider({ children }) {
  const [theme, setTheme] = React.useState("light");

  const toggleTheme = () => {
    setTheme(prev => prev === "light" ? "dark" : "light");
  };

  return (
    <ThemeContext.Provider value={{ theme, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
}
```

</details>

<details>
<summary>ステップ2：useThemeカスタムフックを作る</summary>

```jsx
function useTheme() {
  const context = React.useContext(ThemeContext);
  if (!context) {
    throw new Error("useTheme must be used within a ThemeProvider");
  }
  return context;
}
```

子コンポーネントでは `const { theme, toggleTheme } = useTheme();` で使います。

</details>

</head>
<body>
  <div id="root"></div>
  <script type="text/babel">
    // ==================== テーマ定義 ====================
    const THEMES = {
      light: { bg: "#f8f9fa", text: "#333", card: "#ffffff", border: "#dee2e6", headerBg: "#ffffff" },
      dark:  { bg: "#1a1a2e", text: "#e0e0e0", card: "#16213e", border: "#2a2a4a", headerBg: "#0f3460" },
    };

    const ACCENT_COLORS = {
      blue:   "#2196f3",
      green:  "#4caf50",
      purple: "#9c27b0",
    };

    const FONT_SIZES = {
      small:  { base: 14, heading: 20 },
      medium: { base: 16, heading: 24 },
      large:  { base: 18, heading: 28 },
    };

    // ==================== Context ====================
    const ThemeContext = React.createContext();

    function ThemeProvider({ children }) {
      const [theme, setTheme] = React.useState("light");
      const [accentColor, setAccentColor] = React.useState("blue");
      const [fontSize, setFontSize] = React.useState("medium");

      function toggleTheme() {
        setTheme(theme === "light" ? "dark" : "light");
      }

      var value = {
        theme: theme,
        colors: THEMES[theme],
        accent: ACCENT_COLORS[accentColor],
        accentName: accentColor,
        fontSize: fontSize,
        fontSizes: FONT_SIZES[fontSize],
        toggleTheme: toggleTheme,
        setAccentColor: setAccentColor,
        setFontSize: setFontSize,
      };

      return (
        <ThemeContext.Provider value={value}>
          {children}
        </ThemeContext.Provider>
      );
    }

    // ==================== カスタムフック ====================
    function useTheme() {
      return React.useContext(ThemeContext);
    }

    // ==================== コンポーネント ====================
    function Header() {
      var ctx = useTheme();

      var style = {
        display: "flex", justifyContent: "space-between", alignItems: "center",

        borderBottom: "1px solid " + ctx.colors.border, transition: "all 0.3s",
      };

      return (

          <h1 style={{ fontSize: ctx.fontSizes.heading, color: ctx.accent }}>

          </h1>
          <button
            onClick={ctx.toggleTheme}
            style={{
              padding: "8px 20px", border: "none", borderRadius: 20,
              background: ctx.accent, color: "white", cursor: "pointer", fontSize: 14,

        </header>
      );

    function Card({ title, content }) {

      var style = {
        backgroundColor: ctx.colors.card, border: "1px solid " + ctx.colors.border,
        borderRadius: 12, padding: 20, marginBottom: 16,
        transition: "all 0.3s", fontSize: ctx.fontSizes.base,

      return (
        <div style={style}>
          <h3 style={{ color: ctx.accent, marginBottom: 8, fontSize: ctx.fontSizes.base + 4 }}>
            {title}
          </h3>
          <p style={{ color: ctx.colors.text, lineHeight: 1.6 }}>{content}</p>
        </div>
      );
    }

    function SettingsPanel() {
      var ctx = useTheme();

      var panelStyle = {
        backgroundColor: ctx.colors.card, border: "1px solid " + ctx.colors.border,
        borderRadius: 12, padding: 20, marginBottom: 16, transition: "all 0.3s",
      };

      var btnStyle = function(isActive) {
        return {
          padding: "6px 16px", margin: "0 4px", border: isActive ? "2px solid " + ctx.accent : "1px solid " + ctx.colors.border,

          color: isActive ? ctx.accent : ctx.colors.text, cursor: "pointer", fontSize: ctx.fontSizes.base - 2,

      return (
        <div style={panelStyle}>
          <h3 style={{ color: ctx.accent, marginBottom: 16 }}>設定</h3>

            })}

              var label = colorName === "blue" ? "青" : colorName === "green" ? "緑" : "紫";
              return (
                <button key={colorName} style={btnStyle(ctx.accentName === colorName)} onClick={() => ctx.setAccentColor(colorName)}>
                  <span style={{ color: ACCENT_COLORS[colorName] }}>●</span> {label}
                </button>
              );
            })}
          </div>
        </div>
      );
    }

    function Footer() {
      var ctx = useTheme();

      return (
        <footer style={{
          textAlign: "center", padding: "16px", color: ctx.colors.text,
          borderTop: "1px solid " + ctx.colors.border, fontSize: ctx.fontSizes.base - 2,

    function App() {
      var ctx = useTheme();

      return (
        <div style={{
          minHeight: "100vh", backgroundColor: ctx.colors.bg,

  </script>
</body>

<summary>解答例（改良版 ─ localStorageで永続化）</summary>

テーマ設定をlocalStorageに保存し、次回アクセス時に復元する改良です。差分のみ示します。

```jsx
// localStorageから読み込むカスタムフック
function useLocalStorage(key, defaultValue) {
  const [value, setValue] = React.useState(() => {
    try {
      const saved = localStorage.getItem(key);
      return saved !== null ? JSON.parse(saved) : defaultValue;
    } catch {
      return defaultValue;
    }
  });

  React.useEffect(() => {
    localStorage.setItem(key, JSON.stringify(value));
  }, [key, value]);

  return [value, setValue];
}

// ThemeProviderでuseLocalStorageを使う
function ThemeProvider({ children }) {
  const [theme, setTheme] = useLocalStorage("theme", "light");
  const [accentColor, setAccentColor] = useLocalStorage("accent", "blue");
  const [fontSize, setFontSize] = useLocalStorage("fontSize", "medium");

  // ...以下同じ
}
```

**初心者向けとの違い:**
- `useLocalStorage` カスタムフックで永続化を抽象化
- 遅延初期化 `useState(() => ...)` でlocalStorage読み込み
- `useEffect` で値が変わるたびに自動保存
