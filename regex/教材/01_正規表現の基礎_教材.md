## 1.1 正規表現とは

### 正規表現の定義

正規表現（Regular Expression、略称：regex / regexp）とは、**文字列のパターンを記述するための小さな言語**です。「この文字列はメールアドレスの形式か？」「電話番号を抽出したい」「URLをリンクに置換したい」といった操作を、パターンの記述だけで実現できます。

```
【正規表現のイメージ】

文字列:  "お問い合わせは 03-1234-5678 まで"

パターン: \d{2,4}-\d{2,4}-\d{4}

結果:    "03-1234-5678" がマッチ！
```

### 正規表現でできること

| 操作 | 説明 | 例 |
|---|---|---|
| **検索（Search）** | パターンに一致する部分を見つける | テキストから電話番号を探す |
| **マッチング（Match）** | 文字列全体がパターンに一致するか判定 | 入力値がメールアドレスか検証 |
| **抽出（Extract）** | パターンに一致する部分を取り出す | ログから日付だけを取り出す |
| **置換（Replace）** | パターンに一致する部分を別の文字列に置き換え | 個人情報をマスキングする |
| **分割（Split）** | パターンを区切りとして文字列を分割 | CSVの行をカンマで分割する |

### 正規表現が使える場所

正規表現はPythonだけでなく、多くの場所で使えます。

```
【正規表現が使える場所】

プログラミング言語:  Python, JavaScript, Java, Go, Ruby, PHP ...
テキストエディタ  :  VSCode, Vim, Sublime Text ...
コマンドラインツール: grep, sed, awk ...
データベース      :  MySQL (REGEXP), PostgreSQL (~) ...
その他            :  Nginx設定, Apache .htaccess ...
```

> **実務でのポイント**: 正規表現は一度覚えれば、プログラミング言語を問わず幅広く使えるスキルです。構文に多少の方言（flavor）はありますが、基本的な考え方は共通です。

## 1.2 リテラル文字

### リテラル文字とは

リテラル文字（Literal Character）とは、**その文字自体にマッチする文字**のことです。正規表現の最も基本的な要素です。

```python
import re

text = "Hello, World!"
# "Hello" というリテラル文字でマッチング
result = re.search(r"Hello", text)
print(result.group())  # Hello
```

- 正規表現 `Hello` は文字列中の "Hello" にそのままマッチします
- 英字の大文字と小文字は区別されます（`Hello` ≠ `hello`）

```python
import re

text = "apple banana apple cherry"

# "apple" を検索 → 最初に見つかった位置にマッチ
result = re.search(r"apple", text)
print(result.group())   # apple
print(result.start())   # 0（位置）

# すべての "apple" を検索
results = re.findall(r"apple", text)
print(results)  # ['apple', 'apple']
```

### raw文字列（r""）の重要性

Pythonで正規表現を書くときは、**raw文字列（`r""`）**を使います。これにより、バックスラッシュ `\` がPythonのエスケープシーケンスとして解釈されず、正規表現エンジンにそのまま渡されます。

```python
# ❌ 通常の文字列：\d がPythonに解釈されてしまう場合がある
pattern1 = "\d+"

# ✅ raw文字列：\d がそのまま正規表現として渡される
pattern2 = r"\d+"
```

> **実務でのポイント**: Pythonで正規表現を書くときは、常に `r""` を使う習慣をつけましょう。通常の文字列でも動く場合はありますが、`\b`（正規表現では単語境界）がPythonでは「バックスペース」になるなど、思わぬバグの原因になります。

## 1.3 メタ文字

### メタ文字とは

メタ文字（Meta Character）とは、**特別な意味を持つ文字**のことです。リテラル文字がその文字自体にマッチするのに対し、メタ文字はパターンの「ルール」を表します。

### ドット（`.`）：任意の1文字

ドット `.` は、**改行を除く任意の1文字**にマッチします。

```python
import re

text = "cat cot cut c0t c_t c t"

# c.t → "c" + 任意の1文字 + "t"
results = re.findall(r"c.t", text)
print(results)  # ['cat', 'cot', 'cut', 'c0t', 'c_t', 'c t']
```

```
【ドットのマッチング】

パターン: c.t

c a t  →  マッチ ✓（a は任意の1文字）
c o t  →  マッチ ✓（o は任意の1文字）
c u t  →  マッチ ✓（u は任意の1文字）
c t    →  マッチ ✗（. は1文字必要、0文字はダメ）
c a r t → マッチ ✗（. は1文字のみ、2文字はダメ）
```

### `\d`：数字1文字

`\d` は、**半角数字（0-9）の1文字**にマッチします。`d` は digit（数字）の頭文字です。

```python
import re

text = "注文番号: A-12345"

results = re.findall(r"\d", text)
print(results)  # ['1', '2', '3', '4', '5']

# \d\d\d で連続する3桁の数字
results = re.findall(r"\d\d\d", text)
print(results)  # ['123', '345'] ← 重複あり（後述の量指定子で改善）
```

### `\D`：数字以外の1文字

`\D` は、`\d` の否定（complement）で、**数字以外の任意の1文字**にマッチします。大文字にすると否定になるルールは他のメタ文字にも共通です。

```python
import re

text = "Phone: 090-1234-5678"

results = re.findall(r"\D", text)
print(results)  # ['P', 'h', 'o', 'n', 'e', ':', ' ', '-', '-']
```

### `\w`：単語文字1文字

`\w` は、**英数字とアンダースコア（[a-zA-Z0-9_]）の1文字**にマッチします。`w` は word（単語）の頭文字です。

```python
import re

text = "user_name = hello123"

results = re.findall(r"\w", text)
print(results)
# ['u', 's', 'e', 'r', '_', 'n', 'a', 'm', 'e', 'h', 'e', 'l', 'l', 'o', '1', '2', '3']
```

### `\W`：単語文字以外の1文字

```python
import re

text = "Hello, World! 123"

results = re.findall(r"\W", text)
print(results)  # [',', ' ', '!', ' ']
```

### `\s`：空白文字1文字

`\s` は、**空白文字（スペース、タブ、改行など）の1文字**にマッチします。`s` は space（空白）の頭文字です。

```python
import re

text = "Hello World\tPython\nRegex"

results = re.findall(r"\s", text)
print(results)  # [' ', '\t', '\n']
```

### `\S`：空白以外の1文字

```python
import re

text = "a b c"

results = re.findall(r"\S", text)
print(results)  # ['a', 'b', 'c']
```

### メタ文字のまとめ

```
【基本メタ文字の一覧】

┌──────┬───────────────────────────────┬────────────┐
│ 記号 │ マッチする文字                  │ 同等の表記   │
├──────┼───────────────────────────────┼────────────┤
│  .   │ 改行以外の任意の1文字           │ -          │
│ \d   │ 数字（0-9）                    │ [0-9]      │
│ \D   │ 数字以外                       │ [^0-9]     │
│ \w   │ 英数字とアンダースコア          │ [a-zA-Z0-9_]│
│ \W   │ \w以外                        │ [^a-zA-Z0-9_]│
│ \s   │ 空白文字（スペース、タブ、改行） │ [ \t\n\r\f]│
│ \S   │ 空白以外                       │ [^ \t\n\r\f]│
└──────┴───────────────────────────────┴────────────┘

覚え方：小文字 = マッチ、大文字 = その否定
```

## 1.4 エスケープ

### メタ文字をリテラルとして使う

メタ文字 `.` `*` `+` `?` `(` `)` `[` `]` `{` `}` `\` `^` `$` `|` をリテラル文字（その文字自体）としてマッチさせたい場合は、バックスラッシュ `\` を前に付けてエスケープします。

```python
import re

text = "価格は1,500円です。合計は3.14ドルです。"

# ドットをエスケープせずに使うと、任意の1文字にマッチしてしまう
results = re.findall(r"3.14", text)
print(results)  # ['3.14']  ← たまたま合っているが...

text2 = "3a14 3.14 3x14"
results = re.findall(r"3.14", text2)
print(results)  # ['3a14', '3.14', '3x14']  ← 全部マッチ！

# ドットをエスケープすると、文字通りの「.」にだけマッチ
results = re.findall(r"3\.14", text2)
print(results)  # ['3.14']  ← 正しい
```

```
【エスケープが必要なメタ文字】

.  →  \.    （ドット）
*  →  \*    （アスタリスク）
+  →  \+    （プラス）
?  →  \?    （クエスチョン）
(  →  \(    （丸括弧）
)  →  \)
[  →  \[    （角括弧）
]  →  \]
{  →  \{    （波括弧）
}  →  \}
\  →  \\    （バックスラッシュ）
^  →  \^    （キャレット）
$  →  \$    （ドル記号）
|  →  \|    （パイプ）
```

## 1.5 Pythonで正規表現を体験する

### re.search() で最初のマッチを探す

```python
import re

text = "今日の日付は 2026-04-05 です"

# \d{4}-\d{2}-\d{2} で日付パターンにマッチ
result = re.search(r"\d{4}-\d{2}-\d{2}", text)

if result:
    print(f"マッチした文字列: {result.group()}")  # 2026-04-05
    print(f"開始位置: {result.start()}")           # 7
    print(f"終了位置: {result.end()}")             # 17
else:
    print("マッチしませんでした")
```

### re.findall() ですべてのマッチを探す

```python
import re

text = "連絡先: 090-1111-2222 または 03-3333-4444"

# 電話番号のパターン
results = re.findall(r"\d{2,4}-\d{4}-\d{4}", text)
print(results)  # ['090-1111-2222', '03-3333-4444']
```

### re.sub() で置換する

```python
import re

text = "私の電話番号は 090-1234-5678 です"

# 電話番号をマスキング
masked = re.sub(r"\d{3}-\d{4}-\d{4}", "***-****-****", text)
print(masked)  # 私の電話番号は ***-****-**** です
```
