## 演習1（基本）：三項演算子で表示を切り替えよう

## 演習5（基本）：空リストの処理をしよう

タスクリストを表示するコンポーネントを作り、リストが空の場合は「タスクがありません」というメッセージを表示してください。

**要件：**
- タスクの追加・削除機能
- 全削除したときに空メッセージが表示される

<details>
<summary>ヒント</summary>

`if (tasks.length === 0) return <p>タスクがありません</p>` または三項演算子で空の場合を処理します。

</details>

<details>
<summary>解答例</summary>

```jsx
import { useState } from 'react';

function TaskList() {
  const [tasks, setTasks] = useState(["買い物", "掃除", "勉強"]);
  const [input, setInput] = useState("");

  const addTask = () => {
    if (input.trim() === "") return;
    setTasks([...tasks, input.trim()]);
    setInput("");
  };

  const removeTask = (index) => {
    setTasks(tasks.filter((_, i) => i !== index));
  };

  return (
    <div style={{ maxWidth: '400px', margin: '0 auto', padding: '20px' }}>
      <h2>タスクリスト</h2>

      <div style={{ display: 'flex', gap: '8px', marginBottom: '16px' }}>
        <input
          type="text"
          value={input}
          onChange={(e) => setInput(e.target.value)}
          onKeyDown={(e) => e.key === "Enter" && addTask()}
          placeholder="タスクを追加..."
          style={{ flex: 1, padding: '8px' }}
        />
        <button onClick={addTask}>追加</button>
      </div>

      {tasks.length === 0 ? (
        <p style={{ color: '#999', textAlign: 'center', padding: '20px' }}>
          タスクがありません。新しいタスクを追加しましょう！
        </p>
      ) : (
        <ul style={{ listStyle: 'none', padding: 0 }}>
          {tasks.map((task, index) => (
            <li key={index} style={{
              display: 'flex',
              justifyContent: 'space-between',
              padding: '8px',
              borderBottom: '1px solid #eee',
            }}>
              <span>{task}</span>
              <button onClick={() => removeTask(index)}>削除</button>
            </li>
          ))}
        </ul>
      )}
    </div>
  );
}

export default TaskList;
```

## 演習7（応用）：並び替え機能を追加しよう

演習6のリストに並び替え機能を追加してください。

**要件：**
- 「名前（昇順）」「名前（降順）」「価格（安い順）」「価格（高い順）」で並び替え
- セレクトボックスで並び順を切り替える

<details>
<summary>ヒント</summary>

元の配列を変更しないよう `[...products].sort(...)` でコピーしてからソートします。`localeCompare` で文字列の比較、`a - b` で数値の比較ができます。

</details>

<details>
<summary>解答例</summary>

```jsx
import { useState } from 'react';

function SortableProducts() {
  const products = [
    { id: 1, name: "ノートPC", price: 89800 },
    { id: 2, name: "マウス", price: 3980 },
    { id: 3, name: "キーボード", price: 12800 },
    { id: 4, name: "モニター", price: 39800 },
    { id: 5, name: "ヘッドセット", price: 8980 },
  ];

  const [sortKey, setSortKey] = useState("name-asc");

  const sorted = [...products].sort((a, b) => {
    switch (sortKey) {
      case "name-asc": return a.name.localeCompare(b.name);
      case "name-desc": return b.name.localeCompare(a.name);
      case "price-asc": return a.price - b.price;
      case "price-desc": return b.price - a.price;
      default: return 0;
    }
  });

  return (
    <div style={{ maxWidth: '500px', margin: '0 auto', padding: '20px' }}>
      <h2>商品一覧</h2>

      <select
        value={sortKey}
        onChange={(e) => setSortKey(e.target.value)}
        style={{ padding: '8px', marginBottom: '16px' }}
      >
        <option value="name-asc">名前（昇順）</option>
        <option value="name-desc">名前（降順）</option>
        <option value="price-asc">価格（安い順）</option>
        <option value="price-desc">価格（高い順）</option>
      </select>

      {sorted.map(product => (
        <div key={product.id} style={{
          display: 'flex',
          justifyContent: 'space-between',
          padding: '12px',
          borderBottom: '1px solid #eee',
        }}>
          <span>{product.name}</span>
          <span style={{ fontWeight: 'bold' }}>¥{product.price.toLocaleString()}</span>
        </div>
      ))}
    </div>
  );
}

export default SortableProducts;
```

## 演習10（チャレンジ）：Todoアプリを作ろう

以下の機能を備えたTodoアプリを作成してください。

**要件：**
1. タスクの追加（Enter または「追加」ボタン）
2. タスクの完了/未完了の切り替え（チェックボックス）
3. タスクの削除
4. フィルタ機能（すべて / 未完了 / 完了済み）
5. 件数表示（「残り〇件」）
6. 空の場合のメッセージ表示

<details>
<summary>ヒント</summary>

- タスクのデータ構造：`{ id: Date.now(), text: "...", completed: false }`
- 完了切り替え：`setTodos(todos.map(t => t.id === id ? { ...t, completed: !t.completed } : t))`
- フィルタ：`filter` ステータスのstateを持ち、表示前にfilterをかける

</details>

<details>
<summary>解答例</summary>

```jsx
import { useState } from 'react';

function TodoApp() {
  const [todos, setTodos] = useState([
    { id: 1, text: "Reactの基本を学ぶ", completed: true },
    { id: 2, text: "コンポーネントを作る", completed: false },
    { id: 3, text: "Todoアプリを完成させる", completed: false },
  ]);
  const [input, setInput] = useState("");
  const [filter, setFilter] = useState("all"); // all, active, completed

  const addTodo = () => {
    if (input.trim() === "") return;
    setTodos([...todos, { id: Date.now(), text: input.trim(), completed: false }]);
    setInput("");
  };

  const toggleTodo = (id) => {
    setTodos(todos.map(todo =>
      todo.id === id ? { ...todo, completed: !todo.completed } : todo
    ));
  };

  const deleteTodo = (id) => {
    setTodos(todos.filter(todo => todo.id !== id));
  };

  const filteredTodos = todos.filter(todo => {
    if (filter === "active") return !todo.completed;
    if (filter === "completed") return todo.completed;
    return true;
  });

  const remainingCount = todos.filter(t => !t.completed).length;

  return (
    <div style={{ maxWidth: '500px', margin: '0 auto', padding: '20px' }}>
      <h1>Todo App</h1>

      {/* 入力エリア */}
      <div style={{ display: 'flex', gap: '8px', marginBottom: '16px' }}>
        <input
          type="text"
          value={input}
          onChange={(e) => setInput(e.target.value)}
          onKeyDown={(e) => e.key === "Enter" && addTodo()}
          placeholder="新しいタスクを入力..."
          style={{ flex: 1, padding: '10px' }}
        />
        <button onClick={addTodo} style={{ padding: '10px 20px' }}>追加</button>
      </div>

      {/* フィルタ */}
      <div style={{ display: 'flex', gap: '8px', marginBottom: '12px' }}>
        {["all", "active", "completed"].map(f => (
          <button
            key={f}
            onClick={() => setFilter(f)}
            style={{
              padding: '6px 14px',
              backgroundColor: filter === f ? '#3498db' : '#ecf0f1',
              color: filter === f ? 'white' : '#333',
              border: 'none',
              borderRadius: '4px',
              cursor: 'pointer',
            }}
          >
            {f === "all" ? "すべて" : f === "active" ? "未完了" : "完了済み"}
          </button>
        ))}
      </div>

      <p style={{ color: '#666' }}>残り{remainingCount}件のタスク</p>

      {/* タスクリスト */}
      {filteredTodos.length === 0 ? (
        <p style={{ color: '#999', textAlign: 'center', padding: '20px' }}>
          {filter === "all" ? "タスクがありません。" :
           filter === "active" ? "未完了のタスクはありません。" :
           "完了済みのタスクはありません。"}
        </p>
      ) : (
        filteredTodos.map(todo => (
          <div key={todo.id} style={{
            display: 'flex',
            alignItems: 'center',
            padding: '10px',
            borderBottom: '1px solid #eee',
            gap: '10px',
          }}>
            <input
              type="checkbox"
              checked={todo.completed}
              onChange={() => toggleTodo(todo.id)}
            />
            <span style={{
              flex: 1,
              textDecoration: todo.completed ? 'line-through' : 'none',
              color: todo.completed ? '#999' : '#333',
            }}>
              {todo.text}
            </span>
            <button
              onClick={() => deleteTodo(todo.id)}
              style={{
                backgroundColor: '#e74c3c',
                color: 'white',
                border: 'none',
                borderRadius: '4px',
                padding: '4px 10px',
                cursor: 'pointer',
              }}
            >
              削除
            </button>
          </div>
        ))
      )}
    </div>
  );
}

export default TodoApp;
```
