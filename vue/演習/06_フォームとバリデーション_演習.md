## 演習6-1：v-model で各種フォーム要素を制御しよう（基本）

### 問題

以下のフォーム要素を `v-model` で制御するアンケートフォームを作成してください。

**要件:**
1. テキスト入力（名前）
2. セレクトボックス（職業）
3. ラジオボタン（性別）
4. チェックボックス（興味のある分野：複数選択可）
5. テキストエリア（自由記入欄）
6. 入力内容のリアルタイムプレビュー表示

<details>
<summary>ヒント</summary>

- チェックボックスの複数選択は `v-model` に配列を渡します
- ラジオボタンは同じ `v-model` で同じ変数を共有します
- `.trim` 修飾子で前後の空白を自動除去できます

```vue
<input v-model.trim="name" />
<select v-model="job">...</select>
<input type="radio" v-model="gender" value="male" />
<input type="checkbox" v-model="interests" value="frontend" />
```

</details>

<details>
<summary>解答例</summary>

```vue
<script setup>
import { ref, computed } from 'vue'

const name = ref('')
const job = ref('')
const gender = ref('')
const interests = ref([])
const comment = ref('')

const jobOptions = ['学生', 'エンジニア', 'デザイナー', 'マネージャー', 'その他']
const interestOptions = ['フロントエンド', 'バックエンド', 'モバイル', 'インフラ', 'AI/ML']

const isComplete = computed(() => {
  return name.value.trim() && job.value && gender.value && interests.value.length > 0
})
</script>

<template>
  <div style="display: flex; gap: 24px; max-width: 800px; margin: 0 auto; padding: 20px;">
    <!-- フォーム -->
    <div style="flex: 1;">
      <h2>アンケートフォーム</h2>

      <!-- テキスト入力 -->
      <div style="margin-bottom: 16px;">
        <label style="display: block; font-weight: bold; margin-bottom: 4px;">名前</label>
        <input v-model.trim="name" placeholder="お名前" style="width: 100%; padding: 8px;" />
      </div>

      <!-- セレクトボックス -->
      <div style="margin-bottom: 16px;">
        <label style="display: block; font-weight: bold; margin-bottom: 4px;">職業</label>
        <select v-model="job" style="width: 100%; padding: 8px;">
          <option value="" disabled>選択してください</option>
          <option v-for="opt in jobOptions" :key="opt" :value="opt">{{ opt }}</option>
        </select>
      </div>

      <!-- ラジオボタン -->
      <div style="margin-bottom: 16px;">
        <label style="display: block; font-weight: bold; margin-bottom: 4px;">性別</label>
        <label style="margin-right: 16px;">
          <input type="radio" v-model="gender" value="male" /> 男性
        </label>
        <label style="margin-right: 16px;">
          <input type="radio" v-model="gender" value="female" /> 女性
        </label>
        <label>
          <input type="radio" v-model="gender" value="other" /> その他
        </label>
      </div>

      <!-- チェックボックス（複数選択） -->
      <div style="margin-bottom: 16px;">
        <label style="display: block; font-weight: bold; margin-bottom: 4px;">
          興味のある分野（複数選択可）
        </label>
        <label v-for="opt in interestOptions" :key="opt" style="display: block; margin: 4px 0;">
          <input type="checkbox" v-model="interests" :value="opt" /> {{ opt }}
        </label>
      </div>

      <!-- テキストエリア -->
      <div style="margin-bottom: 16px;">
        <label style="display: block; font-weight: bold; margin-bottom: 4px;">自由記入</label>
        <textarea v-model="comment" rows="3" placeholder="ご意見・ご感想" style="width: 100%; padding: 8px;"></textarea>
      </div>
    </div>

    <!-- プレビュー -->
    <div style="flex: 1; padding: 20px; background: #f8f9fa; border-radius: 12px;">
      <h3>入力プレビュー</h3>
      <p><strong>名前:</strong> {{ name || '（未入力）' }}</p>
      <p><strong>職業:</strong> {{ job || '（未選択）' }}</p>
      <p><strong>性別:</strong> {{ gender === 'male' ? '男性' : gender === 'female' ? '女性' : gender === 'other' ? 'その他' : '（未選択）' }}</p>
      <p><strong>興味:</strong> {{ interests.length > 0 ? interests.join(', ') : '（未選択）' }}</p>
      <p><strong>コメント:</strong> {{ comment || '（なし）' }}</p>

      <div :style="{ marginTop: '16px', padding: '12px', borderRadius: '8px', background: isComplete ? '#e8f5e9' : '#fff3e0' }">
        {{ isComplete ? '全項目入力済み' : '必須項目を入力してください' }}
      </div>
    </div>
  </div>
</template>
```

**ポイント:**
- チェックボックスの `v-model` に配列を渡すと、チェックされた値が配列に追加/削除されます
- ラジオボタンは同じ `v-model` 変数を共有し、`value` の値がセットされます
- `.trim` 修飾子は前後の空白を自動除去します

</details>

## 演習6-2：リアルタイムバリデーションを実装しよう（基本）

### 問題

ユーザー登録フォームにリアルタイムバリデーションを実装してください。

**要件:**
1. 名前: 2文字以上必須
2. メール: メール形式の検証
3. パスワード: 8文字以上、大文字・数字を含む
4. パスワード確認: パスワードと一致
5. フィールドにフォーカスが当たった後（`@blur`）からバリデーション開始
6. すべてのバリデーションが通るまで送信ボタンを無効化

<details>
<summary>ヒント</summary>

- `@blur` イベントでフィールドが「触れられた」ことを記録します
- `computed` でバリデーション結果をリアクティブに算出します
- 全エラーがない場合のみ送信可能にします

</details>

<details>
<summary>解答例</summary>

```vue
<script setup>
import { reactive, computed } from 'vue'

const form = reactive({
  name: '',
  email: '',
  password: '',
  confirmPassword: ''
})

// どのフィールドがタッチされたかを記録
const touched = reactive({
  name: false,
  email: false,
  password: false,
  confirmPassword: false
})

// バリデーションルール（computed で自動再計算）
const errors = computed(() => ({
  name: !form.name.trim() ? '名前は必須です'
    : form.name.trim().length < 2 ? '2文字以上で入力してください' : '',
  email: !form.email.trim() ? 'メールは必須です'
    : !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(form.email) ? '正しいメール形式で入力してください' : '',
  password: !form.password ? 'パスワードは必須です'
    : form.password.length < 8 ? '8文字以上で入力してください'
    : !/[A-Z]/.test(form.password) ? '大文字を含めてください'
    : !/[0-9]/.test(form.password) ? '数字を含めてください' : '',
  confirmPassword: !form.confirmPassword ? 'パスワード確認は必須です'
    : form.confirmPassword !== form.password ? 'パスワードが一致しません' : ''
}))

// 全てのバリデーションが通っているか
const isValid = computed(() => {
  return Object.values(errors.value).every(e => e === '')
})

// パスワード強度
const passwordStrength = computed(() => {
  const p = form.password
  if (!p) return 0
  let s = 0
  if (p.length >= 8) s++
  if (/[A-Z]/.test(p)) s++
  if (/[0-9]/.test(p)) s++
  if (/[^A-Za-z0-9]/.test(p)) s++
  return s
})

const submitted = reactive({ value: false })

function handleSubmit() {
  // 全フィールドをタッチ済みにする
  Object.keys(touched).forEach(key => touched[key] = true)
  if (!isValid.value) return
  submitted.value = true
}
</script>

<template>
  <div style="max-width: 450px; margin: 0 auto; padding: 20px;">
    <h2>ユーザー登録</h2>

    <div v-if="submitted.value" style="text-align: center; padding: 40px; background: #e8f5e9; border-radius: 12px;">
      <h3 style="color: #43a047;">登録完了！</h3>
      <p>{{ form.name }} さん、ようこそ！</p>
    </div>

    <form v-else @submit.prevent="handleSubmit">
      <div v-for="field in [
        { key: 'name', label: '名前', type: 'text', placeholder: '山田太郎' },
        { key: 'email', label: 'メール', type: 'email', placeholder: 'mail@example.com' },
        { key: 'password', label: 'パスワード', type: 'password', placeholder: '8文字以上' },
        { key: 'confirmPassword', label: 'パスワード確認', type: 'password', placeholder: '再入力' },
      ]" :key="field.key" style="margin-bottom: 16px;">
        <label style="display: block; margin-bottom: 4px; font-weight: bold;">{{ field.label }}</label>
        <input
          v-model="form[field.key]"
          :type="field.type"
          :placeholder="field.placeholder"
          @blur="touched[field.key] = true"
          :style="{
            width: '100%', padding: '10px', borderRadius: '6px', outline: 'none',
            border: '2px solid ' + (touched[field.key] && errors[field.key] ? '#e53935' : touched[field.key] && !errors[field.key] ? '#43a047' : '#ddd')
          }"
        />
        <p v-if="touched[field.key] && errors[field.key]" style="color: #e53935; font-size: 13px; margin-top: 4px;">
          {{ errors[field.key] }}
        </p>

        <!-- パスワード強度バー -->
        <div v-if="field.key === 'password' && form.password" style="margin-top: 6px;">
          <div style="display: flex; gap: 4px;">
            <div v-for="i in 4" :key="i" :style="{
              flex: 1, height: '4px', borderRadius: '2px',
              background: i <= passwordStrength ? ['#e53935','#fb8c00','#43a047','#43a047'][passwordStrength - 1] : '#e0e0e0'
            }"></div>
          </div>
          <span style="font-size: 12px; color: #888;">
            {{ ['', '弱い', '普通', '強い', 'とても強い'][passwordStrength] }}
          </span>
        </div>
      </div>

      <button type="submit" :disabled="!isValid" :style="{
        width: '100%', padding: '12px', border: 'none', borderRadius: '8px',
        fontSize: '16px', cursor: isValid ? 'pointer' : 'not-allowed',
        background: isValid ? '#5c6bc0' : '#c5cae9', color: 'white'
      }">
        登録する
      </button>
    </form>
  </div>
</template>
```

**ポイント:**
- `@blur` でフィールドのタッチ状態を記録し、タッチ後のみエラーを表示することでUXを向上します
- `computed` でバリデーション結果をリアクティブに算出すると、入力のたびに自動で検証されます

</details>

## 演習6-3：動的フォームを作ろう（応用）

### 問題

項目を動的に追加・削除できるフォームを作成してください。

**要件:**
1. 「スキルを追加」ボタンでスキル入力行を動的に追加
2. 各行にスキル名とレベル（1-5）の入力欄
3. 各行に削除ボタン
4. 最大5つまでのバリデーション
5. 入力済みスキルの一覧表示

<details>
<summary>ヒント</summary>

- `reactive` の配列に `push` でオブジェクトを追加します
- `splice` で指定位置の要素を削除できます
- `v-for` のインデックスを使って各行を操作します

</details>

<details>
<summary>解答例</summary>

```vue
<script setup>
import { reactive, computed } from 'vue'

const skills = reactive([
  { name: '', level: 3 }
])

function addSkill() {
  if (skills.length >= 5) return
  skills.push({ name: '', level: 3 })
}

function removeSkill(index) {
  if (skills.length <= 1) return
  skills.splice(index, 1)
}

const validSkills = computed(() => {
  return skills.filter(s => s.name.trim())
})

const canAdd = computed(() => skills.length < 5)
const canRemove = computed(() => skills.length > 1)
</script>

<template>
  <div style="max-width: 600px; margin: 0 auto; padding: 20px;">
    <h2>スキル登録</h2>

    <div v-for="(skill, index) in skills" :key="index" style="display: flex; gap: 8px; align-items: center; margin-bottom: 12px; padding: 12px; background: #f8f9fa; border-radius: 8px;">
      <span style="color: #888; width: 24px;">{{ index + 1 }}.</span>

      <input
        v-model="skill.name"
        placeholder="スキル名"
        style="flex: 1; padding: 8px; border: 1px solid #ddd; border-radius: 6px;"
      />

      <div style="display: flex; align-items: center; gap: 4px;">
        <span style="font-size: 13px; color: #888;">Lv.</span>
        <input
          v-model.number="skill.level"
          type="range"
          min="1" max="5"
          style="width: 80px;"
        />
        <span style="width: 20px; text-align: center; font-weight: bold;">{{ skill.level }}</span>
      </div>

      <button
        @click="removeSkill(index)"
        :disabled="!canRemove"
        :style="{ padding: '6px 10px', border: 'none', borderRadius: '6px', cursor: canRemove ? 'pointer' : 'not-allowed', background: canRemove ? '#ffebee' : '#eee', color: canRemove ? '#e53935' : '#999' }"
      >
        削除
      </button>
    </div>

    <button
      @click="addSkill"
      :disabled="!canAdd"
      :style="{ padding: '8px 16px', border: '2px dashed', borderColor: canAdd ? '#5c6bc0' : '#ddd', borderRadius: '8px', background: 'transparent', color: canAdd ? '#5c6bc0' : '#999', cursor: canAdd ? 'pointer' : 'not-allowed', width: '100%' }"
    >
      + スキルを追加（{{ skills.length }}/5）
    </button>

    <!-- プレビュー -->
    <div v-if="validSkills.length > 0" style="margin-top: 24px; padding: 16px; background: #e8eaf6; border-radius: 8px;">
      <h3 style="margin-bottom: 12px;">登録スキル</h3>
      <div v-for="skill in validSkills" :key="skill.name" style="display: flex; align-items: center; gap: 8px; margin-bottom: 4px;">
        <strong>{{ skill.name }}</strong>
        <span v-for="i in 5" :key="i" :style="{ color: i <= skill.level ? '#f6ad55' : '#ddd' }">★</span>
      </div>
    </div>
  </div>
</template>
```

**ポイント:**
- `reactive` の配列に `push` / `splice` すると、Vue が変更を検知して自動で画面が更新されます
- 動的フォームは SaaS の設定画面やアンケート作成ツールなど、実務でよく使われるパターンです

</details>

## 演習6-4：カスタム v-model を持つフォームコンポーネント（チャレンジ）

### 問題

以下の2つのカスタムフォームコンポーネントを作成し、親コンポーネントで `v-model` を使って利用してください。

1. **`RatingInput`** — 星5段階の評価入力（`v-model` で値を双方向バインド）
2. **`TagInput`** — タグの追加・削除（`v-model` で配列を双方向バインド）

<details>
<summary>ヒント</summary>

- カスタム `v-model` は `modelValue` props と `update:modelValue` emit で実現します
- `defineProps` と `defineEmits` を使います

</details>

<details>
<summary>解答例</summary>

```vue
<!-- 全体を1ファイルで表現（SFC の場合は分割） -->
<script setup>
import { ref, computed } from 'vue'

// ========================================
// composable として RatingInput / TagInput のロジックを定義
// （SFCなら別ファイルのコンポーネントにする）
// ========================================

// 親コンポーネントでの使用
const rating = ref(3)
const tags = ref(['Vue.js', 'JavaScript'])
</script>

<template>
  <div style="max-width: 500px; margin: 0 auto; padding: 20px;">
    <h2>カスタムフォームコンポーネント</h2>

    <!-- RatingInput（星の評価） -->
    <div style="margin-bottom: 24px; padding: 16px; background: #f8f9fa; border-radius: 12px;">
      <h3>評価</h3>
      <div>
        <span
          v-for="n in 5"
          :key="n"
          @click="rating = n"
          :style="{
            fontSize: '32px', cursor: 'pointer',
            color: n <= rating ? '#f6ad55' : '#ddd',
            transition: 'transform 0.1s'
          }"
          @mouseover="$event.target.style.transform = 'scale(1.2)'"
          @mouseleave="$event.target.style.transform = 'scale(1)'"
        >
          {{ n <= rating ? '★' : '☆' }}
        </span>
      </div>
      <p>選択中: {{ rating }} / 5</p>
    </div>

    <!-- TagInput（タグ入力） -->
    <div style="padding: 16px; background: #f8f9fa; border-radius: 12px;">
      <h3>タグ</h3>
      <div style="display: flex; flex-wrap: wrap; gap: 8px; margin-bottom: 8px;">
        <span
          v-for="(tag, index) in tags"
          :key="tag"
          style="display: inline-flex; align-items: center; gap: 4px; padding: 4px 10px; background: #e8eaf6; border-radius: 16px; font-size: 14px;"
        >
          {{ tag }}
          <button
            @click="tags.splice(index, 1)"
            style="border: none; background: none; cursor: pointer; color: #999; font-size: 16px; padding: 0 2px;"
          >
            ×
          </button>
        </span>
      </div>

      <input
        placeholder="Enter でタグを追加"
        @keyup.enter="(e) => {
          const val = e.target.value.trim()
          if (val && !tags.includes(val)) {
            tags.push(val)
            e.target.value = ''
          }
        }"
        style="width: 100%; padding: 8px; border: 1px solid #ddd; border-radius: 6px;"
      />
      <p style="font-size: 13px; color: #888; margin-top: 4px;">
        {{ tags.length }} 個のタグ
      </p>
    </div>
  </div>
</template>
```

**SFC での正式な実装（RatingInput.vue）:**
```vue
<script setup>
defineProps({ modelValue: { type: Number, default: 0 } })
const emit = defineEmits(['update:modelValue'])
</script>

<template>
  <span
    v-for="n in 5" :key="n"
    @click="emit('update:modelValue', n)"
    style="font-size: 32px; cursor: pointer;"
    :style="{ color: n <= modelValue ? '#f6ad55' : '#ddd' }"
  >
    {{ n <= modelValue ? '★' : '☆' }}
  </span>
</template>
```

**使い方:**
```vue
<RatingInput v-model="rating" />
<!-- ↓ 展開すると以下と同じ -->
<RatingInput :modelValue="rating" @update:modelValue="val => rating = val" />
```

</details>
