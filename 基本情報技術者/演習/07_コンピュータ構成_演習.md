# 第7章：コンピュータ構成 演習問題

## 基本問題

### 問題1：5大装置
コンピュータの5大装置を全て挙げてください。

<details>
<summary>解答</summary>

1. **制御装置** — 命令の解読と各装置の制御
2. **演算装置（ALU）** — 四則演算、論理演算
3. **記憶装置** — データとプログラムの格納
4. **入力装置** — 外部からデータを取り込む
5. **出力装置** — 処理結果を外部に出す
</details>

---

### 問題2：レジスタの役割
プログラムカウンタ（PC）と命令レジスタ（IR）の違いを説明してください。

<details>
<summary>解答</summary>

- **プログラムカウンタ（PC）：** 次に実行する命令が格納されている**メモリアドレス**を保持する
- **命令レジスタ（IR）：** 現在実行中の**命令そのもの**を保持する

PCは「どこにある命令か」、IRは「何の命令か」を保持します。
</details>

### 問題3：キャッシュメモリの実効アクセス時間
キャッシュメモリのアクセス時間が 10ns、主記憶のアクセス時間が 80ns、ヒット率が 90% のとき、実効アクセス時間を求めてください。

<details>
<summary>解答</summary>

```
実効アクセス時間 = ヒット率 × キャッシュアクセス時間 + (1 - ヒット率) × 主記憶アクセス時間
               = 0.9 × 10 + 0.1 × 80
               = 9 + 8
               = 17ns
```
答え：**17ns**
</details>

### 問題4：四択問題
SRAMとDRAMの説明として正しいものはどれですか。

ア. SRAMはリフレッシュが必要で、DRAMはリフレッシュ不要である
イ. SRAMはキャッシュメモリに使われ、DRAMは主記憶に使われる
ウ. SRAMはDRAMより低速で、大容量に適している
エ. SRAMはコンデンサで、DRAMはフリップフロップで構成される

<details>
<summary>解答</summary>

答え：**イ. SRAMはキャッシュメモリに使われ、DRAMは主記憶に使われる**

- SRAMはフリップフロップ構成、高速、リフレッシュ不要、高価 → キャッシュ
- DRAMはコンデンサ構成、低速、リフレッシュ必要、安価 → 主記憶
</details>

### 問題5：アドレッシングモード
指標アドレッシング（インデックスアドレッシング）において、命令のアドレス部が 300、インデックスレジスタの値が 20 のとき、実効アドレスを求めてください。

<details>
<summary>解答</summary>

```
実効アドレス = 命令のアドレス部 + インデックスレジスタの値
            = 300 + 20
            = 320
```
答え：**320**
</details>

## 応用問題

### 問題6：パイプライン処理時間
4ステージのパイプラインで、1ステージ 15ns の場合、50命令を実行するのにかかる時間を求めてください。パイプラインなしの場合と比較してください。

<details>
<summary>解答</summary>

```
パイプラインあり:
(k + n - 1) × t = (4 + 50 - 1) × 15 = 53 × 15 = 795ns

パイプラインなし:
k × n × t = 4 × 50 × 15 = 3000ns

速度向上比: 3000 / 795 ≒ 3.77倍
```
答え：パイプラインあり **795ns**、パイプラインなし **3000ns**、速度向上比 **約3.8倍**
</details>

### 問題7：MIPS値の計算
クロック周波数が 3GHz、CPI（1命令あたりの平均クロック数）が 6 のプロセッサのMIPS値を求めてください。

<details>
<summary>解答</summary>

```
MIPS = クロック周波数(Hz) / CPI / 10⁶
     = (3 × 10⁹) / 6 / 10⁶
     = 500 MIPS
```
答え：**500 MIPS**（1秒間に5億命令を実行できる）
</details>

### 問題8：間接アドレッシング
間接アドレッシングにおいて、命令のアドレス部が 400、主記憶のアドレス 400 の内容が 700、主記憶のアドレス 700 の内容が 150 のとき、取得されるデータの値を答えてください。

<details>
<summary>解答</summary>

```
1. 命令のアドレス部 = 400
2. 主記憶のアドレス400を参照 → 内容 700（これが実効アドレス）
3. 主記憶のアドレス700を参照 → 内容 150（これが取得データ）
```
答え：取得されるデータは **150**

間接アドレッシングでは2回メモリを参照します。1回目で実効アドレスを取得し、2回目で実データを取得します。
</details>

### 問題9：四択問題
パイプライン処理において、前の命令の演算結果が次の命令で必要とされるために発生するハザードはどれですか。

ア. 構造ハザード　　イ. データハザード　　ウ. 制御ハザード　　エ. タイミングハザード

<details>
<summary>解答</summary>

答え：**イ. データハザード**

- 構造ハザード：ハードウェア資源の競合
- データハザード：前の命令の結果が必要
- 制御ハザード（分岐ハザード）：分岐命令による行き先の不確定
</details>

### 問題10：キャッシュメモリのヒット率
キャッシュアクセス時間が 5ns、主記憶アクセス時間が 50ns のとき、実効アクセス時間を 10ns 以下にするために必要な最低ヒット率を求めてください。

<details>
<summary>ヒント</summary>
実効アクセス時間 = h × 5 + (1-h) × 50 ≦ 10 を解きます。
</details>

<details>
<summary>解答</summary>

```
h × 5 + (1 - h) × 50 ≦ 10
5h + 50 - 50h ≦ 10
-45h ≦ -40
h ≧ 40/45
h ≧ 0.889...
```
答え：ヒット率 **約 88.9% 以上**（89%以上あれば実効アクセス時間は10ns以下）
</details>

## チャレンジ問題

### 問題11：アドレスバスとメモリ空間
アドレスバスが 24ビットのコンピュータが直接アドレッシングできるメモリ空間の最大容量を求めてください。

<details>
<summary>解答</summary>

```
アドレス空間 = 2²⁴ = 16,777,216 バイト = 16MB
```
答え：**16MB**

2¹⁰ = 1KB, 2²⁰ = 1MB なので、2²⁴ = 2²⁰ × 2⁴ = 1MB × 16 = 16MB です。
</details>

### 問題12：DMA方式
DMA（Direct Memory Access）方式の利点と、プログラム制御方式との違いを説明してください。

<details>
<summary>解答</summary>

**DMA方式の利点：**
DMAコントローラが主記憶と入出力装置の間で直接データ転送を行うため、CPUを介さずにデータ転送ができます。その間CPUは別の処理を実行できるため、CPU負荷が低い。

**プログラム制御方式との違い：**
プログラム制御方式ではCPUが直接データ転送を制御するため、転送中はCPUが専有されます。DMA方式ではCPUはデータ転送の開始指示のみ行い、実際の転送はDMAコントローラが担当します。
</details>

### 問題13：総合計算問題
あるCPUのクロック周波数が 2.4GHz、5ステージのパイプライン処理を行い、1ステージが1クロックサイクルで完了するとします。このCPUで1000命令を実行するのにかかる時間を求めてください。

<details>
<summary>解答</summary>

```
1クロックサイクル時間 = 1 / (2.4 × 10⁹) ≒ 0.417ns

パイプライン処理時間 = (k + n - 1) × t
                    = (5 + 1000 - 1) × 0.417ns
                    = 1004 × 0.417ns
                    ≒ 418.7ns
```
答え：約 **418.7ns（≒ 0.42μs）**

パイプラインなしの場合：5 × 1000 × 0.417 = 2085ns → 約5倍の高速化
</details>
