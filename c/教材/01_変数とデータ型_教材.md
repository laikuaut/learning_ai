## 1.1 なぜC言語を学ぶのか

C言語は1972年にデニス・リッチーによって開発された、歴史あるプログラミング言語です。
「古い言語」と思われがちですが、現在でも以下の分野で広く使われています。

| 分野 | 具体例 |
|---|---|
| OS開発 | Linux カーネル、Windows カーネル |
| 組み込み（embedded）開発 | 家電、自動車、医療機器 |
| ゲームエンジン | 高速な処理が必要な部分 |
| データベース | PostgreSQL、SQLite |
| 言語処理系 | Python、Ruby の実装（CPython、CRuby） |

C言語を学ぶメリットは以下のとおりです。

1. **メモリの仕組みを直接理解できる** — ポインタやメモリ管理を通じて、コンピュータの動作原理がわかります
2. **実行速度が速い** — 機械語に近いレベルで最適化できます
3. **他の言語の理解が深まる** — C++、Java、Go など多くの言語がC言語の影響を受けています
4. **長期間のキャリアに役立つ** — 組み込み・OS・システムプログラミングは今後も需要があります

## 1.2 C言語プログラムの基本構造

最初のプログラムとして「Hello, World!」を見てみましょう。

```c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

### 各行の解説

```
#include <stdio.h>    ← 標準入出力ライブラリを読み込む（ヘッダファイル）
                         printf や scanf を使うために必要

int main(void) {      ← プログラムの開始地点（エントリーポイント）
                         int は「整数を返す」という意味
                         void は「引数なし」という意味

    printf("...\n");  ← 画面に文字を出力する関数
                         \n は改行（newline）を意味する

    return 0;         ← プログラムの終了コード（0 = 正常終了）
}
```

### コンパイル（compile）と実行の流れ

C言語は**コンパイル言語**です。PythonやJavaScriptと違い、実行前にソースコードを機械語に変換する必要があります。

```
┌─────────────┐     コンパイル      ┌─────────────┐      実行       ┌─────────────┐
│  hello.c    │  ──────────────→  │   hello     │  ──────────→  │  画面出力   │
│（ソースコード）│   gcc コマンド     │（実行ファイル）│   ./hello      │ Hello, World│
└─────────────┘                   └─────────────┘               └─────────────┘
```

ターミナルでの操作は以下のとおりです。

```bash
# コンパイル（-o は出力ファイル名を指定）
gcc -o hello hello.c

# 実行
./hello
```

実行結果：
```
Hello, World!
```

> **よくある間違い：** `gcc hello.c` だけで実行すると `a.out` という名前の実行ファイルが生成されます。`-o` オプションでわかりやすい名前をつける習慣をつけましょう。

### ポイントまとめ
- `#include <stdio.h>` は入出力に必須のおまじない
- `main` 関数がプログラムの開始地点
- `return 0;` で正常終了を示す
- `gcc -o 出力名 ソースファイル` でコンパイルする

## 1.3 変数の宣言と初期化

変数（variable）とは、**値を入れておく箱**のようなものです。C言語では変数を使う前に必ず**型（type）**を宣言する必要があります。

### 基本の書き方

```c
型名 変数名;           // 宣言のみ
型名 変数名 = 初期値;   // 宣言と初期化を同時に行う
```

```c
#include <stdio.h>

int main(void) {
    int age = 25;          // 整数型の変数
    float height = 170.5;  // 小数型の変数
    char grade = 'A';      // 文字型の変数

    printf("年齢: %d歳\n", age);
    printf("身長: %.1fcm\n", height);
    printf("成績: %c\n", grade);

    return 0;
}
```

実行結果：
```
年齢: 25歳
身長: 170.5cm
成績: A
```

### 変数の命名ルール

| ルール | OK例 | NG例 |
|---|---|---|
| 英字・数字・アンダースコアのみ | `my_age`, `score1` | `my-age`, `score!` |
| 先頭は数字不可 | `age1` | `1age` |
| 予約語（keyword）は使えない | `my_int` | `int`, `return` |
| 大文字と小文字は区別される | `Age` と `age` は別物 | — |

> **現場のベストプラクティス：** 変数名は`snake_case`（小文字＋アンダースコア）で書くのがC言語の慣習です。`maxScore` のようなキャメルケースはC++やJavaで主に使います。

### ポイントまとめ
- 変数は「型 変数名 = 値;」で宣言する
- 使う前に必ず宣言が必要
- 初期化しないと不定値（ゴミ値）が入るので注意

## 1.4 基本データ型

C言語の基本データ型は以下のとおりです。

| 型名 | 用途 | サイズ（一般的） | 範囲（32bit環境） | 書式指定子 |
|---|---|---|---|---|
| `char` | 文字1文字 | 1バイト | -128 〜 127 | `%c` |
| `int` | 整数 | 4バイト | 約-21億 〜 約21億 | `%d` |
| `float` | 単精度小数 | 4バイト | 有効桁数 約6〜7桁 | `%f` |
| `double` | 倍精度小数 | 8バイト | 有効桁数 約15〜16桁 | `%lf`（scanf） / `%f`（printf） |

```c
#include <stdio.h>

int main(void) {
    char   letter = 'Z';
    int    count  = 42;
    float  pi_f   = 3.14159f;   // f をつけると float リテラル
    double pi_d   = 3.141592653589793;

    printf("char:   %c (ASCII値: %d)\n", letter, letter);
    printf("int:    %d\n", count);
    printf("float:  %.5f\n", pi_f);
    printf("double: %.15f\n", pi_d);

    return 0;
}
```

実行結果：
```
char:   Z (ASCII値: 90)
int:    42
float:  3.14159
double: 3.141592653589793
```

> **よくある間違い：** `float` と `double` の精度の違いに注意してください。金額計算などで `float` を使うと誤差が出ることがあります。通常は `double` を使うのが安全です。

### ポイントまとめ
- 整数には `int`、小数には `double` を基本的に使う
- `char` は文字1文字を格納し、内部的にはASCIIコードの整数値
- `float` は精度が低いため、特別な理由がなければ `double` を使う

## 1.5 sizeof演算子

`sizeof` 演算子（operator）は、型や変数のメモリサイズ（バイト数）を調べるために使います。

```c
#include <stdio.h>

int main(void) {
    printf("char   のサイズ: %zu バイト\n", sizeof(char));
    printf("int    のサイズ: %zu バイト\n", sizeof(int));
    printf("float  のサイズ: %zu バイト\n", sizeof(float));
    printf("double のサイズ: %zu バイト\n", sizeof(double));

    int number = 100;
    printf("変数numberのサイズ: %zu バイト\n", sizeof(number));

    return 0;
}
```

実行結果（環境により異なります）：
```
char   のサイズ: 1 バイト
int    のサイズ: 4 バイト
float  のサイズ: 4 バイト
double のサイズ: 8 バイト
変数numberのサイズ: 4 バイト
```

### メモリ上の変数のイメージ

```
メモリ（1マス = 1バイト）
アドレス: 0x100  0x101  0x102  0x103  0x104  0x105  0x106  ...
        ┌──────┐
char c  │  'A' │                                   ← 1バイト
        └──────┘
        ┌──────┬──────┬──────┬──────┐
int n   │      │      │      │  42  │              ← 4バイト
        └──────┴──────┴──────┴──────┘
        ┌──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┐
double d│      │      │      │      │      │      │      │ 3.14 │ ← 8バイト
        └──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

### ポイントまとめ
- `sizeof` は型や変数のバイト数を返す
- 書式指定子は `%zu` を使う
- サイズは環境（OS・コンパイラ）によって異なる場合がある

## 1.6 printf と scanf の基本

### printf（出力）

`printf` 関数は、画面に文字を出力するための関数です。**書式指定子（format specifier）**を使って変数の値を埋め込めます。

```c
#include <stdio.h>

int main(void) {
    int age = 20;
    double height = 172.5;
    char initial = 'T';

    // 基本的な出力
    printf("私は%d歳です。\n", age);
    printf("身長は%.1fcmです。\n", height);
    printf("イニシャルは%cです。\n", initial);

    // 複数の値を一度に出力
    printf("%cさんは%d歳、身長%.1fcmです。\n", initial, age, height);

    // 桁揃え
    printf("[%10d]\n", age);    // 右揃え（幅10）
    printf("[%-10d]\n", age);   // 左揃え（幅10）
    printf("[%05d]\n", age);    // ゼロ埋め（幅5）

    return 0;
}
```

実行結果：
```
私は20歳です。
身長は172.5cmです。
イニシャルはTです。
Tさんは20歳、身長172.5cmです。
[        20]
[20        ]
[00020]
```

### 主な書式指定子一覧

| 書式指定子 | 意味 | 例 |
|---|---|---|
| `%d` | int（10進整数） | `42` |
| `%f` | float / double | `3.140000` |
| `%.2f` | 小数点以下2桁 | `3.14` |
| `%c` | char（1文字） | `A` |
| `%s` | 文字列 | `hello` |
| `%x` | 16進数 | `2a` |
| `%o` | 8進数 | `52` |
| `%%` | %そのもの | `%` |

### scanf（入力）

`scanf` 関数は、キーボードからの入力を受け取るための関数です。変数のアドレスを渡すため、変数名の前に `&`（アンパサンド）をつけます。

```c
#include <stdio.h>

int main(void) {
    int age;
    double height;

    printf("年齢を入力してください: ");
    scanf("%d", &age);

    printf("身長を入力してください(cm): ");
    scanf("%lf", &height);   // double の読み取りは %lf

    printf("あなたは%d歳、身長%.1fcmですね。\n", age, height);

    return 0;
}
```

実行例：
```
年齢を入力してください: 25
身長を入力してください(cm): 170.5
あなたは25歳、身長170.5cmですね。
```

> **よくある間違い：** `scanf` で `&` を忘れるとプログラムがクラッシュします。また、`double` の読み取りには `%lf` を使います（`printf` では `%f` でOKですが、`scanf` では `%lf` が必要です）。

### ポイントまとめ
- `printf` は書式指定子で変数の値を出力する
- `scanf` は `&変数名` でアドレスを渡して入力を受け取る
- `double` の入力には `%lf`、出力には `%f` を使う

## 1.7 算術演算子

C言語で使える基本的な算術演算子（arithmetic operator）は以下のとおりです。

| 演算子 | 意味 | 例 | 結果 |
|---|---|---|---|
| `+` | 加算 | `5 + 3` | `8` |
| `-` | 減算 | `5 - 3` | `2` |
| `*` | 乗算 | `5 * 3` | `15` |
| `/` | 除算 | `5 / 3` | `1`（整数同士） |
| `%` | 剰余 | `5 % 3` | `2` |

```c
#include <stdio.h>

int main(void) {
    int a = 10, b = 3;

    printf("%d + %d = %d\n", a, b, a + b);
    printf("%d - %d = %d\n", a, b, a - b);
    printf("%d * %d = %d\n", a, b, a * b);
    printf("%d / %d = %d\n", a, b, a / b);   // 整数除算 → 小数切り捨て
    printf("%d %% %d = %d\n", a, b, a % b);   // %% で % を表示

    // 小数を含む除算
    double x = 10.0, y = 3.0;
    printf("%.1f / %.1f = %.4f\n", x, y, x / y);

    return 0;
}
```

実行結果：
```
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3
10 % 3 = 1
10.0 / 3.0 = 3.3333
```

> **よくある間違い：** `10 / 3` は `3.333...` ではなく `3` になります。整数同士の除算は**小数部分が切り捨て**られます。小数の結果が欲しい場合は、少なくとも一方を `double` にしてください。

### 複合代入演算子

```c
int x = 10;
x += 5;   // x = x + 5;  → x は 15
x -= 3;   // x = x - 3;  → x は 12
x *= 2;   // x = x * 2;  → x は 24
x /= 4;   // x = x / 4;  → x は 6
x %= 4;   // x = x % 4;  → x は 2
```

### インクリメント・デクリメント

```c
int count = 5;
count++;   // count = count + 1; → 6
count--;   // count = count - 1; → 5
```

### ポイントまとめ
- 整数同士の除算は小数部分が切り捨てられる
- `%` は剰余（余り）を求める演算子
- `+=`, `-=` などの複合代入演算子で短く書ける
- `++`, `--` でインクリメント・デクリメントができる

## 1.8 型変換（キャスト）

異なる型同士で計算するとき、**暗黙の型変換（implicit conversion）**と**明示的な型変換（explicit cast）**があります。

### 暗黙の型変換

```c
#include <stdio.h>

int main(void) {
    int a = 5;
    double b = 2.0;

    // int と double の計算 → 自動的に double になる
    double result = a + b;
    printf("5 + 2.0 = %f\n", result);   // 7.000000

    // int 同士の除算 → 結果も int
    int c = 7 / 2;
    printf("7 / 2 = %d\n", c);          // 3（切り捨て）

    return 0;
}
```

### 明示的な型変換（キャスト）

```c
#include <stdio.h>

int main(void) {
    int a = 7, b = 2;

    // キャストなし → 整数除算
    printf("キャストなし: %d / %d = %d\n", a, b, a / b);

    // キャストあり → 小数の結果が得られる
    double result = (double)a / b;
    printf("キャストあり: %d / %d = %.2f\n", a, b, result);

    // double → int（小数部分が失われる）
    double pi = 3.14159;
    int pi_int = (int)pi;
    printf("double → int: %.5f → %d\n", pi, pi_int);

    return 0;
}
```

実行結果：
```
キャストなし: 7 / 2 = 3
キャストあり: 7 / 2 = 3.50
double → int: 3.14159 → 3
```

### ポイントまとめ
- 型が異なる計算では、小さい型が大きい型に自動変換される
- `(型名)値` で明示的にキャスト（cast）できる
- `double` → `int` の変換では小数部分が失われる

## 1.9 定数

プログラム中で**変化しない値**には定数（constant）を使います。

### const による定数

```c
#include <stdio.h>

int main(void) {
    const double TAX_RATE = 0.10;   // 消費税率（変更不可）
    int price = 1000;

    double total = price * (1 + TAX_RATE);
    printf("税込価格: %.0f円\n", total);

    // TAX_RATE = 0.08;  ← コンパイルエラー！ const は変更できない

    return 0;
}
```

### #define によるマクロ定数

```c
#include <stdio.h>

#define PI 3.14159265358979
#define MAX_STUDENTS 50

int main(void) {
    double radius = 5.0;
    double area = PI * radius * radius;
    printf("半径%.1fの円の面積: %.2f\n", radius, area);
    printf("最大生徒数: %d\n", MAX_STUDENTS);

    return 0;
}
```

実行結果：
```
半径5.0の円の面積: 78.54
最大生徒数: 50
```

### const と #define の違い

| 特徴 | `const` | `#define` |
|---|---|---|
| 型がある | はい | いいえ（テキスト置換） |
| デバッグしやすい | はい | いいえ |
| スコープ | ブロック内 | ファイル全体 |
| 現場での推奨 | 基本はこちら | 配列サイズ等に使う |

### ポイントまとめ
- 変更しない値には `const` をつける
- `#define` はコンパイル前のテキスト置換（マクロ）
- 定数名は**大文字＋アンダースコア**（`MAX_SIZE` など）が慣習

## 1.10 総合例：BMI計算プログラム

ここまで学んだ内容を組み合わせた実用的なプログラムです。

```c
#include <stdio.h>

#define UNDERWEIGHT 18.5
#define NORMAL      25.0

int main(void) {
    double height_cm, weight;

    printf("=== BMI計算プログラム ===\n");
    printf("身長(cm)を入力してください: ");
    scanf("%lf", &height_cm);

    printf("体重(kg)を入力してください: ");
    scanf("%lf", &weight);

    // cm を m に変換
    double height_m = height_cm / 100.0;

    // BMI = 体重(kg) / 身長(m)^2
    double bmi = weight / (height_m * height_m);

    printf("\n--- 結果 ---\n");
    printf("身長: %.1f cm\n", height_cm);
    printf("体重: %.1f kg\n", weight);
    printf("BMI:  %.1f\n", bmi);

    // 判定
    if (bmi < UNDERWEIGHT) {
        printf("判定: 低体重\n");
    } else if (bmi < NORMAL) {
        printf("判定: 普通体重\n");
    } else {
        printf("判定: 肥満\n");
    }

    return 0;
}
```

実行例：
```
=== BMI計算プログラム ===
身長(cm)を入力してください: 170
体重(kg)を入力してください: 65

--- 結果 ---
身長: 170.0 cm
体重: 65.0 kg
BMI:  22.5
判定: 普通体重
```
