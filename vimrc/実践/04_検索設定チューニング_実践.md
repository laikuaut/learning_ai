# 実践課題04：検索設定チューニング ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第4章（検索と置換の設定）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 検索に関する設定（`hlsearch`, `incsearch`, `ignorecase`, `smartcase`）を組み合わせ、快適な検索環境を構築できるようになる

---

## 完成イメージ

```
┌──────────────────────────────────────────────────────────┐
│  検索環境の目標                                           │
├──────────────────────────────────────────────────────────┤
│                                                           │
│  1. /hello と入力中にリアルタイムで候補がハイライト       │
│  2. Enter で確定後、全マッチ箇所がハイライト              │
│  3. Esc または特定キーでハイライトを消せる                │
│  4. /hello → hello, Hello, HELLO すべてにマッチ          │
│  5. /Hello → Hello のみにマッチ（大文字を含むと厳密）    │
│  6. :grep で外部 grep を使ってプロジェクト全体を検索      │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

---

## 課題の要件

1. インクリメンタルサーチ（`incsearch`）を有効にする
2. 検索結果のハイライト（`hlsearch`）を有効にする
3. 大文字小文字を無視する検索（`ignorecase`）を有効にする
4. スマートケース（`smartcase`）を有効にする
5. 検索ハイライトを消すためのキーマッピングを設定する
6. `wrapscan` でファイル末尾から先頭へ検索が回り込むようにする
7. `grepprg` を設定して外部 grep ツールを活用する準備をする

---

## ステップガイド

<details>
<summary>ステップ1：検索の基本4設定を記述する</summary>

```vim
set hlsearch        " 検索結果をハイライト表示
set incsearch       " インクリメンタルサーチ（入力中にリアルタイムで検索）
set ignorecase      " 検索で大文字・小文字を区別しない
set smartcase       " 大文字が含まれる場合のみ大文字・小文字を区別する
```

`ignorecase` と `smartcase` はセットで使うのが一般的です。

</details>

<details>
<summary>ステップ2：ハイライト解除のマッピングを作る</summary>

検索後にハイライトが残って邪魔になることがあります。Esc キーの2回押しや特定キーでハイライトを消せるようにしましょう。

```vim
" Esc 2回押しで検索ハイライトを消す
nnoremap <Esc><Esc> :nohlsearch<CR>
```

</details>

<details>
<summary>ステップ3：wrapscan と grepprg を設定する</summary>

```vim
" ファイル末尾まで検索したら先頭に戻る
set wrapscan

" 外部 grep コマンドの設定（ripgrep が使える場合）
if executable('rg')
  set grepprg=rg\ --vimgrep\ --smart-case
  set grepformat=%f:%l:%c:%m
endif
```

</details>

<details>
<summary>ステップ4：動作を確認する</summary>

以下の手順で確認しましょう。

1. 適当なファイルを開き、`/function` で検索してリアルタイムハイライトを確認する
2. Enter で確定後、すべてのマッチ箇所がハイライトされることを確認する
3. `Esc` を2回押してハイライトが消えることを確認する
4. `/hello` と `/Hello` で検索結果が変わることを確認する（`smartcase` の動作）

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ==================================================
" 検索設定
" ==================================================

" 検索結果をハイライト表示する
set hlsearch

" インクリメンタルサーチ（入力中にリアルタイム検索）
set incsearch

" 大文字・小文字を区別しない
set ignorecase

" 大文字が含まれる場合のみ区別する
set smartcase

" ファイル末尾まで検索したら先頭に戻る
set wrapscan

" ==================================================
" 検索用キーマッピング
" ==================================================

" Esc 2回押しで検索ハイライトを消す
nnoremap <Esc><Esc> :nohlsearch<CR>
```

</details>

<details>
<summary>解答例（改良版 ─ 外部ツール連携と便利マッピング）</summary>

```vim
" ==================================================
" 検索設定（改良版）
" ==================================================

" --- 基本検索設定 ---
set hlsearch
set incsearch
set ignorecase
set smartcase
set wrapscan

" --- 外部 grep ツール連携 ---
" ripgrep が使える場合は grepprg を変更する
if executable('rg')
  set grepprg=rg\ --vimgrep\ --smart-case
  set grepformat=%f:%l:%c:%m
elseif executable('ag')
  " ag（The Silver Searcher）が使える場合
  set grepprg=ag\ --vimgrep
  set grepformat=%f:%l:%c:%m
endif

" ==================================================
" 検索用キーマッピング
" ==================================================

" Esc 2回押しで検索ハイライトを消す
nnoremap <Esc><Esc> :nohlsearch<CR>

" カーソル下の単語をハイライトする（移動はしない）
nnoremap <Leader>* *N

" 選択範囲を検索する（ビジュアルモード）
xnoremap * "zy/\V<C-r>z<CR>

" :grep の結果を quickfix で表示するショートカット
nnoremap <Leader>g :grep!<Space>

" quickfix リストの移動
nnoremap ]q :cnext<CR>
nnoremap [q :cprevious<CR>

" --- 置換の便利設定 ---
" カーソル下の単語を置換するテンプレートを挿入する
nnoremap <Leader>s :%s/\<<C-r><C-w>\>//gc<Left><Left><Left>
```

**初心者向けとの違い:**
- `ripgrep` / `ag` の自動検出でプロジェクト全体の高速検索が可能
- `<Leader>*` でカーソル移動せずにハイライトだけ実行
- ビジュアルモードでの選択範囲検索に対応
- quickfix リストのナビゲーション用マッピングで grep 結果を効率的に閲覧
- `<Leader>s` でカーソル下の単語を一括置換するテンプレートを即座に呼び出し

</details>
