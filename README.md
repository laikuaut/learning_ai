# プログラミング学習教材

初心者〜中級者向けのプログラミング学習教材集です。全15分野・351ファイルで構成されています。

## 特徴

- **教材**: 概念説明 → コード例 → 解説の段階的構成
- **演習**: 基本 → 応用 → チャレンジの3段階。ヒント・解答付き
- **サンプル**: コピペで即動作する完成コード

## ファイル構成

各フォルダは以下の3つのサブフォルダで構成されています。

```
<トピック>/
├── 教材/      # XX_トピック名_教材.md（学習テキスト）
├── 演習/      # XX_トピック名_演習.md（問題集）
└── サンプル/  # XX_サンプル名.拡張子（動作するコード）
```

### 統計

| 種別 | ファイル数 |
|---|:---:|
| 教材 | 125 |
| 演習 | 125 |
| サンプル | 101 |
| **合計** | **351** |

## 使い方

1. 学びたい分野のフォルダを開く
2. `教材/` の01番から順に読み進める
3. 各章を読んだ後、対応する `演習/` で理解度を確認
4. `サンプル/` のコードを実行して動きを体感

### サンプルの実行方法

| 形式 | 実行方法 |
|---|---|
| `.py` | `python ファイル名.py` |
| `.ts` | `tsc ファイル名.ts && node ファイル名.js` |
| `.html` | ブラウザで直接開く |
| `.sh` | `chmod +x ファイル名.sh && ./ファイル名.sh` |
| `.tsx` | Next.jsプロジェクト内に配置して `npm run dev` |
