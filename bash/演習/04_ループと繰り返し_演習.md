## 基本問題

### 問題1: forループの基本
1〜10の数字を1行に1つずつ表示するスクリプトを、ブレース展開を使って書いてください。

期待される出力：
```
1
2
3
...
10
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
# ブレース展開で1〜10を生成し、forループで表示
for i in {1..10}; do
    echo "$i"
done
```

</details>

### 問題2: C言語スタイルのforループ
1〜20の偶数だけを表示するスクリプトを、C言語スタイルのforループで書いてください。

期待される出力：
```
2
4
6
...
20
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
# C言語スタイルで2から20まで2刻み
for ((i = 2; i <= 20; i += 2)); do
    echo "$i"
done
```

</details>

### 問題3: whileループとカウンタ
whileループを使って、10からカウントダウンし「発射！」と表示するスクリプトを書いてください。

期待される出力：
```
10
9
8
...
1
発射！
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
count=10
while ((count >= 1)); do
    echo "$count"
    ((count--))
done
echo "発射！"
```

</details>

## 応用問題

### 問題4: ファイルの行読み取り
以下のCSVデータを変数に格納し、while readで1行ずつ読み取って整形表示してください。

```
data="田中,30,東京
鈴木,25,大阪
佐藤,35,名古屋"
```

期待される出力：
```
名前: 田中, 年齢: 30, 都市: 東京
名前: 鈴木, 年齢: 25, 都市: 大阪
名前: 佐藤, 年齢: 35, 都市: 名古屋
```

<details>
<summary>ヒント</summary>

`IFS=','` を使って区切り文字をカンマに指定します。`<<< "$data"` でヒアストリングを入力に使えます。

</details>

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
data="田中,30,東京
鈴木,25,大阪
佐藤,35,名古屋"

# while read で1行ずつ読み取り、IFS でカンマ区切り
while IFS=',' read -r name age city; do
    echo "名前: $name, 年齢: $age, 都市: $city"
done <<< "$data"
```

</details>

### 問題5: breakとcontinue
1〜20の数字をループし、3の倍数は「Fizz」と表示し、15に達したらループを終了するスクリプトを書いてください。

期待される出力：
```
1
2
Fizz
4
5
Fizz
7
8
Fizz
10
11
Fizz
13
14
Fizz - ループ終了！
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
for i in {1..20}; do
    # 15に達したら特別メッセージで終了
    if ((i == 15)); then
        echo "Fizz - ループ終了！"
        break
    fi
    # 3の倍数はFizzを表示してスキップ
    if ((i % 3 == 0)); then
        echo "Fizz"
        continue
    fi
    echo "$i"
done
```

</details>

### 問題6: select文
select文を使って、以下のメニューを表示し選択に応じた処理を行うスクリプトを書いてください。「終了」を選ぶまでメニューを繰り返し表示します。

期待される動作：
```
操作を選択してください:
1) 日時表示
2) ディレクトリ表示
3) ユーザー表示
4) 終了
#? 1
現在の日時: 2026-03-27 12:00:00
#? 4
終了します
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
PS3="操作を選択してください: "
select action in "日時表示" "ディレクトリ表示" "ユーザー表示" "終了"; do
    case "$action" in
        "日時表示")
            echo "現在の日時: $(date '+%Y-%m-%d %H:%M:%S')"
            ;;
        "ディレクトリ表示")
            echo "カレントディレクトリ: $(pwd)"
            ;;
        "ユーザー表示")
            echo "ユーザー: $(whoami)"
            ;;
        "終了")
            echo "終了します"
            break
            ;;
        *)
            echo "無効な選択です"
            ;;
    esac
done
```

</details>

### 問題7: 九九の表
forループのネストを使って、九九の表（1×1〜9×9）を表示するスクリプトを書いてください。各行は掛ける数ごとに表示し、結果はタブ区切りにしてください。

期待される出力（一部）：
```
1	2	3	4	5	6	7	8	9
2	4	6	8	10	12	14	16	18
...
9	18	27	36	45	54	63	72	81
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
# 九九の表をネストされたforループで作成
for ((i = 1; i <= 9; i++)); do
    for ((j = 1; j <= 9; j++)); do
        # 最後の列以外はタブを付ける
        if ((j < 9)); then
            printf "%d\t" $((i * j))
        else
            printf "%d" $((i * j))
        fi
    done
    echo ""  # 改行
done
```

</details>

### 問題8: リトライ処理
指定されたURLにcurlでアクセスし、失敗した場合は最大3回まで指数バックオフ（1秒、2秒、4秒）でリトライするスクリプトを書いてください。成功したら「成功」、全リトライ失敗で「失敗」と表示します。

<details>
<summary>ヒント</summary>

- `curl -s --max-time 3 URL` で接続タイムアウト3秒
- `sleep $delay` で待機
- `((delay *= 2))` で待ち時間を倍に

</details>

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
set -uo pipefail

url="${1:-http://example.com}"
max_retries=3
delay=1

for ((i = 1; i <= max_retries; i++)); do
    echo "試行 $i/$max_retries: $url"
    if curl -s --max-time 3 "$url" > /dev/null 2>&1; then
        echo "成功！"
        exit 0
    fi
    if ((i < max_retries)); then
        echo "失敗。${delay}秒後にリトライ..."
        sleep "$delay"
        ((delay *= 2))  # 指数バックオフ
    fi
done

echo "失敗: 全リトライが失敗しました" >&2
exit 1
```

</details>
