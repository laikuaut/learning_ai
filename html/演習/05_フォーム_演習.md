## 基本問題

### 問題1：基本的なフォーム
以下の条件を満たすお問い合わせフォームを作成してください。

- form の送信先は `/contact`、メソッドは `post`
- 「お名前」テキスト入力（必須）
- 「メールアドレス」メール入力（必須）
- 「送信」ボタン

<details>
<summary>ヒント</summary>

`<form>` で全体を囲み、各入力欄に `<label>` を付けます。必須にするには `required` 属性を使います。

</details>

<details>
<summary>解答例</summary>

```html
<form action="/contact" method="post">
  <div>
    <!-- for と id で label と input を関連付け -->
    <label for="name">お名前：</label>
    <input type="text" id="name" name="name" required>
  </div>
  <div>
    <label for="email">メールアドレス：</label>
    <input type="email" id="email" name="email" required>
  </div>
  <button type="submit">送信</button>
</form>
```

<!-- required 属性を付けると、空のまま送信しようとしたときにブラウザが警告を表示します。 -->

</details>

### 問題2：input type の使い分け
以下の各入力欄に最も適切な `input type` を答えてください。

1. パスワード入力欄
2. 生年月日の入力欄
3. 電話番号の入力欄
4. 商品の個数（1〜99）
5. Webサイトの URL

<details>
<summary>解答</summary>

1. `type="password"` — 入力文字がマスクされます
2. `type="date"` — カレンダーUIが表示されます
3. `type="tel"` — スマホでテンキーが表示されます
4. `type="number"` — `min="1" max="99"` と組み合わせます
5. `type="url"` — URLの形式チェックが自動で行われます

</details>

### 問題3：label要素の関連付け
以下のフォームには label の関連付けに問題があります。正しく修正してください。

```html
<label>ユーザー名</label>
<input type="text" id="username" name="username">

<label for="mail">メールアドレス</label>
<input type="email" id="email" name="email">
```

<details>
<summary>解答例</summary>

```html
<!-- 修正1: for 属性を追加して id と紐付ける -->
<label for="username">ユーザー名</label>
<input type="text" id="username" name="username">

<!-- 修正2: for の値を id と一致させる（mail → email） -->
<label for="email">メールアドレス</label>
<input type="email" id="email" name="email">
```

<!-- label の for 属性の値と input の id 属性の値が一致しないと関連付けられません。 -->

</details>

### 問題4：select と textarea
以下の条件を満たすフォーム部品を作成してください。

1. 都道府県を選択するドロップダウン（東京都、大阪府、北海道の3つ。初期状態は「選択してください」）
2. お問い合わせ内容を入力するテキストエリア（5行30列、プレースホルダーあり）

<details>
<summary>解答例</summary>

```html
<!-- ドロップダウン -->
<label for="prefecture">都道府県：</label>
<select id="prefecture" name="prefecture">
  <!-- disabled selected で初期値を選択不能にする -->
  <option value="" disabled selected>選択してください</option>
  <option value="tokyo">東京都</option>
  <option value="osaka">大阪府</option>
  <option value="hokkaido">北海道</option>
</select>

<!-- テキストエリア -->
<label for="message">お問い合わせ内容：</label>
<textarea id="message" name="message" rows="5" cols="30"
          placeholder="こちらにお問い合わせ内容を入力してください"></textarea>
```

<!-- select の最初の option に disabled selected を付けると、プレースホルダー的に使えます。 -->

</details>

### 問題5：チェックボックスとラジオボタン
以下の条件を満たすフォーム部品を作成してください。

1. 好きなプログラミング言語（複数選択可）：Python、JavaScript、Java、Go
2. 経験レベル（1つだけ選択）：初心者、中級者、上級者

それぞれ `<fieldset>` と `<legend>` でグループ化してください。

<details>
<summary>解答例</summary>

```html
<!-- チェックボックス：複数選択可 -->
<fieldset>
  <legend>好きなプログラミング言語（複数選択可）</legend>
  <label><input type="checkbox" name="language" value="python"> Python</label>
  <label><input type="checkbox" name="language" value="javascript"> JavaScript</label>
  <label><input type="checkbox" name="language" value="java"> Java</label>
  <label><input type="checkbox" name="language" value="go"> Go</label>
</fieldset>

<!-- ラジオボタン：1つだけ選択 -->
<fieldset>
  <legend>経験レベル</legend>
  <label><input type="radio" name="level" value="beginner"> 初心者</label>
  <label><input type="radio" name="level" value="intermediate"> 中級者</label>
  <label><input type="radio" name="level" value="advanced"> 上級者</label>
</fieldset>
```

<!--
  チェックボックス：同じ name で複数の値を送信できます。
  ラジオボタン：同じ name のグループ内で1つだけ選択できます。
  fieldset/legend はフォーム要素をグループ化し、スクリーンリーダーに「このグループは何か」を伝えます。
-->

</details>

## 応用問題

### 問題6：バリデーション属性
以下の条件を満たす入力欄を作成してください。すべてHTMLのバリデーション属性のみで実装すること（JavaScriptは使わない）。

1. ユーザー名：3〜20文字、英数字とアンダースコアのみ（`pattern` 属性を使用）
2. 年齢：18〜100の整数
3. ウェブサイト：URL形式（任意入力）
4. パスワード：8文字以上（必須）

<details>
<summary>ヒント</summary>

`pattern` 属性には正規表現を指定します。`title` 属性に入力ルールの説明を書くと、バリデーションエラー時にブラウザが表示してくれます。

</details>

<details>
<summary>解答例</summary>

```html
<form action="/register" method="post">
  <div>
    <label for="username">ユーザー名（3〜20文字、英数字と_）：</label>
    <input type="text" id="username" name="username"
           required
           minlength="3" maxlength="20"
           pattern="[a-zA-Z0-9_]{3,20}"
           title="英数字とアンダースコアで3〜20文字">
  </div>

  <div>
    <label for="age">年齢（18〜100）：</label>
    <input type="number" id="age" name="age"
           required
           min="18" max="100">
  </div>

  <div>
    <label for="website">ウェブサイト（任意）：</label>
    <input type="url" id="website" name="website"
           placeholder="https://example.com">
    <!-- required がないので任意入力 -->
  </div>

  <div>
    <label for="password">パスワード（8文字以上）：</label>
    <input type="password" id="password" name="password"
           required
           minlength="8">
  </div>

  <button type="submit">登録</button>
</form>
```

<!--
  pattern: 正規表現で入力形式を指定。^ と $ は自動で付加されます。
  minlength / maxlength: 文字数の制限。
  min / max: 数値の範囲制限。
  title: バリデーションエラー時に表示されるメッセージ。
-->

</details>

### 問題7：ファイルアップロードフォーム
以下の条件を満たすファイルアップロードフォームを作成してください。

- 送信先は `/upload`、メソッドは `post`
- 画像ファイルのみ受け付ける（JPEG, PNG, GIF）
- 複数ファイル選択可能
- enctype を適切に設定

<details>
<summary>解答例</summary>

```html
<!-- ファイルアップロードには enctype="multipart/form-data" が必須 -->
<form action="/upload" method="post" enctype="multipart/form-data">
  <label for="photos">写真を選択（JPEG, PNG, GIF）：</label>
  <input type="file" id="photos" name="photos"
         accept="image/jpeg, image/png, image/gif"
         multiple>
  <button type="submit">アップロード</button>
</form>
```

<!--
  enctype="multipart/form-data" を指定しないとファイルが正しく送信されません。
  accept 属性でファイルの種類を制限できます（ブラウザのファイル選択ダイアログに反映）。
  multiple 属性で複数ファイルの選択が可能になります。
-->

</details>

## チャレンジ問題

### 問題8：会員登録フォームの作成
以下の要件を満たす完全なHTMLページを作成してください。

- ページタイトル「会員登録」
- h1 で「会員登録フォーム」
- form の送信先は `/register`、メソッドは `post`
- 以下の入力欄を設ける：
  - 氏名（テキスト、必須）
  - メールアドレス（メール形式、必須）
  - パスワード（8文字以上、必須）
  - 生年月日（日付）
  - 電話番号（電話番号形式）
  - 性別（ラジオボタン：男性/女性/その他/回答しない）— fieldset で囲む
  - 興味のある分野（チェックボックス：プログラミング/デザイン/マーケティング/データ分析）— fieldset で囲む
  - 自己紹介（テキストエリア、3行40列）
  - プロフィール画像（ファイル、画像のみ）
  - 利用規約への同意（チェックボックス、必須）
- 「登録する」送信ボタンと「リセット」ボタン

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>会員登録</title>
</head>
<body>
  <h1>会員登録フォーム</h1>

  <form action="/register" method="post" enctype="multipart/form-data">
    <!-- 基本情報 -->
    <div>
      <label for="fullname">氏名：</label>
      <input type="text" id="fullname" name="fullname" required>
    </div>

    <div>
      <label for="email">メールアドレス：</label>
      <input type="email" id="email" name="email" required>
    </div>

    <div>
      <label for="password">パスワード（8文字以上）：</label>
      <input type="password" id="password" name="password"
             required minlength="8">
    </div>

    <div>
      <label for="birthday">生年月日：</label>
      <input type="date" id="birthday" name="birthday">
    </div>

    <div>
      <label for="phone">電話番号：</label>
      <input type="tel" id="phone" name="phone"
             placeholder="090-1234-5678">
    </div>

    <!-- 性別（ラジオボタン） -->
    <fieldset>
      <legend>性別</legend>
      <label><input type="radio" name="gender" value="male"> 男性</label>
      <label><input type="radio" name="gender" value="female"> 女性</label>
      <label><input type="radio" name="gender" value="other"> その他</label>
      <label><input type="radio" name="gender" value="noanswer" checked> 回答しない</label>
    </fieldset>

    <!-- 興味のある分野（チェックボックス） -->
    <fieldset>
      <legend>興味のある分野（複数選択可）</legend>
      <label><input type="checkbox" name="interest" value="programming"> プログラミング</label>
      <label><input type="checkbox" name="interest" value="design"> デザイン</label>
      <label><input type="checkbox" name="interest" value="marketing"> マーケティング</label>
      <label><input type="checkbox" name="interest" value="data"> データ分析</label>
    </fieldset>

    <!-- 自己紹介 -->
    <div>
      <label for="bio">自己紹介：</label><br>
      <textarea id="bio" name="bio" rows="3" cols="40"
                placeholder="簡単な自己紹介をお書きください"></textarea>
    </div>

    <!-- プロフィール画像 -->
    <div>
      <label for="avatar">プロフィール画像：</label>
      <input type="file" id="avatar" name="avatar"
             accept="image/*">
    </div>

    <!-- 利用規約への同意 -->
    <div>
      <label>
        <input type="checkbox" name="agree" value="yes" required>
        利用規約に同意します（必須）
      </label>
    </div>

    <!-- ボタン -->
    <div>
      <button type="submit">登録する</button>
      <button type="reset">リセット</button>
    </div>
  </form>
</body>
</html>
```

</details>
