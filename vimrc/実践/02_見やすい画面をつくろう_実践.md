# 実践課題02：見やすい画面をつくろう ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第2章（表示とエディタの基本設定）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 行番号・カラースキーム・ステータスラインの設定を組み合わせて、視認性の高いエディタ画面を構築する

---

## 完成イメージ

```
┌──────────────────────────────────────────────────┐
│  1 │ def calculate_tax(price, rate=0.10):        │
│  2 │     """税込み価格を計算する"""                │
│  3 │     tax = price * rate                       │
│  4 │     return price + tax                       │
│  5 │                                              │
│  6 │ total = calculate_tax(1000)                  │
│  7 │ print(f"税込み: {total}円")                  │
│  ~ │                                              │
│  ~ │                                              │
├──────────────────────────────────────────────────┤
│ [+] example.py [python] [utf-8]   7L  3/15  50% │
└──────────────────────────────────────────────────┘
```

ステータスラインに以下の情報が表示されます。

```
┌────────────────────────────────────────────────────────┐
│ [+]         : 変更あり（未保存マーク）                  │
│ example.py  : ファイル名                               │
│ [python]    : ファイルタイプ                            │
│ [utf-8]     : エンコーディング                         │
│ 7L          : 総行数                                   │
│ 3/15        : カーソル位置（行/列）                     │
│ 50%         : ファイル内のカーソル位置（割合）          │
└────────────────────────────────────────────────────────┘
```

---

## 課題の要件

1. 行番号を表示し、行番号の列幅を4桁に設定する
2. カーソル行とカーソル列をハイライトする
3. シンタックスハイライトを有効にする
4. カラースキームを設定する
5. ステータスラインをカスタマイズして、以下を表示する
   - 変更フラグ、ファイル名、ファイルタイプ、エンコーディング
   - 総行数、カーソル位置（行/列）、ファイル内の位置（%）
6. 長い行の折り返し方法を設定する

---

## ステップガイド

<details>
<summary>ステップ1：行番号と行のハイライト</summary>

```vim
" --- 行番号の表示 ---
set number              " 行番号を表示
set numberwidth=4       " 行番号の表示幅を4桁に

" --- カーソル行・列のハイライト ---
set cursorline          " カーソル行をハイライト
set cursorcolumn        " カーソル列をハイライト（縦ライン）
```

`cursorcolumn` はインデントのズレを発見するのに便利です。ただし、気になる場合は省略しても構いません。

</details>

<details>
<summary>ステップ2：シンタックスハイライトとカラースキーム</summary>

```vim
" --- シンタックスハイライト ---
syntax enable           " シンタックスハイライトを有効化

" --- カラースキーム ---
colorscheme desert      " カラースキームを設定
set background=dark     " 背景色のタイプ（dark/light）
```

Vim組み込みの主なカラースキーム:
- `desert` — 暗めの背景に温かみのある配色
- `slate` — 青系の暗い背景
- `elflord` — 鮮やかな配色
- `peachpuff` — 明るい背景向け

</details>

<details>
<summary>ステップ3：ステータスラインのカスタマイズ</summary>

`statusline` オプションでは、特殊な `%` 記号で情報を埋め込みます。

```vim
" --- ステータスライン ---
set laststatus=2        " 常にステータスラインを表示

" ステータスラインの内容
set statusline=%m       " 変更フラグ [+]
set statusline+=\ %f    " ファイル名（スペースを \  でエスケープ）
set statusline+=\ [%Y]  " ファイルタイプ
set statusline+=\ [%{&fileencoding}]  " エンコーディング
set statusline+=%=      " ← ここから右寄せ
set statusline+=%LL     " 総行数
set statusline+=\ \ %l/%c    " 行/列
set statusline+=\ \ %p%%     " ファイル内の位置（%）
```

主要な `statusline` の書式指定子:

| 指定子 | 説明 |
|--------|------|
| `%f` | ファイル名 |
| `%F` | フルパス |
| `%m` | 変更フラグ（`[+]` / `-`） |
| `%Y` | ファイルタイプ |
| `%l` | 現在の行番号 |
| `%c` | 現在の列番号 |
| `%L` | 総行数 |
| `%p` | ファイル内の位置（%） |
| `%=` | 左右の区切り（右寄せ開始） |
| `%{&option}` | オプションの値を展開 |

</details>

<details>
<summary>ステップ4：行の折り返し設定</summary>

```vim
" --- 折り返し設定 ---
set wrap                " 長い行を折り返して表示
set linebreak           " 単語の途中で折り返さない
set showbreak=↪\        " 折り返し行の先頭に表示する記号
```

`linebreak` を設定すると、単語単位で折り返してくれるので読みやすくなります。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ===========================================
" 見やすい画面設定
" ===========================================

set nocompatible
syntax enable

" --- 行番号 ---
set number

" --- カーソル ---

" --- ステータスライン ---

" --- 折り返し ---
set wrap
set linebreak
set showbreak=↪\ 
```

</details>

<details>
<summary>解答例（改良版：条件付き設定と視覚的工夫）</summary>

```vim
" ===========================================
" 見やすい画面設定 - 改良版
" ===========================================

set nocompatible
syntax enable

" --- 行番号（相対行番号を併用）---
set number              " 絶対行番号を表示
set relativenumber      " 相対行番号を表示（移動の参考に）
set numberwidth=4

" --- カーソル ---
set cursorline

" 挿入モードではカーソルラインを消す（モードの視覚的区別）
autocmd InsertEnter * set nocursorline
autocmd InsertLeave * set cursorline

" --- カラー ---
colorscheme desert
set background=dark

" 行番号の色をカスタマイズ
highlight LineNr ctermfg=grey
highlight CursorLineNr ctermfg=yellow cterm=bold

" --- ステータスライン ---
set laststatus=2

" 読みやすくするため変数に分割して組み立て
let &statusline = ''
let &statusline .= ' %m'                     " 変更フラグ
let &statusline .= ' %f'                     " ファイル名
let &statusline .= ' [%Y]'                   " ファイルタイプ
let &statusline .= ' [%{&fileencoding}]'     " エンコーディング
let &statusline .= ' %r'                     " 読み取り専用フラグ
let &statusline .= '%='                      " 右寄せ
let &statusline .= ' %LL'                    " 総行数
let &statusline .= '  %l/%c'                 " 行/列
let &statusline .= '  %p%% '                 " 位置（%）

" --- 折り返し ---
set wrap
set linebreak
set showbreak=↪\ 
set breakindent         " 折り返し行もインデントを維持

" --- 追加の視認性向上 ---
set list                        " 不可視文字を表示
set listchars=tab:»\ ,trail:·  " タブと行末空白を可視化
set showmatch                   " 対応括弧を表示
set matchtime=1                 " 表示時間を短く
```

**初心者向けとの違い**:

- `relativenumber` で相対行番号を追加（`5j` のような移動が楽に）
- 挿入モードで `cursorline` を切り替えてモードを視覚的に区別
- `highlight` コマンドで行番号の色をカスタマイズ
- `let &statusline` で設定を読みやすく分割
- `breakindent` で折り返し行のインデントを維持
- `listchars` で不可視文字を可視化

</details>

## よくある間違い

| ミス | 正しい書き方 | 説明 |
|------|-------------|------|
| `statusline` のスペースを `\ ` でエスケープし忘れ | `%m\ %f` | スペースは `\ ` で書く |
| `%%` と書くべきところで `%` | `%p%%` | リテラルの `%` は `%%` |
| `syntax on` と `syntax enable` の混同 | `syntax enable` を推奨 | `enable` は既存の色設定を維持する |
| `colorscheme` を `syntax enable` の前に書く | `syntax enable` → `colorscheme` の順 | 順序が逆だと反映されないことがある |
