# 第2章：条件分岐

## この章のゴール

- 比較演算子と論理演算子を使って条件を表現できるようになる
- if文、if-else文、else if文を使い分けられるようになる
- 三項演算子を使った短い条件式が書けるようになる
- switch文で多分岐処理を実装できるようになる
- 条件分岐を使った実用的なプログラムが書けるようになる

---

## 2.1 比較演算子

比較演算子（comparison operator）は、2つの値を比較して**真（1）**または**偽（0）**を返します。

| 演算子 | 意味 | 例 | 結果 |
|---|---|---|---|
| `==` | 等しい | `5 == 5` | 1（真） |
| `!=` | 等しくない | `5 != 3` | 1（真） |
| `<` | より小さい | `3 < 5` | 1（真） |
| `>` | より大きい | `5 > 3` | 1（真） |
| `<=` | 以下 | `5 <= 5` | 1（真） |
| `>=` | 以上 | `3 >= 5` | 0（偽） |

```c
#include <stdio.h>

int main(void) {
    int a = 10, b = 20;

    printf("%d == %d → %d\n", a, b, a == b);   // 0
    printf("%d != %d → %d\n", a, b, a != b);   // 1
    printf("%d <  %d → %d\n", a, b, a < b);    // 1
    printf("%d >  %d → %d\n", a, b, a > b);    // 0
    printf("%d <= %d → %d\n", a, b, a <= b);   // 1
    printf("%d >= %d → %d\n", a, b, a >= b);   // 0

    return 0;
}
```

実行結果：
```
10 == 20 → 0
10 != 20 → 1
10 <  20 → 1
10 >  20 → 0
10 <= 20 → 1
10 >= 20 → 0
```

> **重要：** C言語には `bool` 型がありません（C99から `<stdbool.h>` で使えますが）。真偽は整数で表され、**0が偽、0以外が真**です。

### ポイントまとめ
- 比較演算子は真（1）または偽（0）を返す
- C言語では「0 = 偽、0以外 = 真」

## 2.2 論理演算子

複数の条件を組み合わせるために、論理演算子（logical operator）を使います。

| 演算子 | 意味 | 例 | 説明 |
|---|---|---|---|
| `&&` | AND（かつ） | `a > 0 && a < 100` | 両方とも真なら真 |
| `\|\|` | OR（または） | `a == 0 \|\| a == 1` | どちらかが真なら真 |
| `!` | NOT（否定） | `!(a == 0)` | 真偽を反転 |

```c
#include <stdio.h>

int main(void) {
    int age = 25;
    int income = 3000000;  // 年収300万円

    // AND: 両方の条件を満たす
    if (age >= 20 && income >= 2000000) {
        printf("ローン審査: 条件を満たしています\n");
    }

    // OR: どちらかの条件を満たす
    int day = 0;  // 0=日曜日
    if (day == 0 || day == 6) {
        printf("今日は休日です\n");
    }

    // NOT: 条件の否定
    int is_holiday = 0;  // 0 = 平日
    if (!is_holiday) {
        printf("今日は平日です\n");
    }

    return 0;
}
```

実行結果：
```
ローン審査: 条件を満たしています
今日は休日です
今日は平日です
```

### 真理値表

```
AND (&&)           OR (||)            NOT (!)
A     B   結果     A     B   結果     A   結果
真    真   真      真    真   真      真   偽
真    偽   偽      真    偽   真      偽   真
偽    真   偽      偽    真   真
偽    偽   偽      偽    偽   偽
```

### 短絡評価（short-circuit evaluation）

`&&` は左側が偽なら右側を評価しません。`||` は左側が真なら右側を評価しません。これを**短絡評価**と呼びます。

```c
// b が 0 のとき、a / b は実行されない（0除算を回避）
if (b != 0 && a / b > 2) {
    printf("条件を満たしました\n");
}
```

### ポイントまとめ
- `&&`（AND）は両方真のとき真
- `||`（OR）はどちらかが真のとき真
- `!`（NOT）は真偽を反転する
- 短絡評価により、不要な計算を省略できる

## 2.3 if文

if文は、**条件が真のとき**にブロック内の処理を実行します。

### 基本の if文

```c
#include <stdio.h>

int main(void) {
    int temperature = 35;

    if (temperature >= 30) {
        printf("今日は真夏日です。熱中症に注意してください。\n");
    }

    return 0;
}
```

### if-else文

条件が真のときと偽のときで処理を分けます。

```c
#include <stdio.h>

int main(void) {
    int age;
    printf("年齢を入力してください: ");
    scanf("%d", &age);

    if (age >= 18) {
        printf("あなたは成人です。\n");
    } else {
        printf("あなたは未成年です。\n");
    }

    return 0;
}
```

### else if文（多分岐）

3つ以上の場合分けをするときに使います。

```c
#include <stdio.h>

int main(void) {
    int score;
    printf("点数を入力してください(0-100): ");
    scanf("%d", &score);

    if (score >= 90) {
        printf("評価: A（優秀）\n");
    } else if (score >= 80) {
        printf("評価: B（良好）\n");
    } else if (score >= 70) {
        printf("評価: C（普通）\n");
    } else if (score >= 60) {
        printf("評価: D（可）\n");
    } else {
        printf("評価: F（不可）\n");
    }

    return 0;
}
```

実行例：
```
点数を入力してください(0-100): 85
評価: B（良好）
```

### if文の処理の流れ

```
            score = 85
                │
        score >= 90 ?
       ┌─── Yes ───┐─── No ───┐
       │            │          │
    評価A        score >= 80 ?
                ┌── Yes ──┐── No ──┐
                │         │        │
             評価B     score >= 70 ?
                       ┌ Yes ┐─ No ─┐
                       │     │      │
                    評価C  score >= 60 ?
                          ┌ Yes ┐─ No ─┐
                          │     │      │
                       評価D   評価F
```

> **よくある間違い：** `=` と `==` の混同は、C言語でもっとも多いバグの一つです。

```c
// NG: 代入になってしまう（常に真になる）
if (x = 5) {    // x に 5 を代入し、5 は 0 でないので常に真
    printf("ここは常に実行される！\n");
}

// OK: 比較
if (x == 5) {
    printf("x は 5 です\n");
}
```

> **現場のテクニック：** 定数を左に書くと、`=` にしたときコンパイルエラーになるので安全です（ヨーダ記法）。

```c
if (5 == x) {   // 万が一 5 = x と書くとコンパイルエラー
    printf("x は 5 です\n");
}
```

### ポイントまとめ
- `if` は条件が真のときに処理を実行する
- `else` は条件が偽のときの処理を書く
- `else if` で3つ以上の分岐ができる
- `=`（代入）と `==`（比較）を間違えないこと

## 2.4 三項演算子

三項演算子（ternary operator）は、**if-elseを1行で書ける**便利な記法です。

```
条件 ? 真の値 : 偽の値
```

```c
#include <stdio.h>

int main(void) {
    int age = 20;

    // if-else で書く場合
    // const char *status;
    // if (age >= 18) {
    //     status = "成人";
    // } else {
    //     status = "未成年";
    // }

    // 三項演算子で書く場合（同じ意味）
    const char *status = (age >= 18) ? "成人" : "未成年";
    printf("%d歳: %s\n", age, status);

    // 数値の最大値を求める
    int a = 30, b = 50;
    int max = (a > b) ? a : b;
    printf("%d と %d の大きい方: %d\n", a, b, max);

    // 偶数・奇数の判定
    int num = 7;
    printf("%d は%s数です\n", num, (num % 2 == 0) ? "偶" : "奇");

    return 0;
}
```

実行結果：
```
20歳: 成人
30 と 50 の大きい方: 50
7 は奇数です
```

> **現場のベストプラクティス：** 三項演算子は短い条件式に使いましょう。複雑な処理にはif-else文のほうが読みやすくなります。

### ポイントまとめ
- `条件 ? 値1 : 値2` で短い条件分岐が書ける
- 複雑な条件にはif-elseを使うほうが可読性が高い

## 2.5 switch文

switch文は、**1つの変数の値**に応じて処理を分岐するときに使います。`if-else if` が長くなる場合に有効です。

```c
#include <stdio.h>

int main(void) {
    int month;
    printf("月(1-12)を入力してください: ");
    scanf("%d", &month);

    switch (month) {
        case 1:
            printf("1月: 睦月（むつき）\n");
            break;
        case 2:
            printf("2月: 如月（きさらぎ）\n");
            break;
        case 3:
            printf("3月: 弥生（やよい）\n");
            break;
        case 4:
            printf("4月: 卯月（うづき）\n");
            break;
        case 5:
            printf("5月: 皐月（さつき）\n");
            break;
        case 6:
            printf("6月: 水無月（みなづき）\n");
            break;
        case 7:
            printf("7月: 文月（ふみづき）\n");
            break;
        case 8:
            printf("8月: 葉月（はづき）\n");
            break;
        case 9:
            printf("9月: 長月（ながつき）\n");
            break;
        case 10:
            printf("10月: 神無月（かんなづき）\n");
            break;
        case 11:
            printf("11月: 霜月（しもつき）\n");
            break;
        case 12:
            printf("12月: 師走（しわす）\n");
            break;
        default:
            printf("無効な月です。1〜12を入力してください。\n");
            break;
    }

    return 0;
}
```

実行例：
```
月(1-12)を入力してください: 4
4月: 卯月（うづき）
```

### break の重要性

`break` を書き忘れると、次の `case` の処理も実行されてしまいます（フォールスルー、fall-through）。

```c
#include <stdio.h>

int main(void) {
    int rank = 2;

    printf("=== break なしの場合 ===\n");
    switch (rank) {
        case 1:
            printf("金メダル!\n");
        case 2:
            printf("銀メダル!\n");    // ここから実行開始
        case 3:
            printf("銅メダル!\n");    // break がないのでここも実行される
        default:
            printf("入賞おめでとう!\n"); // ここも実行される
    }

    printf("\n=== break ありの場合 ===\n");
    switch (rank) {
        case 1:
            printf("金メダル!\n");
            break;
        case 2:
            printf("銀メダル!\n");
            break;
        case 3:
            printf("銅メダル!\n");
            break;
        default:
            printf("入賞おめでとう!\n");
            break;
    }

    return 0;
}
```

実行結果：
```
=== break なしの場合 ===
銀メダル!
銅メダル!
入賞おめでとう!

=== break ありの場合 ===
銀メダル!
```

### フォールスルーの活用例

意図的に `break` を省略して、複数のcaseで同じ処理を実行することもあります。

```c
#include <stdio.h>

int main(void) {
    int month;
    printf("月(1-12)を入力してください: ");
    scanf("%d", &month);

    switch (month) {
        case 3: case 4: case 5:
            printf("春です\n");
            break;
        case 6: case 7: case 8:
            printf("夏です\n");
            break;
        case 9: case 10: case 11:
            printf("秋です\n");
            break;
        case 12: case 1: case 2:
            printf("冬です\n");
            break;
        default:
            printf("無効な入力です\n");
            break;
    }

    return 0;
}
```

> **よくある間違い：** switch文は `int` や `char` などの整数型でしか使えません。`float`、`double`、文字列では使えません。

### ポイントまとめ
- switch文は1つの値で多分岐するときに便利
- `break` を忘れるとフォールスルーが起きる
- `default` はどのcaseにも一致しないときの処理
- switch文は整数型の値にのみ使える

## 2.6 ネストした条件分岐

if文の中にif文を書くことを**ネスト（nest、入れ子）**と呼びます。

```c
#include <stdio.h>

int main(void) {
    int age;
    char gender;

    printf("年齢を入力してください: ");
    scanf("%d", &age);
    printf("性別を入力してください (M/F): ");
    scanf(" %c", &gender);   // %c の前のスペースは改行文字を読み飛ばすため

    if (age >= 18) {
        if (gender == 'M' || gender == 'm') {
            printf("成人男性です\n");
        } else if (gender == 'F' || gender == 'f') {
            printf("成人女性です\n");
        } else {
            printf("成人です（性別不明）\n");
        }
    } else {
        if (gender == 'M' || gender == 'm') {
            printf("未成年男性です\n");
        } else if (gender == 'F' || gender == 'f') {
            printf("未成年女性です\n");
        } else {
            printf("未成年です（性別不明）\n");
        }
    }

    return 0;
}
```

> **現場のベストプラクティス：** ネストは深くなるほど読みにくくなります。3段以上のネストは関数に分割するか、早期リターン（early return）で浅くすることを検討しましょう。

### ポイントまとめ
- if文の中にif文を書くことでネストができる
- ネストが深くなると可読性が下がるので注意
- `scanf(" %c", ...)` のスペースで改行文字を読み飛ばせる

## 2.7 実用例：簡易電卓

ここまで学んだ条件分岐を使って、簡易電卓を作ってみましょう。

```c
#include <stdio.h>

int main(void) {
    double num1, num2;
    char op;

    printf("=== 簡易電卓 ===\n");
    printf("計算式を入力してください（例: 10 + 3）: ");
    scanf("%lf %c %lf", &num1, &op, &num2);

    switch (op) {
        case '+':
            printf("%.2f + %.2f = %.2f\n", num1, num2, num1 + num2);
            break;
        case '-':
            printf("%.2f - %.2f = %.2f\n", num1, num2, num1 - num2);
            break;
        case '*':
            printf("%.2f * %.2f = %.2f\n", num1, num2, num1 * num2);
            break;
        case '/':
            if (num2 == 0) {
                printf("エラー: 0で割ることはできません。\n");
            } else {
                printf("%.2f / %.2f = %.2f\n", num1, num2, num1 / num2);
            }
            break;
        case '%':
            if (num2 == 0) {
                printf("エラー: 0で割ることはできません。\n");
            } else {
                printf("%.0f %% %.0f = %d\n", num1, num2, (int)num1 % (int)num2);
            }
            break;
        default:
            printf("エラー: 未対応の演算子 '%c' です。\n", op);
            break;
    }

    return 0;
}
```

実行例：
```
=== 簡易電卓 ===
計算式を入力してください（例: 10 + 3）: 100 / 3
100.00 / 3.00 = 33.33
```

## 2.8 実用例：料金計算（年齢と曜日による割引）

```c
#include <stdio.h>

int main(void) {
    int age;
    int is_weekend;  // 0: 平日, 1: 休日
    const int BASE_PRICE = 1800;  // 基本料金

    printf("=== 映画チケット料金計算 ===\n");
    printf("年齢を入力してください: ");
    scanf("%d", &age);
    printf("休日ですか？ (0: 平日, 1: 休日): ");
    scanf("%d", &is_weekend);

    int price;

    // 年齢による料金設定
    if (age < 4) {
        price = 0;          // 3歳以下: 無料
        printf("区分: 幼児（無料）\n");
    } else if (age < 13) {
        price = 1000;       // 4〜12歳: 子供料金
        printf("区分: 子供\n");
    } else if (age < 65) {
        price = BASE_PRICE; // 13〜64歳: 一般料金
        printf("区分: 一般\n");
    } else {
        price = 1200;       // 65歳以上: シニア料金
        printf("区分: シニア\n");
    }

    // 平日割引（一般のみ）
    if (!is_weekend && age >= 13 && age < 65) {
        int discount = 200;
        price -= discount;
        printf("平日割引: -%d円\n", discount);
    }

    printf("お支払い金額: %d円\n", price);

    return 0;
}
```

実行例：
```
=== 映画チケット料金計算 ===
年齢を入力してください: 30
休日ですか？ (0: 平日, 1: 休日): 0
区分: 一般
平日割引: -200円
お支払い金額: 1600円
```

## 章末まとめ

この章で学んだことを整理します。

```
条件分岐
├── 比較演算子
│   └── ==, !=, <, >, <=, >=
├── 論理演算子
│   ├── && (AND)  … 両方真なら真
│   ├── || (OR)   … どちらか真なら真
│   └── !  (NOT)  … 真偽を反転
├── if文
│   ├── if (条件) { ... }
│   ├── if-else
│   └── else if（多分岐）
├── 三項演算子
│   └── 条件 ? 値1 : 値2
├── switch文
│   ├── case 値: ... break;
│   ├── default: ...
│   └── break忘れ → フォールスルー
└── 注意点
    ├── = と == の混同に注意
    ├── ネストは浅く保つ
    └── switchは整数型のみ対応
```

次の章では、**繰り返し**（for文、while文）を学びます。同じ処理を何度も繰り返す方法を身につけましょう。
