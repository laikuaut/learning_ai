# 第1章：変数とデータ型

> **この章のゴール**: Pythonで「データを扱う」基礎をマスターする。
> 変数・データ型・型変換・演算を理解すれば、あらゆるプログラムの土台が身につきます。

## なぜ「変数とデータ型」を最初に学ぶのか？

プログラミングとは、突き詰めれば **「データを受け取り、加工し、出力する」** 作業です。

例えば、こんな日常的なシステムを考えてみてください：

- **ECサイト**: 商品名（文字列）、価格（数値）、在庫（数値）、販売中か（真偽値）
- **SNS**: ユーザー名（文字列）、フォロワー数（数値）、非公開か（真偽値）
- **天気アプリ**: 気温（小数）、地域名（文字列）、降水確率（小数）

すべてのアプリケーションは、**変数**にデータを入れ、**データ型**に応じた操作をしています。この章を理解することは、プログラミング全体の基礎固めになります。

## 1.1 変数とは？

### 概念：名前のついたデータの入れ物

**変数（variable）** は、データに名前をつけて保存する仕組みです。

```
  変数名      値
  ┌─────┐   ┌─────┐
  │ name │ → │"太郎"│
  └─────┘   └─────┘
  ┌─────┐   ┌─────┐
  │ age  │ → │ 20  │
  └─────┘   └─────┘
```

Pythonでは `=`（代入演算子）を使って変数に値を入れます。

```python
name = "太郎"
age = 20
```

> **📖 正確には**: Pythonの変数は「箱」よりも「名札」に近い概念です。
> データ（オブジェクト）はメモリ上に存在し、変数はそのデータを指し示す「名札」です。
> この違いは、リストなどの「ミュータブル（変更可能な）オブジェクト」を学ぶ第4章で重要になります。

### 変数の値を上書きする

変数には何度でも新しい値を代入できます。

```python
message = "おはよう"
print(message)        # おはよう

message = "こんにちは"
print(message)        # こんにちは（上書きされた）
```

### 変数の値を使って別の変数を作る

```python
price = 1000
tax_rate = 0.1
tax = price * tax_rate      # 100.0
total = price + tax          # 1100.0
```

このように、変数を使って計算し、その結果をさらに別の変数に入れることができます。

---

### 変数の命名ルール

Pythonには変数名に関する厳格なルールがあります。

| ルール | OK | NG | 理由 |
|--------|----|----|------|
| 英字・数字・`_` が使える | `my_name`, `count1` | `my-name` | `-` は引き算と解釈される |
| 先頭に数字は使えない | `item1` | `1item` | 数値リテラルと区別できない |
| 大文字と小文字は区別される | `Name` と `name` は別の変数 | - | - |
| 予約語は使えない | - | `if`, `for`, `class` | Python構文として使われている |
| 日本語も使える（非推奨） | `名前 = "太郎"` | - | 動くが慣習に反する |

### 命名のベストプラクティス

```python
# ✅ 良い変数名：意味が明確で、スネークケースを使用
user_name = "太郎"
total_price = 1500
is_logged_in = True
max_retry_count = 3

# ❌ 悪い変数名：意味がわからない、スタイルが統一されていない
x = "太郎"           # xが何を表すか不明
TotalPrice = 1500     # PascalCaseはクラス名に使う（第9章）
flg = True            # 省略しすぎて意味不明
```

> **📖 コラム：なぜ命名が重要か？**
>
> プログラマーはコードを「書く」時間より「読む」時間の方がはるかに長いです。
> 変数名は「未来の自分」や「チームメンバー」へのメッセージです。
> `d = 7` より `days_in_week = 7` の方が、半年後に読んでも意味がわかります。

### ✏️ やってみよう

Pythonを起動して、以下を入力してみましょう：

```python
my_name = "（あなたの名前）"
my_age = （あなたの年齢）
print(my_name)
print(my_age)
```

エラーが出たら、スペルミスや引用符の閉じ忘れがないか確認してください。

## 1.2 データ型

### なぜ「型」が存在するのか？

コンピュータにとって、`42`（数値）と `"42"`（文字列）は全く別のものです。

```python
print(42 + 8)       # 50   ← 数値の足し算
print("42" + "8")   # 428  ← 文字列の連結
```

同じ `+` でも、データ型によって動作が変わります。型を理解しないと、意図しない結果になります。

### 整数型（int）

小数点を持たない数値です。正の数、負の数、ゼロを扱えます。

```python
age = 25
temperature = -3
count = 0
population = 126_000_000   # _で桁区切り（読みやすくなるだけで値は同じ）
```

> **📖 Pythonの特徴**: Pythonの整数には**サイズの上限がありません**。
> C言語やJavaでは32ビット整数（約21億）が上限ですが、
> Pythonでは `10 ** 1000` のような巨大な数値も扱えます。

### 浮動小数点数型（float）

小数点を持つ数値です。

```python
height = 170.5
pi = 3.14159265
ratio = 0.75
scientific = 1.5e3    # 1.5 × 10³ = 1500.0（科学的記法）
```

> **⚠️ 落とし穴：浮動小数点の誤差**
>
> ```python
> print(0.1 + 0.2)    # 0.30000000000000004 ← 0.3 にならない！
> ```
>
> これはPythonのバグではなく、コンピュータが小数を2進数で表現する際の限界です。
> 金額計算など正確さが必要な場面では `decimal` モジュール（第10章）を使います。
>
> ```python
> from decimal import Decimal
> print(Decimal("0.1") + Decimal("0.2"))  # 0.3（正確）
> ```

### 文字列型（str）

文字の並びです。**シングルクォート** `'` または **ダブルクォート** `"` で囲みます。

```python
name = "太郎"
greeting = 'Hello, World!'
empty = ""                  # 空の文字列（文字がゼロ個）
```

どちらのクォートでも同じですが、プロジェクト内では統一するのが一般的です。

#### クォートの使い分け

```python
# 文字列中にシングルクォートを含むとき → ダブルクォートで囲む
message = "I'm a student"

# 文字列中にダブルクォートを含むとき → シングルクォートで囲む
message = 'He said "Hello"'

# 両方含むとき → エスケープ文字 \ を使う
message = "I'm a \"student\""

# 複数行の文字列 → トリプルクォート
poem = """
春はあけぼの
やうやう白くなりゆく
山ぎは少しあかりて
"""
```

#### エスケープ文字

| エスケープ | 意味 | 例 |
|-----------|------|-----|
| `\n` | 改行 | `"Hello\nWorld"` → 2行に分かれる |
| `\t` | タブ | `"Name\tAge"` → タブで区切る |
| `\\` | バックスラッシュ自体 | `"C:\\Users"` |
| `\'` | シングルクォート | `'I\'m'` |
| `\"` | ダブルクォート | `"He said \"Hi\""` |

#### 文字列の連結と繰り返し

```python
# 連結（+）
first_name = "田中"
last_name = "太郎"
full_name = first_name + last_name    # "田中太郎"

# 繰り返し（*）
line = "-" * 30      # "------------------------------"
print(line)
```

#### f文字列（フォーマット文字列）★重要★

Python 3.6以降で使える、最も読みやすい文字列フォーマット方法です。

```python
name = "太郎"
age = 20
height = 170.5

# f文字列：波括弧 {} の中に変数や式を直接書ける
print(f"{name}さんは{age}歳です")              # 太郎さんは20歳です
print(f"来年は{age + 1}歳になります")          # 来年は21歳になります
print(f"身長は{height:.1f}cmです")             # 身長は170.5cmです
print(f"身長は{height:.0f}cmです")             # 身長は171cmです（四捨五入）
```

**f文字列のフォーマット指定子**（よく使うもの）：

| 指定子 | 意味 | 例 | 結果 |
|--------|------|-----|------|
| `:.2f` | 小数点以下2桁 | `f"{3.14159:.2f}"` | `3.14` |
| `:.0f` | 小数点なし | `f"{3.7:.0f}"` | `4` |
| `:,` | 桁区切りカンマ | `f"{1000000:,}"` | `1,000,000` |
| `:>10` | 右寄せ（幅10） | `f"{'abc':>10}"` | `       abc` |
| `:<10` | 左寄せ（幅10） | `f"{'abc':<10}"` | `abc       ` |
| `:^10` | 中央寄せ（幅10） | `f"{'abc':^10}"` | `   abc    ` |

> **📖 歴史メモ**: f文字列以前は `%` 演算子や `.format()` メソッドが使われていました。
> ```python
> # 古い書き方（今でも動くが、f文字列の方が読みやすい）
> print("{}さんは{}歳です".format(name, age))
> print("%sさんは%d歳です" % (name, age))
> ```

### 真偽値型（bool）

`True`（真）と `False`（偽）の2つだけを持つ型です。条件判断の結果を表します。

```python
is_student = True
is_raining = False

# 比較の結果はboolになる
print(10 > 5)     # True
print(10 < 5)     # False
print(10 == 10)   # True
```

> **📖 True/Falseの正体**: Pythonでは `True` は `1`、`False` は `0` として扱えます。
> ```python
> print(True + True)   # 2
> print(True * 10)     # 10
> print(False + 1)     # 1
> ```
> これが役立つ場面は第2章で詳しく扱います。

### None型

「何もない」ことを表す特別な値です。

```python
result = None
print(result)       # None
print(type(result)) # <class 'NoneType'>
```

「まだ値が決まっていない」変数の初期値としてよく使います。

### ✏️ やってみよう

以下のコードの出力を予想してから、実際に実行して確認してみましょう：

```python
a = 10
b = "10"
c = 10.0
d = True

print(type(a))
print(type(b))
print(type(c))
print(type(d))
print(a == c)       # ← 予想してみよう
print(a == b)       # ← 予想してみよう
```

## 1.3 型の確認と変換

### 型の確認：`type()`

変数にどんな型のデータが入っているか確認できます。

```python
value = 42
print(type(value))   # <class 'int'>

value = "hello"
print(type(value))   # <class 'str'>
```

デバッグ（バグ探し）のとき、「この変数は何型だろう？」と確認するのに便利です。

### 型の判定：`isinstance()`

特定の型かどうかを `True` / `False` で判定します（`type()` より柔軟です）。

```python
age = 25
print(isinstance(age, int))     # True
print(isinstance(age, str))     # False
print(isinstance(age, (int, float)))  # True（intかfloatのどちらか）
```

### 型変換（キャスト）

異なる型の間でデータを変換します。

```
    str("42")          int("42")
  int ──────→ str    str ──────→ int
   42         "42"    "42"         42

    str(3.14)          float("3.14")
  float ─────→ str   str ──────→ float
   3.14       "3.14"  "3.14"       3.14

              float(42)
  int ──────────→ float
   42              42.0
              int(3.14)
  float ─────────→ int （小数点以下切り捨て）
   3.14             3
```

```python
# 文字列 → 数値（ユーザー入力の処理でよく使う）
age = int("25")          # 25
price = float("99.5")    # 99.5

# 数値 → 文字列（文字列連結の際に必要）
text = str(100)          # "100"
text = str(3.14)         # "3.14"

# float → int（小数点以下を切り捨て）
count = int(3.7)         # 3（四捨五入ではなく切り捨て）
count = int(-3.7)        # -3（0に近い方に切り捨て）

# int → float
value = float(10)        # 10.0

# int → bool
print(bool(0))           # False
print(bool(1))           # True
print(bool(-5))          # True（0以外はTrue）
print(bool(""))          # False（空文字列）
print(bool("hello"))     # True（空でない文字列）
```

> **⚠️ よくあるエラー**
>
> ```python
> num = int("hello")      # ValueError: invalid literal for int()
> num = int("3.14")       # ValueError: 小数文字列は直接intに変換できない
> num = int(float("3.14"))  # ← まずfloatにしてからintにする
> ```

### ✏️ やってみよう

以下のコードはエラーになります。なぜエラーになるか考え、修正してみましょう：

```python
age = input("年齢を入力: ")
next_year_age = age + 1
print(f"来年は{next_year_age}歳です")
```

<details>
<summary>ヒントと答え</summary>

`input()` は常に文字列を返すので、`age` は `str` 型です。
`str + int` はできないため、`int()` で変換が必要です。

```python
age = int(input("年齢を入力: "))
next_year_age = age + 1
print(f"来年は{next_year_age}歳です")
```

</details>

## 1.4 演算子

### 算術演算子

| 演算子 | 名前 | 例 | 結果 | 用途 |
|--------|------|-----|------|------|
| `+` | 加算 | `3 + 2` | `5` | 足し算 |
| `-` | 減算 | `3 - 2` | `1` | 引き算 |
| `*` | 乗算 | `3 * 2` | `6` | 掛け算 |
| `/` | 除算 | `7 / 2` | `3.5` | 割り算（**常にfloat**を返す） |
| `//` | 切り捨て除算 | `7 // 2` | `3` | 商（整数部分だけ） |
| `%` | 剰余 | `7 % 2` | `1` | 余り（偶奇判定に便利） |
| `**` | べき乗 | `3 ** 2` | `9` | 累乗（`pow(3, 2)` と同じ） |

> **⚠️ 注意: `/` は常にfloatを返す**
> ```python
> print(6 / 2)     # 3.0（intではなくfloat）
> print(6 // 2)    # 3（整数が欲しいときはこちら）
> ```

> **📖 実務でよく使うパターン**
> ```python
> # 偶数・奇数の判定
> number = 15
> if number % 2 == 0:
>     print("偶数")
> else:
>     print("奇数")
>
> # 秒を「分:秒」に変換
> total_seconds = 185
> minutes = total_seconds // 60    # 3
> seconds = total_seconds % 60     # 5
> print(f"{minutes}分{seconds}秒") # 3分5秒
> ```

### 演算子の優先順位

数学と同じく、掛け算・割り算が足し算・引き算より先に計算されます。

```
高い ←───────────────────────→ 低い
 **  →  *, /, //, %  →  +, -
```

```python
print(2 + 3 * 4)       # 14（3*4が先）
print((2 + 3) * 4)     # 20（括弧が最優先）
```

迷ったら**括弧 `()` を使いましょう**。読みやすくもなります。

### 複合代入演算子

変数の値を更新するときの省略記法です。

```python
score = 100

score += 10     # score = score + 10 → 110
score -= 20     # score = score - 20 → 90
score *= 2      # score = score * 2  → 180
score /= 3      # score = score / 3  → 60.0
score //= 7     # score = score // 7 → 8.0
score %= 3      # score = score % 3  → 2.0
score **= 4     # score = score ** 4 → 16.0
```

## 1.5 print() による出力

`print()` はプログラムの結果を画面に表示する最も基本的な関数です。

### 基本の使い方

```python
print("Hello, World!")     # 文字列を表示
print(42)                   # 数値を表示
print(3.14)                 # floatを表示
print(True)                 # boolを表示
print()                     # 空行を表示
```

### 複数の値をまとめて出力

```python
name = "太郎"
age = 20

# カンマ区切りで渡す（自動的にスペースで区切られる）
print(name, age)                  # 太郎 20

# 区切り文字を変更（sep引数）
print(name, age, sep=", ")        # 太郎, 20
print(2025, 3, 24, sep="/")       # 2025/3/24
print("A", "B", "C", sep=" → ")   # A → B → C
```

### 改行の制御

```python
# デフォルトでは末尾に改行が入る
print("Hello")    # Hello（改行）
print("World")    # World（改行）

# end引数で末尾の文字を変更
print("Hello", end=" ")   # Hello （改行なし）
print("World")             # World
# 出力: Hello World

# ローディング風の表示
for i in range(5):
    print(".", end="")
print(" Done!")
# 出力: ..... Done!
```

## 1.6 input() による入力

### 基本の使い方

`input()` はユーザーからのキーボード入力を受け取ります。

```python
name = input("名前を入力してください: ")
print(f"こんにちは、{name}さん！")
```

**実行の流れ：**
```
名前を入力してください: 太郎    ← ユーザーが入力してEnter
こんにちは、太郎さん！          ← プログラムの出力
```

### 最重要ポイント：input() は常に文字列を返す

```python
age = input("年齢: ")
print(type(age))        # <class 'str'> ← 数字を入力してもstr！

# 数値として使いたい場合は変換が必要
age = int(input("年齢: "))       # 整数
price = float(input("価格: "))   # 小数
```

> **⚠️ よくある間違い**
> ```python
> # ❌ 数値のつもりが文字列のまま計算してしまう
> price = input("価格: ")     # "1000"（文字列）
> total = price * 1.1          # TypeError!
>
> # ✅ 型変換してから計算する
> price = int(input("価格: "))  # 1000（整数）
> total = price * 1.1            # 1100.0
> ```

---

## よくある間違いとトラブルシューティング

| 症状 | 原因 | 解決法 |
|------|------|--------|
| `SyntaxError: EOL while scanning string` | 文字列のクォートが閉じていない | `"` や `'` の対応を確認 |
| `NameError: name 'xxx' is not defined` | 変数名のスペルミス、未定義の変数を使用 | 変数名を正確に入力 |
| `TypeError: can only concatenate str to str` | 文字列と数値を `+` で連結しようとした | `str()` で変換するかf文字列を使う |
| `TypeError: unsupported operand type(s)` | 型が合わない演算 | `type()` で型を確認し変換する |
| `ValueError: invalid literal for int()` | 数値に変換できない文字列を渡した | 入力値を確認、`float()` を試す |
| `ZeroDivisionError` | 0で割った | 割る前に0でないか確認 |
