## 6.1 オブジェクトとは

オブジェクトは、**キー（プロパティ名）と値のペア**を格納するデータ構造です。

```
オブジェクトのイメージ:

  +----------------------------+
  |  person                    |
  +----------------------------+
  |  name:  "太郎"             |
  |  age:   25                 |
  |  city:  "東京"             |
  |  greet: function() {...}   |
  +----------------------------+
```

```javascript
// オブジェクトの作成（リテラル記法）
const person = {
  name: "太郎",
  age: 25,
  city: "東京"
};

// プロパティへのアクセス
console.log(person.name);    // "太郎"（ドット記法）
console.log(person["age"]);  // 25（ブラケット記法）

// 変数をキーとして使う場合はブラケット記法
const key = "city";
console.log(person[key]); // "東京"
```

### 配列とオブジェクトの使い分け

```
  配列: 順番のあるデータ → インデックスでアクセス
  [値0, 値1, 値2, ...]

  オブジェクト: 名前付きデータ → キーでアクセス
  { キー1: 値1, キー2: 値2, ... }
```

## 6.2 プロパティの追加・変更・削除

```javascript
const user = { name: "太郎" };

// プロパティの追加
user.age = 25;
user.email = "taro@example.com";
console.log(user); // { name: "太郎", age: 25, email: "taro@example.com" }

// プロパティの変更
user.age = 26;
console.log(user.age); // 26

// プロパティの削除
delete user.email;
console.log(user); // { name: "太郎", age: 26 }

// プロパティの存在チェック
console.log("name" in user);    // true
console.log("email" in user);   // false
console.log(user.email);        // undefined

// ★ const で宣言してもプロパティは変更できる
// （変数の再代入はできないが、オブジェクトの中身は変更可能）
const obj = { x: 1 };
obj.x = 2;      // OK
obj.y = 3;      // OK
// obj = {};    // NG: TypeError（再代入は不可）
```

## 6.3 省略記法（ショートハンド）

```javascript
const name = "太郎";
const age = 25;

// プロパティ名と変数名が同じ場合、省略できる
const user = { name, age };
// ↑ const user = { name: name, age: age }; と同じ

console.log(user); // { name: "太郎", age: 25 }

// 計算されたプロパティ名（Computed Property Names）
const prefix = "user";
const config = {
  [`${prefix}Name`]: "太郎",
  [`${prefix}Age`]: 25
};
console.log(config); // { userName: "太郎", userAge: 25 }
```

## 6.4 メソッド

オブジェクトのプロパティに関数を持たせたものを**メソッド**と呼びます。

```javascript
const calculator = {
  // メソッドの省略記法（推奨）
  add(a, b) {
    return a + b;
  },

  // 従来の書き方
  subtract: function(a, b) {
    return a - b;
  }
};

console.log(calculator.add(3, 5));      // 8
console.log(calculator.subtract(10, 3)); // 7
```

### this キーワード

```javascript
const user = {
  name: "太郎",
  age: 25,

  // thisはこのオブジェクト自身を指す
  greet() {
    return `こんにちは、${this.name}です。${this.age}歳です。`;
  },

  // 誕生日メソッド
  birthday() {
    this.age++;
    return `${this.name}は${this.age}歳になりました！`;
  }
};

console.log(user.greet());    // "こんにちは、太郎です。25歳です。"
console.log(user.birthday()); // "太郎は26歳になりました！"
```

## 6.5 オブジェクトの反復処理

```javascript
const scores = {
  math: 85,
  english: 72,
  science: 90,
  japanese: 88
};

// Object.keys() — キーの配列を取得
console.log(Object.keys(scores));
// ["math", "english", "science", "japanese"]

// Object.values() — 値の配列を取得
console.log(Object.values(scores));
// [85, 72, 90, 88]

// Object.entries() — [キー, 値] のペア配列を取得
console.log(Object.entries(scores));
// [["math", 85], ["english", 72], ["science", 90], ["japanese", 88]]

// for...in ループ
for (const subject in scores) {
  console.log(`${subject}: ${scores[subject]}点`);
}

// Object.entries + for...of（推奨パターン）
for (const [subject, score] of Object.entries(scores)) {
  console.log(`${subject}: ${score}点`);
}

// 合計・平均の計算
const values = Object.values(scores);
const total = values.reduce((sum, v) => sum + v, 0);
const average = total / values.length;
console.log(`合計: ${total}点, 平均: ${average}点`);
// 合計: 335点, 平均: 83.75点
```

## 6.6 オブジェクトの分割代入

```javascript
const user = {
  name: "太郎",
  age: 25,
  city: "東京",
  email: "taro@example.com"
};

// 基本の分割代入
const { name, age } = user;
console.log(name); // "太郎"
console.log(age);  // 25

// 別名をつける
const { name: userName, age: userAge } = user;
console.log(userName); // "太郎"

// デフォルト値
const { country = "日本" } = user;
console.log(country); // "日本"（userにcountryがないのでデフォルト値）

// 残余プロパティ（rest）
const { name: n, ...others } = user;
console.log(n);      // "太郎"
console.log(others); // { age: 25, city: "東京", email: "taro@example.com" }

// 関数の引数で分割代入（現場で頻出！）
function printUser({ name, age, city = "不明" }) {
  console.log(`${name}（${age}歳）- ${city}`);
}
printUser(user); // "太郎（25歳）- 東京"
```

## 6.7 スプレッド構文とオブジェクトの操作

```javascript
// シャローコピー（浅いコピー）
const original = { a: 1, b: 2, c: 3 };
const copy = { ...original };
copy.a = 100;
console.log(original.a); // 1（元は変わらない）

// オブジェクトのマージ（結合）
const defaults = { color: "赤", size: "M", quantity: 1 };
const options = { size: "L", quantity: 3 };
const merged = { ...defaults, ...options };
console.log(merged); // { color: "赤", size: "L", quantity: 3 }
// ※ 後から展開したものが優先される

// プロパティの追加・上書き
const user = { name: "太郎", age: 25 };
const updated = { ...user, age: 26, city: "大阪" };
console.log(updated); // { name: "太郎", age: 26, city: "大阪" }
```

### Object.assign（レガシーパターン）

```javascript
// Object.assign(ターゲット, ソース1, ソース2, ...)
const target = { a: 1 };
Object.assign(target, { b: 2 }, { c: 3 });
console.log(target); // { a: 1, b: 2, c: 3 }（targetが変更される）

// 非破壊マージ
const merged = Object.assign({}, { a: 1 }, { b: 2 });
```

## 6.8 オブジェクトの凍結

```javascript
// Object.freeze() — プロパティの追加・変更・削除を禁止
const config = Object.freeze({
  apiUrl: "https://api.example.com",
  timeout: 5000
});

config.apiUrl = "https://other.com"; // 無視される（strictモードではエラー）
config.newProp = "test";             // 無視される
console.log(config.apiUrl); // "https://api.example.com"

// Object.isFrozen() で確認
console.log(Object.isFrozen(config)); // true

// Object.freeze は浅い凍結（ネストしたオブジェクトは変更可能）
const data = Object.freeze({
  user: { name: "太郎" }
});
data.user.name = "花子"; // これは変更できてしまう！
console.log(data.user.name); // "花子"
```

## 6.9 クラス構文（ES6+）

```javascript
class User {
  // コンストラクタ（初期化メソッド）
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  // メソッド
  greet() {
    return `こんにちは、${this.name}です。`;
  }

  // ゲッター
  get info() {
    return `${this.name}（${this.age}歳）`;
  }

  // セッター
  set nickname(value) {
    this._nickname = value;
  }

  get nickname() {
    return this._nickname || this.name;
  }
}

const user = new User("太郎", 25);
console.log(user.greet()); // "こんにちは、太郎です。"
console.log(user.info);    // "太郎（25歳）"

user.nickname = "タロー";
console.log(user.nickname); // "タロー"

// 継承
class Student extends User {
  constructor(name, age, grade) {
    super(name, age); // 親クラスのコンストラクタを呼ぶ
    this.grade = grade;
  }

  greet() {
    return `${super.greet()} ${this.grade}年生です。`;
  }
}

const student = new Student("花子", 18, 3);
console.log(student.greet()); // "こんにちは、花子です。 3年生です。"
console.log(student.info);    // "花子（18歳）"
```

## 6.10 現場で使われるパターン

### オプションオブジェクトパターン

```javascript
// 引数が多い関数はオブジェクトで受け取る
function createUser({ name, age, email, role = "user" }) {
  return { name, age, email, role, createdAt: new Date() };
}

const user = createUser({
  name: "太郎",
  age: 25,
  email: "taro@example.com"
});
console.log(user);
```

### データの変換（配列 <-> オブジェクト）

```javascript
// Object.fromEntries — [キー, 値] ペア配列からオブジェクトを作成
const entries = [["name", "太郎"], ["age", 25], ["city", "東京"]];
const obj = Object.fromEntries(entries);
console.log(obj); // { name: "太郎", age: 25, city: "東京" }

// 実用例: URLクエリパラメータの解析
const query = "name=太郎&age=25&city=東京";
const params = Object.fromEntries(
  query.split("&").map(pair => pair.split("=").map(decodeURIComponent))
);
console.log(params); // { name: "太郎", age: "25", city: "東京" }
```

### オブジェクトの深いコピー

```javascript
// structuredClone()（モダンな方法、Node.js 17+、ブラウザ対応済み）
const original = {
  name: "太郎",
  scores: [85, 90, 78],
  address: { city: "東京", zip: "100-0001" }
};

const deepCopy = structuredClone(original);
deepCopy.scores.push(100);
deepCopy.address.city = "大阪";

console.log(original.scores);       // [85, 90, 78]（変わらない）
console.log(original.address.city); // "東京"（変わらない）
```

## よくある間違い

### 1. thisのコンテキスト喪失

```javascript
const user = {
  name: "太郎",
  greet() {
    return `こんにちは、${this.name}です`;
  }
};

// メソッドを変数に代入するとthisが失われる
const greetFn = user.greet;
// console.log(greetFn()); // "こんにちは、undefinedです" or エラー

// 解決策: bind() で固定
const boundGreet = user.greet.bind(user);
console.log(boundGreet()); // "こんにちは、太郎です"
```

### 2. ドット記法で変数は使えない

```javascript
const key = "name";
const obj = { name: "太郎" };

// NG
// console.log(obj.key); // undefined（"key"というプロパティを探す）

// OK
console.log(obj[key]); // "太郎"
```

### 3. オブジェクトの比較は参照比較

```javascript
const a = { x: 1 };
const b = { x: 1 };
console.log(a === b); // false（異なるオブジェクト）
console.log(a === a); // true（同じ参照）

// 内容で比較するには JSON.stringify を使う（簡易的な方法）
console.log(JSON.stringify(a) === JSON.stringify(b)); // true
```

## ポイントまとめ

- オブジェクトは `{ キー: 値 }` の形式でデータを管理します
- プロパティアクセスはドット記法 `.` とブラケット記法 `[]` の2つがあります
- `Object.keys()`, `Object.values()`, `Object.entries()` で反復処理できます
- 分割代入 `const { name, age } = obj` で簡潔にプロパティを取得できます
- スプレッド構文 `{ ...obj }` でシャローコピーやマージができます
- `const` でもオブジェクトの中身は変更可能です（再代入のみ不可）
- クラス構文はオブジェクトの設計図として使います
- `this` はメソッドが属するオブジェクトを指しますが、コンテキストに注意が必要です
