> **難易度**: ★★★★★（総合）
> **前提知識**: 第1章〜第8章（全章）

## 課題の要件

1. **サイドバー**: デスクトップでは左固定、モバイルでは非表示（ハンバーガーメニューで開閉）
2. **統計カード**: 4枚をGridで配置。デスクトップ4列、タブレット2列、モバイル2列
3. **メインコンテンツ**: チャートエリア（大）+ 注文一覧（小）を横並び。モバイルでは縦積み
4. **テーマ切替**: カスタムプロパティでダーク/ライト対応
5. **アニメーション**: カードのホバーエフェクト、サイドバーの開閉トランジション
6. **疑似要素**: アイコン装飾やバッジに `::before` / `::after` を活用
7. **レスポンシブ**: 3段階のブレークポイント（モバイル / タブレット / デスクトップ）

## ステップガイド

<details>
<summary>ステップ1：カスタムプロパティとグローバルスタイルを定義する</summary>

まずテーマカラーとグローバルリセットを設定します。

```css
:root {
  --sidebar-width: 240px;
  --header-height: 64px;
  --color-bg: #f1f5f9;
  --color-surface: #ffffff;
  --color-text: #1e293b;
  --color-primary: #3b82f6;
  /* ... 他の変数 */
}

[data-theme="dark"] {
  --color-bg: #0f172a;
  --color-surface: #1e293b;
  --color-text: #e2e8f0;
  /* ... */
}
```

</details>

<details>
<summary>ステップ2：サイドバー + メインエリアのレイアウト</summary>

Grid でサイドバーとメインを分割します。

```css
.app {
  display: grid;
  grid-template-columns: var(--sidebar-width) 1fr;
  min-height: 100vh;
}

@media (max-width: 768px) {
  .app {
    grid-template-columns: 1fr;
  }
}
```

</details>

<details>
<summary>ステップ3：モバイルサイドバーの開閉</summary>

チェックボックスハック（checkbox hack）を使えば JavaScript なしでも実現できます。

```css
#sidebar-toggle:checked ~ .sidebar {
  transform: translateX(0);
}
```

</details>

<summary>解答例（改良版）との違い ─ 追加すべき要素</summary>

改良版では以下のポイントを追加できます。

### 1. サイドバーの折りたたみ（アイコンのみ表示）

```css
.sidebar.collapsed {
  width: 64px;
}

.sidebar.collapsed .sidebar-nav li a span {
  display: none;
}

.sidebar.collapsed .sidebar-logo {
  text-align: center;
  font-size: 14px;
}
```

### 2. カードに入場アニメーション

```css
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(16px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.stat-card {
  animation: fadeInUp 0.4s ease forwards;
}

.stat-card:nth-child(2) { animation-delay: 0.1s; }
.stat-card:nth-child(3) { animation-delay: 0.2s; }
.stat-card:nth-child(4) { animation-delay: 0.3s; }
```

### 3. `prefers-color-scheme` の対応

```css
@media (prefers-color-scheme: dark) {
  :root:not([data-theme="light"]) {
    --color-bg: #0f172a;
    --color-surface: #1e293b;
    /* ... ダークモードの色 */
  }
}
```

### 4. コンテナクエリ（container query）の活用

```css
.content {
  container-type: inline-size;
}

@container (max-width: 600px) {
  .stats {
    grid-template-columns: 1fr;
  }
}
```

これらはすべてCSSのみで実現できる、実務で使われるテクニックです。
