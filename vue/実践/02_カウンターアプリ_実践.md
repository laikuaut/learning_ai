# 実践課題02：カウンターアプリ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（Vue.jsの基本）、第2章（リアクティビティ）、第3章（テンプレート構文）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `ref` によるリアクティブデータの操作、`v-on`（`@`）イベントハンドリング、`computed` プロパティの活用を実践する

---

## 完成イメージ

複数のカウンターを管理するアプリです。各カウンターは独立して動作し、全体の合計が表示されます。

```
┌──────────────────────────────────────┐
│    🔢 マルチカウンター              │
├──────────────────────────────────────┤
│                                      │
│  カウンター名: [          ] [追加]   │
│                                      │
│  ┌─────────────────────────────┐    │
│  │ 買い物     [-] 3 [+]  [✗]  │    │
│  ├─────────────────────────────┤    │
│  │ 勉強時間   [-] 5 [+]  [✗]  │    │
│  ├─────────────────────────────┤    │
│  │ 運動       [-] 1 [+]  [✗]  │    │
│  └─────────────────────────────┘    │
│                                      │
│  合計: 9    [全リセット]            │
└──────────────────────────────────────┘
```

---

## 課題の要件

1. カウンター名を入力して新しいカウンターを追加できる
2. 各カウンターは `+` / `-` ボタンで値を増減できる（最小値は 0）
3. 各カウンターに削除ボタンがある
4. 全カウンターの合計値を `computed` で表示する
5. 「全リセット」ボタンで全カウンターの値を 0 にする
6. Vue 3 CDN を使い、ブラウザで開くだけで動作する HTML ファイルとして完成させる

---

## ステップガイド

<details>
<summary>ステップ1：カウンターデータの構造を定義する</summary>

```js
const counters = ref([
  { id: 1, name: "買い物", count: 0 },
  { id: 2, name: "勉強時間", count: 0 },
])
const newName = ref("")
```

各カウンターは `id`、`name`、`count` を持つオブジェクトです。

</details>

<details>
<summary>ステップ2：増減・追加・削除の関数を作る</summary>

```js
function increment(id) {
  const counter = counters.value.find(c => c.id === id)
  if (counter) counter.count++
}

function decrement(id) {
  const counter = counters.value.find(c => c.id === id)
  if (counter && counter.count > 0) counter.count--
}

function addCounter() {
  if (!newName.value.trim()) return
  counters.value.push({ id: Date.now(), name: newName.value.trim(), count: 0 })
  newName.value = ""
}
```

</details>

<details>
<summary>ステップ3：computed で合計を算出する</summary>

```js
const total = computed(() =>
  counters.value.reduce((sum, c) => sum + c.count, 0)
)
```

</details>

          total
        }
      }
    }).mount('#app')

    const { createApp, ref, computed, watch } = Vue

    createApp({
      setup() {
        // localStorage からの復元
        const saved = localStorage.getItem('multiCounters')
        const initial = saved
          ? JSON.parse(saved)
          : [
              { id: 1, name: "買い物", count: 0 },
              { id: 2, name: "勉強時間", count: 0 },
              { id: 3, name: "運動", count: 0 },
            ]

        const counters = ref(initial)
        const newName = ref("")

        // watch で localStorage に自動保存
        watch(counters, (val) => {
          localStorage.setItem('multiCounters', JSON.stringify(val))
        }, { deep: true })

        const increment = (id) => {
          const c = counters.value.find(c => c.id === id)
          if (c) c.count++
        }

        const decrement = (id) => {
          const c = counters.value.find(c => c.id === id)
          if (c && c.count > 0) c.count--
        }

        const addCounter = () => {
          if (!newName.value.trim()) return
          counters.value.push({ id: Date.now(), name: newName.value.trim(), count: 0 })
          newName.value = ""
        }

        const removeCounter = (id) => {
          counters.value = counters.value.filter(c => c.id !== id)
        }

        const resetAll = () => counters.value.forEach(c => { c.count = 0 })

        const total = computed(() => counters.value.reduce((s, c) => s + c.count, 0))

        return { counters, newName, increment, decrement, addCounter, removeCounter, resetAll, total }
      }
    }).mount('#app')
  </script>

**初心者向けとの違い:**
- アロー関数で簡潔に記述
- `watch` で `counters` の変更を監視し、`localStorage` に自動保存（ページリロードしてもデータが残る）
- `-` ボタンを `count === 0` のとき `:disabled` で無効化
- 追加ボタンも空文字のとき `:disabled` で無効化
- CSSのトランジションとホバー効果で使いやすさ向上
