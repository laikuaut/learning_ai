# 実践課題05：タイマーアプリ ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第3章（stateとイベント）、第5章（useEffectと副作用）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `useEffect` でのタイマー管理（setInterval/clearInterval）とクリーンアップを実践する

---

## 完成イメージ

```
===== ポモドーロタイマー =====

      [作業: 25:00]

    ┌──────────────────┐
    │                  │  ← プログレスバー
    └──────────────────┘

    [スタート]  [リセット]

    モード: [作業 25分] [休憩 5分] [長休憩 15分]

    完了セッション: 3回
```

---

## 課題の要件

1. カウントダウンタイマーを実装する（mm:ss表示）
2. スタート/一時停止/リセットボタン
3. 3つのモード（作業25分/休憩5分/長休憩15分）を切り替えられる
4. プログレスバーで残り時間を視覚化する
5. 時間が0になったらアラート表示し、完了回数をカウントする
6. `useEffect` でインターバルのセットアップとクリーンアップを適切に行う

---

## ステップガイド

<details>
<summary>ステップ1：基本のカウントダウンを作る</summary>

```jsx
const [seconds, setSeconds] = React.useState(25 * 60);
const [isRunning, setIsRunning] = React.useState(false);

React.useEffect(() => {
  if (!isRunning) return;

  const timer = setInterval(() => {
    setSeconds(prev => {
      if (prev <= 0) return 0;
      return prev - 1;
    });
  }, 1000);

  // クリーンアップ：コンポーネントのアンマウント時やisRunning変更時にタイマーを解除
  return () => clearInterval(timer);
}, [isRunning]);
```

**重要**: `return () => clearInterval(timer)` がクリーンアップ関数です。これを忘れるとタイマーが重複して動きます。

</details>

<details>
<summary>ステップ2：時間が0になったら停止する</summary>

```jsx
React.useEffect(() => {
  if (seconds === 0 && isRunning) {
    setIsRunning(false);
    setCompletedCount(prev => prev + 1);
    alert("タイマー完了！");
  }
}, [seconds, isRunning]);
```

const MODES = {

<summary>解答例（改良版 ─ useRefでインターバルID管理）</summary>

useEffectのクリーンアップの代わりに `useRef` でインターバルIDを保持する方法です。差分のみ示します。

```jsx
function App() {
  const [mode, setMode] = React.useState("work");
  const [seconds, setSeconds] = React.useState(MODES.work.minutes * 60);
  const [isRunning, setIsRunning] = React.useState(false);
  const [completedCount, setCompletedCount] = React.useState(0);
  const timerRef = React.useRef(null);

  // useRefでタイマーを管理するパターン
  const startTimer = () => {
    if (timerRef.current) return; // 重複防止
    timerRef.current = setInterval(() => {
      setSeconds(prev => {
        if (prev <= 1) {
          clearInterval(timerRef.current);
          timerRef.current = null;
          setIsRunning(false);
          setCompletedCount(c => c + 1);
          return 0;
        }
        return prev - 1;
      });
    }, 1000);
  };

  const stopTimer = () => {
    if (timerRef.current) {
      clearInterval(timerRef.current);
      timerRef.current = null;
    }
  };

  // isRunning の変化に応じてタイマーを制御
  React.useEffect(() => {
    if (isRunning) {
      startTimer();
    } else {
      stopTimer();
    }
    return stopTimer; // クリーンアップ
  }, [isRunning]);

  // ...
}
```

**初心者向けとの違い:**
- `useRef` でインターバルIDを保持 → 複数のuseEffectから参照可能
- タイマー完了処理をsetInterval内で直接実行（別useEffect不要）
- 重複起動防止ロジック

</details>
