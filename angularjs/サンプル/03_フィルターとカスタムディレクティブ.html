フィルターとカスタムディレクティブ

AngularJSの強力なフィルター機能とディレクティブを体験しましょう

組み込みフィルター filter | currency | date

currency フィルター
{{ 29800 | currency:'¥':0 }}
date フィルター
{{ today | date:'yyyy年MM月dd日' }}
uppercase フィルター
{{ 'angularjs' | uppercase }}
カスタム truncate フィルター
{{ 'AngularJSのフィルターは非常に強力です' | truncate:12 }}

カスタムディレクティブ collapsible-panel

クリックで開閉する再利用可能なパネルコンポーネントです。

フィルターはテンプレート内でデータを整形する仕組みです。パイプ（|）記号を使って式に適用します。

{{ expression | filterName:arg1:arg2 }}

ディレクティブはHTMLを拡張する仕組みです。独自のHTML要素や属性を定義できます。

要素（E）、属性（A）、クラス（C）、コメント（M）として使用できます。

truncate フィルター: 長いテキストを指定文字数で切り詰めて「...」を追加します。

collapsible-panel ディレクティブ: transclude を使って、パネル内のコンテンツを自由に埋め込めるようにしています。
