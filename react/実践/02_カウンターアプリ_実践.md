# 実践課題02：カウンターアプリ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（Reactの基本）、第3章（stateとイベント）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `useState` によるstate管理とイベントハンドラの基本を実践する

---

## 完成イメージ

```
        ===== カウンター =====

          +1    +5    +10

              [ 23 ]

          -1    -5    -10

            [リセット]

        履歴: 0 → 1 → 6 → 16 → 15 → 23
```

---

## 課題の要件

1. 数値を表示するカウンターを作成する
2. +1, +5, +10, -1, -5, -10 のボタンで値を増減する
3. リセットボタンで0に戻す
4. カウンターの値が0未満にならないようにする（最小値0）
5. 値の変更履歴を配列で保持し、画面下部に表示する

---

## ステップガイド

<details>
<summary>ステップ1：useState でカウント値を管理する</summary>

```jsx
const [count, setCount] = React.useState(0);
```

ボタンクリックで `setCount` を呼んで値を更新します。

</details>

<details>
<summary>ステップ2：履歴を別のstateで管理する</summary>

```jsx
const [history, setHistory] = React.useState([0]);

function handleChange(delta) {
  const newCount = Math.max(0, count + delta);
  setCount(newCount);
  setHistory([...history, newCount]);
}
```

</details>

<details>
<summary>ステップ3：ボタンをコンポーネント化する</summary>

```jsx
function CountButton({ label, delta, onClick }) {
  return <button onClick={() => onClick(delta)}>{label}</button>;
}
```

</details>

<summary>解答例（改良版 ─ ボタン定義をデータ化）</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>カウンターアプリ（改良版）</title>
  <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
  <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
  <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
  <style>
    body { font-family: sans-serif; text-align: center; padding: 40px; background: #1a1a2e; color: white; }
    .display { font-size: 72px; font-weight: bold; margin: 24px 0; }
    .btn-group { margin: 8px 0; display: flex; justify-content: center; gap: 8px; }
    .btn { font-size: 16px; padding: 12px 24px; border: none; border-radius: 8px;
           cursor: pointer; color: white; font-weight: bold; transition: transform 0.1s; }
    .btn:hover { transform: scale(1.05); }
    .btn:active { transform: scale(0.95); }
    .btn-inc { background: #00b894; }
    .btn-dec { background: #d63031; }
    .btn-reset { background: #636e72; padding: 8px 32px; margin-top: 16px; }
    .history { margin-top: 24px; color: #aaa; font-size: 13px; max-width: 400px; margin-left: auto; margin-right: auto; word-wrap: break-word; }
    .min-warning { color: #fdcb6e; font-size: 14px; margin-top: 8px; }
  </style>
</head>
<body>
  <div id="root"></div>
  <script type="text/babel">
    const BUTTON_CONFIG = [
      { delta: 1, label: "+1" },
      { delta: 5, label: "+5" },
      { delta: 10, label: "+10" },
    ];

    function App() {
      const [count, setCount] = React.useState(0);
      const [history, setHistory] = React.useState([0]);

      const handleChange = (delta) => {
        const newCount = Math.max(0, count + delta);
        if (newCount !== count) {
          setCount(newCount);
          setHistory((prev) => [...prev, newCount]);
        }
      };

      const handleReset = () => {
        if (count !== 0) {
          setCount(0);
          setHistory((prev) => [...prev, 0]);
        }
      };

      const atMinimum = count === 0;

      return (
        <div>
          <h1>カウンター</h1>

          <div className="btn-group">
            {BUTTON_CONFIG.map(({ delta, label }) => (
              <button key={delta} className="btn btn-inc" onClick={() => handleChange(delta)}>
                {label}
              </button>
            ))}
          </div>

          <div className="display">{count}</div>

          <div className="btn-group">
            {BUTTON_CONFIG.map(({ delta, label }) => (
              <button
                key={-delta}
                className="btn btn-dec"
                onClick={() => handleChange(-delta)}
                disabled={atMinimum}
                style={{ opacity: atMinimum ? 0.4 : 1 }}
              >
                {label.replace("+", "-")}
              </button>
            ))}
          </div>

          {atMinimum && <div className="min-warning">0より小さくできません</div>}

          <div>
            <button className="btn btn-reset" onClick={handleReset}>リセット</button>
          </div>

          <div className="history">
            履歴 ({history.length - 1}回操作): {history.join(" → ")}
          </div>
        </div>
      );
    }

    ReactDOM.createRoot(document.getElementById("root")).render(<App />);
  </script>
</body>
</html>
```

**初心者向けとの違い:**
- ボタン設定を `BUTTON_CONFIG` 配列で管理 → 重複コード削減
- `setHistory((prev) => [...prev, newCount])` で関数型更新
- 最小値0のときマイナスボタンを `disabled` に
- 同じ値への更新をスキップ（不要な再レンダリング防止）

</details>
