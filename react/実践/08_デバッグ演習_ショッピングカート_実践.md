## 課題の説明

以下の「ショッピングカート」アプリには **8個のバグ** が埋め込まれています。
ブラウザで開いて動作を確認しながら、すべてのバグを見つけて修正してください。

## バグ入りコード

以下を `shopping_cart_buggy.html` として保存し、ブラウザで開いてください。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>ショッピングカート（バグ入り）</title>
  <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
  <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
  <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 600px; margin: 0 auto; padding: 20px; }
    .product { display: flex; align-items: center; padding: 12px; border: 1px solid #ddd; border-radius: 8px; margin-bottom: 8px; }
    .product-info { flex: 1; }
    .cart-item { display: flex; align-items: center; padding: 8px; background: #f9f9f9; border-radius: 6px; margin-bottom: 6px; }
    .cart-item .name { flex: 1; }
    .qty-btn { width: 30px; height: 30px; border: 1px solid #ddd; background: white; cursor: pointer; font-size: 16px; border-radius: 4px; }
    .add-btn { padding: 6px 16px; background: #4caf50; color: white; border: none; border-radius: 6px; cursor: pointer; }
    .total { font-size: 20px; font-weight: bold; text-align: right; margin-top: 16px; }
  </style>
</head>
<body>
  <div id="root"></div>
  <script type="text/babel">
    const PRODUCTS = [
      { id: 1, name: "ノートPC", price: 89000 },
      { id: 2, name: "マウス", price: 3500 },
      { id: 3, name: "キーボード", price: 12000 },
      { id: 4, name: "モニター", price: 45000 },
    ];

    // バグ1: ProductItem の表示
    function ProductItem({ product, onAdd }) {
      return (
        <div className="product">
          <div className="product-info">
            <strong>{product.name}</strong>
            <div>{product.price}円</div>  {/* バグ1 */}
          </div>
          <button className="add-btn" onClick={onAdd(product)}>  {/* バグ2 */}
            カートに追加
          </button>
        </div>
      );
    }

    function CartItem({ item, onIncrement, onDecrement, onRemove }) {
      return (
        <div className="cart-item">
          <span className="name">{item.name} ×{item.quantity}</span>
          <span>{item.price * item.quantity}円</span>  {/* バグ3: 小計の計算 */}
          <button className="qty-btn" onClick={() => onDecrement(item.id)}>-</button>
          <span style={{ margin: "0 8px" }}>{item.quantity}</span>
          <button className="qty-btn" onClick={() => onIncrement(item.id)}>+</button>
          <button style={{ marginLeft: 8, color: "red", border: "none", background: "none", cursor: "pointer" }}
                  onClick={() => onRemove(item.id)}>✗</button>
        </div>
      );
    }

    function App() {
      const [cart, setCart] = React.useState([]);

      function handleAdd(product) {
        var existing = cart.find(item => item.id === product.id);
        if (existing) {
          // バグ4: stateの直接変更
          existing.quantity += 1;
          setCart(cart);
        } else {
          setCart([...cart, { id: product.id, name: product.name, price: product.price, quantity: 1 }]);
        }
      }

      function handleIncrement(id) {
        setCart(cart.map(item =>
          item.id === id ? { ...item, quantity: item.quantity + 1 } : item
        ));
      }

      function handleDecrement(id) {
        setCart(cart.map(item => {
          if (item.id === id) {
            return { ...item, quantity: item.quantity - 1 };  // バグ5: 0以下になる
          }
          return item;
        }));
      }

      function handleRemove(id) {
        setCart(cart.filter(item => item.id === id));  // バグ6: フィルタ条件が逆
      }

      // バグ7: 合計金額の計算
      var total = 0;
      for (var i = 0; i < cart.length; i++) {
        total += cart[i].price;
      }

      // バグ8: useEffect の使い方
      React.useEffect(() => {
        document.title = "カート（" + cart.length + "種類）";
      });

      return (
        <div>
          <h1>ショッピングカート</h1>

          <h2>商品一覧</h2>
          {PRODUCTS.map(product => (
            <ProductItem
              key={product.id}
              product={product}
              onAdd={handleAdd}
            />
          ))}

          <h2>カート ({cart.length}種類)</h2>
          {cart.length === 0 ? (
            <p style={{ color: "#999" }}>カートは空です</p>
          ) : (
            cart.map(item => (
              <CartItem
                item={item}
                onIncrement={handleIncrement}
                onDecrement={handleDecrement}
                onRemove={handleRemove}
              />  /* バグ: keyなし（CartItemのmap） */
            ))
          )}

          <div className="total">合計: {total.toLocaleString()}円</div>
        </div>
      );
    }

    ReactDOM.createRoot(document.getElementById("root")).render(<App />);
  </script>
</body>
</html>
```

## バグの種類

<details>
<summary>ヒント（軽め）：バグのカテゴリ</summary>

| カテゴリ | 件数 |
|---|:---:|
| イベントハンドラの書き方 | 1 |
| stateの不変性違反 | 1 |
| ロジックエラー | 3 |
| React特有の警告（key等） | 1 |
| useEffectの依存配列 | 1 |
| 表示の問題 | 1 |

<details>
<summary>ヒント（詳細）：各バグの場所</summary>

| # | 場所 | ヒント |
|---|---|---|
| 1 | ProductItem の価格表示 | 金額がカンマ区切りでない |
| 2 | ProductItem の onClick | 関数がレンダリング時に実行されてしまう |
| 3 | CartItem の小計 | 小計が2回表示されている（名前欄と金額欄） → 名前欄から金額を消す必要 |
| 4 | handleAdd の既存商品更新 | stateオブジェクトを直接変更している |
| 5 | handleDecrement | 数量が0以下になれる |
| 6 | handleRemove のフィルタ | 条件が逆（削除対象を残してしまう） |
| 7 | total の計算 | 数量を考慮していない |
| 8 | useEffect | 依存配列がない（毎レンダリングで実行される） |

※ cart.map の key なしも修正が必要です。

</details>

## 解答例

<details>
<summary>全バグの修正</summary>

### バグ1：価格表示のフォーマット
```jsx
// 修正前
<div>{product.price}円</div>
// 修正後
<div>{product.price.toLocaleString()}円</div>
```

### バグ2：onClickが即座に実行される
```jsx
// 修正前 — onAdd(product) がレンダリング時に実行される
<button onClick={onAdd(product)}>
// 修正後 — アロー関数で包む
<button onClick={() => onAdd(product)}>
```

### バグ3：CartItem の重複表示
```jsx
// 修正前 — 名前欄に×数量と金額欄に小計が両方ある
<span className="name">{item.name} ×{item.quantity}</span>
<span>{item.price * item.quantity}円</span>
// 修正後 — 名前欄はシンプルに
<span className="name">{item.name}</span>
<span>{(item.price * item.quantity).toLocaleString()}円</span>
```

### バグ4：stateの直接変更
```jsx
// 修正前 — ミュータブルな変更はReactが検知できない
existing.quantity += 1;
setCart(cart);
// 修正後 — 新しい配列と新しいオブジェクトを作る
setCart(cart.map(item =>
  item.id === product.id ? { ...item, quantity: item.quantity + 1 } : item
));
```

### バグ5：数量が0以下になる
```jsx
// 修正前
return { ...item, quantity: item.quantity - 1 };
// 修正後 — 1未満にしない、または0になったら削除
if (item.id === id) {
  var newQty = item.quantity - 1;
  if (newQty <= 0) return null; // 後でfilterで除去
  return { ...item, quantity: newQty };
}
// ＋ setCart の後に .filter(Boolean) を付ける
// または Math.max(1, item.quantity - 1) で最低1個にする
```

### バグ6：フィルタ条件が逆
```jsx
// 修正前 — 削除対象だけを残してしまう
setCart(cart.filter(item => item.id === id));
// 修正後 — 削除対象以外を残す
setCart(cart.filter(item => item.id !== id));
```

### バグ7：合計金額で数量を考慮していない
```jsx
// 修正前
total += cart[i].price;
// 修正後
total += cart[i].price * cart[i].quantity;
```

### バグ8：useEffectの依存配列がない
```jsx
// 修正前 — 毎レンダリングで実行される
React.useEffect(() => { ... });
// 修正後 — cartが変わったときだけ実行
React.useEffect(() => { ... }, [cart]);
```

### 追加修正：CartItemのmap にkeyがない
```jsx
// 修正前
cart.map(item => (<CartItem item={item} ... />))
// 修正後
cart.map(item => (<CartItem key={item.id} item={item} ... />))
```

</details>
