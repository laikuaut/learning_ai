# 第2章：条件分岐 - 演習問題

この演習では、`if`、`else if`、`else`、`switch` 文を使った条件分岐を実践します。

---

## 基本問題

### 問題1：正負判定

整数を1つ入力し、その値が「正の数」「負の数」「ゼロ」のいずれかを判定して表示するプログラムを作成してください。

**期待される出力例：**
```
整数を入力してください: -7
-7 は負の数です。
```

```
整数を入力してください: 0
0 はゼロです。
```

<details>
<summary>ヒント</summary>

- `if (num > 0)` で正の数、`else if (num < 0)` で負の数、`else` でゼロを判定します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int num;

    printf("整数を入力してください: ");
    scanf("%d", &num);

    /* 3つの条件を if-else if-else で分岐 */
    if (num > 0) {
        printf("%d は正の数です。\n", num);
    } else if (num < 0) {
        printf("%d は負の数です。\n", num);
    } else {
        printf("%d はゼロです。\n", num);
    }

    return 0;
}
```

</details>

### 問題2：偶数奇数判定

整数を1つ入力し、偶数か奇数かを判定して表示するプログラムを作成してください。

**期待される出力例：**
```
整数を入力してください: 42
42 は偶数です。
```

```
整数を入力してください: 15
15 は奇数です。
```

<details>
<summary>ヒント</summary>

- 偶数かどうかは「2で割った余りが0かどうか」で判定できます
- 剰余演算子 `%` を使います

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int num;

    printf("整数を入力してください: ");
    scanf("%d", &num);

    /* 2 で割った余りが 0 なら偶数、それ以外は奇数 */
    if (num % 2 == 0) {
        printf("%d は偶数です。\n", num);
    } else {
        printf("%d は奇数です。\n", num);
    }

    return 0;
}
```

</details>

### 問題3：成績判定

テストの点数（0〜100）を入力し、以下の基準で成績を判定して表示するプログラムを作成してください。

| 点数 | 成績 |
|---|---|
| 90〜100 | A |
| 80〜89 | B |
| 70〜79 | C |
| 60〜69 | D |
| 0〜59 | F |

**期待される出力例：**
```
点数を入力してください(0-100): 85
成績: B
```

```
点数を入力してください(0-100): 42
成績: F
```

<details>
<summary>ヒント</summary>

- `if-else if` を上の条件から順に書いていきます
- 範囲外の入力（0未満や100超）のエラーチェックも加えるとより良いプログラムになります

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int score;

    printf("点数を入力してください(0-100): ");
    scanf("%d", &score);

    /* 入力値の範囲チェック */
    if (score < 0 || score > 100) {
        printf("エラー: 0〜100の範囲で入力してください。\n");
    } else if (score >= 90) {
        /* 90点以上はA */
        printf("成績: A\n");
    } else if (score >= 80) {
        /* 80点以上（90未満は上で処理済み）はB */
        printf("成績: B\n");
    } else if (score >= 70) {
        printf("成績: C\n");
    } else if (score >= 60) {
        printf("成績: D\n");
    } else {
        /* 59点以下はF */
        printf("成績: F\n");
    }

    return 0;
}
```

**ポイント：** `else if` は上の条件が偽の場合にのみ評価されるため、`score >= 80` の時点で「90未満である」ことが確定しています。

</details>

## 応用問題

### 問題4：うるう年判定

西暦年を入力し、その年がうるう年かどうかを判定するプログラムを作成してください。

**うるう年の条件：**
1. 4で割り切れる年はうるう年
2. ただし、100で割り切れる年はうるう年ではない
3. ただし、400で割り切れる年はうるう年

**期待される出力例：**
```
西暦を入力してください: 2024
2024年はうるう年です。
```

```
西暦を入力してください: 1900
1900年はうるう年ではありません。
```

```
西暦を入力してください: 2000
2000年はうるう年です。
```

<details>
<summary>ヒント</summary>

- 条件を整理すると「400で割り切れる」または「4で割り切れ、かつ100で割り切れない」場合にうるう年です
- 論理演算子 `&&`（AND）と `||`（OR）を使います

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int year;

    printf("西暦を入力してください: ");
    scanf("%d", &year);

    /*
     * うるう年の判定条件:
     *   400で割り切れる → うるう年
     *   100で割り切れる → うるう年ではない
     *   4で割り切れる   → うるう年
     *   それ以外       → うるう年ではない
     *
     * まとめると:
     *   (400で割り切れる) || (4で割り切れる && 100で割り切れない)
     */
    if (year % 400 == 0 || (year % 4 == 0 && year % 100 != 0)) {
        printf("%d年はうるう年です。\n", year);
    } else {
        printf("%d年はうるう年ではありません。\n", year);
    }

    return 0;
}
```

</details>

### 問題5：じゃんけんプログラム

ユーザーが手を選び（0:グー、1:チョキ、2:パー）、コンピュータの手と比較して勝敗を表示するプログラムを作成してください。コンピュータの手は固定値（例えば `1`）でかまいません。`switch` 文を使って手の名前を表示してください。

**期待される出力例：**
```
じゃんけん！(0:グー 1:チョキ 2:パー): 0
あなた: グー
コンピュータ: チョキ
あなたの勝ちです！
```

<details>
<summary>ヒント</summary>

- `switch` 文で `case 0:` → グー、`case 1:` → チョキ、`case 2:` → パー と変換します
- 勝敗判定は `if` 文で行います
- 勝ちの条件は「(自分 - 相手 + 3) % 3 == 2」で一般化できます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int player, computer;
    /* コンピュータの手を固定（1 = チョキ） */
    computer = 1;

    printf("じゃんけん！(0:グー 1:チョキ 2:パー): ");
    scanf("%d", &player);

    /* switch 文で手の名前を表示 */
    printf("あなた: ");
    switch (player) {
        case 0: printf("グー\n"); break;
        case 1: printf("チョキ\n"); break;
        case 2: printf("パー\n"); break;
        default: printf("不正な入力\n"); return 1;
    }

    printf("コンピュータ: ");
    switch (computer) {
        case 0: printf("グー\n"); break;
        case 1: printf("チョキ\n"); break;
        case 2: printf("パー\n"); break;
    }

    /* 勝敗判定 */
    if (player == computer) {
        printf("あいこです！\n");
    } else if ((player - computer + 3) % 3 == 2) {
        /*
         * 勝ちパターンを数式で一般化:
         * グー(0) vs チョキ(1): (0-1+3)%3 = 2 → 勝ち
         * チョキ(1) vs パー(2): (1-2+3)%3 = 2 → 勝ち
         * パー(2) vs グー(0): (2-0+3)%3 = 2 → 勝ち
         */
        printf("あなたの勝ちです！\n");
    } else {
        printf("あなたの負けです...\n");
    }

    return 0;
}
```

**発展：** `rand()` 関数を使えばコンピュータの手をランダムにできます。`#include <stdlib.h>` と `#include <time.h>` を追加し、`srand(time(NULL));` で初期化してから `computer = rand() % 3;` とします。

</details>

## チャレンジ問題

### 問題6：簡易電卓

2つの実数と演算子（`+`, `-`, `*`, `/`）を入力し、計算結果を表示する電卓プログラムを作成してください。`switch` 文を使って演算子ごとに処理を分岐させてください。ゼロ除算のチェックも行ってください。

**期待される出力例：**
```
1つ目の数を入力: 12.5
演算子を入力(+, -, *, /): *
2つ目の数を入力: 4.0
12.50 * 4.00 = 50.00
```

```
1つ目の数を入力: 10.0
演算子を入力(+, -, *, /): /
2つ目の数を入力: 0
エラー: 0で割ることはできません。
```

<details>
<summary>ヒント</summary>

- 演算子は `char` 型で受け取ります
- `scanf(" %c", &op);` のように `%c` の前にスペースを入れると、改行文字をスキップできます
- `switch` 文は整数型だけでなく `char` 型にも使えます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    double num1, num2, result;
    char op;

    printf("1つ目の数を入力: ");
    scanf("%lf", &num1);

    /*
     * %c の前にスペースを入れることで、
     * 前の入力で残った改行文字(\n)をスキップする
     */
    printf("演算子を入力(+, -, *, /): ");
    scanf(" %c", &op);

    printf("2つ目の数を入力: ");
    scanf("%lf", &num2);

    /* switch 文で演算子ごとに処理を分岐 */
    switch (op) {
        case '+':
            result = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, result);
            break;
        case '-':
            result = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, result);
            break;
        case '*':
            result = num1 * num2;
            printf("%.2f * %.2f = %.2f\n", num1, num2, result);
            break;
        case '/':
            /* ゼロ除算のチェック */
            if (num2 == 0) {
                printf("エラー: 0で割ることはできません。\n");
            } else {
                result = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, result);
            }
            break;
        default:
            /* 対応していない演算子が入力された場合 */
            printf("エラー: '%c' は対応していない演算子です。\n", op);
            break;
    }

    return 0;
}
```

**よくある間違い：** `switch` 文で `break` を書き忘れると、次の `case` にそのまま処理が流れてしまいます（フォールスルー）。意図的でない限り、各 `case` の末尾に必ず `break` を入れましょう。

</details>
