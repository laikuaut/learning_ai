# 第4章：アルゴリズム 演習問題

## 基本問題

### 問題1：計算量
以下のアルゴリズムの計算量を O 記法で答えてください。

(1) 配列のインデックスによる要素アクセス
(2) 線形探索
(3) 二分探索
(4) バブルソート

<details>
<summary>解答</summary>

(1) **O(1)** — インデックスで直接アクセスするため定数時間
(2) **O(n)** — 最悪の場合、全要素を調べる
(3) **O(log n)** — 探索範囲を毎回半分に絞る
(4) **O(n²)** — 二重ループで全ペアを比較
</details>

---

### 問題2：二分探索のトレース
ソート済み配列 A = [3, 7, 12, 18, 25, 30, 45] で、二分探索を使って key = 18 を探す手順をトレースしてください。

<details>
<summary>解答</summary>

```
手順1: left=0, right=6, mid=(0+6)/2=3
       A[3]=18 = key → 発見！
```

答え：**1回の比較で位置 3 に発見**

この例では中央の要素がちょうど探索キーと一致しました。
</details>

---

### 問題3：二分探索の最大比較回数
データ数が 500 件のソート済み配列に対して二分探索を行う場合、最大何回の比較が必要ですか。

<details>
<summary>ヒント</summary>
2^k ≧ 500 を満たす最小の k を求めます。2⁸ = 256, 2⁹ = 512 です。
</details>

<details>
<summary>解答</summary>

```
2⁸ = 256 < 500
2⁹ = 512 ≧ 500
```

答え：最大 **9回**
</details>

### 問題4：四択問題
ソートアルゴリズムのうち、平均計算量が O(n log n) であるが、最悪計算量が O(n²) となるものはどれですか。

ア. マージソート　　イ. ヒープソート　　ウ. クイックソート　　エ. バブルソート

<details>
<summary>解答</summary>

答え：**ウ. クイックソート**

クイックソートは平均 O(n log n) ですが、ピボットの選び方が悪い場合（例：ソート済みデータで先頭をピボットに選ぶ）に最悪 O(n²) になります。マージソートとヒープソートは最悪でも O(n log n) です。
</details>

---

### 問題5：安定ソート
以下のソートアルゴリズムのうち、安定なソートをすべて選んでください。

バブルソート、選択ソート、挿入ソート、クイックソート、マージソート、ヒープソート

<details>
<summary>解答</summary>

安定なソート：**バブルソート、挿入ソート、マージソート**

安定ソートとは、同じ値を持つ要素のソート前後の順序が保たれるソートです。
</details>

## 応用問題

### 問題6：バブルソートのトレース
配列 [6, 2, 8, 3] をバブルソートで昇順にソートするとき、最初のパス（i=0）での各比較・交換の過程を示してください。

<details>
<summary>解答</summary>

```
初期状態: [6, 2, 8, 3]

パス1（i=0）：右端から左に比較していく
j=3: A[2]=8 と A[3]=3 比較 → 8 > 3 なので交換 → [6, 2, 3, 8]
j=2: A[1]=2 と A[2]=3 比較 → 2 < 3 なので交換なし → [6, 2, 3, 8]
j=1: A[0]=6 と A[1]=2 比較 → 6 > 2 なので交換 → [2, 6, 3, 8]

パス1完了後: [2, 6, 3, 8]
→ 最小値 2 が先頭に確定
```
</details>

### 問題7：選択ソートのトレース
配列 [4, 1, 7, 3, 5] を選択ソートで昇順にソートする全過程を示してください。

<details>
<summary>解答</summary>

```
初期状態: [4, 1, 7, 3, 5]

i=0: 最小値=A[1]=1 → A[0]と交換 → [1, 4, 7, 3, 5]
i=1: 最小値=A[3]=3 → A[1]と交換 → [1, 3, 7, 4, 5]
i=2: 最小値=A[3]=4 → A[2]と交換 → [1, 3, 4, 7, 5]
i=3: 最小値=A[4]=5 → A[3]と交換 → [1, 3, 4, 5, 7]

ソート完了: [1, 3, 4, 5, 7]
```
</details>

### 問題8：再帰のトレース
以下の再帰関数 f(n) について、f(4) の値を求めてください。呼び出しの過程も示してください。

```
function f(n):
    if n ≤ 1:
        return 1
    else:
        return n + f(n - 2)
```

<details>
<summary>解答</summary>

```
f(4) = 4 + f(2)
f(2) = 2 + f(0)
f(0) = 1（基底条件: n ≤ 1）

戻り：
f(0) = 1
f(2) = 2 + 1 = 3
f(4) = 4 + 3 = 7
```
答え：**f(4) = 7**
</details>

### 問題9：ユークリッドの互除法
ユークリッドの互除法を使って gcd(56, 21) を求めてください。計算過程を示してください。

<details>
<summary>解答</summary>

```
gcd(56, 21) → gcd(21, 56 mod 21) = gcd(21, 14)
gcd(21, 14) → gcd(14, 21 mod 14) = gcd(14, 7)
gcd(14, 7)  → gcd(7, 14 mod 7)   = gcd(7, 0)
gcd(7, 0)   → b=0 なので return 7
```
答え：**gcd(56, 21) = 7**
</details>

### 問題10：四択問題（ソートの特徴）
「ほぼ整列済みのデータに対して最も効率的に動作する」ソートアルゴリズムはどれですか。

ア. 選択ソート　　イ. バブルソート　　ウ. 挿入ソート　　エ. ヒープソート

<details>
<summary>解答</summary>

答え：**ウ. 挿入ソート**

挿入ソートはほぼ整列済みのデータに対して O(n) で動作します。各要素の挿入位置がすぐ見つかるため、移動がほとんど発生しません。
</details>

## チャレンジ問題

### 問題11：クイックソートのトレース
配列 [5, 3, 8, 1, 4, 7, 2, 6] に対してクイックソートを適用します。ピボットを配列の中央の値とした場合、最初の分割結果を示してください。

<details>
<summary>ヒント</summary>
配列の中央のインデックスは (0+7)/2 = 3。A[3] = 1 がピボット...ではなく、「中央の値」は配列の要素を考慮してください。ここでは mid=(0+7)/2=3、ピボット=A[3]=1 とします。
</details>

<details>
<summary>解答</summary>

```
配列: [5, 3, 8, 1, 4, 7, 2, 6]
mid = (0+7)/2 = 3、ピボット = A[3] = 1

i=0(左端), j=7(右端)

A[i]=5 ≧ 1, A[j]=6 > 1 → j-- → j=6
A[j]=2 > 1 → j-- → j=5
A[j]=7 > 1 → j-- → j=4
A[j]=4 > 1 → j-- → j=3
A[j]=1 ≦ 1 → swap(A[0], A[3]) → [1, 3, 8, 5, 4, 7, 2, 6]
i=1, j=2

左グループ: [1]（ピボット以下）
右グループ: [3, 8, 5, 4, 7, 2, 6]（ピボットより大きい）
```

ピボットの選び方により分割結果は変わりますが、ピボット以下とピボット超のグループに分かれる点は共通です。
</details>

### 問題12：アルゴリズムのパターン識別
以下のアルゴリズムは「分割統治法」「貪欲法」「動的計画法」のどれに該当しますか。

(1) マージソート
(2) つり銭問題で大きい硬貨から使う方法
(3) フィボナッチ数列の計算結果を表に記録して重複計算を避ける方法

<details>
<summary>解答</summary>

(1) **分割統治法** — 配列を半分に分割し、それぞれをソートしてからマージする
(2) **貪欲法** — 各ステップで「その時点で最善」の選択（最大額の硬貨）を行う
(3) **動的計画法** — 部分問題の解を記録（メモ化）して重複計算を避ける
</details>

### 問題13：ソートアルゴリズムの総合比較
以下の表を完成させてください。

| アルゴリズム | 平均計算量 | 最悪計算量 | 安定性 |
|-------------|-----------|-----------|--------|
| バブルソート | ? | ? | ? |
| 挿入ソート | ? | ? | ? |
| クイックソート | ? | ? | ? |
| マージソート | ? | ? | ? |

<details>
<summary>解答</summary>

| アルゴリズム | 平均計算量 | 最悪計算量 | 安定性 |
|-------------|-----------|-----------|--------|
| バブルソート | **O(n²)** | **O(n²)** | **安定** |
| 挿入ソート | **O(n²)** | **O(n²)** | **安定** |
| クイックソート | **O(n log n)** | **O(n²)** | **不安定** |
| マージソート | **O(n log n)** | **O(n log n)** | **安定** |

覚え方：安定ソートは「バブル・挿入・マージ」の3つです。
</details>
