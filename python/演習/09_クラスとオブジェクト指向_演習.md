## 問題1：基本的なクラスの作成（基本）

商品を表す `Product` クラスを作成してください。

- 属性: 名前、価格、在庫数
- メソッド: 商品情報の表示、在庫の増減

```python
p = Product("ノートPC", 98000, 10)
p.show_info()
p.sell(3)
p.restock(5)
p.show_info()
```

**期待される出力：**
```
商品: ノートPC / 98,000円 / 在庫: 10個
3個販売しました（残り7個）
5個入荷しました（残り12個）
商品: ノートPC / 98,000円 / 在庫: 12個
```

<details>
<summary>解答例</summary>

```python
class Product:
    def __init__(self, name, price, stock=0):
        self.name = name
        self.price = price
        self.stock = stock

    def show_info(self):
        """商品情報を表示する"""
        print(f"商品: {self.name} / {self.price:,}円 / 在庫: {self.stock}個")

    def sell(self, quantity):
        """商品を販売する"""
        if quantity > self.stock:
            print(f"在庫不足です（在庫: {self.stock}個）")
            return
        self.stock -= quantity
        print(f"{quantity}個販売しました（残り{self.stock}個）")

    def restock(self, quantity):
        """商品を入荷する"""
        self.stock += quantity
        print(f"{quantity}個入荷しました（残り{self.stock}個）")


p = Product("ノートPC", 98000, 10)
p.show_info()
p.sell(3)
p.restock(5)
p.show_info()
```

```python
e1 = Employee("太郎", "開発部", 350000)
e2 = Employee("花子", "営業部", 320000)
e3 = Employee("次郎", "開発部", 280000)
```

```
=== 社員情報 ===
ABC株式会社 - 太郎（開発部）給与: 350,000円
ABC株式会社 - 花子（営業部）給与: 320,000円
```

<details>
<summary>解答例</summary>

```python
class Employee:
    # クラス変数（全インスタンスで共有）

    employee_count = 0

        print(f"{Employee.company_name} - {self.name}"
              f"（{self.department}）給与: {self.salary:,}円")

e2 = Employee("花子", "営業部", 320000)

    emp.show_info()
```

## 問題3：__str__ と __repr__（基本）

日時を管理する `DateTime` クラスを作成し、`__str__` と `__repr__` を実装してください。

```python
dt = DateTime(2025, 4, 15, 14, 30)
print(dt)        # ユーザー向け表示
print(repr(dt))  # 開発者向け表示
```

**期待される出力：**
```
2025年04月15日 14:30
DateTime(2025, 4, 15, 14, 30)
```

<details>
<summary>解答例</summary>

```python
class DateTime:
    def __init__(self, year, month, day, hour=0, minute=0):
        self.year = year
        self.month = month
        self.day = day
        self.hour = hour
        self.minute = minute

    def __str__(self):
        """ユーザー向けの見やすい表示"""
        return (f"{self.year}年{self.month:02d}月{self.day:02d}日 "
                f"{self.hour:02d}:{self.minute:02d}")

    def __repr__(self):
        """開発者向けの再現可能な表示"""
        return (f"DateTime({self.year}, {self.month}, {self.day}, "
                f"{self.hour}, {self.minute})")


dt = DateTime(2025, 4, 15, 14, 30)
print(dt)
print(repr(dt))
```

## 問題4：@property の活用（基本）

長方形を表す `Rectangle` クラスを作成してください。

- 幅と高さを設定可能（ただし0以下はエラー）
- 面積と周囲長は読み取り専用プロパティ

```python
r = Rectangle(5, 3)
print(f"面積: {r.area}")
print(f"周囲長: {r.perimeter}")
r.width = 10
print(f"面積: {r.area}")
r.width = -5   # → エラー
```

**期待される出力：**
```
面積: 15
周囲長: 16
面積: 30
エラー: 幅は正の数でなければなりません
```

<details>
<summary>解答例</summary>

```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width    # setter が呼ばれる
        self.height = height  # setter が呼ばれる

    @property
    def width(self):
        return self._width

    @width.setter
    def width(self, value):
        if value <= 0:
            raise ValueError("幅は正の数でなければなりません")
        self._width = value

    @property
    def height(self):
        return self._height

    @height.setter
    def height(self, value):
        if value <= 0:
            raise ValueError("高さは正の数でなければなりません")
        self._height = value

    @property
    def area(self):
        """面積（読み取り専用）"""
        return self._width * self._height

    @property
    def perimeter(self):
        """周囲長（読み取り専用）"""
        return 2 * (self._width + self._height)


r = Rectangle(5, 3)
print(f"面積: {r.area}")
print(f"周囲長: {r.perimeter}")

r.width = 10
print(f"面積: {r.area}")

try:
    r.width = -5
except ValueError as e:
    print(f"エラー: {e}")
```

## 問題5：継承とメソッドオーバーライド（応用）

乗り物の基底クラス `Vehicle` を作成し、`Car`, `Bicycle`, `Train` のサブクラスを作成してください。

- 全ての乗り物: 名前、最高速度
- Car: 燃料タイプ（ガソリン/電気）
- Bicycle: ギア数
- Train: 車両数

各クラスに `describe()` メソッドを実装し、ポリモーフィズムで一括表示してください。

**期待される出力：**
```
=== 乗り物一覧 ===
[自動車] プリウス - 最高速度: 180km/h, 燃料: ハイブリッド
[自転車] ロードバイク - 最高速度: 50km/h, ギア: 21段
[電車] のぞみ - 最高速度: 300km/h, 車両数: 16両
[自動車] テスラ - 最高速度: 250km/h, 燃料: 電気

最速の乗り物: のぞみ (300km/h)
```

<details>
<summary>解答例</summary>

```python
class Vehicle:
    """乗り物の基底クラス"""
    def __init__(self, name, max_speed):
        self.name = name
        self.max_speed = max_speed

    def describe(self):
        return f"{self.name} - 最高速度: {self.max_speed}km/h"


class Car(Vehicle):
    """自動車"""
    def __init__(self, name, max_speed, fuel_type):
        super().__init__(name, max_speed)
        self.fuel_type = fuel_type

    def describe(self):
        return f"[自動車] {super().describe()}, 燃料: {self.fuel_type}"


class Bicycle(Vehicle):
    """自転車"""
    def __init__(self, name, max_speed, gears):
        super().__init__(name, max_speed)
        self.gears = gears

    def describe(self):
        return f"[自転車] {super().describe()}, ギア: {self.gears}段"


class Train(Vehicle):
    """電車"""
    def __init__(self, name, max_speed, cars):
        super().__init__(name, max_speed)
        self.cars = cars

    def describe(self):
        return f"[電車] {super().describe()}, 車両数: {self.cars}両"


# ポリモーフィズム: 異なるクラスを同じインターフェースで扱う
vehicles = [
    Car("プリウス", 180, "ハイブリッド"),
    Bicycle("ロードバイク", 50, 21),
    Train("のぞみ", 300, 16),
    Car("テスラ", 250, "電気"),
]

print("=== 乗り物一覧 ===")
for v in vehicles:
    print(v.describe())

# 最速の乗り物を探す
fastest = max(vehicles, key=lambda v: v.max_speed)
print(f"\n最速の乗り物: {fastest.name} ({fastest.max_speed}km/h)")
```

## 問題6：特殊メソッド（応用）

お金を表す `Money` クラスを作成し、以下の特殊メソッドを実装してください。

- `__add__`: お金の足し算
- `__sub__`: お金の引き算
- `__mul__`: 倍率の掛け算
- `__eq__`: 等価比較
- `__lt__`: 大小比較
- `__str__`: 表示用文字列

```python
m1 = Money(1500)
m2 = Money(800)
print(m1 + m2)         # 2,300円
print(m1 - m2)         # 700円
print(m1 * 3)          # 4,500円
print(m1 == Money(1500))  # True
print(m1 > m2)         # True
print(sorted([Money(300), Money(100), Money(200)]))
```

**期待される出力：**
```
2,300円
700円
4,500円
True
True
[100円, 200円, 300円]
```

<details>
<summary>解答例</summary>

```python
class Money:
    def __init__(self, amount):
        if amount < 0:
            raise ValueError("金額は0以上を指定してください")
        self.amount = amount

    def __str__(self):
        return f"{self.amount:,}円"

    def __repr__(self):
        return f"Money({self.amount})"

    def __add__(self, other):
        if isinstance(other, Money):
            return Money(self.amount + other.amount)
        return NotImplemented

    def __sub__(self, other):
        if isinstance(other, Money):
            return Money(self.amount - other.amount)
        return NotImplemented

    def __mul__(self, factor):
        if isinstance(factor, (int, float)):
            return Money(int(self.amount * factor))
        return NotImplemented

    def __eq__(self, other):
        if isinstance(other, Money):
            return self.amount == other.amount
        return NotImplemented

    def __lt__(self, other):
        if isinstance(other, Money):
            return self.amount < other.amount
        return NotImplemented

    def __le__(self, other):
        if isinstance(other, Money):
            return self.amount <= other.amount
        return NotImplemented


m1 = Money(1500)
m2 = Money(800)

print(m1 + m2)
print(m1 - m2)
print(m1 * 3)
print(m1 == Money(1500))
print(m1 > m2)
print(sorted([Money(300), Money(100), Money(200)]))
```

## 問題7：タスク管理システム（チャレンジ）

オブジェクト指向で「タスク管理システム」を設計してください。

クラス構成：
- `Task` - 個別タスク（タイトル、優先度、完了状態）
- `TaskManager` - タスクの管理（追加、完了、一覧、フィルタ）

機能：
1. タスクの追加（優先度: 高/中/低）
2. タスクの完了
3. 全タスク一覧
4. 未完了タスクのフィルタ
5. 優先度別の集計

**期待される出力：**
```
=== タスク管理テスト ===
追加: レポート作成（優先度: 高）
追加: メール返信（優先度: 中）
追加: 資料整理（優先度: 低）
追加: プレゼン準備（優先度: 高）
追加: 経費精算（優先度: 中）

=== 全タスク ===
[ ] [高] レポート作成
[ ] [中] メール返信
[ ] [低] 資料整理
[ ] [高] プレゼン準備
[ ] [中] 経費精算

「メール返信」を完了しました

=== 未完了タスク ===
[ ] [高] レポート作成
[ ] [低] 資料整理
[ ] [高] プレゼン準備
[ ] [中] 経費精算

=== 優先度別集計 ===
高: 2件（未完了: 2件）
中: 2件（未完了: 1件）
低: 1件（未完了: 1件）
```

<details>
<summary>解答例</summary>

```python
class Task:
    """個別タスクを表すクラス"""
    PRIORITY_LABELS = {"high": "高", "medium": "中", "low": "低"}

    def __init__(self, title, priority="medium"):
        self.title = title
        self.priority = priority
        self.completed = False

    def complete(self):
        """タスクを完了にする"""
        self.completed = True

    def __str__(self):
        status = "[x]" if self.completed else "[ ]"
        priority_label = self.PRIORITY_LABELS.get(self.priority, "?")
        return f"{status} [{priority_label}] {self.title}"


class TaskManager:
    """タスクを管理するクラス"""

    def __init__(self):
        self.tasks = []

    def add_task(self, title, priority="medium"):
        """タスクを追加する"""
        task = Task(title, priority)
        self.tasks.append(task)
        label = Task.PRIORITY_LABELS.get(priority, "?")
        print(f"追加: {title}（優先度: {label}）")

    def complete_task(self, title):
        """タスクを完了にする"""
        for task in self.tasks:
            if task.title == title and not task.completed:
                task.complete()
                print(f"「{title}」を完了しました")
                return
        print(f"「{title}」が見つかりません")

    def show_all(self):
        """全タスクを表示する"""
        print("\n=== 全タスク ===")
        for task in self.tasks:
            print(task)

    def show_incomplete(self):
        """未完了タスクを表示する"""
        print("\n=== 未完了タスク ===")
        for task in self.tasks:
            if not task.completed:
                print(task)

    def show_summary(self):
        """優先度別の集計を表示する"""
        print("\n=== 優先度別集計 ===")
        for priority in ["high", "medium", "low"]:
            tasks = [t for t in self.tasks if t.priority == priority]
            incomplete = [t for t in tasks if not t.completed]
            label = Task.PRIORITY_LABELS[priority]
            print(f"{label}: {len(tasks)}件（未完了: {len(incomplete)}件）")


# テスト
print("=== タスク管理テスト ===")
manager = TaskManager()

manager.add_task("レポート作成", "high")
manager.add_task("メール返信", "medium")
manager.add_task("資料整理", "low")
manager.add_task("プレゼン準備", "high")
manager.add_task("経費精算", "medium")

manager.show_all()

print()
manager.complete_task("メール返信")

manager.show_incomplete()
manager.show_summary()
```

## 問題8：ショッピングカートシステム（チャレンジ）

オブジェクト指向でショッピングカートを実装してください。

クラス構成：
- `Product` - 商品（名前、価格、カテゴリ）
- `CartItem` - カート内のアイテム（商品、数量）
- `ShoppingCart` - カート（追加、削除、合計計算、割引適用）

割引ルール：
- 合計5,000円以上: 5%割引
- 合計10,000円以上: 10%割引

**期待される出力：**
```
=== ショッピングカート ===
コーヒー豆 x 2 = 2,560円
紅茶パック x 3 = 1,740円
マグカップ x 1 = 1,200円
------------------------------
小計: 5,500円
割引(5%): -275円
合計: 5,225円（税込 5,748円）
```

<details>
<summary>解答例</summary>

```python
class Product:
    """商品クラス"""
    def __init__(self, name, price, category="一般"):
        self.name = name
        self.price = price
        self.category = category

    def __str__(self):
        return f"{self.name}({self.price:,}円)"


class CartItem:
    """カート内アイテム"""
    def __init__(self, product, quantity=1):
        self.product = product
        self.quantity = quantity

    @property
    def subtotal(self):
        return self.product.price * self.quantity

    def __str__(self):
        return (f"{self.product.name} x {self.quantity} "
                f"= {self.subtotal:,}円")


class ShoppingCart:
    """ショッピングカート"""
    TAX_RATE = 0.10  # 消費税10%

    def __init__(self):
        self.items = []

    def add(self, product, quantity=1):
        """商品を追加（同じ商品なら数量を加算）"""
        for item in self.items:
            if item.product.name == product.name:
                item.quantity += quantity
                return
        self.items.append(CartItem(product, quantity))

    def remove(self, product_name):
        """商品を削除する"""
        self.items = [i for i in self.items if i.product.name != product_name]

    @property
    def subtotal(self):
        """小計"""
        return sum(item.subtotal for item in self.items)

    @property
    def discount_rate(self):
        """割引率の計算"""
        if self.subtotal >= 10000:
            return 0.10
        elif self.subtotal >= 5000:
            return 0.05
        return 0

    @property
    def discount_amount(self):
        """割引額"""
        return int(self.subtotal * self.discount_rate)

    @property
    def total(self):
        """割引後の合計"""
        return self.subtotal - self.discount_amount

    @property
    def total_with_tax(self):
        """税込合計"""
        return int(self.total * (1 + self.TAX_RATE))

    def show(self):
        """カートの内容を表示する"""
        print("=== ショッピングカート ===")
        for item in self.items:
            print(item)
        print("-" * 30)
        print(f"小計: {self.subtotal:,}円")
        if self.discount_rate > 0:
            rate_pct = int(self.discount_rate * 100)
            print(f"割引({rate_pct}%): -{self.discount_amount:,}円")
        print(f"合計: {self.total:,}円（税込 {self.total_with_tax:,}円）")


# テスト
cart = ShoppingCart()
cart.add(Product("コーヒー豆", 1280, "食品"), 2)
cart.add(Product("紅茶パック", 580, "食品"), 3)
cart.add(Product("マグカップ", 1200, "雑貨"), 1)
cart.show()
```

</details>
