## 課題の要件

1. シバン行（`#!/usr/bin/env bash`）を記述する
2. 現在のユーザー名を変数に格納する（`$USER` または `whoami`）
3. 現在時刻を取得し変数に格納する（`date` コマンド）
4. 時間帯に応じて挨拶を変える（午前→おはようございます、午後→こんにちは、夜→こんばんは）
5. ホスト名とOS名を表示する
6. 見やすいフォーマットで出力する

---

## ステップガイド

<details>
<summary>ステップ1：シバン行と基本変数を用意する</summary>

スクリプトの1行目にシバン行を書き、基本的な変数を定義しましょう。

```bash
#!/usr/bin/env bash

# ユーザー名を取得
username="${USER}"

# 現在時刻を取得
current_time=$(date "+%Y-%m-%d %H:%M:%S")

# 時（hour）だけを取得
hour=$(date "+%H")
```

</details>

<details>
<summary>ステップ2：時間帯に応じた挨拶を決める</summary>

`hour` の値に応じて、挨拶文を変数に代入します。
算術比較には `(( ))` を使うと簡潔です。

```bash
if (( hour < 12 )); then
    greeting="おはようございます"
elif (( hour < 18 )); then
    greeting="こんにちは"
else
    greeting="こんばんは"
fi
```

</details>

<details>
<summary>ステップ3：システム情報を取得して表示する</summary>

`hostname` コマンドや `uname` コマンドでシステム情報を取得し、整形して出力します。

```bash
host=$(hostname)
os_name=$(uname -s)

echo "==========================="
echo "  ${greeting}、${username} さん！"
echo "  現在時刻: ${current_time}"
# ... 残りも同様に
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```bash
#!/usr/bin/env bash
# 挨拶スクリプト
# 学べる内容：シバン行、変数、コマンド置換、条件分岐の基本
# 実行方法：chmod +x greeting.sh && ./greeting.sh

# --- 情報取得 ---
username="${USER}"
current_time=$(date "+%Y-%m-%d %H:%M:%S")
hour=$(date "+%H")
host=$(hostname)
os_name=$(uname -s)
shell_path="${SHELL}"

# --- 時間帯に応じた挨拶 ---
if [ "$hour" -lt 12 ]; then
    greeting="おはようございます"
elif [ "$hour" -lt 18 ]; then
    greeting="こんにちは"
else
    greeting="こんばんは"
fi

# --- 表示 ---
echo "==========================="
echo "  ${greeting}、${username} さん！"
echo "  現在時刻: ${current_time}"
echo "  ホスト名: ${host}"
echo "  OS: ${os_name}"
echo "  シェル: ${shell_path}"
echo "==========================="
echo "今日も一日がんばりましょう！"
```

</details>

<details>
<summary>解答例（改良版 ─ パラメータ展開とprintf活用）</summary>

```bash
#!/usr/bin/env bash
# 挨拶スクリプト（改良版）
# パラメータ展開、printf、関数を活用したバージョン
# 実行方法：chmod +x greeting.sh && ./greeting.sh

# --- 設定 ---
readonly BORDER="==========================="

# --- 情報取得 ---
# ${USER} が未定義の場合のフォールバック
username="${USER:-$(whoami)}"
current_time=$(date "+%Y-%m-%d %H:%M:%S")
hour=$(date "+%H")
host=$(hostname -s 2>/dev/null || hostname)
os_name=$(uname -s)
shell_name="${SHELL##*/}"  # パス展開で最後の部分だけ取得（例：/bin/bash → bash）

# --- 時間帯に応じた挨拶 ---
if (( 10#$hour < 12 )); then
    greeting="おはようございます"
elif (( 10#$hour < 18 )); then
    greeting="こんにちは"
else
    greeting="こんばんは"
fi

# --- 表示（printfで整形） ---
echo "${BORDER}"
printf "  %s、%s さん！\n" "${greeting}" "${username}"
printf "  %-8s %s\n" "時刻:" "${current_time}"
printf "  %-8s %s\n" "ホスト:" "${host}"
printf "  %-8s %s\n" "OS:" "${os_name}"
printf "  %-8s %s\n" "シェル:" "${shell_name}"
echo "${BORDER}"
echo "今日も一日がんばりましょう！"
```

**初心者向けとの違い:**
- `${USER:-$(whoami)}` でデフォルト値を設定 → `$USER` が未定義でも安全
- `${SHELL##*/}` でパスからシェル名だけを抽出 → パラメータ展開の活用
- `10#$hour` で8進数誤解釈を防止（例：`08` がエラーにならない）
- `printf` で整形出力 → 桁揃えが容易
- `readonly` で定数を明示 → 意図しない変更を防止
- `hostname -s 2>/dev/null || hostname` で環境差異に対応

</details>
