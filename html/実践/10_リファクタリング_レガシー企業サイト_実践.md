# 実践課題10：リファクタリング ─ レガシー企業サイト ★4

> **難易度**: ★★★★☆（上級）
> **前提知識**: 第1章〜第8章（すべての章）
> **課題の種類**: リファクタリング
> **学習目標**: 古い書き方のHTMLをモダンなHTML5/セマンティックHTMLにリファクタリングし、アクセシビリティとSEOを向上させる

---

## 課題の説明

以下のHTMLは2000年代に書かれた架空の企業サイトです。動作はしますが、以下の問題を抱えています。

1. **テーブルレイアウト** を使って配置している（CSSレイアウトにすべき）
2. **セマンティックでない** タグの使い方（divの多用、適切なタグの不使用）
3. **廃止された要素・属性** の使用（font、center、bgcolor など）
4. **アクセシビリティ** の考慮がない（alt属性の欠落、label の未使用など）
5. **メタ情報** が不足している

### ゴール

このコードを以下の方針でリファクタリングしてください。

- テーブルレイアウトをセマンティックHTML + CSSに置き換える
- 廃止された要素・属性をすべて削除し、CSSで代替する
- 適切なセマンティックHTML要素（header / nav / main / section / article / aside / footer）を使う
- アクセシビリティを向上させる（alt、label、aria属性など）
- メタ情報を充実させる（charset、viewport、description、OGP）

## リファクタリングのチェックリスト

以下の項目をすべて満たすようにリファクタリングしてください。

- [ ] `<!DOCTYPE html>` を追加
- [ ] `<html lang="ja">` に言語指定を追加
- [ ] `<meta charset="utf-8">` を追加
- [ ] `<meta name="viewport">` を追加
- [ ] `<meta name="description">` を追加
- [ ] すべての `<font>` タグを削除し、CSSに置き換え
- [ ] すべての `<center>` タグを削除し、CSSに置き換え
- [ ] `bgcolor`、`text` 属性を削除し、CSSに置き換え
- [ ] テーブルレイアウトを `<header>` + `<nav>` + `<main>` + `<footer>` に置き換え
- [ ] お知らせを適切なリスト要素に変更
- [ ] サービス紹介を `<article>` と `<figure>` を使って構造化
- [ ] `<img>` に `alt` 属性を追加
- [ ] フォームの `<label>` と `<input>` を関連付け
- [ ] メールアドレスの入力を `type="email"` に変更
- [ ] `required` 属性で必須項目を指定
- [ ] `<hr>` の装飾的な使用をCSSに置き換え

## 解答例

<details>
<summary>ヒント：何をどう変えるかの対応表</summary>

| レガシーな書き方 | モダンな書き方 |
|---|---|
| `<html>` | `<html lang="ja">` |
| `<body bgcolor="#FFF" text="#333">` | CSS: `body { background: #fff; color: #333; }` |
| `<center>...</center>` | CSS: `text-align: center;` |
| `<font size="6" color="#003366">` | CSS: `font-size: 2em; color: #003366;` |
| `<b>見出し</b>` | `<h1>見出し</h1>` 等の見出しタグ |
| `<table>` でレイアウト | `<header>` + `<nav>` + `<main>` + `<footer>` |
| `<br><br>` で余白 | CSS: `margin` |
| `<hr>` で区切り | CSS: `border-bottom` |
| `<input type="text">` (メール) | `<input type="email">` |
| `<font size="2">お名前：</font>` | `<label for="name">お名前：</label>` |

</details>

**レガシーコードとの主な変更点:**

| 変更箇所 | レガシー | モダン |
|---|---|---|
| DOCTYPE | なし | `<!DOCTYPE html>` |
| 言語指定 | なし | `<html lang="ja">` |
| 文字コード | なし | `<meta charset="utf-8">` |
| レイアウト | `<table>` | CSS Flexbox |
| ナビゲーション | `<font>` + `<a>` + `<br>` | `<nav>` + `<ul>` + `<li>` |
| 装飾 | `<font>` / `<center>` / `bgcolor` | CSS |
| 見出し | `<font size="4"><b>...</b></font>` | `<h2>` |
| 画像 | `alt` なし | `alt` 付き |
| フォーム | `<label>` なし / `type="text"` | `<label for="">` / `type="email"` / `required` |
| お知らせ | `<table>` | `<ul>` + `<time>` |
| サービス | `<table>` | `<article>` + `<figure>` |
| フッター | `<center>` + `<font>` | `<footer>` |
| レスポンシブ | 非対応 | `@media` クエリ対応 |
