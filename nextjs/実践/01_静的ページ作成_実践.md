# 実践課題01：静的ページ作成 ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（Next.jsの基本）
> **課題の種類**: ミニプロジェクト
> **学習目標**: Next.js プロジェクトの作成、App Router のファイル構造、基本的なページ作成を実践する

---

## 完成イメージ

シンプルな自己紹介サイトをNext.jsで構築します。トップページとAboutページの2ページ構成です。

```
┌─────────────────────────────────────┐
│  🏠 My Portfolio                    │
│  [ホーム]  [自己紹介]                │
├─────────────────────────────────────┤
│                                     │
│  こんにちは、太郎です                  │
│                                     │
│  フロントエンドエンジニアとして         │
│  Web開発をしています。                │
│                                     │
│  ┌──────┐ ┌──────┐ ┌──────┐        │
│  │React │ │Next  │ │TS   │        │
│  └──────┘ └──────┘ └──────┘        │
│                                     │
│  © 2026 My Portfolio                │
└─────────────────────────────────────┘
```

---

## 課題の要件

1. `create-next-app` でプロジェクトを作成する（TypeScript、App Router を選択）
2. トップページ（`/`）に名前・肩書き・スキル一覧を表示する
3. 自己紹介ページ（`/about`）に詳細なプロフィールを表示する
4. 両ページで共通のヘッダーとフッターを表示する（ルートレイアウトを使用）
5. `next/link` を使ってページ間をナビゲーションする

---

### app/about/page.tsx（メタデータ付き）

```tsx
import type { Metadata } from 'next'

// ページ固有のメタデータ — template により「自己紹介 | My Portfolio」になる
export const metadata: Metadata = {
  title: '自己紹介',
  description: 'フロントエンドエンジニア太郎の経歴とスキル',
}

export default function AboutPage() {
  const career = [
    { year: '2020', event: 'Web開発を独学で開始' },
    { year: '2021', event: 'React案件に初参画' },
    { year: '2023', event: 'Next.jsでフルスタック開発' },
    { year: '2024', event: 'チームリーダーとして活動' },
  ]

  return (
    <div>
      <h1>自己紹介</h1>

      <section style={{ marginBottom: '32px' }}>
        <h2>経歴</h2>
        {career.map(({ year, event }) => (
          <div
            key={year}
            style={{
              display: 'flex',
              gap: '16px',
              padding: '8px 0',
              borderBottom: '1px solid #f1f5f9',
            }}
          >
            <span style={{ fontWeight: 'bold', color: '#2563eb' }}>
              {year}
            </span>
            <span>{event}</span>
          </div>
        ))}
      </section>

      <section>
        <h2>好きな技術</h2>
        <ul style={{ lineHeight: '2' }}>
          <li>React / Next.js — UIの構築が好きです</li>
          <li>TypeScript — 型安全なコードを書くのが好きです</li>
          <li>Tailwind CSS — 素早くスタイリングできます</li>
        </ul>
      </section>
    </div>
  )
}
```

**初心者向けとの違い:**
- CSS Modules でスタイルを分離し、保守性を向上
- `Metadata` の `template` 機能でページタイトルを自動フォーマット
- 経歴データを配列で管理し、`map` で描画（データとUIを分離）
- ホバーエフェクトで操作感を改善
