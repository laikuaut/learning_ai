## 課題の要件

2. CIDR表記（/24等）とドット表記（255.255.255.0等）の両方に対応する

4. 各アドレスの2進数表現も表示する

## 背景知識の確認

```
IPアドレスの構造（/26の場合）:

  192.168.1.100 = 11000000.10101000.00000001.01|100100
                  ←── ネットワーク部（26ビット）──→←ホスト部→

サブネットマスク /26 = 255.255.255.192
  = 11111111.11111111.11111111.11000000

ネットワークアドレス = IPアドレス AND サブネットマスク
  11000000.10101000.00000001.01100100  (192.168.1.100)
  AND
  11111111.11111111.11111111.11000000  (255.255.255.192)
  = 11000000.10101000.00000001.01000000  (192.168.1.64)

ブロードキャスト = ネットワークアドレス OR (NOT サブネットマスク)
  = 11000000.10101000.00000001.01111111  (192.168.1.127)

使用可能ホスト数 = 2^(32-プレフィックス長) - 2
```

## ステップガイド

<details>
<summary>ステップ1：IPアドレスを整数に変換する</summary>

IPアドレスを「.」で分割し、各オクテットを8ビットずつ左シフトして32ビット整数にします。

```python
def ip_to_int(ip_str):
    """IPアドレス文字列を32ビット整数に変換"""
    parts = ip_str.split(".")
    return (int(parts[0]) << 24) + (int(parts[1]) << 16) + \
           (int(parts[2]) << 8) + int(parts[3])
```

</details>

<details>
<summary>ステップ2：CIDR表記の処理</summary>

「/26」のような入力を受け取ったらサブネットマスクの整数値に変換します。

```python
def prefix_to_mask(prefix):
    """プレフィックス長（例: 26）からサブネットマスク整数を生成"""
    return (0xFFFFFFFF << (32 - prefix)) & 0xFFFFFFFF
```

</details>

<details>
<summary>ステップ3：ネットワーク計算</summary>

ビット演算で各アドレスを求めます。

```python
network = ip_int & mask_int            # AND演算
broadcast = network | (~mask_int & 0xFFFFFFFF)  # OR NOT演算
host_min = network + 1
host_max = broadcast - 1
host_count = broadcast - network - 1
```

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```python
# IPアドレス／サブネット計算ツール
# 学べる内容：IPアドレス計算、ビット演算、サブネッティング
# 実行方法：python subnet_calc.py

def ip_to_int(ip_str):
    """IPアドレス文字列を32ビット整数に変換"""
    parts = ip_str.split(".")
    return (int(parts[0]) << 24) | (int(parts[1]) << 16) | \
           (int(parts[2]) << 8) | int(parts[3])

def int_to_ip(ip_int):
    """32ビット整数をIPアドレス文字列に変換"""
    return f"{(ip_int >> 24) & 0xFF}.{(ip_int >> 16) & 0xFF}.{(ip_int >> 8) & 0xFF}.{ip_int & 0xFF}"

def int_to_bin(ip_int):
    """32ビット整数を2進数ドット表記に変換"""
    b = format(ip_int, "032b")
    return f"{b[0:8]}.{b[8:16]}.{b[16:24]}.{b[24:32]}"

def prefix_to_mask(prefix):
    """プレフィックス長からサブネットマスク整数を生成"""
    return (0xFFFFFFFF << (32 - prefix)) & 0xFFFFFFFF

def mask_to_prefix(mask_int):
    """サブネットマスク整数からプレフィックス長を計算"""
    count = 0
    for i in range(31, -1, -1):
        if mask_int & (1 << i):
            count += 1
        else:
            break
    return count

# --- メイン ---
print("===== IPアドレス／サブネット計算ツール =====\n")

ip_str = input("IPアドレスを入力（例: 192.168.1.100）: ")
mask_input = input("サブネットマスクを入力（例: 255.255.255.0 または /24）: ")

# IPアドレスを整数に変換
ip_int = ip_to_int(ip_str)

# サブネットマスクの処理
if mask_input.startswith("/"):
    prefix = int(mask_input[1:])
    mask_int = prefix_to_mask(prefix)
else:
    mask_int = ip_to_int(mask_input)
    prefix = mask_to_prefix(mask_int)

# ネットワーク計算
network_int = ip_int & mask_int
broadcast_int = network_int | (~mask_int & 0xFFFFFFFF)
host_min_int = network_int + 1
host_max_int = broadcast_int - 1
host_count = 2 ** (32 - prefix) - 2

# 結果表示
print("\n--- 計算結果 ---")
print(f"IPアドレス:            {ip_str}")
print(f"サブネットマスク:       {int_to_ip(mask_int)} (/{prefix})")
print(f"ネットワークアドレス:   {int_to_ip(network_int)}")
print(f"ブロードキャストアドレス: {int_to_ip(broadcast_int)}")
print(f"使用可能ホスト範囲:     {int_to_ip(host_min_int)} ～ {int_to_ip(host_max_int)}")
print(f"使用可能ホスト数:       {host_count}台")

# ビット表現
print("\n--- ビット表現 ---")
print(f"IPアドレス:   {int_to_bin(ip_int)}")
print(f"サブネットマスク: {int_to_bin(mask_int)}")
print(f"ネットワーク: {int_to_bin(network_int)}")
```

</details>

<summary>解答例（改良版 ─ サブネット分割一覧表示付き）</summary>

```python
# IPアドレス／サブネット計算ツール（改良版）
# 学べる内容：サブネッティング、VLSM、ネットワーク設計
# 実行方法：python subnet_advanced.py

def ip_to_int(ip_str):
    parts = ip_str.split(".")
    return (int(parts[0]) << 24) | (int(parts[1]) << 16) | \
           (int(parts[2]) << 8) | int(parts[3])

def int_to_ip(n):
    return f"{(n >> 24) & 0xFF}.{(n >> 16) & 0xFF}.{(n >> 8) & 0xFF}.{n & 0xFF}"

def prefix_to_mask(prefix):
    return (0xFFFFFFFF << (32 - prefix)) & 0xFFFFFFFF

def subnet_info(network_int, prefix):
    """サブネットの情報を辞書で返す"""
    mask_int = prefix_to_mask(prefix)
    broadcast_int = network_int | (~mask_int & 0xFFFFFFFF)
    host_count = max(0, 2 ** (32 - prefix) - 2)
    return {
        "network": int_to_ip(network_int),
        "broadcast": int_to_ip(broadcast_int),
        "first_host": int_to_ip(network_int + 1) if host_count > 0 else "-",
        "last_host": int_to_ip(broadcast_int - 1) if host_count > 0 else "-",
        "hosts": host_count,
    }

# --- メイン ---
print("===== サブネット分割ツール =====\n")

base_ip = input("ネットワークアドレスを入力（例: 192.168.1.0）: ")
base_prefix = int(input("元のプレフィックス長（例: 24）: "))
new_prefix = int(input("分割後のプレフィックス長（例: 26）: "))

if new_prefix <= base_prefix:
    print("エラー: 分割後のプレフィックス長は元より大きくしてください。")
else:
    base_int = ip_to_int(base_ip)
    subnet_count = 2 ** (new_prefix - base_prefix)
    subnet_size = 2 ** (32 - new_prefix)

    print(f"\n/{base_prefix} を /{new_prefix} に分割 → {subnet_count}個のサブネット")
    print(f"各サブネットのホスト数: {subnet_size - 2}台\n")

    print(f"{'#':>3} | {'ネットワーク':<18} | {'最初のホスト':<18} | {'最後のホスト':<18} | {'ブロードキャスト':<18}")
    print("-" * 90)

    for i in range(subnet_count):
        net_int = base_int + (i * subnet_size)
        info = subnet_info(net_int, new_prefix)
        print(f"{i+1:3d} | {info['network']:<18} | {info['first_host']:<18} | "
              f"{info['last_host']:<18} | {info['broadcast']:<18}")

    print(f"\n合計: {subnet_count}サブネット × {subnet_size - 2}ホスト = {subnet_count * (subnet_size - 2)}ホスト割当可能")
```

- サブネット分割（サブネッティング）のシミュレーション機能を追加
- 分割結果の一覧表を自動生成し、ネットワーク設計の実務に近い体験ができる
- 関数化により計算ロジックを整理し、再利用性を高めている

## 確認問題

手計算してからプログラムで検算しましょう。

**問1:** 172.16.10.130/25 のネットワークアドレスとブロードキャストアドレスは？
→ ネットワーク: 172.16.10.128、ブロードキャスト: 172.16.10.255

**問2:** 10.0.0.0/22 のネットワークで使用可能なホスト数は？
→ 2^(32-22) - 2 = 1024 - 2 = 1,022台

**問3:** 192.168.0.0/24 を /27 に分割すると何個のサブネットができるか？各サブネットのホスト数は？
→ 2^(27-24) = 8個、各サブネット 2^(32-27) - 2 = 30台
