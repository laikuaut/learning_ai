---

## 課題の要件

1. 商品を表す `Product` インターフェースを定義する（名前、価格、カテゴリ、在庫数）
2. カテゴリを表す型エイリアスを定義する
3. 全商品を一覧表示する関数を作る（品切れ表示付き）
4. カテゴリで絞り込む関数を作る
5. 在庫がある商品の合計金額を計算する関数を作る
6. 価格帯でフィルタする関数を作る

---

## ステップガイド

<details>
<summary>ステップ1：型定義を作る</summary>

まずカテゴリと商品の型を定義しましょう。

```typescript
// 型エイリアスでカテゴリを定義
type Category = "書籍" | "電子機器" | "食品" | "衣類";

// インターフェースで商品を定義
interface Product {
  id: number;
  name: string;
  price: number;
  category: Category;
  stock: number;
}
```

`Category` は文字列リテラル型（literal type）のユニオンです。これにより、定義されていないカテゴリ名を指定するとコンパイルエラーになります。

</details>

<details>
<summary>ステップ2：商品データを作る</summary>

```typescript
const products: Product[] = [
  { id: 1, name: "TypeScript入門書", price: 2800, category: "書籍", stock: 15 },
  { id: 2, name: "ワイヤレスマウス", price: 3500, category: "電子機器", stock: 42 },
  // ... さらに追加
];
```

`Product[]` という型注釈を付けることで、配列の中身が必ず `Product` 型であることが保証されます。

</details>

<details>
<summary>ステップ3：関数を実装する</summary>

各関数には引数と戻り値の型注釈を付けましょう。

```typescript
function filterByCategory(products: Product[], category: Category): Product[] {
  return products.filter((p: Product): boolean => p.category === category);
}
```

`filter` は条件に合う要素だけを取り出すメソッドです。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```typescript
// 商品カタログ型定義
// 学べる内容：interface、型エイリアス、リテラル型、配列操作
// 実行方法：npx ts-node 02_product_catalog.ts

// --- 型定義 ---
type Category = "書籍" | "電子機器" | "食品" | "衣類";

interface Product {
  id: number;
  name: string;
  price: number;
  category: Category;
  stock: number;
}

// --- 商品データ ---
const products: Product[] = [
  { id: 1, name: "TypeScript入門書", price: 2800, category: "書籍", stock: 15 },
  { id: 2, name: "ワイヤレスマウス", price: 3500, category: "電子機器", stock: 42 },
  { id: 3, name: "コーヒー豆 500g", price: 1200, category: "食品", stock: 0 },
  { id: 4, name: "プログラミングTシャツ", price: 2500, category: "衣類", stock: 8 },
  { id: 5, name: "USBケーブル", price: 980, category: "電子機器", stock: 120 },
];

// --- 全商品一覧表示 ---
function displayAll(products: Product[]): void {
  console.log("--- 全商品一覧 ---");
  for (let i = 0; i < products.length; i++) {
    const p: Product = products[i];
    const priceStr: string = `¥${p.price.toLocaleString()}`;
    const stockLabel: string = p.stock === 0 ? "在庫: 0 【品切れ】" : `在庫: ${p.stock}`;
    console.log(
      `[${p.id}] ${p.name.padEnd(20)} ${priceStr.padEnd(8)} カテゴリ: ${p.category.padEnd(6)} ${stockLabel}`
    );
  }
}

// --- カテゴリ絞り込み ---
function filterByCategory(products: Product[], category: Category): Product[] {
  const result: Product[] = [];
  for (const p of products) {
    if (p.category === category) {
      result.push(p);
    }
  }
  return result;
}

// --- 在庫あり商品の合計金額 ---
function totalInStockPrice(products: Product[]): { total: number; count: number } {
  let total: number = 0;
  let count: number = 0;
  for (const p of products) {
    if (p.stock > 0) {
      total += p.price;
      count++;
    }
  }
  return { total, count };
}

// --- 価格帯フィルタ ---
function filterByPriceRange(
  products: Product[],
  min: number,
  max: number
): Product[] {
  const result: Product[] = [];
  for (const p of products) {
    if (p.price >= min && p.price <= max) {
      result.push(p);
    }
  }
  return result;
}

// --- メイン処理 ---
console.log("===== 商品カタログ =====\n");
displayAll(products);

console.log(`\n--- カテゴリ「電子機器」の商品 ---`);
const electronics: Product[] = filterByCategory(products, "電子機器");
for (const p of electronics) {
  console.log(`[${p.id}] ${p.name.padEnd(16)} ¥${p.price.toLocaleString().padEnd(6)} 在庫: ${p.stock}`);
}

console.log("\n--- 在庫あり商品の合計金額 ---");
const { total, count } = totalInStockPrice(products);
console.log(`合計: ¥${total.toLocaleString()}（${count}商品）`);

console.log("\n--- 価格帯フィルタ（¥1,000〜¥3,000） ---");
const filtered: Product[] = filterByPriceRange(products, 1000, 3000);
for (const p of filtered) {
  console.log(`[${p.id}] ${p.name.padEnd(20)} ¥${p.price.toLocaleString()}`);
}
```

<summary>解答例（改良版 ─ 高階関数を活用）</summary>

```typescript
// 商品カタログ型定義（改良版）
// 学べる内容：高階関数（filter, reduce, map）、オプショナルプロパティ
// 実行方法：npx ts-node 02_product_catalog_v2.ts

// --- 型定義 ---
type Category = "書籍" | "電子機器" | "食品" | "衣類";

interface Product {
  readonly id: number;
  readonly name: string;
  readonly price: number;
  readonly category: Category;
  readonly stock: number;
  readonly description?: string; // オプショナルプロパティ
}

// フィルタ条件を表す型
interface FilterCondition {
  category?: Category;
  minPrice?: number;
  maxPrice?: number;
  inStockOnly?: boolean;
}

// --- 商品データ ---
const products: Product[] = [
  { id: 1, name: "TypeScript入門書", price: 2800, category: "書籍", stock: 15, description: "初心者向け" },
  { id: 2, name: "ワイヤレスマウス", price: 3500, category: "電子機器", stock: 42 },
  { id: 3, name: "コーヒー豆 500g", price: 1200, category: "食品", stock: 0 },
  { id: 4, name: "プログラミングTシャツ", price: 2500, category: "衣類", stock: 8 },
  { id: 5, name: "USBケーブル", price: 980, category: "電子機器", stock: 120 },
];

// --- 表示ヘルパー ---
function formatPrice(price: number): string {
  return `¥${price.toLocaleString()}`;
}

function formatProduct(p: Product): string {
  const stockLabel: string = p.stock === 0 ? "在庫: 0 【品切れ】" : `在庫: ${p.stock}`;
  return `[${p.id}] ${p.name.padEnd(20)} ${formatPrice(p.price).padEnd(8)} カテゴリ: ${p.category.padEnd(6)} ${stockLabel}`;
}

// --- 汎用フィルタ関数 ---
function filterProducts(products: Product[], condition: FilterCondition): Product[] {
  return products.filter((p: Product): boolean => {
    if (condition.category !== undefined && p.category !== condition.category) return false;
    if (condition.minPrice !== undefined && p.price < condition.minPrice) return false;
    if (condition.maxPrice !== undefined && p.price > condition.maxPrice) return false;
    if (condition.inStockOnly === true && p.stock === 0) return false;
    return true;
  });
}

// --- 集計関数 ---
function calculateTotal(products: Product[]): { total: number; count: number } {
  return products.reduce(
    (acc: { total: number; count: number }, p: Product) => ({
      total: acc.total + p.price,
      count: acc.count + 1,
    }),
    { total: 0, count: 0 }
  );
}

// --- メイン処理 ---
console.log("===== 商品カタログ =====\n");

console.log("--- 全商品一覧 ---");
products.map(formatProduct).forEach((line: string) => console.log(line));

console.log(`\n--- カテゴリ「電子機器」の商品 ---`);
const electronics: Product[] = filterProducts(products, { category: "電子機器" });
electronics.forEach((p: Product): void => {
  console.log(`[${p.id}] ${p.name.padEnd(16)} ${formatPrice(p.price).padEnd(8)} 在庫: ${p.stock}`);
});

console.log("\n--- 在庫あり商品の合計金額 ---");
const inStock: Product[] = filterProducts(products, { inStockOnly: true });
const { total, count } = calculateTotal(inStock);
console.log(`合計: ${formatPrice(total)}（${count}商品）`);

console.log("\n--- 価格帯フィルタ（¥1,000〜¥3,000） ---");
const priceFiltered: Product[] = filterProducts(products, { minPrice: 1000, maxPrice: 3000 });
priceFiltered.forEach((p: Product): void => {
  console.log(`[${p.id}] ${p.name.padEnd(20)} ${formatPrice(p.price)}`);
});
```

**初心者向けとの違い:**
- `FilterCondition` インターフェースで検索条件を統一し、1つの `filterProducts` 関数で複数のフィルタ条件に対応
- `filter`, `reduce`, `map` などの高階関数を活用
- `formatPrice`, `formatProduct` でフォーマット処理を共通化
- オプショナルプロパティ（`?`）の実用例

</details>
