# 第3章 演習：挿入モードとテキスト編集

この演習では、挿入モードへの入り方、テキストオブジェクト、オペレータとモーションの組み合わせを学びます。

## 基本問題

### 問題1：挿入モードの入り方の使い分け

以下のコードで、各シナリオに最適な挿入モードへの入り方を答えてください。

```javascript
function greet() {
    console.log("Hello");
}
```

| シナリオ | キー |
|---|---|
| (1) カーソルが `console` の `c` にあり、その前にテキストを追加したい | （ ） |
| (2) カーソルがどこにあっても、行末にコメントを追加したい | （ ） |
| (3) 現在の行の下に新しい行を追加したい | （ ） |
| (4) 現在の行の上に新しい行を追加したい | （ ） |
| (5) カーソルがどこにあっても、行頭にテキストを挿入したい | （ ） |
| (6) カーソルが `H` にあり、`H` の後ろに追加したい | （ ） |

**期待される解答例：**

```
(1) i  (2) A  (3) o  (4) O  (5) I  (6) a
```

<details>
<summary>ヒント</summary>

`i`/`a` はカーソル前後、`I`/`A` は行頭/行末、`o`/`O` は新しい行の下/上です。

</details>

<details>
<summary>解答と解説</summary>

```vim
i  " カーソルの前に挿入（insert）
a  " カーソルの後ろに追加（append）
I  " 行頭の最初の非空白文字に挿入（大文字で行全体に作用）
A  " 行末に追加（大文字で行全体に作用）
o  " 下に新しい行を作って挿入（open）
O  " 上に新しい行を作って挿入（大文字で逆方向）
```

覚え方：小文字はカーソル付近、大文字は行全体。`o`/`O` は新規行の作成です。

</details>

### 問題2：テキストオブジェクトの理解

カーソルが `H`（Hello）の位置にあるとき、各テキストオブジェクトの範囲を答えてください。

```python
message = "Hello, World!"
```

1. `iw`（inner word）
2. `aw`（a word）
3. `i"`（inner double quote）
4. `a"`（a double quote）

**期待される解答例：**

```
1. Hello（単語のみ）
2. Hello,（単語 + 隣接する区切り文字）
3. Hello, World!（ダブルクォートの中身）
4. "Hello, World!"（ダブルクォートを含む全体）
```

<details>
<summary>ヒント</summary>

`i` は inner（内側、囲み文字を含まない）、`a` は a（ひとまとまり、囲み文字を含む）です。

</details>

<details>
<summary>解答と解説</summary>

```vim
" iw → Hello
"   単語の内側のみ。diw で Hello だけ削除されます

" aw → Hello + 隣接する区切り
"   単語と周囲の空白や区切りを含みます。daw で自然な削除ができます

" i" → Hello, World!
"   ダブルクォートの中身だけ。ci" で中身を書き換えられます

" a" → "Hello, World!"
"   ダブルクォートを含む全体。da" で引用符ごと削除できます

" テキストオブジェクトはカーソルが囲みの中のどこにあっても正しく動作します
```

</details>

### 問題3：基本的なオペレータ操作

カーソルが `q`（quick）の位置にあるとき、各操作のコマンドを答えてください。

```
The quick brown fox jumps over the lazy dog.
```

1. `quick` を削除する
2. `quick` を `slow` に変更する
3. `quick brown fox` をコピーする

**期待される解答例：**

```
1. dw
2. cw → slow → ESC
3. 3yw（または y3w）
```

<details>
<summary>ヒント</summary>

`d`：削除、`c`：削除して挿入モード、`y`：コピー。いずれもモーションと組み合わせます。

</details>

<details>
<summary>解答と解説</summary>

```vim
dw   " d（削除）+ w（次の単語まで）。daw なら空白も適切に処理されます
cw   " c（変更）+ w（単語）。削除して挿入モードに入ります
     " dw → i → slow → ESC より効率的です
3yw  " 3（3回分）+ y（ヤンク）+ w（単語）。p で貼り付けできます

" オペレータの公式: [回数] + オペレータ + モーション
```

</details>

## 応用問題

### 問題4：テキストオブジェクトを使ったコード編集

以下のHTMLで、テキストオブジェクトを使って編集してください。

```html
<div class="container">
    <p>古いテキストがここに入っています</p>
    <a href="https://old-url.com">リンクテキスト</a>
</div>
```

1. `<p>` タグの中身を `新しいテキスト` に書き換える
2. `href` の URL を `https://new-url.com` に書き換える

**期待される出力：**

```html
<div class="container">
    <p>新しいテキスト</p>
    <a href="https://new-url.com">リンクテキスト</a>
</div>
```

<details>
<summary>ヒント</summary>

`cit`：HTMLタグの中身を変更、`ci"`：ダブルクォートの中身を変更です。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 1. <p>タグの中身を書き換え
cit
" c（変更）+ it（inner tag = タグの中身）
" カーソルが <p> 行内のどこにあっても動作します
" 「新しいテキスト」と入力して ESC

" 2. href の URL を書き換え
ci"
" c（変更）+ i"（inner " = ダブルクォートの中身）
" https://new-url.com と入力して ESC

" 他にも使えるテキストオブジェクト:
" ci(  括弧の中身を変更
" ci{  中括弧の中身を変更
" ci[  角括弧の中身を変更
" ci'  シングルクォートの中身を変更
```

</details>

### 問題5：オペレータ+モーションの組み合わせ

以下のPythonコードで、指定された編集を1つのコマンドで行ってください。

```python
def calculate_total(price, tax_rate):
    subtotal = price * 1.0
    tax = subtotal * tax_rate
    total = subtotal + tax
    return total
```

1. カーソルが `subtotal` の `s` にあるとき、`base_price` に変更する
2. カーソルが3行目にあるとき、3〜4行目を削除する
3. カーソルが `return` の行にあるとき、行末の `total` を `result` に変更する

**期待される解答例：**

```
1. ciw → base_price → ESC
2. 2dd
3. $ で行末に移動 → ciw → result → ESC
```

<details>
<summary>ヒント</summary>

`ciw` は単語の途中にカーソルがあっても単語全体を対象にします。`$` は行末への移動です。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 1. subtotal → base_price
ciw
" c + iw（inner word）で単語全体を変更します
" cw との違い: ciw はカーソルが単語の途中でも単語全体が対象です

" 2. 2行を削除
2dd
" 2（回数）+ dd（行削除）。3行目と4行目がまとめて削除されます

" 3. 行末の total → result
$    " 行末に移動
ciw  " カーソル位置の単語（total）を変更
     " result と入力して ESC
```

</details>

## チャレンジ問題

### 問題6：実際のコードをVim文法で効率的に編集

以下のコードを4箇所修正してください。各修正で使用するコマンドを記述してください。

```javascript
function getUserData(id) {
    var user = fetchUser(id);
    var name = user.name;
    var email = user.emal;
    if (age > 0) {
        console.log(name);
    }
    return { name: name, email: email, age: age };
}
```

1. `var` を3箇所すべて `const` に変更（ドットコマンド活用）
2. `emal` を `email` に修正
3. `if (age > 0)` を `if (age >= 18)` に変更
4. `return` 行を `return { name, email, age };` に書き換え

**期待される出力：**

```javascript
function getUserData(id) {
    const user = fetchUser(id);
    const name = user.name;
    const email = user.email;
    if (age >= 18) {
        console.log(name);
    }
    return { name, email, age };
}
```

<details>
<summary>ヒント</summary>

- 修正1：`/var` で検索 → `cw` → `n.` で繰り返し
- 修正2：`/emal` → `ciw` で書き換え
- 修正3：`ci(` でカッコの中身を変更
- 修正4：`ci{` で中括弧の中身を変更

</details>

<details>
<summary>解答と解説</summary>

```vim
" === 修正1: var → const（3箇所）===
/var[Enter]       " 最初の var を検索
cwconst[ESC]      " var を const に変更
n.                " n で次の var にジャンプ、. で同じ変更を繰り返す
n.                " もう1回。3箇所完了

" === 修正2: emal → email ===
/emal[Enter]      " emal を検索
ciwemail[ESC]     " ciw で単語全体を email に変更

" === 修正3: (age > 0) → (age >= 18) ===
/age > 0[Enter]   " 対象箇所を検索
ci(               " カッコの中身を変更
age >= 18[ESC]    " 新しい条件を入力

" === 修正4: return行の書き換え ===
/return[Enter]    " return 行にジャンプ
f{ci{             " f{ で { に移動、ci{ で中括弧の中身を変更
 name, email, age [ESC]

" 使ったテクニックまとめ:
" /検索 + cw + n. : 繰り返し置換の定番パターン
" ciw : 単語単位の書き換え
" ci( : 括弧内の書き換え
" ci{ : 中括弧内の書き換え
```

</details>
