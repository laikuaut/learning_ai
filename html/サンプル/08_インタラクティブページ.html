1. data-* 属性（カスタムデータ属性）

1-1. data属性とCSS属性セレクタの組み合わせ

data属性の値に応じてCSSでスタイルを切り替えます。 ボタンを押すとステータスが変わります。

現在のステータス： アクティブ

ポイント： CSSで [data-status="active"] { color: green; } のように、 data属性の値に応じたスタイルを適用できます。

1-2. data属性によるツールチップ

以下の用語にマウスを乗せると、 HTML、 CSS、 JavaScript のツールチップが表示されます。

ポイント： ::after 疑似要素と content: attr(data-tooltip) で、 data属性の値をツールチップとして表示しています。JavaScriptは不要です。

2. Geolocation API（位置情報の取得）

navigator.geolocation を使って、 ブラウザから現在の位置情報（緯度・経度）を取得できます。 ユーザーの許可が必要です。

取得中...

注意： Geolocation API はユーザーの許可が必要です。 HTTPSまたはlocalhost以外では動作しない場合があります。 また、位置情報はプライバシーに関わるため、取り扱いには十分注意が必要です。

3. localStorage（ブラウザへのデータ保存）

localStorage を使うと、ブラウザにキーと値のペアで データを永続的に保存できます。ページを閉じても、再度開くとデータが残っています。

メモ帳アプリ

保存されたメモ数：0件

ポイント： localStorage.setItem(key, value) でデータを保存し、 localStorage.getItem(key) で読み込みます。 値は文字列のみなので、オブジェクトは JSON.stringify() で変換します。

4. Canvas（グラフィック描画）

<canvas> 要素とJavaScriptを使って、 図形やテキストを描画できます。ゲーム、グラフ、画像処理など幅広く活用されます。
