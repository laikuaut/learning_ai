# 第6章：フォームとバリデーション

## 6.1 フォーム入力の基礎

Vue では `v-model` を使って、さまざまなフォーム要素とデータを双方向にバインドします。

### テキスト入力

```vue
<script setup>
import { ref } from 'vue'

const username = ref('')
const email = ref('')
const password = ref('')
const bio = ref('')
</script>

<template>
  <form @submit.prevent="handleSubmit">
    <!-- テキスト入力 -->
    <div class="form-group">
      <label for="username">ユーザー名</label>
      <input
        id="username"
        v-model="username"
        type="text"
        placeholder="ユーザー名を入力"
        maxlength="20"
      />
      <span class="char-count">{{ username.length }} / 20</span>
    </div>

    <!-- メール入力 -->
    <div class="form-group">
      <label for="email">メールアドレス</label>
      <input id="email" v-model="email" type="email" />
    </div>

    <!-- パスワード入力 -->
    <div class="form-group">
      <label for="password">パスワード</label>
      <input id="password" v-model="password" type="password" />
    </div>

    <!-- テキストエリア -->
    <div class="form-group">
      <label for="bio">自己紹介</label>
      <textarea
        id="bio"
        v-model="bio"
        rows="4"
        placeholder="自己紹介を入力"
      ></textarea>
    </div>
  </form>
</template>
```

### 数値入力

```vue
<script setup>
import { ref } from 'vue'

const age = ref(0)
const price = ref(0)
const rating = ref(3)
</script>

<template>
  <!-- .number 修飾子で数値型に変換 -->
  <input v-model.number="age" type="number" min="0" max="150" />

  <!-- range スライダー -->
  <input v-model.number="rating" type="range" min="1" max="5" />
  <span>{{ rating }} / 5</span>
</template>
```

## 6.2 セレクト、チェックボックス、ラジオボタン

### セレクトボックス

```vue
<script setup>
import { ref } from 'vue'

const selectedCity = ref('')
const selectedSkills = ref([])

const cities = [
  { value: 'tokyo', label: '東京' },
  { value: 'osaka', label: '大阪' },
  { value: 'nagoya', label: '名古屋' },
  { value: 'fukuoka', label: '福岡' },
  { value: 'sapporo', label: '札幌' }

<template>
  <!-- 単一選択 -->
  <div class="form-group">
    <label>都市</label>
    <select v-model="selectedCity">
      <option disabled value="">-- 選択してください --</option>
      <option
        v-for="city in cities"
        :key="city.value"
        :value="city.value"
      >
        {{ city.label }}
      </option>
    </select>

  </div>

  <!-- 複数選択 -->
  <div class="form-group">
    <label>スキル（複数選択可）</label>
    <select v-model="selectedSkills" multiple>
      <option v-for="skill in skills" :key="skill" :value="skill">
        {{ skill }}
      </option>
    </select>
    <p>選択: {{ selectedSkills.join(', ') }}</p>
  </div>
```

### チェックボックス


## 6.3 カスタムバリデーション

### 基本的なバリデーション実装

```vue
<script setup>
import { ref, computed, reactive } from 'vue'

const form = reactive({
  username: '',
  email: '',
  password: '',
  confirmPassword: ''
})

const errors = reactive({
  username: '',
  email: '',
  password: '',
  confirmPassword: ''
})

const touched = reactive({
  username: false,
  email: false,
  password: false,
  confirmPassword: false
})

// バリデーション関数
function validateUsername() {
  if (!form.username) {
    errors.username = 'ユーザー名は必須です'
  } else if (form.username.length < 3) {
    errors.username = 'ユーザー名は3文字以上で入力してください'
  } else if (form.username.length > 20) {
    errors.username = 'ユーザー名は20文字以内で入力してください'
  } else if (!/^[a-zA-Z0-9_]+$/.test(form.username)) {
    errors.username = '半角英数字とアンダースコアのみ使用できます'
  } else {
    errors.username = ''
  }
}

function validateEmail() {
  const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/
  if (!form.email) {
    errors.email = 'メールアドレスは必須です'
  } else if (!emailRegex.test(form.email)) {
    errors.email = '有効なメールアドレスを入力してください'
  } else {
    errors.email = ''
  }
}

function validatePassword() {
  if (!form.password) {
    errors.password = 'パスワードは必須です'
  } else if (form.password.length < 8) {
    errors.password = 'パスワードは8文字以上で入力してください'
  } else if (!/(?=.*[a-z])(?=.*[A-Z])(?=.*\d)/.test(form.password)) {
    errors.password = '大文字、小文字、数字をそれぞれ含めてください'
  } else {
    errors.password = ''
  }
}

function validateConfirmPassword() {
  if (!form.confirmPassword) {
    errors.confirmPassword = '確認用パスワードは必須です'
  } else if (form.confirmPassword !== form.password) {
    errors.confirmPassword = 'パスワードが一致しません'
  } else {
    errors.confirmPassword = ''
  }
}

// フォーム全体の有効性
const isFormValid = computed(() => {
  return (
    form.username &&
    form.email &&
    form.password &&
    form.confirmPassword &&
    !errors.username &&
    !errors.email &&
    !errors.password &&
    !errors.confirmPassword
  )
})

// blur 時にバリデーション実行
function handleBlur(field) {
  touched[field] = true
  switch (field) {
    case 'username': validateUsername(); break
    case 'email': validateEmail(); break
    case 'password': validatePassword(); break
    case 'confirmPassword': validateConfirmPassword(); break
  }
}

// 送信
function handleSubmit() {
  // 全フィールドをバリデーション
  validateUsername()
  validateEmail()
  validatePassword()
  validateConfirmPassword()

  Object.keys(touched).forEach(key => touched[key] = true)

  if (isFormValid.value) {
    console.log('送信データ:', { ...form })
    alert('登録完了！')
  }
}
</script>

<template>
  <form @submit.prevent="handleSubmit" novalidate>
    <div class="form-group" :class="{ error: touched.username && errors.username }">
      <label for="username">ユーザー名 *</label>
      <input
        id="username"
        v-model="form.username"
        @blur="handleBlur('username')"
        type="text"
      />
      <p v-if="touched.username && errors.username" class="error-message">
        {{ errors.username }}
      </p>
    </div>

    <div class="form-group" :class="{ error: touched.email && errors.email }">
      <label for="email">メール *</label>
      <input
        id="email"
        v-model="form.email"
        @blur="handleBlur('email')"
        type="email"
      />
      <p v-if="touched.email && errors.email" class="error-message">
        {{ errors.email }}
      </p>
    </div>

    <div class="form-group" :class="{ error: touched.password && errors.password }">
      <label for="password">パスワード *</label>
      <input
        id="password"
        v-model="form.password"
        @blur="handleBlur('password')"
        type="password"
      />
      <p v-if="touched.password && errors.password" class="error-message">
        {{ errors.password }}
      </p>
    </div>

    <div class="form-group" :class="{ error: touched.confirmPassword && errors.confirmPassword }">
      <label for="confirm">パスワード確認 *</label>
      <input
        id="confirm"
        v-model="form.confirmPassword"
        @blur="handleBlur('confirmPassword')"
        type="password"
      />
      <p v-if="touched.confirmPassword && errors.confirmPassword" class="error-message">
        {{ errors.confirmPassword }}
      </p>
    </div>

    <button type="submit" :disabled="!isFormValid">登録</button>
  </form>
</template>

<style scoped>
.form-group.error input {
  border-color: #e74c3c;
}
.error-message {
  color: #e74c3c;
  font-size: 0.875rem;
  margin-top: 0.25rem;
}
</style>
```

## 6.4 Composable によるバリデーション

バリデーションロジックを composable に切り出して再利用可能にします。

```js
// composables/useFormValidation.js
import { reactive, computed } from 'vue'

export function useFormValidation(rules) {
  const errors = reactive({})
  const touched = reactive({})

  // 各フィールドの初期化
  Object.keys(rules).forEach(field => {
    errors[field] = ''
    touched[field] = false
  })

  // 単一フィールドのバリデーション
  function validateField(field, value) {
    const fieldRules = rules[field]
    if (!fieldRules) return

    for (const rule of fieldRules) {
      const result = rule(value)
      if (result !== true) {
        errors[field] = result  // エラーメッセージ
        return false
      }
    }
    errors[field] = ''
    return true
  }

  // 全フィールドのバリデーション
  function validateAll(formData) {
    let isValid = true
    Object.keys(rules).forEach(field => {
      touched[field] = true
      if (!validateField(field, formData[field])) {
        isValid = false
      }
    })
    return isValid
  }

  // フィールドの blur ハンドラ
  function handleBlur(field, value) {
    touched[field] = true
    validateField(field, value)
  }

  // フォーム全体の有効性
  const isValid = computed(() => {
    return Object.values(errors).every(e => !e) &&
           Object.values(touched).some(t => t)
  })

  return {
    errors,
    touched,
    validateField,
    validateAll,
    handleBlur,
    isValid
  }
}
```

### バリデーションルールの定義

```js
// utils/validationRules.js

// 必須チェック
export const required = (message = '必須項目です') => {
  return (value) => {
    if (!value || (typeof value === 'string' && !value.trim())) {
      return message
    }
    return true
  }
}

// 最小文字数
export const minLength = (min, message) => {
  return (value) => {
    if (value && value.length < min) {
      return message || `${min}文字以上で入力してください`
    }
    return true
  }
}

// 最大文字数
export const maxLength = (max, message) => {
  return (value) => {
    if (value && value.length > max) {
      return message || `${max}文字以内で入力してください`
    }
    return true
  }
}

// メール形式
export const email = (message = '有効なメールアドレスを入力してください') => {
  return (value) => {
    if (value && !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(value)) {
      return message
    }
    return true
  }
}

// パターンマッチ
export const pattern = (regex, message) => {
  return (value) => {
    if (value && !regex.test(value)) {
      return message
    }
    return true
  }
}
```

### 使用例

```vue
<script setup>
import { reactive } from 'vue'
import { useFormValidation } from '@/composables/useFormValidation'
import { required, minLength, maxLength, email } from '@/utils/validationRules'

const form = reactive({
  name: '',
  email: '',
  message: ''
})

const { errors, touched, handleBlur, validateAll, isValid } = useFormValidation({
  name: [required('名前は必須です'), minLength(2), maxLength(50)],
  email: [required('メールは必須です'), email()],
  message: [required('メッセージは必須です'), minLength(10, '10文字以上入力してください')]
})

function onSubmit() {
  if (validateAll(form)) {
    console.log('送信:', form)
  }
}
</script>

<template>
  <form @submit.prevent="onSubmit">
    <div>
      <input
        v-model="form.name"
        @blur="handleBlur('name', form.name)"
        placeholder="名前"
      />
      <span v-if="touched.name && errors.name">{{ errors.name }}</span>
    </div>

    <div>
      <input
        v-model="form.email"
        @blur="handleBlur('email', form.email)"
        placeholder="メール"
      />
      <span v-if="touched.email && errors.email">{{ errors.email }}</span>
    </div>

    <div>
      <textarea
        v-model="form.message"
        @blur="handleBlur('message', form.message)"
        placeholder="メッセージ"
      ></textarea>
      <span v-if="touched.message && errors.message">{{ errors.message }}</span>
    </div>

    <button type="submit" :disabled="!isValid">送信</button>
  </form>
</template>
```

## 6.5 VeeValidate によるバリデーション

大規模なフォームには **VeeValidate** ライブラリが便利です。

### インストール

```bash
npm install vee-validate @vee-validate/rules @vee-validate/i18n
```

### 基本的な使い方

```vue
<script setup>
import { useForm, useField } from 'vee-validate'
import * as yup from 'yup'  // yup と組み合わせることが多い

// yup でバリデーションスキーマを定義
const schema = yup.object({
  name: yup.string().required('名前は必須です').min(2, '2文字以上'),
  email: yup.string().required('メールは必須です').email('有効なメール形式で'),
  age: yup.number().required('年齢は必須です').min(0).max(150)
})

// useForm でフォーム全体を管理
const { handleSubmit, errors, isSubmitting } = useForm({
  validationSchema: schema,
  initialValues: {
    name: '',
    email: '',
    age: null
  }
})

// useField で各フィールドを管理
const { value: name } = useField('name')
const { value: email } = useField('email')
const { value: age } = useField('age')

// 送信処理
const onSubmit = handleSubmit((values) => {
  console.log('送信データ:', values)
})
</script>

<template>
  <form @submit="onSubmit">
    <div>
      <input v-model="name" placeholder="名前" />
      <span v-if="errors.name">{{ errors.name }}</span>
    </div>

    <div>
      <input v-model="email" type="email" placeholder="メール" />
      <span v-if="errors.email">{{ errors.email }}</span>
    </div>

    <div>
      <input v-model.number="age" type="number" placeholder="年齢" />
      <span v-if="errors.age">{{ errors.age }}</span>
    </div>

    <button type="submit" :disabled="isSubmitting">送信</button>
  </form>
</template>
```

## 6.6 動的フォーム

フィールドを動的に追加・削除するフォームの実装パターンです。

```vue
<script setup>
import { ref } from 'vue'

const members = ref([
  { id: 1, name: '', email: '' }
])

let nextId = 2

function addMember() {
  members.value.push({ id: nextId++, name: '', email: '' })
}

function removeMember(id) {
  if (members.value.length > 1) {
    members.value = members.value.filter(m => m.id !== id)
  }
}

function handleSubmit() {
  console.log('メンバー:', members.value)
}
</script>

<template>
  <form @submit.prevent="handleSubmit">
    <div v-for="(member, index) in members" :key="member.id" class="member-row">
      <span>メンバー {{ index + 1 }}</span>
      <input v-model="member.name" placeholder="名前" />
      <input v-model="member.email" placeholder="メール" type="email" />
      <button type="button" @click="removeMember(member.id)" :disabled="members.length <= 1">
        削除
      </button>
    </div>
    <button type="button" @click="addMember">+ メンバーを追加</button>
    <button type="submit">送信</button>
  </form>
</template>
```

## 6.7 本章のまとめ

| 概念 | 説明 |
|------|------|
| `v-model` | フォーム要素との双方向バインディング |
| `.number` `.trim` `.lazy` | v-model の修飾子 |
| カスタムバリデーション | エラー状態を reactive で管理 |
| composable | バリデーションロジックの再利用 |
| VeeValidate | 大規模フォーム向けライブラリ |
| 動的フォーム | フィールドの追加・削除 |
