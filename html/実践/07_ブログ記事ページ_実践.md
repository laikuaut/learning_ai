# 実践課題07：ブログ記事ページ ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第1章〜第7章（HTMLの基本構造、テキストとリンク、画像、リストとテーブル、フォーム、セマンティックHTML、メタ情報とhead）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `<head>` 内のメタ情報（OGP、description、viewport）を正しく設定し、`<article>` を中心としたブログ記事ページを構築する

## 課題の要件

1. `<head>` 内に以下のメタ情報を設定すること
   - `charset`、`viewport`、`description`
   - OGP（Open Graph Protocol）メタタグ（`og:title`、`og:description`、`og:image`、`og:type`）
2. `<article>` で記事本体を囲むこと
3. 記事のメタデータ（投稿日、著者、カテゴリ）を `<time>` や `<address>` で構造化すること
4. 記事内に見出し階層（`<h2>`、`<h3>`）を正しく使うこと
5. `<pre>` と `<code>` でコード例を表示すること
6. `<blockquote>` で引用を表示すること
7. `<aside>` でサイドバーを配置すること
8. 前後の記事へのナビゲーションを含むこと

---

## ステップガイド

<details>
<summary>ステップ1：head内のメタ情報を設定する</summary>

```html
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="HTML初心者向けの入門ガイド。基本構造からタグの使い方まで解説します。">
  
  <!-- OGP (Open Graph Protocol) -->
  <meta property="og:title" content="初心者のためのHTML入門ガイド">
  <meta property="og:description" content="HTML初心者向けの入門ガイド。基本構造からタグの使い方まで解説します。">
  <meta property="og:image" content="https://example.com/images/html-guide.jpg">
  <meta property="og:type" content="article">
  
  <title>初心者のためのHTML入門ガイド | テックブログ</title>
</head>
```

OGPはSNSでシェアされたときに表示されるタイトル・説明文・画像を指定します。

</details>

<details>
<summary>ステップ2：article で記事本体を囲む</summary>

```html
<article>
  <header>
    <h2>初心者のためのHTML入門ガイド</h2>
    <p>
      投稿日: <time datetime="2026-04-01">2026年4月1日</time>
      | カテゴリ: <a href="#">Web開発</a>
    </p>
  </header>
  
  <!-- 記事本文 -->
  
  <footer>
    <p>タグ: <a href="#">#HTML</a> <a href="#">#初心者</a></p>
  </footer>
</article>
```

`<article>` 内にも `<header>` と `<footer>` を使えます。これは記事自体のヘッダー/フッターです。

</details>

<details>
<summary>ステップ3：pre と code でコードを表示する</summary>

```html
<pre><code>&lt;!DOCTYPE html&gt;
&lt;html lang="ja"&gt;
&lt;head&gt;
  &lt;meta charset="utf-8"&gt;
  &lt;title&gt;My Page&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
  &lt;h1&gt;Hello World&lt;/h1&gt;
&lt;/body&gt;
&lt;/html&gt;</code></pre>
```

HTMLタグをそのまま表示するには、`<` を `&lt;`、`>` を `&gt;` にエスケープ（escape）する必要があります。

</details>

- OGPに加えてTwitter Card メタタグを追加 → SNSシェア時の表示が最適化
