## 課題の要件

1. 複数行のテキストを標準入力から受け取る（空行で入力終了）
2. 総行数・総単語数・ユニーク単語数を集計する
3. 各単語の出現回数をカウントする
4. 最も出現回数が多い単語を表示する
5. 出現回数の多い順にソートして表示する

---

## ステップガイド

<details>
<summary>ステップ1：複数行の入力を受け取る</summary>

`loop` でテキストを1行ずつ読み取り、空行が来たら終了するようにします。

```rust
use std::io;

fn main() {
    println!("テキストを入力してください（空行で終了）:");

    let mut lines: Vec<String> = Vec::new();

    loop {
        let mut line = String::new();
        io::stdin().read_line(&mut line).expect("入力エラー");
        let line = line.trim().to_string();

        if line.is_empty() {
            break;
        }
        lines.push(line);
    }
}
```

</details>

<details>
<summary>ステップ2：単語に分割してカウントする</summary>

各行を `.split_whitespace()` で単語に分割し、`Vec<(String, usize)>` で出現回数を管理します。
（標準ライブラリの `HashMap` を使わないバージョンです）

```rust
let mut word_counts: Vec<(String, usize)> = Vec::new();

for line in &lines {
    for word in line.split_whitespace() {
        let word = word.to_string();
        // 既存の単語を探す
        let mut found = false;
        for entry in &mut word_counts {
            if entry.0 == word {
                entry.1 += 1;
                found = true;
                break;
            }
        }
        if !found {
            word_counts.push((word, 1));
        }
    }
}
```

</details>

<details>
<summary>ステップ3：ソートして結果を表示する</summary>

`sort_by` を使って出現回数の降順にソートします。

```rust
word_counts.sort_by(|a, b| b.1.cmp(&a.1));

for (word, count) in &word_counts {
    println!("  {:<14}: {}回", word, count);
}
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```rust
// 単語カウンタ
// 学べる内容：Vec、文字列操作、ループ、ソート
// 実行方法：rustc main.rs && ./main

use std::io;

fn main() {
    println!("===== 単語カウンタ =====");
    println!("テキストを入力してください（空行で終了）:");

    // --- 複数行の入力を受け取る ---
    let mut lines: Vec<String> = Vec::new();

    loop {
        let mut line = String::new();
        io::stdin().read_line(&mut line).expect("入力エラー");
        let trimmed = line.trim().to_string();

        if trimmed.is_empty() {
            break;
        }
        lines.push(trimmed);
    }

    if lines.is_empty() {
        println!("テキストが入力されませんでした。");
        return;
    }

    // --- 単語をカウントする ---
    let mut word_counts: Vec<(String, usize)> = Vec::new();
    let mut total_words: usize = 0;

    for line in &lines {
        for word in line.split_whitespace() {
            total_words += 1;
            let word = word.to_string();

            let mut found = false;
            for entry in &mut word_counts {
                if entry.0 == word {
                    entry.1 += 1;
                    found = true;
                    break;
                }
            }
            if !found {
                word_counts.push((word, 1));
            }
        }
    }

    // --- ソート（出現回数の降順）---
    word_counts.sort_by(|a, b| b.1.cmp(&a.1));

    // --- 結果表示 ---
    let unique_words = word_counts.len();
    let top_word = &word_counts[0];

    println!("\n--- 解析結果 ---");
    println!("総行数: {}", lines.len());
    println!("総単語数: {}", total_words);
    println!("ユニーク単語数: {}", unique_words);
    println!("最頻出単語: \"{}\" ({}回)", top_word.0, top_word.1);

    println!("\n--- 単語出現回数 ---");
    for (word, count) in &word_counts {
        println!("  {:<14}: {}回", word, count);
    }
}
```

</details>

<details>
<summary>解答例（改良版 ─ HashMapとイテレータを活用）</summary>

標準ライブラリの `HashMap` を使うと、単語カウントがはるかに簡潔になります。

```rust
// 単語カウンタ（改良版）
// HashMapとイテレータチェーンを活用したバージョン
// 実行方法：rustc main.rs && ./main

use std::collections::HashMap;
use std::io;

/// 標準入力から複数行を読み取ります（空行で終了）
fn read_lines() -> Vec<String> {
    let mut lines = Vec::new();
    loop {
        let mut line = String::new();
        io::stdin().read_line(&mut line).expect("入力エラー");
        let trimmed = line.trim().to_string();
        if trimmed.is_empty() {
            break;
        }
        lines.push(trimmed);
    }
    lines
}

/// 行のスライスから単語の出現回数を集計します
fn count_words(lines: &[String]) -> HashMap<String, usize> {
    let mut counts = HashMap::new();
    for line in lines {
        for word in line.split_whitespace() {
            *counts.entry(word.to_string()).or_insert(0) += 1;
        }
    }
    counts
}

fn main() {
    println!("===== 単語カウンタ =====");
    println!("テキストを入力してください（空行で終了）:");

    let lines = read_lines();

    if lines.is_empty() {
        println!("テキストが入力されませんでした。");
        return;
    }

    let word_counts = count_words(&lines);

    // 総単語数を計算
    let total_words: usize = word_counts.values().sum();

    // 出現回数の降順にソート
    let mut sorted: Vec<(&String, &usize)> = word_counts.iter().collect();
    sorted.sort_by(|a, b| b.1.cmp(a.1));

    // 最頻出単語
    let (top_word, top_count) = sorted[0];

    println!("\n--- 解析結果 ---");
    println!("総行数: {}", lines.len());
    println!("総単語数: {}", total_words);
    println!("ユニーク単語数: {}", word_counts.len());
    println!("最頻出単語: \"{}\" ({}回)", top_word, top_count);

    println!("\n--- 単語出現回数 ---");
    for (word, count) in &sorted {
        println!("  {:<14}: {}回", word, count);
    }
}
```

**初心者向けとの違い:**
- `HashMap` で単語カウントを O(1) で実現（Vecの線形探索 → ハッシュテーブル）
- `entry().or_insert(0)` パターンで「存在しなければ初期値を挿入」を簡潔に記述
- 処理を `read_lines()` と `count_words()` に関数分離 → テスト可能で再利用しやすい
- `&[String]` スライス参照で所有権を渡さずにデータを利用

</details>
