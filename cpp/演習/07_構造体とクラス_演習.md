## 問題1：座標を表す構造体（基本）

2次元座標を表す構造体 `Point` を作成してください。以下の機能を実装します。

- メンバ変数: `x`, `y`（double型）
- 原点からの距離を返すメンバ関数 `distance()`
- 座標を表示するメンバ関数 `print()`

**期待される出力例：**
```
点1: (3, 4)
原点からの距離: 5
点2: (1, 1)
原点からの距離: 1.41421
```

<details>
<summary>ヒント</summary>

- 原点からの距離は `sqrt(x*x + y*y)` で求められます
- `<cmath>` の `sqrt` を使ってください
- 構造体のメンバ関数は `struct` の中に書けます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <cmath>
using namespace std;

struct Point {
    double x, y;

    // 原点からの距離を返す
    double distance() const {
        return sqrt(x * x + y * y);
    }

    // 座標を表示する
    void print() const {
        cout << "(" << x << ", " << y << ")" << endl;
    }
};

int main() {
    Point p1 = {3.0, 4.0};
    cout << "点1: ";
    p1.print();
    cout << "原点からの距離: " << p1.distance() << endl;

    Point p2 = {1.0, 1.0};
    cout << "点2: ";
    p2.print();
    cout << "原点からの距離: " << p2.distance() << endl;

    return 0;
}
```

</details>

## 問題2：銀行口座クラス（基本）

銀行口座を管理するクラス `BankAccount` を作成してください。

- **メンバ変数（private）**: 口座名義（string）、残高（int）
- **コンストラクタ**: 口座名義と初期残高を受け取る
- **メンバ関数**:
  - `deposit(int amount)` — 入金
  - `withdraw(int amount)` — 出金（残高不足の場合はエラーメッセージ）
  - `showBalance()` — 残高表示

**期待される出力例：**
```
[山田太郎] 残高: 10000円
5000円を入金しました
[山田太郎] 残高: 15000円
3000円を出金しました
[山田太郎] 残高: 12000円
残高不足です（残高: 12000円、出金額: 20000円）
```

<details>
<summary>ヒント</summary>

- `private` セクションにメンバ変数を置き、`public` セクションにメンバ関数を置きます
- コンストラクタはクラス名と同じ名前の関数です
- `withdraw` では、出金額が残高を超えていないかチェックします

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
using namespace std;

class BankAccount {
private:
    string owner;   // 口座名義
    int balance;    // 残高

public:
    // コンストラクタ
    BankAccount(const string& name, int initialBalance)
        : owner(name), balance(initialBalance) {}

    // 入金
    void deposit(int amount) {
        balance += amount;
        cout << amount << "円を入金しました" << endl;
    }

    // 出金
    void withdraw(int amount) {
        if (amount > balance) {
            cout << "残高不足です（残高: " << balance
                 << "円、出金額: " << amount << "円）" << endl;
        } else {
            balance -= amount;
            cout << amount << "円を出金しました" << endl;
        }
    }

    // 残高表示
    void showBalance() const {
        cout << "[" << owner << "] 残高: " << balance << "円" << endl;
    }
};

int main() {
    BankAccount account("山田太郎", 10000);
    account.showBalance();

    account.deposit(5000);
    account.showBalance();

    account.withdraw(3000);
    account.showBalance();

    account.withdraw(20000);  // 残高不足

    return 0;
}
```

</details>

## 問題3：日付クラス（基本）

日付を表すクラス `Date` を作成してください。

- **メンバ変数（private）**: 年、月、日（int）
- **コンストラクタ**: 年月日を受け取る
- **メンバ関数**:
  - `print()` — "YYYY/MM/DD" 形式で表示
  - `isLeapYear()` — 閏年かどうかを返す
  - `daysInMonth()` — その月の日数を返す

**期待される出力例：**
```
日付: 2024/02/29
閏年: はい
2月の日数: 29
日付: 2023/04/15
閏年: いいえ
4月の日数: 30
```

<details>
<summary>ヒント</summary>

- 閏年の条件: 4で割り切れ、かつ100で割り切れない。ただし400で割り切れれば閏年
- `<iomanip>` の `setfill('0')` と `setw(2)` で0埋め表示ができます
- 月の日数は配列で管理すると便利です（2月は閏年で変わります）

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

class Date {
private:
    int year, month, day;

public:
    // コンストラクタ
    Date(int y, int m, int d) : year(y), month(m), day(d) {}

    // YYYY/MM/DD形式で表示
    void print() const {
        cout << year << "/"
             << setfill('0') << setw(2) << month << "/"
             << setfill('0') << setw(2) << day << endl;
    }

    // 閏年判定
    bool isLeapYear() const {
        if (year % 400 == 0) return true;
        if (year % 100 == 0) return false;
        if (year % 4 == 0) return true;
        return false;
    }

    // その月の日数を返す
    int daysInMonth() const {
        int days[] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
        if (month == 2 && isLeapYear()) return 29;
        return days[month];
    }
};

int main() {
    Date d1(2024, 2, 29);
    cout << "日付: ";
    d1.print();
    cout << "閏年: " << (d1.isLeapYear() ? "はい" : "いいえ") << endl;
    cout << "2月の日数: " << d1.daysInMonth() << endl;

    Date d2(2023, 4, 15);
    cout << "日付: ";
    d2.print();
    cout << "閏年: " << (d2.isLeapYear() ? "はい" : "いいえ") << endl;
    cout << "4月の日数: " << d2.daysInMonth() << endl;

    return 0;
}
```

</details>

## 問題4：分数クラスと演算子オーバーロード（応用）

分数を表すクラス `Fraction` を作成し、以下の演算子をオーバーロードしてください。

- `+` — 分数の足し算
- `-` — 分数の引き算
- `*` — 分数の掛け算
- `==` — 等価比較
- `<<`（friend関数）— ストリーム出力

結果は常に約分（最大公約数で割る）してください。

**期待される出力例：**
```
1/2 + 1/3 = 5/6
3/4 - 1/4 = 1/2
2/3 * 3/5 = 2/5
1/2 == 2/4 → 等しい
```

<details>
<summary>ヒント</summary>

- 最大公約数（GCD）を求める関数を用意し、コンストラクタや演算後に約分します
- `a/b + c/d = (a*d + b*c) / (b*d)` です
- `operator<<` は `friend` 関数として定義します
- 負の分数は分子のみ負にする慣例に従います

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <cstdlib>
using namespace std;

class Fraction {
private:
    int num;   // 分子
    int den;   // 分母

    // 最大公約数を求める
    static int gcd(int a, int b) {
        a = abs(a);
        b = abs(b);
        while (b != 0) {
            int t = b;
            b = a % b;
            a = t;
        }
        return a;
    }

    // 約分する
    void reduce() {
        if (den < 0) {  // 分母が負なら符号を反転
            num = -num;
            den = -den;
        }
        int g = gcd(num, den);
        num /= g;
        den /= g;
    }

public:
    // コンストラクタ
    Fraction(int n = 0, int d = 1) : num(n), den(d) {
        reduce();
    }

    // 足し算
    Fraction operator+(const Fraction& other) const {
        return Fraction(num * other.den + den * other.num,
                        den * other.den);
    }

    // 引き算
    Fraction operator-(const Fraction& other) const {
        return Fraction(num * other.den - den * other.num,
                        den * other.den);
    }

    // 掛け算
    Fraction operator*(const Fraction& other) const {
        return Fraction(num * other.num, den * other.den);
    }

    // 等価比較（約分済みなので直接比較できる）
    bool operator==(const Fraction& other) const {
        return num == other.num && den == other.den;
    }

    // ストリーム出力
    friend ostream& operator<<(ostream& os, const Fraction& f) {
        os << f.num << "/" << f.den;
        return os;
    }
};

int main() {
    Fraction a(1, 2), b(1, 3);
    cout << a << " + " << b << " = " << (a + b) << endl;

    Fraction c(3, 4), d(1, 4);
    cout << c << " - " << d << " = " << (c - d) << endl;

    Fraction e(2, 3), f(3, 5);
    cout << e << " * " << f << " = " << (e * f) << endl;

    Fraction g(1, 2), h(2, 4);
    cout << g << " == " << h << " → "
         << (g == h ? "等しい" : "等しくない") << endl;

    return 0;
}
```

</details>

## 問題5：配列ベースのスタック実装（応用）

固定サイズの配列を使ったスタック（LIFO）クラス `Stack` を作成してください。

- **操作**: `push`, `pop`, `top`, `isEmpty`, `isFull`, `print`
- スタックオーバーフロー・アンダーフローをチェックすること

**期待される出力例：**
```
push: 10, 20, 30
スタックの内容: [30, 20, 10]
top: 30
pop: 30
pop: 20
スタックの内容: [10]
isEmpty: いいえ
pop: 10
isEmpty: はい
pop: スタックが空です
```

<details>
<summary>ヒント</summary>

- 固定サイズの配列と、現在の要素数を管理する変数を使います
- `push` はスタックが満杯でないか確認してから追加します
- `pop` はスタックが空でないか確認してから取り出します
- `top` は配列の最後の要素（`data[count-1]`）を返します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

class Stack {
private:
    static const int MAX_SIZE = 100;
    int data[MAX_SIZE];  // データ格納用配列
    int count;           // 現在の要素数

public:
    // コンストラクタ
    Stack() : count(0) {}

    // 要素をプッシュ
    void push(int value) {
        if (isFull()) {
            cout << "スタックが満杯です" << endl;
            return;
        }
        data[count++] = value;
    }

    // 要素をポップ
    int pop() {
        if (isEmpty()) {
            cout << "スタックが空です" << endl;
            return -1;
        }
        return data[--count];
    }

    // 先頭要素を参照
    int top() const {
        if (isEmpty()) {
            cout << "スタックが空です" << endl;
            return -1;
        }
        return data[count - 1];
    }

    // 空かどうか
    bool isEmpty() const {
        return count == 0;
    }

    // 満杯かどうか
    bool isFull() const {
        return count == MAX_SIZE;
    }

    // スタックの内容を表示
    void print() const {
        cout << "[";
        for (int i = count - 1; i >= 0; i--) {
            if (i < count - 1) cout << ", ";
            cout << data[i];
        }
        cout << "]" << endl;
    }
};

int main() {
    Stack s;

    cout << "push: 10, 20, 30" << endl;
    s.push(10);
    s.push(20);
    s.push(30);

    cout << "スタックの内容: ";
    s.print();

    cout << "top: " << s.top() << endl;

    cout << "pop: " << s.pop() << endl;
    cout << "pop: " << s.pop() << endl;

    cout << "スタックの内容: ";
    s.print();

    cout << "isEmpty: " << (s.isEmpty() ? "はい" : "いいえ") << endl;

    cout << "pop: " << s.pop() << endl;
    cout << "isEmpty: " << (s.isEmpty() ? "はい" : "いいえ") << endl;

    cout << "pop: ";
    s.pop();

    return 0;
}
```

</details>

## 問題6：学生管理システム（チャレンジ）

学生情報を管理するシステムを作成してください。

**Student クラス:**
- 名前（string）、学籍番号（int）、3科目の点数（int配列）
- 平均点を返す `average()` メンバ関数
- 情報を表示する `print()` メンバ関数

**StudentManager クラス:**
- 学生を追加する `addStudent()`
- 全学生を表示する `printAll()`
- 平均点の最も高い学生を返す `getTopStudent()`

`vector` を使って可変長のリストを管理してください。

**期待される出力例：**
```
=== 全学生一覧 ===
[1001] 佐藤一郎: 国語=80, 数学=90, 英語=85 (平均: 85)
[1002] 鈴木花子: 国語=95, 数学=70, 英語=90 (平均: 85)
[1003] 田中次郎: 国語=60, 数学=100, 英語=75 (平均: 78.3333)

=== 最優秀学生 ===
[1001] 佐藤一郎: 国語=80, 数学=90, 英語=85 (平均: 85)
```

<details>
<summary>ヒント</summary>

- `Student` クラスに3科目の名前を持たせると表示が分かりやすくなります
- `StudentManager` は `vector<Student>` でリストを管理します
- `getTopStudent` は `vector` を走査して最大平均点の学生を見つけます
- 平均点が同じ場合は最初に見つかった学生を返すとよいでしょう

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

class Student {
private:
    string name;
    int id;
    int scores[3];  // 国語、数学、英語

public:
    // コンストラクタ
    Student(const string& n, int i, int s1, int s2, int s3)
        : name(n), id(i) {
        scores[0] = s1;
        scores[1] = s2;
        scores[2] = s3;
    }

    // 平均点を返す
    double average() const {
        return (scores[0] + scores[1] + scores[2]) / 3.0;
    }

    // 情報を表示
    void print() const {
        cout << "[" << id << "] " << name << ": "
             << "国語=" << scores[0] << ", "
             << "数学=" << scores[1] << ", "
             << "英語=" << scores[2]
             << " (平均: " << average() << ")" << endl;
    }
};

class StudentManager {
private:
    vector<Student> students;

public:
    // 学生を追加
    void addStudent(const Student& s) {
        students.push_back(s);
    }

    // 全学生を表示
    void printAll() const {
        for (const auto& s : students) {
            s.print();
        }
    }

    // 最優秀学生を返す
    const Student& getTopStudent() const {
        int bestIndex = 0;
        for (int i = 1; i < (int)students.size(); i++) {
            if (students[i].average() > students[bestIndex].average()) {
                bestIndex = i;
            }
        }
        return students[bestIndex];
    }
};

int main() {
    StudentManager manager;
    manager.addStudent(Student("佐藤一郎", 1001, 80, 90, 85));
    manager.addStudent(Student("鈴木花子", 1002, 95, 70, 90));
    manager.addStudent(Student("田中次郎", 1003, 60, 100, 75));

    cout << "=== 全学生一覧 ===" << endl;
    manager.printAll();

    cout << "\n=== 最優秀学生 ===" << endl;
    manager.getTopStudent().print();

    return 0;
}
```

</details>

## 問題7：テンプレートクラスで汎用ペア（チャレンジ）

2つの異なる型の値を保持するテンプレートクラス `MyPair` を作成してください。
（`std::pair` を自作するイメージです。）

- テンプレート引数: `T1`, `T2`
- メンバ: `first`（T1型）、`second`（T2型）
- `print()` メンバ関数で `(first, second)` の形式で表示

**期待される出力例：**
```
整数と文字列: (1, hello)
浮動小数と真偽値: (3.14, 1)
文字列と文字列: (key, value)
```

<details>
<summary>ヒント</summary>

- テンプレートクラスの宣言は `template<typename T1, typename T2>` で始めます
- メンバ関数もテンプレートクラス内で定義するのが簡単です
- `bool` 型は `cout` で出力すると `1` または `0` になります

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
using namespace std;

template<typename T1, typename T2>
class MyPair {
public:
    T1 first;
    T2 second;

    // コンストラクタ
    MyPair(const T1& f, const T2& s) : first(f), second(s) {}

    // ペアを表示
    void print() const {
        cout << "(" << first << ", " << second << ")" << endl;
    }
};

int main() {
    MyPair<int, string> p1(1, "hello");
    cout << "整数と文字列: ";
    p1.print();

    MyPair<double, bool> p2(3.14, true);
    cout << "浮動小数と真偽値: ";
    p2.print();

    MyPair<string, string> p3("key", "value");
    cout << "文字列と文字列: ";
    p3.print();

    return 0;
}
```

</details>
