# 実践課題10：リファクタリング ─ レガシー設定スクリプト ★4

> **難易度**: ★★★★☆（上級）
> **前提知識**: 第2章（変数と展開）、第3章（条件分岐）、第5章（関数）、第8章（実践的なスクリプト開発）
> **課題の種類**: リファクタリング
> **学習目標**: 動くが保守性の低いスクリプトを読み解き、モダンなBashの書き方にリファクタリングする力を養う

---

## 課題の説明

以下のスクリプトは「サーバーの初期設定を行うスクリプト」です。動作はしますが、以下の問題を抱えています。

**問題点:**
1. 関数化されておらず、すべてがトップレベルに書かれている
2. エラーハンドリングがない（`set -euo pipefail` なし）
3. クォーティングが不十分
4. コマンドの存在チェックがない
5. ログ出力の仕組みがない
6. ハードコーディングされた値が散在
7. 同じパターンのコードが重複している
8. 終了コードが考慮されていない

これらの問題をすべて修正し、保守性の高いスクリプトにリファクタリングしてください。

---

## リファクタリング対象コード

```bash
#!/bin/bash

# サーバー初期設定スクリプト

echo "サーバー初期設定を開始します"

# ユーザー作成
echo "ユーザーを作成しています..."
useradd -m -s /bin/bash appuser
echo "appuser:P@ssw0rd123" | chpasswd
usermod -aG sudo appuser
echo "ユーザー appuser を作成しました"

# 必要なパッケージのインストール
echo "パッケージをインストールしています..."
apt-get update
apt-get install -y git curl wget vim htop tmux
apt-get install -y nginx
apt-get install -y nodejs npm
echo "パッケージのインストール完了"

# Nginx設定
echo "Nginxを設定しています..."
cat > /etc/nginx/sites-available/myapp <<EOF
server {
    listen 80;
    server_name myapp.example.com;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade \$http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host \$host;
    }
}
EOF
ln -sf /etc/nginx/sites-available/myapp /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t
systemctl restart nginx
echo "Nginx設定完了"

# ファイアウォール設定
echo "ファイアウォールを設定しています..."
ufw allow 22/tcp
ufw allow 80/tcp
ufw allow 443/tcp
ufw --force enable
echo "ファイアウォール設定完了"

# SSH設定
echo "SSHを設定しています..."
sed -i 's/#PasswordAuthentication yes/PasswordAuthentication no/' /etc/ssh/sshd_config
sed -i 's/#PermitRootLogin prohibit-password/PermitRootLogin no/' /etc/ssh/sshd_config
systemctl restart sshd
echo "SSH設定完了"

# スワップファイル作成
echo "スワップファイルを作成しています..."
dd if=/dev/zero of=/swapfile bs=1M count=2048
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo "/swapfile none swap sw 0 0" >> /etc/fstab
echo "スワップファイル作成完了"

# タイムゾーン設定
timedatectl set-timezone Asia/Tokyo
echo "タイムゾーンをAsia/Tokyoに設定しました"

echo "サーバー初期設定が完了しました"
```

## リファクタリングの要件

1. **関数化**: 各セクション（ユーザー作成、パッケージ、Nginx等）を独立した関数にする
2. **堅牢化**: `set -euo pipefail`、エラーハンドリング、`trap` を追加する
3. **設定の外部化**: ハードコーディングされた値を変数または設定ファイルに切り出す
4. **ログ機能**: タイムスタンプ付きのログ出力関数を作成する
5. **冪等性（idempotency）**: 2回実行しても問題ないようにする（既にユーザーが存在する場合はスキップ等）
6. **コマンド存在チェック**: 必要なコマンドが利用可能か事前に確認する
7. **root権限チェック**: root以外で実行された場合にエラーにする
8. **ドライランモード**: `-n` オプションで実行せずに確認できるようにする

<details>
<summary>解答例（初心者向け ─ 基本的なリファクタリング）</summary>

```bash
#!/usr/bin/env bash
# サーバー初期設定スクリプト（リファクタリング版）
# 学べる内容：関数化、エラーハンドリング、冪等性、設定の外部化
# 実行方法：sudo ./server_setup.sh [-n]

set -euo pipefail

# ============================================================
# 設定
# ============================================================
APP_USER="appuser"
APP_PORT=3000
SERVER_NAME="myapp.example.com"
TIMEZONE="Asia/Tokyo"
SWAP_SIZE_MB=2048
PACKAGES=(git curl wget vim htop tmux nginx nodejs npm)
ALLOWED_PORTS=(22 80 443)
DRY_RUN=false

# ============================================================
# ユーティリティ
# ============================================================
log() {
    local level="$1"; shift
    echo "[$(date '+%Y-%m-%d %H:%M:%S')] [$level] $*" >&2
}

run() {
    if $DRY_RUN; then
        log "DRY" "$*"
        return 0
    fi
    "$@"
}

# ============================================================
# オプション解析
# ============================================================
while getopts ":n" opt; do
    case "$opt" in
        n) DRY_RUN=true ;;
        *) echo "使い方: $0 [-n]" >&2; exit 1 ;;
    esac
done

# ============================================================
# 事前チェック
# ============================================================
if [ "$(id -u)" -ne 0 ] && ! $DRY_RUN; then
    log "ERROR" "rootで実行してください: sudo $0"
    exit 1
fi

# ============================================================
# ユーザー作成
# ============================================================
setup_user() {
    log "INFO" "--- ユーザー設定 ---"

    if id "$APP_USER" &>/dev/null; then
        log "INFO" "ユーザー $APP_USER は既に存在します。スキップ"
        return 0
    fi

    run useradd -m -s /bin/bash "$APP_USER"
    log "INFO" "ユーザー $APP_USER を作成しました"
    log "WARN" "パスワードは手動で設定してください: sudo passwd $APP_USER"
    run usermod -aG sudo "$APP_USER"
}

# ============================================================
# パッケージインストール
# ============================================================
install_packages() {
    log "INFO" "--- パッケージインストール ---"

    run apt-get update -qq

    for pkg in "${PACKAGES[@]}"; do
        if dpkg -l "$pkg" &>/dev/null; then
            log "INFO" "$pkg は既にインストール済み。スキップ"
        else
            log "INFO" "$pkg をインストールしています..."
            run apt-get install -y -qq "$pkg"
        fi
    done

    log "INFO" "パッケージインストール完了"
}

# ============================================================
# Nginx設定
# ============================================================
setup_nginx() {
    log "INFO" "--- Nginx設定 ---"

    local config_file="/etc/nginx/sites-available/myapp"

    if [ -f "$config_file" ]; then
        log "INFO" "Nginx設定は既に存在します。スキップ"
        return 0
    fi

    run bash -c "cat > '$config_file' <<NGINX_EOF
server {
    listen 80;
    server_name ${SERVER_NAME};

    location / {
        proxy_pass http://localhost:${APP_PORT};
        proxy_http_version 1.1;
        proxy_set_header Upgrade \\\$http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host \\\$host;
    }
}
NGINX_EOF"

    run ln -sf "$config_file" /etc/nginx/sites-enabled/
    run rm -f /etc/nginx/sites-enabled/default

    if ! $DRY_RUN; then
        nginx -t || { log "ERROR" "Nginx設定にエラーがあります"; return 1; }
    fi

    run systemctl restart nginx
    log "INFO" "Nginx設定完了"
}

# ============================================================
# ファイアウォール設定
# ============================================================
setup_firewall() {
    log "INFO" "--- ファイアウォール設定 ---"

    if ! command -v ufw &>/dev/null; then
        log "WARN" "ufwが見つかりません。スキップ"
        return 0
    fi

    for port in "${ALLOWED_PORTS[@]}"; do
        run ufw allow "${port}/tcp"
    done

    run ufw --force enable
    log "INFO" "ファイアウォール設定完了"
}

# ============================================================
# SSH設定
# ============================================================
setup_ssh() {
    log "INFO" "--- SSH設定 ---"

    local sshd_config="/etc/ssh/sshd_config"
    if [ ! -f "$sshd_config" ]; then
        log "WARN" "sshd_config が見つかりません。スキップ"
        return 0
    fi

    # バックアップ（冪等性: 既にバックアップがあればスキップ）
    if [ ! -f "${sshd_config}.bak" ]; then
        run cp "$sshd_config" "${sshd_config}.bak"
    fi

    run sed -i 's/#\?PasswordAuthentication yes/PasswordAuthentication no/' "$sshd_config"
    run sed -i 's/#\?PermitRootLogin .*/PermitRootLogin no/' "$sshd_config"

    run systemctl restart sshd
    log "INFO" "SSH設定完了"
}

# ============================================================
# スワップファイル作成
# ============================================================
setup_swap() {
    log "INFO" "--- スワップ設定 ---"

    if [ -f /swapfile ]; then
        log "INFO" "スワップファイルは既に存在します。スキップ"
        return 0
    fi

    run dd if=/dev/zero of=/swapfile bs=1M count="$SWAP_SIZE_MB" status=progress
    run chmod 600 /swapfile
    run mkswap /swapfile
    run swapon /swapfile

    # fstabに追記（重複チェック）
    if ! grep -q '/swapfile' /etc/fstab 2>/dev/null; then
        echo "/swapfile none swap sw 0 0" >> /etc/fstab
    fi

    log "INFO" "スワップファイル作成完了 (${SWAP_SIZE_MB}MB)"
}

# ============================================================
# タイムゾーン設定
# ============================================================
setup_timezone() {
    log "INFO" "--- タイムゾーン設定 ---"

    local current_tz
    current_tz=$(timedatectl show --value -p Timezone 2>/dev/null || echo "")

    if [ "$current_tz" = "$TIMEZONE" ]; then
        log "INFO" "タイムゾーンは既に $TIMEZONE です。スキップ"
        return 0
    fi

    run timedatectl set-timezone "$TIMEZONE"
    log "INFO" "タイムゾーンを $TIMEZONE に設定しました"
}

# ============================================================
# メイン処理
# ============================================================
main() {
    log "INFO" "===== サーバー初期設定を開始します ====="
    if $DRY_RUN; then
        log "INFO" "（ドライランモード: 実際の変更は行いません）"
    fi

    setup_user
    install_packages
    setup_nginx
    setup_firewall
    setup_ssh
    setup_swap
    setup_timezone

    log "INFO" "===== サーバー初期設定が完了しました ====="
}

main "$@"
```

<summary>解答例（改良版 ─ 設定ファイル対応とログファイル出力）</summary>

```bash
#!/usr/bin/env bash
# サーバー初期設定スクリプト（改良版）
# 設定ファイル対応、ログファイル出力、セクション選択機能付き
# 実行方法：sudo ./server_setup.sh [-n] [-c config.conf] [-s user,pkg,nginx]

set -euo pipefail

readonly SCRIPT_NAME="$(basename "$0")"
readonly SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
readonly LOG_FILE="/var/log/server_setup_$(date '+%Y%m%d_%H%M%S').log"

# ============================================================
# デフォルト設定
# ============================================================
APP_USER="${APP_USER:-appuser}"
APP_PORT="${APP_PORT:-3000}"
SERVER_NAME="${SERVER_NAME:-myapp.example.com}"
TIMEZONE="${TIMEZONE:-Asia/Tokyo}"
SWAP_SIZE_MB="${SWAP_SIZE_MB:-2048}"
PACKAGES=(git curl wget vim htop tmux nginx nodejs npm)
ALLOWED_PORTS=(22 80 443)
DRY_RUN=false
SECTIONS="all"

# ============================================================
# ユーティリティ
# ============================================================
log() {
    local level="$1"; shift
    local message="$*"
    local timestamp
    timestamp=$(date '+%Y-%m-%d %H:%M:%S')
    local entry="[$timestamp] [$level] $message"

    # 標準エラーに出力（色付き）
    local color='\033[0m'
    if [ -t 2 ]; then
        case "$level" in
            INFO)  color='\033[32m' ;;
            WARN)  color='\033[33m' ;;
            ERROR) color='\033[31m' ;;
            DRY)   color='\033[36m' ;;
        esac
    fi
    echo -e "${color}${entry}\033[0m" >&2

    # ログファイルに記録
    if ! $DRY_RUN && [ -w "$(dirname "$LOG_FILE")" ]; then
        echo "$entry" >> "$LOG_FILE"
    fi
}

run() {
    if $DRY_RUN; then
        log "DRY" "$*"
        return 0
    fi
    "$@"
}

die() {
    log "ERROR" "$@"
    exit 1
}

section_enabled() {
    [ "$SECTIONS" = "all" ] || [[ ",$SECTIONS," == *",$1,"* ]]
}

require_command() {
    local cmd="$1"
    if ! command -v "$cmd" &>/dev/null; then
        die "必要なコマンドが見つかりません: $cmd"
    fi
}

# ============================================================
# 設定ファイル読み込み
# ============================================================
load_config() {
    local config_file="$1"
    [ -f "$config_file" ] || return 0
    log "INFO" "設定ファイル読み込み: $config_file"

    while IFS='=' read -r key value; do
        [[ "$key" =~ ^[[:space:]]*# ]] && continue
        [[ -z "$key" ]] && continue
        key=$(echo "$key" | sed 's/^[[:space:]]*//;s/[[:space:]]*$//')
        value=$(echo "$value" | sed 's/^[[:space:]]*//;s/[[:space:]]*$//')
        case "$key" in
            app_user)    APP_USER="$value" ;;
            app_port)    APP_PORT="$value" ;;
            server_name) SERVER_NAME="$value" ;;
            timezone)    TIMEZONE="$value" ;;
            swap_size)   SWAP_SIZE_MB="$value" ;;
        esac
    done < "$config_file"
}

# ============================================================
# オプション解析
# ============================================================
usage() {
    cat <<EOF
使い方: sudo $SCRIPT_NAME [オプション]

オプション:
  -n              ドライラン
  -c FILE         設定ファイル
  -s SECTIONS     実行セクション（カンマ区切り）
                  user,pkg,nginx,firewall,ssh,swap,timezone
  -h              ヘルプ

例:
  sudo $SCRIPT_NAME                        # 全設定を実行
  sudo $SCRIPT_NAME -n                     # ドライラン
  sudo $SCRIPT_NAME -s user,pkg            # ユーザーとパッケージのみ
  sudo $SCRIPT_NAME -c /path/to/setup.conf # 設定ファイル指定
EOF
}

while getopts ":nc:s:h" opt; do
    case "$opt" in
        n) DRY_RUN=true ;;
        c) load_config "$OPTARG" ;;
        s) SECTIONS="$OPTARG" ;;
        h) usage; exit 0 ;;
        :) die "オプション -$OPTARG には値が必要です" ;;
        *) die "不明なオプション: -$OPTARG" ;;
    esac
done

# ============================================================
# 事前チェック
# ============================================================
preflight_check() {
    if [ "$(id -u)" -ne 0 ] && ! $DRY_RUN; then
        die "rootで実行してください: sudo $SCRIPT_NAME"
    fi
    log "INFO" "事前チェック完了"
}

# ============================================================
# 各セクション（関数）
# ============================================================
setup_user() {
    log "INFO" "--- ユーザー設定 ---"
    if id "$APP_USER" &>/dev/null; then
        log "INFO" "ユーザー $APP_USER は既に存在します。スキップ"
        return 0
    fi
    run useradd -m -s /bin/bash "$APP_USER"
    run usermod -aG sudo "$APP_USER"
    log "WARN" "パスワードを設定してください: sudo passwd $APP_USER"
    log "INFO" "ユーザー $APP_USER を作成しました"
}

install_packages() {
    log "INFO" "--- パッケージインストール ---"
    require_command apt-get
    run apt-get update -qq
    for pkg in "${PACKAGES[@]}"; do
        if dpkg -l "$pkg" &>/dev/null 2>&1; then
            log "INFO" "$pkg: インストール済み"
        else
            log "INFO" "$pkg: インストール中..."
            run apt-get install -y -qq "$pkg"
        fi
    done
}

setup_nginx() {
    log "INFO" "--- Nginx設定 ---"
    require_command nginx

    local config_file="/etc/nginx/sites-available/myapp"
    if [ -f "$config_file" ] && ! $DRY_RUN; then
        log "INFO" "Nginx設定は既に存在します。スキップ"
        return 0
    fi

    run bash -c "cat > '$config_file' <<NGINX_EOF
server {
    listen 80;
    server_name ${SERVER_NAME};
    location / {
        proxy_pass http://localhost:${APP_PORT};
        proxy_http_version 1.1;
        proxy_set_header Upgrade \\\$http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host \\\$host;
    }
}
NGINX_EOF"

    run ln -sf "$config_file" /etc/nginx/sites-enabled/
    run rm -f /etc/nginx/sites-enabled/default
    if ! $DRY_RUN; then
        nginx -t || die "Nginx設定エラー"
    fi
    run systemctl restart nginx
    log "INFO" "Nginx設定完了"
}

setup_firewall() {
    log "INFO" "--- ファイアウォール設定 ---"
    if ! command -v ufw &>/dev/null; then
        log "WARN" "ufwが見つかりません。スキップ"
        return 0
    fi
    for port in "${ALLOWED_PORTS[@]}"; do
        run ufw allow "${port}/tcp"
    done
    run ufw --force enable
    log "INFO" "ファイアウォール設定完了"
}

setup_ssh() {
    log "INFO" "--- SSH設定 ---"
    local sshd_config="/etc/ssh/sshd_config"
    [ -f "$sshd_config" ] || { log "WARN" "sshd_config なし。スキップ"; return 0; }
    [ -f "${sshd_config}.bak" ] || run cp "$sshd_config" "${sshd_config}.bak"
    run sed -i 's/#\?PasswordAuthentication yes/PasswordAuthentication no/' "$sshd_config"
    run sed -i 's/#\?PermitRootLogin .*/PermitRootLogin no/' "$sshd_config"
    run systemctl restart sshd
    log "INFO" "SSH設定完了"
}

setup_swap() {
    log "INFO" "--- スワップ設定 ---"
    if [ -f /swapfile ]; then
        log "INFO" "スワップファイルは既に存在します。スキップ"
        return 0
    fi
    run dd if=/dev/zero of=/swapfile bs=1M count="$SWAP_SIZE_MB" status=progress
    run chmod 600 /swapfile
    run mkswap /swapfile
    run swapon /swapfile
    if ! grep -q '/swapfile' /etc/fstab 2>/dev/null; then
        echo "/swapfile none swap sw 0 0" >> /etc/fstab
    fi
    log "INFO" "スワップファイル作成完了 (${SWAP_SIZE_MB}MB)"
}

setup_timezone() {
    log "INFO" "--- タイムゾーン設定 ---"
    local current_tz
    current_tz=$(timedatectl show --value -p Timezone 2>/dev/null || echo "")
    if [ "$current_tz" = "$TIMEZONE" ]; then
        log "INFO" "タイムゾーンは既に $TIMEZONE。スキップ"
        return 0
    fi
    run timedatectl set-timezone "$TIMEZONE"
    log "INFO" "タイムゾーンを $TIMEZONE に設定しました"
}

# ============================================================
# メイン処理
# ============================================================
main() {
    log "INFO" "===== サーバー初期設定開始 ====="
    $DRY_RUN && log "INFO" "（ドライランモード）"

    preflight_check

    section_enabled "user"     && setup_user
    section_enabled "pkg"      && install_packages
    section_enabled "nginx"    && setup_nginx
    section_enabled "firewall" && setup_firewall
    section_enabled "ssh"      && setup_ssh
    section_enabled "swap"     && setup_swap
    section_enabled "timezone" && setup_timezone

    log "INFO" "===== サーバー初期設定完了 ====="
    ! $DRY_RUN && log "INFO" "ログファイル: $LOG_FILE"
    return 0
}

main "$@"
```

**リファクタリングのポイントまとめ:**

| 改善項目 | Before | After |
|---|---|---|
| 構造 | フラット（100行超の直列処理） | 関数分割（各セクション独立） |
| エラー処理 | なし | `set -euo pipefail` + `die` 関数 |
| クォーティング | 不完全 | すべての変数を `"$var"` で囲む |
| 冪等性 | 2回目で失敗する | 既存チェック付きでスキップ |
| 設定 | ハードコード | 変数化 + 設定ファイル + 環境変数 |
