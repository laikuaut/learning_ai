## 問題1：基本的なtry/except（基本）

ユーザーから2つの数値を入力させ、割り算の結果を表示してください。
以下のエラーを適切にハンドリングしてください。

- 数値以外が入力された場合（ValueError）
- 0で割ろうとした場合（ZeroDivisionError）

**実行例1：**
```
1つ目の数値: 10
2つ目の数値: 3
結果: 10 / 3 = 3.3333
```

**実行例2：**
```
1つ目の数値: 10
2つ目の数値: 0
エラー: 0で割ることはできません
```

**実行例3：**
```
1つ目の数値: abc
エラー: 数値を入力してください
```

<details>
<summary>解答例</summary>

```python
try:
    num1 = float(input("1つ目の数値: "))
    num2 = float(input("2つ目の数値: "))
    result = num1 / num2
except ValueError:
    # float() に数値以外の文字列を渡した場合
    print("エラー: 数値を入力してください")
except ZeroDivisionError:
    # 0で割ろうとした場合
    print("エラー: 0で割ることはできません")
else:
    # エラーなしの場合だけ実行
    print(f"結果: {num1:.0f} / {num2:.0f} = {result:.4f}")
```

</details>

## 問題2：入力検証ループ（基本）

1〜100の整数を正しく入力するまで繰り返し入力を求めるプログラムを作成してください。

**実行例：**
```
1〜100の整数を入力してください: abc
エラー: 数値を入力してください。再度入力してください。
1〜100の整数を入力してください: 150
エラー: 1〜100の範囲で入力してください。
1〜100の整数を入力してください: -5
エラー: 1〜100の範囲で入力してください。
1〜100の整数を入力してください: 42
入力された値: 42
```

<details>
<summary>解答例</summary>

```python
def get_valid_number(prompt, min_val=1, max_val=100):
    """指定範囲の整数を入力させる（エラー時は再入力）"""
    while True:
        try:
            value = int(input(prompt))
            if min_val <= value <= max_val:
                return value
            else:
                print(f"エラー: {min_val}〜{max_val}の範囲で入力してください。")
        except ValueError:
            print("エラー: 数値を入力してください。再度入力してください。")


result = get_valid_number("1〜100の整数を入力してください: ")
print(f"入力された値: {result}")
```

## 問題3：ファイル操作のエラー処理（基本）

ファイルを安全に読み込む関数 `safe_read_file` を作成してください。
以下のエラーを個別にハンドリングしてください。

- ファイルが存在しない（FileNotFoundError）
- アクセス権がない（PermissionError）
- その他のエラー（Exception）

```python
# テスト
print(safe_read_file("existing_file.txt"))   # ファイルの内容
print(safe_read_file("nonexistent.txt"))     # エラーメッセージ
```

**期待される出力：**
```
=== existing_file.txt ===
テストデータです

=== nonexistent.txt ===
エラー: ファイルが見つかりません: nonexistent.txt
```

<details>
<summary>解答例</summary>

```python
def safe_read_file(filepath):
    """ファイルを安全に読み込む"""
    try:
        with open(filepath, "r", encoding="utf-8") as f:
            return f.read()
    except FileNotFoundError:
        return f"エラー: ファイルが見つかりません: {filepath}"
    except PermissionError:
        return f"エラー: アクセス権がありません: {filepath}"
    except Exception as e:
        return f"エラー: 予期しないエラー: {type(e).__name__}: {e}"


# テスト用ファイルを作成
with open("existing_file.txt", "w", encoding="utf-8") as f:
    f.write("テストデータです")

# テスト
for filename in ["existing_file.txt", "nonexistent.txt"]:
    print(f"=== {filename} ===")
    print(safe_read_file(filename))
    print()
```

## 問題4：else と finally の活用（基本）

データベース接続を模擬したプログラムを作成してください。
try/except/else/finally の全てを使い、処理の流れを確認してください。

```python
def process_data(data):
    """データを処理する（Noneや空文字列でエラー）"""
    # この関数を呼び出して、エラー処理を行う
```

**期待される出力（成功時）：**
```
[接続] データベースに接続しました
[処理] データ処理中: Hello World
[成功] データの保存が完了しました
[終了] データベース接続を切断しました
```

**期待される出力（失敗時）：**
```
[接続] データベースに接続しました
[エラー] TypeError: データはNoneです
[終了] データベース接続を切断しました
```

<details>
<summary>解答例</summary>

```python
def process_data(data):
    """データを処理する（Noneや空文字列でエラー）"""
    if data is None:
        raise TypeError("データはNoneです")
    if not isinstance(data, str):
        raise TypeError(f"文字列を期待しましたが{type(data).__name__}を受け取りました")
    if len(data) == 0:
        raise ValueError("データが空です")
    return data.upper()


def run_with_data(data):
    """データベース風の処理フロー"""
    print("[接続] データベースに接続しました")
    try:
        result = process_data(data)
        print(f"[処理] データ処理中: {data}")
    except (TypeError, ValueError) as e:
        # エラーが発生した場合
        print(f"[エラー] {type(e).__name__}: {e}")
    else:
        # エラーが発生しなかった場合のみ実行
        print(f"[成功] データの保存が完了しました")
    finally:
        # 常に実行される（接続の切断など）
        print("[終了] データベース接続を切断しました")
    print()


# テスト
run_with_data("Hello World")  # 成功パターン
run_with_data(None)           # TypeError パターン
run_with_data("")             # ValueError パターン
```

## 問題5：raiseで例外を発生させる（応用）

年齢、メールアドレス、パスワードを検証する関数群を作成してください。
不正な値が渡されたら、適切な例外を `raise` で発生させてください。

検証ルール：
- 年齢: 0〜150の整数
- メールアドレス: `@` を含む
- パスワード: 8文字以上、英字と数字をそれぞれ1つ以上含む

```python
# テスト
test_cases = [
    ("validate_age", -5),
    ("validate_age", 25),
    ("validate_email", "invalid"),
    ("validate_email", "user@example.com"),
    ("validate_password", "abc"),
    ("validate_password", "abcd1234"),
]
```

**期待される出力：**
```
validate_age(-5)           → NG: 年齢は0〜150の範囲で指定してください
validate_age(25)           → OK
validate_email(invalid)    → NG: メールアドレスに@が含まれていません
validate_email(user@ex..)  → OK
validate_password(abc)     → NG: パスワードは8文字以上必要です
validate_password(abcd1..) → OK
```

<details>
<summary>解答例</summary>

```python
def validate_age(age):
    """年齢を検証する"""
    if not isinstance(age, int):
        raise TypeError("年齢は整数で指定してください")
    if age < 0 or age > 150:
        raise ValueError("年齢は0〜150の範囲で指定してください")
    return True


def validate_email(email):
    """メールアドレスを検証する"""
    if not isinstance(email, str):
        raise TypeError("メールアドレスは文字列で指定してください")
    if "@" not in email:
        raise ValueError("メールアドレスに@が含まれていません")
    if "." not in email.split("@")[1]:
        raise ValueError("ドメインが不正です")
    return True


def validate_password(password):
    """パスワードを検証する"""
    if len(password) < 8:
        raise ValueError("パスワードは8文字以上必要です")
    if not any(c.isalpha() for c in password):
        raise ValueError("パスワードには英字を1文字以上含めてください")
    if not any(c.isdigit() for c in password):
        raise ValueError("パスワードには数字を1文字以上含めてください")
    return True


# テスト
test_cases = [
    (validate_age, -5),
    (validate_age, 25),
    (validate_email, "invalid"),
    (validate_email, "user@example.com"),
    (validate_password, "abc"),
    (validate_password, "abcd1234"),
]

for func, value in test_cases:
    try:
        func(value)
        display = str(value)
        if len(display) > 10:
            display = display[:8] + ".."
        print(f"{func.__name__}({display:<14}) → OK")
    except (ValueError, TypeError) as e:
        display = str(value)
        if len(display) > 10:
            display = display[:8] + ".."
        print(f"{func.__name__}({display:<14}) → NG: {e}")
```

## 問題6：カスタム例外クラス（応用）

銀行口座を管理するプログラムで、以下のカスタム例外を定義して使用してください。

- `InsufficientFundsError` - 残高不足
- `InvalidAmountError` - 不正な金額（0以下やマイナス）
- `DailyLimitExceededError` - 1日の引き出し上限超過

```python
class BankAccount:
    DAILY_LIMIT = 500000  # 1日の引き出し上限: 50万円
```

**期待される出力：**
```
=== 口座操作テスト ===
入金 10,000円 → 残高: 10,000円
入金 50,000円 → 残高: 60,000円
引出 20,000円 → 残高: 40,000円
エラー: 残高不足 - 残高40,000円に対して100,000円の引き出し要求
エラー: 不正な金額 - 金額は正の数を指定してください
引出 40,000円 → 残高: 0円
```

<details>
<summary>解答例</summary>

```python
class InsufficientFundsError(Exception):
    """残高不足エラー"""
    def __init__(self, balance, amount):
        self.balance = balance
        self.amount = amount
        super().__init__(
            f"残高不足 - 残高{balance:,}円に対して{amount:,}円の引き出し要求"
        )


class InvalidAmountError(Exception):
    """不正な金額エラー"""
    def __init__(self, message="金額は正の数を指定してください"):
        super().__init__(f"不正な金額 - {message}")


class DailyLimitExceededError(Exception):
    """1日の引き出し上限超過エラー"""
    def __init__(self, limit, total):
        super().__init__(
            f"1日の引き出し上限超過 - 上限{limit:,}円、本日合計{total:,}円"
        )


class BankAccount:
    DAILY_LIMIT = 500000

    def __init__(self, owner, balance=0):
        self.owner = owner
        self._balance = balance
        self._daily_withdrawn = 0

    def deposit(self, amount):
        """入金"""
        if amount <= 0:
            raise InvalidAmountError()
        self._balance += amount
        print(f"入金 {amount:,}円 → 残高: {self._balance:,}円")

    def withdraw(self, amount):
        """引き出し"""
        if amount <= 0:
            raise InvalidAmountError()
        if amount > self._balance:
            raise InsufficientFundsError(self._balance, amount)
        if self._daily_withdrawn + amount > self.DAILY_LIMIT:
            raise DailyLimitExceededError(
                self.DAILY_LIMIT, self._daily_withdrawn + amount
            )
        self._balance -= amount
        self._daily_withdrawn += amount
        print(f"引出 {amount:,}円 → 残高: {self._balance:,}円")


# テスト
print("=== 口座操作テスト ===")
account = BankAccount("太郎")

operations = [
    ("deposit", 10000),
    ("deposit", 50000),
    ("withdraw", 20000),
    ("withdraw", 100000),  # 残高不足
    ("deposit", -5000),    # 不正な金額
    ("withdraw", 40000),
]

for op, amount in operations:
    try:
        getattr(account, op)(amount)
    except (InsufficientFundsError, InvalidAmountError, DailyLimitExceededError) as e:
        print(f"エラー: {e}")
```

## 問題7：リトライ機能付き関数（チャレンジ）

失敗した場合に自動的にリトライする仕組みを作成してください。

1. リトライ回数と待機時間を指定できる `retry` 関数を作成
2. ランダムに失敗する関数を使ってテスト

```python
import random

def unreliable_api():
    """70%の確率で失敗するAPI（模擬）"""
    if random.random() < 0.7:
        raise ConnectionError("サーバーに接続できません")
    return {"status": "ok", "data": [1, 2, 3]}
```

**期待される出力（例）：**
```
=== リトライテスト ===
試行 1/5: ConnectionError - サーバーに接続できません
  1秒後にリトライします...
試行 2/5: ConnectionError - サーバーに接続できません
  1秒後にリトライします...
試行 3/5: 成功！
結果: {'status': 'ok', 'data': [1, 2, 3]}
```

<details>
<summary>ヒント</summary>

- `time.sleep()` で待機（実行時は短い秒数で）
- ループで指定回数までリトライ
- 全て失敗したら最後のエラーを raise

</details>

<details>
<summary>解答例</summary>

```python
import random
import time


def unreliable_api():
    """70%の確率で失敗するAPI（模擬）"""
    if random.random() < 0.7:
        raise ConnectionError("サーバーに接続できません")
    return {"status": "ok", "data": [1, 2, 3]}


def retry(func, max_attempts=5, wait_seconds=1):
    """失敗時にリトライする汎用関数"""
    last_error = None

    for attempt in range(1, max_attempts + 1):
        try:
            result = func()
            print(f"試行 {attempt}/{max_attempts}: 成功！")
            return result
        except Exception as e:
            last_error = e
            print(f"試行 {attempt}/{max_attempts}: "
                  f"{type(e).__name__} - {e}")
            if attempt < max_attempts:
                print(f"  {wait_seconds}秒後にリトライします...")
                time.sleep(wait_seconds)

    # 全試行が失敗した場合
    print(f"\n全{max_attempts}回の試行が失敗しました。")
    raise last_error


# テスト
print("=== リトライテスト ===")
try:
    result = retry(unreliable_api, max_attempts=5, wait_seconds=1)
    print(f"結果: {result}")
except ConnectionError:
    print("最終的に接続できませんでした。")
```

## 問題8：エラーログシステム（チャレンジ）

例外情報をファイルに記録するエラーログシステムを作成してください。

機能：
- 日時、エラーの種類、メッセージ、発生箇所をログファイルに記録
- ログの一覧表示
- エラー種類別の集計

**期待される出力：**
```
=== エラーを発生させてログに記録 ===
[記録] ValueError: 不正な値です
[記録] TypeError: 型が一致しません
[記録] FileNotFoundError: ファイルが見つかりません
[記録] ValueError: 範囲外の値です
[記録] ZeroDivisionError: 0で割りました

=== エラーログ一覧 ===
[2025-04-01 10:30:15] ValueError: 不正な値です
[2025-04-01 10:30:15] TypeError: 型が一致しません
[2025-04-01 10:30:15] FileNotFoundError: ファイルが見つかりません
[2025-04-01 10:30:15] ValueError: 範囲外の値です
[2025-04-01 10:30:15] ZeroDivisionError: 0で割りました

=== エラー種類別集計 ===
ValueError          : 2件
TypeError           : 1件
FileNotFoundError   : 1件
ZeroDivisionError   : 1件
```

<details>
<summary>解答例</summary>

```python
import json
from datetime import datetime
from pathlib import Path
from collections import Counter


class ErrorLogger:
    """エラーログを管理するクラス"""

    def __init__(self, log_file="error_log.json"):
        self.log_file = Path(log_file)
        self.logs = self._load_logs()

    def _load_logs(self):
        """ログファイルを読み込む"""
        if self.log_file.exists():
            with open(self.log_file, "r", encoding="utf-8") as f:
                return json.load(f)
        return []

    def _save_logs(self):
        """ログファイルに保存する"""
        with open(self.log_file, "w", encoding="utf-8") as f:
            json.dump(self.logs, f, ensure_ascii=False, indent=2)

    def log_error(self, error):
        """例外をログに記録する"""
        entry = {
            "timestamp": datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
            "type": type(error).__name__,
            "message": str(error),
        }
        self.logs.append(entry)
        self._save_logs()
        print(f"[記録] {entry['type']}: {entry['message']}")

    def show_logs(self):
        """ログ一覧を表示する"""
        print("\n=== エラーログ一覧 ===")
        if not self.logs:
            print("ログはありません。")
            return
        for entry in self.logs:
            print(f"[{entry['timestamp']}] {entry['type']}: {entry['message']}")

    def show_summary(self):
        """エラー種類別の集計を表示する"""
        print("\n=== エラー種類別集計 ===")
        type_counts = Counter(entry["type"] for entry in self.logs)
        for error_type, count in type_counts.items():
            print(f"{error_type:<20}: {count}件")


# テスト
logger = ErrorLogger()

print("=== エラーを発生させてログに記録 ===")
test_errors = [
    ValueError("不正な値です"),
    TypeError("型が一致しません"),
    FileNotFoundError("ファイルが見つかりません"),
    ValueError("範囲外の値です"),
    ZeroDivisionError("0で割りました"),
]

for error in test_errors:
    try:
        raise error
    except Exception as e:
        logger.log_error(e)

logger.show_logs()
logger.show_summary()
```

</details>
