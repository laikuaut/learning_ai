## 課題の要件

1. ユーザーからパスワード文字列を入力で受け取る
2. 以下の5つの基準でチェックする
   - 8文字以上であること
   - 大文字（A-Z）を含むこと
   - 小文字（a-z）を含むこと
   - 数字（0-9）を含むこと
   - 特殊文字（`!@#$%^&*()-_=+` など）を含むこと
3. 各基準の OK / NG を表示する
4. 満たした基準の数に応じて強度を5段階で表示する
5. 弱い場合は改善のアドバイスを表示する

<details>
<summary>解答例（初心者向け）</summary>

```python
# パスワード強度チェッカー
# 学べる内容：文字列操作、条件分岐、セキュリティの基本
# 実行方法：python password_checker.py

# --- 特殊文字の定義 ---
special_chars = "!@#$%^&*()-_=+[]{}|;:,.<>?/~`"

# --- 入力 ---
print("===== パスワード強度チェッカー =====")
print()
password = input("パスワードを入力してください: ")

# --- 各基準をチェック ---
has_length = len(password) >= 8

has_upper = False
for ch in password:
    if ch.isupper():
        has_upper = True
        break

has_lower = False
for ch in password:
    if ch.islower():
        has_lower = True
        break

has_digit = False
for ch in password:
    if ch.isdigit():
        has_digit = True
        break

has_special = False
for ch in password:
    if ch in special_chars:
        has_special = True
        break

# --- 結果表示 ---
print("--- 判定結果 ---")
print(f"長さ (8文字以上):           {'OK' if has_length else 'NG'}")
print(f"大文字を含む:               {'OK' if has_upper else 'NG'}")
print(f"小文字を含む:               {'OK' if has_lower else 'NG'}")
print(f"数字を含む:                 {'OK' if has_digit else 'NG'}")
print(f"特殊文字を含む:             {'OK' if has_special else 'NG'}")

# --- 強度判定 ---
score = 0
if has_length:
    score += 1
if has_upper:
    score += 1
if has_lower:
    score += 1
if has_digit:
    score += 1
if has_special:
    score += 1

if score <= 1:
    strength = "非常に弱い"
elif score == 2:
    strength = "弱い"
elif score == 3:
    strength = "普通"
elif score == 4:
    strength = "強い"
else:
    strength = "非常に強い"

print(f"\n強度: {strength}（{score}/5）")

# --- アドバイス ---
if score < 5:
    advice = []
    if not has_length:
        advice.append("8文字以上にする")
    if not has_upper:
        advice.append("大文字を追加する")
    if not has_lower:
        advice.append("小文字を追加する")
    if not has_digit:
        advice.append("数字を追加する")
    if not has_special:
        advice.append("特殊文字を追加する")
    print("アドバイス: " + "、".join(advice) + "ことで強度が上がります。")
```

<summary>解答例（改良版 ─ 関数とany()を活用）</summary>

```python
# パスワード強度チェッカー（改良版）
# 学べる内容：関数設計、any()、リスト内包表記
# 実行方法：python password_checker_v2.py

SPECIAL_CHARS = "!@#$%^&*()-_=+[]{}|;:,.<>?/~`"

STRENGTH_LABELS = {
    0: "非常に弱い", 1: "非常に弱い",
    2: "弱い", 3: "普通", 4: "強い", 5: "非常に強い",
}


def check_password(password):
    """パスワードの各基準をチェックし、結果を辞書で返します。"""
    checks = {
        "長さ (8文字以上)": len(password) >= 8,
        "大文字を含む": any(ch.isupper() for ch in password),
        "小文字を含む": any(ch.islower() for ch in password),
        "数字を含む": any(ch.isdigit() for ch in password),
        "特殊文字を含む": any(ch in SPECIAL_CHARS for ch in password),
    }
    return checks


def get_strength(score):
    """スコアから強度ラベルを返します。"""
    return STRENGTH_LABELS.get(score, "不明")


def show_result(checks):
    """チェック結果を表示します。"""
    print("--- 判定結果 ---")
    for label, passed in checks.items():
        status = "OK" if passed else "NG"
        print(f"{label:<24} {status}")

    score = sum(checks.values())
    print(f"\n強度: {get_strength(score)}（{score}/5）")

    # 改善アドバイス
    failed = [label for label, passed in checks.items() if not passed]
    if failed:
        print("アドバイス: 以下を改善しましょう。")
        for item in failed:
            print(f"  - {item}")


def main():
    print("===== パスワード強度チェッカー =====")
    print()
    password = input("パスワードを入力してください: ")
    checks = check_password(password)
    show_result(checks)


if __name__ == "__main__":
    main()
```

**初心者向けとの違い:**
- チェック処理を `check_password()` 関数に分離 → テストしやすい
- `any()` とジェネレータ式で簡潔に判定 → ループを書かなくてよい
- 定数を大文字で定義 → 変更しやすく意図が明確
- `if __name__ == "__main__":` で再利用可能な構造に
