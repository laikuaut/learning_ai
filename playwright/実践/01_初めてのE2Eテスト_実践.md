## 課題の要件

1. `example.com` にアクセスして見出し（h1）のテキストが `"Example Domain"` であることを検証するテストを書く
2. ページのタイトルが `"Example Domain"` であることを検証するテストを書く
3. `"More information..."` というリンクが表示されていて、`href` 属性に `iana.org` が含まれていることを検証するテストを書く
4. 3つのテストを `test.describe` でグループ化する
5. Chromiumブラウザで実行して全テストがパスすることを確認する

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```typescript
// tests/example.spec.ts
// 学べる内容：テストの基本構造、ロケーター、アサーション
// 実行方法：npx playwright test tests/example.spec.ts --project=chromium

import { test, expect } from '@playwright/test';

test.describe('Example.com の基本テスト', () => {

  // テスト1：見出しの検証
  test('見出しが正しく表示される', async ({ page }) => {
    // ページにアクセス
    await page.goto('https://example.com');

    // h1要素を取得してテキストを検証
    const heading = page.getByRole('heading', { level: 1 });
    await expect(heading).toHaveText('Example Domain');
  });

  // テスト2：タイトルの検証
  test('ページタイトルが正しい', async ({ page }) => {
    await page.goto('https://example.com');

    // ページのタイトルを検証
    await expect(page).toHaveTitle('Example Domain');
  });

  // テスト3：リンクの検証
  test('リンクが存在して正しいURLを持つ', async ({ page }) => {
    await page.goto('https://example.com');

    // リンク要素を取得
    const link = page.getByRole('link', { name: 'More information...' });

    // リンクが表示されていることを検証
    await expect(link).toBeVisible();

    // href属性にiana.orgが含まれていることを検証
    await expect(link).toHaveAttribute('href', /iana\.org/);
  });
});
```

</details>

<details>
<summary>解答例（改良版 ─ beforeEachを活用）</summary>

各テストで `page.goto()` が重複しているので、`beforeEach` で共通化します。

```typescript
// tests/example.spec.ts
// 改良版：beforeEachでページ遷移を共通化
// 実行方法：npx playwright test tests/example.spec.ts --project=chromium

import { test, expect } from '@playwright/test';

test.describe('Example.com の基本テスト', () => {

  // 各テストの前にExample.comにアクセスする
  test.beforeEach(async ({ page }) => {
    await page.goto('https://example.com');
  });

  test('見出しが正しく表示される', async ({ page }) => {
    const heading = page.getByRole('heading', { level: 1 });
    await expect(heading).toHaveText('Example Domain');
  });

  test('ページタイトルが正しい', async ({ page }) => {
    await expect(page).toHaveTitle('Example Domain');
  });

  test('リンクが存在して正しいURLを持つ', async ({ page }) => {
    const link = page.getByRole('link', { name: 'More information...' });
    await expect(link).toBeVisible();
    await expect(link).toHaveAttribute('href', /iana\.org/);
  });

  test('ページのURLが正しい', async ({ page }) => {
    await expect(page).toHaveURL('https://example.com/');
  });
});
```

**初心者向けとの違い:**
- `test.beforeEach` で共通処理（ページ遷移）を1箇所にまとめた
- テストの数を追加しても `goto` の修正が1箇所で済む
- テスト名が日本語で「何を検証しているか」が明確

</details>
