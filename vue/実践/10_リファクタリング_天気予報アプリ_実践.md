# 実践課題10：リファクタリング ─ 天気予報アプリ ★4

> **難易度**: ★★★★☆（上級）
> **前提知識**: 第4章（コンポーネント基礎）、第5章（Composition API）、第9章（非同期処理とAPI）、第10章（実践パターン ─ provide/inject、composable）
> **課題の種類**: リファクタリング
> **学習目標**: 巨大な単一コンポーネントを分割し、composable 関数と provide/inject で整理する

---

## 課題の説明

以下の「天気予報アプリ」は**正しく動作しますが**、すべてのロジックが1つの `setup()` に詰まっています。段階的にリファクタリングしてください。

### 問題点

1. `setup()` 内に200行以上のロジックが詰まっている
2. データ取得ロジックとUIが混在している
3. 同じスタイル定義が繰り返されている
4. 天気データ取得ロジックが再利用できない
5. 単位設定（℃/℉）が深い階層まで伝搬しにくい

### ゴール

1. **ステップ1**: UIをコンポーネントに分割する
2. **ステップ2**: データ取得を composable（`useWeather`）に抽出する
3. **ステップ3**: 単位設定を `provide` / `inject` で管理する

## リファクタリングの手順

<details>
<summary>ステップ1：コンポーネントに分割する（構造）</summary>

以下のようにコンポーネントを分けます。

```
App
├── UnitToggle（単位切替ボタン）
├── FavoriteCities（お気に入り都市一覧）
├── SearchBar（検索バー）
├── CurrentWeather（現在の天気カード）
└── ForecastList（3日間予報カード）
```

各コンポーネントで必要なデータは `props` で渡すか、`inject` で取得します。

</details>

<details>
<summary>ステップ2：composable に抽出する</summary>

天気データの取得ロジックを `useWeather()` として抽出します。

```js
function useWeather() {
  const city = ref("東京")
  const weather = ref(null)
  const forecast = ref([])
  const loading = ref(false)
  const error = ref(null)

  async function loadWeather(cityName) {
    // ...
  }

  function selectCity(name) {
    city.value = name
    loadWeather(name)
  }

  // 初期読み込み
  loadWeather(city.value)

  return { city, weather, forecast, loading, error, loadWeather, selectCity }
}
```

</details>

<details>
<summary>ステップ3：provide/inject で単位設定を共有する</summary>

```js
// App の setup()
const unit = ref("celsius")
provide("unit", unit)
provide("convertTemp", (temp) => {
  if (unit.value === "fahrenheit") return Math.round(temp * 9 / 5 + 32)
  return temp
})

// 子コンポーネントのsetup()
const unit = inject("unit")
const convertTemp = inject("convertTemp")
```

</details>

<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>天気予報アプリ（リファクタリング後）</title>

      <template v-else-if="weatherData.weather.value">
        <current-weather :city="weatherData.city.value" :weather="weatherData.weather.value"></current-weather>
        <forecast-list :forecast="weatherData.forecast.value"></forecast-list>

  <script>
    const { createApp, ref, provide, inject, computed } = Vue

    // ==================== composable: useWeather ====================
    function useWeather() {
      const city = ref("東京")
      const weather = ref(null)
      const forecast = ref([])
      const loading = ref(false)
      const error = ref(null)

      function fetchWeatherData(cityName) {
        return new Promise(function(resolve) {
          setTimeout(function() {
            const temps = { "東京": 22, "大阪": 24, "札幌": 15, "福岡": 25, "名古屋": 23, "沖縄": 28, "仙台": 18 }
            const conditions = { "東京": "晴れ", "大阪": "曇り", "札幌": "雨", "福岡": "晴れ", "名古屋": "曇り", "沖縄": "晴れ", "仙台": "雨" }
            const humidity = { "東京": 55, "大阪": 65, "札幌": 70, "福岡": 50, "名古屋": 60, "沖縄": 75, "仙台": 68 }
            if (temps[cityName] !== undefined) {
              resolve({
                current: { temp: temps[cityName], condition: conditions[cityName], humidity: humidity[cityName], wind: Math.floor(Math.random() * 10) + 1 },
                forecast: [
                  { day: "明日", temp: temps[cityName] + Math.floor(Math.random() * 5) - 2, condition: "晴れ" },
                  { day: "明後日", temp: temps[cityName] + Math.floor(Math.random() * 5) - 2, condition: "曇り" },
                  { day: "3日後", temp: temps[cityName] + Math.floor(Math.random() * 5) - 2, condition: "雨" },
                ],
              })
            } else {
              resolve(null)
            }
          }, 800)
        })
      }

      async function loadWeather(cityName) {
        loading.value = true
        error.value = null
        weather.value = null
        try {
          const data = await fetchWeatherData(cityName)
          if (data) { weather.value = data.current; forecast.value = data.forecast }
          else { error.value = "「" + cityName + "」の天気データが見つかりません" }
        } catch (e) { error.value = "データの取得に失敗しました" }
        finally { loading.value = false }
      }

      function selectCity(name) { city.value = name; loadWeather(name) }

      loadWeather(city.value)

      return { city, weather, forecast, loading, error, loadWeather, selectCity }
    }

    // ==================== composable: useUnit ====================
    function useUnit() {
      const unit = ref("celsius")
      function convertTemp(temp) {
        if (unit.value === "fahrenheit") return Math.round(temp * 9 / 5 + 32)
        return temp
      }
      function unitLabel() { return unit.value === "celsius" ? "C" : "F" }
      return { unit, convertTemp, unitLabel }
    }

    // ==================== utility ====================
    function conditionIcon(condition) {
      const icons = { "晴れ": "☀️", "曇り": "☁️", "雨": "🌧️", "雪": "❄️" }
      return icons[condition] || "🌤️"

      setup() {
        const weatherData = useWeather()
        const unitData = useUnit()
        const favorites = ref(["東京", "大阪", "札幌"])

        // provide で共有
        provide("unit", unitData.unit)
        provide("convertTemp", unitData.convertTemp)
        provide("unitLabel", unitData.unitLabel)
        provide("conditionIcon", conditionIcon)

        function handleSearch(cityName) { weatherData.selectCity(cityName) }
        function addFavorite() {
          if (!favorites.value.includes(weatherData.city.value)) {
            favorites.value.push(weatherData.city.value)
          }
        }
        function removeFavorite(name) {
          favorites.value = favorites.value.filter(f => f !== name)
        }

        return { weatherData, favorites, handleSearch, addFavorite, removeFavorite }
      }
    })

    // ==================== UnitToggle ====================
    app.component('unit-toggle', {
      setup() {
        const unit = inject("unit")
        return { unit }
      },
      template: `
        <div style="text-align: center; margin-bottom: 16px;">
          <button v-for="u in ['celsius', 'fahrenheit']" :key="u" class="toggle-btn"
            @click="unit = u"
            :style="{ background: unit === u ? '#1976d2' : '#e0e0e0', color: unit === u ? 'white' : '#333' }"
          >{{ u === 'celsius' ? '℃' : '℉' }}</button>
        </div>
      `
    })

    // ==================== FavoriteCities ====================
    app.component('favorite-cities', {
      props: ['favorites', 'currentCity'],
      emits: ['select', 'remove'],
      template: `
        <div style="display: flex; gap: 8px; justify-content: center; margin-bottom: 16px; flex-wrap: wrap;">
          <button v-for="fav in favorites" :key="fav" class="fav-btn" :class="{ active: currentCity === fav }"
            @click="$emit('select', fav)">{{ fav }}</button>
        </div>
      `
    })

    // ==================== SearchBar ====================
    app.component('search-bar', {
      emits: ['search'],
      setup(props, { emit }) {
        const input = ref("")
        function handleSearch() {
          if (input.value.trim()) { emit('search', input.value.trim()); input.value = "" }
        }
        return { input, handleSearch }
      },
      template: `
        <div style="display: flex; gap: 8px; margin-bottom: 16px;">
          <input v-model="input" @keyup.enter="handleSearch" placeholder="都市名を入力..." class="search-input" />
          <button @click="handleSearch" class="search-btn">検索</button>
        </div>
      `
    })

    // ==================== CurrentWeather ====================
    app.component('current-weather', {
      props: ['city', 'weather'],
      setup() {
        const convertTemp = inject("convertTemp")
        const unitLabel = inject("unitLabel")
        const icon = inject("conditionIcon")
        return { convertTemp, unitLabel, icon }

      template: `
        <div class="card">
          <h2 style="margin: 0 0 8px;">{{ city }}</h2>
          <div style="display: flex; align-items: center; gap: 16px;">
            <span style="font-size: 48px;">{{ icon(weather.condition) }}</span>
            <div>
              <div style="font-size: 36px; font-weight: bold;">{{ convertTemp(weather.temp) }}°{{ unitLabel() }}</div>

            </div>
          </div>

            <span>🌬️ 風速: {{ weather.wind }}m/s</span>
          </div>
        </div>

    })

    app.component('forecast-list', {
      props: ['forecast'],

        const convertTemp = inject("convertTemp")
        const unitLabel = inject("unitLabel")
        const icon = inject("conditionIcon")

        <div class="card">
          <h3 style="margin: 0 0 12px;">📅 3日間予報</h3>

            <span>{{ day.day }}</span>
            <span>{{ icon(day.condition) }} {{ day.condition }}</span>
            <span style="font-weight: bold;">{{ convertTemp(day.temp) }}°{{ unitLabel() }}</span>
          </div>
        </div>

    })

    app.mount('#app')

**リファクタリングのポイント:**

| 改善点 | Before | After |
|--------|--------|-------|
| ロジックの分離 | すべて1つのsetup() | `useWeather` / `useUnit` composable |
| コンポーネント分割 | 1つ | 5つ（UnitToggle, FavoriteCities, SearchBar, CurrentWeather, ForecastList） |
| グローバル設定 | propsバケツリレー | `provide` / `inject` で直接アクセス |
| 再利用性 | なし | composable を別画面でも使い回せる |
| テスタビリティ | 単体テスト困難 | composable / コンポーネント単位でテスト可能 |
