### 問題1：オブジェクトの作成とアクセス

以下の情報を持つオブジェクト `book` を作成し、各プロパティを表示してください。

- タイトル: "JavaScript入門"
- 著者: "山田太郎"
- ページ数: 350
- 出版年: 2025
- 電子版あり: true

**期待される出力：**
```
タイトル: JavaScript入門
著者: 山田太郎
ページ数: 350
出版年: 2025
電子版: あり
```

<details>
<summary>解答例</summary>

```javascript
const book = {
  title: "JavaScript入門",
  author: "山田太郎",
  pages: 350,
  year: 2025,
  hasEbook: true
};

console.log(`タイトル: ${book.title}`);
console.log(`著者: ${book.author}`);
console.log(`ページ数: ${book.pages}`);
console.log(`出版年: ${book.year}`);
console.log(`電子版: ${book.hasEbook ? "あり" : "なし"}`);
```

### 問題2：プロパティの追加・変更・削除

以下のオブジェクトに対して操作を行い、各ステップの結果を表示してください。

```javascript
const user = { name: "太郎", age: 25 };
```

1. `email` プロパティを追加: `"taro@example.com"`
2. `age` を 26 に変更
3. `name` プロパティが存在するかチェック
4. `email` プロパティを削除
5. 最終的なオブジェクトを表示

**期待される出力：**
```
追加後: { name: "太郎", age: 25, email: "taro@example.com" }
変更後: age = 26
nameが存在する: true
削除後: { name: "太郎", age: 26 }
```

<details>
<summary>解答例</summary>

```javascript
const user = { name: "太郎", age: 25 };

// 1. プロパティの追加
user.email = "taro@example.com";
console.log("追加後:", JSON.stringify(user));

// 2. プロパティの変更
user.age = 26;
console.log(`変更後: age = ${user.age}`);

// 3. 存在チェック
console.log(`nameが存在する: ${"name" in user}`);

// 4. プロパティの削除
delete user.email;
console.log("削除後:", JSON.stringify(user));
```

</details>

### 問題3：分割代入の練習

以下のオブジェクトから分割代入を使って値を取り出してください。

```javascript
const config = {
  host: "localhost",
  port: 3000,
  database: {
    name: "mydb",
    user: "admin",
    password: "secret"
  },
  debug: true
};
```

1. `host` と `port` を分割代入で取得
2. `database` の中の `name` と `user` をネストした分割代入で取得
3. `timeout` をデフォルト値 `5000` で取得

**期待される出力：**
```
host: localhost
port: 3000
DB名: mydb
DBユーザー: admin
タイムアウト: 5000
```

<details>
<summary>解答例</summary>

```javascript
const config = {
  host: "localhost",
  port: 3000,
  database: { name: "mydb", user: "admin", password: "secret" },
  debug: true
};

// 1. 基本の分割代入
const { host, port } = config;
console.log(`host: ${host}`);
console.log(`port: ${port}`);

// 2. ネストした分割代入
const { database: { name: dbName, user: dbUser } } = config;
console.log(`DB名: ${dbName}`);
console.log(`DBユーザー: ${dbUser}`);

// 3. デフォルト値付き分割代入
const { timeout = 5000 } = config;
console.log(`タイムアウト: ${timeout}`);
```

</details>

### 問題4：オブジェクトの反復処理

以下の成績データの合計と平均を `Object.entries()` を使って計算してください。

```javascript
const scores = {
  数学: 85,
  英語: 72,
  国語: 90,
  理科: 88,
  社会: 76
};
```

**期待される出力：**
```
数学: 85点
英語: 72点
国語: 90点
理科: 88点
社会: 76点
-----------
合計: 411点
平均: 82.2点
最高: 国語 (90点)
最低: 英語 (72点)
```

<details>
<summary>解答例</summary>

```javascript
const scores = {
  数学: 85,
  英語: 72,
  国語: 90,
  理科: 88,
  社会: 76
};

// 各科目を表示
for (const [subject, score] of Object.entries(scores)) {
  console.log(`${subject}: ${score}点`);
}

// 合計と平均
const values = Object.values(scores);
const total = values.reduce((sum, v) => sum + v, 0);
const average = (total / values.length).toFixed(1);

// 最高・最低の科目を見つける
const entries = Object.entries(scores);
const maxEntry = entries.reduce((max, cur) => cur[1] > max[1] ? cur : max);
const minEntry = entries.reduce((min, cur) => cur[1] < min[1] ? cur : min);

console.log("-----------");
console.log(`合計: ${total}点`);
console.log(`平均: ${average}点`);
console.log(`最高: ${maxEntry[0]} (${maxEntry[1]}点)`);
console.log(`最低: ${minEntry[0]} (${minEntry[1]}点)`);
```

</details>

### 問題5：スプレッド構文でオブジェクトを操作

以下のオブジェクトを使って、スプレッド構文による操作を練習してください。

```javascript
const defaults = {
  theme: "light",
  language: "ja",
  fontSize: 14,
  showSidebar: true
};

const userPrefs = {
  theme: "dark",
  fontSize: 16
};
```

1. `defaults` と `userPrefs` をマージ（`userPrefs` の値が優先）
2. マージ結果に `notifications: true` を追加
3. `showSidebar` を除外した新しいオブジェクトを作成

**期待される出力：**
```
マージ: {"theme":"dark","language":"ja","fontSize":16,"showSidebar":true}
追加: {"theme":"dark","language":"ja","fontSize":16,"showSidebar":true,"notifications":true}
除外: {"theme":"dark","language":"ja","fontSize":16,"notifications":true}
```

<details>
<summary>解答例</summary>

```javascript
const defaults = {
  theme: "light", language: "ja", fontSize: 14, showSidebar: true
};
const userPrefs = { theme: "dark", fontSize: 16 };

// 1. マージ（後から展開したものが優先）
const merged = { ...defaults, ...userPrefs };
console.log(`マージ: ${JSON.stringify(merged)}`);

// 2. プロパティの追加
const withNotify = { ...merged, notifications: true };
console.log(`追加: ${JSON.stringify(withNotify)}`);

// 3. 特定プロパティの除外（分割代入 + rest）
const { showSidebar, ...withoutSidebar } = withNotify;
console.log(`除外: ${JSON.stringify(withoutSidebar)}`);
```

</details>

### 問題6：メソッドと this

以下のようなオブジェクトを作成し、メソッドを使って操作してください。

銀行口座オブジェクトを作成し、以下のメソッドを持たせてください：
- `deposit(amount)`: 入金して残高を返す
- `withdraw(amount)`: 出金して残高を返す（残高不足ならエラーメッセージ）
- `getBalance()`: 現在の残高を返す
- `toString()`: "口座: [名前] 残高: [金額]円" を返す

**期待される出力：**
```
入金 5000円 → 残高: 15000円
入金 3000円 → 残高: 18000円
出金 2000円 → 残高: 16000円
残高不足です（残高: 16000円、出金額: 20000円）
口座: 太郎 残高: 16000円
```

<details>
<summary>解答例</summary>

```javascript
const account = {
  name: "太郎",
  balance: 10000,

  deposit(amount) {
    this.balance += amount;
    return `入金 ${amount}円 → 残高: ${this.balance}円`;
  },

  withdraw(amount) {
    if (amount > this.balance) {
      return `残高不足です（残高: ${this.balance}円、出金額: ${amount}円）`;
    }
    this.balance -= amount;
    return `出金 ${amount}円 → 残高: ${this.balance}円`;
  },

  getBalance() {
    return this.balance;
  },

  toString() {
    return `口座: ${this.name} 残高: ${this.balance}円`;
  }
};

console.log(account.deposit(5000));
console.log(account.deposit(3000));
console.log(account.withdraw(2000));
console.log(account.withdraw(20000));
console.log(account.toString());
```

</details>

### 問題7：クラスで学生管理

`Student` クラスを作成してください。

- コンストラクタ: `name`（名前）, `scores`（科目と点数のオブジェクト）
- メソッド `getAverage()`: 平均点を返す
- メソッド `getHighest()`: 最高点の科目名と点数を返す
- ゲッター `info`: "名前（平均: XX点）" を返す

さらに `StudentList` クラスを作成し、生徒の追加と全体の統計を表示する機能を持たせてください。

**期待される出力：**
```
太郎（平均: 82.3点）
花子（平均: 85.3点）
--- 最高点 ---
太郎: 国語 (90点)
花子: 理科 (95点)
--- 全生徒の平均 ---
83.8点
```

<details>
<summary>解答例</summary>

```javascript
class Student {
  constructor(name, scores) {
    this.name = name;
    this.scores = scores;
  }

  getAverage() {
    const values = Object.values(this.scores);
    const total = values.reduce((sum, v) => sum + v, 0);
    return Math.round((total / values.length) * 10) / 10;
  }

  getHighest() {
    const entries = Object.entries(this.scores);
    const [subject, score] = entries.reduce(
      (max, cur) => cur[1] > max[1] ? cur : max
    );
    return { subject, score };
  }

  get info() {
    return `${this.name}（平均: ${this.getAverage()}点）`;
  }
}

// 生徒の作成
const taro = new Student("太郎", { 数学: 85, 英語: 72, 国語: 90 });
const hanako = new Student("花子", { 数学: 92, 英語: 69, 国語: 95 });

console.log(taro.info);
console.log(hanako.info);

console.log("--- 最高点 ---");
const taroHighest = taro.getHighest();
console.log(`太郎: ${taroHighest.subject} (${taroHighest.score}点)`);

const hanakoHighest = hanako.getHighest();
console.log(`花子: ${hanakoHighest.subject} (${hanakoHighest.score}点)`);

// 全生徒の平均
const students = [taro, hanako];
const overallAvg = students.reduce((sum, s) => sum + s.getAverage(), 0) / students.length;
console.log("--- 全生徒の平均 ---");
console.log(`${overallAvg}点`);
```

</details>
