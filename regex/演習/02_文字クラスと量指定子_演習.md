## 問題1：文字クラスの基本（基本）

以下のテキストから、母音（a, e, i, o, u）をすべて抽出してください。

```python
text = "Hello, World! Programming is fun."
```

**期待される出力：**
```
['e', 'o', 'o', 'r', 'o', 'r', 'a', 'i', 'i', 'u']
```

※ 注意：小文字の母音のみを対象とします。

<details>
<summary>ヒント</summary>

文字クラス `[aeiou]` を使えば、母音のいずれか1文字にマッチします。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "Hello, World! Programming is fun."

# [aeiou] → 小文字の母音のいずれか1文字にマッチ
results = re.findall(r"[aeiou]", text)
print(results)
# ['e', 'o', 'o', 'r', 'o', 'r', 'a', 'i', 'i', 'u']

# 補足：大文字の母音も含めたい場合
results_all = re.findall(r"[aeiouAEIOU]", text)
print(results_all)
# ['e', 'o', 'O', 'o', 'r', 'o', 'r', 'a', 'i', 'i', 'u']
```

</details>

## 問題2：範囲指定と否定（基本）

以下のテキストから、英字以外の文字をすべて抽出してください（スペースや記号、数字など）。

```python
text = "Price: $19.99 (tax included)"
```

**期待される出力：**
```
[':', ' ', '$', '1', '9', '.', '9', '9', ' ', '(', ' ', ')']
```

<details>
<summary>ヒント</summary>

否定文字クラス `[^a-zA-Z]` を使うと、英字以外の1文字にマッチします。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "Price: $19.99 (tax included)"

# [^a-zA-Z] → 英字以外の1文字
results = re.findall(r"[^a-zA-Z]", text)
print(results)
# [':', ' ', '$', '1', '9', '.', '9', '9', ' ', '(', ' ', ')']
```

</details>

## 問題3：量指定子の使い分け（基本）

以下のテキストから、連続する数字（1桁以上）をすべて抽出してください。

```python
text = "注文番号: 12345, 数量: 3, 合計: 45000円"
```

**期待される出力：**
```
['12345', '3', '45000']
```

<details>
<summary>ヒント</summary>

`\d+` で「1文字以上の連続する数字」にマッチします。`+` は「1回以上の繰り返し」です。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "注文番号: 12345, 数量: 3, 合計: 45000円"

# \d+ → 1文字以上の連続する数字
results = re.findall(r"\d+", text)
print(results)  # ['12345', '3', '45000']
```

</details>

## 問題4：{n,m} による桁数指定（基本）

以下のテキストから、2桁以上4桁以下の数字だけを抽出してください。

```python
text = "1 23 456 7890 12345"
```

**期待される出力：**
```
['23', '456', '7890']
```

<details>
<summary>ヒント</summary>

`\d{2,4}` だけでは、5桁の数字「12345」から「1234」が切り出されてしまいます。前後が数字でないことを保証するために、単語境界 `\b` を使うか、工夫が必要です（単語境界は第3章で学びますが、先取りして使ってみましょう）。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "1 23 456 7890 12345"

# \b を使って単語境界で区切り、2〜4桁の数字だけにマッチ
results = re.findall(r"\b\d{2,4}\b", text)
print(results)  # ['23', '456', '7890']

# 補足：\b なしだと 12345 から 1234 が切り出されてしまう
results_no_boundary = re.findall(r"\d{2,4}", text)
print(results_no_boundary)  # ['23', '456', '7890', '1234']
```

</details>

## 問題5：貪欲マッチと非貪欲マッチ（応用）

以下のHTMLテキストから、個別のHTMLタグ（`<b>`, `</b>`, `<i>`, `</i>`）をすべて抽出してください。

```python
html = "<b>重要</b>な<i>お知らせ</i>です"
```

**期待される出力：**
```
['<b>', '</b>', '<i>', '</i>']
```

<details>
<summary>ヒント</summary>

貪欲マッチ `<.+>` だと最初の `<` から最後の `>` まで取ってしまいます。非貪欲マッチ `<.+?>` を使うと、最短マッチで個別のタグを取得できます。

</details>

<details>
<summary>解答例</summary>

```python
import re

html = "<b>重要</b>な<i>お知らせ</i>です"

# ❌ 貪欲マッチ → 全部つながって取れてしまう
results_greedy = re.findall(r"<.+>", html)
print(f"貪欲: {results_greedy}")
# ['<b>重要</b>な<i>お知らせ</i>']

# ✅ 非貪欲マッチ → 個別のタグが取れる
results_lazy = re.findall(r"<.+?>", html)
print(f"非貪欲: {results_lazy}")
# ['<b>', '</b>', '<i>', '</i>']
```

</details>

## 問題6：郵便番号の抽出（応用）

以下のテキストから郵便番号（`〒` 付きまたはなし、3桁-4桁の形式）をすべて抽出してください。

```python
text = """
東京本社: 〒100-0001 千代田区千代田1-1
大阪支社: 〒530-0001 大阪市北区梅田1-2
倉庫住所: 150-0002 渋谷区渋谷2-3
電話番号: 03-1234-5678（これは郵便番号ではない）
"""
```

**期待される出力：**
```
['〒100-0001', '〒530-0001', '150-0002']
```

<details>
<summary>ヒント</summary>

郵便番号は「（〒が0回か1回）+ 数字3桁 + ハイフン + 数字4桁」のパターンです。`〒?` で `〒` の有無に対応し、`\d{3}-\d{4}` で数字部分にマッチします。電話番号と区別するために、ハイフンの数やパターンに注意しましょう。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = """
東京本社: 〒100-0001 千代田区千代田1-1
大阪支社: 〒530-0001 大阪市北区梅田1-2
倉庫住所: 150-0002 渋谷区渋谷2-3
電話番号: 03-1234-5678（これは郵便番号ではない）
"""

# 〒? → 〒が0回か1回、\d{3}-\d{4} → 3桁-4桁
results = re.findall(r"〒?\d{3}-\d{4}", text)
print(results)  # ['〒100-0001', '〒530-0001', '150-0002']

# 補足：電話番号の 03-1234 も3桁-4桁だが、
# 03-1234-5678 全体としては 2桁-4桁-4桁 なので
# findall は左から順にマッチし、03-1234 がマッチした後
# -5678 が残る。結果的に電話番号もマッチする可能性がある。
# より厳密にするには、後続が数字やハイフンでないことを
# 確認するパターンが必要です（第4章の先読みで解説）。
```

</details>

## 問題7：英単語の抽出（応用）

以下のテキストから、英字だけで構成された単語（1文字以上）をすべて抽出してください。数字を含む文字列（`Python3` 等）は除外します。

```python
text = "Use Python3 for web2.0 development with React and Vue"
```

**期待される出力：**
```
['Use', 'for', 'development', 'with', 'React', 'and', 'Vue']
```

<details>
<summary>ヒント</summary>

`[a-zA-Z]+` は英字が1文字以上連続する部分にマッチしますが、`Python3` の `Python` 部分にもマッチしてしまいます。単語境界 `\b` を使って「前後が英字以外」であることを確認するか、別のアプローチが必要です。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "Use Python3 for web2.0 development with React and Vue"

# スペースで区切った各単語のうち、英字だけで構成されたものを抽出
# 方法1: re.findall でスペース区切りの英字のみの単語を取得
results = re.findall(r"\b[a-zA-Z]+\b", text)
# これだと Python3 の Python や web の部分もマッチしてしまう

# 方法2: スペースで分割してから英字のみのものをフィルタ
words = text.split()
results = [w for w in words if re.fullmatch(r"[a-zA-Z]+", w)]
print(results)  # ['Use', 'for', 'development', 'with', 'React', 'and', 'Vue']
```

</details>

## 問題8：クォートで囲まれた文字列の抽出（チャレンジ）

以下のテキストから、ダブルクォートで囲まれた文字列の中身をすべて抽出してください。

```python
text = '設定: name="太郎", age="25", city="東京"'
```

**期待される出力：**
```
['太郎', '25', '東京']
```

<details>
<summary>ヒント</summary>

`"` と `"` の間の文字を取得するには、`"(.+?)"` のように非貪欲マッチとグループ化を組み合わせます。`()` はグループ化（第3章で詳しく学びます）で、`re.findall()` はグループ内の部分だけを返します。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = '設定: name="太郎", age="25", city="東京"'

# ".+?" → ダブルクォートの間を非貪欲でマッチ
# () でグループ化すると、findall はグループ内だけを返す
results = re.findall(r'"(.+?)"', text)
print(results)  # ['太郎', '25', '東京']

# 補足：貪欲マッチだとどうなるか確認
results_greedy = re.findall(r'"(.+)"', text)
print(results_greedy)  # ['太郎", age="25", city="東京']
# ← 最初の " から最後の " まで一気にマッチしてしまう
```

</details>

## 問題9：パスワード形式の検証（チャレンジ）

以下の文字列リストについて、「英字と数字の両方を含み、8文字以上16文字以下」のパスワード形式に合致するものだけを `re.fullmatch()` で判定してください。

```python
passwords = [
    "abc12345",     # ✓ 英字+数字、8文字
    "abcdefgh",     # ✗ 数字なし
    "12345678",     # ✗ 英字なし
    "Ab1",          # ✗ 8文字未満
    "Pass1234Word5", # ✓ 英字+数字、13文字
    "a1b2c3d4e5f6g7h8", # ✗ 16文字超
]
```

**期待される出力：**
```
abc12345 → 合格
abcdefgh → 不合格
12345678 → 不合格
Ab1 → 不合格
Pass1234Word5 → 合格
a1b2c3d4e5f6g7h8 → 不合格
```

<details>
<summary>ヒント</summary>

この問題は文字クラスと量指定子だけでは難しいです。「英字を含む」「数字を含む」という条件を同時に満たすには、先読み（第4章で詳しく学びます）を先取りして使うか、正規表現を2つ組み合わせるアプローチがあります。

</details>

<details>
<summary>解答例</summary>

```python
import re

passwords = [
    "abc12345",
    "abcdefgh",
    "12345678",
    "Ab1",
    "Pass1234Word5",
    "a1b2c3d4e5f6g7h8",
]

for pw in passwords:
    # 方法: 3つの条件を個別にチェック
    is_length_ok = re.fullmatch(r"[a-zA-Z0-9]{8,16}", pw) is not None
    has_alpha = re.search(r"[a-zA-Z]", pw) is not None
    has_digit = re.search(r"[0-9]", pw) is not None

    if is_length_ok and has_alpha and has_digit:
        print(f"{pw} → 合格")
    else:
        print(f"{pw} → 不合格")

# 補足：第4章で学ぶ「先読み」を使えば、1つの正規表現で書けます
# pattern = r"(?=.*[a-zA-Z])(?=.*[0-9])[a-zA-Z0-9]{8,16}"
```

</details>
