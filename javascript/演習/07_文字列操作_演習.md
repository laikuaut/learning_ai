## 基本問題

### 問題1：文字列の検索

以下の文字列に対して、指定された検索操作を行ってください。

```javascript
const text = "JavaScriptは世界で最も人気のあるプログラミング言語です。JavaScriptを学びましょう！";
```

1. `"JavaScript"` が含まれるか
2. `"JavaScript"` の最初の出現位置
3. `"JavaScript"` の最後の出現位置
4. `"Java"` で始まるか
5. `"！"` で終わるか

**期待される出力：**
```
含まれる: true
最初の位置: 0
最後の位置: 25
Javaで始まる: true
！で終わる: true
```

<details>
<summary>解答例</summary>

```javascript
const text = "JavaScriptは世界で最も人気のあるプログラミング言語です。JavaScriptを学びましょう！";

console.log(`含まれる: ${text.includes("JavaScript")}`);
console.log(`最初の位置: ${text.indexOf("JavaScript")}`);
console.log(`最後の位置: ${text.lastIndexOf("JavaScript")}`);
console.log(`Javaで始まる: ${text.startsWith("Java")}`);
console.log(`！で終わる: ${text.endsWith("！")}`);
```

</details>

### 問題2：文字列の変換

以下の文字列に対して各種変換を行ってください。

```javascript
const input = "  Hello, JavaScript World!  ";
```

1. 前後の空白を除去
2. 全て大文字に変換
3. 全て小文字に変換
4. `"JavaScript"` を `"TypeScript"` に置換
5. `","` と `"!"` を除去

**期待される出力：**
```
空白除去: "Hello, JavaScript World!"
大文字: "HELLO, JAVASCRIPT WORLD!"
小文字: "hello, javascript world!"
置換: "Hello, TypeScript World!"
記号除去: "Hello JavaScript World"
```

<details>
<summary>解答例</summary>

```javascript
const input = "  Hello, JavaScript World!  ";

const trimmed = input.trim();
console.log(`空白除去: "${trimmed}"`);
console.log(`大文字: "${trimmed.toUpperCase()}"`);
console.log(`小文字: "${trimmed.toLowerCase()}"`);
console.log(`置換: "${trimmed.replace("JavaScript", "TypeScript")}"`);
console.log(`記号除去: "${trimmed.replace(/[,!]/g, "")}"`);
```

</details>

### 問題3：文字列の分割と結合

以下のCSVデータを処理してください。

```javascript
const csv = "太郎,25,東京,taro@example.com";
```

1. カンマで分割して各フィールドを表示
2. 分割した配列を `" | "` で結合して表示

**期待される出力：**
```
名前: 太郎
年齢: 25
都市: 東京
メール: taro@example.com
---
結合: 太郎 | 25 | 東京 | taro@example.com
```

<details>
<summary>解答例</summary>

```javascript
const csv = "太郎,25,東京,taro@example.com";

// 1. カンマで分割して分割代入
const [name, age, city, email] = csv.split(",");
console.log(`名前: ${name}`);
console.log(`年齢: ${age}`);
console.log(`都市: ${city}`);
console.log(`メール: ${email}`);

// 2. 結合
console.log("---");
const joined = csv.split(",").join(" | ");
console.log(`結合: ${joined}`);
```

</details>

### 問題4：パディングと繰り返し

以下の形式で出力を作成してください。

1. 数字の `1`, `5`, `42`, `100` を3桁にゼロ埋め
2. 罫線 `"="` を40文字分
3. タイトルを中央寄せ（40文字幅）

**期待される出力：**
```
001
005
042
100
========================================
           JavaScript入門
========================================
```

<details>
<summary>解答例</summary>

```javascript
// 1. ゼロ埋め
const numbers = [1, 5, 42, 100];
numbers.forEach(n => {
  console.log(String(n).padStart(3, "0"));
});

// 2. 罫線
const line = "=".repeat(40);
console.log(line);

// 3. 中央寄せ（全体幅から文字列長を引いて左パディング）
const title = "JavaScript入門";
const padding = Math.floor((40 - title.length) / 2);
console.log(" ".repeat(padding) + title);

console.log(line);
```

</details>

## 応用問題

### 問題5：正規表現で情報を抽出

以下のテキストから正規表現を使って情報を抽出してください。

```javascript
const text = "田中さんの電話番号は 090-1234-5678 で、メールは tanaka@example.com です。鈴木さんは 080-9876-5432 で suzuki@test.co.jp です。";
```

1. すべての電話番号を抽出
2. すべてのメールアドレスを抽出

**期待される出力：**
```
電話番号: ["090-1234-5678", "080-9876-5432"]
メール: ["tanaka@example.com", "suzuki@test.co.jp"]
```

<details>
<summary>ヒント</summary>

- 電話番号のパターン: `/\d{3}-\d{4}-\d{4}/g`
- メールのパターン: `/[a-zA-Z0-9._-]+@[a-zA-Z0-9.-]+\.[a-zA-Z]+/g`

</details>

<details>
<summary>解答例</summary>

```javascript
const text = "田中さんの電話番号は 090-1234-5678 で、メールは tanaka@example.com です。鈴木さんは 080-9876-5432 で suzuki@test.co.jp です。";

// 電話番号の抽出
const phones = text.match(/\d{3}-\d{4}-\d{4}/g);
console.log(`電話番号: ${JSON.stringify(phones)}`);

// メールアドレスの抽出
const emails = text.match(/[a-zA-Z0-9._-]+@[a-zA-Z0-9.-]+\.[a-zA-Z]+/g);
console.log(`メール: ${JSON.stringify(emails)}`);
```

</details>

### 問題6：ケース変換ユーティリティ

以下の文字列変換関数を作成してください。

1. `camelToSnake("firstName")` → `"first_name"`
2. `snakeToCamel("first_name")` → `"firstName"`
3. `capitalize("hello world")` → `"Hello World"`（各単語の先頭を大文字に）

**期待される出力：**
```
camelToSnake: user_name, get_user_by_id
snakeToCamel: userName, getUserById
capitalize: Hello World, JavaScript Is Fun
```

<details>
<summary>解答例</summary>

```javascript
// キャメルケース → スネークケース
function camelToSnake(str) {
  return str.replace(/[A-Z]/g, letter => `_${letter.toLowerCase()}`);
}

// スネークケース → キャメルケース
function snakeToCamel(str) {
  return str.replace(/_([a-z])/g, (_, letter) => letter.toUpperCase());
}

// 各単語の先頭を大文字に
function capitalize(str) {
  return str.replace(/\b[a-z]/g, letter => letter.toUpperCase());
}

console.log(`camelToSnake: ${camelToSnake("userName")}, ${camelToSnake("getUserById")}`);
console.log(`snakeToCamel: ${snakeToCamel("user_name")}, ${snakeToCamel("get_user_by_id")}`);
console.log(`capitalize: ${capitalize("hello world")}, ${capitalize("javaScript is fun")}`);
```

</details>

### 問題7：テンプレートエンジン（簡易版）

テンプレート文字列内の `{{変数名}}` を、データオブジェクトの値に置き換える関数 `render` を作成してください。

```javascript
const template = "こんにちは、{{name}}さん！あなたは{{age}}歳で、{{city}}に住んでいます。";
const data = { name: "太郎", age: 25, city: "東京" };
```

**期待される出力：**
```
こんにちは、太郎さん！あなたは25歳で、東京に住んでいます。
```

<details>
<summary>ヒント</summary>

`replace` に正規表現 `/\{\{(\w+)\}\}/g` を使い、コールバック関数でマッチしたキー名に対応する値を返します。

</details>

<details>
<summary>解答例</summary>

```javascript
function render(template, data) {
  // {{変数名}} のパターンにマッチして値に置き換え
  return template.replace(/\{\{(\w+)\}\}/g, (match, key) => {
    // data にキーが存在すればその値を、なければそのまま返す
    return key in data ? data[key] : match;
  });
}

const template = "こんにちは、{{name}}さん！あなたは{{age}}歳で、{{city}}に住んでいます。";
const data = { name: "太郎", age: 25, city: "東京" };

console.log(render(template, data));
// "こんにちは、太郎さん！あなたは25歳で、東京に住んでいます。"

// 存在しないキーはそのまま残る
const template2 = "{{greeting}}、{{name}}！ {{unknown}}";
console.log(render(template2, { greeting: "こんにちは", name: "花子" }));
// "こんにちは、花子！ {{unknown}}"
```

</details>

### 問題8：バリデーション関数群

以下のバリデーション関数を作成し、テストデータで動作を確認してください。

1. `isValidEmail(email)`: メールアドレスの形式チェック
2. `isValidPhone(phone)`: 日本の携帯電話番号チェック（ハイフンあり/なし対応）
3. `isValidPassword(password)`: パスワード強度チェック（8文字以上、英大文字・小文字・数字を含む）

**期待される出力：**
```
--- メール ---
test@example.com: true
invalid-email: false
@example.com: false
--- 電話番号 ---
090-1234-5678: true
09012345678: true
012-3456-7890: false
--- パスワード ---
Abc12345: true
abcdefgh: false
ABC12345: false
Ab1: false
```

<details>
<summary>解答例</summary>

```javascript
function isValidEmail(email) {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}

function isValidPhone(phone) {
  return /^0[789]0-?\d{4}-?\d{4}$/.test(phone);
}

function isValidPassword(password) {
  if (password.length < 8) return false;
  if (!/[a-z]/.test(password)) return false;  // 小文字を含む
  if (!/[A-Z]/.test(password)) return false;  // 大文字を含む
  if (!/[0-9]/.test(password)) return false;  // 数字を含む
  return true;
}

console.log("--- メール ---");
console.log(`test@example.com: ${isValidEmail("test@example.com")}`);
console.log(`invalid-email: ${isValidEmail("invalid-email")}`);
console.log(`@example.com: ${isValidEmail("@example.com")}`);

console.log("--- 電話番号 ---");
console.log(`090-1234-5678: ${isValidPhone("090-1234-5678")}`);
console.log(`09012345678: ${isValidPhone("09012345678")}`);
console.log(`012-3456-7890: ${isValidPhone("012-3456-7890")}`);

console.log("--- パスワード ---");
console.log(`Abc12345: ${isValidPassword("Abc12345")}`);
console.log(`abcdefgh: ${isValidPassword("abcdefgh")}`);
console.log(`ABC12345: ${isValidPassword("ABC12345")}`);
console.log(`Ab1: ${isValidPassword("Ab1")}`);
```

</details>
