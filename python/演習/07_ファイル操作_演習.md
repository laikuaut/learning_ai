## 問題1：テキストファイルの書き込みと読み込み（基本）

以下の手順でテキストファイルを操作してください。

1. `fruits.txt` に果物の名前を1行ずつ書き込む（りんご、みかん、バナナ、ぶどう、メロン）
2. ファイルを読み込んで、行番号付きで表示する

**期待される出力：**
```
=== fruits.txt の内容 ===
1: りんご
2: みかん
3: バナナ
4: ぶどう
5: メロン
合計: 5行
```

<details>
<summary>解答例</summary>

```python
# 書き込み
fruits = ["りんご", "みかん", "バナナ", "ぶどう", "メロン"]

with open("fruits.txt", "w", encoding="utf-8") as f:
    for fruit in fruits:
        f.write(f"{fruit}\n")

# 読み込み
print("=== fruits.txt の内容 ===")
with open("fruits.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    for i, line in enumerate(lines, start=1):
        print(f"{i}: {line.strip()}")
    print(f"合計: {len(lines)}行")
```

## 問題2：ファイルへの追記（基本）

既存のファイルに追記する操作を行ってください。

1. `log.txt` に現在の日時を含むログメッセージを3回書き込む
2. ファイルの内容を全て表示する

**期待される出力（例）：**
```
=== ログファイル ===
[2025-04-01 10:30:15] アプリケーションを起動しました
[2025-04-01 10:30:15] データを読み込みました
[2025-04-01 10:30:15] 処理が完了しました
```

<details>
<summary>ヒント</summary>

- `"a"` モードで追記
- `from datetime import datetime` で日時を取得
- `datetime.now().strftime("%Y-%m-%d %H:%M:%S")` でフォーマット

</details>

<details>
<summary>解答例</summary>

```python
from datetime import datetime

messages = [
    "アプリケーションを起動しました",
    "データを読み込みました",
    "処理が完了しました",
]

# 追記モードで書き込み（初回は新規作成される）
with open("log.txt", "a", encoding="utf-8") as f:
    for msg in messages:
        now = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        f.write(f"[{now}] {msg}\n")

# 読み込みと表示
print("=== ログファイル ===")
with open("log.txt", "r", encoding="utf-8") as f:
    print(f.read(), end="")
```

## 問題3：CSVファイルの読み書き（基本）

以下のデータをCSVファイルに書き込み、読み込んで表形式で表示してください。

```python
students = [
    {"名前": "太郎", "数学": 85, "英語": 78, "国語": 92},
    {"名前": "花子", "数学": 92, "英語": 95, "国語": 88},
    {"名前": "次郎", "数学": 68, "英語": 72, "国語": 75},
]
```

**期待される出力：**
```
=== CSVファイル書き込み完了 ===

=== 成績表 ===
名前     数学  英語  国語  平均
----------------------------
太郎       85    78    92  85.0
花子       92    95    88  91.7
次郎       68    72    75  71.7
```

<details>
<summary>解答例</summary>

```python
import csv

students = [
    {"名前": "太郎", "数学": 85, "英語": 78, "国語": 92},
    {"名前": "花子", "数学": 92, "英語": 95, "国語": 88},
    {"名前": "次郎", "数学": 68, "英語": 72, "国語": 75},
]

# CSVファイルに書き込み（DictWriter）
with open("scores.csv", "w", encoding="utf-8", newline="") as f:
    fieldnames = ["名前", "数学", "英語", "国語"]
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerows(students)

print("=== CSVファイル書き込み完了 ===")

# CSVファイルを読み込んで表形式で表示
print("\n=== 成績表 ===")
with open("scores.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)

    print(f"{'名前':<6} {'数学':>4} {'英語':>4} {'国語':>4} {'平均':>5}")
    print("-" * 30)

    for row in reader:
        math = int(row["数学"])
        eng = int(row["英語"])
        jpn = int(row["国語"])
        avg = (math + eng + jpn) / 3
        print(f"{row['名前']:<6} {math:>4} {eng:>4} {jpn:>4} {avg:>5.1f}")
```

## 問題4：JSONファイルの操作（基本）

設定ファイルをJSON形式で保存・読み込みするプログラムを作成してください。

1. デフォルト設定を作成してJSONファイルに保存
2. ファイルから読み込んで表示
3. 設定値を変更して再度保存

```python
default_config = {
    "app_name": "MyApp",
    "version": "1.0.0",
    "theme": "light",
    "font_size": 14,
    "auto_save": True,
    "recent_files": [],
}
```

**期待される出力：**
```
=== デフォルト設定を保存 ===
config.json に保存しました

=== 設定を読み込み ===
app_name   : MyApp
version    : 1.0.0
theme      : light
font_size  : 14
auto_save  : True
recent_files: []

=== 設定を変更して保存 ===
theme を dark に変更
font_size を 16 に変更
config.json に保存しました
```

<details>
<summary>解答例</summary>

```python
import json

default_config = {
    "app_name": "MyApp",
    "version": "1.0.0",
    "theme": "light",
    "font_size": 14,
    "auto_save": True,
    "recent_files": [],
}

# 1. デフォルト設定を保存
print("=== デフォルト設定を保存 ===")
with open("config.json", "w", encoding="utf-8") as f:

# 2. 読み込みと表示
print("\n=== 設定を読み込み ===")
with open("config.json", "r", encoding="utf-8") as f:

# 3. 変更して保存
print("\n=== 設定を変更して保存 ===")
config["theme"] = "dark"
print("theme を dark に変更")
config["font_size"] = 16
print("font_size を 16 に変更")

with open("config.json", "w", encoding="utf-8") as f:
    json.dump(config, f, ensure_ascii=False, indent=2)
print("config.json に保存しました")
```

## 問題5：pathlibでファイル操作（応用）

`pathlib` を使って、以下の操作を行うプログラムを作成してください。

1. `output/reports` ディレクトリを作成
2. そのディレクトリ内にテスト用の `.txt` ファイルと `.csv` ファイルを各3つ作成
3. ディレクトリ内のファイルを拡張子別に一覧表示
4. `.txt` ファイルだけのサイズ合計を計算

**期待される出力（例）：**
```
=== ディレクトリ作成 ===
output/reports を作成しました

=== テストファイル作成 ===
report_001.txt を作成
report_002.txt を作成
report_003.txt を作成
data_001.csv を作成
data_002.csv を作成
data_003.csv を作成

=== ファイル一覧 ===
[.txt ファイル]
  report_001.txt (26 bytes)
  report_002.txt (26 bytes)
  report_003.txt (26 bytes)

[.csv ファイル]
  data_001.csv (18 bytes)
  data_002.csv (18 bytes)
  data_003.csv (18 bytes)

.txt ファイルの合計サイズ: 78 bytes
```

<details>
<summary>ヒント</summary>

- `Path.mkdir(parents=True, exist_ok=True)` でディレクトリ作成
- `Path.glob("*.txt")` でパターンマッチ
- `Path.stat().st_size` でファイルサイズ取得
- `Path.write_text()` で簡単にファイル書き込み

</details>

<details>
<summary>解答例</summary>

```python
from pathlib import Path

# 1. ディレクトリ作成
print("=== ディレクトリ作成 ===")
output_dir = Path("output/reports")
output_dir.mkdir(parents=True, exist_ok=True)
print(f"{output_dir} を作成しました")

# 2. テストファイル作成
print("\n=== テストファイル作成 ===")
for i in range(1, 4):
    # .txt ファイル
    txt_file = output_dir / f"report_{i:03d}.txt"
    txt_file.write_text(f"レポート{i}の内容です。\n", encoding="utf-8")
    print(f"{txt_file.name} を作成")

    # .csv ファイル
    csv_file = output_dir / f"data_{i:03d}.csv"
    csv_file.write_text(f"id,value\n{i},{i*100}\n", encoding="utf-8")
    print(f"{csv_file.name} を作成")

# 3. 拡張子別の一覧表示
print("\n=== ファイル一覧 ===")
for ext in [".txt", ".csv"]:
    files = sorted(output_dir.glob(f"*{ext}"))
    print(f"[{ext} ファイル]")
    for f in files:
        size = f.stat().st_size
        print(f"  {f.name} ({size} bytes)")
    print()

# 4. .txt ファイルのサイズ合計
txt_total = sum(f.stat().st_size for f in output_dir.glob("*.txt"))
print(f".txt ファイルの合計サイズ: {txt_total} bytes")
```

</details>

## 問題6：ファイルの差分検出（応用）

2つのテキストファイルの内容を比較し、差分を表示するプログラムを作成してください。

まず2つのファイルを作成し、その後比較してください。

**期待される出力：**
```
=== ファイル差分 ===
  共通: りんご
- 旧のみ: みかん
+ 新のみ: オレンジ
  共通: バナナ
- 旧のみ: ぶどう
+ 新のみ: マンゴー
  共通: メロン
+ 新のみ: パイナップル
```

<details>
<summary>ヒント</summary>

- 2つのファイルの行をリストとして読み込む
- 集合（set）で共通・固有の行を判定
- 行の順番を保持するには、リストのまま処理する

</details>

<details>
<summary>解答例</summary>

```python
# テスト用ファイルを作成
old_data = ["りんご", "みかん", "バナナ", "ぶどう", "メロン"]
new_data = ["りんご", "オレンジ", "バナナ", "マンゴー", "メロン", "パイナップル"]

with open("old_list.txt", "w", encoding="utf-8") as f:
    f.write("\n".join(old_data))

with open("new_list.txt", "w", encoding="utf-8") as f:
    f.write("\n".join(new_data))

# ファイルを読み込む
with open("old_list.txt", "r", encoding="utf-8") as f:
    old_lines = [line.strip() for line in f.readlines()]

with open("new_list.txt", "r", encoding="utf-8") as f:
    new_lines = [line.strip() for line in f.readlines()]

# 差分を検出して表示
old_set = set(old_lines)
new_set = set(new_lines)

print("=== ファイル差分 ===")

# 全ての行を順番に処理（両方のリストの順序を保持）
all_items = []
for item in old_lines:
    if item not in all_items:
        all_items.append(item)
for item in new_lines:
    if item not in all_items:
        all_items.append(item)

for item in all_items:
    if item in old_set and item in new_set:
        print(f"  共通: {item}")
    elif item in old_set:
        print(f"- 旧のみ: {item}")
    else:
        print(f"+ 新のみ: {item}")
```

</details>

## 問題7：簡易データベース（チャレンジ）

JSONファイルをデータベースとして使い、住所録の登録・検索・一覧・削除ができるプログラムを作成してください。

機能：
1. 連絡先の追加
2. 連絡先の検索（名前で部分一致）
3. 連絡先の一覧表示
4. 連絡先の削除
5. 終了

**実行例：**
```
=== 住所録管理 ===
1. 追加  2. 検索  3. 一覧  4. 削除  5. 終了
選択: 1
名前: 田中太郎
電話: 090-1234-5678
メール: taro@example.com
田中太郎 を登録しました

選択: 3
=== 連絡先一覧 ===
1. 田中太郎
   電話: 090-1234-5678
   メール: taro@example.com

選択: 5
データを保存して終了します。
```

<details>
<summary>ヒント</summary>

- JSONファイルからの読み込み→操作→保存のパターン
- ファイルがなければ空のリストで開始
- 各操作を関数に分ける

</details>

<details>
<summary>解答例</summary>

```python
import json
from pathlib import Path

DB_FILE = Path("contacts.json")


def load_contacts():
    """連絡先をJSONファイルから読み込む"""
    if DB_FILE.exists():
        with open(DB_FILE, "r", encoding="utf-8") as f:
            return json.load(f)
    return []


def save_contacts(contacts):
    """連絡先をJSONファイルに保存する"""
    with open(DB_FILE, "w", encoding="utf-8") as f:
        json.dump(contacts, f, ensure_ascii=False, indent=2)


def add_contact(contacts):
    """連絡先を追加する"""
    name = input("名前: ").strip()
    phone = input("電話: ").strip()
    email = input("メール: ").strip()

    contacts.append({"name": name, "phone": phone, "email": email})
    print(f"{name} を登録しました")


def search_contacts(contacts):
    """名前で部分一致検索する"""
    keyword = input("検索キーワード: ").strip()
    results = [c for c in contacts if keyword in c["name"]]

    if results:
        print(f"\n=== 検索結果（{len(results)}件）===")
        for c in results:
            print(f"  {c['name']} / {c['phone']} / {c['email']}")
    else:
        print("見つかりませんでした。")


def list_contacts(contacts):
    """連絡先を一覧表示する"""
    if not contacts:
        print("連絡先が登録されていません。")
        return

    print(f"\n=== 連絡先一覧（{len(contacts)}件）===")
    for i, c in enumerate(contacts, start=1):
        print(f"{i}. {c['name']}")
        print(f"   電話: {c['phone']}")
        print(f"   メール: {c['email']}")


def delete_contact(contacts):
    """連絡先を削除する"""
    list_contacts(contacts)
    if not contacts:
        return

    try:
        num = int(input("削除する番号: "))
        if 1 <= num <= len(contacts):
            removed = contacts.pop(num - 1)
            print(f"{removed['name']} を削除しました")
        else:
            print("無効な番号です。")
    except ValueError:
        print("数値を入力してください。")


# === メインループ ===
contacts = load_contacts()
print("=== 住所録管理 ===")

while True:
    print("\n1. 追加  2. 検索  3. 一覧  4. 削除  5. 終了")
    choice = input("選択: ").strip()

    if choice == "1":
        add_contact(contacts)
        save_contacts(contacts)
    elif choice == "2":
        search_contacts(contacts)
    elif choice == "3":
        list_contacts(contacts)
    elif choice == "4":
        delete_contact(contacts)
        save_contacts(contacts)
    elif choice == "5":
        save_contacts(contacts)
        print("データを保存して終了します。")
        break
    else:
        print("1〜5の数字を入力してください。")
```

</details>

## 問題8：テキストファイル統計ツール（チャレンジ）

テキストファイルを読み込んで、以下の統計情報を表示するプログラムを作成してください。

1. 行数
2. 単語数
3. 文字数（空白含む/含まない）
4. 最も長い行と最も短い行
5. 各単語の出現頻度TOP5

まずテスト用のファイルを作成してから分析してください。

**期待される出力（例）：**
```
=== ファイル統計: sample_text.txt ===
行数        : 5
単語数      : 28
文字数(空白含む): 168
文字数(空白除く): 141

最も長い行 (42文字):
  "Pythonはデータ分析やWeb開発など幅広い分野で使われています"

最も短い行 (18文字):
  "Pythonを学びましょう"

=== 単語出現頻度 TOP5 ===
1. Python    : 3回
2. は        : 3回
3. で        : 2回
4. プログラミング: 2回
5. 学ぶ      : 1回
```

<details>
<summary>ヒント</summary>

- `collections.Counter` で単語頻度を集計
- `max()` / `min()` の `key` パラメータで最長・最短行を取得
- 日本語の単語分割は空白で区切る簡易版でOK

</details>

<details>
<summary>解答例</summary>

```python
from collections import Counter
from pathlib import Path

# テスト用ファイルの作成
sample_text = """Pythonは初心者に優しいプログラミング言語です
Pythonはデータ分析やWeb開発など幅広い分野で使われています
プログラミングを学ぶにはまず手を動かすことが大切です
Pythonで楽しくコードを書いてスキルを磨きましょう
Pythonを学びましょう"""

Path("sample_text.txt").write_text(sample_text, encoding="utf-8")

# ファイルを読み込む
with open("sample_text.txt", "r", encoding="utf-8") as f:
    content = f.read()

lines = content.split("\n")

# 基本統計
word_count = sum(len(line.split()) for line in lines)
char_count_with_space = len(content)
char_count_no_space = len(content.replace(" ", "").replace("\n", ""))

print(f"=== ファイル統計: sample_text.txt ===")
print(f"行数        : {len(lines)}")
print(f"単語数      : {word_count}")
print(f"文字数(空白含む): {char_count_with_space}")
print(f"文字数(空白除く): {char_count_no_space}")

# 最も長い行と短い行
longest = max(lines, key=len)
shortest = min(lines, key=len)
print(f'\n最も長い行 ({len(longest)}文字):')
print(f'  "{longest}"')
print(f'\n最も短い行 ({len(shortest)}文字):')
print(f'  "{shortest}"')

# 単語出現頻度（空白区切りの簡易版）
all_words = []
for line in lines:
    all_words.extend(line.split())

counter = Counter(all_words)
print(f"\n=== 単語出現頻度 TOP5 ===")
for rank, (word, count) in enumerate(counter.most_common(5), start=1):
    print(f"{rank}. {word:<10}: {count}回")
```

</details>
