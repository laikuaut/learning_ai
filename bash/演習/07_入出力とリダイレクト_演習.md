### 問題1: リダイレクトの基本
以下の各コマンドの動作を説明してください。

```bash
1. echo "Hello" > output.txt
2. echo "World" >> output.txt
3. ls /nonexistent 2> error.log
4. command > all.log 2>&1
5. command &> /dev/null
```

<details>
<summary>解答</summary>

```bash
# 1. "Hello" を output.txt に上書き（既存の内容は消える）
# 2. "World" を output.txt に追記（既存の内容の後に追加）
# 3. エラー出力のみを error.log に書き込む（画面には表示されない）
# 4. 標準出力を all.log に、標準エラー出力も同じ all.log に書き込む
# 5. 標準出力も標準エラー出力もすべて捨てる（完全に無音）
```

</details>

### 問題2: ヒアドキュメント
ヒアドキュメントを使って、以下の内容のHTMLファイル（/tmp/test.html）を生成するスクリプトを書いてください。タイトルは変数 `title="My Page"` の値を使ってください。

期待される出力ファイル：
```html
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>My Page</h1>
</body>
</html>
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
title="My Page"

# ヒアドキュメントで変数を展開しつつファイル生成
cat <<EOF > /tmp/test.html
<!DOCTYPE html>
<html>
<head>
    <title>${title}</title>
</head>
<body>
    <h1>${title}</h1>
</body>
</html>
EOF

echo "生成しました: /tmp/test.html"
cat /tmp/test.html
```

</details>

### 問題3: パイプの活用
`/etc/passwd` ファイルから、シェルが `/bin/bash` のユーザーだけを抽出し、ユーザー名のみをアルファベット順に表示するワンライナーを書いてください。

期待される出力例：
```
root
tanaka
user1
```

<details>
<summary>解答</summary>

```bash
# /etc/passwd からbashユーザーを抽出し、ユーザー名だけソートして表示
grep '/bin/bash$' /etc/passwd | cut -d: -f1 | sort
```

</details>

### 問題4: teeコマンド
コマンドの実行結果を画面に表示しつつ、ログファイル（/tmp/system_check.log）にも保存するスクリプトを書いてください。以下の情報を出力します。

1. 現在の日時
2. ホスト名
3. ディスク使用率（df -h /）

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
log_file="/tmp/system_check.log"

{
    echo "=== システムチェック ==="
    echo "日時: $(date '+%Y-%m-%d %H:%M:%S')"
    echo "ホスト名: $(hostname)"
    echo ""
    echo "=== ディスク使用率 ==="
    df -h /
} | tee "$log_file"

echo ""
echo "ログ保存先: $log_file"
```

</details>

### 問題5: 標準出力と標準エラー出力の分離
以下の関数は標準出力とエラー出力の両方を出します。標準出力をファイル `stdout.txt` に、標準エラー出力をファイル `stderr.txt` に、それぞれ別々にリダイレクトして実行してください。

```bash
mixed_output() {
    echo "正常な出力1"
    echo "エラーメッセージ1" >&2
    echo "正常な出力2"
    echo "エラーメッセージ2" >&2
}
```

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash

mixed_output() {
    echo "正常な出力1"
    echo "エラーメッセージ1" >&2
    echo "正常な出力2"
    echo "エラーメッセージ2" >&2
}

# stdout と stderr を別々のファイルにリダイレクト
mixed_output > /tmp/stdout.txt 2> /tmp/stderr.txt

echo "--- stdout.txt ---"
cat /tmp/stdout.txt
echo ""
echo "--- stderr.txt ---"
cat /tmp/stderr.txt

rm -f /tmp/stdout.txt /tmp/stderr.txt
```

</details>

### 問題6: ヒアストリングとプロセス置換
以下の2つの処理をそれぞれ書いてください。

1. ヒアストリング（`<<<`）を使って、`"hello:world:bash"` を `:` で分割し3つの変数に代入
2. プロセス置換（`<()`）を使って、`seq 1 5` と `seq 3 7` の出力から共通する行を表示

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash

# 1. ヒアストリングで分割
IFS=':' read -r a b c <<< "hello:world:bash"
echo "a=$a, b=$b, c=$c"
# 出力: a=hello, b=world, c=bash

# 2. プロセス置換で共通行を取得
echo "共通する数字:"
comm -12 <(seq 1 5) <(seq 3 7)
# 出力: 3 4 5
```

</details>

## チャレンジ問題

### 問題7: ログ関数の実装
以下の要件を満たすログ関数を実装してください。

- `log_info`, `log_warn`, `log_error` の3つの関数
- 形式: `[YYYY-MM-DD HH:MM:SS] [LEVEL] メッセージ`
- INFO と WARN は stdout に、ERROR は stderr に出力
- すべてのログを `/tmp/app.log` にも追記保存
- ログレベルに応じた色付き表示（INFO=緑、WARN=黄、ERROR=赤）

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
LOG_FILE="/tmp/app.log"

# 色の定義
readonly GREEN='\033[0;32m'
readonly YELLOW='\033[0;33m'
readonly RED='\033[0;31m'
readonly NC='\033[0m'

_log() {
    local level="$1"
    local color="$2"
    local fd="$3"
    shift 3
    local timestamp
    timestamp=$(date '+%Y-%m-%d %H:%M:%S')
    local plain="[$timestamp] [$level] $*"
    local colored="${color}[$timestamp] [$level]${NC} $*"

    # 色付きで画面に出力
    echo -e "$colored" >&"$fd"
    # プレーンテキストでファイルに追記
    echo "$plain" >> "$LOG_FILE"
}

log_info()  { _log "INFO"  "$GREEN"  1 "$@"; }
log_warn()  { _log "WARN"  "$YELLOW" 1 "$@"; }
log_error() { _log "ERROR" "$RED"    2 "$@"; }

# テスト
log_info "アプリケーションを起動しました"
log_warn "メモリ使用率が80%を超えています"
log_error "データベース接続に失敗しました"

echo ""
echo "--- ログファイルの内容 ---"
cat "$LOG_FILE"
rm -f "$LOG_FILE"
```

</details>

### 問題8: パイプラインの活用
以下の要件を満たすスクリプトを書いてください。

1. `/etc/passwd` から一般ユーザー（UID 1000以上）を抽出
2. ユーザー名とホームディレクトリだけを取り出す
3. 「ユーザー名: ホームディレクトリ」の形式で表示
4. 結果をアルファベット順にソート

<details>
<summary>解答</summary>

```bash
#!/usr/bin/env bash
# パイプラインで/etc/passwdを処理
awk -F: '$3 >= 1000 && $3 < 65534 {print $1 ": " $6}' /etc/passwd | sort
```

別解（パイプを段階的に使う場合）：
```bash
#!/usr/bin/env bash
# 段階的なパイプライン
while IFS=: read -r user _ uid _ _ home _; do
    if ((uid >= 1000 && uid < 65534)); then
        echo "$user: $home"
    fi
done < /etc/passwd | sort
```

</details>
