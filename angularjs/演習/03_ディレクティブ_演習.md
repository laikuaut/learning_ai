## 演習3-1：ng-repeat でリストを表示する（基本）

### 問題

以下の要件を満たす HTML を作成してください。

1. `fruits` 配列に5つ以上のフルーツ名を格納する
2. `ng-repeat` で `<ul><li>` のリストとして表示する
3. 各項目に `$index + 1` で番号を付ける
4. 最初の項目には「★」、最後の項目には「（ラスト）」を表示する

### 期待される表示

```
★ 1. りんご
2. バナナ
3. みかん
4. ぶどう
5. もも（ラスト）
```

<details>
<summary>ヒント</summary>

- `$index` は0始まりのインデックスです
- `$first` と `$last` で先頭・末尾を判定できます
- `ng-if` や `ng-show` と組み合わせて表示を切り替えます

</details>

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="FruitCtrl">
  <h2>フルーツリスト</h2>
  <ul>
    <li ng-repeat="fruit in fruits">
      <!-- $first なら★を表示 -->
      <span ng-if="$first">★ </span>
      <!-- 番号とフルーツ名 -->
      {{ $index + 1 }}. {{ fruit }}
      <!-- $last ならラストを表示 -->
      <span ng-if="$last">（ラスト）</span>
    </li>
  </ul>

  <script>
    angular.module('myApp', [])
      .controller('FruitCtrl', function($scope) {
        // フルーツ配列を定義
        $scope.fruits = ['りんご', 'バナナ', 'みかん', 'ぶどう', 'もも'];
      });
  </script>
</body>
</html>
```

**ポイント:** `$first`, `$last` は `ng-repeat` 内で使える特殊変数で、条件付き表示に便利です。

</details>

## 演習3-2：ng-if と ng-show の違いを体験する（基本）

### 問題

1. チェックボックスで表示/非表示を切り替える
2. `ng-if` と `ng-show` をそれぞれ使って同じ内容を表示する
3. ブラウザの開発者ツールで DOM の違いを確認する

### 期待される動作

- チェックを外すと両方の要素が非表示になる
- `ng-if` の要素は DOM から完全に消える
- `ng-show` の要素は `display: none` になるが DOM に残る

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .box { padding: 15px; margin: 10px 0; border-radius: 8px; }
    .box-if { background: #e3f2fd; border: 2px solid #2196f3; }
    .box-show { background: #e8f5e9; border: 2px solid #4caf50; }
  </style>
</head>
<body ng-controller="ToggleCtrl">
  <h2>ng-if vs ng-show</h2>
  <label>
    <input type="checkbox" ng-model="isVisible"> 表示する
  </label>
  
  <!-- ng-if: DOM から追加/削除される -->
  <div class="box box-if" ng-if="isVisible">
    <strong>ng-if</strong>: この要素は DOM から完全に削除されます
  </div>
  
  <!-- ng-show: display:none で隠れるだけ -->
  <div class="box box-show" ng-show="isVisible">
    <strong>ng-show</strong>: この要素は CSS で非表示になるだけです
  </div>
  
  <p>開発者ツール（F12）でDOMを確認してみましょう。</p>

  <script>
    angular.module('myApp', [])
      .controller('ToggleCtrl', function($scope) {
        $scope.isVisible = true;
      });
  </script>
</body>
</html>
```

**ポイント:** `ng-if` は DOM 要素を削除するため不要時にリソースを節約できますが、切り替え時のコストは `ng-show` より高くなります。

</details>

## 演習3-3：ソート・フィルター付きリスト（応用）

### 問題

生徒の成績一覧を以下の要件で表示してください。

1. 生徒データ（名前、学年、点数）を5人以上用意する
2. テキストボックスで名前の検索ができる
3. セレクトボックスでソート順（名前順、点数順、学年順）を切り替えられる
4. 点数が80以上は緑、60〜79は黄、60未満は赤の背景色を付ける

### 期待される動作

- 検索欄に文字を入力するとリアルタイムに絞り込まれる
- ソート順を切り替えると即座に並び替わる
- 点数に応じた色分けが表示される

<details>
<summary>ヒント</summary>

- `ng-repeat` に `| filter:search | orderBy:sortKey` をチェーンします
- `ng-class` でスコアに応じたクラスを付与します
- `orderBy` に `-score` のようにマイナスを付けると降順になります

</details>

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    table { border-collapse: collapse; width: 100%; margin-top: 10px; }
    th, td { border: 1px solid #ddd; padding: 8px; text-align: left; }
    th { background: #35495e; color: white; }
    input, select { padding: 8px; margin: 4px; border: 1px solid #ddd; border-radius: 4px; }
    /* 点数に応じた背景色 */
    .score-high { background: #c8e6c9; }
    .score-mid { background: #fff9c4; }
    .score-low { background: #ffcdd2; }
  </style>
</head>
<body ng-controller="StudentCtrl">
  <h2>生徒成績一覧</h2>
  
  <!-- 検索 -->
  <input type="text" ng-model="search" placeholder="名前で検索...">
  
  <!-- ソート -->
  <select ng-model="sortKey">
    <option value="name">名前順</option>
    <option value="-score">点数順（高→低）</option>
    <option value="score">点数順（低→高）</option>
    <option value="grade">学年順</option>
  </select>

  <table>
    <tr>
      <th>名前</th><th>学年</th><th>点数</th>
    </tr>
    <!-- filter で検索、orderBy でソート -->
    <tr ng-repeat="s in students | filter:search | orderBy:sortKey"
        ng-class="{
          'score-high': s.score >= 80,
          'score-mid': s.score >= 60 && s.score < 80,
          'score-low': s.score < 60
        }">
      <td>{{ s.name }}</td>
      <td>{{ s.grade }}年</td>
      <td>{{ s.score }}点</td>
    </tr>
  </table>
  
  <p>表示: {{ (students | filter:search).length }}人 / 全{{ students.length }}人</p>

  <script>
    angular.module('myApp', [])
      .controller('StudentCtrl', function($scope) {
        $scope.sortKey = 'name';
        $scope.students = [
          { name: '田中太郎', grade: 1, score: 85 },
          { name: '鈴木花子', grade: 2, score: 92 },
          { name: '佐藤一郎', grade: 1, score: 48 },
          { name: '高橋美咲', grade: 3, score: 73 },
          { name: '渡辺健一', grade: 2, score: 61 },
          { name: '山田次郎', grade: 3, score: 95 }
        ];
      });
  </script>
</body>
</html>
```

</details>

## 演習3-4：ng-class で動的スタイル（応用）

### 問題

タスクリストアプリで以下の要件を実装してください。

1. タスク一覧（タスク名、優先度: high/medium/low、完了フラグ）を表示
2. 優先度に応じて行の左ボーダーの色を変える（高:赤、中:黄、低:緑）
3. 完了したタスクは取り消し線と薄いテキスト色にする
4. チェックボックスで完了/未完了を切り替えられる

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .task { padding: 10px; margin: 4px 0; border-radius: 4px; border-left: 4px solid #ddd; display: flex; align-items: center; gap: 10px; }
    .priority-high { border-left-color: #e74c3c; }
    .priority-medium { border-left-color: #f39c12; }
    .priority-low { border-left-color: #27ae60; }
    .completed { text-decoration: line-through; color: #999; background: #f5f5f5; }
  </style>
</head>
<body ng-controller="TaskCtrl">
  <h2>タスクリスト</h2>
  <div ng-repeat="task in tasks"
       class="task"
       ng-class="[
         'priority-' + task.priority,
         { 'completed': task.done }
       ]">
    <input type="checkbox" ng-model="task.done">
    <span>{{ task.name }}</span>
    <small>({{ task.priority }})</small>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('TaskCtrl', function($scope) {
        $scope.tasks = [
          { name: 'バグ修正', priority: 'high', done: false },
          { name: 'ドキュメント作成', priority: 'medium', done: false },
          { name: 'コードレビュー', priority: 'high', done: true },
          { name: 'テスト追加', priority: 'medium', done: false },
          { name: 'ライブラリ更新', priority: 'low', done: false }
        ];
      });
  </script>
</body>
</html>
```

**ポイント:** `ng-class` の配列構文で、文字列とオブジェクトを組み合わせて使えます。

</details>

## 演習3-5：商品カタログの構築（チャレンジ）

### 問題

以下の機能を持つ商品カタログページを作成してください。

1. 商品データ（名前、カテゴリ、価格、在庫有無）を8つ以上用意する
2. カテゴリで絞り込めるボタン（全て、電子機器、食品、衣類など）
3. 名前で検索できるテキストボックス
4. 価格の安い順/高い順でソートできるセレクトボックス
5. 在庫なしの商品はグレーアウト表示する
6. 検索結果が0件の場合は「該当する商品がありません」と `ng-if` で表示
7. 商品数の表示

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 800px; margin: 20px auto; padding: 0 20px; }
    .controls { background: #f5f5f5; padding: 16px; border-radius: 8px; margin-bottom: 16px; }
    .controls input, .controls select { padding: 8px; border: 1px solid #ddd; border-radius: 4px; margin: 4px; }
    .category-btn { padding: 6px 14px; border: 1px solid #ddd; background: white; border-radius: 20px; cursor: pointer; margin: 2px; }
    .category-btn.active { background: #42b883; color: white; border-color: #42b883; }
    .product-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(180px, 1fr)); gap: 12px; }
    .product-card { border: 1px solid #ddd; border-radius: 8px; padding: 16px; }
    .product-card h3 { margin: 0 0 8px; font-size: 16px; }
    .product-card .price { color: #e74c3c; font-weight: bold; font-size: 18px; }
    .product-card .category { font-size: 12px; color: #666; background: #eee; padding: 2px 8px; border-radius: 10px; }
    .out-of-stock { opacity: 0.5; }
    .out-of-stock::after { content: '在庫切れ'; display: block; color: #e74c3c; font-weight: bold; margin-top: 8px; }
    .no-results { text-align: center; padding: 40px; color: #999; }
  </style>
</head>
<body ng-controller="CatalogCtrl">
  <h2>商品カタログ</h2>
  
  <div class="controls">
    <div>
      <button class="category-btn" ng-class="{ active: selectedCategory === '' }" ng-click="selectedCategory = ''">全て</button>
      <button class="category-btn" ng-repeat="cat in categories" ng-class="{ active: selectedCategory === cat }" ng-click="$parent.selectedCategory = cat">{{ cat }}</button>
    </div>
    <div style="margin-top: 8px;">
      <input type="text" ng-model="searchText" placeholder="商品名で検索...">
      <select ng-model="sortKey">
        <option value="name">名前順</option>
        <option value="price">価格: 安い順</option>
        <option value="-price">価格: 高い順</option>
      </select>
    </div>
  </div>

  <p>{{ filteredProducts().length }}件の商品</p>

  <!-- 検索結果なし -->
  <div class="no-results" ng-if="filteredProducts().length === 0">
    該当する商品がありません
  </div>

  <div class="product-grid">
    <div class="product-card" ng-repeat="p in filteredProducts() | orderBy:sortKey" ng-class="{ 'out-of-stock': !p.inStock }">
      <span class="category">{{ p.category }}</span>
      <h3>{{ p.name }}</h3>
      <p class="price">¥{{ p.price | number }}</p>
    </div>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('CatalogCtrl', function($scope, $filter) {
        $scope.searchText = '';
        $scope.selectedCategory = '';
        $scope.sortKey = 'name';
        $scope.categories = ['電子機器', '食品', '衣類', '書籍'];
        
        $scope.products = [
          { name: 'ワイヤレスイヤホン', category: '電子機器', price: 5980, inStock: true },
          { name: 'スマートウォッチ', category: '電子機器', price: 29800, inStock: true },
          { name: 'USBケーブル', category: '電子機器', price: 980, inStock: false },
          { name: 'オーガニックコーヒー', category: '食品', price: 1280, inStock: true },
          { name: '抹茶セット', category: '食品', price: 3500, inStock: true },
          { name: 'Tシャツ', category: '衣類', price: 2980, inStock: true },
          { name: 'パーカー', category: '衣類', price: 5480, inStock: false },
          { name: 'プログラミング入門', category: '書籍', price: 2640, inStock: true },
          { name: 'デザインパターン', category: '書籍', price: 3960, inStock: true }
        ];
        
        // フィルタリングされた商品を返す関数
        $scope.filteredProducts = function() {
          var result = $scope.products;
          // カテゴリフィルター
          if ($scope.selectedCategory) {
            result = result.filter(function(p) {
              return p.category === $scope.selectedCategory;
            });
          }
          // テキスト検索
          if ($scope.searchText) {
            result = $filter('filter')(result, { name: $scope.searchText });
          }
          return result;
        };
      });
  </script>
</body>
</html>
```

</details>
