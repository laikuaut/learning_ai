## 演習4-1：組み込みフィルターを使う（基本）

### 問題

商品一覧を以下のフィルターを使って表示してください。

1. 価格に `currency` フィルターを適用（¥マーク、小数なし）
2. 登録日に `date` フィルターを適用（yyyy年MM月dd日 形式）
3. 商品名に `uppercase` フィルターを適用して大文字版も表示
4. 説明文に `limitTo` フィルターを適用して30文字まで表示

### 期待される表示例

```
商品名: ワイヤレスマウス（ワイヤレスマウス）
価格: ¥2,980
登録日: 2024年03月15日
説明: 高精度センサー搭載のワイヤレスマウス。人間...
```

## 演習4-2：カスタムフィルターを作成する（基本）

### 問題

電話番号をフォーマットするカスタムフィルター `phoneFormat` を作成してください。

1. `09012345678` → `090-1234-5678`（携帯: 3-4-4）
2. `0312345678` → `03-1234-5678`（固定: 2-4-4 または 3-4-4）
3. 数字以外が含まれていたら除去してからフォーマット

### 期待される表示

```
090-1234-5678
03-1234-5678
```

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="PhoneCtrl">
  <h2>電話番号フォーマット</h2>
  
  <input type="text" ng-model="rawPhone" placeholder="電話番号を入力">
  <p>フォーマット済み: {{ rawPhone | phoneFormat }}</p>
  
  <h3>サンプル</h3>
  <ul>
    <li ng-repeat="phone in phones">
      {{ phone }} → {{ phone | phoneFormat }}
    </li>
  </ul>

  <script>
    angular.module('myApp', [])
      .filter('phoneFormat', function() {
        return function(input) {
          if (!input) return '';
          // 数字以外を除去
          var cleaned = String(input).replace(/\D/g, '');
          // 11桁（携帯）: 3-4-4
          if (cleaned.length === 11) {
            return cleaned.replace(/(\d{3})(\d{4})(\d{4})/, '$1-$2-$3');
          }
          // 10桁（固定）: 先頭2桁が市外局番
          if (cleaned.length === 10) {
            return cleaned.replace(/(\d{2,3})(\d{4})(\d{4})/, '$1-$2-$3');
          }
          return input;
        };
      })
      .controller('PhoneCtrl', function($scope) {
        $scope.rawPhone = '';
        $scope.phones = ['09012345678', '0312345678', '090-9876-5432'];
      });
  </script>
</body>
</html>
```

</details>

## 演習4-3：$http でAPIデータを取得する（応用）

### 問題

JSONPlaceholder API からユーザー一覧を取得して表示してください。

1. ボタンクリックで `https://jsonplaceholder.typicode.com/users` からデータを取得
2. 読み込み中は「読み込み中...」と表示
3. エラー時はエラーメッセージを表示
4. ユーザー名、メール、会社名をカード形式で表示
5. 名前で検索できるフィルター機能を付ける

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 700px; margin: 20px auto; padding: 0 20px; }
    .card { border: 1px solid #ddd; border-radius: 8px; padding: 12px 16px; margin: 8px 0; }
    .card h3 { margin: 0 0 4px; color: #35495e; }
    .card p { margin: 2px 0; font-size: 14px; color: #666; }
    .loading { color: #999; font-style: italic; }
    .error { color: #e74c3c; }
    input { padding: 8px; border: 1px solid #ddd; border-radius: 4px; width: 200px; margin-left: 10px; }
    button { padding: 8px 16px; background: #42b883; color: white; border: none; border-radius: 4px; cursor: pointer; }
  </style>
</head>
<body ng-controller="ApiCtrl">
  <h2>ユーザー一覧（API）</h2>
  
  <button ng-click="loadUsers()" ng-disabled="loading">データを取得</button>
  <input ng-model="search" placeholder="名前で検索..." ng-show="users.length > 0">
  
  <p class="loading" ng-show="loading">読み込み中...</p>
  <p class="error" ng-show="error">{{ error }}</p>
  
  <!-- ユーザーカード -->
  <div class="card" ng-repeat="user in users | filter:{ name: search }">
    <h3>{{ user.name }}</h3>
    <p>📧 {{ user.email }}</p>
    <p>🏢 {{ user.company.name }}</p>
  </div>
  
  <p ng-show="users.length > 0">
    表示: {{ (users | filter:{ name: search }).length }} / {{ users.length }}人
  </p>

  <script>
    angular.module('myApp', [])
      .controller('ApiCtrl', ['$scope', '$http', function($scope, $http) {
        $scope.users = [];
        $scope.loading = false;
        $scope.error = null;
        
        $scope.loadUsers = function() {
          $scope.loading = true;
          $scope.error = null;
          
          // $http.get で API を呼び出し
          $http.get('https://jsonplaceholder.typicode.com/users')
            .then(function(response) {
              // 成功: response.data にデータが入る
              $scope.users = response.data;
            })
            .catch(function(error) {
              // エラー処理
              $scope.error = 'データの取得に失敗しました: ' + error.statusText;
            })
            .finally(function() {
              // 成功でもエラーでも実行
              $scope.loading = false;
            });
        };
      }]);
  </script>
</body>
</html>
```

</details>

## 演習4-4：カスタムサービスを作成する（応用）

### 問題

`CartService` というファクトリーサービスを作成し、買い物カート機能を実装してください。

1. `addItem(product)` — 商品を追加（同じ商品は数量を増やす）
2. `removeItem(product)` — 商品を削除
3. `getItems()` — カート内の商品一覧を返す
4. `getTotal()` — 合計金額を返す
5. `clearCart()` — カートを空にする
6. 商品一覧とカート表示を別々のコントローラで作成し、サービスを共有する

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 700px; margin: 20px auto; }
    .section { border: 1px solid #ddd; border-radius: 8px; padding: 16px; margin: 10px 0; }
    .product-row { display: flex; justify-content: space-between; align-items: center; padding: 8px 0; border-bottom: 1px solid #eee; }
    .cart-item { display: flex; justify-content: space-between; padding: 6px 0; }
    .btn { padding: 6px 12px; border: none; border-radius: 4px; cursor: pointer; color: white; }
    .btn-add { background: #42b883; }
    .btn-remove { background: #e74c3c; }
    .btn-clear { background: #95a5a6; }
    .total { font-size: 1.3em; font-weight: bold; margin-top: 10px; }
  </style>
</head>
<body>
  <!-- 商品一覧（ShopCtrl） -->
  <div ng-controller="ShopCtrl" class="section">
    <h2>商品一覧</h2>
    <div class="product-row" ng-repeat="p in products">
      <span>{{ p.name }} — ¥{{ p.price | number }}</span>
      <button class="btn btn-add" ng-click="addToCart(p)">カートに追加</button>
    </div>
  </div>
  
  <!-- カート表示（CartCtrl） -->
  <div ng-controller="CartCtrl" class="section">
    <h2>カート（{{ getItemCount() }}点）</h2>
    <p ng-if="cartItems.length === 0">カートは空です</p>
    <div class="cart-item" ng-repeat="item in cartItems">
      <span>{{ item.name }} × {{ item.qty }}</span>
      <span>¥{{ item.price * item.qty | number }}
        <button class="btn btn-remove" ng-click="remove(item)">×</button>
      </span>
    </div>
    <div class="total" ng-show="cartItems.length > 0">
      合計: ¥{{ total | number }}
      <button class="btn btn-clear" ng-click="clear()">カートを空にする</button>
    </div>
  </div>

  <script>
    angular.module('myApp', [])
      // カートサービス（factory パターン）
      .factory('CartService', function() {
        var items = [];
        return {
          addItem: function(product) {
            // 同じ商品があれば数量を増やす
            var existing = items.find(function(i) { return i.name === product.name; });
            if (existing) {
              existing.qty++;
            } else {
              items.push({ name: product.name, price: product.price, qty: 1 });
            }
          },
          removeItem: function(item) {
            var idx = items.indexOf(item);
            if (idx > -1) items.splice(idx, 1);
          },
          getItems: function() { return items; },
          getTotal: function() {
            return items.reduce(function(sum, i) { return sum + i.price * i.qty; }, 0);
          },
          getCount: function() {
            return items.reduce(function(sum, i) { return sum + i.qty; }, 0);
          },
          clearCart: function() { items.length = 0; }
        };
      })
      // 商品コントローラ
      .controller('ShopCtrl', ['$scope', 'CartService', function($scope, CartService) {
        $scope.products = [
          { name: 'コーヒー', price: 350 },
          { name: 'サンドイッチ', price: 480 },
          { name: 'ケーキ', price: 550 },
          { name: 'ジュース', price: 200 }
        ];
        $scope.addToCart = function(product) {
          CartService.addItem(product);
        };
      }])
      // カートコントローラ
      .controller('CartCtrl', ['$scope', 'CartService', function($scope, CartService) {
        $scope.$watch(function() {
          return CartService.getCount() + CartService.getTotal();
        }, function() {
          $scope.cartItems = CartService.getItems();
          $scope.total = CartService.getTotal();
        });
        $scope.getItemCount = function() { return CartService.getCount(); };
        $scope.remove = function(item) { CartService.removeItem(item); };
        $scope.clear = function() { CartService.clearCart(); };
      }]);
  </script>
</body>
</html>
```

## 演習4-5：データダッシュボードの構築（チャレンジ）

### 問題

以下の機能を持つデータダッシュボードを作成してください。

1. JSONPlaceholder API から投稿一覧（`/posts`）を取得する
2. カスタムフィルター `truncate` で本文を50文字に省略する
3. カスタムサービス `BookmarkService` でお気に入り登録できる
4. お気に入り登録した投稿は別セクションに表示する
5. 投稿のタイトルで検索できる

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 800px; margin: 20px auto; padding: 0 20px; }
    .post { border: 1px solid #ddd; border-radius: 8px; padding: 12px; margin: 8px 0; }
    .post h3 { margin: 0 0 8px; font-size: 15px; }
    .post p { color: #666; font-size: 14px; margin: 0; }
    .bookmark-section { background: #fff8e1; padding: 16px; border-radius: 8px; margin-bottom: 20px; }
    input[type="text"] { padding: 8px; border: 1px solid #ddd; border-radius: 4px; width: 300px; }
    .btn { padding: 4px 10px; border: none; border-radius: 4px; cursor: pointer; font-size: 13px; }
    .btn-bookmark { background: #f39c12; color: white; }
    .btn-remove { background: #e74c3c; color: white; }
    .loading { color: #999; }
  </style>
</head>
<body ng-controller="DashboardCtrl">
  <h2>投稿ダッシュボード</h2>
  
  <!-- お気に入りセクション -->
  <div class="bookmark-section" ng-show="bookmarks.length > 0">
    <h3>★ お気に入り（{{ bookmarks.length }}件）</h3>
    <div class="post" ng-repeat="post in bookmarks">
      <h3>{{ post.title }}</h3>
      <p>{{ post.body | truncate:50 }}</p>
      <button class="btn btn-remove" ng-click="removeBookmark(post)">お気に入り解除</button>
    </div>
  </div>
  
  <!-- 検索 -->
  <input type="text" ng-model="searchTitle" placeholder="タイトルで検索...">
  <button ng-click="loadPosts()" ng-disabled="loading" style="padding:8px 16px;">データ取得</button>
  
  <p class="loading" ng-show="loading">読み込み中...</p>
  
  <!-- 投稿一覧 -->
  <div class="post" ng-repeat="post in posts | filter:{ title: searchTitle } | limitTo:20">
    <h3>{{ post.title }}</h3>
    <p>{{ post.body | truncate:50 }}</p>
    <button class="btn btn-bookmark" ng-click="addBookmark(post)" ng-hide="isBookmarked(post)">★ お気に入り</button>
    <span ng-show="isBookmarked(post)" style="color: #f39c12;">★ お気に入り済み</span>
  </div>

  <script>
    angular.module('myApp', [])
      // カスタムフィルター: 文字列省略
      .filter('truncate', function() {
        return function(input, maxLen) {
          if (!input) return '';
          maxLen = maxLen || 30;
          if (input.length <= maxLen) return input;
          return input.substring(0, maxLen) + '...';
        };
      })
      // お気に入りサービス
      .factory('BookmarkService', function() {
        var bookmarks = [];
        return {
          add: function(post) {
            if (!bookmarks.find(function(b) { return b.id === post.id; })) {
              bookmarks.push(post);
            }
          },
          remove: function(post) {
            var idx = bookmarks.findIndex(function(b) { return b.id === post.id; });
            if (idx > -1) bookmarks.splice(idx, 1);
          },
          getAll: function() { return bookmarks; },
          isBookmarked: function(post) {
            return bookmarks.some(function(b) { return b.id === post.id; });
          }
        };
      })
      .controller('DashboardCtrl', ['$scope', '$http', 'BookmarkService',
        function($scope, $http, BookmarkService) {
          $scope.posts = [];
          $scope.loading = false;
          $scope.bookmarks = BookmarkService.getAll();
          
          $scope.loadPosts = function() {
            $scope.loading = true;
            $http.get('https://jsonplaceholder.typicode.com/posts')
              .then(function(res) { $scope.posts = res.data; })
              .catch(function() { alert('取得に失敗しました'); })
              .finally(function() { $scope.loading = false; });
          };
          
          $scope.addBookmark = function(post) { BookmarkService.add(post); };
          $scope.removeBookmark = function(post) { BookmarkService.remove(post); };
          $scope.isBookmarked = function(post) { return BookmarkService.isBookmarked(post); };
          
          // 初回読み込み
          $scope.loadPosts();
        }
      ]);
  </script>
</body>
</html>
```

</details>
