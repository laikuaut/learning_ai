# 実践課題01：カウンターアプリ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（基本構文と変数）、第2章（演算子と型変換）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 変数の宣言・代入・演算子を使い、ブラウザで動くシンプルなアプリを作る体験をする

---

## 完成イメージ

```
===== カウンター =====

現在の値: 0

[+1] [-1] [+10] [-10] [リセット]

+1 を押した → 現在の値: 1
+1 を押した → 現在の値: 2
-1 を押した → 現在の値: 1
+10 を押した → 現在の値: 11
リセットを押した → 現在の値: 0
```

ブラウザで開くと、ボタンを押すたびにカウンターの値が増減するページが表示されます。

---

## 課題の要件

1. HTMLファイル1つで完結すること（ブラウザで開くだけで動作）
2. カウンターの現在値を画面に表示する
3. 「+1」「-1」「+10」「-10」「リセット」の5つのボタンを用意する
4. ボタンを押すと即座に画面の表示が更新される
5. `let` を使ってカウンター変数を管理する

---

<summary>ステップ1：HTMLの骨組みを作る</summary>

まず、表示エリアとボタンを配置するHTMLを書きましょう。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">

</head>
<body>
  <h1>カウンター</h1>

  <button onclick="increment(1)">+1</button>
  <button onclick="increment(-1)">-1</button>
  <!-- 残りのボタンも同様に追加 -->

  <script>
    // ここにJavaScriptを書く
  </script>
</body>
</html>
```

</details>

<details>
<summary>ステップ2：変数を宣言して更新関数を作る</summary>

`let` でカウンター変数を宣言し、値を変更して画面に反映する関数を作ります。

```javascript
let count = 0;

function increment(value) {
  count = count + value;
  // 画面の表示を更新する処理をここに書く
}
```

画面の更新には `document.getElementById()` と `.textContent` を使います。

</details>

<details>
<summary>ステップ3：リセット機能を追加する</summary>

リセットボタン用の関数を別に作るか、`increment` の引数を工夫して対応します。

```javascript
function resetCount() {
  count = 0;
  // 画面を更新
}
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>カウンター</title>
  <style>
    body { font-family: sans-serif; text-align: center; margin-top: 50px; }
    button { font-size: 18px; margin: 5px; padding: 10px 20px; cursor: pointer; }
    #count { font-size: 48px; font-weight: bold; color: #333; }
  </style>
</head>
<body>
  <h1>カウンター</h1>
  <p>現在の値:</p>
  <p id="count">0</p>

  <button onclick="increment(1)">+1</button>
  <button onclick="increment(-1)">-1</button>
  <button onclick="increment(10)">+10</button>
  <button onclick="increment(-10)">-10</button>
  <button onclick="resetCount()">リセット</button>

  <script>
    // カウンターの現在値を保持する変数
    let count = 0;

    // 画面の表示を更新する関数
    function updateDisplay() {
      document.getElementById("count").textContent = count;
    }

    // カウンターを増減させる関数
    function increment(value) {
      count = count + value;
      updateDisplay();
    }

    // カウンターをリセットする関数
    function resetCount() {
      count = 0;
      updateDisplay();
    }
  </script>
</body>
</html>
```

</details>

<details>
<summary>解答例（改良版 ─ 第3章・第8章の知識を先取り）</summary>

改良版では、値に応じて色を変え、履歴を表示し、`const` と条件分岐を活用します。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>カウンター（改良版）</title>
  <style>
    body { font-family: sans-serif; text-align: center; margin-top: 30px; }
    button { font-size: 16px; margin: 4px; padding: 8px 16px; cursor: pointer; border: 1px solid #999; border-radius: 4px; }
    button:hover { background-color: #e0e0e0; }
    #count { font-size: 56px; font-weight: bold; transition: color 0.2s; }
    .positive { color: #2e7d32; }
    .negative { color: #c62828; }
    .zero { color: #333; }
    #history { margin-top: 20px; font-size: 14px; color: #666; }
  </style>
</head>
<body>
  <h1>カウンター</h1>
  <p id="count" class="zero">0</p>

  <button onclick="change(1)">+1</button>
  <button onclick="change(-1)">-1</button>
  <button onclick="change(10)">+10</button>
  <button onclick="change(-10)">-10</button>
  <button onclick="reset()">リセット</button>

  <div id="history">操作履歴がここに表示されます</div>

  <script>
    // 現在のカウント値
    let count = 0;
    // 操作履歴を保持する配列
    const history = [];

    // 表示を更新する関数
    function updateDisplay() {
      const el = document.getElementById("count");
      el.textContent = count;

      // 値に応じて色を変更
      if (count > 0) {
        el.className = "positive";
      } else if (count < 0) {
        el.className = "negative";
      } else {
        el.className = "zero";
      }

      // 履歴を表示（直近5件）
      const recent = history.slice(-5).reverse();
      document.getElementById("history").textContent =
        "履歴: " + recent.join(" → ");
    }

    // カウントを変更する関数
    function change(value) {
      count += value;  // 複合代入演算子を使用
      const sign = value > 0 ? "+" : "";
      history.push(sign + value + " → " + count);
      updateDisplay();
    }

    // リセットする関数
    function reset() {
      count = 0;
      history.push("リセット → 0");
      updateDisplay();
    }
  </script>
</body>
</html>
```

**初心者向けとの違い:**
- `const` と `let` を使い分けている（変更しない変数は `const`）
- 複合代入演算子 `+=` を使用
- 三項演算子（`? :`）で符号の表示を切り替え
- 値の正負に応じてCSSクラスで色を動的に変更
- 操作履歴を配列で管理し、`slice()` で直近5件だけ表示

</details>
