# 実践課題01：はじめての vimrc 作成 ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（vimrcの基本と仕組み）
> **課題の種類**: ミニプロジェクト
> **学習目標**: vimrc ファイルを新規作成し、基本的な `set` コマンドでエディタの動作を変更する体験をする

---

## 完成イメージ

以下の設定が反映された vimrc を作成し、Vim を起動した際に設定が有効になっていることを確認します。

```
┌──────────────────────────────────────────────────────────┐
│  Vim 起動後の状態                                        │
├──────────────────────────────────────────────────────────┤
│  - 左端に行番号が表示されている                          │
│  - 入力中のコマンドが右下に表示されている                │
│  - 対応する括弧がハイライトされる                        │
│  - ビープ音が鳴らない                                    │
│  - バックスペースが直感的に動作する                      │
│  - ファイルのエンコーディングが UTF-8 になっている       │
└──────────────────────────────────────────────────────────┘
```

`:set number?` を実行すると `number` と表示され、`:set encoding?` を実行すると `encoding=utf-8` と表示されます。

---

## 課題の要件

1. vimrc ファイルを適切な場所に新規作成する
2. `set nocompatible` で Vi 互換モードを無効にする
3. 文字コードを UTF-8 に設定する（`encoding` と `fileencoding`）
4. 行番号を表示する
5. 入力中のコマンドを右下に表示する（`showcmd`）
6. 対応する括弧のハイライトを有効にする（`showmatch`）
7. バックスペースの挙動を直感的にする
8. ビープ音を無効にする
9. セクションごとにコメントで区切り、読みやすく整理する
10. `:source` コマンドで再読み込みが正しく動作することを確認する

---

## ステップガイド

<details>
<summary>ステップ1：vimrc ファイルを作成する</summary>

ターミナルで以下のコマンドを実行して、vimrc を Vim で開きます。

```bash
# Linux / macOS の場合
vim ~/.vimrc

# ファイルが存在しない場合は新規ファイルとして開かれます
```

まだ何も書かれていない空のファイルが表示されるはずです。

</details>

<details>
<summary>ステップ2：セクションヘッダーを書く</summary>

vimrc を整理するために、コメントでセクションを区切りましょう。

```vim
" ==================================================
" 基本設定
" ==================================================

" ==================================================
" 文字コード
" ==================================================

" ==================================================
" 表示設定
" ==================================================

" ==================================================
" 操作設定
" ==================================================
```

</details>

<details>
<summary>ステップ3：各セクションに設定を記述する</summary>

各セクションの下に、対応する `set` コマンドを書きます。

- 「基本設定」 → `set nocompatible`
- 「文字コード」 → `encoding` と `fileencoding`
- 「表示設定」 → `number`, `showcmd`, `showmatch`
- 「操作設定」 → `backspace`, `belloff`

各 `set` コマンドの後にコメントで説明を付けましょう。

</details>

<details>
<summary>ステップ4：保存して確認する</summary>

1. `:w` で保存する
2. `:source %` で現在のファイルを再読み込みする
3. `:set number?` で設定が反映されているか確認する
4. 一度 Vim を終了（`:q`）して再度起動し、設定が自動で読み込まれることを確認する

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ==================================================
" 基本設定
" ==================================================

" Vi 互換モードを無効にする（Vim のフル機能を使う）
set nocompatible

" ==================================================
" 文字コード
" ==================================================

" Vim 内部のエンコーディング
set encoding=utf-8

" ファイル保存時のエンコーディング
set fileencoding=utf-8

" ==================================================
" 表示設定
" ==================================================

" 行番号を表示する
set number

" 入力中のコマンドを右下に表示する
set showcmd

" 対応する括弧を一瞬ハイライトする
set showmatch

" ==================================================
" 操作設定
" ==================================================

" バックスペースでインデント・改行・挿入開始位置を削除できるようにする
set backspace=indent,eol,start

" ビープ音をすべて無効にする
set belloff=all
```

</details>

<details>
<summary>解答例（改良版 ─ 少し踏み込んだ設定）</summary>

基本の設定に加えて、実用的な設定を追加したバージョンです。

```vim
" ==================================================
" 基本設定
" ==================================================
set nocompatible              " Vi 互換モードを無効にする

" ==================================================
" 文字コード
" ==================================================
set encoding=utf-8            " Vim 内部のエンコーディング
set fileencoding=utf-8        " ファイル保存時のエンコーディング
set fileencodings=utf-8,cp932,euc-jp  " ファイル読み込み時の文字コード自動判別

" ==================================================
" 表示設定
" ==================================================
set number                    " 行番号を表示する
set showcmd                   " 入力中のコマンドを右下に表示する
set showmatch                 " 対応する括弧をハイライトする
set ruler                     " カーソル位置を右下に表示する
set laststatus=2              " 常にステータスラインを表示する
set wildmenu                  " コマンド補完を拡張する

" ==================================================
" 操作設定
" ==================================================
set backspace=indent,eol,start  " バックスペースの挙動を直感的にする
set belloff=all                 " ビープ音をすべて無効にする
set confirm                     " 保存されていない変更がある時に確認ダイアログを出す
set hidden                      " バッファを閉じずに隠す（未保存でも切り替え可能）

" ==================================================
" vimrc 操作用マッピング
" ==================================================
" vimrc を素早く開く（Leader + ev）
nnoremap <Leader>ev :edit $MYVIMRC<CR>
" vimrc を素早く再読み込みする（Leader + sv）
nnoremap <Leader>sv :source $MYVIMRC<CR>
```

**初心者向けとの違い:**
- `fileencodings` で日本語ファイルの文字コード自動判別を追加
- `ruler`, `laststatus`, `wildmenu` で表示情報を充実
- `confirm`, `hidden` で操作の利便性を向上
- `$MYVIMRC` を使ったマッピングで vimrc の編集・再読み込みを高速化

</details>
