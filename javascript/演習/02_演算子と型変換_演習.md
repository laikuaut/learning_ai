## 基本問題

### 問題1：基本の四則演算

変数 `a = 17` と `b = 5` を使って、すべての算術演算の結果を表示してください。

**期待される出力：**
```
加算: 22
減算: 12
乗算: 85
除算: 3.4
剰余: 2
べき乗: 1419857
```

<details>
<summary>解答例</summary>

```javascript
const a = 17;
const b = 5;

console.log(`加算: ${a + b}`);
console.log(`減算: ${a - b}`);
console.log(`乗算: ${a * b}`);
console.log(`除算: ${a / b}`);
console.log(`剰余: ${a % b}`);
console.log(`べき乗: ${a ** b}`);
```

</details>

### 問題2：比較演算子の練習

以下の比較式の結果を予想してから実行し、`===` と `==` の違いを確認してください。

```javascript
console.log(5 === 5);
console.log(5 === "5");
console.log(5 == "5");
console.log(null === undefined);
console.log(null == undefined);
console.log(0 === false);
console.log(0 == false);
console.log("" == false);
```

**期待される出力：**
```
true
false
true
false
true
false
true
true
```

<details>
<summary>解説</summary>

```javascript
// === は型も含めて厳密に比較する（推奨）
console.log(5 === 5);           // true: 型も値も同じ
console.log(5 === "5");         // false: 型が異なる（number vs string）
console.log(5 == "5");          // true: 暗黙の型変換で数値比較

// null と undefined
console.log(null === undefined); // false: 異なる型
console.log(null == undefined);  // true: ==ではnullとundefinedは等しいとみなされる

// 0 と false
console.log(0 === false);       // false: 型が異なる
console.log(0 == false);        // true: falseが0に変換される

// "" と false
console.log("" == false);       // true: ""が0に、falseも0に変換される
```

</details>

### 問題3：論理演算子と短絡評価

以下の変数を使って、条件に応じたメッセージを表示してください。

```javascript
const username = "";
const role = "admin";
const score = 0;
```

1. `username` が空なら `"ゲスト"` を、そうでなければ `username` を表示（`||` を使う）
2. `role` が `"admin"` かつ `score` が 0 以上かどうかを表示（`&&` を使う）
3. `score` の値が `null` か `undefined` でなければ `score` を、そうでなければ `"未設定"` を表示（`??` を使う）

**期待される出力：**
```
表示名: ゲスト
管理者かつスコア有効: true
スコア: 0
```

<details>
<summary>ヒント</summary>

- `||` は左側がfalsyなら右側を返します。空文字列はfalsyです。
- `??` は左側が `null` または `undefined` のときだけ右側を返します。`0` はnullでもundefinedでもないので、そのまま `0` が使われます。

</details>

<details>
<summary>解答例</summary>

```javascript
const username = "";
const role = "admin";
const score = 0;

// || は falsy なら右側を返す（空文字列は falsy）
const displayName = username || "ゲスト";
console.log(`表示名: ${displayName}`);

// && で両方の条件をチェック
const isAdminAndValid = role === "admin" && score >= 0;
console.log(`管理者かつスコア有効: ${isAdminAndValid}`);

// ?? は null/undefined の場合のみ右側を返す（0 はそのまま使われる）
const displayScore = score ?? "未設定";
console.log(`スコア: ${displayScore}`);
```

</details>

### 問題4：三項演算子で成績判定

点数（`score`）に応じて成績を表示する処理を三項演算子で書いてください。

- 90以上: "A"
- 80以上: "B"
- 70以上: "C"
- 60以上: "D"
- 60未満: "F"

`score = 85` での出力を確認してください。

**期待される出力：**
```
85点の成績: B
```

<details>
<summary>解答例</summary>

```javascript
const score = 85;

// 三項演算子のネスト
const grade = score >= 90 ? "A"
            : score >= 80 ? "B"
            : score >= 70 ? "C"
            : score >= 60 ? "D"
            : "F";

console.log(`${score}点の成績: ${grade}`);
```

</details>

## 応用問題

### 問題5：Optional Chaining の活用

以下のオブジェクトに対して、Optional Chaining（`?.`）を使って安全にプロパティにアクセスしてください。存在しないプロパティにはデフォルト値を設定しましょう。

```javascript
const user = {
  name: "太郎",
  address: {
    city: "東京"
  },
  hobbies: ["読書", "映画"]
};

const guest = null;
```

以下を表示してください：
1. `user` の都市名
2. `user` の国名（存在しないのでデフォルト値 `"日本"` を表示）
3. `user` の最初の趣味
4. `guest` の名前（nullなのでデフォルト値 `"ゲスト"` を表示）

**期待される出力：**
```
都市: 東京
国: 日本
趣味: 読書
名前: ゲスト
```

<details>
<summary>解答例</summary>

```javascript
const user = {
  name: "太郎",
  address: { city: "東京" },
  hobbies: ["読書", "映画"]
};
const guest = null;

// ?. で安全にアクセスし、?? でデフォルト値を設定
console.log(`都市: ${user?.address?.city ?? "不明"}`);
console.log(`国: ${user?.address?.country ?? "日本"}`);
console.log(`趣味: ${user?.hobbies?.[0] ?? "なし"}`);
console.log(`名前: ${guest?.name ?? "ゲスト"}`);
```

</details>

### 問題6：型変換チャレンジ

以下の変換をすべて「明示的な型変換」で行い、結果を表示してください。

1. 文字列 `"42"` を数値に変換
2. 数値 `255` を16進数の文字列に変換
3. 数値 `0` を真偽値に変換
4. 文字列 `"hello"` を真偽値に変換
5. `null` を数値に変換
6. `undefined` を数値に変換

**期待される出力：**
```
"42" → 数値: 42
255 → 16進数: ff
0 → 真偽値: false
"hello" → 真偽値: true
null → 数値: 0
undefined → 数値: NaN
```

<details>
<summary>解答例</summary>

```javascript
// Number() で数値変換
console.log(`"42" → 数値: ${Number("42")}`);

// toString(16) で16進数文字列に変換
console.log(`255 → 16進数: ${(255).toString(16)}`);

// Boolean() で真偽値変換
console.log(`0 → 真偽値: ${Boolean(0)}`);
console.log(`"hello" → 真偽値: ${Boolean("hello")}`);

// null と undefined の数値変換
console.log(`null → 数値: ${Number(null)}`);         // 0
console.log(`undefined → 数値: ${Number(undefined)}`); // NaN
```

</details>

## チャレンジ問題

### 問題7：割引価格計算システム

以下の条件で割引価格を計算するプログラムを作成してください。

- 元の価格: 3000円
- 会員の場合: 10%割引
- さらに、クーポンコードが `"SAVE20"` の場合: 追加で20%割引（割引後の価格に対して）
- 税率: 10%

変数 `isMember = true` と `couponCode = "SAVE20"` を使って、最終的な税込価格を計算してください。

**期待される出力：**
```
元の価格: 3000円
会員割引後: 2700円
クーポン割引後: 2160円
税込価格: 2376円
```

<details>
<summary>ヒント</summary>

- 三項演算子を使って会員かどうかで割引率を切り替えます。
- 最終価格は `Math.floor()` で小数を切り捨てましょう。

</details>

<details>
<summary>解答例</summary>

```javascript
const originalPrice = 3000;
const isMember = true;
const couponCode = "SAVE20";
const TAX_RATE = 0.1;

// 会員割引の適用
const memberDiscount = isMember ? 0.1 : 0;
const afterMember = originalPrice * (1 - memberDiscount);

// クーポン割引の適用
const couponDiscount = couponCode === "SAVE20" ? 0.2 : 0;
const afterCoupon = afterMember * (1 - couponDiscount);

// 税込価格の計算（小数切り捨て）
const finalPrice = Math.floor(afterCoupon * (1 + TAX_RATE));

console.log(`元の価格: ${originalPrice}円`);
console.log(`会員割引後: ${afterMember}円`);
console.log(`クーポン割引後: ${afterCoupon}円`);
console.log(`税込価格: ${finalPrice}円`);
```

</details>

### 問題8：falsy値の完全チェック

JavaScriptの7つのfalsy値（`false`, `0`, `-0`, `""`, `null`, `undefined`, `NaN`）をすべてチェックし、それぞれが `Boolean()` で `false` になることを確認してください。また、よく間違えやすい「truthyな値」も3つ以上チェックしてください。

**期待される出力例：**
```
--- falsy値 ---
false → false
0 → false
-0 → false
"" → false
null → false
undefined → false
NaN → false
--- 間違えやすいtruthy値 ---
"0" → true
"false" → true
[] → true
{} → true
```

<details>
<summary>解答例</summary>

```javascript
console.log("--- falsy値 ---");
console.log(`false → ${Boolean(false)}`);
console.log(`0 → ${Boolean(0)}`);
console.log(`-0 → ${Boolean(-0)}`);
console.log(`"" → ${Boolean("")}`);
console.log(`null → ${Boolean(null)}`);
console.log(`undefined → ${Boolean(undefined)}`);
console.log(`NaN → ${Boolean(NaN)}`);

console.log("--- 間違えやすいtruthy値 ---");
console.log(`"0" → ${Boolean("0")}`);         // 文字列の"0"はtruthy
console.log(`"false" → ${Boolean("false")}`);  // 文字列の"false"もtruthy
console.log(`[] → ${Boolean([])}`);             // 空配列はtruthy
console.log(`{} → ${Boolean({})}`);             // 空オブジェクトもtruthy
```

</details>
