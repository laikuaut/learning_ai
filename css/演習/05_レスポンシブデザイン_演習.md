## 基本レベル（問題1〜3）

### 問題1：ビューポートとメディアクエリの基本

以下のHTMLで、画面幅に応じてボックスの色が変わるスタイルを書いてください。

```html
<div class="responsive-box">
  画面の幅を変えてみてください
</div>
```

**要件：**
- `.responsive-box`：padding `24px`、文字色 白、角丸 `8px`、テキスト中央揃え、文字サイズ `18px`
- デフォルト（モバイル）：背景色 `#e74c3c`（赤）
- 画面幅 `768px` 以上：背景色 `#2ecc71`（緑）
- 画面幅 `1024px` 以上：背景色 `#3498db`（青）

**動作イメージ：**
```
〜767px:    ┌──── 赤 ────┐
768〜1023px: ┌──── 緑 ──────────┐
1024px〜:    ┌──── 青 ────────────────────┐
```

<details>
<summary>ヒント</summary>

- `@media (min-width: 768px)` で768px以上の条件を指定
- メディアクエリは小さい順に書く（モバイルファースト）
- 大きい値のメディアクエリが後に来ると、上書きされる
</details>

<details>
<summary>解答例</summary>

```css
.responsive-box {
  padding: 24px;
  color: white;
  border-radius: 8px;
  text-align: center;
  font-size: 18px;
  /* モバイルファースト: デフォルトは赤 */
  background-color: #e74c3c;
}

/* タブレット以上（768px〜）で緑に */
@media (min-width: 768px) {
  .responsive-box {
    background-color: #2ecc71;
  }
}

/* デスクトップ以上（1024px〜）で青に */
@media (min-width: 1024px) {
  .responsive-box {
    background-color: #3498db;
  }
}
```
</details>

### 問題2：レスポンシブなカード列

以下のHTMLで、画面幅に応じてカードの列数が変わるレイアウトを作ってください。

```html
<div class="card-container">
  <div class="card">カード1</div>
  <div class="card">カード2</div>
  <div class="card">カード3</div>
  <div class="card">カード4</div>
</div>
```

**要件：**
- `.card-container`：Gridレイアウト、間隔 `16px`、padding `16px`
- `.card`：背景色 白、padding `20px`、枠線 `1px solid #ddd`、角丸 `8px`
- モバイル（デフォルト）：1列
- タブレット（`768px` 以上）：2列
- デスクトップ（`1024px` 以上）：4列

<details>
<summary>ヒント</summary>

- モバイルファーストで `grid-template-columns: 1fr`（1列）をデフォルトに
- `@media (min-width: 768px)` で `repeat(2, 1fr)`
- `@media (min-width: 1024px)` で `repeat(4, 1fr)`
</details>

<details>
<summary>解答例</summary>

```css
.card-container {
  display: grid;
  /* モバイル: 1列 */
  grid-template-columns: 1fr;
  gap: 16px;
  padding: 16px;
}

.card {
  background-color: white;
  padding: 20px;
  border: 1px solid #ddd;
  border-radius: 8px;
}

/* タブレット: 2列 */
@media (min-width: 768px) {
  .card-container {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* デスクトップ: 4列 */
@media (min-width: 1024px) {
  .card-container {
    grid-template-columns: repeat(4, 1fr);
  }
}
```
</details>

### 問題3：レスポンシブなフォントサイズ

以下のHTMLで、`clamp()` を使って画面幅に応じて文字サイズが滑らかに変化する見出しを作ってください。

```html
<h1 class="fluid-heading">レスポンシブな見出し</h1>
<p class="fluid-text">
  この段落のテキストも画面幅に合わせてサイズが変わります。
  小さい画面では読みやすく、大きい画面では存在感のある表示になります。
</p>
```

**要件：**
- `.fluid-heading`：文字サイズが `24px`〜`56px` の間で画面幅に応じて変化、色 `#2c3e50`、行の高さ `1.2`
- `.fluid-text`：文字サイズが `14px`〜`20px` の間で変化、色 `#555`、行の高さ `1.8`、最大幅 `800px`

<details>
<summary>ヒント</summary>

- `clamp(最小値, 推奨値, 最大値)` を使う
- 推奨値には `vw`（ビューポート幅）を使う
- 例：`clamp(24px, 5vw, 56px)` は最小24px、最大56px、その間は画面幅の5%
</details>

<details>
<summary>解答例</summary>

```css
.fluid-heading {
  /* clamp(最小, 推奨, 最大) */
  /* 画面幅に応じて24px〜56pxの間で滑らかに変化 */
  font-size: clamp(24px, 5vw, 56px);
  color: #2c3e50;
  line-height: 1.2;
}

.fluid-text {
  /* 14px〜20pxの間で変化 */
  font-size: clamp(14px, 1.5vw + 0.5rem, 20px);
  color: #555;
  line-height: 1.8;
  max-width: 800px;
}
```
</details>

### 問題4：ナビゲーションの切り替え

以下のHTMLで、モバイルでは縦並び、デスクトップでは横並びのナビゲーションを作ってください。

```html
<nav class="main-nav">
  <div class="nav-brand">MyApp</div>
  <ul class="nav-menu">
    <li class="nav-item"><a href="#" class="nav-link">ホーム</a></li>
    <li class="nav-item"><a href="#" class="nav-link">サービス</a></li>
    <li class="nav-item"><a href="#" class="nav-link">料金</a></li>
    <li class="nav-item"><a href="#" class="nav-link">ブログ</a></li>
    <li class="nav-item"><a href="#" class="nav-link">お問い合わせ</a></li>
  </ul>
</nav>
```

**モバイル（〜767px）：**
```
┌──────────────────┐
│ MyApp            │
├──────────────────┤
│ ホーム            │
│ サービス          │
│ 料金             │
│ ブログ            │
│ お問い合わせ       │
└──────────────────┘
```

**デスクトップ（768px〜）：**
```
┌──────────────────────────────────────────────┐
│ MyApp       ホーム  サービス  料金  ブログ  お問い合わせ │
└──────────────────────────────────────────────┘
```

**要件：**
- `.main-nav`：背景色 `#2c3e50`、padding `16px`
  - モバイル：通常のブロック表示
  - デスクトップ：Flexbox、`space-between`、垂直中央揃え
- `.nav-brand`：文字色 白、文字サイズ `20px`、太字、margin-bottom `12px`（モバイル時）
- `.nav-menu`：リストスタイルなし、margin `0`、padding `0`
  - モバイル：縦並び
  - デスクトップ：Flexbox、間隔 `20px`
- `.nav-link`：文字色 `#ecf0f1`、下線なし、display `block`、padding `8px 0`

<details>
<summary>ヒント</summary>

- モバイルファーストで、デフォルトは縦並びレイアウト
- `@media (min-width: 768px)` でFlexboxに切り替え
- `.nav-brand` の `margin-bottom` はデスクトップで `0` に
</details>

<details>
<summary>解答例</summary>

```css
/* ===== モバイル（デフォルト） ===== */
.main-nav {
  background-color: #2c3e50;
  padding: 16px;
}

.nav-brand {
  color: white;
  font-size: 20px;
  font-weight: bold;
  margin-bottom: 12px;
}

.nav-menu {
  list-style: none;
  margin: 0;
  padding: 0;
}

.nav-link {
  color: #ecf0f1;
  text-decoration: none;
  display: block;
  padding: 8px 0;
}

/* ===== デスクトップ（768px以上） ===== */
@media (min-width: 768px) {
  .main-nav {
    /* 横並びに切り替え */
    display: flex;
    justify-content: space-between;
    align-items: center;
  }

  .nav-brand {
    /* 下余白を解除 */
    margin-bottom: 0;
  }

  .nav-menu {
    /* リンクを横並びに */
    display: flex;
    gap: 20px;
  }

  .nav-link {
    padding: 0;
  }
}
```
</details>

### 問題5：レスポンシブな2カラム→1カラムレイアウト

以下のHTMLで、デスクトップでは2カラム、モバイルでは1カラムになるレイアウトを作ってください。

```html
<div class="article-layout">
  <main class="article-content">
    <h1>記事タイトル</h1>
    <p>ここに記事の本文が入ります。レスポンシブデザインにより、画面サイズに応じてレイアウトが変化します。</p>
    <p>モバイルではサイドバーが下に回り込み、デスクトップでは横に並びます。</p>
  </main>
  <aside class="article-sidebar">
    <h3>関連記事</h3>
    <ul>
      <li>記事1へのリンク</li>
      <li>記事2へのリンク</li>
      <li>記事3へのリンク</li>
    </ul>
  </aside>
</div>
```

**要件：**
- `.article-layout`：間隔 `24px`、padding `16px`、最大幅 `1200px`、中央揃え
  - モバイル：1カラム
  - デスクトップ（`768px` 以上）：Gridで `1fr 300px` の2カラム
- `.article-content`：背景色 白、padding `24px`、角丸 `8px`
- `.article-sidebar`：背景色 `#f8f9fa`、padding `20px`、角丸 `8px`、枠線 `1px solid #e0e0e0`

<details>
<summary>ヒント</summary>

- モバイルファーストでデフォルトは1カラム（Gridを使わない or `grid-template-columns: 1fr`）
- `@media (min-width: 768px)` で `grid-template-columns: 1fr 300px` に変更
</details>

<details>
<summary>解答例</summary>

```css
.article-layout {
  /* モバイル: 1カラム（デフォルト） */
  display: grid;
  grid-template-columns: 1fr;
  gap: 24px;
  padding: 16px;
  max-width: 1200px;
  margin: 0 auto;
}

.article-content {
  background-color: white;
  padding: 24px;
  border-radius: 8px;
}

.article-sidebar {
  background-color: #f8f9fa;
  padding: 20px;
  border-radius: 8px;
  border: 1px solid #e0e0e0;
}

/* デスクトップ: 2カラム */
@media (min-width: 768px) {
  .article-layout {
    /* メインが可変幅、サイドバーが300px固定 */
    grid-template-columns: 1fr 300px;
  }
}
```
</details>

### 問題6：レスポンシブな画像

以下のHTMLで、画面幅に応じて画像が適切にリサイズされるスタイルを書いてください。

```html
<div class="image-gallery">
  <figure class="gallery-item">
    <div class="image-placeholder" style="background-color: #3498db;">画像1</div>
    <figcaption>美しい風景写真</figcaption>
  </figure>
  <figure class="gallery-item">
    <div class="image-placeholder" style="background-color: #e74c3c;">画像2</div>
    <figcaption>街のスナップ</figcaption>
  </figure>
  <figure class="gallery-item">
    <div class="image-placeholder" style="background-color: #2ecc71;">画像3</div>
    <figcaption>自然の美しさ</figcaption>
  </figure>
</div>
```

**要件：**
- `.image-gallery`：Gridレイアウト、間隔 `16px`、padding `16px`
  - モバイル：1列
  - タブレット（`600px` 以上）：2列
  - デスクトップ（`900px` 以上）：3列
- `.gallery-item`：margin `0`
- `.image-placeholder`：幅 `100%`、`aspect-ratio: 16 / 9`（16:9の比率を維持）、文字色 白、Flexboxで上下左右中央揃え、角丸 `8px`
- `figcaption`：padding `8px 0`、文字サイズ `14px`、色 `#666`

<details>
<summary>ヒント</summary>

- `aspect-ratio: 16 / 9` で画像の縦横比を固定できる
- `width: 100%` と組み合わせると、幅に応じて高さが自動調整される
</details>

<details>
<summary>解答例</summary>

```css
.image-gallery {
  display: grid;
  /* モバイル: 1列 */
  grid-template-columns: 1fr;
  gap: 16px;
  padding: 16px;
}

.gallery-item {
  margin: 0;
}

.image-placeholder {
  width: 100%;
  /* 16:9の比率を維持 */
  aspect-ratio: 16 / 9;
  color: white;
  /* Flexboxで中央揃え */
  display: flex;
  justify-content: center;
  align-items: center;
  border-radius: 8px;
  font-size: 18px;
}

figcaption {
  padding: 8px 0;
  font-size: 14px;
  color: #666;
}

/* タブレット: 2列 */
@media (min-width: 600px) {
  .image-gallery {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* デスクトップ: 3列 */
@media (min-width: 900px) {
  .image-gallery {
    grid-template-columns: repeat(3, 1fr);
  }
}
```
</details>

### 問題7：コンテナクエリを使う

以下のHTMLで、親コンテナの幅に応じてカードのレイアウトが変化するスタイルを書いてください。

```html
<div class="wrapper">
  <div class="card-wrapper">
    <div class="adaptive-card">
      <div class="card-image"></div>
      <div class="card-body">
        <h3 class="card-title">コンテナクエリ対応カード</h3>
        <p class="card-desc">このカードは親コンテナの幅に応じてレイアウトが変わります。</p>
      </div>
    </div>
  </div>
</div>
```

**動作イメージ：**
```
コンテナが狭い時（縦並び）:     コンテナが広い時（横並び）:
┌─────────────┐              ┌──────┬──────────────────┐
│   [画像]     │              │      │ タイトル          │
├─────────────┤              │[画像]│ 説明テキスト       │
│ タイトル      │              │      │                  │
│ 説明テキスト  │              └──────┴──────────────────┘
└─────────────┘
```

**要件：**
- `.card-wrapper`：コンテナクエリのコンテナとして設定
- `.adaptive-card`：背景色 白、角丸 `12px`、overflow hidden、枠線 `1px solid #e0e0e0`
  - デフォルト（コンテナが狭い時）：縦並び
  - コンテナ幅 `500px` 以上：Flexboxで横並び
- `.card-image`：背景色 `#bdc3c7`、最小高さ `150px`
  - コンテナ幅 `500px` 以上：幅 `200px` 固定
- `.card-body`：padding `16px`
- `.card-title`：文字サイズ `18px`、色 `#2c3e50`、margin `0 0 8px 0`
- `.card-desc`：文字サイズ `14px`、色 `#666`、行の高さ `1.6`、margin `0`

<details>
<summary>ヒント</summary>

- `container-type: inline-size` で親要素をコンテナクエリの対象に
- `@container (min-width: 500px)` でコンテナの幅に応じたスタイルを適用
- メディアクエリが画面全体の幅を見るのに対し、コンテナクエリは親要素の幅を見る
</details>

<details>
<summary>解答例</summary>

```css
.card-wrapper {
  /* この要素をコンテナクエリのコンテナとして登録 */
  container-type: inline-size;
}

.adaptive-card {
  background-color: white;
  border-radius: 12px;
  overflow: hidden;
  border: 1px solid #e0e0e0;
}

.card-image {
  background-color: #bdc3c7;
  min-height: 150px;
}

.card-body {
  padding: 16px;
}

.card-title {
  font-size: 18px;
  color: #2c3e50;
  margin: 0 0 8px 0;
}

.card-desc {
  font-size: 14px;
  color: #666;
  line-height: 1.6;
  margin: 0;
}

/* コンテナの幅が500px以上の時、横並びに */
@container (min-width: 500px) {
  .adaptive-card {
    display: flex;
  }

  .card-image {
    /* 横並び時は幅を固定 */
    width: 200px;
    flex-shrink: 0;
    min-height: auto;
  }
}
```
</details>
