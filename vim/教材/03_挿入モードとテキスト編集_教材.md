# 第3章：挿入モードとテキスト編集

## この章のゴール

- 様々な方法で挿入モードに入れる（i, I, a, A, o, O）
- 挿入モード中の便利な操作を使える
- テキストオブジェクトを理解して活用できる
- オペレータとモーションの組み合わせを理解する（Vim の文法）

---

## 1. 挿入モードに入る方法

### 6つの入り方を使い分ける

挿入モードに入る方法は複数あり、状況に応じて使い分けることで効率が大きく変わります。

```
  i  ← カーソルの前（左）に挿入  (insert)
  a  ← カーソルの後（右）に挿入  (append)
  I  ← 行の最初の非空白文字の前に挿入
  A  ← 行末に挿入
  o  ← カーソルの下に新しい行を作って挿入  (open)
  O  ← カーソルの上に新しい行を作って挿入
```

```
  例: "Hello World" というテキストで、カーソルが W の上にある場合

  Hello World
        ^  ← カーソル位置

  i → Hello |World    ← W の前にカーソル（| は入力位置）
  a → Hello W|orld    ← W の後にカーソル
  I → |Hello World    ← 行頭にカーソル
  A → Hello World|    ← 行末にカーソル
  o → Hello World
      |               ← 下に新しい行が追加される
  O → |
      Hello World     ← 上に新しい行が追加される
```

### よくある使い分け

| やりたいこと | 使うコマンド |
|-------------|-------------|
| 行末にセミコロンを追加 | `A;` |
| 行頭にコメント記号を追加 | `I# ` |
| 下に新しい行を追加して入力 | `o` |
| 上に空行を追加 | `O` + `ESC` |
| カーソル位置の前に文字を挿入 | `i` |
| カーソル位置の後に文字を追加 | `a` |

### ポイントまとめ

- `i`/`a` は文字単位、`I`/`A` は行単位、`o`/`O` は新しい行です
- 行末に追記したいときは `A`、行頭に追記したいときは `I` が便利です
- `o`/`O` は新しい行を追加するとき最も効率的です

## 2. 挿入モード中の操作

挿入モード中でも、いくつかの便利なキー操作が使えます。

### 挿入モード中のショートカット

```
  Ctrl+h    ← 1文字削除（Backspaceと同じ）
  Ctrl+w    ← 1単語削除
  Ctrl+u    ← 行頭まで削除
  Ctrl+[    ← ESCと同じ（ノーマルモードに戻る）
  Ctrl+o    ← 一時的にノーマルモードに入る（1コマンドだけ実行）
  Ctrl+t    ← インデントを追加
  Ctrl+d    ← インデントを削除
  Ctrl+n    ← 次の補完候補
  Ctrl+p    ← 前の補完候補
```

### Ctrl+o の活用

`Ctrl+o` は挿入モードから一時的にノーマルモードに入り、1つだけコマンドを実行した後、自動的に挿入モードに戻ります。

```
  挿入モード中に Ctrl+o を押すと:
  -- (insert) --    ← 画面左下の表示が変わる

  例: 挿入モードのまま画面をスクロールしたい場合
  Ctrl+o → Ctrl+d   ← 半画面スクロールして挿入モードに戻る

  例: 挿入モードのままファイルを保存したい場合
  Ctrl+o → :w Enter  ← 保存して挿入モードに戻る
```

### ポイントまとめ

- `Ctrl+w` で単語削除、`Ctrl+u` で行頭まで削除できます
- `Ctrl+[` は `ESC` の代替として人気があります
- `Ctrl+o` で一時的にノーマルモードのコマンドを実行できます

## 3. テキストオブジェクト

テキストオブジェクト（text object）は、Vim の最も強力な機能の1つです。「単語」「文」「段落」「括弧の中身」などの意味のあるテキストの塊を操作対象にできます。

### テキストオブジェクトの基本

テキストオブジェクトには2種類あります。

```
  i{対象}  ← inner（内側）：区切り文字を含まない
  a{対象}  ← a/around（周り）：区切り文字を含む
```

### 主なテキストオブジェクト

```
  iw / aw   ← 単語 (word)
  iW / aW   ← WORD（空白区切り）
  is / as   ← 文 (sentence)
  ip / ap   ← 段落 (paragraph)
  i" / a"   ← ダブルクォートの中 / ダブルクォートを含む
  i' / a'   ← シングルクォートの中 / シングルクォートを含む
  i` / a`   ← バッククォートの中 / バッククォートを含む
  i( / a(   ← 括弧の中 / 括弧を含む（ib / ab も同じ）
  i[ / a[   ← 角括弧の中 / 角括弧を含む
  i{ / a{   ← 波括弧の中 / 波括弧を含む（iB / aB も同じ）
  i< / a<   ← 山括弧の中 / 山括弧を含む
  it / at   ← HTMLタグの中 / タグを含む (tag)
```

### テキストオブジェクトの図解

```
  例: "Hello World" というテキストがある場合

  "Hello World"
   ^^^^^^^^^^^    ← i" の範囲（ダブルクォートの中身）
  ^^^^^^^^^^^^^   ← a" の範囲（ダブルクォートを含む）
```

```
  例: function(arg1, arg2) の場合

  function(arg1, arg2)
           ^^^^^^^^^^    ← i( の範囲（括弧の中身）
          ^^^^^^^^^^^^   ← a( の範囲（括弧を含む）
```

```
  例: Hello World の World にカーソルがある場合

  Hello World
        ^^^^^   ← iw の範囲（単語のみ）
       ^^^^^^   ← aw の範囲（前の空白を含む）
```

### ポイントまとめ

- テキストオブジェクトは `i`（inner）と `a`（around）の2種類があります
- `i` は区切り文字を含まず、`a` は含みます
- カーソルが対象の中にあれば、どこにいても範囲全体を操作できます

## 4. オペレータとモーション — Vim の文法

Vim のコマンドは **オペレータ（動詞）+ モーション/テキストオブジェクト（名詞）** の組み合わせで構成されます。これが「Vim の文法」です。

### Vim の文法

```
  ┌──────────────┐   ┌──────────────────────┐
  │ オペレータ    │ + │ モーション/            │
  │（何をする）   │   │ テキストオブジェクト    │
  │              │   │（どこに対して）         │
  └──────────────┘   └──────────────────────┘

  d（削除）  + w（次の単語まで） = dw  （次の単語まで削除）
  c（変更）  + i"（"の中身）     = ci" （"の中身を変更）
  y（コピー）+ ap（段落全体）    = yap （段落全体をコピー）
```

### 主なオペレータ

```
  d  ← 削除 (delete)
  c  ← 変更 (change) = 削除 + 挿入モード
  y  ← ヤンク/コピー (yank)
  >  ← インデント追加
  <  ← インデント削除
  =  ← 自動インデント
  gU ← 大文字に変換 (uppercase)
  gu ← 小文字に変換 (lowercase)
```

### 組み合わせの例

```
  d + w   = dw    次の単語まで削除
  d + $   = d$    行末まで削除
  d + iw  = diw   カーソル上の単語を削除
  d + i(  = di(   括弧の中身を削除
  d + ap  = dap   段落全体を削除

  c + w   = cw    次の単語まで変更
  c + i"  = ci"   ダブルクォートの中身を変更
  c + it  = cit   HTMLタグの中身を変更

  y + y   = yy    行全体をヤンク
  y + iw  = yiw   カーソル上の単語をヤンク
  y + a[  = ya[   角括弧を含めてヤンク

  > + >   = >>    現在行をインデント
  gU + iw = gUiw  カーソル上の単語を大文字に変換
```

### ポイントまとめ

- Vim のコマンドは「オペレータ + モーション/テキストオブジェクト」の文法に従います
- 新しいオペレータを覚えれば、既知のモーションすべてと組み合わせられます
- 新しいモーションを覚えれば、既知のオペレータすべてと組み合わせられます
- この文法こそが Vim の効率性の源です

## 5. 実践：効率的なテキスト編集

### シナリオ1：変数名の変更

```python
# 元のコード
name = "Alice"
print("Hello, " + name)
```

```
  1. name の n にカーソルを置く
  2. ciw と入力（単語を変更）
  3. username と入力して ESC
  → name が username に変わる
```

### シナリオ2：文字列の中身を変更

```javascript
// 元のコード
const message = "Hello, World!";
```

```
  1. カーソルを "Hello, World!" のどこかに置く
  2. ci" と入力（ダブルクォートの中身を変更）
  3. Goodbye! と入力して ESC
  → const message = "Goodbye!"; に変わる
```

### シナリオ3：関数の引数を削除

```python
# 元のコード
result = calculate(100, 200, 300)
```

```
  1. カーソルを括弧の中のどこかに置く
  2. di( と入力（括弧の中身を削除）
  → result = calculate() に変わる
```

### シナリオ4：HTMLタグの中身を書き換え

```html
<!-- 元のコード -->
<h1>古いタイトル</h1>
```

```
  1. カーソルをタグ内のどこかに置く
  2. cit と入力（タグの中身を変更）
  3. 新しいタイトル と入力して ESC
  → <h1>新しいタイトル</h1> に変わる
```

### シナリオ5：ドットコマンドとの組み合わせ

```python
# 複数の print を console.log に変更したい
print("message 1")
print("message 2")
print("message 3")
```

```
  1. 最初の print にカーソルを合わせる
  2. ciwconsole.log と入力して ESC
  3. 次の print に移動（n で検索 or w で移動）
  4. . を押す（ドットコマンドで同じ変更を繰り返し）
  5. 次の print に移動して . を押す
```

### ポイントまとめ

- `ciw` は変数名の変更に最適です
- `ci"`, `ci(`, `cit` でクォートや括弧、タグの中身を効率的に変更できます
- テキストオブジェクト + ドットコマンドの組み合わせで反復作業を大幅に効率化できます
