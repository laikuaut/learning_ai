# 第7章 演習：関数とカスタムコマンド

---

## 演習の目標

この演習では、以下のスキルを確認します。

- Vim script の変数とスコープ（`g:` `s:` `l:` `a:`）を理解している
- `if` / `elseif` / `endif` による条件分岐を書ける
- `has()` や `exists()` で機能や変数の存在を判定できる
- `function!` で関数を定義できる
- `command!` でカスタムコマンドを作成できる
- 実用的な関数（トグル、削除、整形など）を実装できる

---

## 基本問題

### 問題1：変数のスコープ

以下の変数宣言について、それぞれのスコープ（有効範囲）を答えてください。

```vim
let g:my_var = 1
let s:my_var = 2
let l:my_var = 3
let b:my_var = 4
let w:my_var = 5
```

また、Vim のコマンドラインで以下を実行して、グローバル変数を設定・確認する方法を示してください。

**期待される出力例：**

```
:let g:author_name = "Taro"
:echo g:author_name
→ Taro
```

<details>
<summary>ヒント</summary>

- `g:` はグローバル、`s:` はスクリプトローカル、`l:` は関数ローカル、`b:` はバッファローカル、`w:` はウィンドウローカルです
- `:let` で変数を設定し、`:echo` で表示できます

</details>

<details>
<summary>解答例</summary>

```vim
" g: → グローバル変数（どこからでもアクセス可能）
let g:my_var = 1

" s: → スクリプトローカル変数（定義されたスクリプトファイル内のみ）
let s:my_var = 2

" l: → 関数ローカル変数（定義された関数内のみ）
let l:my_var = 3

" b: → バッファローカル変数（現在のバッファ内のみ）
let b:my_var = 4

" w: → ウィンドウローカル変数（現在のウィンドウ内のみ）
let w:my_var = 5

" グローバル変数の設定と確認
:let g:author_name = "Taro"
:echo g:author_name
" → Taro
```

</details>

### 問題2：条件分岐の基本

以下の要件を満たす Vim script を vimrc に記述してください。

1. クリップボード機能（`clipboard`）がある場合は `set clipboard=unnamedplus` を設定する
2. True Color（`termguicolors`）に対応している場合は `set termguicolors` を設定する
3. Neovim の場合は `set inccommand=split` を設定する

**期待される動作：**

```
環境に応じて適切な設定が自動的に適用される
```

<details>
<summary>ヒント</summary>

- `has('clipboard')` でクリップボード機能の有無を判定できます
- `has('termguicolors')` で True Color 対応を判定できます
- `has('nvim')` で Neovim かどうかを判定できます

</details>

<details>
<summary>解答例</summary>

```vim
" クリップボード機能の判定
if has('clipboard')
  set clipboard=unnamedplus
endif

" True Color 対応の判定
if has('termguicolors')
  set termguicolors
endif

" Neovim の判定
if has('nvim')
  set inccommand=split
endif
```

</details>

### 問題3：シンプルな関数の定義

以下の要件を満たす関数 `ShowFileInfo()` を定義してください。

1. 現在のファイル名を表示する
2. ファイルタイプを表示する
3. 総行数を表示する

**期待される出力例：**

```
ファイル名: main.py
ファイルタイプ: python
総行数: 42
```

<details>
<summary>ヒント</summary>

- `expand('%:t')` でファイル名を取得できます
- `&filetype` でファイルタイプを取得できます
- `line('$')` で総行数を取得できます
- `echo` または `echomsg` で出力します

</details>

<details>
<summary>解答例</summary>

```vim
" ファイル情報を表示する関数
function! ShowFileInfo()
  " expand('%:t') で現在のファイル名（パスなし）を取得
  echo "ファイル名: " . expand('%:t')
  " &filetype でファイルタイプオプションの値を取得
  echo "ファイルタイプ: " . &filetype
  " line('$') でファイルの最終行番号（= 総行数）を取得
  echo "総行数: " . line('$')
endfunction

" コマンドラインから :call ShowFileInfo() で実行
" または以下のマッピングを設定
nnoremap <Leader>i :call ShowFileInfo()<CR>
```

</details>

### 問題4：トグル関数

行番号の表示を切り替えるトグル関数 `ToggleLineNumber()` を作成してください。

要件：
1. 呼び出すたびに3つのモードを切り替える
   - モード1: 絶対行番号（`number` のみ）
   - モード2: ハイブリッド（`number` + `relativenumber`）
   - モード3: 非表示（両方 OFF）
2. 現在のモードをメッセージで表示する
3. `<Leader>n` にマッピングする

**期待される動作：**

```
1回目: 行番号: 絶対行番号
2回目: 行番号: ハイブリッド（絶対 + 相対）
3回目: 行番号: 非表示
4回目: 行番号: 絶対行番号（ループ）
```

<details>
<summary>ヒント</summary>

- グローバル変数 `g:line_number_mode` で現在のモードを追跡します
- `exists()` で変数の存在を確認し、初期化します
- `%` 演算子でループを実現できます

</details>

<details>
<summary>解答例</summary>

```vim
" 行番号表示を3モードで切り替えるトグル関数
function! ToggleLineNumber()
  " 変数が存在しない場合は初期化
  if !exists('g:line_number_mode')
    let g:line_number_mode = 0
  endif

  " 次のモードへ
  let g:line_number_mode = (g:line_number_mode + 1) % 3

  if g:line_number_mode == 0
    " モード1: 絶対行番号
    set number
    set norelativenumber
    echo "行番号: 絶対行番号"
  elseif g:line_number_mode == 1
    " モード2: ハイブリッド
    set number
    set relativenumber
    echo "行番号: ハイブリッド（絶対 + 相対）"
  else
    " モード3: 非表示
    set nonumber
    set norelativenumber
    echo "行番号: 非表示"
  endif
endfunction

" Leader + n でトグル
nnoremap <Leader>n :call ToggleLineNumber()<CR>
```

</details>

### 問題5：末尾空白削除関数

以下の要件を満たす末尾空白削除関数 `StripTrailingWhitespace()` を作成してください。

要件：
1. ファイル全体の行末空白を削除する
2. 削除前のカーソル位置を保存し、削除後に復元する
3. 削除した行数をメッセージで表示する
4. カスタムコマンド `:StripWhitespace` で呼び出せるようにする

**期待される動作：**

```
:StripWhitespace
→ 末尾空白を 5 行から削除しました
```

<details>
<summary>ヒント</summary>

- `winsaveview()` / `winrestview()` でカーソル位置を保存・復元できます
- `search()` で末尾空白がある行数をカウントできます
- `:%s/\s\+$//e` で末尾空白を削除できます
- `command!` でカスタムコマンドを定義します

</details>

<details>
<summary>解答例</summary>

```vim
" 末尾空白を削除する関数
function! StripTrailingWhitespace()
  " カーソル位置を保存
  let l:save = winsaveview()

  " 末尾空白がある行数をカウント
  let l:count = 0
  let l:line = 1
  while l:line <= line('$')
    if getline(l:line) =~ '\s\+$'
      let l:count += 1
    endif
    let l:line += 1
  endwhile

  " 末尾空白を削除（e フラグでエラーを無視）
  :%s/\s\+$//e

  " カーソル位置を復元
  call winrestview(l:save)

  " 結果を表示
  if l:count > 0
    echo "末尾空白を " . l:count . " 行から削除しました"
  else
    echo "末尾空白はありませんでした"
  endif
endfunction

" カスタムコマンドとして登録
command! StripWhitespace call StripTrailingWhitespace()
```

</details>

### 問題6：カスタムコマンドの作成

以下の要件を満たすカスタムコマンドを作成してください。

1. `:Today` → 現在の日付（YYYY-MM-DD 形式）をカーソル位置に挿入する
2. `:Config` → vimrc を新しいタブで開く
3. `:Reload` → vimrc を再読み込みする

**期待される動作：**

```
:Today   → カーソル位置に 2026-03-29 が挿入される
:Config  → 新しいタブで ~/.vimrc が開かれる
:Reload  → vimrc が再読み込みされ「vimrc を再読み込みしました」と表示
```

<details>
<summary>ヒント</summary>

- `strftime('%Y-%m-%d')` で現在日付を取得できます
- `put =` で文字列をカーソル位置に挿入できます
- `$MYVIMRC` で vimrc のパスを参照できます
- `source` で vimrc を再読み込みできます

</details>

<details>
<summary>解答例</summary>

```vim
" 今日の日付をカーソル位置に挿入
" -nargs=0 は引数なし（デフォルトなので省略可能）
command! Today put =strftime('%Y-%m-%d')

" vimrc を新しいタブで開く
" $MYVIMRC は Vim が自動設定する vimrc のパス
command! Config tabedit $MYVIMRC

" vimrc を再読み込み
" execute を使って source コマンドを実行し、メッセージを表示
command! Reload source $MYVIMRC | echo "vimrc を再読み込みしました"
```

</details>

### 問題7：汎用トグルコマンドの作成

任意のブール型オプションを切り替えられる汎用的なトグルコマンド `:Toggle` を作成してください。

要件：
1. `:Toggle wrap` で `wrap` の ON/OFF を切り替える
2. `:Toggle number` で `number` の ON/OFF を切り替える
3. 切り替え後に、オプション名と新しい状態を表示する
4. 存在しないオプションが指定された場合はエラーメッセージを表示する

**期待される動作：**

```
:Toggle wrap
→ wrap: ON

:Toggle wrap
→ wrap: OFF

:Toggle number
→ number: ON

:Toggle foobar
→ エラー: 'foobar' は有効なオプションではありません
```

<details>
<summary>ヒント</summary>

- `command! -nargs=1` で1つの引数を受け取るコマンドを定義します
- `exists('+オプション名')` でオプションの存在を確認できます
- `execute 'set ' . option . '!'` でトグルできます
- `eval('&' . option)` でオプションの値を取得できます

</details>

<details>
<summary>解答例</summary>

```vim
" 汎用トグル関数
function! ToggleOption(option)
  " オプションが存在するか確認
  " exists('+option') でオプションの存在をチェック
  if !exists('+' . a:option)
    echohl ErrorMsg
    echo "エラー: '" . a:option . "' は有効なオプションではありません"
    echohl None
    return
  endif

  " オプションをトグル（! で反転）
  execute 'set ' . a:option . '!'

  " 新しい値を取得して表示
  " eval('&option') でオプションの現在値を取得
  let l:value = eval('&' . a:option)
  if l:value
    echo a:option . ": ON"
  else
    echo a:option . ": OFF"
  endif
endfunction

" カスタムコマンドとして登録
" -nargs=1: 引数を1つ受け取る
" -complete=option: オプション名の補完を有効化
command! -nargs=1 -complete=option Toggle call ToggleOption(<q-args>)
```

</details>

### 問題8：ステータスライン用関数

以下の情報をステータスラインに表示するための関数群を作成し、`statusline` に組み込んでください。

要件：
1. `GetMode()` → 現在のモードを日本語で返す（ノーマル、挿入、ビジュアル等）
2. `GetGitBranch()` → Git ブランチ名を返す（Git リポジトリでない場合は空文字）
3. これらの関数を使ったステータスラインを設定する

**期待される表示：**

```
[ノーマル] main | src/main.py [+] [python] utf-8 42:10 65%
```

<details>
<summary>ヒント</summary>

- `mode()` で現在のモード文字を取得できます（`n`, `i`, `v` 等）
- `system()` で外部コマンドを実行できます
- `trim()` で末尾の改行を削除できます
- `statusline` に `%{関数名()}` で関数の戻り値を埋め込めます

</details>

<details>
<summary>解答例</summary>

```vim
" 現在のモードを日本語で返す関数
function! GetMode()
  " mode() はモードを表す1文字を返す
  let l:mode_map = {
    \ 'n':  'ノーマル',
    \ 'i':  '挿入',
    \ 'v':  'ビジュアル',
    \ 'V':  'Vライン',
    \ "\<C-v>": 'Vブロック',
    \ 'R':  '置換',
    \ 'c':  'コマンド',
    \ 't':  'ターミナル',
    \ }
  let l:current = mode()
  " get() でキーが存在しない場合のデフォルト値を指定
  return get(l:mode_map, l:current, l:current)
endfunction

" Git ブランチ名を返す関数
function! GetGitBranch()
  " system() で git コマンドを実行
  let l:branch = trim(system('git -C ' . expand('%:p:h') . ' rev-parse --abbrev-ref HEAD 2>/dev/null'))
  " v:shell_error が 0 でなければ Git リポジトリではない
  if v:shell_error != 0
    return ''
  endif
  return l:branch
endfunction

" ステータスラインの設定
set laststatus=2
set statusline=
set statusline+=[%{GetMode()}]           " モード表示
set statusline+=\ %{GetGitBranch()}      " Git ブランチ
set statusline+=\ \|\ %f                 " ファイルパス
set statusline+=\ %m%r                   " 変更・読取専用フラグ
set statusline+=%=                       " 右寄せ開始
set statusline+=\ %y                     " ファイルタイプ
set statusline+=\ %{&encoding}           " エンコーディング
set statusline+=\ %l:%c                  " 行:列
set statusline+=\ %p%%                   " 位置（パーセント）
```

</details>
