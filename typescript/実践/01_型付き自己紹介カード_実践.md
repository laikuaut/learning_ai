# 実践課題01：型付き自己紹介カード ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（TypeScriptの基本）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 基本的な型注釈（type annotation）を使って、安全にデータを扱う体験をする

---

## 完成イメージ

```
===== 自己紹介カード =====

名前: 田中太郎
年齢: 28歳
職業: エンジニア
趣味: プログラミング, 読書, ランニング
自己紹介: TypeScriptを勉強中です！

----- プロフィール更新 -----
名前を更新: 田中太郎 → 田中次郎
年齢を1歳増やしました: 28 → 29
趣味を追加: 映画鑑賞
更新後の趣味: プログラミング, 読書, ランニング, 映画鑑賞

===== 複数人のカード =====
[1] 田中次郎 (29歳) - エンジニア
[2] 佐藤花子 (24歳) - デザイナー
[3] 鈴木一郎 (35歳) - マネージャー
```

---

## 課題の要件

1. 自己紹介カードの情報を**型注釈付き**の変数で管理する
2. 名前（`string`）、年齢（`number`）、職業（`string`）、趣味（`string[]`）、自己紹介文（`string`）を保持する
3. プロフィールを表示する関数を作る
4. 名前の変更、年齢の加算、趣味の追加ができるようにする
5. 複数人のカードを配列で管理し、一覧表示する

---

## ステップガイド

<details>
<summary>ステップ1：型注釈付きの変数を用意する</summary>

まず基本的な型注釈を使って変数を宣言しましょう。

```typescript
// 基本型の変数
const name: string = "田中太郎";
const age: number = 28;
const job: string = "エンジニア";
const hobbies: string[] = ["プログラミング", "読書", "ランニング"];
const introduction: string = "TypeScriptを勉強中です！";
```

`let` と `const` の使い分けを意識してください。変更する可能性がある変数は `let` を使います。

</details>

<details>
<summary>ステップ2：表示用の関数を作る</summary>

関数の引数と戻り値にも型注釈を付けます。

```typescript
function displayCard(
  name: string,
  age: number,
  job: string,
  hobbies: string[],
  introduction: string
): void {
  console.log("===== 自己紹介カード =====");
  console.log(`\n名前: ${name}`);
  console.log(`年齢: ${age}歳`);
  // ... 残りの表示
}
```

`void` は「戻り値がない」ことを表す型です。

</details>

<details>
<summary>ステップ3：更新用の関数を作る</summary>

趣味を追加する関数を例にします。

```typescript
function addHobby(hobbies: string[], newHobby: string): string[] {
  return [...hobbies, newHobby];
}
```

スプレッド構文（`...`）を使って元の配列を変更せずに新しい配列を返す書き方を覚えましょう。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```typescript
// 型付き自己紹介カード
// 学べる内容：基本的な型注釈、関数の型、配列型
// 実行方法：npx ts-node 01_profile_card.ts

// --- 基本データ ---
let userName: string = "田中太郎";
let userAge: number = 28;
const userJob: string = "エンジニア";
let userHobbies: string[] = ["プログラミング", "読書", "ランニング"];
const userIntro: string = "TypeScriptを勉強中です！";

// --- 表示関数 ---
function displayCard(
  name: string,
  age: number,
  job: string,
  hobbies: string[],
  introduction: string
): void {
  console.log("===== 自己紹介カード =====");
  console.log(`\n名前: ${name}`);
  console.log(`年齢: ${age}歳`);
  console.log(`職業: ${job}`);
  console.log(`趣味: ${hobbies.join(", ")}`);
  console.log(`自己紹介: ${introduction}`);
}

// --- 更新関数 ---
function updateName(oldName: string, newName: string): string {
  console.log(`名前を更新: ${oldName} → ${newName}`);
  return newName;
}

function incrementAge(currentAge: number): number {
  const newAge: number = currentAge + 1;
  console.log(`年齢を1歳増やしました: ${currentAge} → ${newAge}`);
  return newAge;
}

function addHobby(hobbies: string[], newHobby: string): string[] {
  const updated: string[] = [...hobbies, newHobby];
  console.log(`趣味を追加: ${newHobby}`);
  console.log(`更新後の趣味: ${updated.join(", ")}`);
  return updated;
}

// --- 一覧表示関数 ---
function displayCardList(
  cards: { name: string; age: number; job: string }[]
): void {
  console.log("\n===== 複数人のカード =====");
  for (let i: number = 0; i < cards.length; i++) {
    const card = cards[i];
    console.log(`[${i + 1}] ${card.name} (${card.age}歳) - ${card.job}`);
  }
}

// --- メイン処理 ---
displayCard(userName, userAge, userJob, userHobbies, userIntro);

console.log("\n----- プロフィール更新 -----");
userName = updateName(userName, "田中次郎");
userAge = incrementAge(userAge);
userHobbies = addHobby(userHobbies, "映画鑑賞");

// 複数人のカードを配列で管理
const cards: { name: string; age: number; job: string }[] = [
  { name: userName, age: userAge, job: userJob },
  { name: "佐藤花子", age: 24, job: "デザイナー" },
  { name: "鈴木一郎", age: 35, job: "マネージャー" },
];

displayCardList(cards);
```

</details>

<details>
<summary>解答例（改良版 ─ 型エイリアスを先取り）</summary>

```typescript
// 型付き自己紹介カード（改良版）
// 学べる内容：型エイリアスの先取り、readonly、関数型
// 実行方法：npx ts-node 01_profile_card_v2.ts

// --- 型定義（第2章の先取り） ---
type ProfileCard = {
  readonly name: string;
  readonly age: number;
  readonly job: string;
  readonly hobbies: readonly string[];
  readonly introduction: string;
};

// --- カードを生成する関数 ---
function createCard(
  name: string,
  age: number,
  job: string,
  hobbies: string[],
  introduction: string
): ProfileCard {
  return { name, age, job, hobbies, introduction };
}

// --- 表示関数 ---
function displayCard(card: ProfileCard): void {
  console.log("===== 自己紹介カード =====");
  console.log(`\n名前: ${card.name}`);
  console.log(`年齢: ${card.age}歳`);
  console.log(`職業: ${card.job}`);
  console.log(`趣味: ${card.hobbies.join(", ")}`);
  console.log(`自己紹介: ${card.introduction}`);
}

// --- 更新関数（イミュータブルに新しいカードを返す） ---
function withName(card: ProfileCard, newName: string): ProfileCard {
  console.log(`名前を更新: ${card.name} → ${newName}`);
  return { ...card, name: newName };
}

function withIncrementedAge(card: ProfileCard): ProfileCard {
  const newAge: number = card.age + 1;
  console.log(`年齢を1歳増やしました: ${card.age} → ${newAge}`);
  return { ...card, age: newAge };
}

function withHobby(card: ProfileCard, newHobby: string): ProfileCard {
  const updatedHobbies: string[] = [...card.hobbies, newHobby];
  console.log(`趣味を追加: ${newHobby}`);
  console.log(`更新後の趣味: ${updatedHobbies.join(", ")}`);
  return { ...card, hobbies: updatedHobbies };
}

// --- 一覧表示 ---
function displayCardList(cards: ProfileCard[]): void {
  console.log("\n===== 複数人のカード =====");
  cards.forEach((card: ProfileCard, index: number): void => {
    console.log(`[${index + 1}] ${card.name} (${card.age}歳) - ${card.job}`);
  });
}

// --- メイン処理 ---
let myCard: ProfileCard = createCard(
  "田中太郎",
  28,
  "エンジニア",
  ["プログラミング", "読書", "ランニング"],
  "TypeScriptを勉強中です！"
);

displayCard(myCard);

console.log("\n----- プロフィール更新 -----");
myCard = withName(myCard, "田中次郎");
myCard = withIncrementedAge(myCard);
myCard = withHobby(myCard, "映画鑑賞");

const allCards: ProfileCard[] = [
  myCard,
  createCard("佐藤花子", 24, "デザイナー", ["イラスト"], "UIが好きです"),
  createCard("鈴木一郎", 35, "マネージャー", ["ゴルフ"], "チームリードです"),
];

displayCardList(allCards);
```

**初心者向けとの違い:**
- `type` エイリアスでデータ構造を一箇所にまとめている
- `readonly` でうっかり変更を防いでいる
- 更新時に元のオブジェクトを書き換えず、新しいオブジェクトを返す（イミュータブル設計）
- `forEach` を使って配列を走査している

</details>
