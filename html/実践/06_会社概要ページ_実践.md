# 実践課題06：会社概要ページ ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第1章〜第6章（HTMLの基本構造、テキストとリンク、画像、リストとテーブル、フォーム、セマンティックHTML）
> **課題の種類**: ミニプロジェクト
> **学習目標**: セマンティックHTML（header / nav / main / section / article / aside / footer）を正しく使い分けて、構造的に正しい複数セクションのページを構築する

---

## 完成イメージ

ブラウザで開くと以下のような会社概要ページが表示されます。

```
[ナビゲーションバー]
  ホーム | 会社概要 | サービス | お問い合わせ

====================================
    株式会社テックサンプル 会社概要
====================================

■ 会社情報
  社名：株式会社テックサンプル
  設立：2020年4月1日
  代表：田中一郎
  ...

  ・モバイルアプリ開発
  ・ITコンサルティング

■ アクセス
  [地図のプレースホルダー]
  〒100-0001 東京都千代田区...

■ 沿革
  | 年月     | 出来事              |
  |----------|-------------------|
  | 2020年4月 | 会社設立           |
  | 2021年1月 | 社員数10名突破     |
  | ...      | ...               |

■ お問い合わせ
  [簡易お問い合わせフォーム]

─────────────────────────────
  © 2026 株式会社テックサンプル
```

---

## 課題の要件

1. 正しいHTML基本構造を持つこと
2. `<header>` に会社名とナビゲーション（`<nav>`）を含むこと
3. `<main>` にメインコンテンツを配置すること
4. `<section>` で各セクション（会社情報、事業内容、アクセス、沿革）を分けること
5. `<dl>`（定義リスト）で会社の基本情報を表示すること
6. `<ul>` で事業内容をリスト表示すること
7. `<table>` で沿革を年表形式で表示すること
8. `<aside>` でサイドバー的な補足情報を配置すること
9. `<footer>` に著作権表記とリンクを配置すること
10. 簡易お問い合わせフォームを含むこと
11. ページ内リンク（`id` と `#` アンカー）でナビゲーションからジャンプできること

## ステップガイド

<details>
<summary>ステップ1：ページ全体の骨格を作る</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>会社概要 | 株式会社テックサンプル</title>
</head>
<body>
  <header>
    <!-- ロゴ・ナビゲーション -->
  </header>
  <main>
    <!-- メインコンテンツ -->
  </main>
  <footer>
    <!-- フッター -->
  </footer>
</body>
</html>
```

</details>

<details>
<summary>ステップ2：ナビゲーションとページ内リンクを作る</summary>

```html
<header>
  <h1>株式会社テックサンプル</h1>
  <nav>
    <ul>
      <li><a href="#about">会社情報</a></li>
      <li><a href="#service">事業内容</a></li>
      <li><a href="#access">アクセス</a></li>
      <li><a href="#history">沿革</a></li>
      <li><a href="#contact">お問い合わせ</a></li>
    </ul>
  </nav>
</header>
```

リンク先のセクションに `id` を設定します。

```html
<section id="about">
  <h2>会社情報</h2>
  <!-- ... -->
</section>
```

</details>

<details>
<summary>ステップ3：定義リスト（dl）で会社情報を表示する</summary>

```html
<dl>
  <dt>社名</dt>
  <dd>株式会社テックサンプル</dd>
  <dt>設立</dt>
  <dd>2020年4月1日</dd>
</dl>
```

`<dl>` は「用語（dt）と説明（dd）のペア」を表します。会社情報のような「項目名：値」の形式に最適です。

</details>

**初心者向けとの違い:**
- `<time datetime="">` で日付をマシンリーダブルに → 検索エンジンが日付を正確に理解
- `<address>` で住所・連絡先を明示 → セマンティクスが正確
- `aria-label` でナビゲーションの用途を明示 → スクリーンリーダー対応
- `<aside>` でサイドバーを配置 → 補足情報の構造が明確
- `<dl>` にCSSグリッドを適用 → ラベルと値が整列
- `tel:` リンクでスマートフォンから直接発信可能
- `autocomplete` 属性でブラウザの自動入力を支援
