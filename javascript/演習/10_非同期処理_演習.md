## 基本問題

### 問題1：setTimeout の基本

`setTimeout` を使って、以下の順番でメッセージを表示してください。

**期待される出力（時間差あり）：**
```
1. 開始
2. 次の処理（すぐに表示）
3. 1秒後のメッセージ
4. 2秒後のメッセージ
5. 3秒後のメッセージ
```

<details>
<summary>ヒント</summary>

`console.log` の順番に注意してください。`setTimeout` は非同期なので、後に書いたコードが先に実行されます。

</details>

<details>
<summary>解答例</summary>

```javascript
console.log("1. 開始");

setTimeout(() => console.log("3. 1秒後のメッセージ"), 1000);
setTimeout(() => console.log("4. 2秒後のメッセージ"), 2000);
setTimeout(() => console.log("5. 3秒後のメッセージ"), 3000);

// これはsetTimeoutより先に実行される
console.log("2. 次の処理（すぐに表示）");
```

</details>

### 問題2：Promiseの作成

指定した秒数後に値を返すPromise関数 `delay` を作成してください。

```javascript
// delay(秒数, 値) → 指定秒数後に値をresolveするPromiseを返す
```

**期待される出力：**
```
待機中...
1秒後: Hello
2秒後: World
```

<details>
<summary>解答例</summary>

```javascript
// 指定秒数後に値を返すPromise
function delay(seconds, value) {
  return new Promise((resolve) => {
    setTimeout(() => resolve(value), seconds * 1000);
  });
}

console.log("待機中...");

delay(1, "Hello").then(result => {
  console.log(`1秒後: ${result}`);
});

delay(2, "World").then(result => {
  console.log(`2秒後: ${result}`);
});
```

</details>

### 問題3：async/await の基本

問題2の `delay` 関数を使って、`async/await` で順番にメッセージを表示する関数を作成してください。

**期待される出力（時間差あり）：**
```
処理開始
→ ステップ1完了（1秒後）
→ ステップ2完了（さらに1秒後）
→ ステップ3完了（さらに1秒後）
全ステップ完了！（約3秒後）
```

<details>
<summary>解答例</summary>

```javascript
function delay(seconds, value) {
  return new Promise((resolve) => {
    setTimeout(() => resolve(value), seconds * 1000);
  });
}

async function runSteps() {
  console.log("処理開始");

  await delay(1);
  console.log("→ ステップ1完了（1秒後）");

  await delay(1);
  console.log("→ ステップ2完了（さらに1秒後）");

  await delay(1);
  console.log("→ ステップ3完了（さらに1秒後）");

  console.log("全ステップ完了！（約3秒後）");
}

runSteps();
```

</details>

### 問題4：fetch APIでデータ取得

JSONPlaceholder API からユーザーデータを取得し、名前とメールの一覧を表示してください。

URL: `https://jsonplaceholder.typicode.com/users`

**期待される出力（一部）：**
```
ユーザー一覧（10人）:
1. Leanne Graham (Sincere@april.biz)
2. Ervin Howell (Shanna@melissa.tv)
...
```

<details>
<summary>解答例</summary>

```javascript
async function fetchUsers() {
  try {
    const response = await fetch("https://jsonplaceholder.typicode.com/users");

    if (!response.ok) {
      throw new Error(`HTTP Error: ${response.status}`);
    }

    const users = await response.json();

    console.log(`ユーザー一覧（${users.length}人）:`);
    users.forEach((user, index) => {
      console.log(`${index + 1}. ${user.name} (${user.email})`);
    });
  } catch (error) {
    console.error("データの取得に失敗:", error.message);
  }
}

fetchUsers();
```

</details>

## 応用問題

### 問題5：Promise.all で並行処理

3つの異なるAPIエンドポイントから同時にデータを取得し、それぞれの件数を表示してください。

- ユーザー: `https://jsonplaceholder.typicode.com/users`
- 投稿: `https://jsonplaceholder.typicode.com/posts`
- コメント: `https://jsonplaceholder.typicode.com/comments`

**期待される出力：**
```
データ取得中...
--- 取得結果 ---
ユーザー: 10件
投稿: 100件
コメント: 500件
合計取得時間: XXXms
```

<details>
<summary>ヒント</summary>

`Promise.all` で3つの `fetch` を同時に実行します。`performance.now()` で実行時間を計測できます。

</details>

<details>
<summary>解答例</summary>

```javascript
async function fetchAllData() {
  console.log("データ取得中...");
  const startTime = performance.now();

  try {
    // 3つのリクエストを同時に実行
    const [users, posts, comments] = await Promise.all([
      fetch("https://jsonplaceholder.typicode.com/users").then(r => r.json()),
      fetch("https://jsonplaceholder.typicode.com/posts").then(r => r.json()),
      fetch("https://jsonplaceholder.typicode.com/comments").then(r => r.json())
    ]);

    const elapsed = Math.round(performance.now() - startTime);

    console.log("--- 取得結果 ---");
    console.log(`ユーザー: ${users.length}件`);
    console.log(`投稿: ${posts.length}件`);
    console.log(`コメント: ${comments.length}件`);
    console.log(`合計取得時間: ${elapsed}ms`);
  } catch (error) {
    console.error("取得に失敗:", error.message);
  }
}

fetchAllData();
```

</details>

### 問題6：リトライ処理

失敗時に自動的に再試行する `fetchWithRetry` 関数を作成してください。最大3回までリトライし、各試行の間に1秒の待機時間を入れてください。

**期待される出力（成功時）：**
```
試行 1/3...
データ取得成功
```

**期待される出力（失敗時）：**
```
試行 1/3... 失敗
試行 2/3... 失敗
試行 3/3... 失敗
全ての試行が失敗しました
```

<details>
<summary>解答例</summary>

```javascript
async function fetchWithRetry(url, maxRetries = 3) {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      console.log(`試行 ${attempt}/${maxRetries}...`);
      const response = await fetch(url);

      if (!response.ok) {
        throw new Error(`HTTP ${response.status}`);
      }

      const data = await response.json();
      console.log("データ取得成功");
      return data;
    } catch (error) {
      console.log(`試行 ${attempt}/${maxRetries}... 失敗: ${error.message}`);

      if (attempt === maxRetries) {
        console.log("全ての試行が失敗しました");
        throw error;
      }

      // リトライ前に待機
      await new Promise(resolve => setTimeout(resolve, 1000));
    }
  }
}

// テスト（正常なURL）
fetchWithRetry("https://jsonplaceholder.typicode.com/users/1")
  .then(data => console.log("結果:", data.name))
  .catch(err => console.error("最終エラー:", err.message));
```

</details>

## チャレンジ問題

### 問題7：非同期処理のタイムアウト

任意のPromiseにタイムアウトを設定する `withTimeout` 関数を作成してください。指定時間内に完了しなければタイムアウトエラーをスローします。

```javascript
// withTimeout(promise, timeoutMs) → タイムアウト付きのPromiseを返す
```

**期待される出力：**
```
--- テスト1: 正常完了 ---
結果: 正常に完了
--- テスト2: タイムアウト ---
エラー: タイムアウト（1000ms）
```

<details>
<summary>ヒント</summary>

`Promise.race` を使って、元のPromiseとタイムアウト用のPromiseを競争させます。

</details>

<details>
<summary>解答例</summary>

```javascript
function withTimeout(promise, timeoutMs) {
  // タイムアウト用のPromise
  const timeout = new Promise((_, reject) => {
    setTimeout(() => {
      reject(new Error(`タイムアウト（${timeoutMs}ms）`));
    }, timeoutMs);
  });

  // 元のPromiseとタイムアウトを競争させる
  return Promise.race([promise, timeout]);
}

// テスト用の遅い処理
function slowTask(ms) {
  return new Promise(resolve => {
    setTimeout(() => resolve("正常に完了"), ms);
  });
}

// テスト1: 正常完了（500ms の処理に 2000ms のタイムアウト）
async function test1() {
  console.log("--- テスト1: 正常完了 ---");
  try {
    const result = await withTimeout(slowTask(500), 2000);
    console.log(`結果: ${result}`);
  } catch (error) {
    console.log(`エラー: ${error.message}`);
  }
}

// テスト2: タイムアウト（3000ms の処理に 1000ms のタイムアウト）
async function test2() {
  console.log("--- テスト2: タイムアウト ---");
  try {
    const result = await withTimeout(slowTask(3000), 1000);
    console.log(`結果: ${result}`);
  } catch (error) {
    console.log(`エラー: ${error.message}`);
  }
}

test1().then(() => test2());
```

</details>
