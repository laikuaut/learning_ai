### 問題1：挨拶関数を作ろう

名前と時間帯を引数に取り、適切な挨拶を返す関数 `greet` を作成してください。時間帯のデフォルト値は `"朝"` とします。

- `"朝"`: "おはようございます"
- `"昼"`: "こんにちは"
- `"夜"`: "こんばんは"

**期待される出力：**
```
おはようございます、太郎さん！
こんにちは、花子さん！
こんばんは、次郎さん！
```

<details>
<summary>解答例</summary>

```javascript
function greet(name, timeOfDay = "朝") {
  // 時間帯に応じた挨拶を選択
  let greeting;
  if (timeOfDay === "朝") {
    greeting = "おはようございます";
  } else if (timeOfDay === "昼") {
    greeting = "こんにちは";
  } else {
    greeting = "こんばんは";
  }
  return `${greeting}、${name}さん！`;
}

console.log(greet("太郎"));           // デフォルトで「朝」
console.log(greet("花子", "昼"));
console.log(greet("次郎", "夜"));
```

</details>

### 問題2：アロー関数で書き換え

以下の関数をそれぞれアロー関数に書き換えてください。

```javascript
function double(n) {
  return n * 2;
}

function isEven(n) {
  return n % 2 === 0;
}

function getFullName(firstName, lastName) {
  return `${lastName} ${firstName}`;
}
```

**期待される出力：**
```
10
true
山田 太郎
```

<details>
<summary>解答例</summary>

```javascript
// アロー関数で書き換え
const double = n => n * 2;
const isEven = n => n % 2 === 0;
const getFullName = (firstName, lastName) => `${lastName} ${firstName}`;

console.log(double(5));
console.log(isEven(4));
console.log(getFullName("太郎", "山田"));
```

</details>

### 問題3：残余引数を使った合計関数

任意の数の引数を受け取って合計を返す関数 `sum` を作成してください。残余引数（`...`）を使ってください。

**期待される出力：**
```
3
15
0
```

<details>
<summary>解答例</summary>

```javascript
// 残余引数で任意の数の引数を受け取る
const sum = (...numbers) => {
  return numbers.reduce((total, n) => total + n, 0);
};

console.log(sum(1, 2));              // 3
console.log(sum(1, 2, 3, 4, 5));     // 15
console.log(sum());                   // 0
```

</details>

### 問題4：コールバック関数の練習

配列の各要素に対して処理を適用する関数 `processArray` を作成してください。第1引数に配列、第2引数にコールバック関数を受け取ります。

```javascript
const numbers = [1, 2, 3, 4, 5];
```

この関数を使って以下を実現してください：
1. 各要素を2倍にした配列
2. 各要素を文字列に変換した配列

**期待される出力：**
```
2倍: [2, 4, 6, 8, 10]
文字列: ["No.1", "No.2", "No.3", "No.4", "No.5"]
```

<details>
<summary>解答例</summary>

```javascript
// コールバック関数を受け取って各要素を変換する関数
function processArray(arr, callback) {
  const result = [];
  for (const item of arr) {
    result.push(callback(item));
  }
  return result;
}

const numbers = [1, 2, 3, 4, 5];

// アロー関数をコールバックとして渡す
const doubled = processArray(numbers, n => n * 2);
console.log(`2倍: [${doubled.join(", ")}]`);

const strings = processArray(numbers, n => `"No.${n}"`);
console.log(`文字列: [${strings.join(", ")}]`);
```

</details>

### 問題5：クロージャでカウンターを作ろう

クロージャを使って、以下の機能を持つカウンターを作成してください。

- `increment()`: カウントを1増やして現在値を返す
- `decrement()`: カウントを1減らして現在値を返す
- `reset()`: カウントを0に戻す
- `getCount()`: 現在のカウント値を返す

**期待される出力：**
```
1
2
3
2
カウント: 2
リセット後: 0
```

<details>
<summary>ヒント</summary>

外側の関数でプライベートな変数 `count` を宣言し、内側の関数（メソッド）からその変数を操作するオブジェクトを返します。

</details>

<details>
<summary>解答例</summary>

```javascript
function createCounter() {
  // プライベート変数（外部から直接アクセスできない）
  let count = 0;

  return {
    increment() { return ++count; },
    decrement() { return --count; },
    reset() { count = 0; return count; },
    getCount() { return count; }
  };
}

const counter = createCounter();
console.log(counter.increment()); // 1
console.log(counter.increment()); // 2
console.log(counter.increment()); // 3
console.log(counter.decrement()); // 2
console.log(`カウント: ${counter.getCount()}`); // 2
console.log(`リセット後: ${counter.reset()}`);  // 0
```

</details>

### 問題6：高階関数 — バリデーター生成

特定のバリデーションルールを持つ関数を生成する高階関数を作成してください。

1. `createMinLengthValidator(min)`: 文字列が最低 `min` 文字以上かチェックする関数を返す
2. `createRangeValidator(min, max)`: 数値が `min` 以上 `max` 以下かチェックする関数を返す

**期待される出力：**
```
"ab" は3文字以上か: false
"abc" は3文字以上か: true
"abcdef" は3文字以上か: true
5 は1〜10の範囲内か: true
15 は1〜10の範囲内か: false
```

<details>
<summary>解答例</summary>

```javascript
// 最小文字数バリデーターを生成する高階関数
function createMinLengthValidator(min) {
  return (str) => str.length >= min;
}

// 範囲バリデーターを生成する高階関数
function createRangeValidator(min, max) {
  return (num) => num >= min && num <= max;
}

// バリデーターの生成
const isMinLength3 = createMinLengthValidator(3);
const isInRange1to10 = createRangeValidator(1, 10);

// テスト
console.log(`"ab" は3文字以上か: ${isMinLength3("ab")}`);
console.log(`"abc" は3文字以上か: ${isMinLength3("abc")}`);
console.log(`"abcdef" は3文字以上か: ${isMinLength3("abcdef")}`);
console.log(`5 は1〜10の範囲内か: ${isInRange1to10(5)}`);
console.log(`15 は1〜10の範囲内か: ${isInRange1to10(15)}`);
```

</details>

### 問題7：再帰関数 — 配列のフラット化

ネストされた配列を再帰的にフラットにする関数 `flatten` を作成してください。`Array.isArray()` を使って要素が配列かどうかを判定します。

```javascript
const nested = [1, [2, 3], [4, [5, 6]], [[7, 8], 9]];
```

**期待される出力：**
```
[1, 2, 3, 4, 5, 6, 7, 8, 9]
```

<details>
<summary>ヒント</summary>

- 配列の各要素をチェックし、要素が配列なら再帰的に `flatten` を呼び出します。
- 配列でなければ結果にそのまま追加します。

</details>

<details>
<summary>解答例</summary>

```javascript
function flatten(arr) {
  const result = [];
  for (const item of arr) {
    if (Array.isArray(item)) {
      // 配列なら再帰的にフラット化してスプレッドで追加
      result.push(...flatten(item));
    } else {
      result.push(item);
    }
  }
  return result;
}

const nested = [1, [2, 3], [4, [5, 6]], [[7, 8], 9]];
console.log(flatten(nested));
// [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

</details>

### 問題8：メモ化関数を実装しよう

計算結果をキャッシュする `memoize` 関数を作成してください。同じ引数で呼ばれた場合は、計算を行わずにキャッシュから結果を返します。

フィボナッチ数列の計算でその効果を確認しましょう。

**期待される出力例：**
```
計算中: fib(10)
結果: 55
キャッシュ使用: fib(10)
結果: 55
fib(40)の結果: 102334155
```

<details>
<summary>解答例</summary>

```javascript
function memoize(fn) {
  const cache = new Map();
  return function(...args) {
    const key = JSON.stringify(args);
    if (cache.has(key)) {
      console.log(`キャッシュ使用: ${fn.name}(${args})`);
      return cache.get(key);
    }
    console.log(`計算中: ${fn.name}(${args})`);
    const result = fn(...args);
    cache.set(key, result);
    return result;
  };
}

// 再帰的なフィボナッチ（メモ化なしだと遅い）
function fib(n) {
  if (n <= 1) return n;
  return memoizedFib(n - 1) + memoizedFib(n - 2);
}

// メモ化を適用
const memoizedFib = memoize(fib);

console.log(`結果: ${memoizedFib(10)}`);
console.log(`結果: ${memoizedFib(10)}`); // キャッシュから取得
console.log(`fib(40)の結果: ${memoizedFib(40)}`);
```

</details>
