# 実践課題12：ポートフォリオサイト ★5

> **難易度**: ★★★★★（総合）
> **前提知識**: 第1章〜第8章（全章）
> **課題の種類**: ミニプロジェクト
> **学習目標**: CSSの全知識を駆使して、実務レベルの完成度を持つレスポンシブなポートフォリオサイトを1ファイルで構築する

---

ブラウザで開くと、以下のセクションで構成されたシングルページのポートフォリオサイトが表示されます。

---

## 課題の要件

1. **1つのHTMLファイルだけ**で完結すること（外部ライブラリ不使用）
2. **ヒーローセクション**: グラデーション背景、中央寄せ、CTAボタン2つ
3. **Aboutセクション**: 自己紹介文、スキルバー（プログレスバー風のCSS表現）
4. **Worksセクション**: 3列のプロジェクトカード（Grid）。ホバーでオーバーレイ表示
5. **Contactセクション**: お問い合わせフォーム（送信は不要、見た目のみ）
6. **レスポンシブ対応**: モバイル、タブレット、デスクトップの3段階
7. **ダーク/ライトテーマ対応**: カスタムプロパティで管理
8. **アニメーション**: スクロールに合わせたフェードイン風エフェクト（CSS only）
9. **疑似クラス/疑似要素**: `:hover`、`:focus`、`::before`、`::after` を活用
10. **アクセシビリティ**: フォーム要素に `<label>`、ボタンに適切な色コントラスト

## ステップガイド

<details>
<summary>ステップ1：セクション構造とカスタムプロパティを作る</summary>

HTML の骨組みを先に作り、`:root` にカラーパレットを定義します。

```html
<nav class="nav">...</nav>
<section id="hero" class="hero">...</section>
<section id="about" class="about">...</section>
<section id="works" class="works">...</section>
<section id="contact" class="contact">...</section>
<footer class="footer">...</footer>
```

```css
:root {
  --color-bg: #f8fafc;
  --color-surface: #ffffff;
  --color-text: #1e293b;
  --color-primary: #3b82f6;
  --color-gradient-start: #667eea;
  --color-gradient-end: #764ba2;
  /* ... */
}
```

</details>

<details>
<summary>ステップ2：ヒーローセクションを作る</summary>

全画面高さのグラデーション背景に中央寄せのコンテンツを配置します。

```css
.hero {
  min-height: 100vh;
  background: linear-gradient(135deg, var(--color-gradient-start), var(--color-gradient-end));
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  text-align: center;
  color: #ffffff;
}
```

</details>

<details>
<summary>ステップ3：スキルバーをCSSで作る</summary>

`width` をパーセンテージで指定した内側の `<div>` で表現します。

```css
.skill-bar {
  height: 8px;
  background-color: var(--color-border);
  border-radius: 4px;
  overflow: hidden;
}

.skill-fill {
  height: 100%;
  background-color: var(--color-primary);
  border-radius: 4px;
  transition: width 1s ease;
}
```

```html
<div class="skill-bar">
  <div class="skill-fill" style="width: 90%;"></div>
</div>
```

</details>

<details>
<summary>ステップ4：プロジェクトカードのホバーオーバーレイ</summary>

```css
.project-card::after {
  content: "詳細を見る";
  position: absolute;
  inset: 0;
  background: rgba(0, 0, 0, 0.6);
  display: flex;
  align-items: center;
  justify-content: center;
  color: #fff;
  opacity: 0;
  transition: opacity 0.3s;
}

.project-card:hover::after {
  opacity: 1;
}
```

</details>

### 1. スクロール連動フェードイン（CSS animation + IntersectionObserver）

```css
/* フェードインアニメーション */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(24px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animate {
  opacity: 0;
}

.animate.visible {
  animation: fadeInUp 0.6s ease forwards;
}

/* 子要素に遅延を付ける */
.animate.visible:nth-child(2) { animation-delay: 0.1s; }
.animate.visible:nth-child(3) { animation-delay: 0.2s; }
```

```javascript
// 最小限のIntersectionObserver
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
    }
  });
}, { threshold: 0.1 });

document.querySelectorAll('.animate').forEach(el => observer.observe(el));
```
