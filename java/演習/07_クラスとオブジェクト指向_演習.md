### 問題1：Studentクラス

以下の仕様で`Student`クラスを作成してください。

- フィールド：`name`（名前）、`score`（点数、0〜100）
- コンストラクタ：名前と点数を受け取る
- メソッド：
  - `getGrade()`：点数に応じた評価を返す（90以上→A、80以上→B、70以上→C、60以上→D、それ未満→F）
  - `toString()`：「名前: スコア点 (評価)」の形式で返す

**期待される出力例：**
```
田中太郎: 85点 (B)
佐藤花子: 92点 (A)
鈴木一郎: 58点 (F)
```

<details><summary>ヒント</summary>

- `getGrade()`メソッドではif-else文で点数の範囲を判定します
- `toString()`メソッドをオーバーライドすると、`System.out.println()`に直接オブジェクトを渡せます

</details>

<details><summary>解答例</summary>

```java
public class Student {
    // フィールド（インスタンス変数）
    private String name;
    private int score;

    // コンストラクタ：オブジェクト生成時に名前と点数を設定
    public Student(String name, int score) {
        this.name = name;
        this.score = score;
    }

    // 点数に応じた評価を返すメソッド
    public String getGrade() {
        if (score >= 90) return "A";
        if (score >= 80) return "B";
        if (score >= 70) return "C";
        if (score >= 60) return "D";
        return "F";
    }

    // オブジェクトの文字列表現をオーバーライド
    @Override
    public String toString() {
        return name + ": " + score + "点 (" + getGrade() + ")";
    }

    public static void main(String[] args) {
        // Studentオブジェクトを生成
        Student s1 = new Student("田中太郎", 85);
        Student s2 = new Student("佐藤花子", 92);
        Student s3 = new Student("鈴木一郎", 58);

        // toString()が自動的に呼ばれる
        System.out.println(s1);
        System.out.println(s2);
        System.out.println(s3);
    }
}
```

</details>

### 問題2：Rectangleクラス

以下の仕様で`Rectangle`クラスを作成してください。

- フィールド：`width`（幅）、`height`（高さ）、どちらもdouble型
- コンストラクタ：幅と高さを受け取る
- メソッド：
  - `getArea()`：面積を返す
  - `getPerimeter()`：周囲の長さを返す
  - `isSquare()`：正方形かどうかを返す
  - `toString()`：長方形の情報を文字列で返す

**期待される出力例：**
```
長方形(5.0 x 3.0) - 面積: 15.0, 周囲: 16.0, 正方形: false
長方形(4.0 x 4.0) - 面積: 16.0, 周囲: 16.0, 正方形: true
```

<details><summary>ヒント</summary>

- 面積は`width * height`、周囲の長さは`2 * (width + height)`で求められます
- 正方形の判定は`width == height`で行います

</details>

<details><summary>解答例</summary>

```java
public class Rectangle {
    // privateフィールドでカプセル化
    private double width;
    private double height;

    // コンストラクタ
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }

    // 面積を計算するメソッド
    public double getArea() {
        return width * height;
    }

    // 周囲の長さを計算するメソッド
    public double getPerimeter() {
        return 2 * (width + height);
    }

    // 正方形かどうかを判定するメソッド
    public boolean isSquare() {
        return width == height;
    }

    @Override
    public String toString() {
        return "長方形(" + width + " x " + height + ") - 面積: " + getArea()
                + ", 周囲: " + getPerimeter() + ", 正方形: " + isSquare();
    }

    public static void main(String[] args) {
        Rectangle r1 = new Rectangle(5.0, 3.0);
        Rectangle r2 = new Rectangle(4.0, 4.0);

        System.out.println(r1);
        System.out.println(r2);
    }
}
```

</details>

### 問題3：ゲッターとセッター

`Person`クラスを作成し、カプセル化（フィールドをprivateにしてgetter/setterでアクセス）を実践してください。

- フィールド：`name`（String）、`age`（int）
- セッターに検証を入れる：`age`は0〜150の範囲のみ受け付ける
- `introduce()`メソッド：自己紹介を表示

**期待される出力例：**
```
こんにちは、田中太郎です。25歳です。
年齢を30に変更しました。
こんにちは、田中太郎です。30歳です。
エラー：年齢は0〜150の範囲で指定してください。（入力値：-5）
```

<details><summary>ヒント</summary>

- フィールドを`private`にし、`getName()`/`setName()`などのメソッドを公開します
- `setAge()`内でバリデーション（値の検証）を行います

</details>

<details><summary>解答例</summary>

```java
public class Person {
    // privateフィールド：外部から直接アクセスできない
    private String name;
    private int age;

    // コンストラクタ
    public Person(String name, int age) {
        this.name = name;
        setAge(age); // セッター経由で検証を通す
    }

    // ゲッター：値を取得する
    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    // セッター：値を設定する（検証付き）
    public void setName(String name) {
        this.name = name;
    }

    public void setAge(int age) {
        // バリデーション：不正な値を拒否する
        if (age < 0 || age > 150) {
            System.out.println("エラー：年齢は0〜150の範囲で指定してください。（入力値：" + age + "）");
            return;
        }
        this.age = age;
        System.out.println("年齢を" + age + "に変更しました。");
    }

    // 自己紹介メソッド
    public void introduce() {
        System.out.println("こんにちは、" + name + "です。" + age + "歳です。");
    }

    public static void main(String[] args) {
        Person p = new Person("田中太郎", 25);
        p.introduce();

        p.setAge(30);       // 正常な値
        p.introduce();

        p.setAge(-5);       // 不正な値 → エラーメッセージ
    }
}
```

</details>

### 問題4：BankAccountクラス

銀行口座を表す`BankAccount`クラスを作成してください。

- フィールド：`ownerName`（口座名義）、`balance`（残高、double型）、`accountNumber`（口座番号、String型）
- コンストラクタ：口座名義と口座番号を受け取り、残高は0で初期化
- メソッド：
  - `deposit(double amount)`：入金（0以下は拒否）
  - `withdraw(double amount)`：出金（残高不足・0以下は拒否）
  - `getBalance()`：残高を返す
  - `showHistory()`：取引履歴を表示（入金・出金の記録を配列で保持）

**期待される出力例：**
```
口座番号: 001 (田中太郎)
10000円を入金しました。残高: 10000.0円
3000円を出金しました。残高: 7000.0円
5000円を入金しました。残高: 12000.0円
出金失敗：残高不足です。（出金額: 20000.0円, 残高: 12000.0円）

=== 取引履歴 ===
入金: 10000.0円
出金: 3000.0円
入金: 5000.0円
```

<details><summary>ヒント</summary>

- 取引履歴は`ArrayList<String>`で管理すると便利です
- `deposit()`と`withdraw()`内で入力値のチェックを行います

</details>

<details><summary>解答例</summary>

```java
import java.util.ArrayList;
import java.util.List;

public class BankAccount {
    private String ownerName;
    private String accountNumber;
    private double balance;
    // 取引履歴をリストで管理
    private List<String> history;

    public BankAccount(String ownerName, String accountNumber) {
        this.ownerName = ownerName;
        this.accountNumber = accountNumber;
        this.balance = 0;
        this.history = new ArrayList<>();
    }

    // 入金メソッド
    public void deposit(double amount) {
        if (amount <= 0) {
            System.out.println("入金失敗：正の金額を指定してください。");
            return;
        }
        balance += amount;
        history.add("入金: " + amount + "円");
        System.out.println(amount + "円を入金しました。残高: " + balance + "円");
    }

    // 出金メソッド
    public void withdraw(double amount) {
        if (amount <= 0) {
            System.out.println("出金失敗：正の金額を指定してください。");
            return;
        }
        if (amount > balance) {
            System.out.println("出金失敗：残高不足です。（出金額: " + amount + "円, 残高: " + balance + "円）");
            return;
        }
        balance -= amount;
        history.add("出金: " + amount + "円");
        System.out.println(amount + "円を出金しました。残高: " + balance + "円");
    }

    public double getBalance() {
        return balance;
    }

    // 取引履歴を表示するメソッド
    public void showHistory() {
        System.out.println("\n=== 取引履歴 ===");
        if (history.isEmpty()) {
            System.out.println("取引履歴はありません。");
        } else {
            for (String record : history) {
                System.out.println(record);
            }
        }
    }

    public static void main(String[] args) {
        BankAccount account = new BankAccount("田中太郎", "001");
        System.out.println("口座番号: " + account.accountNumber + " (" + account.ownerName + ")");

        account.deposit(10000);
        account.withdraw(3000);
        account.deposit(5000);
        account.withdraw(20000); // 残高不足

        account.showHistory();
    }
}
```

</details>

### 問題5：BookCollectionクラス

書籍管理システムを作成してください。`Book`クラスと`BookCollection`クラスの2つを使います。

- `Book`クラス：`title`（タイトル）、`author`（著者）、`price`（価格）
- `BookCollection`クラス：
  - `addBook(Book book)`：書籍を追加
  - `findByAuthor(String author)`：著者名で検索
  - `showAllBooks()`：全書籍を表示
  - `getTotalPrice()`：全書籍の合計金額を返す

**期待される出力例：**
```
=== 全書籍一覧 ===
[1] 「Java入門」 著者: 田中太郎 価格: 2800円
[2] 「Python実践」 著者: 佐藤花子 価格: 3200円
[3] 「Java応用」 著者: 田中太郎 価格: 3500円
合計金額: 9500円

--- 著者「田中太郎」の書籍 ---
「Java入門」 著者: 田中太郎 価格: 2800円
「Java応用」 著者: 田中太郎 価格: 3500円
```

<details><summary>ヒント</summary>

- `BookCollection`内に`ArrayList<Book>`を持たせてリストとして管理します
- `findByAuthor()`では全書籍をループして著者名が一致するものを探します

</details>

<details><summary>解答例</summary>

```java
import java.util.ArrayList;
import java.util.List;

public class BookCollection {

    // 内部クラスとしてBookを定義
    static class Book {
        private String title;
        private String author;
        private int price;

        public Book(String title, String author, int price) {
            this.title = title;
            this.author = author;
            this.price = price;
        }

        @Override
        public String toString() {
            return "「" + title + "」 著者: " + author + " 価格: " + price + "円";
        }

        public String getAuthor() {
            return author;
        }

        public int getPrice() {
            return price;
        }
    }

    // 書籍リスト
    private List<Book> books;

    public BookCollection() {
        this.books = new ArrayList<>();
    }

    // 書籍を追加するメソッド
    public void addBook(Book book) {
        books.add(book);
    }

    // 著者名で検索するメソッド
    public List<Book> findByAuthor(String author) {
        List<Book> result = new ArrayList<>();
        for (Book book : books) {
            if (book.getAuthor().equals(author)) {
                result.add(book);
            }
        }
        return result;
    }

    // 全書籍を表示するメソッド
    public void showAllBooks() {
        System.out.println("=== 全書籍一覧 ===");
        for (int i = 0; i < books.size(); i++) {
            System.out.println("[" + (i + 1) + "] " + books.get(i));
        }
        System.out.println("合計金額: " + getTotalPrice() + "円");
    }

    // 合計金額を計算するメソッド
    public int getTotalPrice() {
        int total = 0;
        for (Book book : books) {
            total += book.getPrice();
        }
        return total;
    }

    public static void main(String[] args) {
        BookCollection collection = new BookCollection();

        // 書籍を追加
        collection.addBook(new Book("Java入門", "田中太郎", 2800));
        collection.addBook(new Book("Python実践", "佐藤花子", 3200));
        collection.addBook(new Book("Java応用", "田中太郎", 3500));

        // 全書籍表示
        collection.showAllBooks();

        // 著者で検索
        System.out.println("\n--- 著者「田中太郎」の書籍 ---");
        List<Book> found = collection.findByAuthor("田中太郎");
        for (Book book : found) {
            System.out.println(book);
        }
    }
}
```

### 問題6：ショッピングカート

`Product`クラスと`Cart`クラスを作成し、ショッピングカートシステムを実装してください。

- `Product`クラス：`name`（商品名）、`price`（単価）、`quantity`（数量）
- `Cart`クラス：
  - `addProduct(Product product)`：商品をカートに追加（同じ商品名なら数量を加算）
  - `removeProduct(String name)`：商品をカートから削除
  - `getSubtotal()`：小計を計算
  - `getTax()`：消費税（10%）を計算
  - `getTotal()`：合計金額（税込）を計算
  - `showReceipt()`：レシート形式で表示

**期待される出力例：**
```
「りんご」をカートに追加しました。
「パン」をカートに追加しました。
「牛乳」をカートに追加しました。
「りんご」の数量を更新しました。（合計: 5個）

==============================
        レシート
==============================
りんご      x5    ¥1,500
パン        x2      ¥600
牛乳        x1      ¥250
------------------------------
小計:              ¥2,350
消費税(10%):         ¥235
------------------------------
合計:              ¥2,585
==============================
```

<details><summary>ヒント</summary>

- `Cart`内に`ArrayList<Product>`を持たせます
- `addProduct()`では既存の商品名と一致するか検索し、あれば数量を加算します
- `String.format()`を使うと出力のフォーマットを整えやすくなります

</details>

<details><summary>解答例</summary>

```java
import java.util.ArrayList;
import java.util.List;

public class ShoppingCart {

    // 商品クラス
    static class Product {
        private String name;
        private int price;
        private int quantity;

        public Product(String name, int price, int quantity) {
            this.name = name;
            this.price = price;
            this.quantity = quantity;
        }

        public String getName() { return name; }
        public int getPrice() { return price; }
        public int getQuantity() { return quantity; }

        // 数量を加算するメソッド
        public void addQuantity(int amount) {
            this.quantity += amount;
        }

        // この商品の小計を返す
        public int getSubtotal() {
            return price * quantity;
        }
    }

    // カートクラス
    static class Cart {
        private List<Product> products;

        public Cart() {
            this.products = new ArrayList<>();
        }

        // 商品を追加（同名商品があれば数量を加算）
        public void addProduct(Product product) {
            for (Product p : products) {
                if (p.getName().equals(product.getName())) {
                    p.addQuantity(product.getQuantity());
                    System.out.println("「" + product.getName() + "」の数量を更新しました。（合計: " + p.getQuantity() + "個）");
                    return;
                }
            }
            products.add(product);
            System.out.println("「" + product.getName() + "」をカートに追加しました。");
        }

        // 商品を削除
        public void removeProduct(String name) {
            for (int i = 0; i < products.size(); i++) {
                if (products.get(i).getName().equals(name)) {
                    products.remove(i);
                    System.out.println("「" + name + "」をカートから削除しました。");
                    return;
                }
            }
            System.out.println("「" + name + "」はカートにありません。");
        }

        // 小計を計算
        public int getSubtotal() {
            int total = 0;
            for (Product p : products) {
                total += p.getSubtotal();
            }
            return total;
        }

        // 消費税を計算（10%）
        public int getTax() {
            return (int) (getSubtotal() * 0.10);
        }

        // 合計金額（税込）
        public int getTotal() {
            return getSubtotal() + getTax();
        }

        // レシートを表示
        public void showReceipt() {
            System.out.println("\n==============================");
            System.out.println("        レシート");
            System.out.println("==============================");

            for (Product p : products) {
                // String.formatで桁を揃える
                System.out.printf("%-8s x%d %,8d円%n",
                        p.getName(), p.getQuantity(), p.getSubtotal());
            }

            System.out.println("------------------------------");
            System.out.printf("小計:         %,8d円%n", getSubtotal());
            System.out.printf("消費税(10%%):  %,8d円%n", getTax());
            System.out.println("------------------------------");
            System.out.printf("合計:         %,8d円%n", getTotal());
            System.out.println("==============================");
        }
    }

    public static void main(String[] args) {
        Cart cart = new Cart();

        // 商品をカートに追加
        cart.addProduct(new Product("りんご", 300, 3));
        cart.addProduct(new Product("パン", 300, 2));
        cart.addProduct(new Product("牛乳", 250, 1));
        cart.addProduct(new Product("りんご", 300, 2)); // 既存商品 → 数量加算

        // レシートを表示
        cart.showReceipt();
    }
}
```

### 問題7：成績管理システム

`Student`クラスと`Classroom`クラスを作成し、クラス全体の成績管理を行うプログラムを作成してください。

- `Student`クラス：`name`、`scores`（科目ごとの点数、int配列）
- `Classroom`クラス：
  - `addStudent(Student student)`：生徒を追加
  - `getClassAverage()`：クラス全体の平均点
  - `getTopStudent()`：最高平均点の生徒を返す
  - `showReport()`：成績表を表示

**期待される出力例：**
```
=== クラス成績表 ===
名前        国語  数学  英語  平均    評価
----------------------------------------------
田中太郎     85    92    78   85.0    B
佐藤花子     95    88    96   93.0    A
鈴木一郎     72    65    80   72.3    C
----------------------------------------------
クラス平均:                  83.4
最高得点者: 佐藤花子 (93.0点)
```

<details><summary>ヒント</summary>

- `Student`に`getAverage()`メソッドを追加して個人の平均を計算します
- `Classroom`内で全生徒をループして各種統計を計算します
- `String.format()`を使って表を整えます

</details>

<details><summary>解答例</summary>

```java
import java.util.ArrayList;
import java.util.List;

public class ClassroomReport {

    static class Student {
        private String name;
        private int[] scores; // 科目ごとの点数

        public Student(String name, int[] scores) {
            this.name = name;
            this.scores = scores;
        }

        public String getName() { return name; }
        public int[] getScores() { return scores; }

        // 個人の平均点を計算
        public double getAverage() {
            int sum = 0;
            for (int score : scores) {
                sum += score;
            }
            return (double) sum / scores.length;
        }

        // 平均点に応じた評価を返す
        public String getGrade() {
            double avg = getAverage();
            if (avg >= 90) return "A";
            if (avg >= 80) return "B";
            if (avg >= 70) return "C";
            if (avg >= 60) return "D";
            return "F";
        }
    }

    static class Classroom {
        private List<Student> students;
        private String[] subjects; // 科目名

        public Classroom(String[] subjects) {
            this.students = new ArrayList<>();
            this.subjects = subjects;
        }

        public void addStudent(Student student) {
            students.add(student);
        }

        // クラス全体の平均点
        public double getClassAverage() {
            double sum = 0;
            for (Student s : students) {
                sum += s.getAverage();
            }
            return sum / students.size();
        }

        // 最高平均点の生徒を返す
        public Student getTopStudent() {
            Student top = students.get(0);
            for (Student s : students) {
                if (s.getAverage() > top.getAverage()) {
                    top = s;
                }
            }
            return top;
        }

        // 成績表を表示
        public void showReport() {
            System.out.println("=== クラス成績表 ===");

            // ヘッダー行
            System.out.printf("%-10s", "名前");
            for (String subject : subjects) {
                System.out.printf("%6s", subject);
            }
            System.out.printf("%7s%6s%n", "平均", "評価");

            // 区切り線
            System.out.println("----------------------------------------------");

            // 各生徒の行
            for (Student s : students) {
```
