# 第6章：ポインタ - 演習問題

この演習では、C言語のポインタの基本概念、ポインタ経由の値の操作、動的メモリ確保について練習します。

---

## 基本問題

### 問題1：変数のアドレスと値を表示

`int` 型変数と `double` 型変数をそれぞれ宣言し、ポインタを使ってアドレスと値を表示するプログラムを作成してください。

- 変数を宣言し、その変数を指すポインタも宣言する
- ポインタ経由で値を表示する（`*p` で間接参照）
- アドレスは `%p` フォーマット指定子で表示する

**期待される出力例：**
```
int型変数:
  値: 42
  アドレス: 0x7ffd5e8a3b2c
  ポインタ経由の値: 42

double型変数:
  値: 3.140000
  アドレス: 0x7ffd5e8a3b30
  ポインタ経由の値: 3.140000
```

※アドレスの値は実行環境によって異なります。

<details>
<summary>ヒント</summary>

- `int *p = &x;` でポインタを宣言・初期化します
- `*p` で間接参照（ポインタが指す先の値）を取得できます
- アドレスを表示するには `printf("%p", (void *)p)` を使います

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    /* int型変数とそのポインタ */
    int x = 42;
    int *px = &x;

    /* double型変数とそのポインタ */
    double y = 3.14;
    double *py = &y;

    /* int型の情報を表示 */
    printf("int型変数:\n");
    printf("  値: %d\n", x);
    printf("  アドレス: %p\n", (void *)px);
    printf("  ポインタ経由の値: %d\n", *px);

    printf("\n");

    /* double型の情報を表示 */
    printf("double型変数:\n");
    printf("  値: %f\n", y);
    printf("  アドレス: %p\n", (void *)py);
    printf("  ポインタ経由の値: %f\n", *py);

    return 0;
}
```

</details>

### 問題2：swap関数をポインタで実装

2つの `int` 型変数の値を交換する関数 `swap` を、ポインタを使って実装してください。

- `swap` 関数は2つの `int *` 型引数を受け取る
- `main` 関数で交換前・交換後の値を表示して動作を確認する

**期待される出力例：**
```
交換前: a = 10, b = 20
交換後: a = 20, b = 10
```

<details>
<summary>ヒント</summary>

- 値の交換には一時変数（`temp`）が必要です
- `*a` と `*b` を使ってポインタが指す先の値を操作します
- 関数を `swap(&a, &b)` のようにアドレスを渡して呼び出します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

/* プロトタイプ宣言 */
void swap(int *a, int *b);

int main(void) {
    int a = 10;
    int b = 20;

    printf("交換前: a = %d, b = %d\n", a, b);

    /* アドレスを渡して値を交換 */
    swap(&a, &b);

    printf("交換後: a = %d, b = %d\n", a, b);

    return 0;
}

/* ポインタを使って2つの値を交換する関数 */
void swap(int *a, int *b) {
    int temp = *a;  /* aが指す値を一時保存 */
    *a = *b;        /* bが指す値をaの場所にコピー */
    *b = temp;      /* 一時保存した値をbの場所にコピー */
}
```

</details>

### 問題3：配列をポインタで走査

`int` 型配列の全要素を、配列の添字（`arr[i]`）ではなくポインタのインクリメントを使って走査し、各要素を表示してください。

- 配列の先頭を指すポインタを用意する
- ポインタを1つずつ進めながら値を表示する

**期待される出力例：**
```
配列の要素（ポインタで走査）:
arr[0] = 10 (アドレス: 0x7ffd1234ab00)
arr[1] = 20 (アドレス: 0x7ffd1234ab04)
arr[2] = 30 (アドレス: 0x7ffd1234ab08)
arr[3] = 40 (アドレス: 0x7ffd1234ab0c)
arr[4] = 50 (アドレス: 0x7ffd1234ab10)
```

※アドレスの値は実行環境によって異なります。`int` が4バイトの環境ではアドレスが4ずつ増えることを確認してみましょう。

<details>
<summary>ヒント</summary>

- `int *p = arr;` で配列の先頭を指すポインタを作れます
- `p++` でポインタを次の要素に進められます
- ループの終了条件は `p < arr + size` とすると分かりやすいです

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int arr[] = {10, 20, 30, 40, 50};
    int size = sizeof(arr) / sizeof(arr[0]);
    int *p = arr;  /* 配列の先頭を指すポインタ */
    int i = 0;

    printf("配列の要素（ポインタで走査）:\n");

    /* ポインタを進めながら全要素を表示 */
    while (p < arr + size) {
        printf("arr[%d] = %d (アドレス: %p)\n", i, *p, (void *)p);
        p++;  /* ポインタを次の要素に進める */
        i++;
    }

    return 0;
}
```

</details>

## 応用問題

### 問題4：動的配列の確保と使用（malloc/free）

ユーザーから要素数を入力で受け取り、`malloc` でその分の `int` 型配列を動的に確保してください。各要素にインデックスの2乗を格納し、すべて表示した後に `free` で解放します。

- `malloc` の戻り値が `NULL` でないかチェックする
- 使い終わったら必ず `free` で解放する

**期待される出力例：**
```
要素数を入力してください: 5
動的配列の内容:
arr[0] = 0
arr[1] = 1
arr[2] = 4
arr[3] = 9
arr[4] = 16
メモリを解放しました。
```

<details>
<summary>ヒント</summary>

- `int *arr = (int *)malloc(sizeof(int) * n);` で `n` 個分の領域を確保します
- `malloc` は `<stdlib.h>` をインクルードして使います
- `malloc` が失敗すると `NULL` が返るので、必ずチェックしましょう

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <stdlib.h>  /* malloc, free に必要 */

int main(void) {
    int n, i;
    int *arr;

    /* 要素数を入力 */
    printf("要素数を入力してください: ");
    scanf("%d", &n);

    /* 動的にメモリを確保 */
    arr = (int *)malloc(sizeof(int) * n);

    /* メモリ確保の成否をチェック */
    if (arr == NULL) {
        printf("メモリの確保に失敗しました。\n");
        return 1;
    }

    /* 各要素にインデックスの2乗を格納 */
    for (i = 0; i < n; i++) {
        arr[i] = i * i;
    }

    /* 内容を表示 */
    printf("動的配列の内容:\n");
    for (i = 0; i < n; i++) {
        printf("arr[%d] = %d\n", i, arr[i]);
    }

    /* メモリを解放 */
    free(arr);
    printf("メモリを解放しました。\n");

    return 0;
}
```

</details>

### 問題5：文字列をポインタで逆順にする

文字列を受け取り、その内容を逆順に並べ替える関数 `reverse_string` を作成してください。

- 文字列の操作はポインタで行う
- 元の配列の中身を直接書き換える（新しい配列は作らない）
- `strlen` を使ってもよい

**期待される出力例：**
```
元の文字列: Hello, World!
逆順の文字列: !dlroW ,olleH
```

<details>
<summary>ヒント</summary>

- 文字列の先頭と末尾を指す2つのポインタを用意します
- 先頭ポインタを後ろに、末尾ポインタを前に進めながら文字を交換します
- 2つのポインタがすれ違ったら（`left >= right`）終了です

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>  /* strlen に必要 */

/* プロトタイプ宣言 */
void reverse_string(char *str);

int main(void) {
    char text[] = "Hello, World!";

    printf("元の文字列: %s\n", text);

    /* 文字列を逆順にする */
    reverse_string(text);

    printf("逆順の文字列: %s\n", text);

    return 0;
}

/* ポインタを使って文字列を逆順にする関数 */
void reverse_string(char *str) {
    char *left = str;                      /* 先頭を指すポインタ */
    char *right = str + strlen(str) - 1;   /* 末尾を指すポインタ */
    char temp;

    /* 先頭と末尾から中央に向かって文字を交換 */
    while (left < right) {
        temp = *left;
        *left = *right;
        *right = temp;
        left++;   /* 先頭側を後ろに進める */
        right--;  /* 末尾側を前に進める */
    }
}
```

</details>

## チャレンジ問題

### 問題6：動的に配列サイズを拡張するプログラム

ユーザーが数値を入力するたびに動的配列に追加し、`0` が入力されたら終了して全要素を表示するプログラムを作成してください。

- 最初に小さなサイズ（例：2）で `malloc` する
- 配列が満杯になったら `realloc` でサイズを2倍に拡張する
- 拡張するたびにメッセージを表示する
- 終了後にすべての要素と合計を表示する

**期待される出力例：**
```
数値を入力してください（0で終了）:
> 10
> 20
> 30
配列を拡張しました（容量: 2 → 4）
> 40
> 50
配列を拡張しました（容量: 4 → 8）
> 0

入力された値:
[0] 10
[1] 20
[2] 30
[3] 40
[4] 50
合計: 150
要素数: 5
```

<details>
<summary>ヒント</summary>

- 現在の要素数（`size`）と容量（`capacity`）を別の変数で管理します
- `size == capacity` になったら `realloc` で拡張します
- `realloc` が `NULL` を返した場合、元のポインタはまだ有効なので別の変数で受け取りましょう
- `realloc` は `<stdlib.h>` に含まれています

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <stdlib.h>  /* malloc, realloc, free に必要 */

int main(void) {
    int *arr;
    int capacity = 2;   /* 初期容量 */
    int size = 0;        /* 現在の要素数 */
    int value;
    int sum = 0;
    int i;
    int *temp;

    /* 初期サイズでメモリ確保 */
    arr = (int *)malloc(sizeof(int) * capacity);
    if (arr == NULL) {
        printf("メモリの確保に失敗しました。\n");
        return 1;
    }

    printf("数値を入力してください（0で終了）:\n");

    while (1) {
        printf("> ");
        scanf("%d", &value);

        /* 0が入力されたら終了 */
        if (value == 0) {
            break;
        }

        /* 配列が満杯なら容量を2倍に拡張 */
        if (size == capacity) {
            int old_capacity = capacity;
            capacity *= 2;

            /* reallocで拡張（失敗時に元のメモリを失わないよう一時変数で受ける） */
            temp = (int *)realloc(arr, sizeof(int) * capacity);
            if (temp == NULL) {
                printf("メモリの再確保に失敗しました。\n");
                free(arr);
                return 1;
            }
            arr = temp;

            printf("配列を拡張しました（容量: %d → %d）\n", old_capacity, capacity);
        }

        /* 値を追加 */
        arr[size] = value;
        size++;
    }

    /* 結果を表示 */
    printf("\n入力された値:\n");
    for (i = 0; i < size; i++) {
        printf("[%d] %d\n", i, arr[i]);
        sum += arr[i];
    }
    printf("合計: %d\n", sum);
    printf("要素数: %d\n", size);

    /* メモリを解放 */
    free(arr);

    return 0;
}
```

</details>
