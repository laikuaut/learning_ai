## 課題パート1：システム性能計算問題

### 基本問題

**Q1. MIPS（Million Instructions Per Second）計算**

クロック周波数が 3GHz、平均CPI（Cycles Per Instruction）が 5 のプロセッサのMIPS値を求めてください。

```
MIPS = クロック周波数 / (CPI × 10⁶)
```

**Q2. 命令ミックス**

あるプロセッサの命令構成が以下の通りです。平均CPIとMIPS値を求めてください。（クロック: 2GHz）

| 命令タイプ | 出現頻度 | CPI |
|-----------|---------|-----|
| 算術演算 | 40% | 2 |
| メモリアクセス | 30% | 5 |
| 分岐 | 20% | 3 |
| その他 | 10% | 4 |

**Q3. 稼働率（Availability）計算**

以下の構成の全体稼働率を求めてください。各装置の稼働率は R = 0.95 です。

```
(a) 直列構成: [装置A] → [装置B] → [装置C]

(b) 並列構成:  ┌─[装置A]─┐
               │         │
               └─[装置B]─┘

(c) 直並列構成: [装置A] → ┌─[装置B]─┐
                         │         │
                         └─[装置C]─┘
```

**Q4. MTBF と MTTR**

あるシステムのMTBF（Mean Time Between Failures, 平均故障間隔）が 990 時間、MTTR（Mean Time To Repair, 平均修復時間）が 10 時間のとき、稼働率を求めてください。

```
稼働率 = MTBF / (MTBF + MTTR)
```

### 応用問題

**Q5. キャッシュメモリのヒット率**

主記憶のアクセス時間が 50ns、キャッシュメモリのアクセス時間が 5ns、ヒット率が 0.9 のとき、実効アクセス時間を求めてください。

```
実効アクセス時間 = ヒット率 × キャッシュ時間 + (1 - ヒット率) × 主記憶時間
```

**Q6. プロセススケジューリング**

以下のプロセスがこの順番で到着したとき、FCFS（First Come First Served）とSJF（Shortest Job First）それぞれの平均待ち時間を求めてください。

| プロセス | 到着時刻 | 実行時間 |
|---------|---------|---------|
| P1 | 0 | 8 |
| P2 | 0 | 4 |
| P3 | 0 | 2 |
| P4 | 0 | 6 |

**Q7. ページ置換アルゴリズム**

ページ参照列 `1, 2, 3, 4, 1, 2, 5, 1, 2, 3, 4, 5` に対して、ページフレーム数 3 のとき、FIFO とLRU それぞれのページフォルト回数を求めてください。

```python
def fcfs(processes):
    """First Come First Served"""
    wait_times = []
    current_time = 0
    for name, arrival, burst in processes:
        wait = max(0, current_time - arrival)
        wait_times.append(wait)
        current_time = max(current_time, arrival) + burst
    return sum(wait_times) / len(wait_times)
```

## 解答例

<details>
<summary>計算問題の解答</summary>

**Q1.** MIPS = 3000 / 5 = **600 MIPS**

**Q2.** 
- 平均CPI = 0.40×2 + 0.30×5 + 0.20×3 + 0.10×4 = 0.8 + 1.5 + 0.6 + 0.4 = **3.3**
- MIPS = 2000 / 3.3 ≒ **606 MIPS**

**Q3.**
- (a) 直列: 0.95 × 0.95 × 0.95 = **0.8574** (約85.7%)
- (b) 並列: 1 - (1-0.95)² = 1 - 0.0025 = **0.9975** (約99.8%)
- (c) 直並列: 0.95 × [1 - (1-0.95)²] = 0.95 × 0.9975 = **0.9476** (約94.8%)

**Q4.** 稼働率 = 990 / (990 + 10) = 990 / 1000 = **0.99** (99%)

**Q5.** 実効アクセス時間 = 0.9 × 5 + 0.1 × 50 = 4.5 + 5.0 = **9.5ns**

**Q6.**
- FCFS (P1→P2→P3→P4):
  - P1: 待ち0, P2: 待ち8, P3: 待ち12, P4: 待ち14
  - 平均 = (0 + 8 + 12 + 14) / 4 = **8.5**
- SJF (P3→P2→P4→P1):
  - P3: 待ち0, P2: 待ち2, P4: 待ち6, P1: 待ち12
  - 平均 = (0 + 2 + 6 + 12) / 4 = **5.0**

**Q7.**

**FIFO（ページフレーム3）:**
```
参照: 1  2  3  4  1  2  5  1  2  3  4  5
F1:  [1] 1  1 [4] 4  4 [5] 5  5  5 [4] 4
F2:      [2] 2  2 [1] 1  1  1 [2] 2  2 [5]
F3:          [3] 3  3 [2] 2  2  2 [3] 3  3
PF:   *  *  *  *  *  *  *        *  *  *
```
ページフォルト: **10回**

**LRU（ページフレーム3）:**
```
参照: 1  2  3  4  1  2  5  1  2  3  4  5
F1:  [1] 1  1 [4] 4  4 [5] 5  5  5 [4] 4
F2:      [2] 2  2 [1] 1  1  1  1 [3] 3 [5]
F3:          [3] 3  3 [2] 2  2  2  2  2  2
PF:   *  *  *  *  *  *  *        *  *  *
```
ページフォルト: **10回**

※ この参照列では FIFO と LRU の結果が同じになります。

```python
# システム性能計算ツール
# 学べる内容：MIPS、稼働率、スケジューリング
# 実行方法：python 10_system_performance.py

def mips_calculator():
    """MIPS計算"""
    print("\n--- MIPS計算 ---")
    clock = float(input("クロック周波数 (GHz): "))
    cpi = float(input("平均CPI: "))

    mips = (clock * 1000) / cpi
    print(f"\nMIPS値: {mips:.0f} MIPS")
    print(f"1秒間に {mips * 1_000_000:,.0f} 命令を実行できます。")

    # 処理時間の計算
    data_count = int(input("\n処理データ件数 (例: 1000000): "))
    inst_per_data = int(input("1件あたりの命令数 (例: 1000): "))
    total_inst = data_count * inst_per_data
    time_sec = total_inst / (mips * 1_000_000)
    print(f"\n処理時間: {time_sec:.2f}秒")


def availability_calculator():
    """稼働率計算"""
    print("\n--- 稼働率計算 ---")
    print("構成を選んでください:")
    print("  1. 直列構成")
    print("  2. 並列構成")
    print("  3. MTBF/MTTRから計算")

    choice = input("選択 (1-3): ")

    if choice == "1":
        n = int(input("装置の数: "))
        total = 1.0
        for i in range(n):
            r = float(input(f"  装置{i+1}の稼働率: "))
            total = total * r
        print(f"\n全体稼働率: {total:.6f} ({total * 100:.4f}%)")

    elif choice == "2":
        n = int(input("装置の数: "))
        fail_prob = 1.0
        for i in range(n):
            r = float(input(f"  装置{i+1}の稼働率: "))
            fail_prob = fail_prob * (1 - r)
        total = 1 - fail_prob
        print(f"\n全体稼働率: {total:.6f} ({total * 100:.4f}%)")

    elif choice == "3":
        mtbf = float(input("MTBF (時間): "))
        mttr = float(input("MTTR (時間): "))
        avail = mtbf / (mtbf + mttr)
        print(f"\n稼働率: {avail:.6f} ({avail * 100:.4f}%)")


def scheduling_simulator():
    """プロセススケジューリング"""
    print("\n--- プロセススケジューリング ---")

    n = int(input("プロセス数: "))
    processes = []
    for i in range(n):
        name = f"P{i+1}"
        burst = int(input(f"  {name}の実行時間: "))
        processes.append((name, burst))

    # FCFS
    print("\n[FCFS - 先着順]")
    current = 0
    total_wait = 0
    for name, burst in processes:
        print(f"  {name}: 待ち{current}, 実行{current}〜{current + burst}")
        total_wait += current
        current += burst
    avg_fcfs = total_wait / n
    print(f"  平均待ち時間: {avg_fcfs:.2f}")

    # SJF
    print("\n[SJF - 最短ジョブ優先]")
    sorted_procs = sorted(processes, key=lambda x: x[1])
    current = 0
    total_wait = 0
    for name, burst in sorted_procs:
        print(f"  {name}: 待ち{current}, 実行{current}〜{current + burst}")
        total_wait += current
        current += burst
    avg_sjf = total_wait / n
    print(f"  平均待ち時間: {avg_sjf:.2f}")

    print(f"\n→ SJFはFCFSより {avg_fcfs - avg_sjf:.2f} 短い")


# --- メイン ---
print("===== システム性能計算ツール =====")
print("  1. MIPS計算")
print("  2. 稼働率計算")
print("  3. スケジューリングシミュレーション")

choice = input("\n選択 (1-3): ")

if choice == "1":
    mips_calculator()
elif choice == "2":
    availability_calculator()
elif choice == "3":
    scheduling_simulator()
else:
    print("無効な選択です。")
```

```python
# システム性能計算ツール（改良版）
# 命令ミックス対応、ページ置換アルゴリズムのシミュレーション付き


def mips_with_mix():
    """命令ミックスを考慮したMIPS計算"""
    print("\n--- 命令ミックスMIPS計算 ---")
    clock = float(input("クロック周波数 (GHz): "))

    print("命令タイプごとの出現頻度とCPIを入力（空行で終了）:")
    instructions = []
    while True:
        name = input("  命令名 (空でEnd): ").strip()
        if not name:
            break
        freq = float(input(f"  {name}の出現頻度 (例: 0.4): "))
        cpi = float(input(f"  {name}のCPI: "))
        instructions.append((name, freq, cpi))

    avg_cpi = sum(f * c for _, f, c in instructions)
    mips = (clock * 1000) / avg_cpi

    print(f"\n--- 結果 ---")
    for name, freq, cpi in instructions:
        print(f"  {name}: 頻度{freq:.0%}, CPI={cpi}, 寄与={freq * cpi:.2f}")
    print(f"  平均CPI: {avg_cpi:.2f}")
    print(f"  MIPS: {mips:.0f}")


def cache_calculator():
    """キャッシュメモリの実効アクセス時間"""
    print("\n--- キャッシュ実効アクセス時間 ---")
    cache_time = float(input("キャッシュアクセス時間 (ns): "))
    mem_time = float(input("主記憶アクセス時間 (ns): "))
    hit_rate = float(input("ヒット率 (例: 0.9): "))

    effective = hit_rate * cache_time + (1 - hit_rate) * mem_time
    speedup = mem_time / effective

    print(f"\n実効アクセス時間: {effective:.2f} ns")
    print(f"高速化率: {speedup:.2f}倍")

    # ヒット率を変えた場合のシミュレーション
    print("\n--- ヒット率別シミュレーション ---")
    print(f"{'ヒット率':>8} | {'実効時間':>8} | {'高速化率':>8}")
    print("-" * 30)
    for hr in [0.7, 0.8, 0.85, 0.9, 0.95, 0.99]:
        eff = hr * cache_time + (1 - hr) * mem_time
        print(f"{hr:>8.0%} | {eff:>7.2f}ns | {mem_time / eff:>7.2f}x")


def page_replacement_sim():
    """ページ置換アルゴリズムのシミュレーション"""
    print("\n--- ページ置換シミュレーション ---")

    ref_str = input("ページ参照列 (カンマ区切り, 例: 1,2,3,4,1,2,5,1,2,3,4,5): ")
    references = [int(x.strip()) for x in ref_str.split(",")]
    frames_count = int(input("ページフレーム数: "))

    # FIFO
    print(f"\n[FIFO]")
    frames = []
    faults_fifo = 0
    for page in references:
        if page in frames:
            status = "HIT "
        else:
            faults_fifo += 1
            status = "MISS"
            if len(frames) >= frames_count:
                frames.pop(0)  # 最も古いページを追い出す
            frames.append(page)
        print(f"  参照{page}: {status} フレーム={frames}")
    print(f"  ページフォルト: {faults_fifo}回")

    # LRU
    print(f"\n[LRU]")
    frames = []
    faults_lru = 0
    for page in references:
        if page in frames:
            status = "HIT "
            frames.remove(page)
            frames.append(page)  # 最近使ったのでリストの末尾に移動
        else:
            faults_lru += 1
            status = "MISS"
            if len(frames) >= frames_count:
                frames.pop(0)  # 最も長く使われていないページを追い出す
            frames.append(page)
        print(f"  参照{page}: {status} フレーム={frames}")
    print(f"  ページフォルト: {faults_lru}回")

    print(f"\n--- 比較 ---")
    print(f"  FIFO: {faults_fifo}回、LRU: {faults_lru}回")


def main():
    print("===== システム性能計算ツール =====")
    print("  1. 命令ミックスMIPS計算")
    print("  2. キャッシュ実効アクセス時間")
    print("  3. ページ置換シミュレーション")

    choice = input("\n選択 (1-3): ").strip()
    if choice == "1":
        mips_with_mix()
    elif choice == "2":
        cache_calculator()
    elif choice == "3":
        page_replacement_sim()


if __name__ == "__main__":
    main()
```
