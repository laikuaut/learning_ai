# 実践課題01：温度変換ツール ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（変数とデータ型）、第2章（条件分岐）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 変数の宣言・代入、printf/scanf による入出力、条件分岐を組み合わせて動くプログラムを作る

## 完成イメージ

```
===== 温度変換ツール =====
1. 摂氏 → 華氏
2. 華氏 → 摂氏

変換モードを選んでください (1-2): 1
温度を入力してください: 100.0

変換結果: 100.0°C = 212.0°F
```

```
変換モードを選んでください (1-2): 2
温度を入力してください: 32.0

変換結果: 32.0°F = 0.0°C
```

```
変換モードを選んでください (1-2): 5

無効な選択です。1 または 2 を入力してください。
```

---

## 課題の要件

1. 摂氏から華氏、華氏から摂氏の2つの変換モードを選択できる
2. ユーザーから変換モードと温度を `scanf` で受け取る
3. 無効な選択肢（1, 2 以外）にはエラーメッセージを表示する
4. 変換結果を小数点以下1桁で表示する
5. 変換公式：
   - 華氏 = 摂氏 × 9.0 / 5.0 + 32.0
   - 摂氏 = (華氏 - 32.0) × 5.0 / 9.0

---

## ステップガイド

<details>
<summary>ステップ1：メニューを表示して選択を受け取る</summary>

`printf` でメニューを表示し、`scanf` で選択番号を読み取ります。

```c
#include <stdio.h>

int main(void) {
    int choice;

    printf("===== 温度変換ツール =====\n");
    printf("1. 摂氏 → 華氏\n");
    printf("2. 華氏 → 摂氏\n\n");
    printf("変換モードを選んでください (1-2): ");
    scanf("%d", &choice);

    return 0;
}
```

</details>

<details>
<summary>ステップ2：温度を入力して変換する</summary>

温度は小数を扱うため `double` 型を使います。`scanf` では `%lf` を使います。

```c
double temp, result;

printf("温度を入力してください: ");
scanf("%lf", &temp);

if (choice == 1) {
    result = temp * 9.0 / 5.0 + 32.0;
    // 結果を表示
}
```

</details>

<details>
<summary>ステップ3：結果の表示と無効入力の処理</summary>

`if-else if-else` で分岐し、`%.1f` で小数点以下1桁に揃えます。

```c
if (choice == 1) {
    result = temp * 9.0 / 5.0 + 32.0;
    printf("\n変換結果: %.1f°C = %.1f°F\n", temp, result);
} else if (choice == 2) {
    result = (temp - 32.0) * 5.0 / 9.0;
    printf("\n変換結果: %.1f°F = %.1f°C\n", temp, result);
} else {
    printf("\n無効な選択です。1 または 2 を入力してください。\n");
}
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```c
/*
 * 温度変換ツール
 * 学べる内容：変数宣言、scanf/printf、条件分岐、浮動小数点演算
 * コンパイル：gcc -o temp_converter temp_converter.c
 * 実行：./temp_converter
 */
#include <stdio.h>

int main(void) {
    int choice;
    double temp;
    double result;

    /* メニュー表示 */
    printf("===== 温度変換ツール =====\n");
    printf("1. 摂氏 → 華氏\n");
    printf("2. 華氏 → 摂氏\n\n");

    /* モード選択 */
    printf("変換モードを選んでください (1-2): ");
    scanf("%d", &choice);

    if (choice == 1) {
        /* 摂氏 → 華氏 */
        printf("温度を入力してください: ");
        scanf("%lf", &temp);
        result = temp * 9.0 / 5.0 + 32.0;
        printf("\n変換結果: %.1f°C = %.1f°F\n", temp, result);

    } else if (choice == 2) {
        /* 華氏 → 摂氏 */
        printf("温度を入力してください: ");
        scanf("%lf", &temp);
        result = (temp - 32.0) * 5.0 / 9.0;
        printf("\n変換結果: %.1f°F = %.1f°C\n", temp, result);

    } else {
        printf("\n無効な選択です。1 または 2 を入力してください。\n");
    }

    return 0;
}
```

</details>

<details>
<summary>解答例（改良版 ─ 繰り返しで連続変換）</summary>

`while` ループを使って、終了を選ぶまで何度でも変換できるバージョンです。
第3章（繰り返し）を学んだ後に挑戦してみてください。

```c
/*
 * 温度変換ツール（改良版）
 * 学べる内容：ループによる繰り返し処理、入力バリデーション
 * コンパイル：gcc -o temp_converter2 temp_converter2.c
 * 実行：./temp_converter2
 */
#include <stdio.h>

int main(void) {
    int choice;
    double temp;
    double result;

    printf("===== 温度変換ツール =====\n");

    while (1) {
        printf("\n1. 摂氏 → 華氏\n");
        printf("2. 華氏 → 摂氏\n");
        printf("0. 終了\n\n");
        printf("変換モードを選んでください: ");
        scanf("%d", &choice);

        if (choice == 0) {
            printf("終了します。\n");
            break;
        }

        if (choice != 1 && choice != 2) {
            printf("無効な選択です。0, 1, 2 のいずれかを入力してください。\n");
            continue;
        }

        printf("温度を入力してください: ");
        scanf("%lf", &temp);

        if (choice == 1) {
            result = temp * 9.0 / 5.0 + 32.0;
            printf("\n変換結果: %.1f°C = %.1f°F\n", temp, result);
        } else {
            result = (temp - 32.0) * 5.0 / 9.0;
            printf("\n変換結果: %.1f°F = %.1f°C\n", temp, result);
        }
    }

    return 0;
}
```

**初心者向けとの違い:**
- `while (1)` の無限ループで連続変換を実現 → ユーザーが「0」を入力するまで何度でも使える
- `continue` で無効入力時にメニューに戻る → 不正入力でプログラムが終了しない
- 温度入力の `scanf` を一箇所にまとめている → コードの重複が減少

</details>
