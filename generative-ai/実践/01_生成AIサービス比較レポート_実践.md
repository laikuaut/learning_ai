## 課題の要件

1. **テキスト生成AI**を3サービス以上選び、以下の観点で比較表を作成する
   - サービス名・提供元
   - 主な得意分野（要約、コード生成、創作など）
   - 料金体系（無料枠の有無、月額、API従量課金）
   - 入力トークン上限（コンテキストウィンドウ（context window））
   - 主な制約事項・注意点
2. 用途別（ビジネスメール作成、プログラミング支援、文章校正、翻訳）に、どのサービスが最適かを表形式でまとめる
3. 各評価に対して「なぜそう判断したか」の根拠を1〜2文で記述する
4. 最後にまとめ・考察を200文字以上で記述する

## ステップガイド

<details>
<summary>ステップ1：調査対象のサービスを選定する</summary>

まず比較対象を決めましょう。以下は代表的なサービスです。

- **ChatGPT**（OpenAI）
- **Claude**（Anthropic）
- **Gemini**（Google）
- **Copilot**（Microsoft）
- **Command R**（Cohere）

最低3つを選びましょう。公式サイトやドキュメントから情報を収集します。

**よくある間違い**: 古い情報を参照してしまうこと。生成AIの世界は変化が激しいため、必ず調査日を記録しましょう。

</details>

|---------|---------|--------|--------|

<details>
<summary>ステップ3：用途別おすすめマトリクスを作成する</summary>

用途（行）×サービス（列）で、◎（最適）/ ○（適） / △（可能だが不向き）の3段階で評価します。

```markdown
| 用途 | ChatGPT | Claude | Gemini |
|------|---------|--------|--------|
| ビジネスメール作成 | ○ | ◎ | ○ |
| プログラミング支援 | ◎ | ○ | ○ |
| 文章校正 | ○ | ◎ | △ |
| 翻訳 | ○ | ○ | ◎ |
```

各セルの評価に対して、根拠を別途箇条書きで記述しましょう。

</details>

<details>
<summary>ステップ4：まとめ・考察を書く</summary>

以下の観点を含めるとよいでしょう。

- どのサービスが「万能」に近いか
- 使い分けが必要な場面はあるか
- 料金面でのコストパフォーマンス
- 今後の動向予測

200文字以上で、自分の意見を含めて記述しましょう。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

### 生成AIサービス比較レポート

**調査日**: 2026年4月

#### 1. サービス比較表

| 比較項目 | ChatGPT (GPT-4o) | Claude (Claude 3.5 Sonnet) | Gemini (Gemini 1.5 Pro) |
|---------|-------------------|---------------------------|------------------------|
| 提供元 | OpenAI | Anthropic | Google |
| 得意分野 | 汎用的な対話・コード生成 | 長文分析・丁寧な文章作成 | マルチモーダル・検索連携 |
| 無料枠 | あり（制限付き） | あり（制限付き） | あり（制限付き） |
| 有料プラン | 月額20ドル（Plus） | 月額20ドル（Pro） | 月額19.99ドル（Advanced） |
| API従量課金 | 入力/出力トークン単位 | 入力/出力トークン単位 | 入力/出力トークン単位 |
| コンテキスト上限 | 128Kトークン | 200Kトークン | 1Mトークン |
| 主な制約 | 学習データのカットオフあり | リアルタイム検索非対応 | 一部地域で制限あり |

#### 2. 用途別おすすめマトリクス

| 用途 | ChatGPT | Claude | Gemini |
|------|:-------:|:------:|:------:|
| ビジネスメール作成 | ○ | ◎ | ○ |
| プログラミング支援 | ◎ | ○ | ○ |
| 文章校正 | ○ | ◎ | △ |
| 翻訳 | ○ | ○ | ◎ |

#### 3. 評価根拠

- **ビジネスメール作成**: Claudeは丁寧で自然な日本語生成に定評があり、ビジネス文書に適しています
- **プログラミング支援**: ChatGPTはコード生成の精度とエコシステム（プラグイン等）が充実しています
- **文章校正**: Claudeは長文の文脈を正確に把握でき、細かなニュアンスの修正が得意です
- **翻訳**: Geminiは多言語対応に強く、Google翻訳の知見も活かされています

#### 4. まとめ・考察

現時点で「すべての用途で最高」と言える単一のサービスは存在しません。ChatGPTは汎用性が高くプログラミング支援で強みを持ち、Claudeは長文処理と丁寧な日本語生成で優れています。Geminiはマルチモーダル対応とGoogle検索との連携が魅力です。実務では「メインで使うサービス」を1つ決めつつ、用途に応じて使い分けるのがベストプラクティスです。料金面では各社とも無料枠を提供しているため、まず無料で試してから有料プランを検討することをおすすめします。生成AIの性能は急速に向上しており、半年後には評価が大きく変わる可能性があるため、定期的な再調査が重要です。

### 生成AIサービス比較レポート 改良版

**調査日**: 2026年4月
**調査者**: （氏名）
**更新予定**: 3ヶ月後に再調査

改良版では、以下の点を強化しています。

- **比較対象を5サービスに拡大**
- **定量的なベンチマークスコアを追記**
- **コスト試算（月間利用量ベース）を追加**

#### 1. サービス比較表（拡張版）

| 比較項目 | ChatGPT | Claude | Gemini | Copilot | Command R+ |
|---------|---------|--------|--------|---------|-----------|
| 提供元 | OpenAI | Anthropic | Google | Microsoft | Cohere |
| 得意分野 | 汎用対話・コード | 長文分析・安全性 | マルチモーダル | Office連携 | RAG・検索 |
| 無料枠 | あり | あり | あり | あり | あり |
| 月額プラン | $20 | $20 | $19.99 | $30(M365込) | 従量のみ |
| コンテキスト | 128K | 200K | 1M | 128K | 128K |
| 日本語品質（5段階） | 4 | 5 | 3 | 4 | 2 |
| コード生成力（5段階） | 5 | 4 | 4 | 5 | 3 |

#### 2. 用途別おすすめ（1位〜3位）

| 用途 | 1位 | 2位 | 3位 |
|------|-----|-----|-----|
| ビジネスメール | Claude | ChatGPT | Copilot |
| プログラミング | ChatGPT | Copilot | Claude |
| 文章校正 | Claude | ChatGPT | Gemini |
| 翻訳 | Gemini | ChatGPT | Claude |
| 資料要約 | Claude | Gemini | ChatGPT |
| データ分析 | ChatGPT | Gemini | Command R+ |

#### 3. コスト試算（月間100万トークン利用想定）

```
ChatGPT API  : 入力 $2.50 + 出力 $10.00 = 約 $12.50/月
Claude API   : 入力 $3.00 + 出力 $15.00 = 約 $18.00/月
Gemini API   : 入力 $1.25 + 出力 $5.00  = 約 $6.25/月
```

※ 上記は概算です。実際の料金はモデルのバージョンや利用方法によって変動します。

#### 4. 選定フローチャート

```
スタート
  │
  ├─ 主な用途は？
  │    ├─ コード生成 ──→ ChatGPT / Copilot
  │    ├─ 長文分析   ──→ Claude
  │    ├─ 画像+テキスト ─→ Gemini
  │    └─ RAG構築    ──→ Command R+
  │
  ├─ 予算は？
  │    ├─ 無料で試したい ──→ 各社無料枠を活用
  │    ├─ 月$20以内     ──→ ChatGPT Plus / Claude Pro
  │    └─ API大量利用   ──→ Gemini API（コスト最安）
  │
  └─ 結論：用途×予算で最適なサービスを選定
```

#### 5. まとめ・考察

生成AIサービスの選定では「万能な1つ」を探すのではなく、用途・予算・組織のセキュリティ要件に応じた最適解を見つけることが重要です。初心者向けの改良ポイントとして、単純な機能比較だけでなく、コスト試算と選定フローチャートを追加しました。これにより「結局どれを使えばいいの？」という疑問に対して、論理的な判断基準を提供できます。また、生成AIの進化スピードは非常に速いため、レポートには必ず「調査日」と「次回更新予定日」を記載し、定期的にアップデートする運用を推奨します。実務では、1つのサービスに依存するロックイン（lock-in）リスクを避けるため、APIの抽象化レイヤーを設けておくこともベストプラクティスです。

**初心者向けとの違い**:
- 比較対象を3→5サービスに拡大し、より多角的な視点を提供
- 定性評価（◎○△）に加えて定量的な5段階評価を導入
- コスト試算を追加し、予算面での意思決定を支援
- 選定フローチャートで実際の判断プロセスを可視化
