## 基本問題

### 問題1：見出しの階層構造
以下のHTML の見出し構造には問題があります。問題点を指摘し、修正してください。

```html
<h1>HTML入門講座</h1>
<h3>第1部：基礎編</h3>
<h4>HTMLとは</h4>
<h2>第2部：応用編</h2>
<h6>フォームの作り方</h6>
```

<details>
<summary>ヒント</summary>

見出しのレベルを飛ばしてはいけません。h1 の次は h2、h2 の次は h3 という順番を守りましょう。

</details>

<details>
<summary>解答例</summary>

問題点：
- `<h1>` の直後に `<h3>` が来ている（h2 を飛ばしている）
- `<h2>` の直後に `<h6>` が来ている（h3〜h5 を飛ばしている）

修正後：
```html
<h1>HTML入門講座</h1>
<h2>第1部：基礎編</h2>
<h3>HTMLとは</h3>
<h2>第2部：応用編</h2>
<h3>フォームの作り方</h3>
```

</details>

### 問題2：strong と em の使い分け
次の文章を、適切な要素を使ってマークアップしてください。

- 「注意」は重要な文言として太字に
- 「必ず」は強調して斜体に

> 注意：パスワードは必ず8文字以上で設定してください。

<details>
<summary>解答例</summary>

```html
<p><strong>注意：</strong>パスワードは<em>必ず</em>8文字以上で設定してください。</p>
```

`<strong>` は重要性を表し（デフォルト太字）、`<em>` は強調を表します（デフォルト斜体）。

</details>

### 問題3：リンクの基本
以下の条件でリンクを作成してください。

1. Google（https://www.google.com）へのリンク。新しいタブで開く。
2. 同じフォルダにある `about.html` へのリンク。同じタブで開く。
3. メールアドレス `info@example.com` へのメールリンク。

<details>
<summary>解答例</summary>

```html
<!-- 1. 外部リンク（新しいタブ） -->
<a href="https://www.google.com" target="_blank" rel="noopener noreferrer">Google</a>

<!-- 2. 内部リンク（同じタブ） -->
<a href="about.html">自己紹介ページ</a>

<!-- 3. メールリンク -->
<a href="mailto:info@example.com">お問い合わせ</a>
```

</details>

### 問題4：特殊文字
以下のテキストを正しいHTMLで記述してください（ブラウザに正しく表示されるように）。

表示したいテキスト：`HTMLのタグは <要素名> と書きます。価格は ¥500 です。`

<details>
<summary>ヒント</summary>

`<` と `>` はHTMLのタグとして解釈されてしまうため、文字実体参照を使います。

</details>

<details>
<summary>解答例</summary>

```html
<p>HTMLのタグは &lt;要素名&gt; と書きます。価格は &yen;500 です。</p>
```

`&lt;` は `<`、`&gt;` は `>`、`&yen;` は `¥` を表します。

</details>

### 問題5：ページ内リンク
以下の要件を満たす目次とセクションを作成してください。

- 「セクション1」「セクション2」「セクション3」への目次リンクを作成
- 各セクションには id 属性を付ける
- 各セクションの末尾に「ページ先頭に戻る」リンクを配置

<details>
<summary>解答例</summary>

```html
<h1 id="top">ドキュメント</h1>
<nav>
  <ul>
    <li><a href="#section1">セクション1</a></li>
    <li><a href="#section2">セクション2</a></li>
    <li><a href="#section3">セクション3</a></li>
  </ul>
</nav>

<h2 id="section1">セクション1</h2>
<p>セクション1の内容です。</p>
<a href="#top">ページ先頭に戻る</a>

<h2 id="section2">セクション2</h2>
<p>セクション2の内容です。</p>
<a href="#top">ページ先頭に戻る</a>

<h2 id="section3">セクション3</h2>
<p>セクション3の内容です。</p>
<a href="#top">ページ先頭に戻る</a>
```

</details>

## 応用問題

### 問題6：相対パスの計算
以下のフォルダ構造があるとき、`pages/blog.html` から見た各ファイルの相対パスを答えてください。

```
my-site/
├── index.html
├── pages/
│   ├── blog.html     ← 現在のファイル
│   └── contact.html
├── images/
│   └── logo.png
└── css/
    └── style.css
```

1. `index.html` への相対パス
2. `contact.html` への相対パス
3. `logo.png` への相対パス
4. `style.css` への相対パス

<details>
<summary>解答</summary>

1. `../index.html`（1つ上の階層）
2. `contact.html`（同じフォルダ）
3. `../images/logo.png`（1つ上 → images フォルダ）
4. `../css/style.css`（1つ上 → css フォルダ）

</details>

### 問題7：インライン要素の総合
以下の要件をすべて満たすHTMLを記述してください。

- 「HTML」を略語として `<abbr>` でマークアップ（title属性で正式名称を記述）
- 「Ctrl + S」をキーボード入力として `<kbd>` でマークアップ
- 公開日を `<time>` 要素で記述（datetime属性付き）
- 著作権表示を `<small>` で記述

<details>
<summary>解答例</summary>

```html
<p><abbr title="HyperText Markup Language">HTML</abbr>の学習を始めましょう。</p>
<p>ファイルを保存するには <kbd>Ctrl</kbd> + <kbd>S</kbd> を押します。</p>
<p>公開日：<time datetime="2024-03-15">2024年3月15日</time></p>
<p><small>&copy; 2024 My Website. All rights reserved.</small></p>
```

</details>

### 問題8：ナビゲーション付きページの作成
以下の要件を満たす完全なHTMLページを作成してください。

- ページタイトル「技術ブログ | My Site」
- h1 で「技術ブログ」
- ナビゲーション：「ホーム」「ブログ」「お問い合わせ」のリンク（href は # でOK）
- 3つのセクション（h2 見出し + 段落 + 「続きを読む」リンク）
- フッターに著作権表示と「ページ先頭に戻る」リンク
- 外部リンクには `target="_blank"` と `rel="noopener noreferrer"` を付ける

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>技術ブログ | My Site</title>
</head>
<body id="top">
  <h1>技術ブログ</h1>
  <nav>
    <a href="#">ホーム</a> |
    <a href="#">ブログ</a> |
    <a href="#">お問い合わせ</a>
  </nav>

  <hr>

  <h2>HTMLの基礎を学ぼう</h2>
  <p>HTMLはWebページの構造を記述するマークアップ言語です。この記事では基本的なタグの使い方を解説します。</p>
  <a href="#">続きを読む</a>

  <h2>CSSでデザインを整える</h2>
  <p>CSSを使えば、HTMLに色やレイアウトを適用できます。FlexboxやGridなどのモダンな手法も紹介します。</p>
  <a href="#">続きを読む</a>

  <h2>JavaScriptで動きを付ける</h2>
  <p>JavaScriptを使えば、ユーザーの操作に応じた動的なページを作成できます。</p>
  <a href="#">続きを読む</a>

  <hr>

  <footer>
    <p><small>&copy; 2024 My Site. All rights reserved.</small></p>
    <a href="#top">ページ先頭に戻る</a>
  </footer>
</body>
</html>
```

</details>
