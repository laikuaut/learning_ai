---

## 2.1 比較演算子

2つの値を比較して、`True` か `False` を返します。

| 演算子 | 意味 | 例 | 結果 |
|--------|------|-----|------|
| `==` | 等しい | `5 == 5` | `True` |
| `!=` | 等しくない | `5 != 3` | `True` |
| `>` | より大きい | `5 > 3` | `True` |
| `<` | より小さい | `5 < 3` | `False` |
| `>=` | 以上 | `5 >= 5` | `True` |
| `<=` | 以下 | `5 <= 3` | `False` |

```python
age = 20
print(age >= 18)   # True
print(age == 15)   # False
```

> **注意**: `=` は代入、`==` は比較です。間違えやすいので注意！

## 2.2 if文の基本

条件が `True` のとき、インデント（字下げ）されたブロックが実行されます。

```python
age = 20

if age >= 18:
    print("成人です")
```

### インデントのルール

- Pythonでは**インデント（スペース4つ）**でブロックを表します
- 同じブロック内のインデントは揃える必要があります

```python
if age >= 18:
    print("成人です")      # OK：同じインデント
    print("投票できます")  # OK：同じインデント
```

---

## 2.3 if-else

条件が `True` のときと `False` のときで処理を分けます。

```python
age = 15

if age >= 18:
    print("成人です")
else:
    print("未成年です")
```

**出力：** `未成年です`

---

## 2.4 if-elif-else

3つ以上の条件で分岐したいときは `elif`（else ifの略）を使います。

```python
score = 75

if score >= 90:
    print("評価: A")
elif score >= 80:
    print("評価: B")
elif score >= 70:
    print("評価: C")
elif score >= 60:
    print("評価: D")
else:
    print("評価: F")
```

**出力：** `評価: C`

### ポイント

- `elif` は上から順に評価され、**最初に `True` になった条件だけ**が実行されます
- `else` はどの条件にも当てはまらないときに実行されます
- `else` は省略可能です

## 2.5 論理演算子

複数の条件を組み合わせます。

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `and` | 両方True | `age >= 18 and age <= 65` |
| `or` | どちらかTrue | `day == "土" or day == "日"` |
| `not` | 否定 | `not is_student` |

```python
age = 25
income = 300

if age >= 20 and income >= 200:
    print("ローン審査に通る可能性があります")
```

```python
day = "日"

if day == "土" or day == "日":
    print("週末です！")
else:
    print("平日です")
```

```python
is_raining = False

if not is_raining:
    print("傘は要りません")
```

## 2.6 条件分岐のネスト（入れ子）

if文の中にさらにif文を書くことができます。

```python
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("入場できます")
    else:
        print("身分証明書が必要です")
else:
    print("18歳未満は入場できません")
```

> **コツ**: ネストが深くなりすぎると読みにくくなります。2〜3段階が目安です。

---

## 2.7 in 演算子

リストや文字列に特定の値が含まれているか確認できます。

```python
fruit = "りんご"
fruits = ["りんご", "みかん", "バナナ"]

if fruit in fruits:
    print(f"{fruit}はリストにあります")
```

```python
message = "今日は晴れです"

if "晴れ" in message:
    print("いい天気ですね！")
```

---

## 2.8 三項演算子（条件式）

1行で条件分岐を書く方法です。簡単な条件に使います。

```python
age = 20
status = "成人" if age >= 18 else "未成年"
print(status)  # 成人
```

これは以下と同じ意味です：

```python
if age >= 18:
    status = "成人"
else:
    status = "未成年"
```

---

## まとめ

| 概念 | ポイント |
|------|----------|
| `if` | 条件がTrueのとき実行 |
| `elif` | 前の条件がFalseで、この条件がTrueのとき実行 |
| `else` | どの条件にも当てはまらないとき実行 |
| `and` | 両方Trueで全体がTrue |
| `or` | どちらかTrueで全体がTrue |
| `not` | TrueとFalseを反転 |
| `in` | 要素が含まれているか確認 |
| インデント | Pythonではブロックをインデント（スペース4つ）で表す |
