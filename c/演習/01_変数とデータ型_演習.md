# 第1章：変数とデータ型 - 演習問題

この演習では、C言語の変数宣言、基本データ型、`printf`/`scanf` による入出力を実践します。

---

## 基本問題

### 問題1：自己紹介プログラム

変数を使って、名前（文字列）と年齢（整数）を格納し、以下のように表示するプログラムを作成してください。

**期待される出力例：**
```
私の名前は田中太郎です。
年齢は25歳です。
よろしくお願いします！
```

<details>
<summary>ヒント</summary>

- 文字列は `char name[] = "田中太郎";` のように宣言します
- 整数は `int age = 25;` のように宣言します
- `printf` の書式指定子は、文字列が `%s`、整数が `%d` です

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    /* 文字列型の変数に名前を格納 */
    char name[] = "田中太郎";
    /* 整数型の変数に年齢を格納 */
    int age = 25;

    /* printf で書式指定子を使って変数の値を埋め込む */
    printf("私の名前は%sです。\n", name);
    printf("年齢は%d歳です。\n", age);
    printf("よろしくお願いします！\n");

    return 0;
}
```

</details>

### 問題2：四則演算

2つの整数変数 `a` と `b` に好きな値を代入し、加算・減算・乗算・除算・剰余の結果をすべて表示してください。

**期待される出力例（a=17, b=5 の場合）：**
```
a = 17, b = 5
加算: 17 + 5 = 22
減算: 17 - 5 = 12
乗算: 17 * 5 = 85
除算: 17 / 5 = 3
剰余: 17 % 5 = 2
```

<details>
<summary>ヒント</summary>

- 整数同士の除算では小数点以下が切り捨てられます
- 剰余演算子 `%` は割り算の余りを求めます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int a = 17;
    int b = 5;

    printf("a = %d, b = %d\n", a, b);

    /* 各演算の結果を表示 */
    printf("加算: %d + %d = %d\n", a, b, a + b);
    printf("減算: %d - %d = %d\n", a, b, a - b);
    printf("乗算: %d * %d = %d\n", a, b, a * b);

    /* 整数除算では小数部分が切り捨てられる点に注意 */
    printf("除算: %d / %d = %d\n", a, b, a / b);

    /* 剰余（余り）を求める */
    printf("剰余: %d %% %d = %d\n", a, b, a % b);

    return 0;
}
```

**補足：** `printf` の中で `%` を文字として表示するには `%%` と書きます。

</details>

### 問題3：sizeof で各型のサイズを確認

`char`、`short`、`int`、`long`、`float`、`double` それぞれの型が何バイトかを `sizeof` 演算子で調べて表示してください。

**期待される出力例（環境により値は異なります）：**
```
char   : 1 バイト
short  : 2 バイト
int    : 4 バイト
long   : 8 バイト
float  : 4 バイト
double : 8 バイト
```

<details>
<summary>ヒント</summary>

- `sizeof(型名)` で型のバイト数を取得できます
- `sizeof` の戻り値は `size_t` 型なので、書式指定子は `%zu` を使います

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    /* sizeof 演算子で各データ型のサイズ（バイト数）を取得 */
    /* %zu は size_t 型に対応する書式指定子 */
    printf("char   : %zu バイト\n", sizeof(char));
    printf("short  : %zu バイト\n", sizeof(short));
    printf("int    : %zu バイト\n", sizeof(int));
    printf("long   : %zu バイト\n", sizeof(long));
    printf("float  : %zu バイト\n", sizeof(float));
    printf("double : %zu バイト\n", sizeof(double));

    return 0;
}
```

**補足：** 型のサイズは環境（OS・コンパイラ）によって異なることがあります。C言語の仕様では `sizeof(char)` は常に 1 と定められています。

</details>

## 応用問題

### 問題4：円の面積と円周を計算

`scanf` で半径（実数）を入力し、円の面積と円周を計算して表示するプログラムを作成してください。円周率は `3.14159265358979` を使ってください。

- 面積 = π × r²
- 円周 = 2 × π × r

**期待される出力例：**
```
半径を入力してください: 5.0
面積: 78.54
円周: 31.42
```

<details>
<summary>ヒント</summary>

- 半径は `double` 型で宣言します
- `scanf` で `double` を読み込むには `%lf` を使います
- `printf` で小数点以下2桁まで表示するには `%.2f` を使います

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    /* 円周率を定数として定義 */
    const double PI = 3.14159265358979;
    double radius;
    double area, circumference;

    printf("半径を入力してください: ");

    /* scanf で double を読み込むには %lf を使う */
    scanf("%lf", &radius);

    /* 面積 = π × r × r */
    area = PI * radius * radius;

    /* 円周 = 2 × π × r */
    circumference = 2.0 * PI * radius;

    /* %.2f で小数点以下2桁まで表示 */
    printf("面積: %.2f\n", area);
    printf("円周: %.2f\n", circumference);

    return 0;
}
```

</details>

### 問題5：温度変換（華氏→摂氏）

華氏温度（Fahrenheit）を入力し、摂氏温度（Celsius）に変換して表示するプログラムを作成してください。

変換式：**摂氏 = (華氏 - 32) × 5 / 9**

**期待される出力例：**
```
華氏温度を入力してください: 100.0
摂氏温度: 37.78 ℃
```

<details>
<summary>ヒント</summary>

- 整数同士の除算 `5 / 9` は `0` になってしまいます
- `5.0 / 9.0` のように小数で書くか、変数を `double` にして計算しましょう

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    double fahrenheit, celsius;

    printf("華氏温度を入力してください: ");
    scanf("%lf", &fahrenheit);

    /*
     * 変換式: 摂氏 = (華氏 - 32) × 5 / 9
     * 5.0 / 9.0 と書くことで浮動小数点演算にする
     * 整数同士の 5 / 9 だと結果が 0 になるので注意
     */
    celsius = (fahrenheit - 32.0) * 5.0 / 9.0;

    printf("摂氏温度: %.2f ℃\n", celsius);

    return 0;
}
```

**よくある間違い：** `(fahrenheit - 32) * 5 / 9` と書いた場合、`fahrenheit` が `double` なので演算全体は浮動小数点で行われますが、`(int型 - 32) * 5 / 9` のように整数変数を使うと `5 / 9` が `0` になります。

</details>

## チャレンジ問題

### 問題6：BMI計算プログラム

身長（cm）と体重（kg）を入力し、BMI を計算して判定結果を表示するプログラムを作成してください。

**BMI = 体重(kg) / 身長(m)²**

| BMI | 判定 |
|---|---|
| 18.5 未満 | 低体重 |
| 18.5 以上 25.0 未満 | 普通体重 |
| 25.0 以上 30.0 未満 | 肥満（1度） |
| 30.0 以上 | 肥満（2度以上） |

**期待される出力例：**
```
身長(cm)を入力してください: 170.0
体重(kg)を入力してください: 65.0
あなたのBMI: 22.49
判定: 普通体重
```

<details>
<summary>ヒント</summary>

- 身長を cm → m に変換するには `height_m = height_cm / 100.0` とします
- BMI の判定には `if - else if - else` 文を使います（第2章の先取りになります）

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    double height_cm, weight_kg;
    double height_m, bmi;

    printf("身長(cm)を入力してください: ");
    scanf("%lf", &height_cm);

    printf("体重(kg)を入力してください: ");
    scanf("%lf", &weight_kg);

    /* cm を m に変換 */
    height_m = height_cm / 100.0;

    /* BMI = 体重(kg) / 身長(m)の2乗 */
    bmi = weight_kg / (height_m * height_m);

    printf("あなたのBMI: %.2f\n", bmi);

    /* BMI に基づく判定（if-else if で範囲を絞り込む） */
    if (bmi < 18.5) {
        printf("判定: 低体重\n");
    } else if (bmi < 25.0) {
        printf("判定: 普通体重\n");
    } else if (bmi < 30.0) {
        printf("判定: 肥満（1度）\n");
    } else {
        printf("判定: 肥満（2度以上）\n");
    }

    return 0;
}
```

**補足：** この問題では条件分岐（`if` 文）を先取りして使っています。まだ学習していない場合は、BMI の計算と表示だけでも十分です。

</details>
