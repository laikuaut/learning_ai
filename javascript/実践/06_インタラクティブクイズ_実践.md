# 実践課題06：インタラクティブクイズアプリ ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第1章〜第9章（変数・演算子・条件分岐・関数・配列・オブジェクト・文字列操作・DOM操作・イベント処理）
> **課題の種類**: ミニプロジェクト
> **学習目標**: DOM操作とイベント処理を組み合わせて、複数画面を持つインタラクティブなWebアプリを構築する

## 課題の要件

1. HTMLファイル1つで完結すること
2. 5問以上のクイズ（4択）をオブジェクトの配列で管理する
3. 1問ずつ表示し、選択肢をラジオボタンで表示する
4. 「確認する」ボタンで正誤を判定し、解説を表示する
5. 全問終了後に結果画面（正解数・正答率・評価メッセージ）を表示する
6. 「もう一度挑戦する」で最初からやり直せる
7. DOM操作で画面の表示を動的に切り替える（ページ遷移なし）
8. イベントリスナー（`addEventListener`）を使う
