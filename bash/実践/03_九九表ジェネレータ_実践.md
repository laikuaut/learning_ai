## 課題の要件

1. 引数なしで実行すると、1〜9の九九表を表形式で表示する
2. 引数で段（1〜9）を指定すると、その段だけを縦に表示する
3. 引数が1〜9の範囲外の場合、エラーメッセージを表示する
4. 桁揃え（右寄せ）して見やすくする
5. 特定の段を表示する場合は合計も表示する

---

## ステップガイド

<details>
<summary>ステップ1：引数なしの九九表を作る</summary>

ネストした `for` ループで9x9の表を出力します。`printf` の `%3d` で3桁右寄せにします。

```bash
for i in $(seq 1 9); do
    for j in $(seq 1 9); do
        printf "%4d" $((i * j))
    done
    echo ""  # 行末で改行
done
```

</details>

<details>
<summary>ステップ2：ヘッダー行と罫線を追加する</summary>

1行目にヘッダー（1〜9）、2行目に罫線を表示してから本体を出力します。

```bash
# ヘッダー行
printf "     |"
for j in $(seq 1 9); do
    printf "%4d" "$j"
done
echo ""

# 罫線
echo "-----+------------------------------------"
```

</details>

<details>
<summary>ステップ3：引数ありの場合を処理する</summary>

`$1` が指定されている場合、その段だけを表示するモードに切り替えます。

```bash
if [ -n "$1" ]; then
    # 特定の段を表示
    dan=$1
    # ... 1段分のループ
else
    # 全体の九九表を表示
    # ...
fi
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```bash
#!/usr/bin/env bash
# 九九表ジェネレータ
# 学べる内容：ネストしたforループ、算術展開、printf整形
# 実行方法：chmod +x kuku.sh && ./kuku.sh [段]

# --- 特定の段を表示する場合 ---
if [ -n "${1:-}" ]; then
    dan=$1

    # バリデーション
    if [ "$dan" -lt 1 ] 2>/dev/null || [ "$dan" -gt 9 ] 2>/dev/null; then
        echo "エラー: 1〜9の数字を指定してください"
        exit 1
    fi

    echo ""
    echo "===== ${dan}の段 ====="

    total=0
    for j in $(seq 1 9); do
        result=$((dan * j))
        printf "%d x %d = %2d\n" "$dan" "$j" "$result"
        total=$((total + result))
    done

    echo "合計: ${total}"
    exit 0
fi

# --- 九九表を表示する場合 ---
echo ""
echo "===== 九九表 ====="
echo ""

# ヘッダー行
printf "     |"
for j in $(seq 1 9); do
    printf "%4d" "$j"
done
echo ""

# 罫線
echo "-----+------------------------------------"

# 本体
for i in $(seq 1 9); do
    printf "  %d  |" "$i"
    for j in $(seq 1 9); do
        printf "%4d" $((i * j))
    done
    echo ""
done
```

</details>

<details>
<summary>解答例（改良版 ─ 関数化とカラー表示）</summary>

```bash
#!/usr/bin/env bash
# 九九表ジェネレータ（改良版）
# 関数化とカラー表示に対応
# 実行方法：chmod +x kuku.sh && ./kuku.sh [段]

# --- 色定義 ---
if [ -t 1 ]; then
    # 端末出力の場合のみ色を有効化
    BOLD='\033[1m'
    GREEN='\033[32m'
    CYAN='\033[36m'
    RESET='\033[0m'
else
    BOLD="" GREEN="" CYAN="" RESET=""
fi

# --- 特定の段を表示 ---
show_single() {
    local dan=$1
    local total=0

    echo ""
    echo -e "${BOLD}===== ${dan}の段 =====${RESET}"

    for j in $(seq 1 9); do
        local result=$((dan * j))
        printf "%d x %d = %2d\n" "$dan" "$j" "$result"
        total=$((total + result))
    done

    echo -e "${GREEN}合計: ${total}${RESET}"
}

# --- 全九九表を表示 ---
show_table() {
    local max=${1:-9}

    echo ""
    echo -e "${BOLD}===== 九九表 =====${RESET}"
    echo ""

    # ヘッダー行
    printf "     |"
    for j in $(seq 1 "$max"); do
        printf "%4d" "$j"
    done
    echo ""

    # 罫線（動的に長さを計算）
    local line_len=$((max * 4))
    printf "-----+"
    printf '%*s' "$line_len" '' | tr ' ' '-'
    echo ""

    # 本体
    for i in $(seq 1 "$max"); do
        printf -v row_header "  %d  |" "$i"
        echo -ne "${CYAN}${row_header}${RESET}"
        for j in $(seq 1 "$max"); do
            local result=$((i * j))
            # 対角線（同じ数同士の積）を強調
            if [ "$i" -eq "$j" ]; then
                printf "${BOLD}%4d${RESET}" "$result"
            else
                printf "%4d" "$result"
            fi
        done
        echo ""
    done
}

# --- メイン処理 ---
if [ -n "${1:-}" ]; then
    # 引数のバリデーション
    if ! [[ "$1" =~ ^[0-9]+$ ]] || [ "$1" -lt 1 ] || [ "$1" -gt 9 ]; then
        echo "エラー: 1〜9の数字を指定してください" >&2
        exit 1
    fi
    show_single "$1"
else
    show_table 9
fi
```

**初心者向けとの違い:**
- 関数化 → 表示ロジックを `show_single` / `show_table` に分離し見通しが良い
- カラー表示 → 端末判定 `[ -t 1 ]` 付きで安全にカラーを使用
- 罫線を動的生成 → `max` が変わっても自動調整
- `printf -v` で変数に格納 → カラーと組み合わせやすい
- 正規表現バリデーション `[[ "$1" =~ ^[0-9]+$ ]]` → 数値以外の入力も安全に弾く
- 対角線の強調 → 同じ数同士の積（平方数）をボールドで表示

</details>
