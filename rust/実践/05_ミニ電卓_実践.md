# 実践課題05：ミニ電卓 ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第1章（変数と所有権）、第2章（条件分岐）、第3章（ループとイテレータ）、第4章（ベクタと文字列）、第5章（関数とクロージャ）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 文字列のパース処理と関数設計を組み合わせ、入力に応じた計算処理を正しく実装する

---

## 完成イメージ

```
===== ミニ電卓 =====
式を入力してください（'quit' で終了）

> 10 + 3
= 13

> 100 / 7
= 14.285714285714286

> 2 ** 10
= 1024

> 10 / 0

> 10 % 3
= 1

> history
--- 計算履歴 ---
1. 10 + 3 = 13
2. 100 / 7 = 14.285714285714286
3. 2 ** 10 = 1024
4. 10 % 3 = 1

> quit
お疲れさまでした。4回の計算を行いました。
```

---

## 課題の要件

1. `数値 演算子 数値` の形式で入力を受け取る
2. 対応する演算子：`+`（加算）、`-`（減算）、`*`（乗算）、`/`（除算）、`%`（剰余）、`**`（累乗）
3. ゼロ除算のエラーを検出して適切にメッセージを表示する
4. `history` コマンドで計算履歴を表示する
5. `quit` で終了し、合計計算回数を表示する
6. 不正な入力にはエラーメッセージを表示してループを継続する

## ステップガイド

<details>
<summary>ステップ1：式を分割して解析する</summary>

`**` 演算子は2文字なので、先に `**` を検出してから1文字演算子を処理する必要があります。

```rust
fn parse_expression(input: &str) -> Option<(f64, String, f64)> {
    // "**" を含むか先にチェック
    if input.contains("**") {
        let parts: Vec<&str> = input.splitn(2, "**").collect();
        if parts.len() == 2 {
            let left: f64 = parts[0].trim().parse().ok()?;
            let right: f64 = parts[1].trim().parse().ok()?;
            return Some((left, "**".to_string(), right));
        }
    }

    // 1文字演算子で分割
    let parts: Vec<&str> = input.split_whitespace().collect();
    if parts.len() != 3 {
        return None;
    }
    let left: f64 = parts[0].parse().ok()?;
    let op = parts[1].to_string();
    let right: f64 = parts[2].parse().ok()?;
    Some((left, op, right))
}
```

<summary>解答例（初心者向け）</summary>

```rust
// ミニ電卓
// 学べる内容：文字列パース、関数、Result、ループ、Vec
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

fn calculate(left: f64, op: &str, right: f64) -> Result<f64, String> {
    match op {
        "+" => Ok(left + right),
        "-" => Ok(left - right),
        "*" => Ok(left * right),
        "/" => {
            if right == 0.0 {
                Err("0で割ることはできません。".to_string())
            } else {
                Ok(left / right)
            }
        }
        "%" => {
            if right == 0.0 {
                Err("0で割ることはできません。".to_string())
            } else {
                Ok(left % right)
            }
        }
        "**" => Ok(left.powf(right)),
        _ => Err(format!("不明な演算子: {}", op)),
    }
}

fn parse_expression(input: &str) -> Option<(f64, String, f64)> {
    // "**" 演算子を先にチェック
    if input.contains("**") {
        let parts: Vec<&str> = input.splitn(2, "**").collect();
        if parts.len() == 2 {
            let left: f64 = parts[0].trim().parse().ok()?;
            let right: f64 = parts[1].trim().parse().ok()?;
            return Some((left, "**".to_string(), right));
        }
    }

    // スペース区切りで分割
    let parts: Vec<&str> = input.split_whitespace().collect();
    if parts.len() != 3 {
        return None;
    }
    let left: f64 = parts[0].parse().ok()?;
    let op = parts[1].to_string();
    let right: f64 = parts[2].parse().ok()?;
    Some((left, op, right))
}

fn main() {
    println!("===== ミニ電卓 =====");
    println!("式を入力してください（'quit' で終了）");

    let mut history: Vec<String> = Vec::new();

    loop {
        print!("\n> ");
        io::stdout().flush().unwrap();

        let mut input = String::new();
        io::stdin().read_line(&mut input).expect("入力エラー");
        let input = input.trim();

        if input == "quit" {
            println!("お疲れさまでした。{}回の計算を行いました。", history.len());
            break;
        }

        if input == "history" {
            if history.is_empty() {
                println!("計算履歴はありません。");
            } else {
                println!("--- 計算履歴 ---");
                for (i, entry) in history.iter().enumerate() {
                    println!("{}. {}", i + 1, entry);
                }
            }
            continue;
        }

        match parse_expression(input) {
            Some((left, op, right)) => {
                match calculate(left, &op, right) {
                    Ok(result) => {
                        println!("= {}", result);
                        history.push(format!("{} {} {} = {}", left, op, right, result));
                    }
                    Err(msg) => {
                        println!("エラー: {}", msg);
                    }
                }
            }
            None => {
                println!("無効な式です。「数値 演算子 数値」の形式で入力してください。");
            }
        }
    }
}
```

<summary>解答例（改良版 ─ 列挙型とクロージャを活用）</summary>

```rust
// ミニ電卓（改良版）
// 列挙型とクロージャで拡張性を高めたバージョン
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

/// コマンドの種類を表す列挙型
enum Command {
    Calculate(f64, String, f64),
    History,
    Quit,
    Invalid(String),
}

/// 履歴エントリ
struct HistoryEntry {
    expression: String,
    result: f64,
}

/// 入力をコマンドに変換します
fn parse_command(input: &str) -> Command {
    match input {
        "quit" => Command::Quit,
        "history" => Command::History,
        _ => {
            // "**" 演算子を先にチェック
            if input.contains("**") {
                let parts: Vec<&str> = input.splitn(2, "**").collect();
                if parts.len() == 2 {
                    if let (Ok(l), Ok(r)) = (
                        parts[0].trim().parse::<f64>(),
                        parts[1].trim().parse::<f64>(),
                    ) {
                        return Command::Calculate(l, "**".to_string(), r);
                    }
                }
            }

            let parts: Vec<&str> = input.split_whitespace().collect();
            if parts.len() == 3 {
                if let (Ok(l), Ok(r)) = (parts[0].parse::<f64>(), parts[2].parse::<f64>()) {
                    return Command::Calculate(l, parts[1].to_string(), r);
                }
            }
            Command::Invalid(input.to_string())
        }
    }
}

/// 計算を実行します
fn calculate(left: f64, op: &str, right: f64) -> Result<f64, String> {
    match op {
        "+"  => Ok(left + right),
        "-"  => Ok(left - right),
        "*"  => Ok(left * right),
        "/"  if right == 0.0 => Err("0で割ることはできません。".to_string()),
        "/"  => Ok(left / right),
        "%"  if right == 0.0 => Err("0で割ることはできません。".to_string()),
        "%"  => Ok(left % right),
        "**" => Ok(left.powf(right)),
        _    => Err(format!("不明な演算子: {}", op)),
    }
}

fn prompt(msg: &str) -> String {
    print!("{}", msg);
    io::stdout().flush().unwrap();
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

fn main() {
    println!("===== ミニ電卓 =====");
    println!("式を入力してください（'quit' で終了）");

    let mut history: Vec<HistoryEntry> = Vec::new();

    loop {
        let input = prompt("\n> ");

        match parse_command(&input) {
            Command::Quit => {
                println!("お疲れさまでした。{}回の計算を行いました。", history.len());
                break;
            }
            Command::History => {
                if history.is_empty() {
                    println!("計算履歴はありません。");
                } else {
                    println!("--- 計算履歴 ---");
                    for (i, entry) in history.iter().enumerate() {
                        println!("{}. {} = {}", i + 1, entry.expression, entry.result);
                    }
                }
            }
            Command::Calculate(left, op, right) => {
                match calculate(left, &op, right) {
                    Ok(result) => {
                        println!("= {}", result);
                        history.push(HistoryEntry {
                            expression: format!("{} {} {}", left, op, right),
                            result,
                        });
                    }
                    Err(msg) => println!("エラー: {}", msg),
                }
            }
            Command::Invalid(_) => {
                println!("無効な式です。「数値 演算子 数値」の形式で入力してください。");
            }
        }
    }
}
```

**初心者向けとの違い:**
- `Command` 列挙型で入力の種類を明示的に分類 → 処理の見通しが良い
- `HistoryEntry` 構造体で履歴データを構造化
- `match` のガードパターン（`"/" if right == 0.0`）でゼロ除算チェックを簡潔に記述
- `parse_command()` で入力解析ロジックを完全に分離
