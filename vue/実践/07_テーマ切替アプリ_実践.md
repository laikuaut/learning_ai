# 実践課題07：テーマ切替アプリ ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第4章（コンポーネント基礎）、第5章（Composition API）、第10章（実践パターン ─ provide/inject）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `provide` / `inject` によるグローバル状態共有、composable の作成と利用、CSS変数とリアクティブデータの連携を実践する

---

## 完成イメージ

テーマ（ライト/ダーク/カスタム色）を切り替えられるダッシュボード風UIです。

---

## 課題の要件

1. ライトモード/ダークモードの切替ボタンを設ける
2. `provide` / `inject` でテーマ情報を全コンポーネントで共有する
3. プライマリ色をカラーピッカーで変更できる
4. フォントサイズ・角丸をスライダーで調整できる
5. テーマ設定を `localStorage` に保存して永続化する
6. ダッシュボードカード、ボタンなどのUI要素に設定が反映される
7. 最低3つのコンポーネントに分割する

---

## ステップガイド

<details>
<summary>ステップ1：provide / inject でテーマを共有する</summary>

```js
// App（親コンポーネント）のsetup内
const theme = ref("light")
const primaryColor = ref("#1976d2")

provide("theme", theme)
provide("primaryColor", primaryColor)
provide("toggleTheme", () => {
  theme.value = theme.value === "light" ? "dark" : "light"
})
```

```js
// 子コンポーネントのsetup内
const theme = inject("theme")
const primaryColor = inject("primaryColor")
```

</details>

<details>
<summary>ステップ2：CSS変数をリアクティブに更新する</summary>

`watch` を使って、テーマの変更をCSS変数に反映します。

```js
watch([theme, primaryColor, fontSize], () => {
  const root = document.documentElement
  root.style.setProperty('--primary', primaryColor.value)
  root.style.setProperty('--bg', theme.value === 'dark' ? '#1a1a2e' : '#f5f5f5')
  root.style.setProperty('--card-bg', theme.value === 'dark' ? '#16213e' : '#fff')
  root.style.setProperty('--text', theme.value === 'dark' ? '#e0e0e0' : '#333')
  root.style.setProperty('--font-size', fontSize.value + 'px')
})
```

</details>

<details>
<summary>ステップ3：composable にテーマロジックを抽出する</summary>

```js
function useTheme() {
  const saved = localStorage.getItem('themeSettings')
  const defaults = { mode: 'light', primary: '#1976d2', fontSize: 16, borderRadius: 8 }
  const settings = ref(saved ? { ...defaults, ...JSON.parse(saved) } : defaults)

  watch(settings, (val) => {
    localStorage.setItem('themeSettings', JSON.stringify(val))
  }, { deep: true })

  return settings
}
```

</details>

<summary>解答例（改良版 ─ コンポーネント分割 + provide/inject 活用）</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>テーマ切替ダッシュボード（改良版）</title>
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
  <style>
    :root { --primary: #1976d2; --bg: #f5f5f5; --card: #fff; --text: #333; --sub: #888; --border: #e0e0e0; --radius: 8px; --fs: 16px; }
    * { margin: 0; padding: 0; box-sizing: border-box; transition: background 0.3s, color 0.3s, border-color 0.3s; }
    body { font-family: sans-serif; background: var(--bg); color: var(--text); font-size: var(--fs); }
    #app { max-width: 720px; margin: 0 auto; padding: 20px; }
  </style>
</head>
<body>
  <div id="app">
    <app-header></app-header>
    <theme-customizer></theme-customizer>
    <stats-grid></stats-grid>
    <button-preview></button-preview>
  </div>

  <script>
    const { createApp, ref, reactive, computed, watch, provide, inject, onMounted } = Vue

    // ========== composable: useThemeSettings ==========
    function useThemeSettings() {
      const saved = localStorage.getItem('dashTheme2')
      const defaults = { mode: 'light', primary: '#1976d2', fontSize: 16, borderRadius: 8 }
      const settings = reactive(saved ? { ...defaults, ...JSON.parse(saved) } : { ...defaults })

      function apply() {
        const d = settings.mode === 'dark'
        const r = document.documentElement
        r.style.setProperty('--primary', settings.primary)
        r.style.setProperty('--bg', d ? '#1a1a2e' : '#f5f5f5')
        r.style.setProperty('--card', d ? '#16213e' : '#fff')
        r.style.setProperty('--text', d ? '#e0e0e0' : '#333')
        r.style.setProperty('--sub', d ? '#8899aa' : '#888')
        r.style.setProperty('--border', d ? '#2a3a5a' : '#e0e0e0')
        r.style.setProperty('--radius', settings.borderRadius + 'px')
        r.style.setProperty('--fs', settings.fontSize + 'px')
      }

      apply() // 初期適用

      watch(() => ({ ...settings }), () => {
        apply()
        localStorage.setItem('dashTheme2', JSON.stringify(settings))
      }, { deep: true })

      function toggleMode() {
        settings.mode = settings.mode === 'light' ? 'dark' : 'light'
      }

      return { settings, toggleMode }
    }

    // ========== App ==========
    const app = createApp({
      setup() {
        const { settings, toggleMode } = useThemeSettings()
        provide('themeSettings', settings)
        provide('toggleMode', toggleMode)
        return {}
      }
    })

    // ========== Header ==========
    app.component('app-header', {
      setup() {
        const settings = inject('themeSettings')
        const toggleMode = inject('toggleMode')
        return { settings, toggleMode }
      },
      template: `
        <div style="display: flex; justify-content: space-between; align-items: center; padding: 16px 0; margin-bottom: 20px; border-bottom: 1px solid var(--border);">
          <h1 style="font-size: 20px;">🎨 ダッシュボード</h1>
          <button @click="toggleMode" style="background: none; border: 1px solid var(--border); border-radius: var(--radius); padding: 8px 14px; cursor: pointer; font-size: 16px; color: var(--text);">
            {{ settings.mode === 'light' ? '🌙 ダーク' : '☀️ ライト' }}
          </button>
        </div>
      `
    })

    // ========== ThemeCustomizer ==========
    app.component('theme-customizer', {
      setup() {
        const settings = inject('themeSettings')
        const presetColors = ['#1976d2', '#e91e63', '#4caf50', '#ff9800', '#9c27b0', '#00bcd4']
        return { settings, presetColors }
      },
      template: `
        <div style="background: var(--card); border-radius: var(--radius); padding: 20px; margin-bottom: 16px; box-shadow: 0 1px 4px rgba(0,0,0,0.06);">
          <div style="font-weight: bold; margin-bottom: 16px;">🎨 テーマカスタマイズ</div>

          <div style="display: flex; align-items: center; justify-content: space-between; margin-bottom: 14px;">
            <label style="font-size: 14px; color: var(--sub);">プライマリ色</label>
            <div style="display: flex; align-items: center; gap: 6px;">
              <button v-for="c in presetColors" :key="c" @click="settings.primary = c"
                :style="{ width: '24px', height: '24px', borderRadius: '50%', background: c, border: settings.primary === c ? '3px solid var(--text)' : '2px solid transparent', cursor: 'pointer' }"></button>
              <input type="color" v-model="settings.primary" style="width: 32px; height: 32px; border: none; cursor: pointer; border-radius: 4px;" />
            </div>
          </div>

          <div style="display: flex; align-items: center; justify-content: space-between; margin-bottom: 14px;">
            <label style="font-size: 14px; color: var(--sub);">フォントサイズ</label>
            <div style="display: flex; align-items: center; gap: 8px;">
              <input type="range" v-model.number="settings.fontSize" min="12" max="24" style="width: 140px;" />
              <span style="font-size: 13px; color: var(--sub); width: 45px; text-align: right;">{{ settings.fontSize }}px</span>
            </div>
          </div>

          <div style="display: flex; align-items: center; justify-content: space-between;">
            <label style="font-size: 14px; color: var(--sub);">角丸</label>
            <div style="display: flex; align-items: center; gap: 8px;">
              <input type="range" v-model.number="settings.borderRadius" min="0" max="24" style="width: 140px;" />
              <span style="font-size: 13px; color: var(--sub); width: 45px; text-align: right;">{{ settings.borderRadius }}px</span>
            </div>
          </div>
        </div>
      `
    })

    // ========== StatsGrid ==========
    app.component('stats-grid', {
      setup() {
        const settings = inject('themeSettings')
        const stats = [
          { value: 42, label: "タスク完了", icon: "📋" },
          { value: 3, label: "新着通知", icon: "🔔" },
          { value: "89%", label: "達成率", icon: "📈" },
          { value: 7, label: "メンバー", icon: "👥" },
        ]
        return { settings, stats }
      },
      template: `
        <div style="display: grid; grid-template-columns: 1fr 1fr; gap: 12px; margin-bottom: 16px;">
          <div v-for="s in stats" :key="s.label"
            style="background: var(--card); border-radius: var(--radius); padding: 20px; text-align: center; box-shadow: 0 1px 4px rgba(0,0,0,0.06);">
            <div style="font-size: 20px; margin-bottom: 4px;">{{ s.icon }}</div>
            <div :style="{ fontSize: '26px', fontWeight: 'bold', color: settings.primary }">{{ s.value }}</div>
            <div style="font-size: 13px; color: var(--sub); margin-top: 4px;">{{ s.label }}</div>
          </div>
        </div>
      `
    })

    // ========== ButtonPreview ==========
    app.component('button-preview', {
      setup() {
        const settings = inject('themeSettings')
        const buttons = [
          { label: "プライマリ", useTheme: true },
          { label: "成功", color: "#4caf50" },
          { label: "警告", color: "#ff9800" },
          { label: "エラー", color: "#f44336" },
        ]
        return { settings, buttons }
      },
      template: `
        <div style="background: var(--card); border-radius: var(--radius); padding: 20px; box-shadow: 0 1px 4px rgba(0,0,0,0.06);">
          <div style="font-weight: bold; margin-bottom: 14px;">ボタンプレビュー</div>
          <div style="display: flex; gap: 8px; flex-wrap: wrap;">
            <button v-for="b in buttons" :key="b.label"
              :style="{
                padding: '10px 22px',
                border: 'none',
                borderRadius: settings.borderRadius + 'px',
                background: b.useTheme ? settings.primary : b.color,
                color: 'white',
                cursor: 'pointer',
                fontWeight: 'bold',
                fontSize: '14px'
              }">{{ b.label }}</button>
          </div>
        </div>
      `
    })

    app.mount('#app')
  </script>
</body>
</html>
```

**初心者向けとの違い:**
- `useThemeSettings` composable にテーマロジックを集約し、再利用可能に
- 4つのコンポーネント（`app-header`、`theme-customizer`、`stats-grid`、`button-preview`）に分離
- `provide` / `inject` で全コンポーネントからテーマ設定にアクセス
- プリセットカラーパレットをクリックで選択可能に
- `reactive` でテーマ設定をオブジェクトとして管理し、`watch` で一括監視
