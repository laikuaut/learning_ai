## 基本問題

### 問題1：img要素の基本
以下の条件で画像を表示するHTMLを書いてください。

- 画像ファイル：`images/photo.jpg`
- 代替テキスト：「桜並木の風景写真」
- 表示サイズ：幅640px、高さ480px

<details>
<summary>解答例</summary>

```html
<img src="images/photo.jpg" alt="桜並木の風景写真" width="640" height="480">
```

width と height を指定すると、ブラウザが事前にスペースを確保し、レイアウトシフト（CLS）を防げます。

</details>

---

### 問題2：alt属性の判断
以下の画像に対して、適切な alt 属性の値を考えてください。

1. 企業ロゴ（会社名：ABC株式会社）
2. ページ装飾用の区切り線画像
3. 「検索」ボタンの虫眼鏡アイコン画像
4. 売上推移を示すグラフ（2024年Q1: 100万, Q2: 150万, Q3: 200万）

<details>
<summary>解答例</summary>

1. `alt="ABC株式会社"` — ロゴは会社名・サービス名を記述
2. `alt=""` — 装飾画像は空の alt を指定
3. `alt="検索"` — ボタン画像はボタンの機能を記述
4. `alt="2024年売上推移グラフ。Q1は100万円、Q2は150万円、Q3は200万円と増加傾向"` — データ画像は要約を記述

</details>

### 問題3：figure / figcaption
画像とキャプションをセットにして表示するHTMLを書いてください。

- 画像：`sunset.jpg`、代替テキスト「海に沈む夕日」
- キャプション：「2024年8月、湘南海岸にて撮影」

<details>
<summary>解答例</summary>

```html
<figure>
  <img src="sunset.jpg" alt="海に沈む夕日" width="800" height="600">
  <figcaption>2024年8月、湘南海岸にて撮影</figcaption>
</figure>
```

</details>

### 問題4：video要素
以下の条件で動画を埋め込むHTMLを書いてください。

- 動画ファイル：`video/tutorial.mp4`
- 再生コントロールを表示
- サムネイル画像：`video/thumbnail.jpg`
- サイズ：幅640、高さ360
- ブラウザ非対応時は「お使いのブラウザは動画に対応していません」と表示

<details>
<summary>解答例</summary>

```html
<video src="video/tutorial.mp4" controls
       poster="video/thumbnail.jpg"
       width="640" height="360">
  <p>お使いのブラウザは動画に対応していません。</p>
</video>
```

</details>

### 問題5：loading属性の判断
以下の画像のうち、`loading="lazy"` を付けるべきものと付けるべきでないものを分類してください。

A. ページ先頭のヒーロー画像
B. ページ中盤のギャラリー画像（10枚）
C. フッター付近のバナー画像
D. ヘッダーのサイトロゴ

<details>
<summary>解答</summary>

**`loading="lazy"` を付けるべき**：B（ギャラリー画像）、C（フッターバナー）

**付けるべきでない**：A（ヒーロー画像）、D（サイトロゴ）

ファーストビュー（画面上部、スクロールせずに見える領域）の画像には `loading="lazy"` を付けると LCP（Largest Contentful Paint）が遅くなるため、付けません。

</details>

## 応用問題

### 問題6：picture要素によるレスポンシブ画像
以下の要件を満たす picture 要素を書いてください。

- デスクトップ（幅1024px以上）：`hero-large.jpg`
- タブレット（幅768px以上）：`hero-medium.jpg`
- スマートフォン（それ以外）：`hero-small.jpg`
- 代替テキスト：「商品のメインビジュアル」

<details>
<summary>解答例</summary>

```html
<picture>
  <source media="(min-width: 1024px)" srcset="hero-large.jpg">
  <source media="(min-width: 768px)" srcset="hero-medium.jpg">
  <img src="hero-small.jpg" alt="商品のメインビジュアル">
</picture>
```

ブラウザは `<source>` を上から順に評価し、条件に合う最初のものを使用します。どの条件にも合わない場合は `<img>` がフォールバックです。

</details>

### 問題7：フォーマット切り替え
モダンブラウザには WebP、非対応ブラウザには JPEG を提供するHTMLを書いてください。

- WebPファイル：`photo.webp`
- JPEGファイル：`photo.jpg`
- 代替テキスト：「チームの集合写真」

<details>
<summary>解答例</summary>

```html
<picture>
  <source srcset="photo.webp" type="image/webp">
  <img src="photo.jpg" alt="チームの集合写真" width="800" height="600">
</picture>
```

</details>

## チャレンジ問題

### 問題8：メディアギャラリーページの作成
以下の要件を満たす完全なHTMLページを作成してください。

- ページタイトル「フォトギャラリー」
- h1 で「フォトギャラリー」
- 3枚の画像を figure + figcaption で配置（画像ファイル名は自由、alt属性は具体的に）
- すべての画像に width / height と `loading="lazy"` を指定
- 1つの動画を video 要素で配置（controls、poster 付き）
- audio 要素で BGM を配置（controls 付き、loop で自動ループ）

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>フォトギャラリー</title>
</head>
<body>
  <h1>フォトギャラリー</h1>

  <h2>写真</h2>
  <figure>
    <img src="images/mountain.jpg" alt="雪をかぶった富士山の全景"
         width="800" height="600" loading="lazy">
    <figcaption>富士山（2024年1月撮影）</figcaption>
  </figure>

  <figure>
    <img src="images/ocean.jpg" alt="青い海と白い砂浜が広がる沖縄のビーチ"
         width="800" height="600" loading="lazy">
    <figcaption>沖縄のビーチ（2024年7月撮影）</figcaption>
  </figure>

  <figure>
    <img src="images/city.jpg" alt="夜の渋谷スクランブル交差点、ネオンが輝いている"
         width="800" height="600" loading="lazy">
    <figcaption>渋谷の夜景（2024年12月撮影）</figcaption>
  </figure>

  <h2>動画</h2>
  <video controls poster="images/video-thumb.jpg" width="640" height="360">
    <source src="videos/travel.webm" type="video/webm">
    <source src="videos/travel.mp4" type="video/mp4">
    <p>お使いのブラウザは動画再生に対応していません。</p>
  </video>

  <h2>BGM</h2>
  <audio controls loop>
    <source src="audio/bgm.ogg" type="audio/ogg">
    <source src="audio/bgm.mp3" type="audio/mpeg">
    <p>お使いのブラウザは音声再生に対応していません。</p>
  </audio>
</body>
</html>
```

</details>
