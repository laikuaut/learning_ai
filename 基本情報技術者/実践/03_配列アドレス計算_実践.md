## 課題パート1：手計算問題

### 基本問題（1次元配列）

**Q1.** 1次元配列 A[0]〜A[99] がある。先頭アドレスが 200、要素サイズが 2 バイトのとき、A[30] のアドレスを求めよ。

**Q2.** 1次元配列の先頭アドレスが 500、要素サイズが 8 バイトで、A[i] のアドレスが 564 のとき、i の値を求めよ。

### 応用問題（2次元配列）

**Q3.** 2次元配列 A[5][4]（5行4列）、先頭アドレス 1000、要素サイズ 4 バイト、**行優先（row-major）** のとき、A[3][2] のアドレスを求めよ。

**Q4.** 同じ配列が**列優先（column-major）** で格納されている場合、A[3][2] のアドレスを求めよ。

**Q5.** 2次元配列 A[M][N] の先頭アドレスが a、要素サイズが s のとき、行優先で A[i][j] のアドレスを求める一般公式を書け。

### チャレンジ問題

**Q6.** 添字が0始まりではなく、A[2..6][3..7]（行の添字が2〜6、列の添字が3〜7）の場合、先頭アドレス 100、要素サイズ 2 バイト、行優先で A[4][5] のアドレスを求めよ。

> **ヒント：** 添字のオフセットを引いてから公式に代入します。
> A[4][5] → 行方向: 4-2=2、列方向: 5-3=2、列数: 7-3+1=5

## 課題パート2：アドレス計算シミュレータを作る

### 要件

1. 先頭アドレス、要素サイズ、行数、列数を入力で受け取る
2. 行優先・列優先を選択できる
3. 全要素のアドレス一覧を表形式で表示する
4. 指定した要素のアドレスを計算して確認できる

---

## ステップガイド

<details>
<summary>ステップ1：1次元配列のアドレス公式を確認する</summary>

```
要素 A[i] のアドレス = 先頭アドレス + i × 要素サイズ
```

例：先頭アドレス 200、要素サイズ 2 のとき
```
A[30] = 200 + 30 × 2 = 260
```

</details>

<details>
<summary>ステップ2：2次元配列のアドレス公式を確認する</summary>

**行優先（row-major）:**
```
A[i][j] = 先頭アドレス + (i × 列数 + j) × 要素サイズ
```

**列優先（column-major）:**
```
A[i][j] = 先頭アドレス + (j × 行数 + i) × 要素サイズ
```

</details>

## 解答例

<details>
<summary>手計算問題の解答</summary>

**Q1.** `200 + 30 × 2 = **260**`

**Q2.** `564 = 500 + i × 8` → `i = (564 - 500) / 8 = **8**`

**Q3.** 行優先: `1000 + (3 × 4 + 2) × 4 = 1000 + 14 × 4 = **1056**`

**Q4.** 列優先: `1000 + (2 × 5 + 3) × 4 = 1000 + 13 × 4 = **1052**`

**Q5.** 一般公式:
- 行優先: `A[i][j] = a + (i × N + j) × s`
- 列優先: `A[i][j] = a + (j × M + i) × s`

**Q6.** 添字をオフセットで調整:
- 実効行: 4 - 2 = 2
- 実効列: 5 - 3 = 2
- 列数: 7 - 3 + 1 = 5
- `100 + (2 × 5 + 2) × 2 = 100 + 12 × 2 = **124**`

</details>

```python
# 配列アドレス計算シミュレータ
# 学べる内容：配列のメモリ配置、アドレス計算公式
# 実行方法：python 03_array_address.py

print("===== 配列アドレス計算シミュレータ =====\n")

# --- 設定入力 ---
base = int(input("先頭アドレス: "))
size = int(input("要素サイズ (バイト): "))
rows = int(input("行数: "))
cols = int(input("列数: "))

print("\n格納方式を選んでください:")
print("  1. 行優先 (row-major)")
print("  2. 列優先 (column-major)")
mode = int(input("選択 (1-2): "))

# --- アドレス一覧表示 ---
print(f"\n--- アドレス一覧 ---")

for i in range(rows):
    line = ""
    for j in range(cols):
        if mode == 1:
            # 行優先
            addr = base + (i * cols + j) * size
        else:
            # 列優先
            addr = base + (j * rows + i) * size
        line += f"A[{i}][{j}]={addr:<8}"
    print(line)

# --- クイズ ---
print("\n--- アドレス計算クイズ ---")
qi = int(input("行番号 i を入力: "))
qj = int(input("列番号 j を入力: "))

if mode == 1:
    correct = base + (qi * cols + qj) * size
    formula = f"{base} + ({qi}×{cols} + {qj}) × {size}"
else:
    correct = base + (qj * rows + qi) * size
    formula = f"{base} + ({qj}×{rows} + {qi}) × {size}"

user_answer = int(input(f"A[{qi}][{qj}] のアドレスは？: "))

if user_answer == correct:
    print(f"✓ 正解！ {formula} = {correct}")
else:
    print(f"✗ 不正解... 正解は {correct} です。")
    print(f"  計算式: {formula} = {correct}")
```

<summary>解答例（改良版 ─ 添字オフセット対応・連続クイズ）</summary>

```python
# 配列アドレス計算シミュレータ（改良版）
# 添字の開始値を指定可能、連続クイズ付き

import random


def calc_address(base, size, rows, cols, i, j, row_major=True):
    """2次元配列のアドレスを計算する"""
    if row_major:
        return base + (i * cols + j) * size
    else:
        return base + (j * rows + i) * size


def show_address_table(base, size, rows, cols, row_start, col_start, row_major):
    """アドレス一覧を表形式で表示する"""
    mode_name = "行優先" if row_major else "列優先"
    print(f"\n--- アドレス一覧（{mode_name}） ---")

    for i in range(rows):
        parts = []
        for j in range(cols):
            addr = calc_address(base, size, rows, cols, i, j, row_major)
            label = f"A[{i + row_start}][{j + col_start}]"
            parts.append(f"{label}={addr:<6}")
        print("  ".join(parts))


def quiz_mode(base, size, rows, cols, row_start, col_start, row_major):
    """ランダムにアドレスを問うクイズ"""
    correct_count = 0
    total = 5

    print(f"\n===== アドレス計算クイズ（全{total}問） =====")

    for q in range(1, total + 1):
        i = random.randint(0, rows - 1)
        j = random.randint(0, cols - 1)
        display_i = i + row_start
        display_j = j + col_start
        correct = calc_address(base, size, rows, cols, i, j, row_major)

        print(f"\n【問題{q}】A[{display_i}][{display_j}] のアドレスは？")

        try:
            answer = int(input("  回答: "))
        except ValueError:
            print("  数値を入力してください。")
            continue

        if answer == correct:
            print(f"  ✓ 正解！")
            correct_count += 1
        else:
            if row_major:
                formula = f"{base} + ({i}×{cols} + {j}) × {size}"
            else:
                formula = f"{base} + ({j}×{rows} + {i}) × {size}"
            print(f"  ✗ 不正解... 正解は {correct}")
            print(f"    計算式: {formula} = {correct}")

    rate = int(correct_count / total * 100)
    print(f"\n結果: {total}問中 {correct_count}問正解（{rate}%）")


def main():
    print("===== 配列アドレス計算シミュレータ =====\n")

    base = int(input("先頭アドレス (例: 1000): "))
    size = int(input("要素サイズ [バイト] (例: 4): "))
    rows = int(input("行数 (例: 3): "))
    cols = int(input("列数 (例: 4): "))

    print("\n添字の開始値（通常は0）:")
    row_start = int(input("  行の開始値 (例: 0): "))
    col_start = int(input("  列の開始値 (例: 0): "))

    print("\n格納方式:")
    print("  1. 行優先 (row-major)")
    print("  2. 列優先 (column-major)")
    row_major = int(input("選択 (1-2): ")) == 1

    # アドレス一覧を表示
    show_address_table(base, size, rows, cols, row_start, col_start, row_major)

    # クイズモード
    quiz = input("\nクイズに挑戦しますか？ (y/n): ")
    if quiz.lower() == "y":
        quiz_mode(base, size, rows, cols, row_start, col_start, row_major)

    print("\nお疲れさまでした！")


if __name__ == "__main__":
    main()
```

**初心者向けとの違い:**
- 添字の開始値を指定可能 → A[2..6][3..7] のような試験問題にも対応
- アドレス計算を関数化 → 再利用しやすい設計
- クイズモードでランダム出題 → 繰り返し練習が可能
- 不正解時に計算式を表示 → どこで間違えたかがわかる
