## 基本問題

### 問題1：自己紹介プログラム

名前（String）、年齢（int）、身長（double）、Javaが好きか（boolean）を変数に格納し、自己紹介文として出力するプログラムを作成してください。

**期待される出力例：**
```
===== 自己紹介 =====
名前：田中太郎
年齢：25歳
身長：170.5cm
Javaが好き：はい
====================
```

<details><summary>ヒント</summary>

- `String` は文字列型、`int` は整数型、`double` は小数型、`boolean` は真偽値型です
- `boolean` の値を「はい」「いいえ」に変換するには三項演算子 `条件 ? 真の値 : 偽の値` が使えます
- `System.out.println()` で文字列と変数を `+` で連結して出力します

</details>

<details><summary>解答例</summary>

```java
public class SelfIntroduction {
    public static void main(String[] args) {
        // 各データ型の変数を宣言・初期化
        String name = "田中太郎";       // 文字列型
        int age = 25;                   // 整数型
        double height = 170.5;          // 小数型（倍精度浮動小数点）
        boolean likesJava = true;       // 真偽値型

        // 自己紹介を出力
        System.out.println("===== 自己紹介 =====");
        System.out.println("名前：" + name);
        System.out.println("年齢：" + age + "歳");
        System.out.println("身長：" + height + "cm");
        // 三項演算子でbooleanを日本語に変換
        System.out.println("Javaが好き：" + (likesJava ? "はい" : "いいえ"));
        System.out.println("====================");
    }
}
```

</details>

### 問題2：四則演算電卓

2つの整数値を変数に格納し、加算・減算・乗算・除算・剰余の5つの計算結果を表示するプログラムを作成してください。除算では整数同士の除算と `double` にキャストした場合の両方を表示してください。

**期待される出力例：**
```
a = 17, b = 5
加算：17 + 5 = 22
減算：17 - 5 = 12
乗算：17 * 5 = 85
除算（整数）：17 / 5 = 3
除算（小数）：17 / 5 = 3.4
剰余：17 % 5 = 2
```

<details><summary>ヒント</summary>

- 整数同士の除算は小数部分が切り捨てられます（例：`17 / 5` → `3`）
- 小数の結果を得るには、片方を `double` にキャストします（例：`(double) a / b`）
- `%` 演算子で剰余（余り）を求められます

</details>

<details><summary>解答例</summary>

```java
public class ArithmeticCalculator {
    public static void main(String[] args) {
        int a = 17;
        int b = 5;

        System.out.println("a = " + a + ", b = " + b);

        // 加算
        System.out.println("加算：" + a + " + " + b + " = " + (a + b));

        // 減算
        System.out.println("減算：" + a + " - " + b + " = " + (a - b));

        // 乗算
        System.out.println("乗算：" + a + " * " + b + " = " + (a * b));

        // 除算（整数同士 → 小数部分が切り捨てられる）
        System.out.println("除算（整数）：" + a + " / " + b + " = " + (a / b));

        // 除算（キャストして小数結果を取得）
        System.out.println("除算（小数）：" + a + " / " + b + " = " + ((double) a / b));

        // 剰余（余り）
        System.out.println("剰余：" + a + " % " + b + " = " + (a % b));
    }
}
```

</details>

### 問題3：摂氏から華氏への温度変換

摂氏温度（`double`）を変数に格納し、華氏温度に変換して表示するプログラムを作成してください。変換式は `華氏 = 摂氏 × 9 / 5 + 32` です。小数第1位まで表示してください。

**期待される出力例：**
```
=== 温度変換 ===
摂氏：36.5℃
華氏：97.7°F
```

<details><summary>ヒント</summary>

- `String.format("%.1f", 値)` で小数第1位まで表示できます
- `System.out.printf()` を使う方法もあります
- 計算式のかっこの位置に注意しましょう

</details>

<details><summary>解答例</summary>

```java
public class TemperatureConverter {
    public static void main(String[] args) {
        // 摂氏温度を格納
        double celsius = 36.5;

        // 華氏に変換：F = C × 9/5 + 32
        double fahrenheit = celsius * 9.0 / 5.0 + 32.0;

        System.out.println("=== 温度変換 ===");
        System.out.println("摂氏：" + celsius + "℃");
        // String.formatで小数第1位まで表示
        System.out.println("華氏：" + String.format("%.1f", fahrenheit) + "°F");
    }
}
```

</details>

### 問題4：BMI計算プログラム

身長（cm）と体重（kg）を変数に格納し、BMI値を計算して判定結果を表示するプログラムを作成してください。BMIの計算式は `BMI = 体重(kg) / (身長(m))²` です。BMI値は小数第2位まで表示してください。

判定基準：
- 18.5未満：低体重
- 18.5以上25.0未満：普通体重
- 25.0以上30.0未満：肥満（1度）
- 30.0以上：肥満（2度以上）

**期待される出力例：**
```
===== BMI計算 =====
身長：170.0cm
体重：65.0kg
BMI：22.49
判定：普通体重
===================
```

<details><summary>ヒント</summary>

- 身長をcmからmに変換するには `/ 100.0` します
- べき乗は `Math.pow(値, 2)` または単純に `値 * 値` で計算できます
- 判定にはif-else文を使います（第2章の内容ですが、ここでは簡単なif文で挑戦してみましょう）
- `String.format("%.2f", 値)` で小数第2位まで表示できます

</details>

<details><summary>解答例</summary>

```java
public class BmiCalculator {
    public static void main(String[] args) {
        // 身長（cm）と体重（kg）
        double heightCm = 170.0;
        double weightKg = 65.0;

        // 身長をメートルに変換
        double heightM = heightCm / 100.0;

        // BMI計算：体重(kg) / 身長(m)の2乗
        double bmi = weightKg / (heightM * heightM);

        // 判定
        String result;
        if (bmi < 18.5) {
            result = "低体重";
        } else if (bmi < 25.0) {
            result = "普通体重";
        } else if (bmi < 30.0) {
            result = "肥満（1度）";
        } else {
            result = "肥満（2度以上）";
        }

        // 結果表示
        System.out.println("===== BMI計算 =====");
        System.out.println("身長：" + heightCm + "cm");
        System.out.println("体重：" + weightKg + "kg");
        System.out.println("BMI：" + String.format("%.2f", bmi));
        System.out.println("判定：" + result);
        System.out.println("===================");
    }
}
```

</details>

### 問題5：通貨変換プログラム

日本円の金額を変数に格納し、米ドル（USD）、ユーロ（EUR）、英ポンド（GBP）に変換して表示するプログラムを作成してください。各通貨の為替レートも変数として定義してください。結果は小数第2位まで表示してください。

**期待される出力例：**
```
====== 通貨変換 ======
元の金額：10,000円

為替レート：
  1 USD = 150.00 円
  1 EUR = 163.00 円
  1 GBP = 190.00 円

変換結果：
  10,000円 → 66.67 USD
  10,000円 → 61.35 EUR
  10,000円 → 52.63 GBP
======================
```

<details><summary>ヒント</summary>

- 通貨変換は `外貨 = 日本円 / 為替レート` で計算します
- `String.format("%,.0f", 値)` でカンマ区切りの整数表示ができます
- `String.format("%.2f", 値)` で小数第2位まで表示できます

</details>

<details><summary>解答例</summary>

```java
public class CurrencyConverter {
    public static void main(String[] args) {
        // 日本円の金額
        int yen = 10000;

        // 為替レート（1外貨あたりの円）
        double rateUsd = 150.00;  // 米ドル
        double rateEur = 163.00;  // ユーロ
        double rateGbp = 190.00;  // 英ポンド

        // 通貨変換
        double usd = yen / rateUsd;
        double eur = yen / rateEur;
        double gbp = yen / rateGbp;

        // 結果表示
        System.out.println("====== 通貨変換 ======");
        System.out.println("元の金額：" + String.format("%,d", yen) + "円");
        System.out.println();
        System.out.println("為替レート：");
        System.out.printf("  1 USD = %.2f 円%n", rateUsd);
        System.out.printf("  1 EUR = %.2f 円%n", rateEur);
        System.out.printf("  1 GBP = %.2f 円%n", rateGbp);
        System.out.println();
        System.out.println("変換結果：");
        System.out.printf("  %,d円 → %.2f USD%n", yen, usd);
        System.out.printf("  %,d円 → %.2f EUR%n", yen, eur);
        System.out.printf("  %,d円 → %.2f GBP%n", yen, gbp);
        System.out.println("======================");
    }
}
```

</details>

### 問題6：データ型の範囲エクスプローラー

Javaの主要な基本データ型（`byte`, `short`, `int`, `long`, `float`, `double`, `char`）の最小値・最大値・サイズ（バイト数）を表示するプログラムを作成してください。各ラッパークラスの `MIN_VALUE`, `MAX_VALUE`, `BYTES` 定数を活用してください。

**期待される出力例：**
```
===== Javaデータ型の範囲 =====
型       | サイズ | 最小値                     | 最大値
---------|--------|---------------------------|---------------------------
byte     | 1 byte | -128                      | 127
short    | 2 byte | -32768                    | 32767
int      | 4 byte | -2147483648               | 2147483647
long     | 8 byte | -9223372036854775808      | 9223372036854775807
float    | 4 byte | 1.4E-45                   | 3.4028235E38
double   | 8 byte | 4.9E-324                  | 1.7976931348623157E308
char     | 2 byte | 0                         | 65535
================================
```

<details><summary>ヒント</summary>

- 各ラッパークラス（`Byte`, `Short`, `Integer`, `Long`, `Float`, `Double`, `Character`）に定数が定義されています
- `Integer.MIN_VALUE`, `Integer.MAX_VALUE`, `Integer.BYTES` のように参照します
- `Character` の場合、`MIN_VALUE` と `MAX_VALUE` は `char` 型なので、整数として表示するには `(int)` キャストが必要です
- `printf` を使うと桁揃えが便利です

</details>

<details><summary>解答例</summary>

```java
public class DataTypeExplorer {
    public static void main(String[] args) {
        System.out.println("===== Javaデータ型の範囲 =====");
        // ヘッダー
        System.out.printf("%-8s | %-6s | %-27s | %s%n",
                "型", "サイズ", "最小値", "最大値");
        System.out.println("---------+--------+-----------------------------+---------------------------");

        // byte: 1バイト整数
        System.out.printf("%-8s | %d byte | %-27d | %d%n",
                "byte", Byte.BYTES, Byte.MIN_VALUE, Byte.MAX_VALUE);

        // short: 2バイト整数
        System.out.printf("%-8s | %d byte | %-27d | %d%n",
                "short", Short.BYTES, Short.MIN_VALUE, Short.MAX_VALUE);

        // int: 4バイト整数
        System.out.printf("%-8s | %d byte | %-27d | %d%n",
                "int", Integer.BYTES, Integer.MIN_VALUE, Integer.MAX_VALUE);

        // long: 8バイト整数
        System.out.printf("%-8s | %d byte | %-27d | %d%n",
                "long", Long.BYTES, Long.MIN_VALUE, Long.MAX_VALUE);

        // float: 4バイト浮動小数点（MIN_VALUEは最小の正の値）
        System.out.printf("%-8s | %d byte | %-27s | %s%n",
                "float", Float.BYTES, Float.MIN_VALUE, Float.MAX_VALUE);

        // double: 8バイト浮動小数点（MIN_VALUEは最小の正の値）
        System.out.printf("%-8s | %d byte | %-27s | %s%n",
                "double", Double.BYTES, Double.MIN_VALUE, Double.MAX_VALUE);

        // char: 2バイト文字（Unicode）
        // charのMIN_VALUE/MAX_VALUEはchar型なので、intにキャストして数値表示
        System.out.printf("%-8s | %d byte | %-27d | %d%n",
                "char", Character.BYTES, (int) Character.MIN_VALUE, (int) Character.MAX_VALUE);

        System.out.println("================================");
    }
}
```

</details>

### 問題7：数値の桁分析プログラム

5桁の整数を変数に格納し、以下の情報を表示するプログラムを作成してください。
- 各桁の数字（万の位〜一の位）
- 全桁の合計
- 桁を逆順にした数値

**期待される出力例：**
```
===== 桁分析 =====
元の数値：12345

各桁の数字：
  万の位：1
  千の位：2
  百の位：3
  十の位：4
  一の位：5

全桁の合計：15
逆順の数値：54321
==================
```

<details><summary>ヒント</summary>

- 各桁の取り出しには `/`（除算）と `%`（剰余）を組み合わせます
- 万の位：`number / 10000`
- 千の位：`number / 1000 % 10`
- 百の位：`number / 100 % 10`
- 逆順の数値は各桁に位の重み（10000, 1000, ...）を掛けて足し合わせます

</details>

<details><summary>解答例</summary>

```java
public class DigitAnalyzer {
    public static void main(String[] args) {
        // 分析対象の5桁の整数
        int number = 12345;

        // 各桁を取り出す（除算と剰余を使用）
        int digit1 = number / 10000;       // 万の位
        int digit2 = number / 1000 % 10;   // 千の位
        int digit3 = number / 100 % 10;    // 百の位
        int digit4 = number / 10 % 10;     // 十の位
        int digit5 = number % 10;          // 一の位

        // 全桁の合計
        int sum = digit1 + digit2 + digit3 + digit4 + digit5;

        // 桁を逆順にした数値
        int reversed = digit5 * 10000 + digit4 * 1000 + digit3 * 100 + digit2 * 10 + digit1;

        // 結果表示
        System.out.println("===== 桁分析 =====");
        System.out.println("元の数値：" + number);
        System.out.println();
        System.out.println("各桁の数字：");
        System.out.println("  万の位：" + digit1);
        System.out.println("  千の位：" + digit2);
        System.out.println("  百の位：" + digit3);
        System.out.println("  十の位：" + digit4);
        System.out.println("  一の位：" + digit5);
        System.out.println();
        System.out.println("全桁の合計：" + sum);
        System.out.println("逆順の数値：" + reversed);
        System.out.println("==================");
    }
}
```

</details>
