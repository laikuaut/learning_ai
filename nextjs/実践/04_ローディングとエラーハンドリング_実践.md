# 実践課題04：ローディングとエラーハンドリング ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第2章（App Routerとルーティング）、第4章（データフェッチング）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `loading.tsx`、`error.tsx`、`not-found.tsx` の使い方と、Suspense を活用したストリーミングレンダリングを実践する

---

## 完成イメージ

ユーザー情報を外部APIから取得して表示するページです。ローディング中、エラー発生時、ユーザーが見つからない場合のそれぞれで適切なUIを表示します。

```
【ローディング中】               【正常表示】
┌─────────────────┐      ┌─────────────────┐
│  ユーザー情報     │      │  ユーザー情報     │
│                  │      │                  │
│  ░░░░░░░░░░░░░  │      │  田中太郎        │
│  ░░░░░░░░       │  →   │  tanaka@mail.com │
│  ░░░░░░░░░░░    │      │  東京都           │
│                  │      │  投稿数: 42      │
└─────────────────┘      └─────────────────┘

【エラー発生時】               【404】
┌─────────────────┐      ┌─────────────────┐
│                  │      │                  │
│  ⚠ エラーが       │      │  ユーザーが       │
│  発生しました     │      │  見つかりません    │
│                  │      │                  │
│  [再試行]        │      │  [一覧に戻る]     │
└─────────────────┘      └─────────────────┘
```

---

## 課題の要件

1. `app/users/page.tsx` にユーザー一覧を表示する
2. `app/users/[id]/page.tsx` でユーザー詳細を表示する
3. `app/users/loading.tsx` でローディングUI（スケルトン）を実装する
4. `app/users/[id]/error.tsx` でエラーUIを実装する（再試行ボタン付き）
5. `app/users/[id]/not-found.tsx` で404UIを実装する
6. `fetch` で JSONPlaceholder API（`https://jsonplaceholder.typicode.com/users`）からデータを取得する

<details>
<summary>ステップ1：ファイル構造を理解する</summary>

```
app/users/
├── page.tsx          → ユーザー一覧（/users）
├── loading.tsx       → 一覧のローディングUI
└── [id]/
    ├── page.tsx      → ユーザー詳細（/users/1 など）
    ├── loading.tsx   → 詳細のローディングUI
    ├── error.tsx     → エラーUI
    └── not-found.tsx → 404UI
```

Next.js が自動的に以下のようにラップします。

```tsx
<Layout>
  <ErrorBoundary fallback={<Error />}>
    <Suspense fallback={<Loading />}>
      <Page />
    </Suspense>
  </ErrorBoundary>
</Layout>
```

### app/users/page.tsx

```tsx
import Link from 'next/link'

type User = {
  id: number
  name: string
  email: string
}

async function getUsers(): Promise<User[]> {
  const res = await fetch('https://jsonplaceholder.typicode.com/users', {
    cache: 'no-store', // 常に最新データを取得
  })

  if (!res.ok) {
    throw new Error('ユーザー一覧の取得に失敗しました')
  }

  return res.json()
}

export default async function UsersPage() {
  const users = await getUsers()

  return (
    <div style={{ maxWidth: '600px', margin: '0 auto', padding: '24px' }}>
      <h1>ユーザー一覧</h1>
      <div style={{ display: 'flex', flexDirection: 'column', gap: '12px' }}>
        {users.map(function (user) {
          return (
            <Link
              key={user.id}
              href={`/users/${user.id}`}
              style={{
                display: 'block',
                padding: '16px',
                border: '1px solid #e2e8f0',
                borderRadius: '8px',
                textDecoration: 'none',
                color: '#1e293b',
              }}
            >
              <strong>{user.name}</strong>
              <div style={{ color: '#64748b', fontSize: '14px' }}>
                {user.email}
              </div>
            </Link>
          )
        })}
      </div>
    </div>
  )
}
```

<summary>解答例（改良版 ─ Suspense + パラレルデータ取得）</summary>

### app/users/[id]/page.tsx

```tsx
import { Suspense } from 'react'
import Link from 'next/link'
import { notFound } from 'next/navigation'

type UserDetail = {
  id: number
  name: string
  email: string
  phone: string
  website: string
  company: { name: string }
  address: { city: string }
}

type Post = {
  id: number
  title: string
}

type Todo = {
  id: number
  title: string
  completed: boolean
}

async function getUser(id: string): Promise<UserDetail | null> {
  const res = await fetch(`https://jsonplaceholder.typicode.com/users/${id}`)
  if (res.status === 404) return null
  if (!res.ok) throw new Error('ユーザー情報の取得に失敗しました')
  return res.json()
}

async function getUserPosts(userId: string): Promise<Post[]> {
  const res = await fetch(
    `https://jsonplaceholder.typicode.com/posts?userId=${userId}`
  )
  if (!res.ok) throw new Error('投稿の取得に失敗しました')
  return res.json()
}

async function getUserTodos(userId: string): Promise<Todo[]> {
  const res = await fetch(
    `https://jsonplaceholder.typicode.com/todos?userId=${userId}`
  )
  if (!res.ok) throw new Error('TODOの取得に失敗しました')
  return res.json()
}

// 投稿一覧セクション（個別にSuspenseでラップ）
async function UserPosts({ userId }: { userId: string }) {
  const posts = await getUserPosts(userId)

  return (
    <section>
      <h2>投稿 ({posts.length}件)</h2>
      <ul style={{ listStyle: 'none', padding: 0 }}>
        {posts.slice(0, 5).map((post) => (
          <li
            key={post.id}
            style={{
              padding: '8px 0',
              borderBottom: '1px solid #f1f5f9',
            }}
          >
            {post.title}
          </li>
        ))}
      </ul>
      {posts.length > 5 && (
        <p style={{ color: '#64748b', fontSize: '14px' }}>
          他 {posts.length - 5} 件
        </p>
      )}
    </section>
  )
}

// TODO一覧セクション
async function UserTodos({ userId }: { userId: string }) {
  const todos = await getUserTodos(userId)
  const completed = todos.filter((t) => t.completed).length

  return (
    <section>
      <h2>TODO ({completed}/{todos.length} 完了)</h2>
      <div
        style={{
          height: '8px',
          backgroundColor: '#e2e8f0',
          borderRadius: '4px',
          overflow: 'hidden',
        }}
      >
        <div
          style={{
            height: '100%',
            width: `${(completed / todos.length) * 100}%`,
            backgroundColor: '#22c55e',
            borderRadius: '4px',
          }}
        />
      </div>
    </section>
  )
}

// ローディング用のスケルトン
function SectionSkeleton() {
  return (
    <div style={{ padding: '16px 0' }}>
      <div
        style={{
          height: '24px',
          width: '150px',
          backgroundColor: '#e2e8f0',
          borderRadius: '4px',
          marginBottom: '12px',
        }}
      />
      {[1, 2, 3].map((i) => (
        <div
          key={i}
          style={{
            height: '16px',
            width: `${70 + i * 5}%`,
            backgroundColor: '#f1f5f9',
            borderRadius: '4px',
            marginBottom: '8px',
          }}
        />
      ))}
    </div>
  )
}

export default async function UserDetailPage({
  params,
}: {
  params: Promise<{ id: string }>
}) {
  const { id } = await params
  const user = await getUser(id)

  if (!user) {
    notFound()
  }

  return (
    <div style={{ maxWidth: '600px', margin: '0 auto', padding: '24px' }}>
      <Link href="/users" style={{ color: '#2563eb', textDecoration: 'none' }}>
        ← ユーザー一覧に戻る
      </Link>

      <div style={{ marginTop: '24px' }}>
        <h1>{user.name}</h1>
        <p style={{ color: '#64748b' }}>{user.email} / {user.address.city}</p>
      </div>

      {/* 各セクションを個別にSuspenseでラップ — 独立してストリーミング */}
      <Suspense fallback={<SectionSkeleton />}>
        <UserPosts userId={id} />
      </Suspense>

      <Suspense fallback={<SectionSkeleton />}>
        <UserTodos userId={id} />
      </Suspense>
    </div>
  )
}
```

**初心者向けとの違い:**
- `Suspense` で各セクションを個別にラップし、独立してストリーミング表示
- ユーザー基本情報が先に表示され、投稿・TODOは非同期で順次表示される
- プログレスバーでTODOの完了率を視覚的に表示
- `async` なサーバーコンポーネントを子コンポーネントとして分離
- データの取得失敗は `error.tsx` が自動でキャッチする
