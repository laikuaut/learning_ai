## 課題の要件

1. サンプルのアクセスログを生成する（またはサンプルデータを使用する）
2. ステータスコード別にリクエスト数を集計する
3. IPアドレス別のアクセス数をランキング表示する
4. URL別のアクセス数をランキング表示する
5. 時間帯別のアクセス数を棒グラフで表示する
6. エラー（404, 500）のリクエストを抽出して表示する

## ステップガイド

<details>
<summary>ステップ1：ログのフォーマットを理解する</summary>

Apacheのアクセスログ（Combined Log Format）の構造を確認します。

```
192.168.1.10 - - [01/Apr/2024:09:00:01 +0900] "GET /index.html HTTP/1.1" 200 1234
│            │ │ │                             │                          │   │
│            │ │ │                             │                          │   └─ レスポンスサイズ(bytes)
│            │ │ │                             │                          └─ ステータスコード
│            │ │ │                             └─ リクエスト行(メソッド パス プロトコル)
│            │ │ └─ 日時
│            │ └─ ユーザー名(認証)
│            └─ リモートログ名
└─ IPアドレス
```

`awk` でフィールドを抽出する場合：
- `$1` → IPアドレス
- `$4` → 日時（`[01/Apr/2024:09:00:01` の部分）
- `$7` → リクエストパス
- `$9` → ステータスコード

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```bash
#!/bin/bash
# ログファイル解析スクリプト
# 学べる内容：grep, awk, sort, uniq, パイプライン
# 実行方法：bash log_analyzer.sh access.log

LOG_FILE="${1:-access.log}"

# ファイル存在チェック
if [ ! -f "$LOG_FILE" ]; then
    echo "エラー: $LOG_FILE が見つかりません"
    exit 1
fi

TOTAL=$(wc -l < "$LOG_FILE")

echo "===== アクセスログ解析レポート ====="
echo "解析対象: $LOG_FILE"
echo "解析日時: $(date '+%Y-%m-%d %H:%M:%S')"
echo "総リクエスト数: $TOTAL"
echo ""

# --- ステータスコード別集計 ---
echo "【ステータスコード別集計】"
awk '{print $9}' "$LOG_FILE" | sort | uniq -c | sort -rn | while read -r count code; do
    percent=$(echo "scale=1; $count * 100 / $TOTAL" | bc)
    printf "  %s : %5d件 (%5s%%)\n" "$code" "$count" "$percent"
done
echo ""

# --- IPアドレス別 TOP10 ---
echo "【アクセス数 TOP10 ─ IPアドレス】"
awk '{print $1}' "$LOG_FILE" | sort | uniq -c | sort -rn | head -10 | while read -r count ip; do
    printf "  %5d件  %s\n" "$count" "$ip"
done
echo ""

# --- URL別 TOP10 ---
echo "【アクセス数 TOP10 ─ URL】"
awk '{print $7}' "$LOG_FILE" | sort | uniq -c | sort -rn | head -10 | while read -r count url; do
    printf "  %5d件  %s\n" "$count" "$url"
done
echo ""

# --- 時間帯別アクセス数 ---
echo "【時間帯別アクセス数】"
awk '{print $4}' "$LOG_FILE" | cut -d: -f2 | sort | uniq -c | sort -k2n | while read -r count hour; do
    bar=$(printf '%*s' "$count" '' | tr ' ' '█')
    printf "  %s時: %s (%d)\n" "$hour" "$bar" "$count"
done
echo ""

# --- エラーログ抽出 ---
echo "【エラーログ（404/500）】"
awk '$9 == 404 || $9 == 500 {
    printf "  %-18s %s  %-25s %s\n", $1, $9, $7, $4
}' "$LOG_FILE"
echo ""

echo "解析完了"
```

<details>
<summary>解答例（改良版 ─ 関数化とファイル出力）</summary>

```bash
#!/bin/bash
# ログファイル解析スクリプト（改良版）
# 学べる内容：関数化、awk応用、レポート出力
# 実行方法：bash log_analyzer_v2.sh access.log [出力ファイル]

set -e

LOG_FILE="${1:-access.log}"
OUTPUT="${2:-}"

if [ ! -f "$LOG_FILE" ]; then
    echo "エラー: $LOG_FILE が見つかりません" >&2
    exit 1
fi

TOTAL=$(wc -l < "$LOG_FILE")

# 出力先の設定（ファイル指定があればtee、なければそのまま）
output() {
    if [ -n "$OUTPUT" ]; then
        tee -a "$OUTPUT"
    else
        cat
    fi
}

# レポートファイルを初期化
[ -n "$OUTPUT" ] && > "$OUTPUT"

# --- レポート生成 ---
{
    echo "===== アクセスログ解析レポート ====="
    echo "解析対象: $LOG_FILE"
    echo "解析日時: $(date '+%Y-%m-%d %H:%M:%S')"
    echo "総リクエスト数: $TOTAL"
    echo ""

    # ステータスコード別集計（awkで一括処理）
    echo "【ステータスコード別集計】"
    awk -v total="$TOTAL" '
    {status[$9]++}
    END {
        # ステータスコードの意味を定義
        desc["200"] = "OK"
        desc["301"] = "Redirect"
        desc["302"] = "Found"
        desc["304"] = "Not Modified"
        desc["400"] = "Bad Request"
        desc["401"] = "Unauthorized"
        desc["403"] = "Forbidden"
        desc["404"] = "Not Found"
        desc["500"] = "Server Error"
        desc["502"] = "Bad Gateway"
        desc["503"] = "Unavailable"

        # ソートのために配列に格納
        n = asorti(status, sorted)
        for (i = 1; i <= n; i++) {
            code = sorted[i]
            count = status[code]
            pct = count * 100.0 / total
            label = (code in desc) ? desc[code] : "Unknown"
            printf "  %s (%-12s) : %5d件 (%5.1f%%)\n", code, label, count, pct
        }
    }' "$LOG_FILE"
    echo ""

    # HTTPメソッド別集計
    echo "【HTTPメソッド別集計】"
    awk '{
        gsub(/"/, "", $6)  # 先頭の " を除去
        methods[$6]++
    }
    END {
        for (m in methods) printf "  %-8s: %d件\n", m, methods[m]
    }' "$LOG_FILE" | sort -t: -k2 -rn
    echo ""

    # IPアドレス別 TOP10
    echo "【アクセス数 TOP10 ─ IPアドレス】"
    awk '{print $1}' "$LOG_FILE" | sort | uniq -c | sort -rn | head -10 | \
        awk '{printf "  %5d件  %s\n", $1, $2}'
    echo ""

    # URL別 TOP10
    echo "【アクセス数 TOP10 ─ URL】"
    awk '{print $7}' "$LOG_FILE" | sort | uniq -c | sort -rn | head -10 | \
        awk '{printf "  %5d件  %s\n", $1, $2}'
    echo ""

    # 時間帯別アクセス数（0〜23時すべて表示）
    echo "【時間帯別アクセス数】"
    awk '{print $4}' "$LOG_FILE" | cut -d: -f2 | sort | uniq -c | \
        awk '
        BEGIN { for (h = 0; h < 24; h++) hours[sprintf("%02d", h)] = 0 }
        { hours[$2] = $1; if ($1 > max) max = $1 }
        END {
            for (h = 0; h < 24; h++) {
                key = sprintf("%02d", h)
                count = hours[key]
                # 最大値に対する割合で棒グラフの長さを決定（最大40文字）
                if (max > 0) {
                    bar_len = int(count * 40 / max)
                } else {
                    bar_len = 0
                }
                bar = ""
                for (i = 0; i < bar_len; i++) bar = bar "#"
                printf "  %s時: %-40s (%d)\n", key, bar, count
            }
        }'
    echo ""

    # レスポンスサイズ合計
    echo "【レスポンスサイズ】"
    awk '{total += $10} END {
        if (total >= 1073741824) printf "  合計: %.2f GB\n", total/1073741824
        else if (total >= 1048576) printf "  合計: %.2f MB\n", total/1048576
        else if (total >= 1024) printf "  合計: %.2f KB\n", total/1024
        else printf "  合計: %d bytes\n", total
        printf "  平均: %.0f bytes/リクエスト\n", total/NR
    }' "$LOG_FILE"
    echo ""

    # エラーログ抽出
    echo "【エラーログ（4xx/5xx）】"
    error_count=$(awk '$9 >= 400' "$LOG_FILE" | wc -l)
    if [ "$error_count" -gt 0 ]; then
        echo "  エラー数: ${error_count}件 ($(echo "scale=1; $error_count * 100 / $TOTAL" | bc)%)"
        echo ""
        awk '$9 >= 400 {
            printf "  %-18s %s  %-25s %s\n", $1, $9, $7, $4
        }' "$LOG_FILE"
    else
        echo "  エラーなし"
    fi

} | output

if [ -n "$OUTPUT" ]; then
    echo ""
    echo "レポートを ${OUTPUT} に保存しました。"
fi
```

**初心者向けとの違い:**
- `awk` の連想配列を活用してステータスコードの意味を自動表示
- HTTPメソッド別の集計を追加
- 時間帯別グラフを0〜23時すべて表示（アクセスがない時間帯も含む）
- レスポンスサイズの合計・平均を計算
- 4xx/5xx 系のエラーをすべて抽出（404/500だけでなく403等も）
- オプションでファイル出力に対応
