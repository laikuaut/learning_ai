## 問題1：1から100までの合計（基本）

`for` ループを使って、1から100までの整数の合計を計算して表示するプログラムを作成してください。

さらに、1からNまでの合計を計算できるよう、Nをユーザーから入力として受け取るバージョンも作成してください。

### 期待される出力例

```
1から100までの合計: 5050
```

```
上限値Nを入力してください: 10
1から10までの合計: 55
```

<details>
<summary>ヒント</summary>

- 合計用の変数 `sum` を 0 で初期化し、ループ内で加算していきます
- `for (int i = 1; i <= n; i++)` の形式を使います
- 検算：1〜100の合計は `100 * 101 / 2 = 5050`（ガウスの公式）

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    // 固定版: 1〜100の合計
    int sum_fixed = 0;
    for (int i = 1; i <= 100; i++) {
        sum_fixed += i; // sum_fixed = sum_fixed + i と同じ
    }
    std::cout << "1から100までの合計: " << sum_fixed << std::endl;

    // 入力版: 1〜Nの合計
    int n;
    std::cout << "上限値Nを入力してください: ";
    std::cin >> n;

    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    std::cout << "1から" << n << "までの合計: " << sum << std::endl;

    return 0;
}
```

</details>

## 問題2：九九の表を表示しよう（基本）

二重ループを使って、九九の表を表示するプログラムを作成してください。

- 各数値は右揃え（2桁分のスペース）で表示してください
- 行と列の区切りが分かるよう、タブまたはスペースで整列させてください

### 期待される出力例

```
--- 九九の表 ---
  1  2  3  4  5  6  7  8  9
  2  4  6  8 10 12 14 16 18
  3  6  9 12 15 18 21 24 27
  4  8 12 16 20 24 28 32 36
  5 10 15 20 25 30 35 40 45
  6 12 18 24 30 36 42 48 54
  7 14 21 28 35 42 49 56 63
  8 16 24 32 40 48 56 64 72
  9 18 27 36 45 54 63 72 81
```

<details>
<summary>ヒント</summary>

- 外側のループが段（行）、内側のループが列を担当します
- `#include <iomanip>` の `std::setw(3)` を使うと3桁分の幅で右揃え表示できます
- 内側のループが終わったら `std::endl` で改行します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>

int main() {
    std::cout << "--- 九九の表 ---" << std::endl;

    // 二重ループで九九を表示
    for (int i = 1; i <= 9; i++) {         // 行（段）
        for (int j = 1; j <= 9; j++) {     // 列
            // setw(3) で3桁幅の右揃え表示
            std::cout << std::setw(3) << i * j;
        }
        std::cout << std::endl; // 1行分出力したら改行
    }

    return 0;
}
```

</details>

## 問題3：FizzBuzz（基本）

1から30までの数を表示するプログラムを作成してください。ただし以下のルールに従います。

- 3の倍数のときは数値の代わりに「Fizz」と表示
- 5の倍数のときは数値の代わりに「Buzz」と表示
- 3と5の両方の倍数のときは「FizzBuzz」と表示
- それ以外は数値をそのまま表示

### 期待される出力例

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
Fizz
22
23
Fizz
Buzz
26
Fizz
28
29
FizzBuzz
```

<details>
<summary>ヒント</summary>

- 条件判定の順番が重要です。「3と5の両方の倍数」を最初にチェックしないと正しく動きません
- `i % 15 == 0` は「3と5の両方の倍数」と同じ意味です
- `if` → `else if` → `else if` → `else` の順に判定します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    for (int i = 1; i <= 30; i++) {
        // 重要: 15の倍数（3と5の公倍数）を最初にチェック
        if (i % 15 == 0) {
            std::cout << "FizzBuzz" << std::endl;
        }
        else if (i % 3 == 0) {
            std::cout << "Fizz" << std::endl;
        }
        else if (i % 5 == 0) {
            std::cout << "Buzz" << std::endl;
        }
        else {
            std::cout << i << std::endl;
        }
    }

    return 0;
}
```

</details>

## 問題4：素数判定プログラム（応用）

整数を入力として受け取り、その数が素数かどうかを判定するプログラムを作成してください。

さらに、2からNまでの素数をすべて表示する機能も追加してください。

### 期待される出力例

```
整数を入力してください: 17
17は素数です

2から17までの素数:
2 3 5 7 11 13 17
合計: 7個
```

```
整数を入力してください: 12
12は素数ではありません（12 = 2 × 6）

2から12までの素数:
2 3 5 7 11
合計: 5個
```

<details>
<summary>ヒント</summary>

- 素数とは「1と自分自身以外で割り切れない、2以上の整数」です
- 2からn-1まで（または√nまで）の数で割って、余りが0になるものがあれば素数ではありません
- `break` を使って、割り切れた時点でループを抜けると効率的です
- 素数でない場合、最小の約数を見つけると `n = 約数 × 商` の形で表示できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int n;
    std::cout << "整数を入力してください: ";
    std::cin >> n;

    // --- 単一の数の素数判定 ---
    bool is_prime = true;
    int divisor = 0; // 素数でない場合の約数を記録

    if (n < 2) {
        is_prime = false;
    } else {
        // 2から n-1 まで割ってみる
        for (int i = 2; i * i <= n; i++) {
            if (n % i == 0) {
                is_prime = false;
                divisor = i;
                break; // 割り切れたらそこで終了
            }
        }
    }

    if (is_prime) {
        std::cout << n << "は素数です" << std::endl;
    } else {
        std::cout << n << "は素数ではありません";
        if (divisor > 0) {
            std::cout << "（" << n << " = " << divisor
                      << " × " << n / divisor << "）";
        }
        std::cout << std::endl;
    }

    // --- 2からNまでの素数一覧 ---
    std::cout << std::endl;
    std::cout << "2から" << n << "までの素数:" << std::endl;

    int count = 0;
    for (int num = 2; num <= n; num++) {
        bool prime = true;
        for (int i = 2; i * i <= num; i++) {
            if (num % i == 0) {
                prime = false;
                break;
            }
        }
        if (prime) {
            std::cout << num << " ";
            count++;
        }
    }
    std::cout << std::endl;
    std::cout << "合計: " << count << "個" << std::endl;

    return 0;
}
```

</details>

## 問題5：階乗計算プログラム（応用）

整数N（0以上）を入力として受け取り、Nの階乗（N!）を計算して表示するプログラムを作成してください。

- `while` ループを使って計算してください
- 計算過程も表示してください（例：`5! = 5 × 4 × 3 × 2 × 1 = 120`）
- 0! = 1 であることに注意してください
- 負の数が入力された場合はエラーメッセージを表示してください

### 期待される出力例

```
整数を入力してください: 5
5! = 5 × 4 × 3 × 2 × 1 = 120
```

```
整数を入力してください: 0
0! = 1
```

```
整数を入力してください: 1
1! = 1
```

```
整数を入力してください: -3
エラー: 負の数の階乗は定義されていません
```

<details>
<summary>ヒント</summary>

- `while` ループのカウンタを N から 1 まで減らしていく方法が自然です
- 結果を格納する変数は `long long` 型を使うと大きな数にも対応できます
- 計算過程の `×` 記号は最初の数字の後から表示する工夫が必要です
- `do-while` ループでも実装できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int n;
    std::cout << "整数を入力してください: ";
    std::cin >> n;

    // 負の数チェック
    if (n < 0) {
        std::cout << "エラー: 負の数の階乗は定義されていません" << std::endl;
        return 1;
    }

    // 0! と 1! の特別なケース
    if (n == 0 || n == 1) {
        std::cout << n << "! = 1" << std::endl;
        return 0;
    }

    // 階乗の計算（whileループ使用）
    long long result = 1;
    int current = n;

    // 計算過程の表示を準備
    std::cout << n << "! = ";

    while (current >= 1) {
        // 最初の数字以外は「× 」を前に付ける
        if (current != n) {
            std::cout << " × ";
        }
        std::cout << current;

        result *= current;
        current--; // カウンタを1減らす
    }

    std::cout << " = " << result << std::endl;

    return 0;
}
```

</details>

## 問題6：ピラミッドを表示しよう（応用）

段数Nを入力として受け取り、以下のようなピラミッド（三角形）を表示するプログラムを作成してください。

### 期待される出力例

```
段数を入力してください: 5

パターン1: 左寄せ三角形
*
**
***
****
*****

パターン2: ピラミッド（中央寄せ）
    *
   ***
  *****
 *******
*********
```

<details>
<summary>ヒント</summary>

- **パターン1:** 外側のループで行数、内側のループで `*` の個数を制御します。i行目には i 個の `*` を表示します。
- **パターン2:** i行目（1始まり）では、先頭に `(n - i)` 個の空白、続けて `(2 * i - 1)` 個の `*` を表示します。
- ネスト（入れ子）ループで空白と `*` をそれぞれ別のループで出力しましょう。

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int n;
    std::cout << "段数を入力してください: ";
    std::cin >> n;

    // パターン1: 左寄せ三角形
    std::cout << std::endl;
    std::cout << "パターン1: 左寄せ三角形" << std::endl;
    for (int i = 1; i <= n; i++) {
        for (int j = 0; j < i; j++) {
            std::cout << "*";
        }
        std::cout << std::endl;
    }

    // パターン2: ピラミッド（中央寄せ）
    std::cout << std::endl;
    std::cout << "パターン2: ピラミッド（中央寄せ）" << std::endl;
    for (int i = 1; i <= n; i++) {
        // 空白を出力: (n - i) 個
        for (int j = 0; j < n - i; j++) {
            std::cout << " ";
        }
        // *を出力: (2 * i - 1) 個
        for (int j = 0; j < 2 * i - 1; j++) {
            std::cout << "*";
        }
        std::cout << std::endl;
    }

    return 0;
}
```

</details>

## 問題7：数当てゲーム（チャレンジ）

コンピューターが決めた「秘密の数」（1〜100の範囲）をユーザーが当てるゲームを作成してください。

- 秘密の数は固定値（例えば42）でかまいません
- ユーザーが数を入力するたびに「大きすぎます」「小さすぎます」のヒントを出す
- 正解したら「おめでとう！○回目で正解です！」と表示して終了
- `do-while` ループを使って実装してください
- `continue` を使って範囲外入力をスキップする処理も入れてください

### 期待される出力例

```
=== 数当てゲーム ===
1〜100の数を当ててください！

予想を入力: 50
大きすぎます！

予想を入力: 25
小さすぎます！

予想を入力: 37
小さすぎます！

予想を入力: 150
1〜100の範囲で入力してください

予想を入力: 42
おめでとう！4回目で正解です！
```

<details>
<summary>ヒント</summary>

- `do-while` ループは「少なくとも1回は実行する」ループです
- ループの継続条件は「予想 != 秘密の数」です
- `continue` を使うと、範囲外入力時にカウントを増やさずに次の入力を求められます
- 試行回数用のカウンタ変数を用意しましょう

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    // 秘密の数（固定値）
    // 実際のゲームでは乱数を使いますが、ここでは固定で練習
    const int secret = 42;

    int guess;
    int attempts = 0;

    std::cout << "=== 数当てゲーム ===" << std::endl;
    std::cout << "1〜100の数を当ててください！" << std::endl;
    std::cout << std::endl;

    // do-whileループ: 少なくとも1回は実行される
    do {
        std::cout << "予想を入力: ";
        std::cin >> guess;

        // 範囲外チェック（continueでループの先頭に戻る）
        if (guess < 1 || guess > 100) {
            std::cout << "1〜100の範囲で入力してください" << std::endl;
            std::cout << std::endl;
            continue; // カウントせずに次のループへ
        }

        attempts++; // 有効な試行回数をカウント

        if (guess > secret) {
            std::cout << "大きすぎます！" << std::endl;
        } else if (guess < secret) {
            std::cout << "小さすぎます！" << std::endl;
        } else {
            std::cout << "おめでとう！" << attempts
                      << "回目で正解です！" << std::endl;
        }

        std::cout << std::endl;

    } while (guess != secret); // 正解するまでループ

    return 0;
}
```

</details>
