# 第10章：実践パターン

## 10.1 provide / inject — 深いコンポーネント間のデータ共有

props のバケツリレー（prop drilling）を避けて、祖先から子孫にデータを直接渡す仕組みです。

### 基本的な使い方

```vue
<!-- 祖先コンポーネント（App.vue） -->
<script setup>
import { ref, provide, readonly } from 'vue'

const theme = ref('light')
const user = ref({ name: '太郎', role: 'admin' })

// provide(キー, 値)
provide('theme', theme)

// readonly で渡すと子孫から変更されない（推奨）
provide('user', readonly(user))

// 変更関数も provide できる
function toggleTheme() {
  theme.value = theme.value === 'light' ? 'dark' : 'light'
}
provide('toggleTheme', toggleTheme)
</script>
```

```vue
<!-- 子孫コンポーネント（深い階層でもOK） -->
<script setup>
import { inject } from 'vue'

// inject(キー, デフォルト値)
const theme = inject('theme', 'light')
const user = inject('user')
const toggleTheme = inject('toggleTheme')
</script>

<template>
  <div :class="['container', `theme-${theme}`]">
    <p>こんにちは、{{ user?.name }}さん</p>
    <button @click="toggleTheme">テーマ切り替え</button>
  </div>
</template>
```

### Symbol キーでの provide/inject（大規模アプリ向け）

```js
// injection-keys.js
export const THEME_KEY = Symbol('theme')
export const USER_KEY = Symbol('user')
```

```vue
<!-- 提供側 -->
<script setup>
import { provide, ref } from 'vue'
import { THEME_KEY, USER_KEY } from '@/injection-keys'

provide(THEME_KEY, ref('dark'))
provide(USER_KEY, ref({ name: '太郎' }))
</script>
```

```vue
<!-- 注入側 -->
<script setup>
import { inject } from 'vue'
import { THEME_KEY, USER_KEY } from '@/injection-keys'

const theme = inject(THEME_KEY)
const user = inject(USER_KEY)
</script>
```

## 10.2 Teleport — DOM の別の場所にレンダリング

コンポーネントの描画結果を、DOM ツリーの別の場所に移動します。モーダル、トースト、ツールチップなどに最適です。

```vue
<!-- components/Modal.vue -->
<script setup>
defineProps({
  show: Boolean,
  title: { type: String, default: '' }
})

const emit = defineEmits(['close'])
</script>

<template>
  <!-- body 直下にテレポート -->
  <Teleport to="body">
    <Transition name="modal">
      <div v-if="show" class="modal-overlay" @click.self="emit('close')">
        <div class="modal-content">
          <header class="modal-header">
            <h2>{{ title }}</h2>
            <button @click="emit('close')" class="close-btn">&times;</button>
          </header>
          <main class="modal-body">
            <slot />
          </main>
          <footer class="modal-footer">
            <slot name="footer">
              <button @click="emit('close')">閉じる</button>
            </slot>
          </footer>
        </div>
      </div>
    </Transition>
  </Teleport>
</template>

<style scoped>
.modal-overlay {
  position: fixed;
  inset: 0;
  background: rgba(0, 0, 0, 0.5);
  display: flex;
  align-items: center;
  justify-content: center;
  z-index: 1000;
}

.modal-content {
  background: white;
  border-radius: 12px;
  width: 90%;
  max-width: 500px;
  max-height: 80vh;
  overflow-y: auto;
}

.modal-header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem 1.5rem;
  border-bottom: 1px solid #eee;
}

.modal-body {
  padding: 1.5rem;
}

.modal-footer {
  padding: 1rem 1.5rem;
  border-top: 1px solid #eee;
  text-align: right;
}

.close-btn {
  background: none;
  border: none;
  font-size: 1.5rem;
  cursor: pointer;
}

/* トランジション */
.modal-enter-active,
.modal-leave-active {
  transition: opacity 0.3s;
}
.modal-enter-from,
.modal-leave-to {
  opacity: 0;
}
</style>
```

### 使用例

```vue
<script setup>
import { ref } from 'vue'
import Modal from '@/components/Modal.vue'

const showModal = ref(false)
</script>

<template>
  <button @click="showModal = true">モーダルを開く</button>

  <Modal :show="showModal" title="確認" @close="showModal = false">
    <p>この操作を実行しますか？</p>

    <template #footer>
      <button @click="showModal = false">キャンセル</button>
      <button @click="handleConfirm">OK</button>
    </template>
  </Modal>
</template>
```

## 10.3 Transition / TransitionGroup — アニメーション

### 単一要素のトランジション

```vue
<script setup>
import { ref } from 'vue'

const show = ref(true)
</script>

<template>
  <button @click="show = !show">トグル</button>

  <!-- name 属性でCSSクラス名のプレフィックスを指定 -->
  <Transition name="fade">
    <p v-if="show">フェードイン/アウトする要素</p>
  </Transition>
</template>

<style>
/* Enter: 要素が表示される時 */
.fade-enter-from {
  opacity: 0;            /* 開始状態 */
}
.fade-enter-active {
  transition: opacity 0.5s ease;  /* アニメーション設定 */
}
.fade-enter-to {
  opacity: 1;            /* 終了状態（省略可: デフォルト） */
}

/* Leave: 要素が非表示になる時 */
.fade-leave-from {
  opacity: 1;
}
.fade-leave-active {
  transition: opacity 0.5s ease;
}
.fade-leave-to {
  opacity: 0;
}
</style>
```

### スライドトランジション

```vue
<template>
  <Transition name="slide-fade">
    <div v-if="show" class="box">スライドフェード</div>
  </Transition>
</template>

<style>
.slide-fade-enter-active {
  transition: all 0.3s ease-out;
}
.slide-fade-leave-active {
  transition: all 0.3s cubic-bezier(1, 0.5, 0.8, 1);
}
.slide-fade-enter-from,
.slide-fade-leave-to {
  transform: translateX(20px);
  opacity: 0;
}
</style>
```

### mode="out-in" — 要素の入れ替え

```vue
<template>
  <Transition name="fade" mode="out-in">
    <!-- key が変わると古い要素が退出 → 新しい要素が入場 -->
    <div :key="currentTab">
      <component :is="tabs[currentTab]" />
    </div>
  </Transition>
</template>
```

### TransitionGroup — リストのアニメーション

```vue
<script setup>
import { ref } from 'vue'

const items = ref([1, 2, 3, 4, 5])
let nextId = 6

function addItem() {
  const index = Math.floor(Math.random() * (items.value.length + 1))
  items.value.splice(index, 0, nextId++)
}

function removeItem(index) {
  items.value.splice(index, 1)
}

function shuffle() {
  items.value = items.value.sort(() => Math.random() - 0.5)
}
</script>

<template>
  <button @click="addItem">追加</button>
  <button @click="shuffle">シャッフル</button>

  <TransitionGroup name="list" tag="ul">
    <li
      v-for="item in items"
      :key="item"
      @click="removeItem(items.indexOf(item))"
    >
      {{ item }}
    </li>
  </TransitionGroup>
</template>

<style>
.list-enter-active,
.list-leave-active {
  transition: all 0.5s ease;
}
.list-enter-from,
.list-leave-to {
  opacity: 0;
  transform: translateX(30px);
}

/* FLIP アニメーション（移動時のスムーズな遷移） */
.list-move {
  transition: transform 0.5s ease;
}

/* 退出時に他の要素がスムーズに移動するように */
.list-leave-active {
  position: absolute;
}
</style>
```

### JavaScript フック

```vue
<template>
  <Transition
    @before-enter="onBeforeEnter"
    @enter="onEnter"
    @after-enter="onAfterEnter"
    @before-leave="onBeforeLeave"
    @leave="onLeave"
    @after-leave="onAfterLeave"
    :css="false"
  >
    <div v-if="show">要素</div>
  </Transition>
</template>

<script setup>
function onEnter(el, done) {
  // JavaScript でアニメーション（GSAP などのライブラリを使う場合）
  el.style.opacity = 0
  el.offsetHeight  // reflow をトリガー

  el.style.transition = 'opacity 0.5s'
  el.style.opacity = 1

  el.addEventListener('transitionend', done)
}

function onLeave(el, done) {
  el.style.transition = 'opacity 0.5s'
  el.style.opacity = 0
  el.addEventListener('transitionend', done)
}
</script>
```

## 10.4 動的コンポーネント

実行時にコンポーネントを切り替える仕組みです。

```vue
<script setup>
import { ref, shallowRef, markRaw } from 'vue'
import TabHome from './tabs/TabHome.vue'
import TabProfile from './tabs/TabProfile.vue'
import TabSettings from './tabs/TabSettings.vue'

const tabs = {
  home: markRaw(TabHome),
  profile: markRaw(TabProfile),
  settings: markRaw(TabSettings)
}

const tabLabels = {
  home: 'ホーム',
  profile: 'プロフィール',
  settings: '設定'
}

const currentTab = ref('home')
</script>

<template>
  <div class="tabs">
    <button
      v-for="(label, key) in tabLabels"
      :key="key"
      @click="currentTab = key"
      :class="{ active: currentTab === key }"
    >
      {{ label }}
    </button>
  </div>

  <!-- 動的コンポーネント -->
  <component :is="tabs[currentTab]" />
</template>
```

### KeepAlive — 状態の保持

デフォルトでは、コンポーネントが切り替わると状態が失われます。`<KeepAlive>` で囲むと、切り替えても状態が保持されます。

```vue
<template>
  <KeepAlive>
    <component :is="tabs[currentTab]" />
  </KeepAlive>

  <!-- 特定のコンポーネントだけキャッシュ -->
  <KeepAlive include="TabHome,TabProfile">
    <component :is="tabs[currentTab]" />
  </KeepAlive>

  <!-- 最大キャッシュ数 -->
  <KeepAlive :max="5">
    <component :is="tabs[currentTab]" />
  </KeepAlive>
</template>
```

### KeepAlive のライフサイクル

```vue
<script setup>
import { onActivated, onDeactivated } from 'vue'

// KeepAlive でキャッシュされたコンポーネント専用のフック
onActivated(() => {
  console.log('コンポーネントが再アクティブ化された')
  // データの再取得など
})

onDeactivated(() => {
  console.log('コンポーネントが非アクティブ化された')
  // タイマーの一時停止など
})
</script>
```

## 10.5 TypeScript との統合

### props の型定義

```vue
<script setup lang="ts">
// TypeScript の interface で props を定義
interface User {
  id: number
  name: string
  email: string
  role: 'admin' | 'editor' | 'viewer'
  avatar?: string
}

interface Props {
  user: User
  showAvatar?: boolean
  maxWidth?: number
}

const props = withDefaults(defineProps<Props>(), {
  showAvatar: true,
  maxWidth: 400
})

// props.user は User 型として推論される
console.log(props.user.name)
</script>
```

### emits の型定義

```vue
<script setup lang="ts">
interface UserFormData {
  name: string
  email: string
}

// emits の型定義
const emit = defineEmits<{
  (e: 'submit', data: UserFormData): void
  (e: 'cancel'): void
  (e: 'update:modelValue', value: string): void
}>()

function handleSubmit() {
  emit('submit', { name: '太郎', email: 'taro@example.com' })
}
</script>
```

### ref の型定義

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'

// 型推論（通常は自動推論で十分）
const count = ref(0)         // Ref<number>
const message = ref('hello') // Ref<string>

// 明示的な型指定（null を含む場合など）
const user = ref<User | null>(null)
const items = ref<string[]>([])

// computed も型推論される
const doubled = computed(() => count.value * 2)  // ComputedRef<number>
</script>
```

### テンプレート参照の型

```vue
<script setup lang="ts">
import { ref, onMounted } from 'vue'

// HTML要素の型を指定
const inputRef = ref<HTMLInputElement | null>(null)
const canvasRef = ref<HTMLCanvasElement | null>(null)

onMounted(() => {
  inputRef.value?.focus()  // オプショナルチェーンで安全にアクセス

  const ctx = canvasRef.value?.getContext('2d')
  if (ctx) {
    ctx.fillStyle = 'red'
    ctx.fillRect(0, 0, 100, 100)
  }
})
</script>

<template>
  <input ref="inputRef" />
  <canvas ref="canvasRef" width="200" height="200"></canvas>
</template>
```

### Composable の型定義

```ts
// composables/useCounter.ts
import { ref, computed, type Ref, type ComputedRef } from 'vue'

interface UseCounterReturn {
  count: Ref<number>
  doubled: ComputedRef<number>
  increment: () => void
  decrement: () => void
  reset: () => void
}

export function useCounter(initialValue = 0): UseCounterReturn {
  const count = ref(initialValue)
  const doubled = computed(() => count.value * 2)

  function increment() { count.value++ }
  function decrement() { count.value-- }
  function reset() { count.value = initialValue }

  return { count, doubled, increment, decrement, reset }
}
```

## 10.6 カスタムディレクティブ

DOM要素に対して低レベルの操作を行うカスタムディレクティブを作成できます。

```js
// directives/vFocus.js
// 要素にフォーカスを当てるディレクティブ
export const vFocus = {
  mounted(el) {
    el.focus()
  }
}
```

```js
// directives/vClickOutside.js
// 要素の外側をクリックした時に発火するディレクティブ
export const vClickOutside = {
  mounted(el, binding) {
    el.__clickOutsideHandler = (event) => {
      if (!el.contains(event.target)) {
        binding.value(event)  // コールバック実行
      }
    }
    document.addEventListener('click', el.__clickOutsideHandler)
  },
  unmounted(el) {
    document.removeEventListener('click', el.__clickOutsideHandler)
  }
}
```

```vue
<!-- 使用例 -->
<script setup>
import { ref } from 'vue'
import { vFocus } from '@/directives/vFocus'
import { vClickOutside } from '@/directives/vClickOutside'

const isDropdownOpen = ref(false)

function closeDropdown() {
  isDropdownOpen.value = false
}
</script>

<template>
  <!-- 自動フォーカス -->
  <input v-focus placeholder="自動フォーカス" />

  <!-- 外側クリックで閉じる -->
  <div v-click-outside="closeDropdown" class="dropdown">
    <button @click="isDropdownOpen = !isDropdownOpen">メニュー</button>
    <ul v-show="isDropdownOpen">
      <li>項目1</li>
      <li>項目2</li>
    </ul>
  </div>
</template>
```

## 10.7 非同期コンポーネント

大きなコンポーネントを必要な時にだけ読み込みます。

```vue
<script setup>
import { defineAsyncComponent, ref } from 'vue'

// 非同期コンポーネント
const HeavyChart = defineAsyncComponent(() =>
  import('@/components/HeavyChart.vue')
)

// ローディング/エラー付き
const AsyncModal = defineAsyncComponent({
  loader: () => import('@/components/Modal.vue'),
  loadingComponent: () => import('@/components/LoadingSpinner.vue'),
  errorComponent: () => import('@/components/ErrorMessage.vue'),
  delay: 200,       // ローディング表示までの遅延（ms）
  timeout: 10000    // タイムアウト（ms）
})

const showChart = ref(false)
</script>

<template>
  <button @click="showChart = true">グラフを表示</button>

  <!-- ボタンが押されて初めてコンポーネントが読み込まれる -->
  <HeavyChart v-if="showChart" />
</template>
```

## 10.8 パフォーマンス最適化

### v-once — 一度だけレンダリング

```vue
<template>
  <!-- 静的なコンテンツ: 一度レンダリングしたら更新しない -->
  <div v-once>
    <h1>{{ title }}</h1>
    <p>この内容は変更されません</p>
  </div>
</template>
```

### v-memo — 条件付きメモ化

```vue
<template>
  <!-- items と selected が同じなら再レンダリングをスキップ -->
  <div v-for="item in items" :key="item.id" v-memo="[item.id === selected]">
    <p>{{ item.name }}</p>
    <span v-if="item.id === selected">選択中</span>
  </div>
</template>
```

### 大きなリストの仮想スクロール

```bash
npm install @tanstack/vue-virtual
```

```vue
<script setup>
import { ref } from 'vue'
import { useVirtualizer } from '@tanstack/vue-virtual'

// 10,000件のデータ
const items = ref(Array.from({ length: 10000 }, (_, i) => ({
  id: i,
  name: `アイテム ${i + 1}`
})))

const parentRef = ref(null)

const virtualizer = useVirtualizer({
  count: items.value.length,
  getScrollElement: () => parentRef.value,
  estimateSize: () => 40,  // 各アイテムの推定高さ
})
</script>

<template>
  <div ref="parentRef" style="height: 400px; overflow: auto;">
    <div :style="{ height: `${virtualizer.getTotalSize()}px`, position: 'relative' }">
      <div
        v-for="row in virtualizer.getVirtualItems()"
        :key="row.key"
        :style="{
          position: 'absolute',
          top: `${row.start}px`,
          height: `${row.size}px`,
          width: '100%'
        }"
      >
        {{ items[row.index].name }}
      </div>
    </div>
  </div>
</template>
```

## 10.10 本章のまとめ

| 概念 | 用途 |
|------|------|
| `provide` / `inject` | 深い階層へのデータ受け渡し |
| `Teleport` | DOM の別の場所にレンダリング |
| `Transition` | 単一要素の入退場アニメーション |
| `TransitionGroup` | リストのアニメーション |
| `KeepAlive` | コンポーネントの状態を保持 |
| 動的コンポーネント | `<component :is>` で実行時に切り替え |
| TypeScript | 型安全な Vue 開発 |
| カスタムディレクティブ | DOM要素への低レベル操作 |
| 非同期コンポーネント | 遅延ロードによるパフォーマンス最適化 |
| `v-once` / `v-memo` | レンダリングの最適化 |
