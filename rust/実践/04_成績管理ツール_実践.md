## 課題の要件

1. 生徒の名前と点数をベクタで管理する
2. 以下のメニュー操作を実装する：
   - **追加**: 名前と点数を入力して登録
   - **一覧**: 全生徒の名前・点数・評価を表示
   - **平均**: 全生徒の平均点を計算して表示
   - **最高点**: 最高得点の生徒を表示
   - **検索**: 名前の一部で生徒を検索
   - **終了**: プログラムを終了
3. 評価基準：90点以上=A、70点以上=B、50点以上=C、それ未満=D
4. 生徒が0人のときに平均・最高点を選んだ場合、適切なメッセージを表示する

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```rust
// 成績管理ツール
// 学べる内容：Vec、タプル、ループ、match、文字列操作
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

fn get_grade(score: i32) -> &'static str {
    if score >= 90 {
        "A（優秀）"
    } else if score >= 70 {
        "B（良好）"
    } else if score >= 50 {
        "C（合格）"
    } else {
        "D（要努力）"
    }
}

fn read_input(prompt: &str) -> String {
    print!("{}", prompt);
    io::stdout().flush().unwrap();
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

fn main() {
    println!("===== 成績管理ツール =====");

    let mut students: Vec<(String, i32)> = Vec::new();

    loop {
        println!("\n1.追加  2.一覧  3.平均  4.最高点  5.検索  6.終了");
        let choice = read_input("操作: ");

        match choice.as_str() {
            "1" => {
                // 追加
                let name = read_input("名前: ");
                let score_str = read_input("点数: ");
                let score: i32 = match score_str.parse() {
                    Ok(n) => n,
                    Err(_) => {
                        println!("無効な点数です。");
                        continue;
                    }
                };
                println!("{}さん（{}点）を追加しました。", name, score);
                students.push((name, score));
            }
            "2" => {
                // 一覧
                if students.is_empty() {
                    println!("生徒が登録されていません。");
                    continue;
                }
                println!("\n--- 成績一覧 ---");
                for (i, (name, score)) in students.iter().enumerate() {
                    let grade = get_grade(*score);
                    println!(" {}. {:<8} {}点  {}", i + 1, name, score, grade);
                }
            }
            "3" => {
                // 平均
                if students.is_empty() {
                    println!("生徒が登録されていません。");
                    continue;
                }
                let total: i32 = students.iter().map(|(_, s)| s).sum();
                let average = total as f64 / students.len() as f64;
                println!("平均点: {:.1}点", average);
            }
            "4" => {
                // 最高点
                if students.is_empty() {
                    println!("生徒が登録されていません。");
                    continue;
                }
                let top = students.iter().max_by_key(|(_, s)| s).unwrap();
                println!("最高得点: {}さん（{}点）", top.0, top.1);
            }
            "5" => {
                // 検索
                let keyword = read_input("検索する名前: ");
                let results: Vec<&(String, i32)> = students
                    .iter()
                    .filter(|(name, _)| name.contains(&keyword))
                    .collect();

                if results.is_empty() {
                    println!("見つかりませんでした。");
                } else {
                    for (name, score) in results {
                        println!("  {} {}点", name, score);
                    }
                }
            }
            "6" => {
                println!("終了します。");
                break;
            }
            _ => {
                println!("無効な操作です。1〜6で選んでください。");
            }
        }
    }
}
```

<summary>解答例（改良版 ─ 構造体を使った設計）</summary>

第6章（構造体と列挙型）の知識を先取りして、構造体でデータを管理するバージョンです。

```rust
// 成績管理ツール（改良版）
// 構造体とメソッドで設計したバージョン
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

/// 生徒を表す構造体
struct Student {
    name: String,
    score: i32,
}

impl Student {
    fn new(name: String, score: i32) -> Student {
        Student { name, score }
    }

    fn grade(&self) -> &'static str {
        match self.score {
            90..=100 => "A（優秀）",
            70..=89  => "B（良好）",
            50..=69  => "C（合格）",
            _        => "D（要努力）",
        }
    }
}

/// 成績管理を行う構造体
struct GradeManager {
    students: Vec<Student>,
}

impl GradeManager {
    fn new() -> GradeManager {
        GradeManager {
            students: Vec::new(),
        }
    }

    fn add(&mut self, name: String, score: i32) {
        println!("{}さん（{}点）を追加しました。", name, score);
        self.students.push(Student::new(name, score));
    }

    fn show_all(&self) {
        if self.students.is_empty() {
            println!("生徒が登録されていません。");
            return;
        }
        println!("\n--- 成績一覧 ---");
        for (i, s) in self.students.iter().enumerate() {
            println!(" {}. {:<8} {}点  {}", i + 1, s.name, s.score, s.grade());
        }
    }

    fn average(&self) {
        if self.students.is_empty() {
            println!("生徒が登録されていません。");
            return;
        }
        let total: i32 = self.students.iter().map(|s| s.score).sum();
        let avg = total as f64 / self.students.len() as f64;
        println!("平均点: {:.1}点", avg);
    }

    fn top(&self) {
        if self.students.is_empty() {
            println!("生徒が登録されていません。");
            return;
        }
        let top = self.students.iter().max_by_key(|s| s.score).unwrap();
        println!("最高得点: {}さん（{}点）", top.name, top.score);
    }

    fn search(&self, keyword: &str) {
        let results: Vec<&Student> = self
            .students
            .iter()
            .filter(|s| s.name.contains(keyword))
            .collect();

        if results.is_empty() {
            println!("見つかりませんでした。");
        } else {
            for s in results {
                println!("  {} {}点  {}", s.name, s.score, s.grade());
            }
        }
    }
}

fn read_input(prompt: &str) -> String {
    print!("{}", prompt);
    io::stdout().flush().unwrap();
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

fn main() {
    println!("===== 成績管理ツール =====");
    let mut manager = GradeManager::new();

    loop {
        println!("\n1.追加  2.一覧  3.平均  4.最高点  5.検索  6.終了");
        let choice = read_input("操作: ");

        match choice.as_str() {
            "1" => {
                let name = read_input("名前: ");
                let score_str = read_input("点数: ");
                match score_str.parse::<i32>() {
                    Ok(score) => manager.add(name, score),
                    Err(_) => println!("無効な点数です。"),
                }
            }
            "2" => manager.show_all(),
            "3" => manager.average(),
            "4" => manager.top(),
            "5" => {
                let keyword = read_input("検索する名前: ");
                manager.search(&keyword);
            }
            "6" => {
                println!("終了します。");
                break;
            }
            _ => println!("無効な操作です。1〜6で選んでください。"),
        }
    }
}
```

**初心者向けとの違い:**
- `Student` 構造体でデータと振る舞い（`grade()` メソッド）をまとめて管理
- `GradeManager` 構造体で操作ロジックをカプセル化 → `main()` がシンプルに
- `match` 式のレンジパターン（`90..=100`）で評価判定を簡潔に記述
- `&self` / `&mut self` で所有権と借用を正しく使い分け
