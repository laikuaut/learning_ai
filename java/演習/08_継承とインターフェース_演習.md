## 基本問題

### 問題1：動物クラスの継承

`Animal` クラスを親クラスとして、`Dog` クラスと `Cat` クラスを作成してください。それぞれ `speak()` メソッドをオーバーライドして、動物ごとの鳴き声を表示してください。

**期待される出力例：**
```
ポチ: ワンワン！
タマ: ニャーニャー！
```

<details>
<summary>ヒント</summary>

- 親クラス `Animal` に `name` フィールドとコンストラクタ、`speak()` メソッドを定義します
- `Dog` と `Cat` で `extends Animal` して `speak()` を `@Override` します
- `super(name)` で親のコンストラクタを呼び出します

</details>

<details>
<summary>解答例</summary>

```java
/* 動物クラスの継承 */
class Animal {
    String name;

    Animal(String name) {
        this.name = name;
    }

    void speak() {
        System.out.println(name + ": ...");
    }
}

class Dog extends Animal {
    Dog(String name) {
        super(name);  /* 親クラスのコンストラクタを呼び出し */
    }

    @Override
    void speak() {
        System.out.println(name + ": ワンワン！");
    }
}

class Cat extends Animal {
    Cat(String name) {
        super(name);
    }

    @Override
    void speak() {
        System.out.println(name + ": ニャーニャー！");
    }
}

public class AnimalTest {
    public static void main(String[] args) {
        Dog dog = new Dog("ポチ");
        Cat cat = new Cat("タマ");
        dog.speak();
        cat.speak();
    }
}
```

</details>

### 問題2：図形クラスの継承

抽象クラス `Shape` を作成し、`Circle` と `Rectangle` で `area()`（面積）と `describe()`（説明表示）を実装してください。

**期待される出力例：**
```
円（半径: 5.0） → 面積: 78.54
長方形（幅: 4.0, 高さ: 6.0） → 面積: 24.00
```

<details>
<summary>ヒント</summary>

- `abstract class Shape` に `abstract double area()` と `abstract void describe()` を定義します
- `Circle` は半径、`Rectangle` は幅と高さをフィールドに持ちます
- `Math.PI` を円の面積計算に使います

</details>

<details>
<summary>解答例</summary>

```java
/* 抽象クラスで図形を定義 */
abstract class Shape {
    abstract double area();
    abstract void describe();
}

class Circle extends Shape {
    double radius;

    Circle(double radius) {
        this.radius = radius;
    }

    @Override
    double area() {
        return Math.PI * radius * radius;
    }

    @Override
    void describe() {
        System.out.printf("円（半径: %.1f） → 面積: %.2f%n", radius, area());
    }
}

class Rectangle extends Shape {
    double width, height;

    Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }

    @Override
    double area() {
        return width * height;
    }

    @Override
    void describe() {
        System.out.printf("長方形（幅: %.1f, 高さ: %.1f） → 面積: %.2f%n", width, height, area());
    }
}

public class ShapeTest {
    public static void main(String[] args) {
        Shape circle = new Circle(5.0);
        Shape rect = new Rectangle(4.0, 6.0);
        circle.describe();
        rect.describe();
    }
}
```

</details>

### 問題3：インターフェースの基本

`Printable` インターフェースを作成し、`printInfo()` メソッドを定義してください。`Student` クラスと `Teacher` クラスでこのインターフェースを実装してください。

**期待される出力例：**
```
[学生] 田中太郎 - 3年生
[教師] 鈴木先生 - 数学担当
```

<details>
<summary>ヒント</summary>

- `interface Printable` に `void printInfo()` を定義します
- 各クラスで `implements Printable` として `printInfo()` を実装します

</details>

<details>
<summary>解答例</summary>

```java
/* インターフェースの定義 */
interface Printable {
    void printInfo();
}

class Student implements Printable {
    String name;
    int grade;

    Student(String name, int grade) {
        this.name = name;
        this.grade = grade;
    }

    @Override
    public void printInfo() {
        System.out.println("[学生] " + name + " - " + grade + "年生");
    }
}

class Teacher implements Printable {
    String name;
    String subject;

    Teacher(String name, String subject) {
        this.name = name;
        this.subject = subject;
    }

    @Override
    public void printInfo() {
        System.out.println("[教師] " + name + " - " + subject + "担当");
    }
}

public class PrintableTest {
    public static void main(String[] args) {
        Printable student = new Student("田中太郎", 3);
        Printable teacher = new Teacher("鈴木先生", "数学");
        student.printInfo();
        teacher.printInfo();
    }
}
```

</details>

### 問題4：社員クラスの階層

`Employee` クラスを親クラスとして、`Manager` クラスと `Engineer` クラスを作成してください。各クラスで `getBonus()` メソッドを実装し、役職に応じたボーナスを計算してください。

- Employee: 基本給 × 1.0
- Manager: 基本給 × 2.0
- Engineer: 基本給 × 1.5

**期待される出力例：**
```
佐藤（一般社員）: 基本給 250,000円 → ボーナス 250,000円
山田（マネージャー）: 基本給 350,000円 → ボーナス 700,000円
田中（エンジニア）: 基本給 300,000円 → ボーナス 450,000円
```

<details>
<summary>ヒント</summary>

- `Employee` に `name`, `baseSalary` フィールドと `getBonus()` メソッドを定義します
- 子クラスで `getBonus()` をオーバーライドして倍率を変えます
- `String.format("%,d", value)` で3桁カンマ区切り表示ができます

</details>

<details>
<summary>解答例</summary>

```java
class Employee {
    String name;
    int baseSalary;

    Employee(String name, int baseSalary) {
        this.name = name;
        this.baseSalary = baseSalary;
    }

    /* ボーナスを計算（デフォルトは基本給×1.0） */
    int getBonus() {
        return baseSalary;
    }

    String getRole() {
        return "一般社員";
    }

    void printInfo() {
        System.out.printf("%s（%s）: 基本給 %,d円 → ボーナス %,d円%n",
            name, getRole(), baseSalary, getBonus());
    }
}

class Manager extends Employee {
    Manager(String name, int baseSalary) {
        super(name, baseSalary);
    }

    @Override
    int getBonus() {
        return baseSalary * 2;
    }

    @Override
    String getRole() {
        return "マネージャー";
    }
}

class Engineer extends Employee {
    Engineer(String name, int baseSalary) {
        super(name, baseSalary);
    }

    @Override
    int getBonus() {
        return (int)(baseSalary * 1.5);
    }

    @Override
    String getRole() {
        return "エンジニア";
    }
}

public class EmployeeTest {
    public static void main(String[] args) {
        Employee[] employees = {
            new Employee("佐藤", 250000),
            new Manager("山田", 350000),
            new Engineer("田中", 300000)
        };

        for (Employee e : employees) {
            e.printInfo();
        }
    }
}
```

</details>

### 問題5：Comparable インターフェースの実装

`Score` クラスを作成し、`Comparable<Score>` を実装して、得点順にソートできるようにしてください。

**期待される出力例：**
```
ソート前: [田中:75, 鈴木:92, 佐藤:88, 山田:65]
ソート後: [山田:65, 田中:75, 佐藤:88, 鈴木:92]
```

<details>
<summary>ヒント</summary>

- `Comparable<Score>` を実装し、`compareTo` メソッドを定義します
- `compareTo` では `this.point - other.point` を返すと昇順になります
- `java.util.Arrays.sort()` でソートできます

</details>

<details>
<summary>解答例</summary>

```java
import java.util.Arrays;

class Score implements Comparable<Score> {
    String name;
    int point;

    Score(String name, int point) {
        this.name = name;
        this.point = point;
    }

    /* 得点の昇順で比較 */
    @Override
    public int compareTo(Score other) {
        return this.point - other.point;
    }

    @Override
    public String toString() {
        return name + ":" + point;
    }
}

public class ScoreSort {
    public static void main(String[] args) {
        Score[] scores = {
            new Score("田中", 75),
            new Score("鈴木", 92),
            new Score("佐藤", 88),
            new Score("山田", 65)
        };

        System.out.println("ソート前: " + Arrays.toString(scores));
        Arrays.sort(scores);
        System.out.println("ソート後: " + Arrays.toString(scores));
    }
}
```

</details>

### 問題6：デフォルトメソッド付きインターフェース

`Logger` インターフェースを作成し、デフォルトメソッドで共通のログフォーマットを提供してください。`ConsoleLogger` と `SimpleLogger` で実装を変えてください。

**期待される出力例：**
```
[2026-03-29] [INFO] アプリケーション起動
[SIMPLE] アプリケーション起動
```

<details>
<summary>ヒント</summary>

- インターフェースに `default` メソッドを定義できます（Java 8+）
- `java.time.LocalDate.now()` で現在日付を取得できます
- `log(String level, String message)` を抽象メソッドにします

</details>

<details>
<summary>解答例</summary>

```java
import java.time.LocalDate;

interface Logger {
    /* 抽象メソッド：各クラスで実装 */
    void log(String level, String message);

    /* デフォルトメソッド：info レベルのショートカット */
    default void info(String message) {
        log("INFO", message);
    }

    /* デフォルトメソッド：error レベルのショートカット */
    default void error(String message) {
        log("ERROR", message);
    }
}

class ConsoleLogger implements Logger {
    @Override
    public void log(String level, String message) {
        System.out.printf("[%s] [%s] %s%n", LocalDate.now(), level, message);
    }
}

class SimpleLogger implements Logger {
    @Override
    public void log(String level, String message) {
        System.out.printf("[%s] %s%n", level.substring(0, Math.min(level.length(), 6)), message);
    }
}

public class LoggerTest {
    public static void main(String[] args) {
        Logger console = new ConsoleLogger();
        Logger simple = new SimpleLogger();

        console.info("アプリケーション起動");
        simple.info("アプリケーション起動");
    }
}
```

</details>

## チャレンジ問題

### 問題7：メディアライブラリ

以下の構造でメディアライブラリシステムを作成してください。

- `Playable` インターフェース：`play()` メソッドと `getDuration()` メソッド
- `MediaItem` 抽象クラス：`title` フィールドと `getInfo()` 抽象メソッド
- `Music` クラス：Playable を実装し MediaItem を継承。アーティスト名を持つ
- `Video` クラス：Playable を実装し MediaItem を継承。解像度を持つ

全メディアを一覧表示し、合計再生時間を計算してください。

**期待される出力例：**
```
=== メディアライブラリ ===
♪ 夜に駆ける / YOASOBI (3分45秒)
  → 再生中... ♪
▶ Java入門講座 / 1080p (15分30秒)
  → 再生中... ▶
♪ Lemon / 米津玄師 (4分16秒)
  → 再生中... ♪

合計再生時間: 23分31秒
```

<details>
<summary>ヒント</summary>

- `getDuration()` は秒数を返し、表示時に分秒に変換します
- `MediaItem` を継承しつつ `Playable` を実装する形にします
- ポリモーフィズムを使って `Playable[]` や `MediaItem[]` で統一的に扱えます

</details>

<details>
<summary>解答例</summary>

```java
/* Playable インターフェース */
interface Playable {
    void play();
    int getDuration();  /* 秒数で返す */
}

/* MediaItem 抽象クラス */
abstract class MediaItem {
    String title;

    MediaItem(String title) {
        this.title = title;
    }

    abstract String getInfo();
}

/* 音楽クラス */
class Music extends MediaItem implements Playable {
    String artist;
    int durationSec;

    Music(String title, String artist, int durationSec) {
        super(title);
        this.artist = artist;
        this.durationSec = durationSec;
    }

    @Override
    public void play() {
        System.out.println("  → 再生中... ♪");
    }

    @Override
    public int getDuration() {
        return durationSec;
    }

    @Override
    String getInfo() {
        return String.format("♪ %s / %s (%d分%d秒)",
            title, artist, durationSec / 60, durationSec % 60);
    }
}

/* 動画クラス */
class Video extends MediaItem implements Playable {
    String resolution;
    int durationSec;

    Video(String title, String resolution, int durationSec) {
        super(title);
        this.resolution = resolution;
        this.durationSec = durationSec;
    }

    @Override
    public void play() {
        System.out.println("  → 再生中... ▶");
    }

    @Override
    public int getDuration() {
        return durationSec;
    }

    @Override
    String getInfo() {
        return String.format("▶ %s / %s (%d分%d秒)",
            title, resolution, durationSec / 60, durationSec % 60);
    }
}

public class MediaLibrary {
    public static void main(String[] args) {
        /* メディアアイテムの配列（ポリモーフィズム） */
        MediaItem[] items = {
            new Music("夜に駆ける", "YOASOBI", 225),
            new Video("Java入門講座", "1080p", 930),
            new Music("Lemon", "米津玄師", 256)
        };

        System.out.println("=== メディアライブラリ ===");
        int totalSec = 0;

        for (MediaItem item : items) {
            System.out.println(item.getInfo());
            if (item instanceof Playable) {
                Playable p = (Playable) item;
                p.play();
                totalSec += p.getDuration();
            }
        }

        System.out.printf("%n合計再生時間: %d分%d秒%n", totalSec / 60, totalSec % 60);
    }
}
```

</details>

### 問題8：複数インターフェースの実装

`Saveable`（保存可能）と `Loadable`（読み込み可能）の2つのインターフェースを定義し、`Document` クラスで両方を実装してください。保存と読み込みの動作をシミュレーションしてください。

**期待される出力例：**
```
[保存] "企画書.docx" を保存しました（1024バイト）
[読み込み] "企画書.docx" を読み込みました
内容: これは企画書の内容です。
[保存] "報告書.pdf" を保存しました（2048バイト）
[読み込み] "報告書.pdf" を読み込みました
内容: これは報告書の内容です。
```

<details>
<summary>ヒント</summary>

- `Saveable` に `void save()` メソッド
- `Loadable` に `String load()` メソッド
- `Document` で `implements Saveable, Loadable` とします

</details>

<details>
<summary>解答例</summary>

```java
interface Saveable {
    void save();
}

interface Loadable {
    String load();
}

class Document implements Saveable, Loadable {
    String filename;
    String content;
    int size;

    Document(String filename, String content, int size) {
        this.filename = filename;
        this.content = content;
        this.size = size;
    }

    @Override
    public void save() {
        System.out.printf("[保存] \"%s\" を保存しました（%dバイト）%n", filename, size);
    }

    @Override
    public String load() {
        System.out.printf("[読み込み] \"%s\" を読み込みました%n", filename);
        return content;
    }
}

public class DocumentTest {
    public static void main(String[] args) {
        Document[] docs = {
            new Document("企画書.docx", "これは企画書の内容です。", 1024),
            new Document("報告書.pdf", "これは報告書の内容です。", 2048)
        };

        for (Document doc : docs) {
            doc.save();
            String content = doc.load();
            System.out.println("内容: " + content);
        }
    }
}
```

</details>
