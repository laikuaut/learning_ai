# 第2章 演習：ノーマルモードの操作

この演習では、カーソル移動、単語移動、削除、コピーペースト、ドットコマンドを実践的に学びます。

---

## 基本問題

### 問題1：hjkl移動

以下のテキストをVimで開き、`h`/`j`/`k`/`l` のみでカーソルを移動してください。

```
ABCDEFGHIJ
KLMNOPQRST
UVWXYZ0123
```

1. `A` の位置から `F` に移動するには？
2. `A` の位置から `K` に移動するには？
3. `S` の位置から `N` に移動するには？

**期待される解答例：**

```
1. lllll または 5l
2. j
3. hhhhh または 5h
```

<details>
<summary>ヒント</summary>

`h`：左、`j`：下、`k`：上、`l`：右です。数字を前に付けると繰り返せます（例：`5l`）。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 1. A → F：右に5文字移動
5l       " 数値プレフィックスで一気に移動できます

" 2. A → K：1行下に移動
j        " j/k は同じ列位置を保って上下移動します

" 3. S → N：左に5文字移動
5h       " h を5回押すのと同じです
```

数値プレフィックスを活用すると、`h`/`j`/`k`/`l` でも効率的に移動できます。

</details>

### 問題2：単語移動

以下のコードでカーソルが行頭（`c`）にあるとき、各操作後のカーソル位置を答えてください。

```
const message = "Hello World";
```

1. `w` を1回押す
2. `w` を3回押す
3. `e` を1回押す
4. 行末 `;` から `b` を2回押す

**期待される解答例：**

```
1. m（message の先頭）
2. "（ダブルクォートの位置。= も単語扱い）
3. t（const の末尾）
4. World の先頭
```

<details>
<summary>ヒント</summary>

- `w`（word）：次の単語の先頭
- `b`（back）：前の単語の先頭
- `e`（end）：現在/次の単語の末尾
- `=` や `"` などの記号も「単語」として扱われます

</details>

<details>
<summary>解答と解説</summary>

```vim
" 1. w → m（message の先頭）
"    次の単語の先頭にジャンプします

" 2. www → "（ダブルクォート）
"    1回目: m (message)、2回目: = (記号も単語)、3回目: " (ダブルクォート)

" 3. e → t（const の末尾）
"    現在の単語の末尾に移動します

" 4. ;からbb → World の先頭
"    b は前の単語の先頭に戻ります
"    1回目: " (閉じ引用符)、2回目: World の先頭
```

`W`/`B`/`E`（大文字）は空白区切りの「WORD」単位で移動します。記号を飛ばしたい場合に便利です。

</details>

### 問題3：削除操作

以下のテキストで指定された削除を行うコマンドを答えてください。

```
This is a sample text for Vim practice.
```

1. カーソルが `s`（sample）にあるとき、`sample` を削除する
2. 行全体を削除する
3. カーソルが `f`（for）にあるとき、カーソルから行末まで削除する

**期待される解答例：**

```
1. dw    2. dd    3. D（または d$）
```

<details>
<summary>ヒント</summary>

`d` は削除オペレータです。`dw`：単語削除、`dd`：行削除、`D`/`d$`：行末まで削除です。

</details>

<details>
<summary>解答と解説</summary>

```vim
dw   " d（削除）+ w（次の単語まで）。空白も削除されます
     " daw なら周囲の空白を適切に処理して削除します
dd   " d を2回で行全体を削除。削除内容はレジスタに保存されます
D    " d$ のショートカット。カーソル位置から行末まで削除します
```

</details>

## 応用問題

### 問題4：数値プレフィックスの活用

以下のテキスト（10行の `line 1` 〜 `line 10`）に対して操作してください。

```
line 1
line 2
line 3
line 4
line 5
```

1. 1行目から5行目に一気に移動するコマンドは？
2. 1行目から3行分を一度に削除するコマンドは？
3. `line` の上で `row` に置き換える最も効率的なコマンドは？

**期待される解答例：**

```
1. 4j（下に4行移動）または 5G（5行目へジャンプ）
2. 3dd
3. cw → row → ESC
```

<details>
<summary>ヒント</summary>

数値 + コマンドでその回数分繰り返せます。`c`（change）は削除して挿入モードに入るオペレータです。

</details>

<details>
<summary>解答と解説</summary>

```vim
4j    " 下に4行移動。5G で5行目に直接ジャンプすることも可能です
3dd   " 3行まとめて削除。結果: line 4 以降だけ残ります
cw    " c（変更）+ w（単語）で削除して挿入モードに入ります
      " row と入力して ESC。dw → i → row → ESC より1ステップ少ないです
```

</details>

### 問題5：コピーペースト操作

以下のHTMLコードで操作してください。

```html
<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>
```

1. `<li>りんご</li>` をコピーして `<li>みかん</li>` の下に貼り付ける
2. `<li>みかん</li>` を `<li>りんご</li>` の上に移動する

**期待される出力（問題1）：**

```html
<ul>
  <li>りんご</li>
  <li>みかん</li>
  <li>りんご</li>
</ul>
```

<details>
<summary>ヒント</summary>

`yy`：行コピー、`p`：下に貼り付け、`dd`：行カット、`P`：上に貼り付けです。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 問題1: コピー＆ペースト
yy   " りんご行でコピー（ヤンク）
j    " みかん行に移動
p    " カーソルの下の行に貼り付け

" 問題2: 行の移動
dd   " みかん行で削除（カット）。レジスタに保存されます
k    " りんご行に移動
P    " カーソルの上の行に貼り付け（大文字 P）

" Vimでは dd で削除した内容はレジスタに保存されるため
" 「カット＆ペースト」として使えます
```

</details>

## チャレンジ問題

### 問題6：ドットコマンドを使った効率的な編集

すべての `var` を `const` に変更してください。置換コマンド（`:s`）は使わず `.` を活用すること。

```javascript
var name = "Taro";
var age = 25;
var city = "Tokyo";
var hobby = "programming";
var language = "JavaScript";
```

**期待される出力：**

```javascript
const name = "Taro";
const age = 25;
const city = "Tokyo";
const hobby = "programming";
const language = "JavaScript";
```

<details>
<summary>ヒント</summary>

`cw` で単語を変更し、`.` で直前の変更操作を繰り返します。`j0` で次の行の先頭に移動します。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 1行目の var にカーソルを合わせる
cw          " var を削除して挿入モードに入る
const[ESC]  " const と入力して ESC

" 残りの行で繰り返す
j0.         " 次の行の先頭に移動して . で同じ変更を繰り返す
j0.         " 3行目
j0.         " 4行目
j0.         " 5行目

" ドットコマンドのポイント:
" - . は「cw で var を削除し、const を入力して ESC」を丸ごと繰り返します
" - 1回の操作 + 4回の . で5箇所の変更が完了します

" 別解: 検索と組み合わせる方法
" /var[Enter]    → 最初の var を検索
" cwconst[ESC]   → 変更
" n.             → 次の var にジャンプして変更を繰り返し
" n. を繰り返す
```

</details>
