はじめてのAngularJS

AngularJSの基本機能をインタラクティブに体験しましょう

双方向バインディング ng-model

テキストを入力すると、リアルタイムに表示が更新されます

こんにちは、{{ greeting || 'ゲスト' }} さん！AngularJSへようこそ！

カウンター ng-click

ボタンをクリックしてカウントを操作します

{{ count }}
カウントは {{ count >= 0 ? '正の数' : '負の数' }} です

リスト表示 ng-repeat

アイテムを追加・削除できます

  • {{ $index + 1 }}. {{ item }}
アイテムがありません。上のフォームから追加してください。

条件付き表示 ng-if / ng-show

チェックボックスで表示を切り替えます

秘密のメッセージ: AngularJSの学習、頑張りましょう！ (ng-if: DOMに存在)
チェックを入れるとメッセージが表示されます (ng-if: 要素が除去されます)
ng-if は条件が偽のとき DOM から要素を完全に削除します。
ng-show は CSS の display で表示/非表示を切り替えます（DOM には残ります）。
