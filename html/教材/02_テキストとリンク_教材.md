# 第2章：テキストとリンク

## この章で学ぶこと
- 見出しタグ（h1〜h6）の使い方と階層構造

- ハイパーリンク（a要素）の基本と応用
- target属性によるリンクの開き方制御
- 相対パスと絶対パスの違い

## 2.1 見出しタグ（h1〜h6）

HTMLには6段階の見出しレベルがあります。`<h1>` が最も重要な見出し、`<h6>` が最も下位の見出しです。

```html
<h1>メインタイトル（最重要）</h1>
<h2>セクションの見出し</h2>
<h3>サブセクションの見出し</h3>
<h4>小見出し</h4>
<h5>さらに小さい見出し</h5>
<h6>最も小さい見出し</h6>
```

### 見出しの使い方のルール

#### 1. 階層構造を守る
見出しは「アウトライン」（目次）を構成します。レベルを飛ばしてはいけません。

```html
<!-- 良い例 -->
<h1>HTML入門</h1>
  <h2>基本構造</h2>
    <h3>DOCTYPE宣言</h3>
    <h3>head要素</h3>
  <h2>テキスト要素</h2>
    <h3>見出し</h3>
    <h3>段落</h3>

<!-- 悪い例：h2を飛ばしている -->
<h1>HTML入門</h1>
  <h3>基本構造</h3>  <!-- h2を飛ばしてはダメ -->
```

#### 2. h1は1ページに1つだけ
- `<h1>` はページ全体の主題を表す
- 技術的には複数書けるが、SEOとアクセシビリティの観点から**1つが推奨**
- `<h2>` 以降は複数使ってOK

#### 3. 見出しを見た目の調整に使わない
```html
<!-- 悪い例：文字を大きくしたいだけでh1を使っている -->
<h1>お知らせ</h1>
<h3>2024年1月のイベント情報</h3>  <!-- 文字サイズ調整のためにh3を使うのはNG -->

<!-- 良い例：見た目はCSSで調整する -->
<h2>お知らせ</h2>
<h3>2024年1月のイベント情報</h3>
```

### デフォルトの見出しサイズ（参考）
| タグ | デフォルトサイズ（おおよそ） |
|---|---|
| `<h1>` | 2em（32px） |
| `<h2>` | 1.5em（24px） |
| `<h3>` | 1.17em（18.72px） |
| `<h4>` | 1em（16px） |
| `<h5>` | 0.83em（13.28px） |
| `<h6>` | 0.67em（10.72px） |

> **ポイント**: サイズはCSSで自由に変更できます。意味的に正しいレベルを選ぶことが大切です。

## 2.2 段落（p要素）

`<p>` 要素は**段落**（パラグラフ）を表します。テキストの基本的なまとまりです。

```html
<p>HTMLはWebページの構造を記述するマークアップ言語です。</p>
<p>CSSはWebページの見た目を装飾するスタイルシート言語です。</p>
<p>JavaScriptはWebページに動きを加えるプログラミング言語です。</p>
```

### p要素の特徴
- **ブロック要素**：前後に改行が入り、横幅いっぱいに広がる
- 段落間にはデフォルトで**マージン**（余白）が入る
- p要素の中にブロック要素（div, h1, ul等）を入れることはできない

```html
<!-- OK -->
<p>テキストと<strong>強調テキスト</strong>は入れられる。</p>

<!-- NG：p要素の中にdivは入れられない -->
<p>
  <div>これはエラー</div>
</p>
```

## 2.3 改行（br要素）と水平線（hr要素）

### br要素：改行
`<br>` は文中で**強制的に改行**するための要素です。

```html
<p>
  〒100-0001<br>
  東京都千代田区千代田1-1<br>
  皇居
</p>
```

#### brの注意点
- `<br>` は**空要素**（終了タグなし）
- 段落の間隔を空けるために `<br>` を連続で使うのは**NG**
- 見た目の調整はCSSのmarginやpaddingで行う

```html
<!-- 悪い例 -->
<p>最初の段落</p>
<br><br><br>
<p>次の段落</p>

<!-- 良い例：CSSで余白を制御 -->
<p style="margin-bottom: 3em;">最初の段落</p>
<p>次の段落</p>
```

### hr要素：水平線（テーマの区切り）
`<hr>` は**テーマの区切り**を表す水平線を描画します。

```html
<h2>第1部：基礎編</h2>
<p>基礎的な内容を学びます。</p>
<hr>
<h2>第2部：応用編</h2>
<p>応用的な内容を学びます。</p>
```

- `<hr>` も空要素
- 視覚的な線引きではなく、**意味的な区切り**を表す
- 装飾目的の線はCSSの `border` を使う

## 2.4 テキストの強調と装飾

### strong要素：重要なテキスト
`<strong>` は**重要性**を持つテキストを表します。デフォルトでは太字で表示されます。

```html
<p><strong>注意：</strong>この操作は取り消せません。</p>
```

### em要素：強調テキスト
`<em>` は**強勢**（アクセント）を付けるテキストを表します。デフォルトでは斜体で表示されます。

```html
<p>HTMLは<em>構造</em>を、CSSは<em>見た目</em>を担当します。</p>
```

### strongとemの違い
| 要素 | 意味 | 見た目（デフォルト） | 使い場面 |
|---|---|---|---|
| `<strong>` | 重要性 | **太字** | 警告、注意事項 |
| `<em>` | 強調 | *斜体* | ニュアンスの違いを伝える |

> **注意**: `<b>` タグ（太字）や `<i>` タグ（斜体）もありますが、これらは「意味を持たない視覚的な装飾」です。意味がある場合は `<strong>` や `<em>` を使いましょう。

### その他のインライン要素

```html
<!-- 小さいテキスト（注意書き、著作権表示など） -->
<p><small>Copyright 2024 Example Inc.</small></p>

<!-- 打ち消し線（古い情報や訂正） -->
<p>価格：<s>5,000円</s> → 3,980円</p>

<!-- 下線付きテキスト（スペルミスの指摘などに使用） -->
<p><u>Teh</u> → The</p>

<!-- マーカー（ハイライト） -->
<p>ここが<mark>重要なポイント</mark>です。</p>

<!-- 上付き文字・下付き文字 -->
<p>E = mc<sup>2</sup></p>
<p>H<sub>2</sub>O</p>

<!-- 引用 -->
<p>彼は<q>HTMLは楽しい</q>と言いました。</p>

<!-- 略語 -->
<p><abbr title="HyperText Markup Language">HTML</abbr>を学びましょう。</p>

<!-- コード -->
<p>変数は<code>let x = 10;</code>のように宣言します。</p>

<!-- キーボード入力 -->
<p><kbd>Ctrl</kbd> + <kbd>C</kbd> でコピーできます。</p>

<!-- 日時 -->
<p>記事公開日：<time datetime="2024-01-15">2024年1月15日</time></p>
```

## 2.5 ハイパーリンク（a要素）

リンクはWebの最も基本的な機能です。`<a>`（anchor）要素で作成します。

### 基本構文
```html
<a href="リンク先のURL">リンクテキスト</a>
```

### 基本的な使い方
```html
<!-- 外部サイトへのリンク -->
<a href="https://www.google.com">Googleへ</a>

<!-- 同じサイト内のページへのリンク -->
<a href="about.html">自己紹介ページ</a>

<!-- メールリンク -->
<a href="mailto:info@example.com">メールを送る</a>

<!-- 電話リンク（スマートフォン向け） -->
<a href="tel:03-1234-5678">03-1234-5678</a>
```

## 2.6 target属性：リンクの開き方

```html
<!-- 同じタブで開く（デフォルト） -->
<a href="https://example.com">同じタブで開く</a>

<!-- 新しいタブで開く -->
<a href="https://example.com" target="_blank">新しいタブで開く</a>
```

### target属性の値
| 値 | 動作 |
|---|---|
| `_self` | 同じタブで開く（デフォルト） |
| `_blank` | 新しいタブ（ウィンドウ）で開く |
| `_parent` | 親フレームで開く |
| `_top` | 最上位フレームで開く |

### セキュリティ対策：rel="noopener noreferrer"
`target="_blank"` を使う場合は、セキュリティのために `rel` 属性を追加します。

```html
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  外部サイト
</a>
```

- **noopener**: 新しいタブから元のページを操作できないようにする
- **noreferrer**: リファラー情報（どこから来たか）を送信しない
- 現代のブラウザでは `target="_blank"` に対して `noopener` が自動適用されるが、明示的に書くのが安全

## 2.7 ページ内リンク（アンカーリンク）

同じページ内の特定の位置にジャンプするリンクです。

```html
<!-- リンク元：#でid属性を指定 -->
<a href="#section1">セクション1へジャンプ</a>
<a href="#section2">セクション2へジャンプ</a>

<!-- リンク先：id属性を付けた要素 -->
<h2 id="section1">セクション1</h2>
<p>セクション1の内容...</p>

<h2 id="section2">セクション2</h2>
<p>セクション2の内容...</p>

<!-- ページトップに戻る -->
<a href="#top">ページの先頭に戻る</a>
```

### id属性のルール
- ページ内で**一意**（重複不可）
- 半角英数字、ハイフン、アンダースコアを使用
- **数字から始めてはいけない**（CSS セレクタの制約）
- 大文字と小文字は区別される

## 2.8 相対パスと絶対パス

### 絶対パス
完全なURLを指定する方法です。

```html
<!-- 外部サイトへの絶対パス -->
<a href="https://www.example.com/about.html">About</a>

<!-- 同じサーバーのルートからの絶対パス -->
<a href="/images/logo.png">ロゴ</a>
```

### 相対パス
現在のファイルの位置を基準にする方法です。

```html
<!-- 同じフォルダのファイル -->
<a href="about.html">About</a>

<!-- 子フォルダのファイル -->
<a href="pages/contact.html">Contact</a>

<!-- 親フォルダのファイル -->
<a href="../index.html">トップに戻る</a>

<!-- 親の親フォルダ -->
<a href="../../assets/style.css">CSS</a>
```

### フォルダ構造の例

```
my-website/
├── index.html          ← 現在のファイル
├── about.html
├── pages/
│   ├── contact.html
│   └── blog.html
├── images/
│   ├── photo.jpg
│   └── logo.png
└── css/
    └── style.css
```

この場合、`index.html` から見た相対パス：
| ファイル | 相対パス |
|---|---|
| about.html | `about.html` または `./about.html` |
| contact.html | `pages/contact.html` |
| photo.jpg | `images/photo.jpg` |
| style.css | `css/style.css` |

`pages/contact.html` から見た相対パス：
| ファイル | 相対パス |
|---|---|
| index.html | `../index.html` |
| about.html | `../about.html` |
| photo.jpg | `../images/photo.jpg` |
| blog.html | `blog.html` |

### 相対パスと絶対パスの使い分け
| 種類 | メリット | デメリット | 使い場面 |
|---|---|---|---|
| 相対パス | ファイル移動に柔軟 | 階層が深いと複雑 | サイト内リンク |
| 絶対パス（URL） | 常に正確 | サーバー変更時に修正必要 | 外部サイトリンク |
| 絶対パス（ルート相対） | どこからでも同じ指定 | ローカルで動かない | 大規模サイト |

## 2.9 リンクの応用テクニック

### 画像をリンクにする
```html
<a href="https://example.com">
  <img src="logo.png" alt="Example.comのロゴ">
</a>
```

### ダウンロードリンク
```html
<a href="files/document.pdf" download>PDFをダウンロード</a>
<a href="files/data.csv" download="レポート.csv">CSVをダウンロード</a>
```

### リンクにツールチップを表示
```html
<a href="glossary.html" title="用語集ページへ移動します">用語集</a>
```

### ブロック要素をリンクにする（HTML5以降）
```html
<a href="article.html">
  <article>
    <h2>記事のタイトル</h2>
    <p>記事の概要テキスト...</p>
  </article>
</a>
```

## 2.10 特殊文字（文字実体参照）

HTMLで特別な意味を持つ文字を表示するには、**文字実体参照**を使います。

| 表示 | 記述 | 名前 |
|---|---|---|
| `<` | `&lt;` | 小なり |
| `>` | `&gt;` | 大なり |
| `&` | `&amp;` | アンパサンド |
| `"` | `&quot;` | ダブルクォート |
| `'` | `&apos;` | シングルクォート |
| （空白） | `&nbsp;` | ノーブレークスペース |
| `©` | `&copy;` | コピーライト |
| `®` | `&reg;` | 登録商標 |
| `™` | `&trade;` | 商標 |
| `¥` | `&yen;` | 円記号 |
| `→` | `&rarr;` | 右矢印 |
| `←` | `&larr;` | 左矢印 |

```html
<p>HTMLのタグは &lt;タグ名&gt; のように書きます。</p>
<p>&copy; 2024 My Website. All rights reserved.</p>
<p>価格：&yen;1,000</p>
```

## まとめ

| 要素 | 用途 | 例 |
|---|---|---|
| `<h1>`〜`<h6>` | 見出し（6段階） | `<h1>タイトル</h1>` |
| `<p>` | 段落 | `<p>テキスト</p>` |
| `<br>` | 改行（空要素） | `住所<br>東京都` |
| `<hr>` | テーマの区切り線 | セクション間に使用 |
| `<strong>` | 重要なテキスト | `<strong>警告</strong>` |
| `<em>` | 強調テキスト | `<em>重要</em>` |
| `<a href="">` | ハイパーリンク | `<a href="url">テキスト</a>` |
| `target="_blank"` | 新しいタブで開く | 外部リンクに使用 |
| 相対パス | 現在位置基準のパス | `../images/photo.jpg` |
| 絶対パス | 完全なURL | `https://example.com` |
