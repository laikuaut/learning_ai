## 基本問題

### 問1（四択・SWOT分析）
SWOT分析において、「外部環境のマイナス要因」に該当するものはどれか。

ア 独自の特許技術を保有している
イ 社員のIT人材が不足している
ウ 新しい市場への参入機会がある
エ 競合他社の新規参入が増加している

<details><summary>解答・解説</summary>

**正解: エ**

- ア: 強み（Strengths）- 内部環境のプラス要因
- イ: 弱み（Weaknesses）- 内部環境のマイナス要因
- ウ: 機会（Opportunities）- 外部環境のプラス要因
- エ: 脅威（Threats）- 外部環境のマイナス要因

</details>

### 問2（四択・PPM）
BCGマトリクス（PPM）において、市場成長率が高く市場シェアも高い事業を何というか。

ア 花形（Star）　　イ 金のなる木（Cash Cow）　　ウ 問題児（Question Mark）　　エ 負け犬（Dog）

<details><summary>解答・解説</summary>

**正解: ア 花形（Star）**

| 市場成長率/市場シェア | シェア高 | シェア低 |
|---|---|---|
| 成長率高 | 花形(Star) | 問題児(Question Mark) |
| 成長率低 | 金のなる木(Cash Cow) | 負け犬(Dog) |

</details>

### 問3（四択・BSC）
バランススコアカード（BSC）の4つの視点に含まれないものはどれか。

ア 財務の視点　　イ 技術の視点　　ウ 顧客の視点　　エ 学習と成長の視点

<details><summary>解答・解説</summary>

**正解: イ 技術の視点**

BSCの4つの視点:
1. 財務の視点
2. 顧客の視点
3. 業務プロセスの視点
4. 学習と成長の視点

「技術の視点」はBSCの標準的な4つの視点には含まれません。

</details>

### 問4（四択・EA）
エンタープライズアーキテクチャ（EA）の4つの階層のうち、システムの技術基盤（インフラ）を対象とするものはどれか。

ア ビジネスアーキテクチャ（BA）
イ データアーキテクチャ（DA）
ウ アプリケーションアーキテクチャ（AA）
エ テクノロジアーキテクチャ（TA）

<details><summary>解答・解説</summary>

**正解: エ テクノロジアーキテクチャ（TA）**

EAの4階層:
- BA: 業務プロセスの構造
- DA: データ・情報の構造
- AA: アプリケーションの構造
- TA: 技術基盤（サーバ、ネットワーク等）の構造

</details>

### 問5（四択・SOA）
SOA（サービス指向アーキテクチャ）に関する記述として、正しいものはどれか。

ア 全ての機能を1つのアプリケーションにまとめるアーキテクチャである
イ 業務機能をサービスとして部品化し、組み合わせてシステムを構築するアーキテクチャである
ウ データベースを中心にシステムを構築するアーキテクチャである
エ クライアント側に全ての処理を集約するアーキテクチャである

<details><summary>解答・解説</summary>

**正解: イ**

SOAは業務機能を独立した「サービス」として定義し、それらをネットワーク経由で連携させてシステムを構築するアーキテクチャです。サービスの再利用性と疎結合が特徴です。

アはモノリシックアーキテクチャの説明です。

</details>

## 応用問題

### 問6（四択・NPV）
あるIT投資案件で、初期投資が500万円、年間キャッシュフローが200万円（3年間）、割引率が10%の場合のNPVに最も近い値はどれか。ただし、1/(1.1)=0.909、1/(1.1)²=0.826、1/(1.1)³=0.751とする。

ア -3万円　　イ 0万円　　ウ -3万円　　エ -3万円

（注: 選択肢を正しく計算して判定してください）

ア 約-3万円　　イ 約+3万円　　ウ 約-50万円　　エ 約+100万円

<details><summary>ヒント</summary>

NPV = Σ{CFt/(1+r)^t} - 初期投資

</details>

<details><summary>解答・解説</summary>

**正解: ア 約-3万円**

```
NPV = 200×0.909 + 200×0.826 + 200×0.751 - 500
    = 181.8 + 165.2 + 150.2 - 500
    = 497.2 - 500
    = -2.8万円 ≈ 約-3万円
```

NPV < 0 なので、この投資案件は割引率10%の条件では投資すべきではないと判断されます。ただし、4年目以降もキャッシュフローが見込める場合は再計算が必要です。

</details>

### 問7（四択・DX）
DX（デジタルトランスフォーメーション）の3段階のうち、「ビジネスモデル自体を根本的に変革すること」に該当する段階はどれか。

ア デジタイゼーション
イ デジタライゼーション
ウ デジタルトランスフォーメーション
エ デジタルオプティマイゼーション

<details><summary>解答・解説</summary>

**正解: ウ デジタルトランスフォーメーション**

3段階の違い:
- デジタイゼーション: アナログ情報のデジタル化（例: 紙の帳票を電子化）
- デジタライゼーション: デジタル技術で個別の業務プロセスを改善（例: ワークフローシステムの導入）
- デジタルトランスフォーメーション: デジタル技術によるビジネスモデル自体の変革（例: 製造業がサブスクリプションモデルに転換）

</details>

### 問8（記述・SWOT分析）
あるソフトウェア開発会社のSWOT分析結果が以下の通りである。クロスSWOT分析を行い、4つの戦略を立案せよ。

- **強み(S):** AI技術に精通したエンジニアが多い、自社開発のAIフレームワークを保有
- **弱み(W):** 営業部門が弱い、大企業との取引実績が少ない
- **機会(O):** AI需要の急増、政府のDX推進補助金
- **脅威(T):** 大手IT企業のAI市場参入、エンジニアの人材不足・引き抜き

<details><summary>解答・解説</summary>

**SO戦略（強み×機会）: 積極戦略**
AI技術力と自社フレームワークを活かし、急増するAI需要を取り込む。政府のDX補助金を活用して中小企業向けのAI導入支援サービスを展開する。自社フレームワークをSaaS化して提供することで、市場シェアの拡大を図る。

**WO戦略（弱み×機会）: 改善戦略**
DX推進の流れに乗るため、営業力を強化する。具体的には、AI導入のセミナーや事例紹介を通じたインバウンド営業を強化する。大企業との取引実績を増やすため、DX補助金対象の実証事業に参加して実績を作る。

**ST戦略（強み×脅威）: 差別化戦略**
大手IT企業との差別化として、自社AIフレームワークのカスタマイズ性や特定業界への専門性を訴求する。エンジニアの引き抜き対策として、技術的な裁量の高さや研究開発への投資を強みとした待遇・環境整備を行い人材を定着させる。

**WT戦略（弱み×脅威）: 防衛戦略**
大手との直接競合を避け、特定のニッチ市場（医療AI、農業AI等）に特化する。少数精鋭で運営できるサービスモデルに集中し、営業リソースの不足を補うパートナー戦略（SI企業との協業）を推進する。

</details>

## チャレンジ問題

### 問9（記述・午後試験形式）
J社の情報システム戦略について、以下の設問に答えなさい。

J社は製造業で、以下のシステムを運用している。
- 基幹系システム（ERP）: 会計、生産管理、在庫管理（15年稼働）
- 営業支援システム（SFA）: 営業活動管理（5年稼働）
- 顧客向けECサイト: 製品の直販（3年稼働）

経営層はDXを推進し、データ活用による意思決定の高度化を目指している。

**設問1:** J社のシステムをSoR/SoE/SoIの観点で分類し、各システムの特徴と今後の方向性を述べよ。

**設問2:** データ活用基盤の構築について、EAの4階層（BA/DA/AA/TA）それぞれの観点からの検討事項を述べよ。

<details><summary>解答・解説</summary>

**設問1の解答:**

| システム | 分類 | 特徴 | 今後の方向性 |
|---------|------|------|------------|
| 基幹系(ERP) | SoR | 会計・生産・在庫の記録を正確に管理。安定性・正確性重視。ただし15年稼働で技術的負債が蓄積 | 段階的なモダナイゼーション（クラウドERP移行やAPI化）により、データ連携可能な基盤に更新する |
| 営業支援(SFA) | SoE | 営業担当者と顧客の関わりを管理。顧客体験の向上とスピード重視 | モバイル対応の強化、AIによる営業活動の予測・推薦機能の追加を検討する |
| 顧客向けECサイト | SoE | 顧客との直接的な接点。顧客体験・UI/UX重視 | 顧客行動データの分析によるパーソナライゼーション、レコメンデーション機能の強化を図る |

さらに、SoI（洞察のシステム）として、上記3システムのデータを統合するデータ活用基盤（BIツール、データウェアハウス、AI分析基盤）を新規構築する必要があります。

**設問2の解答:**

**BA（ビジネスアーキテクチャ）:**
- データ活用による意思決定プロセスの定義。どの業務プロセスでどのようなデータ分析が必要かを整理する
- データ活用の推進体制（データガバナンス組織、データスチュワードの設置）を検討する

**DA（データアーキテクチャ）:**
- 3つのシステムのデータ定義の統一（マスタデータ管理）。顧客コード、商品コードなどの共通コード体系を策定する
- データウェアハウス/データレイクのスキーマ設計。分析に必要なデータモデルを定義する
- データ品質の管理ルール（クレンジング、バリデーション）を策定する

**AA（アプリケーションアーキテクチャ）:**
- ETL/ELTツールによる各システムからのデータ抽出・変換・ロードの設計
- BIツール/ダッシュボードの導入計画
- 基幹系ERPのAPI化（データ連携のインタフェース設計）
- AIモデルの開発・運用基盤（MLOps）の検討

**TA（テクノロジアーキテクチャ）:**
- データ基盤のインフラ選定（オンプレミス vs クラウド）。スケーラビリティを考慮しクラウド（AWS/Azure/GCP）を推奨
- ネットワーク構成（基幹系とデータ基盤間の接続、セキュリティ）
- データのバックアップ・DR設計

</details>

### 問10（記述・午後試験形式）
IT投資の評価について、以下の設問に答えなさい。

K社は2つのIT投資案件（案件Aと案件B）を比較検討している。

| 項目 | 案件A | 案件B |
|------|-------|-------|
| 初期投資 | 3,000万円 | 5,000万円 |
| 年間コスト削減効果 | 1,000万円 | 1,800万円 |
| 効果期間 | 5年 | 5年 |
| 割引率 | 5% | 5% |

ただし、1/(1.05)=0.952、1/(1.05)²=0.907、1/(1.05)³=0.864、1/(1.05)⁴=0.823、1/(1.05)⁵=0.784とする。

**設問1:** 各案件のNPVを計算し、投資すべきか判断せよ。

**設問2:** 各案件の投資回収期間（単純回収期間）を求めよ。

**設問3:** 両案件のうち、K社が優先すべき案件を推薦し、その理由を述べよ。

<details><summary>解答・解説</summary>

**設問1の解答:**

```
現在価値係数の合計 = 0.952 + 0.907 + 0.864 + 0.823 + 0.784 = 4.330

案件A:
  NPV = 1,000 × 4.330 - 3,000 = 4,330 - 3,000 = +1,330万円
  NPV > 0 → 投資すべき

案件B:
  NPV = 1,800 × 4.330 - 5,000 = 7,794 - 5,000 = +2,794万円
  NPV > 0 → 投資すべき
```

**設問2の解答:**

```
案件A: 回収期間 = 3,000 / 1,000 = 3.0年
案件B: 回収期間 = 5,000 / 1,800 ≈ 2.78年
```

**設問3の解答:**

NPVで比較すると案件B（+2,794万円）が案件A（+1,330万円）を上回り、投資回収期間も案件B（2.78年）が案件A（3.0年）より短いため、案件Bを優先すべきです。

ただし、以下の点も考慮が必要です:
- 初期投資額: 案件Bは5,000万円と案件Aの3,000万円より大きい。資金調達の制約がある場合は案件Aの方が実現しやすい
- リスク: 投資額が大きい案件Bの方がリスクも大きい。コスト削減効果が計画通り実現しなかった場合の影響も大きい
- ROI: 案件AのROI=1,330/3,000≈44%、案件BのROI=2,794/5,000≈56%。投資効率は案件Bが上回る

資金に制約がなくリスクを許容できるなら案件B、資金制約やリスク回避を重視するなら案件Aを選択します。

</details>
