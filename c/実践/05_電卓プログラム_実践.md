# 実践課題05：関数型電卓プログラム ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第2章（条件分岐）、第3章（繰り返し）、第5章（関数）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 関数の定義・呼び出し・戻り値を使いこなし、処理を適切に分割する設計力を養う

---

## 完成イメージ

```
===== 関数型電卓 =====
使い方: 数値 演算子 数値（例: 10 + 3）
終了するには q を入力

> 10 + 3
= 13.00

> 100 / 7
= 14.29

> 2 ^ 8
= 256.00

> 10 / 0
エラー: ゼロで割ることはできません。

> 10 % 3
= 1.00

> q
終了します。
```

---

## 課題の要件

1. 四則演算（+, -, *, /）に加え、剰余（%）とべき乗（^）をサポートする
2. 各演算を個別の関数として定義する
3. ゼロ除算のエラーチェックを行う
4. ループで繰り返し計算でき、`q` で終了する
5. 結果は小数点以下2桁で表示する

---

## ステップガイド

<details>
<summary>ステップ1：演算関数を定義する</summary>

各演算を関数にします。まず四則演算から始めましょう。

```c
double add(double a, double b) {
    return a + b;
}

double subtract(double a, double b) {
    return a - b;
}

double multiply(double a, double b) {
    return a * b;
}
```

</details>

<details>
<summary>ステップ2：除算のエラーチェック</summary>

ゼロ除算を検出するために、成功/失敗を返す工夫が必要です。
簡単な方法として、特別な値を使うか、ポインタで結果を返す方法があります。

```c
/* 簡易版: 成功なら1, 失敗なら0を返し、結果はポインタで受け取る */
int divide_safe(double a, double b, double *result) {
    if (b == 0.0) {
        return 0;  /* 失敗 */
    }
    *result = a / b;
    return 1;  /* 成功 */
}
```

</details>

<details>
<summary>ステップ3：メインループと入力処理</summary>

`scanf` で数値と演算子を読み取り、`switch` で処理を振り分けます。

```c
while (1) {
    char op;
    double a, b;

    printf("\n> ");
    /* 最初の文字が 'q' なら終了 */
    char first;
    scanf(" %c", &first);
    if (first == 'q' || first == 'Q') {
        break;
    }
    /* first を数値の一部として戻す */
    ungetc(first, stdin);
    scanf("%lf %c %lf", &a, &op, &b);

    /* op に応じて関数を呼び出す */
}
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```c
/*
 * 関数型電卓プログラム
 * 学べる内容：関数定義、戻り値、switch文、ループ
 * コンパイル：gcc -o calculator calculator.c -lm
 * 実行：./calculator
 */
#include <stdio.h>
#include <math.h>

/* --- 演算関数 --- */

double add(double a, double b) {
    return a + b;
}

double subtract(double a, double b) {
    return a - b;
}

double multiply(double a, double b) {
    return a * b;
}

double power(double base, double exp) {
    return pow(base, exp);
}

double modulo(double a, double b) {
    return (double)((int)a % (int)b);
}

/* --- メイン処理 --- */

int main(void) {
    double a, b, result;
    char op;
    char line[256];

    printf("===== 関数型電卓 =====\n");
    printf("使い方: 数値 演算子 数値（例: 10 + 3）\n");
    printf("終了するには q を入力\n");

    while (1) {
        printf("\n> ");
        fgets(line, sizeof(line), stdin);

        /* 終了判定 */
        if (line[0] == 'q' || line[0] == 'Q') {
            printf("終了します。\n");
            break;
        }

        /* 入力のパース */
        if (sscanf(line, "%lf %c %lf", &a, &op, &b) != 3) {
            printf("入力形式が正しくありません。例: 10 + 3\n");
            continue;
        }

        switch (op) {
            case '+':
                result = add(a, b);
                printf("= %.2f\n", result);
                break;
            case '-':
                result = subtract(a, b);
                printf("= %.2f\n", result);
                break;
            case '*':
                result = multiply(a, b);
                printf("= %.2f\n", result);
                break;
            case '/':
                if (b == 0.0) {
                    printf("エラー: ゼロで割ることはできません。\n");
                } else {
                    result = a / b;
                    printf("= %.2f\n", result);
                }
                break;
            case '%':
                if (b == 0.0) {
                    printf("エラー: ゼロで割ることはできません。\n");
                } else {
                    result = modulo(a, b);
                    printf("= %.2f\n", result);
                }
                break;
            case '^':
                result = power(a, b);
                printf("= %.2f\n", result);
                break;
            default:
                printf("未対応の演算子です: %c\n", op);
                break;
        }
    }

    return 0;
}
```

</details>

<details>
<summary>解答例（改良版 ─ 関数ポインタで演算を管理）</summary>

関数ポインタ（function pointer）を使って演算を配列で管理するバージョンです。
第6章（ポインタ）を学んだ後に挑戦してみてください。

```c
/*
 * 関数型電卓（改良版）
 * 学べる内容：関数ポインタ、構造体による演算テーブル、拡張性の高い設計
 * コンパイル：gcc -o calculator2 calculator2.c -lm
 * 実行：./calculator2
 */
#include <stdio.h>
#include <math.h>

/* 演算関数の型定義 */
typedef double (*operation_fn)(double, double);

/* 演算関数 */
double op_add(double a, double b)      { return a + b; }
double op_subtract(double a, double b) { return a - b; }
double op_multiply(double a, double b) { return a * b; }
double op_divide(double a, double b)   { return a / b; }
double op_modulo(double a, double b)   { return (double)((int)a % (int)b); }
double op_power(double a, double b)    { return pow(a, b); }

/* 演算テーブル */
typedef struct {
    char symbol;
    operation_fn func;
    int check_zero;  /* 1ならゼロ除算チェックが必要 */
} Operation;

Operation operations[] = {
    {'+', op_add,      0},
    {'-', op_subtract, 0},
    {'*', op_multiply, 0},
    {'/', op_divide,   1},
    {'%', op_modulo,   1},
    {'^', op_power,    0},
};
int num_operations = sizeof(operations) / sizeof(operations[0]);

/* 演算子に対応する Operation を検索 */
Operation *find_operation(char symbol) {
    int i;
    for (i = 0; i < num_operations; i++) {
        if (operations[i].symbol == symbol) {
            return &operations[i];
        }
    }
    return NULL;
}

int main(void) {
    double a, b;
    char op;
    char line[256];

    printf("===== 関数型電卓（改良版）=====\n");
    printf("使い方: 数値 演算子 数値（例: 10 + 3）\n");
    printf("対応演算子: ");
    int i;
    for (i = 0; i < num_operations; i++) {
        printf("%c ", operations[i].symbol);
    }
    printf("\n終了するには q を入力\n");

    while (1) {
        printf("\n> ");
        fgets(line, sizeof(line), stdin);

        if (line[0] == 'q' || line[0] == 'Q') {
            printf("終了します。\n");
            break;
        }

        if (sscanf(line, "%lf %c %lf", &a, &op, &b) != 3) {
            printf("入力形式が正しくありません。\n");
            continue;
        }

        Operation *oper = find_operation(op);
        if (oper == NULL) {
            printf("未対応の演算子です: %c\n", op);
            continue;
        }

        if (oper->check_zero && b == 0.0) {
            printf("エラー: ゼロで割ることはできません。\n");
            continue;
        }

        double result = oper->func(a, b);
        printf("= %.2f\n", result);
    }

    return 0;
}
```

**初心者向けとの違い:**
- 関数ポインタで演算を統一的に管理 → 新しい演算子を追加するとき、配列に1行足すだけ
- 構造体で演算子・関数・ゼロチェックフラグをセットにまとめている → switch 文の繰り返しが不要
- `find_operation` で演算子を検索する設計 → 処理の流れがシンプルになる

</details>
