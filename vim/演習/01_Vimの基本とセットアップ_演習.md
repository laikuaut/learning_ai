# 第1章 演習：Vimの基本とセットアップ

この演習では、Vimの起動・終了、モードの概念、基本操作を実践的に学びます。

---

## 基本問題

### 問題1：モードの理解

Vimの主要な3つのモードについて、以下の表の空欄を埋めてください。

| モード名 | 役割 | 入り方 | 抜け方 |
|---|---|---|---|
| ノーマルモード | （A） | （B） | - |
| 挿入モード | （C） | （D） | （E） |
| コマンドラインモード | （F） | （G） | （H） |

**期待される解答例：**

| モード名 | 役割 | 入り方 | 抜け方 |
|---|---|---|---|
| ノーマルモード | カーソル移動・テキスト操作 | Vim起動時のデフォルト / `ESC` | - |
| 挿入モード | テキスト入力 | `i` | `ESC` |
| コマンドラインモード | 保存・終了等のコマンド実行 | `:` | `Enter` または `ESC` |

<details>
<summary>ヒント</summary>

Vimを起動した直後はノーマルモードです。テキスト入力には挿入モード、保存や終了にはコマンドラインモードを使います。

</details>

<details>
<summary>解答と解説</summary>

```
(A) カーソル移動・テキスト操作  # Vimのホームポジション。すべての操作の起点です
(B) Vim起動時のデフォルト       # 他モードからは ESC で戻ります
(C) テキスト入力               # 実際に文字を打ち込むモードです
(D) i（insert の頭文字）        # カーソル位置の前から入力が始まります
(E) ESC                        # 挿入モードから抜ける唯一の方法です
(F) 保存・終了等のコマンド実行   # :w や :q などを入力します
(G) :（コロン）                 # 画面下部にコマンド入力欄が現れます
(H) Enter（実行）または ESC     # 実行かキャンセルの2通りです
```

</details>

### 問題2：起動と終了の操作

以下の操作で入力するコマンドを答えてください。

1. ターミナルからVimを起動する（ファイル名指定なし）
2. Vimを保存せずに終了する
3. `hello.txt` を指定してVimを起動する
4. ファイルを保存して終了する

**期待される解答例：**

```
1. vim
2. :q
3. vim hello.txt
4. :wq
```

<details>
<summary>ヒント</summary>

`:q` は quit、`:w` は write、`:wq` は保存して終了の組み合わせです。

</details>

<details>
<summary>解答と解説</summary>

```bash
vim            # ファイル名なしで起動。無名バッファが開きます
:q             # 変更がない場合にそのまま終了します
vim hello.txt  # ファイルが存在しなければ新規作成モードで開きます
:wq            # :w（保存）と :q（終了）の連続実行。ZZ でも同じ動作です
```

テキストを編集した後に保存せず終了したい場合は `:q!` を使います。`!` は「強制」を意味します。

</details>

### 問題3：保存コマンドの使い分け

各シナリオで使うべきコマンドを答えてください。

| シナリオ | コマンド |
|---|---|
| 保存して作業を続けたい | （A） |
| 保存して終了したい | （B） |
| 変更を破棄して終了したい | （C） |
| 別名（`backup.txt`）で保存したい | （D） |

**期待される解答例：**

```
(A) :w    (B) :wq    (C) :q!    (D) :w backup.txt
```

<details>
<summary>ヒント</summary>

`:w` の後にファイル名を指定すると「名前を付けて保存」になります。

</details>

<details>
<summary>解答と解説</summary>

```vim
:w              " 上書き保存して編集を続行。こまめに使いましょう
:wq             " 保存と終了を一度に実行。最頻出コマンドの一つです
:q!             " 未保存の変更を破棄して強制終了。:q だけではエラーになります
:w backup.txt   " 現在の内容を別ファイルに保存。元ファイルには影響しません
```

</details>

## 応用問題

### 問題4：モード遷移の実践

各ステップでVimがどのモードにあるか答えてください。

```
1. vim test.txt を実行  → モード：（  ）
2. i を押す             → モード：（  ）
3. "Hello Vim" と入力   → モード：（  ）
4. ESC を押す           → モード：（  ）
5. : を押す             → モード：（  ）
6. wq Enter             → 終了
```

**期待される解答例：**

```
1. ノーマルモード  2. 挿入モード  3. 挿入モード
4. ノーマルモード  5. コマンドラインモード  6. 終了
```

<details>
<summary>ヒント</summary>

Vim起動直後は必ずノーマルモード。テキスト入力中はモードが変わりません。

</details>

<details>
<summary>解答と解説</summary>

```
1. ノーマルモード     # Vimは常にノーマルモードで起動します
2. 挿入モード         # 画面下部に「-- INSERT --」と表示されます
3. 挿入モード         # テキスト入力中はモードが変わりません
4. ノーマルモード     # ESC は「ノーマルモードに戻る」万能キーです
5. コマンドラインモード # 画面最下部に : が表示されます
6. 終了               # :wq で保存して終了します
```

**重要：** 迷ったら `ESC` を押してノーマルモードに戻る。これがVim操作の基本原則です。

</details>

### 問題5：ファイルを作成して保存

以下の内容のファイルをVimで新規作成してください。操作手順を書き出し、`cat` で結果を確認してください。

**作成するファイル：** `profile.txt`

```
名前: Taro
年齢: 25
趣味: プログラミング
```

**期待される操作手順と出力：**

```bash
$ vim profile.txt
# i → テキスト入力 → ESC → :wq
$ cat profile.txt
名前: Taro
年齢: 25
趣味: プログラミング
```

<details>
<summary>ヒント</summary>

改行は挿入モード中に `Enter` キーで入力できます。間違えた場合は `Backspace` で修正できます。

</details>

<details>
<summary>解答と解説</summary>

```bash
vim profile.txt    # 新規ファイルとして開く
```

```
i                  # 挿入モードに入る（画面下部に -- INSERT -- と表示）
名前: Taro[Enter]  # テキスト入力。Enter で改行
年齢: 25[Enter]
趣味: プログラミング
ESC                # ノーマルモードに戻る
:wq                # 保存して終了
```

```bash
cat profile.txt    # 保存内容を確認
# 名前: Taro
# 年齢: 25
# 趣味: プログラミング
```

`o` コマンドを使う方法もあります：`i` → 1行入力 → `ESC` → `o` → 次行入力 の繰り返しです。

</details>

## チャレンジ問題

### 問題6：vimtutorに挑戦

Vimの公式チュートリアル `vimtutor` のレッスン1を完了し、学んだコマンドをすべて書き出してください。

```bash
vimtutor ja    # 日本語版
vimtutor       # 英語版
```

**期待される解答例：**

```
- h / j / k / l  : カーソル移動（左 / 下 / 上 / 右）
- x              : カーソル位置の文字を削除
- i              : カーソル位置の前に挿入
- A              : 行末に追加
- ESC            : ノーマルモードに戻る
- :q!            : 保存せずに終了
- u              : Undo（元に戻す）
```

<details>
<summary>ヒント</summary>

`vimtutor` は実際のVimで操作しながら学べるチュートリアルです。途中で終了する場合は `:q!` で抜けられます。

</details>

<details>
<summary>解答と解説</summary>

```
# カーソル移動
h  → 左（キーボード上で一番左だから「左」）
j  → 下（j のフックが下向きだから「下」）
k  → 上（k が上に突き出ているから「上」）
l  → 右（キーボード上で一番右だから「右」）

# テキスト編集
x  → カーソル位置の1文字を削除
i  → カーソル位置の前に挿入モードで入る
A  → 行末に追加する形で挿入モードに入る（Append）

# モード切替・終了
ESC  → ノーマルモードに戻る
:q!  → 変更を破棄して強制終了

# 元に戻す
u      → 直前の操作を元に戻す（Undo）
Ctrl+r → Undoをやり直す（Redo）
```

`vimtutor` は繰り返し実行して体に覚えさせることが上達のコツです。

</details>
