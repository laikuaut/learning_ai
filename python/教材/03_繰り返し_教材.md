## 3.1 for ループ

リストや文字列などの要素を1つずつ取り出して処理します。

```python
fruits = ["りんご", "みかん", "バナナ"]

for fruit in fruits:
    print(fruit)
```

**出力：**
```
りんご
みかん
バナナ
```

### 文字列の1文字ずつ

```python
for char in "Python":
    print(char)
```

**出力：**
```
P
y
t
h
o
n
```

---

## 3.2 range() 関数

指定した回数だけ繰り返したいときに使います。

### 基本の使い方

```python
# 0から4まで（5回）
for i in range(5):
    print(i)
```

**出力：** `0 1 2 3 4`（それぞれ改行）

### 開始と終了を指定

```python
# 1から5まで
for i in range(1, 6):
    print(i)
```

**出力：** `1 2 3 4 5`

### ステップ（増分）を指定

```python
# 0から10まで2ずつ
for i in range(0, 11, 2):
    print(i)
```

**出力：** `0 2 4 6 8 10`

### カウントダウン

```python
for i in range(5, 0, -1):
    print(i)
```

**出力：** `5 4 3 2 1`

### range() のまとめ

| 書き方 | 意味 | 生成される数列 |
|--------|------|----------------|
| `range(5)` | 0から4まで | 0, 1, 2, 3, 4 |
| `range(2, 7)` | 2から6まで | 2, 3, 4, 5, 6 |
| `range(0, 10, 3)` | 0から9まで3ずつ | 0, 3, 6, 9 |
| `range(5, 0, -1)` | 5から1まで逆順 | 5, 4, 3, 2, 1 |

## 3.3 enumerate() でインデックス付きループ

要素と一緒にインデックス（番号）も取得できます。

```python
fruits = ["りんご", "みかん", "バナナ"]

for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")
```

**出力：**
```
0: りんご
1: みかん
2: バナナ
```

### 開始番号を指定

```python
for i, fruit in enumerate(fruits, start=1):
    print(f"{i}: {fruit}")
```

**出力：**
```
1: りんご
2: みかん
3: バナナ
```

## 3.4 while ループ

条件が `True` の間、繰り返し続けます。

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

**出力：** `0 1 2 3 4`（それぞれ改行）

> **注意**: `count += 1` を忘れると**無限ループ**になります！

### while の使いどころ

- 繰り返し回数が事前にわからないとき
- 特定の条件を満たすまで繰り返したいとき

```python
# ユーザーが"quit"と入力するまで繰り返す
while True:
    text = input("入力してください（quitで終了）: ")
    if text == "quit":
        break
    print(f"入力: {text}")
```

## 3.5 break と continue

### break：ループを途中で抜ける

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

**出力：** `0 1 2 3 4`

### continue：今の回をスキップして次へ

```python
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)
```

**出力：** `1 3 5 7 9`（偶数をスキップ）

---

## 3.6 ネストしたループ（二重ループ）

ループの中にループを書くことができます。

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} × {j} = {i * j}")
    print("---")
```

**出力：**
```
1 × 1 = 1
1 × 2 = 2
1 × 3 = 3
---
2 × 1 = 2
2 × 2 = 4
2 × 3 = 6
---
3 × 1 = 3
3 × 2 = 6
3 × 3 = 9
---
```

---

## 3.7 ループとelse

Pythonでは `for` や `while` に `else` をつけることができます。
ループが `break` されずに最後まで実行されたときに `else` ブロックが実行されます。

```python
for i in range(5):
    if i == 10:  # この条件はTrueにならない
        break
else:
    print("ループは最後まで実行されました")
```

**出力：** `ループは最後まで実行されました`

---

## まとめ

| 概念 | ポイント |
|------|----------|
| `for` | リストや範囲の要素を1つずつ処理 |
| `range()` | 数値の範囲を生成 |
| `enumerate()` | インデックスと要素を同時に取得 |
| `while` | 条件がTrueの間繰り返す |
| `break` | ループを途中で終了 |
| `continue` | 今の回をスキップして次へ |
| ネスト | ループの中にループを書ける |
