## 基本問題

### 問題1：テキストファイルに文字列を書き込む

ユーザーが入力した3行分のテキストを `output.txt` というファイルに書き込むプログラムを作成してください。

- `fopen` で書き込みモード（`"w"`）で開く
- `fgets` で標準入力から読み取り、`fprintf` でファイルに書き込む
- 書き込み完了後にメッセージを表示する
- ファイルオープンの失敗チェックを行う

**期待される出力例：**
```
3行のテキストを入力してください:
> こんにちは
> C言語の勉強中です
> ファイル操作は便利ですね
output.txt に書き込みました。
```

**output.txt の中身：**
```
こんにちは
C言語の勉強中です
ファイル操作は便利ですね
```

<details>
<summary>ヒント</summary>

- `FILE *fp = fopen("output.txt", "w");` で書き込み用にファイルを開きます
- `fgets(buf, sizeof(buf), stdin)` で1行読み取れます（末尾に改行が含まれます）
- `fprintf(fp, "%s", buf)` でファイルに書き込みます
- 忘れずに `fclose(fp)` で閉じましょう

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    FILE *fp;
    char buf[256];
    int i;

    /* ファイルを書き込みモードで開く */
    fp = fopen("output.txt", "w");
    if (fp == NULL) {
        printf("ファイルを開けませんでした。\n");
        return 1;
    }

    printf("3行のテキストを入力してください:\n");

    for (i = 0; i < 3; i++) {
        printf("> ");
        /* 標準入力から1行読み取る */
        fgets(buf, sizeof(buf), stdin);
        /* ファイルに書き込む */
        fprintf(fp, "%s", buf);
    }

    /* ファイルを閉じる */
    fclose(fp);

    printf("output.txt に書き込みました。\n");

    return 0;
}
```

</details>

### 問題2：テキストファイルを1行ずつ読み込んで表示

テキストファイル `output.txt`（問題1で作成したもの）を1行ずつ読み込み、行番号をつけて表示するプログラムを作成してください。

- `fopen` で読み込みモード（`"r"`）で開く
- `fgets` でEOFまで繰り返し読み込む
- 各行に行番号を付けて表示する

**期待される出力例：**
```
===== output.txt の内容 =====
1: こんにちは
2: C言語の勉強中です
3: ファイル操作は便利ですね
===== 読み込み完了（3行）=====
```

<details>
<summary>ヒント</summary>

- `fgets` は読み込みに成功すると文字列のポインタを、EOFに達すると `NULL` を返します
- `while (fgets(buf, sizeof(buf), fp) != NULL)` のようにループ条件にできます
- `fgets` で読み込んだ文字列には末尾に改行（`\n`）が含まれることがあります

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    FILE *fp;
    char buf[256];
    int line_num = 0;

    /* ファイルを読み込みモードで開く */
    fp = fopen("output.txt", "r");
    if (fp == NULL) {
        printf("ファイルを開けませんでした。\n");
        return 1;
    }

    printf("===== output.txt の内容 =====\n");

    /* EOFまで1行ずつ読み込む */
    while (fgets(buf, sizeof(buf), fp) != NULL) {
        line_num++;

        /* 末尾の改行を除去（表示フォーマットを整えるため） */
        buf[strcspn(buf, "\n")] = '\0';

        printf("%d: %s\n", line_num, buf);
    }

    printf("===== 読み込み完了（%d行）=====\n", line_num);

    /* ファイルを閉じる */
    fclose(fp);

    return 0;
}
```

</details>

### 問題3：コマンドライン引数を表示するプログラム

コマンドライン引数をすべて表示するプログラムを作成してください。

- `main` 関数を `int main(int argc, char *argv[])` の形式にする
- `argc`（引数の数）と各 `argv` の値を表示する
- 引数が1つも指定されなかった場合は使い方メッセージを表示する

**実行例：**
```
$ ./args Hello World 123
引数の数: 4
argv[0] = ./args
argv[1] = Hello
argv[2] = World
argv[3] = 123
```

**引数なしの場合：**
```
$ ./args
使い方: ./args <引数1> <引数2> ...
引数を1つ以上指定してください。
```

<details>
<summary>ヒント</summary>

- `argc` はプログラム名自身を含んだ引数の数です（引数なしでも `1`）
- `argv[0]` はプログラム自身のパスです
- 引数がないとき `argc == 1` となります

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(int argc, char *argv[]) {
    int i;

    /* 引数が指定されなかった場合 */
    if (argc == 1) {
        printf("使い方: %s <引数1> <引数2> ...\n", argv[0]);
        printf("引数を1つ以上指定してください。\n");
        return 1;
    }

    /* 引数の数を表示 */
    printf("引数の数: %d\n", argc);

    /* 各引数を表示 */
    for (i = 0; i < argc; i++) {
        printf("argv[%d] = %s\n", i, argv[i]);
    }

    return 0;
}
```

</details>

### 問題4：単語の出現回数をカウントしてファイルに保存

テキストファイルから指定された単語が何回出現するかをカウントし、結果を別のファイルに保存するプログラムを作成してください。

- 検索対象のファイル名と検索する単語をプログラム内で指定する（定数として）
- `fgets` で1行ずつ読み込み、`strstr` で単語を検索する
- 結果を `result.txt` に書き込む

**入力ファイル `sample.txt` の中身（事前に用意）：**
```
C language is a powerful language.
Many programs are written in C.
Learning C is fun and rewarding.
C is used in systems programming.
```

**期待される出力例：**
```
ファイル「sample.txt」内の単語「C」の出現回数を調べます...
1行目: 1回 → C language is a powerful language.
2行目: 1回 → Many programs are written in C.
3行目: 1回 → Learning C is fun and rewarding.
4行目: 1回 → C is used in systems programming.

合計: 4回
結果を result.txt に保存しました。
```

<details>
<summary>ヒント</summary>

- `strstr(haystack, needle)` は文字列中に部分文字列が見つかるとその位置のポインタを返します
- 1行に同じ単語が複数回出現する場合は、見つかった位置の次から再度検索を繰り返します
- 結果ファイルにも `fprintf` で書き込みます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>

/* 1行中に指定単語が何回出現するか数える関数 */
int count_in_line(const char *line, const char *word) {
    int count = 0;
    int word_len = strlen(word);
    const char *p = line;

    /* 単語が見つかる限り繰り返す */
    while ((p = strstr(p, word)) != NULL) {
        count++;
        p += word_len;  /* 見つかった位置の次から検索を再開 */
    }

    return count;
}

int main(void) {
    const char *input_file = "sample.txt";
    const char *output_file = "result.txt";
    const char *word = "C";
    FILE *fp_in, *fp_out;
    char buf[256];
    int line_num = 0;
    int total = 0;
    int count;

    /* 入力ファイルを開く */
    fp_in = fopen(input_file, "r");
    if (fp_in == NULL) {
        printf("ファイル「%s」を開けませんでした。\n", input_file);
        return 1;
    }

    /* 出力ファイルを開く */
    fp_out = fopen(output_file, "w");
    if (fp_out == NULL) {
        printf("ファイル「%s」を作成できませんでした。\n", output_file);
        fclose(fp_in);
        return 1;
    }

    printf("ファイル「%s」内の単語「%s」の出現回数を調べます...\n", input_file, word);

    /* 1行ずつ読み込んで検索 */
    while (fgets(buf, sizeof(buf), fp_in) != NULL) {
        line_num++;

        /* 末尾の改行を除去 */
        buf[strcspn(buf, "\n")] = '\0';

        /* この行に含まれる単語の数をカウント */
        count = count_in_line(buf, word);

        if (count > 0) {
            printf("%d行目: %d回 → %s\n", line_num, count, buf);
        }

        total += count;
    }

    /* 結果を表示・保存 */
    printf("\n合計: %d回\n", total);

    fprintf(fp_out, "ファイル: %s\n", input_file);
    fprintf(fp_out, "検索語: %s\n", word);
    fprintf(fp_out, "出現回数: %d回\n", total);

    /* ファイルを閉じる */
    fclose(fp_in);
    fclose(fp_out);

    printf("結果を %s に保存しました。\n", output_file);

    return 0;
}
```

</details>

### 問題5：CSVファイルの読み込みと集計

以下の形式のCSVファイル `scores.csv` を読み込み、各生徒の合計点と平均点を計算して表示するプログラムを作成してください。

**scores.csv の中身（事前に用意）：**
```
名前,国語,数学,英語
田中太郎,80,90,75
鈴木花子,95,85,92
佐藤一郎,70,60,80
```

**期待される出力例：**
```
===== 成績集計 =====
名前         国語  数学  英語  合計  平均
田中太郎       80    90    75   245  81.67
鈴木花子       95    85    92   272  90.67
佐藤一郎       70    60    80   210  70.00
```

<details>
<summary>ヒント</summary>

- `fgets` で1行読み込み、`sscanf` でカンマ区切りのデータを解析します
- `sscanf(buf, "%[^,],%d,%d,%d", name, &kokugo, &sugaku, &eigo)` のように `%[^,]` でカンマまでの文字列を読み取れます
- 1行目はヘッダーなので読み飛ばします

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    FILE *fp;
    char buf[256];
    char name[50];
    int kokugo, sugaku, eigo;
    int total;
    double average;
    int is_header = 1;  /* ヘッダー行フラグ */

    /* CSVファイルを開く */
    fp = fopen("scores.csv", "r");
    if (fp == NULL) {
        printf("scores.csv を開けませんでした。\n");
        return 1;
    }

    printf("===== 成績集計 =====\n");
    printf("%-12s %4s %4s %4s %5s %6s\n",
           "名前", "国語", "数学", "英語", "合計", "平均");

    /* 1行ずつ読み込む */
    while (fgets(buf, sizeof(buf), fp) != NULL) {
        /* ヘッダー行はスキップ */
        if (is_header) {
            is_header = 0;
            continue;
        }

        /* カンマ区切りのデータを解析 */
        if (sscanf(buf, "%[^,],%d,%d,%d", name, &kokugo, &sugaku, &eigo) == 4) {
            total = kokugo + sugaku + eigo;
            average = total / 3.0;

            printf("%-12s %4d  %4d  %4d  %4d  %.2f\n",
                   name, kokugo, sugaku, eigo, total, average);
        }
    }

    fclose(fp);

    return 0;
}
```

</details>

## チャレンジ問題

### 問題6：簡易アドレス帳（構造体+ファイル保存）

構造体とファイル操作を組み合わせた簡易アドレス帳プログラムを作成してください。

- 連絡先の構造体：名前、電話番号、メールアドレス
- 以下の機能をメニューで選択できるようにする
  1. 連絡先の追加
  2. 一覧表示
  3. ファイルに保存
  4. ファイルから読み込み
  5. 終了
- データはCSV形式でファイルに保存・読み込みする
- 最大20件まで登録可能

**期待される出力例：**
```
===== 簡易アドレス帳 =====
1. 連絡先を追加
2. 一覧表示
3. ファイルに保存
4. ファイルから読み込み
5. 終了
選択: 1

--- 連絡先の追加 ---
名前: 田中太郎
電話番号: 090-1234-5678
メール: tanaka@example.com
登録しました。（現在1件）

===== 簡易アドレス帳 =====
1. 連絡先を追加
2. 一覧表示
3. ファイルに保存
4. ファイルから読み込み
5. 終了
選択: 2

--- 連絡先一覧（1件）---
[1] 田中太郎
    TEL: 090-1234-5678
    Mail: tanaka@example.com

===== 簡易アドレス帳 =====
1. 連絡先を追加
2. 一覧表示
3. ファイルに保存
4. ファイルから読み込み
5. 終了
選択: 3
contacts.csv に保存しました（1件）。

===== 簡易アドレス帳 =====
1. 連絡先を追加
2. 一覧表示
3. ファイルに保存
4. ファイルから読み込み
5. 終了
選択: 5
アドレス帳を終了します。
```

<details>
<summary>ヒント</summary>

- 構造体配列とカウンタ変数で連絡先を管理します
- メニューは `while` ループと `switch` 文で実装します
- ファイル保存は `fprintf(fp, "%s,%s,%s\n", ...)` でCSV形式にします
- ファイル読み込みは `fgets` + `sscanf` でCSVを解析します
- `scanf` で文字列を読む際は `fgets` を使う方が安全です（スペースを含む入力に対応）

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <string.h>

#define MAX_CONTACTS 20
#define FILENAME "contacts.csv"

/* 連絡先構造体 */
typedef struct {
    char name[50];
    char phone[20];
    char email[50];
} Contact;

/* プロトタイプ宣言 */
void add_contact(Contact contacts[], int *count);
void list_contacts(const Contact contacts[], int count);
void save_to_file(const Contact contacts[], int count);
int load_from_file(Contact contacts[]);
void trim_newline(char *str);

int main(void) {
    Contact contacts[MAX_CONTACTS];
    int count = 0;
    int choice;

    while (1) {
        /* メニュー表示 */
        printf("\n===== 簡易アドレス帳 =====\n");
        printf("1. 連絡先を追加\n");
        printf("2. 一覧表示\n");
        printf("3. ファイルに保存\n");
        printf("4. ファイルから読み込み\n");
        printf("5. 終了\n");
        printf("選択: ");
        scanf("%d", &choice);

        /* scanfの後に残る改行を読み飛ばす */
        while (getchar() != '\n');

        switch (choice) {
            case 1:
                add_contact(contacts, &count);
                break;
            case 2:
                list_contacts(contacts, count);
                break;
            case 3:
                save_to_file(contacts, count);
                break;
            case 4:
                count = load_from_file(contacts);
                break;
            case 5:
                printf("アドレス帳を終了します。\n");
                return 0;
            default:
                printf("無効な選択です。\n");
                break;
        }
    }

    return 0;
}

/* 末尾の改行を除去するユーティリティ関数 */
void trim_newline(char *str) {
    str[strcspn(str, "\n")] = '\0';
}

/* 連絡先を追加する関数 */
void add_contact(Contact contacts[], int *count) {
    if (*count >= MAX_CONTACTS) {
        printf("登録上限（%d件）に達しています。\n", MAX_CONTACTS);
        return;
    }

    printf("\n--- 連絡先の追加 ---\n");

    printf("名前: ");
    fgets(contacts[*count].name, sizeof(contacts[*count].name), stdin);
    trim_newline(contacts[*count].name);

    printf("電話番号: ");
    fgets(contacts[*count].phone, sizeof(contacts[*count].phone), stdin);
    trim_newline(contacts[*count].phone);

    printf("メール: ");
    fgets(contacts[*count].email, sizeof(contacts[*count].email), stdin);
    trim_newline(contacts[*count].email);

    (*count)++;
    printf("登録しました。（現在%d件）\n", *count);
}

/* 連絡先を一覧表示する関数 */
void list_contacts(const Contact contacts[], int count) {
    int i;

    if (count == 0) {
        printf("\n登録されている連絡先はありません。\n");
        return;
    }

    printf("\n--- 連絡先一覧（%d件）---\n", count);
    for (i = 0; i < count; i++) {
        printf("[%d] %s\n", i + 1, contacts[i].name);
        printf("    TEL: %s\n", contacts[i].phone);
        printf("    Mail: %s\n", contacts[i].email);
    }
}

/* ファイルに保存する関数 */
void save_to_file(const Contact contacts[], int count) {
    FILE *fp;
    int i;

    fp = fopen(FILENAME, "w");
    if (fp == NULL) {
        printf("ファイルを開けませんでした。\n");
        return;
    }

    /* CSV形式で書き込む */
    for (i = 0; i < count; i++) {
        fprintf(fp, "%s,%s,%s\n",
                contacts[i].name, contacts[i].phone, contacts[i].email);
    }

    fclose(fp);
    printf("%s に保存しました（%d件）。\n", FILENAME, count);
}

/* ファイルから読み込む関数 */
int load_from_file(Contact contacts[]) {
    FILE *fp;
    char buf[256];
    int count = 0;

    fp = fopen(FILENAME, "r");
    if (fp == NULL) {
        printf("ファイル「%s」が見つかりません。\n", FILENAME);
        return 0;
    }

    /* 1行ずつCSVを解析 */
    while (fgets(buf, sizeof(buf), fp) != NULL && count < MAX_CONTACTS) {
        trim_newline(buf);

        /* カンマ区切りで3つのフィールドを読み取る */
        if (sscanf(buf, "%[^,],%[^,],%[^\n]",
                   contacts[count].name,
                   contacts[count].phone,
                   contacts[count].email) == 3) {
            count++;
        }
    }

    fclose(fp);
    printf("%s から読み込みました（%d件）。\n", FILENAME, count);

    return count;
}
```

</details>
