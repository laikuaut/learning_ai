1. テンプレート構文 {{ '{{ }}' }}

名前を入力すると挨拶が表示されます

2. カウンター（リアクティビティ）

3. リスト表示（v-for）

4. 表示切替（v-if と v-show）

v-if: この要素はDOMから削除/追加されます

v-show: display: none で非表示（DOMには存在し続ける）
