## 4.1 配列の基本

**配列（array）** は、同じ型のデータを連続して格納するデータ構造です。

### 配列のイメージ

```
インデックス:  [0]    [1]    [2]    [3]    [4]
              +------+------+------+------+------+
scores:       |  85  |  92  |  78  |  95  |  88  |
              +------+------+------+------+------+
              ← 長さ（length）= 5 →
```

### 配列の宣言と初期化

```java
public class ArrayBasic {
    public static void main(String[] args) {
        // 方法1：サイズを指定して宣言（デフォルト値で初期化される）
        int[] scores = new int[5];  // [0, 0, 0, 0, 0]
        scores[0] = 85;
        scores[1] = 92;
        scores[2] = 78;
        scores[3] = 95;
        scores[4] = 88;

        // 方法2：初期値を指定して宣言（推奨）
        int[] grades = {85, 92, 78, 95, 88};

        // 方法3：new と初期値の組み合わせ
        String[] names = new String[]{"太郎", "花子", "次郎"};

        // 配列の長さ
        System.out.println("要素数: " + grades.length);

        // 要素へのアクセス
        System.out.println("1番目の要素: " + grades[0]);
        System.out.println("最後の要素: " + grades[grades.length - 1]);

        // 要素の変更
        grades[2] = 80;
        System.out.println("変更後の3番目: " + grades[2]);

        // for文で全要素を表示
        System.out.println("\n--- 全要素 ---");
        for (int i = 0; i < grades.length; i++) {
            System.out.println("grades[" + i + "] = " + grades[i]);
        }

        // 拡張for文で全要素を表示
        System.out.println("\n--- 名前一覧 ---");
        for (String name : names) {
            System.out.println(name);
        }
    }
}
```

実行方法：
```bash
javac ArrayBasic.java && java ArrayBasic
```

実行結果：
```
要素数: 5
1番目の要素: 85
最後の要素: 88
変更後の3番目: 80

--- 全要素 ---
grades[0] = 85
grades[1] = 92
grades[2] = 80
grades[3] = 95
grades[4] = 88

--- 名前一覧 ---
太郎
花子
次郎
```

> **よくある間違い：** 配列のインデックスは **0から始まります**。長さ5の配列の最後の要素は `array[4]` です。`array[5]` にアクセスすると `ArrayIndexOutOfBoundsException` が発生します。

### 各型のデフォルト値

| 型 | デフォルト値 |
|---|---|
| `int`, `long` 等の整数型 | `0` |
| `float`, `double` | `0.0` |
| `boolean` | `false` |
| `char` | `'\u0000'`（空文字） |
| 参照型（`String` 等） | `null` |

### ポイントまとめ

- 配列は `型[] 変数名 = new 型[サイズ]` または `型[] 変数名 = {値, ...}` で作成
- インデックスは0から始まる
- `length` プロパティで配列の長さを取得（メソッドではないので `()` は付けない）
- 一度作成した配列のサイズは変更できない

## 4.2 配列の操作パターン

```java
public class ArrayPatterns {
    public static void main(String[] args) {
        int[] numbers = {34, 12, 56, 78, 23, 45, 89, 67};

        // パターン1：合計と平均
        int sum = 0;
        for (int num : numbers) {
            sum += num;
        }
        double average = (double) sum / numbers.length;
        System.out.printf("合計: %d, 平均: %.1f%n", sum, average);

        // パターン2：最大値・最小値
        int max = numbers[0];
        int min = numbers[0];
        for (int num : numbers) {
            if (num > max) max = num;
            if (num < min) min = num;
        }
        System.out.println("最大値: " + max);
        System.out.println("最小値: " + min);

        // パターン3：線形探索
        int target = 56;
        int index = -1;
        for (int i = 0; i < numbers.length; i++) {
            if (numbers[i] == target) {
                index = i;
                break;
            }
        }
        if (index >= 0) {
            System.out.println(target + "はインデックス" + index + "にあります。");
        } else {
            System.out.println(target + "は見つかりませんでした。");
        }

        // パターン4：要素のカウント
        int count = 0;
        for (int num : numbers) {
            if (num >= 50) {
                count++;
            }
        }
        System.out.println("50以上の要素: " + count + "個");

        // パターン5：配列の反転
        int[] reversed = new int[numbers.length];
        for (int i = 0; i < numbers.length; i++) {
            reversed[i] = numbers[numbers.length - 1 - i];
        }
        System.out.print("反転: ");
        for (int num : reversed) {
            System.out.print(num + " ");
        }
        System.out.println();
    }
}
```

実行方法：
```bash
javac ArrayPatterns.java && java ArrayPatterns
```

実行結果：
```
合計: 404, 平均: 50.5
最大値: 89
最小値: 12
56はインデックス2にあります。
50以上の要素: 4個
反転: 67 89 45 23 78 56 12 34
```

### ポイントまとめ

- 配列の操作は「合計」「最大最小」「探索」「カウント」「反転」が定番パターン
- 探索で見つからない場合のデフォルト値（-1 など）を決めておく
- 拡張for文はインデックス不要な場合に便利

## 4.3 多次元配列

**多次元配列（multi-dimensional array）** は、配列の中に配列を入れたものです。最もよく使うのは **2次元配列** です。

```
2次元配列のイメージ:

          列0    列1    列2
        +------+------+------+
行0     |  1   |  2   |  3   |   matrix[0][0]=1, matrix[0][1]=2, matrix[0][2]=3
        +------+------+------+
行1     |  4   |  5   |  6   |   matrix[1][0]=4, matrix[1][1]=5, matrix[1][2]=6
        +------+------+------+
行2     |  7   |  8   |  9   |   matrix[2][0]=7, matrix[2][1]=8, matrix[2][2]=9
        +------+------+------+
```

```java
public class MultiArray {
    public static void main(String[] args) {
        // 2次元配列の宣言と初期化
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // 全要素の表示
        System.out.println("--- 行列の表示 ---");
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.printf("%3d", matrix[i][j]);
            }
            System.out.println();
        }

        // 実用例：生徒ごとの成績表
        System.out.println("\n--- 成績表 ---");
        String[] subjects = {"国語", "数学", "英語"};
        String[] students = {"太郎", "花子", "次郎"};
        int[][] scores = {
            {80, 90, 85},  // 太郎の成績
            {75, 95, 80},  // 花子の成績
            {90, 70, 88}   // 次郎の成績
        };

        // ヘッダー
        System.out.printf("%-6s", "名前");
        for (String subject : subjects) {
            System.out.printf("%6s", subject);
        }
        System.out.printf("%6s%n", "平均");
        System.out.println("------------------------------");

        // データ
        for (int i = 0; i < students.length; i++) {
            System.out.printf("%-6s", students[i]);
            int total = 0;
            for (int j = 0; j < scores[i].length; j++) {
                System.out.printf("%6d", scores[i][j]);
                total += scores[i][j];
            }
            double avg = (double) total / scores[i].length;
            System.out.printf("%6.1f%n", avg);
        }
    }
}
```

実行方法：
```bash
javac MultiArray.java && java MultiArray
```

実行結果：
```
--- 行列の表示 ---
  1  2  3
  4  5  6
  7  8  9

--- 成績表 ---
名前      国語    数学    英語    平均
------------------------------
太郎        80    90    85  85.0
花子        75    95    80  83.3
次郎        90    70    88  82.7
```

### ポイントまとめ

- 2次元配列は `型[][] 変数名` で宣言
- `matrix.length` で行数、`matrix[i].length` で列数を取得
- 成績表や座標データなどの表形式データに適している

## 4.4 Arraysユーティリティクラス

`java.util.Arrays` クラスには、配列を操作する便利なメソッドが多数用意されています。

```java
import java.util.Arrays;

public class ArraysDemo {
    public static void main(String[] args) {
        int[] numbers = {34, 12, 56, 78, 23, 45, 89, 67};

        // toString：配列を文字列として表示
        System.out.println("元の配列: " + Arrays.toString(numbers));

        // sort：ソート（昇順）
        int[] sorted = Arrays.copyOf(numbers, numbers.length);
        Arrays.sort(sorted);
        System.out.println("ソート後: " + Arrays.toString(sorted));

        // binarySearch：二分探索（ソート済み配列で使用）
        int index = Arrays.binarySearch(sorted, 56);
        System.out.println("56のインデックス: " + index);

        // fill：全要素を同じ値で埋める
        int[] filled = new int[5];
        Arrays.fill(filled, 99);
        System.out.println("fill後: " + Arrays.toString(filled));

        // copyOf：配列のコピー
        int[] copy = Arrays.copyOf(numbers, numbers.length);
        System.out.println("コピー: " + Arrays.toString(copy));

        // copyOfRange：部分コピー
        int[] partial = Arrays.copyOfRange(numbers, 2, 5);
        System.out.println("部分コピー[2,5): " + Arrays.toString(partial));

        // equals：配列の比較
        int[] a = {1, 2, 3};
        int[] b = {1, 2, 3};
        int[] c = {1, 2, 4};
        System.out.println("a equals b: " + Arrays.equals(a, b));  // true
        System.out.println("a equals c: " + Arrays.equals(a, c));  // false

        // String配列のソート
        String[] fruits = {"バナナ", "りんご", "みかん", "いちご"};
        Arrays.sort(fruits);
        System.out.println("果物ソート: " + Arrays.toString(fruits));
    }
}
```

実行方法：
```bash
javac ArraysDemo.java && java ArraysDemo
```

実行結果：
```
元の配列: [34, 12, 56, 78, 23, 45, 89, 67]
ソート後: [12, 23, 34, 45, 56, 67, 78, 89]
56のインデックス: 4
fill後: [99, 99, 99, 99, 99]
コピー: [34, 12, 56, 78, 23, 45, 89, 67]
部分コピー[2,5): [56, 78, 23]
a equals b: true
a equals c: false
果物ソート: [いちご, バナナ, みかん, りんご]
```

### 主なメソッド一覧

| メソッド | 説明 |
|---|---|
| `Arrays.toString(配列)` | 配列を文字列に変換して表示 |
| `Arrays.sort(配列)` | 昇順にソート |
| `Arrays.binarySearch(配列, 値)` | ソート済み配列から値を検索 |
| `Arrays.fill(配列, 値)` | 全要素を指定値で埋める |
| `Arrays.copyOf(配列, 長さ)` | 配列をコピー |
| `Arrays.copyOfRange(配列, from, to)` | 部分コピー |
| `Arrays.equals(配列1, 配列2)` | 内容が等しいか比較 |

> **よくある間違い：** `System.out.println(配列)` とすると、配列の中身ではなくメモリアドレスのような文字列が表示されます。中身を見たい場合は `Arrays.toString(配列)` を使いましょう。

### ポイントまとめ

- `Arrays` クラスは `import java.util.Arrays` で使える
- `Arrays.sort()` はソート、`Arrays.toString()` は表示に必須
- 配列を直接printlnすると中身が見えないので注意

## 4.5 ArrayList

**ArrayList（アレイリスト）** は、サイズが動的に変わるリストです。配列と違い、要素の追加・削除が自由にできます。

```java
import java.util.ArrayList;

public class ArrayListDemo {
    public static void main(String[] args) {
        // ArrayListの作成
        ArrayList<String> fruits = new ArrayList<>();

        // 要素の追加
        fruits.add("りんご");
        fruits.add("バナナ");
        fruits.add("みかん");
        fruits.add("ぶどう");
        System.out.println("果物リスト: " + fruits);
        System.out.println("要素数: " + fruits.size());

        // 要素の取得
        System.out.println("1番目: " + fruits.get(0));
        System.out.println("3番目: " + fruits.get(2));

        // 要素の変更
        fruits.set(1, "メロン");
        System.out.println("変更後: " + fruits);

        // 要素の挿入
        fruits.add(1, "いちご");
        System.out.println("挿入後: " + fruits);

        // 要素の削除
        fruits.remove("ぶどう");
        System.out.println("削除後: " + fruits);

        // インデックスで削除
        fruits.remove(0);
        System.out.println("先頭削除: " + fruits);

        // 存在確認
        System.out.println("みかんを含む: " + fruits.contains("みかん"));

        // 拡張for文で反復
        System.out.println("\n--- 全要素 ---");
        for (String fruit : fruits) {
            System.out.println("- " + fruit);
        }

        // 空かどうかの確認
        System.out.println("空リスト: " + fruits.isEmpty());

        // 全要素の削除
        fruits.clear();
        System.out.println("clear後: " + fruits);
        System.out.println("空リスト: " + fruits.isEmpty());
    }
}
```

実行方法：
```bash
javac ArrayListDemo.java && java ArrayListDemo
```

実行結果：
```
果物リスト: [りんご, バナナ, みかん, ぶどう]
要素数: 4
1番目: りんご
3番目: みかん
変更後: [りんご, メロン, みかん, ぶどう]
挿入後: [りんご, いちご, メロン, みかん, ぶどう]
削除後: [りんご, いちご, メロン, みかん]
先頭削除: [いちご, メロン, みかん]
みかんを含む: true

--- 全要素 ---
- いちご
- メロン
- みかん
空リスト: false
clear後: []
空リスト: true
```

### ArrayList の主なメソッド

| メソッド | 説明 |
|---|---|
| `add(要素)` | 末尾に要素を追加 |
| `add(位置, 要素)` | 指定位置に挿入 |
| `get(位置)` | 指定位置の要素を取得 |
| `set(位置, 要素)` | 指定位置の要素を変更 |
| `remove(位置)` | 指定位置の要素を削除 |
| `remove(要素)` | 指定要素を削除（最初の1つ） |
| `size()` | 要素数を返す |
| `contains(要素)` | 要素が含まれるか |
| `isEmpty()` | 空かどうか |
| `clear()` | 全要素を削除 |

> **よくある間違い：** `ArrayList<int>` とは書けません。プリミティブ型は使えないため、**ラッパークラス**（`Integer`, `Double`, `Boolean` など）を使います。`ArrayList<Integer>` が正しい書き方です。

### ポイントまとめ

- `ArrayList<型>` でジェネリクスを使って型を指定する
- プリミティブ型は使えない（Integer, Double 等のラッパークラスを使う）
- `size()` で要素数を取得（配列の `length` と異なりメソッド）
- 要素の追加・削除が自由にできるのが配列との最大の違い

## 4.6 HashMap

**HashMap（ハッシュマップ）** は、**キー（key）** と **値（value）** のペアでデータを管理するデータ構造です。

```
HashMapのイメージ:
+----------+---------+
|   キー    |   値     |
+----------+---------+
| "太郎"   |   85    |
| "花子"   |   92    |
| "次郎"   |   78    |
+----------+---------+
```

```java
import java.util.HashMap;

public class HashMapDemo {
    public static void main(String[] args) {
        // HashMapの作成
        HashMap<String, Integer> scores = new HashMap<>();

        // 要素の追加
        scores.put("太郎", 85);
        scores.put("花子", 92);
        scores.put("次郎", 78);
        scores.put("美咲", 95);
        System.out.println("成績表: " + scores);

        // 値の取得
        System.out.println("太郎の点数: " + scores.get("太郎"));

        // 値の更新
        scores.put("次郎", 82);  // 同じキーでputすると上書き
        System.out.println("次郎の更新後: " + scores.get("次郎"));

        // キーの存在確認
        System.out.println("花子がいる: " + scores.containsKey("花子"));
        System.out.println("五郎がいる: " + scores.containsKey("五郎"));

        // 値の存在確認
        System.out.println("95点がある: " + scores.containsValue(95));

        // 要素数
        System.out.println("人数: " + scores.size());

        // 要素の削除
        scores.remove("美咲");
        System.out.println("削除後: " + scores);

        // getOrDefault：キーが存在しない場合のデフォルト値
        int score = scores.getOrDefault("五郎", 0);
        System.out.println("五郎の点数: " + score);

        // 全要素の反復
        System.out.println("\n--- 全生徒の成績 ---");
        for (String name : scores.keySet()) {
            System.out.println(name + ": " + scores.get(name) + "点");
        }

        // entrySetで反復（キーと値を同時に取得）
        System.out.println("\n--- entrySet ---");
        for (var entry : scores.entrySet()) {
            System.out.println(entry.getKey() + " => " + entry.getValue());
        }
    }
}
```

実行方法：
```bash
javac HashMapDemo.java && java HashMapDemo
```

実行結果：
```
成績表: {花子=92, 太郎=85, 次郎=78, 美咲=95}
太郎の点数: 85
次郎の更新後: 82
花子がいる: true
五郎がいる: false
95点がある: true
人数: 4
削除後: {花子=92, 太郎=85, 次郎=82}
五郎の点数: 0

--- 全生徒の成績 ---
花子: 92点
太郎: 85点
次郎: 82点

--- entrySet ---
花子 => 92
太郎 => 85
次郎 => 82
```

> **よくある間違い：** 存在しないキーで `get()` を呼ぶと `null` が返ります。`null` に対して操作するとNullPointerExceptionが発生します。`getOrDefault()` や `containsKey()` で安全にアクセスしましょう。

### ポイントまとめ

- `HashMap<キーの型, 値の型>` で作成
- `put()` で追加・更新、`get()` で取得
- `keySet()` でキーの集合、`entrySet()` でキーと値のペアを取得
- 順序は保証されない（順序が必要なら `LinkedHashMap` を使う）

## 4.7 配列 vs ArrayList 比較

| 特徴 | 配列（array） | ArrayList |
|---|---|---|
| サイズ | **固定**（作成後は変更不可） | **可変**（自動で拡張） |
| 型 | プリミティブ型も使える | 参照型のみ（ラッパークラス） |
| 宣言 | `int[] arr = new int[5]` | `ArrayList<Integer> list = new ArrayList<>()` |
| 要素数 | `.length` | `.size()` |
| 要素追加 | 不可（新しい配列を作成） | `add()` で可能 |
| パフォーマンス | 高速（メモリ効率が良い） | やや遅い（オーバーヘッドあり） |
| 便利メソッド | `Arrays` クラスで提供 | 豊富なメソッドが組み込み |

### 使い分けの目安

- **配列を使う場合**：サイズが固定、プリミティブ型、高パフォーマンスが必要
- **ArrayListを使う場合**：サイズが変わる、要素の追加・削除が頻繁

### ポイントまとめ

- サイズが固定なら配列、可変ならArrayList
- 実務ではArrayListを使う場面が多い
- パフォーマンスが重要な場合は配列を検討

## 4.8 実践例：単語の出現回数カウンター

HashMapを使った実践的な例です。

```java
import java.util.HashMap;

public class WordCounter {
    public static void main(String[] args) {
        String text = "java is great java is fun java is powerful and java is everywhere";
        String[] words = text.split(" ");

        // 単語の出現回数をカウント
        HashMap<String, Integer> wordCount = new HashMap<>();

        for (String word : words) {
            wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
        }

        // 結果を表示
        System.out.println("=== 単語の出現回数 ===");
        System.out.printf("%-12s %s%n", "単語", "回数");
        System.out.println("-------------------");

        for (var entry : wordCount.entrySet()) {
            System.out.printf("%-12s %d回%n", entry.getKey(), entry.getValue());
        }

        // 最も多く出現した単語
        String mostFrequent = "";
        int maxCount = 0;
        for (var entry : wordCount.entrySet()) {
            if (entry.getValue() > maxCount) {
                maxCount = entry.getValue();
                mostFrequent = entry.getKey();
            }
        }
        System.out.println("\n最頻出単語: \"" + mostFrequent + "\" (" + maxCount + "回)");
    }
}
```

実行方法：
```bash
javac WordCounter.java && java WordCounter
```

実行結果：
```
=== 単語の出現回数 ===
単語          回数
```

## 4.9 実践例：学生成績管理

ArrayListとHashMapを組み合わせた実践例です。

```java
import java.util.ArrayList;
import java.util.HashMap;

public class StudentScores {
    public static void main(String[] args) {
        // 学生名リスト
        ArrayList<String> students = new ArrayList<>();
        students.add("太郎");
        students.add("花子");
        students.add("次郎");
        students.add("美咲");

        // 成績マップ（名前 → 点数の配列）
        HashMap<String, int[]> scoreMap = new HashMap<>();
        scoreMap.put("太郎", new int[]{80, 90, 85});
        scoreMap.put("花子", new int[]{75, 95, 80});
        scoreMap.put("次郎", new int[]{90, 70, 88});
        scoreMap.put("美咲", new int[]{95, 85, 92});

        String[] subjects = {"国語", "数学", "英語"};

        // 成績表の表示
        System.out.println("==============================");
        System.out.println("       成績一覧表");
        System.out.println("==============================");
        System.out.printf("%-6s", "名前");
        for (String subject : subjects) {
            System.out.printf("%6s", subject);
        }
        System.out.printf("%6s%n", "平均");
        System.out.println("------------------------------");

        double classTotal = 0;
        int studentCount = 0;

        for (String student : students) {
            int[] scores = scoreMap.get(student);
            System.out.printf("%-6s", student);

            int total = 0;
            for (int score : scores) {
                System.out.printf("%6d", score);
                total += score;
            }
            double avg = (double) total / scores.length;
            System.out.printf("%6.1f%n", avg);
            classTotal += avg;
            studentCount++;
        }

        System.out.println("------------------------------");
        System.out.printf("クラス平均: %.1f%n", classTotal / studentCount);
    }
}
```

実行方法：
```bash
javac StudentScores.java && java StudentScores
```

## まとめ

この章では、データを効率的に管理するための配列とコレクションを学びました。

1. **配列**：固定サイズ、同じ型のデータを連続格納
2. **多次元配列**：表形式のデータを管理
3. **Arraysクラス**：sort, toString, copyOf などの便利メソッド
4. **ArrayList**：可変長リスト、要素の追加・削除が自由
5. **HashMap**：キーと値のペアでデータを管理

| データ構造 | 特徴 | 用途 |
|---|---|---|
| 配列 | 固定サイズ、高速 | サイズ固定のデータ |
| ArrayList | 可変サイズ、便利 | 動的なリスト |
| HashMap | キーと値のペア | 名前で検索するデータ |
