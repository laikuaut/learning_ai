# 実践課題01：自己紹介カード ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（AngularJSの基本）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `ng-app`、モジュール、コントローラー、式展開 `{{ }}` と `ng-model` による双方向バインディング（Two-way Data Binding）の基本を実践する

---

## 完成イメージ

名前・役職・自己紹介を入力すると、リアルタイムにカード表示が更新されます。

```
┌─────────────────────────────────────┐
│  自己紹介カード作成                    │
│                                     │
│  名前:  [山田太郎          ]         │
│  役職:  [フロントエンドエンジニア]      │
│  紹介:  [AngularJSを勉強中です]       │
│  色:    [🔵 青 ] ← セレクトボックス    │
│                                     │
│  ── プレビュー ──                    │
│  ┌───────────────────┐              │
│  │      [T]           │              │
│  │    山田太郎         │              │
│  │ フロントエンドエンジニア│             │
│  │ AngularJSを勉強中です │             │
│  └───────────────────┘              │
└─────────────────────────────────────┘
```

---

## 課題の要件

1. AngularJS の CDN を読み込み、`ng-app` と `ng-controller` を設定する
2. `ng-model` で名前・役職・自己紹介文・テーマカラーを入力フォームにバインドする
3. 入力内容がリアルタイムにカードプレビューへ反映される
4. 名前の頭文字をアバターとして表示する
5. テーマカラーをセレクトボックスで選択でき、アバターの色が変わる
6. ブラウザで開くだけで動作する HTML ファイルとして完成させる

---

## ステップガイド

<details>
<summary>ステップ1：HTMLの雛形を作る</summary>

CDN版AngularJSを使って、ビルド不要で動くHTMLファイルを作ります。

```html
<!DOCTYPE html>
<html ng-app="profileApp">
<head>
  <meta charset="UTF-8">
  <title>自己紹介カード</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="ProfileCtrl">
  <!-- ここにフォームとカードを書く -->

  <script>
    angular.module('profileApp', [])
      .controller('ProfileCtrl', function($scope) {
        // 初期値を設定
      });
  </script>
</body>
</html>
```

</details>

<details>
<summary>ステップ2：入力フォームを作る</summary>

`ng-model` を使って各入力とスコープの変数をバインドします。

```html
<input type="text" ng-model="name" placeholder="名前を入力">
<input type="text" ng-model="role" placeholder="役職を入力">
<textarea ng-model="bio" placeholder="自己紹介を入力"></textarea>
<select ng-model="color">
  <option value="#2196f3">青</option>
  <option value="#4caf50">緑</option>
  <!-- 他の色も追加 -->
</select>
```

</details>

<details>
<summary>ステップ3：カードプレビューを作る</summary>

式展開 `{{ }}` で変数の値を表示します。

```html
<div class="card">
  <div class="avatar" ng-style="{'background-color': color}">
    {{ name.charAt(0) }}
  </div>
  <div class="name">{{ name }}</div>
  <div class="role">{{ role }}</div>
  <div class="bio">{{ bio }}</div>
</div>
```

</details>

<summary>解答例（改良版 ─ Controller As 構文 + 文字数カウント）</summary>

```html
<!DOCTYPE html>
<html ng-app="profileApp">
<head>
  <meta charset="UTF-8">
  <title>自己紹介カード（改良版）</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; background: #f0f2f5; max-width: 500px; margin: 0 auto; padding: 40px 20px; }
    h1 { text-align: center; color: #333; }
    .form-group { margin-bottom: 14px; }
    .form-group label { display: block; font-weight: bold; margin-bottom: 4px; color: #555; font-size: 13px; }
    .form-group input, .form-group textarea, .form-group select {
      width: 100%; padding: 10px 12px; border: 1px solid #ddd; border-radius: 8px;
      font-size: 14px; box-sizing: border-box; transition: border-color 0.2s;
    }
    .form-group input:focus, .form-group textarea:focus { border-color: #2196f3; outline: none; }
    .form-group textarea { height: 60px; resize: vertical; }
    .char-count { text-align: right; font-size: 11px; color: #999; margin-top: 2px; }
    .preview-title { text-align: center; color: #aaa; margin: 28px 0 12px; font-size: 13px; letter-spacing: 2px; }
    .card { background: white; border-radius: 16px; padding: 28px; text-align: center;
            box-shadow: 0 4px 16px rgba(0,0,0,0.08); transition: transform 0.2s; }
    .card:hover { transform: translateY(-2px); box-shadow: 0 6px 20px rgba(0,0,0,0.12); }
    .avatar { width: 72px; height: 72px; border-radius: 50%; display: flex; align-items: center;
              justify-content: center; margin: 0 auto 16px; color: white; font-size: 32px; font-weight: bold;
              transition: background-color 0.3s; }
    .card .name { font-size: 22px; font-weight: bold; margin-bottom: 4px; color: #333; }
    .card .role { color: #888; font-size: 14px; margin-bottom: 14px; }
    .card .bio { color: #555; font-size: 13px; line-height: 1.6; white-space: pre-wrap; }
  </style>
</head>
<body ng-controller="ProfileCtrl as vm">

  <h1>自己紹介カード作成</h1>

  <div class="form-group">
    <label>名前（20文字まで）</label>
    <input type="text" ng-model="vm.profile.name" placeholder="名前を入力" maxlength="20">
    <div class="char-count">{{ vm.profile.name.length }} / 20</div>
  </div>
  <div class="form-group">
    <label>役職</label>
    <input type="text" ng-model="vm.profile.role" placeholder="役職を入力" maxlength="30">
  </div>
  <div class="form-group">
    <label>自己紹介（100文字まで）</label>
    <textarea ng-model="vm.profile.bio" placeholder="自己紹介を入力" maxlength="100"></textarea>
    <div class="char-count">{{ vm.profile.bio.length }} / 100</div>
  </div>
  <div class="form-group">
    <label>テーマカラー</label>
    <select ng-model="vm.profile.color" ng-options="c.value as c.label for c in vm.colors"></select>
  </div>

  <div class="preview-title">── PREVIEW ──</div>
  <div class="card">
    <div class="avatar" ng-style="{'background-color': vm.profile.color}">
      {{ vm.profile.name.charAt(0) }}
    </div>
    <div class="name">{{ vm.profile.name || '名前未入力' }}</div>
    <div class="role">{{ vm.profile.role || '役職未入力' }}</div>
    <div class="bio">{{ vm.profile.bio || '自己紹介を入力してください' }}</div>
  </div>

  <script>
    angular.module('profileApp', [])
      .controller('ProfileCtrl', function() {
        // Controller As 構文：$scope を使わず this にバインド
        var vm = this;

        vm.colors = [
          { value: '#2196f3', label: '青' },
          { value: '#4caf50', label: '緑' },
          { value: '#e91e63', label: 'ピンク' },
          { value: '#ff9800', label: 'オレンジ' },
          { value: '#9c27b0', label: '紫' }
        ];

        vm.profile = {
          name: '山田太郎',
          role: 'フロントエンドエンジニア',
          bio: 'AngularJSを勉強中です。よろしくお願いします！',
          color: '#2196f3'
        };
      });
  </script>

</body>
</html>
```

**初心者向けとの違い:**
- `Controller As` 構文（`$scope` を使わない）でスコープの衝突を回避
- プロファイル情報をオブジェクト `vm.profile` にまとめて管理（ドット記法ルール）
- `ng-options` でセレクトボックスの選択肢を動的生成
- 文字数カウントを表示して入力制限を可視化
- CSS トランジションでカードにホバーエフェクトを追加
