## 基本問題

### 問題1：クリックイベント

ボタンをクリックするたびにカウントが増えるカウンターを作成してください。

```html
<p>カウント: <span id="count">0</span></p>
<button id="incrementBtn">+1</button>
<button id="resetBtn">リセット</button>
```

**動作：**
- 「+1」ボタンをクリックするとカウントが1増える
- 「リセット」ボタンをクリックするとカウントが0に戻る

<details>
<summary>解答例</summary>

```javascript
let count = 0;
const countDisplay = document.getElementById("count");

document.getElementById("incrementBtn").addEventListener("click", () => {
  count++;
  countDisplay.textContent = count;
});

document.getElementById("resetBtn").addEventListener("click", () => {
  count = 0;
  countDisplay.textContent = count;
});
```

</details>

### 問題2：キーボードイベント

テキスト入力欄に文字を入力するたびに、入力文字数をリアルタイムで表示してください。

```html
<textarea id="textArea" rows="4" cols="40" placeholder="テキストを入力してください"></textarea>
<p>文字数: <span id="charCount">0</span> / 100</p>
```

**動作：**
- 入力するたびに文字数を更新
- 100文字を超えたら文字数の色を赤にする

<details>
<summary>解答例</summary>

```javascript
const textArea = document.getElementById("textArea");
const charCount = document.getElementById("charCount");
const MAX_LENGTH = 100;

textArea.addEventListener("input", (e) => {
  const length = e.target.value.length;
  charCount.textContent = length;

  // 100文字を超えたら赤色に
  if (length > MAX_LENGTH) {
    charCount.style.color = "red";
  } else {
    charCount.style.color = "";
  }
});
```

</details>

### 問題3：フォーム送信の処理

フォーム送信時にページ遷移を防止し、入力内容をコンソールに表示してください。

```html
<form id="myForm">
  <input type="text" name="name" placeholder="名前">
  <input type="email" name="email" placeholder="メール">
  <button type="submit">送信</button>
</form>
<div id="result"></div>
```

**動作：**
- 送信ボタンをクリックすると、ページ遷移せずに入力内容を `#result` に表示
- 名前が空の場合はエラーメッセージを表示

<details>
<summary>解答例</summary>

```javascript
const form = document.getElementById("myForm");
const result = document.getElementById("result");

form.addEventListener("submit", (e) => {
  e.preventDefault(); // ページ遷移を防止

  const formData = new FormData(form);
  const name = formData.get("name").trim();
  const email = formData.get("email").trim();

  if (!name) {
    result.textContent = "名前を入力してください";
    result.style.color = "red";
    return;
  }

  result.textContent = `名前: ${name}, メール: ${email}`;
  result.style.color = "green";
});
```

</details>

### 問題4：マウスイベント

マウスの座標をリアルタイムで表示し、クリックした位置にドットを表示するプログラムを作成してください。

```html
<div id="area" style="width: 400px; height: 300px; border: 2px solid #333; position: relative;">
  <p id="coords">座標: (0, 0)</p>
</div>
```

<details>
<summary>解答例</summary>

```javascript
const area = document.getElementById("area");
const coords = document.getElementById("coords");

// マウス移動で座標を更新
area.addEventListener("mousemove", (e) => {
  const rect = area.getBoundingClientRect();
  const x = Math.round(e.clientX - rect.left);
  const y = Math.round(e.clientY - rect.top);
  coords.textContent = `座標: (${x}, ${y})`;
});

// クリックでドットを表示
area.addEventListener("click", (e) => {
  const rect = area.getBoundingClientRect();
  const x = e.clientX - rect.left;
  const y = e.clientY - rect.top;

  const dot = document.createElement("div");
  Object.assign(dot.style, {
    position: "absolute",
    left: `${x - 5}px`,
    top: `${y - 5}px`,
    width: "10px",
    height: "10px",
    backgroundColor: "red",
    borderRadius: "50%"
  });
  area.append(dot);
});
```

</details>

### 問題5：イベント委譲

イベント委譲を使って、リスト項目のクリックを処理してください。動的に追加された項目もクリックで反応すること。

```html
<ul id="todoList">
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ul>
<button id="addItem">項目を追加</button>
```

**動作：**
- リスト項目をクリックすると取り消し線をトグル
- 「項目を追加」ボタンで新しい項目を追加
- 追加された項目もクリックで反応する

<details>
<summary>ヒント</summary>

親要素（`<ul>`）に1つだけイベントリスナーを登録し、`e.target` でクリックされた要素を判定します。

</details>

<details>
<summary>解答例</summary>

```javascript
const list = document.getElementById("todoList");
let itemCount = 3;

// イベント委譲: 親要素にリスナーを1つ登録
list.addEventListener("click", (e) => {
  if (e.target.tagName === "LI") {
    e.target.style.textDecoration =
      e.target.style.textDecoration === "line-through" ? "" : "line-through";
  }
});

// 項目の動的追加
document.getElementById("addItem").addEventListener("click", () => {
  itemCount++;
  const li = document.createElement("li");
  li.textContent = `項目${itemCount}`;
  list.append(li);
  // イベント委譲のおかげで、追加した項目にも自動でイベントが効く
});
```

</details>

### 問題6：デバウンスの実装

検索入力欄にデバウンスを適用し、タイピングが止まってから300ms後に検索処理を実行するプログラムを作成してください。

```html
<input type="text" id="searchInput" placeholder="検索...">
<div id="searchResults"></div>
```

**動作：**
- 入力するたびにタイマーがリセットされる
- 最後の入力から300ms後に「検索中...」と表示される

<details>
<summary>解答例</summary>

```javascript
// デバウンス関数
function debounce(func, delay) {
  let timeoutId;
  return function(...args) {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => func.apply(this, args), delay);
  };
}

const searchInput = document.getElementById("searchInput");
const searchResults = document.getElementById("searchResults");

// デバウンスを適用した検索処理
const handleSearch = debounce((e) => {
  const query = e.target.value.trim();
  if (query) {
    searchResults.textContent = `「${query}」を検索中...`;
    console.log(`検索実行: ${query}`);
  } else {
    searchResults.textContent = "";
  }
}, 300);

searchInput.addEventListener("input", handleSearch);
```

</details>

## チャレンジ問題

### 問題7：ドラッグ&ドロップ風の要素移動

マウスのドラッグで要素を移動できるプログラムを作成してください。

```html
<div id="draggable" style="width: 100px; height: 100px; background: #3498db; position: absolute; cursor: grab; border-radius: 8px; display: flex; align-items: center; justify-content: center; color: white;">
  ドラッグ
</div>
```

**動作：**
- 要素をマウスで押すとドラッグ開始
- マウスを動かすと要素が追従
- マウスを離すとドラッグ終了

<details>
<summary>ヒント</summary>

`mousedown` でドラッグ開始、`mousemove` で要素の位置を更新、`mouseup` でドラッグ終了を処理します。

</details>

<details>
<summary>解答例</summary>

```javascript
const draggable = document.getElementById("draggable");
let isDragging = false;
let offsetX, offsetY;

draggable.addEventListener("mousedown", (e) => {
  isDragging = true;
  // クリック位置と要素左上の差分を記録
  offsetX = e.clientX - draggable.offsetLeft;
  offsetY = e.clientY - draggable.offsetTop;
  draggable.style.cursor = "grabbing";
});

document.addEventListener("mousemove", (e) => {
  if (!isDragging) return;
  // マウス位置 - 差分 = 要素の新しい位置
  draggable.style.left = `${e.clientX - offsetX}px`;
  draggable.style.top = `${e.clientY - offsetY}px`;
});

document.addEventListener("mouseup", () => {
  isDragging = false;
  draggable.style.cursor = "grab";
});
```

</details>
