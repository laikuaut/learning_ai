CSSボックスモデル サンプル

1. ボックスモデルの構造

すべてのHTML要素は4層のボックスで構成されています。

content
(コンテンツ)

外側から: margin → border → padding → content

2. box-sizing の違い

同じ width: 200px を指定しても、box-sizing によって実際の表示幅が変わります。

content-box
width: 200px
padding: 20px
border: 5px
→ 実際: 250px

content-box: 幅 = content のみ

border-box
width: 200px
padding: 20px
border: 5px
→ 実際: 200px

border-box: 幅 = 全体（推奨）

3. display プロパティ

block（幅いっぱい、縦に並ぶ）:

block 要素 1
block 要素 2
block 要素 3

inline（内容に合わせた幅、横に並ぶ、width/height 指定不可）:

inline 1 inline 2 inline 3

inline-block（横に並びつつ、width/height 指定可能）:

メニュー1 メニュー2 メニュー3

4. margin の相殺（Collapsing）

隣接するブロック要素の上下 margin は相殺され、大きい方の値だけが適用されます。
下の例: 上のボックスの margin-bottom: 30px、下のボックスの margin-top: 20px → 間隔は 30px（50px ではない）

margin-bottom: 30px
margin-top: 20px

※ 間隔は30px + 20px = 50px ではなく、大きい方の30pxになります

5. width / max-width

ブラウザの幅を変えて、各ボックスの挙動を確認してみましょう。

width: 300px（固定幅）
width: 50%（親要素の半分）
width: 100%, max-width: 500px
min-width: 200px, max-width: 80%

6. overflow（はみ出し制御）

コンテンツがボックスからはみ出した場合の挙動を制御します。

visible（デフォルト）

このテキストはボックスからはみ出して表示されます。overflow: visible はデフォルト値で、はみ出した部分もそのまま表示されます。

hidden

このテキストはボックスからはみ出した部分が隠されます。overflow: hidden ではみ出した部分が非表示になります。

scroll

このテキストにはスクロールバーが常に表示されます。overflow: scroll でスクロール可能になります。

auto

はみ出した場合のみスクロールバーが表示されます。overflow: auto は必要な時だけスクロールバーを出す実用的な設定です。
