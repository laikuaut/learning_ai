# 第1章：Vimの基本とセットアップ

## この章のゴール

- テキストエディタとしての Vim の特徴と利点を理解する
- Vim のインストールと起動・終了ができる
- Vim のモード（ノーマル、挿入、コマンドライン、ビジュアル）の概念を理解する
- 基本的なモード切り替えができる
- チュートリアル（vimtutor）の使い方を知る

---

## 1. Vimとは何か

### テキストエディタの歴史とVimの位置づけ

Vim（ヴィム）は、**Vi IMproved** の略で、Unix 系のテキストエディタ vi を大幅に改良したエディタです。プログラマーやシステム管理者に広く愛用されています。

```
テキストエディタの進化

  ed (1969)        ← 行指向エディタ（1行ずつ編集）
   │
   ▼
  ex (1976)        ← ed の拡張版
   │
   ▼
  vi (1976)        ← ex にビジュアルモードを追加（Bill Joy）
   │
   ▼
  Vim (1991)       ← vi の大幅改良版（Bram Moolenaar）
   │
   ▼
  Neovim (2014)    ← Vim のモダンなフォーク
```

### Vimの特徴

Vim が多くのプロフェッショナルに選ばれる理由は以下の通りです。

| 特徴 | 説明 |
|------|------|
| **モーダル編集** | モードを切り替えて操作するため、少ないキー入力で多くの操作が可能 |
| **キーボード中心** | マウスに手を伸ばす必要がなく、ホームポジションから手を離さずに操作できる |
| **軽量・高速** | 巨大なファイルでもサクサク動作する |
| **どこでも使える** | ほぼすべての Unix/Linux 環境にインストールされている |
| **高い拡張性** | プラグインや設定でいくらでもカスタマイズ可能 |
| **長い歴史** | 30年以上の歴史があり、膨大な知識が蓄積されている |

### ポイントまとめ

- Vim は vi を改良した高機能テキストエディタです
- モーダル編集とキーボード中心の操作が最大の特徴です
- ほぼすべての Unix/Linux 環境で使用できます

## 2. Vimのインストールと起動

### インストール方法

多くの Linux ディストリビューションには vi や vim がプリインストールされています。インストールされていない場合は以下のコマンドでインストールします。

```bash
# Ubuntu / Debian
$ sudo apt update
$ sudo apt install vim

# CentOS / RHEL / Fedora
$ sudo dnf install vim

# macOS（Homebrew）
$ brew install vim

# バージョン確認
$ vim --version | head -1
VIM - Vi IMproved 9.0
```

### Vimの起動と終了

```bash
# ファイルを指定して起動
$ vim filename.txt

# 新しいファイルを作成して起動
$ vim newfile.txt

# ファイル名を指定せずに起動
$ vim
```

Vim を起動すると、**ノーマルモード**（後述）の状態で開始されます。

### 終了方法

Vim の終了は、ノーマルモードで以下のコマンドを入力します。

```
:q          ← 終了（変更がない場合）
:q!         ← 変更を破棄して強制終了
:w          ← 保存（終了しない）
:wq         ← 保存して終了
:x          ← 保存して終了（:wq と同じ）
ZZ          ← 保存して終了（: 不要）
ZQ          ← 保存せず終了（: 不要）
```

> **よくある間違い**: Vim を起動したものの終了方法がわからずパニックになるのは、初心者あるあるです。まず `ESC` を何度か押してノーマルモードに戻り、`:q!` で強制終了できます。

### ポイントまとめ

- `vim ファイル名` で起動します
- `:wq` で保存して終了、`:q!` で保存せず終了します
- 困ったらまず `ESC` を押してノーマルモードに戻りましょう

## 3. Vimのモードを理解する

Vim 最大の特徴は **モーダル編集**（modal editing）です。他のエディタとは異なり、Vim では「モード」によってキーの意味が変わります。

### 4つの主要モード

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│            ┌──────────────────┐                      │
│     ESC    │                  │    i, a, o ...       │
│   ┌───────▶│  ノーマルモード   │◀──────────┐          │
│   │        │  (Normal Mode)   │           │          │
│   │        └───┬──────┬───┘           │          │
│   │            │      │               │          │
│   │       v,V  │      │ :             │ ESC      │
│   │     Ctrl+v │      │              │          │
│   │            ▼      ▼               │          │
│   │   ┌────────────┐ ┌──────────────┐ │          │
│   │   │ビジュアル   │ │コマンドライン │ │          │
│   │   │モード      │ │モード        │ │          │
│   │   │(Visual)    │ │(Command-line)│ │          │
│   │   └────────────┘ └──────────────┘ │          │
│   │        │               │          │          │
│   │        └───────┬───────┘          │          │
│   │                │ ESC              │          │
│   │                ▼                  │          │
│   │        ┌──────────────┐           │          │
│   └────────│  挿入モード   │───────────┘          │
│            │ (Insert Mode) │                      │
│            └──────────────┘                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 各モードの役割

| モード | 役割 | 入り方 | 抜け方 |
|--------|------|--------|--------|
| **ノーマルモード** | カーソル移動、テキスト操作、コマンド実行 | `ESC` | — |
| **挿入モード** | テキストの入力 | `i`, `a`, `o` 等 | `ESC` |
| **ビジュアルモード** | テキストの選択 | `v`, `V`, `Ctrl+v` | `ESC` |
| **コマンドラインモード** | Exコマンドの実行 | `:`, `/`, `?` | `Enter` or `ESC` |

### ノーマルモード（Normal Mode）

Vim を起動すると最初に入るモードです。**最も多くの時間を過ごすモード**で、カーソル移動やテキスト操作を行います。

```
ノーマルモードで使える主なキー:

  h j k l     ← カーソル移動（左 下 上 右）
  w b         ← 単語単位で移動
  dd          ← 行を削除
  yy          ← 行をコピー（ヤンク）
  p           ← 貼り付け（ペースト）
  u           ← 元に戻す（アンドゥ）
```

### 挿入モード（Insert Mode）

テキストを入力するモードです。普通のエディタと同じように文字を入力できます。

```
挿入モードに入る方法:

  i    ← カーソルの前に挿入
  a    ← カーソルの後に挿入
  I    ← 行頭に挿入
  A    ← 行末に挿入
  o    ← 下に新しい行を作って挿入
  O    ← 上に新しい行を作って挿入
```

画面左下に `-- INSERT --` と表示されていれば挿入モードです。

### コマンドラインモード（Command-line Mode）

`:` を押すと画面下部にコマンドラインが表示され、Ex コマンドを入力できます。

```
よく使うコマンド:

  :w           ← 保存
  :q           ← 終了
  :wq          ← 保存して終了
  :e filename  ← ファイルを開く
  :set number  ← 行番号を表示
```

### ビジュアルモード（Visual Mode）

テキストを選択するモードです。選択した範囲に対して操作（削除、コピーなど）を行えます。

```
ビジュアルモードの種類:

  v       ← 文字単位の選択
  V       ← 行単位の選択
  Ctrl+v  ← 矩形（ブロック）選択
```

### モードの確認方法

画面左下の表示でどのモードにいるかわかります。

```
-- INSERT --          ← 挿入モード
-- VISUAL --          ← ビジュアルモード（文字単位）
-- VISUAL LINE --     ← ビジュアルモード（行単位）
-- VISUAL BLOCK --    ← ビジュアルモード（矩形選択）
（何も表示なし）        ← ノーマルモード
:                     ← コマンドラインモード
```

### ポイントまとめ

- Vim には4つの主要モードがあります
- ノーマルモードが「ホームポジション」です。迷ったら `ESC` を押しましょう
- 挿入モードでテキスト入力、ノーマルモードで操作、という使い分けが基本です
- 画面左下でどのモードにいるか確認できます

## 4. はじめての編集操作

### 実際にファイルを編集してみる

以下の手順で、Vim でファイルを作成・編集・保存してみましょう。

```bash
# 1. Vim を起動
$ vim hello.txt
```

```
# 2. i を押して挿入モードに入る
#    画面左下に -- INSERT -- と表示される

# 3. テキストを入力
Hello, Vim!
これは最初のファイルです。

# 4. ESC を押してノーマルモードに戻る
#    画面左下の -- INSERT -- が消える

# 5. :wq と入力して保存・終了
:wq
```

```bash
# 6. ファイルの内容を確認
$ cat hello.txt
Hello, Vim!
これは最初のファイルです。
```

### 基本的な編集の流れ

```
Vim の基本ワークフロー:

  ┌─────────┐     i, a, o     ┌─────────┐
  │ ノーマル  │ ──────────────▶ │  挿入    │
  │ モード   │                 │ モード   │
  │         │ ◀────────────── │         │
  │(移動・操作)│     ESC        │(テキスト │
  └─────────┘                 │  入力)   │
       │                      └─────────┘
       │ :wq
       ▼
    保存・終了
```

プログラマーは、**ノーマルモードで移動・操作 → 挿入モードで入力 → ESC で戻る** を繰り返します。

### ポイントまとめ

- `vim ファイル名` → `i` → テキスト入力 → `ESC` → `:wq` が最も基本的な流れです
- ノーマルモードと挿入モードの切り替えに慣れることが上達の第一歩です

## 5. vimtutorで学ぶ

Vim には公式のチュートリアルが付属しています。ターミナルで以下を実行するだけで始められます。

```bash
# 英語版
$ vimtutor

# 日本語版（利用可能な場合）
$ vimtutor ja
```

vimtutor は、実際に Vim を操作しながら基本操作を学べるインタラクティブなチュートリアルです。所要時間は約25〜30分で、以下のレッスンが含まれています。

| レッスン | 内容 |
|----------|------|
| レッスン1 | カーソル移動、削除、挿入、追記 |
| レッスン2 | 削除コマンド、オペレータとモーション |
| レッスン3 | 貼り付け、置き換え、変更 |
| レッスン4 | 位置とファイル情報、検索、対応する括弧 |
| レッスン5 | 外部コマンド、ファイル操作 |
| レッスン6 | 新しい行の追加、置換コマンド |
| レッスン7 | ヘルプの使い方、設定 |

> **おすすめ**: Vim を始めたら、まず vimtutor を2〜3回繰り返してください。基本操作が手になじむまで練習するのが上達の近道です。

### ポイントまとめ

- `vimtutor` コマンドで公式チュートリアルを起動できます
- 約30分で基本操作を学べます
- 2〜3回繰り返すと手になじみます

## 6. よくある初心者のつまずきポイント

### Vimから出られない

世界で最も検索されている Stack Overflow の質問の1つが「How do I exit Vim?」です。

```
解決方法:

1. まず ESC を何度か押す（ノーマルモードに戻る）
2. :q! と入力して Enter（変更を破棄して終了）

  または

1. ESC を何度か押す
2. :wq と入力して Enter（保存して終了）
```

### 日本語入力モードとの衝突

日本語入力（IME）がオンの状態で ESC を押すと、期待通りに動作しないことがあります。

```
対策:
- ESC の前に必ず IME をオフにする習慣をつける
- Ctrl+[ は ESC と同じ機能を持ち、IME の影響を受けにくい
```

### 意図しない操作をしてしまった

```
慌てないでください:

  u        ← 直前の操作を元に戻す（アンドゥ）
  Ctrl+r   ← アンドゥを取り消す（リドゥ）
  :e!      ← ファイルを最後に保存した状態に戻す
```

### 画面が固まったように見える

`Ctrl+s` を押すと端末の出力が一時停止します（Vim の機能ではなく、端末の機能です）。

```
解決方法:
  Ctrl+q  ← 端末の出力を再開する
```

### ポイントまとめ

- 困ったらまず `ESC` を連打してノーマルモードに戻りましょう
- `:q!` で強制終了、`u` でアンドゥができます
- 画面が固まったら `Ctrl+q` を試してください
