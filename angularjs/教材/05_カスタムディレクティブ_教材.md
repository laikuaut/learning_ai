# 第5章：カスタムディレクティブ

## この章のゴール

- カスタムディレクティブの必要性と利点を理解する
- ディレクティブ定義オブジェクト（DDO）の構造を把握する
- `restrict` オプションの種類と使い分けを学ぶ
- スコープ分離（`@`、`=`、`&`）を使った再利用可能なコンポーネントを作れる
- `link` 関数でDOM操作を行える
- `transclude` でコンテンツ差し込みができる

---

## 5.1 なぜカスタムディレクティブを作るのか

カスタムディレクティブを使うと、**再利用可能なUI部品**を HTML タグとして定義できます。

```
【ディレクティブのメリット】
┌──────────────────────────────────┐
│ 再利用性    → 何度でも使い回せる   │
│ カプセル化  → ロジックとUIを一体化 │
│ 宣言的      → HTMLを読むだけで理解 │
│ テスト容易  → 単体テストしやすい   │
└──────────────────────────────────┘
```

### 使用前と使用後の比較

```html
<!-- ❌ ディレクティブなし: HTMLが複雑になる -->
<div class="alert alert-warning">
  <span class="alert-icon">⚠️</span>
  <span class="alert-text">注意してください</span>
  <button class="alert-close" ng-click="...">×</button>
</div>

<!-- ✅ ディレクティブあり: シンプルで意味が明確 -->
<alert-box type="warning">注意してください</alert-box>
```

## 5.2 ディレクティブの基本構造

### ディレクティブ定義オブジェクト（DDO: Directive Definition Object）

```javascript
angular.module('myApp')
  .directive('myDirective', function() {
    return {
      restrict: 'E',          // 使用方法の制限
      scope: {},               // スコープの設定
      template: '<div>...</div>', // テンプレート
      templateUrl: 'tpl.html', // 外部テンプレート
      link: function(scope, element, attrs) { }, // DOM操作
      transclude: false,       // コンテンツ差し込み
      replace: false,          // 要素の置換
      controller: function() { } // コントローラ
    };
  });
```

### 命名規則

```
JavaScript:  myDirective     （キャメルケース）
HTML:        my-directive    （ケバブケース）
             ↕ 自動変換
```

### 最もシンプルな例

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body>
  <hello-world></hello-world>
  <hello-world></hello-world>

  <script>
    angular.module('myApp', [])
      .directive('helloWorld', function() {
        return {
          restrict: 'E',
          template: '<h2 style="color: #42b883;">こんにちは、AngularJS！</h2>'
        };
      });
  </script>
</body>
</html>
```

---

## 5.3 restrict — 使用方法の制限

| 値 | 意味 | HTML での使い方 |
|----|------|-----------------|
| `'E'` | Element（要素） | `<my-directive></my-directive>` |
| `'A'` | Attribute（属性） | `<div my-directive></div>` |
| `'C'` | Class（クラス） | `<div class="my-directive"></div>` |
| `'M'` | Comment（コメント） | `<!-- directive: my-directive -->` |
| `'EA'` | 要素＋属性（両方OK） | 組み合わせ可能 |

```javascript
// 実務では 'E'（要素）または 'A'（属性）が一般的
// 'EA' にしておくと柔軟
.directive('myWidget', function() {
  return {
    restrict: 'EA',  // <my-widget> でも <div my-widget> でもOK
    template: '<div>ウィジェット</div>'
  };
})
```

> **よくある間違い：** `restrict` を省略すると、デフォルトは `'EA'` です。`'E'` だけにしたい場合は明示的に指定しましょう。

## 5.4 スコープの分離（Isolated Scope）

ディレクティブに専用のスコープを与えることで、**外部のスコープと分離**し、再利用性を高めます。

### 3つのバインディング記号

| 記号 | 名前 | 方向 | 用途 |
|------|------|------|------|
| `@` | 文字列バインディング | 親 → 子（単方向） | 文字列の受け渡し |
| `=` | 双方向バインディング | 親 ⇄ 子 | データの同期 |
| `&` | コールバックバインディング | 子 → 親 | イベント通知 |

```
【バインディングの方向】

  @ 文字列（単方向）
  親 ─────→ 子ディレクティブ

  = 双方向
  親 ⇄ 子ディレクティブ

  & コールバック
  親 ←───── 子ディレクティブ（関数を呼ぶ）
```

### @ 文字列バインディング

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .greeting-card { border: 2px solid #42b883; border-radius: 8px; padding: 16px; margin: 8px; display: inline-block; }
    .greeting-card h3 { color: #42b883; margin: 0 0 8px; }
  </style>
</head>
<body ng-controller="MainCtrl">
  <!-- @ は文字列として渡す（{{}} で式を埋め込める） -->
  <greeting-card name="田中太郎" message="ようこそ！"></greeting-card>
  <greeting-card name="{{ dynamicName }}" message="こんにちは！"></greeting-card>

  <script>
    angular.module('myApp', [])
      .controller('MainCtrl', function($scope) {
        $scope.dynamicName = '鈴木花子';
      })
      .directive('greetingCard', function() {
        return {
          restrict: 'E',
          scope: {
            name: '@',     // 文字列として受け取る
            message: '@'   // 文字列として受け取る
          },
          template: '<div class="greeting-card">' +
                    '  <h3>{{ name }}さん</h3>' +
                    '  <p>{{ message }}</p>' +
                    '</div>'
        };
      });
  </script>
</body>
</html>
```

### = 双方向バインディング

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="MainCtrl">
  <h3>親のカウント: {{ count }}</h3>
  <button ng-click="count = count + 1">親で+1</button>
  
  <!-- = で双方向バインド -->
  <counter-widget value="count"></counter-widget>

  <script>
    angular.module('myApp', [])
      .controller('MainCtrl', function($scope) {
        $scope.count = 0;
      })
      .directive('counterWidget', function() {
        return {
          restrict: 'E',
          scope: {
            value: '='  // 双方向バインディング
          },
          template: '<div style="border: 1px solid #ddd; padding: 10px; margin: 10px 0;">' +
                    '  <p>ディレクティブ内のカウント: {{ value }}</p>' +
                    '  <button ng-click="value = value + 1">ディレクティブで+1</button>' +
                    '</div>'
        };
      });
  </script>
</body>
</html>
```

### & コールバックバインディング

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .delete-btn { background: #e74c3c; color: white; border: none; padding: 6px 12px; border-radius: 4px; cursor: pointer; }
  </style>
</head>
<body ng-controller="MainCtrl">
  <h2>タスク一覧</h2>
  <task-item ng-repeat="task in tasks"
             name="task.name"
             on-delete="removeTask(task)">
  </task-item>
  
  <p>残り: {{ tasks.length }}件</p>

  <script>
    angular.module('myApp', [])
      .controller('MainCtrl', function($scope) {
        $scope.tasks = [
          { name: '買い物' },
          { name: 'メール返信' },
          { name: 'レポート作成' }
        ];
        $scope.removeTask = function(task) {
          var idx = $scope.tasks.indexOf(task);
          if (idx > -1) $scope.tasks.splice(idx, 1);
        };
      })
      .directive('taskItem', function() {
        return {
          restrict: 'E',
          scope: {
            name: '=',       // タスク名（双方向）
            onDelete: '&'    // 削除コールバック
          },
          template: '<div style="padding: 8px; border-bottom: 1px solid #eee;">' +
                    '  {{ name }} ' +
                    '  <button class="delete-btn" ng-click="onDelete()">削除</button>' +
                    '</div>'
        };
      });
  </script>
</body>
</html>
```

### ポイントまとめ

- `@` は文字列の受け渡し、HTML属性には `{{ }}` で式を埋め込める
- `=` は変数の双方向バインド、HTML属性には変数名をそのまま書く
- `&` は親のメソッドを呼び出すコールバック

## 5.5 link 関数 — DOM操作

`link` 関数は、**DOM操作やイベントリスナーの登録**を行う場所です。

```javascript
link: function(scope, element, attrs) {
  // scope   — ディレクティブのスコープ
  // element — jqLite でラップされたDOM要素
  // attrs   — HTML属性のオブジェクト
}
```

### jqLite の主なメソッド

| メソッド | 説明 |
|----------|------|
| `element.css(prop, val)` | スタイルを設定 |
| `element.addClass(cls)` | クラスを追加 |
| `element.removeClass(cls)` | クラスを削除 |
| `element.on(event, fn)` | イベントリスナー登録 |
| `element.find(tag)` | 子要素を検索（タグ名のみ） |
| `element.text()` | テキスト取得/設定 |
| `element.html()` | HTML取得/設定 |

### 例：ツールチップディレクティブ

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .tooltip-container { position: relative; display: inline-block; }
    .tooltip-popup { 
      position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%);
      background: #333; color: white; padding: 6px 12px; border-radius: 4px;
      font-size: 13px; white-space: nowrap; margin-bottom: 6px;
    }
    .tooltip-popup::after {
      content: ''; position: absolute; top: 100%; left: 50%;
      margin-left: -5px; border: 5px solid transparent; border-top-color: #333;
    }
  </style>
</head>
<body ng-controller="MainCtrl">
  <div style="padding: 60px;">
    <button my-tooltip="保存します">保存</button>
    <button my-tooltip="データを削除します">削除</button>
    <button my-tooltip="{{ dynamicTip }}">ヘルプ</button>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('MainCtrl', function($scope) {
        $scope.dynamicTip = 'ヘルプを表示します';
      })
      .directive('myTooltip', function() {
        return {
          restrict: 'A',
          link: function(scope, element, attrs) {
            var tooltip = null;
            
            element.css('cursor', 'pointer');
            element.addClass('tooltip-container');
            
            element.on('mouseenter', function() {
              tooltip = angular.element(
                '<div class="tooltip-popup">' + attrs.myTooltip + '</div>'
              );
              element.append(tooltip);
            });
            
            element.on('mouseleave', function() {
              if (tooltip) {
                tooltip.remove();
                tooltip = null;
              }
            });
            
            // クリーンアップ
            scope.$on('$destroy', function() {
              element.off('mouseenter mouseleave');
            });
          }
        };
      });
  </script>
</body>
</html>
```

## 5.6 transclude — コンテンツ差し込み

`transclude` を使うと、**ディレクティブの利用者が中にHTMLコンテンツを差し込める**ようになります。

```html
<!-- 使う側 -->
<my-panel title="設定">
  <p>ここに好きな内容を入れられる</p>
</my-panel>

<!-- ディレクティブの template -->
<div class="panel">
  <h3>{{ title }}</h3>
  <div ng-transclude></div>  ← ここに差し込まれる
</div>
```

### 折りたたみパネルの例

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .panel { border: 1px solid #ddd; border-radius: 8px; margin: 10px 0; overflow: hidden; }
    .panel-header { background: #35495e; color: white; padding: 12px 16px; cursor: pointer; display: flex; justify-content: space-between; }
    .panel-body { padding: 16px; }
  </style>
</head>
<body>
  <h2>折りたたみパネル</h2>

  <collapsible-panel title="パネル1: 基本情報">
    <p>名前: 田中太郎</p>
    <p>メール: tanaka@example.com</p>
  </collapsible-panel>

  <collapsible-panel title="パネル2: 詳細設定">
    <ul>
      <li>設定A: 有効</li>
      <li>設定B: 無効</li>
      <li>設定C: 有効</li>
    </ul>
  </collapsible-panel>

  <script>
    angular.module('myApp', [])
      .directive('collapsiblePanel', function() {
        return {
          restrict: 'E',
          transclude: true,
          scope: {
            title: '@'
          },
          template: '<div class="panel">' +
                    '  <div class="panel-header" ng-click="isOpen = !isOpen">' +
                    '    {{ title }}' +
                    '    <span>{{ isOpen ? "▲" : "▼" }}</span>' +
                    '  </div>' +
                    '  <div class="panel-body" ng-show="isOpen" ng-transclude></div>' +
                    '</div>',
          link: function(scope) {
            scope.isOpen = true;
          }
        };
      });
  </script>
</body>
</html>
```

### ポイントまとめ

- `transclude: true` を設定し、テンプレートに `ng-transclude` を配置する
- 差し込まれたコンテンツは**親スコープ**を参照する（ディレクティブの isolated scope ではない）
- 再利用可能なレイアウト部品（パネル、モーダル、タブ）に最適

## 5.7 まとめ

| トピック | ポイント |
|----------|----------|
| DDO | ディレクティブの設定を定義するオブジェクト |
| restrict | `'E'`（要素）、`'A'`（属性）が実務で一般的 |
| scope `@` | 文字列の単方向バインディング |
| scope `=` | 変数の双方向バインディング |
| scope `&` | 親メソッドのコールバック |
| link | DOM操作・イベント登録の場所 |
| transclude | 利用者がコンテンツを差し込める仕組み |
