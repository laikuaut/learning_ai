# 実践課題05：レスポンシブブログレイアウト ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第2章（ボックスモデル）、第3章（Flexbox）、第4章（Grid）、第5章（レスポンシブデザイン）
> **課題の種類**: ミニプロジェクト
> **学習目標**: メディアクエリ（media query）とGrid/Flexboxを組み合わせ、画面幅に応じてレイアウトが変化するブログページを作る

---

## 完成イメージ

### デスクトップ（1024px以上）

```
┌──────────────────────────────────────────┐
│  ヘッダー（ロゴ + ナビ）                    │
├──────────────────────┬───────────────────┤
│                      │                   │
│  記事カード  記事カード │  サイドバー        │
│  記事カード  記事カード │  ・カテゴリ一覧    │
│  （2列グリッド）       │  ・人気記事        │
│                      │                   │
├──────────────────────┴───────────────────┤
│  フッター                                 │
└──────────────────────────────────────────┘
```

### タブレット（768px〜1023px）

```
┌──────────────────────────┐
│  ヘッダー                 │
├──────────────────────────┤
│  記事カード  記事カード     │
│  記事カード  記事カード     │
│  （2列グリッド）           │
├──────────────────────────┤
│  サイドバー（横幅100%）     │
├──────────────────────────┤
│  フッター                 │
└──────────────────────────┘
```

### モバイル（767px以下）

```
┌─────────────────┐
│  ヘッダー        │
├─────────────────┤
│  記事カード       │
│  記事カード       │
│  記事カード       │
│  記事カード       │
│  （1列）         │
├─────────────────┤
│  サイドバー       │
├─────────────────┤
│  フッター        │
└─────────────────┘
```

---

## 課題の要件

1. 3つのブレークポイント（breakpoint）でレイアウトを切り替える
   - モバイル: 767px以下（1列）
   - タブレット: 768px〜1023px（2列、サイドバーは下に移動）
   - デスクトップ: 1024px以上（2列+サイドバー横配置）
2. ヘッダーにロゴとナビゲーションを配置する
3. メインエリアに記事カードを4枚以上配置する
4. サイドバーにカテゴリ一覧と人気記事を配置する
5. フッターを作る
6. 記事カードには「カテゴリ」「タイトル」「抜粋」「日付」を含める

---

## ステップガイド

<details>
<summary>ステップ1：モバイルファースト（mobile first）でベースを書く</summary>

まずモバイル用のスタイルを書き、次に大きい画面用を `@media` で追加します。

```css
/* モバイル（デフォルト） */
.page {
  display: grid;
  grid-template-columns: 1fr;
}

/* タブレット */
@media (min-width: 768px) {
  .articles {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* デスクトップ */
@media (min-width: 1024px) {
  .page {
    grid-template-columns: 1fr 300px;
  }
}
```

</details>

<details>
<summary>ステップ2：記事カードを Grid で並べる</summary>

記事一覧のエリアも Grid にします。

```css
.articles {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .articles {
    grid-template-columns: repeat(2, 1fr);
  }
}
```

</details>

<details>
<summary>ステップ3：サイドバーの配置を切り替える</summary>

デスクトップではメインの横に、タブレット以下では下に配置します。`grid-template-areas` を使うと直感的です。

```css
@media (min-width: 1024px) {
  .page {
    grid-template-areas:
      "header  header"
      "main    sidebar"
      "footer  footer";
    grid-template-columns: 1fr 280px;
  }
}
```

</details>
