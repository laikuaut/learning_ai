## 6.1 文字列の基本操作

文字列はインデックスでアクセスでき、スライスで部分を取り出せます。

```python
text = "Python入門"

# インデックス（0から始まる）
print(text[0])    # P
print(text[-1])   # 門

# スライス [開始:終了:ステップ]
print(text[0:6])   # Python
print(text[6:])    # 入門
print(text[::-1])  # 門入nohtyP（逆順）
```

### 文字列の長さと繰り返し

```python
text = "Hello"
print(len(text))     # 5
print(text * 3)      # HelloHelloHello
print("H" in text)   # True
```

## 6.2 split() と join()

### split() - 文字列を分割してリストにする

```python
sentence = "りんご みかん バナナ"
fruits = sentence.split()  # スペースで分割
print(fruits)  # ['りんご', 'みかん', 'バナナ']

csv_line = "太郎,20,東京"
data = csv_line.split(",")  # カンマで分割
print(data)  # ['太郎', '20', '東京']

# 分割回数を制限
text = "a-b-c-d-e"
print(text.split("-", 2))  # ['a', 'b', 'c-d-e']
```

### join() - リストを結合して文字列にする

```python
words = ["Python", "は", "楽しい"]
sentence = "".join(words)
print(sentence)  # Pythonは楽しい

# 区切り文字を指定
fruits = ["りんご", "みかん", "バナナ"]
print(", ".join(fruits))  # りんご, みかん, バナナ

# パス結合の例
parts = ["Users", "taro", "documents"]
path = "/".join(parts)
print(path)  # Users/taro/documents
```

## 6.3 replace() - 文字列の置換

```python
text = "私はPythonが好きです。Pythonは楽しいです。"

# 置換
new_text = text.replace("Python", "プログラミング")
print(new_text)  # 私はプログラミングが好きです。プログラミングは楽しいです。

# 置換回数の制限
new_text = text.replace("Python", "Java", 1)  # 最初の1つだけ
print(new_text)  # 私はJavaが好きです。Pythonは楽しいです。

# チェーンで複数の置換
cleaned = "  Hello,  World!  ".replace(" ", "").replace(",", " ")
print(cleaned)  # Hello World!
```

## 6.4 strip() - 前後の空白・文字を除去

```python
text = "  Hello, World!  "
print(text.strip())    # "Hello, World!"（両端）
print(text.lstrip())   # "Hello, World!  "（左端のみ）
print(text.rstrip())   # "  Hello, World!"（右端のみ）

# 特定の文字を除去
url = "###重要なお知らせ###"
print(url.strip("#"))  # 重要なお知らせ

# 改行の除去（ファイル読み込みで活躍）
line = "データ行\n"
print(line.strip())  # "データ行"
```

## 6.5 find() と検索系メソッド

### find() - 文字列の位置を検索

```python
text = "Python Programming Python"

# 最初に見つかった位置を返す（なければ-1）
print(text.find("Python"))     # 0
print(text.find("Program"))    # 7
print(text.find("Java"))       # -1

# 検索開始位置を指定
print(text.find("Python", 1))  # 20（2番目のPython）

# index() は見つからないとエラーになる
# text.index("Java")  # ValueError!
```

### count() - 出現回数を数える

```python
text = "banana"
print(text.count("a"))    # 3
print(text.count("na"))   # 2
```

### startswith() / endswith() - 先頭・末尾の判定

```python
filename = "report_2024.csv"
print(filename.startswith("report"))  # True
print(filename.endswith(".csv"))      # True
print(filename.endswith((".csv", ".tsv")))  # True（タプルで複数指定）
```

## 6.6 大文字・小文字の変換

```python
text = "Hello World"
print(text.upper())      # HELLO WORLD
print(text.lower())      # hello world
print(text.title())      # Hello World
print(text.capitalize()) # Hello world
print(text.swapcase())   # hELLO wORLD
```

## 6.7 判定メソッド（isXxx）

```python
print("123".isdigit())      # True（数字のみ）
print("abc".isalpha())      # True（文字のみ）
print("abc123".isalnum())   # True（文字と数字のみ）
print("   ".isspace())      # True（空白のみ）
print("HELLO".isupper())    # True（大文字のみ）
print("hello".islower())    # True（小文字のみ）
```

### 実践例：入力値の検証

```python
age_input = "25"
if age_input.isdigit():
    age = int(age_input)
    print(f"年齢: {age}")
else:
    print("数値を入力してください")
```

## 6.8 f文字列と書式指定

### 基本のf文字列

```python
name = "太郎"
age = 20
print(f"名前: {name}, 年齢: {age}")
```

### 書式指定

```python
# 数値のフォーマット
pi = 3.14159265
print(f"円周率: {pi:.2f}")         # 3.14（小数点2桁）
print(f"パーセント: {0.856:.1%}")  # 85.6%

# 桁区切り
big_num = 1234567890
print(f"金額: {big_num:,}円")      # 1,234,567,890円

# 幅指定と揃え
for item, price in [("りんご", 150), ("バナナ", 200), ("メロン", 3000)]:
    print(f"{item:<6} {price:>6,}円")
# りんご      150円
# バナナ      200円
# メロン    3,000円

# 0埋め
for i in range(1, 4):
    print(f"ファイル_{i:03d}.txt")
# ファイル_001.txt
# ファイル_002.txt
# ファイル_003.txt
```

### format() メソッド

```python
# f文字列が使えない場面で活躍（テンプレートなど）
template = "{}さん、こんにちは！今日は{}です。"
print(template.format("太郎", "月曜日"))

# 名前付き
template = "{name}さんの点数は{score}点です。"
print(template.format(name="花子", score=85))
```

## 6.9 正規表現の基本（reモジュール）

正規表現を使うと、パターンで文字列を検索・置換できます。

```python
import re

# パターンにマッチするか検索
text = "電話番号は 090-1234-5678 です"
match = re.search(r"\d{3}-\d{4}-\d{4}", text)
if match:
    print(match.group())  # 090-1234-5678
```

### 主要なパターン

| パターン | 意味 |
|---------|------|
| `\d` | 数字1文字 |
| `\w` | 英数字またはアンダースコア |
| `\s` | 空白文字 |
| `.` | 任意の1文字 |
| `*` | 0回以上の繰り返し |
| `+` | 1回以上の繰り返し |
| `?` | 0回または1回 |
| `{n}` | n回の繰り返し |
| `[abc]` | a, b, cのいずれか |
| `^` | 行の先頭 |
| `$` | 行の末尾 |

### よく使うre関数

```python
import re

text = "今日は2024年3月15日、明日は2024年3月16日です"

# findall() - マッチする全てをリストで返す
dates = re.findall(r"\d{4}年\d{1,2}月\d{1,2}日", text)
print(dates)  # ['2024年3月15日', '2024年3月16日']

# sub() - パターンに一致する部分を置換
censored = re.sub(r"\d", "*", "電話: 090-1234-5678")
print(censored)  # 電話: ***-****-****

# split() - パターンで分割
parts = re.split(r"[,;、。]", "りんご、みかん。バナナ,メロン")
print(parts)  # ['りんご', 'みかん', 'バナナ', 'メロン']

# match() - 先頭からマッチ
if re.match(r"^[a-zA-Z0-9_.+-]+@[a-zA-Z0-9-]+\.[a-zA-Z]+$", "user@example.com"):
    print("有効なメールアドレスです")
```

### グループ化

```python
import re

text = "山田太郎(30歳), 鈴木花子(25歳)"
pattern = r"(\w+)\((\d+)歳\)"

for match in re.finditer(pattern, text):
    name = match.group(1)
    age = match.group(2)
    print(f"{name}: {age}歳")
# 山田太郎: 30歳
# 鈴木花子: 25歳
```

## 6.10 便利な文字列テクニック

### 複数行文字列

```python
message = """
件名: お知らせ
本文:
  来週の会議は火曜日に変更されました。
  よろしくお願いします。
""".strip()
print(message)
```

### 文字列の比較とソート

```python
# 大文字小文字を無視した比較
a = "Hello"
b = "hello"
print(a.lower() == b.lower())  # True

# 日本語のソート（基本はUnicodeコードポイント順）
names = ["田中", "鈴木", "佐藤", "高橋"]
print(sorted(names))  # Unicodeポイント順
```

### raw文字列

```python
# バックスラッシュをエスケープしない
path = r"C:\Users\taro\documents"
print(path)  # C:\Users\taro\documents

# 正規表現でよく使う
pattern = r"\d+\.\d+"  # 小数点を含む数値
```

## まとめ

| メソッド | 用途 | 例 |
|---------|------|-----|
| `split()` | 分割 | `"a,b".split(",")` → `['a','b']` |
| `join()` | 結合 | `",".join(['a','b'])` → `"a,b"` |
| `replace()` | 置換 | `"ab".replace("a","x")` → `"xb"` |
| `strip()` | 前後除去 | `" ab ".strip()` → `"ab"` |
| `find()` | 検索 | `"abc".find("b")` → `1` |
| `upper()/lower()` | 大小変換 | `"Ab".upper()` → `"AB"` |
| `startswith()` | 先頭判定 | `"abc".startswith("a")` → `True` |
| `isdigit()` | 数字判定 | `"123".isdigit()` → `True` |
| `format()` | 書式設定 | `"{}歳".format(20)` → `"20歳"` |
