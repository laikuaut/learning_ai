## 基本問題

### 問題1：順序なしリストの作成
以下の買い物リストを `<ul>` で作成してください。

- 牛乳
- パン
- 卵
- バター
- 砂糖

<details>
<summary>解答例</summary>

```html
<h2>買い物リスト</h2>
<ul>
  <li>牛乳</li>
  <li>パン</li>
  <li>卵</li>
  <li>バター</li>
  <li>砂糖</li>
</ul>
```

<!-- ul の直接の子要素は li のみです。li の中にはテキストや他の要素を入れられます。 -->

</details>

---

### 問題2：順序付きリストの作成
以下の料理の手順を `<ol>` で作成してください。

1. フライパンにバターを溶かす
2. 卵を割り入れる
3. 弱火で3分焼く
4. 塩こしょうで味付けする
5. お皿に盛り付ける

<details>
<summary>解答例</summary>

```html
<h2>目玉焼きの作り方</h2>
<ol>
  <li>フライパンにバターを溶かす</li>
  <li>卵を割り入れる</li>
  <li>弱火で3分焼く</li>
  <li>塩こしょうで味付けする</li>
  <li>お皿に盛り付ける</li>
</ol>
```

<!-- 順序が重要な手順にはolを使います。olの番号はブラウザが自動で付けてくれます。 -->

</details>

---

### 問題3：定義リスト
以下のWeb用語とその意味を定義リスト（`<dl>`）で作成してください。

- HTML：Webページの構造を定義するマークアップ言語
- CSS：Webページの見た目を装飾するスタイルシート言語
- JavaScript：Webページに動的な機能を追加するプログラミング言語

<details>
<summary>解答例</summary>

```html
<h2>Web用語集</h2>
<dl>
  <dt>HTML</dt>
  <dd>Webページの構造を定義するマークアップ言語</dd>

  <dt>CSS</dt>
  <dd>Webページの見た目を装飾するスタイルシート言語</dd>

  <dt>JavaScript</dt>
  <dd>Webページに動的な機能を追加するプログラミング言語</dd>
</dl>
```

<!-- dt が用語、dd がその定義です。dl の直接の子要素は dt と dd のみ配置できます。 -->

</details>

### 問題4：基本テーブル
以下のデータをテーブルで表示してください。ヘッダー行を `<th>` で作成すること。

| 商品名 | 価格 | 在庫 |
|---|---|---|
| りんご | 150円 | 20個 |
| みかん | 80円 | 35個 |
| バナナ | 100円 | 15個 |

<details>
<summary>解答例</summary>

```html
<table border="1">
  <tr>
    <th>商品名</th>
    <th>価格</th>
    <th>在庫</th>
  </tr>
  <tr>
    <td>りんご</td>
    <td>150円</td>
    <td>20個</td>
  </tr>
  <tr>
    <td>みかん</td>
    <td>80円</td>
    <td>35個</td>
  </tr>
  <tr>
    <td>バナナ</td>
    <td>100円</td>
    <td>15個</td>
  </tr>
</table>
```

<!-- th はヘッダーセル（太字・中央揃えで表示）、td は通常のデータセルです。 -->

</details>

## 応用問題

### 問題6：テーブルのグループ化
以下の売上データを `<thead>`, `<tbody>`, `<tfoot>` を使ってグループ化したテーブルを作成してください。

- ヘッダー：月、売上、経費
- データ：1月 100万円 60万円、2月 120万円 70万円、3月 90万円 55万円
- フッター：合計 310万円 185万円

<details>
<summary>ヒント</summary>

`<thead>` にはヘッダー行、`<tbody>` にはデータ行、`<tfoot>` にはフッター行（合計など）を入れます。HTMLの記述順序は thead → tfoot → tbody でも tbody → tfoot でもどちらでも構いませんが、 thead → tbody → tfoot の順が一般的です。

</details>

<details>
<summary>解答例</summary>

```html
<table border="1">
  <thead>
    <tr>
      <th>月</th>
      <th>売上</th>
      <th>経費</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1月</td>
      <td>100万円</td>
      <td>60万円</td>
    </tr>
    <tr>
      <td>2月</td>
      <td>120万円</td>
      <td>70万円</td>
    </tr>
    <tr>
      <td>3月</td>
      <td>90万円</td>
      <td>55万円</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th>合計</th>
      <td>310万円</td>
      <td>185万円</td>
    </tr>
  </tfoot>
</table>
```

<!-- thead/tbody/tfoot のグループ化は、CSSでのスタイリングやアクセシビリティの向上に役立ちます。 -->

</details>

### 問題7：セルの結合
以下のような時間割をテーブルで作成してください。「体育」は2コマ連続（rowspan）、「昼休み」は全列にまたがる（colspan）構造にしてください。

| 時限 | 月曜 | 火曜 |
|---|---|---|
| 1限 | 国語 | 数学 |
| 2限 | 国語 | 体育（2コマ） |
| 昼休み（2列結合） | |
| 3限 | 英語 | 体育（上から続く） |

<details>
<summary>ヒント</summary>

`colspan="2"` で列方向の結合、`rowspan="2"` で行方向の結合を行います。結合された分だけ後続の `<td>` を省略します。

</details>

<details>
<summary>解答例</summary>

```html
<table border="1">
  <thead>
    <tr>
      <th>時限</th>
      <th>月曜</th>
      <th>火曜</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1限</td>
      <td>国語</td>
      <td>数学</td>
    </tr>
    <tr>
      <td>2限</td>
      <td>国語</td>
      <!-- rowspan="2" で2行分を結合。次の行の火曜セルは不要 -->
      <td rowspan="2">体育</td>
    </tr>
    <tr>
      <!-- colspan="2" で2列分を結合 -->
      <td colspan="2">昼休み</td>
      <!-- 火曜は上のrowspanで結合済みなので不要 -->
    </tr>
    <tr>
      <td>3限</td>
      <td>英語</td>
      <!-- 火曜は上のrowspanの範囲内なので不要 -->
    </tr>
  </tbody>
</table>
```

<!-- rowspan/colspan を使うときは、結合先のセルを省略することを忘れがちです。結合した分だけtd/thが減ります。 -->

</details>

## チャレンジ問題

### 問題8：総合問題 - レストランのメニューページ
以下の要件を満たす完全なHTMLページを作成してください。

- ページタイトル「レストランメニュー」
- h1 で「レストラン ABC メニュー」
- 「前菜」「メイン」「デザート」の3カテゴリを順序なしリストのネストで表示
  - 各料理名の横に価格を記述
- 注文ランキングを順序付きリスト（`<ol>`）で表示
- 栄養成分表をテーブルで作成（thead/tbody/tfoot を使用、合計行に colspan を使用）
  - 列：メニュー、カロリー、たんぱく質、価格
  - 3品分のデータ
  - 合計カロリーをフッターに表示

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>レストランメニュー</title>
</head>
<body>
  <h1>レストラン ABC メニュー</h1>

  <!-- カテゴリ別メニュー（ネストしたリスト） -->
  <h2>メニュー一覧</h2>
  <ul>
    <li>前菜
      <ul>
        <li>シーザーサラダ — 800円</li>
        <li>コーンスープ — 500円</li>
        <li>カルパッチョ — 1,200円</li>
      </ul>
    </li>
    <li>メイン
      <ul>
        <li>ハンバーグステーキ — 1,500円</li>
        <li>チキングリル — 1,300円</li>
        <li>パスタ（日替わり） — 1,100円</li>
      </ul>
    </li>
    <li>デザート
      <ul>
        <li>ティラミス — 600円</li>
        <li>季節のフルーツ — 500円</li>
        <li>チョコレートケーキ — 700円</li>
      </ul>
    </li>
  </ul>

  <!-- 人気ランキング（順序付きリスト） -->
  <h2>注文ランキング</h2>
  <ol>
    <li>ハンバーグステーキ</li>
    <li>シーザーサラダ</li>
    <li>ティラミス</li>
  </ol>

  <!-- 栄養成分表（テーブル） -->
  <h2>栄養成分表</h2>
  <table border="1">
    <thead>
      <tr>
        <th>メニュー</th>
        <th>カロリー</th>
        <th>たんぱく質</th>
        <th>価格</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>ハンバーグステーキ</td>
        <td>650 kcal</td>
        <td>30g</td>
        <td>1,500円</td>
      </tr>
      <tr>
        <td>チキングリル</td>
        <td>450 kcal</td>
        <td>35g</td>
        <td>1,300円</td>
      </tr>
      <tr>
        <td>シーザーサラダ</td>
        <td>250 kcal</td>
        <td>8g</td>
        <td>800円</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <!-- colspanで「合計」ラベルを2列分に広げる -->
        <th colspan="2">合計カロリー</th>
        <td colspan="2">1,350 kcal</td>
      </tr>
    </tfoot>
  </table>
</body>
</html>
```

</details>
