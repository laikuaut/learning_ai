# 第4章：配列と文字列 - 演習問題

この演習では、配列の操作と文字列（`char` 配列）の扱いを実践します。

---

## 基本問題

### 問題1：配列の最大値・最小値を求める

10個の整数を配列に格納し、最大値と最小値を求めて表示するプログラムを作成してください。配列の値は初期化子で直接代入してかまいません。

**期待される出力例：**
```
配列: 34 72 15 89 43 56 21 67 98 10
最大値: 98
最小値: 10
```

<details>
<summary>ヒント</summary>

- 最大値用の変数を配列の最初の要素で初期化します
- ループで全要素を順に比較し、より大きい（小さい）値が見つかれば更新します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int data[] = {34, 72, 15, 89, 43, 56, 21, 67, 98, 10};
    int size = sizeof(data) / sizeof(data[0]);  /* 配列の要素数を計算 */
    int max, min;

    /* 最大値・最小値を最初の要素で初期化 */
    max = data[0];
    min = data[0];

    /* 配列の内容を表示 */
    printf("配列: ");
    for (int i = 0; i < size; i++) {
        printf("%d ", data[i]);
    }
    printf("\n");

    /* 2番目の要素から順に比較 */
    for (int i = 1; i < size; i++) {
        if (data[i] > max) {
            max = data[i];  /* より大きい値が見つかれば更新 */
        }
        if (data[i] < min) {
            min = data[i];  /* より小さい値が見つかれば更新 */
        }
    }

    printf("最大値: %d\n", max);
    printf("最小値: %d\n", min);

    return 0;
}
```

**ポイント：** `sizeof(data) / sizeof(data[0])` で配列の要素数を求められます。配列全体のバイト数を1要素のバイト数で割る定番テクニックです。

</details>

### 問題2：配列の要素を逆順に表示

整数配列の要素を逆順に表示するプログラムを作成してください。元の配列の内容も表示して比較できるようにしてください。

**期待される出力例：**
```
元の配列:   1 2 3 4 5 6 7 8
逆順の配列: 8 7 6 5 4 3 2 1
```

<details>
<summary>ヒント</summary>

- 逆順に表示するだけなら、ループを `size - 1` から `0` まで逆向きに回せばOKです
- `for (int i = size - 1; i >= 0; i--)` のように書きます

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    int data[] = {1, 2, 3, 4, 5, 6, 7, 8};
    int size = sizeof(data) / sizeof(data[0]);

    /* 元の配列を表示 */
    printf("元の配列:   ");
    for (int i = 0; i < size; i++) {
        printf("%d ", data[i]);
    }
    printf("\n");

    /* 逆順に表示（インデックスを末尾から先頭に向かって回す） */
    printf("逆順の配列: ");
    for (int i = size - 1; i >= 0; i--) {
        printf("%d ", data[i]);
    }
    printf("\n");

    return 0;
}
```

**発展：** 表示だけでなく、配列の中身自体を逆順に並べ替える場合は、先頭と末尾の要素を交換しながら中央に向かって進むアルゴリズムを使います。

</details>

### 問題3：文字列の長さを手動で数える

文字列を `char` 配列で定義し、`strlen` を使わずにループで文字数を数えて表示するプログラムを作成してください。

**期待される出力例：**
```
文字列: "Hello, World!"
文字数: 13
```

<details>
<summary>ヒント</summary>

- C言語の文字列は末尾にヌル文字 `'\0'` が入っています
- `while (str[i] != '\0')` でヌル文字が見つかるまでカウントします

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    char str[] = "Hello, World!";
    int length = 0;

    /*
     * C言語の文字列はヌル文字('\0')で終端される
     * ヌル文字が現れるまでカウントすれば文字数がわかる
     */
    while (str[length] != '\0') {
        length++;
    }

    printf("文字列: \"%s\"\n", str);
    printf("文字数: %d\n", length);

    return 0;
}
```

**補足：** 標準ライブラリの `strlen` 関数（`#include <string.h>`）も内部的には同様の処理を行っています。この仕組みを理解することは、C言語の文字列操作を学ぶ上で非常に重要です。

</details>

## 応用問題

### 問題4：配列の平均と標準偏差

実数の配列（5〜10個程度）から平均値と標準偏差を計算して表示するプログラムを作成してください。

**標準偏差の計算式：**
- 平均 = 全要素の合計 / 要素数
- 分散 = 各要素と平均の差の2乗の合計 / 要素数
- 標準偏差 = 分散の平方根

**期待される出力例：**
```
データ: 80.0 75.0 90.0 85.0 70.0 95.0 88.0
要素数: 7
平均: 83.29
標準偏差: 8.38
```

<details>
<summary>ヒント</summary>

- 平方根の計算には `math.h` の `sqrt` 関数を使います
- コンパイル時に `-lm` オプションが必要な場合があります（例：`gcc -o prog prog.c -lm`）
- 分散の計算では、各要素について `(data[i] - average)` の2乗を合計します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>
#include <math.h>  /* sqrt 関数を使うために必要 */

int main(void) {
    double data[] = {80.0, 75.0, 90.0, 85.0, 70.0, 95.0, 88.0};
    int size = sizeof(data) / sizeof(data[0]);
    double sum = 0.0, average, variance = 0.0, stddev;

    /* データを表示 */
    printf("データ: ");
    for (int i = 0; i < size; i++) {
        printf("%.1f ", data[i]);
    }
    printf("\n");
    printf("要素数: %d\n", size);

    /* 合計を求める */
    for (int i = 0; i < size; i++) {
        sum += data[i];
    }

    /* 平均を計算 */
    average = sum / size;

    /* 分散を計算（各要素と平均の差の2乗の合計 / 要素数） */
    for (int i = 0; i < size; i++) {
        double diff = data[i] - average;
        variance += diff * diff;
    }
    variance /= size;

    /* 標準偏差 = 分散の平方根 */
    stddev = sqrt(variance);

    printf("平均: %.2f\n", average);
    printf("標準偏差: %.2f\n", stddev);

    return 0;
}
```

**コンパイル方法：** `gcc -o stats stats.c -lm`（`-lm` は数学ライブラリのリンク指定）

</details>

### 問題5：文字列の大文字小文字変換

文字列を入力し、大文字を小文字に、小文字を大文字に変換して表示するプログラムを作成してください。`toupper`/`tolower` は使わず、ASCII値の差を利用して変換してください。

**期待される出力例：**
```
文字列を入力してください: Hello, World!
変換結果: hELLO, wORLD!
```

<details>
<summary>ヒント</summary>

- ASCII コードでは `'A'` は 65、`'a'` は 97 です。差は 32 です
- 大文字の判定：`'A' <= c && c <= 'Z'`
- 大文字→小文字：`c + 32` または `c + ('a' - 'A')`
- アルファベット以外の文字はそのまま出力します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

int main(void) {
    char str[256];
    int i = 0;

    printf("文字列を入力してください: ");
    /*
     * fgets で1行読み込む（scanf は空白で区切ってしまうため）
     * 改行文字も含まれる点に注意
     */
    fgets(str, sizeof(str), stdin);

    /* 末尾の改行文字を除去 */
    while (str[i] != '\0') {
        i++;
    }
    if (i > 0 && str[i - 1] == '\n') {
        str[i - 1] = '\0';
    }

    printf("変換結果: ");
    i = 0;
    while (str[i] != '\0') {
        if (str[i] >= 'A' && str[i] <= 'Z') {
            /*
             * 大文字→小文字: ASCIIコードに32を加算
             * 'A'(65) + 32 = 'a'(97)
             */
            printf("%c", str[i] + ('a' - 'A'));
        } else if (str[i] >= 'a' && str[i] <= 'z') {
            /*
             * 小文字→大文字: ASCIIコードから32を減算
             * 'a'(97) - 32 = 'A'(65)
             */
            printf("%c", str[i] - ('a' - 'A'));
        } else {
            /* アルファベット以外はそのまま出力 */
            printf("%c", str[i]);
        }
        i++;
    }
    printf("\n");

    return 0;
}
```

**ポイント：** `'a' - 'A'` と書くことで、マジックナンバー `32` を避けて意図が明確になります。

</details>

## チャレンジ問題

### 問題6：文字列の回文判定

文字列を入力し、それが回文（前から読んでも後ろから読んでも同じ文字列）かどうかを判定するプログラムを作成してください。英数字のみ（半角）を対象とし、大文字小文字は区別しないものとします。

**期待される出力例：**
```
文字列を入力してください: level
"level" は回文です。
```

```
文字列を入力してください: Hello
"Hello" は回文ではありません。
```

```
文字列を入力してください: Madam
"Madam" は回文です。
```

<details>
<summary>ヒント</summary>

- まず文字列の長さを求めます
- 先頭と末尾から同時に比較していき、中央で出会えば回文です
- 大文字小文字を区別しない比較のため、比較前に統一（例：小文字に変換）します

</details>

<details>
<summary>解答例</summary>

```c
#include <stdio.h>

/* 大文字を小文字に変換する関数 */
char to_lower(char c) {
    if (c >= 'A' && c <= 'Z') {
        return c + ('a' - 'A');
    }
    return c;
}

int main(void) {
    char str[256];
    int len = 0;
    int is_palindrome = 1;  /* 回文フラグ（1:回文, 0:回文でない） */

    printf("文字列を入力してください: ");
    fgets(str, sizeof(str), stdin);

    /* 文字列の長さを求める（改行文字を除く） */
    while (str[len] != '\0' && str[len] != '\n') {
        len++;
    }
    /* 末尾の改行を除去 */
    if (len > 0 && str[len - 1] == '\n') {
        str[len - 1] = '\0';
        len--;
    }

    /*
     * 先頭(left)と末尾(right)から中央に向かって比較
     * 一つでも異なる文字があれば回文ではない
     */
    int left = 0;
    int right = len - 1;

    while (left < right) {
        /* 大文字小文字を区別せずに比較 */
        if (to_lower(str[left]) != to_lower(str[right])) {
            is_palindrome = 0;
            break;  /* 異なる文字が見つかったら即終了 */
        }
        left++;
        right--;
    }

    if (is_palindrome) {
        printf("\"%s\" は回文です。\n", str);
    } else {
        printf("\"%s\" は回文ではありません。\n", str);
    }

    return 0;
}
```

**発展課題：** スペースや記号を無視して判定するバージョンを作ってみましょう。例えば `"A man a plan a canal Panama"` のような文も回文と判定できるようにします。

</details>
