# 第4章：フィルターとサービス

## この章のゴール

- 組み込みフィルター（Built-in Filter）を使ってデータを整形できる
- カスタムフィルターを作成できる
- 組み込みサービス（`$http`、`$timeout` など）を理解する
- `$http` でAPIと通信できる
- カスタムサービスを `factory`、`service`、`provider` で作成できる
- 依存性注入（Dependency Injection）の仕組みを理解する

---

## 4.1 フィルターとは

フィルター（Filter）は、**テンプレート内でデータを表示用に整形する仕組み**です。パイプ記号 `|` を使って適用します。

```
{{ 式 | フィルター名 }}
{{ 式 | フィルター名:引数1:引数2 }}
{{ 式 | フィルター1 | フィルター2 }}   ← チェーン可能
```

---

## 4.2 組み込みフィルター

### 一覧

| フィルター | 用途 | 例 |
|-----------|------|-----|
| `currency` | 通貨表示 | `{{ 1234 | currency:'¥' }}` → ¥1,234.00 |
| `number` | 数値フォーマット | `{{ 3.14159 | number:2 }}` → 3.14 |
| `date` | 日付フォーマット | `{{ dt | date:'yyyy/MM/dd' }}` |
| `uppercase` | 大文字変換 | `{{ 'hello' | uppercase }}` → HELLO |
| `lowercase` | 小文字変換 | `{{ 'HELLO' | lowercase }}` → hello |
| `json` | JSON整形表示 | `{{ obj | json }}` |
| `limitTo` | 配列・文字列の切り出し | `{{ items | limitTo:3 }}` |
| `orderBy` | 配列のソート | `ng-repeat="x in items | orderBy:'name'"` |
| `filter` | 配列の検索・絞り込み | `ng-repeat="x in items | filter:query"` |

### 実用例：商品リスト

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    table { border-collapse: collapse; width: 100%; margin-top: 10px; }
    th, td { border: 1px solid #ddd; padding: 8px; }
    th { background: #35495e; color: white; }
    input, select { padding: 6px; margin: 4px; }
  </style>
</head>
<body ng-controller="ProductCtrl">
  <h2>商品一覧</h2>
  
  <!-- 検索 -->
  <input ng-model="search" placeholder="商品を検索...">
  
  <!-- ソート -->
  <select ng-model="sortKey">
    <option value="name">名前順</option>
    <option value="price">価格順</option>
    <option value="-price">価格順（降順）</option>
  </select>

  <table>
    <tr>
      <th>商品名</th><th>価格</th><th>在庫</th><th>登録日</th>
    </tr>
    <tr ng-repeat="p in products | filter:search | orderBy:sortKey">
      <td>{{ p.name }}</td>
      <td>{{ p.price | currency:'¥':0 }}</td>
      <td>{{ p.stock | number }}個</td>
      <td>{{ p.date | date:'yyyy年MM月dd日' }}</td>
    </tr>
  </table>
  
  <p>表示件数: {{ (products | filter:search).length }} / {{ products.length }}</p>

  <script>
    angular.module('myApp', [])
      .controller('ProductCtrl', function($scope) {
        $scope.sortKey = 'name';
        $scope.products = [
          { name: 'ノートPC', price: 89800, stock: 15, date: new Date('2024-01-15') },
          { name: 'マウス', price: 2980, stock: 150, date: new Date('2024-03-01') },
          { name: 'キーボード', price: 7980, stock: 80, date: new Date('2024-02-10') },
          { name: 'モニター', price: 34800, stock: 25, date: new Date('2024-01-20') },
          { name: 'USBメモリ', price: 1280, stock: 200, date: new Date('2024-04-05') }
        ];
      });
  </script>
</body>
</html>
```

### コントローラ内でフィルターを使う

テンプレートだけでなく、コントローラ内でも `$filter` サービスを注入して使えます。

```javascript
angular.module('myApp', [])
  .controller('MyCtrl', function($scope, $filter) {
    var price = 1234.5;
    
    // $filter('フィルター名')(値, 引数...)
    $scope.formatted = $filter('currency')(price, '¥', 0);
    // → "¥1,235"
    
    var items = [3, 1, 4, 1, 5, 9, 2, 6];
    $scope.sorted = $filter('orderBy')(items);
    // → [1, 1, 2, 3, 4, 5, 6, 9]
  });
```

### ポイントまとめ

- フィルターはテンプレートで `|` を使って適用する
- `orderBy` と `filter` は `ng-repeat` と組み合わせて強力に使える
- コントローラ内では `$filter` サービス経由で使用できる

## 4.3 カスタムフィルターの作成

```javascript
angular.module('myApp')
  .filter('フィルター名', function() {
    return function(input, arg1, arg2) {
      // input を加工して返す
      return 加工結果;
    };
  });
```

### 例：文字列を省略するフィルター

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="CustomFilterCtrl">
  <h2>カスタムフィルター</h2>
  
  <!-- truncate フィルター -->
  <p ng-repeat="text in texts">
    {{ text | truncate:20 }}
  </p>
  
  <hr>
  
  <!-- 電話番号フォーマットフィルター -->
  <p>{{ '09012345678' | phoneFormat }}</p>
  <!-- → 090-1234-5678 -->

  <script>
    angular.module('myApp', [])
      // 文字列省略フィルター
      .filter('truncate', function() {
        return function(input, maxLength) {
          if (!input) return '';
          maxLength = maxLength || 30;
          if (input.length <= maxLength) return input;
          return input.substring(0, maxLength) + '...';
        };
      })
      // 電話番号フォーマットフィルター
      .filter('phoneFormat', function() {
        return function(input) {
          if (!input) return '';
          var cleaned = input.replace(/\D/g, '');
          if (cleaned.length === 11) {
            return cleaned.replace(/(\d{3})(\d{4})(\d{4})/, '$1-$2-$3');
          }
          return input;
        };
      })
      .controller('CustomFilterCtrl', function($scope) {
        $scope.texts = [
          'AngularJSは強力なフレームワークです。',
          'これは短い文。',
          'フィルターを使うとテンプレート内でデータを簡単に整形できます。とても便利な機能です。'
        ];
      });
  </script>
</body>
</html>
```

## 4.4 サービスとは

サービス（Service）は、**コントローラ間で共有できるビジネスロジックやデータを格納するオブジェクト**です。

```
┌──────────────┐     ┌──────────────┐
│ ControllerA  │     │ ControllerB  │
│   $scope     │     │   $scope     │
└──────┬───────┘     └──────┬───────┘
       │                     │
       └─────┐   ┌──────────┘
             ▼   ▼
        ┌──────────────┐
        │   Service    │  ← 共有データ・ロジック
        │  (シングルトン)│
        └──────────────┘
```

### 組み込みサービス

| サービス | 用途 |
|----------|------|
| `$http` | HTTP通信（API呼び出し） |
| `$timeout` | 遅延実行（setTimeoutのAngular版） |
| `$interval` | 定期実行（setIntervalのAngular版） |
| `$log` | コンソールログ |
| `$window` | windowオブジェクトへのアクセス |
| `$location` | URLの操作 |
| `$filter` | フィルターの利用 |
| `$q` | Promise の作成 |

---

## 4.5 $http — API通信

`$http` は**サーバーとHTTP通信を行うサービス**です。

### 基本的な使い方

```javascript
// GET リクエスト
$http.get('https://jsonplaceholder.typicode.com/users')
  .then(function(response) {
    // 成功時
    $scope.users = response.data;
  })
  .catch(function(error) {
    // エラー時
    console.error('エラー:', error);
  });

// POST リクエスト
$http.post('https://jsonplaceholder.typicode.com/posts', {
  title: '新しい投稿',
  body: '本文です',
  userId: 1
}).then(function(response) {
  console.log('作成成功:', response.data);
});
```

### 完全な例：ユーザー一覧の取得

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .loading { color: #999; font-style: italic; }
    .error { color: #e74c3c; }
    .card { border: 1px solid #ddd; border-radius: 8px; padding: 12px; margin: 8px 0; }
  </style>
</head>
<body ng-controller="ApiCtrl">
  <h2>ユーザー一覧（JSONPlaceholder API）</h2>
  
  <button ng-click="loadUsers()">データを取得</button>
  
  <p ng-show="loading" class="loading">読み込み中...</p>
  <p ng-show="error" class="error">{{ error }}</p>
  
  <div class="card" ng-repeat="user in users">
    <strong>{{ user.name }}</strong> ({{ user.username }})<br>
    Email: {{ user.email }}<br>
    会社: {{ user.company.name }}
  </div>

  <script>
    angular.module('myApp', [])
      .controller('ApiCtrl', function($scope, $http) {
        $scope.users = [];
        $scope.loading = false;
        $scope.error = null;
        
        $scope.loadUsers = function() {
          $scope.loading = true;
          $scope.error = null;
          
          $http.get('https://jsonplaceholder.typicode.com/users')
            .then(function(response) {
              $scope.users = response.data;
            })
            .catch(function(error) {
              $scope.error = 'データの取得に失敗しました';
            })
            .finally(function() {
              $scope.loading = false;
            });
        };
      });
  </script>
</body>
</html>
```

### よくある間違い

- **`$http` の戻り値はPromise** — `.then()` で結果を受け取る必要があります。直接 `var data = $http.get(...)` のように代入しても動きません
- **レスポンスのデータは `response.data`** — `response` 自体にはヘッダーやステータスコードも含まれます

## 4.6 $timeout と $interval

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body ng-controller="TimerCtrl">
  <h2>タイマーデモ</h2>
  
  <!-- $timeout の例 -->
  <button ng-click="showDelayed()">3秒後にメッセージ表示</button>
  <p ng-show="delayedMessage">{{ delayedMessage }}</p>
  
  <hr>
  
  <!-- $interval の例 -->
  <h3>経過時間: {{ seconds }}秒</h3>
  <button ng-click="startTimer()" ng-disabled="running">開始</button>
  <button ng-click="stopTimer()" ng-disabled="!running">停止</button>

  <script>
    angular.module('myApp', [])
      .controller('TimerCtrl', function($scope, $timeout, $interval) {
        $scope.delayedMessage = '';
        $scope.seconds = 0;
        $scope.running = false;
        var timer = null;
        
        // $timeout: 一度だけ遅延実行
        $scope.showDelayed = function() {
          $scope.delayedMessage = '';
          $timeout(function() {
            $scope.delayedMessage = '3秒経ちました！';
          }, 3000);
        };
        
        // $interval: 繰り返し実行
        $scope.startTimer = function() {
          $scope.running = true;
          timer = $interval(function() {
            $scope.seconds++;
          }, 1000);
        };
        
        $scope.stopTimer = function() {
          if (timer) {
            $interval.cancel(timer);
            $scope.running = false;
          }
        };
        
        // コントローラ破棄時にタイマーをクリーンアップ
        $scope.$on('$destroy', function() {
          if (timer) $interval.cancel(timer);
        });
      });
  </script>
</body>
</html>
```

### ポイントまとめ

- `$timeout` は `setTimeout` の AngularJS 版（`$apply` が自動）
- `$interval` は `setInterval` の AngularJS 版
- `$interval.cancel(timer)` で停止できる
- `$destroy` イベントでクリーンアップを忘れずに

## 4.7 カスタムサービスの作成

### factory / service / provider の比較

| 方式 | 返すもの | 用途 |
|------|----------|------|
| `factory` | 任意のオブジェクトを返す | 最も一般的、柔軟 |
| `service` | `new` でインスタンス化される | クラス的な書き方 |
| `provider` | `$get` メソッドを持つ | 設定可能なサービス |

### factory の例：買い物カートサービス

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .cart-item { display: flex; justify-content: space-between; padding: 8px; border-bottom: 1px solid #eee; }
    .total { font-size: 1.2em; font-weight: bold; margin-top: 10px; }
    button { padding: 6px 12px; margin: 2px; cursor: pointer; }
  </style>
</head>
<body>
  <!-- 商品リスト -->
  <div ng-controller="ShopCtrl">
    <h2>商品</h2>
    <div ng-repeat="product in products">
      {{ product.name }} - ¥{{ product.price }}
      <button ng-click="addToCart(product)">カートに追加</button>
    </div>
  </div>
  
  <!-- カート表示（別のコントローラでもサービスを共有） -->
  <div ng-controller="CartCtrl">
    <h2>カート ({{ cartItemCount }}点)</h2>
    <div class="cart-item" ng-repeat="item in cartItems">
      {{ item.name }} x {{ item.qty }} = ¥{{ item.price * item.qty }}
      <button ng-click="removeFromCart(item)">削除</button>
    </div>
    <p class="total">合計: ¥{{ cartTotal }}</p>
  </div>

  <script>
    angular.module('myApp', [])
      // カートサービス（factory）
      .factory('CartService', function() {
        var items = [];
        
        return {
          getItems: function() { return items; },
          
          addItem: function(product) {
            var existing = items.find(function(i) { return i.name === product.name; });
            if (existing) {
              existing.qty++;
            } else {
              items.push({ name: product.name, price: product.price, qty: 1 });
            }
          },
          
          removeItem: function(item) {
            var index = items.indexOf(item);
            if (index > -1) items.splice(index, 1);
          },
          
          getTotal: function() {
            return items.reduce(function(sum, item) {
              return sum + item.price * item.qty;
            }, 0);
          },
          
          getCount: function() {
            return items.reduce(function(sum, item) { return sum + item.qty; }, 0);
          }
        };
      })
      // 商品コントローラ
      .controller('ShopCtrl', function($scope, CartService) {
        $scope.products = [
          { name: 'コーヒー', price: 300 },
          { name: 'サンドイッチ', price: 450 },
          { name: 'ケーキ', price: 500 }
        ];
        $scope.addToCart = function(product) {
          CartService.addItem(product);
        };
      })
      // カートコントローラ
      .controller('CartCtrl', function($scope, CartService) {
        // $watch でカートの変更を監視
        $scope.$watch(function() {
          return CartService.getItems().length + CartService.getTotal();
        }, function() {
          $scope.cartItems = CartService.getItems();
          $scope.cartTotal = CartService.getTotal();
          $scope.cartItemCount = CartService.getCount();
        });
        
        $scope.removeFromCart = function(item) {
          CartService.removeItem(item);
        };
      });
  </script>
</body>
</html>
```

## 4.8 依存性注入（DI）

依存性注入（Dependency Injection, DI）とは、**必要なサービスを外部から自動的に渡してもらう仕組み**です。

```
┌───────────────────────┐
│    AngularJS の DI     │
│    コンテナ            │
│                       │
│  $http ─┐             │
│  $scope ─┤   注入      │
│  CartService ─┘ ──→ Controller │
└───────────────────────┘
```

### minification（圧縮）対策

JavaScriptを圧縮すると引数名が変わるため、配列記法で明示的に依存関係を指定します。

```javascript
// NG: 圧縮すると壊れる
app.controller('MyCtrl', function($scope, $http) { ... });

// OK: 配列記法（推奨）
app.controller('MyCtrl', ['$scope', '$http', function($scope, $http) {
  // 文字列は圧縮されないため安全
}]);

// OK: $inject プロパティ
function MyCtrl($scope, $http) { ... }
MyCtrl.$inject = ['$scope', '$http'];
app.controller('MyCtrl', MyCtrl);
```

> **よくある間違い：** 本番環境でJavaScriptを圧縮（minify）したら動かなくなった → 配列記法を使っていないことが原因です。**常に配列記法を使う**ことを推奨します。

### ポイントまとめ

- サービスはコントローラ間でデータとロジックを共有するシングルトン
- `factory` が最も一般的で柔軟な作成方法
- 依存性注入により、テスト可能で疎結合なコードが書ける
- **圧縮対策として配列記法を常に使う**

## 4.9 まとめ

| トピック | ポイント |
|----------|----------|
| 組み込みフィルター | `currency`, `date`, `orderBy`, `filter` が特に便利 |
| カスタムフィルター | `.filter()` で独自のデータ整形ロジックを定義 |
| $http | API通信の基本、`.then()` でPromise処理 |
| $timeout / $interval | setTimeout/setInterval のAngular版、`$apply` 不要 |
| factory | カスタムサービス作成の推奨パターン |
| DI | 配列記法で圧縮安全な依存性注入を行う |
