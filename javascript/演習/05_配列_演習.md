## 基本問題

### 問題1：配列の基本操作

以下の配列に対して、指定された操作を順番に行い、各ステップの結果を表示してください。

```javascript
const fruits = ["りんご", "みかん", "バナナ"];
```

1. 末尾に `"ぶどう"` を追加
2. 先頭に `"メロン"` を追加
3. 末尾から1つ削除
4. 先頭から1つ削除
5. 最終的な配列を表示

**期待される出力：**
```
1. 追加後: ["りんご", "みかん", "バナナ", "ぶどう"]
2. 追加後: ["メロン", "りんご", "みかん", "バナナ", "ぶどう"]
3. 削除: ぶどう → ["メロン", "りんご", "みかん", "バナナ"]
4. 削除: メロン → ["りんご", "みかん", "バナナ"]
5. 最終結果: ["りんご", "みかん", "バナナ"]
```

<details>
<summary>解答例</summary>

```javascript
const fruits = ["りんご", "みかん", "バナナ"];

// 1. 末尾に追加
fruits.push("ぶどう");
console.log(`1. 追加後: [${fruits.map(f => `"${f}"`).join(", ")}]`);

// 2. 先頭に追加
fruits.unshift("メロン");
console.log(`2. 追加後: [${fruits.map(f => `"${f}"`).join(", ")}]`);

// 3. 末尾から削除
const removed1 = fruits.pop();
console.log(`3. 削除: ${removed1} → [${fruits.map(f => `"${f}"`).join(", ")}]`);

// 4. 先頭から削除
const removed2 = fruits.shift();
console.log(`4. 削除: ${removed2} → [${fruits.map(f => `"${f}"`).join(", ")}]`);

// 5. 最終結果
console.log(`5. 最終結果: [${fruits.map(f => `"${f}"`).join(", ")}]`);
```

</details>

### 問題2：map で配列を変換

以下の配列を `map` を使って変換してください。

```javascript
const prices = [100, 250, 480, 1200, 3500];
```

1. 全商品に消費税10%を加算した配列を作成
2. 各価格を `"¥1,100"` のような表示形式に変換

**期待される出力：**
```
税込価格: [110, 275, 528, 1320, 3850]
表示形式: ["¥110", "¥275", "¥528", "¥1,320", "¥3,850"]
```

<details>
<summary>解答例</summary>

```javascript
const prices = [100, 250, 480, 1200, 3500];

// 1. 消費税10%を加算
const withTax = prices.map(price => Math.floor(price * 1.1));
console.log(`税込価格: [${withTax.join(", ")}]`);

// 2. 表示形式に変換（toLocaleString でカンマ区切り）
const formatted = withTax.map(price => `"¥${price.toLocaleString()}"`);
console.log(`表示形式: [${formatted.join(", ")}]`);
```

</details>

### 問題3：filter で条件に合う要素を抽出

以下の生徒データから、条件に合うデータを `filter` で抽出してください。

```javascript
const students = [
  { name: "太郎", score: 85, grade: "A" },
  { name: "花子", score: 72, grade: "B" },
  { name: "次郎", score: 90, grade: "A" },
  { name: "美咲", score: 65, grade: "C" },
  { name: "健太", score: 78, grade: "B" },
  { name: "さくら", score: 95, grade: "A" }
];
```

1. スコアが80以上の生徒
2. グレードが "A" の生徒の名前一覧

**期待される出力：**
```
80点以上: 太郎(85), 次郎(90), さくら(95)
Aグレード: 太郎, 次郎, さくら
```

<details>
<summary>解答例</summary>

```javascript
const students = [
  { name: "太郎", score: 85, grade: "A" },
  { name: "花子", score: 72, grade: "B" },
  { name: "次郎", score: 90, grade: "A" },
  { name: "美咲", score: 65, grade: "C" },
  { name: "健太", score: 78, grade: "B" },
  { name: "さくら", score: 95, grade: "A" }
];

// 1. スコア80以上をフィルタ
const highScorers = students.filter(s => s.score >= 80);
console.log(`80点以上: ${highScorers.map(s => `${s.name}(${s.score})`).join(", ")}`);

// 2. Aグレードの名前一覧（filter + map のチェーン）
const gradeA = students.filter(s => s.grade === "A").map(s => s.name);
console.log(`Aグレード: ${gradeA.join(", ")}`);
```

</details>

### 問題4：reduce で集計しよう

以下の買い物リストから `reduce` を使って合計金額を計算してください。

```javascript
const cart = [
  { name: "りんご", price: 150, quantity: 3 },
  { name: "牛乳", price: 200, quantity: 1 },
  { name: "パン", price: 300, quantity: 2 },
  { name: "卵", price: 250, quantity: 1 }
];
```

**期待される出力：**
```
りんご: 150円 × 3 = 450円
牛乳: 200円 × 1 = 200円
パン: 300円 × 2 = 600円
卵: 250円 × 1 = 250円
---
合計: 1500円
```

<details>
<summary>解答例</summary>

```javascript
const cart = [
  { name: "りんご", price: 150, quantity: 3 },
  { name: "牛乳", price: 200, quantity: 1 },
  { name: "パン", price: 300, quantity: 2 },
  { name: "卵", price: 250, quantity: 1 }
];

// 各商品の小計を表示
cart.forEach(item => {
  const subtotal = item.price * item.quantity;
  console.log(`${item.name}: ${item.price}円 × ${item.quantity} = ${subtotal}円`);
});

// reduce で合計を計算
const total = cart.reduce((sum, item) => sum + item.price * item.quantity, 0);
console.log("---");
console.log(`合計: ${total}円`);
```

</details>

### 問題5：メソッドチェーンで変換

以下の商品データから、メソッドチェーン（filter → sort → map）を使って処理してください。

```javascript
const products = [
  { name: "ノートPC", price: 89000, category: "電子機器" },
  { name: "マウス", price: 3500, category: "電子機器" },
  { name: "デスク", price: 25000, category: "家具" },
  { name: "キーボード", price: 8000, category: "電子機器" },
  { name: "チェア", price: 45000, category: "家具" },
  { name: "モニター", price: 35000, category: "電子機器" }
];
```

カテゴリが「電子機器」の商品を、価格の安い順に並べて、名前と価格の文字列に変換してください。

**期待される出力：**
```
電子機器（価格順）:
マウス: ¥3,500
キーボード: ¥8,000
モニター: ¥35,000
ノートPC: ¥89,000
```

<details>
<summary>解答例</summary>

```javascript
const products = [
  { name: "ノートPC", price: 89000, category: "電子機器" },
  { name: "マウス", price: 3500, category: "電子機器" },
  { name: "デスク", price: 25000, category: "家具" },
  { name: "キーボード", price: 8000, category: "電子機器" },
  { name: "チェア", price: 45000, category: "家具" },
  { name: "モニター", price: 35000, category: "電子機器" }
];

// メソッドチェーンで filter → sort → map
const result = products
  .filter(p => p.category === "電子機器")
  .sort((a, b) => a.price - b.price)
  .map(p => `${p.name}: ¥${p.price.toLocaleString()}`);

console.log("電子機器（価格順）:");
result.forEach(item => console.log(item));
```

</details>

### 問題6：配列の重複除去とグルーピング

以下のタグ配列から重複を除去し、さらに文字数でグルーピングしてください。

```javascript
const tags = ["JS", "CSS", "HTML", "JS", "React", "Vue", "CSS", "Node", "HTML", "Deno"];
```

**期待される出力：**
```
重複除去: ["JS", "CSS", "HTML", "React", "Vue", "Node", "Deno"]
--- 文字数別グルーピング ---
2文字: JS
3文字: CSS, Vue
4文字: HTML, Node, Deno
5文字: React
```

<details>
<summary>解答例</summary>

```javascript
const tags = ["JS", "CSS", "HTML", "JS", "React", "Vue", "CSS", "Node", "HTML", "Deno"];

// 重複除去（Set を使用）
const unique = [...new Set(tags)];
console.log(`重複除去: [${unique.map(t => `"${t}"`).join(", ")}]`);

// 文字数でグルーピング（reduce を使用）
const grouped = unique.reduce((acc, tag) => {
  const len = tag.length;
  if (!acc[len]) acc[len] = [];
  acc[len].push(tag);
  return acc;
}, {});

console.log("--- 文字数別グルーピング ---");
for (const [len, items] of Object.entries(grouped)) {
  console.log(`${len}文字: ${items.join(", ")}`);
}
```

</details>

### 問題7：配列の分割代入とスプレッド構文

以下の操作を分割代入とスプレッド構文を使って行ってください。

```javascript
const scores = [95, 87, 72, 63, 88, 91, 76];
```

1. 最初の要素を `top` に、残りを `rest` に分割代入
2. スプレッド構文を使って `scores` の先頭に `100` を追加した新しい配列を作成
3. 2つの配列 `[1, 2, 3]` と `[4, 5, 6]` をスプレッド構文で結合
4. `scores` の非破壊ソート（元の配列を変更せずにソート）

**期待される出力：**
```
top: 95
rest: [87, 72, 63, 88, 91, 76]
先頭追加: [100, 95, 87, 72, 63, 88, 91, 76]
結合: [1, 2, 3, 4, 5, 6]
ソート済み: [63, 72, 76, 87, 88, 91, 95]
元の配列: [95, 87, 72, 63, 88, 91, 76]
```

<details>
<summary>解答例</summary>

```javascript
const scores = [95, 87, 72, 63, 88, 91, 76];

// 1. 分割代入
const [top, ...rest] = scores;
console.log(`top: ${top}`);
console.log(`rest: [${rest.join(", ")}]`);

// 2. 先頭に要素を追加（非破壊）
const withNew = [100, ...scores];
console.log(`先頭追加: [${withNew.join(", ")}]`);

// 3. 配列の結合
const combined = [...[1, 2, 3], ...[4, 5, 6]];
console.log(`結合: [${combined.join(", ")}]`);

// 4. 非破壊ソート（スプレッドでコピーしてからソート）
const sorted = [...scores].sort((a, b) => a - b);
console.log(`ソート済み: [${sorted.join(", ")}]`);
console.log(`元の配列: [${scores.join(", ")}]`);
```

</details>

### 問題8：成績処理システム

以下のデータを使って、各種統計を計算する処理を配列メソッドで実装してください。

```javascript
const results = [
  { name: "太郎", math: 85, english: 72, science: 90 },
  { name: "花子", math: 92, english: 88, science: 76 },
  { name: "次郎", math: 68, english: 95, science: 82 },
  { name: "美咲", math: 78, english: 80, science: 95 },
  { name: "健太", math: 95, english: 65, science: 70 }
];
```

1. 各生徒の平均点を追加した新しい配列を作成
2. 平均点の高い順にソート
3. 平均点が80以上の生徒だけ抽出
4. 全員の数学の平均点

**期待される出力：**
```
--- 成績一覧（平均点順）---
花子: 数学92, 英語88, 理科76 → 平均85.3
美咲: 数学78, 英語80, 理科95 → 平均84.3
太郎: 数学85, 英語72, 理科90 → 平均82.3
次郎: 数学68, 英語95, 理科82 → 平均81.7
健太: 数学95, 英語65, 理科70 → 平均76.7
--- 平均80以上 ---
花子, 美咲, 太郎, 次郎
--- 数学の全体平均 ---
83.6点
```

<details>
<summary>解答例</summary>

```javascript
const results = [
  { name: "太郎", math: 85, english: 72, science: 90 },
  { name: "花子", math: 92, english: 88, science: 76 },
  { name: "次郎", math: 68, english: 95, science: 82 },
  { name: "美咲", math: 78, english: 80, science: 95 },
  { name: "健太", math: 95, english: 65, science: 70 }
];

// 1. 平均点を追加
const withAvg = results.map(s => ({
  ...s,
  average: Math.round((s.math + s.english + s.science) / 3 * 10) / 10
}));

// 2. 平均点の高い順にソート
const sorted = [...withAvg].sort((a, b) => b.average - a.average);

// 成績一覧を表示
console.log("--- 成績一覧（平均点順）---");
sorted.forEach(s => {
  console.log(`${s.name}: 数学${s.math}, 英語${s.english}, 理科${s.science} → 平均${s.average}`);
});

// 3. 平均80以上の生徒
const highAvg = sorted.filter(s => s.average >= 80);
console.log("--- 平均80以上 ---");
console.log(highAvg.map(s => s.name).join(", "));

// 4. 数学の全体平均
const mathAvg = results.reduce((sum, s) => sum + s.math, 0) / results.length;
console.log("--- 数学の全体平均 ---");
console.log(`${mathAvg}点`);
```

</details>
