# 実践課題07：ユーザー登録フォーム ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第3章（ディレクティブ）、第4章（フィルターとサービス）、第6章（フォームとバリデーション）
> **課題の種類**: ミニプロジェクト
> **学習目標**: AngularJSのフォーム管理機能（`$pristine`、`$dirty`、`$valid`、`$error`）を使ったリアルタイムバリデーション付き登録フォームを構築する

---

## 完成イメージ

各フィールドにバリデーションが設定され、エラーメッセージがリアルタイムに表示されます。すべてのバリデーションを通過するまで送信ボタンは無効です。

## 課題の要件

1. ユーザー名（3〜20文字、英数字とアンダースコアのみ）
2. メールアドレス（`type="email"` + `required`）
3. パスワード（8文字以上、パスワード強度メーターを表示）
4. パスワード確認（カスタムバリデータで一致チェック）
5. 利用規約チェックボックス
6. 各フィールドに適切なエラーメッセージをリアルタイム表示する
7. フォーム状態に応じてCSSクラスを切り替える（`ng-valid` / `ng-invalid` など）
8. すべてのバリデーション通過時のみ送信ボタンを有効にする
9. 送信時にフォームデータをまとめて表示する

          vm.submittedData = angular.copy(vm.user);
        }
      };
    });
  </script>

</body>
</html>

- エラーメッセージにアイコン（`&#10007;` / `&#10003;`）で視覚的なフィードバック
- 文字数カウントをリアルタイム表示
