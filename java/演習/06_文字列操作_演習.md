## 基本問題

### 問題1：文字列の長さと文字カウント

ユーザーから文字列を受け取り、以下の情報を表示するプログラムを作成してください。
- 文字列の長さ
- 大文字の数
- 小文字の数
- 数字の数
- その他の文字の数

**期待される出力例：**
```
文字列を入力してください: Hello World 123
=== 文字解析結果 ===
文字列: Hello World 123
長さ: 15
大文字: 2
小文字: 8
数字: 3
その他: 2
```

<details><summary>ヒント</summary>

- `String`の`length()`メソッドで文字列の長さを取得できます。
- `charAt(i)`で各文字を取り出せます。
- `Character.isUpperCase()`, `Character.isLowerCase()`, `Character.isDigit()`で文字の種類を判定できます。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class CharCounter {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("文字列を入力してください: ");
        String input = scanner.nextLine();

        // 各カテゴリのカウンターを初期化
        int upperCount = 0;
        int lowerCount = 0;
        int digitCount = 0;
        int otherCount = 0;

        // 1文字ずつ調べてカウント
        for (int i = 0; i < input.length(); i++) {
            char c = input.charAt(i);
            if (Character.isUpperCase(c)) {
                upperCount++;
            } else if (Character.isLowerCase(c)) {
                lowerCount++;
            } else if (Character.isDigit(c)) {
                digitCount++;
            } else {
                otherCount++;
            }
        }

        // 結果を表示
        System.out.println("=== 文字解析結果 ===");
        System.out.println("文字列: " + input);
        System.out.println("長さ: " + input.length());
        System.out.println("大文字: " + upperCount);
        System.out.println("小文字: " + lowerCount);
        System.out.println("数字: " + digitCount);
        System.out.println("その他: " + otherCount);

        scanner.close();
    }
}
```

</details>

### 問題2：母音カウンター

文字列を受け取り、母音（a, e, i, o, u）の出現回数をそれぞれカウントして表示するプログラムを作成してください。大文字・小文字の区別なくカウントしてください。

**期待される出力例：**
```
文字列を入力してください: Hello Java Programming
=== 母音カウント結果 ===
a: 3
e: 1
i: 2
o: 2
u: 0
母音の合計: 8
```

<details><summary>ヒント</summary>

- `String`の`toLowerCase()`メソッドで小文字に変換してから処理すると簡単です。
- 各母音ごとにカウンターを用意しましょう。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class VowelCounter {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("文字列を入力してください: ");
        String input = scanner.nextLine();

        // 小文字に統一して比較しやすくする
        String lower = input.toLowerCase();

        // 各母音のカウンター
        int countA = 0, countE = 0, countI = 0, countO = 0, countU = 0;

        for (int i = 0; i < lower.length(); i++) {
            char c = lower.charAt(i);
            // switch文で各母音を判定
            switch (c) {
                case 'a': countA++; break;
                case 'e': countE++; break;
                case 'i': countI++; break;
                case 'o': countO++; break;
                case 'u': countU++; break;
            }
        }

        int total = countA + countE + countI + countO + countU;

        System.out.println("=== 母音カウント結果 ===");
        System.out.println("a: " + countA);
        System.out.println("e: " + countE);
        System.out.println("i: " + countI);
        System.out.println("o: " + countO);
        System.out.println("u: " + countU);
        System.out.println("母音の合計: " + total);

        scanner.close();
    }
}
```

### 問題3：文字列の反転

文字列を受け取り、反転させた文字列を表示するプログラムを作成してください。`StringBuilder`の`reverse()`メソッドを使わず、自力で反転のロジックを実装してください。

**期待される出力例：**
```
文字列を入力してください: Hello Java
元の文字列: Hello Java
反転した文字列: avaJ olleH
```

<details><summary>ヒント</summary>

- 文字列の末尾から先頭に向かってループし、1文字ずつ新しい文字列に追加する方法があります。
- `String`に直接追加する代わりに`StringBuilder`の`append()`を使うとパフォーマンスが良いです。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class StringReverser {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("文字列を入力してください: ");
        String input = scanner.nextLine();

        // StringBuilderを使って効率よく文字列を構築
        StringBuilder reversed = new StringBuilder();

        // 末尾から先頭へ1文字ずつ追加
        for (int i = input.length() - 1; i >= 0; i--) {
            reversed.append(input.charAt(i));
        }

        System.out.println("元の文字列: " + input);
        System.out.println("反転した文字列: " + reversed.toString());

        scanner.close();
    }
}
```

</details>

### 問題4：回文判定プログラム

文字列を受け取り、それが回文（前から読んでも後ろから読んでも同じ）かどうかを判定するプログラムを作成してください。判定時にはスペースを無視し、大文字・小文字を区別しないようにしてください。

**期待される出力例：**
```
文字列を入力してください: A Santa at NASA
=== 回文判定結果 ===
元の文字列: A Santa at NASA
正規化後: asantaatnasa
結果: 回文です！
```

```
文字列を入力してください: Hello World
=== 回文判定結果 ===
元の文字列: Hello World
正規化後: helloworld
結果: 回文ではありません。
```

<details><summary>ヒント</summary>

- まず`toLowerCase()`で小文字に変換し、`replace(" ", "")`でスペースを除去して「正規化」します。
- 正規化した文字列を反転させて、元の正規化文字列と比較します。
- `StringBuilder`の`reverse()`メソッドを使うと便利です。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class PalindromeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("文字列を入力してください: ");
        String input = scanner.nextLine();

        // 正規化: 小文字変換＋スペース除去
        String normalized = input.toLowerCase().replace(" ", "");

        // StringBuilderで反転文字列を作成
        String reversed = new StringBuilder(normalized).reverse().toString();

        // 回文かどうか判定（equals()で文字列を比較）
        boolean isPalindrome = normalized.equals(reversed);

        System.out.println("=== 回文判定結果 ===");
        System.out.println("元の文字列: " + input);
        System.out.println("正規化後: " + normalized);
        System.out.println("結果: " + (isPalindrome ? "回文です！" : "回文ではありません。"));

        scanner.close();
    }
}
```

</details>

### 問題5：単語カウンター

文を受け取り、以下の情報を表示するプログラムを作成してください。
- 単語数
- 各単語とその文字数
- 最も長い単語

単語はスペースで区切られているものとします。

**期待される出力例：**
```
文を入力してください: Java programming is fun and powerful
=== 単語分析結果 ===
単語数: 6
--- 各単語の文字数 ---
  Java: 4文字
  programming: 11文字
  is: 2文字
  fun: 3文字
  and: 3文字
  powerful: 8文字
最も長い単語: programming (11文字)
```

<details><summary>ヒント</summary>

- `String`の`split(" ")`メソッドで文字列をスペースで分割し、配列にできます。
- 配列の`length`で単語数が取得できます。
- 最長の単語を見つけるには、ループで各単語の長さを比較しましょう。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class WordCounter {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("文を入力してください: ");
        String input = scanner.nextLine();

        // スペースで分割して単語配列を取得
        // "\\s+"は1つ以上の空白にマッチする正規表現
        String[] words = input.trim().split("\\s+");

        System.out.println("=== 単語分析結果 ===");
        System.out.println("単語数: " + words.length);
        System.out.println("--- 各単語の文字数 ---");

        // 最長単語を追跡するための変数
        String longestWord = "";

        for (String word : words) {
            System.out.println("  " + word + ": " + word.length() + "文字");
            // 現在の単語が最長単語より長ければ更新
            if (word.length() > longestWord.length()) {
                longestWord = word;
            }
        }

        System.out.println("最も長い単語: " + longestWord + " (" + longestWord.length() + "文字)");

        scanner.close();
    }
}
```

</details>

### 問題6：シーザー暗号

文字列とシフト量（整数）を受け取り、シーザー暗号で暗号化・復号するプログラムを作成してください。アルファベットのみシフトし、大文字・小文字を維持してください。その他の文字はそのままにしてください。

**期待される出力例：**
```
文字列を入力してください: Hello World
シフト量を入力してください: 3
=== シーザー暗号 ===
元の文字列: Hello World
暗号化: Khoor Zruog
復号: Hello World
```

<details><summary>ヒント</summary>

- アルファベットの文字コードを利用します。`'A'`は65、`'Z'`は90、`'a'`は97、`'z'`は122です。
- シフト後にアルファベットの範囲を超えた場合は、26で割った余りを使って先頭に戻します。
- 復号は暗号化の逆操作（シフト量を負にする）です。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class CaesarCipher {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("文字列を入力してください: ");
        String input = scanner.nextLine();
        System.out.print("シフト量を入力してください: ");
        int shift = scanner.nextInt();

        // 暗号化
        String encrypted = caesarShift(input, shift);
        // 復号（逆方向にシフト）
        String decrypted = caesarShift(encrypted, -shift);

        System.out.println("=== シーザー暗号 ===");
        System.out.println("元の文字列: " + input);
        System.out.println("暗号化: " + encrypted);
        System.out.println("復号: " + decrypted);

        scanner.close();
    }

    /**
     * シーザー暗号のシフト処理
     * アルファベットのみシフトし、その他の文字はそのまま返す
     */
    static String caesarShift(String text, int shift) {
        StringBuilder result = new StringBuilder();

        for (int i = 0; i < text.length(); i++) {
            char c = text.charAt(i);

            if (Character.isUpperCase(c)) {
                // 大文字: 'A'を基準にシフト計算
                // +26してから%26することで負のシフトにも対応
                char shifted = (char) ('A' + (c - 'A' + shift % 26 + 26) % 26);
                result.append(shifted);
            } else if (Character.isLowerCase(c)) {
                // 小文字: 'a'を基準にシフト計算
                char shifted = (char) ('a' + (c - 'a' + shift % 26 + 26) % 26);
                result.append(shifted);
            } else {
                // アルファベット以外はそのまま
                result.append(c);
            }
        }

        return result.toString();
    }
}
```

### 問題7：メールアドレス形式バリデーター

メールアドレスの文字列を受け取り、以下のルールに基づいて形式が正しいかどうかを検証するプログラムを作成してください（正規表現を使わずに実装してください）。

**検証ルール：**
1. `@`がちょうど1つ含まれること
2. `@`の前（ローカル部）が1文字以上であること
3. `@`の後（ドメイン部）に`.`が少なくとも1つ含まれること
4. ドメイン部の最後の`.`の後に2文字以上あること
5. 先頭と末尾がスペースでないこと

**期待される出力例：**
```
メールアドレスを入力してください: user@example.com
=== メールアドレス検証結果 ===
入力: user@example.com
@の数: 1 -> OK
ローカル部: user -> OK
ドメイン部にドットあり: example.com -> OK
末尾ドメイン長: com (3文字) -> OK
結果: 有効なメールアドレスです。
```

```
メールアドレスを入力してください: invalid@com
=== メールアドレス検証結果 ===
入力: invalid@com
@の数: 1 -> OK
ローカル部: invalid -> OK
ドメイン部にドットあり: com -> NG
結果: 無効なメールアドレスです。
```

<details><summary>ヒント</summary>

- `indexOf('@')`と`lastIndexOf('@')`を比較して`@`が1つだけか確認できます。
- `substring()`でローカル部とドメイン部を切り出しましょう。
- ドメイン部に対して`lastIndexOf('.')`を使い、`.`の後の長さを確認します。

</details>

<details><summary>解答例</summary>

```java
import java.util.Scanner;

public class EmailValidator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("メールアドレスを入力してください: ");
        String email = scanner.nextLine().trim();

        System.out.println("=== メールアドレス検証結果 ===");
        System.out.println("入力: " + email);

        boolean isValid = true;

        // ルール1: @がちょうど1つ
        int atIndex = email.indexOf('@');
        int lastAtIndex = email.lastIndexOf('@');
        long atCount = email.chars().filter(c -> c == '@').count();

        if (atIndex == -1 || atIndex != lastAtIndex) {
            System.out.println("@の数: " + atCount + " -> NG");
            isValid = false;
        } else {
            System.out.println("@の数: " + atCount + " -> OK");

            // @の前後を分割
            String localPart = email.substring(0, atIndex);
            String domainPart = email.substring(atIndex + 1);

            // ルール2: ローカル部が1文字以上
            if (localPart.isEmpty()) {
                System.out.println("ローカル部: (空) -> NG");
                isValid = false;
            } else {
                System.out.println("ローカル部: " + localPart + " -> OK");
            }

            // ルール3: ドメイン部に.が含まれること
            int dotIndex = domainPart.lastIndexOf('.');
            if (dotIndex == -1) {
                System.out.println("ドメイン部にドットあり: " + domainPart + " -> NG");
                isValid = false;
            } else {
                System.out.println("ドメイン部にドットあり: " + domainPart + " -> OK");

                // ルール4: 最後の.の後に2文字以上
                String tld = domainPart.substring(dotIndex + 1);
                if (tld.length() < 2) {
                    System.out.println("末尾ドメイン長: " + tld + " (" + tld.length() + "文字) -> NG");
                    isValid = false;
                } else {
                    System.out.println("末尾ドメイン長: " + tld + " (" + tld.length() + "文字) -> OK");
                }
            }
        }

        // 最終結果
        System.out.println("結果: " + (isValid ? "有効なメールアドレスです。" : "無効なメールアドレスです。"));

        scanner.close();
    }
}
```

### 問題8：CSV行パーサー

CSV形式の1行データを受け取り、各フィールドを解析して表示するプログラムを作成してください。以下の仕様を満たしてください。

**仕様：**
- カンマ（`,`）で区切られた各フィールドを取り出す
- ダブルクォート（`"`）で囲まれたフィールド内のカンマはフィールド区切りとして扱わない
- 各フィールドの前後の空白はトリムする

**期待される出力例：**
```
CSV行を入力してください: John,25,"New York, USA",Engineer
=== CSV解析結果 ===
フィールド数: 4
[0]: John
[1]: 25
[2]: New York, USA
[3]: Engineer
```

```
CSV行を入力してください: "Smith, Bob",30,"Tokyo, Japan","Software Developer"
=== CSV解析結果 ===
フィールド数: 4
[0]: Smith, Bob
[1]: 30
[2]: Tokyo, Japan
[3]: Software Developer
```

<details><summary>ヒント</summary>

- 1文字ずつ走査し、ダブルクォートの内側かどうかを`boolean`フラグで管理します。
- ダブルクォートの外でカンマに出会ったらフィールドの区切りです。
- `ArrayList<String>`を使ってフィールドを蓄積しましょう。
- 各フィールドの前後のダブルクォートとスペースを`trim()`で除去します。

</details>

<details><summary>解答例</summary>

```java
import java.util.ArrayList;
import java.util.Scanner;

public class CsvParser {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("CSV行を入力してください: ");
        String line = scanner.nextLine();

        // CSV行を解析してフィールドのリストを取得
        ArrayList<String> fields = parseCsvLine(line);

        System.out.println("=== CSV解析結果 ===");
        System.out.println("フィールド数: " + fields.size());

        for (int i = 0; i < fields.size(); i++) {
            System.out.println("[" + i + "]: " + fields.get(i));
        }

        scanner.close();
    }

    /**
     * CSV行を解析してフィールドのリストを返す
     * ダブルクォート内のカンマは区切りとして扱わない
     */
    static ArrayList<String> parseCsvLine(String line) {
        ArrayList<String> fields = new ArrayList<>();
        StringBuilder currentField = new StringBuilder();
        boolean inQuotes = false; // ダブルクォートの内側かどうか

        for (int i = 0; i < line.length(); i++) {
            char c = line.charAt(i);

            if (c == '"') {
                // ダブルクォートのON/OFFを切り替え
                inQuotes = !inQuotes;
            } else if (c == ',' && !inQuotes) {
                // クォートの外のカンマ = フィールド区切り
                fields.add(currentField.toString().trim());
                currentField = new StringBuilder();
            } else {
                // 通常の文字はフィールドに追加
                currentField.append(c);
            }
        }

        // 最後のフィールドを追加（末尾にカンマがないため）
        fields.add(currentField.toString().trim());

        return fields;
    }
}
```

**お疲れさまでした！** 文字列操作はプログラミングの基礎中の基礎です。`String`クラスのメソッドを使いこなせるようになると、データ処理の幅が大きく広がります。
