## 課題の要件

1. トランプのカード（スート4種 x 数値13種 = 52枚）をデータで表現する
2. デッキのシャッフル機能を実装する（簡易的な擬似乱数で可）
3. プレイヤーとディーラーの手札を管理する
4. Aは1点または11点として計算する（合計が21を超えない最大値）
5. ブラックジャックのルール：
   - プレイヤーは「ヒット（もう1枚引く）」か「スタンド（止める）」を選ぶ
   - 21を超えたらバスト（即負け）
   - ディーラーは合計17以上になるまで引き続ける
   - 最初の2枚で21ならブラックジャック（即勝ち）
6. 勝敗判定と戦績の記録を行う

<summary>ステップ2：簡易的な擬似乱数を実装する</summary>

外部クレートを使わずにシャッフルするため、線形合同法（linear congruential generator）による簡易乱数を使います。

```rust
struct SimpleRng {
    state: u64,
}

impl SimpleRng {
    fn new(seed: u64) -> SimpleRng {
        SimpleRng { state: seed }
    }

    fn next(&mut self) -> u64 {
        // 線形合同法
        self.state = self.state.wrapping_mul(6364136223846793005).wrapping_add(1);
        self.state >> 33
    }

    /// ベクタをシャッフルする（Fisher-Yatesアルゴリズム）
    fn shuffle<T>(&mut self, vec: &mut Vec<T>) {
        let len = vec.len();
        for i in (1..len).rev() {
            let j = (self.next() as usize) % (i + 1);
            vec.swap(i, j);
        }
    }
}
```

</details>

<details>
<summary>ステップ3：手札の合計を計算する（Aの扱い）</summary>

Aは11として計算し、合計が21を超えたら1に切り替えます。

```rust
fn hand_value(hand: &[Card]) -> u32 {
    let mut total: u32 = 0;
    let mut aces: u32 = 0;

    for card in hand {
        let value = match card.rank {
            1 => { aces += 1; 11 }
            11 | 12 | 13 => 10,
            n => n as u32,
        };
        total += value;
    }

    // Aを11→1に変換してバストを回避
    while total > 21 && aces > 0 {
        total -= 10;
        aces -= 1;
    }
    total
}
```

```rust
// --- カード ---
#[derive(Clone, Copy)]
enum Suit {
    Heart,
    Diamond,
    Club,
    Spade,
}

impl Suit {
    fn symbol(&self) -> &'static str {
        match self {
            Suit::Heart => "ハート",
            Suit::Diamond => "ダイヤ",
            Suit::Club => "クラブ",
            Suit::Spade => "スペード",
        }
    }

#[derive(Clone, Copy)]
struct Card {
    suit: Suit,
    rank: u8,
}

impl Card {
    fn display(&self) -> String {
        let rank_str = match self.rank {
            1 => "A".to_string(),
            11 => "J".to_string(),
            12 => "Q".to_string(),
            13 => "K".to_string(),
            n => n.to_string(),
        };
        format!("[{}{}]", self.suit.symbol(), rank_str)
    }
}

// --- デッキ ---
fn create_deck() -> Vec<Card> {
    let suits = [Suit::Heart, Suit::Diamond, Suit::Club, Suit::Spade];
    let mut deck = Vec::new();
    for &suit in &suits {
        for rank in 1..=13 {
            deck.push(Card { suit, rank });
        }
    }
    deck
}

// --- 手札の合計 ---
fn hand_value(hand: &[Card]) -> u32 {
    let mut total: u32 = 0;
    let mut aces: u32 = 0;

    for card in hand {
        let value = match card.rank {
            1 => {
                aces += 1;
                11
            }
            11 | 12 | 13 => 10,
            n => n as u32,
        };
        total += value;
    }

    while total > 21 && aces > 0 {
        total -= 10;
        aces -= 1;
    }
    total
}

// --- 手札の表示 ---
fn show_hand(hand: &[Card]) -> String {
    hand.iter().map(|c| c.display()).collect::<Vec<_>>().join(" ")
}

fn read_input(prompt: &str) -> String {
    print!("{}", prompt);
    io::stdout().flush().unwrap();
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

fn main() {
    println!("===== ブラックジャック =====");

    // シードとして現在時刻の代わりにカウンタを使用
    let mut seed: u64 = 12345;
    let mut wins: u32 = 0;
    let mut losses: u32 = 0;
    let mut draws: u32 = 0;

    loop {
        seed += 67890;
        let mut rng = SimpleRng::new(seed);
        let mut deck = create_deck();
        rng.shuffle(&mut deck);

        // 初期配布
        let mut player_hand: Vec<Card> = Vec::new();
        let mut dealer_hand: Vec<Card> = Vec::new();

        player_hand.push(deck.pop().unwrap());
        dealer_hand.push(deck.pop().unwrap());
        player_hand.push(deck.pop().unwrap());
        dealer_hand.push(deck.pop().unwrap());

        // プレイヤーの手札表示
        println!(
            "\nあなたの手札: {}  → 合計: {}",
            show_hand(&player_hand),
            hand_value(&player_hand)
        );
        println!(
            "ディーラーの手札: {} [??]",
            dealer_hand[0].display()
        );

        // ブラックジャック判定
        if hand_value(&player_hand) == 21 {
            println!("\nブラックジャック！ あなたの勝ちです！");
            wins += 1;
        } else {
            // プレイヤーのターン
            let mut player_bust = false;

            loop {
                let action = read_input("\nヒット(h) / スタンド(s): ");

                if action == "h" {
                    let card = deck.pop().unwrap();
                    println!("{} を引きました。", card.display());
                    player_hand.push(card);

                    let total = hand_value(&player_hand);
                    println!(
                        "あなたの手札: {}  → 合計: {}",
                        show_hand(&player_hand),
                        total
                    );

                    if total > 21 {
                        println!("\nバスト！ あなたの負けです。");
                        player_bust = true;
                        losses += 1;
                        break;
                    }
                } else if action == "s" {
                    break;
                } else {
                    println!("'h' または 's' を入力してください。");
                }
            }

            if !player_bust {
                // ディーラーのターン
                println!("\nディーラーのターン:");
                println!(
                    "ディーラーの手札: {}  → 合計: {}",
                    show_hand(&dealer_hand),
                    hand_value(&dealer_hand)
                );

                while hand_value(&dealer_hand) < 17 {
                    let card = deck.pop().unwrap();
                    dealer_hand.push(card);
                    println!("ディーラーが引きました: {}", card.display());
                    println!(
                        "ディーラーの手札: {}  → 合計: {}",
                        show_hand(&dealer_hand),
                        hand_value(&dealer_hand)
                    );
                }

                let player_total = hand_value(&player_hand);
                let dealer_total = hand_value(&dealer_hand);

                if dealer_total > 21 {
                    println!("\nディーラーがバスト！ あなたの勝ちです！");
                    wins += 1;
                } else if player_total > dealer_total {
                    println!(
                        "\nあなたの勝ち！（あなた: {} vs ディーラー: {}）",
                        player_total, dealer_total
                    );
                    wins += 1;
                } else if player_total < dealer_total {
                    println!(
                        "\nディーラーの勝ち！（ディーラー: {} vs あなた: {}）",
                        dealer_total, player_total
                    );
                    losses += 1;

                    println!("\n引き分け！（両者: {}）", player_total);
```

```rust
// ブラックジャック（改良版）
// ゲーム状態を構造体で管理する設計
// 実行方法：rustc main.rs && ./main

use std::io;
use std::io::Write;

// --- 擬似乱数 ---
struct SimpleRng {
    state: u64,
}

impl SimpleRng {
    fn new(seed: u64) -> SimpleRng {
        SimpleRng { state: seed }
    }

    fn next(&mut self) -> u64 {
        self.state = self.state.wrapping_mul(6364136223846793005).wrapping_add(1);
        self.state >> 33
    }

    fn shuffle<T>(&mut self, vec: &mut Vec<T>) {
        let len = vec.len();
        for i in (1..len).rev() {
            let j = (self.next() as usize) % (i + 1);
            vec.swap(i, j);
        }
    }
}

// --- カード ---
#[derive(Clone, Copy)]
enum Suit { Heart, Diamond, Club, Spade }

impl Suit {
    fn symbol(&self) -> &'static str {
        match self {
            Suit::Heart => "ハート",
            Suit::Diamond => "ダイヤ",
            Suit::Club => "クラブ",
            Suit::Spade => "スペード",
        }
    }
}

#[derive(Clone, Copy)]
struct Card { suit: Suit, rank: u8 }

impl Card {
    fn display(&self) -> String {
        let rank_str = match self.rank {
            1 => "A".to_string(),
            11 => "J".to_string(),
            12 => "Q".to_string(),
            13 => "K".to_string(),
            n => n.to_string(),
        };
        format!("[{}{}]", self.suit.symbol(), rank_str)
    }
}

/// 手札を表す構造体
struct Hand {
    cards: Vec<Card>,
}

impl Hand {
    fn new() -> Hand {
        Hand { cards: Vec::new() }
    }

    fn add(&mut self, card: Card) {
        self.cards.push(card);
    }

    fn value(&self) -> u32 {
        let mut total: u32 = 0;
        let mut aces: u32 = 0;
        for card in &self.cards {
            match card.rank {
                1 => { aces += 1; total += 11; }
                11 | 12 | 13 => total += 10,
                n => total += n as u32,
            }
        }
        while total > 21 && aces > 0 {
            total -= 10;
            aces -= 1;
        }
        total
    }

    fn is_bust(&self) -> bool {
        self.value() > 21
    }

    fn is_blackjack(&self) -> bool {
        self.cards.len() == 2 && self.value() == 21
    }

    fn display(&self) -> String {
        self.cards.iter().map(|c| c.display()).collect::<Vec<_>>().join(" ")
    }
}

/// ゲーム結果
enum GameResult {
    PlayerWin(String),   // 理由
    DealerWin(String),
    Draw,
}

/// ゲーム状態を管理する構造体
struct Game {
    deck: Vec<Card>,
    player: Hand,
    dealer: Hand,
    rng: SimpleRng,
}

impl Game {
    fn new(seed: u64) -> Game {
        let mut rng = SimpleRng::new(seed);
        let mut deck = Vec::new();
        let suits = [Suit::Heart, Suit::Diamond, Suit::Club, Suit::Spade];
        for &suit in &suits {
            for rank in 1..=13 {
                deck.push(Card { suit, rank });
            }
        }
        rng.shuffle(&mut deck);

        Game {
            deck,
            player: Hand::new(),
            dealer: Hand::new(),
            rng,
        }
    }

    fn deal_initial(&mut self) {
        for _ in 0..2 {
            let card = self.deck.pop().unwrap();
            self.player.add(card);
            let card = self.deck.pop().unwrap();
            self.dealer.add(card);
        }
    }

    fn draw_card(&mut self) -> Card {
        self.deck.pop().unwrap()
    }

    fn show_initial_state(&self) {
        println!(
            "\nあなたの手札: {}  → 合計: {}",
            self.player.display(),
            self.player.value()
        );
        println!(
            "ディーラーの手札: {} [??]",
            self.dealer.cards[0].display()
        );
    }

    fn dealer_turn(&mut self) {
        println!("\nディーラーのターン:");
        println!(
            "ディーラーの手札: {}  → 合計: {}",
            self.dealer.display(),
            self.dealer.value()
        );

        while self.dealer.value() < 17 {
            let card = self.draw_card();
            println!("ディーラーが引きました: {}", card.display());
            self.dealer.add(card);
            println!(
                "ディーラーの手札: {}  → 合計: {}",
                self.dealer.display(),
                self.dealer.value()
            );
        }
    }

    fn judge(&self) -> GameResult {
        let p = self.player.value();
        let d = self.dealer.value();

        if self.player.is_blackjack() {
            GameResult::PlayerWin("ブラックジャック！".to_string())
        } else if self.player.is_bust() {
            GameResult::DealerWin("バスト！".to_string())
        } else if self.dealer.is_bust() {
            GameResult::PlayerWin("ディーラーがバスト！".to_string())
        } else if p > d {
            GameResult::PlayerWin(format!("あなた: {} vs ディーラー: {}", p, d))
        } else if p < d {
            GameResult::DealerWin(format!("ディーラー: {} vs あなた: {}", d, p))
        } else {
            GameResult::Draw
        }
    }
}

fn read_input(prompt: &str) -> String {
    print!("{}", prompt);
    io::stdout().flush().unwrap();
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

fn main() {
    println!("===== ブラックジャック =====");

    let mut seed: u64 = 42;
    let mut wins: u32 = 0;
    let mut losses: u32 = 0;
    let mut draws: u32 = 0;

    loop {
        seed = seed.wrapping_add(98765);
        let mut game = Game::new(seed);
        game.deal_initial();
        game.show_initial_state();

        // ブラックジャック即勝ち判定
        if !game.player.is_blackjack() {
            // プレイヤーのターン
            loop {
                let action = read_input("\nヒット(h) / スタンド(s): ");
                match action.as_str() {
                    "h" => {
                        let card = game.draw_card();
                        println!("{} を引きました。", card.display());
                        game.player.add(card);
                        println!(
                            "あなたの手札: {}  → 合計: {}",
                            game.player.display(),
                            game.player.value()
                        );
                        if game.player.is_bust() {
                            break;
                        }
                    }
                    "s" => break,
                    _ => println!("'h' または 's' を入力してください。"),
                }
            }

            if !game.player.is_bust() {
                game.dealer_turn();
            }
        }

        // 勝敗判定
        match game.judge() {
            GameResult::PlayerWin(reason) => {
                println!("\n{} あなたの勝ちです！", reason);
                wins += 1;
            }
            GameResult::DealerWin(reason) => {
                println!("\n{} あなたの負けです。", reason);
                losses += 1;
            }
            GameResult::Draw => {
                println!("\n引き分け！（両者: {}）", game.player.value());
                draws += 1;
            }
        }

        let total = wins + losses + draws;
        println!("\n--- 戦績 ---");
        println!(
            "プレイ回数: {}  勝ち: {}  負け: {}  引き分け: {}",
            total, wins, losses, draws
        );

        if read_input("\nもう1回プレイしますか？ (y/n): ") != "y" {
            println!("お疲れさまでした！");
            break;
        }
    }
}
```
