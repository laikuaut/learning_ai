# 実践課題01：自己紹介ページ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（HTMLの基本構造）、第2章（テキストとリンク）
> **課題の種類**: ミニプロジェクト
> **学習目標**: HTMLの基本構造（DOCTYPE、html、head、body）を使い、見出し・段落・リンクを組み合わせた1ページを作る体験をする

---

## 完成イメージ

ブラウザで開くと以下のような自己紹介ページが表示されます。

```
自己紹介ページ（ブラウザのタブに表示）

===========================
  山田太郎の自己紹介
===========================

■ プロフィール
名前：山田太郎
年齢：25歳
住所：東京都渋谷区

■ 趣味
プログラミングが好きです。特にWebサイトを
作ることに興味があります。

休日は読書や映画鑑賞をしています。

■ 好きな技術
HTMLとCSSを勉強中です。将来はフルスタック
エンジニアを目指しています。

■ リンク
→ Googleで検索する（リンク）
→ MDN Web Docsを見る（リンク）
```

---

## 課題の要件

1. `<!DOCTYPE html>` で始まり、正しいHTML基本構造を持つこと
2. `<title>` にページタイトルを設定すること
3. `<meta charset="utf-8">` を指定すること
4. `<h1>` で名前を含むページ見出しを作ること
5. `<h2>` でセクション見出し（プロフィール、趣味、好きな技術、リンク）を作ること
6. `<p>` で自己紹介文を書くこと
7. `<a>` で外部サイトへのリンクを2つ以上作ること（新しいタブで開くようにする）
8. `<br>` を適切に使って改行すること

---

## ステップガイド

<details>
<summary>ステップ1：HTML基本構造を書く</summary>

まずはHTMLファイルの骨格を作りましょう。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>自己紹介ページ</title>
</head>
<body>
  <!-- ここにコンテンツを書いていきます -->
</body>
</html>
```

`lang="ja"` は日本語のページであることを示します。

</details>

<details>
<summary>ステップ2：見出しと段落を追加する</summary>

`<h1>` でメインの見出し、`<h2>` で各セクションの見出しを作ります。

```html
<h1>山田太郎の自己紹介</h1>

<h2>プロフィール</h2>
<p>
  名前：山田太郎<br>
  年齢：25歳<br>
  住所：東京都渋谷区
</p>
```

`<br>` は段落内での改行に使います。

</details>

<details>
<summary>ステップ3：リンクを追加する</summary>

`<a>` タグでリンクを作ります。`target="_blank"` で新しいタブで開きます。

```html
<h2>リンク</h2>
<p>
  <a href="https://www.google.com" target="_blank">Googleで検索する</a>
</p>
```

</details>

<details>
<summary>解答例（改良版 ─ セマンティクスを意識）</summary>

第6章の知識を先取りして、セマンティックHTML（semantic HTML）を使った改良版です。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>山田太郎の自己紹介</title>
</head>
<body>
  <header>
    <h1>山田太郎の自己紹介</h1>
  </header>

  <main>
    <section>
      <h2>プロフィール</h2>
      <dl>
        <dt>名前</dt>
        <dd>山田太郎</dd>
        <dt>年齢</dt>
        <dd>25歳</dd>
        <dt>住所</dt>
        <dd>東京都渋谷区</dd>
      </dl>
    </section>

    <section>
      <h2>趣味</h2>
      <p>プログラミングが好きです。特にWebサイトを作ることに興味があります。</p>
      <p>休日は読書や映画鑑賞をしています。</p>
    </section>

    <section>
      <h2>好きな技術</h2>
      <p>HTMLとCSSを勉強中です。将来はフルスタックエンジニアを目指しています。</p>
    </section>

    <nav>
      <h2>リンク</h2>
      <ul>
        <li><a href="https://www.google.com" target="_blank" rel="noopener noreferrer">Googleで検索する</a></li>
        <li><a href="https://developer.mozilla.org/ja/" target="_blank" rel="noopener noreferrer">MDN Web Docsを見る</a></li>
      </ul>
    </nav>
  </main>

  <footer>
    <p><small>&copy; 2026 山田太郎</small></p>
  </footer>
</body>
</html>
```

**初心者向けとの違い:**
- `<header>`、`<main>`、`<section>`、`<nav>`、`<footer>` でページの構造を明示（セマンティックHTML）
- プロフィールを `<dl>`（定義リスト）で構造化 → 情報の関係性が明確に
- リンクを `<ul>` でリスト化 → スクリーンリーダーが「リンクが2つあります」と伝えられる
- `rel="noopener noreferrer"` を追加 → `target="_blank"` のセキュリティ対策
- `<meta name="viewport">` を追加 → スマートフォンでも正しく表示される
- `<small>` と `&copy;` で著作権表記を追加

</details>
