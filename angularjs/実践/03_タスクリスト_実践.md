# 実践課題03：タスクリスト ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第1章（AngularJSの基本）、第2章（データバインディングとスコープ）、第3章（ディレクティブ）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `ng-repeat` でリストを動的に表示し、`ng-if` / `ng-show` / `ng-class` / `ng-click` を組み合わせてインタラクティブなタスク管理UIを構築する

---

## 完成イメージ

タスクの追加・完了・削除・フィルタリングができるタスクリストです。

```
┌─────────────────────────────────────┐
│  タスクリスト                         │
│                                     │
│  [新しいタスクを入力...     ] [追加]   │
│                                     │
│  フィルタ: [すべて] [未完了] [完了済]   │
│  残り: 3件 / 全5件                   │
│                                     │
│  ☑ 買い物に行く          [削除]      │
│  ☐ レポートを書く         [削除]      │
│  ☐ 部屋を掃除する         [削除]      │
│  ☑ メールを返信する        [削除]      │
│  ☐ AngularJSを勉強する    [削除]      │
│                                     │
│  [完了済をすべて削除]                  │
└─────────────────────────────────────┘
```

---

## 課題の要件

1. テキスト入力とボタンでタスクを追加する
2. 各タスクにチェックボックスを表示し、クリックで完了/未完了を切り替える
3. 完了したタスクには取り消し線を表示する（`ng-class`）
4. 各タスクに削除ボタンを付ける
5. 「すべて」「未完了」「完了済」のフィルタボタンを実装する
6. 残りの未完了タスク数を表示する
7. 「完了済をすべて削除」ボタンを実装する
8. 空のタスクは追加できないようにする

---

## ステップガイド

<details>
<summary>ステップ1：タスクデータの構造を決める</summary>

```javascript
$scope.tasks = [
  { id: 1, text: '買い物に行く', done: true },
  { id: 2, text: 'レポートを書く', done: false },
  { id: 3, text: '部屋を掃除する', done: false }
];
$scope.nextId = 4;
```

</details>

<details>
<summary>ステップ2：ng-repeat でタスクを表示する</summary>

```html
<div ng-repeat="task in tasks" ng-class="{'completed': task.done}">
  <input type="checkbox" ng-model="task.done">
  <span>{{ task.text }}</span>
  <button ng-click="removeTask(task.id)">削除</button>
</div>
```

</details>

<details>
<summary>ステップ3：フィルタリングを実装する</summary>

`ng-show` を使って表示/非表示を切り替えます。

```html
<div ng-repeat="task in tasks"
     ng-show="filter === 'all' || (filter === 'active' && !task.done) || (filter === 'done' && task.done)">
```

</details>

  <script>
    angular.module('todoApp', [])
      .controller('TodoCtrl', function($scope) {
        $scope.tasks = [
          { id: 1, text: '買い物に行く', done: true },
          { id: 2, text: 'レポートを書く', done: false },
          { id: 3, text: '部屋を掃除する', done: false },
          { id: 4, text: 'メールを返信する', done: true },
          { id: 5, text: 'AngularJSを勉強する', done: false }
        ];
        $scope.nextId = 6;
        $scope.newTask = '';
        $scope.filter = 'all';

        $scope.addTask = function() {
          var text = $scope.newTask.trim();
          if (text) {
            $scope.tasks.push({
              id: $scope.nextId++,
              text: text,
              done: false
            });
            $scope.newTask = '';
          }
        };

        $scope.removeTask = function(id) {
          $scope.tasks = $scope.tasks.filter(function(task) {
            return task.id !== id;
          });
        };

        $scope.getRemainingCount = function() {
          return $scope.tasks.filter(function(task) {
            return !task.done;
          }).length;
        };

        $scope.getCompletedCount = function() {
          return $scope.tasks.filter(function(task) {
            return task.done;
          }).length;
        };

        $scope.clearCompleted = function() {
          $scope.tasks = $scope.tasks.filter(function(task) {
            return !task.done;
          });
        };
      });
  </script>

</body>
</html>

<summary>解答例（改良版 ─ Controller As + 編集機能 + アニメーション）</summary>

```html
<!DOCTYPE html>
<html ng-app="todoApp">
<head>
  <meta charset="UTF-8">
  <title>タスクリスト（改良版）</title>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; background: #f0f2f5; max-width: 520px; margin: 0 auto; padding: 40px 20px; }
    h1 { text-align: center; color: #333; }
    .input-area { display: flex; gap: 8px; margin-bottom: 16px; }
    .input-area input { flex: 1; padding: 10px 14px; border: 1px solid #ddd; border-radius: 8px; font-size: 14px; }
    .input-area input:focus { border-color: #2196f3; outline: none; }
    .input-area button { padding: 10px 20px; background: #4caf50; color: white; border: none; border-radius: 8px; cursor: pointer; }
    .input-area button:disabled { background: #ccc; }
    .filters { display: flex; gap: 8px; margin-bottom: 12px; }
    .fbtn { padding: 5px 12px; border: 1px solid #ddd; border-radius: 14px; background: white; cursor: pointer; font-size: 12px; }
    .fbtn.on { background: #2196f3; color: white; border-color: #2196f3; }
    .stats { color: #888; font-size: 13px; margin-bottom: 10px; }
    .item { display: flex; align-items: center; padding: 10px 12px; background: white; border-radius: 8px;
            margin-bottom: 6px; box-shadow: 0 1px 4px rgba(0,0,0,0.06); transition: opacity 0.3s; }
    .item.done-item { opacity: 0.6; }
    .item input[type="checkbox"] { margin-right: 10px; width: 18px; height: 18px; }
    .item-text { flex: 1; font-size: 14px; cursor: pointer; }
    .item-text.done { text-decoration: line-through; color: #bbb; }
    .edit-input { flex: 1; padding: 4px 8px; border: 1px solid #2196f3; border-radius: 4px; font-size: 14px; }
    .del-btn { background: none; border: none; color: #f44336; cursor: pointer; font-size: 18px; padding: 0 4px; }
    .clear-btn { display: block; width: 100%; padding: 10px; margin-top: 16px;
                 background: #ff5722; color: white; border: none; border-radius: 8px; cursor: pointer; }
    .empty { text-align: center; color: #ccc; padding: 24px; }
    .progress { height: 4px; background: #e0e0e0; border-radius: 2px; margin-bottom: 16px; overflow: hidden; }
    .progress-bar { height: 100%; background: #4caf50; transition: width 0.3s; }
  </style>
</head>
<body ng-controller="TodoCtrl as vm">

  <h1>タスクリスト</h1>

  <!-- 進捗バー -->
  <div class="progress">
    <div class="progress-bar" ng-style="{'width': vm.getProgress() + '%'}"></div>
  </div>

  <div class="input-area">
    <input type="text" ng-model="vm.newTask" placeholder="新しいタスクを入力..."
           ng-keypress="$event.keyCode === 13 && vm.addTask()">
    <button ng-click="vm.addTask()" ng-disabled="!vm.newTask || !vm.newTask.trim()">追加</button>
  </div>

  <div class="filters">
    <button class="fbtn" ng-class="{'on': vm.filter === 'all'}" ng-click="vm.filter = 'all'">
      すべて ({{ vm.tasks.length }})
    </button>
    <button class="fbtn" ng-class="{'on': vm.filter === 'active'}" ng-click="vm.filter = 'active'">
      未完了 ({{ vm.remaining() }})
    </button>
    <button class="fbtn" ng-class="{'on': vm.filter === 'done'}" ng-click="vm.filter = 'done'">
      完了 ({{ vm.completed() }})
    </button>
  </div>

  <div class="stats">
    {{ vm.remaining() }}件の未完了タスク（完了率: {{ vm.getProgress() | number:0 }}%）
  </div>

  <div ng-repeat="task in vm.tasks track by task.id"
       ng-show="vm.isVisible(task)"
       class="item" ng-class="{'done-item': task.done}">
    <input type="checkbox" ng-model="task.done">
    <!-- 通常表示 / 編集モード切替 -->
    <span class="item-text" ng-class="{'done': task.done}"
          ng-hide="task.editing" ng-dblclick="vm.startEdit(task)">{{ task.text }}</span>
    <input class="edit-input" ng-show="task.editing" ng-model="task.text"
           ng-keypress="$event.keyCode === 13 && vm.endEdit(task)"
           ng-blur="vm.endEdit(task)">
    <button class="del-btn" ng-click="vm.remove(task.id)">&#10005;</button>
  </div>

  <div class="empty" ng-if="vm.tasks.length === 0">
    タスクがありません
  </div>

  <button class="clear-btn" ng-click="vm.clearDone()" ng-show="vm.completed() > 0">
    完了済をすべて削除（{{ vm.completed() }}件）
  </button>

  <script>
    angular.module('todoApp', [])
      .controller('TodoCtrl', function() {
        var vm = this;
        vm.nextId = 6;
        vm.newTask = '';
        vm.filter = 'all';
        vm.tasks = [
          { id: 1, text: '買い物に行く', done: true, editing: false },
          { id: 2, text: 'レポートを書く', done: false, editing: false },
          { id: 3, text: '部屋を掃除する', done: false, editing: false },
          { id: 4, text: 'メールを返信する', done: true, editing: false },
          { id: 5, text: 'AngularJSを勉強する', done: false, editing: false }
        ];

        vm.addTask = function() {
          var text = vm.newTask.trim();
          if (text) {
            vm.tasks.push({ id: vm.nextId++, text: text, done: false, editing: false });
            vm.newTask = '';
          }
        };

        vm.remove = function(id) {
          vm.tasks = vm.tasks.filter(function(t) { return t.id !== id; });
        };

        vm.remaining = function() {
          return vm.tasks.filter(function(t) { return !t.done; }).length;
        };

        vm.completed = function() {
          return vm.tasks.filter(function(t) { return t.done; }).length;
        };

        vm.clearDone = function() {
          vm.tasks = vm.tasks.filter(function(t) { return !t.done; });
        };

        vm.getProgress = function() {
          if (vm.tasks.length === 0) return 0;
          return (vm.completed() / vm.tasks.length) * 100;
        };

        vm.isVisible = function(task) {
          if (vm.filter === 'all') return true;
          if (vm.filter === 'active') return !task.done;
          return task.done;
        };

        // ダブルクリックで編集開始
        vm.startEdit = function(task) {
          task.editing = true;
        };

        vm.endEdit = function(task) {
          task.editing = false;
          if (!task.text.trim()) {
            vm.remove(task.id);
          }
        };
      });
  </script>

</body>
</html>
```

**初心者向けとの違い:**
- `Controller As` 構文で `$scope` を使わない設計
- ダブルクリックでタスク名をインライン編集できる機能を追加
- 進捗バー（プログレスバー）で完了率を視覚化
- フィルタボタンに各カテゴリの件数を表示
- `number` フィルターで完了率をフォーマット
- 完了済タスクの行全体に `opacity` を下げる視覚効果
