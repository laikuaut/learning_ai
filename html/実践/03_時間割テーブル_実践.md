# 実践課題03：時間割テーブル ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第1章（HTMLの基本構造）、第2章（テキストとリンク）、第4章（リストとテーブル）
> **課題の種類**: ミニプロジェクト
> **学習目標**: テーブル要素（table、thead、tbody、th、td）を使いこなし、セル結合（colspan、rowspan）を含む実用的な表を作る

---

## 完成イメージ

ブラウザで開くと以下のような学校の時間割表が表示されます。

```
=================================
    2026年度 3年A組 時間割
=================================

| 時限 |  月曜   |  火曜   |  水曜   |  木曜   |  金曜   |
|------|---------|---------|---------|---------|---------|
| 1限  | 国語    | 数学    | 英語    | 理科    | 社会    |
| 2限  | 数学    | 英語    | 国語    | 社会    | 理科    |
| 3限  | 英語    | 理科    | 数学    | 国語    | 英語    |
|      |     昼休み（全日共通）                          |
| 4限  | 体育    | 音楽    | 美術    | 体育    | 総合    |
| 5限  | 体育    | 技術    |  HR     | 家庭科  | 総合    |

※ 体育は月曜と木曜で2時間連続です
※ 総合は金曜の午後に行います
```

---

## 課題の要件

1. 正しいHTML基本構造を持つこと
2. `<table>` でテーブルを作成すること
3. `<thead>` と `<tbody>` でヘッダーとボディを分離すること
4. `<th>` で見出しセル、`<td>` でデータセルを使い分けること
5. `<caption>` でテーブルのタイトルを付けること
6. `colspan` を使って「昼休み」を横方向に結合すること
7. `rowspan` を使って「体育」と「総合」を縦方向に結合すること
8. `border` 属性でテーブルの枠線を表示すること

---

## ステップガイド

<details>
<summary>ステップ1：テーブルの骨格を作る</summary>

```html
<table border="1">
  <caption>2026年度 3年A組 時間割</caption>
  <thead>
    <tr>
      <th>時限</th>
      <th>月曜</th>
      <th>火曜</th>
      <th>水曜</th>
      <th>木曜</th>
      <th>金曜</th>
    </tr>
  </thead>
  <tbody>
    <!-- ここに各行を追加 -->
  </tbody>
</table>
```

</details>

<details>
<summary>ステップ2：通常の行を追加する</summary>

```html
<tr>
  <th>1限</th>
  <td>国語</td>
  <td>数学</td>
  <td>英語</td>
  <td>理科</td>
  <td>社会</td>
</tr>
```

時限の列は見出しなので `<th>` を使います。

</details>

<details>
<summary>ステップ3：colspan でセルを横に結合する</summary>

「昼休み」は全曜日共通なので、1つのセルで5列分を結合します。

```html
<tr>
  <th>昼</th>
  <td colspan="5">昼休み</td>
</tr>
```

`colspan="5"` は「このセルを横に5列分の幅にする」という意味です。

</details>

<details>
<summary>ステップ4：rowspan でセルを縦に結合する</summary>

月曜の「体育」は4限と5限で連続なので、4限のセルを縦に2行分結合します。

```html
<!-- 4限の行 -->
<tr>
  <th>4限</th>
  <td rowspan="2">体育</td>  <!-- 月曜：2行分 -->
  <td>音楽</td>
  <td>美術</td>
  <td>体育</td>
  <td rowspan="2">総合</td>  <!-- 金曜：2行分 -->
</tr>
<!-- 5限の行（月曜と金曜のセルは不要） -->
<tr>
  <th>5限</th>
  <!-- 月曜は上の行で rowspan しているので書かない -->
  <td>技術</td>
  <td>HR</td>
  <td>家庭科</td>
  <!-- 金曜も上の行で rowspan しているので書かない -->
</tr>
```

`rowspan="2"` を指定した列は、次の行ではセルを書きません。

</details>

<summary>解答例（改良版 ─ tfoot とスタイル属性を活用）</summary>

`<tfoot>`（テーブルフッター）やスタイル属性を使って見やすくした改良版です。

```html

  <meta name="viewport" content="width=device-width, initial-scale=1.0">

    table {
      border-collapse: collapse;
      width: 100%;
      max-width: 700px;
    }
    th, td {
      border: 1px solid #333;
      padding: 8px 12px;
      text-align: center;
    }
    thead th {
      background-color: #4a90d9;
      color: white;
    }
    tbody th {
      background-color: #e8f0fe;
    }
    tfoot td {
      background-color: #f5f5f5;
      font-size: 0.9em;
      text-align: left;
    }
    .lunch {
      background-color: #fff3e0;
      font-weight: bold;
    }
    .pe {
      background-color: #e8f5e9;
    }
    .sogo {
      background-color: #fce4ec;
    }
  </style>
</head>
<body>
  <h1>2026年度 3年A組 時間割</h1>

  <table>
    <caption>前期 時間割（4月〜9月）</caption>
    <thead>
      <tr>
        <th>時限</th>
        <th>月曜</th>
        <th>火曜</th>
        <th>水曜</th>
        <th>木曜</th>
        <th>金曜</th>
      </tr>
    </thead>
    <tfoot>
      <tr>
        <td colspan="6">
          ※ 体育は運動着を持参してください ／
          ※ 総合学習は金曜午後に実施 ／
          ※ 行事等で変更になる場合があります
        </td>
      </tr>
    </tfoot>
    <tbody>
      <tr>
        <th>1限<br><small>8:50-9:40</small></th>
        <td>国語</td>
        <td>数学</td>
        <td>英語</td>
        <td>理科</td>
        <td>社会</td>
      </tr>
      <tr>
        <th>2限<br><small>9:50-10:40</small></th>

```

**初心者向けとの違い:**
- `<tfoot>` で注意書きをテーブルフッターに配置 → テーブルの構造が正しくなる
- `<style>` タグでCSSを使い、見やすいデザインに → `border-collapse: collapse` で二重線を解消
- 各時限に時間帯を `<small>` で追加 → 実用的な情報が増える
- CSSクラスで背景色を変えて視覚的に区別 → 体育・総合・昼休みが一目で分かる
- `<tfoot>` はHTML上で `<tbody>` より前に書けるのが特徴（ブラウザは正しい位置に表示する）
