## 1. CSSとは？

**CSS（Cascading Style Sheets）** は、HTMLで作った文書の「見た目」を装飾するための言語です。

HTMLが「文書の構造（骨組み）」を担当し、CSSが「デザイン（見た目）」を担当します。

```
HTML = 家の骨組み（柱、壁、屋根）
CSS  = 内装・外装（壁の色、床材、カーテン）
```

### CSSの基本的な書き方

```css
セレクタ {
  プロパティ: 値;
  プロパティ: 値;
}
```

**例：**

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

この例では：
- `h1` → **セレクタ**（どの要素に適用するか）
- `color` → **プロパティ**（何を変えるか）
- `blue` → **値**（どう変えるか）
- 最後の `;`（セミコロン）を忘れずに！

## 2. CSSをHTMLに適用する3つの方法

### 方法1：外部スタイルシート（推奨）

別ファイル（`.css`）にCSSを書き、HTMLから読み込みます。

**style.css:**
```css
h1 {
  color: red;
}
```

**index.html:**
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="style.css">
  <title>ページタイトル</title>
</head>
<body>
  <h1>こんにちは</h1>
</body>
</html>
```

> **メリット：** 複数のHTMLファイルで同じCSSを共有でき、管理しやすい。実務ではほぼこの方法を使います。

### 方法2：内部スタイルシート

HTMLの `<head>` 内に `<style>` タグで書きます。

```html
<head>
  <style>
    h1 {
      color: red;
    }
  </style>
</head>
```

> **用途：** ちょっとした練習やテスト時に便利。

### 方法3：インラインスタイル

HTML要素に直接 `style` 属性で書きます。

```html
<h1 style="color: red;">こんにちは</h1>
```

> **注意：** 管理が難しくなるため、基本的には避けましょう。

### 3つの方法の比較

| 方法 | 書く場所 | おすすめ度 | 用途 |
|------|---------|-----------|------|
| 外部スタイルシート | 別の `.css` ファイル | ★★★ | 実務・本格的な開発 |
| 内部スタイルシート | `<style>` タグ内 | ★★☆ | 練習・小規模なページ |
| インラインスタイル | `style` 属性 | ★☆☆ | 緊急の上書き程度 |

## 3. セレクタの基本

セレクタは「どのHTML要素にスタイルを適用するか」を指定するものです。

### 3.1 要素セレクタ（タグ名で指定）

HTMLタグ名をそのまま書きます。

```css
p {
  color: gray;
}
```

→ すべての `<p>` 要素の文字色がグレーになります。

### 3.2 クラスセレクタ（`.`で指定）

HTML要素に付けた `class` 属性を使って指定します。最もよく使います。

```html
<p class="highlight">重要なテキスト</p>
<p>普通のテキスト</p>
```

```css
.highlight {
  color: red;
  font-weight: bold;
}
```

→ `class="highlight"` が付いた要素だけが赤い太字になります。

> **ポイント：** クラス名の前に **ドット（`.`）** を付けます。同じクラスは複数の要素に使えます。

### 3.3 IDセレクタ（`#`で指定）

HTML要素の `id` 属性を使って指定します。

```html
<h1 id="main-title">メインタイトル</h1>
```

```css
#main-title {
  color: navy;
  text-align: center;
}
```

→ `id="main-title"` の要素だけに適用されます。

> **注意：** 同じIDはページ内で1回だけ使えます。スタイル指定にはクラスを使うのが一般的です。

### 3.4 子孫セレクタ（スペースで区切る）

ある要素の中にある要素を指定します。

```css
nav a {
  color: white;
  text-decoration: none;
}
```

→ `<nav>` の中にある `<a>` だけに適用されます。

### 3.5 グループセレクタ（カンマで区切る）

複数のセレクタに同じスタイルを適用します。

```css
h1, h2, h3 {
  color: darkblue;
  font-family: sans-serif;
}
```

### セレクタまとめ表

| セレクタ | 書き方 | 例 | 意味 |
|---------|--------|-----|------|
| 要素 | `タグ名` | `p` | すべての `<p>` |
| クラス | `.クラス名` | `.highlight` | `class="highlight"` の要素 |
| ID | `#ID名` | `#main-title` | `id="main-title"` の要素 |
| 子孫 | `A B` | `nav a` | A の中にある B |
| グループ | `A, B` | `h1, h2` | A と B の両方 |

## 4. よく使うプロパティ

### 4.1 文字に関するプロパティ

```css
p {
  color: #333333;           /* 文字色 */
  font-size: 16px;          /* 文字サイズ */
  font-weight: bold;        /* 文字の太さ（bold / normal / 100〜900） */
  font-family: sans-serif;  /* フォントの種類 */
  text-align: center;       /* 文字の揃え方（left / center / right） */
  text-decoration: none;    /* 下線など（underline / none / line-through） */
  line-height: 1.8;         /* 行の高さ（1.5〜2.0が読みやすい） */
  letter-spacing: 1px;      /* 文字の間隔 */
}
```

### 4.2 背景に関するプロパティ

```css
div {
  background-color: #f0f0f0;                /* 背景色 */
  background-image: url("bg.jpg");           /* 背景画像 */
  background-size: cover;                    /* 背景画像のサイズ */
  background-position: center;               /* 背景画像の位置 */
  background-repeat: no-repeat;              /* 背景画像の繰り返し */
}
```

### 4.3 その他のよく使うプロパティ

```css
.box {
  width: 300px;             /* 幅 */
  height: 200px;            /* 高さ */
  border: 1px solid #ccc;   /* 枠線（太さ 種類 色） */
  border-radius: 8px;       /* 角の丸み */
  opacity: 0.8;             /* 透明度（0が完全透明、1が不透明） */
  cursor: pointer;          /* マウスカーソルの形 */
}
```

## 5. 色の指定方法

CSSでは色をさまざまな方法で指定できます。

### 5.1 色名（キーワード）

```css
color: red;
color: blue;
color: white;
color: black;
color: gray;
color: tomato;
color: cornflowerblue;
```

> 約140種類の色名が使えます。

### 5.2 16進数（HEX）

`#RRGGBB` の形式で指定します。最もよく使われます。

```css
color: #ff0000;   /* 赤 */
color: #00ff00;   /* 緑 */
color: #0000ff;   /* 青 */
color: #333333;   /* 濃いグレー */
color: #333;      /* ↑の省略形（同じ値が並ぶ場合） */
```

### 5.3 RGB

赤（Red）・緑（Green）・青（Blue）を0〜255で指定します。

```css
color: rgb(255, 0, 0);     /* 赤 */
color: rgb(100, 100, 100);  /* グレー */
```

### 5.4 RGBA（透明度付き）

RGBに透明度（Alpha：0〜1）を追加します。

```css
background-color: rgba(0, 0, 0, 0.5);   /* 半透明の黒 */
background-color: rgba(255, 0, 0, 0.3);  /* 薄い赤 */
```

### 5.5 HSL

色相（Hue：0〜360）・彩度（Saturation：0〜100%）・明度（Lightness：0〜100%）で指定します。

```css
color: hsl(0, 100%, 50%);    /* 赤 */
color: hsl(120, 100%, 50%);  /* 緑 */
color: hsl(240, 100%, 50%);  /* 青 */
```

### 色の指定方法まとめ

| 方法 | 書き方の例 | 特徴 |
|------|-----------|------|
| 色名 | `red` | 直感的で分かりやすい |
| HEX | `#ff0000` | 最もよく使われる |
| RGB | `rgb(255, 0, 0)` | 数値で細かく指定 |
| RGBA | `rgba(255, 0, 0, 0.5)` | 透明度を指定できる |
| HSL | `hsl(0, 100%, 50%)` | 色の調整がしやすい |

## 6. 単位の種類と使い分け

### 6.1 絶対単位

画面のサイズに関係なく、固定のサイズになります。

```css
font-size: 16px;   /* ピクセル：最もよく使う絶対単位 */
```

### 6.2 相対単位

基準となるサイズに対する割合で決まります。レスポンシブデザインに重要です。

```css
/* em：親要素のフォントサイズ基準 */
font-size: 1.5em;    /* 親の1.5倍 */

/* rem：ルート要素（html）のフォントサイズ基準 ← おすすめ */
font-size: 1.5rem;   /* htmlのフォントサイズの1.5倍 */

/* %：親要素のサイズ基準 */
width: 50%;           /* 親の幅の50% */

/* vw / vh：画面サイズ基準 */
width: 100vw;         /* 画面の幅いっぱい */
height: 100vh;        /* 画面の高さいっぱい */
```

### 単位の使い分けガイド

| 単位 | 基準 | よく使う場面 |
|------|------|------------|
| `px` | 固定 | ボーダー、小さい余白、アイコンサイズ |
| `rem` | html要素のフォントサイズ | フォントサイズ、余白（推奨） |
| `em` | 親要素のフォントサイズ | パディングなど親に連動させたい場合 |
| `%` | 親要素のサイズ | 幅の指定、レイアウト |
| `vw` | 画面の幅 | 画面いっぱいのセクション |
| `vh` | 画面の高さ | ヒーローセクション、フルスクリーン |

> **初心者へのアドバイス：** まずは `px` と `%` から始め、慣れてきたら `rem` を使いましょう。

## 7. CSSのコメント

CSSにメモを残すには `/* */` で囲みます。

```css
/* ここはコメントです。ブラウザには無視されます */
h1 {
  color: blue;   /* メインカラーを青に設定 */
  /* font-size: 32px; ← この行は無効化されている */
}
```

---

## 8. CSSの優先順位（カスケード）

同じ要素に複数のスタイルが指定された場合、以下の優先順位で適用されます。

**優先度（低い → 高い）:**

1. ブラウザのデフォルトスタイル
2. 要素セレクタ（`p`, `h1` など）
3. クラスセレクタ（`.highlight` など）
4. IDセレクタ（`#main-title` など）
5. インラインスタイル（`style="..."`）
6. `!important`（なるべく使わない）

```css
p { color: blue; }            /* 優先度：低 */
.text { color: green; }       /* 優先度：中 */
#special { color: red; }      /* 優先度：高 */
```

```html
<p class="text" id="special">この文字は赤になります</p>
```

> **大原則：** `!important` はデバッグ以外ではなるべく使わず、セレクタの設計で優先度を管理しましょう。

## 9. カスタムプロパティ（CSS変数）

繰り返し使う値を変数として定義できます。

```css
:root {
  --main-color: #3498db;
  --sub-color: #2ecc71;
  --font-size-base: 16px;
}

h1 {
  color: var(--main-color);
}

p {
  font-size: var(--font-size-base);
}

a {
  color: var(--sub-color);
}
```

> **メリット：** 色やサイズを一箇所で管理でき、変更が楽になります。

## まとめ

| 項目 | 要点 |
|------|------|
| CSSの役割 | HTMLの見た目を装飾する |
| 適用方法 | 外部スタイルシートが推奨 |
| セレクタ | 要素、クラス（`.`）、ID（`#`）など |
| 色の指定 | HEXが最も一般的、透明度にはRGBA |
| 単位 | `px`（固定）、`rem`（相対・推奨）、`%`（割合） |
| 優先順位 | 要素 < クラス < ID < インライン |
| CSS変数 | `--変数名` で定義、`var()` で使用 |
