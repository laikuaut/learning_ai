## 1. カスタムプロパティとは？

**カスタムプロパティ**（通称CSS変数）は、CSSの中で値を変数として保存し、再利用できる仕組みです。

```css
/* 変数を定義 */
:root {
  --primary-color: #3498db;
  --font-size-base: 16px;
  --spacing: 8px;
}

/* 変数を使う */
h1 {
  color: var(--primary-color);
  font-size: var(--font-size-base);
}

.card {
  padding: var(--spacing);
  border: 2px solid var(--primary-color);
}
```

### 命名ルール

- 必ず `--` から始める
- 大文字と小文字は区別される（`--Color` と `--color` は別の変数）
- ハイフンで単語を区切るのが一般的（`--primary-color`）

```css
/* OK */
--main-color: blue;
--fontSize: 16px;
--spacing-lg: 24px;

/* NG */
--: blue;           /* 名前がない */
color: blue;        /* --がない（これは通常のプロパティ） */
```

---

## 2. var() 関数

### 基本的な使い方

```css
.element {
  color: var(--text-color);
  padding: var(--spacing);
}
```

### フォールバック値（デフォルト値）

変数が定義されていない場合に使われる値を指定できます。

```css
.element {
  /* --accent-colorが未定義なら #e74c3c を使う */
  color: var(--accent-color, #e74c3c);

  /* フォールバックにも変数を使える */
  background-color: var(--bg-color, var(--default-bg, white));

  /* カンマを含む値も指定可能 */
  font-family: var(--font-main, "Helvetica Neue", Arial, sans-serif);
}
```

> **ポイント：** フォールバック値を設定しておくと、変数の定義忘れによるデザイン崩れを防げます。

---

## 3. スコープ（適用範囲）

カスタムプロパティはCSSのカスケードに従います。定義された要素とその子孫要素で使えます。

### グローバルスコープ（:root）

```css
/* :root はhtml要素と同じ。ページ全体で使える */
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
}
```

### ローカルスコープ

```css
.card {
  --card-padding: 16px;       /* .card とその子要素でのみ有効 */
  padding: var(--card-padding);
}

.card .title {
  padding-bottom: var(--card-padding);  /* 使える（子要素だから） */
}

.other-element {
  padding: var(--card-padding);  /* 効かない（.cardの子ではないから） */
}
```

### スコープの上書き

```css
:root {
  --text-color: #333;
}

.dark-section {
  --text-color: #fff;          /* このセクション内だけ白に */
}

p {
  color: var(--text-color);    /* 通常は#333、.dark-section内では#fff */
}
```

```
┌──────────────────────────────┐
│ :root (--text-color: #333)   │
│                              │
│  ┌────────────────────────┐  │
│  │ .dark-section           │  │
│  │ (--text-color: #fff)    │  │
│  │                        │  │
│  │  <p>白いテキスト</p>     │  │
│  └────────────────────────┘  │
│                              │
│  <p>黒いテキスト</p>          │
└──────────────────────────────┘
```

## 4. テーマの切り替え

カスタムプロパティの最大の活用場面がテーマ機能です。

### ダーク/ライトテーマ

```css
/* ライトテーマ（デフォルト） */
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
  --card-bg: #f8f9fa;
  --border-color: #dee2e6;
  --primary: #3498db;
  --shadow: rgba(0, 0, 0, 0.1);
}

/* ダークテーマ */
[data-theme="dark"] {
  --bg-color: #1a1a2e;
  --text-color: #e0e0e0;
  --card-bg: #16213e;
  --border-color: #0f3460;
  --primary: #64b5f6;
  --shadow: rgba(0, 0, 0, 0.3);
}

/* テーマ変数を使ってスタイルを書く */
body {
  background-color: var(--bg-color);
  color: var(--text-color);
}

.card {
  background-color: var(--card-bg);
  border: 1px solid var(--border-color);
  box-shadow: 0 2px 8px var(--shadow);
}

a {
  color: var(--primary);
}
```

```html
<!-- HTMLでテーマを切り替え -->
<html data-theme="light">  <!-- または "dark" -->
```

> **JavaScriptで切り替え：**
> ```javascript
> document.documentElement.setAttribute('data-theme', 'dark');
> ```

### OSのテーマ設定に連動

```css
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a2e;
    --text-color: #e0e0e0;
    /* ...その他のダークテーマ変数 */
  }
}
```

## 5. calc() との組み合わせ

`calc()` 関数を使うと、変数を使った計算ができます。

### 基本的な計算

```css
:root {
  --spacing-unit: 8px;
  --base-font-size: 16px;
}

.element {
  /* 変数を使った計算 */
  padding: calc(var(--spacing-unit) * 2);          /* 16px */
  margin-bottom: calc(var(--spacing-unit) * 3);    /* 24px */
  font-size: calc(var(--base-font-size) * 1.25);   /* 20px */
}
```

### スペーシングシステムの構築

```css
:root {
  --space-unit: 8px;
  --space-xs: calc(var(--space-unit) * 0.5);   /* 4px */
  --space-sm: var(--space-unit);                /* 8px */
  --space-md: calc(var(--space-unit) * 2);      /* 16px */
  --space-lg: calc(var(--space-unit) * 3);      /* 24px */
  --space-xl: calc(var(--space-unit) * 4);      /* 32px */
  --space-2xl: calc(var(--space-unit) * 6);     /* 48px */
}

.section {
  padding: var(--space-xl) var(--space-lg);
}

.card {
  padding: var(--space-md);
  margin-bottom: var(--space-lg);
}
```

> **メリット：** `--space-unit` を変更するだけで、すべての余白が比例して変わります。

### タイポグラフィスケール

```css
:root {
  --font-size-base: 16px;
  --font-size-sm: calc(var(--font-size-base) * 0.875);   /* 14px */
  --font-size-lg: calc(var(--font-size-base) * 1.125);   /* 18px */
  --font-size-xl: calc(var(--font-size-base) * 1.5);     /* 24px */
  --font-size-2xl: calc(var(--font-size-base) * 2);      /* 32px */
  --font-size-3xl: calc(var(--font-size-base) * 2.5);    /* 40px */
}
```

## 6. 動的な値の活用

### レスポンシブな変数

メディアクエリの中で変数の値を変更できます。

```css
:root {
  --container-padding: 16px;
  --heading-size: 24px;
  --grid-columns: 1;
}

@media (min-width: 768px) {
  :root {
    --container-padding: 32px;
    --heading-size: 32px;
    --grid-columns: 2;
  }
}

@media (min-width: 1200px) {
  :root {
    --container-padding: 48px;
    --heading-size: 40px;
    --grid-columns: 3;
  }
}

/* 変数を使うだけ。メディアクエリの重複を減らせる */
.container {
  padding: 0 var(--container-padding);
}

h1 {
  font-size: var(--heading-size);
}

.grid {
  display: grid;
  grid-template-columns: repeat(var(--grid-columns), 1fr);
}
```

### カウンターやインデックス的な使い方

```css
/* リスト項目に順番に遅延をつける */
.item:nth-child(1) { --delay: 0; }
.item:nth-child(2) { --delay: 1; }
.item:nth-child(3) { --delay: 2; }
.item:nth-child(4) { --delay: 3; }

.item {
  animation: fadeIn 0.5s ease forwards;
  animation-delay: calc(var(--delay) * 0.1s);
}
```

### カラーシステム

```css
:root {
  /* ベースカラーをHSLで定義 */
  --hue-primary: 210;
  --hue-success: 120;
  --hue-danger: 0;

  /* 明度を変えてバリエーションを作る */
  --primary-light: hsl(var(--hue-primary), 70%, 80%);
  --primary: hsl(var(--hue-primary), 70%, 50%);
  --primary-dark: hsl(var(--hue-primary), 70%, 30%);

  --success: hsl(var(--hue-success), 60%, 45%);
  --danger: hsl(var(--hue-danger), 70%, 55%);
}
```

> **メリット：** `--hue-primary` の値を変えるだけで、サイト全体の色調が変わります。

## 7. 実践：デザイントークンとしての活用

実務では、カスタムプロパティを「デザイントークン」として体系的に管理します。

```css
:root {
  /* ========== カラー ========== */
  --color-primary: #3498db;
  --color-primary-hover: #2980b9;
  --color-secondary: #2ecc71;
  --color-danger: #e74c3c;
  --color-warning: #f39c12;

  --color-text: #333333;
  --color-text-light: #666666;
  --color-text-muted: #999999;

  --color-bg: #ffffff;
  --color-bg-secondary: #f8f9fa;
  --color-border: #dee2e6;

  /* ========== タイポグラフィ ========== */
  --font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
  --font-size-base: 16px;
  --line-height-base: 1.6;

  /* ========== スペーシング ========== */
  --space-unit: 8px;
  --space-sm: calc(var(--space-unit) * 1);
  --space-md: calc(var(--space-unit) * 2);
  --space-lg: calc(var(--space-unit) * 3);
  --space-xl: calc(var(--space-unit) * 4);

  /* ========== ボーダー ========== */
  --border-radius: 8px;
  --border-radius-sm: 4px;
  --border-radius-lg: 12px;
  --border-radius-full: 9999px;

  /* ========== シャドウ ========== */
  --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
  --shadow-lg: 0 10px 25px rgba(0, 0, 0, 0.15);

  /* ========== トランジション ========== */
  --transition-fast: 0.15s ease;
  --transition-normal: 0.3s ease;
  --transition-slow: 0.5s ease;
}
```

## まとめ

| 項目 | 要点 |
|------|------|
| 定義 | `--変数名: 値` で定義 |
| 使用 | `var(--変数名)` で参照 |
| フォールバック | `var(--変数名, デフォルト値)` で未定義時の値を指定 |
| スコープ | `:root` でグローバル、要素内でローカル |
| テーマ | 変数の値を切り替えるだけでテーマが変わる |
| calc() | `calc(var(--変数) * 2)` で計算 |
| デザイントークン | カラー・サイズ・余白を体系的に管理 |
