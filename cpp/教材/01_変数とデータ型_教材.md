# 第1章：変数とデータ型

> **この章のゴール**: C++の基本的な仕組みを理解し、変数の宣言・初期化・使用ができるようになる。標準入出力を使った簡単なプログラムを書けるようになる。

## 1.1 C++とは

C++は、1979年にビャーネ・ストロヴストルップ（Bjarne Stroustrup）が設計したプログラミング言語です。C言語を拡張して作られており、以下の特徴があります。

- **高速な実行速度** — ゲーム、OS、組み込みシステムなどで広く使われています
- **オブジェクト指向** — クラスや継承などの仕組みをサポートしています
- **標準ライブラリが充実** — 文字列操作、コンテナ、アルゴリズムなどが揃っています

本教材では **C++17** を基準に、モダンなC++の書き方を学んでいきます。

## 1.2 最初のプログラム ― Hello, World!

まずは最も基本的なプログラムを見てみましょう。

```cpp
// hello.cpp — 最初のC++プログラム
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

### コンパイル（compile）と実行

C++は **コンパイル言語** です。ソースコードを書いたら、コンパイラ（compiler）で実行ファイルに変換してから実行します。

```
ソースコード(.cpp)  →  コンパイラ  →  実行ファイル  →  実行
    hello.cpp          g++           a.out (Linux)      ./a.out
                                     hello.exe (Windows)
```

ターミナルでの操作：

```bash
# コンパイル（C++17 を指定）
g++ -std=c++17 -o hello hello.cpp

# 実行
./hello
```

出力：
```
Hello, World!
```

### プログラムの構造を分解して理解する

```
#include <iostream>        ← ① ヘッダファイルの読み込み

int main() {              ← ② main関数（プログラムの入口）
    std::cout << "Hello"; ← ③ 標準出力に文字列を表示
    return 0;             ← ④ プログラムの正常終了を表す
}
```

| 要素 | 説明 |
|---|---|
| `#include <iostream>` | 入出力に必要なヘッダファイル（header file）を読み込みます |
| `int main()` | プログラムは必ずここから始まります |
| `std::cout` | 標準出力（standard output）へデータを送ります |
| `<<` | 挿入演算子（insertion operator）。データを出力ストリームに流します |
| `std::endl` | 改行して出力バッファを flush します |
| `return 0;` | OSに「正常終了」を伝えます |

### ポイントまとめ
- C++プログラムは `main()` 関数から実行が始まります
- `#include` でヘッダファイルを読み込んで機能を使えるようにします
- コンパイル → 実行 の2ステップが必要です

## 1.3 名前空間（namespace）

先ほどのプログラムで `std::cout` と書きました。`std` は **名前空間（namespace）** と呼ばれるものです。

名前空間は、名前の衝突を防ぐための仕組みです。たとえば、自分で `cout` という変数を作っても、`std::cout` と区別できます。

```
┌─────────────────────────────┐
│  名前空間 std               │
│  ┌───────┐ ┌───────┐       │
│  │ cout  │ │ cin   │ ...   │
│  └───────┘ └───────┘       │
└─────────────────────────────┘
```

### using 宣言

毎回 `std::` と書くのが面倒な場合、`using` 宣言を使うことができます。

```cpp
#include <iostream>
using namespace std;  // std名前空間をすべて使えるようにする

int main() {
    cout << "Hello!" << endl;  // std:: が不要になる
    return 0;
}
```

ただし、**大規模なプログラムでは `using namespace std;` を避け、`std::` を明示するのが推奨** されます。本教材ではコードの簡潔さのために `using namespace std;` を使う場合がありますが、実務では注意してください。

### よくある間違い

```cpp
// NG: using namespace をヘッダファイルに書く
// → ヘッダを include した全ファイルに影響が及ぶため危険
// ヘッダファイルでは必ず std:: を明示しましょう
```

## 1.4 基本データ型（data type）

C++には様々なデータ型があります。まず最も基本的なものを学びましょう。

### 主要なデータ型一覧

```
┌──────────────┬──────────────────┬─────────────────────────────────┐
│  型名        │  サイズ(典型値)  │  説明                           │
├──────────────┼──────────────────┼─────────────────────────────────┤
│  int         │  4バイト         │  整数（-21億 〜 +21億）         │
│  double      │  8バイト         │  小数（倍精度浮動小数点数）     │
│  char        │  1バイト         │  1文字（'A', '0' など）         │
│  bool        │  1バイト         │  真偽値（true / false）         │
│  std::string │  可変            │  文字列（"Hello" など）         │
├──────────────┼──────────────────┼─────────────────────────────────┤
│  long long   │  8バイト         │  大きな整数（約±920京）         │
│  float       │  4バイト         │  小数（単精度浮動小数点数）     │
│  unsigned int│  4バイト         │  符号なし整数（0 〜 約42億）    │
└──────────────┴──────────────────┴─────────────────────────────────┘
```

### コード例：基本データ型を使う

```cpp
#include <iostream>
#include <string>

int main() {
    // 整数型（integer）
    int age = 25;
    long long world_population = 8000000000LL;

    // 浮動小数点型（floating point）
    double pi = 3.14159265358979;
    float temperature = 36.5f;  // float にはサフィックス f を付ける

    // 文字型（character）
    char grade = 'A';  // シングルクォートで囲む

    // 真偽値型（boolean）
    bool is_student = true;

    // 文字列型（string）
    std::string name = "太郎";  // ダブルクォートで囲む

    // 表示
    std::cout << "名前: " << name << std::endl;
    std::cout << "年齢: " << age << std::endl;
    std::cout << "世界人口: " << world_population << std::endl;
    std::cout << "円周率: " << pi << std::endl;
    std::cout << "体温: " << temperature << std::endl;
    std::cout << "成績: " << grade << std::endl;
    std::cout << "学生か: " << std::boolalpha << is_student << std::endl;

    return 0;
}
```

出力：
```
名前: 太郎
年齢: 25
世界人口: 8000000000
円周率: 3.14159
体温: 36.5
成績: A
学生か: true
```

### ポイントまとめ
- 用途に合ったデータ型を選びましょう
- `int` は日常的な整数に、`double` は小数に使います
- `std::string` を使うには `#include <string>` が必要です
- `char` はシングルクォート `' '`、`std::string` はダブルクォート `" "` で囲みます

## 1.5 変数（variable）の宣言と初期化

### 変数の宣言方法

C++では変数を使う前に **宣言（declaration）** が必要です。

```cpp
// 基本の形: 型名 変数名;
int count;          // 宣言のみ（値は不定 — 危険！）
int count = 0;      // 宣言と同時に初期化（推奨）
```

### 初期化の方法（C++11以降）

C++には複数の初期化方法があります。

```cpp
#include <iostream>
#include <string>

int main() {
    // ① コピー初期化（copy initialization）
    int a = 10;

    // ② 直接初期化（direct initialization）
    int b(20);

    // ③ 統一初期化 / リスト初期化（uniform / list initialization）★推奨
    int c{30};

    // ④ 代入初期化（文字列など）
    std::string greeting = "こんにちは";

    // ⑤ リスト初期化（文字列）
    std::string farewell{"さようなら"};

    std::cout << a << ", " << b << ", " << c << std::endl;
    std::cout << greeting << " / " << farewell << std::endl;

    return 0;
}
```

出力：
```
10, 20, 30
こんにちは / さようなら
```

### リスト初期化 `{}` が推奨される理由

リスト初期化 `{}` は **縮小変換（narrowing conversion）** を防いでくれます。

```cpp
int x = 3.14;   // OK（警告は出るが通る） → x は 3 になる（小数部が消える！）
// int y{3.14};  // コンパイルエラー！ 縮小変換を検出してくれる
```

### よくある間違い

```cpp
// NG: 初期化せずに変数を使う
int value;
std::cout << value << std::endl;  // 未定義動作！何が表示されるか分からない

// OK: 必ず初期化する
int value = 0;
std::cout << value << std::endl;  // 0 と表示される
```

**ルール：変数は宣言と同時に初期化しましょう。**

## 1.6 const と auto

### const — 変更できない値

`const` を付けると、その変数の値を後から変更できなくなります。**定数（constant）** を表現するために使います。

```cpp
#include <iostream>

int main() {
    const double TAX_RATE = 0.10;  // 消費税率（変更不可）
    const int MAX_PLAYERS = 11;    // サッカーの1チーム人数

    int price = 1000;
    double total = price * (1.0 + TAX_RATE);

    std::cout << "税込価格: " << total << "円" << std::endl;

    // TAX_RATE = 0.08;  // コンパイルエラー！ const は変更できない

    return 0;
}
```

出力：
```
税込価格: 1100円
```

### auto — 型の自動推論

`auto` を使うと、コンパイラが右辺の値から型を **自動推論（type deduction）** してくれます。

```cpp
#include <iostream>
#include <string>

int main() {
    auto count = 42;           // int と推論される
    auto pi = 3.14159;         // double と推論される
    auto letter = 'X';         // char と推論される
    auto flag = true;          // bool と推論される
    auto message = std::string("Hello");  // std::string と推論される

    std::cout << "count の型サイズ: " << sizeof(count) << " バイト" << std::endl;
    std::cout << "pi の型サイズ: " << sizeof(pi) << " バイト" << std::endl;

    return 0;
}
```

出力：
```
count の型サイズ: 4 バイト
pi の型サイズ: 8 バイト
```

### よくある間違い

```cpp
// NG: auto を初期化なしで使う
// auto x;  // コンパイルエラー！右辺がないと型を推論できない

// NG: auto で意図しない型になる
auto price = 100;       // int（整数）になる
auto tax = price * 0.1; // double（小数）になる ← 意図通りか確認！
```

### ポイントまとめ
- `const` は変更されるべきでない値に必ず付けましょう
- `auto` は型が明らかな場面で使うと、コードが簡潔になります
- `auto` を使いすぎると可読性が下がるので、バランスが大切です

## 1.7 算術演算子（arithmetic operator）

### 基本的な演算子

```
┌──────────┬────────┬───────────────────────┐
│ 演算子   │ 意味   │ 例                    │
├──────────┼────────┼───────────────────────┤
│    +     │ 加算   │ 5 + 3  → 8            │
│    -     │ 減算   │ 5 - 3  → 2            │
│    *     │ 乗算   │ 5 * 3  → 15           │
│    /     │ 除算   │ 5 / 3  → 1（整数除算）│
│    %     │ 剰余   │ 5 % 3  → 2            │
└──────────┴────────┴───────────────────────┘
```

### コード例：算術演算

```cpp
#include <iostream>

int main() {
    int a = 17;
    int b = 5;

    std::cout << "a + b = " << (a + b) << std::endl;  // 22
    std::cout << "a - b = " << (a - b) << std::endl;  // 12
    std::cout << "a * b = " << (a * b) << std::endl;  // 85
    std::cout << "a / b = " << (a / b) << std::endl;  // 3（整数同士の除算は切り捨て）
    std::cout << "a % b = " << (a % b) << std::endl;  // 2（余り）

    // 小数の除算
    double x = 17.0;
    double y = 5.0;
    std::cout << "x / y = " << (x / y) << std::endl;  // 3.4

    return 0;
}
```

出力：
```
a + b = 22
a - b = 12
a * b = 85
a / b = 3
a % b = 2
x / y = 3.4
```

### 複合代入演算子とインクリメント

```cpp
#include <iostream>

int main() {
    int score = 100;

    score += 20;   // score = score + 20  → 120
    score -= 5;    // score = score - 5   → 115
    score *= 2;    // score = score * 2   → 230
    score /= 10;   // score = score / 10  → 23
    score %= 7;    // score = score % 7   → 2

    std::cout << "score: " << score << std::endl;

    int count = 0;
    count++;       // 後置インクリメント → count は 1
    ++count;       // 前置インクリメント → count は 2
    std::cout << "count: " << count << std::endl;

    return 0;
}
```

出力：
```
score: 2
count: 2
```

### よくある間違い

```cpp
// NG: 整数同士の除算で小数を期待する
int a = 7, b = 2;
double result = a / b;  // 3.0（3.5 にはならない！）
// → 整数同士の除算は整数の結果になり、それが double に変換される

// OK: 片方を double にする
double result = static_cast<double>(a) / b;  // 3.5
```

## 1.8 型変換（type conversion）

### 暗黙の型変換（implicit conversion）

C++は異なる型を混ぜた演算で、自動的に型変換を行います。

```
変換の方向（データが失われにくい方向に変換される）:

  int → long → long long → float → double → long double
         ↑
        char
```

```cpp
#include <iostream>

int main() {
    int i = 42;
    double d = 3.14;

    // int が自動的に double に変換される
    double result = i + d;  // 42.0 + 3.14 = 45.14
    std::cout << result << std::endl;

    // double が int に代入される → 小数部が失われる（警告あり）
    int truncated = d;  // 3（小数部が切り捨て）
    std::cout << truncated << std::endl;

    return 0;
}
```

### 明示的な型変換 ― static_cast

安全に型変換を行うには `static_cast<変換先の型>(値)` を使います。

```cpp
#include <iostream>

int main() {
    int total_score = 273;
    int num_subjects = 5;

    // 整数同士の除算 → 小数部が消える
    std::cout << "整数除算: " << total_score / num_subjects << std::endl;  // 54

    // static_cast で明示的に double に変換してから除算
    double average = static_cast<double>(total_score) / num_subjects;
    std::cout << "平均点: " << average << std::endl;  // 54.6

    // char と int の変換
    char ch = 'A';
    int ascii_value = static_cast<int>(ch);
    std::cout << ch << " の ASCII コード: " << ascii_value << std::endl;  // 65

    return 0;
}
```

出力：
```
整数除算: 54
平均点: 54.6
A の ASCII コード: 65
```

### よくある間違い

```cpp
// NG: Cスタイルのキャスト（非推奨）
double d = (double)total_score / num_subjects;

// OK: static_cast を使う（意図が明確で安全）
double d = static_cast<double>(total_score) / num_subjects;
```

### ポイントまとめ
- 整数同士の除算は整数結果になることに注意しましょう
- 型変換には `static_cast` を使いましょう（Cスタイルキャストは避ける）
- 暗黙の型変換では情報が失われる場合があるので注意しましょう

## 1.9 標準入出力（cin / cout）

### 出力 ― std::cout

```cpp
#include <iostream>
#include <string>

int main() {
    std::string name = "花子";
    int age = 20;

    // << を連続して使える（チェーン）
    std::cout << "名前: " << name << "、年齢: " << age << "歳" << std::endl;

    // 改行の方法
    std::cout << "1行目" << std::endl;  // endl: 改行 + flush
    std::cout << "2行目\n";             // \n: 改行のみ（高速）
    std::cout << "3行目" << '\n';       // これも OK

    return 0;
}
```

### 入力 ― std::cin

```cpp
#include <iostream>
#include <string>

int main() {
    // 整数の入力
    std::cout << "年齢を入力してください: ";
    int age;
    std::cin >> age;

    // 文字列の入力（スペースまで）
    std::cout << "名前を入力してください: ";
    std::string name;
    std::cin >> name;

    std::cout << name << "さん（" << age << "歳）、ようこそ！" << std::endl;

    return 0;
}
```

実行例：
```
年齢を入力してください: 25
名前を入力してください: 太郎
太郎さん（25歳）、ようこそ！
```

### スペースを含む文字列の入力 ― getline

`std::cin >>` はスペースで区切られるため、スペースを含む文字列を読むには `std::getline` を使います。

```cpp
#include <iostream>
#include <string>

int main() {
    std::cout << "フルネームを入力してください: ";
    std::string full_name;
    std::getline(std::cin, full_name);

    std::cout << "こんにちは、" << full_name << "さん！" << std::endl;

    return 0;
}
```

実行例：
```
フルネームを入力してください: 山田 太郎
こんにちは、山田 太郎さん！
```

### よくある間違い

```cpp
// NG: cin >> の後に getline を使うと、改行文字が残って空読みされる
int age;
std::cin >> age;

std::string name;
std::getline(std::cin, name);  // 改行文字を読んで空文字列になる！

// OK: cin.ignore() で改行文字を捨てる
int age;
std::cin >> age;
std::cin.ignore();  // バッファに残った改行を読み飛ばす

std::string name;
std::getline(std::cin, name);  // 正常に入力を受け取れる
```

### ポイントまとめ
- `std::cout <<` で出力、`std::cin >>` で入力します
- スペースを含む入力には `std::getline` を使います
- `cin >>` と `getline` を混在させる場合は `cin.ignore()` を入れましょう
- 大量出力時は `std::endl` より `'\n'` の方が高速です

## 1.10 実践例：BMI計算プログラム

学んだ内容を組み合わせた実践的なプログラムです。

```cpp
// bmi_calculator.cpp — BMI計算プログラム
// コンパイル: g++ -std=c++17 -o bmi bmi_calculator.cpp
// 実行: ./bmi

#include <iostream>
#include <string>
#include <iomanip>  // std::fixed, std::setprecision

int main() {
    std::cout << "===== BMI計算プログラム =====" << std::endl;
    std::cout << std::endl;

    // 名前の入力
    std::cout << "名前を入力してください: ";
    std::string name;
    std::getline(std::cin, name);

    // 身長の入力（cm）
    std::cout << "身長（cm）を入力してください: ";
    double height_cm;
    std::cin >> height_cm;

    // 体重の入力（kg）
    std::cout << "体重（kg）を入力してください: ";
    double weight_kg;
    std::cin >> weight_kg;

    // BMI の計算: 体重(kg) / 身長(m)^2
    const double height_m = height_cm / 100.0;
    const double bmi = weight_kg / (height_m * height_m);

    // 判定
    std::string category;
    if (bmi < 18.5) {
        category = "低体重";
    } else if (bmi < 25.0) {
        category = "普通体重";
    } else if (bmi < 30.0) {
        category = "肥満（1度）";
    } else {
        category = "肥満（2度以上）";
    }

    // 結果表示
    std::cout << std::endl;
    std::cout << "===== 結果 =====" << std::endl;
    std::cout << "名前: " << name << std::endl;
    std::cout << "身長: " << height_cm << " cm" << std::endl;
    std::cout << "体重: " << weight_kg << " kg" << std::endl;
    std::cout << std::fixed << std::setprecision(1);
    std::cout << "BMI: " << bmi << std::endl;
    std::cout << "判定: " << category << std::endl;

    return 0;
}
```

実行例：
```
===== BMI計算プログラム =====

名前を入力してください: 山田太郎
身長（cm）を入力してください: 170
体重（kg）を入力してください: 65

===== 結果 =====
名前: 山田太郎
身長: 170.0 cm
体重: 65.0 kg
BMI: 22.5
判定: 普通体重
```
