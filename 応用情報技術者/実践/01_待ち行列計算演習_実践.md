---

## 課題の要件

1. ユーザーから平均到着率（λ）と平均サービス率（μ）を入力で受け取る
2. λ ≧ μ の場合は「系が安定しない」旨のエラーメッセージを表示する
3. M/M/1モデルの各指標（ρ, Wq, Ws, Lq, Ls）を計算する
4. 利用率に応じたアドバイスメッセージを表示する（70%未満: 余裕あり / 70〜90%: 注意 / 90%以上: 危険）

---

## 背景知識の確認

M/M/1モデルの公式を整理しておきましょう。

```
利用率:            ρ = λ / μ
平均待ち時間:      Wq = ρ / (μ - λ)
平均系内時間:      Ws = 1 / (μ - λ)
平均待ち行列長:    Lq = ρ² / (1 - ρ)
平均系内人数:      Ls = ρ / (1 - ρ)
```

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```python
# M/M/1 待ち行列シミュレータ
# 学べる内容：待ち行列理論の基本公式、条件分岐、数値計算
# 実行方法：python mm1_queue.py

print("===== M/M/1 待ち行列シミュレータ =====\n")

# --- 入力 ---
lam = float(input("平均到着率λ（件/秒）を入力: "))
mu = float(input("平均サービス率μ（件/秒）を入力: "))

# --- 安定条件チェック ---
if lam >= mu:
    print("\nエラー: λ ≧ μ のため系は安定しません。")
    print("μ > λ となるようサービス率を上げるか、到着率を下げてください。")
else:
    # --- 計算 ---
    rho = lam / mu                    # 利用率
    wq = rho / (mu - lam)            # 平均待ち時間
    ws = 1 / (mu - lam)              # 平均系内時間
    lq = rho ** 2 / (1 - rho)        # 平均待ち行列長
    ls = rho / (1 - rho)             # 平均系内人数

    # --- 結果表示 ---
    print("\n--- 計算結果 ---")
    print(f"利用率 ρ         = {rho:.3f}")
    print(f"平均待ち時間 Wq  = {wq:.3f} 秒")
    print(f"平均系内時間 Ws  = {ws:.3f} 秒")
    print(f"平均待ち行列長 Lq = {lq:.3f} 件")
    print(f"平均系内人数 Ls  = {ls:.3f} 件")

    # --- 判定 ---
    if rho < 0.7:
        print("\n[判定] 余裕があります。現状維持で問題ありません。")
    elif rho < 0.9:
        print("\n[判定] 利用率が高めです。負荷の推移を注視してください。")
    else:
        print("\n[判定] 利用率が90%を超えています。サーバ増強を検討してください。")
```

</details>

<details>
<summary>解答例（改良版 ─ 複数サーバ比較機能付き）</summary>

サービス率を段階的に変えたときの性能変化を一覧表示する改良版です。

```python
# M/M/1 待ち行列シミュレータ（改良版）
# 学べる内容：待ち行列理論、サーバ増強のシミュレーション
# 実行方法：python mm1_queue_advanced.py

def calc_mm1(lam, mu):
    """M/M/1モデルの各指標を計算して辞書で返す"""
    if lam >= mu:
        return None
    rho = lam / mu
    return {
        "rho": rho,
        "wq": rho / (mu - lam),
        "ws": 1 / (mu - lam),
        "lq": rho ** 2 / (1 - rho),
        "ls": rho / (1 - rho),
    }

def judge(rho):
    """利用率に応じた判定を返す"""
    if rho < 0.7:
        return "余裕"
    elif rho < 0.9:
        return "注意"
    else:
        return "危険"

# --- メイン ---
print("===== M/M/1 待ち行列シミュレータ（改良版） =====\n")

lam = float(input("平均到着率λ（件/秒）を入力: "))
mu = float(input("平均サービス率μ（件/秒）を入力: "))

result = calc_mm1(lam, mu)
if result is None:
    print("\nエラー: λ ≧ μ のため系は安定しません。")
else:
    print(f"\n--- 現在の性能（μ={mu}） ---")
    print(f"利用率 ρ = {result['rho']:.3f}  [{judge(result['rho'])}]")
    print(f"平均系内時間 Ws = {result['ws']:.3f} 秒")

    # μを20%ずつ増やしたときの比較
    print("\n--- サービス率を変更したときの比較 ---")
    print(f"{'μ':>6} | {'ρ':>6} | {'Ws(秒)':>8} | {'Lq(件)':>8} | 判定")
    print("-" * 50)
    for factor in [1.0, 1.2, 1.5, 2.0]:
        new_mu = mu * factor
        r = calc_mm1(lam, new_mu)
        if r:
            print(f"{new_mu:6.1f} | {r['rho']:6.3f} | {r['ws']:8.3f} | {r['lq']:8.3f} | {judge(r['rho'])}")

**初心者向けとの違い:**
- 関数に分離して計算ロジックを再利用可能にしている
- サービス率を変えたときの比較表を自動生成し、増強の判断材料を提供
- 辞書で計算結果を返すことで、データの取り回しが容易になっている

</details>
