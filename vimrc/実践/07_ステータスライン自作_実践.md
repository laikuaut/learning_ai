# 実践課題07：ステータスライン自作 ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第2章（表示設定）、第6章（autocmd）、第7章（関数とカスタムコマンド）
> **課題の種類**: ミニプロジェクト
> **学習目標**: Vim Script の関数と `statusline` の書式指定を組み合わせて、情報量が豊富で視認性の高いステータスラインを自作する

---

## 完成イメージ

```
ノーマルモード:
┌──────────────────────────────────────────────────────────┐
│  NORMAL │ example.py [+] │ python │ utf-8 │  42/150 : 15 │
└──────────────────────────────────────────────────────────┘
    ↑          ↑        ↑       ↑       ↑        ↑      ↑
  モード    ファイル名  変更  種別  文字コード  行/総行 列

挿入モード（色が変わる）:
┌──────────────────────────────────────────────────────────┐
│ INSERT │ example.py [+] │ python │ utf-8 │  42/150 : 15  │
└──────────────────────────────────────────────────────────┘

ビジュアルモード:
┌──────────────────────────────────────────────────────────┐
│ VISUAL │ example.py [+] │ python │ utf-8 │  42/150 : 15  │
└──────────────────────────────────────────────────────────┘
```

---

## 課題の要件

1. 現在のモード名をステータスラインに表示する
2. モードに応じてステータスラインの色を変える
3. ファイル名、変更フラグ、ファイルタイプを表示する
4. エンコーディングと改行コードを表示する
5. カーソル位置情報（行番号、総行数、列番号）を表示する
6. 関数を使って表示内容を組み立てる

---

## ステップガイド

<details>
<summary>ステップ1：モード名を返す関数</summary>

```vim
function! GetMode() abort
    let l:mode = mode()
    let l:mode_map = {
        \ 'n':      'NORMAL',
        \ 'i':      'INSERT',
        \ 'v':      'VISUAL',
        \ 'V':      'V-LINE',
        \ "\<C-v>": 'V-BLOCK',
        \ 'R':      'REPLACE',
        \ 'c':      'COMMAND',
        \ 't':      'TERMINAL',
        \ }
    return get(l:mode_map, l:mode, l:mode)
endfunction
```

`mode()` 関数は現在のモードを1文字で返します。それを読みやすい名前に変換しています。
`get()` は辞書にキーが存在しない場合に第3引数（デフォルト値）を返します。

</details>

<details>
<summary>ステップ2：モードに応じた色変更</summary>

`autocmd` の `InsertEnter` / `InsertLeave` イベントでステータスラインの色を変えます。

```vim
" ステータスラインの色定義
function! UpdateStatuslineColor() abort
    let l:mode = mode()
    if l:mode ==# 'i'
        highlight StatusLine ctermfg=white ctermbg=blue
    elseif l:mode ==# 'v' || l:mode ==# 'V' || l:mode ==# "\<C-v>"
        highlight StatusLine ctermfg=white ctermbg=magenta
    elseif l:mode ==# 'R'
        highlight StatusLine ctermfg=white ctermbg=red
    else
        highlight StatusLine ctermfg=white ctermbg=darkgreen
    endif
endfunction

augroup StatuslineColor
    autocmd!
    autocmd InsertEnter * call UpdateStatuslineColor()
    autocmd InsertLeave * call UpdateStatuslineColor()
    autocmd CursorMoved * call UpdateStatuslineColor()
augroup END
```

</details>

<details>
<summary>ステップ3：ステータスラインの組み立て</summary>

```vim
set laststatus=2

" ステータスラインの内容
set statusline=
set statusline+=\ %{GetMode()}         " モード名
set statusline+=\ │                     " セパレータ
set statusline+=\ %f                    " ファイル名
set statusline+=\ %m                    " 変更フラグ [+]
set statusline+=\ │                     " セパレータ
set statusline+=\ %Y                    " ファイルタイプ
set statusline+=\ │                     " セパレータ
set statusline+=\ %{&fileencoding}      " エンコーディング
set statusline+=%=                      " 右寄せ開始
set statusline+=\ %l/%L                 " 現在行/総行数
set statusline+=\ :\ %c                " 列番号
set statusline+=\ 
```

`│` (Box Drawing Light Vertical, U+2502) をセパレータに使うと見た目が整います。

</details>

<details>
<summary>ステップ4：Git情報の表示（発展）</summary>

Gitのブランチ名を表示する関数を追加します。

```vim
function! GetGitBranch() abort
    let l:branch = system('git rev-parse --abbrev-ref HEAD 2>/dev/null')
    let l:branch = substitute(l:branch, '\n', '', 'g')
    if l:branch ==# ''
        return ''
    endif
    return ' ' . l:branch . ' │'
endfunction
```

**注意**: `system()` はファイルを開くたびに外部コマンドを実行するので、頻繁に呼ぶとパフォーマンスに影響します。実用的にはタイマーでキャッシュする工夫が必要です。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ===========================================
" カスタムステータスライン
" ===========================================

set nocompatible
set laststatus=2

" --- モード名を返す関数 ---
function! GetMode() abort
    let l:mode = mode()
    let l:mode_map = {
        \ 'n':      'NORMAL',
        \ 'i':      'INSERT',
        \ 'v':      'VISUAL',
        \ 'V':      'V-LINE',
        \ "\<C-v>": 'V-BLOCK',
        \ 'R':      'REPLACE',
        \ 'c':      'COMMAND',
        \ }
    return get(l:mode_map, l:mode, l:mode)
endfunction

" --- ステータスラインの組み立て ---
set statusline=
set statusline+=\ %{GetMode()}         " モード名
set statusline+=\ │                     " セパレータ
set statusline+=\ %f                    " ファイル名
set statusline+=\ %m                    " 変更フラグ
set statusline+=\ │                     " セパレータ
set statusline+=\ %Y                    " ファイルタイプ
set statusline+=\ │                     " セパレータ
set statusline+=\ %{&fileencoding}      " エンコーディング
set statusline+=%=                      " 右寄せ
set statusline+=\ %l/%L                 " 行番号/総行数
set statusline+=\ :\ %c                " 列番号
set statusline+=\ 

" --- モード切替時の色変更 ---
augroup StatuslineColor
    autocmd!
    autocmd InsertEnter * highlight StatusLine ctermbg=blue ctermfg=white

highlight StatusLine ctermbg=darkgreen ctermfg=white
```

</details>

<details>
<summary>解答例（改良版：ハイライトグループとGit情報）</summary>

```vim
" ===========================================
" カスタムステータスライン - 改良版
" ===========================================

set nocompatible
set laststatus=2
set noshowmode          " モード表示はステータスラインに任せるので消す

" ------------------------------------------
" ハイライトグループの定義
" ------------------------------------------
highlight StatusMode     ctermbg=green  ctermfg=black cterm=bold
highlight StatusFile     ctermbg=237    ctermfg=white
highlight StatusType     ctermbg=235    ctermfg=grey
highlight StatusPos      ctermbg=240    ctermfg=white

highlight StatusModeI    ctermbg=blue   ctermfg=white cterm=bold
highlight StatusModeV    ctermbg=magenta ctermfg=white cterm=bold
highlight StatusModeR    ctermbg=red    ctermfg=white cterm=bold

" ------------------------------------------
" モード関連の関数
" ------------------------------------------
function! GetMode() abort
    let l:mode = mode()
    let l:mode_map = {
        \ 'n':      'NORMAL',
        \ 'i':      'INSERT',
        \ 'v':      'VISUAL',
        \ 'V':      'V-LINE',
        \ "\<C-v>": 'V-BLOCK',
        \ 'R':      'REPLACE',
        \ 'c':      'COMMAND',
        \ 't':      'TERMINAL',
        \ }
    return get(l:mode_map, l:mode, l:mode)
endfunction

function! UpdateModeHighlight() abort
    let l:mode = mode()
    if l:mode ==# 'i'
        highlight! link StatusMode StatusModeI
    elseif l:mode =~# '[vV\x16]'
        highlight! link StatusMode StatusModeV
    elseif l:mode ==# 'R'
        highlight! link StatusMode StatusModeR
    else
        highlight StatusMode ctermbg=green ctermfg=black cterm=bold
    endif
endfunction

" ------------------------------------------
" Git ブランチ名（キャッシュ付き）
" ------------------------------------------
let s:git_branch_cache = ''
let s:git_branch_timer = 0

function! GetGitBranch() abort
    return s:git_branch_cache
endfunction

function! RefreshGitBranch(timer) abort
    let l:branch = system('git rev-parse --abbrev-ref HEAD 2>/dev/null')
    let s:git_branch_cache = substitute(l:branch, '\n', '', 'g')
endfunction

" 5秒ごとにブランチ名を更新
if has('timers')
    let s:git_branch_timer = timer_start(5000, 'RefreshGitBranch', {'repeat': -1})
    call RefreshGitBranch(0)
endif

" ------------------------------------------
" ファイルサイズを返す関数
" ------------------------------------------
function! GetFileSize() abort
    let l:size = getfsize(expand('%'))
    if l:size < 0
        return ''
    elseif l:size < 1024
        return l:size . 'B'
    elseif l:size < 1048576
        return printf('%.1fK', l:size / 1024.0)
    else
        return printf('%.1fM', l:size / 1048576.0)
    endif
endfunction

" ------------------------------------------
" ステータスラインの組み立て
" ------------------------------------------
set statusline=
set statusline+=%#StatusMode#          " モード用の色
set statusline+=\ %{GetMode()}\ 
set statusline+=%#StatusFile#          " ファイル用の色
set statusline+=\ %f\ %m%r            " ファイル名 + 変更/読取専用フラグ
set statusline+=%{GetGitBranch()!=''?'\ │\ '.GetGitBranch():''}
set statusline+=%#StatusType#          " タイプ用の色
set statusline+=\ %Y                   " ファイルタイプ
set statusline+=\ │\ %{&fileencoding}  " エンコーディング
set statusline+=\ │\ %{&fileformat}    " 改行コード（unix/dos）
set statusline+=%=                     " 右寄せ
set statusline+=%#StatusPos#           " 位置用の色
set statusline+=\ %{GetFileSize()}     " ファイルサイズ
set statusline+=\ │\ %l/%L:%c         " 行/総行:列
set statusline+=\ │\ %p%%\            " 位置（%）

" ------------------------------------------
" autocmd
" ------------------------------------------
augroup StatuslineUpdate
    autocmd!
    autocmd InsertEnter,InsertLeave * call UpdateModeHighlight()
    " バッファ切替時にGit情報を更新
    autocmd BufEnter * call RefreshGitBranch(0)
augroup END

" 初期化
call UpdateModeHighlight()
```

**初心者向けとの違い**:

- `%#HighlightGroup#` でセクションごとに色を変えて視認性を向上
- `highlight! link` でモード別の色を切り替え
- Git ブランチ名をタイマーでキャッシュ（パフォーマンス対策）
- ファイルサイズ表示関数
- 改行コード（`fileformat`）の表示
- `noshowmode` で組み込みのモード表示を非表示に

</details>

## よくある間違い

| ミス | 正しい書き方 | 説明 |
|------|-------------|------|
| `statusline` に `%{}` なしで関数を書く | `%{GetMode()}` | 関数呼び出しは `%{}` で囲む |
| `mode()` の戻り値を `==` で比較 | `==#` で厳密比較 | `==` は `ignorecase` に影響される |
| `system()` を `statusline` 内で毎回呼ぶ | キャッシュする | 描画のたびに外部コマンド実行で遅くなる |
| `highlight` の色名を間違える | `:highlight` で確認 | 使える色名はターミナルに依存する |
