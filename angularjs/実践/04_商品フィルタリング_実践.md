# 実践課題04：商品フィルタリング ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第3章（ディレクティブ）、第4章（フィルターとサービス）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 組み込みフィルター（`filter`、`orderBy`、`currency`）とカスタムフィルターを活用して、商品データの検索・絞り込み・並べ替えを実装する

---

## 完成イメージ

キーワード検索・カテゴリ絞り込み・価格帯フィルタ・並べ替えが連動する商品カタログです。

```
┌─────────────────────────────────────────┐
│  商品カタログ                             │
│                                         │
│  検索: [キーワードを入力...        ]      │
│  カテゴリ: [すべて] [食品] [飲料] [日用品]  │
│  価格帯:  [すべて] [〜500] [500〜1000] [1000〜] │
│  並べ替え: [名前▲] [名前▼] [安い順] [高い順] │
│                                         │
│  10件中 4件表示                           │
│                                         │
│  ┌───────────┐ ┌───────────┐            │
│  │ 有機りんご  │ │ 国産みかん  │            │
│  │ 食品       │ │ 食品       │            │
│  │ ¥380       │ │ ¥450      │            │
│  │ ★★★★☆     │ │ ★★★☆☆     │            │
│  └───────────┘ └───────────┘            │
└─────────────────────────────────────────┘
```

---

## 課題の要件

1. 商品データ（名前・カテゴリ・価格・評価）を配列で管理する（10件以上）
2. `filter` フィルターでキーワード検索を実装する
3. カテゴリボタンで絞り込む
4. カスタムフィルターで価格帯フィルタリングを実装する
5. `orderBy` フィルターで並べ替え機能を追加する
6. `currency` フィルターで価格を通貨表示にする
7. 該当件数を動的に表示する

---

## ステップガイド

<details>
<summary>ステップ1：商品データと組み込みフィルターを使う</summary>

AngularJS の `filter` フィルターはテンプレート内でそのまま使えます。

```html
<div ng-repeat="product in products | filter:searchText | orderBy:sortKey">
  {{ product.name }} - {{ product.price | currency:'¥':0 }}
</div>
```

</details>

<details>
<summary>ステップ2：カテゴリフィルタを追加する</summary>

`filter` フィルターにオブジェクトを渡すことで、特定のプロパティで絞り込めます。

```html
<div ng-repeat="product in products | filter:searchText | filter:categoryFilter | orderBy:sortKey">
```

```javascript
$scope.setCategory = function(cat) {
  $scope.selectedCategory = cat;
  $scope.categoryFilter = (cat === 'all') ? {} : { category: cat };
};
```

</details>

<details>
<summary>ステップ3：カスタムフィルターを作る</summary>

価格帯フィルタリングにはカスタムフィルターが便利です。

```javascript
angular.module('shopApp')
  .filter('priceRange', function() {
    return function(products, range) {
      if (!range || range === 'all') return products;
      return products.filter(function(p) {
        if (range === 'low') return p.price < 500;
        if (range === 'mid') return p.price >= 500 && p.price < 1000;
        if (range === 'high') return p.price >= 1000;
        return true;
      });
    };
  });
```

</details>

          if (range === 'low') return p.price < 500;
          if (range === 'mid') return p.price >= 500 && p.price < 1000;
          return p.price >= 1000;
        });
      };

    // カスタムフィルター：カテゴリ
    app.filter('categoryFilter', function() {
      return function(items, category) {
        if (!items || !category || category === 'all') return items;

- `ng-bind-html` でHTMLを安全にレンダリング
- カテゴリごとに色分けしたラベルを表示
- ホバーエフェクトで商品カードにインタラクティブ性を追加
