---

## 課題の要件

1. 商品（名前・個数・単価）をリストで管理する
2. メニューをループで繰り返し表示し、ユーザーの操作を受け付ける
3. 以下の5つの操作を実装する
   - **追加**: 商品名・個数・単価を入力してリストに追加
   - **一覧表示**: 全商品を番号付きで表示（小計も表示）
   - **削除**: 番号を指定して商品を削除
   - **合計金額**: 全商品の合計金額を計算して表示
   - **終了**: プログラムを終了する
4. 不正な番号の入力にはエラーメッセージを表示する

<summary>ステップ2：メニューのループを作る</summary>

`while True` で無限ループを作り、「終了」が選ばれたら `break` します。

```python
while True:
    print("\n===== 買い物リスト =====")
    print("1. 追加")
    # ...
    choice = input("操作を選んでください: ")

    if choice == "5":
        print("お買い物リストを終了します。")
        break
    elif choice == "1":
        # 追加処理
        pass
```

</details>

追加・表示・削除・合計の各処理を `if-elif` の中に書きます。

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```python
# 買い物リスト管理
# 学べる内容：リスト操作、辞書、whileループ、対話型プログラム

shopping_list = []

while True:
    print("\n===== 買い物リスト =====")
    print("1. 追加")
    print("2. 一覧表示")
    print("3. 削除")
    print("4. 合計金額")
    print("5. 終了")

    choice = input("\n操作を選んでください: ")

    if choice == "1":
        # 追加
        name = input("商品名: ")
        quantity = int(input("個数: "))
        price = int(input("単価: "))
        item = {"name": name, "quantity": quantity, "price": price}
        shopping_list.append(item)
        print(f"「{name}」を追加しました。")

    elif choice == "2":
        # 一覧表示
        if len(shopping_list) == 0:
            print("\nリストは空です。")
        else:
            print("\n--- 買い物リスト ---")
            total = 0
            for i in range(len(shopping_list)):
                item = shopping_list[i]
                subtotal = item["quantity"] * item["price"]
                total = total + subtotal
                print(f" {i + 1}. {item['name']:<6} x{item['quantity']}  @{item['price']}円  = {subtotal}円")
            print("-" * 30)
            print(f"合計: {total}円")

    elif choice == "3":
        # 削除
        if len(shopping_list) == 0:
            print("\nリストは空です。")
        else:
            num = int(input("削除する番号: "))
            if 1 <= num <= len(shopping_list):
                removed = shopping_list.pop(num - 1)
                print(f"「{removed['name']}」を削除しました。")
            else:
                print("無効な番号です。")

    elif choice == "4":
        # 合計金額
        total = 0
        for item in shopping_list:
            total = total + item["quantity"] * item["price"]
        print(f"\n合計金額: {total}円")

    elif choice == "5":
        print("お買い物リストを終了します。")
        break

    else:
        print("1〜5の番号を入力してください。")
```

</details>

<details>
<summary>解答例（改良版 ─ 関数で整理）</summary>

```python
# 買い物リスト管理（改良版）
# 各操作を関数に分離して見通しを良くしたバージョン

def add_item(shopping_list):
    """商品を追加する"""
    name = input("商品名: ")
    quantity = int(input("個数: "))
    price = int(input("単価: "))
    shopping_list.append({"name": name, "quantity": quantity, "price": price})
    print(f"「{name}」を追加しました。")


def show_list(shopping_list):
    """一覧を表示する"""
    if not shopping_list:
        print("\nリストは空です。")
        return

    print("\n--- 買い物リスト ---")
    total = 0
    for i, item in enumerate(shopping_list, 1):
        subtotal = item["quantity"] * item["price"]
        total += subtotal
        print(f" {i}. {item['name']:<6} x{item['quantity']}  @{item['price']}円  = {subtotal}円")
    print("-" * 30)
    print(f"合計: {total}円")


def delete_item(shopping_list):
    """商品を削除する"""
    if not shopping_list:
        print("\nリストは空です。")
        return

    num = int(input("削除する番号: "))
    if 1 <= num <= len(shopping_list):
        removed = shopping_list.pop(num - 1)
        print(f"「{removed['name']}」を削除しました。")
    else:
        print("無効な番号です。")


def show_total(shopping_list):
    """合計金額を表示する"""
    total = sum(item["quantity"] * item["price"] for item in shopping_list)
    print(f"\n合計金額: {total}円")


# --- メイン処理 ---
shopping_list = []

menu = """
===== 買い物リスト =====
1. 追加
2. 一覧表示
3. 削除
4. 合計金額
5. 終了"""

while True:
    print(menu)
    choice = input("\n操作を選んでください: ")

    if choice == "1":
        add_item(shopping_list)
    elif choice == "2":
        show_list(shopping_list)
    elif choice == "3":
        delete_item(shopping_list)
    elif choice == "4":
        show_total(shopping_list)
    elif choice == "5":
        print("お買い物リストを終了します。")
        break
    else:
        print("1〜5の番号を入力してください。")
```

**初心者向けとの違い:**
- 各操作を関数に切り出し → メインループが見やすい
- `enumerate()` で番号付きループ
- `sum()` とジェネレータ式で合計計算を1行に
- `not shopping_list` による空チェック（Pythonicな書き方）

</details>
