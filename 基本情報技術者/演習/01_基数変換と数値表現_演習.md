# 第1章：基数変換と数値表現 演習問題

---

## 基本問題

### 問題1：基数変換（n進数 → 10進数）
(1101)₂ を10進数に変換してください。

<details>
<summary>ヒント</summary>
各桁に 2 の累乗の重みをかけて足し合わせます。右端が 2⁰ = 1 です。
</details>

<details>
<summary>解答</summary>

```
1×2³ + 1×2² + 0×2¹ + 1×2⁰
= 8 + 4 + 0 + 1
= 13
```
答え：**13**
</details>

---

### 問題2：基数変換（10進数 → 2進数）
25 を2進数に変換してください。

<details>
<summary>ヒント</summary>
25 を 2 で繰り返し割り、余りを下から上へ並べます。
</details>

<details>
<summary>解答</summary>

```
25 ÷ 2 = 12 余り 1  ↑
12 ÷ 2 = 6  余り 0  ↑
 6 ÷ 2 = 3  余り 0  ↑
 3 ÷ 2 = 1  余り 1  ↑
 1 ÷ 2 = 0  余り 1  ↑（読む方向）
```
答え：**(11001)₂**
</details>

---

### 問題3：基数変換（16進数 → 10進数）
(3A)₁₆ を10進数に変換してください。

<details>
<summary>ヒント</summary>
16進数の A = 10 です。各桁に 16 の累乗の重みをかけます。
</details>

<details>
<summary>解答</summary>

```
3×16¹ + A(=10)×16⁰
= 48 + 10
= 58
```
答え：**58**
</details>

### 問題4：2進数 ↔ 16進数
(10110110)₂ を16進数に変換してください。

<details>
<summary>ヒント</summary>
2進数を右端から4桁ずつ区切り、それぞれを16進数に変換します。
</details>

<details>
<summary>解答</summary>

```
1011 | 0110
 B   |  6
```
答え：**(B6)₁₆**
</details>

### 問題5：2の補数
8ビットの2の補数表現で -35 を表してください。

<details>
<summary>ヒント</summary>
まず 35 を8ビットの2進数にし、全ビットを反転（1の補数）してから 1 を足します。
</details>

<details>
<summary>解答</summary>

```
35 = (00100011)₂
ビット反転（1の補数）: (11011100)₂
+1: (11011101)₂
```
答え：**(11011101)₂**

検証：11011101 を10進数に戻すと、最上位ビットが1なので負の数。
反転: 00100010、+1: 00100011 = 35 → よって -35 で正しい。
</details>

## 応用問題

### 問題6：四択問題
16進数 (2B)₁₆ を10進数に変換した値として正しいものはどれですか。

ア. 39　　イ. 43　　ウ. 47　　エ. 53

<details>
<summary>解答</summary>

```
2×16¹ + B(=11)×16⁰
= 32 + 11
= 43
```
答え：**イ. 43**
</details>

### 問題7：小数の基数変換
0.75 を2進数に変換してください。

<details>
<summary>ヒント</summary>
小数部は 2 を繰り返しかけ、整数部を順に取り出します。
</details>

<details>
<summary>解答</summary>

```
0.75 × 2 = 1.5  → 整数部 1 ↓
0.5  × 2 = 1.0  → 整数部 1 ↓（読む方向）
```
答え：**(0.11)₂**

検証：1×2⁻¹ + 1×2⁻² = 0.5 + 0.25 = 0.75 ✓
</details>

### 問題8：2の補数の範囲
nビットの2の補数表現で表現できる整数の範囲を答えてください。また、16ビットの場合の具体的な範囲を求めてください。

<details>
<summary>解答</summary>

nビットの範囲：**-2^(n-1) 〜 2^(n-1) - 1**

16ビットの場合：
- 最小値：-2¹⁵ = -32768
- 最大値：2¹⁵ - 1 = 32767

答え：**-32768 〜 32767**
</details>

### 問題9：四択問題（浮動小数点数）
IEEE 754 単精度浮動小数点数の構成として正しいものはどれですか。

ア. 符号1ビット、指数部7ビット、仮数部24ビット
イ. 符号1ビット、指数部8ビット、仮数部23ビット
ウ. 符号1ビット、指数部10ビット、仮数部21ビット
エ. 符号2ビット、指数部8ビット、仮数部22ビット

<details>
<summary>解答</summary>

答え：**イ. 符号1ビット、指数部8ビット、仮数部23ビット**

単精度は合計32ビット = 1 + 8 + 23。バイアス値は127です。
</details>

### 問題10：シフト演算
8ビットの2進数 (00010100)₂ を左に3ビット論理シフトした結果と、その10進数の値を求めてください。

<details>
<summary>ヒント</summary>
左に1ビットシフトすると値が2倍になります。3ビットなら 2³ = 8 倍です。
</details>

<details>
<summary>解答</summary>

```
元の値: 00010100 = 20
左3ビットシフト: 10100000 = 160

検証: 20 × 2³ = 20 × 8 = 160 ✓
```
答え：**(10100000)₂ = 160**
</details>

## チャレンジ問題

### 問題11：誤差の識別
次の(1)〜(3)の現象に該当する誤差の名称を答えてください。

(1) 1.23457 - 1.23452 を有効数字6桁で計算したところ、結果の有効数字が1桁になった。
(2) 10¹⁰ + 0.5 を計算したところ、0.5の部分が反映されなかった。
(3) 円周率 π を 3.14 として計算に使用した。

<details>
<summary>解答</summary>

(1) **桁落ち** — ほぼ等しい値の引き算で有効数字が減少する現象。
(2) **情報落ち** — 絶対値が大きく異なる数の加減算で、小さい方の情報が失われる現象。
(3) **丸め誤差** — 無限に続く値を有限桁で打ち切る（四捨五入・切り捨て等）ことで生じる誤差。
</details>

### 問題12：総合問題
ある8ビット固定小数点コンピュータで、2の補数表現が (11100101)₂ であるとき、この値が表す10進数の値を求めてください。

<details>
<summary>ヒント</summary>
最上位ビットが1なので負の数です。2の補数から元の値を求めるには、全ビット反転して+1します。
</details>

<details>
<summary>解答</summary>

```
元の2進数: 11100101（最上位ビット=1 → 負の数）
ビット反転: 00011010
+1:        00011011 = 16 + 8 + 2 + 1 = 27
```
答え：**-27**
</details>

### 問題13：四択問題（総合）
2進数 (11010.101)₂ を10進数に変換した値として正しいものはどれですか。

ア. 25.5　　イ. 26.5　　ウ. 26.625　　エ. 27.625

<details>
<summary>解答</summary>

```
整数部: 1×2⁴ + 1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 16 + 8 + 0 + 2 + 0 = 26
小数部: 1×2⁻¹ + 0×2⁻² + 1×2⁻³ = 0.5 + 0 + 0.125 = 0.625

合計: 26 + 0.625 = 26.625
```
答え：**ウ. 26.625**
</details>
