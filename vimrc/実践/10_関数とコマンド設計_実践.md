## 完成イメージ

以下の4つのカスタム機能を vimrc に実装します。

```
┌──────────────────────────────────────────────────────────┐
│  実装する機能                                             │
├──────────────────────────────────────────────────────────┤
│                                                           │
│  1. :TrimWhitespace                                       │
│     → 末尾の空白を全行から削除（カーソル位置維持）       │
│                                                           │
│  2. :ToggleNumber                                         │
│     → 行番号の表示モードを切り替える                     │
│       （非表示 → 絶対 → 相対 → ハイブリッド → 非表示）   │
│                                                           │
│  3. :InsertDate                                           │
│     → カーソル位置に現在の日付を挿入する                 │
│                                                           │
│  4. :ScratchBuffer                                        │
│     → 一時的なメモ用バッファを新規作成する               │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

## 課題の要件

### 機能1: TrimWhitespace
1. 全行から末尾の空白文字（trailing whitespace）を削除する
2. 実行前のカーソル位置とスクロール位置を維持する
3. 検索履歴を汚さない（`keeppatterns` を使用する）
4. `:TrimWhitespace` コマンドと `<Leader>tw` マッピングの両方で呼び出せる

### 機能2: ToggleNumber
1. 行番号の表示モードを以下の順序で切り替える
   - 非表示 → 絶対行番号 → 相対行番号 → ハイブリッド → 非表示
2. 現在のモードをメッセージで表示する
3. `:ToggleNumber` コマンドと `<Leader>tn` マッピングで呼び出せる

### 機能3: InsertDate
1. カーソル位置に `YYYY-MM-DD` 形式の日付を挿入する
2. `:InsertDate` コマンドと `<Leader>id` マッピングで呼び出せる

### 機能4: ScratchBuffer
1. 新しい空バッファを作成し、以下の設定を適用する
   - バッファタイプを `nofile` にする（ファイルと関連付けない）
   - スワップファイルを作成しない
   - 保存確認をスキップする
2. `:ScratchBuffer` コマンドと `<Leader>sc` マッピングで呼び出せる

## ステップガイド

<details>
<summary>ステップ1：関数定義の基本を確認する</summary>

Vim script の関数定義の基本構文を確認しましょう。

```vim
" 関数名は大文字で始める（ユーザー定義関数の規則）
" s: プレフィックスでスクリプトローカルにする
function! s:MyFunction() abort
  " abort を付けると、エラー発生時に即座に関数を終了する
  " 処理...
endfunction

" カスタムコマンドの定義
command! MyCommand call s:MyFunction()

" マッピングとの連携
nnoremap <Leader>m :MyCommand<CR>
```

`function!` の `!` は、関数がすでに存在する場合に上書きを許可します。`:source` で再読み込みした時のエラーを防ぎます。

</details>

<details>
<summary>ステップ2：TrimWhitespace を実装する</summary>

末尾空白の削除は置換コマンドで行います。ポイントはカーソル位置の保存・復元です。

```vim
function! s:TrimWhitespace() abort
  " 現在のウィンドウ状態を保存
  let l:save = winsaveview()
  " 末尾の空白を削除（e フラグでマッチしなくてもエラーにしない）
  keeppatterns %s/\s\+$//e
  " ウィンドウ状態を復元
  call winrestview(l:save)
endfunction
```

</details>

<details>
<summary>ステップ3：ToggleNumber を実装する</summary>

`&number` と `&relativenumber` のオプション値を条件分岐で切り替えます。

```vim
function! s:ToggleNumber() abort
  if &number && &relativenumber
    " ハイブリッド → 非表示
    set nonumber norelativenumber
    echo "行番号: 非表示"
  elseif &relativenumber
    " 相対 → ハイブリッド
    set number
    echo "行番号: ハイブリッド"
  elseif &number
    " 絶対 → 相対
    set nonumber relativenumber
    echo "行番号: 相対"
  else
    " 非表示 → 絶対
    set number norelativenumber
    echo "行番号: 絶対"
  endif
endfunction
```

</details>

<details>
<summary>ステップ4：InsertDate と ScratchBuffer を実装する</summary>

```vim
" 日付挿入
function! s:InsertDate() abort
  let l:date = strftime('%Y-%m-%d')

" スクラッチバッファ
function! s:ScratchBuffer() abort
  enew
  setlocal buftype=nofile
  setlocal bufhidden=wipe
  setlocal noswapfile
  echo "スクラッチバッファを作成しました"
endfunction
```

<details>
<summary>解答例（改良版 ─ 引数対応と機能拡張）</summary>

```vim
" ==================================================
" カスタム関数とコマンド（改良版）
" ==================================================

" --- 1. 末尾空白の削除（範囲指定対応） ---
function! s:TrimWhitespace() range abort
  let l:save = winsaveview()
  execute a:firstline . ',' . a:lastline . 'keeppatterns s/\s\+$//e'
  call winrestview(l:save)
  echo "末尾空白を削除しました (" . a:firstline . "-" . a:lastline . "行)"
endfunction

" -range=% でデフォルトはファイル全体、ビジュアル選択で範囲指定可能
command! -range=% TrimWhitespace <line1>,<line2>call s:TrimWhitespace()
nnoremap <Leader>tw :TrimWhitespace<CR>

" --- 2. 行番号モードの切り替え ---
function! s:ToggleNumber() abort
  if &number && &relativenumber
    set nonumber norelativenumber
    echo "行番号: 非表示"
  elseif &relativenumber
    set number
    echo "行番号: ハイブリッド"
  elseif &number
    set nonumber relativenumber
    echo "行番号: 相対"
  else
    set number norelativenumber
    echo "行番号: 絶対"
  endif
endfunction

command! ToggleNumber call s:ToggleNumber()
nnoremap <Leader>tn :ToggleNumber<CR>

" --- 3. 日付・時刻の挿入（フォーマット引数対応） ---
function! s:InsertDateTime(format) abort
  if a:format ==# 'date'
    let l:text = strftime('%Y-%m-%d')
  elseif a:format ==# 'time'
    let l:text = strftime('%H:%M:%S')
  elseif a:format ==# 'datetime'
    let l:text = strftime('%Y-%m-%d %H:%M:%S')
  elseif a:format ==# 'iso'
    let l:text = strftime('%Y-%m-%dT%H:%M:%S%z')
  else
    let l:text = strftime(a:format)
  endif
  execute "normal! a" . l:text
endfunction

" Tab 補完で候補を表示
command! -nargs=? -complete=customlist,s:DateFormatComplete InsertDate
  \ call s:InsertDateTime(<q-args> != '' ? <q-args> : 'date')

function! s:DateFormatComplete(ArgLead, CmdLine, CursorPos) abort
  return filter(['date', 'time', 'datetime', 'iso'], 'v:val =~ "^" . a:ArgLead')
endfunction

nnoremap <Leader>id :InsertDate<CR>
nnoremap <Leader>it :InsertDate time<CR>
nnoremap <Leader>idt :InsertDate datetime<CR>

" --- 4. スクラッチバッファ（分割方向対応） ---
function! s:ScratchBuffer(split_cmd) abort
  execute a:split_cmd
  setlocal buftype=nofile
  setlocal bufhidden=wipe
  setlocal nobuflisted
  setlocal noswapfile
  setlocal modifiable
  echo "スクラッチバッファを作成しました (" . a:split_cmd . ")"
endfunction

command! ScratchBuffer call s:ScratchBuffer('enew')
command! ScratchSplit  call s:ScratchBuffer('new')
command! ScratchVSplit call s:ScratchBuffer('vnew')

nnoremap <Leader>sc :ScratchBuffer<CR>
nnoremap <Leader>ss :ScratchSplit<CR>
nnoremap <Leader>sv :ScratchVSplit<CR>

" --- 5. Quickfix トグル ---
function! s:ToggleQuickfix() abort
  let l:nr = winnr('$')
  cclose
  if l:nr == winnr('$')
    copen
  endif
endfunction

command! ToggleQuickfix call s:ToggleQuickfix()
nnoremap <Leader>cc :ToggleQuickfix<CR>
```

**初心者向けとの違い:**
- TrimWhitespace がビジュアル選択による範囲指定に対応
- InsertDate が引数で日付フォーマットを切り替え可能（Tab 補完付き）
- ScratchBuffer が分割方向（水平・垂直・現在ウィンドウ）を選択可能
- Quickfix ウィンドウのトグル機能を追加
- カスタムコマンドに `-nargs`, `-range`, `-complete` オプションを活用し、より Vim ネイティブな操作感を実現
