## 問題1：同じ文字の繰り返し検出（基本）

以下のテキストから、同じ文字が2回連続している箇所を見つけてください。

```python
text = "aabbcc hello book coffee"
```

**期待される出力：**
```
['a', 'b', 'c', 'l', 'o', 'f', 'e']
```

<details>
<summary>ヒント</summary>

`(.)\1` で「任意の1文字をキャプチャし、その後に同じ文字が続く」パターンを書けます。`re.findall()` はキャプチャグループの中身を返します。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "aabbcc hello book coffee"

# (.)\1 → 任意の1文字 + 同じ文字の繰り返し
results = re.findall(r"(.)\1", text)
print(results)  # ['a', 'b', 'c', 'l', 'o', 'f', 'e']

# 実際の連続文字を確認したい場合
for m in re.finditer(r"(.)\1", text):
    print(f"'{m.group()}' (位置: {m.start()})")
# 'aa' (位置: 0)
# 'bb' (位置: 2)
# 'cc' (位置: 4)
# 'll' (位置: 9)
# 'oo' (位置: 14)
# 'ff' (位置: 17)
# 'ee' (位置: 20)
```

</details>

## 問題2：重複単語の検出（基本）

以下のテキストから、連続して重複している英単語を見つけてください。

```python
text = "I have have a very very good good idea"
```

**期待される出力：**
```
['have', 'very', 'good']
```

<details>
<summary>ヒント</summary>

`(\b\w+)\s+\1\b` で「単語をキャプチャし、空白の後に同じ単語が続く」パターンを書けます。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "I have have a very very good good idea"

# (\b\w+)\s+\1\b → 単語 + 空白 + 同じ単語
results = re.findall(r"(\b\w+)\s+\1\b", text)
print(results)  # ['have', 'very', 'good']

# 重複を1つに修正
fixed = re.sub(r"(\b\w+)\s+\1\b", r"\1", text)
print(fixed)  # I have a very good idea
```

</details>

## 問題3：名前付きグループ（基本）

以下のテキストから日付を抽出し、名前付きグループを使って年・月・日を辞書形式で取得してください。

```python
text = "会議日: 2026-04-15"
```

**期待される出力：**
```
{'year': '2026', 'month': '04', 'day': '15'}
```

<details>
<summary>ヒント</summary>

`(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})` で名前付きグループを定義し、`.groupdict()` で辞書として取得できます。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "会議日: 2026-04-15"

# 名前付きグループで年・月・日をキャプチャ
result = re.search(r"(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})", text)

if result:
    print(result.groupdict())
    # {'year': '2026', 'month': '04', 'day': '15'}

    # 個別に取得
    print(f"{result.group('year')}年{result.group('month')}月{result.group('day')}日")
    # 2026年04月15日
```

</details>

## 問題4：肯定先読み（基本）

以下のテキストから、「円」が後に続く数値だけを抽出してください（「円」自体は結果に含めないでください）。

```python
text = "りんご 150円、バナナ 200円、オレンジ $3、ぶどう 500円"
```

**期待される出力：**
```
['150', '200', '500']
```

<details>
<summary>ヒント</summary>

`\d+(?=円)` で「数字の後に円が続く」パターンを書きます。先読みはゼロ幅なので、「円」自体はマッチ結果に含まれません。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "りんご 150円、バナナ 200円、オレンジ $3、ぶどう 500円"

# \d+(?=円) → 数字の後に「円」が続くもの（円はマッチに含まない）
results = re.findall(r"\d+(?=円)", text)
print(results)  # ['150', '200', '500']
```

</details>

## 問題5：肯定後読み（応用）

以下のテキストから、`$` 記号の後に続く金額（数値部分のみ）を抽出してください。

```python
text = "商品A: $49.99, 商品B: €29.99, 商品C: $199.50, 送料: ¥500"
```

**期待される出力：**
```
['49.99', '199.50']
```

<details>
<summary>ヒント</summary>

`(?<=\$)` で「前に `$` がある位置」を指定し、その後の数値パターン `\d+\.\d+` にマッチさせます。`$` は正規表現のメタ文字なのでエスケープが必要です。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "商品A: $49.99, 商品B: €29.99, 商品C: $199.50, 送料: ¥500"

# (?<=\$) → 前に $ がある位置
# [\d.]+ → 数字とドットの1文字以上
results = re.findall(r"(?<=\$)\d+\.\d+", text)
print(results)  # ['49.99', '199.50']
```

</details>

## 問題6：否定先読みでフィルタリング（応用）

以下のログから、`ERROR` を含まない行だけを抽出してください。

```python
log = """2026-04-05 INFO  起動完了
2026-04-05 ERROR 接続失敗
2026-04-05 INFO  処理開始
2026-04-05 WARN  遅延検出
2026-04-05 ERROR タイムアウト
2026-04-05 INFO  処理完了"""
```

**期待される出力：**
```
['2026-04-05 INFO  起動完了', '2026-04-05 INFO  処理開始', '2026-04-05 WARN  遅延検出', '2026-04-05 INFO  処理完了']
```

<details>
<summary>ヒント</summary>

`^(?!.*ERROR).+$` で「行の先頭から見て、どこにも ERROR がない行」にマッチさせます。`re.MULTILINE` フラグが必要です。

</details>

<details>
<summary>解答例</summary>

```python
import re

log = """2026-04-05 INFO  起動完了
2026-04-05 ERROR 接続失敗
2026-04-05 INFO  処理開始
2026-04-05 WARN  遅延検出
2026-04-05 ERROR タイムアウト
2026-04-05 INFO  処理完了"""

# ^(?!.*ERROR).+$ → 行の中に ERROR がない行
results = re.findall(r"^(?!.*ERROR).+$", log, re.MULTILINE)
print(results)
# ['2026-04-05 INFO  起動完了', '2026-04-05 INFO  処理開始',
#  '2026-04-05 WARN  遅延検出', '2026-04-05 INFO  処理完了']
```

</details>

## 問題7：HTMLタグの対応チェック（応用）

以下のHTMLテキストで、開始タグと閉じタグが正しく対応しているもの（`<tag>中身</tag>`）だけを抽出してください。

```python
html = "<h1>タイトル</h1> <p>本文</div> <div>コンテンツ</div> <span>テキスト</p>"
```

**期待される出力：**
```
[('h1', 'タイトル'), ('div', 'コンテンツ')]
```

<details>
<summary>ヒント</summary>

`<(\w+)>(.+?)</\1>` で、開始タグのタグ名をキャプチャし、閉じタグで同じタグ名を後方参照 `\1` で確認します。

</details>

<details>
<summary>解答例</summary>

```python
import re

html = "<h1>タイトル</h1> <p>本文</div> <div>コンテンツ</div> <span>テキスト</p>"

# <(\w+)> で開始タグ名をキャプチャ、</\1> で同じタグ名の閉じタグを後方参照
results = re.findall(r"<(\w+)>(.+?)</\1>", html)
print(results)  # [('h1', 'タイトル'), ('div', 'コンテンツ')]

# <p>本文</div> → p ≠ div なのでマッチしない
# <span>テキスト</p> → span ≠ p なのでマッチしない
```

</details>

## 問題8：パスワード強度チェック（チャレンジ）

先読みを使って、以下の条件をすべて満たすパスワードかどうかを1つの正規表現で判定してください。

条件：
- 8文字以上
- 英大文字を少なくとも1文字含む
- 英小文字を少なくとも1文字含む
- 数字を少なくとも1文字含む

```python
passwords = [
    "Abcdef12",    # ✓
    "abcdefgh",    # ✗（大文字なし）
    "ABCDEFGH",    # ✗（小文字なし）
    "Abcdefgh",    # ✗（数字なし）
    "Ab1",         # ✗（8文字未満）
    "P@ssw0rd",    # ✓
    "Hello123",    # ✓
]
```

**期待される出力：**
```
Abcdef12 → 合格
abcdefgh → 不合格
ABCDEFGH → 不合格
Abcdefgh → 不合格
Ab1 → 不合格
P@ssw0rd → 合格
Hello123 → 合格
```

<details>
<summary>ヒント</summary>

`^(?=.*[A-Z])(?=.*[a-z])(?=.*\d).{8,}$` のように、先読みを複数組み合わせます。各先読みは「文字列のどこかに条件を満たす文字がある」ことを確認します。

</details>

<details>
<summary>解答例</summary>

```python
import re

passwords = [
    "Abcdef12",
    "abcdefgh",
    "ABCDEFGH",
    "Abcdefgh",
    "Ab1",
    "P@ssw0rd",
    "Hello123",
]

# 先読みを組み合わせた1つの正規表現
pattern = r"^(?=.*[A-Z])(?=.*[a-z])(?=.*\d).{8,}$"

for pw in passwords:
    if re.match(pattern, pw):
        print(f"{pw} → 合格")
    else:
        print(f"{pw} → 不合格")

# パターンの分解：
# ^           → 文字列の先頭
# (?=.*[A-Z]) → どこかに大文字がある（先読み）
# (?=.*[a-z]) → どこかに小文字がある（先読み）
# (?=.*\d)    → どこかに数字がある（先読み）
# .{8,}       → 任意の文字が8文字以上
# $           → 文字列の末尾
```

</details>

## 問題9：日付形式の変換（チャレンジ）

名前付きグループと `re.sub()` を使って、以下のテキスト内の日付を `YYYY-MM-DD` 形式から `YYYY年MM月DD日` 形式に変換してください。

```python
text = "開始: 2026-04-01, 中間: 2026-06-15, 終了: 2026-09-30"
```

**期待される出力：**
```
開始: 2026年04月01日, 中間: 2026年06月15日, 終了: 2026年09月30日
```

<details>
<summary>ヒント</summary>

`(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})` で名前付きキャプチャし、`re.sub()` の置換文字列で `\g<year>年\g<month>月\g<day>日` のように参照します。

</details>

<details>
<summary>解答例</summary>

```python
import re

text = "開始: 2026-04-01, 中間: 2026-06-15, 終了: 2026-09-30"

# 名前付きグループでキャプチャし、置換で \g<name> で参照
result = re.sub(
    r"(?P<year>\d{4})-(?P<month>\d{2})-(?P<day>\d{2})",
    r"\g<year>年\g<month>月\g<day>日",
    text
)
print(result)
# 開始: 2026年04月01日, 中間: 2026年06月15日, 終了: 2026年09月30日

# 番号での参照でも同じ結果
result2 = re.sub(
    r"(\d{4})-(\d{2})-(\d{2})",
    r"\1年\2月\3日",
    text
)
print(result2)
# 開始: 2026年04月01日, 中間: 2026年06月15日, 終了: 2026年09月30日
```

</details>

## 問題10：数値への3桁カンマ挿入（チャレンジ）

先読みと後読みを使って、以下の数値に3桁ごとのカンマを挿入してください。

```python
numbers = ["1234567", "100", "12345", "9876543210"]
```

**期待される出力：**
```
1234567 → 1,234,567
100 → 100
12345 → 12,345
9876543210 → 9,876,543,210
```

<details>
<summary>ヒント</summary>

`(?<=\d)(?=(\d{3})+(?!\d))` で「前が数字で、後ろに3桁ずつの数字のかたまりがあり、その後は数字でない」位置にカンマを挿入します。

</details>

<details>
<summary>解答例</summary>

```python
import re

numbers = ["1234567", "100", "12345", "9876543210"]

for num in numbers:
    # (?<=\d)     → 前が数字
    # (?=(\d{3})+ → 後ろに3桁の数字のかたまりが1回以上
    # (?!\d))     → そのさらに後ろは数字でない
    result = re.sub(r"(?<=\d)(?=(\d{3})+(?!\d))", ",", num)
    print(f"{num} → {result}")

# 1234567 → 1,234,567
# 100 → 100
# 12345 → 12,345
# 9876543210 → 9,876,543,210
```

</details>
