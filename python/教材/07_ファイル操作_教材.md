## 7.1 ファイル操作の基本

### ファイルを開いて読む

```python
# 基本形（非推奨 - closeを忘れやすい）
f = open("sample.txt", "r", encoding="utf-8")
content = f.read()
print(content)
f.close()
```

### with文を使う（推奨）

```python
# with文なら自動でcloseされる
with open("sample.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
# ← ここでファイルが自動的に閉じられる
```

### ファイルモード

| モード | 意味 | 説明 |
|-------|------|------|
| `"r"` | 読み取り | ファイルがないとエラー |
| `"w"` | 書き込み | ファイルを新規作成（既存は上書き） |
| `"a"` | 追記 | ファイルの末尾に追加 |
| `"x"` | 排他作成 | ファイルが既に存在するとエラー |
| `"r+"` | 読み書き | ファイルがないとエラー |

## 7.2 テキストファイルの読み込み

### 全体を読む

```python
with open("sample.txt", "r", encoding="utf-8") as f:
    content = f.read()  # 全体を1つの文字列として読む
    print(content)
```

### 1行ずつ読む

```python
# readlines() - 行のリストとして読む
with open("sample.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()  # ['1行目\n', '2行目\n', ...]
    for line in lines:
        print(line.strip())  # 改行を除去して表示

# ファイルオブジェクトを直接ループ（メモリ効率が良い）
with open("sample.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

### readline() - 1行ずつ手動で読む

```python
with open("sample.txt", "r", encoding="utf-8") as f:
    first_line = f.readline()   # 1行目
    second_line = f.readline()  # 2行目
    print(first_line.strip())
    print(second_line.strip())
```

## 7.3 テキストファイルへの書き込み

### 新規作成・上書き（"w"モード）

```python
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("1行目のテキスト\n")
    f.write("2行目のテキスト\n")

# リストを一括で書き込む
lines = ["りんご\n", "みかん\n", "バナナ\n"]
with open("fruits.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)
```

### 追記（"a"モード）

```python
# 既存のファイルに追加
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("2024-03-15: ログイン\n")
    f.write("2024-03-15: ファイル作成\n")
```

### print() でファイルに書き込む

```python
with open("output.txt", "w", encoding="utf-8") as f:
    print("名前: 太郎", file=f)
    print("年齢: 20", file=f)
    print(f"合計: {100 + 200}", file=f)
```

## 7.4 CSVファイルの操作

CSVファイル（カンマ区切りファイル）は、表形式のデータ保存によく使われます。

### csvモジュールで読み込む

```python
import csv

# CSVファイルの読み込み
with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.reader(f)
    header = next(reader)  # ヘッダー行を取得
    print(header)  # ['名前', '年齢', '点数']

    for row in reader:
        name, age, score = row
        print(f"{name}: {age}歳, {score}点")
```

### 辞書形式で読み込む（DictReader）

```python
import csv

with open("students.csv", "r", encoding="utf-8") as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(f"{row['名前']}: {row['年齢']}歳, {row['点数']}点")
```

### CSVファイルへの書き込み

```python
import csv

students = [
    ["名前", "年齢", "点数"],
    ["太郎", 20, 85],
    ["花子", 19, 92],
    ["次郎", 21, 78],
]

with open("students.csv", "w", encoding="utf-8", newline="") as f:
    writer = csv.writer(f)
    writer.writerows(students)  # 全行を一括書き込み
```

### 辞書形式で書き込む（DictWriter）

```python
import csv

students = [
    {"名前": "太郎", "年齢": 20, "点数": 85},
    {"名前": "花子", "年齢": 19, "点数": 92},
]

with open("students.csv", "w", encoding="utf-8", newline="") as f:
    fieldnames = ["名前", "年齢", "点数"]
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()    # ヘッダー行を書き込む
    writer.writerows(students)
```

## 7.5 JSONファイルの操作

JSONは、データの保存や通信で広く使われる形式です。

### JSONファイルの読み込み

```python
import json

with open("config.json", "r", encoding="utf-8") as f:
    data = json.load(f)  # 辞書やリストとして読み込む
    print(data)
    print(data["name"])
```

### JSONファイルへの書き込み

```python
import json

config = {
    "name": "太郎",
    "age": 20,
    "hobbies": ["読書", "プログラミング", "映画"],
    "address": {
        "city": "東京",
        "zip": "100-0001"
    }
}

with open("config.json", "w", encoding="utf-8") as f:
    json.dump(config, f, ensure_ascii=False, indent=2)
```

### JSON文字列との変換

```python
import json

# 辞書 → JSON文字列
data = {"name": "太郎", "age": 20}
json_str = json.dumps(data, ensure_ascii=False)
print(json_str)  # {"name": "太郎", "age": 20}

# JSON文字列 → 辞書
text = '{"name": "花子", "age": 19}'
data = json.loads(text)
print(data["name"])  # 花子
```

> **注意**: `ensure_ascii=False` を指定しないと、日本語が `\uXXXX` の形式に変換されます。

## 7.6 パス操作（pathlibモジュール）

```python
from pathlib import Path

# パスの作成
p = Path("data") / "reports" / "2024"
print(p)  # data/reports/2024

# カレントディレクトリ
current = Path.cwd()
print(current)

# ホームディレクトリ
home = Path.home()
print(home)
```

### パスの情報を取得

```python
from pathlib import Path

p = Path("data/report_2024.csv")

print(p.name)      # report_2024.csv（ファイル名）
print(p.stem)      # report_2024（拡張子なし）
print(p.suffix)    # .csv（拡張子）
print(p.parent)    # data（親ディレクトリ）
print(p.exists())  # True / False（存在確認）
print(p.is_file()) # True / False（ファイルか？）
print(p.is_dir())  # True / False（ディレクトリか？）
```

### ディレクトリの操作

```python
from pathlib import Path

# ディレクトリ作成（exist_ok=Trueで既存でもエラーにならない）
Path("output/data").mkdir(parents=True, exist_ok=True)

# ディレクトリ内のファイル一覧
for file in Path("data").iterdir():
    print(file)

# パターンで検索（glob）
for csv_file in Path("data").glob("*.csv"):
    print(csv_file)

# 再帰的に検索
for py_file in Path(".").glob("**/*.py"):
    print(py_file)
```

## 7.7 実践的なパターン

### ファイルの存在確認してから読む

```python
from pathlib import Path

filepath = Path("data.txt")
if filepath.exists():
    content = filepath.read_text(encoding="utf-8")
    print(content)
else:
    print("ファイルが見つかりません")
```

### 設定ファイルの読み書き

```python
import json
from pathlib import Path

CONFIG_FILE = Path("settings.json")

def load_config():
    """設定を読み込む（なければデフォルト）"""
    if CONFIG_FILE.exists():
        with open(CONFIG_FILE, "r", encoding="utf-8") as f:
            return json.load(f)
    return {"theme": "light", "font_size": 14, "language": "ja"}

def save_config(config):
    """設定を保存する"""
    with open(CONFIG_FILE, "w", encoding="utf-8") as f:
        json.dump(config, f, ensure_ascii=False, indent=2)

# 使い方
config = load_config()
config["theme"] = "dark"
save_config(config)
```

## まとめ

| 操作 | 方法 |
|------|------|
| テキスト読み | `open("file", "r")` + `read()` / `readlines()` |
| テキスト書き | `open("file", "w")` + `write()` |
| 追記 | `open("file", "a")` + `write()` |
| CSV読み | `csv.reader()` / `csv.DictReader()` |
| CSV書き | `csv.writer()` / `csv.DictWriter()` |
| JSON読み | `json.load()` / `json.loads()` |
| JSON書き | `json.dump()` / `json.dumps()` |
| パス操作 | `pathlib.Path` |

> **重要**: ファイル操作は必ず `with` 文を使いましょう！
