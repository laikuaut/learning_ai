## 課題の要件

### 機能要件

1. タスクの追加・完了切替・削除ができる
2. タブで「全て / 未完了 / 完了」を切り替えてフィルタできる
3. 統計情報（件数・完了率）がリアルタイムに更新される
4. 削除時にモーダル（確認ダイアログ）を表示する
5. `localStorage` でデータを永続化する

### 設計要件

1. **基底クラス `Component`** を定義し、全UIパーツがこれを継承する
2. 各コンポーネントは `render()` メソッドを持ち、自分のDOM要素を生成する
3. **イベント駆動（EventEmitter パターン）** でコンポーネント間を連携させる
4. **状態（state）の変更が自動的にUIに反映される**仕組みを作る
5. HTMLファイル1つで完結すること

<summary>ステップ1：EventEmitter クラスを作る</summary>

コンポーネント間の通信を担う「イベントの仕組み」を作ります。

```javascript
class EventEmitter {
  constructor() {
    this._listeners = {};
  }

  on(event, callback) {
    if (!this._listeners[event]) {
      this._listeners[event] = [];
    }
    this._listeners[event].push(callback);
  }

  emit(event, data) {
    const callbacks = this._listeners[event];
    if (callbacks) {
      callbacks.forEach(cb => cb(data));
    }
  }

  off(event, callback) {
    if (!this._listeners[event]) return;
    this._listeners[event] = this._listeners[event].filter(cb => cb !== callback);
  }
}
```

<summary>ステップ3：Store（状態管理）を作る</summary>

アプリ全体の状態を1つのオブジェクトで管理し、変更があったらUIに通知します。

```javascript
class Store extends EventEmitter {
  constructor() {
    super();
    this.tasks = this._load();
  }

  _load() {
    try {
      const json = localStorage.getItem("tasks");
      return json ? JSON.parse(json) : [];
    } catch {
      return [];
    }
  }

  _save() {
    localStorage.setItem("tasks", JSON.stringify(this.tasks));
    this.emit("change", this.tasks);  // 変更を通知
  }

  addTask(name, priority) {
    this.tasks.push({
      id: Date.now(),
      name,
      priority,
      done: false,
      createdAt: new Date().toISOString()
    });
    this._save();
  }

  toggleTask(id) {
    const task = this.tasks.find(t => t.id === id);
    if (task) {
      task.done = !task.done;
      this._save();
    }
  }

  removeTask(id) {
    this.tasks = this.tasks.filter(t => t.id !== id);
    this._save();
  }

  getFiltered(filter) {
    if (filter === "active") return this.tasks.filter(t => !t.done);
    if (filter === "done") return this.tasks.filter(t => t.done);
    return [...this.tasks];
  }

  getStats() {
    const total = this.tasks.length;
    const done = this.tasks.filter(t => t.done).length;
    return { total, done, remaining: total - done, percent: total > 0 ? Math.round(done / total * 100) : 0 };
  }
}
```

### テクニック1：Proxy を使った自動再描画

```javascript
// 状態が変更されたら自動的にUIを更新するProxyラッパー
function createReactiveStore(store, onChange) {
  return new Proxy(store, {
    set(target, property, value) {
      target[property] = value;
      onChange();
      return true;
    }
  });
}

// 使用例
const reactiveStore = createReactiveStore(store, () => {
  taskList.render();
  stats.render();
});
```

```javascript
class Component {
```

### テクニック3：イベント委譲（Event Delegation）

```javascript
class TaskList extends Component {
  render() {
    // 各タスクにイベントを付けるのではなく、親要素1つに委譲
    this.container.addEventListener("click", (e) => {
      const target = e.target;
      const taskEl = target.closest("[data-task-id]");
      if (!taskEl) return;

      const taskId = Number(taskEl.dataset.taskId);

      if (target.classList.contains("task-check")) {
        this.store.toggleTask(taskId);
      } else if (target.classList.contains("task-delete")) {
        this.onDeleteRequest(taskId);
      }
    });
  }
}
```
