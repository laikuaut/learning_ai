# 実践課題02：ボックスモデル装飾 ─ 料金プラン表 ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（CSSの基本）、第2章（ボックスモデルとレイアウト）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `margin`・`padding`・`border`・`box-sizing` を使い分けて、整った見た目のコンポーネントを作る

---

---

## 課題の要件

1. 3つの料金プランを横に並べる
2. 各カードに「プラン名」「料金」「機能リスト」「ボタン」を含める
3. 中央のカード（おすすめプラン）を強調する
   - 枠線の色を変える
   - 上部に「おすすめ」ラベルを付ける
4. `box-sizing: border-box` を使い、幅の計算を分かりやすくする
5. `margin` と `padding` を適切に設定して、読みやすい余白を確保する
6. ボタンには `border` と `border-radius` でスタイルを付ける

---

## ステップガイド

<details>
<summary>ステップ1：HTMLの構造を作る</summary>

3つのカードを並べるコンテナと、各カードの中身を準備します。

```html
<div class="pricing">
  <div class="plan">
    <h2 class="plan-name">Free</h2>
    <p class="plan-price"><span class="price">¥0</span>/月</p>
    <ul class="plan-features">
      <li>基本機能</li>
      <li>1GBストレージ</li>
    </ul>
    <button class="plan-btn">選択する</button>
  </div>
  <!-- Pro, Business も同様 -->
</div>
```

</details>

<details>
<summary>ステップ2：box-sizing を設定する</summary>

全要素に `border-box` を指定すると、`padding` や `border` を含めた幅で管理できます。

```css
*, *::before, *::after {
  box-sizing: border-box;
}
```

これはCSSのリセットとして現場でも定番のテクニックです。

</details>

<details>
<summary>ステップ3：おすすめプランを強調する</summary>

中央のカードにクラスを追加し、差をつけます。

```css
.plan.recommended {
  border: 2px solid #4a90d9;
  transform: scale(1.05);
}
```

</details>
