## 基本問題

### 問題1：自己紹介を表示しよう

変数を使って、名前・年齢・趣味をコンソールに表示するプログラムを作成してください。テンプレートリテラルを使いましょう。

**期待される出力例：**
```
名前: 太郎
年齢: 20歳
趣味: プログラミング
```

<details>
<summary>ヒント</summary>

`const` で変数を宣言し、テンプレートリテラル `` `${変数}` `` を使って `console.log()` で出力しましょう。

</details>

<details>
<summary>解答例</summary>

```javascript
// 変数を const で宣言（値が変わらないため）
const name = "太郎";
const age = 20;
const hobby = "プログラミング";

// テンプレートリテラルで出力
console.log(`名前: ${name}`);
console.log(`年齢: ${age}歳`);
console.log(`趣味: ${hobby}`);
```

</details>

### 問題2：データ型を調べよう

以下の変数それぞれの型を `typeof` で確認し、表示してください。

```javascript
const val1 = 42;
const val2 = "hello";
const val3 = true;
const val4 = null;
const val5 = undefined;
const val6 = [1, 2, 3];
```

**期待される出力：**
```
42 の型: number
hello の型: string
true の型: boolean
null の型: object
undefined の型: undefined
[1,2,3] の型: object
```

<details>
<summary>ヒント</summary>

`typeof` 演算子を使います。`null` の `typeof` が `"object"` になるのはJavaScriptの歴史的なバグです。

</details>

<details>
<summary>解答例</summary>

```javascript
const val1 = 42;
const val2 = "hello";
const val3 = true;
const val4 = null;
const val5 = undefined;
const val6 = [1, 2, 3];

// typeof で型を調べて表示
console.log(`${val1} の型: ${typeof val1}`);
console.log(`${val2} の型: ${typeof val2}`);
console.log(`${val3} の型: ${typeof val3}`);
console.log(`${val4} の型: ${typeof val4}`);       // "object"（歴史的バグ）
console.log(`${val5} の型: ${typeof val5}`);
console.log(`[${val6}] の型: ${typeof val6}`);     // 配列も "object"
```

</details>

### 問題3：const と let の使い分け

以下の要件に合わせて変数を宣言し、結果を表示してください。

1. 税率（10%）を定数として宣言
2. 商品価格 1000 を変数として宣言
3. 税込価格を計算して表示
4. 商品価格を 2000 に変更して、再度税込価格を表示

**期待される出力：**
```
税込価格: 1100円
税込価格: 2200円
```

<details>
<summary>ヒント</summary>

変わらない値は `const`、変わる値は `let` で宣言します。税率は変わらないので `const`、価格は後から変更するので `let` です。

</details>

<details>
<summary>解答例</summary>

```javascript
// 税率は変わらないので const
const TAX_RATE = 0.1;

// 商品価格は後から変更するので let
let price = 1000;

// 税込価格を計算して表示
console.log(`税込価格: ${price + price * TAX_RATE}円`);

// 商品価格を変更
price = 2000;
console.log(`税込価格: ${price + price * TAX_RATE}円`);
```

</details>

### 問題4：文字列の基本操作

文字列 `"JavaScript Programming"` に対して以下の操作を行い、結果を表示してください。

1. 文字数を表示
2. 先頭の文字を表示
3. `"Script"` が含まれるか確認
4. すべて大文字に変換
5. `"Java"` の部分だけ切り出し

**期待される出力：**
```
文字数: 22
先頭の文字: J
"Script"を含む: true
大文字: JAVASCRIPT PROGRAMMING
切り出し: Java
```

<details>
<summary>解答例</summary>

```javascript
const str = "JavaScript Programming";

// 各種文字列操作
console.log(`文字数: ${str.length}`);
console.log(`先頭の文字: ${str[0]}`);
console.log(`"Script"を含む: ${str.includes("Script")}`);
console.log(`大文字: ${str.toUpperCase()}`);
console.log(`切り出し: ${str.slice(0, 4)}`);
```

</details>

## 応用問題

### 問題5：プロフィールカードを作ろう

以下の変数を使って、テンプレートリテラルで見やすいプロフィールカードをコンソールに表示してください。

```javascript
const name = "JavaScript太郎";
const age = 25;
const height = 170.5;
const isStudent = true;
```

**期待される出力例：**
```
+------------------------------+
|       プロフィールカード       |
+------------------------------+
| 名前: JavaScript太郎
| 年齢: 25歳（2001年生まれ）
| 身長: 170.5cm
| 学生: はい
+------------------------------+
```

<details>
<summary>ヒント</summary>

- 生年は `2026 - age` で計算できます。
- 「はい/いいえ」の表示は三項演算子 `条件 ? "はい" : "いいえ"` が使えます。
- `"+" + "-".repeat(30) + "+"` で罫線を作れます。

</details>

<details>
<summary>解答例</summary>

```javascript
const name = "JavaScript太郎";
const age = 25;
const height = 170.5;
const isStudent = true;

// 生年の計算
const birthYear = 2026 - age;

// 罫線の作成
const line = "+" + "-".repeat(30) + "+";

// プロフィールカードの表示
console.log(line);
console.log("|       プロフィールカード       |");
console.log(line);
console.log(`| 名前: ${name}`);
console.log(`| 年齢: ${age}歳（${birthYear}年生まれ）`);
console.log(`| 身長: ${height}cm`);
console.log(`| 学生: ${isStudent ? "はい" : "いいえ"}`);
console.log(line);
```

</details>

### 問題6：ランダムな数値の生成

1～100 のランダムな整数を3回生成し、それぞれの値と、3つの中の最大値・最小値を表示してください。

**期待される出力例：**
```
数値1: 42
数値2: 87
数値3: 15
最大値: 87
最小値: 15
```

<details>
<summary>ヒント</summary>

- `Math.random()` は 0以上1未満のランダムな数を返します。
- `Math.floor(Math.random() * 100) + 1` で 1～100 のランダムな整数が得られます。
- `Math.max()` と `Math.min()` で最大値・最小値を求められます。

</details>

<details>
<summary>解答例</summary>

```javascript
// 1～100のランダムな整数を3つ生成
const num1 = Math.floor(Math.random() * 100) + 1;
const num2 = Math.floor(Math.random() * 100) + 1;
const num3 = Math.floor(Math.random() * 100) + 1;

// 各値を表示
console.log(`数値1: ${num1}`);
console.log(`数値2: ${num2}`);
console.log(`数値3: ${num3}`);

// 最大値・最小値を表示
console.log(`最大値: ${Math.max(num1, num2, num3)}`);
console.log(`最小値: ${Math.min(num1, num2, num3)}`);
```

</details>

## チャレンジ問題

### 問題7：数値の書式変換

変数 `const number = 255;` に対して、以下の変換を行って表示してください。

1. 2進数表記
2. 8進数表記
3. 16進数表記
4. 3桁にゼロ埋め（例: "005"）

**期待される出力：**
```
10進数: 255
2進数: 11111111
8進数: 377
16進数: ff
ゼロ埋め: 255
```

<details>
<summary>ヒント</summary>

- `(数値).toString(基数)` で進数変換ができます。
- `String(数値).padStart(桁数, "0")` でゼロ埋めができます。

</details>

<details>
<summary>解答例</summary>

```javascript
const number = 255;

// 各進数への変換
console.log(`10進数: ${number}`);
console.log(`2進数: ${number.toString(2)}`);
console.log(`8進数: ${number.toString(8)}`);
console.log(`16進数: ${number.toString(16)}`);
console.log(`ゼロ埋め: ${String(number).padStart(3, "0")}`);
```

</details>

### 問題8：型変換の結果を予想しよう

以下の式の結果を**まず自分で予想**してから、実際に実行して確認してください。

```javascript
console.log(typeof null);
console.log(typeof undefined);
console.log(typeof NaN);
console.log("5" + 3);
console.log("5" - 3);
console.log(true + true);
console.log("" + 0);
console.log(null + 1);
console.log(undefined + 1);
console.log(0.1 + 0.2 === 0.3);
```

**期待される出力：**
```
object
undefined
number
53
2
2
0
1
NaN
false
```

<details>
<summary>解説</summary>

```javascript
// typeof null は "object"（歴史的なバグ）
console.log(typeof null);         // "object"

// typeof undefined は "undefined"
console.log(typeof undefined);    // "undefined"

// NaN は Number 型
console.log(typeof NaN);          // "number"

// + 演算子は文字列が含まれると文字列連結になる
console.log("5" + 3);            // "53"（文字列連結）

// - 演算子は数値変換してから計算する
console.log("5" - 3);            // 2（数値演算）

// true は数値では 1
console.log(true + true);        // 2

// "" は数値変換で 0 になる
console.log("" + 0);             // "0"（文字列連結）

// null は数値変換で 0 になる
console.log(null + 1);           // 1

// undefined は数値変換で NaN になる
console.log(undefined + 1);      // NaN

// 浮動小数点の誤差
console.log(0.1 + 0.2 === 0.3);  // false（0.30000000000000004）
```

</details>
