# 実践課題01：プロフィールカード ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（CSSの基本）
> **課題の種類**: ミニプロジェクト
> **学習目標**: セレクタ・色・フォント・余白など基本プロパティを組み合わせて、見栄えのするカードを作る

---

## 完成イメージ

```
┌──────────────────────────────┐
│                              │
│       ◯◯◯（アイコン）        │
│                              │
│        山田 太郎              │
│     Webデザイナー             │
│                              │
│  好きな技術: CSS / React      │
│  趣味: カフェ巡り             │
│                              │
│  ─────────────────────────   │
│  自己紹介テキストが           │
│  ここに入ります。             │
│                              │
└──────────────────────────────┘
```

ブラウザで開くと、中央寄せされたカードに名前・肩書き・自己紹介が表示されます。

---

## 課題の要件

1. 1つの HTML ファイルだけで動作すること（`<style>` タグ使用）
2. カードは画面中央に配置する
3. カード内に以下を含める
   - 丸いアイコン（背景色で代用してよい）
   - 名前（大きめの文字）
   - 肩書き（小さめ・薄い色）
   - プロフィール情報（2行以上）
   - 区切り線
   - 自己紹介文
4. カードに影（`box-shadow`）を付ける
5. 適切な余白（`margin` / `padding`）を設定する

---

## ステップガイド

<details>
<summary>ステップ1：HTMLの骨組みを作る</summary>

まず必要な要素を配置しましょう。

```html
<div class="card">
  <div class="avatar"></div>
  <h1 class="name">山田 太郎</h1>
  <p class="title">Webデザイナー</p>
  <div class="info">
    <p>好きな技術: CSS / React</p>
    <p>趣味: カフェ巡り</p>
  </div>
  <hr>
  <p class="bio">Webが大好きなデザイナーです。...</p>
</div>
```

</details>

<details>
<summary>ステップ2：カードのスタイルを設定する</summary>

カードの幅、角丸、影を設定します。

```css
.card {
  width: 320px;
  background: #fff;
  border-radius: 12px;
  box-shadow: 0 2px 12px rgba(0, 0, 0, 0.15);
  padding: 32px;
  text-align: center;
}
```

</details>

<details>
<summary>ステップ3：画面中央に配置する</summary>

`body` に最小限のスタイルを当てれば中央に置けます。

```css
body {
  margin: 0;
  min-height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
  background: #f0f2f5;
}
```

> **ポイント**: `display: flex` はまだ第3章の内容ですが、中央配置の定番パターンとして覚えてしまいましょう。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>プロフィールカード</title>
  <style>
    /* --- 全体 --- */
    body {
      margin: 0;
      min-height: 100vh;
      display: flex;
      justify-content: center;
      align-items: center;
      background-color: #f0f2f5;
      font-family: sans-serif;
    }

    /* --- カード --- */
    .card {
      width: 320px;
      background-color: #ffffff;
      border-radius: 12px;
      box-shadow: 0 2px 12px rgba(0, 0, 0, 0.15);
      padding: 32px;
      text-align: center;
    }

    /* --- アバター（丸い背景色で代用） --- */
    .avatar {
      width: 80px;
      height: 80px;
      background-color: #4a90d9;
      border-radius: 50%;
      margin: 0 auto 16px;
    }

    /* --- 名前 --- */
    .name {
      font-size: 22px;
      color: #333333;
      margin: 0 0 4px;
    }

    /* --- 肩書き --- */
    .title {
      font-size: 14px;
      color: #888888;
      margin: 0 0 20px;
    }

    /* --- 情報 --- */
    .info p {
      font-size: 14px;
      color: #555555;
      margin: 4px 0;
    }

    /* --- 区切り線 --- */
    hr {
      border: none;
      border-top: 1px solid #e0e0e0;
      margin: 20px 0;
    }

    /* --- 自己紹介 --- */
    .bio {
      font-size: 14px;
      color: #555555;
      line-height: 1.6;
    }
  </style>
</head>
<body>
  <div class="card">
    <div class="avatar"></div>
    <h1 class="name">山田 太郎</h1>
    <p class="title">Webデザイナー</p>
    <div class="info">
      <p>好きな技術: CSS / React</p>
      <p>趣味: カフェ巡り</p>
    </div>
    <hr>
    <p class="bio">
      Webが大好きなデザイナーです。<br>
      きれいなUIを作ることに情熱を注いでいます。
    </p>
  </div>
</body>
</html>
```

<details>
<summary>解答例（改良版 ─ より洗練されたデザイン）</summary>

色のグラデーションやトランジション（transition）を加えた、少しリッチなバージョンです。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>プロフィールカード（改良版）</title>
  <style>
    body {
      margin: 0;
      min-height: 100vh;
      display: flex;
      justify-content: center;
      align-items: center;
      background: linear-gradient(135deg, #667eea, #764ba2);
      font-family: "Helvetica Neue", Arial, sans-serif;
    }

    .card {
      width: 340px;
      background: #ffffff;
      border-radius: 16px;
      box-shadow: 0 8px 32px rgba(0, 0, 0, 0.2);
      padding: 40px 32px;
      text-align: center;
      transition: transform 0.3s;
    }

    .card:hover {
      transform: translateY(-4px);
    }

    .avatar {
      width: 90px;
      height: 90px;
      background: linear-gradient(135deg, #667eea, #764ba2);
      border-radius: 50%;
      margin: 0 auto 20px;
      display: flex;
      justify-content: center;
      align-items: center;
      color: #ffffff;
      font-size: 36px;
      font-weight: bold;
    }

    .name {
      font-size: 24px;
      color: #222222;
      margin: 0 0 4px;
      font-weight: 700;
    }

    .title {
      font-size: 13px;
      color: #999999;
      margin: 0 0 24px;
      letter-spacing: 1px;
      text-transform: uppercase;
    }

    .info {
      display: flex;
      justify-content: center;
      gap: 24px;
      margin-bottom: 20px;
    }

    .info-item {
      font-size: 13px;
      color: #666666;
    }

    .info-item strong {
      display: block;
      font-size: 18px;
      color: #333333;
      margin-bottom: 2px;
    }

    .divider {
      border: none;
      border-top: 1px solid #eeeeee;
      margin: 0 0 20px;
    }

    .bio {
      font-size: 14px;
      color: #666666;
      line-height: 1.7;
      margin: 0;
    }
  </style>
</head>
<body>
  <div class="card">
    <div class="avatar">T</div>
    <h1 class="name">山田 太郎</h1>
    <p class="title">Web Designer</p>
    <div class="info">
      <div class="info-item">
        <strong>5年</strong>
        経験
      </div>
      <div class="info-item">
        <strong>120+</strong>
        プロジェクト
      </div>
      <div class="info-item">
        <strong>3</strong>
        受賞歴
      </div>
    </div>
    <hr class="divider">
    <p class="bio">
      Webが大好きなデザイナーです。<br>
      きれいなUIを作ることに情熱を注いでいます。
    </p>
  </div>
</body>
</html>
```

**初心者向けとの違い:**
- 背景にグラデーションを使い、奥行き感を演出
- `transition` でホバー時にカードが浮き上がるアニメーション
- アバターにイニシャル文字を表示
- 情報セクションを数値+ラベルの形式に変更し、実績を見やすく表示
- `letter-spacing` や `text-transform` で細部のタイポグラフィを調整

</details>
