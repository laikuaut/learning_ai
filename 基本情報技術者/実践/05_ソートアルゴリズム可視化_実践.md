## ステップガイド

<details>
<summary>ステップ1：バブルソートの仕組み</summary>

隣接する要素を比較し、順序が逆なら交換します。1パスごとに最大値が末尾に移動します。

```
パス1: [5,3,8,1,2]
  5>3 → 交換 [3,5,8,1,2]
  5<8 → そのまま
  8>1 → 交換 [3,5,1,8,2]
  8>2 → 交換 [3,5,1,2,8]  ← 8が確定
```

</details>

<details>
<summary>ステップ2：選択ソートの仕組み</summary>

```
[5,3,8,1,2] → 最小値1 → 先頭と交換 → [1,3,8,5,2]
[1,|3,8,5,2] → 最小値2 → 2番目と交換 → [1,2,8,5,3]
...
```

<summary>ステップ3：挿入ソートの仕組み</summary>

各要素をソート済み部分の適切な位置に挿入します。

```
[3,5,|8,1,2] → 8はそのまま → [3,5,8,|1,2]
[3,5,8,|1,2] → 1を先頭に → [1,3,5,8,|2]
...
```

## 解答例

<details>
<summary>手計算問題の解答</summary>

**Q1. バブルソート** `[5, 3, 8, 1, 2]`
- パス1: [3, 5, 1, 2, **8**]（8が確定）
- パス2: [3, 1, 2, **5**, 8]（5が確定）
- パス3: [1, 2, **3**, 5, 8]（3が確定）
- パス4: [1, **2**, 3, 5, 8]（ソート完了）

**Q2. 選択ソート** `[5, 3, 8, 1, 2]`
- パス1: 最小値=1（位置3）→ 位置0と交換 → [**1**, 3, 8, 5, 2]
- パス2: 最小値=2（位置4）→ 位置1と交換 → [1, **2**, 8, 5, 3]
- パス3: 最小値=3（位置4）→ 位置2と交換 → [1, 2, **3**, 5, 8]
- パス4: 最小値=5（位置3）→ そのまま → [1, 2, 3, **5**, 8]

**Q3. 挿入ソート** `[5, 3, 8, 1, 2]`
- ステップ1: 3を挿入 → [**3, 5**, 8, 1, 2]
- ステップ2: 8はそのまま → [3, 5, **8**, 1, 2]
- ステップ3: 1を先頭に挿入 → [**1, 3, 5, 8**, 2]
- ステップ4: 2を位置1に挿入 → [1, **2, 3, 5, 8**]

**Q4. 計算量**

| アルゴリズム | 最良 | 平均 | 最悪 | 安定性 |
|-------------|------|------|------|--------|
| バブルソート | **O(n)** | **O(n²)** | **O(n²)** | **安定** |
| 選択ソート | **O(n²)** | **O(n²)** | **O(n²)** | **不安定** |
| 挿入ソート | **O(n)** | **O(n²)** | **O(n²)** | **安定** |

※バブルソートと挿入ソートは、すでにソート済みのデータでは O(n) になります。

**Q5.** 安定なソートとは、同じキー値を持つ要素の相対的な順序がソート前後で変わらないソートです。例えば、[(田中, 80), (鈴木, 80), (佐藤, 90)] を点数でソートしたとき、安定ソートなら田中と鈴木の順序が保たれます。

```python
# ソートアルゴリズム可視化
# 学べる内容：3種のソート、計算量の違い
# 実行方法：python 05_sort_visualizer.py

def bubble_sort(data):
    """バブルソート（各パスの状態を表示）"""
    arr = data.copy()
    n = len(arr)
    compare = 0
    swap = 0

    print("\n--- バブルソート (Bubble Sort) ---")
    for i in range(n - 1):
        for j in range(n - 1 - i):
            compare += 1
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swap += 1
        print(f"  パス{i + 1}: {arr}")

    print(f"  比較回数: {compare}回、交換回数: {swap}回")
    return arr


def selection_sort(data):
    """選択ソート（各パスの状態を表示）"""
    arr = data.copy()
    n = len(arr)
    compare = 0
    swap = 0

    print("\n--- 選択ソート (Selection Sort) ---")
    for i in range(n - 1):
        min_idx = i
        for j in range(i + 1, n):
            compare += 1
            if arr[j] < arr[min_idx]:
                min_idx = j
        if min_idx != i:
            arr[i], arr[min_idx] = arr[min_idx], arr[i]
            swap += 1
        print(f"  パス{i + 1}: {arr}  (最小値 {arr[i]} を位置{i}に配置)")

    print(f"  比較回数: {compare}回、交換回数: {swap}回")
    return arr


def insertion_sort(data):
    """挿入ソート（各ステップの状態を表示）"""
    arr = data.copy()
    n = len(arr)
    compare = 0
    shift = 0

    print("\n--- 挿入ソート (Insertion Sort) ---")
    for i in range(1, n):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            compare += 1
            arr[j + 1] = arr[j]
            shift += 1
            j -= 1
        if j >= 0:
            compare += 1  # whileの終了条件でも1回比較
        arr[j + 1] = key
        print(f"  ステップ{i}: {arr}  ({key}を挿入)")

    print(f"  比較回数: {compare}回、シフト回数: {shift}回")
    return arr


# --- メイン ---
print("===== ソートアルゴリズム可視化 =====")

data = [64, 34, 25, 12, 22, 11, 90]
print(f"\n元データ: {data}")

bubble_sort(data)
selection_sort(data)
insertion_sort(data)
```

<summary>解答例（改良版 ─ ベンチマーク比較付き）</summary>

```python
# ソートアルゴリズム可視化（改良版）
# ベンチマークで計算量の差を実測

import random
import time


def bubble_sort(arr, verbose=False):
    """バブルソート"""
    a = arr.copy()
    n = len(a)
    comp, swap = 0, 0
    for i in range(n - 1):
        swapped = False
        for j in range(n - 1 - i):
            comp += 1
            if a[j] > a[j + 1]:
                a[j], a[j + 1] = a[j + 1], a[j]
                swap += 1
                swapped = True
        if verbose:
            print(f"  パス{i + 1}: {a}")
        if not swapped:
            break  # 交換がなければ早期終了
    return a, comp, swap


def selection_sort(arr, verbose=False):
    """選択ソート"""
    a = arr.copy()
    n = len(a)
    comp, swap = 0, 0
    for i in range(n - 1):
        min_idx = i
        for j in range(i + 1, n):
            comp += 1
            if a[j] < a[min_idx]:
                min_idx = j
        if min_idx != i:
            a[i], a[min_idx] = a[min_idx], a[i]
            swap += 1
        if verbose:
            print(f"  パス{i + 1}: {a}")
    return a, comp, swap


def insertion_sort(arr, verbose=False):
    """挿入ソート"""
    a = arr.copy()
    n = len(a)
    comp, shift = 0, 0
    for i in range(1, n):
        key = a[i]
        j = i - 1
        while j >= 0 and a[j] > key:
            comp += 1
            a[j + 1] = a[j]
            shift += 1
            j -= 1
        if j >= 0:
            comp += 1
        a[j + 1] = key
        if verbose:
            print(f"  ステップ{i}: {a}")
    return a, comp, shift


def visualize_mode():
    """可視化モード"""
    print("\n--- 可視化モード ---")
    data = [64, 34, 25, 12, 22, 11, 90]
    print(f"元データ: {data}")

    algorithms = [
        ("バブルソート", bubble_sort),
        ("選択ソート", selection_sort),
        ("挿入ソート", insertion_sort),
    ]

    for name, func in algorithms:
        print(f"\n=== {name} ===")
        result, comp, ops = func(data, verbose=True)
        print(f"  → 比較: {comp}回、操作: {ops}回")


def benchmark_mode():
    """ベンチマークモード"""
    print("\n--- ベンチマーク ---")
    sizes = [100, 500, 1000, 3000, 5000]

    algorithms = [
        ("バブル", bubble_sort),
        ("選択", selection_sort),
        ("挿入", insertion_sort),
    ]

    # ヘッダー
    header = f"{'件数':>6}"
    for name, _ in algorithms:
        header += f" | {name + '(比較)':>12} {name + '(時間)':>10}"
    print(header)
    print("-" * len(header))

    for n in sizes:
        data = list(range(n))
        random.shuffle(data)

        row = f"{n:>6}"
        for name, func in algorithms:
            start = time.time()
            _, comp, _ = func(data)
            elapsed = time.time() - start
            row += f" | {comp:>12,} {elapsed:>9.4f}s"
        print(row)

    # 理論値の説明
    print(f"\n理論上の比較回数（平均）:")
    for n in sizes:
        print(f"  n={n}: n²/2 = {n * n // 2:,}")


def main():
    print("===== ソートアルゴリズム可視化 =====")
    print("  1. 可視化モード（ステップ表示）")
    print("  2. ベンチマーク（件数別比較）")

    choice = input("\n選択 (1-2): ").strip()
    if choice == "1":
        visualize_mode()
    elif choice == "2":
        benchmark_mode()
    else:
        print("無効な選択です。")


if __name__ == "__main__":
    main()
```

**初心者向けとの違い:**
- バブルソートに早期終了の最適化を追加 → 最良ケース O(n) を体験できる
- ベンチマークモードで件数別の実測データを表示 → O(n²) を実感
- 実行時間も計測 → 理論値と実測の対応を確認
- 関数をリストで管理 → アルゴリズム追加が容易
