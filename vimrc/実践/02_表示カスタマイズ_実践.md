# 実践課題02：表示カスタマイズ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第2章（表示とエディタの基本設定）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 行番号・カーソル表示・ステータスライン・不可視文字の可視化など、Vim の見た目に関する設定を自在にカスタマイズできるようになる

---

## 完成イメージ

```
┌──────────────────────────────────────────────────────────┐
│  3  function hello() {                                    │
│  2    console.log("Hello");                               │
│  1    return true;                                        │
│ 15  }  ← カーソル行（ハイライト + 絶対行番号）            │
│  1                                                        │
│  2  function world() {                                    │
│  3    console.log("World");                               │
│  4  }                                                     │
│                                                           │
│  タブ → »···  末尾スペース → ·  改行 → ↲                  │
│                                                           │
├──────────────────────────────────────────────────────────┤
│ [+] app.js [utf-8][unix]          行:15/42 列:3   42%    │
└──────────────────────────────────────────────────────────┘
```

- ハイブリッド行番号（カーソル行は絶対番号、他は相対番号）
- カーソル行がハイライトされている
- 不可視文字（タブ・末尾スペース・改行）が可視化されている
- ステータスラインにファイル情報と位置情報が表示されている

---

## 課題の要件

1. ハイブリッド行番号（`number` + `relativenumber`）を有効にする
2. カーソル行をハイライトする（`cursorline`）
3. シンタックスハイライトを有効にする
4. 不可視文字を可視化する（`list` + `listchars`）
   - タブ文字を `»` とスペースで表示
   - 末尾の空白を `·` で表示
   - 改行を `↲` で表示
5. ステータスラインをカスタマイズして以下を表示する
   - ファイル名、変更フラグ、エンコーディング、ファイル形式
   - 行番号/総行数、列番号、ファイル内の位置（%）
6. ステータスラインを常に表示する

---

## ステップガイド

<details>
<summary>ステップ1：行番号とカーソル行を設定する</summary>

```vim
" ハイブリッド行番号
set number
set relativenumber

" カーソル行をハイライト
set cursorline
```

設定後、Vim でカーソルを上下に移動して、行番号の表示が変わることを確認しましょう。

</details>

<details>
<summary>ステップ2：シンタックスハイライトを有効にする</summary>

```vim
" シンタックスハイライトを有効にする
syntax enable

" ファイルタイプ検出を有効にする
filetype plugin indent on
```

`.py` や `.js` などのファイルを開いて、コードに色が付くことを確認しましょう。

</details>

<details>
<summary>ステップ3：不可視文字を可視化する</summary>

```vim
" 不可視文字を表示する
set list

" 不可視文字の表示スタイルを設定する
set listchars=tab:»\ ,trail:·,eol:↲,extends:>,precedes:<
```

`listchars` の各要素の意味：
- `tab:»\ ` → タブ文字を `»` + スペースで表示
- `trail:·` → 行末の余分なスペースを `·` で表示
- `eol:↲` → 改行を `↲` で表示
- `extends:>` → 画面に収まらない行の右端を `>` で表示
- `precedes:<` → 画面に収まらない行の左端を `<` で表示

</details>

<details>
<summary>ステップ4：ステータスラインをカスタマイズする</summary>

```vim
" 常にステータスラインを表示する
set laststatus=2

" ステータスラインの内容を設定する
set statusline=%F%m%r%h%w\ [%{&enc}][%{&ff}]%=%l/%L\ 列:%c\ %p%%
```

ステータスラインの各要素：
- `%F` → ファイルのフルパス
- `%m` → 変更フラグ `[+]`
- `%r` → 読み取り専用フラグ `[RO]`
- `%=` → 左右の区切り（右寄せ）
- `%l/%L` → 現在行/総行数
- `%c` → 列番号
- `%p%%` → ファイル内の位置（%）

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ==================================================
" 表示カスタマイズ vimrc
" ==================================================

" --- 行番号 ---
set number                " 絶対行番号を表示
set relativenumber        " 相対行番号を表示（ハイブリッド）

" --- カーソル ---
set cursorline            " カーソル行をハイライト

" --- シンタックスハイライト ---
syntax enable             " シンタックスハイライトを有効にする
filetype plugin indent on " ファイルタイプ検出を有効にする

" --- 不可視文字 ---
set list                  " 不可視文字を表示する
set listchars=tab:»\ ,trail:·,eol:↲,extends:>,precedes:<

" --- ステータスライン ---
set laststatus=2          " 常にステータスラインを表示する
set statusline=%F%m%r%h%w\ [%{&enc}][%{&ff}]%=%l/%L\ 列:%c\ %p%%
```

</details>

<details>
<summary>解答例（改良版 ─ より実用的なカスタマイズ）</summary>

```vim
" ==================================================
" 表示カスタマイズ vimrc（改良版）
" ==================================================

" --- 行番号 ---
set number
set relativenumber
set signcolumn=yes        " サイン列を常に表示（Git 差分マーク等で使用）

" --- カーソル ---
set cursorline            " カーソル行をハイライト
set scrolloff=5           " カーソルの上下に常に5行の余白を確保
set sidescrolloff=5       " 横スクロール時も余白を確保

" --- シンタックスハイライト ---
syntax enable
filetype plugin indent on

" ターミナルが True Color に対応している場合
if has('termguicolors')
  set termguicolors
endif

" --- 不可視文字 ---
set list
set listchars=tab:»\ ,trail:·,eol:↲,extends:>,precedes:<,nbsp:+

" --- ステータスライン ---
set laststatus=2
" 区切り線を使ったステータスライン
set statusline=
set statusline+=\ %F                     " ファイルパス
set statusline+=\ %m%r%h%w               " 各種フラグ
set statusline+=\ [%{&enc}/%{&ff}]       " エンコーディング / 改行形式
set statusline+=%=                        " --- 右寄せ ---
set statusline+=\ %Y                     " ファイルタイプ
set statusline+=\ \ 行:%l/%L             " 行番号/総行数
set statusline+=\ 列:%c                  " 列番号
set statusline+=\ %p%%\                  " パーセント

" --- その他の表示設定 ---
set showcmd               " 入力中のコマンドを右下に表示
set showmode              " 現在のモードを表示
set wrap                  " 長い行を折り返して表示
set linebreak             " 単語の途中で折り返さない
set display=lastline      " 最終行が長くても表示を省略しない
```

**初心者向けとの違い:**
- `scrolloff` / `sidescrolloff` でカーソル周囲に常に余白を確保し、前後の文脈が見やすくなる
- `termguicolors` で True Color 対応ターミナルの表示品質を向上
- ステータスラインを `+=` で段階的に組み立てることで、可読性が高い
- `linebreak` + `display=lastline` で長い行の表示を改善

</details>
