## 演習8-1：.component() への書き換え（基本）

### 問題

以下の `.controller()` ベースのコードを `.component()` を使って書き換えてください。

```html
<div ng-controller="GreetCtrl">
  <h2>{{ greeting }}</h2>
  <input ng-model="name">
  <p>こんにちは、{{ name }}さん！</p>
</div>
<script>
  angular.module('myApp', [])
    .controller('GreetCtrl', function($scope) {
      $scope.greeting = 'あいさつアプリ';
      $scope.name = 'ゲスト';
    });
</script>
```

**要件：**
- `<greet-app>` コンポーネントとして使えるようにする
- テンプレート内では `$ctrl` を使う
- `$scope` を使わない

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body>
  <!-- コンポーネントとして使う -->
  <greet-app></greet-app>

  <script>
    angular.module('myApp', [])
      // .component() で定義
      .component('greetApp', {
        // $ctrl がデフォルトのプレフィックス
        template: '<div>' +
                  '  <h2>{{ $ctrl.greeting }}</h2>' +
                  '  <input ng-model="$ctrl.name">' +
                  '  <p>こんにちは、{{ $ctrl.name }}さん！</p>' +
                  '</div>',
        // $scope の代わりに this を使う
        controller: function() {
          this.greeting = 'あいさつアプリ';
          this.name = 'ゲスト';
        }
      });
  </script>
</body>
</html>
```

**ポイント:** `.component()` では `$scope` を使わず、`this`（テンプレートでは `$ctrl`）でデータを管理します。

</details>

## 演習8-2：ライフサイクルフックの実装（基本）

### 問題

`$onInit` と `$onDestroy` を使ったタイマーコンポーネントを作成してください。

1. `<timer-widget>` コンポーネントを作成
2. `$onInit` で1秒ごとにカウントアップするタイマーを開始
3. `$onDestroy` でタイマーを停止（メモリリーク防止）
4. 「表示/非表示」ボタンでコンポーネントの表示を `ng-if` で切り替え
5. 非表示にしたとき `$onDestroy` が呼ばれることをコンソールで確認

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    .timer { background: #e3f2fd; padding: 20px; border-radius: 8px; text-align: center; margin: 10px 0; }
    .timer .count { font-size: 48px; font-family: monospace; }
    button { padding: 10px 20px; margin: 5px; border: none; border-radius: 4px; cursor: pointer; font-size: 14px; }
    .btn-toggle { background: #42b883; color: white; }
    .log { background: #f5f5f5; padding: 10px; border-radius: 4px; font-family: monospace; font-size: 13px; max-height: 150px; overflow-y: auto; }
  </style>
</head>
<body ng-controller="AppCtrl">
  <h2>タイマーコンポーネント</h2>
  
  <button class="btn-toggle" ng-click="showTimer = !showTimer">
    {{ showTimer ? 'タイマーを非表示' : 'タイマーを表示' }}
  </button>
  
  <!-- ng-if で切り替え → $onInit/$onDestroy が呼ばれる -->
  <timer-widget ng-if="showTimer"></timer-widget>
  
  <h3>ログ</h3>
  <div class="log">
    <p ng-repeat="msg in logs track by $index">{{ msg }}</p>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('AppCtrl', function($scope) {
        $scope.showTimer = true;
        $scope.logs = [];
      })
      .component('timerWidget', {
        template: '<div class="timer">' +
                  '  <div class="count">{{ $ctrl.seconds }}</div>' +
                  '  <p>秒</p>' +
                  '</div>',
        // $interval を配列記法で注入
        controller: ['$interval', '$scope', function($interval, $scope) {
          var ctrl = this;
          var timer;
          
          // 初期化時に呼ばれる
          ctrl.$onInit = function() {
            ctrl.seconds = 0;
            // 親のlogsにアクセス（$scope.$parent経由）
            $scope.$parent.logs.push('[' + new Date().toLocaleTimeString() + '] $onInit: タイマー開始');
            timer = $interval(function() {
              ctrl.seconds++;
            }, 1000);
          };
          
          // 破棄時に呼ばれる
          ctrl.$onDestroy = function() {
            $scope.$parent.logs.push('[' + new Date().toLocaleTimeString() + '] $onDestroy: タイマー停止（' + ctrl.seconds + '秒）');
            // タイマーを確実に停止
            $interval.cancel(timer);
          };
        }]
      });
  </script>
</body>
</html>
```

</details>

## 演習8-3：単方向バインディングとコールバック（応用）

### 問題

親子コンポーネント間で単方向データフロー（`<` と `&`）を使った通信を実装してください。

1. 親コンポーネント `<app-root>` が商品リストを管理
2. 子コンポーネント `<product-card>` は商品情報を `<` で受け取り表示
3. 子で「カートに追加」ボタンを押すと `&` コールバックで親に通知
4. 親はカートに追加された商品数を表示

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 700px; margin: 20px auto; }
    .product-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); gap: 12px; }
    .card { border: 1px solid #ddd; border-radius: 8px; padding: 16px; }
    .card h3 { margin: 0 0 8px; font-size: 16px; }
    .card .price { color: #e74c3c; font-weight: bold; font-size: 18px; }
    .btn { padding: 8px 16px; border: none; border-radius: 4px; cursor: pointer; color: white; width: 100%; margin-top: 8px; }
    .btn-add { background: #42b883; }
    .btn-add:disabled { background: #ccc; }
    .cart-badge { background: #e74c3c; color: white; padding: 4px 12px; border-radius: 20px; font-size: 14px; }
  </style>
</head>
<body>
  <app-root></app-root>

  <script>
    angular.module('myApp', [])
      // 親コンポーネント
      .component('appRoot', {
        template: '<h2>商品ストア <span class="cart-badge" ng-show="$ctrl.cartCount > 0">カート: {{ $ctrl.cartCount }}点</span></h2>' +
                  '<div class="product-grid">' +
                  '  <product-card ng-repeat="p in $ctrl.products"' +
                  '    product="p"' +
                  '    on-add-to-cart="$ctrl.addToCart(product)">' +
                  '  </product-card>' +
                  '</div>',
        controller: function() {
          var ctrl = this;
          ctrl.cartCount = 0;
          ctrl.products = [
            { name: 'ノートPC', price: 89800, inStock: true },
            { name: 'マウス', price: 2980, inStock: true },
            { name: 'モニター', price: 34800, inStock: false },
            { name: 'キーボード', price: 7980, inStock: true }
          ];
          // 子から呼ばれるコールバック
          ctrl.addToCart = function(product) {
            ctrl.cartCount++;
            alert(product.name + ' をカートに追加しました！');
          };
        }
      })
      // 子コンポーネント
      .component('productCard', {
        bindings: {
          product: '<',       // 単方向バインディング
          onAddToCart: '&'    // コールバック
        },
        template: '<div class="card">' +
                  '  <h3>{{ $ctrl.product.name }}</h3>' +
                  '  <p class="price">¥{{ $ctrl.product.price | number }}</p>' +
                  '  <p>{{ $ctrl.product.inStock ? "在庫あり" : "在庫切れ" }}</p>' +
                  '  <button class="btn btn-add" ng-click="$ctrl.onAddToCart({ product: $ctrl.product })" ng-disabled="!$ctrl.product.inStock">' +
                  '    カートに追加' +
                  '  </button>' +
                  '</div>'
      });
  </script>
</body>
</html>
```

**ポイント:** `&` コールバックでパラメータを渡すときは `{ key: value }` のオブジェクト形式で渡します。

</details>

## 演習8-4：ワンタイムバインディングの適用（応用）

### 問題

大量のデータリスト（100件以上）を表示し、ワンタイムバインディング `::` の効果を体験してください。

1. 100件のダミーデータを生成
2. 通常バインディング版とワンタイムバインディング版を並べて表示
3. ウォッチャー数を表示して比較する
4. `track by` も適用する

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="myApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; max-width: 900px; margin: 20px auto; }
    .columns { display: grid; grid-template-columns: 1fr 1fr; gap: 20px; }
    .column { border: 1px solid #ddd; border-radius: 8px; padding: 12px; max-height: 400px; overflow-y: auto; }
    .column h3 { margin: 0 0 8px; }
    .item { padding: 4px 0; font-size: 13px; border-bottom: 1px solid #f0f0f0; }
    .stats { background: #f5f5f5; padding: 16px; border-radius: 8px; margin-bottom: 16px; }
    .highlight { color: #e74c3c; font-weight: bold; }
  </style>
</head>
<body ng-controller="PerfCtrl">
  <h2>ワンタイムバインディング比較</h2>
  
  <div class="stats">
    <p>データ件数: {{ items.length }}</p>
    <p>ウォッチャー数: <span class="highlight">{{ getWatcherCount() }}</span></p>
    <button ng-click="regenerate()">データ再生成</button>
  </div>
  
  <div class="columns">
    <!-- 通常バインディング -->
    <div class="column">
      <h3>通常バインディング</h3>
      <div class="item" ng-repeat="item in items track by item.id">
        {{ item.id }}. {{ item.name }} — {{ item.value }}
      </div>
    </div>
    
    <!-- ワンタイムバインディング :: -->
    <div class="column">
      <h3>ワンタイムバインディング (::)</h3>
      <div class="item" ng-repeat="item in ::items track by item.id">
        {{ ::item.id }}. {{ ::item.name }} — {{ ::item.value }}
      </div>
    </div>
  </div>

  <script>
    angular.module('myApp', [])
      .controller('PerfCtrl', function($scope) {
        // データ生成
        function generateItems() {
          var items = [];
          for (var i = 1; i <= 100; i++) {
            items.push({
              id: i,
              name: 'アイテム' + i,
              value: Math.floor(Math.random() * 1000)
            });
          }
          return items;
        }
        
        $scope.items = generateItems();
        
        $scope.regenerate = function() {
          $scope.items = generateItems();
        };
        
        // ウォッチャー数を計算
        $scope.getWatcherCount = function() {
          var root = angular.element(document.getElementsByTagName('body'));
          var watchers = 0;
          var f = function(element) {
            var scope = element.data() && (element.data().$scope || element.data().$isolateScope);
            if (scope && scope.$$watchers) {
              watchers += scope.$$watchers.length;
            }
            angular.forEach(element.children(), function(child) {
              f(angular.element(child));
            });
          };
          f(root);
          return watchers;
        };
      });
  </script>
</body>
</html>
```

**ポイント:** `::` を付けると値が安定した後はウォッチャーが解除され、パフォーマンスが向上します。「データ再生成」ボタンを押しても、ワンタイム側は更新されないことが確認できます。

</details>

## 演習8-5：コンポーネントベース TODO アプリ（チャレンジ）

### 問題

以下のコンポーネント構成でTODOアプリを作成してください。

```
<todo-app>
├── <todo-header on-add="...">        — タイトルと入力フォーム
├── <todo-list items="..." on-toggle="..." on-delete="...">  — リスト表示
│   └── <todo-item>                   — 各アイテム
└── <todo-footer count="..." on-clear="...">  — 残数表示とクリアボタン
```

**要件：**
- すべて `.component()` で作成
- データは親（todo-app）が管理し、子へは `<` で渡す
- イベントは `&` コールバックで親に通知
- ライフサイクルフック `$onInit` を使用
- `track by` を使用

<details>
<summary>解答例</summary>

```html
<!DOCTYPE html>
<html ng-app="todoApp">
<head>
  <meta charset="UTF-8">
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: sans-serif; background: #f5f5f5; margin: 0; }
    .todo-container { max-width: 500px; margin: 40px auto; }
    .header { background: #42b883; color: white; padding: 20px; border-radius: 12px 12px 0 0; text-align: center; }
    .header h1 { margin: 0 0 12px; }
    .header .input-row { display: flex; gap: 8px; }
    .header input { flex: 1; padding: 10px; border: none; border-radius: 4px; font-size: 14px; }
    .header button { padding: 10px 16px; background: #35495e; color: white; border: none; border-radius: 4px; cursor: pointer; }
    .list { background: white; }
    .todo-item { display: flex; align-items: center; padding: 12px 16px; border-bottom: 1px solid #eee; }
    .todo-item.done .text { text-decoration: line-through; color: #999; }
    .todo-item .text { flex: 1; margin-left: 10px; }
    .todo-item .delete { color: #e74c3c; cursor: pointer; border: none; background: none; font-size: 18px; padding: 0 4px; }
    .todo-item .delete:hover { font-weight: bold; }
    .footer { background: white; padding: 12px 16px; border-radius: 0 0 12px 12px; border-top: 1px solid #eee; display: flex; justify-content: space-between; align-items: center; }
    .footer .count { font-size: 14px; color: #666; }
    .footer button { padding: 6px 12px; background: #e74c3c; color: white; border: none; border-radius: 4px; cursor: pointer; font-size: 13px; }
    .empty { text-align: center; padding: 30px; color: #999; background: white; }
  </style>
</head>
<body>
  <div class="todo-container">
    <todo-app></todo-app>
  </div>

  <script>
    angular.module('todoApp', [])
      // ルートコンポーネント
      .component('todoApp', {
        template: '<todo-header on-add="$ctrl.addTodo(text)"></todo-header>' +
                  '<todo-list items="$ctrl.todos" on-toggle="$ctrl.toggleTodo(item)" on-delete="$ctrl.deleteTodo(item)"></todo-list>' +
                  '<todo-footer remaining="$ctrl.remainingCount()" has-completed="$ctrl.hasCompleted()" on-clear="$ctrl.clearCompleted()"></todo-footer>',
        controller: function() {
          var ctrl = this;
          
          ctrl.$onInit = function() {
            ctrl.nextId = 4;
            ctrl.todos = [
              { id: 1, text: 'AngularJSを学ぶ', done: true },
              { id: 2, text: 'コンポーネントを理解する', done: false },
              { id: 3, text: 'TODOアプリを作る', done: false }
            ];
          };
          
          ctrl.addTodo = function(text) {
            ctrl.todos.push({ id: ctrl.nextId++, text: text, done: false });
          };
          ctrl.toggleTodo = function(item) { item.done = !item.done; };
          ctrl.deleteTodo = function(item) {
            ctrl.todos = ctrl.todos.filter(function(t) { return t.id !== item.id; });
          };
          ctrl.remainingCount = function() {
            return ctrl.todos.filter(function(t) { return !t.done; }).length;
          };
          ctrl.hasCompleted = function() {
            return ctrl.todos.some(function(t) { return t.done; });
          };
          ctrl.clearCompleted = function() {
            ctrl.todos = ctrl.todos.filter(function(t) { return !t.done; });
          };
        }
      })
      // ヘッダーコンポーネント
      .component('todoHeader', {
        bindings: { onAdd: '&' },
        template: '<div class="header">' +
                  '  <h1>TODO</h1>' +
                  '  <div class="input-row">' +
                  '    <input placeholder="新しいタスクを入力..." ng-model="$ctrl.newText" ng-keyup="$event.keyCode === 13 && $ctrl.add()">' +
                  '    <button ng-click="$ctrl.add()">追加</button>' +
                  '  </div>' +
                  '</div>',
        controller: function() {
          this.newText = '';
          this.add = function() {
            if (this.newText.trim()) {
              this.onAdd({ text: this.newText });
              this.newText = '';
            }
          };
        }
      })
      // リストコンポーネント
      .component('todoList', {
        bindings: { items: '<', onToggle: '&', onDelete: '&' },
        template: '<div class="list">' +
                  '  <p class="empty" ng-if="$ctrl.items.length === 0">タスクがありません</p>' +
                  '  <todo-item ng-repeat="item in $ctrl.items track by item.id"' +
                  '    item="item"' +
                  '    on-toggle="$ctrl.onToggle({ item: item })"' +
                  '    on-delete="$ctrl.onDelete({ item: item })">' +
                  '  </todo-item>' +
                  '</div>'
      })
      // アイテムコンポーネント
      .component('todoItem', {
        bindings: { item: '<', onToggle: '&', onDelete: '&' },
        template: '<div class="todo-item" ng-class="{ done: $ctrl.item.done }">' +
                  '  <input type="checkbox" ng-checked="$ctrl.item.done" ng-click="$ctrl.onToggle()">' +
                  '  <span class="text">{{ $ctrl.item.text }}</span>' +
                  '  <button class="delete" ng-click="$ctrl.onDelete()">×</button>' +
                  '</div>'
      })
      // フッターコンポーネント
      .component('todoFooter', {
        bindings: { remaining: '<', hasCompleted: '<', onClear: '&' },
        template: '<div class="footer">' +
                  '  <span class="count">残り {{ $ctrl.remaining }}件</span>' +
                  '  <button ng-show="$ctrl.hasCompleted" ng-click="$ctrl.onClear()">完了済みを削除</button>' +
                  '</div>'
      });
  </script>
</body>
</html>
```

</details>
