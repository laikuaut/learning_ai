# 実践課題03：インデント設定チューニング ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第1章（vimrcの基本）、第3章（インデントとフォーマット設定）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `tabstop`・`shiftwidth`・`expandtab` の関係を理解し、言語の慣習に合ったインデント設定を作成する

---

## 完成イメージ

タブ文字とスペースの違いが可視化され、言語に合ったインデントが適用される状態を目指します。

```
Tab文字を使う場合（Makefile向け）:
┌──────────────────────────────────────┐
│  1 target:                           │
│  2 →   echo "Hello"    ← Tab文字    │
│  3 →   echo "World"                  │
└──────────────────────────────────────┘

スペースを使う場合（Python向け）:
┌──────────────────────────────────────┐
│  1 def greet(name):                  │
│  2 ····print(f"Hello, {name}")       │
│  3 ····if name == "World":           │
│  4 ········print("!")  ← 4スペース  │
└──────────────────────────────────────┘
```

---

## 課題の要件

1. デフォルトのインデントをスペース4つに設定する
2. タブ文字の表示幅を設定する
3. Tab キーでスペースを挿入する設定にする
4. 自動インデントを有効にする
5. 不可視文字（タブ・行末スペース）を可視化する
6. 既存ファイルのインデントを一括で整える方法を理解する

---

## 背景知識：タブとスペースの3つのオプション

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   tabstop     : タブ文字（\t）の「表示幅」                  │
│                 → 既存のタブ文字が何桁に見えるか            │
│                                                             │
│   shiftwidth  : >> や自動インデントの「インデント幅」       │
│                 → インデント操作で何桁ずれるか              │
│                                                             │
│   softtabstop : Tabキーを押したときの「挿入幅」             │
│                 → Tabキーで何桁分のスペースが入るか         │
│                                                             │
│   expandtab   : Tabキーでタブ文字の代わりにスペースを挿入   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## ステップガイド

<details>
<summary>ステップ1：デフォルトのインデント設定</summary>

まず、すべてのファイルに適用されるデフォルト設定を書きます。

```vim
" --- インデントの基本設定 ---
set tabstop=4           " タブ文字の表示幅
set shiftwidth=4        " 自動インデント・>>の幅
set softtabstop=4       " Tabキーで挿入する幅
set expandtab           " Tabキーでスペースを挿入
```

3つの値を同じにするのが最もシンプルで混乱しにくい設定です。

</details>

<details>
<summary>ステップ2：自動インデントを有効にする</summary>

```vim
" --- 自動インデント ---
set autoindent          " 前の行のインデントを引き継ぐ
set smartindent         " C言語風の構文でインデントを自動調整
filetype plugin indent on  " ファイルタイプ別のインデント設定を有効化
```

- `autoindent` だけだと、前の行のインデントを単純コピーするだけです
- `smartindent` を加えると、`{` の後で自動的にインデントが深くなります
- `filetype plugin indent on` を加えると、Pythonなら `:` の後でインデントが深くなるなど、言語に合った動作になります

</details>

<details>
<summary>ステップ3：不可視文字の可視化</summary>

タブ文字と行末のスペースを目に見えるようにします。

```vim
" --- 不可視文字の表示 ---
set list
set listchars=tab:▸\ ,trail:·,extends:»,precedes:«
```

| 記号 | 意味 |
|------|------|
| `tab:▸\ ` | タブ文字を `▸` + スペースで表示 |
| `trail:·` | 行末の余分なスペースを `·` で表示 |
| `extends:»` | 折り返さない行の右端 |
| `precedes:«` | 折り返さない行の左端 |

</details>

<details>
<summary>ステップ4：インデント操作の便利設定</summary>

```vim
" --- インデント操作 ---
set shiftround          " インデントをshiftwidthの倍数に揃える
```

`shiftround` を設定すると、`>>` で3桁の位置にいた場合、4桁にスナップしてくれます（shiftwidth=4のとき）。

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ===========================================
" インデント設定
" ===========================================

set nocompatible
filetype plugin indent on

" --- インデントの基本 ---
set tabstop=4
set shiftwidth=4
set softtabstop=4
set expandtab
set autoindent
set smartindent

" --- 不可視文字の表示 ---
set list
set listchars=tab:▸\ ,trail:·,extends:»,precedes:«

" --- インデント操作 ---
set shiftround
```

</details>

<details>
<summary>解答例（改良版：インデントの確認コマンド付き）</summary>

```vim
" ===========================================
" インデント設定 - 改良版
" ===========================================

set nocompatible
filetype plugin indent on

" --- インデントの基本（4スペース）---
set tabstop=4
set shiftwidth=4
set softtabstop=4
set expandtab
set autoindent
set smartindent
set shiftround

" --- 不可視文字の表示 ---
set list
set listchars=tab:▸\ ,trail:·,extends:»,precedes:«,nbsp:␣

" 行末スペースを赤くハイライト
highlight TrailingWhitespace ctermbg=red guibg=red
match TrailingWhitespace /\s\+$/

" --- 便利コマンド ---
" 行末の余分なスペースを削除するコマンド
command! TrimWhitespace %s/\s\+$//e

" タブをスペースに変換するコマンド
command! TabToSpace %retab

" 現在のインデント設定を表示するコマンド
command! ShowIndent echo printf(
    \ "tabstop=%d shiftwidth=%d softtabstop=%d expandtab=%s",
    \ &tabstop, &shiftwidth, &softtabstop,
    \ &expandtab ? "on" : "off")
```

**初心者向けとの違い**:

- `nbsp:␣` で改行なしスペースも可視化
- 行末スペースを赤でハイライトして見つけやすく
- `:TrimWhitespace` で行末スペースを一括削除
- `:TabToSpace` でタブ→スペース変換
- `:ShowIndent` で現在の設定値を即座に確認

</details>

## 確認方法

設定が正しく動作しているか、以下の手順で確認しましょう。

1. **Tabキーの動作確認**: 挿入モードで `Tab` を押してスペースが入ることを確認
2. **不可視文字の確認**: タブ文字を含むファイル（Makefileなど）を開いて `▸` が表示されることを確認
3. **自動インデント確認**: `if (true) {` の後に改行して自動的にインデントされることを確認
4. **設定値の確認**: `:set tabstop?` でオプションの現在値を確認

## よくある間違い

| ミス | 正しい設定 | 説明 |
|------|-----------|------|
| `tabstop` だけ変えて `shiftwidth` を揃えない | 3つの値を統一する | `>>` のインデント幅がズレる |
| `expandtab` を設定し忘れる | `set expandtab` を明記 | Tabキーでタブ文字が入ってしまう |
| `listchars` の `tab:` に文字を1つだけ指定 | `tab:▸\ `（2文字必要） | エラーになる。2文字必要 |
| `smartindent` だけで `filetype indent on` を省略 | 両方設定する | 言語別の高度なインデントが効かない |
