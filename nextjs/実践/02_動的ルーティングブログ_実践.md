# 実践課題02：動的ルーティングブログ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（Next.jsの基本）、第2章（App Routerとルーティング）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 動的ルート（Dynamic Routes）、`params` の取得、`Link` コンポーネントによるナビゲーションを実践する

---

## 完成イメージ

ブログ記事の一覧ページと、各記事の詳細ページを持つシンプルなブログサイトです。

```
/ (トップ)
├── /blog           → 記事一覧
├── /blog/nextjs    → 「Next.js入門」記事
├── /blog/react     → 「React基礎」記事
└── /blog/typescript → 「TypeScript活用」記事

┌──────────────────────────────────┐
│  📝 My Blog                     │
├──────────────────────────────────┤
│                                  │
│  記事一覧                        │
│                                  │
│  ┌────────────────────────────┐  │
│  │ Next.js入門                │  │
│  │ 2026-04-01  #Next.js #React│  │
│  │ Next.jsの基本的な使い方を...│  │
│  └────────────────────────────┘  │
│  ┌────────────────────────────┐  │
│  │ React基礎                  │  │
│  │ 2026-03-20  #React         │  │
│  │ Reactのコンポーネント...    │  │
│  └────────────────────────────┘  │
└──────────────────────────────────┘
```

---

## 課題の要件

1. `app/blog/page.tsx` に記事一覧を表示する
2. `app/blog/[slug]/page.tsx` で動的ルーティングを実装する
3. 記事データは配列（ハードコード）で管理する
4. 一覧ページから各記事へ `Link` で遷移できるようにする
5. 存在しないスラッグにアクセスした場合、`notFound()` で404ページを表示する

---

  return posts

<details>
<summary>ステップ3：動的ルートの詳細ページを作成する</summary>

`[slug]` フォルダを作り、`page.tsx` で `params` を受け取ります。

```tsx
// app/blog/[slug]/page.tsx
import { getPostBySlug } from '@/lib/posts'
import { notFound } from 'next/navigation'

export default async function PostPage({
  params,
}: {
  params: Promise<{ slug: string }>
}) {
  const { slug } = await params
  const post = getPostBySlug(slug)

  if (!post) {
    notFound()
  }

  return (
    <article>
      <h1>{post.title}</h1>
      <p>{post.content}</p>
    </article>
  )
}
```

<summary>解答例（改良版 ─ generateStaticParams + generateMetadata）</summary>

### lib/posts.ts（同じ）

初心者向けと同じファイルを使用します。

### app/blog/[slug]/page.tsx

```tsx
import Link from 'next/link'
import type { Metadata } from 'next'
import { getPostBySlug, getAllPosts } from '@/lib/posts'
import { notFound } from 'next/navigation'

// ビルド時に静的生成するパスを指定
export function generateStaticParams() {
  const posts = getAllPosts()
  return posts.map((post) => ({
    slug: post.slug,
  }))
}

// 動的メタデータ — 記事タイトルがページタイトルに反映される
export async function generateMetadata({
  params,
}: {
  params: Promise<{ slug: string }>
}): Promise<Metadata> {
  const { slug } = await params
  const post = getPostBySlug(slug)

  if (!post) {
    return { title: '記事が見つかりません' }
  }

  return {
    title: post.title,
    description: post.excerpt,
    openGraph: {
      title: post.title,
      description: post.excerpt,
      type: 'article',
      publishedTime: post.date,
      tags: post.tags,
    },
  }
}

export default async function PostPage({
  params,
}: {
  params: Promise<{ slug: string }>
}) {
  const { slug } = await params
  const post = getPostBySlug(slug)

  if (!post) {
    notFound()
  }

  // 前後の記事を取得
  const allPosts = getAllPosts()
  const currentIndex = allPosts.findIndex((p) => p.slug === slug)
  const prevPost = currentIndex < allPosts.length - 1 ? allPosts[currentIndex + 1] : null
  const nextPost = currentIndex > 0 ? allPosts[currentIndex - 1] : null

  return (
    <div style={{ maxWidth: '700px', margin: '0 auto', padding: '24px' }}>
      <Link
        href="/blog"
        style={{ color: '#2563eb', textDecoration: 'none', fontSize: '14px' }}
      >
        ← 記事一覧に戻る
      </Link>

      <article style={{ marginTop: '24px' }}>
        <h1 style={{ fontSize: '28px', lineHeight: '1.4' }}>{post.title}</h1>
        <div style={{ fontSize: '14px', color: '#64748b', marginBottom: '32px' }}>
          <time dateTime={post.date}>{post.date}</time>
          {post.tags.map((tag) => (
            <span
              key={tag}
              style={{
                marginLeft: '8px',
                padding: '2px 8px',
                backgroundColor: '#eff6ff',
                color: '#2563eb',
                borderRadius: '12px',
                fontSize: '12px',
              }}
            >
              #{tag}
            </span>
          ))}
        </div>
        <div style={{ lineHeight: '1.8', color: '#334155' }}>
          {post.content}
        </div>
      </article>

      {/* 前後の記事リンク */}
      <nav
        style={{
          display: 'flex',
          justifyContent: 'space-between',
          marginTop: '48px',
          paddingTop: '24px',
          borderTop: '1px solid #e2e8f0',
        }}
      >
        {prevPost ? (
          <Link href={`/blog/${prevPost.slug}`} style={{ textDecoration: 'none' }}>
            ← {prevPost.title}
          </Link>
        ) : (
          <span />
        )}
        {nextPost ? (
          <Link href={`/blog/${nextPost.slug}`} style={{ textDecoration: 'none' }}>
            {nextPost.title} →
          </Link>
        ) : (
          <span />
        )}
      </nav>
    </div>
  )
}
```

**初心者向けとの違い:**
- `generateStaticParams` でビルド時に全記事ページを静的生成（SSG）
- `generateMetadata` で記事ごとにタイトル・OGP情報を動的設定
- 前後の記事へのナビゲーションリンクを追加
- `<time>` タグの `dateTime` 属性でセマンティックHTMLを実現
- アロー関数でコードを簡潔に記述
