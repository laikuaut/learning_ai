## 問題1：2つの値の最大値を返す関数（基本）

2つの整数を受け取り、大きい方を返す関数 `maxValue` を作成してください。
`std::max` を使わずに自分で実装してください。

**期待される出力例：**
```
maxValue(10, 20) = 20
maxValue(-3, -7) = -3
maxValue(5, 5) = 5
```

<details>
<summary>ヒント</summary>

- 条件演算子（三項演算子） `? :` を使うと1行で書けます
- if-else でも問題ありません

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// 2つの整数のうち大きい方を返す関数
int maxValue(int a, int b) {
    return (a >= b) ? a : b;
}

int main() {
    cout << "maxValue(10, 20) = " << maxValue(10, 20) << endl;
    cout << "maxValue(-3, -7) = " << maxValue(-3, -7) << endl;
    cout << "maxValue(5, 5) = " << maxValue(5, 5) << endl;
    return 0;
}
```

</details>

## 問題2：素数判定関数（基本）

整数を受け取り、素数であれば `true`、そうでなければ `false` を返す関数 `isPrime` を作成してください。

**期待される出力例：**
```
2: 素数です
4: 素数ではありません
7: 素数です
13: 素数です
15: 素数ではありません
1: 素数ではありません
```

<details>
<summary>ヒント</summary>

- 1以下は素数ではありません
- 2は素数です（唯一の偶数の素数）
- 3以上の数は、2から `sqrt(n)` まで割り切れるかチェックします
- `<cmath>` の `sqrt` を使うか、`i * i <= n` の条件で判定できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// 素数判定関数
bool isPrime(int n) {
    if (n <= 1) return false;   // 1以下は素数ではない
    if (n == 2) return true;    // 2は素数
    if (n % 2 == 0) return false; // 2以外の偶数は素数ではない

    // 3以上の奇数で割り切れるかチェック
    for (int i = 3; i * i <= n; i += 2) {
        if (n % i == 0) return false;
    }
    return true;
}

int main() {
    int numbers[] = {2, 4, 7, 13, 15, 1};
    for (int n : numbers) {
        cout << n << ": "
             << (isPrime(n) ? "素数です" : "素数ではありません")
             << endl;
    }
    return 0;
}
```

</details>

## 問題3：フィボナッチ数列 - 再帰（基本）

n番目のフィボナッチ数を再帰で求める関数 `fibonacci` を作成してください。
フィボナッチ数列は F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2) です。

**期待される出力例：**
```
F(0) = 0
F(1) = 1
F(2) = 1
F(5) = 5
F(10) = 55
```

<details>
<summary>ヒント</summary>

- 基底条件（base case）は `n == 0` と `n == 1` の2つです
- それ以外は `fibonacci(n-1) + fibonacci(n-2)` を返します
- 大きな値（40以上）を渡すと非常に遅くなることに注意してください

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// 再帰でフィボナッチ数を求める
int fibonacci(int n) {
    if (n == 0) return 0;  // 基底条件1
    if (n == 1) return 1;  // 基底条件2
    return fibonacci(n - 1) + fibonacci(n - 2);  // 再帰呼び出し
}

int main() {
    int values[] = {0, 1, 2, 5, 10};
    for (int n : values) {
        cout << "F(" << n << ") = " << fibonacci(n) << endl;
    }
    return 0;
}
```

</details>

## 問題4：配列の合計と平均を求める関数（応用）

整数の配列とそのサイズを受け取り、合計と平均を**参照渡し**で返す関数 `calcStats` を作成してください。
戻り値は使わず、引数の参照を通じて結果を呼び出し元に返してください。

**期待される出力例：**
```
データ: 10 20 30 40 50
合計: 150
平均: 30
```

<details>
<summary>ヒント</summary>

- 関数のシグネチャは `void calcStats(int arr[], int size, int& sum, double& avg)` のようにします
- `sum` と `avg` は参照渡しなので、関数内で値を変更すると呼び出し元にも反映されます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
using namespace std;

// 配列の合計と平均を参照渡しで返す関数
void calcStats(int arr[], int size, int& sum, double& avg) {
    sum = 0;
    for (int i = 0; i < size; i++) {
        sum += arr[i];
    }
    avg = static_cast<double>(sum) / size;
}

int main() {
    int data[] = {10, 20, 30, 40, 50};
    int size = 5;
    int sum = 0;
    double avg = 0.0;

    calcStats(data, size, sum, avg);

    cout << "データ:";
    for (int i = 0; i < size; i++) {
        cout << " " << data[i];
    }
    cout << endl;
    cout << "合計: " << sum << endl;
    cout << "平均: " << avg << endl;
    return 0;
}
```

</details>

## 問題5：関数オーバーロード - 面積計算（応用）

同じ関数名 `area` で、以下の3つのオーバーロードを作成してください。

1. `area(double radius)` — 円の面積（π × r²）
2. `area(double width, double height)` — 長方形の面積
3. `area(double a, double b, double c)` — ヘロンの公式による三角形の面積

**期待される出力例：**
```
円の面積（半径5）: 78.5398
長方形の面積（3x4）: 12
三角形の面積（3,4,5）: 6
```

<details>
<summary>ヒント</summary>

- 円周率は `M_PI`（`<cmath>` をインクルード）または `3.14159265358979` を使います
- ヘロンの公式：`s = (a+b+c)/2` とすると、面積 = `sqrt(s*(s-a)*(s-b)*(s-c))`
- `<cmath>` の `sqrt` を使ってください

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <cmath>
using namespace std;

// 円の面積
double area(double radius) {
    return M_PI * radius * radius;
}

// 長方形の面積
double area(double width, double height) {
    return width * height;
}

// 三角形の面積（ヘロンの公式）
double area(double a, double b, double c) {
    double s = (a + b + c) / 2.0;
    return sqrt(s * (s - a) * (s - b) * (s - c));
}

int main() {
    cout << "円の面積（半径5）: " << area(5.0) << endl;
    cout << "長方形の面積（3x4）: " << area(3.0, 4.0) << endl;
    cout << "三角形の面積（3,4,5）: " << area(3.0, 4.0, 5.0) << endl;
    return 0;
}
```

</details>

## 問題6：文字列処理関数（応用）

以下の文字列処理関数を作成してください。

1. `countChar(const string& s, char c)` — 文字列 `s` に文字 `c` が何回出現するか返す
2. `reverseString(const string& s)` — 文字列を逆順にして返す
3. `toUpperCase(const string& s)` — すべて大文字に変換して返す

**期待される出力例：**
```
"hello world" に 'l' は 3 回出現
"hello" の逆順: "olleh"
"Hello World" の大文字: "HELLO WORLD"
```

<details>
<summary>ヒント</summary>

- `string` を使うには `<string>` をインクルードします
- `reverseString` は `<algorithm>` の `reverse` を使うか、ループで自作します
- `toupper()` は `<cctype>` に含まれ、1文字を大文字に変換します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>
#include <cctype>
using namespace std;

// 文字の出現回数をカウント
int countChar(const string& s, char c) {
    int count = 0;
    for (char ch : s) {
        if (ch == c) count++;
    }
    return count;
}

// 文字列を逆順にする
string reverseString(const string& s) {
    string result = s;
    int n = result.size();
    for (int i = 0; i < n / 2; i++) {
        swap(result[i], result[n - 1 - i]);
    }
    return result;
}

// すべて大文字に変換
string toUpperCase(const string& s) {
    string result = s;
    for (char& ch : result) {
        ch = toupper(ch);
    }
    return result;
}

int main() {
    cout << "\"hello world\" に 'l' は "
         << countChar("hello world", 'l')
         << " 回出現" << endl;

    cout << "\"hello\" の逆順: \""
         << reverseString("hello") << "\"" << endl;

    cout << "\"Hello World\" の大文字: \""
         << toUpperCase("Hello World") << "\"" << endl;

    return 0;
}
```

</details>

## 問題7：ラムダ式でカスタムソート（チャレンジ）

整数の `vector` を以下の条件でソートするプログラムを、ラムダ式を使って作成してください。

1. 昇順ソート
2. 降順ソート
3. 絶対値の小さい順にソート
4. 偶数を先に、奇数を後に（同じグループ内では昇順）

**期待される出力例：**
```
元のデータ: -5 3 -1 4 -2 8 7
昇順: -5 -2 -1 3 4 7 8
降順: 8 7 4 3 -1 -2 -5
絶対値順: -1 -2 3 4 -5 7 8
偶数優先: -2 4 8 -5 -1 3 7
```

<details>
<summary>ヒント</summary>

- `<algorithm>` の `sort` を使います
- ラムダ式の書き方: `[](int a, int b) { return a < b; }`
- 絶対値は `abs()`（`<cstdlib>` または `<cmath>`）を使います
- 偶数優先ソートでは、まず偶奇を比較し、同じグループなら値で比較します
- ソート前に毎回元のデータに戻すことを忘れないでください

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstdlib>
using namespace std;

// vectorの中身を表示するヘルパー関数
void printVec(const vector<int>& v) {
    for (int i = 0; i < (int)v.size(); i++) {
        if (i > 0) cout << " ";
        cout << v[i];
    }
    cout << endl;
}

int main() {
    vector<int> original = {-5, 3, -1, 4, -2, 8, 7};
    vector<int> data;

    cout << "元のデータ: ";
    printVec(original);

    // 1. 昇順ソート
    data = original;
    sort(data.begin(), data.end(), [](int a, int b) {
        return a < b;
    });
    cout << "昇順: ";
    printVec(data);

    // 2. 降順ソート
    data = original;
    sort(data.begin(), data.end(), [](int a, int b) {
        return a > b;
    });
    cout << "降順: ";
    printVec(data);

    // 3. 絶対値の小さい順
    data = original;
    sort(data.begin(), data.end(), [](int a, int b) {
        return abs(a) < abs(b);
    });
    cout << "絶対値順: ";
    printVec(data);

    // 4. 偶数優先（同グループ内は昇順）
    data = original;
    sort(data.begin(), data.end(), [](int a, int b) {
        bool aEven = (a % 2 == 0);
        bool bEven = (b % 2 == 0);
        if (aEven != bEven) return aEven;  // 偶数を先に
        return a < b;                       // 同グループ内は昇順
    });
    cout << "偶数優先: ";
    printVec(data);

    return 0;
}
```

</details>

## 問題8：メモ化フィボナッチと実行時間比較（チャレンジ）

問題3の再帰フィボナッチは大きな値で非常に遅くなります。
`std::map` を使ってメモ化（memoization）したバージョン `fibMemo` を作成し、
通常の再帰版と実行時間を比較してください。

**期待される出力例：**
```
=== 再帰版 ===
F(40) = 102334155
実行時間: 523ms

=== メモ化版 ===
F(40) = 102334155
実行時間: 0ms

F(50) をメモ化版で計算: 12586269025
```

※ 実行時間はマシンにより変わります。

<details>
<summary>ヒント</summary>

- `<map>` をインクルードし、`map<int, long long>` をメモとして使います
- メモ化関数では、まず `map` にキーが存在するか確認し、あればその値を返します
- 時間計測には `<chrono>` を使います
- F(50) は `int` の範囲を超えるため、`long long` を使ってください

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <map>
#include <chrono>
using namespace std;

// 通常の再帰版
long long fibonacci(int n) {
    if (n <= 0) return 0;
    if (n == 1) return 1;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

// メモ化版
map<int, long long> memo;

long long fibMemo(int n) {
    if (n <= 0) return 0;
    if (n == 1) return 1;

    // メモにあればそれを返す
    if (memo.find(n) != memo.end()) {
        return memo[n];
    }

    // 計算してメモに保存
    memo[n] = fibMemo(n - 1) + fibMemo(n - 2);
    return memo[n];
}

int main() {
    int n = 40;

    // 再帰版の時間計測
    cout << "=== 再帰版 ===" << endl;
    auto start = chrono::high_resolution_clock::now();
    long long result1 = fibonacci(n);
    auto end = chrono::high_resolution_clock::now();
    auto duration1 = chrono::duration_cast<chrono::milliseconds>(end - start);
    cout << "F(" << n << ") = " << result1 << endl;
    cout << "実行時間: " << duration1.count() << "ms" << endl;

    // メモ化版の時間計測
    cout << "\n=== メモ化版 ===" << endl;
    start = chrono::high_resolution_clock::now();
    long long result2 = fibMemo(n);
    end = chrono::high_resolution_clock::now();
    auto duration2 = chrono::duration_cast<chrono::milliseconds>(end - start);
    cout << "F(" << n << ") = " << result2 << endl;
    cout << "実行時間: " << duration2.count() << "ms" << endl;

    // メモ化版ならF(50)も瞬時に計算可能
    cout << "\nF(50) をメモ化版で計算: " << fibMemo(50) << endl;

    return 0;
}
```

</details>
