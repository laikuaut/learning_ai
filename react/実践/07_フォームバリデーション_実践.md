# 実践課題07：フォームバリデーション ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第3章（stateとイベント）、第5章（useEffectと副作用）、第6章（カスタムフック）
> **課題の種類**: ミニプロジェクト
> **学習目標**: フォーム入力のリアルタイムバリデーションをカスタムフックで汎用的に実装する

---

## 完成イメージ

```
===== ユーザー登録 =====

名前:      [太郎                    ] ✓
メール:    [taro@example             ] ✗ 有効なメールアドレスを入力してください
パスワード: [••••••                   ] ⚠ 8文字以上必要です（現在6文字）
確認:      [••••••                   ] ✓

パスワード強度: ████░░░░░░ 普通

  [  登録する  ]  ← バリデーション全通過で有効化
```

---

## 課題の要件

1. 4つのフィールド（名前・メール・パスワード・パスワード確認）を持つフォーム
2. 各フィールドにバリデーションルールを設定する
   - 名前: 必須、2文字以上
   - メール: 必須、`@` と `.` を含む
   - パスワード: 必須、8文字以上、大文字・小文字・数字を含む
   - 確認: パスワードと一致
3. リアルタイムでバリデーション結果を表示する（入力中に即反映）
4. パスワード強度インジケーターを表示する
5. すべてのバリデーションが通過したときのみ送信ボタンを有効化する
6. `useFormValidation` カスタムフックでバリデーションロジックを汎用化する

<details>
<summary>ステップ2：useFormValidation カスタムフックを作る</summary>

```jsx
function useFormValidation(initialValues, rules) {
  const [values, setValues] = React.useState(initialValues);
  const [errors, setErrors] = React.useState({});
  const [touched, setTouched] = React.useState({});

  function handleChange(field, value) {
    setValues({ ...values, [field]: value });
    // バリデーション実行
    if (rules[field]) {
      setErrors({ ...errors, [field]: rules[field](value, values) });
    }
  }

  function handleBlur(field) {
    setTouched({ ...touched, [field]: true });
  }

  var isValid = Object.keys(rules).every(field => !rules[field](values[field], values));

  return { values, errors, touched, handleChange, handleBlur, isValid };
}
```

<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>フォームバリデーション</title>
  <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
  <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
  <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
  <style>
    body { font-family: sans-serif; background: #f5f5f5; display: flex; justify-content: center; padding: 40px; }
    .form-container { background: white; border-radius: 16px; padding: 32px; width: 420px; box-shadow: 0 4px 16px rgba(0,0,0,0.1); }
    h1 { text-align: center; margin-bottom: 24px; }

    .field input:focus { outline: none; border-color: #2196f3; }
    .field input.error { border-color: #f44336; }
    .field input.valid { border-color: #4caf50; }

    .strength-segment { height: 6px; flex: 1; border-radius: 3px; background: #e0e0e0; transition: background 0.3s; }
    .strength-label { font-size: 13px; margin-top: 4px; }

    .submit-btn:disabled { background: #ccc; cursor: not-allowed; }
  </style>
</head>
<body>
  <div id="root"></div>
  <script type="text/babel">
    // ==================== バリデーションルール ====================
    function validateName(value) {
      if (!value) return "名前は必須です";
      if (value.length < 2) return "2文字以上で入力してください";
      return "";
    }

    function validateEmail(value) {
      if (!value) return "メールアドレスは必須です";
      if (!value.includes("@") || !value.includes(".")) return "有効なメールアドレスを入力してください";
      return "";
    }

    function validatePassword(value) {
      if (!value) return "パスワードは必須です";
      if (value.length < 8) return "8文字以上必要です（現在" + value.length + "文字）";
      return "";
    }

    function validateConfirm(value, allValues) {
      if (!value) return "パスワード確認は必須です";
      if (value !== allValues.password) return "パスワードが一致しません";
      return "";
    }

    // ==================== カスタムフック ====================
    function useFormValidation(initialValues, validators) {
      const [values, setValues] = React.useState(initialValues);
      const [errors, setErrors] = React.useState({});
      const [touched, setTouched] = React.useState({});

      function handleChange(field, value) {
        var newValues = { ...values, [field]: value };
        setValues(newValues);

        // 触れたフィールドのバリデーション実行
        if (touched[field] && validators[field]) {
          setErrors(function(prev) {
            return { ...prev, [field]: validators[field](value, newValues) };
          });
        }

        // パスワード変更時に確認フィールドも再バリデーション
        if (field === "password" && touched.confirm && validators.confirm) {
          setErrors(function(prev) {
            return { ...prev, confirm: validators.confirm(newValues.confirm, newValues) };
          });
        }
      }

      function handleBlur(field) {
        setTouched(function(prev) { return { ...prev, [field]: true }; });
        // blur時にバリデーション実行
        if (validators[field]) {
          setErrors(function(prev) {
            return { ...prev, [field]: validators[field](values[field], values) };
          });
        }
      }

      // 全フィールドがバリデーション通過しているか
      var isValid = Object.keys(validators).every(function(field) {
        return !validators[field](values[field], values);
      });

      return { values: values, errors: errors, touched: touched, handleChange: handleChange, handleBlur: handleBlur, isValid: isValid };
    }

    // ==================== パスワード強度 ====================
    function getPasswordStrength(password) {
      if (!password) return { score: 0, label: "", color: "#e0e0e0" };
      var score = 0;
      if (password.length >= 8) score++;
      if (password.length >= 12) score++;
      if (/[A-Z]/.test(password)) score++;
      if (/[a-z]/.test(password)) score++;
      if (/[0-9]/.test(password)) score++;
      if (/[^A-Za-z0-9]/.test(password)) score++;

      var labels = [
        { min: 0, label: "非常に弱い", color: "#f44336" },
        { min: 2, label: "弱い", color: "#ff9800" },
        { min: 3, label: "普通", color: "#ffc107" },
        { min: 4, label: "強い", color: "#8bc34a" },
        { min: 5, label: "非常に強い", color: "#4caf50" },
      ];

      var result = labels[0];
      for (var i = 0; i < labels.length; i++) {
        if (score >= labels[i].min) result = labels[i];
      }
      return { score: score, label: result.label, color: result.color };
    }

    function StrengthBar({ password }) {
      var strength = getPasswordStrength(password);
      var segments = [];
      for (var i = 0; i < 6; i++) {
        segments.push(
          <div
            key={i}
            className="strength-segment"
            style={{ background: i < strength.score ? strength.color : "#e0e0e0" }}
          />
        );
      }

      return (
        <div>
          <div className="strength-bar">{segments}</div>
          {password && <div className="strength-label" style={{ color: strength.color }}>強度: {strength.label}</div>}
        </div>
      );
    }

    // ==================== フォームフィールド ====================
    function FormField({ label, type, value, error, touched, onChange, onBlur }) {
      var hasError = touched && error;
      var isValid = touched && !error && value;
      var inputClass = hasError ? "error" : isValid ? "valid" : "";
      var icon = hasError ? "✗" : isValid ? "✓" : "";
      var iconColor = hasError ? "#f44336" : "#4caf50";

      return (
        <div className="field">
          <label>{label}</label>
          <div className="input-wrapper">
            <input
              type={type || "text"}
              className={inputClass}
              value={value}
              onChange={(e) => onChange(e.target.value)}
              onBlur={onBlur}
            />
            <span className="status-icon" style={{ color: iconColor }}>{icon}</span>
          </div>
          {hasError && <div className="error-message">{error}</div>}
        </div>
      );
    }

    // ==================== メインコンポーネント ====================
    function App() {
      var form = useFormValidation(
        { name: "", email: "", password: "", confirm: "" },
        { name: validateName, email: validateEmail, password: validatePassword, confirm: validateConfirm }
      );

      function handleSubmit(e) {
        e.preventDefault();
        if (form.isValid) {
          alert("登録成功！\n名前: " + form.values.name + "\nメール: " + form.values.email);
        }
      }

      return (
        <div className="form-container">
          <h1>ユーザー登録</h1>
          <form onSubmit={handleSubmit}>
            <FormField
              label="名前" value={form.values.name}
              error={form.errors.name} touched={form.touched.name}
              onChange={(v) => form.handleChange("name", v)}
              onBlur={() => form.handleBlur("name")}
            />
            <FormField
              label="メールアドレス" type="email" value={form.values.email}
              error={form.errors.email} touched={form.touched.email}
              onChange={(v) => form.handleChange("email", v)}
              onBlur={() => form.handleBlur("email")}
            />
            <FormField
              label="パスワード" type="password" value={form.values.password}
              error={form.errors.password} touched={form.touched.password}
              onChange={(v) => form.handleChange("password", v)}
              onBlur={() => form.handleBlur("password")}
            />
            <StrengthBar password={form.values.password} />
            <FormField
              label="パスワード（確認）" type="password" value={form.values.confirm}
              error={form.errors.confirm} touched={form.touched.confirm}
              onChange={(v) => form.handleChange("confirm", v)}
              onBlur={() => form.handleBlur("confirm")}
            />
            <button type="submit" className="submit-btn" disabled={!form.isValid}>
              登録する
            </button>
          </form>
        </div>
      );
    }

    ReactDOM.createRoot(document.getElementById("root")).render(<App />);
  </script>
</body>
</html>

<summary>解答例（改良版 ─ バリデーションルールを宣言的に定義）</summary>

ルールを宣言的なオブジェクトとして定義し、カスタムフックが自動でバリデーションを組み立てる方式です。差分のみ示します。

```jsx
// 宣言的なルール定義
const FIELD_RULES = {
  name: [
    { test: (v) => !!v, message: "名前は必須です" },
    { test: (v) => v.length >= 2, message: "2文字以上で入力してください" },
  ],
  email: [
    { test: (v) => !!v, message: "メールアドレスは必須です" },
    { test: (v) => /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(v), message: "有効なメールアドレスを入力してください" },
  ],
  password: [
    { test: (v) => !!v, message: "パスワードは必須です" },
    { test: (v) => v.length >= 8, message: (v) => `8文字以上必要です（現在${v.length}文字）` },
  ],
  confirm: [
    { test: (v) => !!v, message: "パスワード確認は必須です" },
    { test: (v, all) => v === all.password, message: "パスワードが一致しません" },
  ],
};

// 汎用バリデーション実行関数
function runValidation(rules, value, allValues) {
  for (const rule of rules) {
    if (!rule.test(value, allValues)) {
      return typeof rule.message === "function" ? rule.message(value) : rule.message;
    }
  }
  return "";
}
```

**初心者向けとの違い:**
- バリデーションルールがデータとして分離 → ロジックの見通しが良い
- ルールの追加・変更が宣言的 → テストしやすい
- メッセージを関数にすることで動的な値を含めることが可能
