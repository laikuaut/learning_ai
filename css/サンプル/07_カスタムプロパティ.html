2. ローカルスコープ（コンポーネント変数）

クラスごとに変数を上書きすることで、同じ構造のカードの色を変えられます。

お知らせ

--card-color と --card-bg をローカルで定義しています。

成功

同じ CSS 構造で、色だけ変数で切り替えています。

エラー

変数のスコープにより、各カードに異なる色が適用されます。

6. テーマ切り替えの仕組み

右上の「テーマ切替」ボタンを押すと、ライトモードとダークモードが切り替わります。

仕組み: html 要素に data-theme="dark" を付与し、CSSの属性セレクタ [data-theme="dark"] で変数を上書きしています。JavaScriptは属性を切り替えるだけで、スタイルの変更はすべてCSSが担当します。
