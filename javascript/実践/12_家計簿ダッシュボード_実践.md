## 課題の要件

### 必須機能

1. **取引記録**: 収入/支出の登録（日付・金額・カテゴリ・メモ）
2. **月別サマリー**: 収入合計・支出合計・残高を表示
3. **カテゴリ別分析**: 支出カテゴリ別の割合と棒グラフ
4. **取引履歴**: 月別フィルタ・キーワード検索・日付順ソート
5. **データ永続化**: `localStorage` でJSON形式で保存/読み込み
6. **CSV出力**: 現在表示中の取引をCSV文字列として出力
7. **エラー処理**: 不正な入力に対する適切なバリデーションとメッセージ

### 設計要件

1. **ES6+ の機能** を積極的に使うこと（クラス・アロー関数・テンプレートリテラル・分割代入・スプレッド構文・`Map` / `Set`・`Promise`）
2. **配列の高階関数**（`map`, `filter`, `reduce`, `sort`, `find`）を活用すること
3. **エラーハンドリング**（`try-catch`）を適切に入れること
4. HTMLファイル1つで完結すること

### カテゴリ定義

- **収入カテゴリ**: 給与、副業、投資、その他
- **支出カテゴリ**: 食費、交通費、日用品、娯楽、通信費、医療、教育、その他

          id: Date.now(),

          date,
          amount,
          category,
          memo: memo || ""

        this._save();

const validationRules = [
  { test: (d) => d.type === "income" || d.type === "expense",
    message: "種別を選択してください" },
  { test: (d) => d.date && !isNaN(new Date(d.date).getTime()),

];

function validate(data) {
  const failed = validationRules.find(rule => !rule.test(data));

}

async function exportCSV(transactions) {
  const csv = toCSV(transactions);

  try {
    await navigator.clipboard.writeText(csv);
    showNotification("CSVをクリップボードにコピーしました");
  } catch (err) {
    // フォールバック: Blob + download link
    const blob = new Blob([csv], { type: "text/csv;charset=utf-8;" });
    const url = URL.createObjectURL(blob);
    const a = document.createElement("a");
    a.href = url;
    a.download = `kakeibo_${currentYear}_${currentMonth}.csv`;
    a.click();
    URL.revokeObjectURL(url);
  }
}

### テクニック4：デバウンス（debounce）で検索を最適化

```javascript
// 入力のたびに即座に検索するのではなく、
// 300ms入力が止まったら検索する
function debounce(fn, delay) {
  let timerId;
  return function(...args) {
    clearTimeout(timerId);
    timerId = setTimeout(() => fn(...args), delay);
  };
}

const debouncedSearch = debounce((keyword) => {
  searchKeyword = keyword;
  renderHistory();
}, 300);

searchInput.addEventListener("input", (e) => {
  debouncedSearch(e.target.value);
});
```

- バリデーションルールをデータ駆動で定義（ルール追加が容易）
- `Intl.NumberFormat` で国際化対応のフォーマット
- `async/await` でクリップボードAPIを呼び出し + Blobフォールバック
- デバウンスで検索パフォーマンスを最適化
- これらは実務のWebアプリケーションで頻繁に使われるパターンです
