# 実践課題03：文字列カウンター ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第3章（繰り返し）、第4章（配列と文字列）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 文字配列の操作、ループによる文字列走査、条件判定を組み合わせてテキスト分析ツールを作る

---

## 完成イメージ

```
===== 文字列カウンター =====
文字列を入力してください: Hello, World! 123

--- 分析結果 ---
文字列の長さ    : 17
英大文字        : 2
英小文字        : 8
数字            : 3
空白            : 2
その他の文字    : 2
```

---

## 課題の要件

1. ユーザーから1行の文字列を入力で受け取る（`fgets` を使用）
2. 文字列を1文字ずつ走査し、以下をカウントする
   - 英大文字（A-Z）
   - 英小文字（a-z）
   - 数字（0-9）
   - 空白文字
   - その他の文字（記号など）
3. 文字列の長さと各カテゴリのカウントを表示する
4. 改行文字は文字数に含めない

---

## ステップガイド

<details>
<summary>ステップ1：文字列を入力で受け取る</summary>

`scanf("%s", ...)` は空白で区切られてしまうため、空白を含む文字列を読むには `fgets` を使います。

```c
#include <stdio.h>
#include <string.h>

int main(void) {
    char str[256];

    printf("===== 文字列カウンター =====\n");
    printf("文字列を入力してください: ");
    fgets(str, sizeof(str), stdin);

    /* fgets は改行を含むので除去する */
    size_t len = strlen(str);
    if (len > 0 && str[len - 1] == '\n') {
        str[len - 1] = '\0';
        len--;
    }

    return 0;
}
```

</details>

<details>
<summary>ステップ2：1文字ずつ判定してカウントする</summary>

文字の種類を判定するには、文字コード（ASCII）の範囲を使います。

```c
int upper = 0, lower = 0, digit = 0, space = 0, other = 0;

for (int i = 0; i < (int)len; i++) {
    char c = str[i];
    if (c >= 'A' && c <= 'Z') {
        upper++;
    } else if (c >= 'a' && c <= 'z') {
        lower++;
    } else if (c >= '0' && c <= '9') {
        digit++;
    } else if (c == ' ') {
        space++;
    } else {
        other++;
    }
}
```

</details>

<details>
<summary>ステップ3：結果を整形して表示する</summary>

`%-16s` で左寄せにすると見やすくなります。

```c
printf("\n--- 分析結果 ---\n");
printf("%-16s: %d\n", "文字列の長さ", (int)len);
printf("%-16s: %d\n", "英大文字", upper);
/* ... 残りも同様 ... */
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```c
/*
 * 文字列カウンター
 * 学べる内容：文字配列、fgets、文字種判定、ループ処理
 * コンパイル：gcc -o str_counter str_counter.c
 * 実行：./str_counter
 */
#include <stdio.h>
#include <string.h>

int main(void) {
    char str[256];
    int upper = 0;
    int lower = 0;
    int digit = 0;
    int space = 0;
    int other = 0;
    int i;
    size_t len;

    printf("===== 文字列カウンター =====\n");
    printf("文字列を入力してください: ");
    fgets(str, sizeof(str), stdin);

    /* 末尾の改行を除去 */
    len = strlen(str);
    if (len > 0 && str[len - 1] == '\n') {
        str[len - 1] = '\0';
        len--;
    }

    /* 1文字ずつ分類してカウント */
    for (i = 0; i < (int)len; i++) {
        char c = str[i];
        if (c >= 'A' && c <= 'Z') {
            upper++;
        } else if (c >= 'a' && c <= 'z') {
            lower++;
        } else if (c >= '0' && c <= '9') {
            digit++;
        } else if (c == ' ') {
            space++;
        } else {
            other++;
        }
    }

    /* 結果表示 */
    printf("\n--- 分析結果 ---\n");
    printf("文字列の長さ    : %d\n", (int)len);
    printf("英大文字        : %d\n", upper);
    printf("英小文字        : %d\n", lower);
    printf("数字            : %d\n", digit);
    printf("空白            : %d\n", space);
    printf("その他の文字    : %d\n", other);

    return 0;
}
```

<details>
<summary>解答例（改良版 ─ ctype.h を活用）</summary>

`<ctype.h>` ヘッダに用意されている文字判定関数を使うと、コードが簡潔で移植性も高くなります。

```c
/*
 * 文字列カウンター（改良版）
 * 学べる内容：ctype.h の文字分類関数、配列による頻度集計
 * コンパイル：gcc -o str_counter2 str_counter2.c
 * 実行：./str_counter2
 */
#include <stdio.h>
#include <string.h>
#include <ctype.h>

int main(void) {
    char str[256];
    int upper = 0, lower = 0, digit = 0, space = 0, other = 0;
    int freq[26] = {0};  /* アルファベット頻度 */
    int i;
    size_t len;

    printf("===== 文字列カウンター（改良版）=====\n");
    printf("文字列を入力してください: ");
    fgets(str, sizeof(str), stdin);

    /* 末尾の改行を除去 */
    len = strlen(str);
    if (len > 0 && str[len - 1] == '\n') {
        str[len - 1] = '\0';
        len--;
    }

    /* 文字分類とアルファベット頻度集計 */
    for (i = 0; i < (int)len; i++) {
        unsigned char c = (unsigned char)str[i];
        if (isupper(c)) {
            upper++;
            freq[c - 'A']++;
        } else if (islower(c)) {
            lower++;
            freq[c - 'a']++;
        } else if (isdigit(c)) {
            digit++;
        } else if (isspace(c)) {
            space++;
        } else {
            other++;
        }
    }

    /* 基本情報 */
    printf("\n--- 分析結果 ---\n");
    printf("文字列の長さ    : %d\n", (int)len);
    printf("英大文字        : %d\n", upper);
    printf("英小文字        : %d\n", lower);
    printf("数字            : %d\n", digit);
    printf("空白            : %d\n", space);
    printf("その他の文字    : %d\n", other);

    /* アルファベット頻度（出現したもののみ） */
    printf("\n--- アルファベット頻度 ---\n");
    for (i = 0; i < 26; i++) {
        if (freq[i] > 0) {
            printf("  %c : %d回\n", 'a' + i, freq[i]);
        }
    }

    return 0;
}
```

**初心者向けとの違い:**
- `<ctype.h>` の `isupper()`, `islower()`, `isdigit()`, `isspace()` を使用 → 文字コードの範囲を直接書くより安全で移植性が高い
- アルファベット頻度の集計機能を追加 → 配列をカウンタとして活用するテクニック
- `unsigned char` にキャストして `ctype.h` 関数に渡す → 負の値を防ぐベストプラクティス

</details>
