### 問題1：リンクの状態に応じたスタイル

以下のHTMLで、リンクの各状態（通常・ホバー・アクティブ・訪問済み）にスタイルを適用してください。

```html
<nav class="link-demo">
  <a href="#section1" class="styled-link">セクション1</a>
  <a href="#section2" class="styled-link">セクション2</a>
  <a href="#section3" class="styled-link">セクション3</a>
</nav>
```

**要件：**
- `.styled-link`（通常状態）：文字色 `#3498db`、下線なし、padding `8px 16px`、角丸 `4px`、トランジション `0.2s ease`
- `:hover`（マウスを乗せた時）：背景色 `#ebf5fb`、文字色 `#2980b9`
- `:active`（クリック中）：背景色 `#d4e6f1`、`transform: scale(0.98)`
- `:visited`（訪問済み）：文字色 `#8e44ad`
- `:focus-visible`（キーボードフォーカス時）：`outline: 2px solid #3498db`、`outline-offset: 2px`

<details>
<summary>ヒント</summary>

- リンクの疑似クラスには推奨される記述順があります：`:link` → `:visited` → `:hover` → `:active`（LVHA順）
- `:focus-visible` はキーボード操作時のみフォーカスリングを表示（マウスクリック時は表示しない）
</details>

<details>
<summary>解答例</summary>

```css
.link-demo {
  display: flex;
  gap: 8px;
  padding: 20px;
}

/* 通常状態 */
.styled-link {
  color: #3498db;
  text-decoration: none;
  padding: 8px 16px;
  border-radius: 4px;
  transition: all 0.2s ease;
}

/* 訪問済み */
.styled-link:visited {
  color: #8e44ad;
}

/* ホバー */
.styled-link:hover {
  background-color: #ebf5fb;
  color: #2980b9;
}

/* クリック中 */
.styled-link:active {
  background-color: #d4e6f1;
  transform: scale(0.98);
}

/* キーボードフォーカス時のみ（アクセシビリティ対応） */
.styled-link:focus-visible {
  outline: 2px solid #3498db;
  outline-offset: 2px;
}
```
</details>

### 問題2：nth-childで交互の色

以下のHTMLで、偶数行と奇数行に異なるスタイルを適用してください。

```html
<table class="styled-table">
  <thead>
    <tr>
      <th>名前</th>
      <th>役割</th>
      <th>状態</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>山田太郎</td><td>開発者</td><td>アクティブ</td></tr>
    <tr><td>佐藤花子</td><td>デザイナー</td><td>アクティブ</td></tr>
    <tr><td>鈴木一郎</td><td>マネージャー</td><td>休止中</td></tr>
    <tr><td>田中美咲</td><td>開発者</td><td>アクティブ</td></tr>
    <tr><td>高橋健太</td><td>デザイナー</td><td>アクティブ</td></tr>
  </tbody>
</table>
```

**要件：**
- `.styled-table`：幅 `100%`、最大幅 `600px`、`border-collapse: collapse`
- `th`：背景色 `#2c3e50`、文字色 白、padding `12px 16px`、テキスト左揃え
- `td`：padding `10px 16px`、枠線下 `1px solid #e0e0e0`
- `tbody tr:nth-child(even)`（偶数行）：背景色 `#f8f9fa`
- `tbody tr:hover`：背景色 `#ebf5fb`
- `tbody tr:last-child td`：枠線下なし

<details>
<summary>ヒント</summary>

- `:nth-child(even)` で偶数番目を選択（`odd` で奇数番目）
- `:last-child` で最後の要素を選択
- `border-collapse: collapse` でセルの枠線を重ねる
</details>

<details>
<summary>解答例</summary>

```css
.styled-table {
  width: 100%;
  max-width: 600px;
  /* セルの枠線を重ねて1本にする */
  border-collapse: collapse;
}

th {
  background-color: #2c3e50;
  color: white;
  padding: 12px 16px;
  text-align: left;
}

td {
  padding: 10px 16px;
  border-bottom: 1px solid #e0e0e0;
}

/* 偶数行に背景色 */
tbody tr:nth-child(even) {
  background-color: #f8f9fa;
}

/* ホバー時のハイライト */
tbody tr:hover {
  background-color: #ebf5fb;
}

/* 最後の行は下線なし */
tbody tr:last-child td {
  border-bottom: none;
}
```
</details>

### 問題3：::beforeと::afterの基本

以下のHTMLで、疑似要素を使ってラベルの前にアイコンを表示してください。

```html
<ul class="icon-list">
  <li class="icon-item icon-check">完了したタスク</li>
  <li class="icon-item icon-warn">注意が必要</li>
  <li class="icon-item icon-info">参考情報</li>
  <li class="icon-item icon-check">もう一つの完了タスク</li>
</ul>
```

**完成イメージ：**
```
✓ 完了したタスク
⚠ 注意が必要
ℹ 参考情報
✓ もう一つの完了タスク
```

**要件：**
- `.icon-list`：リストスタイルなし、padding `0`
- `.icon-item`：padding `10px 0`、枠線下 `1px solid #f0f0f0`、padding-left `28px`、position `relative`
- `.icon-check::before`：content `"✓"`、color `#2ecc71`、位置 `absolute`、left `0`、太字
- `.icon-warn::before`：content `"⚠"`、color `#f39c12`、位置 `absolute`、left `0`
- `.icon-info::before`：content `"ℹ"`、color `#3498db`、位置 `absolute`、left `0`

<details>
<summary>ヒント</summary>

- `::before` は要素の前にコンテンツを挿入する疑似要素
- `content` プロパティは必須（空でも `content: ""` が必要）
- `position: absolute` + 親の `position: relative` でアイコンを正確に配置
</details>

<details>
<summary>解答例</summary>

```css
.icon-list {
  list-style: none;
  padding: 0;
}

.icon-item {
  padding: 10px 0;
  border-bottom: 1px solid #f0f0f0;
  /* アイコンの分だけ左にスペースを確保 */
  padding-left: 28px;
  /* 疑似要素の位置決めの基準 */
  position: relative;
}

/* ✓ チェックアイコン */
.icon-check::before {
  content: "✓";
  color: #2ecc71;
  position: absolute;
  left: 0;
  font-weight: bold;
}

/* ⚠ 警告アイコン */
.icon-warn::before {
  content: "⚠";
  color: #f39c12;
  position: absolute;
  left: 0;
}

/* ℹ 情報アイコン */
.icon-info::before {
  content: "ℹ";
  color: #3498db;
  position: absolute;
  left: 0;
}
```
</details>

### 問題4：フォームの状態スタイリング

以下のHTMLで、フォーム要素の各状態に応じたスタイルを適用してください。

```html
<form class="styled-form">
  <div class="form-group">
    <label for="name">名前 <span class="required">*</span></label>
    <input type="text" id="name" class="form-input" required placeholder="山田太郎">
  </div>
  <div class="form-group">
    <label for="email">メール <span class="required">*</span></label>
    <input type="email" id="email" class="form-input" required placeholder="example@mail.com">
  </div>
  <div class="form-group">
    <label for="note">備考</label>
    <input type="text" id="note" class="form-input" placeholder="任意入力">
  </div>
  <div class="form-group">
    <label for="disabled-input">無効な項目</label>
    <input type="text" id="disabled-input" class="form-input" disabled value="変更できません">
  </div>
</form>
```

**要件：**
- `.form-group`：margin-bottom `16px`
- `label`：display `block`、文字サイズ `14px`、色 `#555`、margin-bottom `4px`、太字
- `.required`：色 `#e74c3c`
- `.form-input`（通常状態）：幅 `100%`、padding `10px 12px`、枠線 `2px solid #ddd`、角丸 `6px`、文字サイズ `15px`、トランジション `border-color 0.2s`
- `:focus`：枠線色 `#3498db`、outline なし、影 `0 0 0 3px rgba(52, 152, 219, 0.15)`
- `:disabled`：背景色 `#f5f5f5`、色 `#999`、cursor `not-allowed`
- `:placeholder-shown`（プレースホルダーが表示されている = 未入力）：枠線色 `#ddd`
- `:not(:placeholder-shown):valid`（入力済みかつ有効）：枠線色 `#2ecc71`
- `:not(:placeholder-shown):invalid`（入力済みかつ無効）：枠線色 `#e74c3c`

<details>
<summary>ヒント</summary>

- `:placeholder-shown` はプレースホルダーが表示されている状態（＝まだ何も入力していない）
- `:not(:placeholder-shown)` で「入力済み」を判定
- `:valid` / `:invalid` はHTML5バリデーションの結果に連動
- 組み合わせることで「入力済みかつ有効」を表現
</details>

<details>
<summary>解答例</summary>

```css
.styled-form {
  max-width: 400px;
  padding: 24px;
}

.form-group {
  margin-bottom: 16px;
}

label {
  display: block;
  font-size: 14px;
  color: #555;
  margin-bottom: 4px;
  font-weight: bold;
}

.required {
  color: #e74c3c;
}

/* ===== インプットの通常状態 ===== */
.form-input {
  width: 100%;
  padding: 10px 12px;
  border: 2px solid #ddd;
  border-radius: 6px;
  font-size: 15px;
  transition: border-color 0.2s;
  /* box-sizingでpaddingを含めた幅計算 */
  box-sizing: border-box;
}

/* フォーカス時 */
.form-input:focus {
  border-color: #3498db;
  outline: none;
  /* フォーカスリングの代替 */
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.15);
}

/* 無効状態 */
.form-input:disabled {
  background-color: #f5f5f5;
  color: #999;
  cursor: not-allowed;
}

/* 入力済み かつ 有効 → 緑の枠線 */
.form-input:not(:placeholder-shown):valid {
  border-color: #2ecc71;
}

/* 入力済み かつ 無効 → 赤の枠線 */
.form-input:not(:placeholder-shown):invalid {
  border-color: #e74c3c;
}
```
</details>

### 問題5：::beforeと::afterで装飾線付き見出し

以下のHTMLで、疑似要素を使って見出しの左右に装飾線を表示してください。

```html
<h2 class="decorated-heading">セクションタイトル</h2>
<h2 class="quote-heading">引用風見出し</h2>
<h2 class="badge-heading" data-badge="NEW">最新機能</h2>
```

**完成イメージ：**
```
────── セクションタイトル ──────

「 引用風見出し 」

● 最新機能 [NEW]
```

**要件：**

1. `.decorated-heading`：Flexboxで中央揃え、間隔 `16px`、色 `#2c3e50`
   - `::before` と `::after`：content `""`、高さ `2px`、`flex: 1`、背景色 `#ddd`

2. `.quote-heading`：色 `#2c3e50`、position `relative`、padding `0 40px`
   - `::before`：content `"「"`、色 `#3498db`、文字サイズ `32px`、位置 `absolute`、left `0`、top `-4px`
   - `::after`：content `"」"`、色 `#3498db`、文字サイズ `32px`、位置 `absolute`、right `0`、top `-4px`

3. `.badge-heading`：色 `#2c3e50`
   - `::before`：content `"● "`、色 `#e74c3c`
   - `::after`：content `attr(data-badge)` を使用、背景色 `#e74c3c`、文字色 白、文字サイズ `11px`、padding `3px 8px`、角丸 `4px`、margin-left `8px`、`vertical-align: middle`

<details>
<summary>ヒント</summary>

- `::before` / `::after` は `content` プロパティが必須
- `content: attr(data-badge)` でHTML属性の値を表示できる
- 装飾線は `flex: 1` + `height` + `background-color` で実現
- Flexboxの `align-items: center` で線とテキストを垂直中央に揃える
</details>

<details>
<summary>解答例</summary>

```css
/* ===== 1. 装飾線付き見出し ===== */
.decorated-heading {
  /* Flexboxで線─テキスト─線の横並び */
  display: flex;
  align-items: center;
  gap: 16px;
  color: #2c3e50;
}

.decorated-heading::before,
.decorated-heading::after {
  /* 空のコンテンツ（必須） */
  content: "";
  /* 残りのスペースを均等に埋める */
  flex: 1;
  height: 2px;
  background-color: #ddd;
}

/* ===== 2. 引用風見出し ===== */
.quote-heading {
  color: #2c3e50;
  position: relative;
  padding: 0 40px;
  display: inline-block;
}

.quote-heading::before {
  content: "「";
  color: #3498db;
  font-size: 32px;
  position: absolute;
  left: 0;
  top: -4px;
}

.quote-heading::after {
  content: "」";
  color: #3498db;
  font-size: 32px;
  position: absolute;
  right: 0;
  top: -4px;
}

/* ===== 3. バッジ付き見出し ===== */
.badge-heading {
  color: #2c3e50;
}

.badge-heading::before {
  content: "● ";
  color: #e74c3c;
}

.badge-heading::after {
  /* data-badge属性の値を表示 */
  content: attr(data-badge);
  background-color: #e74c3c;
  color: white;
  font-size: 11px;
  padding: 3px 8px;
  border-radius: 4px;
  margin-left: 8px;
  vertical-align: middle;
}
```
</details>

### 問題6：:has() セレクタを使う

以下のHTMLで、`:has()` セレクタを使って親要素のスタイルを子要素の状態に応じて変化させてください。

```html
<div class="has-demo">
  <div class="input-group">
    <label>ユーザー名</label>
    <input type="text" class="text-input" placeholder="入力してください">
  </div>

  <div class="input-group">
    <label>メールアドレス</label>
    <input type="email" class="text-input" placeholder="example@mail.com">
  </div>

  <div class="feature-card">
    <h3>通常のカード</h3>
    <p>画像のないカードです。</p>
  </div>

  <div class="feature-card">
    <div class="card-img" style="background: #3498db;"></div>
    <h3>画像付きカード</h3>
    <p>画像があるカードは別のスタイルになります。</p>
  </div>
</div>
```

**要件：**
- `.input-group`：padding `12px`、margin-bottom `12px`、角丸 `8px`、枠線 `2px solid #e0e0e0`、トランジション `border-color 0.2s`
- `.input-group:has(input:focus)`：枠線色 `#3498db`、背景色 `#f8fbff`
- `.text-input`：幅 `100%`、padding `8px`、枠線なし、outline なし、文字サイズ `15px`、背景 `transparent`
- `.feature-card`：背景色 白、padding `20px`、角丸 `12px`、枠線 `1px solid #e0e0e0`、margin-bottom `12px`
- `.feature-card:has(.card-img)`：padding `0`、overflow `hidden`
  - この場合の `h3` と `p`：padding `0 20px`
- `.card-img`：高さ `150px`

<details>
<summary>ヒント</summary>

- `:has()` は「特定の子要素を持つ親要素」を選択できる
- `.input-group:has(input:focus)` → 「フォーカス中のinputを含む.input-group」
- `.feature-card:has(.card-img)` → 「.card-imgを含む.feature-card」
- これにより、子の状態に応じて親のスタイルを変えられる
</details>

<details>
<summary>解答例</summary>

```css
.has-demo {
  max-width: 500px;
  padding: 20px;
}

/* ===== 入力グループ ===== */
.input-group {
  padding: 12px;
  margin-bottom: 12px;
  border-radius: 8px;
  border: 2px solid #e0e0e0;
  transition: border-color 0.2s, background-color 0.2s;
}

/* 子のinputがフォーカスされたら、親のスタイルを変更 */
.input-group:has(input:focus) {
  border-color: #3498db;
  background-color: #f8fbff;
}

.input-group label {
  display: block;
  font-size: 12px;
  color: #888;
  margin-bottom: 4px;
}

.text-input {
  width: 100%;
  padding: 8px;
  border: none;
  outline: none;
  font-size: 15px;
  background: transparent;
  box-sizing: border-box;
}

/* ===== フィーチャーカード ===== */
.feature-card {
  background-color: white;
  padding: 20px;
  border-radius: 12px;
  border: 1px solid #e0e0e0;
  margin-bottom: 12px;
}

/* .card-img を含むカードはpaddingをなくす */
.feature-card:has(.card-img) {
  padding: 0;
  overflow: hidden;
}

/* 画像付きカードのテキストにpaddingを付ける */
.feature-card:has(.card-img) h3,
.feature-card:has(.card-img) p {
  padding: 0 20px;
}

.card-img {
  height: 150px;
}
```
</details>

### 問題7：カスタムチェックボックスとラジオボタン

以下のHTMLで、疑似要素を使ったカスタムチェックボックスとラジオボタンを作ってください。

```html
<div class="custom-controls">
  <h3>チェックボックス</h3>
  <label class="custom-checkbox">
    <input type="checkbox">
    <span class="checkbox-label">利用規約に同意する</span>
  </label>
  <label class="custom-checkbox">
    <input type="checkbox" checked>
    <span class="checkbox-label">ニュースレターを受け取る</span>
  </label>

  <h3>ラジオボタン</h3>
  <label class="custom-radio">
    <input type="radio" name="plan" checked>
    <span class="radio-label">月額プラン</span>
  </label>
  <label class="custom-radio">
    <input type="radio" name="plan">
    <span class="radio-label">年額プラン</span>
  </label>
</div>
```

**要件：**
- ネイティブの `input` は非表示にする（`appearance: none` + サイズ0）
- `.custom-checkbox`：display `flex`、align-items `center`、gap `8px`、padding `8px`、cursor `pointer`
- `.checkbox-label::before`：content `""`、幅 `20px`、高さ `20px`、枠線 `2px solid #ccc`、角丸 `4px`、display `inline-flex`、justify-content `center`、align-items `center`、トランジション `0.2s`
- チェック済み：`.custom-checkbox:has(input:checked) .checkbox-label::before`
  - 背景色 `#3498db`、枠線色 `#3498db`、content `"✓"`、文字色 白、文字サイズ `14px`

- `.custom-radio`：同様の構造
- `.radio-label::before`：角丸 `50%`（円形）
- チェック済み：背景色 白、枠線色 `#3498db`、`box-shadow: inset 0 0 0 4px #3498db`

<details>
<summary>ヒント</summary>

- `appearance: none` でネイティブのチェックボックス/ラジオの見た目を消す
- `:has(input:checked)` でチェック状態の判定ができる
- ラジオボタンの内側の塗りは `box-shadow: inset` で実現
</details>

<details>
<summary>解答例</summary>

```css
.custom-controls {
  max-width: 400px;
  padding: 20px;
}

/* ===== ネイティブinputを非表示 ===== */
.custom-checkbox input,
.custom-radio input {
  /* ブラウザデフォルトの見た目を消す */
  appearance: none;
  width: 0;
  height: 0;
  margin: 0;
  position: absolute;
}

/* ===== チェックボックス ===== */
.custom-checkbox {
  display: flex;
  align-items: center;
  gap: 8px;
  padding: 8px;
  cursor: pointer;
}

/* 疑似要素でカスタムチェックボックスを描画 */
.checkbox-label::before {
  content: "";
  width: 20px;
  height: 20px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-flex;
  justify-content: center;
  align-items: center;
  transition: all 0.2s;
  flex-shrink: 0;
}

/* チェック済みの状態 */
.custom-checkbox:has(input:checked) .checkbox-label::before {
  background-color: #3498db;
  border-color: #3498db;
  content: "✓";
  color: white;
  font-size: 14px;
}

/* ===== ラジオボタン ===== */
.custom-radio {
  display: flex;
  align-items: center;
  gap: 8px;
  padding: 8px;
  cursor: pointer;
}

/* 疑似要素でカスタムラジオを描画 */
.radio-label::before {
  content: "";
```

### 問題8：ミニプロジェクト ─ ツールチップ＆ドロップダウンメニュー

以下のHTMLで、CSSのみでツールチップとドロップダウンメニューを作成してください。

```html
<div class="demo-area">
  <!-- ツールチップ -->
  <p>
    これは
    <span class="tooltip" data-tip="CSSの疑似要素でツールチップを作れます！">ツールチップ</span>
    のデモです。テキストにホバーしてみてください。
  </p>

  <!-- ドロップダウンメニュー -->
  <nav class="dropdown-nav">
    <div class="dropdown">
      <button class="dropdown-trigger">メニュー ▼</button>
      <ul class="dropdown-menu">
        <li><a href="#">プロフィール</a></li>
        <li><a href="#">設定</a></li>
        <li><a href="#">ヘルプ</a></li>
        <li class="dropdown-divider"></li>
        <li><a href="#" class="dropdown-danger">ログアウト</a></li>
      </ul>
    </div>
  </nav>
</div>
```

**要件：**

**ツールチップ：**
- `.tooltip`：position `relative`、枠線下 `1px dashed #3498db`、色 `#3498db`、cursor `help`
- `.tooltip::after`：content `attr(data-tip)`、位置 `absolute`、bottom `calc(100% + 8px)`、left `50%`、`transform: translateX(-50%)`、背景色 `#2c3e50`、文字色 白、padding `8px 12px`、角丸 `6px`、文字サイズ `13px`、`white-space: nowrap`
  - 通常時：`opacity: 0`、`pointer-events: none`、`transform: translateX(-50%) translateY(4px)`
  - ホバー時：`opacity: 1`、`transform: translateX(-50%) translateY(0)`
  - トランジション：`0.2s ease`
- `.tooltip::before`（吹き出しの三角形）：content `""`、位置 `absolute`、bottom `calc(100% + 2px)`、left `50%`、`transform: translateX(-50%)`、border `6px solid transparent`、`border-top-color: #2c3e50`
  - 通常時とホバー時は `::after` と同じ表示/非表示

**ドロップダウンメニュー：**
- `.dropdown`：position `relative`、display `inline-block`
- `.dropdown-trigger`：背景色 `#3498db`、文字色 白、padding `10px 20px`、枠線なし、角丸 `6px`、cursor `pointer`、文字サイズ `14px`
- `.dropdown-menu`：position `absolute`、top `calc(100% + 4px)`、left `0`、背景色 白、枠線 `1px solid #e0e0e0`、角丸 `8px`、影 `0 4px 16px rgba(0,0,0,0.12)`、リストスタイルなし、padding `8px 0`、margin `0`、最小幅 `180px`
  - 通常時：`opacity: 0`、`visibility: hidden`、`transform: translateY(-8px)`
  - ホバー時（`.dropdown:hover .dropdown-menu`）：`opacity: 1`、`visibility: visible`、`transform: translateY(0)`
  - トランジション：`0.2s ease`
- `.dropdown-menu a`：display `block`、padding `8px 16px`、文字色 `#333`、下線なし、文字サイズ `14px`
  - ホバー時：背景色 `#f0f0f0`
- `.dropdown-divider`：高さ `1px`、背景色 `#e0e0e0`、margin `4px 0`
- `.dropdown-danger`：色 `#e74c3c`

<details>
<summary>ヒント</summary>

- ツールチップは `::after` でテキスト、`::before` で三角形を表示
- `attr(data-tip)` でHTML属性の値をcontentに使える
- ドロップダウンは `:hover` でメニューを表示/非表示
- `visibility: hidden` は `display: none` と違いトランジションが効く
- `pointer-events: none` でホバー判定を無効にする（消えている時にマウスが当たらないように）
</details>

<details>
<summary>解答例</summary>
