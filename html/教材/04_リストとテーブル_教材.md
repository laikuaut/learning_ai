# 第4章：リストとテーブル

## この章で学ぶこと
- 順序なしリスト（ul / li）
- 順序付きリスト（ol / li）
- 定義リスト（dl / dt / dd）
- リストの入れ子（ネスト）

## 4.1 順序なしリスト（ul / li）

順序が重要でない項目の一覧には `<ul>`（Unordered List）を使います。

### 基本構文
```html
<ul>
  <li>りんご</li>
  <li>みかん</li>
  <li>バナナ</li>
</ul>
```

### 表示結果（デフォルト）
- りんご
- みかん
- バナナ

### ulの特徴
- 各項目の前に**ビュレット**（・）が表示される
- 項目の順番を入れ替えても意味が変わらない場合に使う
- `<ul>` の直接の子要素は `<li>` のみ

```html
<!-- 良い例 -->
<ul>
  <li>項目1</li>
  <li>項目2</li>
</ul>

<!-- 悪い例：ulの直接の子要素にli以外を入れてはいけない -->
<ul>
  <p>これは間違い</p>
  <li>項目</li>
</ul>
```

### CSSでビュレットのスタイルを変更
```html
<!-- disc（デフォルト）、circle、square、none -->
<ul style="list-style-type: circle;">
  <li>白い丸</li>
  <li>白い丸</li>
</ul>

<ul style="list-style-type: none;">
  <li>ビュレットなし（ナビゲーション等に便利）</li>
</ul>
```

## 4.2 順序付きリスト（ol / li）

順序が重要な項目の一覧には `<ol>`（Ordered List）を使います。

### 基本構文
```html
<ol>
  <li>卵を割ってボウルに入れる</li>
  <li>砂糖を加えてよく混ぜる</li>
  <li>薄力粉をふるい入れる</li>
  <li>オーブンで180度で30分焼く</li>
</ol>
```

### 表示結果（デフォルト）
1. 卵を割ってボウルに入れる
2. 砂糖を加えてよく混ぜる
3. 薄力粉をふるい入れる
4. オーブンで180度で30分焼く

### ol要素の属性
| 属性 | 説明 | 例 |
|---|---|---|
| `start` | 開始番号を指定 | `<ol start="5">` |
| `reversed` | 逆順にする | `<ol reversed>` |
| `type` | 番号の種類 | `<ol type="a">` |

### type属性の値
| 値 | 表示 | 例 |
|---|---|---|
| `1` | 数字（デフォルト） | 1, 2, 3... |
| `a` | 小文字アルファベット | a, b, c... |
| `A` | 大文字アルファベット | A, B, C... |
| `i` | 小文字ローマ数字 | i, ii, iii... |
| `I` | 大文字ローマ数字 | I, II, III... |

```html
<!-- 5番から開始 -->
<ol start="5">
  <li>5番目の項目</li>
  <li>6番目の項目</li>
</ol>

<!-- アルファベット表記 -->
<ol type="a">
  <li>選択肢a</li>
  <li>選択肢b</li>
  <li>選択肢c</li>
</ol>

<!-- 逆順カウントダウン -->
<ol reversed>
  <li>3位：Cチーム</li>
  <li>2位：Bチーム</li>
  <li>1位：Aチーム</li>
</ol>
```

### li要素のvalue属性
個別の項目の番号を指定できます。

```html
<ol>
  <li>通常（1）</li>
  <li value="10">10番にジャンプ</li>
  <li>11（続きから）</li>
</ol>
```

## 4.3 定義リスト（dl / dt / dd）

用語とその定義のペアを記述するには `<dl>`（Description List）を使います。

### 基本構文
```html
<dl>
  <dt>HTML</dt>
  <dd>Webページの構造を記述するマークアップ言語</dd>

  <dt>CSS</dt>
  <dd>Webページの見た目を装飾するスタイルシート言語</dd>

  <dt>JavaScript</dt>
  <dd>Webページに動的な機能を追加するプログラミング言語</dd>
</dl>
```

### 各要素の役割
| 要素 | 名前 | 役割 |
|---|---|---|
| `<dl>` | Description List | リスト全体のコンテナ |
| `<dt>` | Description Term | 用語（名前） |
| `<dd>` | Description Details | 定義（説明） |

### 1つの用語に複数の定義
```html
<dl>
  <dt>SET</dt>
  <dd>（動詞）設定する</dd>
  <dd>（名詞）セット、一組</dd>
  <dd>（形容詞）固定された</dd>
</dl>
```

### 複数の用語に1つの定義
```html
<dl>
  <dt>著者</dt>
  <dt>Author</dt>
  <dd>本や記事を書いた人</dd>
</dl>
```

### 定義リストの活用例
```html
<!-- メタデータ的な情報 -->
<dl>
  <dt>投稿日</dt>
  <dd><time datetime="2024-03-15">2024年3月15日</time></dd>

  <dt>カテゴリ</dt>
  <dd>プログラミング</dd>

  <dt>タグ</dt>
  <dd>HTML, Web開発, 入門</dd>
</dl>

<!-- FAQ -->
<dl>
  <dt>送料はいくらですか？</dt>
  <dd>全国一律500円です。5,000円以上のご注文で送料無料になります。</dd>

  <dt>返品はできますか？</dt>
  <dd>商品到着から7日以内であれば返品可能です。</dd>
</dl>
```

## 4.4 リストの入れ子（ネスト）

リストの中にリストを入れることができます。

```html
<ul>
  <li>フロントエンド
    <ul>
      <li>HTML</li>
      <li>CSS
        <ul>
          <li>Flexbox</li>
          <li>Grid</li>
        </ul>
      </li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>バックエンド
    <ul>
      <li>Python</li>
      <li>Node.js</li>
      <li>PHP</li>
    </ul>
  </li>
</ul>
```

### 入れ子のルール
- 内側のリストは `<li>` の**中に**配置する
- `<ul>` と `<ol>` を混在させることもできる
- 深い入れ子はビュレットのスタイルが自動で変わる（●→○→■）

```html
<!-- ulとolの混在 -->
<ol>
  <li>朝食のメニュー
    <ul>
      <li>トースト</li>
      <li>目玉焼き</li>
      <li>サラダ</li>
    </ul>
  </li>
  <li>昼食のメニュー
    <ul>
      <li>パスタ</li>
      <li>スープ</li>
    </ul>
  </li>
</ol>
```

## 4.5 テーブルの基本構造

### 最小限のテーブル
```html
<table>
  <tr>
    <th>名前</th>
    <th>年齢</th>
    <th>職業</th>
  </tr>
  <tr>
    <td>田中太郎</td>
    <td>30</td>
    <td>エンジニア</td>
  </tr>
  <tr>
    <td>鈴木花子</td>
    <td>25</td>
    <td>デザイナー</td>
  </tr>
</table>
```

### 各要素の役割
| 要素 | 名前 | 役割 |
|---|---|---|
| `<table>` | Table | テーブル全体のコンテナ |
| `<tr>` | Table Row | テーブルの行 |
| `<th>` | Table Header | ヘッダーセル（太字・中央寄せ） |
| `<td>` | Table Data | データセル |

### thとtdの違い
- `<th>` はデフォルトで**太字**かつ**中央揃え**
- `<td>` はデフォルトで**通常の太さ**かつ**左揃え**
- `<th>` はスクリーンリーダーが「見出し」として読み上げる
- 列の見出しだけでなく、行の見出しにも `<th>` を使う

```html
<table>
  <tr>
    <th></th>
    <th>1月</th>
    <th>2月</th>
    <th>3月</th>
  </tr>
  <tr>
    <th>売上</th>  <!-- 行の見出しにもthを使う -->
    <td>100万円</td>
    <td>120万円</td>
    <td>150万円</td>
  </tr>
  <tr>
    <th>利益</th>
    <td>30万円</td>
    <td>40万円</td>
    <td>50万円</td>
  </tr>
</table>
```

## 4.6 テーブルのグループ化（thead / tbody / tfoot）

大きなテーブルでは、ヘッダー・ボディ・フッターをグループ化します。

```html
<table>
  <thead>
    <tr>
      <th>商品名</th>
      <th>単価</th>
      <th>数量</th>
      <th>小計</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ノートPC</td>
      <td>150,000円</td>
      <td>2</td>
      <td>300,000円</td>
    </tr>
    <tr>
      <td>マウス</td>
      <td>3,000円</td>
      <td>5</td>
      <td>15,000円</td>
    </tr>
    <tr>
      <td>キーボード</td>
      <td>8,000円</td>
      <td>3</td>
      <td>24,000円</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th colspan="3">合計</th>
      <td>339,000円</td>
    </tr>
  </tfoot>
</table>
```

### グループ化のメリット
| 要素 | 役割 | メリット |
|---|---|---|
| `<thead>` | ヘッダー行 | 印刷時に各ページにヘッダーが繰り返される |
| `<tbody>` | データ行 | CSSでスタイルを一括指定しやすい |
| `<tfoot>` | フッター行 | 合計値など。tbodyの前に書いてもbodyの後に表示される |

> **ポイント**: `<tbody>` は省略しても自動的に補完されますが、明示的に書くのが良い習慣です。

## 4.7 セルの結合（colspan / rowspan）

### colspan：列の結合（横方向）
```html
<table>
  <tr>
    <th colspan="3">2024年度 売上報告</th>
  </tr>
  <tr>
    <th>Q1</th>
    <th>Q2</th>
    <th>Q3</th>
  </tr>
  <tr>
    <td>100万</td>
    <td>120万</td>
    <td>150万</td>
  </tr>
</table>
```

`colspan="3"` は「このセルは3列分の幅を占める」という意味です。

### rowspan：行の結合（縦方向）
```html
<table>
  <tr>
    <th>カテゴリ</th>
    <th>商品</th>
    <th>価格</th>
  </tr>
  <tr>
    <td rowspan="2">果物</td>
    <td>りんご</td>
    <td>200円</td>
  </tr>
  <tr>
    <!-- 「果物」のセルは上の行に結合されているのでここには書かない -->
    <td>みかん</td>
    <td>100円</td>
  </tr>
  <tr>
    <td rowspan="2">野菜</td>
    <td>にんじん</td>
    <td>150円</td>
  </tr>
  <tr>
    <td>キャベツ</td>
    <td>200円</td>
  </tr>
</table>
```

### colspan と rowspan の組み合わせ
```html
<table>
  <tr>
    <th rowspan="2">時間</th>
    <th colspan="2">月曜日</th>
    <th colspan="2">火曜日</th>
  </tr>
  <tr>
    <th>科目</th>
    <th>教室</th>
    <th>科目</th>
    <th>教室</th>
  </tr>
  <tr>
    <td>9:00</td>
    <td>数学</td>
    <td>101</td>
    <td>英語</td>
    <td>202</td>
  </tr>
  <tr>
    <td>10:00</td>
    <td colspan="2">体育（体育館）</td>
    <td>理科</td>
    <td>301</td>
  </tr>
</table>
```

### セル結合時の注意点
- 結合によって「消える」セルは、HTMLコード上では**記述しない**
- 行と列の数が合わなくなると、テーブルの表示が崩れる
- 複雑な結合は表を見ながら慎重に組み立てる

## 4.8 テーブルのアクセシビリティ

### caption要素：テーブルのキャプション
```html
<table>
  <caption>2024年度 四半期別売上一覧</caption>
  <thead>
    <tr>
      <th>四半期</th>
      <th>売上</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Q1</td>
      <td>100万円</td>
    </tr>
  </tbody>
</table>
```

- `<caption>` は `<table>` の**最初の子要素**として配置
- テーブルの内容を要約するタイトルを記述
- スクリーンリーダーがテーブルの内容を理解する助けになる

### scope属性：ヘッダーの適用方向
```html
<table>
  <tr>
    <th scope="col">名前</th>
    <th scope="col">点数</th>
  </tr>
  <tr>
    <th scope="row">田中</th>
    <td>85</td>
  </tr>
  <tr>
    <th scope="row">鈴木</th>
    <td>92</td>
  </tr>
</table>
```

| 値 | 意味 |
|---|---|
| `col` | この見出しは列（下方向）に適用 |
| `row` | この見出しは行（右方向）に適用 |
| `colgroup` | 列グループに適用 |
| `rowgroup` | 行グループに適用 |

## 4.9 テーブルの列グループ（colgroup / col）

列に対してスタイルを一括適用するために `<colgroup>` と `<col>` を使います。

```html
<table>
  <colgroup>
    <col>
    <col style="background-color: #f0f0f0;">
    <col style="background-color: #e0e0e0;">
  </colgroup>
  <tr>
    <th>名前</th>
    <th>数学</th>
    <th>英語</th>
  </tr>
  <tr>
    <td>田中</td>
    <td>85</td>
    <td>90</td>
  </tr>
</table>
```

### span属性で複数列をまとめる
```html
<colgroup>
  <col>                          <!-- 1列目 -->
  <col span="3" style="background-color: #f0f0f0;">  <!-- 2〜4列目 -->
  <col span="2" style="background-color: #e0e0e0;">  <!-- 5〜6列目 -->
</colgroup>
```

## 4.10 テーブルの使い方のルール

### テーブルを使うべき場面
- 表形式のデータ（スプレッドシート的な情報）
- スケジュール、時刻表
- 比較表
- 統計データ

### テーブルを使うべきでない場面
- ページのレイアウト（CSSのFlexboxやGridを使う）
- 視覚的な整列のためだけ
- リスト形式で十分な場合

```html
<!-- 悪い例：レイアウトのためにテーブルを使っている -->
<table>
  <tr>
    <td>サイドバー</td>
    <td>メインコンテンツ</td>
  </tr>
</table>

<!-- 良い例：CSSでレイアウトする -->
<div style="display: flex;">
  <aside>サイドバー</aside>
  <main>メインコンテンツ</main>
</div>
```

## まとめ

| 要素 | 用途 |
|---|---|
| `<ul>` / `<li>` | 順序なしリスト |

| `<td>` | データセル |

| `<caption>` | テーブルのキャプション |
| `<colgroup>` / `<col>` | 列のグループ化・スタイル |
