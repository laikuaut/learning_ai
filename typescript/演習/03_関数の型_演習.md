## 問題1：基本的な関数の型注釈（基本）

以下の3つの関数に適切な型注釈（パラメータ型・戻り値型）を付けて実装してください。

1. `add(a, b)` — 2つの数値を足す
2. `greet(name)` — 名前を受け取り挨拶文を返す
3. `isEven(n)` — 数値が偶数なら `true` を返す

**期待される出力：**
```
add(3, 5) = 8
greet("太郎") = こんにちは、太郎さん！
isEven(4) = true
isEven(7) = false
```

<details>
<summary>解答例</summary>

```typescript
// 関数宣言に型注釈を付ける
function add(a: number, b: number): number {
  return a + b;
}

function greet(name: string): string {
  return `こんにちは、${name}さん！`;
}

function isEven(n: number): boolean {
  return n % 2 === 0;
}

console.log(`add(3, 5) = ${add(3, 5)}`);
console.log(`greet("太郎") = ${greet("太郎")}`);
console.log(`isEven(4) = ${isEven(4)}`);
console.log(`isEven(7) = ${isEven(7)}`);
```

</details>

## 問題2：オプショナルパラメータとデフォルト値（基本）

以下の関数を実装してください。

`formatName(firstName, lastName, honorific?)` — フルネームを整形して返す
- `firstName`: 名（必須）
- `lastName`: 姓（必須）
- `honorific`: 敬称（省略時は `"さん"`）

**期待される出力：**
```
田中太郎さん
山田花子先生
鈴木一郎様
```

<details>
<summary>ヒント</summary>

- デフォルト値は `パラメータ: 型 = デフォルト値` で指定します。
- デフォルト値がある場合、`?` は不要です。

</details>

<details>
<summary>解答例</summary>

```typescript
// デフォルト値付きパラメータ
function formatName(
  firstName: string,
  lastName: string,
  honorific: string = "さん" // デフォルト値
): string {
  return `${lastName}${firstName}${honorific}`;
}

console.log(formatName("太郎", "田中"));           // デフォルトの「さん」
console.log(formatName("花子", "山田", "先生"));   // 指定した敬称
console.log(formatName("一郎", "鈴木", "様"));     // 指定した敬称
```

</details>

## 問題3：残余パラメータ（基本）

可変長引数を受け取る以下の2つの関数を実装してください。

1. `sum(...numbers)` — 全ての数値の合計を返す
2. `joinWithSeparator(separator, ...items)` — 区切り文字で文字列を結合する

**期待される出力：**
```
sum(1, 2, 3, 4, 5) = 15
joinWithSeparator(" | ", "A", "B", "C") = A | B | C
```

<details>
<summary>解答例</summary>

```typescript
// 残余パラメータで可変長引数を受け取る
function sum(...numbers: number[]): number {
  return numbers.reduce((total, n) => total + n, 0);
}

// 固定パラメータとの併用
function joinWithSeparator(separator: string, ...items: string[]): string {
  return items.join(separator);
}

console.log(`sum(1, 2, 3, 4, 5) = ${sum(1, 2, 3, 4, 5)}`);
console.log(
  `joinWithSeparator(" | ", "A", "B", "C") = ${joinWithSeparator(" | ", "A", "B", "C")}`
);
```

</details>

## 問題4：関数型の定義（応用）

以下の要件を満たすコードを書いてください。

1. `MathOperation` 型エイリアス：2つの `number` を受け取り `number` を返す関数型
2. 4つの演算関数を作成：`add`, `subtract`, `multiply`, `divide`
3. `calculate(a, b, operation)` 関数：`MathOperation` 型の関数を受け取って計算結果を返す

**期待される出力：**
```
10 + 3 = 13
10 - 3 = 7
10 * 3 = 30
10 / 3 = 3.33
```

<details>
<summary>ヒント</summary>

- 関数型は `type Name = (引数: 型) => 戻り値型` で定義します。
- 高階関数（関数を引数に取る関数）を使います。

</details>

<details>
<summary>解答例</summary>

```typescript
// 関数型の定義
type MathOperation = (a: number, b: number) => number;

// 各演算関数（アロー関数で定義）
const add: MathOperation = (a, b) => a + b;
const subtract: MathOperation = (a, b) => a - b;
const multiply: MathOperation = (a, b) => a * b;
const divide: MathOperation = (a, b) => a / b;

// 高階関数：MathOperation型の関数を受け取る
function calculate(a: number, b: number, operation: MathOperation): number {
  return operation(a, b);
}

// 計算と表示
const a = 10;
const b = 3;

console.log(`${a} + ${b} = ${calculate(a, b, add)}`);
console.log(`${a} - ${b} = ${calculate(a, b, subtract)}`);
console.log(`${a} * ${b} = ${calculate(a, b, multiply)}`);
console.log(`${a} / ${b} = ${calculate(a, b, divide).toFixed(2)}`);
```

</details>

## 問題5：関数オーバーロード（応用）

`stringify` 関数をオーバーロードで実装してください。

- `string` を受け取ったら → そのまま返す
- `number` を受け取ったら → 小数点2桁の文字列にして返す
- `boolean` を受け取ったら → `"はい"` または `"いいえ"` を返す
- `Date` を受け取ったら → `"YYYY/MM/DD"` 形式の文字列を返す

**期待される出力：**
```
stringify("hello") = hello
stringify(3.14159) = 3.14
stringify(true) = はい
stringify(false) = いいえ
stringify(new Date("2026-03-25")) = 2026/03/25
```

<details>
<summary>ヒント</summary>

- オーバーロードシグネチャを複数書き、最後に実装シグネチャを書きます。
- `typeof` と `instanceof` で型を判定します。

</details>

<details>
<summary>解答例</summary>

```typescript
// オーバーロードシグネチャ
function stringify(value: string): string;
function stringify(value: number): string;
function stringify(value: boolean): string;
function stringify(value: Date): string;

// 実装シグネチャ
function stringify(value: string | number | boolean | Date): string {
  if (typeof value === "string") {
    return value;
  } else if (typeof value === "number") {
    return value.toFixed(2);
  } else if (typeof value === "boolean") {
    return value ? "はい" : "いいえ";
  } else {
    // Date型
    const y = value.getFullYear();
    const m = String(value.getMonth() + 1).padStart(2, "0");
    const d = String(value.getDate()).padStart(2, "0");
    return `${y}/${m}/${d}`;
  }
}

console.log(`stringify("hello") = ${stringify("hello")}`);
console.log(`stringify(3.14159) = ${stringify(3.14159)}`);
console.log(`stringify(true) = ${stringify(true)}`);
console.log(`stringify(false) = ${stringify(false)}`);
console.log(`stringify(new Date("2026-03-25")) = ${stringify(new Date("2026-03-25"))}`);
```

</details>

## 問題6：コールバック関数の型（応用）

以下の要件を満たすイベントシステムの型を定義してください。

1. `EventCallback` 型：`(eventName: string, data: unknown) => void`
2. `EventSystem` インターフェース：`on(event: string, callback: EventCallback): void` と `emit(event: string, data: unknown): void`
3. `EventSystem` を実装したオブジェクトを作成し、イベントの登録と発火を行う

**期待される出力：**
```
イベント受信 [click]: {"x":100,"y":200}
イベント受信 [submit]: {"form":"login"}
```

<details>
<summary>解答例</summary>

```typescript
// コールバックの型定義
type EventCallback = (eventName: string, data: unknown) => void;

// イベントシステムのインターフェース
interface EventSystem {
  on(event: string, callback: EventCallback): void;
  emit(event: string, data: unknown): void;
}

// 実装
function createEventSystem(): EventSystem {
  const listeners: { [event: string]: EventCallback[] } = {};

  return {
    on(event: string, callback: EventCallback): void {
      if (!listeners[event]) {
        listeners[event] = [];
      }
      listeners[event].push(callback);
    },
    emit(event: string, data: unknown): void {
      const callbacks = listeners[event];
      if (callbacks) {
        callbacks.forEach((cb) => cb(event, data));
      }
    },
  };
}

// 動作確認
const events = createEventSystem();

// コールバックを登録
events.on("click", (eventName, data) => {
  console.log(`イベント受信 [${eventName}]: ${JSON.stringify(data)}`);
});

events.on("submit", (eventName, data) => {
  console.log(`イベント受信 [${eventName}]: ${JSON.stringify(data)}`);
});

// イベント発火
events.emit("click", { x: 100, y: 200 });
events.emit("submit", { form: "login" });
```

</details>

## 問題7：アサーション関数（チャレンジ）

`asserts` キーワードを使って、以下の2つのアサーション関数を実装してください。

1. `assertIsString(value)` — 文字列でなければ例外を投げる
2. `assertInRange(value, min, max)` — 数値が範囲内でなければ例外を投げる

これらを使って、ユーザー入力のバリデーションを行ってください。

**期待される出力：**
```
名前: 太郎（有効）
年齢: 25（有効）
エラー: Expected string, got number
エラー: 値 200 は 0〜150 の範囲外です
```

<details>
<summary>解答例</summary>

```typescript
// アサーション関数：文字列であることを保証
function assertIsString(value: unknown): asserts value is string {
  if (typeof value !== "string") {
    throw new Error(`Expected string, got ${typeof value}`);
  }
}

// アサーション関数：範囲内であることを保証
function assertInRange(
  value: number,
  min: number,
  max: number
): asserts value is number {
  if (value < min || value > max) {
    throw new Error(`値 ${value} は ${min}〜${max} の範囲外です`);
  }
}

// バリデーション関数
function validateUserInput(name: unknown, age: unknown): void {
  // 名前のバリデーション
  try {
    assertIsString(name);
    // ここでnameはstring型として使える
    console.log(`名前: ${name}（有効）`);
  } catch (e) {
    console.log(`エラー: ${(e as Error).message}`);
  }

  // 年齢のバリデーション
  try {
    if (typeof age !== "number") throw new Error(`Expected number, got ${typeof age}`);
    assertInRange(age, 0, 150);
    console.log(`年齢: ${age}（有効）`);
  } catch (e) {
    console.log(`エラー: ${(e as Error).message}`);
  }
}

// 正常なデータ
validateUserInput("太郎", 25);

// 不正なデータ
validateUserInput(42, 200);
```

</details>

## 問題8：ビルダーパターンの型付け（チャレンジ）

HTMLタグを組み立てるビルダーを型安全に実装してください。

```typescript
interface HtmlBuilder {
  tag(name: string): HtmlBuilder;
  attr(key: string, value: string): HtmlBuilder;
  text(content: string): HtmlBuilder;
  build(): string;
}
```

**期待される出力：**
```
<a href="https://example.com" target="_blank">こちらをクリック</a>
<div class="container" id="main">Hello, TypeScript!</div>
```

<details>
<summary>解答例</summary>

```typescript
interface HtmlBuilder {
  tag(name: string): HtmlBuilder;
  attr(key: string, value: string): HtmlBuilder;
  text(content: string): HtmlBuilder;
  build(): string;
}

function createHtmlBuilder(): HtmlBuilder {
  let tagName = "div";
  const attrs: { key: string; value: string }[] = [];
  let textContent = "";

  const builder: HtmlBuilder = {
    tag(name: string): HtmlBuilder {
      tagName = name;
      return builder;
    },
    attr(key: string, value: string): HtmlBuilder {
      attrs.push({ key, value });
      return builder;
    },
    text(content: string): HtmlBuilder {
      textContent = content;
      return builder;
    },
    build(): string {
      const attrStr = attrs
        .map((a) => ` ${a.key}="${a.value}"`)
        .join("");
      return `<${tagName}${attrStr}>${textContent}</${tagName}>`;
    },
  };

  return builder;
}

// メソッドチェーンで型安全にHTMLを組み立て
const link = createHtmlBuilder()
  .tag("a")
  .attr("href", "https://example.com")
  .attr("target", "_blank")
  .text("こちらをクリック")
  .build();

const div = createHtmlBuilder()
  .tag("div")
  .attr("class", "container")
  .attr("id", "main")
  .text("Hello, TypeScript!")
  .build();

console.log(link);
console.log(div);
```

</details>
