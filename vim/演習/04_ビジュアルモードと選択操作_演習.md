# 第4章 演習：ビジュアルモードと選択操作

この演習では、ビジュアルモードの3種類（文字・行・矩形選択）と選択範囲への操作を学びます。

---

## 基本問題

### 問題1：ビジュアルモードの種類

3種類のビジュアルモードについて、以下の表の空欄を埋めてください。

| キー | モード名 | 選択単位 | 画面下部の表示 |
|---|---|---|---|
| `v` | （A） | （B） | （C） |
| `V` | （D） | （E） | （F） |
| `Ctrl+v` | （G） | （H） | （I） |

**期待される解答例：**

| キー | モード名 | 選択単位 | 画面下部の表示 |
|---|---|---|---|
| `v` | ビジュアルモード | 文字単位 | `-- VISUAL --` |
| `V` | ビジュアルラインモード | 行単位 | `-- VISUAL LINE --` |
| `Ctrl+v` | ビジュアルブロックモード | 矩形単位 | `-- VISUAL BLOCK --` |

<details>
<summary>ヒント</summary>

小文字 `v` は文字単位、大文字 `V` は行単位、`Ctrl+v` は矩形（ブロック）単位の選択です。

</details>

<details>
<summary>解答と解説</summary>

```vim
v       " ビジュアルモード（文字単位）。テキストの一部分を選択する場合に使います
V       " ビジュアルラインモード（行単位）。最もよく使うモードです
Ctrl+v  " ビジュアルブロックモード（矩形）。複数行の同じ列を一括編集できます

" 共通操作:
" 選択後に d で削除、y でコピー、c で変更ができます
" ESC で選択を解除してノーマルモードに戻ります
```

</details>

---

### 問題2：行選択でのコピーと削除

以下のコードで `V`（ビジュアルラインモード）を使って操作してください。

```python
def greet(name):
    print("こんにちは")
    print("お元気ですか")
    print("さようなら")
    return True
```

1. 2〜3行目をコピーして `return True` の下に貼り付ける
2. 4行目（`さようなら`）を削除する

**期待される出力（問題1の後）：**

```python
def greet(name):
    print("こんにちは")
    print("お元気ですか")
    print("さようなら")
    return True
    print("こんにちは")
    print("お元気ですか")
```

<details>
<summary>ヒント</summary>

`V` で行選択 → `j`/`k` で範囲拡張 → `y` でコピー / `d` で削除。`p` で下に貼り付けです。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 問題1: 2〜3行目をコピーして貼り付け
:2[Enter]  " 2行目に移動
V          " 行選択モードに入る
j          " 3行目まで選択範囲を拡張
y          " コピー（「2 lines yanked」と表示されます）
/return[Enter]  " return の行に移動
p          " 下に貼り付け

" 問題2: 4行目を削除
" 4行目にカーソルを移動して:
Vd         " 行選択して即削除
" 1行だけなら dd の方が速いですが、V は目視確認しながら操作できます
```

</details>

### 問題3：インデント操作

以下のインデントが崩れたコードを修正してください。

**修正前：**

```javascript
function process() {
var x = 10;
var y = 20;
if (x > 0) {
console.log(x);
console.log(y);
}
}
```

**期待される出力：**

```javascript
function process() {
    var x = 10;
    var y = 20;
    if (x > 0) {
        console.log(x);
        console.log(y);
    }
}
```

<details>
<summary>ヒント</summary>

`V` で行選択 → `>` でインデント追加、`<` でインデント削除です。`=` で自動インデントもできます。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 方法1: 手動でインデント調整
" 2〜7行目を選択して1段インデント
:2[Enter]
V5j>       " 2〜7行目を選択して > でインデント追加

" 5〜6行目（console.log）をさらに1段インデント
:5[Enter]
Vj>        " 5〜6行目を選択して > でインデント追加

" 方法2: 自動インデント（最も簡単）
ggVG=      " gg で先頭、V で行選択、G で最終行まで、= で自動インデント

" 方法3: ノーマルモードで
6>>        " 2行目で実行。6行分を1段インデント（>> は現在行をインデント）

" > : インデント追加、< : インデント削除、= : 自動インデント
```

</details>

## 応用問題

### 問題4：矩形選択で複数行コメントアウト

矩形選択（`Ctrl+v`）を使って、2〜5行目をコメントアウト（行頭に `# ` を追加）してください。

**修正前：**

```python
def calculate():
    x = 10
    y = 20
    z = x + y
    print(z)
    return z
```

**期待される出力：**

```python
def calculate():
    # x = 10
    # y = 20
    # z = x + y
    # print(z)
    return z
```

<details>
<summary>ヒント</summary>

`Ctrl+v` → 範囲選択 → `I`（大文字）→ 文字入力 → `ESC` で全行に一括挿入されます。

</details>

<details>
<summary>解答と解説</summary>

```vim
" 1. 2行目のインデント後の最初の文字に移動
:2[Enter]
^          " 行の最初の非空白文字（x）に移動

" 2. 矩形選択して4行分選択
Ctrl+v     " 矩形選択モードに入る
3j         " 下に3行（2〜5行目の同じ列が選択される）

" 3. 一括挿入
I          " 大文字 I でブロック挿入モードに入る
# [スペース] " 「# 」と入力

" 4. 確定
ESC        " 全選択行に「# 」が挿入されます

" ポイント:
" - ESC を押した直後は1行だけに見えますが、全行に反映されます
" - I は左端に挿入、A は右端に追加です
```

</details>

### 問題5：矩形選択での文字列挿入

以下のCSVデータに対して、矩形選択を使って2つの操作を行ってください。

```
apple,100,fruit
banana,80,fruit
carrot,120,vegetable
daikon,90,vegetable
```

**操作1：** 全行の先頭に `ID,` を一括挿入する

**操作1の出力：**

```
ID,apple,100,fruit
ID,banana,80,fruit
ID,carrot,120,vegetable
ID,daikon,90,vegetable
```

**操作2：** さらに全行の末尾に `,available` を一括追加する

**操作2の出力：**

```
ID,apple,100,fruit,available
ID,banana,80,fruit,available
ID,carrot,120,vegetable,available
ID,daikon,90,vegetable,available
```

<details>
<summary>ヒント</summary>

行頭：`Ctrl+v` → 範囲選択 → `I` → 入力 → `ESC`。行末：`Ctrl+v` → 範囲選択 → `$` → `A` → 入力 → `ESC`。

</details>

<details>
<summary>解答と解説</summary>

```vim
" === 操作1: 行頭に「ID,」を挿入 ===
gg0        " ファイル先頭の行頭に移動
Ctrl+v     " 矩形選択モードに入る
3j         " 最終行まで選択
I          " ブロック挿入モード
ID,        " テキスト入力
ESC        " 全行に反映

" === 操作2: 行末に「,available」を追加 ===
gg         " 先頭行に移動
Ctrl+v     " 矩形選択モードに入る
3j         " 最終行まで選択
$          " 各行の末尾まで選択（行の長さが違っても対応）
A          " ブロック追加モード（各行の末尾に追加）
,available " テキスト入力
ESC        " 全行に反映

" $ を使った矩形選択は行の長さが不揃いでも正しく動作します
```

</details>

## チャレンジ問題

### 問題6：矩形選択を活用したデータ整形

以下のログデータに対して3つの操作を行ってください。

```
2024-01-15 09:00:00 INFO  UserLogin    admin
2024-01-15 09:05:23 WARN  DiskSpace    server01
2024-01-15 09:10:45 ERROR DBConnect    db-master
2024-01-15 09:15:12 INFO  FileUpload   user123
2024-01-15 09:20:33 ERROR APITimeout   gateway
```

**操作1：** 日付列（`2024-01-15 `）を矩形選択で削除する

```
09:00:00 INFO  UserLogin    admin
09:05:23 WARN  DiskSpace    server01
09:10:45 ERROR DBConnect    db-master
09:15:12 INFO  FileUpload   user123
09:20:33 ERROR APITimeout   gateway
```

**操作2：** ログレベル列（`INFO `/`WARN `/`ERROR`）を矩形選択でコピーし、末尾に貼り付ける

**操作3：** 全行の先頭に `[LOG] ` を矩形選択で一括挿入する

```
[LOG] 09:00:00 INFO  UserLogin    admin
[LOG] 09:05:23 WARN  DiskSpace    server01
[LOG] 09:10:45 ERROR DBConnect    db-master
[LOG] 09:15:12 INFO  FileUpload   user123
[LOG] 09:20:33 ERROR APITimeout   gateway
```

<details>
<summary>ヒント</summary>

- 操作1：`Ctrl+v` → 日付部分を矩形選択 → `d`
- 操作2：`Ctrl+v` → ログレベル列を選択 → `y` → `G` → `p`
- 操作3：`Ctrl+v` → 全行選択 → `I` → `[LOG] ` → `ESC`

</details>

<details>
<summary>解答と解説</summary>

```vim
" === 操作1: 日付列の削除 ===
gg0        " 先頭行の行頭に移動
Ctrl+v     " 矩形選択モード
10l        " 「2024-01-15 」の範囲を選択（11文字分）
4j         " 最終行まで選択
d          " 矩形範囲を一括削除

" === 操作2: ログレベル列のコピー ===
gg         " 先頭行に移動
9l         " INFO の先頭（I）に移動（時刻の後）
Ctrl+v     " 矩形選択モード
4l         " 5文字分選択（ERROR が最長）
4j         " 最終行まで選択
y          " コピー
G          " ファイル末尾に移動
o[ESC]     " 空行を作成
p          " 矩形コピーは矩形のまま貼り付けられます

" === 操作3: プレフィックスの一括挿入 ===
gg0        " 先頭行の行頭に移動
Ctrl+v     " 矩形選択モード
4j         " 最終行まで選択
I          " ブロック挿入モード
[LOG]      " テキスト入力（[LOG] + スペース）
ESC        " 全行の先頭に挿入されます

" 矩形選択の主な使い方まとめ:
" 矩形削除:       Ctrl+v → 範囲選択 → d
" 矩形コピー:     Ctrl+v → 範囲選択 → y → 移動 → p
" 一括挿入（左端）: Ctrl+v → 範囲選択 → I → 入力 → ESC
" 一括追加（右端）: Ctrl+v → 範囲選択 → $ → A → 入力 → ESC
```

</details>
