## 課題の説明

以下の「タスク管理システム」には **10個のバグ** が埋め込まれています。
プログラムをブラウザで開き、エラーメッセージや意図しない動作を手がかりにすべてのバグを見つけて修正してください。

### 進め方

1. 以下のコードをそのまま HTMLファイルとして保存する
2. ブラウザで開いてコンソール（F12）を確認する
3. 各機能を試して不具合を発見する
4. 1つずつバグを修正し、再読み込みで確認する
5. 全機能が正常に動作すれば完了

## バグ入りコード

以下のコードをそのまま `task_manager_buggy.html` として保存してください。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>タスク管理（バグ入り版）</title>
  <style>
    body { font-family: sans-serif; max-width: 600px; margin: 30px auto; padding: 0 20px; }
    .input-row { display: flex; gap: 8px; margin-bottom: 16px; }
    .input-row input, .input-row select { font-size: 14px; padding: 6px; }
    .input-row input { flex: 1; }
    .input-row button { padding: 6px 16px; cursor: pointer; }
    .task { display: flex; align-items: center; padding: 8px; border-bottom: 1px solid #eee; }
    .task.done .task-name { text-decoration: line-through; color: #999; }
    .task-name { flex: 1; cursor: pointer; }
    .task-priority { margin: 0 8px; font-size: 12px; padding: 2px 8px; border-radius: 4px; }
    .high { background: #ffcdd2; }
    .medium { background: #fff9c4; }
    .low { background: #c8e6c9; }
    .delete-btn { background: #e74c3c; color: white; border: none; padding: 4px 8px;
                  cursor: pointer; border-radius: 4px; }
    #stats { margin-top: 16px; padding: 12px; background: #f5f5f5; border-radius: 8px; }
  </style>
</head>
<body>
  <h1>タスク管理</h1>
  <div class="input-row">
    <input type="text" id="taskInput" placeholder="タスク名">
    <select id="prioritySelect">
      <option value="high">高</option>
      <option value="medium" selected>中</option>
      <option value="low">低</option>
    </select>
    <button onclick="addTask()">追加</button>
  </div>
  <div id="taskList"></div>
  <div id="stats"></div>

  <script>
    let tasks = [];
    let nextId = 1;

    function addTask() {
      const input = document.getElementById("taskInput");
      const name = input.value;  // バグ1
      const priority = document.getElementById("prioritySelect").value;

      if (name == "") {  // バグ2
        alert("タスク名を入力してください");
        return;
      }

      const task = {
        id: nextId++,
        name: name,
        priority: priority,
        done: false,
        createdAt: Date.now  // バグ3
      };
      tasks.push(task);
      input.value = "";
      renderTasks();
    }

    function toggleTask(id) {
      for (let i = 0; i <= tasks.length; i++) {  // バグ4
        if (tasks[i].id === id) {
          tasks[i].done = !tasks[i].done;
          break;
        }
      }
      renderTasks();
    }

    function deleteTask(id) {
      tasks = tasks.filter(function(task) {
        return task.id === id;  // バグ5
      });
      renderTasks();
    }

    function renderTasks() {
      const listEl = document.getElementById("taskList");
      listEl.innerHTML = "";

      // 優先度順にソート
      const sorted = tasks.sort(function(a, b) {  // バグ6
        const order = { high: 1, medium: 2, low: 3 };
        return order[a.priority] - order[b.priority];
      });

      for (var i = 0; i < sorted.length; i++) {
        const task = sorted[i];
        const div = document.createElement("div");
        div.className = "task" + (task.done ? " done" : "");

        const nameSpan = document.createElement("span");
        nameSpan.className = "task-name";
        nameSpan.textContent = task.name;
        nameSpan.addEventListener("click", function() {
          toggleTask(task.id);
        });

        const prioritySpan = document.createElement("span");
        prioritySpan.className = "task-priority " + task.priority;
        const priorityLabels = { high: "高", medium: "中", low: "低" };
        prioritySpan.textContent = priorityLabels[task.prioritiy];  // バグ7

        const deleteBtn = document.createElement("button");
        deleteBtn.className = "delete-btn";
        deleteBtn.textContent = "削除";
        deleteBtn.addEventListener("click", function() {
          deleteTask(task.id);
        });

        div.appendChild(nameSpan);
        div.appendChild(prioritySpan);
        div.appendChild(deleteBtn);
        listEl.appendChild(div);
      }

      updateStats();
    }

    function updateStats() {
      const total = tasks.length;
      let doneCount = 0;
      for (let i = 0; i < tasks.length; i++) {
        if (tasks[i].done) doneCount++;
      }
      const remaining = total - doneCount;

      // 完了率を計算
      const percent = (doneCount / total * 100).toFixed(1);  // バグ8

      const highCount = tasks.filter(t => t.priority === "high" && !t.done);  // バグ9

      document.getElementById("stats").innerHTML =
        "全" + total + "件 | 完了: " + doneCount + "件 | " +
        "残り: " + remaining + "件 | 完了率: " + percent + "% | " +
        "未完了(高): " + highCount + "件";
    }

    // Enterキーで追加
    document.getElementById("taskInput").addEvenListener("keydown", function(e) {  // バグ10
      if (e.key === "Enter") {
        addTask();
      }
    });

    renderTasks();
  </script>
</body>
</html>
```

## 解答例

<details>
<summary>全10個のバグと修正方法</summary>

### バグ1：文字列の前後の空白が除去されていない

```javascript
// 修正前
const name = input.value;
// 修正後
const name = input.value.trim();
```

空白だけのタスク名が追加できてしまいます。`.trim()` で前後の空白を除去しましょう。

### バグ2：`==` ではなく `===` を使うべき

```javascript
// 修正前
if (name == "") {
// 修正後
if (name === "") {
```

`==` は型変換を行うため予期しない一致が起こりえます。JavaScriptでは `===`（厳密等価演算子）を使うのがベストプラクティスです。このケースでは動作に影響しませんが、`===` に統一すべきです。

### バグ3：`Date.now` はメソッドなので `()` が必要

```javascript
// 修正前
createdAt: Date.now
// 修正後
createdAt: Date.now()
```

`Date.now` は関数への参照で、`Date.now()` が実際にタイムスタンプを返す呼び出しです。`()` をつけないと関数オブジェクトそのものが代入されます。

### バグ4：ループ条件の境界エラー（off-by-one）

```javascript
// 修正前
for (let i = 0; i <= tasks.length; i++) {
// 修正後
for (let i = 0; i < tasks.length; i++) {
```

`<=` だと `tasks[tasks.length]` にアクセスしてしまい、`undefined` でエラーになります。配列のインデックスは `0` 〜 `length - 1` です。

### バグ5：フィルタの条件が逆

```javascript
// 修正前（IDが一致するものだけ残す＝削除したいものだけ残してしまう）
return task.id === id;
// 修正後（IDが一致しないものを残す）
return task.id !== id;
```

`filter()` は条件が `true` の要素を残します。削除したいIDの要素を「残す」条件にしてしまっています。

### バグ6：`sort()` が元の配列を破壊する

```javascript
// 修正前（tasks配列自体がソートされてしまう）
const sorted = tasks.sort(function(a, b) {
// 修正後（コピーしてからソート）
const sorted = tasks.slice().sort(function(a, b) {
```

`Array.prototype.sort()` は元の配列を変更する破壊的メソッドです。`.slice()` でコピーを作ってからソートしましょう。

### バグ7：プロパティ名のタイポ

```javascript
// 修正前
prioritySpan.textContent = priorityLabels[task.prioritiy];
// 修正後
prioritySpan.textContent = priorityLabels[task.priority];
```

`prioritiy` は `priority` のタイポです。存在しないプロパティは `undefined` を返します。

### バグ8：タスクが0件のときゼロ除算

```javascript
// 修正前
const percent = (doneCount / total * 100).toFixed(1);
// 修正後
const percent = total > 0 ? (doneCount / total * 100).toFixed(1) : "0.0";
```

`total` が 0 のとき `0 / 0` は `NaN` になります。

### バグ9：`filter()` は配列を返す（数値ではない）

```javascript
// 修正前（filterの結果は配列であり、件数ではない）
const highCount = tasks.filter(t => t.priority === "high" && !t.done);
// 修正後（.lengthで件数を取得）
const highCount = tasks.filter(t => t.priority === "high" && !t.done).length;
```

`filter()` は条件に合う要素の**配列**を返します。件数を知りたいなら `.length` が必要です。

### バグ10：メソッド名のタイポ

```javascript
// 修正前
document.getElementById("taskInput").addEvenListener("keydown", function(e) {
// 修正後
document.getElementById("taskInput").addEventListener("keydown", function(e) {
```

`addEvenListener` ではなく `addEventListener` です。1文字欠けているため `TypeError` が発生します。
