## 問題1：偶数・奇数判定（基本）

変数 `num = 7` が偶数か奇数かを判定して表示してください。

**期待される出力：**
```
7は奇数です
```

<details>
<summary>ヒント</summary>

余り演算子 `%` を使います。`num % 2 == 0` なら偶数です。

</details>

<details>
<summary>解答例</summary>

```python
num = 7

if num % 2 == 0:
    print(f"{num}は偶数です")
else:
    print(f"{num}は奇数です")
```

</details>

## 問題2：成績判定（基本）

テストの点数 `score = 82` に対して、以下の基準で評価を表示してください。

- 90点以上：「優秀！」
- 80点以上：「良い成績です」
- 60点以上：「合格です」
- 60点未満：「不合格です」

**期待される出力：**
```
82点: 良い成績です
```

<details>
<summary>解答例</summary>

```python
score = 82

if score >= 90:
    result = "優秀！"
elif score >= 80:
    result = "良い成績です"
elif score >= 60:
    result = "合格です"
else:
    result = "不合格です"

print(f"{score}点: {result}")
```

</details>

## 問題3：年齢区分（基本）

年齢に応じて料金区分を表示してください。

- 6歳未満：「無料」
- 6歳以上12歳以下：「子ども料金（500円）」
- 13歳以上64歳以下：「大人料金（1000円）」
- 65歳以上：「シニア料金（700円）」

`age = 10` のときの出力を確認してください。

**期待される出力：**
```
10歳: 子ども料金（500円）
```

<details>
<summary>解答例</summary>

```python
age = 10

if age < 6:
    category = "無料"
elif age <= 12:
    category = "子ども料金（500円）"
elif age <= 64:
    category = "大人料金（1000円）"
else:
    category = "シニア料金（700円）"

print(f"{age}歳: {category}")
```

</details>

## 問題4：論理演算子の練習（基本）

以下の条件をプログラムで判定してください。

- `temperature = 28`, `is_sunny = True` のとき
- 気温が25度以上**かつ**晴れなら「海に行こう！」
- 気温が25度以上**だが**晴れでないなら「プールに行こう！」
- それ以外なら「家にいよう」

**期待される出力：**
```
海に行こう！
```

<details>
<summary>解答例</summary>

```python
temperature = 28
is_sunny = True

if temperature >= 25 and is_sunny:
    print("海に行こう！")
elif temperature >= 25 and not is_sunny:
    print("プールに行こう！")
else:
    print("家にいよう")
```

</details>

## 問題5：うるう年判定（応用）

年を表す変数 `year` がうるう年かどうかを判定してください。

**うるう年の条件：**
1. 4で割り切れる年はうるう年
2. ただし、100で割り切れる年はうるう年ではない
3. ただし、400で割り切れる年はうるう年

`year = 2024` で確認してください。

**期待される出力：**
```
2024年はうるう年です
```

<details>
<summary>ヒント</summary>

条件を `and` / `or` で組み合わせるか、ネストした `if` 文を使います。

</details>

<details>
<summary>解答例</summary>

```python
year = 2024

if (year % 400 == 0) or (year % 4 == 0 and year % 100 != 0):
    print(f"{year}年はうるう年です")
else:
    print(f"{year}年はうるう年ではありません")
```

</details>

## 問題6：じゃんけん（応用）

プレイヤーの手 `player = "グー"` とコンピュータの手 `computer = "チョキ"` で、勝敗を判定してください。

**期待される出力：**
```
プレイヤー: グー vs コンピュータ: チョキ
結果: あなたの勝ち！
```

<details>
<summary>ヒント</summary>

勝つパターンを列挙しましょう：
- グー vs チョキ
- チョキ vs パー
- パー vs グー

</details>

<details>
<summary>解答例</summary>

```python
player = "グー"
computer = "チョキ"

print(f"プレイヤー: {player} vs コンピュータ: {computer}")

if player == computer:
    print("結果: あいこ！")
elif (player == "グー" and computer == "チョキ") or \
     (player == "チョキ" and computer == "パー") or \
     (player == "パー" and computer == "グー"):
    print("結果: あなたの勝ち！")
else:
    print("結果: あなたの負け...")
```

</details>

## 問題7：BMI判定（応用）

第1章で作成したBMI計算に、以下の判定を追加してください。

| BMI | 判定 |
|-----|------|
| 18.5未満 | 低体重 |
| 18.5以上25未満 | 普通体重 |
| 25以上30未満 | 肥満（1度） |
| 30以上 | 肥満（2度以上） |

`height_cm = 170`, `weight_kg = 65` で確認してください。

**期待される出力：**
```
BMI: 22.49
判定: 普通体重
```

<details>
<summary>解答例</summary>

```python
height_cm = 170
weight_kg = 65

height_m = height_cm / 100
bmi = weight_kg / (height_m ** 2)

if bmi < 18.5:
    category = "低体重"
elif bmi < 25:
    category = "普通体重"
elif bmi < 30:
    category = "肥満（1度）"
else:
    category = "肥満（2度以上）"

print(f"BMI: {bmi:.2f}")
print(f"判定: {category}")
```

</details>

## 問題8：月の日数（チャレンジ）

変数 `month`（1〜12）に対応する月の日数を表示してください。
2月はうるう年かどうかで日数が変わります。`year = 2024`, `month = 2` で確認してください。

**期待される出力：**
```
2024年2月は29日あります
```

<details>
<summary>ヒント</summary>

- 31日の月: 1, 3, 5, 7, 8, 10, 12月
- 30日の月: 4, 6, 9, 11月
- 2月: うるう年なら29日、そうでなければ28日
- `in` 演算子を使うと便利です

</details>

<details>
<summary>解答例</summary>

```python
year = 2024
month = 2

if month in [1, 3, 5, 7, 8, 10, 12]:
    days = 31
elif month in [4, 6, 9, 11]:
    days = 30
elif month == 2:
    if (year % 400 == 0) or (year % 4 == 0 and year % 100 != 0):
        days = 29
    else:
        days = 28

print(f"{year}年{month}月は{days}日あります")
```

</details>

## 問題9：電気料金計算（チャレンジ）

電気使用量（kWh）に応じて料金を計算してください。

| 使用量 | 単価（円/kWh） |
|--------|----------------|
| 120kWhまで | 20円 |
| 121〜300kWh | 25円 |
| 301kWh以上 | 30円 |

※ 段階的に計算します。例えば250kWhなら：
- 最初の120kWh × 20円 = 2400円
- 残り130kWh × 25円 = 3250円
- 合計: 5650円

`usage = 250` で確認してください。

**期待される出力：**
```
使用量: 250kWh
電気料金: 5650円
```

<details>
<summary>ヒント</summary>

段階ごとに計算して合計します。条件分岐で場合分けしましょう。

</details>

<details>
<summary>解答例</summary>

```python
usage = 250

if usage <= 120:
    cost = usage * 20
elif usage <= 300:
    cost = 120 * 20 + (usage - 120) * 25
else:
    cost = 120 * 20 + 180 * 25 + (usage - 300) * 30

print(f"使用量: {usage}kWh")
print(f"電気料金: {cost}円")
```

</details>

## 問題10：パスワードチェック（チャレンジ）

パスワード文字列が以下のすべての条件を満たすか判定してください。

1. 8文字以上であること
2. 数字が含まれていること（`any(c.isdigit() for c in password)`）
3. 英字が含まれていること（`any(c.isalpha() for c in password)`）

条件を満たさない場合は、どの条件が不足しているか表示してください。

`password = "abc123"` で確認してください。

**期待される出力：**
```
パスワード: abc123
NG: 8文字以上にしてください（現在6文字）
```

<details>
<summary>ヒント</summary>

- `len(password)` で文字数を取得
- `any()` は1つでもTrueがあればTrueを返す
- 条件ごとにチェックして、NGの理由を表示しましょう

</details>

<details>
<summary>解答例</summary>

```python
password = "abc123"
is_valid = True

print(f"パスワード: {password}")

if len(password) < 8:
    print(f"NG: 8文字以上にしてください（現在{len(password)}文字）")
    is_valid = False

if not any(c.isdigit() for c in password):
    print("NG: 数字を含めてください")
    is_valid = False

if not any(c.isalpha() for c in password):
    print("NG: 英字を含めてください")
    is_valid = False

if is_valid:
    print("OK: パスワードは有効です")
```

</details>
