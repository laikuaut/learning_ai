# 実践課題02：レシピカード ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（HTMLの基本構造）、第2章（テキストとリンク）、第3章（画像とマルチメディア）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 見出し・段落・リスト・画像を組み合わせて、実用的なレシピカードを作る

---

## 完成イメージ

ブラウザで開くと以下のようなレシピカードが表示されます。

```
============================
  カレーライスの作り方
============================

[カレーの画像（プレースホルダー）]

調理時間：約45分  |  4人分

■ 材料
  ・ 豚肉 ........... 300g
  ・ じゃがいも ..... 3個
  ・ にんじん ....... 1本
  ・ 玉ねぎ ......... 2個
  ・ カレールー ..... 1箱
  ・ 水 ............. 800ml
  ・ サラダ油 ....... 大さじ1

■ 作り方
  1. 野菜と肉を一口大に切る
  2. 鍋に油を熱し、肉を炒める
  3. 野菜を加えて炒める
  4. 水を加えて沸騰させる
  5. アクを取りながら20分煮込む
  6. 火を止めてルーを入れる
  7. 再び弱火で10分煮込む

■ ポイント
  ルーを入れるときは必ず火を止めて
  ください。ダマになるのを防げます。
```

---

## 課題の要件

1. 正しいHTML基本構造を持つこと
2. `<h1>` でレシピ名を表示すること
3. `<img>` で画像を表示すること（画像がない場合はプレースホルダーを使用）
4. 調理時間と分量を `<p>` で表示すること
5. 材料を `<ul>`（順序なしリスト）で表示すること
6. 作り方を `<ol>`（順序付きリスト）で表示すること
7. ポイントを `<p>` と `<strong>` で強調して表示すること

---

## ステップガイド

<details>
<summary>ステップ1：基本構造と見出しを作る</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>カレーライスのレシピ</title>
</head>
<body>
  <h1>カレーライスの作り方</h1>
</body>
</html>
```

</details>

<details>
<summary>ステップ2：画像を追加する</summary>

画像がない場合は、プレースホルダー画像サービスを使うか、`alt` 属性だけで表現します。

```html
<!-- 方法1: プレースホルダー画像を使う -->
<img src="https://via.placeholder.com/400x250?text=Curry+Rice"
     alt="カレーライスの完成写真"
     width="400" height="250">

<!-- 方法2: 画像なしでaltテキストだけ表示させる -->
<img src="curry.jpg"
     alt="カレーライスの完成写真"
     width="400" height="250">
```

`alt` 属性は画像が表示できないときの代替テキスト（alternative text）です。必ず設定しましょう。

</details>

<details>
<summary>ステップ3：リストを作る</summary>

材料は順番が関係ないので `<ul>`、作り方は順番が重要なので `<ol>` を使います。

```html
<h2>材料</h2>
<ul>
  <li>豚肉 ... 300g</li>
  <li>じゃがいも ... 3個</li>
</ul>

<h2>作り方</h2>
<ol>
  <li>野菜と肉を一口大に切る</li>
  <li>鍋に油を熱し、肉を炒める</li>
</ol>
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>カレーライスのレシピ</title>
</head>
<body>
  <h1>カレーライスの作り方</h1>

  <img src="https://via.placeholder.com/400x250/FF8C00/FFFFFF?text=Curry+Rice"
       alt="カレーライスの完成写真"
       width="400" height="250">

  <p>調理時間：約45分 ｜ 4人分</p>

  <h2>材料</h2>
  <ul>
    <li>豚肉 ─ 300g</li>
    <li>じゃがいも ─ 3個</li>
    <li>にんじん ─ 1本</li>
    <li>玉ねぎ ─ 2個</li>
    <li>カレールー ─ 1箱</li>
    <li>水 ─ 800ml</li>
    <li>サラダ油 ─ 大さじ1</li>
  </ul>

  <h2>作り方</h2>
  <ol>
    <li>野菜と肉を一口大に切る</li>
    <li>鍋にサラダ油を熱し、肉を中火で炒める</li>
    <li>肉の色が変わったら野菜を加えて炒める</li>
    <li>水800mlを加えて強火で沸騰させる</li>
    <li>アクを取りながら中火で約20分煮込む</li>
    <li><strong>火を止めて</strong>カレールーを割り入れてよく溶かす</li>
    <li>再び弱火で約10分煮込んで完成</li>
  </ol>

  <h2>ポイント</h2>
  <p><strong>ルーを入れるときは必ず火を止めてください。</strong>沸騰した状態でルーを入れるとダマになりやすくなります。</p>
  <p>一晩寝かせると味がなじんでさらに美味しくなります。</p>
</body>
</html>
```

</details>

<details>
<summary>解答例（改良版 ─ テーブルと構造化を活用）</summary>

第4章（リストとテーブル）の知識を先取りして、材料をテーブルで表現した改良版です。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>カレーライスのレシピ</title>
</head>
<body>
  <header>
    <h1>カレーライスの作り方</h1>
  </header>

  <main>
    <figure>
      <img src="https://via.placeholder.com/400x250/FF8C00/FFFFFF?text=Curry+Rice"
           alt="カレーライスの完成写真"
           width="400" height="250">
      <figcaption>カレーライスの完成イメージ</figcaption>
    </figure>

    <p><strong>調理時間：</strong>約45分 ｜ <strong>分量：</strong>4人分</p>

    <section>
      <h2>材料（4人分）</h2>
      <table border="1">
        <thead>
          <tr>
            <th>材料</th>
            <th>分量</th>
          </tr>
        </thead>
        <tbody>
          <tr><td>豚肉</td><td>300g</td></tr>
          <tr><td>じゃがいも</td><td>3個</td></tr>
          <tr><td>にんじん</td><td>1本</td></tr>
          <tr><td>玉ねぎ</td><td>2個</td></tr>
          <tr><td>カレールー</td><td>1箱</td></tr>
          <tr><td>水</td><td>800ml</td></tr>
          <tr><td>サラダ油</td><td>大さじ1</td></tr>
        </tbody>
      </table>
    </section>

    <section>
      <h2>作り方</h2>
      <ol>
        <li>野菜と肉を一口大に切る</li>
        <li>鍋にサラダ油を熱し、肉を中火で炒める</li>
        <li>肉の色が変わったら野菜を加えて炒める</li>
        <li>水800mlを加えて強火で沸騰させる</li>
        <li>アクを取りながら中火で約20分煮込む</li>
        <li><strong>火を止めて</strong>カレールーを割り入れてよく溶かす</li>
        <li>再び弱火で約10分煮込んで完成</li>
      </ol>
    </section>

    <aside>
      <h2>ポイント</h2>
      <ul>
        <li><strong>ルーを入れるときは必ず火を止める</strong> ─ ダマ防止のため</li>
        <li><strong>一晩寝かせる</strong> ─ 味がなじんで美味しくなる</li>
        <li><strong>隠し味</strong> ─ ケチャップ大さじ1を加えるとコクが出る</li>
      </ul>
    </aside>
  </main>

  <footer>
    <p><small>参考：<a href="https://www.house-foods.co.jp/" target="_blank" rel="noopener noreferrer">ハウス食品公式サイト</a></small></p>
  </footer>
</body>
</html>
```

**初心者向けとの違い:**
- `<figure>` と `<figcaption>` で画像にキャプションを付加 → 画像の説明が構造的に明確
- `<table>` で材料を表形式に → 材料名と分量の対応が一目瞭然
- `<section>` でコンテンツを意味的に分割 → ページ構造が明確
- `<aside>` でポイントを補足情報として明示 → 本文と補足の区別がつく
- `<footer>` で参考リンクを追加 → 出典の明示

</details>
