## 5.1 関数とは？

**関数**は、処理をひとまとめにして名前をつけたものです。
同じ処理を何度も書かなくてよくなります。

```python
# 関数の定義
def greet():
    print("こんにちは！")

# 関数の呼び出し
greet()  # こんにちは！
greet()  # こんにちは！
```

### なぜ関数を使うのか？

1. **再利用**: 同じコードを何度も書かなくてよい
2. **整理**: プログラムを機能ごとに分けて読みやすくする
3. **修正が楽**: 1箇所を直せば全部に反映される

## 5.2 引数（パラメータ）

関数に値を渡すことができます。

```python
def greet(name):
    print(f"こんにちは、{name}さん！")

greet("太郎")   # こんにちは、太郎さん！
greet("花子")   # こんにちは、花子さん！
```

### 複数の引数

```python
def introduce(name, age):
    print(f"{name}さんは{age}歳です")

introduce("太郎", 20)  # 太郎さんは20歳です
```

---

## 5.3 戻り値（return）

関数から値を返すことができます。

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8
```

### 複数の値を返す

```python
def min_max(numbers):
    return min(numbers), max(numbers)

minimum, maximum = min_max([3, 1, 4, 1, 5, 9])
print(f"最小: {minimum}, 最大: {maximum}")  # 最小: 1, 最大: 9
```

### returnがない関数

`return` を書かないと `None` が返ります。

```python
def greet(name):
    print(f"こんにちは、{name}さん！")

result = greet("太郎")
print(result)  # None
```

## 5.4 デフォルト引数

引数にデフォルト値を設定できます。呼び出し時に省略できます。

```python
def greet(name, greeting="こんにちは"):
    print(f"{greeting}、{name}さん！")

greet("太郎")              # こんにちは、太郎さん！
greet("太郎", "おはよう")  # おはよう、太郎さん！
```

> **注意**: デフォルト引数は、デフォルト値のない引数の**後**に書く必要があります。

## 5.5 キーワード引数

引数を名前で指定して渡せます。順番を気にしなくてよくなります。

```python
def create_profile(name, age, city):
    print(f"名前: {name}, 年齢: {age}, 住所: {city}")

# 位置引数（順番通り）
create_profile("太郎", 20, "東京")

# キーワード引数（名前で指定）
create_profile(city="大阪", name="花子", age=25)
```

## 5.6 可変長引数

引数の数が決まっていないとき。

### *args（位置引数の可変長）

```python
def total(*numbers):
    return sum(numbers)

print(total(1, 2, 3))        # 6
print(total(10, 20, 30, 40)) # 100
```

### **kwargs（キーワード引数の可変長）

```python
def show_info(**info):
    for key, value in info.items():
        print(f"{key}: {value}")

show_info(name="太郎", age=20, city="東京")
```

**出力：**
```
name: 太郎
age: 20
city: 東京
```

## 5.7 スコープ（変数の有効範囲）

関数の中で作った変数は、関数の外からは使えません。

```python
def my_function():
    local_var = "ローカル変数"
    print(local_var)

my_function()       # ローカル変数
# print(local_var)  # エラー！関数の外からは見えない
```

### グローバル変数とローカル変数

```python
message = "グローバル"  # グローバル変数

def show_message():
    message = "ローカル"  # ローカル変数（別の変数）
    print(message)

show_message()    # ローカル
print(message)    # グローバル（変わっていない）
```

> **コツ**: グローバル変数をなるべく使わず、引数と戻り値でデータをやり取りするのが良い設計です。

## 5.8 関数の設計のコツ

### 1つの関数は1つの仕事

```python
# 悪い例：やることが多すぎる
def process_student(name, scores):
    total = sum(scores)
    avg = total / len(scores)
    if avg >= 60:
        status = "合格"
    else:
        status = "不合格"
    print(f"{name}: {avg:.1f}点 ({status})")

# 良い例：機能を分割
def calculate_average(scores):
    return sum(scores) / len(scores)

def judge_pass(average):
    return "合格" if average >= 60 else "不合格"

def show_result(name, scores):
    avg = calculate_average(scores)
    status = judge_pass(avg)
    print(f"{name}: {avg:.1f}点 ({status})")
```

### 意味のある名前をつける

```python
# 悪い例
def f(x, y):
    return x * y * 0.1

# 良い例
def calculate_tax(price, tax_rate):
    return price * tax_rate * 0.1
```

## 5.9 よく使う組み込み関数

| 関数 | 機能 | 例 |
|------|------|-----|
| `len()` | 長さ・要素数 | `len([1, 2, 3])` → `3` |
| `sum()` | 合計 | `sum([1, 2, 3])` → `6` |
| `max()` / `min()` | 最大・最小 | `max(3, 1, 4)` → `4` |
| `abs()` | 絶対値 | `abs(-5)` → `5` |
| `round()` | 四捨五入 | `round(3.14, 1)` → `3.1` |
| `sorted()` | ソート（新しいリスト） | `sorted([3, 1, 2])` → `[1, 2, 3]` |
| `reversed()` | 逆順イテレータ | `list(reversed([1, 2, 3]))` → `[3, 2, 1]` |
| `enumerate()` | インデックス付きループ | 第3章参照 |
| `zip()` | 複数リストの同時ループ | 下記参照 |
| `map()` | 全要素に関数を適用 | 下記参照 |

### zip() の使い方

```python
names = ["太郎", "花子", "次郎"]
scores = [85, 92, 78]

for name, score in zip(names, scores):
    print(f"{name}: {score}点")
```

### map() の使い方

```python
numbers = ["1", "2", "3", "4", "5"]
int_numbers = list(map(int, numbers))
print(int_numbers)  # [1, 2, 3, 4, 5]
```
