# 第8章：疑似クラスと疑似要素

## この章で学ぶこと

- 疑似クラス（:hover, :focus, :active など）
- 構造疑似クラス（:nth-child, :first-child, :not() など）
- 関数型疑似クラス（:has(), :is(), :where()）
- 疑似要素（::before, ::after, ::placeholder, ::selection）
- 実践的な活用パターン

---

## 1. 疑似クラスとは？

**疑似クラス** は、要素の「状態」や「位置」に基づいてスタイルを適用します。HTMLにクラスを追加せずに、特定の条件に合う要素を選択できます。

```css
/* 疑似クラスの書き方：セレクタの後に :疑似クラス名 */
a:hover {
  color: red;
}
```

---

## 2. ユーザーアクション疑似クラス

### :hover（マウスを乗せた時）

```css
.button {
  background-color: #3498db;
  color: white;
  padding: 12px 24px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: #2980b9;
}

/* リンクのホバー */
a:hover {
  color: #e74c3c;
  text-decoration: underline;
}
```

### :focus（フォーカスされた時）

キーボード操作やクリックで要素がフォーカスされた時に適用されます。

```css
input:focus {
  outline: none;                      /* デフォルトの枠線を消す */
  border-color: #3498db;
  box-shadow: 0 0 0 3px rgba(52, 152, 219, 0.3);
}

/* キーボードフォーカスのみ（マウスクリックでは適用されない） */
.button:focus-visible {
  outline: 2px solid #3498db;
  outline-offset: 2px;
}
```

> **アクセシビリティ：** `outline: none` だけを書くとキーボード操作者が困ります。必ず代替のフォーカススタイルを用意しましょう。

### :active（クリック中）

マウスボタンを押している間に適用されます。

```css
.button:active {
  transform: scale(0.98);         /* 少し縮む */
  background-color: #1a6da7;
}
```

### リンクの疑似クラス

```css
a:link { color: #3498db; }      /* 未訪問のリンク */
a:visited { color: #8e44ad; }   /* 訪問済みのリンク */
a:hover { color: #e74c3c; }     /* ホバー中 */
a:active { color: #c0392b; }    /* クリック中 */
```

> **LVHA順序：** `:link` → `:visited` → `:hover` → `:active` の順に書きます（Love HAte と覚える）。

---

## 3. フォーム関連の疑似クラス

```css
/* 無効な入力欄 */
input:disabled {
  background-color: #f0f0f0;
  cursor: not-allowed;
}

/* 有効な入力欄 */
input:enabled {
  background-color: white;
}

/* チェックされたチェックボックス */
input:checked {
  accent-color: #3498db;
}

/* チェックされたチェックボックスの隣のラベル */
input:checked + label {
  font-weight: bold;
  color: #3498db;
}

/* 必須の入力欄 */
input:required {
  border-left: 3px solid #e74c3c;
}

/* 任意の入力欄 */
input:optional {
  border-left: 3px solid #95a5a6;
}

/* バリデーションが通った入力欄 */
input:valid {
  border-color: #2ecc71;
}

/* バリデーションが通らない入力欄 */
input:invalid {
  border-color: #e74c3c;
}

/* プレースホルダーが表示されている（＝空の）入力欄 */
input:placeholder-shown {
  border-style: dashed;
}
```

## 4. 構造疑似クラス

要素の位置（何番目の子要素か）に基づいてスタイルを適用します。

### :first-child / :last-child

```css
/* 最初の子要素 */
li:first-child {
  font-weight: bold;
}

/* 最後の子要素 */
li:last-child {
  border-bottom: none;
}
```

### :nth-child()

```css
/* 3番目の子要素 */
li:nth-child(3) {
  color: red;
}

/* 偶数番目（2, 4, 6...） */
tr:nth-child(even) {
  background-color: #f8f9fa;
}

/* 奇数番目（1, 3, 5...） */
tr:nth-child(odd) {
  background-color: white;
}

/* 3の倍数（3, 6, 9...） */
li:nth-child(3n) {
  color: blue;
}

/* 3の倍数 + 1（1, 4, 7, 10...） */
li:nth-child(3n+1) {
  font-weight: bold;
}

/* 最初の5つ */
li:nth-child(-n+5) {
  border-left: 3px solid #3498db;
}

/* 6番目以降 */
li:nth-child(n+6) {
  opacity: 0.7;
}
```

### :nth-last-child()

後ろから数えます。

```css
/* 最後から2番目 */
li:nth-last-child(2) {
  color: green;
}

/* 最後の3つ */
li:nth-last-child(-n+3) {
  font-style: italic;
}
```

### :only-child

子要素が1つしかない場合に適用されます。

```css
.list-item:only-child {
  text-align: center;
}
```

### :empty

中身が空の要素に適用されます。

```css
.card:empty {
  display: none;
}

/* または空の場合にプレースホルダーを表示 */
.container:empty::before {
  content: "コンテンツがありません";
  color: #999;
}
```

---

## 5. :not()（否定疑似クラス）

指定したセレクタに**一致しない**要素を選択します。

```css
/* .special以外のすべての段落 */
p:not(.special) {
  color: #666;
}

/* 最後の子要素以外にボーダーを付ける */
li:not(:last-child) {
  border-bottom: 1px solid #eee;
}

/* disabled以外のボタン */
button:not(:disabled) {
  cursor: pointer;
}

/* 複数条件の否定 */
input:not([type="submit"]):not([type="button"]) {
  border: 1px solid #ddd;
  padding: 8px;
}
```

---

## 6. :has()（親セレクタ）

**:has()** は、特定の子要素を持つ親要素を選択できる画期的なセレクタです。

```css
/* 画像を含むカード */
.card:has(img) {
  padding: 0;
}

/* チェックされたinputを持つラベル */
label:has(input:checked) {
  background-color: #e8f4fd;
  border-color: #3498db;
}

/* .errorクラスを持つ入力欄を含むフォームグループ */
.form-group:has(.error) {
  background-color: #fef2f2;
}

/* 画像がない場合のカードスタイル */
.card:not(:has(img)) {
  background-color: #f0f0f0;
}

/* 次の兄弟要素がh2であるh1 */
h1:has(+ h2) {
  margin-bottom: 8px;
}
```

> **:has()** は「親を子の条件で選択できる」ため、以前はJavaScriptが必要だった多くのパターンがCSSだけで実現できます。

---

## 7. :is() と :where()

### :is()（セレクタリストの簡略化）

```css
/* :is() なし（冗長） */
header a:hover,
nav a:hover,
footer a:hover {
  color: #e74c3c;
}

/* :is() あり（簡潔） */
:is(header, nav, footer) a:hover {
  color: #e74c3c;
}

/* ネストしたリスト */
:is(ol, ul) :is(ol, ul) {
  margin-left: 20px;
}
```

### :where()

`:is()` と同じ動作ですが、**詳細度が常に0** です。

```css
/* :is() → 詳細度にセレクタの詳細度が加算される */
:is(.card) .title { color: blue; }

/* :where() → 詳細度は0（上書きしやすい） */
:where(.card) .title { color: blue; }
```

> **使い分け：** リセットCSSやベーススタイルでは `:where()` を使い、上書きしやすくするのがベストプラクティスです。

---

## 8. 疑似要素とは？

**疑似要素** は、HTMLを変更せずに要素の一部や仮想的な要素にスタイルを適用します。`::` を使います。

> **:と::の違い：** 疑似クラスは `:` 1つ、疑似要素は `::` 2つです。

---

## 9. ::before と ::after

要素の前（before）または後（after）に仮想的な要素を挿入します。`content` プロパティが必須です。

### 基本的な使い方

```css
.label::before {
  content: "★ ";       /* テキストを挿入 */
  color: gold;
}

.link::after {
  content: " →";       /* テキストを挿入 */
}
```

### 装飾的な使い方

```css
/* アンダーライン風の装飾 */
.heading {
  position: relative;
  display: inline-block;
}

.heading::after {
  content: "";
  position: absolute;
  bottom: -4px;
  left: 0;
  width: 100%;
  height: 3px;
  background-color: #3498db;
}
```

### ホバーで動くアンダーライン

```css
.nav-link {
  position: relative;
  text-decoration: none;
}

.nav-link::after {
  content: "";
  position: absolute;
  bottom: -2px;
  left: 0;
  width: 0;
  height: 2px;
  background-color: #3498db;
  transition: width 0.3s ease;
}

.nav-link:hover::after {
  width: 100%;
}
```

### 引用符

```css
blockquote::before {
  content: "\201C";     /* 左の二重引用符 " */
  font-size: 3em;
  color: #ddd;
  line-height: 0;
  vertical-align: -0.5em;
}
```

### アイコン風の装飾

```css
.required-label::after {
  content: "*";
  color: #e74c3c;
  margin-left: 4px;
}

.external-link::after {
  content: " ↗";
  font-size: 0.8em;
}

.checkbox-label::before {
  content: "";
  display: inline-block;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 3px;
  margin-right: 8px;
  vertical-align: middle;
}
```

> **重要：** `::before` と `::after` は `content` プロパティがないと表示されません。空でも `content: ""` を書きます。

---

## 10. ::placeholder

入力フィールドのプレースホルダーテキストのスタイルを変更します。

```css
input::placeholder {
  color: #aaa;
  font-style: italic;
  font-size: 14px;
}

textarea::placeholder {
  color: #bbb;
}
```

---

## 11. ::selection

ユーザーがテキストを選択（ドラッグ）した時の色を変更します。

```css
::selection {
  background-color: #3498db;
  color: white;
}

/* 特定の要素だけ変える */
.highlight::selection {
  background-color: #f1c40f;
  color: #333;
}
```

---

## 12. その他の疑似要素

### ::first-line（最初の行）

```css
p::first-line {
  font-weight: bold;
  font-size: 1.1em;
}
```

### ::first-letter（最初の文字）

```css
p::first-letter {
  font-size: 2em;
  font-weight: bold;
  color: #3498db;
  float: left;
  margin-right: 4px;
}
```

### ::marker（リストマーカー）

```css
li::marker {
  color: #3498db;
  font-weight: bold;
  font-size: 1.2em;
}
```

---

## まとめ

| 項目 | 要点 |
|------|------|
| :hover / :focus / :active | ユーザー操作に応じたスタイル |
| :nth-child() | n番目の子要素を選択。even/oddで偶数奇数 |
| :first-child / :last-child | 最初/最後の子要素 |
| :not() | 条件に一致しない要素を選択 |
| :has() | 特定の子要素を持つ親を選択（新機能） |
| :is() / :where() | セレクタの簡略化。:where()は詳細度0 |
| ::before / ::after | 仮想要素を挿入。contentが必須 |
| ::placeholder | プレースホルダーのスタイル |
| ::selection | テキスト選択時のスタイル |
