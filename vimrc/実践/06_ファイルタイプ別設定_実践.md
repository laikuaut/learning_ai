## 課題の要件

1. `filetype plugin indent on` でファイルタイプ検出を有効にする
2. `augroup` を使って autocmd の重複登録を防止する
3. 以下のファイルタイプ別設定を `autocmd FileType` で記述する
   - **Python**: インデント4スペース、`colorcolumn=79`、実行マッピング
   - **JavaScript / TypeScript**: インデント2スペース、`colorcolumn=100`
   - **HTML / CSS**: インデント2スペース
   - **Makefile**: `noexpandtab`（タブ文字を使用）
   - **Markdown**: `wrap`, `linebreak`, `spell`
4. すべてのオプション設定に `setlocal` を使用する（グローバル設定を汚さない）
5. ファイルタイプ別マッピングに `<buffer>` を付ける

## ステップガイド

<details>
<summary>ステップ1：augroup の基本構文を理解する</summary>

`augroup` を使わないと、`:source ~/.vimrc` で再読み込みするたびに同じ autocmd が重複して登録されます。

```vim
" 良い例：augroup で囲んで autocmd! でクリアする
augroup MyFileTypes
  autocmd!
  autocmd FileType python setlocal tabstop=4 shiftwidth=4 expandtab
augroup END
```

`autocmd!` は、この augroup 内の既存の autocmd をすべて削除してから再登録します。これにより重複が防止されます。

</details>

<details>
<summary>ステップ2：各ファイルタイプの設定を記述する</summary>

```vim
augroup MyFileTypes
  autocmd!

  " Python
  autocmd FileType python setlocal tabstop=4 shiftwidth=4 softtabstop=4 expandtab
  autocmd FileType python setlocal colorcolumn=79

  " JavaScript / TypeScript
  autocmd FileType javascript,typescript setlocal tabstop=2 shiftwidth=2 softtabstop=2 expandtab

  " Makefile（タブ文字を使う）
  autocmd FileType make setlocal noexpandtab
augroup END
```

カンマ区切りで複数のファイルタイプを指定できます。

</details>

<details>
<summary>ステップ3：ファイルタイプ別のマッピングを追加する</summary>

`<buffer>` を付けることで、そのバッファ（ファイル）内でのみ有効なマッピングになります。

```vim
augroup MyFileTypes
  autocmd!
  " Python: Leader + r で実行
  autocmd FileType python nnoremap <buffer> <Leader>r :!python3 %<CR>
augroup END
```

</details>

<details>
<summary>ステップ4：動作を確認する</summary>

1. `vim test.py` で Python ファイルを開き、`:set tabstop?` で 4 になっていることを確認
2. `vim test.js` で JavaScript ファイルを開き、`:set tabstop?` で 2 になっていることを確認
3. `vim Makefile` を開き、`:set expandtab?` で `noexpandtab` になっていることを確認
4. Python ファイルで `<Leader>r` を押してスクリプトが実行されることを確認

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ==================================================
" ファイルタイプ検出
" ==================================================
filetype plugin indent on

" ==================================================
" ファイルタイプ別設定
" ==================================================
augroup MyFileTypes
  autocmd!

  " --- Python ---
  autocmd FileType python setlocal tabstop=4 shiftwidth=4 softtabstop=4 expandtab
  autocmd FileType python setlocal colorcolumn=79
  autocmd FileType python nnoremap <buffer> <Leader>r :!python3 %<CR>

  " --- JavaScript / TypeScript ---
  autocmd FileType javascript,typescript setlocal tabstop=2 shiftwidth=2 softtabstop=2 expandtab
  autocmd FileType javascript,typescript setlocal colorcolumn=100

  " --- HTML / CSS ---
  autocmd FileType html,css setlocal tabstop=2 shiftwidth=2 softtabstop=2 expandtab

  " --- Makefile ---
  autocmd FileType make setlocal noexpandtab tabstop=4 shiftwidth=4

  " --- Markdown ---
  autocmd FileType markdown setlocal wrap linebreak
  autocmd FileType markdown setlocal spell spelllang=en,cjk
augroup END
```

</details>

<details>
<summary>解答例（改良版 ─ ftplugin による設定分離）</summary>

改良版では、vimrc 内の autocmd を最小限にして、ファイルタイプ別の設定を `ftplugin` ディレクトリに分離します。

**vimrc に記述する部分:**

```vim
" ==================================================
" ファイルタイプ検出
" ==================================================
filetype plugin indent on

" ==================================================
" ファイルタイプ別の共通 autocmd
" ==================================================
augroup MyFileTypes
  autocmd!

  " 新規ファイル作成時にテンプレートを挿入
  autocmd BufNewFile *.py 0r ~/.vim/templates/python.txt
  autocmd BufNewFile *.sh 0r ~/.vim/templates/bash.txt

  " 保存時に末尾の空白を自動削除
  autocmd BufWritePre * :%s/\s\+$//ge

  " ファイルを開いた時に前回のカーソル位置に戻る
  autocmd BufReadPost *
    \ if line("'\"") > 0 && line("'\"") <= line("$") |
    \   exe "normal! g'\"" |
    \ endif
augroup END
```

**~/.vim/ftplugin/python.vim:**

```vim
" Python 用設定（このファイルは Python を開いた時だけ読み込まれる）
setlocal tabstop=4 shiftwidth=4 softtabstop=4 expandtab
setlocal colorcolumn=79
setlocal textwidth=79

" Python 実行
nnoremap <buffer> <Leader>r :!python3 %<CR>

" Python デバッグプリント挿入
nnoremap <buffer> <Leader>p oprint(f"DEBUG: {}")<Esc>F}i
```

**~/.vim/ftplugin/javascript.vim:**

```vim
" JavaScript 用設定
setlocal tabstop=2 shiftwidth=2 softtabstop=2 expandtab
setlocal colorcolumn=100

" console.log 挿入
nnoremap <buffer> <Leader>l oconsole.log();<Esc>F(a
```

**~/.vim/ftplugin/markdown.vim:**

```vim
" Markdown 用設定
setlocal wrap linebreak
setlocal spell spelllang=en,cjk
setlocal conceallevel=2
setlocal textwidth=80
```

**初心者向けとの違い:**
- `ftplugin` ディレクトリで設定を分離し、vimrc の肥大化を防止
- 各言語専用のファイルに設定を集約するため、管理が容易
- テンプレート自動挿入やカーソル位置復元など、実用的な autocmd を追加
- ファイル保存時の末尾空白自動削除で、コードの品質を維持

</details>
