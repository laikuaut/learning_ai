# 実践課題04：お問い合わせフォーム ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第1章（HTMLの基本構造）、第2章（テキストとリンク）、第5章（フォーム）
> **課題の種類**: ミニプロジェクト
> **学習目標**: フォーム要素（input、textarea、select、button）とバリデーション属性（required、pattern、min/max）を使って、実用的な問い合わせフォームを作る

---

## 完成イメージ

ブラウザで開くと以下のような問い合わせフォームが表示されます。

```
================================
  お問い合わせフォーム
================================

【お客様情報】
  お名前*:     [____________]
  メール*:     [____________]
  電話番号:    [____________]

【お問い合わせ内容】
  カテゴリ*:   [▼ 選択してください]
  件名*:       [____________]
  内容*:
  [                          ]
  [                          ]
  [                          ]

【その他】
  ☐ メールマガジンを受け取る
  ☐ 利用規約に同意する*

  [  送信  ]  [リセット]

  * は必須項目です
```

---

## 課題の要件

1. 正しいHTML基本構造を持つこと
2. `<form>` 要素でフォーム全体を囲むこと
3. `<fieldset>` と `<legend>` でフォームをグループ化すること
4. `<label>` と `<input>` を正しく関連付けること（`for` と `id` の対応）
5. 以下の入力要素を含むこと
   - `type="text"` ─ お名前
   - `type="email"` ─ メールアドレス
   - `type="tel"` ─ 電話番号
   - `<select>` ─ カテゴリ選択
   - `type="text"` ─ 件名
   - `<textarea>` ─ 問い合わせ内容
   - `type="checkbox"` ─ メルマガ受信、利用規約同意
6. `required` 属性で必須項目を指定すること
7. `placeholder` 属性で入力例を示すこと
8. 送信ボタンとリセットボタンを配置すること

---

## ステップガイド

<details>
<summary>ステップ1：フォームの骨格を作る</summary>

```html
<form action="#" method="post">
  <fieldset>
    <legend>お客様情報</legend>
    <!-- ここに入力欄を追加 -->
  </fieldset>

  <fieldset>
    <legend>お問い合わせ内容</legend>
    <!-- ここに入力欄を追加 -->
  </fieldset>

  <button type="submit">送信</button>
</form>
```

`action="#"` は送信先がないことを示します（学習用なので実際にはデータは送信されません）。

</details>

<details>
<summary>ステップ2：label と input を関連付ける</summary>

`<label>` の `for` 属性と `<input>` の `id` 属性を一致させます。

```html
<label for="name">お名前：</label>
<input type="text" id="name" name="name" required>
```

こうすると、ラベルをクリックしたときに入力欄にフォーカスが当たります。

</details>

<details>
<summary>ステップ3：select でドロップダウンを作る</summary>

```html
<label for="category">カテゴリ：</label>
<select id="category" name="category" required>
  <option value="">選択してください</option>
  <option value="product">商品について</option>
  <option value="service">サービスについて</option>
  <option value="other">その他</option>
</select>
```

最初の `<option>` は `value=""` にして、未選択状態を表します。

</details>

<details>
<summary>ステップ4：textarea で複数行入力欄を作る</summary>

```html
<label for="message">お問い合わせ内容：</label>
<textarea id="message" name="message" rows="5" cols="40"
          placeholder="お問い合わせ内容をご記入ください" required></textarea>
```

`rows` は表示行数、`cols` は表示列数です。

</details>

<details>
<summary>解答例（改良版 ─ バリデーションとアクセシビリティ強化）</summary>

パターンバリデーション、`aria` 属性、フォーカス管理を使った改良版です。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>お問い合わせフォーム</title>
  <style>
    body {
      font-family: sans-serif;
      max-width: 600px;
      margin: 20px auto;
      padding: 0 16px;
    }
    fieldset {
      margin-bottom: 20px;
      padding: 16px;
      border: 1px solid #ccc;
      border-radius: 8px;
    }
    legend {
      font-weight: bold;
      padding: 0 8px;
    }
    label {
      display: block;
      margin-top: 12px;
      font-weight: bold;
    }
    input[type="text"],
    input[type="email"],
    input[type="tel"],
    select,
    textarea {
      width: 100%;
      padding: 8px;
      margin-top: 4px;
      border: 1px solid #ccc;
      border-radius: 4px;
      box-sizing: border-box;
      font-size: 1em;
    }
    input:focus,
    select:focus,
    textarea:focus {
      outline: 2px solid #4a90d9;
      border-color: #4a90d9;
    }
    input:invalid:not(:placeholder-shown) {
      border-color: #e74c3c;
    }
    .required::after {
      content: " *";
      color: #e74c3c;
    }
    .checkbox-group {
      margin-top: 12px;
    }
    .checkbox-group label {
      display: inline;
      font-weight: normal;
    }
    .buttons {
      margin-top: 20px;
      text-align: center;
    }
    button {
      padding: 10px 24px;
      font-size: 1em;
      border: none;
      border-radius: 4px;
      cursor: pointer;
      margin: 0 8px;
    }
    button[type="submit"] {
      background-color: #4a90d9;
      color: white;
    }
    button[type="submit"]:hover {
      background-color: #357abd;
    }
    button[type="reset"] {
      background-color: #95a5a6;
      color: white;
    }
    button[type="reset"]:hover {
      background-color: #7f8c8d;
    }
    .note {
      color: #666;
      font-size: 0.9em;
    }
  </style>
</head>
<body>
  <h1>お問い合わせフォーム</h1>
  <p class="note"><span style="color:#e74c3c">*</span> は必須項目です</p>

  <form action="#" method="post" novalidate>

    <fieldset>
      <legend>お客様情報</legend>

      <label for="name" class="required">お名前</label>
      <input type="text" id="name" name="name" required
             placeholder="例：山田太郎"
             autocomplete="name"
             aria-required="true">

      <label for="email" class="required">メールアドレス</label>
      <input type="email" id="email" name="email" required
             placeholder="例：yamada@example.com"
             autocomplete="email"
             aria-required="true">

      <label for="tel">電話番号</label>
      <input type="tel" id="tel" name="tel"
             placeholder="例：090-1234-5678"
             pattern="[0-9\-]{10,13}"
             title="ハイフン付きまたはなしの電話番号を入力してください"
             autocomplete="tel">
    </fieldset>

    <fieldset>
      <legend>お問い合わせ内容</legend>

      <label for="category" class="required">カテゴリ</label>
      <select id="category" name="category" required aria-required="true">
        <option value="">選択してください</option>
        <optgroup label="商品関連">
          <option value="product-info">商品の情報</option>
          <option value="product-stock">在庫の確認</option>
        </optgroup>
        <optgroup label="注文関連">
          <option value="delivery">配送について</option>
          <option value="return">返品・交換について</option>
        </optgroup>
        <option value="other">その他</option>
      </select>

      <label for="subject" class="required">件名</label>
      <input type="text" id="subject" name="subject" required
             placeholder="例：商品の在庫について"
             maxlength="100"
             aria-required="true">

      <label for="message" class="required">お問い合わせ内容</label>
      <textarea id="message" name="message" rows="6"
                placeholder="お問い合わせ内容をご記入ください"
                required minlength="10" maxlength="1000"
                aria-required="true"></textarea>
      <small class="note">10文字以上1000文字以内でご記入ください</small>
    </fieldset>

    <fieldset>
      <legend>その他</legend>

      <div class="checkbox-group">
        <input type="checkbox" id="newsletter" name="newsletter" value="yes">
        <label for="newsletter">メールマガジンを受け取る</label>
      </div>

      <div class="checkbox-group">
        <input type="checkbox" id="agree" name="agree" required aria-required="true">
        <label for="agree" class="required">利用規約に同意する</label>
      </div>
    </fieldset>

    <div class="buttons">
      <button type="submit">送信</button>
      <button type="reset">リセット</button>
    </div>

  </form>
</body>
</html>
```

**初心者向けとの違い:**
- `pattern` 属性で電話番号の形式をバリデーション → 不正な入力を防止
- `minlength` / `maxlength` で文字数制限 → 短すぎる・長すぎる入力を防止
- `autocomplete` 属性でブラウザの自動入力を支援 → ユーザー体験の向上
- `aria-required="true"` でスクリーンリーダーに必須項目を伝達 → アクセシビリティ向上
- `<optgroup>` でセレクトボックスの選択肢をグループ化 → 見つけやすい
- CSSで見やすいデザインとフォーカス時のスタイル → 実務レベルのフォーム
- `:invalid:not(:placeholder-shown)` で入力後のバリデーションエラーを表示 → 初期状態では赤くならない

</details>
