## 読解対象コード

```python
import socket
import time
import json
from datetime import datetime
from urllib.request import urlopen, Request
from urllib.error import URLError, HTTPError


# ==================== 設定 ====================

TARGETS = [
    {"name": "Google DNS", "host": "8.8.8.8", "port": 53, "type": "tcp"},
    {"name": "Example.com", "url": "http://example.com", "type": "http"},
    {"name": "Local SSH", "host": "localhost", "port": 22, "type": "tcp"},
    {"name": "Google", "url": "https://www.google.com", "type": "http"},
]

CHECK_INTERVAL = 60  # 秒
MAX_HISTORY = 100


# ==================== チェッカー ====================

def check_tcp(host, port, timeout=3):
    """TCPポートの疎通確認を行う"""
    start = time.time()
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(timeout)
        result = sock.connect_ex((host, port))
        elapsed = (time.time() - start) * 1000
        sock.close()
        return {
            "reachable": result == 0,
            "latency_ms": round(elapsed, 1),
            "error": None if result == 0 else f"Connection failed (code: {result})",
        }
    except socket.timeout:
        return {"reachable": False, "latency_ms": timeout * 1000, "error": "Timeout"}
    except Exception as e:
        return {"reachable": False, "latency_ms": 0, "error": str(e)}


def check_http(url, timeout=5):
    """HTTPリクエストでWebサイトの応答を確認する"""
    start = time.time()
    try:
        req = Request(url, headers={"User-Agent": "NetworkMonitor/1.0"})
        response = urlopen(req, timeout=timeout)
        elapsed = (time.time() - start) * 1000
        content_length = len(response.read())
        response.close()
        return {
            "reachable": True,
            "latency_ms": round(elapsed, 1),
            "status_code": response.status,
            "content_length": content_length,
            "error": None,
        }
    except HTTPError as e:
        elapsed = (time.time() - start) * 1000
        return {
            "reachable": False,
            "latency_ms": round(elapsed, 1),
            "status_code": e.code,
            "content_length": 0,
            "error": f"HTTP {e.code}: {e.reason}",
        }
    except URLError as e:
        return {"reachable": False, "latency_ms": 0, "status_code": 0,
                "content_length": 0, "error": str(e.reason)}


# ==================== モニター ====================

class NetworkMonitor:
    def __init__(self, targets):
        self.targets = targets
        self.history = {t["name"]: [] for t in targets}
        self.alerts = []

    def run_check(self):
        """全ターゲットをチェックする"""
        timestamp = datetime.now().isoformat()
        results = []

        for target in self.targets:
            if target["type"] == "tcp":
                result = check_tcp(target["host"], target["port"])
            elif target["type"] == "http":
                result = check_http(target["url"])
            else:
                result = {"reachable": False, "latency_ms": 0, "error": "Unknown type"}

            result["name"] = target["name"]
            result["timestamp"] = timestamp
            results.append(result)

            # 履歴に追加
            history = self.history[target["name"]]
            history.append(result)
            if len(history) > MAX_HISTORY:
                history.pop(0)

            # アラート判定
            if not result["reachable"]:
                self.alerts.append({
                    "timestamp": timestamp,
                    "target": target["name"],
                    "error": result.get("error", "Unknown"),
                })

        return results

    def get_statistics(self, target_name):
        """指定ターゲットの統計情報を取得する"""
        history = self.history.get(target_name, [])
        if not history:
            return None

        total = len(history)
        reachable_count = sum(1 for r in history if r["reachable"])
        latencies = [r["latency_ms"] for r in history if r["reachable"]]

        stats = {
            "target": target_name,
            "total_checks": total,
            "successful": reachable_count,
            "failed": total - reachable_count,
            "availability": round(reachable_count / total * 100, 1),
        }

        if latencies:
            stats["avg_latency"] = round(sum(latencies) / len(latencies), 1)
            stats["min_latency"] = min(latencies)
            stats["max_latency"] = max(latencies)

        return stats

    def generate_report(self):
        """全ターゲットのレポートを生成する"""
        report_lines = []
        report_lines.append("=" * 60)
        report_lines.append(f"  ネットワークモニタリングレポート")
        report_lines.append(f"  生成日時: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
        report_lines.append("=" * 60)

        for target in self.targets:
            stats = self.get_statistics(target["name"])
            if stats is None:
                continue

            status_icon = "OK" if stats["availability"] >= 99 else \
                          "WARN" if stats["availability"] >= 90 else "FAIL"

            report_lines.append(f"\n  [{status_icon}] {stats['target']}")
            report_lines.append(f"    稼働率     : {stats['availability']}%")
            report_lines.append(f"    チェック数 : {stats['total_checks']} "
                                f"(成功: {stats['successful']}, 失敗: {stats['failed']})")
            if "avg_latency" in stats:
                report_lines.append(f"    平均遅延   : {stats['avg_latency']}ms")
                report_lines.append(f"    最小/最大  : {stats['min_latency']}ms / "
                                    f"{stats['max_latency']}ms")

        # アラート
        if self.alerts:
            report_lines.append(f"\n{'─' * 60}")
            report_lines.append(f"  直近のアラート（{len(self.alerts)}件）:")
            for alert in self.alerts[-5:]:
                report_lines.append(f"    [{alert['timestamp'][:19]}] "
                                    f"{alert['target']}: {alert['error']}")

        report_lines.append("=" * 60)
        return "\n".join(report_lines)

    def save_history(self, filename):
        """履歴をJSONファイルに保存する"""
        data = {
            "saved_at": datetime.now().isoformat(),
            "history": self.history,
            "alerts": self.alerts,
        }
        with open(filename, "w", encoding="utf-8") as f:
            json.dump(data, f, ensure_ascii=False, indent=2)

    def load_history(self, filename):
        """JSONファイルから履歴を読み込む"""
        try:
            with open(filename, "r", encoding="utf-8") as f:
                data = json.load(f)
            self.history = data.get("history", {})
            self.alerts = data.get("alerts", [])
            return True
        except (FileNotFoundError, json.JSONDecodeError):
            return False


# ==================== メイン ====================

def main():
    monitor = NetworkMonitor(TARGETS)
    print("===== ネットワークモニタ =====\n")

    while True:
        print("1.チェック実行  2.レポート表示  3.統計表示  4.保存  5.読込  6.終了")
        choice = input("操作: ").strip()

        if choice == "1":
            print("\nチェック実行中...")
            results = monitor.run_check()
            for r in results:
                status = "OK" if r["reachable"] else "NG"
                print(f"  [{status}] {r['name']:<20} {r['latency_ms']:>7.1f}ms"
                      f"  {r.get('error', '') or ''}")

        elif choice == "2":
            print(monitor.generate_report())

        elif choice == "3":
            for target in TARGETS:
                stats = monitor.get_statistics(target["name"])
                if stats:
                    print(f"\n  {stats['target']}: "
                          f"稼働率 {stats['availability']}% "
                          f"({stats['total_checks']}回)")

        elif choice == "4":
            monitor.save_history("monitor_data.json")
            print("保存しました。")

        elif choice == "5":
            if monitor.load_history("monitor_data.json"):
                print("読み込みました。")
            else:
                print("読み込みに失敗しました。")

        elif choice == "6":
            print("終了します。")
            break


if __name__ == "__main__":
    main()
```

## 設問

以下の10問に答えてください。

<details>
<summary>設問1：`check_tcp()` 関数は何を返しますか？ 返り値の型と主なキーを答えてください。</summary>

**解答:**

辞書（dict）を返します。主なキーは以下の通りです：
- `reachable`（bool）: ポートに接続できたかどうか
- `latency_ms`（float）: 接続にかかった時間（ミリ秒）
- `error`（str or None）: エラーメッセージ（成功時は `None`）

`connect_ex()` の戻り値が `0` であれば接続成功（`reachable: True`）となります。

</details>

<details>
<summary>設問2：`check_http()` で `HTTPError` と `URLError` を分けてキャッチしているのはなぜですか？</summary>

**解答:**

`HTTPError` は `URLError` のサブクラスです。`HTTPError` はHTTPレスポンスは返ってきたがステータスコードがエラー（4xx, 5xxなど）の場合に発生します。この場合、サーバは応答しているのでレイテンシやステータスコードの情報が取得できます。

一方、`URLError` はDNS解決の失敗やネットワーク接続不可など、サーバに到達できなかった場合に発生します。

先に `HTTPError` をキャッチすることで、「サーバは応答したがエラーだった」ケースと「そもそもサーバに到達できなかった」ケースを区別できます。

</details>

<details>
<summary>設問3：`NetworkMonitor.history` のデータ構造を説明してください。</summary>

**解答:**

`self.history` は辞書で、キーがターゲット名（文字列）、値がチェック結果の辞書のリストです。

```python
{
    "Google DNS": [
        {"reachable": True, "latency_ms": 12.3, "error": None, "name": "Google DNS", "timestamp": "..."},
        {"reachable": True, "latency_ms": 11.5, "error": None, "name": "Google DNS", "timestamp": "..."},
        ...
    ],
    "Example.com": [...],
    ...
}
```

`__init__` で各ターゲット名をキーにした空リストで初期化され、`run_check()` の都度追加されます。`MAX_HISTORY`（100）を超えると先頭を `pop(0)` で削除して最大件数を維持します。

</details>

<details>
<summary>設問4：`history.pop(0)` の計算量は O(n) です。これを O(1) に改善するにはどうしますか？</summary>

**解答:**

`collections.deque` を使えば、先頭の削除が O(1) で行えます。

```python
from collections import deque

# __init__ で
self.history = {t["name"]: deque(maxlen=MAX_HISTORY) for t in targets}
```

`deque(maxlen=MAX_HISTORY)` とすれば、`maxlen` を超えたときに自動的に先頭が削除されるため、手動での `pop(0)` も不要になります。

</details>

<details>
<summary>設問5：`get_statistics()` メソッドの稼働率（availability）はどのように計算されていますか？</summary>

**解答:**

`reachable` が `True` のチェック結果の数を全チェック数で割り、100を掛けて百分率にしています。

```python
availability = round(reachable_count / total * 100, 1)
```

例えば、100回チェックして95回成功なら `95.0%` となります。

注意点として、`total` が 0 の場合はゼロ除算になりますが、メソッドの冒頭で `if not history: return None` としてガードしています。

</details>

<details>
<summary>設問6：`generate_report()` でステータスアイコンはどのように決定されますか？</summary>

**解答:**

稼働率に基づいて3段階で判定しています：

| 稼働率 | アイコン | 意味 |
|--------|----------|------|
| 99%以上 | `OK` | 正常稼働 |
| 90%以上99%未満 | `WARN` | 警告（一部障害あり） |
| 90%未満 | `FAIL` | 障害 |

三項演算子のチェーンで実装されています：
```python
status_icon = "OK" if stats["availability"] >= 99 else \
              "WARN" if stats["availability"] >= 90 else "FAIL"
```

</details>

<details>
<summary>設問7：`save_history()` と `load_history()` の実装で、エラーに対するアプローチが異なります。その違いを説明してください。</summary>

**解答:**

- `save_history()` はエラー処理を行っていません。ファイル書き込みに失敗するとそのまま例外が伝播します。
- `load_history()` は `FileNotFoundError` と `json.JSONDecodeError` を `try-except` でキャッチし、失敗時は `False` を返します。

この設計の意図は、「保存の失敗はデータ損失に繋がるため例外で通知すべき」「読み込みの失敗は初回起動時等に正常に起こり得るので静かに失敗してよい」という考え方です。ただし、実務では保存時にもエラーハンドリングを行うのが望ましいです。

</details>

<details>
<summary>設問8：`TARGETS` リストに新しいターゲット（UDP通信のDNSチェック）を追加するには、コードのどこを変更する必要がありますか？</summary>

**解答:**

2箇所の変更が必要です：

1. **`TARGETS` リストにエントリを追加**:
```python
{"name": "UDP DNS", "host": "8.8.8.8", "port": 53, "type": "udp"}
```

2. **`run_check()` メソッド内の分岐に `udp` タイプの処理を追加** し、それに対応する **`check_udp()` 関数を新規作成** する必要があります。

現在は `tcp` と `http` のみ対応しており、それ以外は `"Unknown type"` エラーになります。

</details>

<details>
<summary>設問9：このコードのセキュリティ上の懸念点を1つ以上挙げてください。</summary>

**解答:**

1. **URLの入力検証がない**: `check_http()` に任意のURLを渡せるため、内部ネットワークへのリクエスト（SSRF: Server-Side Request Forgery）のリスクがある。
2. **HTTPSの証明書検証**: `urlopen()` はデフォルトで証明書を検証しますが、明示的な制御がない。
3. **ファイルパスの固定**: `monitor_data.json` がハードコードされており、パストラバーサル等の対策がない（ただし現状はハードコードなので問題は軽微）。
4. **User-Agent の偽装**: `NetworkMonitor/1.0` としているが、監視ツールであることを示す情報が対象サーバに送信される。

</details>

<details>
<summary>設問10：このコードを改善するとしたら、どのような変更を提案しますか？ 3つ以上挙げてください。</summary>

**解答例:**

1. **`deque` の使用**: `history.pop(0)` を `collections.deque(maxlen=N)` に置き換えてパフォーマンスを改善する。
2. **並列チェック**: `concurrent.futures.ThreadPoolExecutor` で複数ターゲットを同時チェックし、チェック時間を短縮する。
3. **設定の外部化**: `TARGETS` や `CHECK_INTERVAL` を設定ファイル（JSON/YAML）に分離し、コード変更なしでターゲットを追加できるようにする。
4. **ロギング**: `print()` の代わりに `logging` モジュールを使い、ログレベルに応じた出力とファイル記録を行う。
5. **型ヒントの追加**: 関数の引数と返り値に型ヒントを付けて可読性と保守性を向上させる。
6. **テストの追加**: `check_tcp()` や `get_statistics()` のユニットテストを作成し、リグレッションを防ぐ。
7. **UDP チェッカーの追加**: DNS の UDP チェック等、TCP 以外のプロトコルにも対応する。

</details>
