### 問題1: grep の基本
以下の要件を満たす `grep` コマンドを書いてください。

1. `/var/log/syslog` から "error" を含む行を検索する（大文字小文字を区別しない）
2. `/etc/ssh/sshd_config` からコメント行（#で始まる行）を除外して表示する
3. `/home` ディレクトリ以下で "TODO" を含むファイルを再帰的に検索し、行番号も表示する

<details>
<summary>解答</summary>

```bash
# 1. 大文字小文字を区別しない検索（-i）
$ grep -i "error" /var/log/syslog

# 2. コメント行を除外（-v で逆マッチ）+ 空行も除外
$ grep -v "^#" /etc/ssh/sshd_config | grep -v "^$"

# 3. 再帰検索 + 行番号表示（-rn）
$ grep -rn "TODO" /home/
```

</details>

### 問題2: grep と正規表現
以下のパターンにマッチする `grep` コマンドを書いてください。

1. IPアドレスの形式（例: 192.168.1.100）にマッチする行を `access.log` から検索する
2. "error" または "warning" を含む行を検索する
3. 行頭が数字で始まる行を検索する

<details>
<summary>解答</summary>

```bash
# 1. IPアドレスパターン（拡張正規表現 -E を使用）
$ grep -E "[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+" access.log

# 2. error または warning（OR 条件）
$ grep -E "error|warning" /var/log/syslog
# または
$ grep -e "error" -e "warning" /var/log/syslog

# 3. 行頭が数字で始まる行
$ grep "^[0-9]" file.txt
```

</details>

### 問題3: sed の基本
以下の操作を行う `sed` コマンドを書いてください。

1. ファイル内のすべての "localhost" を "192.168.1.100" に置換する（画面に出力）
2. ファイルのコメント行（#で始まる行）を削除する
3. ファイルの空行を削除する
4. ファイルの3行目だけを表示する

<details>
<summary>解答</summary>

```bash
# 1. 文字列の置換（g フラグで行内の全マッチを置換）
$ sed 's/localhost/192.168.1.100/g' config.txt

# 2. コメント行を削除
$ sed '/^#/d' config.txt

# 3. 空行を削除
$ sed '/^$/d' config.txt

# 4. 特定の行だけ表示（-n と p の組み合わせ）
$ sed -n '3p' config.txt
```

</details>

### 問題4: awk の基本
以下の操作を行う `awk` コマンドを書いてください。

1. `/etc/passwd` から1列目（ユーザー名）と3列目（UID）を表示する
2. `/etc/passwd` からUIDが1000以上のユーザーだけを表示する
3. スペース区切りのテキストから最後のフィールドを表示する

<details>
<summary>解答</summary>

```bash
# 1. 特定のフィールドを表示（-F で区切り文字を指定）
$ awk -F: '{print $1, $3}' /etc/passwd
root 0
daemon 1
tanaka 1000

# 2. 条件フィルタ（$3 >= 1000）
$ awk -F: '$3 >= 1000 {print $1, $3}' /etc/passwd
tanaka 1000
suzuki 1001

# 3. 最後のフィールド（$NF）
$ echo "one two three four" | awk '{print $NF}'
four
```

</details>

### 問題5: sort と uniq
以下のデータを処理してください。

```
apple
banana
apple
cherry
banana
apple
```

1. 重複を除去してアルファベット順に表示する
2. 各項目の出現回数を数え、多い順に表示する

期待される出力（問題2）：
```
      3 apple
      2 banana
      1 cherry
```

<details>
<summary>解答</summary>

```bash
# テスト用にファイルを作成
$ echo -e "apple\nbanana\napple\ncherry\nbanana\napple" > fruits.txt

# 1. 重複を除去してソート
$ sort fruits.txt | uniq
apple
banana
cherry
# または
$ sort -u fruits.txt

# 2. 出現回数を数え、多い順に表示（定番パイプライン）
$ sort fruits.txt | uniq -c | sort -rn
      3 apple
      2 banana
      1 cherry
```

</details>

### 問題6: cut と tr
以下の操作を行うコマンドを書いてください。

1. `/etc/passwd` からユーザー名（1列目）とホームディレクトリ（6列目）だけを取り出す
2. 文字列 "Hello, World!" をすべて大文字に変換する
3. "Phone: 090-1234-5678" から数字だけを取り出す

<details>
<summary>解答</summary>

```bash
# 1. cut で特定フィールドを切り出し（コロン区切り）
$ cut -d: -f1,6 /etc/passwd
root:/root
daemon:/usr/sbin
tanaka:/home/tanaka

# 2. 大文字に変換
$ echo "Hello, World!" | tr 'a-z' 'A-Z'
HELLO, WORLD!

# 3. 数字だけを取り出す（-dc で数字以外を削除）
$ echo "Phone: 090-1234-5678" | tr -dc '0-9\n'
09012345678
```

</details>

### 問題7: awk の集計処理
以下のCSVファイル `sales.csv` があるとき、指定された処理を行ってください。

```csv
商品,個数,単価
りんご,5,100
バナナ,3,80
みかん,10,50
りんご,2,100
バナナ,7,80
```

1. 各行の売上（個数 x 単価）を計算して表示する
2. 売上の合計を表示する

期待される出力（問題1）：
```
りんご: 500円
バナナ: 240円
みかん: 500円
りんご: 200円
バナナ: 560円
```

<details>
<summary>解答</summary>

```bash
# 1. 各行の売上を計算（ヘッダーを除外するため NR>1）
$ awk -F, 'NR>1 {printf "%s: %d円\n", $1, $2*$3}' sales.csv
りんご: 500円
バナナ: 240円
みかん: 500円
りんご: 200円
バナナ: 560円

# 2. 売上の合計を表示
$ awk -F, 'NR>1 {total += $2*$3} END {printf "合計: %d円\n", total}' sales.csv
合計: 2000円
```

</details>

### 問題8: sed の実践的な使い方
以下の操作を `sed` で行ってください。

1. 設定ファイルのコメント行と空行を除外し、有効行だけ表示する
2. ログファイル内のIPアドレスを「xxx.xxx.xxx.xxx」に置換する
3. ファイルの10〜20行目だけを表示する

<details>
<summary>解答</summary>

```bash
# 1. コメント行と空行を除外（複数の操作をセミコロンで連結）
$ sed '/^#/d; /^$/d' /etc/ssh/sshd_config

# 2. IPアドレスをマスク（拡張正規表現 -E を使用）
$ sed -E 's/[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+/xxx.xxx.xxx.xxx/g' access.log

# 3. 10〜20行目だけ表示（-n と 範囲指定 + p）
$ sed -n '10,20p' file.txt
```

</details>

### 問題9: パイプラインの組み立て
以下の処理を1行のコマンド（パイプライン）で書いてください。

1. `/etc/passwd` から一般ユーザー（UID 1000以上）のユーザー名だけを取り出し、アルファベット順にソートして表示する
2. テキストファイルの単語の出現頻度を集計し、頻度の多い順にトップ10を表示する
3. `ps aux` の出力から、メモリ使用率が1%以上のプロセス名を使用率順にソートして表示する

<details>
<summary>解答</summary>

```bash
# 1. 一般ユーザーのユーザー名をソートして表示
$ awk -F: '$3 >= 1000 {print $1}' /etc/passwd | sort

# 2. 単語の出現頻度トップ10
$ cat document.txt | tr -s ' ' '\n' | tr 'A-Z' 'a-z' | sort | uniq -c | sort -rn | head -10

# 3. メモリ使用率1%以上のプロセス名（使用率順）
$ ps aux | awk 'NR>1 && $4 >= 1.0 {print $4, $11}' | sort -rn
```

解説（問題2）：
- `tr -s ' ' '\n'` : スペースを改行に変換（1行1単語に）
- `tr 'A-Z' 'a-z'` : 大文字を小文字に統一
- `sort | uniq -c` : ソートしてから重複カウント
- `sort -rn` : 数値の降順でソート
- `head -10` : トップ10を表示

</details>

### 問題10: アクセスログ解析
以下のような Apache のアクセスログ形式のデータがあるとします。

```
192.168.1.50 - - [20/Mar/2024:10:15:23 +0900] "GET /index.html HTTP/1.1" 200 1234
203.0.113.100 - - [20/Mar/2024:10:15:24 +0900] "GET /about.html HTTP/1.1" 200 5678
192.168.1.50 - - [20/Mar/2024:10:15:25 +0900] "POST /api/data HTTP/1.1" 404 89
198.51.100.25 - - [20/Mar/2024:10:15:26 +0900] "GET /index.html HTTP/1.1" 200 1234
192.168.1.50 - - [20/Mar/2024:10:15:27 +0900] "GET /style.css HTTP/1.1" 200 456
203.0.113.100 - - [20/Mar/2024:10:15:28 +0900] "GET /index.html HTTP/1.1" 301 0
```

以下の情報を取得するコマンドを書いてください。

1. アクセス数の多いIPアドレスのランキング
2. ステータスコード別のアクセス数
3. 404エラーが発生したURLの一覧

<details>
<summary>解答</summary>

```bash
# 1. IPアドレス別アクセス数ランキング
$ awk '{print $1}' access.log | sort | uniq -c | sort -rn
      3 192.168.1.50
      2 203.0.113.100
      1 198.51.100.25

# 2. ステータスコード別アクセス数
$ awk '{print $9}' access.log | sort | uniq -c | sort -rn
      4 200
      1 404
      1 301

# 3. 404エラーが発生したURL
$ awk '$9 == 404 {print $7}' access.log
/api/data
```

</details>

### 問題11: テキスト処理の総合問題
以下のスペース区切りのデータファイル `employees.txt` を処理してください。

```
田中 営業部 450000
鈴木 開発部 520000
佐藤 営業部 480000
山田 開発部 550000
高橋 総務部 420000
伊藤 開発部 510000
渡辺 営業部 460000
```

1. 部署別の人数を表示する
2. 給与の平均値を計算する
3. 給与の高い順にソートして表示する
4. 開発部の社員だけを表示し、給与の合計を計算する

<details>
<summary>解答</summary>

```bash
# 1. 部署別の人数
$ awk '{print $2}' employees.txt | sort | uniq -c
      3 営業部
      1 総務部
      3 開発部

# 2. 給与の平均値
$ awk '{sum += $3; count++} END {printf "平均給与: %d円\n", sum/count}' employees.txt
平均給与: 484285円

# 3. 給与の高い順にソート
$ sort -k3 -rn employees.txt
山田 開発部 550000
鈴木 開発部 520000
伊藤 開発部 510000
佐藤 営業部 480000
渡辺 営業部 460000
田中 営業部 450000
高橋 総務部 420000

# 4. 開発部の社員の給与合計
$ awk '$2 == "開発部" {print $0; sum += $3} END {printf "合計: %d円\n", sum}' employees.txt
鈴木 開発部 520000
山田 開発部 550000
伊藤 開発部 510000
合計: 1580000円
```

</details>
