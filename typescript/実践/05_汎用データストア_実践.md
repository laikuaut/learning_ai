# 実践課題05：汎用データストア ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第2章（インターフェースと型エイリアス）、第5章（ジェネリクス）
> **課題の種類**: ミニプロジェクト
> **学習目標**: ジェネリクス（generics）を使って、さまざまなデータ型に対応できる汎用的なデータストアを作る

---

## 課題の要件

1. ジェネリクスを使った `DataStore<T>` クラス（またはオブジェクト）を作る
2. `T` は「`id` プロパティを持つ型」に制約する（`extends` を使う）
3. CRUD操作（追加・取得・更新・削除）を実装する
4. 同じ `DataStore` をユーザーデータにも商品データにも使えることを示す
5. ID検索と全件取得の機能を作る

---

## ステップガイド

<details>
<summary>ステップ1：IDを持つ型の制約を作る</summary>

ジェネリクスの制約（constraint）を使って、`id` プロパティを必ず持つ型に限定します。

```typescript
// idプロパティを持つことを保証するインターフェース
interface HasId {
  id: string;
}

// T は HasId を拡張する型のみ受け付ける
interface DataStore<T extends HasId> {
  items: T[];
  add(item: T): void;
  findById(id: string): T | undefined;
  getAll(): T[];
}
```

`T extends HasId` により、`DataStore` に入れるデータは必ず `id: string` を持つことが保証されます。

</details>

<details>
<summary>ステップ2：具体的なデータ型を定義する</summary>

```typescript
interface User extends HasId {
  id: string;
  name: string;
  email: string;
}

interface Product extends HasId {
  id: string;
  name: string;
  price: number;
}
```

`User` も `Product` も `HasId` を拡張しているので、`DataStore<User>` や `DataStore<Product>` として使えます。

</details>

<details>
<summary>ステップ3：DataStoreを実装する</summary>

```typescript
function createStore<T extends HasId>(): DataStore<T> {
  const items: T[] = [];
  return {
    items,
    add(item: T): void {
      items.push(item);
    },
    findById(id: string): T | undefined {
      return items.find((item: T): boolean => item.id === id);
    },
    getAll(): T[] {
      return [...items]; // コピーを返す
    },
  };
}
```

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```typescript
// 汎用データストア
// 学べる内容：ジェネリクス、制約（extends）、ユニオン型（T | undefined）
// 実行方法：npx ts-node 05_data_store.ts

// --- 基本型定義 ---
interface HasId {
  id: string;
}

interface User extends HasId {
  id: string;
  name: string;
  email: string;
}

interface Product extends HasId {
  id: string;
  name: string;
  price: number;
}

// --- 汎用データストア ---
class DataStore<T extends HasId> {
  private items: T[] = [];

  add(item: T): void {
    this.items.push(item);
  }

  findById(id: string): T | undefined {
    for (const item of this.items) {
      if (item.id === id) {
        return item;
      }
    }
    return undefined;
  }

  getAll(): T[] {
    return [...this.items];
  }

  remove(id: string): boolean {
    const index: number = this.items.findIndex(
      (item: T): boolean => item.id === id
    );
    if (index === -1) {
      return false;
    }
    this.items.splice(index, 1);
    return true;
  }

  count(): number {
    return this.items.length;
  }
}

// --- ユーザーストアの使用 ---
console.log("===== 汎用データストア =====\n");
console.log("--- ユーザーストア ---");

const userStore: DataStore<User> = new DataStore<User>();

const users: User[] = [
  { id: "u1", name: "田中太郎", email: "tanaka@example.com" },
  { id: "u2", name: "佐藤花子", email: "sato@example.com" },
  { id: "u3", name: "鈴木一郎", email: "suzuki@example.com" },
];

for (const user of users) {
  userStore.add(user);
  console.log(`追加: ${user.name} (${user.email})`);
}

console.log("\n全ユーザー:");
const allUsers: User[] = userStore.getAll();
for (const user of allUsers) {
  console.log(`  [${user.id}] ${user.name} (${user.email})`);
}

const found: User | undefined = userStore.findById("u2");
if (found !== undefined) {
  console.log(`\nID検索: u2 → ${found.name}`);
} else {
  console.log("\nID検索: u2 → 見つかりません");
}

// --- 商品ストアの使用 ---
console.log("\n--- 商品ストア ---");

const productStore: DataStore<Product> = new DataStore<Product>();

const products: Product[] = [
  { id: "p1", name: "TypeScript本", price: 2800 },
  { id: "p2", name: "マウス", price: 3500 },
];

for (const product of products) {
  productStore.add(product);
  console.log(`追加: ${product.name} (¥${product.price.toLocaleString()})`);
}

console.log("\n全商品:");
const allProducts: Product[] = productStore.getAll();
for (const product of allProducts) {
  console.log(
    `  [${product.id}] ${product.name} ¥${product.price.toLocaleString()}`
  );
}

// --- 統計 ---
console.log("\n--- ストア統計 ---");
console.log(`ユーザーストア: ${userStore.count()}件`);
console.log(`商品ストア: ${productStore.count()}件`);
```

<details>
<summary>解答例（改良版 ─ ジェネリック関数と複数の型パラメータ）</summary>

```typescript
// 汎用データストア（改良版）
// 学べる内容：複数の型パラメータ、ジェネリック関数、keyof制約
// 実行方法：npx ts-node 05_data_store_v2.ts

// --- 基本型定義 ---
interface HasId {
  readonly id: string;
}

interface User extends HasId {
  readonly id: string;
  readonly name: string;
  readonly email: string;
  readonly age: number;
}

interface Product extends HasId {
  readonly id: string;
  readonly name: string;
  readonly price: number;
  readonly category: string;
}

// --- 汎用データストア（改良版） ---
class DataStore<T extends HasId> {
  private items: Map<string, T> = new Map();

  add(item: T): void {
    this.items.set(item.id, item);
  }

  findById(id: string): T | undefined {
    return this.items.get(id);
  }

  getAll(): T[] {
    return Array.from(this.items.values());
  }

  remove(id: string): boolean {
    return this.items.delete(id);
  }

  count(): number {
    return this.items.size;
  }

  // ジェネリックメソッド：任意のプロパティで検索
  findByProperty<K extends keyof T>(key: K, value: T[K]): T[] {
    return this.getAll().filter((item: T): boolean => item[key] === value);
  }

  // 更新（部分更新に対応）
  update(id: string, updates: Partial<Omit<T, "id">>): T | undefined {
    const existing: T | undefined = this.items.get(id);
    if (existing === undefined) return undefined;

    const updated: T = { ...existing, ...updates };
    this.items.set(id, updated);
    return updated;
  }
}

// --- ストア間で変換するジェネリック関数 ---
function mapStore<T extends HasId, U extends HasId>(
  store: DataStore<T>,
  transform: (item: T) => U
): DataStore<U> {
  const newStore: DataStore<U> = new DataStore<U>();
  for (const item of store.getAll()) {
    newStore.add(transform(item));
  }
  return newStore;
}

// --- 2つのストアをマージするジェネリック関数 ---
function mergeStores<T extends HasId>(
  store1: DataStore<T>,
  store2: DataStore<T>
): DataStore<T> {
  const merged: DataStore<T> = new DataStore<T>();
  for (const item of store1.getAll()) merged.add(item);
  for (const item of store2.getAll()) merged.add(item);
  return merged;
}

// --- メイン処理 ---
console.log("===== 汎用データストア =====\n");

// ユーザーストア
const userStore: DataStore<User> = new DataStore<User>();
userStore.add({ id: "u1", name: "田中太郎", email: "tanaka@example.com", age: 28 });
userStore.add({ id: "u2", name: "佐藤花子", email: "sato@example.com", age: 24 });
userStore.add({ id: "u3", name: "鈴木一郎", email: "suzuki@example.com", age: 35 });

console.log("--- ユーザーストア ---");
for (const user of userStore.getAll()) {
  console.log(`  [${user.id}] ${user.name} (${user.email})`);
}

// プロパティ検索
console.log("\n--- 年齢28のユーザー ---");
const age28: User[] = userStore.findByProperty("age", 28);
for (const user of age28) {
  console.log(`  ${user.name} (${user.age}歳)`);
}

// 部分更新
console.log("\n--- ユーザー更新 ---");
const updated: User | undefined = userStore.update("u1", { name: "田中次郎", age: 29 });
if (updated) {
  console.log(`  更新: ${updated.name} (${updated.age}歳)`);
}

// 商品ストア
const productStore: DataStore<Product> = new DataStore<Product>();
productStore.add({ id: "p1", name: "TypeScript本", price: 2800, category: "書籍" });
productStore.add({ id: "p2", name: "マウス", price: 3500, category: "電子機器" });

console.log("\n--- 商品ストア ---");
for (const p of productStore.getAll()) {
  console.log(`  [${p.id}] ${p.name} ¥${p.price.toLocaleString()} (${p.category})`);
}

// mapStore: User → 簡易プロフィール型への変換
interface SimpleProfile extends HasId {
  readonly id: string;
  readonly displayName: string;
}

const profileStore: DataStore<SimpleProfile> = mapStore(
  userStore,
  (user: User): SimpleProfile => ({
    id: user.id,
    displayName: `${user.name} <${user.email}>`,
  })
);

console.log("\n--- 変換後プロフィール ---");
for (const p of profileStore.getAll()) {
  console.log(`  [${p.id}] ${p.displayName}`);
}

// 統計
console.log("\n--- ストア統計 ---");
console.log(`ユーザーストア: ${userStore.count()}件`);
console.log(`商品ストア: ${productStore.count()}件`);
console.log(`プロフィールストア: ${profileStore.count()}件`);
```

**初心者向けとの違い:**
- `Map<string, T>` で内部データを管理（ID検索が O(1) になる）
- `findByProperty<K extends keyof T>(key: K, value: T[K])` で任意のプロパティ検索（`keyof` と型パラメータの組み合わせ）
- `Partial<Omit<T, "id">>` で部分更新の型安全性を実現
- `mapStore<T, U>` で2つの型パラメータを持つジェネリック関数
- ストアの変換・マージといった実務的なパターン

</details>
