# 実践課題07：カスタムプロパティ ─ ダーク/ライトテーマ切替 ★3

> **難易度**: ★★★☆☆（中級）
> **前提知識**: 第1章（CSSの基本）、第7章（カスタムプロパティ）、第8章（疑似クラスと疑似要素）
> **課題の種類**: ミニプロジェクト
> **学習目標**: CSS カスタムプロパティ（CSS変数）を使って、ダークモードとライトモードを切り替えるUIを作る（JavaScriptは最小限）

---

## 完成イメージ

### ライトモード

```
┌──────────────────────────────────┐  ← 白背景
│  MyApp            [🌙]           │
├──────────────────────────────────┤
│                                  │
│  ようこそ                         │
│  これはテーマ切替のデモです        │
│                                  │
│  ┌──────┐ ┌──────┐ ┌──────┐     │
│  │カード1│ │カード2│ │カード3│     │  ← 白カード
│  └──────┘ └──────┘ └──────┘     │
└──────────────────────────────────┘
```

### ダークモード

```
┌──────────────────────────────────┐  ← 暗い背景
│  MyApp            [☀️]           │
├──────────────────────────────────┤
│                                  │
│  ようこそ                         │
│  これはテーマ切替のデモです        │
│                                  │
│  ┌──────┐ ┌──────┐ ┌──────┐     │
│  │カード1│ │カード2│ │カード3│     │  ← 暗いカード
│  └──────┘ └──────┘ └──────┘     │
└──────────────────────────────────┘
```

ボタンを押すとライト/ダークがなめらかに切り替わります。

---

## 課題の要件

1. CSSカスタムプロパティ（CSS variables）で色を一括管理する
2. `:root` にライトモードの色を定義する
3. `[data-theme="dark"]` にダークモードの色を定義する
4. テーマ切替ボタンを設置する
5. 切替時にトランジション（transition）でなめらかに変化させる
6. ヘッダー、メインコンテンツ、カード、フッターを含むページを構成する

---

## ステップガイド

<details>
<summary>ステップ1：カスタムプロパティを定義する</summary>

`:root` にデフォルト（ライトモード）の色を設定し、`[data-theme="dark"]` でダーク用に上書きします。

```css
:root {
  --color-bg: #ffffff;
  --color-text: #333333;
  --color-card: #f5f5f5;
  --color-primary: #4a90d9;
}

[data-theme="dark"] {
  --color-bg: #1a1a2e;
  --color-text: #e0e0e0;
  --color-card: #2a2a40;
  --color-primary: #667eea;
}
```

</details>

<details>
<summary>ステップ2：カスタムプロパティを使う</summary>

`var()` 関数で参照します。

```css
body {
  background-color: var(--color-bg);
  color: var(--color-text);
}
```

</details>

<details>
<summary>ステップ3：テーマ切替ボタンの仕組み</summary>

CSSだけでは属性の切替ができないので、ごく簡単なJavaScriptを1行使います。

```html
<button onclick="document.documentElement.dataset.theme =
  document.documentElement.dataset.theme === 'dark' ? 'light' : 'dark'">
  テーマ切替
</button>
```

</details>
