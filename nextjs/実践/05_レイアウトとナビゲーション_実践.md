# 実践課題05：レイアウトとナビゲーション ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第2章（App Routerとルーティング）、第6章（レイアウトとメタデータ）
> **課題の種類**: ミニプロジェクト
> **学習目標**: ネストされたレイアウト（Nested Layouts）、Route Groups、アクティブリンクの実装を実践する

## 完成イメージ

管理画面（ダッシュボード）を構築します。サイドバー付きのレイアウトで、複数のセクションを持ちます。


---

## 課題の要件

1. ルートレイアウト（`app/layout.tsx`）でヘッダーとフッターを共有する
2. ダッシュボード用のネストされたレイアウト（`app/dashboard/layout.tsx`）でサイドバーを追加する
3. サイドバーのリンクは現在のページに応じてアクティブ状態を表示する
4. Route Groups を使ってマーケティングページ（`/`、`/about`）とダッシュボードで異なるレイアウトを適用する
5. ダッシュボードの各ページ（概要・売上・ユーザー・設定）を作成する

<details>
<summary>ステップ2：アクティブリンクのナビゲーションを作る</summary>

`usePathname` はクライアントコンポーネントで使用する必要があります。

```tsx
// app/dashboard/SidebarNav.tsx
'use client'

import Link from 'next/link'
import { usePathname } from 'next/navigation'

const links = [
  { href: '/dashboard', label: '概要' },
  { href: '/dashboard/sales', label: '売上' },
  // ...
]

export default function SidebarNav() {
  const pathname = usePathname()

  return (
    <nav>
      {links.map((link) => (
        <Link
          key={link.href}
          href={link.href}
          style={{
            fontWeight: pathname === link.href ? 'bold' : 'normal',
          }}
        >
          {link.label}
        </Link>
      ))}
    </nav>
  )
}
```

### app/dashboard/SidebarNav.tsx

```tsx
'use client'

import Link from 'next/link'
import { usePathname } from 'next/navigation'

const navItems = [
  { href: '/dashboard', label: '概要', icon: '📊' },
  { href: '/dashboard/sales', label: '売上', icon: '💰' },
  { href: '/dashboard/users', label: 'ユーザー', icon: '👥' },
  { href: '/dashboard/settings', label: '設定', icon: '⚙️' },
]

export default function SidebarNav() {
  const pathname = usePathname()

  return (
    <nav style={{ display: 'flex', flexDirection: 'column', gap: '4px' }}>
      {navItems.map(function (item) {
        const isActive = pathname === item.href

        return (
          <Link
            key={item.href}
            href={item.href}
            style={{
              display: 'flex',
              alignItems: 'center',
              gap: '8px',
              padding: '10px 16px',
              borderRadius: '8px',
              textDecoration: 'none',
              color: isActive ? '#2563eb' : '#64748b',
              backgroundColor: isActive ? '#eff6ff' : 'transparent',
              fontWeight: isActive ? 'bold' : 'normal',
              fontSize: '14px',
            }}
          >
            <span>{item.icon}</span>
            <span>{item.label}</span>
          </Link>
        )
      })}
    </nav>
  )
}
```

### app/dashboard/sales/page.tsx

```tsx
import type { Metadata } from 'next'

export const metadata: Metadata = {
  title: '売上',
}

export default function SalesPage() {
  const salesData = [
    { month: '1月', amount: 980000 },
    { month: '2月', amount: 1120000 },
    { month: '3月', amount: 1200000 },
    { month: '4月', amount: 1050000 },
  ]

  const maxAmount = Math.max(...salesData.map((d) => d.amount))

  return (
    <div>
      <h1 style={{ marginTop: 0 }}>売上</h1>
      <div style={{ backgroundColor: 'white', padding: '24px', borderRadius: '12px', border: '1px solid #e2e8f0' }}>
        {salesData.map(function (data) {
          const width = (data.amount / maxAmount) * 100
          return (
            <div key={data.month} style={{ marginBottom: '16px' }}>
              <div style={{ display: 'flex', justifyContent: 'space-between', marginBottom: '4px' }}>
                <span>{data.month}</span>
                <span>¥{data.amount.toLocaleString()}</span>
              </div>
              <div style={{ height: '24px', backgroundColor: '#e2e8f0', borderRadius: '4px', overflow: 'hidden' }}>
                <div
                  style={{
                    height: '100%',
                    width: `${width}%`,
                    backgroundColor: '#2563eb',
                    borderRadius: '4px',
                  }}
                />
              </div>
            </div>
          )
        })}

  )
}
```

### ファイル構造の変更

```
app/
├── layout.tsx                    ← ルートレイアウト（最小限）
├── (marketing)/
│   ├── layout.tsx                ← マーケティング用レイアウト
│   ├── page.tsx                  ← / (トップ)

├── (dashboard)/
```

```tsx
// template.tsx は遷移のたびに再マウントされる（layout.tsx とは異なる）

export default function DashboardTemplate({
  children,

      style={{

      }}

          from { opacity: 0; transform: translateY(8px); }
          to { opacity: 1; transform: translateY(0); }
        }
      `}</style>
      {children}
    </div>
  )
}
```

**初心者向けとの違い:**
- Route Groups `(marketing)` と `(dashboard)` で完全に異なるレイアウトを適用
- `template.tsx` でページ遷移時のフェードインアニメーションを実装（`layout.tsx` との違いを体感）
- マーケティングページとダッシュボードで異なるナビゲーション構造
- ルートレイアウトは `<html>` と `<body>` のみに限定し、責務を分離
