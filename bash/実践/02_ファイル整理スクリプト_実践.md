## 課題の要件

1. コマンドライン引数でターゲットディレクトリを受け取る
2. 引数がない場合やディレクトリが存在しない場合、エラーメッセージを表示する
3. 拡張子に応じてサブディレクトリに振り分ける
   - `.txt`, `.csv`, `.log` → `text/`
   - `.jpg`, `.jpeg`, `.png`, `.gif` → `images/`
   - `.pdf`, `.doc`, `.docx` → `documents/`
   - `.sh`, `.bash` → `scripts/`
   - その他 → `others/`
4. サブディレクトリが存在しない場合は自動作成する
5. 移動したファイル数をカウントして最後に表示する

## ステップガイド

<details>
<summary>ステップ1：引数チェックを書く</summary>

`$1` で引数を取得し、`-z` で空チェック、`-d` でディレクトリ存在チェックを行います。

```bash
#!/usr/bin/env bash

target_dir="$1"

if [ -z "$target_dir" ]; then
    echo "使い方: $0 <ディレクトリ>"
    exit 1
fi

if [ ! -d "$target_dir" ]; then
    echo "エラー: '$target_dir' はディレクトリではありません"
    exit 1
fi
```

</details>

<details>
<summary>ステップ2：拡張子を取得して振り分け先を決める</summary>

ファイル名から拡張子を取り出すには、パラメータ展開 `${filename##*.}` を使います。
`case` 文で振り分け先を決定しましょう。

```bash
ext="${filename##*.}"
ext=$(echo "$ext" | tr '[:upper:]' '[:lower:]')  # 小文字に変換

case "$ext" in
    txt|csv|log) dest="text" ;;
    jpg|jpeg|png|gif) dest="images" ;;
    # ... 残りも同様に
esac
```

</details>

<details>
<summary>ステップ3：ファイルを移動してカウントする</summary>

振り分け先ディレクトリを `mkdir -p` で作成し、`mv` でファイルを移動します。

```bash
mkdir -p "${target_dir}/${dest}"
mv "$filepath" "${target_dir}/${dest}/"
count=$((count + 1))
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```bash
#!/usr/bin/env bash
# ファイル整理スクリプト
# 学べる内容：引数処理、条件分岐、ファイルテスト、case文、ループ
# 実行方法：chmod +x organize.sh && ./organize.sh <ディレクトリ>

# --- 引数チェック ---
target_dir="$1"

if [ -z "$target_dir" ]; then
    echo "使い方: $0 <ディレクトリ>"
    exit 1
fi

if [ ! -d "$target_dir" ]; then
    echo "エラー: '$target_dir' はディレクトリではありません"
    exit 1
fi

echo ""
echo "ファイル整理を開始します: $target_dir"

count=0

# --- ファイルを1つずつ処理 ---
for filepath in "$target_dir"/*; do
    # ディレクトリはスキップ
    if [ ! -f "$filepath" ]; then
        continue
    fi

    # ファイル名と拡張子を取得
    filename=$(basename "$filepath")
    ext="${filename##*.}"
    ext=$(echo "$ext" | tr '[:upper:]' '[:lower:]')

    # 拡張子に応じて振り分け先を決定
    case "$ext" in
        txt|csv|log)
            dest="text"
            ;;
        jpg|jpeg|png|gif)
            dest="images"
            ;;
        pdf|doc|docx)
            dest="documents"
            ;;
        sh|bash)
            dest="scripts"
            ;;
        *)
            dest="others"
            ;;
    esac

    # サブディレクトリ作成 & 移動
    mkdir -p "${target_dir}/${dest}"
    mv "$filepath" "${target_dir}/${dest}/"
    echo "  ${filename} → ${dest}/ に移動しました"

    count=$((count + 1))
done

echo ""
echo "整理完了！ ${count} 個のファイルを移動しました。"
```

</details>

<details>
<summary>解答例（改良版 ─ 連想配列とドライランモード付き）</summary>

```bash
#!/usr/bin/env bash
# ファイル整理スクリプト（改良版）
# 連想配列による拡張子マッピングとドライランモードを追加
# 実行方法：chmod +x organize.sh && ./organize.sh [-n] <ディレクトリ>
#   -n : ドライラン（実際には移動しない）

# --- オプション解析 ---
dry_run=false
if [ "$1" = "-n" ]; then
    dry_run=true
    shift
fi

target_dir="$1"

if [ -z "$target_dir" ]; then
    echo "使い方: $0 [-n] <ディレクトリ>"
    echo "  -n : ドライラン（移動をシミュレート）"
    exit 1
fi

if [ ! -d "$target_dir" ]; then
    echo "エラー: '$target_dir' はディレクトリではありません" >&2
    exit 1
fi

# --- 拡張子 → カテゴリのマッピング（連想配列） ---
declare -A ext_map
ext_map=(
    [txt]="text" [csv]="text" [log]="text" [md]="text"
    [jpg]="images" [jpeg]="images" [png]="images" [gif]="images" [svg]="images"
    [pdf]="documents" [doc]="documents" [docx]="documents"
    [sh]="scripts" [bash]="scripts" [py]="scripts" [rb]="scripts"
    [tar]="archives" [gz]="archives" [zip]="archives" [bz2]="archives"
)

echo ""
if $dry_run; then
    echo "[ドライラン] ファイル整理をシミュレートします: $target_dir"
else
    echo "ファイル整理を開始します: $target_dir"
fi

count=0

for filepath in "$target_dir"/*; do
    [ -f "$filepath" ] || continue

    filename=$(basename "$filepath")
    ext="${filename##*.}"
    ext="${ext,,}"  # Bash 4.0+ の小文字変換

    # 拡張子がない場合（ファイル名と拡張子が同じ）
    if [ "$ext" = "$filename" ]; then
        dest="others"
    else
        dest="${ext_map[$ext]:-others}"
    fi

    if $dry_run; then
        echo "  [DRY] ${filename} → ${dest}/"
    else
        mkdir -p "${target_dir}/${dest}"
        mv "$filepath" "${target_dir}/${dest}/"
        echo "  ${filename} → ${dest}/ に移動しました"
    fi

    count=$((count + 1))
done

echo ""
echo "整理完了！ ${count} 個のファイルを処理しました。"
```

**初心者向けとの違い:**
- 連想配列 `declare -A` で拡張子マッピングを管理 → カテゴリ追加が1行で済む
- `-n` オプションでドライラン → 実行前に結果を確認でき安全
- `${ext,,}` で小文字変換（Bash 4.0+）→ `tr` コマンド不要
- `${ext_map[$ext]:-others}` でデフォルト値指定 → case文の `*)`が不要
- エラーメッセージを `>&2` で標準エラーに出力 → パイプ利用時に安全
- 拡張子がないファイルのハンドリングを追加

</details>
