# 第4章：ビジュアルモードと選択操作

## この章のゴール

- 3種類のビジュアルモード（文字、行、矩形）を使い分けられる
- ビジュアルモードでテキストを選択して操作できる
- 矩形選択で複数行を同時に編集できる
- インデント操作ができる

---

## 1. ビジュアルモードの種類

ビジュアルモードは、テキストを視覚的に選択してから操作を行うモードです。3つの種類があります。

### 3つのビジュアルモード

```
  v       ← 文字単位の選択 (character-wise)
  V       ← 行単位の選択 (line-wise)
  Ctrl+v  ← 矩形選択 (block-wise)
```

### 選択範囲の違い

```
  元のテキスト:
  ┌──────────────────────┐
  │ Hello World          │
  │ Vim is great         │
  │ Let's learn          │
  └──────────────────────┘

  【v（文字単位選択）】HからVまで選択した場合:
  ┌──────────────────────┐
  │ [Hello World]        │  ← ] は選択範囲
  │ [Vim] is great       │
  │ Let's learn          │
  └──────────────────────┘

  【V（行単位選択）】1行目と2行目を選択した場合:
  ┌──────────────────────┐
  │[Hello World         ]│  ← 行全体が選択される
  │[Vim is great        ]│
  │ Let's learn          │
  └──────────────────────┘

  【Ctrl+v（矩形選択）】矩形で選択した場合:
  ┌──────────────────────┐
  │ [Hello] World        │  ← 矩形（ブロック）で選択
  │ [Vim i]s great       │
  │ [Let's] learn        │
  └──────────────────────┘
```

### ポイントまとめ

- `v` は文字単位、`V` は行単位、`Ctrl+v` は矩形選択です
- 用途に応じて使い分けることで効率的に編集できます
- 選択後に操作（削除・コピーなど）を適用します

---

## 2. ビジュアルモードでの選択と操作

### 基本の流れ

```
  1. v / V / Ctrl+v でビジュアルモードに入る
  2. hjkl やモーション（w, $, G など）で選択範囲を広げる
  3. 操作コマンドを実行する
```

### 選択後に使える操作

```
  d    ← 選択範囲を削除
  y    ← 選択範囲をヤンク（コピー）
  c    ← 選択範囲を変更（削除して挿入モードへ）
  >    ← 選択範囲をインデント
  <    ← 選択範囲のインデントを減らす
  =    ← 選択範囲を自動インデント
  ~    ← 大文字/小文字を反転
  U    ← 大文字に変換
  u    ← 小文字に変換
  J    ← 選択行を結合
  :    ← 選択範囲に対してコマンドを実行
```

### 便利な選択テクニック

```
  gv     ← 前回の選択範囲を再選択
  o      ← 選択範囲の反対側にカーソルを移動
  O      ← 矩形選択で角にカーソルを移動
```

### 実践例：関数全体を選択してコピー

```python
def greet(name):
    message = f"Hello, {name}!"
    print(message)
    return message
```

```
  1. def の行にカーソルを置く
  2. V で行単位ビジュアルモードに入る
  3. } で段落末尾まで移動（または 3j で3行下へ）
  4. y でヤンク
  → 関数全体がコピーされた
```

### ポイントまとめ

- ビジュアルモードでは「選択 → 操作」の順で作業します
- `gv` で前回の選択範囲を再選択できます
- `o` で選択範囲の反対端にカーソルを移動できます

## 3. 矩形選択（ブロック選択）の活用

矩形選択（`Ctrl+v`）は、列方向の編集に非常に強力です。

### 複数行の先頭に文字を挿入

```
  元のテキスト:
  line 1
  line 2
  line 3
  line 4

  操作手順:
  1. 1行目の先頭にカーソルを置く
  2. Ctrl+v で矩形選択モードに入る
  3. 3j で4行分選択する
  4. I（大文字のI）を押す
  5. "# " と入力する
  6. ESC を押す

  結果:
  # line 1
  # line 2
  # line 3
  # line 4
```

### 複数行の末尾に文字を追加

```
  元のテキスト:
  item1
  item2
  item3

  操作手順:
  1. 1行目にカーソルを置く
  2. Ctrl+v で矩形選択モードに入る
  3. 2j で3行分選択する
  4. $ で行末まで選択を広げる
  5. A（大文字のA）を押す
  6. "," と入力する
  7. ESC を押す

  結果:
  item1,
  item2,
  item3,
```

### 列の削除

```
  元のテキスト（CSVデータ）:
  name,age,city
  Alice,30,Tokyo
  Bob,25,Osaka

  2列目（age）を削除したい場合:
  1. 1行目の最初のカンマの次の文字にカーソルを置く
  2. Ctrl+v で矩形選択
  3. 2j で3行選択
  4. f, でカンマまで選択を広げる
  5. d で削除

  結果:
  name,city
  Alice,Tokyo
  Bob,Osaka
```

### ポイントまとめ

- `Ctrl+v` → 選択 → `I` で複数行の先頭に文字を挿入できます
- `Ctrl+v` → `$` → `A` で複数行の末尾に文字を追加できます
- 列単位の削除・追加はCSVやテーブルデータの編集に非常に便利です

## 4. インデント操作

### ノーマルモードでのインデント

```
  >>    ← 現在行を右にインデント
  <<    ← 現在行を左にインデント
  ==    ← 現在行を自動インデント

  3>>   ← 3行を右にインデント
  >G    ← 現在行からファイル末尾までインデント
  >ip   ← 段落全体をインデント
```

### ビジュアルモードでのインデント

```
  1. V で行単位ビジュアルモードに入る
  2. 対象行を選択する
  3. > で右にインデント
  4. < で左にインデント
```

> **便利なテクニック**: ビジュアルモードでインデントした後、`gv` で再選択して `>` を押すと、さらにインデントを追加できます。

### 自動インデント

```
  =    ← オペレータとして使用

  gg=G   ← ファイル全体を自動インデント
          （gg でファイル先頭に移動、=G で末尾まで自動インデント）

  =ip    ← 現在の段落を自動インデント
  =i{    ← 波括弧内を自動インデント
```

### ポイントまとめ

- `>>` / `<<` で行のインデントを調整します
- ビジュアルモード + `>` / `<` で範囲指定のインデントが便利です
- `gg=G` でファイル全体の自動インデントができます

## 5. 実践：ビジュアルモードの活用例

### 例1：コードブロックのコメントアウト

```python
# 以下の3行をコメントアウトしたい
print("debug 1")
print("debug 2")
print("debug 3")
```

```
  1. 最初の print の行頭にカーソルを置く
  2. Ctrl+v で矩形選択
  3. 2j で3行選択
  4. I を押す
  5. "# " と入力
  6. ESC を押す

  結果:
  # print("debug 1")
  # print("debug 2")
  # print("debug 3")
```

### 例2：行の並べ替え

```
  元: 以下の行をアルファベット順にソートしたい
  Cherry
  Apple
  Banana

  1. V で1行目を選択
  2. 2j で3行すべてを選択
  3. :sort と入力して Enter

  結果:
  Apple
  Banana
  Cherry
```

### 例3：選択範囲の文字を大文字に変換

```
  元: let status = "active";

  1. v で文字単位ビジュアルモードに入る
  2. iwで "active" の単語を選択
  3. U で大文字に変換

  結果: let STATUS = "active";
```

### 例4：複数行を1行にまとめる

```
  元:
  SELECT name,
         age,
         city
  FROM users

  1. V で SELECT の行を選択
  2. 2j で3行選択
  3. J で行を結合

  結果:
  SELECT name, age, city
  FROM users
```

### ポイントまとめ

- 矩形選択 + `I` でコメントアウトが簡単にできます
- ビジュアル選択 + `:sort` で行のソートができます
- `J` で複数行を1行に結合できます
- ビジュアルモードは「まず選択して、それから操作する」という直感的な方法です
