# 第6章：バッファ・ウィンドウ・タブ

## この章のゴール

- バッファ・ウィンドウ・タブの違いを理解する
- 複数ファイルを効率的に編集できる
- 画面分割を活用できる
- タブページを使い分けられる

---

## 1. バッファ・ウィンドウ・タブの関係

Vim では複数のファイルを扱う仕組みが3つの層に分かれています。

### 階層構造

```
  ┌─────────────────────────────────────────────┐
  │ タブページ (Tab Page)                        │
  │  ┌──────────────────┬──────────────────┐    │
  │  │ ウィンドウ1       │ ウィンドウ2       │    │
  │  │                  │                  │    │
  │  │ バッファA        │ バッファB        │    │
  │  │ (file1.py)       │ (file2.py)       │    │
  │  │                  │                  │    │
  │  └──────────────────┴──────────────────┘    │
  └─────────────────────────────────────────────┘

  バッファ: ファイルの内容をメモリに読み込んだもの
  ウィンドウ: バッファを表示する「窓」
  タブページ: ウィンドウの集合（レイアウト）
```

### それぞれの関係

```
  ・1つのバッファは複数のウィンドウに表示できる
  ・1つのウィンドウには1つのバッファが表示される
  ・1つのタブページには複数のウィンドウを配置できる

  バッファ ──── メモリ上のファイル（開いたファイルの数だけ存在）
  ウィンドウ ── バッファを見る「窓」（画面分割で増減）
  タブページ ── ウィンドウの配置セット（作業場を切り替え）
```

> **よくある間違い**: Vim のタブは、ブラウザのタブとは概念が異なります。ブラウザではタブ = ファイルですが、Vim ではタブ = ウィンドウのレイアウト（配置）です。ファイルの切り替えにはバッファを使うのが Vim の流儀です。

### ポイントまとめ

- バッファ = ファイルの内容、ウィンドウ = 表示領域、タブ = レイアウトです
- ファイルの切り替えにはバッファ操作を使うのが基本です
- タブはウィンドウのレイアウトを切り替えるために使います

---

## 2. バッファ操作

### ファイルを開く

```vim
:e filename       " ファイルを開く（edit）
:e .              " カレントディレクトリのファイルエクスプローラを開く
:e %:h/           " 現在のファイルと同じディレクトリを開く
```

### バッファの一覧と切り替え

```vim
:ls               " バッファ一覧を表示（:buffers も同じ）
:bn               " 次のバッファへ (buffer next)
:bp               " 前のバッファへ (buffer previous)
:b {番号}         " 指定番号のバッファへ
:b {名前}         " ファイル名（部分一致）で切り替え
:b#               " 直前のバッファへ（Ctrl+^ でも可）
```

### バッファ一覧の見方

```
  :ls の出力例:

    1 %a   "file1.py"      line 10    ← 現在アクティブ
    2 #    "file2.py"      line 1     ← 直前のバッファ
    3      "file3.py"      line 1
    4  a   "file4.py"      line 5     ← アクティブ（他のウィンドウ）

  記号の意味:
    %  ← 現在のウィンドウに表示中
    #  ← 直前のバッファ（Ctrl+^ で切り替え可能）
    a  ← アクティブ（ウィンドウに表示中）
    h  ← 隠れている（ウィンドウに表示されていない）
    +  ← 変更あり（未保存）
```

### バッファの削除

```vim
:bd               " 現在のバッファを削除 (buffer delete)
:bd {番号}        " 指定番号のバッファを削除
:bd file1.py      " ファイル名を指定して削除
:%bd              " 全バッファを削除
```

### ポイントまとめ

- `:e ファイル名` でファイルを開き、`:ls` で一覧を確認します
- `:bn`/`:bp` または `:b 番号` でバッファを切り替えます
- `Ctrl+^` で直前のバッファとトグルできます

## 3. ウィンドウ分割

### 画面を分割する

```vim
:sp               " 水平分割（同じバッファ）(split)
:sp filename      " 水平分割して別のファイルを開く
:vs               " 垂直分割（同じバッファ）(vertical split)
:vs filename      " 垂直分割して別のファイルを開く
:new              " 水平分割して新しい空バッファ
:vnew             " 垂直分割して新しい空バッファ
```

```
  【水平分割 :sp】           【垂直分割 :vs】
  ┌──────────────────┐      ┌─────────┬─────────┐
  │                  │      │         │         │
  │   ウィンドウ1     │      │ ウィンドウ│ ウィンドウ│
  │                  │      │    1    │    2    │
  ├──────────────────┤      │         │         │
  │                  │      │         │         │
  │   ウィンドウ2     │      │         │         │
  │                  │      │         │         │
  └──────────────────┘      └─────────┴─────────┘
```

### ウィンドウ間の移動

```
  Ctrl+w の後に方向キーを入力:

  Ctrl+w h  ← 左のウィンドウへ
  Ctrl+w j  ← 下のウィンドウへ
  Ctrl+w k  ← 上のウィンドウへ
  Ctrl+w l  ← 右のウィンドウへ
  Ctrl+w w  ← 次のウィンドウへ（順番にサイクル）
  Ctrl+w p  ← 直前のウィンドウへ
```

### ウィンドウのサイズ調整

```
  Ctrl+w =    ← 全ウィンドウを均等サイズにする
  Ctrl+w _    ← 現在のウィンドウの高さを最大にする
  Ctrl+w |    ← 現在のウィンドウの幅を最大にする
  Ctrl+w +    ← 高さを1行増やす
  Ctrl+w -    ← 高さを1行減らす
  Ctrl+w >    ← 幅を1列増やす
  Ctrl+w <    ← 幅を1列減らす
  10 Ctrl+w + ← 高さを10行増やす
```

### ウィンドウを閉じる

```vim
:close            " 現在のウィンドウを閉じる（Ctrl+w c）
:only             " 現在のウィンドウ以外を全て閉じる（Ctrl+w o）
:q                " ウィンドウを閉じる（最後のウィンドウなら Vim を終了）
```

### ウィンドウの移動と回転

```
  Ctrl+w H  ← 現在のウィンドウを左端に移動
  Ctrl+w J  ← 現在のウィンドウを下端に移動
  Ctrl+w K  ← 現在のウィンドウを上端に移動
  Ctrl+w L  ← 現在のウィンドウを右端に移動
  Ctrl+w r  ← ウィンドウを回転
```

### ポイントまとめ

- `:sp` で水平分割、`:vs` で垂直分割です
- `Ctrl+w` + `h`/`j`/`k`/`l` でウィンドウ間を移動します
- `Ctrl+w =` で均等サイズ、`Ctrl+w _` / `Ctrl+w |` で最大化します

## 4. タブページ

### タブの操作

```vim
:tabnew           " 新しい空のタブを開く
:tabedit filename " 新しいタブでファイルを開く（:tabe も可）
:tabclose         " 現在のタブを閉じる（:tabc も可）
:tabonly          " 現在のタブ以外を全て閉じる（:tabo も可）
```

### タブの切り替え

```
  gt          ← 次のタブへ
  gT          ← 前のタブへ
  {数字}gt    ← 指定番号のタブへ（1gt で1番目のタブ）
  :tabn       ← 次のタブへ (tab next)
  :tabp       ← 前のタブへ (tab previous)
  :tabfirst   ← 最初のタブへ
  :tablast    ← 最後のタブへ
```

### タブの表示

```
  画面上部にタブが表示されます:

  [file1.py] [file2.py] [file3.py]
                ↑ 現在のタブ（ハイライトされる）
```

### タブの実用的な使い方

タブは異なる作業コンテキストを切り替えるのに適しています。

```
  例：Web開発での使い方

  タブ1: フロントエンド作業
    ┌───────────┬───────────┐
    │ App.jsx   │ style.css │
    └───────────┴───────────┘

  タブ2: バックエンド作業
    ┌───────────┬───────────┐
    │ server.py │ models.py │
    └───────────┴───────────┘

  タブ3: テスト
    ┌─────────────────────┐
    │ test_server.py      │
    └─────────────────────┘
```

### ポイントまとめ

- `:tabedit ファイル名` で新しいタブを開きます
- `gt`/`gT` でタブを切り替えます
- タブは作業コンテキスト（フロント/バック/テストなど）の切り替えに使います

## 5. 実践的なファイル管理

### netrw ファイルエクスプローラ

Vim にはファイルエクスプローラ（netrw）が組み込まれています。

```vim
:Explore          " 現在のウィンドウでファイルエクスプローラを開く（:Ex も可）
:Sexplore         " 水平分割で開く（:Sex も可）
:Vexplore         " 垂直分割で開く（:Vex も可）
:Texplore         " 新しいタブで開く（:Tex も可）
```

### netrw での操作

```
  Enter    ← ファイルを開く / ディレクトリに入る
  -        ← 親ディレクトリに移動
  %        ← 新しいファイルを作成
  d        ← 新しいディレクトリを作成
  D        ← ファイル/ディレクトリを削除
  R        ← ファイル名を変更
  i        ← 表示形式を切り替え（リスト/詳細/ツリー）
```

### 引数リスト

```vim
" 複数ファイルを一度に開く（コマンドラインから）
$ vim file1.py file2.py file3.py

" Vim 内で引数リストを確認
:args

" 引数リストのファイル間を移動
:next             " 次のファイルへ（:n も可）
:prev             " 前のファイルへ（:N も可）
:first            " 最初のファイルへ
:last             " 最後のファイルへ

" 全ファイルに対してコマンドを実行
:argdo %s/old/new/g | update
```

### 複数ファイルの一括操作

```vim
" 全バッファに対してコマンドを実行
:bufdo %s/old/new/g | update

" 全ウィンドウに対してコマンドを実行
:windo diffthis

" 全タブに対してコマンドを実行
:tabdo %s/old/new/g | update
```

### ポイントまとめ

- `:Explore` で組み込みファイルエクスプローラを使えます
- `:args` と `:next`/`:prev` で複数ファイルを順番に編集できます
- `:bufdo`、`:argdo` で全バッファ/引数に対して一括操作できます
