# 実践課題05：キーマッピング構築 ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第5章（キーマッピング）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `nnoremap` / `inoremap` / `vnoremap` を使い分け、モード別に実用的なキーマッピングを構築できるようになる

---

## 完成イメージ

以下のキーマッピングが動作する vimrc を作成します。

```
┌──────────────────────────────────────────────────────────┐
│  キーマッピング一覧                                       │
├──────────────────────────────────────────────────────────┤
│                                                           │
│  【ノーマルモード】                                       │
│    Y       → カーソル位置から行末までヤンク               │
│    j / k   → 表示行単位で移動（折り返し行対応）          │
│    <Space> → Leader キーとして機能                        │
│    <Leader>w → ファイルを保存                             │
│    <Leader>q → Vim を終了                                 │
│    + / -   → ウィンドウサイズを拡大・縮小                │
│                                                           │
│  【挿入モード】                                           │
│    jj      → ノーマルモードに戻る                        │
│    Ctrl+h  → カーソルを左に移動                          │
│    Ctrl+l  → カーソルを右に移動                          │
│                                                           │
│  【ビジュアルモード】                                     │
│    < / >   → インデント後に選択を維持                    │
│    J / K   → 選択行を上下に移動                          │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

---

## 課題の要件

1. Leader キーを `<Space>` に設定する
2. **ノーマルモード**で以下のマッピングを設定する
   - `Y` → `y$`（行末までヤンク）
   - `j` / `k` → `gj` / `gk`（表示行移動）
   - `<Leader>w` → `:w<CR>`（保存）
   - `<Leader>q` → `:q<CR>`（終了）
   - `+` / `-` → ウィンドウの高さを拡大・縮小
3. **挿入モード**で以下のマッピングを設定する
   - `jj` → `<Esc>`（ノーマルモードに戻る）
   - `Ctrl+h` / `Ctrl+l` → カーソル左右移動
4. **ビジュアルモード**で以下のマッピングを設定する
   - `<` / `>` → インデント操作後に選択を維持
   - `J` / `K` → 選択行を上下に移動
5. すべてのマッピングに `noremap` 系を使用すること

## ステップガイド

<details>
<summary>ステップ1：Leader キーを設定する</summary>

Leader キーはカスタムマッピングのプレフィックスとして使います。

```vim
" Leader キーを Space に設定する
let mapleader = "\<Space>"
```

この設定以降、`<Leader>` は Space キーとして解釈されます。

</details>

<details>
<summary>ステップ2：ノーマルモードのマッピングを記述する</summary>

すべて `nnoremap`（ノーマルモード非再帰マッピング）を使います。

```vim
" Y を y$（行末までヤンク）にする（C や D との一貫性）
nnoremap Y y$

" 表示行単位で上下移動（折り返し行がある場合に便利）
nnoremap j gj
nnoremap k gk

" Leader + w でファイル保存
nnoremap <Leader>w :w<CR>
```

</details>

<details>
<summary>ステップ3：挿入モードのマッピングを記述する</summary>

`inoremap`（挿入モード非再帰マッピング）を使います。

```vim
" jj でノーマルモードに戻る
inoremap jj <Esc>

" 挿入モードでのカーソル移動
inoremap <C-h> <Left>
inoremap <C-l> <Right>
```

</details>

<details>
<summary>ステップ4：ビジュアルモードのマッピングを記述する</summary>

`vnoremap`（ビジュアルモード非再帰マッピング）を使います。

```vim
" インデント操作後に選択を維持する
vnoremap < <gv
vnoremap > >gv

" 選択行を上下に移動する
vnoremap J :move '>+1<CR>gv=gv
vnoremap K :move '<-2<CR>gv=gv
```

</details>

" ==================================================

" Leader キーを Space に設定する
let mapleader = "\<Space>"

<details>
<summary>解答例（改良版 ─ より実践的なマッピング集）</summary>

```vim
" ==================================================
" キーマッピング設定（改良版）
" ==================================================

" Leader キーを Space に設定
let mapleader = "\<Space>"
" タイムアウトを短くして Leader キーの反応を速くする
set timeoutlen=500

" ==================================================
" ノーマルモード ─ 基本操作の改善
" ==================================================

" Y で行末までヤンク
nnoremap Y y$

" 表示行単位で移動（ただしカウント付きの場合は論理行で移動）
nnoremap <expr> j v:count == 0 ? 'gj' : 'j'
nnoremap <expr> k v:count == 0 ? 'gk' : 'k'

" 検索結果を画面中央に表示する
nnoremap n nzzzv
nnoremap N Nzzzv

" 行の結合後にカーソル位置を維持する
nnoremap J mzJ`z

" ==================================================
" ノーマルモード ─ Leader マッピング
" ==================================================

" ファイル操作
nnoremap <Leader>w :w<CR>
nnoremap <Leader>q :q<CR>
nnoremap <Leader>x :x<CR>

" バッファ操作
nnoremap <Leader>bn :bnext<CR>
nnoremap <Leader>bp :bprevious<CR>
nnoremap <Leader>bd :bdelete<CR>

" ウィンドウ操作
nnoremap <Leader>v :vsplit<CR>
nnoremap <Leader>h :split<CR>
nnoremap + <C-w>+
nnoremap - <C-w>-

" ウィンドウ間移動を Ctrl + hjkl で行う
nnoremap <C-h> <C-w>h
nnoremap <C-j> <C-w>j
nnoremap <C-k> <C-w>k
nnoremap <C-l> <C-w>l

" ==================================================
" 挿入モード
" ==================================================

" jj でノーマルモードに戻る
inoremap jj <Esc>

" 挿入モードでのカーソル移動（Emacs 風）
inoremap <C-h> <Left>
inoremap <C-l> <Right>
inoremap <C-a> <Home>
inoremap <C-e> <End>

" Ctrl+s で挿入モードのまま保存
inoremap <C-s> <Esc>:w<CR>a

" ==================================================
" ビジュアルモード
" ==================================================

" インデント操作後に選択を維持
vnoremap < <gv
vnoremap > >gv

" 選択行を上下に移動
vnoremap J :move '>+1<CR>gv=gv
vnoremap K :move '<-2<CR>gv=gv

" ペースト後にヤンクレジスタを保持する
vnoremap p "_dP

" ==================================================
" コマンドラインモード
" ==================================================

" Ctrl+a で行頭、Ctrl+e で行末に移動
cnoremap <C-a> <Home>
cnoremap <C-e> <End>
```

**初心者向けとの違い:**
- `<expr>` を使ったスマートな j/k マッピング（カウント付きは論理行で移動）
- `nzzzv` で検索移動後に画面中央に自動スクロール
- バッファ操作・ウィンドウ操作の Leader マッピングで複数ファイル編集が効率化
- `Ctrl+hjkl` でウィンドウ間移動が直感的に
- コマンドラインモードにも Emacs 風のナビゲーションを追加
- ビジュアルモードのペーストで元のヤンク内容を保持

</details>
