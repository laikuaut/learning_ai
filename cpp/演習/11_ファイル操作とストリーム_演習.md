## 問題1：テキストファイル読み書き（基本）

`ofstream` を使ってテキストファイルに複数行を書き込み、`ifstream` を使ってその内容を読み込んで表示するプログラムを作成してください。

- ファイル名は `output.txt` とします
- 3行のテキストを書き込んでください
- 読み込み時は行番号付きで表示します
- ファイルのオープン失敗時にはエラーメッセージを出力してください

```cpp
#include <iostream>
#include <fstream>
#include <string>

int main() {
    // ここに処理を記述
    return 0;
}
```

**期待される出力：**
```
ファイルに書き込みました: output.txt
--- ファイルの内容 ---
1: C++のファイル操作を学びましょう
2: ifstreamで読み込み、ofstreamで書き込みます
3: ストリームは非常に便利です
```

<details>
<summary>ヒント</summary>

- `std::ofstream ofs("output.txt");` でファイルを書き込みモードで開けます
- 書き込みは `ofs << "テキスト" << std::endl;` のように `cout` と同じ要領で行えます
- `std::ifstream ifs("output.txt");` で読み込みモードで開けます
- `std::getline(ifs, line)` で1行ずつ読み込めます
- `ofs.is_open()` や `ifs.is_open()` でオープン成功を確認できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <fstream>
#include <string>

int main() {
    const std::string filename = "output.txt";

    // === ファイルへの書き込み ===
    std::ofstream ofs(filename);
    if (!ofs.is_open()) {
        std::cerr << "エラー: ファイルを開けませんでした: " << filename << std::endl;
        return 1;
    }

    // 3行のテキストを書き込む
    ofs << "C++のファイル操作を学びましょう" << std::endl;
    ofs << "ifstreamで読み込み、ofstreamで書き込みます" << std::endl;
    ofs << "ストリームは非常に便利です" << std::endl;
    ofs.close();  // 明示的にクローズ（デストラクタでも閉じられる）

    std::cout << "ファイルに書き込みました: " << filename << std::endl;

    // === ファイルの読み込み ===
    std::ifstream ifs(filename);
    if (!ifs.is_open()) {
        std::cerr << "エラー: ファイルを開けませんでした: " << filename << std::endl;
        return 1;
    }

    std::cout << "--- ファイルの内容 ---" << std::endl;
    std::string line;
    int lineNum = 1;
    // getlineで1行ずつ読み込む
    while (std::getline(ifs, line)) {
        std::cout << lineNum << ": " << line << std::endl;
        lineNum++;
    }
    ifs.close();

    return 0;
}
```

</details>

## 問題2：CSV読み込み（基本）

CSVファイルを読み込んで、テーブル形式で表示するプログラムを作成してください。

- `stringstream` を使ってカンマ区切りの各フィールドを分割します
- ヘッダー行を特別扱いし、区切り線で分離して表示します
- CSVデータは `stringstream` からの読み込みで代用して構いません（実ファイルがなくても動作するように）

```cpp
#include <iostream>
#include <sstream>
#include <string>
#include <vector>

int main() {
    // CSVデータ（実際にはファイルから読む想定）
    std::string csvData =
        "名前,年齢,職業\n"
        "田中太郎,28,エンジニア\n"
        "佐藤花子,35,デザイナー\n"
        "鈴木一郎,42,マネージャー\n";

    // ここに処理を記述
    return 0;
}
```

**期待される出力：**
```
名前            年齢    職業
------------------------------------
田中太郎        28      エンジニア
佐藤花子        35      デザイナー
鈴木一郎        42      マネージャー

合計: 3件のレコード
```

<details>
<summary>ヒント</summary>

- `std::istringstream` をCSVデータ全体に対して作成し、`std::getline` で1行ずつ取り出します
- 各行をさらに `std::istringstream` に入れ、`std::getline(iss, field, ',')` でカンマ区切りのフィールドを取り出せます
- 表示幅の調整には `\t`（タブ文字）を使うと簡単です
- 1行目はヘッダーとして扱い、区切り線を出力します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <sstream>
#include <string>
#include <vector>

// 1行のCSVをフィールドに分割する関数
std::vector<std::string> parseCsvLine(const std::string& line) {
    std::vector<std::string> fields;
    std::istringstream iss(line);
    std::string field;
    // カンマを区切り文字として1フィールドずつ取り出す
    while (std::getline(iss, field, ',')) {
        fields.push_back(field);
    }
    return fields;
}

int main() {
    // CSVデータ（実際にはファイルから読む想定）
    std::string csvData =
        "名前,年齢,職業\n"
        "田中太郎,28,エンジニア\n"
        "佐藤花子,35,デザイナー\n"
        "鈴木一郎,42,マネージャー\n";

    std::istringstream stream(csvData);
    std::string line;
    bool isHeader = true;
    int recordCount = 0;

    while (std::getline(stream, line)) {
        if (line.empty()) continue;  // 空行をスキップ

        std::vector<std::string> fields = parseCsvLine(line);

        // フィールドをタブ区切りで表示
        for (size_t i = 0; i < fields.size(); i++) {
            std::cout << fields[i];
            if (i < fields.size() - 1) {
                std::cout << "\t";
            }
        }
        std::cout << std::endl;

        if (isHeader) {
            // ヘッダーの後に区切り線を表示
            std::cout << "------------------------------------" << std::endl;
            isHeader = false;
        } else {
            recordCount++;
        }
    }

    std::cout << std::endl;
    std::cout << "合計: " << recordCount << "件のレコード" << std::endl;

    return 0;
}
```

</details>

## 問題3：設定ファイルパーサ（応用）

`key=value` 形式の設定ファイルを解析（パース）するプログラムを作成してください。

- コメント行（`#` で始まる行）と空行をスキップします
- `key=value` のペアを `std::map` に格納します
- キーと値の前後の空白はトリム（除去）します
- 存在しないキーを取得しようとした場合はデフォルト値を返します
- 設定データは `stringstream` で代用して構いません

```cpp
#include <iostream>
#include <sstream>
#include <string>
#include <map>

int main() {
    std::string configData =
        "# アプリケーション設定\n"
        "\n"
        "app_name = MyApplication\n"
        "version = 1.2.3\n"
        "# データベース設定\n"
        "db_host = localhost\n"
        "db_port = 5432\n"
        "  db_name  =  my_database  \n"
        "debug = true\n";

    // ここに処理を記述
    return 0;
}
```

**期待される出力：**
```
=== 設定ファイルの解析結果 ===
app_name = MyApplication
db_host = localhost
db_name = my_database
db_port = 5432
debug = true
version = 1.2.3

=== 値の取得 ===
app_name: MyApplication
db_port: 5432
timeout（未設定）: 30
```

<details>
<summary>ヒント</summary>

- `line[0] == '#'` でコメント行を判定できます（先頭空白トリム後にチェック）
- `line.find('=')` で `=` の位置を探し、その前後でキーと値を分割します
- 文字列のトリムには `find_first_not_of` と `find_last_not_of` を使うと便利です
- `std::map` は自動的にキーの辞書順にソートされるので、出力時もアルファベット順になります
- デフォルト値を返す関数を用意すると便利です：`std::string getValue(const std::map<...>& config, const std::string& key, const std::string& defaultVal)`

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <sstream>
#include <string>
#include <map>

// 文字列の前後の空白を除去する関数
std::string trim(const std::string& str) {
    size_t start = str.find_first_not_of(" \t\r\n");
    if (start == std::string::npos) return "";
    size_t end = str.find_last_not_of(" \t\r\n");
    return str.substr(start, end - start + 1);
}

// 設定値を取得する関数（キーが存在しない場合はデフォルト値を返す）
std::string getValue(const std::map<std::string, std::string>& config,
                     const std::string& key,
                     const std::string& defaultVal = "") {
    auto it = config.find(key);
    if (it != config.end()) {
        return it->second;
    }
    return defaultVal;
}

int main() {
    std::string configData =
        "# アプリケーション設定\n"
        "\n"
        "app_name = MyApplication\n"
        "version = 1.2.3\n"
        "# データベース設定\n"
        "db_host = localhost\n"
        "db_port = 5432\n"
        "  db_name  =  my_database  \n"
        "debug = true\n";

    // 設定を格納するmap
    std::map<std::string, std::string> config;
    std::istringstream stream(configData);
    std::string line;

    while (std::getline(stream, line)) {
        // 前後の空白を除去
        std::string trimmedLine = trim(line);

        // 空行とコメント行をスキップ
        if (trimmedLine.empty() || trimmedLine[0] == '#') {
            continue;
        }

        // '='の位置を探す
        size_t eqPos = trimmedLine.find('=');
        if (eqPos == std::string::npos) {
            continue;  // '='がない行はスキップ
        }

        // キーと値を分割してトリム
        std::string key = trim(trimmedLine.substr(0, eqPos));
        std::string value = trim(trimmedLine.substr(eqPos + 1));

        config[key] = value;
    }

    // 解析結果を表示
    std::cout << "=== 設定ファイルの解析結果 ===" << std::endl;
    for (const auto& pair : config) {
        std::cout << pair.first << " = " << pair.second << std::endl;
    }

    // 値の取得テスト
    std::cout << std::endl;
    std::cout << "=== 値の取得 ===" << std::endl;
    std::cout << "app_name: " << getValue(config, "app_name") << std::endl;
    std::cout << "db_port: " << getValue(config, "db_port") << std::endl;
    // 存在しないキーはデフォルト値を返す
    std::cout << "timeout（未設定）: " << getValue(config, "timeout", "30") << std::endl;

    return 0;
}
```

</details>

## 問題4：ログ解析（応用）

サーバーのアクセスログデータを解析し、統計情報を出力するプログラムを作成してください。

- ログの各行は `[日時] レベル メッセージ` の形式です
- ログレベル（INFO / WARNING / ERROR）ごとの件数を集計します
- ERROR レベルのログのみを抽出して表示します
- `stringstream` を活用して各フィールドを分割してください

```cpp
#include <iostream>
#include <sstream>
#include <string>
#include <vector>
#include <map>

int main() {
    std::string logData =
        "[2026-03-28 10:00:01] INFO サーバー起動完了\n"
        "[2026-03-28 10:05:23] INFO ユーザーログイン: user001\n"
        "[2026-03-28 10:12:45] WARNING ディスク使用率80%超過\n"
        "[2026-03-28 10:15:00] ERROR データベース接続タイムアウト\n"
        "[2026-03-28 10:15:30] INFO 自動リトライ実行\n"
        "[2026-03-28 10:16:00] ERROR データベース接続タイムアウト（2回目）\n"
        "[2026-03-28 10:20:00] INFO データベース接続回復\n"
        "[2026-03-28 10:30:00] WARNING メモリ使用率75%超過\n"
        "[2026-03-28 10:45:00] ERROR ファイル書き込み失敗: /var/log/app.log\n"
        "[2026-03-28 11:00:00] INFO 定期バックアップ開始\n";

    // ここに処理を記述
    return 0;
}
```

**期待される出力：**
```
=== ログ解析レポート ===
総ログ件数: 10

--- レベル別集計 ---
ERROR:   3件
INFO:    5件
WARNING: 2件

--- ERRORログ一覧 ---
[2026-03-28 10:15:00] データベース接続タイムアウト
[2026-03-28 10:16:00] データベース接続タイムアウト（2回目）
[2026-03-28 10:45:00] ファイル書き込み失敗: /var/log/app.log
```

<details>
<summary>ヒント</summary>

- まず `std::getline` で1行ずつ取り出します
- 日時部分は `[` と `]` の位置を `find` で探して抽出します
- `]` の後の部分を `std::istringstream` に入れて、最初のトークンがログレベル、残りがメッセージです
- `std::map<std::string, int>` でレベル別カウントを管理します
- ERRORログは `std::vector` に格納しておき、最後にまとめて表示すると便利です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <sstream>
#include <string>
#include <vector>
#include <map>

// ログエントリを表す構造体
struct LogEntry {
    std::string datetime;
    std::string level;
    std::string message;
};

// ログ行をパースする関数
LogEntry parseLogLine(const std::string& line) {
    LogEntry entry;

    // '[' と ']' の位置を探して日時を抽出
    size_t bracketOpen = line.find('[');
    size_t bracketClose = line.find(']');
    if (bracketOpen != std::string::npos && bracketClose != std::string::npos) {
        entry.datetime = line.substr(bracketOpen + 1, bracketClose - bracketOpen - 1);
    }

    // ']' 以降の部分からレベルとメッセージを抽出
    std::string rest = line.substr(bracketClose + 2);  // "] "の後
    std::istringstream iss(rest);
    iss >> entry.level;  // 最初のトークンがレベル

    // 残りがメッセージ（先頭の空白をスキップ）
    std::getline(iss >> std::ws, entry.message);

    return entry;
}

int main() {
    std::string logData =
        "[2026-03-28 10:00:01] INFO サーバー起動完了\n"
        "[2026-03-28 10:05:23] INFO ユーザーログイン: user001\n"
        "[2026-03-28 10:12:45] WARNING ディスク使用率80%超過\n"
        "[2026-03-28 10:15:00] ERROR データベース接続タイムアウト\n"
        "[2026-03-28 10:15:30] INFO 自動リトライ実行\n"
        "[2026-03-28 10:16:00] ERROR データベース接続タイムアウト（2回目）\n"
        "[2026-03-28 10:20:00] INFO データベース接続回復\n"
        "[2026-03-28 10:30:00] WARNING メモリ使用率75%超過\n"
        "[2026-03-28 10:45:00] ERROR ファイル書き込み失敗: /var/log/app.log\n"
        "[2026-03-28 11:00:00] INFO 定期バックアップ開始\n";

    std::istringstream stream(logData);
    std::string line;

    std::map<std::string, int> levelCount;         // レベル別カウント
    std::vector<LogEntry> errorLogs;               // ERRORログを保持
    int totalCount = 0;

    while (std::getline(stream, line)) {
        if (line.empty()) continue;

        LogEntry entry = parseLogLine(line);
        totalCount++;
        levelCount[entry.level]++;

        // ERRORログを記録
        if (entry.level == "ERROR") {
            errorLogs.push_back(entry);
        }
    }

    // レポート出力
    std::cout << "=== ログ解析レポート ===" << std::endl;
    std::cout << "総ログ件数: " << totalCount << std::endl;
    std::cout << std::endl;

    std::cout << "--- レベル別集計 ---" << std::endl;
    for (const auto& pair : levelCount) {
        std::cout << pair.first << ":   " << pair.second << "件" << std::endl;
    }
    std::cout << std::endl;

    std::cout << "--- ERRORログ一覧 ---" << std::endl;
    for (const auto& entry : errorLogs) {
        std::cout << "[" << entry.datetime << "] "
                  << entry.message << std::endl;
    }

    return 0;
}
```

</details>

## 問題5：ファイルコピー（チャレンジ）

`std::filesystem`（C++17）と `ifstream` / `ofstream` を使って、ファイルコピー機能を持つプログラムを作成してください。

- コピー元ファイルの存在確認を行います
- コピー先に同名ファイルが存在する場合は上書き確認メッセージを表示します
- コピー後にファイルサイズを表示して検証します
- `std::filesystem::path` を使ってファイル名やディレクトリ名を操作します
- 実行時にはまずテスト用のファイルを自動生成してからコピーを行います

```cpp
#include <iostream>
#include <fstream>
#include <string>
#include <filesystem>

namespace fs = std::filesystem;

int main() {
    // ここに処理を記述
    return 0;
}
```

**期待される出力：**
```
=== ファイルコピーツール ===
テスト用ファイルを作成しました: source.txt (45 bytes)

コピー元: source.txt
コピー先: backup_source.txt

コピーを実行します...
コピー完了！

--- 検証 ---
コピー元サイズ: 45 bytes
コピー先サイズ: 45 bytes
結果: ファイルサイズが一致しました
```

<details>
<summary>ヒント</summary>

- `std::filesystem::exists(path)` でファイルの存在確認ができます
- `std::filesystem::file_size(path)` でファイルサイズを取得できます
- `std::filesystem::path` の `.filename()` でファイル名部分、`.stem()` で拡張子なしのファイル名を取得できます
- バイナリモードでコピーする場合は `std::ios::binary` フラグを使います
- `rdbuf()` を使うと効率的にストリーム間でデータをコピーできます：`ofs << ifs.rdbuf();`

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <fstream>
#include <string>
#include <filesystem>

namespace fs = std::filesystem;

// ファイルをコピーする関数（ストリームを使った手動コピー）
bool copyFile(const fs::path& src, const fs::path& dst) {
    // バイナリモードで読み込み
    std::ifstream ifs(src, std::ios::binary);
    if (!ifs.is_open()) {
        std::cerr << "エラー: コピー元を開けません: " << src << std::endl;
        return false;
    }

    // バイナリモードで書き込み
    std::ofstream ofs(dst, std::ios::binary);
    if (!ofs.is_open()) {
        std::cerr << "エラー: コピー先を開けません: " << dst << std::endl;
        return false;
    }

    // rdbuf()で効率的にコピー
    ofs << ifs.rdbuf();

    return ofs.good();
}

int main() {
    std::cout << "=== ファイルコピーツール ===" << std::endl;

    fs::path srcPath = "source.txt";

    // テスト用ファイルを作成
    {
        std::ofstream ofs(srcPath);
        ofs << "これはテスト用のファイルです。" << std::endl;
        ofs << "コピー機能の検証に使います。" << std::endl;
    }

    auto srcSize = fs::file_size(srcPath);
    std::cout << "テスト用ファイルを作成しました: "
              << srcPath.filename().string()
              << " (" << srcSize << " bytes)" << std::endl;
    std::cout << std::endl;

    // コピー先のパスを生成（"backup_" + 元のファイル名）
    fs::path dstPath = "backup_" + srcPath.filename().string();

    std::cout << "コピー元: " << srcPath.filename().string() << std::endl;
    std::cout << "コピー先: " << dstPath.filename().string() << std::endl;
    std::cout << std::endl;

    // コピー先に同名ファイルが存在する場合の確認
    if (fs::exists(dstPath)) {
        std::cout << "警告: コピー先にファイルが既に存在します。上書きします。" << std::endl;
    }

    // コピー実行
    std::cout << "コピーを実行します..." << std::endl;
    if (!copyFile(srcPath, dstPath)) {
        std::cerr << "コピーに失敗しました。" << std::endl;
        return 1;
    }
    std::cout << "コピー完了！" << std::endl;
    std::cout << std::endl;

    // 検証
    std::cout << "--- 検証 ---" << std::endl;
    auto copiedSize = fs::file_size(dstPath);
    std::cout << "コピー元サイズ: " << srcSize << " bytes" << std::endl;
    std::cout << "コピー先サイズ: " << copiedSize << " bytes" << std::endl;

    if (srcSize == copiedSize) {
        std::cout << "結果: ファイルサイズが一致しました" << std::endl;
    } else {
        std::cout << "結果: ファイルサイズが一致しません！" << std::endl;
    }

    return 0;
}
```

**コンパイル方法（C++17が必要）：**
```bash
g++ -std=c++17 -o file_copy file_copy.cpp
```

## 問題6：簡易データベース（チャレンジ）

ファイルベースの簡易データベースシステムを作成してください。レコードの追加・一覧表示・検索・保存・読み込みの機能を実装します。

- レコードは `ID,名前,メールアドレス,部署` の形式でCSVファイルに保存します
- `stringstream` を使ってCSVの読み書きを行います
- IDは自動採番とします
- 検索は名前の部分一致で行います
- プログラム内で一連の操作をデモンストレーションしてください（対話式ではなく自動実行）

```cpp
#include <iostream>
#include <fstream>
#include <sstream>
#include <string>
#include <vector>
#include <algorithm>

int main() {
    // ここに処理を記述
    return 0;
}
```

<details>
<summary>ヒント</summary>

- レコードを表す構造体 `Record` を定義しましょう（`id`, `name`, `email`, `department`）
- データベースをクラスとして設計すると見通しが良くなります
- `add()`, `listAll()`, `search()`, `save()`, `load()` などのメソッドを用意します
- CSVへの書き出しは `ofstream` と `<<` 演算子で行えます
- CSVからの読み込みは `ifstream` と `getline` + `istringstream` で行います
- 検索の部分一致には `std::string::find` を使い、戻り値が `std::string::npos` でなければ一致です
- 表示幅の調整には `<iomanip>` の `std::setw` と `std::left` が便利です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <fstream>
#include <sstream>
#include <string>
#include <vector>
#include <algorithm>
#include <iomanip>

// レコードを表す構造体
struct Record {
    int id;
    std::string name;
    std::string email;
    std::string department;
};

// 簡易データベースクラス
class SimpleDatabase {
private:
    std::vector<Record> records;
    int nextId = 1;

    // レコードを表形式で表示する共通関数
    void printRecords(const std::vector<Record>& recs) const {
        std::cout << "  " << std::left
                  << std::setw(4) << "ID"
                  << std::setw(14) << "名前"
                  << std::setw(27) << "メール"
                  << "部署" << std::endl;
        std::cout << "  -------------------------------------------------------" << std::endl;
        for (const auto& r : recs) {
            std::cout << "  " << std::left
                      << std::setw(4) << r.id
                      << std::setw(14) << r.name
                      << std::setw(27) << r.email
                      << r.department << std::endl;
        }
        std::cout << "  -------------------------------------------------------" << std::endl;
    }

public:
    // レコードを追加
    void add(const std::string& name, const std::string& email, const std::string& dept) {
        Record r{nextId++, name, email, dept};
        records.push_back(r);
        std::cout << "  追加: ID=" << r.id << " " << r.name
                  << " (" << r.email << ") " << r.department << std::endl;
    }

    // 全レコードを表示
    void listAll() const {
        printRecords(records);
        std::cout << "  合計: " << records.size() << "件" << std::endl;
    }

    // 名前で部分一致検索
    void search(const std::string& keyword) const {
        std::vector<Record> results;
        for (const auto& r : records) {
            // 部分一致: findの戻り値がnposでなければ一致
            if (r.name.find(keyword) != std::string::npos) {
                results.push_back(r);
            }
        }
        printRecords(results);
        std::cout << "  検索結果: " << results.size() << "件" << std::endl;
    }

    // CSVファイルに保存
    void save(const std::string& filename) const {
        std::ofstream ofs(filename);
        if (!ofs.is_open()) {
            std::cerr << "  エラー: ファイルを開けません: " << filename << std::endl;
            return;
        }
        // ヘッダー行
        ofs << "id,name,email,department" << std::endl;
        // 各レコードをCSV形式で書き出し
        for (const auto& r : records) {
            ofs << r.id << "," << r.name << "," << r.email << "," << r.department << std::endl;
        }
        std::cout << "  " << records.size() << "件のレコードを保存しました" << std::endl;
    }

    // CSVファイルから読み込み
    void load(const std::string& filename) {
        std::ifstream ifs(filename);
        if (!ifs.is_open()) {
            std::cerr << "  エラー: ファイルを開けません: " << filename << std::endl;
            return;
        }

        records.clear();
        std::string line;
        bool isHeader = true;
        int maxId = 0;

        while (std::getline(ifs, line)) {
            if (line.empty()) continue;

            // ヘッダー行をスキップ
            if (isHeader) {
                isHeader = false;
                continue;
            }

            // CSVをパース
            std::istringstream iss(line);
            std::string field;
            Record r;

            std::getline(iss, field, ',');
            r.id = std::stoi(field);
            std::getline(iss, r.name, ',');
            std::getline(iss, r.email, ',');
            std::getline(iss, r.department);

            records.push_back(r);
            if (r.id > maxId) maxId = r.id;
        }

        // 次のIDを設定（既存の最大ID + 1）
        nextId = maxId + 1;
        std::cout << "  " << records.size() << "件のレコードを読み込みました" << std::endl;
    }
};

int main() {
    std::cout << "=== 簡易データベースシステム ===" << std::endl;
    std::cout << std::endl;

    SimpleDatabase db;
    const std::string dbFile = "database.csv";

    // レコード追加
    std::cout << "[操作] レコード追加" << std::endl;
    db.add("田中太郎", "tanaka@example.com", "開発部");
    db.add("佐藤花子", "sato@example.com", "デザイン部");
    db.add("鈴木一郎", "suzuki@example.com", "開発部");
    db.add("田中美咲", "tanaka.m@example.com", "営業部");
    db.add("高橋健太", "takahashi@example.com", "開発部");
    std::cout << std::endl;

    // 全レコード一覧
    std::cout << "[操作] 全レコード一覧" << std::endl;
    db.listAll();
    std::cout << std::endl;

    // ファイルに保存
    std::cout << "[操作] ファイルに保存: " << dbFile << std::endl;
    db.save(dbFile);
    std::cout << std::endl;

    // 名前で検索
    std::cout << "[操作] 名前で検索: \"田中\"" << std::endl;
    db.search("田中");
    std::cout << std::endl;

    // ファイルから読み込み（保存したファイルを再読み込み）
    SimpleDatabase db2;
    std::cout << "[操作] ファイルから読み込み: " << dbFile << std::endl;
    db2.load(dbFile);

    return 0;
}
```
