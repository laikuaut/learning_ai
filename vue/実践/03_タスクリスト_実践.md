# 実践課題03：タスクリスト ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第1章（Vue.jsの基本）、第2章（リアクティビティ）、第3章（テンプレート構文）
> **課題の種類**: ミニプロジェクト
> **学習目標**: `v-model` による双方向バインディング、`v-if` / `v-show` による条件分岐、`computed` によるフィルタリングを実践する

---

## 完成イメージ

タスクの追加・完了切替・削除・フィルタリングができるアプリです。

```
┌────────────────────────────────────┐
│    ✅ タスクリスト                  │
├────────────────────────────────────┤
│  [新しいタスクを入力...    ] [追加] │
│                                    │
│  [すべて] [未完了] [完了済み]       │
│                                    │
│  ☑ 牛乳を買う          [✗]        │
│  ☐ Vue.jsの復習         [✗]        │
│  ☑ メール返信           [✗]        │
│  ☐ 報告書作成           [✗]        │
│                                    │
│  4件中 2件完了（50%）              │
│  [完了済みを一括削除]              │
└────────────────────────────────────┘
```

---

## 課題の要件

1. テキスト入力 + Enter キーまたは追加ボタンでタスクを追加する
2. チェックボックスでタスクの完了/未完了を切り替える
3. 各タスクに削除ボタンを付ける
4. 「すべて」「未完了」「完了済み」のフィルタを設ける
5. `computed` でフィルタ後のタスク一覧と進捗情報を算出する
6. 「完了済みを一括削除」ボタンを用意する
7. 完了済みタスクには取り消し線を表示する

---

## ステップガイド

<details>
<summary>ステップ1：データ構造を定義する</summary>

```js
const tasks = ref([
  { id: 1, text: "牛乳を買う", done: true },
  { id: 2, text: "Vue.jsの復習", done: false },
])
const newTask = ref("")
const filter = ref("all") // "all" | "active" | "done"
```

</details>

<details>
<summary>ステップ2：フィルタリングを computed で実装する</summary>

```js
const filteredTasks = computed(() => {
  if (filter.value === "active") return tasks.value.filter(t => !t.done)
  if (filter.value === "done") return tasks.value.filter(t => t.done)
  return tasks.value
})

const doneCount = computed(() => tasks.value.filter(t => t.done).length)
const progress = computed(() => {
  if (tasks.value.length === 0) return 0
  return Math.round(doneCount.value / tasks.value.length * 100)
})
```

</details>

<details>
<summary>ステップ3：完了タスクのスタイルを動的に適用する</summary>

`:class` バインディングまたは `:style` バインディングを使います。

```html
<span :style="{ textDecoration: task.done ? 'line-through' : 'none', color: task.done ? '#999' : '#333' }">
  {{ task.text }}
</span>
```

</details>

  <div id="app">
    <h1>✅ タスクリスト</h1>

    <div class="add-form">
      <input v-model="newTask" placeholder="新しいタスクを入力..." @keyup.enter="addTask" />

    const { createApp, ref, computed, watch, nextTick } = Vue

    createApp({
      setup() {
        const saved = localStorage.getItem('vueTasks')
        const tasks = ref(saved ? JSON.parse(saved) : [
          { id: 1, text: "牛乳を買う", done: true },
          { id: 2, text: "Vue.jsの復習", done: false },
          { id: 3, text: "メール返信", done: true },
          { id: 4, text: "報告書作成", done: false },
        ])
        const newTask = ref("")
        const filter = ref("all")
        const editingId = ref(null)

        // localStorage への自動保存
        watch(tasks, (val) => {
          localStorage.setItem('vueTasks', JSON.stringify(val))
        }, { deep: true })

        const addTask = () => {
          if (!newTask.value.trim()) return
          tasks.value.push({ id: Date.now(), text: newTask.value.trim(), done: false })
          newTask.value = ""
        }

        const removeTask = (id) => {
          tasks.value = tasks.value.filter(t => t.id !== id)
        }

        const clearDone = () => {
          tasks.value = tasks.value.filter(t => !t.done)
        }

        const startEdit = (id) => {
          editingId.value = id
        }

        const finishEdit = (task, event) => {
          const text = event.target.value.trim()
          if (text) task.text = text
          editingId.value = null
        }

        const doneCount = computed(() => tasks.value.filter(t => t.done).length)
        const progress = computed(() => {
          if (tasks.value.length === 0) return 0
          return Math.round(doneCount.value / tasks.value.length * 100)
        })

        const filteredTasks = computed(() => {
          if (filter.value === "active") return tasks.value.filter(t => !t.done)
          if (filter.value === "done") return tasks.value.filter(t => t.done)
          return tasks.value
        })

        // フィルタボタンに件数を表示するための computed
        const filterOptions = computed(() => [
          { value: "all", label: "すべて", count: tasks.value.length },
          { value: "active", label: "未完了", count: tasks.value.filter(t => !t.done).length },
          { value: "done", label: "完了済み", count: doneCount.value },
        ])

        return {
          tasks, newTask, filter, editingId,
          addTask, removeTask, clearDone, startEdit, finishEdit,
          filteredTasks, doneCount, progress, filterOptions
        }
      }
    }).mount('#app')
  </script>
</body>
</html>

- `watch` + `localStorage` でデータ永続化
- ダブルクリックによるインライン編集機能
- フィルタボタンに件数を表示（`computed` の活用）
- プログレスバーで視覚的に進捗を表示（`:style` バインディング）
- `v-for` でフィルタボタンを動的生成
