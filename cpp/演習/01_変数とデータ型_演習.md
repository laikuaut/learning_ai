## 問題1：自己紹介を表示しよう（基本）

名前（string）、年齢（int）、身長（double）を変数に格納し、以下のフォーマットで表示するプログラムを作成してください。

### 期待される出力例

```
===== 自己紹介 =====
名前: 田中太郎
年齢: 25歳
身長: 170.5cm
====================
```

<details>
<summary>ヒント</summary>

- `#include <iostream>` と `#include <string>` が必要です
- `std::string` で文字列を扱えます
- `std::cout` と `<<` 演算子で出力します
- `std::endl` で改行できます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <string>

int main() {
    // 変数の宣言と初期化
    std::string name = "田中太郎";
    int age = 25;
    double height = 170.5;

    // フォーマットに沿って出力
    std::cout << "===== 自己紹介 =====" << std::endl;
    std::cout << "名前: " << name << std::endl;
    std::cout << "年齢: " << age << "歳" << std::endl;
    std::cout << "身長: " << height << "cm" << std::endl;
    std::cout << "====================" << std::endl;

    return 0;
}
```

</details>

## 問題2：BMI計算プログラム（基本）

ユーザーから身長（cm）と体重（kg）を入力として受け取り、BMIを計算して表示するプログラムを作成してください。

**BMIの計算式：** BMI = 体重(kg) / (身長(m))^2

小数点以下1桁まで表示してください。

### 期待される出力例

```
身長をcmで入力してください: 170
体重をkgで入力してください: 65
あなたのBMI: 22.5
```

<details>
<summary>ヒント</summary>

- 身長は cm で入力されるので、m に変換する必要があります（÷100）
- `#include <iomanip>` を使い、`std::fixed` と `std::setprecision(1)` で小数点以下の桁数を指定できます
- `std::cin >> 変数名` でキーボード入力を受け取れます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 変数の宣言
    double height_cm, weight;

    // 入力の受け取り
    std::cout << "身長をcmで入力してください: ";
    std::cin >> height_cm;

    std::cout << "体重をkgで入力してください: ";
    std::cin >> weight;

    // cm → m に変換
    double height_m = height_cm / 100.0;

    // BMI計算
    double bmi = weight / (height_m * height_m);

    // 小数点以下1桁で出力
    std::cout << "あなたのBMI: "
              << std::fixed << std::setprecision(1)
              << bmi << std::endl;

    return 0;
}
```

</details>

## 問題3：温度変換プログラム（基本）

摂氏温度（Celsius）を入力として受け取り、華氏温度（Fahrenheit）に変換して表示するプログラムを作成してください。

**変換式：** F = C × 9 / 5 + 32

### 期待される出力例

```
摂氏温度を入力してください: 100
摂氏 100.0°C = 華氏 212.0°F
```

```
摂氏温度を入力してください: 0
摂氏 0.0°C = 華氏 32.0°F
```

<details>
<summary>ヒント</summary>

- 整数同士の割り算では小数部分が切り捨てられます。`9.0 / 5.0` のように小数リテラルを使いましょう
- `std::fixed` と `std::setprecision(1)` で小数点以下1桁に揃えられます

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double celsius;

    std::cout << "摂氏温度を入力してください: ";
    std::cin >> celsius;

    // 摂氏 → 華氏 変換
    // 注意: 9/5 だと整数除算になり 1 になってしまう
    double fahrenheit = celsius * 9.0 / 5.0 + 32.0;

    // 出力
    std::cout << std::fixed << std::setprecision(1);
    std::cout << "摂氏 " << celsius << "°C = "
              << "華氏 " << fahrenheit << "°F" << std::endl;

    return 0;
}
```

</details>

## 問題4：円の面積と円周を計算しよう（応用）

半径を入力として受け取り、円の面積と円周の長さを計算して表示するプログラムを作成してください。

- 円周率は `M_PI`（`<cmath>` ヘッダ）もしくは自分で `3.14159265358979` として定義してください
- `const` を使って円周率を定数として宣言しましょう
- 結果は小数点以下2桁で表示してください

**公式：**
- 面積 = π × r²
- 円周 = 2 × π × r

### 期待される出力例

```
円の半径を入力してください: 5
--- 計算結果 ---
半径: 5.00
面積: 78.54
円周: 31.42
```

<details>
<summary>ヒント</summary>

- `const double PI = 3.14159265358979;` のように定数を定義します
- `r * r` で半径の2乗を計算できます（`<cmath>` の `pow(r, 2)` でもOK）
- `std::setprecision(2)` で小数点以下2桁に指定します

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 円周率を定数として定義
    const double PI = 3.14159265358979;

    double radius;
    std::cout << "円の半径を入力してください: ";
    std::cin >> radius;

    // 面積と円周を計算
    double area = PI * radius * radius;
    double circumference = 2.0 * PI * radius;

    // 小数点以下2桁で出力
    std::cout << std::fixed << std::setprecision(2);
    std::cout << "--- 計算結果 ---" << std::endl;
    std::cout << "半径: " << radius << std::endl;
    std::cout << "面積: " << area << std::endl;
    std::cout << "円周: " << circumference << std::endl;

    return 0;
}
```

</details>

## 問題5：割り勘計算プログラム（応用）

合計金額と人数を入力として受け取り、1人あたりの支払い金額と余りを計算するプログラムを作成してください。

- 1人あたりの金額は切り捨て（整数除算）とします
- 余りは幹事が負担するものとして表示してください

### 期待される出力例

```
合計金額を入力してください: 10000
人数を入力してください: 3
===== 割り勘結果 =====
合計金額: 10000円
人数: 3人
1人あたり: 3333円
余り（幹事負担）: 1円
======================
```

```
合計金額を入力してください: 8000
人数を入力してください: 4
===== 割り勘結果 =====
合計金額: 8000円
人数: 4人
1人あたり: 2000円
余り（幹事負担）: 0円
======================
```

<details>
<summary>ヒント</summary>

- 整数同士の割り算 `/` は商（切り捨て）を返します
- 余りは `%`（剰余演算子）で求められます
- 金額と人数は `int` 型が適切です

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>

int main() {
    int total, people;

    // 入力
    std::cout << "合計金額を入力してください: ";
    std::cin >> total;

    std::cout << "人数を入力してください: ";
    std::cin >> people;

    // 割り勘計算
    // 整数同士の除算なので自動的に切り捨てになる
    int per_person = total / people;

    // 剰余演算子 % で余りを計算
    int remainder = total % people;

    // 結果表示
    std::cout << "===== 割り勘結果 =====" << std::endl;
    std::cout << "合計金額: " << total << "円" << std::endl;
    std::cout << "人数: " << people << "人" << std::endl;
    std::cout << "1人あたり: " << per_person << "円" << std::endl;
    std::cout << "余り（幹事負担）: " << remainder << "円" << std::endl;
    std::cout << "======================" << std::endl;

    return 0;
}
```

</details>

## 問題6：秒を時・分・秒に変換しよう（チャレンジ）

合計秒数（int）を入力として受け取り、「○時間○分○秒」の形式に変換して表示するプログラムを作成してください。

さらに、元の秒数を `double` 型でも入力できるようにし、小数部分は切り捨てて整数秒として処理してください（`static_cast<int>` を使った型変換の練習）。

### 期待される出力例

```
秒数を入力してください（小数可）: 3661.7
--- 変換結果 ---
入力値: 3661.7秒
整数秒: 3661秒
変換: 1時間1分1秒
```

```
秒数を入力してください（小数可）: 7200.0
--- 変換結果 ---
入力値: 7200.0秒
整数秒: 7200秒
変換: 2時間0分0秒
```

```
秒数を入力してください（小数可）: 90.5
--- 変換結果 ---
入力値: 90.5秒
整数秒: 90秒
変換: 0時間1分30秒
```

<details>
<summary>ヒント</summary>

- まず `double` で入力を受け取り、`static_cast<int>(値)` で整数に変換します
- 時間 = 合計秒 / 3600
- 分 = (合計秒 % 3600) / 60
- 秒 = 合計秒 % 60
- `/` と `%` を組み合わせて段階的に計算しましょう

</details>

<details>
<summary>解答例</summary>

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double input_seconds;

    std::cout << "秒数を入力してください（小数可）: ";
    std::cin >> input_seconds;

    // double → int への明示的な型変換（小数部分を切り捨て）
    int total_seconds = static_cast<int>(input_seconds);

    // 時・分・秒に分解
    int hours = total_seconds / 3600;          // 3600秒 = 1時間
    int minutes = (total_seconds % 3600) / 60; // 残りの秒を60で割って分を取得
    int seconds = total_seconds % 60;          // 60で割った余りが秒

    // 結果表示
    std::cout << "--- 変換結果 ---" << std::endl;
    std::cout << std::fixed << std::setprecision(1);
    std::cout << "入力値: " << input_seconds << "秒" << std::endl;
    std::cout << "整数秒: " << total_seconds << "秒" << std::endl;
    std::cout << "変換: " << hours << "時間"
              << minutes << "分"
              << seconds << "秒" << std::endl;

    return 0;
}
```

</details>
