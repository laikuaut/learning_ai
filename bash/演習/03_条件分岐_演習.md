# 第3章 演習：条件分岐

## 演習の目的
Bashの条件分岐（`if` 文、`case` 文、`test` コマンド）を使いこなし、ファイルテスト・文字列比較・数値比較・複合条件を正しく記述できるようになることを目指します。

---

## 基本問題

### 問題1：testコマンドの基本
以下の条件式を `test` コマンド（`[ ]` 記法）で書き、結果を表示してください。

1. 変数 `count` が `10` と等しいか（数値比較）
2. 変数 `name` が空文字列でないか
3. 変数 `str1` と `str2` が同じ文字列か

**期待される出力例：**
```
count(10) は 10 と等しい: true
name(Bash) は空ではない: true
str1(hello) と str2(world) は同じ: false
```

<details>
<summary>ヒント</summary>

- 数値比較: `-eq`（等しい）、`-ne`（等しくない）、`-gt`（より大きい）、`-lt`（より小さい）
- 文字列の空チェック: `-n`（空でない）、`-z`（空である）
- 文字列比較: `=`（等しい）、`!=`（等しくない）
- `[ 条件 ] && echo "true" || echo "false"` で結果を簡潔に表示できます

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# testコマンドの基本的な使い方

count=10
name="Bash"
str1="hello"
str2="world"

# 1. 数値比較: -eq は「等しい」を意味する
[ "$count" -eq 10 ] && result="true" || result="false"
echo "count($count) は 10 と等しい: $result"

# 2. 文字列の空チェック: -n は「空でない」を意味する
[ -n "$name" ] && result="true" || result="false"
echo "name($name) は空ではない: $result"

# 3. 文字列比較: = は文字列が同一かを比較する
# 注意: == も使えるがPOSIX準拠は =
[ "$str1" = "$str2" ] && result="true" || result="false"
echo "str1($str1) と str2($str2) は同じ: $result"
```

</details>

### 問題2：ファイルテスト
以下のファイルテストを行い、結果を表示するスクリプトを作成してください。

対象: `/etc/passwd`, `/tmp`, `/nonexistent`, `/bin/bash`

各対象について以下を確認してください。
- 存在するか（`-e`）
- 通常ファイルか（`-f`）
- ディレクトリか（`-d`）
- 実行可能か（`-x`）

**期待される出力例：**
```
===== /etc/passwd =====
  存在する    : はい
  通常ファイル: はい
  ディレクトリ: いいえ
  実行可能    : いいえ

===== /tmp =====
  存在する    : はい
  通常ファイル: いいえ
  ディレクトリ: はい
  実行可能    : はい
...
```

<details>
<summary>ヒント</summary>

- `-e` → 存在する
- `-f` → 通常ファイルである
- `-d` → ディレクトリである
- `-x` → 実行権限がある
- `-r` → 読み取り権限がある
- `-w` → 書き込み権限がある

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# ファイルテストを行うスクリプト

# テスト対象のファイル・ディレクトリ
targets=("/etc/passwd" "/tmp" "/nonexistent" "/bin/bash")

# 各対象についてテストを実行
for target in "${targets[@]}"; do
    echo "===== $target ====="

    # -e: 存在確認
    [ -e "$target" ] && echo "  存在する    : はい" || echo "  存在する    : いいえ"

    # -f: 通常ファイルか
    [ -f "$target" ] && echo "  通常ファイル: はい" || echo "  通常ファイル: いいえ"

    # -d: ディレクトリか
    [ -d "$target" ] && echo "  ディレクトリ: はい" || echo "  ディレクトリ: いいえ"

    # -x: 実行権限があるか
    [ -x "$target" ] && echo "  実行可能    : はい" || echo "  実行可能    : いいえ"

    echo ""
done
```

</details>

### 問題3：if文の基本
ユーザーから数値を受け取り、以下の判定を行う `if` 文を書いてください。

- 0より大きい → 「正の数です」
- 0より小さい → 「負の数です」
- 0と等しい → 「ゼロです」

**期待される出力例：**
```
数値を入力してください: 42
42 は正の数です。

数値を入力してください: -5
-5 は負の数です。

数値を入力してください: 0
0 はゼロです。
```

<details>
<summary>ヒント</summary>

- `read -p "メッセージ" 変数` でユーザー入力を受け取ります
- `if`-`elif`-`else`-`fi` の構文を使います
- 数値比較には `-gt`（より大きい）、`-lt`（より小さい）、`-eq`（等しい）を使います

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# 数値の正負判定スクリプト

# ユーザーから入力を受け取る
read -p "数値を入力してください: " num

# if-elif-else で条件分岐
# -gt: greater than（より大きい）
# -lt: less than（より小さい）
# -eq: equal（等しい）
if [ "$num" -gt 0 ]; then
    echo "$num は正の数です。"
elif [ "$num" -lt 0 ]; then
    echo "$num は負の数です。"
else
    echo "$num はゼロです。"
fi
```

</details>

### 問題4：文字列比較と[[ ]]の活用
以下の機能を持つスクリプトを作成してください。

ユーザーからファイル名を入力してもらい、拡張子に応じて種類を判定します。

- `.sh` → 「シェルスクリプト」
- `.py` → 「Pythonスクリプト」
- `.js` → 「JavaScriptファイル」
- `.txt` または `.md` → 「テキストファイル」
- それ以外 → 「不明なファイル種別」

**期待される出力例：**
```
ファイル名を入力してください: script.sh
script.sh はシェルスクリプトです。

ファイル名を入力してください: README.md
README.md はテキストファイルです。

ファイル名を入力してください: photo.jpg
photo.jpg は不明なファイル種別です。
```

<details>
<summary>ヒント</summary>

- `[[ ]]` 内では `==` でパターンマッチが使えます（`*.sh` のようなグロブ）
- `||` で「または」の条件を記述できます
- パラメータ展開 `${var##*.}` で拡張子を取り出す方法もあります

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# ファイル種別判定スクリプト
# [[ ]] のパターンマッチ機能を活用

read -p "ファイル名を入力してください: " filename

# [[ ]] 内での == はパターンマッチとして動作する
# * はワイルドカード（任意の文字列にマッチ）
if [[ "$filename" == *.sh ]]; then
    echo "$filename はシェルスクリプトです。"
elif [[ "$filename" == *.py ]]; then
    echo "$filename はPythonスクリプトです。"
elif [[ "$filename" == *.js ]]; then
    echo "$filename はJavaScriptファイルです。"
elif [[ "$filename" == *.txt || "$filename" == *.md ]]; then
    # || で複数条件を「または」で結合
    echo "$filename はテキストファイルです。"
else
    echo "$filename は不明なファイル種別です。"
fi
```

</details>

### 問題5：case文
以下のメニューシステムを `case` 文で実装してください。

```
===== システム管理メニュー =====
1) ディスク使用量を表示
2) メモリ使用量を表示
3) ログインユーザーを表示
4) 現在時刻を表示
q) 終了
```

**期待される出力例：**
```
===== システム管理メニュー =====
1) ディスク使用量を表示
2) メモリ使用量を表示
3) ログインユーザーを表示
4) 現在時刻を表示
q) 終了

選択してください: 1
--- ディスク使用量 ---
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1        50G   20G   28G  42% /
...
```

<details>
<summary>ヒント</summary>

- `case $変数 in パターン) コマンド ;; esac` の構文を使います
- 各パターンは `)` で終わり、処理は `;;` で終わります
- `*` はどのパターンにもマッチしないときのデフォルトケースです
- `|` で複数のパターンをまとめられます（例: `y|Y)`）

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# case文を使ったシステム管理メニュー

echo "===== システム管理メニュー ====="
echo "1) ディスク使用量を表示"
echo "2) メモリ使用量を表示"
echo "3) ログインユーザーを表示"
echo "4) 現在時刻を表示"
echo "q) 終了"
echo ""

read -p "選択してください: " choice

# case文で選択に応じた処理を実行
# 各パターンは ) で終わり、;; で処理を区切る
case "$choice" in
    1)
        echo "--- ディスク使用量 ---"
        df -h
        ;;
    2)
        echo "--- メモリ使用量 ---"
        free -h
        ;;
    3)
        echo "--- ログインユーザー ---"
        who
        ;;
    4)
        echo "--- 現在時刻 ---"
        date '+%Y年%m月%d日 %H:%M:%S'
        ;;
    q|Q)
        # | で複数パターンをまとめる（q または Q）
        echo "終了します。"
        ;;
    *)
        # どのパターンにもマッチしなかった場合
        echo "エラー: 無効な選択です。1〜4 または q を入力してください。"
        ;;
esac
```

</details>

### 問題6：複合条件
以下の条件を全て満たすかチェックするスクリプトを作成してください。

**パスワード強度チェッカー:**
入力されたパスワードが以下の条件を満たすか判定してください。

1. 8文字以上であること
2. 数字を含むこと
3. 大文字のアルファベットを含むこと

**期待される出力例：**
```
パスワードを入力してください: abc
結果: 不合格
  [NG] 8文字以上が必要です（現在: 3文字）
  [NG] 数字を含めてください
  [NG] 大文字を含めてください

パスワードを入力してください: MyPass99
結果: 合格！全ての条件を満たしています。
```

<details>
<summary>ヒント</summary>

- `${#var}` で文字数を取得できます
- `[[ "$var" =~ [0-9] ]]` で正規表現マッチ（数字を含むか）ができます
- `[[ "$var" =~ [A-Z] ]]` で大文字を含むかチェックできます
- 複数の条件チェックにはフラグ変数を使うと便利です

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# パスワード強度チェッカー
# 複合条件と正規表現マッチを活用

read -p "パスワードを入力してください: " password

# 合格フラグ（0=合格、1=不合格）
pass=0

echo ""

# 条件1: 8文字以上
# ${#password} でパスワードの文字数を取得
if [ "${#password}" -lt 8 ]; then
    echo "  [NG] 8文字以上が必要です（現在: ${#password}文字）"
    pass=1
else
    echo "  [OK] 文字数: ${#password}文字"
fi

# 条件2: 数字を含むか
# [[ ]] 内の =~ で正規表現マッチ
# [0-9] は任意の数字1文字にマッチ
if [[ ! "$password" =~ [0-9] ]]; then
    echo "  [NG] 数字を含めてください"
    pass=1
else
    echo "  [OK] 数字を含んでいます"
fi

# 条件3: 大文字のアルファベットを含むか
# [A-Z] は任意の大文字1文字にマッチ
if [[ ! "$password" =~ [A-Z] ]]; then
    echo "  [NG] 大文字を含めてください"
    pass=1
else
    echo "  [OK] 大文字を含んでいます"
fi

echo ""

# 最終判定
if [ "$pass" -eq 0 ]; then
    echo "結果: 合格！全ての条件を満たしています。"
else
    echo "結果: 不合格"
fi
```

</details>

### 問題7：ファイル分類スクリプト
指定されたディレクトリ内のファイルを種類別にカウントし、レポートを出力するスクリプトを作成してください。

**分類カテゴリ:**
- テキスト系: `.txt`, `.md`, `.csv`, `.log`
- スクリプト系: `.sh`, `.py`, `.js`, `.rb`
- 画像系: `.jpg`, `.png`, `.gif`, `.svg`
- その他

**期待される出力例：**
```
ディレクトリを入力してください: /home/user/projects
===== ファイル分類レポート =====
対象ディレクトリ: /home/user/projects

テキスト系    : 5 ファイル
スクリプト系  : 12 ファイル
画像系        : 3 ファイル
その他        : 8 ファイル
--------------------------
合計          : 28 ファイル
===============================
```

<details>
<summary>ヒント</summary>

- `for file in "$dir"/*` でディレクトリ内のファイルをループ処理できます
- `case` 文でファイル拡張子による分類ができます
- パターンの `|` で複数拡張子をまとめられます（例: `*.txt|*.md`）
- カウンタ変数を `(( count++ ))` でインクリメントします

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# ファイル分類スクリプト
# 指定ディレクトリ内のファイルを拡張子で分類しカウントする

read -p "ディレクトリを入力してください: " target_dir

# ディレクトリの存在チェック
if [ ! -d "$target_dir" ]; then
    echo "エラー: '$target_dir' はディレクトリではありません。"
    exit 1
fi

# カウンタの初期化
text_count=0
script_count=0
image_count=0
other_count=0

# ディレクトリ内のファイルをループ処理
for file in "$target_dir"/*; do
    # 通常ファイルのみを対象にする（ディレクトリは除外）
    [ -f "$file" ] || continue

    # case文でファイル拡張子による分類
    # | で複数のパターンを「または」で結合
    case "$file" in
        *.txt|*.md|*.csv|*.log)
            (( text_count++ ))
            ;;
        *.sh|*.py|*.js|*.rb)
            (( script_count++ ))
            ;;
        *.jpg|*.png|*.gif|*.svg)
            (( image_count++ ))
            ;;
        *)
            (( other_count++ ))
            ;;
    esac
done

# 合計を計算
total=$(( text_count + script_count + image_count + other_count ))

# レポート出力
echo "===== ファイル分類レポート ====="
echo "対象ディレクトリ: $target_dir"
echo ""
echo "テキスト系    : $text_count ファイル"
echo "スクリプト系  : $script_count ファイル"
echo "画像系        : $image_count ファイル"
echo "その他        : $other_count ファイル"
echo "--------------------------"
echo "合計          : $total ファイル"
echo "==============================="
```

</details>

### 問題8：成績評価システム
科目名と点数を受け取り、成績を判定するスクリプトを作成してください。

**評価基準:**
- 90〜100点: 秀（S）
- 80〜89点: 優（A）
- 70〜79点: 良（B）
- 60〜69点: 可（C）
- 0〜59点: 不可（F）
- 範囲外: エラー

**追加要件:**
- 入力値が数値かどうかのバリデーション
- 複数科目を連続入力できる（空入力で終了）
- 最後に全科目の平均点と総合評価を表示

**期待される出力例：**
```
===== 成績評価システム =====
科目名と点数をスペース区切りで入力してください。
（空入力で終了）

> 数学 85
  数学: 85点 → 優（A）

> 英語 92
  英語: 92点 → 秀（S）

> 国語 68
  国語: 68点 → 可（C）

>

===== 成績表 =====
  数学: 85点 [A]
  英語: 92点 [S]
  国語: 68点 [C]
------------------
  科目数  : 3
  合計点  : 245
  平均点  : 81
  総合評価: 優（A）
==================
```

<details>
<summary>ヒント</summary>

- `read subject score` でスペース区切りの2つの値を読み取れます
- `[[ "$score" =~ ^[0-9]+$ ]]` で数値かどうかを確認できます
- 配列を使って科目と点数を記録できます
- 平均は `$(( total / count ))` で計算できます（整数除算）

</details>

<details>
<summary>解答例</summary>

```bash
#!/bin/bash
# 成績評価システム
# 科目名と点数を入力し、成績を判定・集計する

# 成績判定関数
# 引数: 点数
# 出力: 評価ランク
get_grade() {
    local score=$1
    if [ "$score" -ge 90 ]; then
        echo "秀（S）"
    elif [ "$score" -ge 80 ]; then
        echo "優（A）"
    elif [ "$score" -ge 70 ]; then
        echo "良（B）"
    elif [ "$score" -ge 60 ]; then
        echo "可（C）"
    else
        echo "不可（F）"
    fi
}

# ランク記号を返す関数
get_rank() {
    local score=$1
    if [ "$score" -ge 90 ]; then echo "S"
    elif [ "$score" -ge 80 ]; then echo "A"
    elif [ "$score" -ge 70 ]; then echo "B"
    elif [ "$score" -ge 60 ]; then echo "C"
    else echo "F"
    fi
}

echo "===== 成績評価システム ====="
echo "科目名と点数をスペース区切りで入力してください。"
echo "（空入力で終了）"
echo ""

# 配列で科目名・点数・ランクを記録
subjects=()
scores=()
ranks=()
total=0
count=0

# 入力ループ
while true; do
    read -p "> " subject score

    # 空入力で終了
    if [ -z "$subject" ]; then
        break
    fi

    # 点数が未入力の場合
    if [ -z "$score" ]; then
        echo "  エラー: 点数を入力してください（例: 数学 85）"
        continue
    fi

    # 数値バリデーション: 正規表現で数字のみかチェック
    if [[ ! "$score" =~ ^[0-9]+$ ]]; then
        echo "  エラー: 点数は数値で入力してください。"
        continue
    fi

    # 範囲チェック
    if [ "$score" -lt 0 ] || [ "$score" -gt 100 ]; then
        echo "  エラー: 点数は0〜100の範囲で入力してください。"
        continue
    fi

    # 評価を取得
    grade=$(get_grade "$score")
    rank=$(get_rank "$score")

    echo "  $subject: ${score}点 → $grade"
    echo ""

    # 配列に記録
    subjects+=("$subject")
    scores+=("$score")
    ranks+=("$rank")
    total=$(( total + score ))
    (( count++ ))
done

# 結果がない場合
if [ "$count" -eq 0 ]; then
    echo "科目が入力されませんでした。"
    exit 0
fi

# 成績表の出力
echo ""
echo "===== 成績表 ====="

for i in $(seq 0 $(( count - 1 ))); do
    echo "  ${subjects[$i]}: ${scores[$i]}点 [${ranks[$i]}]"
done

# 平均点を計算（整数除算）
average=$(( total / count ))
avg_grade=$(get_grade "$average")

echo "------------------"
echo "  科目数  : $count"
echo "  合計点  : $total"
echo "  平均点  : $average"
echo "  総合評価: $avg_grade"
echo "=================="
```

</details>
