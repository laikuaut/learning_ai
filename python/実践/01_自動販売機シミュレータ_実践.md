## 課題の要件

1. 3種類の飲み物（名前と価格）を変数で管理する
2. ユーザーから「投入金額」と「選択番号」を入力で受け取る
3. 選択番号が 1〜3 以外の場合、エラーメッセージを表示する
4. 投入金額が足りない場合、不足金額を表示する
5. 購入成功時、商品名とおつりを表示する

---

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```python
# 自動販売機シミュレータ
# 学べる内容：変数、条件分岐、入出力、型変換

# --- 商品データ ---
drink1_name = "お茶"
drink1_price = 120

drink2_name = "コーヒー"
drink2_price = 150

drink3_name = "ジュース"
drink3_price = 160

# --- メニュー表示 ---
print("===== 自動販売機 =====")
print(f"1. {drink1_name}      {drink1_price}円")
print(f"2. {drink2_name}  {drink2_price}円")
print(f"3. {drink3_name}  {drink3_price}円")
print()

# --- 入力 ---
money = int(input("投入金額を入力してください: "))
choice = int(input("番号を選んでください (1-3): "))

# --- 選択した商品を決定 ---
if choice == 1:
    selected_name = drink1_name
    selected_price = drink1_price
elif choice == 2:
    selected_name = drink2_name
    selected_price = drink2_price
elif choice == 3:
    selected_name = drink3_name
    selected_price = drink3_price
else:
    print("無効な番号です。1〜3で選んでください。")
    selected_name = ""
    selected_price = 0

# --- 購入判定 ---
if selected_name != "":
    if money >= selected_price:
        change = money - selected_price
        print(f"\n{selected_name} を購入しました！")
        print(f"おつり: {change}円")
    else:
        shortage = selected_price - money
        print(f"\nお金が足りません。あと {shortage}円 必要です。")
```

</details>

<details>
<summary>解答例（改良版 ─ 第4章の知識を先取り）</summary>

リストや辞書を使うと、商品の追加が簡単になります。
第4章を学んだ後にもう一度挑戦してみてください。

```python
# 自動販売機シミュレータ（改良版）
# 辞書のリストで商品を管理するバージョン

drinks = [
    {"name": "お茶",     "price": 120},
    {"name": "コーヒー", "price": 150},
    {"name": "ジュース", "price": 160},
]

print("===== 自動販売機 =====")
for i, drink in enumerate(drinks, 1):
    print(f"{i}. {drink['name']:<6} {drink['price']}円")
print()

money = int(input("投入金額を入力してください: "))
choice = int(input(f"番号を選んでください (1-{len(drinks)}): "))

if 1 <= choice <= len(drinks):
    selected = drinks[choice - 1]
    if money >= selected["price"]:
        print(f"\n{selected['name']} を購入しました！")
        print(f"おつり: {money - selected['price']}円")
    else:
        print(f"\nお金が足りません。あと {selected['price'] - money}円 必要です。")
else:
    print("無効な番号です。")
```

**初心者向けとの違い:**
- 商品データを辞書のリストで一元管理 → 商品追加が1行で済む
- `enumerate()` でループ表示 → 商品数が変わってもコード修正不要
- 変数の重複がなくなり、バグが生まれにくい

</details>
