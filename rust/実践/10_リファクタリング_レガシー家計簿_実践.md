## 課題の説明

以下は「家計簿アプリケーション」の全コードです。**正しく動作します**が、以下の問題を抱えています。

- すべての処理が `main()` 関数に詰まっている（300行超の巨大関数）
- 金額がマジックナンバー（`1` = 収入、`2` = 支出）で管理されている
- エラー処理が `expect()` で全て異常終了してしまう
- データが `Vec<(String, String, i64, i32)>` のタプルで管理されている（何が何だか分からない）
- 同じ入力処理のパターンがコピー&ペーストされている

**このコードを段階的にリファクタリング（構造改善）してください。**

### ゴール

1. **ステップ1**: データをタプルから構造体に変換する
2. **ステップ2**: 処理を関数に分割する
3. **ステップ3**: 列挙型でカテゴリと種別を型安全にする
4. **ステップ4**: `Result` 型でエラーハンドリングを改善する

## リファクタリング対象のコード

以下を `ledger_legacy.rs` として保存してください。

```rust
// 家計簿アプリケーション（レガシー版）
// 動作するが保守性が低いコード。リファクタリングしてください。
// 実行方法：rustc ledger_legacy.rs && ./ledger_legacy

use std::io;
use std::io::Write;

fn main() {
    println!("===== 家計簿 =====");

    // (日付, カテゴリ, 金額, 種別:1=収入,2=支出)
    let mut records: Vec<(String, String, i64, i32)> = Vec::new();

    loop {
        println!("\n1.追加 2.一覧 3.集計 4.検索 5.削除 6.終了");
        print!("操作: ");
        io::stdout().flush().unwrap();
        let mut c = String::new();
        io::stdin().read_line(&mut c).expect("入力エラー");
        let c = c.trim();

        if c == "1" {
            // --- 追加 ---
            print!("種別 (1:収入 2:支出): ");
            io::stdout().flush().unwrap();
            let mut t = String::new();
            io::stdin().read_line(&mut t).expect("入力エラー");
            let t: i32 = t.trim().parse().expect("数値エラー");

            if t != 1 && t != 2 {
                println!("無効な種別です。");
                continue;
            }

            print!("日付 (例: 2024-03-15): ");
            io::stdout().flush().unwrap();
            let mut d = String::new();
            io::stdin().read_line(&mut d).expect("入力エラー");
            let d = d.trim().to_string();

            print!("カテゴリ: ");
            io::stdout().flush().unwrap();
            let mut cat = String::new();
            io::stdin().read_line(&mut cat).expect("入力エラー");
            let cat = cat.trim().to_string();

            print!("金額: ");
            io::stdout().flush().unwrap();
            let mut a = String::new();
            io::stdin().read_line(&mut a).expect("入力エラー");
            let a: i64 = a.trim().parse().expect("数値エラー");

            if a <= 0 {
                println!("金額は正の値を入力してください。");
                continue;
            }

            records.push((d.clone(), cat.clone(), a, t));

            if t == 1 {
                println!("収入「{}」{}円 を追加しました。", cat, a);
            } else {
                println!("支出「{}」{}円 を追加しました。", cat, a);
            }
        } else if c == "2" {
            // --- 一覧 ---
            if records.is_empty() {
                println!("記録がありません。");
                continue;
            }
            println!("\n--- 収支一覧 ---");
            let mut i = 0;
            while i < records.len() {
                let r = &records[i];
                let kind = if r.3 == 1 { "収入" } else { "支出" };
                let sign = if r.3 == 1 { "+" } else { "-" };
                println!(
                    " {}. {} {:<10} {:<8} {}{:>8}円",
                    i + 1,
                    r.0,
                    r.1,
                    kind,
                    sign,
                    r.2
                );
                i += 1;
            }

            // 合計計算
            let mut total_income: i64 = 0;
            let mut total_expense: i64 = 0;
            let mut j = 0;
            while j < records.len() {
                if records[j].3 == 1 {
                    total_income += records[j].2;
                } else {
                    total_expense += records[j].2;
                }
                j += 1;
            }
            println!("\n  収入合計: +{}円", total_income);
            println!("  支出合計: -{}円", total_expense);
            println!("  差引残高:  {}円", total_income - total_expense);
        } else if c == "3" {
            // --- カテゴリ別集計 ---
            if records.is_empty() {
                println!("記録がありません。");
                continue;
            }
            println!("\n--- カテゴリ別集計 ---");

            // カテゴリ一覧を収集
            let mut categories: Vec<String> = Vec::new();
            for r in &records {
                let mut found = false;
                for cat in &categories {
                    if cat == &r.1 {
                        found = true;
                        break;
                    }
                }
                if !found {
                    categories.push(r.1.clone());
                }
            }

            // カテゴリごとに集計
            for cat in &categories {
                let mut cat_total: i64 = 0;
                let mut cat_count: i32 = 0;
                for r in &records {
                    if &r.1 == cat {
                        if r.3 == 1 {
                            cat_total += r.2;
                        } else {
                            cat_total -= r.2;
                        }
                        cat_count += 1;
                    }
                }
                println!("  {:<10}: {:>8}円 ({}件)", cat, cat_total, cat_count);
            }
        } else if c == "4" {
            // --- 検索 ---
            print!("検索キーワード: ");
            io::stdout().flush().unwrap();
            let mut kw = String::new();
            io::stdin().read_line(&mut kw).expect("入力エラー");
            let kw = kw.trim();

            let mut found = false;
            for (i, r) in records.iter().enumerate() {
                if r.0.contains(kw) || r.1.contains(kw) {
                    let kind = if r.3 == 1 { "収入" } else { "支出" };
                    println!(
                        " {}. {} {:<10} {} {:>8}円",
                        i + 1,
                        r.0,
                        r.1,
                        kind,
                        r.2
                    );
                    found = true;
                }
            }
            if !found {
                println!("見つかりませんでした。");
            }
        } else if c == "5" {
            // --- 削除 ---
            print!("削除する番号: ");
            io::stdout().flush().unwrap();
            let mut n = String::new();
            io::stdin().read_line(&mut n).expect("入力エラー");
            let n: usize = n.trim().parse().expect("数値エラー");

            if n < 1 || n > records.len() {
                println!("無効な番号です。");
                continue;
            }

            let removed = records.remove(n - 1);
            println!("{}の「{}」を削除しました。", removed.0, removed.1);
        } else if c == "6" {
            println!("終了します。");
            break;
        } else {
            println!("無効な操作です。");
        }
    }
}
```

## リファクタリングのステップガイド

<details>
<summary>ステップ1：タプルを構造体に変換する</summary>

`(String, String, i64, i32)` の各フィールドが何を意味するか一目では分かりません。構造体に名前を付けましょう。

```rust
struct Record {
    date: String,
    category: String,
    amount: i64,
    kind: i32,  // まだマジックナンバー（次のステップで改善）
}
```

**変更点:**
- `records: Vec<(String, String, i64, i32)>` → `records: Vec<Record>`
- `r.0` → `r.date`、`r.1` → `r.category`、`r.2` → `r.amount`、`r.3` → `r.kind`

</details>

<details>
<summary>ステップ2：処理を関数に分割する</summary>

`main()` 内の各 `if/else` ブロックを関数に切り出します。

```rust
fn read_input(prompt: &str) -> String { /* ... */ }
fn add_record(records: &mut Vec<Record>) { /* ... */ }
fn show_all(records: &[Record]) { /* ... */ }
fn show_summary(records: &[Record]) { /* ... */ }
fn search(records: &[Record], keyword: &str) { /* ... */ }
fn delete_record(records: &mut Vec<Record>, index: usize) { /* ... */ }
```

`main()` は `match` 式でメニュー操作を振り分けるだけの薄い関数になります。

</details>

<details>
<summary>ステップ3：列挙型でマジックナンバーを排除する</summary>

`1` = 収入、`2` = 支出というマジックナンバーを列挙型に置き換えます。

```rust
enum RecordKind {
    Income,
    Expense,
}

impl RecordKind {
    fn label(&self) -> &'static str {
        match self {
            RecordKind::Income => "収入",
            RecordKind::Expense => "支出",
        }
    }

    fn sign(&self) -> &'static str {
        match self {
            RecordKind::Income => "+",
            RecordKind::Expense => "-",
        }
    }
}
```

</details>

<details>
<summary>ステップ4：Result型でエラーハンドリングを改善する</summary>

`expect()` を使うとエラー時にパニックしてプログラムが落ちます。`Result` 型で回復可能なエラーにします。

```rust
fn read_i64(prompt: &str) -> Result<i64, String> {
    let input = read_input(prompt);
    input.parse::<i64>().map_err(|_| "数値を入力してください。".to_string())
}
```

</details>

## 解答例

<details>
<summary>解答例（段階的改善の最終形）</summary>

```rust
// 家計簿アプリケーション（リファクタリング後）
// 構造体・列挙型・関数分割・エラーハンドリングを適用
// 実行方法：rustc ledger_refactored.rs && ./ledger_refactored

use std::collections::HashMap;
use std::io;
use std::io::Write;

// ==================== データモデル ====================

/// 収支の種別を表す列挙型
enum RecordKind {
    Income,
    Expense,
}

impl RecordKind {
    fn label(&self) -> &'static str {
        match self {
            RecordKind::Income => "収入",
            RecordKind::Expense => "支出",
        }
    }

    fn sign(&self) -> &'static str {
        match self {
            RecordKind::Income => "+",
            RecordKind::Expense => "-",
        }
    }

    fn from_input(input: &str) -> Option<RecordKind> {
        match input {
            "1" => Some(RecordKind::Income),
            "2" => Some(RecordKind::Expense),
            _ => None,
        }
    }
}

/// 収支の1件を表す構造体
struct Record {
    date: String,
    category: String,
    amount: i64,
    kind: RecordKind,
}

impl Record {
    fn new(date: String, category: String, amount: i64, kind: RecordKind) -> Record {
        Record { date, category, amount, kind }
    }

    /// 符号付き金額を返します（収入は正、支出は負）
    fn signed_amount(&self) -> i64 {
        match self.kind {
            RecordKind::Income => self.amount,
            RecordKind::Expense => -self.amount,
        }
    }

    fn display(&self, index: usize) {
        println!(
            " {}. {} {:<10} {:<4} {}{:>8}円",
            index,
            self.date,
            self.category,
            self.kind.label(),
            self.kind.sign(),
            self.amount
        );
    }
}

// ==================== 入力ヘルパー ====================

fn read_input(prompt: &str) -> String {
    print!("{}", prompt);
    io::stdout().flush().unwrap();
    let mut input = String::new();
    io::stdin().read_line(&mut input).expect("入力エラー");
    input.trim().to_string()
}

fn read_positive_i64(prompt: &str) -> Result<i64, String> {
    let input = read_input(prompt);
    let value: i64 = input
        .parse()
        .map_err(|_| "数値を入力してください。".to_string())?;
    if value <= 0 {
        Err("正の値を入力してください。".to_string())
    } else {
        Ok(value)
    }
}

fn read_usize(prompt: &str) -> Result<usize, String> {
    let input = read_input(prompt);
    input
        .parse()
        .map_err(|_| "数値を入力してください。".to_string())
}

// ==================== 操作関数 ====================

fn add_record(records: &mut Vec<Record>) {
    // 種別
    let kind_input = read_input("種別 (1:収入 2:支出): ");
    let kind = match RecordKind::from_input(&kind_input) {
        Some(k) => k,
        None => {
            println!("無効な種別です。");
            return;
        }
    };

    // 日付
    let date = read_input("日付 (例: 2024-03-15): ");

    // カテゴリ
    let category = read_input("カテゴリ: ");

    // 金額
    let amount = match read_positive_i64("金額: ") {
        Ok(a) => a,
        Err(msg) => {
            println!("{}", msg);
            return;
        }
    };

    println!("{}「{}」{}円 を追加しました。", kind.label(), category, amount);
    records.push(Record::new(date, category, amount, kind));
}

fn show_all(records: &[Record]) {
    if records.is_empty() {
        println!("記録がありません。");
        return;
    }

    println!("\n--- 収支一覧 ---");
    for (i, record) in records.iter().enumerate() {
        record.display(i + 1);
    }

    // 合計計算
    let total_income: i64 = records
        .iter()
        .filter(|r| matches!(r.kind, RecordKind::Income))
        .map(|r| r.amount)
        .sum();

    let total_expense: i64 = records
        .iter()
        .filter(|r| matches!(r.kind, RecordKind::Expense))
        .map(|r| r.amount)
        .sum();

    println!("\n  収入合計: +{}円", total_income);
    println!("  支出合計: -{}円", total_expense);
    println!("  差引残高:  {}円", total_income - total_expense);
}

fn show_summary(records: &[Record]) {
    if records.is_empty() {
        println!("記録がありません。");
        return;
    }

    println!("\n--- カテゴリ別集計 ---");

    // HashMapでカテゴリごとに集計
    let mut summary: HashMap<String, (i64, u32)> = HashMap::new();
    for record in records {
        let entry = summary
            .entry(record.category.clone())
            .or_insert((0, 0));
        entry.0 += record.signed_amount();
        entry.1 += 1;
    }

    // ソートして表示
    let mut sorted: Vec<(&String, &(i64, u32))> = summary.iter().collect();
    sorted.sort_by(|a, b| b.1 .0.abs().cmp(&a.1 .0.abs()));

    for (category, (total, count)) in sorted {
        println!("  {:<10}: {:>8}円 ({}件)", category, total, count);
    }
}

fn search(records: &[Record], keyword: &str) {
    let results: Vec<(usize, &Record)> = records
        .iter()
        .enumerate()
        .filter(|(_, r)| r.date.contains(keyword) || r.category.contains(keyword))
        .collect();

    if results.is_empty() {
        println!("見つかりませんでした。");
    } else {
        println!("\n--- 検索結果 ---");
        for (i, record) in results {
            record.display(i + 1);
        }
    }
}

fn delete_record(records: &mut Vec<Record>, num: usize) {
    if num < 1 || num > records.len() {
        println!("無効な番号です。");
        return;
    }
    let removed = records.remove(num - 1);
    println!("{}の「{}」を削除しました。", removed.date, removed.category);
}

// ==================== メイン ====================

fn main() {
    println!("===== 家計簿 =====");
    let mut records: Vec<Record> = Vec::new();

    loop {
        println!("\n1.追加 2.一覧 3.集計 4.検索 5.削除 6.終了");
        let choice = read_input("操作: ");

        match choice.as_str() {
            "1" => add_record(&mut records),
            "2" => show_all(&records),
            "3" => show_summary(&records),
            "4" => {
                let keyword = read_input("検索キーワード: ");
                search(&records, &keyword);
            }
            "5" => {
                match read_usize("削除する番号: ") {
                    Ok(num) => delete_record(&mut records, num),
                    Err(msg) => println!("{}", msg),
                }
            }
            "6" => {
                println!("終了します。");
                break;
            }
            _ => println!("無効な操作です。1〜6で選んでください。"),
        }
    }
}
```

<details>
<summary>レガシー版と改良版の比較</summary>

| 観点 | レガシー版 | 改良版 |
|------|-----------|--------|
| `main()` の行数 | 約170行 | 約20行 |
| データ型 | `(String, String, i64, i32)` タプル | `Record` 構造体 + `RecordKind` 列挙型 |
| 種別の管理 | マジックナンバー `1`, `2` | `RecordKind::Income`, `RecordKind::Expense` |
| エラー処理 | `expect()` でパニック | `Result` で回復 |
| 入力処理 | 毎回コピペ | `read_input()`, `read_positive_i64()` に共通化 |
| 集計処理 | `while` ループ + 手動カウント | `HashMap` + イテレータ |
| カテゴリ収集 | 手動の重複チェックループ | `HashMap::entry()` で自動管理 |

**改善のポイント:**
- **型安全性**: マジックナンバーを列挙型に置き換え → コンパイラが型チェック
- **関心の分離**: 入力・表示・データ操作を関数に分割 → 各関数が1つの責務に集中
- **DRY原則**: 入力処理の共通化 → コードの重複を排除
- **エラー耐性**: `Result` 型 → 不正入力でプログラムが落ちない
- **Rustらしさ**: イテレータ、`match`、`HashMap` の活用

</details>
