### 問題1：配列の最大値・最小値を求める

int型の配列を用意し、その中の最大値と最小値を求めて表示するプログラムを作成してください。配列の全要素も表示してください。

**期待される出力例：**
```
=== 配列の最大値・最小値 ===
配列の要素：[34, 12, 78, 5, 91, 23, 67, 45]
要素数：8
最大値：91
最小値：5
========================
```

<details><summary>ヒント</summary>

- 最大値・最小値用の変数を配列の最初の要素で初期化します
- for文で配列の全要素を走査し、より大きい（小さい）値が見つかったら更新します
- `Arrays.toString(配列)` で配列を `[要素1, 要素2, ...]` 形式で表示できます

</details>

<details><summary>解答例</summary>

```java
import java.util.Arrays;

public class ArrayMinMax {
    public static void main(String[] args) {
        // 配列の宣言と初期化
        int[] numbers = {34, 12, 78, 5, 91, 23, 67, 45};

        System.out.println("=== 配列の最大値・最小値 ===");
        System.out.println("配列の要素：" + Arrays.toString(numbers));
        System.out.println("要素数：" + numbers.length);

        // 最大値・最小値を配列の最初の要素で初期化
        int max = numbers[0];
        int min = numbers[0];

        // 2番目の要素から走査して比較
        for (int i = 1; i < numbers.length; i++) {
            if (numbers[i] > max) {
                max = numbers[i]; // より大きい値で更新
            }
            if (numbers[i] < min) {
                min = numbers[i]; // より小さい値で更新
            }
        }

        System.out.println("最大値：" + max);
        System.out.println("最小値：" + min);
        System.out.println("========================");
    }
}
```

</details>

### 問題2：配列の逆順表示

int型の配列を用意し、元の配列と逆順にした配列を表示するプログラムを作成してください。新しい配列を作成して逆順にコピーしてください。

**期待される出力例：**
```
=== 配列の逆順 ===
元の配列：  [10, 20, 30, 40, 50, 60, 70]
逆順の配列：[70, 60, 50, 40, 30, 20, 10]
===================
```

<details><summary>ヒント</summary>

- 元の配列と同じサイズの新しい配列を作成します
- 元の配列の末尾から順に、新しい配列の先頭にコピーします
- `reversed[i] = original[original.length - 1 - i]` で対応付けできます

</details>

<details><summary>解答例</summary>

```java
import java.util.Arrays;

public class ArrayReverse {
    public static void main(String[] args) {
        int[] original = {10, 20, 30, 40, 50, 60, 70};

        // 同じサイズの新しい配列を作成
        int[] reversed = new int[original.length];

        // 逆順にコピー
        for (int i = 0; i < original.length; i++) {
            // 元の配列の末尾から取得して新しい配列に格納
            reversed[i] = original[original.length - 1 - i];
        }

        System.out.println("=== 配列の逆順 ===");
        System.out.println("元の配列：  " + Arrays.toString(original));
        System.out.println("逆順の配列：" + Arrays.toString(reversed));
        System.out.println("===================");
    }
}
```

</details>

### 問題3：配列の合計と平均

double型の配列を用意し、合計値と平均値を計算して表示するプログラムを作成してください。拡張for文（for-each文）を使用してください。

**期待される出力例：**
```
=== 配列の合計と平均 ===
データ：[85.5, 72.0, 91.3, 68.7, 95.0, 77.8, 83.2]
要素数：7
合計：573.50
平均：81.93
========================
```

<details><summary>ヒント</summary>

- 拡張for文は `for (型 変数 : 配列) { ... }` の構文です
- 合計を要素数で割ると平均が求められます
- `String.format("%.2f", 値)` で小数第2位まで表示できます

</details>

<details><summary>解答例</summary>

```java
public class ArraySumAverage {
    public static void main(String[] args) {
        double[] scores = {85.5, 72.0, 91.3, 68.7, 95.0, 77.8, 83.2};

        // 配列を文字列として表示するためのフォーマット
        StringBuilder sb = new StringBuilder("[");
        for (int i = 0; i < scores.length; i++) {
            if (i > 0) sb.append(", ");
            sb.append(scores[i]);
        }
        sb.append("]");

        System.out.println("=== 配列の合計と平均 ===");
        System.out.println("データ：" + sb);
        System.out.println("要素数：" + scores.length);

        // 拡張for文（for-each文）で合計を計算
        double sum = 0;
        for (double score : scores) {
            sum += score;
        }

        // 平均を計算
        double average = sum / scores.length;

        System.out.println("合計：" + String.format("%.2f", sum));
        System.out.println("平均：" + String.format("%.2f", average));
        System.out.println("========================");
    }
}
```

</details>

### 問題4：学生成績管理プログラム

ArrayListを使って学生の成績を管理するプログラムを作成してください。以下の機能を実装してください。
- 学生の点数をリストに追加
- 全学生の成績一覧表示
- 最高点・最低点・平均点の表示
- 合格者数（60点以上）のカウント

**期待される出力例：**
```
===== 学生成績管理 =====
--- 成績一覧 ---
学生1：85点
学生2：72点
学生3：91点
学生4：58点
学生5：67点
学生6：95点
学生7：43点
学生8：78点

--- 統計情報 ---
人数：8人
最高点：95点
最低点：43点
平均点：73.63点
合格者数：6人（合格率：75.0%）
========================
```

<details><summary>ヒント</summary>

- `ArrayList<Integer>` でint型のリストを作成します
- `Collections.max()` と `Collections.min()` で最大値・最小値を取得できます
- リストのサイズは `.size()` メソッドで取得します

</details>

<details><summary>解答例</summary>

```java
import java.util.ArrayList;
import java.util.Collections;

public class StudentScoreManager {
    public static void main(String[] args) {
        // ArrayListで学生の点数を管理
        ArrayList<Integer> scores = new ArrayList<>();

        // 学生の点数を追加
        scores.add(85);
        scores.add(72);
        scores.add(91);
        scores.add(58);
        scores.add(67);
        scores.add(95);
        scores.add(43);
        scores.add(78);

        System.out.println("===== 学生成績管理 =====");

        // 成績一覧表示
        System.out.println("--- 成績一覧 ---");
        for (int i = 0; i < scores.size(); i++) {
            System.out.println("学生" + (i + 1) + "：" + scores.get(i) + "点");
        }

        // 統計情報の計算
        int max = Collections.max(scores); // 最高点
        int min = Collections.min(scores); // 最低点

        // 合計と合格者数の計算
        int sum = 0;
        int passCount = 0;
        for (int score : scores) {
            sum += score;
            if (score >= 60) {
                passCount++; // 60点以上で合格
            }
        }

        double average = (double) sum / scores.size();
        double passRate = (double) passCount / scores.size() * 100;

        // 統計情報の表示
        System.out.println();
        System.out.println("--- 統計情報 ---");
        System.out.println("人数：" + scores.size() + "人");
        System.out.println("最高点：" + max + "点");
        System.out.println("最低点：" + min + "点");
        System.out.printf("平均点：%.2f点%n", average);
        System.out.printf("合格者数：%d人（合格率：%.1f%%）%n", passCount, passRate);
        System.out.println("========================");
    }
}
```

</details>

### 問題5：単語出現頻度カウンター

HashMapを使って、文章内の各単語の出現回数をカウントして表示するプログラムを作成してください。

**期待される出力例：**
```
===== 単語頻度カウンター =====
文章：java is fun and java is powerful and java is everywhere

--- 単語の出現回数 ---
java → 3回
is → 3回
fun → 1回
and → 2回
powerful → 1回
everywhere → 1回

総単語数：10
異なる単語数：6
最頻出単語：java（3回）
==============================
```

<details><summary>ヒント</summary>

- `String.split(" ")` で文字列をスペース区切りで分割して配列にできます
- `HashMap<String, Integer>` で単語と出現回数を管理します
- `map.getOrDefault(key, 0)` でキーが存在しない場合のデフォルト値を指定できます
- `map.entrySet()` でキーと値のペアを走査できます

</details>

<details><summary>解答例</summary>

```java
import java.util.HashMap;
import java.util.Map;

public class WordFrequencyCounter {
    public static void main(String[] args) {
        String sentence = "java is fun and java is powerful and java is everywhere";

        System.out.println("===== 単語頻度カウンター =====");
        System.out.println("文章：" + sentence);

        // スペース区切りで単語に分割
        String[] words = sentence.split(" ");

        // HashMapで単語の出現回数をカウント
        HashMap<String, Integer> wordCount = new HashMap<>();

        for (String word : words) {
            // getOrDefaultでキーが存在しない場合は0を返す
            wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
        }

        // 結果表示
        System.out.println();
        System.out.println("--- 単語の出現回数 ---");

        // 最頻出単語を探しながら表示
        String mostFrequent = "";
        int maxCount = 0;

        for (Map.Entry<String, Integer> entry : wordCount.entrySet()) {
            System.out.println(entry.getKey() + " → " + entry.getValue() + "回");

            // 最頻出単語の更新
            if (entry.getValue() > maxCount) {
                maxCount = entry.getValue();
                mostFrequent = entry.getKey();
            }
        }

        System.out.println();
        System.out.println("総単語数：" + words.length);
        System.out.println("異なる単語数：" + wordCount.size());
        System.out.println("最頻出単語：" + mostFrequent + "（" + maxCount + "回）");
        System.out.println("==============================");
    }
}
```

</details>

### 問題6：行列の加算

2つの2次元配列（行列）を用意し、行列の加算を行って結果を表示するプログラムを作成してください。行列を見やすくフォーマットして表示してください。

**期待される出力例：**
```
===== 行列の加算 =====
行列A：
| 1  2  3 |
| 4  5  6 |
| 7  8  9 |

行列B：
| 9  8  7 |
| 6  5  4 |
| 3  2  1 |

A + B の結果：
| 10 10 10 |
| 10 10 10 |
| 10 10 10 |
======================
```

<details><summary>ヒント</summary>

- 2次元配列は `int[][] matrix = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};` のように宣言します
- 行列の加算は対応する要素同士を足すだけです：`result[i][j] = a[i][j] + b[i][j]`
- ネストしたfor文（二重ループ）で全要素にアクセスします
- 行列の表示を別のメソッドにしたい場合はメソッドを作っても構いません

</details>

<details><summary>解答例</summary>

```java
public class MatrixAddition {
    public static void main(String[] args) {
        // 2つの3x3行列を定義
        int[][] matrixA = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        int[][] matrixB = {
            {9, 8, 7},
            {6, 5, 4},
            {3, 2, 1}
        };

        // 行列の加算（同じサイズの行列同士）
        int rows = matrixA.length;
        int cols = matrixA[0].length;
        int[][] result = new int[rows][cols];

        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                // 対応する要素同士を加算
                result[i][j] = matrixA[i][j] + matrixB[i][j];
            }
        }

        // 結果表示
        System.out.println("===== 行列の加算 =====");
        System.out.println("行列A：");
        printMatrix(matrixA);
        System.out.println();
        System.out.println("行列B：");
        printMatrix(matrixB);
        System.out.println();
        System.out.println("A + B の結果：");
        printMatrix(result);
        System.out.println("======================");
    }

    // 行列を見やすく表示するメソッド
    static void printMatrix(int[][] matrix) {
        for (int[] row : matrix) {
            System.out.print("|");
            for (int val : row) {
                System.out.printf(" %2d", val);
            }
            System.out.println(" |");
        }
    }
}
```

</details>

### 問題7：簡易電話帳プログラム

HashMapを使った簡易電話帳プログラムを作成してください。以下の機能を実装してください。
- 連絡先の登録（名前と電話番号）
- 全連絡先の一覧表示
- 名前で検索
- 名前の一部で検索（部分一致）

**期待される出力例：**
```
===== 簡易電話帳 =====
--- 連絡先を登録 ---
登録：田中太郎 → 090-1234-5678
登録：鈴木花子 → 080-9876-5432
登録：佐藤次郎 → 070-1111-2222
登録：田中美咲 → 090-3333-4444
登録：山田太郎 → 080-5555-6666

--- 全連絡先一覧 ---
名前：佐藤次郎 | 電話番号：070-1111-2222
名前：山田太郎 | 電話番号：080-5555-6666
名前：田中太郎 | 電話番号：090-1234-5678
名前：田中美咲 | 電話番号：090-3333-4444
名前：鈴木花子 | 電話番号：080-9876-5432
登録件数：5件

--- 名前で検索：田中太郎 ---
見つかりました：田中太郎 → 090-1234-5678

--- 部分一致検索：田中 ---
田中太郎 → 090-1234-5678
田中美咲 → 090-3333-4444
「田中」の検索結果：2件
==========================
```

<details><summary>ヒント</summary>

- `HashMap<String, String>` で名前（キー）と電話番号（値）を管理します
- 完全一致検索は `map.get(key)` で直接取得できます
- 部分一致検索は `map.keySet()` で全キーを取得し、`key.contains(検索文字列)` で判定します
- `TreeMap` を使うとキーが自動的にソートされます

</details>

<details><summary>解答例</summary>

```java
import java.util.Map;
import java.util.TreeMap;

public class PhoneBook {
    public static void main(String[] args) {
        // TreeMapで名前順にソートされた電話帳を作成
        TreeMap<String, String> phoneBook = new TreeMap<>();

        System.out.println("===== 簡易電話帳 =====");

        // 連絡先の登録
        System.out.println("--- 連絡先を登録 ---");
        registerContact(phoneBook, "田中太郎", "090-1234-5678");
        registerContact(phoneBook, "鈴木花子", "080-9876-5432");
        registerContact(phoneBook, "佐藤次郎", "070-1111-2222");
        registerContact(phoneBook, "田中美咲", "090-3333-4444");
        registerContact(phoneBook, "山田太郎", "080-5555-6666");

        // 全連絡先一覧
        System.out.println();
        System.out.println("--- 全連絡先一覧 ---");
        for (Map.Entry<String, String> entry : phoneBook.entrySet()) {
            System.out.println("名前：" + entry.getKey() + " | 電話番号：" + entry.getValue());
        }
        System.out.println("登録件数：" + phoneBook.size() + "件");

        // 名前で完全一致検索
        System.out.println();
        String searchName = "田中太郎";
        System.out.println("--- 名前で検索：" + searchName + " ---");
        String result = phoneBook.get(searchName);
        if (result != null) {
            System.out.println("見つかりました：" + searchName + " → " + result);
        } else {
            System.out.println("「" + searchName + "」は見つかりませんでした。");
        }

        // 部分一致検索
        System.out.println();
        String partialSearch = "田中";
        System.out.println("--- 部分一致検索：" + partialSearch + " ---");
        int matchCount = 0;
        for (Map.Entry<String, String> entry : phoneBook.entrySet()) {
            // キー（名前）に検索文字列が含まれるかチェック
            if (entry.getKey().contains(partialSearch)) {
                System.out.println(entry.getKey() + " → " + entry.getValue());
                matchCount++;
            }
        }
        System.out.println("「" + partialSearch + "」の検索結果：" + matchCount + "件");
        System.out.println("==========================");
    }

    // 連絡先を登録するメソッド
    static void registerContact(TreeMap<String, String> book, String name, String phone) {
        book.put(name, phone);
        System.out.println("登録：" + name + " → " + phone);
    }
}
```

</details>
