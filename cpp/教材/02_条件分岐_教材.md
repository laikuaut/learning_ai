# 第2章：条件分岐

> **この章のゴール**: if文、switch文、三項演算子を使って、条件に応じた処理の分岐を書けるようになる。比較演算子・論理演算子を正しく使えるようになる。

---

## 2.1 条件分岐とは

プログラムは通常、上から下へ順番に実行されます。しかし、「もし〜なら〜する」という **条件分岐（conditional branch）** を使うことで、状況に応じて異なる処理を実行できます。

```
               ┌─────────┐
               │ 条件判定 │
               └────┬────┘
                    │
              ┌─────┴─────┐
              │           │
         [true]       [false]
              │           │
        ┌─────┴──┐  ┌────┴─────┐
        │ 処理A  │  │ 処理B    │
        └────────┘  └──────────┘
              │           │
              └─────┬─────┘
                    │
              （次の処理へ）
```

## 2.2 比較演算子（comparison operator）

条件を書くには、まず **比較演算子** を知る必要があります。

```
┌──────────┬──────────────────┬────────────────────────┐
│ 演算子   │ 意味             │ 例                     │
├──────────┼──────────────────┼────────────────────────┤
│   ==     │ 等しい           │ a == b                 │
│   !=     │ 等しくない       │ a != b                 │
│   <      │ より小さい       │ a < b                  │
│   >      │ より大きい       │ a > b                  │
│   <=     │ 以下             │ a <= b                 │
│   >=     │ 以上             │ a >= b                 │
└──────────┴──────────────────┴────────────────────────┘
```

```cpp
#include <iostream>

int main() {
    int a = 10;
    int b = 20;

    std::cout << std::boolalpha;  // true/false を文字列で表示
    std::cout << "a == b: " << (a == b) << std::endl;  // false
    std::cout << "a != b: " << (a != b) << std::endl;  // true
    std::cout << "a < b:  " << (a < b)  << std::endl;  // true
    std::cout << "a > b:  " << (a > b)  << std::endl;  // false
    std::cout << "a <= b: " << (a <= b) << std::endl;  // true
    std::cout << "a >= b: " << (a >= b) << std::endl;  // false

    return 0;
}
```

出力：
```
a == b: false
a != b: true
a < b:  true
a > b:  false
a <= b: true
a >= b: false
```

### よくある間違い

```cpp
// NG: = は代入、== は比較。間違えやすい！
int x = 5;
if (x = 10) {  // 代入してしまう！ x が 10 になり、常に true
    // ...
}

// OK: == を使う
if (x == 10) {
    // ...
}
```

## 2.3 if 文

### 基本の if 文

```cpp
#include <iostream>

int main() {
    int temperature = 35;

    if (temperature >= 30) {
        std::cout << "真夏日です。熱中症に注意しましょう。" << std::endl;
    }

    return 0;
}
```

出力：
```
真夏日です。熱中症に注意しましょう。
```

### if-else 文

条件が偽（false）の場合の処理も書けます。

```cpp
#include <iostream>

int main() {
    int age = 15;

    if (age >= 18) {
        std::cout << "成人です。投票権があります。" << std::endl;
    } else {
        std::cout << "未成年です。" << std::endl;
    }

    return 0;
}
```

出力：
```
未成年です。
```

### if-else if-else 文

複数の条件を順に判定できます。

```cpp
#include <iostream>

int main() {
    int score = 75;

    if (score >= 90) {
        std::cout << "評価: A（優秀）" << std::endl;
    } else if (score >= 80) {
        std::cout << "評価: B（良好）" << std::endl;
    } else if (score >= 70) {
        std::cout << "評価: C（合格）" << std::endl;
    } else if (score >= 60) {
        std::cout << "評価: D（可）" << std::endl;
    } else {
        std::cout << "評価: F（不合格）" << std::endl;
    }

    return 0;
}
```

出力：
```
評価: C（合格）
```

条件は **上から順に評価** され、最初に `true` になった分岐だけが実行されます。

```
score = 75 の場合:

  score >= 90?  → false  → 次へ
  score >= 80?  → false  → 次へ
  score >= 70?  → true   → ★ ここが実行される
  score >= 60?  → 評価されない
  else          → 評価されない
```

### ポイントまとめ
- `if` は条件が `true` のときだけブロック内を実行します
- `else if` で複数条件を連鎖でき、`else` がどれにも当てはまらない場合を処理します
- 条件は上から順に評価され、最初に `true` になったものだけが実行されます

## 2.4 論理演算子（logical operator）

複数の条件を組み合わせるために **論理演算子** を使います。

```
┌──────────┬────────────┬───────────────────────────────┐
│ 演算子   │ 意味       │ 説明                          │
├──────────┼────────────┼───────────────────────────────┤
│   &&     │ AND（かつ）│ 両方 true なら true            │
│   ||     │ OR（または）│ どちらか true なら true       │
│   !      │ NOT（否定）│ true を false に、逆も同様     │
└──────────┴────────────┴───────────────────────────────┘
```

### 真理値表

```
AND (&&)            OR (||)             NOT (!)
┌───┬───┬────┐     ┌───┬───┬────┐     ┌───┬────┐
│ A │ B │ 結果│     │ A │ B │ 結果│     │ A │ 結果│
├───┼───┼────┤     ├───┼───┼────┤     ├───┼────┤
│ T │ T │  T │     │ T │ T │  T │     │ T │  F │
│ T │ F │  F │     │ T │ F │  T │     │ F │  T │
│ F │ T │  F │     │ F │ T │  T │     └───┴────┘
│ F │ F │  F │     │ F │ F │  F │
└───┴───┴────┘     └───┴───┴────┘
```

### コード例：入場資格の判定

```cpp
#include <iostream>

int main() {
    int age = 22;
    bool has_ticket = true;
    bool is_vip = false;

    // AND: 年齢18以上 かつ チケットあり
    if (age >= 18 && has_ticket) {
        std::cout << "入場できます。" << std::endl;
    }

    // OR: VIP または 年齢65以上 → 優先入場
    if (is_vip || age >= 65) {
        std::cout << "優先入場できます。" << std::endl;
    } else {
        std::cout << "一般入場です。" << std::endl;
    }

    // NOT: チケットがない場合
    if (!has_ticket) {
        std::cout << "チケットを購入してください。" << std::endl;
    }

    return 0;
}
```

出力：
```
入場できます。
一般入場です。
```

### 短絡評価（short-circuit evaluation）

論理演算子は **短絡評価** を行います。結果が確定した時点で、残りの条件は評価されません。

```cpp
#include <iostream>

int main() {
    int x = 0;

    // && の短絡評価: 左辺が false なら右辺は評価されない
    if (x != 0 && 100 / x > 10) {
        std::cout << "条件成立" << std::endl;
    } else {
        std::cout << "条件不成立（ゼロ除算を回避できた！）" << std::endl;
    }

    // || の短絡評価: 左辺が true なら右辺は評価されない
    bool result = true || false;  // false は評価されない

    return 0;
}
```

出力：
```
条件不成立（ゼロ除算を回避できた！）
```

短絡評価を活用すると、安全にゼロ除算などのエラーを防げます。

### ポイントまとめ
- `&&`（AND）は両方 true のとき true です
- `||`（OR）はどちらか true のとき true です
- `!`（NOT）は真偽を反転します
- 短絡評価を理解して、安全な条件式を書きましょう

## 2.5 switch 文

**switch文（switch statement）** は、1つの値に対して複数の選択肢を判定する場合に便利です。

### 基本構文

```cpp
#include <iostream>

int main() {
    int day = 3;

    switch (day) {
        case 1:
            std::cout << "月曜日" << std::endl;
            break;
        case 2:
            std::cout << "火曜日" << std::endl;
            break;
        case 3:
            std::cout << "水曜日" << std::endl;
            break;
        case 4:
            std::cout << "木曜日" << std::endl;
            break;
        case 5:
            std::cout << "金曜日" << std::endl;
            break;
        case 6:
            std::cout << "土曜日" << std::endl;
            break;
        case 7:
            std::cout << "日曜日" << std::endl;
            break;
        default:
            std::cout << "無効な値です" << std::endl;
            break;
    }

    return 0;
}
```

出力：
```
水曜日
```

### switch 文の流れ

```
switch (day) ──→ case 1 → 不一致 → case 2 → 不一致 → case 3 → 一致！
                                                          │
                                                    "水曜日" を表示
                                                          │
                                                      break で脱出
```

### フォールスルー（fall-through）

`break` を書き忘れると、次の `case` の処理も実行されてしまいます。これを **フォールスルー** といいます。

```cpp
#include <iostream>

int main() {
    int month = 3;

    // 意図的なフォールスルーの例：季節の判定
    switch (month) {
        case 3:
        case 4:
        case 5:
            std::cout << "春です" << std::endl;
            break;
        case 6:
        case 7:
        case 8:
            std::cout << "夏です" << std::endl;
            break;
        case 9:
        case 10:
        case 11:
            std::cout << "秋です" << std::endl;
            break;
        case 12:
        case 1:
        case 2:
            std::cout << "冬です" << std::endl;
            break;
        default:
            std::cout << "無効な月です" << std::endl;
            break;
    }

    return 0;
}
```

出力：
```
春です
```

### C++17: switch 文での変数宣言

C++17 では、switch 文の条件部分で変数を宣言できます。

```cpp
#include <iostream>
#include <string>

int main() {
    std::string input = "admin";

    // C++17: init-statement in switch
    // （この例では簡単なものですが、関数の戻り値を受けたい場合に便利です）
    switch (int role = (input == "admin") ? 1 : (input == "user") ? 2 : 0; role) {
        case 1:
            std::cout << "管理者権限です" << std::endl;
            break;
        case 2:
            std::cout << "一般ユーザーです" << std::endl;
            break;
        default:
            std::cout << "不明なロールです" << std::endl;
            break;
    }

    return 0;
}
```

### よくある間違い

```cpp
// NG: break を書き忘れる
switch (value) {
    case 1:
        std::cout << "1です" << std::endl;
        // break がない！→ case 2 の処理も実行されてしまう
    case 2:
        std::cout << "2です" << std::endl;
        break;
}
// value が 1 のとき、"1です" と "2です" の両方が表示される

// NG: switch に文字列を直接使う
// switch (name) {  // コンパイルエラー！ switch は整数型・列挙型のみ
```

### if 文と switch 文の使い分け

```
┌─────────────────────────────┬──────────────────────────────┐
│  if 文が向いている場合      │  switch 文が向いている場合   │
├─────────────────────────────┼──────────────────────────────┤
│ 範囲の比較（< > <=）       │ 値の完全一致の比較           │
│ 複雑な条件の組み合わせ     │ 整数や列挙型の分岐           │
│ 文字列の比較               │ 選択肢が多い場合             │
│ 条件が2〜3個の場合         │ メニュー選択など             │
└─────────────────────────────┴──────────────────────────────┘
```

### ポイントまとめ
- `switch` は整数型や列挙型の値に対する多分岐に適しています
- 各 `case` の最後に `break` を忘れずに書きましょう
- 意図的なフォールスルーは複数のケースをまとめるのに有効です
- `default` でどのケースにも当てはまらない場合を処理しましょう

## 2.6 三項演算子（ternary operator）

簡単な条件分岐は **三項演算子（条件演算子）** で1行にまとめられます。

```
構文: 条件式 ? 真の場合の値 : 偽の場合の値
```

```cpp
#include <iostream>
#include <string>

int main() {
    int age = 20;

    // if-else の代わりに三項演算子を使う
    std::string status = (age >= 18) ? "成人" : "未成年";
    std::cout << status << std::endl;  // 成人

    // 直接出力に使うこともできる
    int score = 85;
    std::cout << "結果: " << ((score >= 60) ? "合格" : "不合格") << std::endl;

    // 数値を選ぶ場合
    int a = 10, b = 20;
    int max_val = (a > b) ? a : b;
    std::cout << "大きい方: " << max_val << std::endl;  // 20

    return 0;
}
```

出力：
```
成人
結果: 合格
大きい方: 20
```

### よくある間違い

```cpp
// NG: 三項演算子をネストしすぎて読めなくなる
std::string grade = (score >= 90) ? "A" : (score >= 80) ? "B" : (score >= 70) ? "C" : "F";
// → 読みにくい！ この場合は if-else if を使いましょう

// OK: 単純な2択の場合に使う
std::string result = (score >= 60) ? "合格" : "不合格";
```

## 2.7 ネストした条件分岐

if 文の中にさらに if 文を書くことを **ネスト（nest）** といいます。

```cpp
#include <iostream>

int main() {
    int age = 25;
    bool has_license = true;

    if (age >= 18) {
        if (has_license) {
            std::cout << "運転できます。" << std::endl;
        } else {
            std::cout << "免許を取得してください。" << std::endl;
        }
    } else {
        std::cout << "18歳未満は運転できません。" << std::endl;
    }

    return 0;
}
```

出力：
```
運転できます。
```

### ネストを浅くする（早期リターン / ガード節）

ネストが深くなると読みにくくなります。**ガード節（guard clause）** を使ってネストを浅くするのがベストプラクティスです。

```cpp
#include <iostream>
#include <string>

// NG: ネストが深い
std::string evaluate_deep(int age, bool has_license, bool has_car) {
    if (age >= 18) {
        if (has_license) {
            if (has_car) {
                return "ドライブに行けます";
            } else {
                return "車が必要です";
            }
        } else {
            return "免許が必要です";
        }
    } else {
        return "18歳未満は不可です";
    }
}

// OK: ガード節でネストを浅く
std::string evaluate_flat(int age, bool has_license, bool has_car) {
    if (age < 18) {
        return "18歳未満は不可です";
    }
    if (!has_license) {
        return "免許が必要です";
    }
    if (!has_car) {
        return "車が必要です";
    }
    return "ドライブに行けます";
}

int main() {
    std::cout << evaluate_flat(25, true, true) << std::endl;
    std::cout << evaluate_flat(25, true, false) << std::endl;
    std::cout << evaluate_flat(25, false, true) << std::endl;
    std::cout << evaluate_flat(16, true, true) << std::endl;

    return 0;
}
```

出力：
```
ドライブに行けます
車が必要です
免許が必要です
18歳未満は不可です
```

ガード節を使った `evaluate_flat` の方が、条件の優先順位が明確で読みやすくなっています。

## 2.8 スコープ（scope）

**スコープ** とは、変数が有効な範囲のことです。`{}` で囲まれたブロック内で宣言された変数は、そのブロックの外からはアクセスできません。

```cpp
#include <iostream>

int main() {
    int x = 10;  // main 関数全体で有効

    if (true) {
        int y = 20;  // この if ブロック内でのみ有効
        std::cout << "x = " << x << std::endl;  // OK: x はアクセス可能
        std::cout << "y = " << y << std::endl;  // OK: y はアクセス可能
    }

    std::cout << "x = " << x << std::endl;  // OK
    // std::cout << "y = " << y << std::endl;  // コンパイルエラー！ y はスコープ外

    return 0;
}
```

### スコープの図

```
main() {
│  int x = 10;         ← x のスコープ開始
│
│  if (...) {
│  │  int y = 20;      ← y のスコープ開始
│  │  // x, y 両方使える
│  }                   ← y のスコープ終了（y は破棄される）
│
│  // x だけ使える
│  // y は使えない
}                      ← x のスコープ終了
```

### C++17: if 文での変数宣言

C++17 では、if 文の条件部分で変数を宣言できます。変数のスコープを if ブロック内に限定できるため便利です。

```cpp
#include <iostream>
#include <string>
#include <map>

int main() {
    std::map<std::string, int> scores = {
        {"太郎", 85},
        {"花子", 92},
        {"次郎", 78}
    };

    // C++17: if 文の中で変数を宣言
    if (auto it = scores.find("花子"); it != scores.end()) {
        std::cout << "花子の点数: " << it->second << std::endl;
    } else {
        std::cout << "花子は見つかりません" << std::endl;
    }
    // it はここではアクセスできない（スコープ外）

    return 0;
}
```

出力：
```
花子の点数: 92
```

### ポイントまとめ
- 変数は宣言されたブロック `{}` 内でのみ有効です
- スコープを意識して、変数の寿命を必要最小限にしましょう
- C++17 の `if` 初期化構文で変数のスコープを限定できます

## 2.9 実践例：簡易電卓

この章の内容を組み合わせた実践プログラムです。

```cpp
// calculator.cpp — 簡易電卓
// コンパイル: g++ -std=c++17 -o calc calculator.cpp
// 実行: ./calc

#include <iostream>
#include <limits>

int main() {
    std::cout << "===== 簡易電卓 =====" << std::endl;
    std::cout << std::endl;

    // 数値1の入力
    std::cout << "1つ目の数値を入力: ";
    double num1;
    std::cin >> num1;

    // 演算子の入力
    std::cout << "演算子を入力 (+, -, *, /): ";
    char op;
    std::cin >> op;

    // 数値2の入力
    std::cout << "2つ目の数値を入力: ";
    double num2;
    std::cin >> num2;

    // 計算と結果表示
    std::cout << std::endl;

    switch (op) {
        case '+':
            std::cout << num1 << " + " << num2 << " = " << (num1 + num2) << std::endl;
            break;
        case '-':
            std::cout << num1 << " - " << num2 << " = " << (num1 - num2) << std::endl;
            break;
        case '*':
            std::cout << num1 << " * " << num2 << " = " << (num1 * num2) << std::endl;
            break;
        case '/':
            if (num2 == 0) {
                std::cout << "エラー: ゼロで割ることはできません。" << std::endl;
            } else {
                std::cout << num1 << " / " << num2 << " = " << (num1 / num2) << std::endl;
            }
            break;
        default:
            std::cout << "エラー: 無効な演算子です。" << std::endl;
            break;
    }

    return 0;
}
```

実行例：
```
===== 簡易電卓 =====

1つ目の数値を入力: 15.5
演算子を入力 (+, -, *, /): *
2つ目の数値を入力: 3

15.5 * 3 = 46.5
```
