# 実践課題02：おみくじアプリ ★1

> **難易度**: ★☆☆☆☆（入門）
> **前提知識**: 第1章（基本構文と変数）、第2章（演算子と型変換）、第3章（条件分岐とループ）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 条件分岐・ランダム値生成・基本的なDOM操作を組み合わせて、インタラクティブなアプリを作る

---

## 完成イメージ

```
===== おみくじ =====

[おみくじを引く]

結果: 大吉
「素晴らしい一日になるでしょう！」

--- 統計 ---
大吉: 1回  中吉: 0回  小吉: 0回  吉: 0回  末吉: 0回  凶: 0回
合計: 1回
```

ボタンを押すたびにランダムな結果が表示され、各結果の出現回数が記録されます。

---

## 課題の要件

1. HTMLファイル1つで完結すること
2. 「おみくじを引く」ボタンを押すと、6種類（大吉・中吉・小吉・吉・末吉・凶）のうち1つがランダムに表示される
3. 各結果に対応するメッセージ（一言コメント）を表示する
4. 各結果の出現回数を集計して画面に表示する
5. `Math.random()` と `Math.floor()` を使ってランダム値を生成する

---

<details>
<summary>ステップ2：ランダムに結果を選ぶ</summary>

`Math.random()` は 0以上1未満の小数を返します。配列のインデックスにするには：

```javascript
const index = Math.floor(Math.random() * fortunes.length);
// 例: Math.random() → 0.73 → 0.73 * 6 → 4.38 → Math.floor → 4
```

</details>

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>おみくじ</title>
  <style>
    body { font-family: sans-serif; text-align: center; margin-top: 40px; background: #fdf6e3; }
    h1 { color: #b71c1c; }
    #result { font-size: 48px; font-weight: bold; margin: 20px 0; }
    #message { font-size: 18px; color: #555; margin-bottom: 20px; }
    button { font-size: 20px; padding: 12px 30px; cursor: pointer;
             background: #c62828; color: white; border: none; border-radius: 8px; }
    button:hover { background: #e53935; }
    #stats { margin-top: 30px; font-size: 14px; color: #666; }
  </style>
</head>
<body>
  <h1>おみくじ</h1>

  <button onclick="drawFortune()">おみくじを引く</button>

  <p id="result">---</p>
  <p id="message">ボタンを押しておみくじを引きましょう</p>
  <div id="stats"></div>

  <script>
    // おみくじの種類
    const fortunes = ["大吉", "中吉", "小吉", "吉", "末吉", "凶"];

    // 各結果に対応するメッセージ
    const messages = [
      "素晴らしい一日になるでしょう！",
      "良いことがありそうです。",
      "小さな幸せが見つかります。",
      "穏やかな一日です。",
      "焦らず進みましょう。",
      "慎重に行動しましょう。"
    ];

    // 各結果の出現回数（初期値は全て0）
    let count0 = 0;  // 大吉
    let count1 = 0;  // 中吉
    let count2 = 0;  // 小吉
    let count3 = 0;  // 吉
    let count4 = 0;  // 末吉
    let count5 = 0;  // 凶

    // 合計回数
    let total = 0;

    function drawFortune() {
      // ランダムに1つ選ぶ
      const index = Math.floor(Math.random() * fortunes.length);
      const fortune = fortunes[index];
      const msg = messages[index];

      // 結果を画面に表示
      document.getElementById("result").textContent = fortune;
      document.getElementById("message").textContent = "「" + msg + "」";

      // カウントを増やす
      if (index === 0) { count0 = count0 + 1; }
      else if (index === 1) { count1 = count1 + 1; }
      else if (index === 2) { count2 = count2 + 1; }
      else if (index === 3) { count3 = count3 + 1; }
      else if (index === 4) { count4 = count4 + 1; }
      else if (index === 5) { count5 = count5 + 1; }
      total = total + 1;

      // 統計を表示
      document.getElementById("stats").textContent =
        "--- 統計 --- " +
        "大吉:" + count0 + "回  " +
        "中吉:" + count1 + "回  " +
        "小吉:" + count2 + "回  " +
        "吉:" + count3 + "回  " +
        "末吉:" + count4 + "回  " +
        "凶:" + count5 + "回  " +
        "| 合計:" + total + "回";
    }
  </script>
</body>
</html>
```

<details>
<summary>解答例（改良版 ─ 第5章・第6章の知識を先取り）</summary>

配列とオブジェクトを使うと、コードが格段に短くなり、種類を増やすのも簡単になります。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>おみくじ（改良版）</title>
  <style>
    body { font-family: sans-serif; text-align: center; margin-top: 40px; background: #fdf6e3; }
    h1 { color: #b71c1c; }
    #result { font-size: 48px; font-weight: bold; margin: 20px 0; min-height: 60px; }
    #message { font-size: 18px; color: #555; min-height: 24px; }
    button { font-size: 20px; padding: 12px 30px; cursor: pointer;
             background: #c62828; color: white; border: none; border-radius: 8px; }
    button:hover { background: #e53935; }
    #stats { margin-top: 30px; font-size: 14px; color: #666; }
    .stat-item { display: inline-block; margin: 4px 8px; }
  </style>
</head>
<body>
  <h1>おみくじ</h1>

  <button onclick="drawFortune()">おみくじを引く</button>

  <p id="result">---</p>
  <p id="message">ボタンを押しておみくじを引きましょう</p>
  <div id="stats"></div>

  <script>
    // おみくじデータをオブジェクトの配列で管理
    const fortuneData = [
      { name: "大吉", message: "素晴らしい一日になるでしょう！", color: "#d4af37" },
      { name: "中吉", message: "良いことがありそうです。", color: "#e65100" },
      { name: "小吉", message: "小さな幸せが見つかります。", color: "#2e7d32" },
      { name: "吉",   message: "穏やかな一日です。", color: "#1565c0" },
      { name: "末吉", message: "焦らず進みましょう。", color: "#6a1b9a" },
      { name: "凶",   message: "慎重に行動しましょう。", color: "#757575" },
    ];

    // 統計をオブジェクトで管理（キー: おみくじ名, 値: 回数）
    const stats = {};
    for (let i = 0; i < fortuneData.length; i++) {
      stats[fortuneData[i].name] = 0;
    }
    let total = 0;

    function drawFortune() {
      // ランダムに1つ選ぶ
      const index = Math.floor(Math.random() * fortuneData.length);
      const selected = fortuneData[index];

      // 結果を表示（色も変更）
      const resultEl = document.getElementById("result");
      resultEl.textContent = selected.name;
      resultEl.style.color = selected.color;
      document.getElementById("message").textContent = "「" + selected.message + "」";

      // 統計を更新
      stats[selected.name] += 1;
      total += 1;

      // 統計テキストを組み立てて表示
      let statsText = "--- 統計（合計 " + total + " 回）---\n";
      for (let i = 0; i < fortuneData.length; i++) {
        const name = fortuneData[i].name;
        statsText += name + ": " + stats[name] + "回  ";
      }
      document.getElementById("stats").textContent = statsText;
    }
  </script>
</body>
</html>
```

**初心者向けとの違い:**
- おみくじデータをオブジェクトの配列で一元管理 → 種類の追加が1行で済む
- 統計をオブジェクトで管理 → `if-else` の連鎖が不要
- `for` ループで統計テキストを動的に組み立て → 種類が増えてもコード変更不要
- 各おみくじに対応する色を設定して視覚的にリッチに

</details>
