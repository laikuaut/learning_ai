# 実践課題03：タスクリスト ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第3章（stateとイベント）、第4章（条件分岐とリスト）
> **課題の種類**: ミニプロジェクト
> **学習目標**: 配列stateの操作（追加・削除・更新）とリスト描画を実践する

---

## 完成イメージ

```
===== タスクリスト =====
[タスクを入力...          ] [追加]

フィルタ: [すべて] [未完了] [完了済み]

☐ 買い物に行く                [削除]
☑ レポートを書く (完了)       [削除]
☐ Reactの勉強                [削除]

残り: 2件 / 全3件
```

---

## 課題の要件

1. テキスト入力 + 追加ボタンでタスクを追加できる
2. 各タスクのチェックボックスで完了/未完了を切り替えられる
3. 削除ボタンで個別のタスクを削除できる
4. フィルタ（すべて/未完了/完了済み）で表示を切り替えられる
5. 空文字のタスクは追加できないようにする
6. 残りタスク数を表示する

---

## ステップガイド

<details>
<summary>ステップ1：タスクのデータ構造を決める</summary>

```jsx
const [tasks, setTasks] = React.useState([
  { id: 1, text: "買い物に行く", done: false },
  { id: 2, text: "レポートを書く", done: true },
]);
```

各タスクに `id`（一意の識別子）、`text`（内容）、`done`（完了フラグ）を持たせます。

</details>

<details>
<summary>ステップ2：追加・削除・完了切替の関数を作る</summary>

```jsx
// 追加
function handleAdd() {
  if (input.trim() === "") return;
  setTasks([...tasks, { id: Date.now(), text: input, done: false }]);
  setInput("");
}

// 完了切替
function handleToggle(id) {
  setTasks(tasks.map(task =>
    task.id === id ? { ...task, done: !task.done } : task
  ));
}

// 削除
function handleDelete(id) {
  setTasks(tasks.filter(task => task.id !== id));
}
```

</details>

<details>
<summary>ステップ3：フィルタ機能を追加する</summary>

```jsx
const [filter, setFilter] = React.useState("all"); // "all", "active", "completed"

const filteredTasks = tasks.filter(task => {
  if (filter === "active") return !task.done;
  if (filter === "completed") return task.done;
  return true;
});
```

</details>

```html
  </style>
</head>
<body>
  <div id="root"></div>
  <script type="text/babel">
    function TaskItem({ task, onToggle, onDelete }) {
      return (
        <div className={"task-item" + (task.done ? " done" : "")}>
          <input
            type="checkbox"
            checked={task.done}
            onChange={() => onToggle(task.id)}
          />
          <span className={"text" + (task.done ? " done" : "")}>
            {task.text}
          </span>
          <button className="delete-btn" onClick={() => onDelete(task.id)}>
            削除
          </button>
        </div>
      );

    function App() {
      const [tasks, setTasks] = React.useState([
        { id: 1, text: "買い物に行く", done: false },
        { id: 2, text: "レポートを書く", done: true },
        { id: 3, text: "Reactの勉強", done: false },
      ]);
      const [input, setInput] = React.useState("");
      const [filter, setFilter] = React.useState("all");

      function handleAdd() {
        if (input.trim() === "") return;
        var newTask = { id: Date.now(), text: input.trim(), done: false };
        setTasks([...tasks, newTask]);
        setInput("");
      }

      function handleToggle(id) {
        setTasks(tasks.map(function(task) {
          if (task.id === id) {
            return { ...task, done: !task.done };
          }
          return task;
        }));
      }

      function handleDelete(id) {
        setTasks(tasks.filter(function(task) {
          return task.id !== id;
        }));
      }

      function handleKeyDown(e) {
        if (e.key === "Enter") handleAdd();
      }

      // フィルタ適用
      var filteredTasks = tasks;
      if (filter === "active") {
        filteredTasks = tasks.filter(function(t) { return !t.done; });
      } else if (filter === "completed") {
        filteredTasks = tasks.filter(function(t) { return t.done; });
      }

      var remaining = tasks.filter(function(t) { return !t.done; }).length;

      return (
        <div>
          <h1>タスクリスト</h1>

          <div className="input-row">
            <input
              type="text"
              placeholder="タスクを入力..."
              value={input}
              onChange={(e) => setInput(e.target.value)}
              onKeyDown={handleKeyDown}
            />
            <button onClick={handleAdd}>追加</button>
          </div>

          <div className="filter-row">
            {["all", "active", "completed"].map(function(f) {
              var label = f === "all" ? "すべて" : f === "active" ? "未完了" : "完了済み";
              return (
                <button
                  key={f}
                  className={"filter-btn" + (filter === f ? " active" : "")}
                  onClick={() => setFilter(f)}
                >
                  {label}
                </button>
              );
            })}
          </div>

          {filteredTasks.map(function(task) {
            return (
              <TaskItem
                key={task.id}
                task={task}
                onToggle={handleToggle}
                onDelete={handleDelete}
              />
            );
          })}

          {filteredTasks.length === 0 && (
            <p style={{ textAlign: "center", color: "#999" }}>タスクがありません</p>
          )}

          <div className="summary">
            残り: {remaining}件 / 全{tasks.length}件
          </div>
        </div>
      );
    }

    ReactDOM.createRoot(document.getElementById("root")).render(<App />);
  </script>
</body>
</html>
```

<summary>解答例（改良版 ─ 編集機能追加）</summary>

初心者向けに以下を追加した改良版です。差分のみ示します。

```jsx
// TaskItem に編集モードを追加
function TaskItem({ task, onToggle, onDelete, onEdit }) {
  const [editing, setEditing] = React.useState(false);
  const [editText, setEditText] = React.useState(task.text);

  const handleSave = () => {
    if (editText.trim()) {
      onEdit(task.id, editText.trim());
      setEditing(false);
    }
  };

  return (
    <div className={"task-item" + (task.done ? " done" : "")}>
      <input type="checkbox" checked={task.done} onChange={() => onToggle(task.id)} />
      {editing ? (
        <>
          <input
            value={editText}
            onChange={(e) => setEditText(e.target.value)}
            onKeyDown={(e) => e.key === "Enter" && handleSave()}
            style={{ flex: 1, marginLeft: 12, padding: "4px 8px" }}
            autoFocus
          />
          <button onClick={handleSave} style={{ background: "#4caf50", color: "white", border: "none", borderRadius: 6, padding: "4px 12px", cursor: "pointer", marginRight: 4 }}>
            保存
          </button>
        </>
      ) : (
        <>
          <span className={"text" + (task.done ? " done" : "")} onDoubleClick={() => setEditing(true)}>
            {task.text}
          </span>
          <button className="delete-btn" onClick={() => onDelete(task.id)}>削除</button>
        </>
      )}
    </div>
  );
}

// App に編集ハンドラを追加
function handleEdit(id, newText) {
  setTasks(tasks.map(task =>
    task.id === id ? { ...task, text: newText } : task
  ));
}
```

**初心者向けとの違い:**
- ダブルクリックで編集モードに切り替え
- `TaskItem` 内にローカルstateを持つ（`editing`, `editText`）
- Enterキーで保存、空文字は保存不可
