## 基本問題

### 問題1：偶数・奇数判定プログラム

整数を変数に格納し、その数が偶数か奇数かを判定して表示するプログラムを作成してください。

**期待される出力例：**
```
数値：7
判定：7は奇数です。
```

<details><summary>ヒント</summary>

- 偶数かどうかは `%`（剰余）演算子で2で割った余りが0かどうかで判定します
- `number % 2 == 0` なら偶数、そうでなければ奇数です

</details>

<details><summary>解答例</summary>

```java
public class EvenOddChecker {
    public static void main(String[] args) {
        int number = 7;

        System.out.println("数値：" + number);

        // 2で割った余りが0なら偶数、それ以外は奇数
        if (number % 2 == 0) {
            System.out.println("判定：" + number + "は偶数です。");
        } else {
            System.out.println("判定：" + number + "は奇数です。");
        }
    }
}
```

</details>

### 問題2：成績評価プログラム

テストの点数（0〜100）を変数に格納し、以下の基準で成績を評価して表示するプログラムを作成してください。

| 点数 | 評価 |
|------|------|
| 90〜100 | A（優秀） |
| 80〜89 | B（良好） |
| 70〜79 | C（普通） |
| 60〜69 | D（要努力） |
| 0〜59 | F（不合格） |

**期待される出力例：**
```
点数：85点
評価：B（良好）
コメント：よく頑張りました！
```

<details><summary>ヒント</summary>

- if-else if-else で上の条件から順に判定します
- 点数が0未満または100超の場合のエラー処理も入れると丁寧です
- 評価に応じたコメントも変数で管理すると整理しやすいです

</details>

<details><summary>解答例</summary>

```java
public class GradeEvaluator {
    public static void main(String[] args) {
        int score = 85;

        System.out.println("点数：" + score + "点");

        // 範囲外チェック
        if (score < 0 || score > 100) {
            System.out.println("エラー：点数は0〜100の範囲で指定してください。");
            return; // プログラム終了
        }

        // 成績判定（上の条件から順に判定）
        String grade;
        String comment;

        if (score >= 90) {
            grade = "A（優秀）";
            comment = "素晴らしい成績です！";
        } else if (score >= 80) {
            grade = "B（良好）";
            comment = "よく頑張りました！";
        } else if (score >= 70) {
            grade = "C（普通）";
            comment = "もう少し頑張りましょう。";
        } else if (score >= 60) {
            grade = "D（要努力）";
            comment = "復習が必要です。";
        } else {
            grade = "F（不合格）";
            comment = "基礎から見直しましょう。";
        }

        System.out.println("評価：" + grade);
        System.out.println("コメント：" + comment);
    }
}
```

</details>

### 問題3：正・負・ゼロ判定プログラム

整数を変数に格納し、その数が正の数、負の数、またはゼロのいずれかを判定して表示するプログラムを作成してください。正の数の場合はさらに1桁か2桁以上かも表示してください。

**期待される出力例：**
```
数値：42
判定：42は正の数です。
補足：2桁以上の数です。
```

<details><summary>ヒント</summary>

- `number > 0` で正、`number < 0` で負、それ以外はゼロです
- 正の数の桁数チェックは `number < 10` で1桁かどうかを判定できます

</details>

<details><summary>解答例</summary>

```java
public class PositiveNegativeZero {
    public static void main(String[] args) {
        int number = 42;

        System.out.println("数値：" + number);

        if (number > 0) {
            System.out.println("判定：" + number + "は正の数です。");
            // 1桁か2桁以上かを追加判定
            if (number < 10) {
                System.out.println("補足：1桁の数です。");
            } else {
                System.out.println("補足：2桁以上の数です。");
            }
        } else if (number < 0) {
            System.out.println("判定：" + number + "は負の数です。");
        } else {
            System.out.println("判定：0（ゼロ）です。");
        }
    }
}
```

</details>

### 問題4：うるう年判定プログラム

西暦年を変数に格納し、うるう年かどうかを判定して表示するプログラムを作成してください。

うるう年の判定ルール：
1. 4で割り切れる年はうるう年
2. ただし、100で割り切れる年はうるう年ではない
3. ただし、400で割り切れる年はうるう年

**期待される出力例：**
```
西暦：2024年
判定：2024年はうるう年です。（2月は29日まであります）
```

<details><summary>ヒント</summary>

- 判定は「400で割り切れる」→「100で割り切れる」→「4で割り切れる」の順で行うとシンプルです
- 論理演算子 `&&`（AND）や `||`（OR）を使って1つの条件式にまとめることもできます
- `(year % 400 == 0) || (year % 4 == 0 && year % 100 != 0)` で一度に判定可能です

</details>

<details><summary>解答例</summary>

```java
public class LeapYearChecker {
    public static void main(String[] args) {
        int year = 2024;

        System.out.println("西暦：" + year + "年");

        // うるう年判定
        // 400で割り切れる → うるう年
        // 100で割り切れる → うるう年ではない
        // 4で割り切れる → うるう年
        boolean isLeapYear = (year % 400 == 0) || (year % 4 == 0 && year % 100 != 0);

        if (isLeapYear) {
            System.out.println("判定：" + year + "年はうるう年です。（2月は29日まであります）");
        } else {
            System.out.println("判定：" + year + "年はうるう年ではありません。（2月は28日までです）");
        }
    }
}
```

</details>

### 問題5：チケット料金計算プログラム

年齢を変数に格納し、以下の料金体系に基づいてチケット料金を表示するプログラムを作成してください。さらに、土日かどうかのフラグ（boolean）を追加し、土日は全料金200円引きとしてください。

| 年齢 | 区分 | 料金 |
|------|------|------|
| 0〜5歳 | 無料 | 0円 |
| 6〜12歳 | 子供 | 500円 |
| 13〜17歳 | 中高生 | 800円 |
| 18〜64歳 | 大人 | 1500円 |
| 65歳以上 | シニア | 1000円 |

**期待される出力例：**
```
===== チケット料金 =====
年齢：10歳
区分：子供
曜日：土日
通常料金：500円
割引額：-200円
お支払い額：300円
========================
```

<details><summary>ヒント</summary>

- if-else if文で年齢に応じた区分と料金を決定します
- 土日割引は最後に適用しますが、割引後に負の値にならないよう `Math.max(0, price)` を使いましょう
- 無料（0円）のチケットには割引を適用しないようにする配慮も大切です

</details>

<details><summary>解答例</summary>

```java
public class TicketPriceCalculator {
    public static void main(String[] args) {
        int age = 10;
        boolean isWeekend = true; // 土日かどうか

        // 年齢に応じた区分と料金を決定
        String category;
        int price;

        if (age < 0) {
            System.out.println("エラー：年齢が不正です。");
            return;
        } else if (age <= 5) {
            category = "無料";
            price = 0;
        } else if (age <= 12) {
            category = "子供";
            price = 500;
        } else if (age <= 17) {
            category = "中高生";
            price = 800;
        } else if (age <= 64) {
            category = "大人";
            price = 1500;
        } else {
            category = "シニア";
            price = 1000;
        }

        // 土日割引の計算（無料チケットには適用しない）
        int discount = 0;
        if (isWeekend && price > 0) {
            discount = 200;
        }
        // 割引後の金額（0円未満にならないようにする）
        int finalPrice = Math.max(0, price - discount);

        // 結果表示
        System.out.println("===== チケット料金 =====");
        System.out.println("年齢：" + age + "歳");
        System.out.println("区分：" + category);
        System.out.println("曜日：" + (isWeekend ? "土日" : "平日"));
        System.out.println("通常料金：" + price + "円");
        if (discount > 0) {
            System.out.println("割引額：-" + discount + "円");
        }
        System.out.println("お支払い額：" + finalPrice + "円");
        System.out.println("========================");
    }
}
```

</details>

### 問題6：じゃんけんプログラム

プレイヤーとコンピューターの手を変数（0=グー, 1=チョキ, 2=パー）に格納し、勝敗を判定して表示するプログラムを作成してください。

**期待される出力例：**
```
===== じゃんけん =====
プレイヤー：グー
コンピューター：チョキ
結果：あなたの勝ちです！
======================
```

<details><summary>ヒント</summary>

- 手の名前は配列またはswitch文で変換できます
- 勝敗判定は `(player - computer + 3) % 3` で簡潔に書けます（0:あいこ, 1:負け, 2:勝ち）
- if文で個別に判定しても構いません

</details>

<details><summary>解答例</summary>

```java
public class RockPaperScissors {
    public static void main(String[] args) {
        // 0=グー, 1=チョキ, 2=パー
        int player = 0;    // プレイヤーの手
        int computer = 1;  // コンピューターの手

        // 手の名前を配列で管理
        String[] hands = {"グー", "チョキ", "パー"};

        System.out.println("===== じゃんけん =====");
        System.out.println("プレイヤー：" + hands[player]);
        System.out.println("コンピューター：" + hands[computer]);

        // 勝敗判定
        // (player - computer + 3) % 3 の結果：
        //   0 → あいこ
        //   1 → プレイヤーの負け
        //   2 → プレイヤーの勝ち
        int result = (player - computer + 3) % 3;

        if (result == 0) {
            System.out.println("結果：あいこです！");
        } else if (result == 2) {
            System.out.println("結果：あなたの勝ちです！");
        } else {
            System.out.println("結果：あなたの負けです...");
        }

        System.out.println("======================");
    }
}
```

### 問題7：switch文で電卓プログラム

2つの数値と演算子（`+`, `-`, `*`, `/`）を変数に格納し、switch文を使って計算結果を表示するプログラムを作成してください。0での除算エラーも処理してください。

**期待される出力例：**
```
===== 電卓 =====
計算式：15.0 / 4.0
結果：15.0 / 4.0 = 3.75
================
```

<details><summary>ヒント</summary>

- switch文は `switch (変数) { case 値: 処理; break; ... }` の構文です
- Java 14以降ではswitch式（`->` 構文）も使えます
- 除算の場合、`b == 0` のチェックを忘れずに行いましょう
- `char` 型の変数で演算子を管理できます

</details>

<details><summary>解答例</summary>

```java
public class SwitchCalculator {
    public static void main(String[] args) {
        double a = 15.0;
        double b = 4.0;
        char operator = '/'; // 演算子

        System.out.println("===== 電卓 =====");
        System.out.println("計算式：" + a + " " + operator + " " + b);

        double result;
        boolean valid = true; // 計算が有効かどうか

        // switch文で演算子に応じた計算を実行
        switch (operator) {
            case '+':
                result = a + b;
                break;
            case '-':
                result = a - b;
                break;
            case '*':
                result = a * b;
                break;
            case '/':
                // 0除算チェック
                if (b == 0) {
                    System.out.println("エラー：0で割ることはできません。");
                    valid = false;
                    result = 0; // コンパイルエラー回避用
                } else {
                    result = a / b;
                }
                break;
            default:
                System.out.println("エラー：未対応の演算子です → " + operator);
                valid = false;
                result = 0; // コンパイルエラー回避用
                break;
        }

        // 有効な計算結果を表示
        if (valid) {
            System.out.println("結果：" + a + " " + operator + " " + b + " = " + result);
        }

        System.out.println("================");
    }
}
```

### 問題8：日付妥当性チェックプログラム

年・月・日を変数に格納し、その日付が存在するかどうかを判定するプログラムを作成してください。うるう年の判定も含めてください。

**期待される出力例：**
```
===== 日付チェック =====
入力日付：2024年2月29日
うるう年：はい
判定結果：有効な日付です。
========================
```

<details><summary>ヒント</summary>

- まず月が1〜12の範囲かチェックします
- 各月の最大日数を判定します（1,3,5,7,8,10,12月は31日、4,6,9,11月は30日、2月は28日または29日）
- 2月の場合はうるう年かどうかで最大日数が変わります
- switch文を使うと月ごとの最大日数を効率的に管理できます

</details>

<details><summary>解答例</summary>

```java
public class DateValidator {
    public static void main(String[] args) {
        int year = 2024;
        int month = 2;
        int day = 29;

        System.out.println("===== 日付チェック =====");
        System.out.println("入力日付：" + year + "年" + month + "月" + day + "日");

        // うるう年判定
        boolean isLeapYear = (year % 400 == 0) || (year % 4 == 0 && year % 100 != 0);
        System.out.println("うるう年：" + (isLeapYear ? "はい" : "いいえ"));

        // 日付の妥当性チェック
        boolean isValid = true;
        String errorMessage = "";

        // 年のチェック
        if (year < 1) {
            isValid = false;
            errorMessage = "年は1以上である必要があります。";
        }
        // 月のチェック
        else if (month < 1 || month > 12) {
            isValid = false;
            errorMessage = "月は1〜12の範囲である必要があります。";
        }
        // 日のチェック
        else {
            // 各月の最大日数を求める
            int maxDay;
            switch (month) {
                case 2:
                    maxDay = isLeapYear ? 29 : 28;
                    break;
                case 4: case 6: case 9: case 11:
                    maxDay = 30;
                    break;
                default: // 1, 3, 5, 7, 8, 10, 12月
                    maxDay = 31;
                    break;
            }

            if (day < 1 || day > maxDay) {
                isValid = false;
                errorMessage = month + "月の日付は1〜" + maxDay + "日の範囲です。";
            }
        }

        // 結果表示
        if (isValid) {
            System.out.println("判定結果：有効な日付です。");
        } else {
            System.out.println("判定結果：無効な日付です。");
            System.out.println("理由：" + errorMessage);
        }

        System.out.println("========================");
    }
}
```

</details>
