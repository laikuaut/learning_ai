# 実践課題03：インデント設定マスター ★2

> **難易度**: ★★☆☆☆（基礎）
> **前提知識**: 第3章（インデントとフォーマット設定）
> **課題の種類**: ミニプロジェクト
> **学習目標**: タブとスペースに関する4つの設定（`tabstop`, `shiftwidth`, `softtabstop`, `expandtab`）を正しく理解し、言語ごとに適切なインデント設定を構成できるようになる

---

## 完成イメージ

以下の動作を実現する vimrc 設定を作成します。

```
┌──────────────────────────────────────────────────────────┐
│  インデント設定の目標                                      │
├──────────────────────────────────────────────────────────┤
│                                                           │
│  デフォルト:                                              │
│    - タブ幅 = 4スペース                                   │
│    - Tab キーでスペースを挿入（タブ文字を使わない）       │
│    - >> / << で4スペース分シフト                           │
│    - 自動インデントが有効                                 │
│                                                           │
│  動作テスト:                                              │
│    if (true) {     ← Enter を押すと...                    │
│        |           ← 自動で4スペース分インデントされる    │
│                                                           │
│    >> で右に4スペースシフト                               │
│    << で左に4スペースシフト                               │
│                                                           │
│  :retab で既存のタブ文字をスペースに変換できる            │
│                                                           │
└──────────────────────────────────────────────────────────┘
```

---

## 課題の要件

1. `tabstop` を 4 に設定する
2. `shiftwidth` を 4 に設定する
3. `softtabstop` を 4 に設定する
4. `expandtab` を有効にする（タブ文字の代わりにスペースを挿入）
5. `autoindent` を有効にする
6. `smartindent` を有効にする
7. 80文字目にカラーカラムを表示する
8. 以下のインデント操作が正しく動作することを確認する
   - Tab キーで4スペースが挿入される
   - `>>` / `<<` で4スペース分シフトされる
   - 改行時に前の行のインデントが引き継がれる
   - `:retab` でタブ文字がスペースに変換される

---

## ステップガイド

<details>
<summary>ステップ1：タブ関連の4つの設定を理解する</summary>

まず、4つの設定の役割を確認しましょう。

```
┌──────────────────────────────────────────────────────┐
│  tabstop (ts)      : タブ文字 \t の「見た目の幅」     │
│  shiftwidth (sw)   : >> / << のシフト幅              │
│  softtabstop (sts) : Tab キーを押した時の挿入幅      │
│  expandtab (et)    : Tab キーでスペースを挿入するか  │
└──────────────────────────────────────────────────────┘
```

この4つを揃えることで、一貫したインデントになります。

```vim
set tabstop=4
set shiftwidth=4
set softtabstop=4
set expandtab
```

</details>

<details>
<summary>ステップ2：自動インデントを有効にする</summary>

```vim
" 前の行のインデントを引き継ぐ
set autoindent

" 構文に応じて自動インデントする（{ の後など）
set smartindent
```

`autoindent` は単純に前の行のインデントをコピーします。`smartindent` は `{` や `}` などの構文を認識して、自動でインデントレベルを調整します。

</details>

<details>
<summary>ステップ3：カラーカラムを設定する</summary>

```vim
" 80文字目にガイドラインを表示する
set colorcolumn=80
```

1行が長くなりすぎていないかを視覚的に確認できます。

</details>

<details>
<summary>ステップ4：動作を確認する</summary>

以下の手順で確認しましょう。

1. 適当なファイルを開き、Tab キーを押して4スペースが挿入されることを確認
2. `>>` で右に4スペースシフトされることを確認
3. `if (true) {` と入力して Enter を押し、自動インデントされることを確認
4. タブ文字を含むファイルを開き、`:retab` でスペースに変換されることを確認

```vim

```

## 解答例

<details>
<summary>解答例（初心者向け）</summary>

```vim
" ==================================================
" インデント設定
" ==================================================

" タブ幅を4スペースに統一
set tabstop=4             " タブ文字の表示幅
set shiftwidth=4          " >> / << のシフト幅
set softtabstop=4         " Tab キーの挿入幅
set expandtab             " タブ文字の代わりにスペースを挿入

" 自動インデント
set autoindent            " 前の行のインデントを引き継ぐ
set smartindent           " 構文に応じて自動インデント

" 行幅のガイドライン
set colorcolumn=80        " 80文字目にカラーカラムを表示
```

</details>

<details>
<summary>解答例（改良版 ─ 細かい調整を追加）</summary>

```vim
" ==================================================
" インデント設定（改良版）
" ==================================================

" --- 基本インデント ---
set tabstop=4
set shiftwidth=4
set softtabstop=4
set expandtab

" --- 自動インデント ---
set autoindent
set smartindent

" --- シフト操作の改善 ---
set shiftround            " >> / << でインデントを shiftwidth の倍数に揃える

" --- 行幅ガイド ---
set colorcolumn=80
" カラーカラムの色を控えめにする（カラースキームによって調整）
highlight ColorColumn ctermbg=235 guibg=#2c2d27

" --- formatoptions の調整 ---
" コメント行で Enter を押した時に自動でコメント記号を挿入する設定
set formatoptions+=r      " 挿入モードで Enter を押した時
set formatoptions+=o      " ノーマルモードで o / O を押した時
set formatoptions-=t      " テキストを自動折り返ししない

" --- インデントの可視化 ---
set list
set listchars=tab:▸\ ,trail:·
```

**初心者向けとの違い:**
- `shiftround` で `>>` / `<<` の結果が `shiftwidth` の倍数に揃い、インデントの乱れを防止
- `formatoptions` の調整でコメント記号の自動挿入をコントロール
- `listchars` でタブ文字を視覚的に区別できるようにし、タブとスペースの混在を防止
- カラーカラムの色をカスタマイズして視覚的に邪魔にならないように調整

</details>
