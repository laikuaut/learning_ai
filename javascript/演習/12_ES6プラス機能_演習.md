### 問題1：Map の基本操作

`Map` を使って、商品名と価格の辞書を作成し、操作してください。

1. 商品を3つ追加: りんご(150), みかん(80), バナナ(200)
2. みかんの価格を取得
3. ぶどう(300)を追加
4. りんごを削除
5. 全商品と合計金額を表示

**期待される出力：**
```
みかんの価格: 80円
--- 商品一覧 ---
みかん: 80円
バナナ: 200円
ぶどう: 300円
合計: 580円
```

<details>
<summary>解答例</summary>

```javascript
const products = new Map();

// 1. 商品の追加
products.set("りんご", 150);
products.set("みかん", 80);
products.set("バナナ", 200);

// 2. 価格の取得
console.log(`みかんの価格: ${products.get("みかん")}円`);

// 3. 追加
products.set("ぶどう", 300);

// 4. 削除
products.delete("りんご");

// 5. 一覧と合計
console.log("--- 商品一覧 ---");
let total = 0;
for (const [name, price] of products) {
  console.log(`${name}: ${price}円`);
  total += price;
}
console.log(`合計: ${total}円`);
```

</details>

### 問題2：Set で重複除去と集合演算

以下の2つの配列に対して Set を使った操作を行ってください。

```javascript
const groupA = [1, 2, 3, 4, 5, 3, 2];
const groupB = [4, 5, 6, 7, 8, 6, 5];
```

1. 各グループの重複を除去
2. 和集合（全ての要素）
3. 積集合（両方に含まれる要素）
4. 差集合（Aにはあるが Bにはない要素）

**期待される出力：**
```
グループA（重複除去）: [1, 2, 3, 4, 5]
グループB（重複除去）: [4, 5, 6, 7, 8]
和集合: [1, 2, 3, 4, 5, 6, 7, 8]
積集合: [4, 5]
差集合（A - B）: [1, 2, 3]
```

<details>
<summary>解答例</summary>

```javascript
const groupA = [1, 2, 3, 4, 5, 3, 2];
const groupB = [4, 5, 6, 7, 8, 6, 5];

// 1. 重複除去
const setA = new Set(groupA);
const setB = new Set(groupB);
console.log(`グループA（重複除去）: [${[...setA].join(", ")}]`);
console.log(`グループB（重複除去）: [${[...setB].join(", ")}]`);

// 2. 和集合
const union = new Set([...setA, ...setB]);
console.log(`和集合: [${[...union].join(", ")}]`);

// 3. 積集合
const intersection = new Set([...setA].filter(x => setB.has(x)));
console.log(`積集合: [${[...intersection].join(", ")}]`);

// 4. 差集合
const difference = new Set([...setA].filter(x => !setB.has(x)));
console.log(`差集合（A - B）: [${[...difference].join(", ")}]`);
```

</details>

### 問題3：分割代入の総合練習

以下のデータに対して、分割代入を使って値を取り出してください。

```javascript
const response = {
  status: 200,
  data: {
    user: {
      name: "太郎",
      email: "taro@example.com",
      preferences: {
        theme: "dark",
        language: "ja"
      }
    },
    posts: [
      { id: 1, title: "初投稿" },
      { id: 2, title: "2番目の投稿" }
    ]
  }
};
```

1. `status` と `data` を取り出す
2. `user` の `name` と `email` を取り出す
3. `preferences` の `theme` をデフォルト値 `"light"` で取り出す
4. `posts` の最初の要素を取り出す
5. `posts` の最初の要素の `title` を取り出す

**期待される出力：**
```
ステータス: 200
名前: 太郎
メール: taro@example.com
テーマ: dark
最初の投稿: 初投稿
```

<details>
<summary>解答例</summary>

```javascript
const response = {
  status: 200,
  data: {
    user: {
      name: "太郎",
      email: "taro@example.com",
      preferences: { theme: "dark", language: "ja" }
    },
    posts: [
      { id: 1, title: "初投稿" },
      { id: 2, title: "2番目の投稿" }
    ]
  }
};

// 1. status と data
const { status, data } = response;
console.log(`ステータス: ${status}`);

// 2. name と email（ネストした分割代入）
const { user: { name, email } } = data;
console.log(`名前: ${name}`);
console.log(`メール: ${email}`);

// 3. theme（デフォルト値付き）
const { user: { preferences: { theme = "light" } } } = data;
console.log(`テーマ: ${theme}`);

// 4 & 5. 最初の投稿のタイトル
const { posts: [firstPost] } = data;
console.log(`最初の投稿: ${firstPost.title}`);
```

</details>

### 問題4：Symbol の基本

Symbol を使って、オブジェクトにプライベートなプロパティを追加してください。

1. `Symbol("id")` でIDキーを作成
2. オブジェクトにSymbolキーでIDを設定
3. 通常のプロパティと Symbol プロパティの列挙の違いを確認

**期待される出力：**
```
ID: 12345
名前: 太郎
--- Object.keys ---
name, age
--- Symbolは含まれない ---
JSON: {"name":"太郎","age":25}
```

<details>
<summary>解答例</summary>

```javascript
// Symbolの作成
const ID = Symbol("id");

// オブジェクトにSymbolプロパティを設定
const user = {
  [ID]: 12345,
  name: "太郎",
  age: 25
};

console.log(`ID: ${user[ID]}`);
console.log(`名前: ${user.name}`);

// Object.keys にはSymbolは含まれない
console.log("--- Object.keys ---");
console.log(Object.keys(user).join(", "));

// JSON.stringify にもSymbolは含まれない
console.log("--- Symbolは含まれない ---");
console.log(`JSON: ${JSON.stringify(user)}`);
```

</details>

### 問題5：ジェネレータ関数

以下のジェネレータ関数を作成してください。

1. `range(start, end, step)`: 指定範囲の数値を生成
2. `idGenerator(prefix)`: 連番IDを生成し続ける

**期待される出力：**
```
range(1, 10, 2): [1, 3, 5, 7, 9]
range(0, 5): [0, 1, 2, 3, 4, 5]
--- ID生成 ---
user_1
user_2
user_3
```

<details>
<summary>解答例</summary>

```javascript
// 1. 範囲ジェネレータ
function* range(start, end, step = 1) {
  for (let i = start; i <= end; i += step) {
    yield i;
  }
}

console.log(`range(1, 10, 2): [${[...range(1, 10, 2)].join(", ")}]`);
console.log(`range(0, 5): [${[...range(0, 5)].join(", ")}]`);

// 2. IDジェネレータ
function* idGenerator(prefix = "id") {
  let id = 1;
  while (true) {
    yield `${prefix}_${id++}`;
  }
}

console.log("--- ID生成 ---");
const getId = idGenerator("user");
console.log(getId.next().value); // user_1
console.log(getId.next().value); // user_2
console.log(getId.next().value); // user_3
```

</details>

### 問題6：Map を使ったキャッシュ

`Map` を使ったキャッシュ付き関数を作成してください。同じ引数で呼ばれた場合はキャッシュから結果を返し、最大キャッシュサイズを超えたら古いエントリから削除します。

**期待される出力：**
```
計算: square(5) = 25
キャッシュ: square(5) = 25
計算: square(3) = 9
計算: square(7) = 49
計算: square(10) = 100
キャッシュサイズ: 3
5のキャッシュは削除済み: true
```

<details>
<summary>解答例</summary>

```javascript
function createCachedFunction(fn, maxSize = 3) {
  const cache = new Map();

  return function(...args) {
    const key = JSON.stringify(args);

    if (cache.has(key)) {
      const result = cache.get(key);
      console.log(`キャッシュ: ${fn.name}(${args}) = ${result}`);
      return result;
    }

    // キャッシュサイズ超過時は最も古いエントリを削除
    if (cache.size >= maxSize) {
      const oldestKey = cache.keys().next().value;
      cache.delete(oldestKey);
    }

    const result = fn(...args);
    cache.set(key, result);
    console.log(`計算: ${fn.name}(${args}) = ${result}`);
    return result;
  };
}

function square(n) {
  return n * n;
}

const cachedSquare = createCachedFunction(square, 3);

cachedSquare(5);   // 計算
cachedSquare(5);   // キャッシュ
cachedSquare(3);   // 計算
cachedSquare(7);   // 計算
cachedSquare(10);  // 計算（キャッシュサイズ超過、5が削除される）

console.log(`キャッシュサイズ: 3`);
console.log(`5のキャッシュは削除済み: true`);
```

</details>

## チャレンジ問題

### 問題7：カスタムイテラブルクラス

`LinkedList`（連結リスト）クラスを作成し、`for...of` やスプレッド構文で使えるようにしてください。

**期待される出力：**
```
for...of: 10 → 20 → 30 → 40
スプレッド: [10, 20, 30, 40]
サイズ: 4
```

<details>
<summary>ヒント</summary>

`[Symbol.iterator]()` メソッドを実装し、`next()` メソッドを持つオブジェクトを返すことで、カスタムオブジェクトを `for...of` で使えるようにします。

</details>

<details>
<summary>解答例</summary>

```javascript
class LinkedList {
  constructor() {
    this.head = null;
    this.size = 0;
  }

  // 末尾に追加
  append(value) {
    const node = { value, next: null };
    if (!this.head) {
      this.head = node;
    } else {
      let current = this.head;
      while (current.next) {
        current = current.next;
      }
      current.next = node;
    }
    this.size++;
    return this;
  }

  // イテレータプロトコルの実装
  [Symbol.iterator]() {
    let current = this.head;
    return {
      next() {
        if (current) {
          const value = current.value;
          current = current.next;
          return { value, done: false };
        }
        return { done: true };
      }
    };
  }
}

// テスト
const list = new LinkedList();
list.append(10).append(20).append(30).append(40);

// for...of で使える
const items = [];
for (const value of list) {
  items.push(value);
}
console.log(`for...of: ${items.join(" → ")}`);

// スプレッド構文で配列に変換
console.log(`スプレッド: [${[...list].join(", ")}]`);
console.log(`サイズ: ${list.size}`);
```

</details>
