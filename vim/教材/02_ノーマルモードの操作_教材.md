# 第2章：ノーマルモードの操作

## この章のゴール

- カーソル移動（h/j/k/l）を使いこなせる
- 単語単位・行単位・画面単位の移動ができる
- 削除・コピー・貼り付けの基本操作ができる
- ドット（.）コマンドで操作を繰り返せる
- 数値プレフィックスの使い方を理解する

---

## 1. 基本のカーソル移動

### h / j / k / l

Vim では矢印キーの代わりに `h`, `j`, `k`, `l` でカーソルを移動します。

```
        k
        ↑
   h ←     → l
        ↓
        j

  h = 左   j = 下   k = 上   l = 右
```

なぜこの4つのキーなのでしょうか？ Vim の前身である vi が開発された当時のキーボード（ADM-3A）では、h/j/k/l のキーに矢印が刻印されていたためです。

```
ADM-3A キーボード配列（該当部分）:

  ┌───┐ ┌───┐ ┌───┐ ┌───┐
  │ h │ │ j │ │ k │ │ l │
  │ ← │ │ ↓ │ │ ↑ │ │ → │
  └───┘ └───┘ └───┘ └───┘
```

ホームポジションに手を置いたまま移動できるため、矢印キーよりも格段に効率的です。

> **よくある間違い**: 最初は矢印キーを使いたくなりますが、`hjkl` に慣れることが Vim 上達の第一歩です。矢印キーを使うと右手がホームポジションから離れてしまいます。

### ポイントまとめ

- `h`（左）、`j`（下）、`k`（上）、`l`（右）で移動します
- ホームポジションから手を離さずに移動できるのが大きな利点です
- 矢印キーも使えますが、`hjkl` に慣れることを強くおすすめします

## 2. 単語単位の移動

1文字ずつの移動だけでは効率が悪いため、Vim には単語単位で移動するコマンドが用意されています。

### w / b / e

```
  w  = 次の単語の先頭へ (word)
  b  = 前の単語の先頭へ (back)
  e  = 現在/次の単語の末尾へ (end)

  例: "Hello World Vim Editor" というテキストで

  Hello World Vim Editor
  ^                          ← カーソル位置

  w を押すと:
  Hello World Vim Editor
        ^                    ← 次の単語の先頭

  もう一度 w:
  Hello World Vim Editor
              ^              ← さらに次の単語の先頭
```

### W / B / E（WORD 単位）

大文字の `W`, `B`, `E` は空白で区切られた「WORD」単位で移動します。

```
  小文字 w: 記号も単語の区切りとみなす
  大文字 W: 空白のみを区切りとみなす

  例: "file-name.txt (2023)" で w を押していくと:

  file-name.txt (2023)
  ^
  file-name.txt (2023)
      ^                    ← "-" で区切られる
  file-name.txt (2023)
       ^
  file-name.txt (2023)
           ^               ← "." で区切られる

  W を押していくと:
  file-name.txt (2023)
  ^
  file-name.txt (2023)
                ^          ← 空白で区切られる
  file-name.txt (2023)
                 ^
```

### ポイントまとめ

- `w`/`b`/`e` で単語単位の移動ができます
- 大文字 `W`/`B`/`E` は空白区切りの WORD 単位です
- コード中では小文字と大文字を使い分けると効率的です

## 3. 行内の移動

### 行頭・行末への移動

```
  0    ← 行の先頭（0文字目）
  ^    ← 行の最初の非空白文字
  $    ← 行末

  例:
      Hello, World!
  0 → ↑              ← 行の先頭（空白含む）
  ^ →     ↑          ← 最初の非空白文字
  $ →              ↑ ← 行末
```

### 行内の文字検索

```
  f{文字}  ← 右方向に{文字}を検索してジャンプ (find)
  F{文字}  ← 左方向に{文字}を検索してジャンプ
  t{文字}  ← 右方向に{文字}の1つ手前にジャンプ (till)
  T{文字}  ← 左方向に{文字}の1つ後にジャンプ
  ;        ← 同じ方向に繰り返し検索
  ,        ← 逆方向に繰り返し検索

  例: "function getData(url, callback)" で

  function getData(url, callback)
  ^                                   ← カーソル位置

  f( を押すと:
  function getData(url, callback)
                   ^                  ← ( にジャンプ

  fa を押すと:
  function getData(url, callback)
                            ^         ← 最初の a にジャンプ
```

### ポイントまとめ

- `0`/`^`/`$` で行頭・行末に素早く移動できます
- `f`/`t` で行内の特定文字に直接ジャンプできます
- `;` と `,` で検索を繰り返せます

---

## 4. ファイル内の移動

### ファイルの先頭・末尾

```
  gg       ← ファイルの先頭へ
  G        ← ファイルの末尾へ
  {数字}G  ← 指定した行番号へ（例: 50G で50行目へ）
  :{数字}  ← 指定した行番号へ（例: :50 で50行目へ）
```

### 画面単位のスクロール

```
  Ctrl+d  ← 半画面下にスクロール (down)
  Ctrl+u  ← 半画面上にスクロール (up)
  Ctrl+f  ← 1画面下にスクロール (forward)
  Ctrl+b  ← 1画面上にスクロール (backward)
```

### 画面内の移動

```
  H  ← 画面の最上行へ (High)
  M  ← 画面の中央行へ (Middle)
  L  ← 画面の最下行へ (Low)

  ┌─────────────────────┐
  │ H → この行へ移動    │ ← 画面の最上行
  │                     │
  │                     │
  │ M → この行へ移動    │ ← 画面の中央
  │                     │
  │                     │
  │ L → この行へ移動    │ ← 画面の最下行
  └─────────────────────┘
```

### ポイントまとめ

- `gg`/`G` でファイルの先頭・末尾に移動します
- `Ctrl+d`/`Ctrl+u` で効率的にスクロールできます
- `H`/`M`/`L` で画面内の移動ができます

## 5. 削除操作

ノーマルモードでの削除操作を学びます。削除したテキストはレジスタに保存されるため、後で貼り付け（ペースト）できます。

### 基本の削除コマンド

```
  x     ← カーソル位置の1文字を削除
  X     ← カーソルの左の1文字を削除
  dd    ← 現在の行を削除
  D     ← カーソルから行末まで削除（d$ と同じ）

  dw    ← 次の単語の先頭まで削除
  db    ← 前の単語の先頭まで削除
  d$    ← 行末まで削除
  d0    ← 行頭まで削除
  d^    ← 最初の非空白文字まで削除
  dG    ← ファイル末尾まで削除
  dgg   ← ファイル先頭まで削除
```

### 削除の実例

```
  元のテキスト: "Hello, World! Goodbye!"
                 ^  ← カーソル位置

  dw を実行すると:
  ", World! Goodbye!"

  dd を実行すると:
  （行全体が削除される）

  d$ を実行すると:
  "Hello"   ← カーソルから行末まで削除
```

### ポイントまとめ

- `d` + モーション で範囲を指定して削除できます
- `dd` で行全体、`D`(=`d$`)で行末まで削除します
- 削除したテキストはレジスタに保存され、`p` で貼り付けできます

---

## 6. コピー（ヤンク）と貼り付け

Vim では「コピー」のことを「ヤンク（yank）」と呼びます。

### ヤンクコマンド

```
  yy    ← 現在の行をヤンク（コピー）
  Y     ← 現在の行をヤンク（yy と同じ）
  yw    ← 次の単語の先頭までヤンク
  y$    ← 行末までヤンク
  y0    ← 行頭までヤンク
  yG    ← ファイル末尾までヤンク
  ygg   ← ファイル先頭までヤンク
```

### 貼り付け（ペースト）

```
  p   ← カーソルの後（または下の行）に貼り付け
  P   ← カーソルの前（または上の行）に貼り付け
```

行単位でヤンクした場合は行として、文字単位でヤンクした場合は文字として貼り付けられます。

### 実践例：行の入れ替え

```
  行を入れ替えたい場合:

  1行目: BBB      ← カーソルがここにある
  2行目: AAA

  dd を実行（1行目を削除＝カット）:
  1行目: AAA      ← カーソルがここに移動

  p を実行（下に貼り付け）:
  1行目: AAA
  2行目: BBB      ← 入れ替え完了！
```

### ポイントまとめ

- `y` + モーション でヤンク（コピー）、`p`/`P` で貼り付けです
- `dd` + `p` で行の移動ができます
- 削除（`d`）したテキストも `p` で貼り付けできます（カット＆ペースト）

## 7. 変更操作

「変更（change）」は、削除して挿入モードに入る操作です。テキストを別の内容に置き換えたい時に便利です。

### 変更コマンド

```
  cc    ← 行全体を変更（行を削除して挿入モードへ）
  C     ← カーソルから行末まで変更（c$ と同じ）
  cw    ← 次の単語の先頭まで変更
  cb    ← 前の単語の先頭まで変更
  c$    ← 行末まで変更
  c0    ← 行頭まで変更
  s     ← 1文字を変更（文字を削除して挿入モードへ）
  S     ← 行全体を変更（cc と同じ）
  r{文字} ← 1文字を置き換え（挿入モードに入らない）
  R     ← 上書きモード（タイプした文字で既存の文字を上書き）
```

### 変更の実例

```
  元のテキスト: "let count = 0;"
                     ^  ← カーソルが count の c にある

  cw を実行すると:
  "let  = 0;"        ← count が削除され、挿入モードに入る
                         ここで total と入力して ESC を押すと:
  "let total = 0;"   ← 変更完了
```

### ポイントまとめ

- `c` は「削除 + 挿入モード」のショートカットです
- `cw` は変数名の変更などに非常に便利です
- `r` は1文字だけの修正に最適です（挿入モードに入りません）

---

## 8. 操作の繰り返しとアンドゥ

### ドット（.）コマンド

ドットコマンドは **直前の変更操作を繰り返す** 非常に強力なコマンドです。

```
  .   ← 直前の変更操作を繰り返す

  例: 複数の行を削除したい場合

  dd を実行（1行削除）
  .  を実行（もう1行削除）
  .  を実行（さらに1行削除）
```

```
  例: 複数の単語を同じテキストに変更したい場合

  元: "var a = 1; var b = 2; var c = 3;"

  1. カーソルを最初の var に合わせる
  2. cw と入力して let と入力し ESC
     → "let a = 1; var b = 2; var c = 3;"
  3. 次の var に移動（w w w で移動）
  4. . を押す
     → "let a = 1; let b = 2; var c = 3;"
  5. 次の var に移動
  6. . を押す
     → "let a = 1; let b = 2; let c = 3;"
```

### アンドゥとリドゥ

```
  u       ← アンドゥ（元に戻す）
  Ctrl+r  ← リドゥ（アンドゥを取り消す）
```

Vim のアンドゥは非常に強力で、操作の全履歴をツリー構造で記録しています。

### 数値プレフィックス

ほとんどのコマンドの前に数値を付けると、その回数だけ操作を繰り返せます。

```
  3dd    ← 3行削除
  5j     ← 5行下に移動
  10w    ← 10単語分進む
  2yy    ← 2行ヤンク
  4p     ← 4回貼り付け
  3dw    ← 3単語削除
  50G    ← 50行目にジャンプ
```

### ポイントまとめ

- `.` コマンドは直前の変更操作を繰り返す、Vim 最強のコマンドの1つです
- `u` でアンドゥ、`Ctrl+r` でリドゥできます
- 数値プレフィックスで操作を効率化できます（例: `3dd` で3行削除）
