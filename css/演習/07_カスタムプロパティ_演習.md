## 基本レベル（問題1〜3）

### 問題1：CSS変数を定義して使う

以下のHTMLに対して、CSS変数を使ってスタイルを書いてください。色はすべて変数で管理します。

```html
<div class="card">
  <h2 class="card-title">CSS変数入門</h2>
  <p class="card-body">CSS変数を使うと、一箇所で値を変更するだけで全体に反映できます。</p>
  <a href="#" class="card-link">詳しく見る</a>
</div>
```

**要件：**
- `:root` に以下の変数を定義：
  - `--color-primary`: `#3498db`
  - `--color-text`: `#333333`
  - `--color-text-light`: `#666666`
  - `--color-bg`: `#ffffff`
  - `--spacing-md`: `16px`
  - `--spacing-lg`: `24px`
  - `--radius`: `8px`
- `.card`：背景色 `--color-bg`、padding `--spacing-lg`、角丸 `--radius`、影 `0 2px 8px rgba(0,0,0,0.1)`、最大幅 `400px`
- `.card-title`：色 `--color-text`、margin `0 0 var(--spacing-md) 0`
- `.card-body`：色 `--color-text-light`、行の高さ `1.6`
- `.card-link`：色 `--color-primary`、下線なし、太字

<details>
<summary>ヒント</summary>

- `:root` はHTML文書のルート要素で、全体で使える変数を定義する場所
- `var(--変数名)` で変数を使用
- marginやpaddingにも変数を使える
</details>

<details>
<summary>解答例</summary>

```css
/* グローバルな変数定義 */
:root {
  --color-primary: #3498db;
  --color-text: #333333;
  --color-text-light: #666666;
  --color-bg: #ffffff;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --radius: 8px;
}

.card {
  /* 変数を使ってスタイリング */
  background-color: var(--color-bg);
  padding: var(--spacing-lg);
  border-radius: var(--radius);
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  max-width: 400px;
}

.card-title {
  color: var(--color-text);
  margin: 0 0 var(--spacing-md) 0;
}

.card-body {
  color: var(--color-text-light);
  line-height: 1.6;
}

.card-link {
  color: var(--color-primary);
  text-decoration: none;
  font-weight: bold;
}
```
</details>

### 問題2：フォールバック値を使う

以下のHTMLで、変数が未定義の場合にフォールバック値が使われるスタイルを書いてください。

```html
<div class="box box-default">デフォルト色</div>
<div class="box box-custom" style="--box-color: #e74c3c;">カスタム色</div>
<div class="box box-custom" style="--box-color: #2ecc71;">カスタム色2</div>
```

**要件：**
- `.box`：padding `20px`、文字色 白、角丸 `8px`、margin-bottom `8px`、テキスト中央揃え
  - 背景色は `var(--box-color, #3498db)` を使用（変数が未定義なら `#3498db`）
- `.box-default` には `--box-color` を設定しない（フォールバックが使われる）
- `.box-custom` にはインラインスタイルで `--box-color` が設定済み

<details>
<summary>ヒント</summary>

- `var(--変数名, フォールバック値)` の第2引数がフォールバック値
- 変数が定義されていない場合にフォールバック値が使われる
- インラインスタイルで `--box-color` を設定すると、それが優先される
</details>

<details>
<summary>解答例</summary>

```css
.box {
  padding: 20px;
  color: white;
  border-radius: 8px;
  margin-bottom: 8px;
  text-align: center;
  /* --box-color が未定義なら #3498db を使用 */
  background-color: var(--box-color, #3498db);
}

/*
 * .box-default: --box-color が設定されていないため
 *               フォールバック値の #3498db（青）が適用される
 *
 * .box-custom:  インラインスタイルで --box-color が設定されているため
 *               その値が使われる
 */
```
</details>

### 問題3：スコープを使った変数の上書き

以下のHTMLで、セクションごとに異なる配色をCSS変数のスコープで実現してください。

```html
<section class="section section-blue">
  <h2>ブルーセクション</h2>
  <p>青い配色のセクションです。</p>
  <button class="btn">ボタン</button>
</section>

<section class="section section-green">
  <h2>グリーンセクション</h2>
  <p>緑の配色のセクションです。</p>
  <button class="btn">ボタン</button>
</section>

<section class="section section-red">
  <h2>レッドセクション</h2>
  <p>赤い配色のセクションです。</p>
  <button class="btn">ボタン</button>
</section>
```

**要件：**
- `.section` 共通スタイル：padding `32px`、margin-bottom `16px`、角丸 `12px`
  - 背景色は `var(--section-bg)` を使用
  - テキスト色は `var(--section-text)` を使用
- `.btn`：背景色 `var(--section-accent)`、文字色 白、padding `10px 24px`、枠線なし、角丸 `6px`、cursor `pointer`
- `.section-blue` に変数を定義：`--section-bg: #ebf5fb`、`--section-text: #1a5276`、`--section-accent: #3498db`
- `.section-green` に変数を定義：`--section-bg: #eafaf1`、`--section-text: #1e8449`、`--section-accent: #2ecc71`
- `.section-red` に変数を定義：`--section-bg: #fdedec`、`--section-text: #922b21`、`--section-accent: #e74c3c`

<details>
<summary>ヒント</summary>

- CSS変数はスコープ（適用範囲）を持つ
- `.section-blue` で定義した変数は、その要素とその子孫要素内でのみ有効
- 共通のスタイルは1回だけ書き、色の違いは変数の値を変えるだけ
</details>

<details>
<summary>解答例</summary>

```css
/* ===== セクションごとに変数を定義 ===== */
.section-blue {
  --section-bg: #ebf5fb;
  --section-text: #1a5276;
  --section-accent: #3498db;
}

.section-green {
  --section-bg: #eafaf1;
  --section-text: #1e8449;
  --section-accent: #2ecc71;
}

.section-red {
  --section-bg: #fdedec;
  --section-text: #922b21;
  --section-accent: #e74c3c;
}

/* ===== 共通スタイル（変数を参照） ===== */
.section {
  padding: 32px;
  margin-bottom: 16px;
  border-radius: 12px;
  /* スコープ内の変数を使用 */
  background-color: var(--section-bg);
  color: var(--section-text);
}

.btn {
  /* セクションごとに異なるアクセント色が適用される */
  background-color: var(--section-accent);
  color: white;
  padding: 10px 24px;
  border: none;
  border-radius: 6px;
  cursor: pointer;
}
```
</details>

### 問題4：ダーク/ライトテーマ切り替え

以下のHTMLで、bodyのクラスを変えるだけでテーマが切り替わる設計を作ってください。

```html
<body class="theme-light">
  <div class="app">
    <header class="app-header">
      <h1>テーマ切り替えアプリ</h1>
    </header>
    <main class="app-main">
      <div class="content-card">
        <h2>コンテンツカード</h2>
        <p>このカードのスタイルはCSS変数で制御されています。</p>
      </div>
      <div class="content-card">
        <h2>もう一つのカード</h2>
        <p>テーマを変更すると、すべてのカードが一斉に変わります。</p>
      </div>
    </main>
  </div>
</body>
```

**要件：**
- `.theme-light` に以下の変数を定義：
  - `--bg-primary`: `#ffffff`、`--bg-secondary`: `#f5f6fa`
  - `--text-primary`: `#2c3e50`、`--text-secondary`: `#666666`
  - `--border-color`: `#e0e0e0`
  - `--shadow`: `0 2px 8px rgba(0, 0, 0, 0.08)`
  - `--accent`: `#3498db`
- `.theme-dark` に以下の変数を定義：
  - `--bg-primary`: `#1a1a2e`、`--bg-secondary`: `#16213e`
  - `--text-primary`: `#e0e0e0`、`--text-secondary`: `#a0a0a0`
  - `--border-color`: `#2a2a4a`
  - `--shadow`: `0 2px 8px rgba(0, 0, 0, 0.3)`
  - `--accent`: `#64b5f6`
- `.app`：背景色 `--bg-secondary`、色 `--text-primary`、最小高さ `100vh`
- `.app-header`：背景色 `--bg-primary`、padding `16px 24px`、枠線下 `1px solid var(--border-color)`
- `.app-main`：padding `24px`、Gridレイアウト、`repeat(auto-fit, minmax(300px, 1fr))`、間隔 `16px`
- `.content-card`：背景色 `--bg-primary`、padding `24px`、角丸 `12px`、影 `--shadow`、枠線 `1px solid var(--border-color)`
  - `h2`：色 `--accent`
  - `p`：色 `--text-secondary`

<details>
<summary>ヒント</summary>

- 2つのテーマクラスで同じ変数名に異なる値を定義
- bodyのクラスを `theme-light` → `theme-dark` に変えるだけで全体が切り替わる
- 色を直接書かず、すべて `var()` を使うのがポイント
</details>

<details>
<summary>解答例</summary>

```css
/* ===== ライトテーマの変数 ===== */
.theme-light {
  --bg-primary: #ffffff;
  --bg-secondary: #f5f6fa;
  --text-primary: #2c3e50;
  --text-secondary: #666666;
  --border-color: #e0e0e0;
  --shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
  --accent: #3498db;
}

/* ===== ダークテーマの変数 ===== */
.theme-dark {
  --bg-primary: #1a1a2e;
  --bg-secondary: #16213e;
  --text-primary: #e0e0e0;
  --text-secondary: #a0a0a0;
  --border-color: #2a2a4a;
  --shadow: 0 2px 8px rgba(0, 0, 0, 0.3);
  --accent: #64b5f6;
}

/* ===== 共通スタイル（変数のみ参照） ===== */
.app {
  background-color: var(--bg-secondary);
  color: var(--text-primary);
  min-height: 100vh;
}

.app-header {
  background-color: var(--bg-primary);
  padding: 16px 24px;
  border-bottom: 1px solid var(--border-color);
}

.app-main {
  padding: 24px;
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 16px;
}

.content-card {
  background-color: var(--bg-primary);
  padding: 24px;
  border-radius: 12px;
  box-shadow: var(--shadow);
  border: 1px solid var(--border-color);
}

.content-card h2 {
  color: var(--accent);
  margin-top: 0;
}

.content-card p {
  color: var(--text-secondary);
  line-height: 1.6;
}
```
</details>

### 問題5：calc()との組み合わせ

以下のHTMLで、CSS変数と `calc()` を組み合わせて一貫したスペーシングシステムを作ってください。

```html
<div class="spacing-demo">
  <div class="space-box space-xs">XS (4px)</div>
  <div class="space-box space-sm">SM (8px)</div>
  <div class="space-box space-md">MD (16px)</div>
  <div class="space-box space-lg">LG (32px)</div>
  <div class="space-box space-xl">XL (64px)</div>
</div>
```

**要件：**
- `:root` に基準値を定義：`--space-unit: 4px`
- 各サイズの変数を `calc()` で定義：
  - `--space-xs`: `var(--space-unit)`（4px）
  - `--space-sm`: `calc(var(--space-unit) * 2)`（8px）
  - `--space-md`: `calc(var(--space-unit) * 4)`（16px）
  - `--space-lg`: `calc(var(--space-unit) * 8)`（32px）
  - `--space-xl`: `calc(var(--space-unit) * 16)`（64px）
- `.spacing-demo`：padding `var(--space-lg)`、Flexbox、縦方向、間隔 `var(--space-sm)`
- `.space-box`：背景色 `#3498db`、文字色 白、角丸 `4px`
  - 各 `.space-*` クラスで padding にそれぞれの変数を使用

<details>
<summary>ヒント</summary>

- `calc()` でCSS変数を使って計算できる
- 基準値を1つ変えるだけで全スケールが変わる設計
- 例：`--space-unit` を `4px` → `5px` にすると全サイズが変わる
</details>

<details>
<summary>解答例</summary>

```css
:root {
  /* 基準となるスペース単位 */
  --space-unit: 4px;

  /* calc() で基準値から各サイズを計算 */
  --space-xs: var(--space-unit);                    /* 4px */
  --space-sm: calc(var(--space-unit) * 2);          /* 8px */
  --space-md: calc(var(--space-unit) * 4);          /* 16px */
  --space-lg: calc(var(--space-unit) * 8);          /* 32px */
  --space-xl: calc(var(--space-unit) * 16);         /* 64px */
}

.spacing-demo {
  padding: var(--space-lg);
  display: flex;
  flex-direction: column;
  gap: var(--space-sm);
}

.space-box {
  background-color: #3498db;
  color: white;
  border-radius: 4px;
}

/* 各サイズのpaddingを変数で指定 */
.space-xs { padding: var(--space-xs); }
.space-sm { padding: var(--space-sm); }
.space-md { padding: var(--space-md); }
.space-lg { padding: var(--space-lg); }
.space-xl { padding: var(--space-xl); }
```
</details>

### 問題6：CSS変数でコンポーネントのバリエーション

以下のHTMLで、CSS変数を使って1つのクラスから複数のボタンバリエーションを作ってください。

```html
<button class="var-btn">デフォルト</button>
<button class="var-btn var-btn-danger">削除</button>
<button class="var-btn var-btn-success">保存</button>
<button class="var-btn var-btn-outline">詳細</button>
<button class="var-btn var-btn-lg">大きいボタン</button>
<button class="var-btn var-btn-sm">小さい</button>
```

**要件：**
- `.var-btn` のデフォルト変数：
  - `--btn-bg`: `#3498db`、`--btn-color`: `#ffffff`
  - `--btn-border`: `transparent`
  - `--btn-padding-y`: `10px`、`--btn-padding-x`: `20px`
  - `--btn-radius`: `8px`、`--btn-font-size`: `15px`
- `.var-btn` のスタイル：すべて変数を使って適用、cursor `pointer`、枠線 `2px solid var(--btn-border)`
  - ホバー時：`filter: brightness(0.9)`
- バリエーション（変数の値を上書きするだけ）：
  - `.var-btn-danger`：`--btn-bg: #e74c3c`
  - `.var-btn-success`：`--btn-bg: #2ecc71`
  - `.var-btn-outline`：`--btn-bg: transparent`、`--btn-color: #3498db`、`--btn-border: #3498db`
  - `.var-btn-lg`：`--btn-padding-y: 16px`、`--btn-padding-x: 32px`、`--btn-font-size: 18px`
  - `.var-btn-sm`：`--btn-padding-y: 6px`、`--btn-padding-x: 12px`、`--btn-font-size: 12px`

<details>
<summary>ヒント</summary>

- `.var-btn` でデフォルト変数とスタイルを定義
- バリエーションクラスでは変数の値を上書きするだけ
- `filter: brightness(0.9)` で少し暗くする（どの色でも動作する汎用的なホバー効果）
</details>

<details>
<summary>解答例</summary>

```css
.var-btn {
  /* デフォルトの変数値 */
  --btn-bg: #3498db;
  --btn-color: #ffffff;
  --btn-border: transparent;
  --btn-padding-y: 10px;
  --btn-padding-x: 20px;
  --btn-radius: 8px;
  --btn-font-size: 15px;

  /* 変数を使ってスタイリング */
  background-color: var(--btn-bg);
  color: var(--btn-color);
  border: 2px solid var(--btn-border);
  padding: var(--btn-padding-y) var(--btn-padding-x);
  border-radius: var(--btn-radius);
  font-size: var(--btn-font-size);
  cursor: pointer;
}

.var-btn:hover {
  /* どの色でも使える汎用的なホバー効果 */
  filter: brightness(0.9);
}

/* ===== バリエーション（変数の上書きだけ） ===== */
.var-btn-danger {
  --btn-bg: #e74c3c;
}

.var-btn-success {
  --btn-bg: #2ecc71;
}

.var-btn-outline {
  --btn-bg: transparent;
  --btn-color: #3498db;
  --btn-border: #3498db;
}

.var-btn-lg {
  --btn-padding-y: 16px;
  --btn-padding-x: 32px;
  --btn-font-size: 18px;
}

.var-btn-sm {
  --btn-padding-y: 6px;
  --btn-padding-x: 12px;
  --btn-font-size: 12px;
}
```
</details>

### 問題7：prefers-color-schemeとCSS変数

以下のHTMLで、OSのダーク/ライトモード設定を自動検出してテーマが切り替わるスタイルを書いてください。

```html
<div class="auto-theme-card">
  <h2>自動テーマ検出</h2>
  <p>OSの設定に応じて、自動的にライト/ダークテーマが適用されます。</p>
  <span class="theme-indicator">現在のテーマ: 自動検出</span>
</div>
```

**要件：**
- `:root` にライトテーマ変数をデフォルトで定義
- `@media (prefers-color-scheme: dark)` 内で `:root` の変数をダーク用に上書き
- ライト変数：`--auto-bg: #ffffff`、`--auto-text: #333`、`--auto-accent: #3498db`、`--auto-card-bg: #f8f9fa`、`--auto-border: #e0e0e0`
- ダーク変数：`--auto-bg: #121212`、`--auto-text: #e0e0e0`、`--auto-accent: #64b5f6`、`--auto-card-bg: #1e1e1e`、`--auto-border: #333`
- `body`：背景色 `--auto-bg`、色 `--auto-text`
- `.auto-theme-card`：背景色 `--auto-card-bg`、枠線 `1px solid var(--auto-border)`、padding `24px`、角丸 `12px`、最大幅 `500px`、margin `40px auto`
- `.theme-indicator`：背景色 `--auto-accent`、文字色 白、padding `4px 12px`、角丸 `4px`、文字サイズ `13px`

<details>
<summary>ヒント</summary>

- `@media (prefers-color-scheme: dark)` はOSのダークモード設定を検出
- この中で `:root` の変数を上書きするだけで、全体のテーマが切り替わる
- JavaScriptは不要で、CSSだけで自動検出できる
</details>

<details>
<summary>解答例</summary>

```css
/* ===== デフォルト（ライトテーマ） ===== */
:root {
  --auto-bg: #ffffff;
  --auto-text: #333;
  --auto-accent: #3498db;
  --auto-card-bg: #f8f9fa;
  --auto-border: #e0e0e0;
}

/* ===== OSがダークモードの場合、変数を上書き ===== */
@media (prefers-color-scheme: dark) {
  :root {
    --auto-bg: #121212;
    --auto-text: #e0e0e0;
    --auto-accent: #64b5f6;
    --auto-card-bg: #1e1e1e;
    --auto-border: #333;
  }
}

/* ===== スタイル（変数のみ参照） ===== */
body {
  background-color: var(--auto-bg);
  color: var(--auto-text);
  font-family: sans-serif;
  margin: 0;
  padding: 20px;
}

.auto-theme-card {
  background-color: var(--auto-card-bg);
  border: 1px solid var(--auto-border);
  padding: 24px;
  border-radius: 12px;
  max-width: 500px;
  margin: 40px auto;
}

.auto-theme-card h2 {
  margin-top: 0;
  color: var(--auto-accent);
}

.theme-indicator {
  display: inline-block;
  background-color: var(--auto-accent);
  color: white;
  padding: 4px 12px;
  border-radius: 4px;
  font-size: 13px;
}
```
</details>

### 問題8：ミニプロジェクト ─ デザインシステムのトークン定義

以下のHTMLで、デザインシステムの基盤となるCSS変数体系（デザイントークン）を構築してください。

- 命名規則を統一する（接頭辞 `--ds-`）と管理しやすい
